# Contributing — 山河中国

> 给人或 AI agent：改这个项目前先读完。

## TL;DR

```bash
npm install
npm run build:dem            # 首次必跑：生成秦岭切片资产
npm run dev                  # vite dev server
npm run test:ci              # 改代码前后必跑 (typecheck + fast tests)
```

## 改代码该跑哪组测试

| 改动类型 | 必跑命令 | 时长 |
|---|---|---|
| 普通 TS 改动（util / 单 module）| `npm run test:ci` | ~0.5s |
| 地形 / 水系 / atlas 改动 | `npm run test:visual && npm run test:data` | ~37s |
| 音频 / mixer / sparse scheduler | `npm run test:audio` | ~0.1s |
| Player / camera / mount | `npm run test:visual` | ~36s |
| 准备 commit | `npm test` | ~38s |
| 改 `qinlingRegion.js` / manifest.json | `npm run test:fast`（含 ssot-drift 守卫） | ~0.2s |

## 文件位置约定

| 内容 | 位置 |
|---|---|
| 新 module（runtime 代码） | `src/game/Xxx.ts` |
| 静态 authoring data | `src/data/*.ts` |
| Auto-generated data | `src/game/data/*.js` + 配套 `scripts/build-*.mjs` |
| 测试 | `scripts/Xxx.test.mjs`（S6 后可能迁 `tests/`） |
| Build script | `scripts/build-*.mjs` |
| Fetch script | `scripts/download-*.mjs` 或 `fetch-*.mjs` |
| Verify script | `scripts/verify-*.mjs` |
| Active 文档 | `docs/*.md` |
| 归档文档 | `.archive/docs/*.md` |
| Runtime artifact | `public/data/*`（不要手改，build script 生成） |

## 命名约定

- `src/**`：`camelCase.ts`
- `scripts/**`：`kebab-case.mjs`
- `public/data/**`：`kebab-case.json` / `.bin`
- Generated 文件：顶部必须有 `// AUTO-GENERATED by scripts/build-xxx.mjs — 不要手改`
- `.js` shim：与 `.ts` 同名（不加 `.shim.js` 后缀）

## SSOT-by-design

写代码前问：**这个事实已经在哪里 author？**
- 已有 → import / derive；不要新建副本
- 新事实 → 一个 authoring source；其他文件 import

详见 [CLAUDE.md](./CLAUDE.md) SSOT 一节。违反前问 user。

## 加新 test 文件

1. 写 `scripts/yourThing.test.mjs`
2. 在 `scripts/test-groups.mjs` 对应 group 数组里 append `"yourThing"`（不带 `.test.mjs` 后缀）
3. 跑 `node scripts/test-groups.mjs <group>` 验证 picks up

不要直接改 `package.json` 里 test 命令。

## 加新 npm 路径

如果加 `public/data/*` 文件 path 引用：

1. 先在 `scripts/data-paths.mjs` 里加 export
2. build/test/runtime 各处 import
3. 不要直接 `path.resolve("public/data/...")` 硬编码

## 改 region bounds / world size

唯一 authoring source: `src/data/qinlingRegion.js`。改完：

1. `npm run build:dem` 重新生成 manifest
2. `npm run test:fast`（含 `ssot-drift.test.mjs` 验证 manifest sync）
3. 不要手改 `public/data/regions/qinling/manifest.json`——它是 build artifact
