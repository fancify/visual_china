## 目录索引

### Tests (~80 files, `*.test.mjs`)

测试组织见 `test-groups.mjs`。改/加测试：

1. 写 `Xxx.test.mjs`
2. 在 `test-groups.mjs` 对应 group append `"Xxx"`
3. `node scripts/test-groups.mjs <group>` 验证

Group 划分：
- **fast** — 契约 / 数学 / 状态机 / SSOT 守卫，<500ms 总
- **visual** — 渲染 / 几何 / runtime mock scene，~36s
- **data** — DEM / POI / atlas validation，~1s
- **audio** — 音频子系统，~100ms

### Build scripts (`build-*.mjs`)

- `build-qinling-real-dem.mjs` — FABDEM → slice DEM master
- `build-qinling-region-assets.mjs` — slice → region manifest + chunks
- `build-qinling-hydrography-report.mjs` — DEM vs hydrography 一致性报告
- `build-qinling-hydrography-from-ne.mjs` — Natural Earth 河流 → hydrography asset
- `build-qinling-osm-hydrography.mjs` — OSM Overpass → hydrography asset
- `build-qinling-primary-hydrography.mjs` — primary 水系数据生成
- `build-qinling-route-anchors.mjs` — historical-anchor → route anchors data
- `build-shu-road-paths.mjs` — anchors → 3D-ready path geometry
- `build-china-national-dem.mjs` — 全国 FABDEM 拼合 (~107MB binary)
- `build-china-major-rivers.mjs` / `build-china-major-lakes.mjs` — NE major water
- `build-china-lowres-dem.mjs` — ETOPO 低分辨率全国 DEM
