千里江山图 · 全国主干河湖原型 v3

数据：Natural Earth 10m rivers + lakes (public domain) · 选择：手工评分（干支级别 + 历史意义） · 匹配按 NE name 严格相等 + bbox 消歧 · NE 收不进的小湖（洱海/滇池/天池）用 fallback disk
河流评分：score = (4-trunk) + area + story
• trunk：1=干流 / 2=一级支 / 3=二级支
• area：5=>1M km² / 4=>100k / 3=>10k / 2=>1k / 1=<1k
• story：0-5（华夏文明源/朝代核心/南北分界/重大战事/漕运/地方文化）

湖泊评分：score = (4-trunk)*3 + story
• trunk：1=国家级（五大淡水湖/高原圣湖/边境民族）/ 2=区域
• story：0-5（同河流维度）

线宽：score 12+=3px，10-11=2px，7-9=1.4px，<7=1px。
颜色：青长江·蓝黄河·绿珠/南方·银白雅澜怒·浅蓝塞北 / 湖泊统一青蓝。
湖 NE 与 disk：disk = NE 10m 收不进的小湖（<500 km²），按已知中心坐标和半径生成的圆形近似。
