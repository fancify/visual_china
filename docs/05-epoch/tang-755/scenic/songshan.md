# 嵩山

## 一句话

五岳中岳，唐河南府登封县境；周人称嵩高，汉武立太室祠——武周登封以后，山名便带着一层天中帝气。

## 形貌

嵩山东西七十里，横亘洛阳东南——伊洛盆地南缘第一道屏障。山分两脉：东曰**太室**，西曰**少室**——两山相距约二十里，中间一段山谷，唐人谓之"嵩岳"。太室山主峰峻极峰一千四百九十二米，少室山主峰连天峰一千五百一十二米——少室略高、太室略尊。两山皆石灰岩，山势不险，唯峻极、连天两峰岩壁陡立；山腹缓阔，多松柏、栎、栾，秋来漫坡赤褐。

自洛阳定鼎门东南行九十里便到山下登封县——县城正卡在太室、少室之间的山口，南北望皆见峰岭。中岳庙在太室南麓、登封县东八里——汉武帝元封元年（前 110）所立"太室祠"的旧基。少林寺在少室北麓、登封县西十三里——北魏孝文帝太和十九年（495）为天竺僧跋陀立。嵩阳观在太室南麓、中岳庙西北——隋改自北魏嵩阳寺。嵩岳寺塔孤立于太室西南山脚——北魏皇室离宫旧址。这几处皆在五十里之内，从太室到少室一日可遍历。

物候是嵩山的一笔。春三月山桃开遍峻极峰下，五月卢崖瀑布水盛——伏雨季嵩岳云烟尤胜，唐人写"嵩云"便是此景。秋来栎叶赤褐、栾子悬枝，冬日太室北坡积雪不化——洛阳人晴日北望，能见嵩山一线雪痕。山中泉眼极多，少林寺前的卓锡泉、中岳庙前的玉案池、嵩阳观的将军柏下井——皆是终年不涸的。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

嵩山在五岳之中坐"中"——这地位远早于唐。

《尚书·禹贡》"外方"即指嵩山，《诗·大雅·崧高》"崧高维岳，骏极于天"——周人把它称作"嵩高"。**汉武帝元封元年（前 110）东巡，登嵩山**——闻"万岁"三呼之声，遂立"太室祠"，加山高三百户为"奉邑"——这是嵩山作"中岳"地位的国家级定立。东汉光武建武中元元年（56）刻"开母庙石阙"、"少室石阙"、"太室石阙"——汉三阙今存于山下，是中国现存最早的庙阙实物。

北魏迁洛（494）后大兴佛事。太和十九年（495）孝文帝为西天僧跋陀立**少林寺**于少室山阴——以其立于"少室之林"得名。**嵩岳寺**亦建于此期——一说北魏正光元年（520）宣武帝离宫改寺，一说永平二年（509）始建；寺中**嵩岳寺塔年代有正光元年（520）、正光四年（523）两说并存**——河南省文物局采 523，学界通行旧说作 520，塔上无纪年铭故未能定论。塔形**密檐十二角十五层、砖造**——是中国现存最早的砖塔之一。北魏一朝，嵩山从一座"五岳之祠所在"加上一层"中原佛教初地"。

隋开皇年间，嵩阳寺改**嵩阳观**——道家收回此地。大业末年王世充据洛阳，王世充侄王仁则屯兵少室山下的轘州（柏谷坞）——少林寺正在轘州东侧。**唐武德四年（621）四月，少林寺僧志操、惠玚、昙宗等十三人协助秦王李世民部擒王仁则、献于唐军**——次年（622）秦王赐少林寺田四十顷、水碾一具、嘉奖立碑。这便是后世"**十三棍僧救唐王**"传说的史源——**但开元十一年（723）裴漼撰《少林寺碑》原文只载擒仁则、赐田，并无"棍僧"、"救李世民"、"十三人格斗御林军"诸节**——"棍僧救唐王"的完整故事，是明代《少林寺志》以下渐渐附会的事。

太宗朝起，少林、中岳庙、嵩阳观三处皆受官方倚重。**贞观十一年（637）太宗令置玄奘别舍于少林寺——但玄奘出关、归长安、入玉华，终其一生未居少林**；玄奘曾屡上表请赴少林译经，太宗、高宗皆不许，史中明文。**至于"玄奘晚年居中岳"，是后世误传，唐史不支持**——玄奘麟德元年（664）卒于玉华寺，距嵩山尚远。

武周一朝，嵩山地位骤升至顶。武则天自垂拱年间便屡赴嵩山——她不喜远赴泰山，更愿就近封禅。《资治通鉴》卷二〇五：**万岁通天元年腊月甲申（农历，公历 696 年 1 月），武则天登嵩岳、封神岳天中王，改元"万岁登封"**——五岳之中头一个被封"王"。改阳城县为"登封县"、嵩阳县为"告成县"——**这是唐代唯一一次嵩山封禅，也是中国历史上唯一一次女主封禅**。次年（万岁登封元年）二月辛巳，加尊神岳天中王为"神岳天中黄帝"、夫人为"灵妃"——把嵩山的神格从"王"升到"帝"。

玄宗朝嵩山复入常制。开元十三年（725）玄宗封禅泰山，于山下一并下诏加五岳爵号，封中岳为"中天王"——但**玄宗本人未亲至嵩山**，只是诏书所及。开元年间中岳庙大修——殿宇阶碑多自此时新立。少林寺在玄宗朝也兴：开元十一年（723）河南尹裴漼撰《少林寺碑》——这是少林寺现存最早的官方记述，把"武德赐田"事刻于石上、永为存证。嵩阳观也存——唐人把它作五岳之中道教重地，与南岳衡山、北岳恒山三观并列。

### 此刻

天宝十四载（755）的嵩山，离洛阳九十里——快马半日可至。山下登封县是河南府所辖之县——**这"登封"之名，三十年前还叫"阳城"，是武则天封禅那一年改的，至今未变**。盛唐人写"嵩岳"、写"中岳"、写"登封"，背后皆默认这层来历。

山中三处大寺道观皆正盛：少林寺禅宗北宗渐兴——神秀（卒 706）一系的法嗣此时多在洛阳、长安二京宫禁讲法，少林本山反而冷清，但寺院规模与田产仍在；中岳庙是国家祭祀正典——盛唐"五岳常祀"由太常寺主，每年遣使祭中岳；嵩阳观为河南道道教重地——观中数株汉武帝东巡时手植的柏，唐人称"将军柏"，玄宗朝犹存。

文学上嵩山是盛唐诗人的近郊山。李白开元二十二年（734）于嵩山东麓元丹丘的颍阳山居住过——《将进酒》"岑夫子，丹丘生，将进酒，杯莫停"中的"丹丘生"便是元丹丘；李白另有《送杨山人归嵩山》《赠嵩山焦炼师》数首咏嵩。王维 745 年前后亦有《送方尊师归嵩山》——盛唐第一流诗人，多有嵩岳之作。（注：杜甫《望岳》三首通行作东岳泰山、南岳衡山、西岳华山，中岳嵩山一首之说不立。）

朝廷的目光此刻在范阳。距嵩山东北一千二百里的范阳郡，安禄山的大军正在校阅——但嵩山道士、少林禅僧、中岳庙祝皆未知。山下登封县仍照常向洛阳输粮、向中岳庙输祭——这是嵩山的最后一个"无事之秋"。

## 子条目

### 太室山

嵩山东脉，**主峰峻极峰一千四百九十二米**。"太室"之名出《尚书·禹贡》——古以山有石室、可居神，故称室。汉武立太室祠便在此山南麓。山势浑厚、坡度缓阔——盛唐人登嵩多从峻极峰路上，路上多松柏、有泉。山中有逍遥谷、卢崖瀑布、君召口诸处。中岳庙、嵩阳观、嵩岳寺塔皆在太室南麓与西南麓——**唐人"登中岳"多指登太室**。

### 少室山

嵩山西脉，**主峰连天峰一千五百一十二米**，比太室略高。"少室"之名亦出《禹贡》——因其相对太室为少（小）也。少室峰岭比太室峻——其北麓即少林寺所在。山中有水帘洞、二祖庵（注：二祖庵之名据传起于禅宗二祖慧可断臂典故，但二祖庵建置年代不可考，盛唐时是否已立无文献支撑）、卓锡泉。**唐人"登少室"则多从少林寺起步**——禅宗禅僧入山参禅之路。

### 中岳庙

太室山南麓、登封县东八里。**汉武帝元封元年（前 110）东巡时立"太室祠"——此地为始**。北魏太延元年（435）扩建、唐玄宗开元十八年（730）大修——殿、阙、廊、亭已成规模。**唐代中岳庙是五岳常祀之首**——每年由太常寺遣使来祭，祭日选在四季的"立"日（立春、立夏、立秋、立冬），与京中告祭。武周万岁登封元年（696）武则天封中岳为"神岳天中王"、后又封"天中黄帝"——这些封号册命的原件即藏于此庙。庙前有汉太室阙、东汉中元元年（56）所立——今存。庙中柏木亦多为唐代以前手植。

### 少林寺

少室山北麓、登封县西十三里。**北魏孝文帝太和十九年（495）为西天僧跋陀立**——以立于"少室之林"得名。北魏宣武帝时印度僧菩提达摩据传亦至少林——但"达摩面壁九年"故事最早见于《景德传灯录》（北宋景德元年 1004 道原撰），盛唐之前的传记中并无此节。**慧可断臂"立雪"之事亦同——见于《景德传灯录》以下，盛唐少林寺碑无载**。

寺中真正的盛唐核心事是**武德四年（621）四月十三日僧志操、惠玚、昙宗等十三人擒王仁则降唐——次年秦王赐田四十顷、水碾一具**。开元十一年（723）河南尹裴漼撰《少林寺碑》刻此事于石——碑今存少林寺。**这是"十三棍僧救唐王"传说的全部史源**：擒一叛将、献功受赐，无"棍僧"、无"格斗御林军"、无救李世民本人——后世故事，明清渐附。

755 年的少林寺寺众千余，禅宗北宗（神秀一系）多由此辐射两京。寺中田产、水碾、磨坊在山下登封县境——是盛唐少林寺经济基础。寺塔林（今存唐至清塔二百四十余）盛唐时已有数座唐塔。

### 嵩岳寺塔

太室山西南麓。**北魏正光年间所造**——具体年代有正光元年（520）、正光四年（523）两说并存：学界旧说取 520，河南省文物局取 523；塔上无纪年铭故未能定论。塔形**密檐十二角十五层、砖造**——周长三十六米、通高约三十八米。**这是中国现存最早的砖塔之一**，也是现存唯一一座十二角形古塔——盛唐之时已立两百三十余年。嵩岳寺本是北魏宣武帝离宫改寺——隋大业末年寺毁、塔独存，唐代仍称嵩岳寺塔但寺院已大幅缩减。唐人过嵩岳南山，可见此塔孤立松柏间——日影之中，是北魏的剪影。

### 嵩阳观

太室山南麓、中岳庙西北。**北魏太和八年（484）始建为嵩阳寺**——隋大业八年（612）改嵩阳观，唐代为道教重地。观中有两株古柏——汉武帝元封元年（前 110）东巡时手植，封"大将军"、"二将军"——故称"将军柏"。盛唐时柏龄已逾八百年。唐玄宗开元年间道教鼎盛，嵩阳观为河南道道众朝集之所——道士司马承祯曾居此观（注：司马承祯一生历居天台、王屋、嵩阳诸山，嵩阳为其晚年居所之一，开元十五年 727 卒于王屋山）。**至北宋至道二年（996）赐名"太室书院"、景祐二年（1035）改"嵩阳书院"——成北宋四大书院之一**。但这是唐之后的事；盛唐之时，此地是一座道观。

### 法王寺

太室山玉柱峰南麓。**东汉永平十四年（71）始建——传为中国最早的佛寺之一**（与白马寺并称"佛教传入华夏后最早一批寺院"）。北魏改名"法王寺"——意为"佛"。寺中有北魏砖塔一座、唐塔数座——唐塔多为高僧墓塔。盛唐时法王寺规模虽不及少林，但因其传"东土最古"之名，仍受唐人重视。

### 会善寺

太室山积翠峰下。**北魏孝文帝时（约 471-499）始建**——原为离宫，后改寺。隋大业年间赐名"会善寺"。**唐高宗永淳二年（683）武则天敕赐为"道场"**——武周一朝大兴。寺中有北齐天保八年（557）所造石室、唐代"净藏禅师塔"（开元二十四年 736 所造，今存）——是盛唐嵩山禅僧墓塔的代表作。唐代天文学家僧一行少年时曾在会善寺师事普寂——开元九年（721）受玄宗命修《大衍历》。**会善寺因此是盛唐天文学的一处源头**。

### 武周封中岳：从"王"到"帝"

**这是中国历史上唯一一次女主亲行封禅**——选在嵩山而非泰山，因嵩山近神都洛阳、且"中岳"取"天中"之名，与武周"周虽旧邦其命维新"的政治意象正相合。

## 历史长河（755 → 今）

**中晚唐**　至德、乾元间兵火（755-762）嵩山未直接陷于叛军——但中岳常祀因两京残破而辍。少林寺亦受波及——田产、水碾在战乱中或损或失。建中四年（783）李希烈据襄阳叛唐，关东一时震动——少林寺南境再次紧张。中晚唐禅宗渐转——北宗（神秀一系，少林为本山）日衰，南宗（慧能一系，岭南曹溪为本山）大兴；少林寺在禅宗谱系中的地位渐被江南诸寺取代。会昌五年（845）武宗灭佛，少林寺、嵩岳寺、法王寺、会善寺皆毁——拆毁殿宇、僧众还俗。次年（846）宣宗复佛，诸寺渐复，但元气已伤。**盛唐嵩山的"少林田、中岳祀、嵩阳道、嵩岳塔"四盛景中，至晚唐只塔（嵩岳寺塔）和庙（中岳庙）大体保存**——其余皆有破有补。

**五代至宋**　五代乱世嵩山诸寺再衰。少林寺至**北宋开宝七年（974）始得重修**——宋廷赐田、补僧。北宋一朝嵩山反成中央关注之地：**至道二年（996）太宗赐名"太室书院"、景祐二年（1035）仁宗改"嵩阳书院"**——嵩阳观由道入儒，成北宋四大书院之一（与岳麓、白鹿洞、应天府书院并称）。司马光、范仲淹、二程皆讲学于此——程颢、程颐之"洛学"，根脚便扎在嵩阳书院。盛唐的道观之地，到北宋成了理学之源——**山未变，学术之主人换了**。元至元年间少林寺复立，元代福裕禅师（卒 1275）重整少林宗法、置"曹洞、临济"两宗常住——少林寺武术的早期记录始见于此期。嵩岳寺塔在五代北宋间已显残破——但因孤立无人扰，幸存。

**明清**　明初洪武年间少林寺再修，置"僧兵"为山中常备——这是明代少林武术系统化的开端。嘉靖年间（1522-1566）少林僧兵数次随官军南下御倭——少林武术从山中走入战场。明末少林寺众千余、有名拳谱。清初顺治、康熙、雍正三朝对少林僧兵颇有戒备——传雍正十三年（1735）曾遣兵围山，但史载不实；少林寺武术此后转入民间秘传。乾隆四十一年（1776），陕西毕沅修《关中胜迹图志》、河南诸学者亦在嵩山遍立碑——能辨识的多是唐宋遗迹。**距武则天封中岳已逾千年**——但中岳庙、少林寺、嵩岳寺塔仍在原位，是嵩山的稳。明清两代，嵩山从国家祭祀正典渐成地方香火——五岳常祀虽在制度上保留，实际祭礼已多由地方官代行。

**民国至今**　1928 年军阀石友三据少林寺、与樊钟秀部交战——纵火焚毁少林寺主体殿宇、藏经阁、钟楼、鼓楼——**盛唐至明清的一千四百年寺院主体在一夜之间烬**。仅余山门、立雪亭、千佛殿等少数建筑及塔林二百四十余座。1973 年起嵩岳寺塔大修——这是该塔自北魏正光元年（520）以来第一次系统修缮，距其立成已一千四百五十年。1981 年起少林寺重修，1982 年电影《少林寺》出，少林武术成为大众文化符号。2010 年"登封'天地之中'历史建筑群"列入世界文化遗产——包括中岳庙、汉三阙、嵩岳寺塔、嵩阳书院、少林寺塔林、会善寺、净藏禅师塔在内的八处十一项——**嵩山一百五十里之内的盛唐遗迹，几乎全数包入这一份名录**。

至于太室、少室——仍东西并列；至于洛阳——仍在西北九十里。两件最稳的东西，是地形。

## 额外章节

### 登封"天地之中"申遗 8 组 11 项（盛唐遗存核对）

UNESCO 2010 列入名录的"登封天地之中历史建筑群"分为 8 组、共 11 项；本表按组归类，标注盛唐时状态：

| 组 | 时代 | 盛唐时状态 |
|---|---|---|
| 中岳庙 | 汉立、唐玄宗开元十八年（730）大修 | 盛唐五岳常祀正典所在 |
| 嵩岳寺塔 | 北魏正光年间（520 / 523 两说） | 已立两百三十余年，盛唐人见的就是这座塔 |
| 太室阙、少室阙、启母阙（汉三阙，1 组 3 项） | 东汉中元元年-延光二年（56-123） | 盛唐人见的就是这三阙 |
| 嵩阳书院（唐时嵩阳观） | 北魏太和八年（484）立、隋大业八年（612）改观 | 唐为河南道道教重地，将军柏盛唐时柏龄逾八百年 |
| 少林寺建筑群（常住院 + 初祖庵 + 塔林，1 组 3 项） | 常住院北魏太和十九年（495）立；塔林唐至清；初祖庵北宋宣和七年（1125） | 常住院盛唐为禅宗北宗本山，塔林盛唐塔今存数座；初祖庵盛唐时未立 |
| 会善寺 | 北魏（约 471-499）立 | 唐武周敕赐为道场，盛唐天文学家僧一行少年居此 |
| 周公测景台、观星台（1 组 2 项） | 测景台原迹相传周公测影所留、唐开元十一年（723）一行测影刻石仍在；观星台元至元十三年（1276）郭守敬筑 | 测景台传为周公所留、一行开元十一年测影时所用；观星台盛唐未立 |

8 组 11 项之中，**主体形制在盛唐之前已立的有中岳庙、嵩岳寺塔、汉三阙、嵩阳观、少林寺常住院、会善寺六处**——嵩山是中国少数几座"盛唐之前已经成熟"的山岳。

### 嵩山与唐代禅宗北宗谱系

| 高僧 | 在嵩山的活动 | 卒年 |
|---|---|---|
| 慧可（禅宗二祖） | 据《续高僧传》卒于少林——但《续高僧传》载其于邺城被害，少林为传说 | 593 / 后说 |
| 道信（四祖） | 唐武德七年（624）住少林短暂讲法 | 651 |
| 弘忍（五祖） | 未居嵩山——在湖北黄梅东山 | 674 |
| 神秀（北宗祖） | 武周年间被召入两京宫禁——少林为其门派本山但本人晚年不居山 | 706 |
| 普寂（神秀法嗣） | 神龙二年（706）后住少林、会善寺；玄宗开元年间召入长安——僧一行师事之 | 739 |

**北宗的"少林本山"地位，自神秀-普寂-义福一代为顶**——开元、天宝间渐转两京宫廷讲法，少林本山反而清。755 年的少林本山，已不见这一代宗师身影。
