# 峨眉山

## 一句话

蜀地最大的一座山——嘉州西七里、青衣江北、岷江西；东晋慧持自庐山西入蜀立白水寺、自此佛教在峨眉立根；李白少年蜀人、登峨眉留"蜀国多仙山、峨眉邈难匹"——**755 时是蜀地名山、佛道并修**。

## 形貌

峨眉山在剑南道嘉州犍为郡峨眉县境内——县城西七里（《元和郡县图志》），是岷江西、青衣江北的一座大山。

**主体三峰**：大峨、二峨、三峨——盛唐人通称"三峨"；本山即"大峨"，海拔约 3099 米（主峰万佛顶）。山势之奇——**自岷江平原西望可见整座山如屏立**，由低至高凡数千米高差；山顶常年云雾、雪线、佛光、云海四象兼具，是中国南方少有的"由低到高一山尽具垂直气候带"的大山。

山形得名"峨眉"——《尚书·禹贡》《华阳国志》皆已记。盛唐两说并存：一说山形似蛾眉（女子细眉之状）；一说"峨"训"高"、"眉"训"前面屏障"——皆传说，正史无明指。**山北麓即青衣江口**——青衣江北南至嘉州城注入岷江、是蜀中西部水系的一处主要孔口；**山东麓即岷江**——盛唐时已可行舟。

山中物候极特殊——**自山下至山顶垂直气候带九重**：山下亚热带常绿阔叶林（樟、楠、栎）→ 中段常绿落叶混交林（杉、桦）→ 高段亚高山针叶林（冷杉、铁杉、巴山冷杉）→ 山顶灌木与高山草甸。山中**药材资源极丰**——盛唐峨眉是蜀中药材采集核心地之一。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

峨眉山在唐人的回望里，是一座**蜀地早立、佛道并修的山**。

**先秦至两汉**——峨眉已是蜀中名山、《华阳国志》记杨雄注《方言》于此。**东晋隆安二年（398）慧持**（慧远胞弟、东晋名僧）自庐山西入蜀，"登峨眉山、立白水寺于山中"（《续高僧传》卷六·慧持传）——**这是佛教在峨眉立寺的最早实证**。慧持在峨眉的具体年数史源略异（约 398-412 之间），但其立白水寺一事是峨眉佛教史的起点。

**南北朝至隋**——峨眉山中陆续有佛寺增建、亦有道士隐修。**唐开国后**——峨眉佛教渐盛，但具体寺院规模、僧侣数目盛唐史源散乱。**白水寺**至唐时已具规模、是峨眉山中最重要的一座佛寺——后世改名"白水普贤寺"、"万年寺"（北宋大中祥符元年/1008 改万年寺；755 时仍称白水寺或白水普贤寺）。

**李白与峨眉的关系是盛唐峨眉最重的一笔**。李白（701-762）祖籍陇西成纪、生于碎叶城（异说）、**自幼随父迁入剑南绵州彰明县（昌明县/今四川江油）**——青莲乡童年。开元八年（720）前后李白十九岁起出游、留下《登峨眉山》《峨眉山月歌》等数首峨眉诗（**是否多次实登峨眉、史源不强定**；《访戴天山道士不遇》之"戴天山"在江油一带，非峨眉本山）。其中《峨眉山月歌》"峨眉山月半轮秋、影入平羌江水流"——**是峨眉山进入中国文学最有名的一笔**；《登峨眉山》"蜀国多仙山、峨眉邈难匹"——把峨眉立为蜀中诸山之最。

### 此刻

755 年的峨眉山归**嘉州**（治龙游县、今四川乐山市）管辖；山在嘉州属下**峨眉县**境内——县城西七里。**嘉州城**在岷江、青衣江、大渡河三江汇流口——是峨眉山下的州城；**乐山大佛（凌云寺大弥勒石像）已凿 42 年但尚未成**——**开元元年（713）海通和尚始凿、贞元十九年（803）剑南节度使韦皋续凿至成、咸通元年（860）面部修整毕**；755 时凌云寺已立、大佛工程在中段（见 [嘉州.md](../cities/嘉州.md)）。

**山中佛寺**：**白水寺一系**（后世万年寺）自东晋慧持立以来已三百五十余年；**寺名链待考**——传慧持入蜀后于峨眉创建普贤寺/白水寺前身，"白水寺、白水普贤寺、万年寺"三名为唐宋以后陆续演变（具体年代异说较多，本 doc 严守"东晋立寺、唐代沿用白水寺称"口径）。**其他寺院**——盛唐峨眉应有数处佛寺，但具体名号史源零散；中峰寺、龙门寺、雷音寺等多为唐宋以后陆续兴建。**山中道士亦多**——峨眉是蜀中道教重镇之一；具体盛唐道观规模待考。

**普贤道场地位**：峨眉作为"普贤菩萨道场"、"佛教四大名山"（与五台、九华、普陀并列）的专称——**主要在宋元间逐步定型**。**唐代峨眉已是蜀地名山 + 佛教兴盛地**，普贤信仰可能已在山中传播；但"普贤道场"作为山的官方身份定型——是 755 时尚未完成的过程。**本 doc 严守此口径**——不预支宋元后才有的"佛教四大名山"概念。

天宝十四载这一年的峨眉——**蜀地名山、白水寺为佛教中心、道观分布全山、李白诗已传**。再过半年——安禄山起兵于范阳；再过一年潼关失、玄宗西奔入蜀；**玄宗一行 天宝十五载（至德元载/756）七月车驾至蜀郡（成都）、未亲登峨眉**（具体史载未详；本 doc 不预设）。

### 此刻意味

对天宝十四载的嘉州 / 峨眉县人来说，峨眉山是**州西那座蜀中第一山**——日常采药、寺中礼佛、道士隐修。对蜀中士人来说——峨眉是**生平所登的第一座大山**：**李白以蜀人身份登峨眉、留诗传天下**——这是峨眉在盛唐最深的文化 anchor。对长安那一代士人来说，峨眉是**蜀中诗的具象**——读李白《峨眉山月歌》《登峨眉山》者皆知"蜀地之峨眉"。

唐人尚不知道——再过 48 年（贞元十九年/803）韦皋会续凿乐山大佛（开元元年/713 海通已始凿，至 755 时已凿 42 年但未成；860 大佛修毕）；再过 250 年（北宋大中祥符元年/1008）白水寺改名"万年寺"、宋代峨眉佛教达鼎盛；再过千年——**峨眉山的"佛教四大名山"+ "普贤道场"地位会在宋元逐步定型**——盛唐 755 时所见的峨眉，是这条历史线的最前端。

## 子条目

### 白水寺一系（万年寺前身）

山中半山——**东晋隆安二年（398）慧持自庐山西入蜀于峨眉立寺**（《续高僧传》卷六·慧持传；具体寺名链异说颇多）。**这是峨眉山有文献依据的最早佛寺**——立寺已 350 余年（398→755）。**盛唐时此寺规模可观**——是峨眉佛教的中心；具体盛唐殿宇形制与寺名史载未详。**寺名变迁**：传慧持创"普贤寺"或"白水寺"前身→白水寺/白水普贤寺（唐间渐显）→万年寺（北宋大中祥符元年/1008 改名）——具体年代异说，本 doc 严守"东晋立寺、唐代沿用白水寺称"口径。**今万年寺**为明清重修，含宝鼎殿、毗卢殿、般若堂、行愿楼等。

### 三峨（大峨、二峨、三峨）

**大峨**：即峨眉本山、主峰万佛顶约 3099 米——盛唐人嘴里"峨眉"主要指此。
**二峨**：在大峨之南、海拔约 2027 米——又名绥山。
**三峨**：在大峨之西、海拔约 2208 米——又名美女山。
**三峨之中以大峨为最高、最显**——盛唐诗咏皆以大峨为主。

### 万佛顶（主峰）

大峨山最高处——海拔约 3099 米。**今万佛顶名号是后世**——盛唐峨眉主峰名号史载略异、本 doc 不预设具体唐代峰名。**盛唐人是否登顶**史载未详——李白《登峨眉山》是否登至主峰、诗中亦无明指；可能登至中山带某高处。**今万佛顶**有金顶华藏寺、佛光观赏点等——皆明清以来陆续建。

### 李白与峨眉

**李白（701-762）**——青莲乡（剑南绵州彰明县境）长大、自幼临峨眉山下、**是否多次实登峨眉、史源不强定**（盛唐有"登峨眉"诗、亦有"望峨眉"诗，"多次实登"非确据）。盛唐峨眉相关李白诗有：
- **《登峨眉山》**："蜀国多仙山、峨眉邈难匹；周流试登览、绝怪安可悉。"
- **《峨眉山月歌》**："峨眉山月半轮秋、影入平羌江水流；夜发清溪向三峡、思君不见下渝州。"
- **《峨眉山月歌送蜀僧晏入中京》**（年代略晚、广德间作）。

**李白蜀中诗的核心地标即峨眉**——盛唐峨眉的文学位置由李白一人定型。**755 时李白 55 岁**——已离蜀三十年、但峨眉诗早已传天下。

### 蜀中佛教初传与峨眉

**佛教入蜀**：东汉末已有；魏晋间渐盛。**慧持入峨眉（398）**是蜀中佛教山林化的标志事件之一。**南北朝至隋**——峨眉山中陆续有佛寺增建；隋朝峨眉佛教受隋文帝崇佛之惠。**唐代峨眉佛教**——盛唐时已是蜀中佛教中心之一，与成都大慈寺、大圣慈寺、宝光寺等共构蜀中佛教格局。**峨眉作为"普贤道场"的专称**——主要在宋元间定型；盛唐峨眉是"蜀地佛教名山"，"普贤道场"地位尚未完全定型。

## 历史长河（755 → 今）

**中晚唐**　**贞元十九年（803）韦皋续凿乐山大佛于凌云山**（713 海通始凿、803 韦皋接续、860 修毕）——峨眉山东麓青衣江口、是中晚唐蜀中佛教最大的一项工程。**中晚唐峨眉佛教继续发展**——白水寺一系、中峰寺等续修。**白居易、刘禹锡、温庭筠**等过蜀地或贬蜀者多有峨眉咏——但中晚唐峨眉诗仍以李白为最高峰。

**两宋**　**北宋初峨眉佛教达鼎盛**——朝廷诏修白水寺；**大中祥符元年（1008）白水普贤寺改名"万年寺"**——此名沿用至今。**南宋时峨眉作为"普贤菩萨道场"的专称定型**——南宋范成大《吴船录》记峨眉时已明称"普贤道场"。**南宋范成大游峨眉**（淳熙四年/1177）——留《峨眉山记》——是峨眉山最早的成熟游记。

**元明清**　元代峨眉佛寺多有大修；**明洪武初敕修万年寺**；**明嘉靖、万历间峨眉佛寺扩为'峨眉八大寺'**（万年、报国、伏虎、清音阁、洗象池、华严顶、金顶、洪椿坪等）——格局至今未易。**清代峨眉为蜀中第一佛教名山**——康熙、雍正、乾隆三朝皆有敕修。**"佛教四大名山"专称（与五台、九华、普陀并列）在明清间逐步固化**——具体某年朝廷"正式列"之说史源不稳，本 doc 不预设。

**民国至今**　1949 年后峨眉山列国家级风景名胜区、1996 年与乐山大佛一起列联合国教科文组织世界自然与文化双重遗产。**今峨眉山**——主峰金顶华藏寺、报国寺、伏虎寺、万年寺、清音阁等为主要寺院；李白《峨眉山月歌》之"半轮秋"仍为今日游人来峨眉的核心诗境。**盛唐 755 时人在白水寺所见的那座山——其物理主体一千二百多年未易**；具体寺院、佛像、设施——多为明清以来累积。
