# 恒山

## 一句话

北岳——唐人口中的"恒山"——天宝五载玄宗诏封"安天王"；**实际指河北曲阳之山**（古恒山 / 大茂山）、非今山西浑源恒山——浑源改祀是清顺治十七年（1660）以后的事。

## 形貌

唐时**北岳恒山在定州博陵郡恒阳县（盛唐县名；后改曲阳；今河北曲阳）北一百四十里**（《元和郡县图志》卷十七）——即今河北省保定市曲阳县西北、阜平县西南、唐县西北一带的山系，主峰**大茂山**海拔约 1900 米——是太行山北段的一座大山。

地势：**北岳是太行山北段最高的一组山系**——主峰大茂山立于晋冀界、东俯华北平原、西望桑干河谷、北接灵丘 / 飞狐口（太行八陉之一）。**整座山系南北长约 60 里、东西宽 40 里**——是河北道北部一片大山。

**唐时北岳庙**——**在恒阳县（即后世曲阳）境**——非在山上。**这是国家祀北岳的正坛**——每岁立冬遣使致祭。**今河北曲阳北岳庙**主体为元、明、清三朝重修，但**庙址北魏 / 隋唐以来当未大易**（汉代具体祀址是否即此庙、本 doc 不预设）——是北岳祀典在曲阳的物理 anchor。

物候：恒山地处太行山北段、冀北寒冷气候——四时分明、冬日大雪、夏日凉爽。山中林深、松柏密、有狐狼之兽——盛唐时多有道士、采药者隐居山中。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

恒山在唐人的回望里，是一座**走过两段地理变迁的山**——周代北岳指霍山、汉宣帝改定恒山为北岳。

最早一笔在《尚书·禹贡》——"恒、卫既从"、"太行恒山、至于碣石"，恒山列冀州山系。**但《周礼·职方氏》记冀州'其山镇曰霍山'**——周代北岳实指霍山（晋南、今山西霍州一带），非恒山。**汉武帝、汉宣帝两朝重定五岳**——**汉宣帝神爵元年（前 61）制诏太常立五岳四渎之祀**、**北岳改定恒山**（《汉书·郊祀志》）——自此恒山正式为北岳，已七百余年。

**汉初避文帝刘恒讳，恒山改称"常山"**——直至北魏复改"恒山"。**北魏、北齐、北周、隋皆有遣使祭北岳之礼**——但具体祭于何处史源略异：**多在曲阳县境**——这是唐沿承的地理坐标。

**唐开国后承隋制**——北岳恒山仍在曲阳。**贞观、永徽、武则天朝皆有遣官祀北岳之礼**——具体祭于曲阳北岳庙。**玄宗朝再推一程**：先天二年（713）封华岳为"金天王"——西岳得封号最早；**天宝五载（746）下诏册封中、南、北三岳**：中岳为"中天王"，南岳为"司天王"，**北岳恒山为"安天王"**——东岳为"天齐王"在开元年间已封。至此五岳尽得王号——北岳的安天王号、自此永驻。

### 此刻

755 年的北岳恒山在**定州博陵郡恒阳县**（后改曲阳）境内——**山与下庙俱在恒阳县**；定州治安喜（今河北定州市）、恒州治真定（今河北正定市）——两州相邻，恒州因古恒山而名，但**此山的直接行政归属是定州，非恒州**。盛唐时定州、恒州常并称"恒定"。**山下恒阳县城**距大茂山主峰约一百四十里（《元和郡县图志》）——盛唐人自定州城北行、过恒阳县、再北行入山——这是 755 时礼北岳的实际行程。

**恒阳北岳庙**——是国家祀北岳的正坛、每岁立冬遣使致祭。盛唐庙宇规模史载未详——但自汉宣帝定恒山为北岳（前 61）至此已 816 年；具体庙址在恒阳的稳定记录可上溯北魏至隋唐。**山上是另一界**——盛唐恒山中道观甚多、佛寺亦有数处。**北魏以来恒山即佛道兼修地**——但具体盛唐山中寺观规模史载零散、本 doc 不一一预设。

**安史前夕**（即 755 这一刻）——再过几个月，**安禄山起兵于范阳（幽州）**、自范阳经定州、恒州南下取洛阳——**北岳恒山一带是安禄山起兵后第一波过境地**。**至德间史思明陷河朔诸州**——这一段安史中恒山一带反复易主，北岳大庙、山中寺观应均有兵燹（具体毁坏程度史载未详）。

### 此刻意味

对天宝十四载的曲阳人来说，恒山是**县西北那座管辖国家祀典之北岳**——日常游赏、岁岁致祭。对长安那一代士人来说，北岳恒山是**五岳之北**——出长安东北行二千余里方至、多数人一生未必能至。但北岳是国家祭五岳之一，凡读《禹贡》《周礼》者皆知"北岳恒山"——是地理认知层面的常识。

**对盛唐文人来说**，恒山的诗咏不如华山、嵩山密集——主要因北岳偏远、文人少至。但**北岳在国家祀典中的位置是固定的**——历代帝王巡狩时若不能亲至，亦遣使祭。盛唐玄宗朝五岳并尊、恒山的政治地位与其他四岳相侔。

唐人尚不知道——再过半年这一带成为安禄山起兵后第一波过境地；再过 900 年（清顺治十七年/1660），**朝廷正式将北岳改祀山西浑源恒山**——自此"北岳"二字在地理上发生根本变迁；再过千年，**今河北曲阳的北岳庙仍立**——但庙中所祀已不再是北岳本山（**北岳本山改在山西浑源**），曲阳北岳庙转为"古北岳"祭祀地。**北岳恒山在唐人脑中所指的那座山——河北曲阳之大茂山——千年之后已退出官方北岳祀典**。

## 子条目

### 大茂山（唐时北岳主峰）

唐恒州 + 定州交界处——海拔约 1900 米。**唐时北岳主峰即此**（《元和郡县图志》"恒山在曲阳县西北一百四十里"——指的便是大茂山一带）。**今大茂山在河北省阜平县与涞源县、唐县三县交界处**——是太行山北段最高峰之一。**今大茂山别名"神仙山"**——是民间通称。**盛唐人来恒山所登的便是这一座**——但唐时是否有上山道、史载未详。

### 恒阳北岳庙（今曲阳北岳庙）

恒阳县城（盛唐县名；后改曲阳——今河北曲阳县城）——**国家祀北岳的正坛**。**汉宣帝定恒山为北岳（前 61）始至此已 816 年；具体庙址在恒阳的稳定记录可上溯北魏至隋唐**，汉代是否即此庙址本 doc 不预设。**隋唐皆有遣使致祭记录**；唐玄宗朝亦多次致祭。**今曲阳北岳庙**主体为元、明、清三朝重修——含德宁之殿（祀北岳安天王）、飞石殿、御香亭等；最古遗存是**元代壁画**（德宁之殿内）。**唐代庙宇格局**已不可考。**北岳改祀浑源后**（清顺治十七年/1660）——曲阳北岳庙转为"古北岳庙"地位，但物理建筑仍在。

### 飞狐口 / 飞狐道

恒山东北麓——**飞狐口是太行八陉之一**（《吕氏春秋·有始览》），连接河北灵丘 / 蔚县与河北涞源、易县——是太行山北段的咽喉道。**飞狐口非北岳本身**、是恒山北侧的关道。**盛唐飞狐道是恒州、定州一线连接山西河东道代州、朔州一线的主要北上通道**——亦是 755 后安禄山起兵 / 唐军围剿往返的重要孔道。

### 北岳 vs 浑源恒山（地理迁移注）

**唐时北岳 = 河北曲阳大茂山一带**（本 doc 严守此口径）。**山西浑源恒山改祀** 是明清以后的事——**明嘉靖年间始议改祀浑源**（《明实录》《明史·礼志》皆有载），但当时未行；**清顺治十七年（1660）正式改祀于浑源恒山**——自此"北岳"二字在地理上发生根本变迁。**盛唐人嘴里的"北岳恒山"**与**今人嘴里的"北岳恒山"是两座不同的山**——这是恒山在中国历史地理上最特殊的一笔。

## 历史长河（755 → 今）

**中晚唐**　**至德二载（757）史思明陷恒州**——北岳恒山一带成为安史拉锯战场；曲阳北岳庙在乱中应有兵燹（具体毁坏程度史载未详）。**贞元间唐德宗诏修北岳庙**——这是中唐北岳祀典恢复的标志。**中晚唐恒山诗咏不密集**——岭北诗人少至此地、北岳偏远使然。

**两宋**　**北宋大中祥符四年（1011）宋真宗诏修北岳庙**——曲阳北岳庙得宋朝一次大修。**北宋庆历七年（1047）德宁之殿建**——是现存德宁之殿的祖型。**南宋失中原后**——金代恒山归金、曲阳北岳庙继续作为北岳祀典正坛。

**金元**　**元代曲阳北岳庙大修**——德宁之殿内今存元代壁画。**元代北岳仍在曲阳**。

**明清**　**明嘉靖朝始议改祀北岳于浑源**——但当时未行。**清顺治十七年（1660）朝廷正式诏改祀北岳于山西浑源恒山**——自此曲阳北岳庙地位骤降、"古北岳"成新名号。**清代浑源恒山列为新北岳**——历代帝王致祭转至浑源。**曲阳北岳庙转为地方祠庙**。

**民国至今**　1949 年后曲阳北岳庙列河北省重点文物保护单位、1982 年列全国重点；浑源恒山列国家级风景名胜区、1982 年列首批。**今"北岳"二字一指山西浑源恒山（自 1660 改祀）+ 一指河北曲阳大茂山（古北岳）**——两山并立；地理学界、宗教学界至今仍有"古北岳"的研究与考察。**唐人在 755 所见的北岳恒山——河北曲阳大茂山——千年之后已退出官方北岳祀典**；但大茂山仍立、神仙山的民间称谓延续至今。
