# 渼陂湖

## 一句话

鄠县西南、终南山下一片唐时大湖——杜甫与岑参兄弟泛舟于此作《渼陂行》，**那是 755 前夜的盛唐最后一夏**；今缩为田畔小水。

## 形貌

渼陂在鄠县城西南五里——南屏终南山，东接樊川水系，西侧引涝水（涝峪之水）入陂。

唐时陂面**周十四里**（《十道志》口径，后世地志沿引；《嘉庆重修一统志》渼陂条同），水深处可没桅杆，足以浮中等舫船。陂形不规则——东南阔、西北窄，岸边多生芦苇、菱荇、水葵。陂北是鄠县城所辖一片水田，陂南是终南山北麓的台塬——自陂上北望，可见塬上的麦田连成一线；自陂上南望，可见秦岭主脊的青影。

陂得名于"渼"——字书有水波纹义（《类篇》口径）；亦有训"陂水鱼美、可酿"两说。盛唐两说并存，但**唐人嘴里"渼陂"两字主要指的是这片水本身**。陂水自南山六谷之一的**涝峪**自南北流而来，注入陂中——涝峪是终南北麓六谷的西路之一，谷中产**黄米**与**栗**，水自上流来时带松针的清气，是渼陂水自带的一脉物候。

陂上有鄠县源氏（**源大、源二少府**）家的画舫与水亭——盛唐 750s 间杜甫数次至此宴游，所记的"水亭"、"碧浪"、"波底翻天舞日"皆是此处。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

渼陂的史源比汉昆明池晚——但盛唐人对它的痴迷，比对昆明池更深。

陂的形成年代不明——后世地志追述"自隋以前已为陂"。先秦至隋的渼陂史料几乎是空白：既无"汉武凿陂"的官修工程记录，也无"齐梁文人泛陂"的私家诗文。**陂大约是涝峪自南北下行、自然积水成湖的产物**——人工只是加了堤、岸、桥、亭。隋唐间鄠县已是**京畿西南二十县之一**——县境内陂、塘、堰、渠星罗，**渼陂是其中最大的一片**。

到盛唐——**渼陂的文学地位骤然拔高**。这与几个因素相关：一是盛唐文人的"水亭—画舫—宴集"风气兴盛，鄠县离长安近（八十里）、又有真水真山，是士人短游的首选；二是渼陂上有源氏家学营建的水亭画舫，提供了"文人主家"的接待平台；三是杜甫——**他是把渼陂从一片地方水景写成全国诗境的人**。

### 此刻状态

天宝十四载这一年的渼陂——**水依旧、人已散**。

杜甫《渼陂行》作于**天宝十三载（754）秋**——比 755 仅早一年（浦起龙《读杜心解》系年；姚鼐《今体诗钞》同）。诗记"岑参兄弟皆好奇，携我远来游渼陂"——这一夜杜甫与岑参、岑长兄等同游渼陂，舟行水上，至深夜方还。**这首诗是渼陂在唐诗中的最高一笔**——也是杜甫在长安最后一个完整的盛唐之夜。

755 时杜甫是否还在长安？——按通行年谱（萧涤非《杜甫全集校注》、莫砺锋《杜甫诗歌讲演录》），杜甫 752 年至长安，**755 秋在长安任右卫率府兵曹参军**，闰十一月奔回奉先县看望家人——途中即作《自京赴奉先县咏怀五百字》。**这一年的渼陂，杜甫已不再泛舟**——他守在长安城内一间狭屋里写诗。岑参 754 已赴北庭幕府——亦不在长安。**陂上去年的游伴皆散**。

陂本身水依旧、亭依旧、画舫依旧——但 754 年那一夜的"船舷暝戛云际寺、水面月出蓝田关"，再不会重来。

### 此刻意味

对天宝十四载的鄠县人来说，渼陂是**县境西南最大的一片水**——耕渔灌溉皆赖此。对长安士人来说，渼陂是**盛唐宴游的一处地标**——但 755 时已开始进入"前几年的旧事"调子；从这一年起，**渼陂的盛唐宴游气脉断了**。对杜甫——再过半年长安陷、他被困两京之间；再过十二年（766）他在夔州写《秋兴八首》追忆长安诸景——其中第八首"昆吾御宿自逶迤，紫阁峰阴入渼陂"七字——**便是渼陂在他笔下的最后一笔**。

## 子条目

### 渼陂水亭

陂上鄠县源氏（源大少府家）所营建的水亭——杜甫《与鄠县源大少府宴渼陂》主角处。亭立陂北水中——四面环水，桥栈通岸。亭内宴坐可观陂上全景——南望终南、北望鄠县平畴。盛唐 750s 间杜甫与岑参兄弟、源氏诸子多次在此宴集，写诗、品酒、夜泛舟。**渼陂水亭是盛唐"文人主家+水亭+画舫" 这一套审美的典型样本**——其后的辋川别业（王维 730s 已营）、城南别业（高适、岑参各家）都从这种模式发展。

### 涝峪入陂口

陂西南角，涝峪之水自此入陂——这是渼陂的主水口。涝峪是终南北麓六谷的西路之一，谷中产黄米与栗。涝水自谷口出后向北、向东折，至渼陂西南角入陂——盛唐时此口岸有水栅、水堰，岁有岁修。**唐人称这一段水为"涝水入陂"**，是渼陂水量的根本来源。涝水若枯——陂便枯。

### 紫阁峰

终南北麓诸峰之一，**距渼陂直线约二十里**——是渼陂上南望可见的主峰。杜甫《秋兴八首》其八"紫阁峰阴入渼陂"即此——意为紫阁峰的山影在傍晚时投入渼陂水中。**紫阁峰是渼陂的文学背景**——凡咏渼陂者皆涉紫阁。紫阁今属西安市鄠邑区，主峰海拔约 2150 米，山势峭丽，唐时为终南名峰之一。

### 杜甫《渼陂行》一夜

天宝十三载（754）秋——岑参、岑参兄长（"岑参兄弟"《渼陂行》开篇即点）携杜甫游渼陂。一行人傍晚至陂，在陂北水亭宴酒——夜深泛舟入陂中央。陂上鼓乐喧腾，舟上看月——**月自蓝田关方向升起**（蓝田关在长安东南、终南山一段，与渼陂相去百余里——杜甫诗中所写是月升的方位感，不是直见关）。**舟行至深夜方归岸**——杜甫归后作《渼陂行》。

诗云"半陂以南纯浸山，动影袅窕冲融间"——意为陂南一半水面，全是终南山的倒影，影子随波动摇。又云"船舷暝戛云际寺，水面月出蓝田关"——前句写暮色中舟舷戛响、声影及云际寺一带（云际寺/云际山为渼陂南畔实有寺刹，盛唐古注有据，非泛指）；后句写月光自东南方向（蓝田一线）升起、洒落水面。**全诗七言二十八句、约一百九十六字，是盛唐写水景的一首长篇七言乐府**——也是渼陂在唐诗中的最高一笔。

## 历史长河（755 → 今）

**中晚唐**　安史乱后渼陂物理上未受兵燹直接破坏——但源氏一家在战乱中散——鄠县唐时的几个士族画舫水亭随之废。**中唐渼陂只剩水**——文学位次随杜甫诗永驻盛唐顶层，物理实存自此与盛唐脱节。元和年间（806-820）韦应物、白居易等过鄠县，均未再写渼陂——**渼陂的中唐文学几乎是空白**——盛唐之后再无人能写它。

**五代至宋**　北宋程大昌《雍录》卷六记渼陂"今为田，唯余涝水入陂遗迹"——可见**北宋时渼陂已大半涸为田**。宋敏求《长安志》卷一五记鄠县渼陂"周回十数里"是追述唐时旧状，时人所见已是缩水。陂上水亭、画舫——皆已无踪。**北宋人来鄠县只能凭杜甫诗"想象渼陂"**——但脚下踏的，是已成耕田的旧陂基。

**明清**　明代鄠县志记渼陂"周回数里"——已从北宋的旧状再缩。**《嘉庆重修一统志》渼陂条记"元末决水取鱼，陂涸为田"**——这是渼陂物理终点的关键史料：元末有人为决堤取鱼，自此陂底成田、再未复水。**清代渼陂只剩涝水一线下行的水道，与唐时大湖再无关系**。乾隆四十年（1775）陕西巡抚毕沅立"渼陂故址"碑——是清代"复古考"的一笔。

**民国至今**　1949 年后渼陂故址范围内仅存数处小水塘——分别为"东坡陂"、"西坡陂"、"北坡陂"等农村小水面，皆远小于唐时全陂。**2017 年西安市鄠邑区始建渼陂湖湿地公园**——人工恢复水面约 4180 亩（公开规划口径，约 2.8 平方公里；另有 5300 亩近期扩建口径）——但**这片水的位置与唐渼陂略偏**，且水来自人工补给（部分自涝峪、部分自市政管网），与唐时"涝水自南来、自流入陂"已不是同一种水。今游人至渼陂湖湿地公园，看到的水亭、画舫、湖心岛——皆是现代景观；唐杜甫笔下那种"半陂以南纯浸山"的真山水倒影——**考古至今也没回到那个水位**。
