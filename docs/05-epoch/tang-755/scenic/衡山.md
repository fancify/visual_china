# 衡山

## 一句话

南岳，唐人口中"配朱鸟"的山——天宝五载玄宗诏封"司天王"；天台宗祖庭之一、南宗禅怀让一脉之祖庭——**衡山在 755 时既是国家祀典、亦是佛道双圣山**。

## 形貌

衡山自湘水之东拔地而起——是衡州城北约百里、湘水东岸的一座长山。

不是孤峰，是数十峰相次连绵的一整片山系——主峰**祝融峰**海拔约 1300 米，居山势之最高、立于全山东北端；从祝融峰南下、有紫盖峰、芙蓉峰、天柱峰、石廪峰等数座大峰相次，至最南端的**回雁峰**（在衡州城北郊）止——**"七十二峰"为后世南岳志书系统化的说法，《南岳总胜集》（南宋陈田夫，1163 撰）详记**；755 时是否已有完整"七十二峰"名目，本 doc 不预支。整片山系南北长约百二十里、东西宽约六十里，是江南西道腹心一片大山。

山下衡州州治在湘水西岸的衡阳县——衡州城与衡山主峰直线相距约百里。**南岳庙**（盛唐通称；神号"司天王"自天宝五载/746 起）在衡山主脊东南麓、山下湘水东岸的衡山县境——是国家祀典南岳的正坛。盛唐人自衡州城北行百里、过湘水入山——这是 755 时礼南岳的实际行程。

物候：衡山地处江南西道中段、亚热带季风山地——四时湿润、夏日清凉、冬日少雪。山中林深、竹丛、瀑布多——是岭北诸山中最适合长居的一处。**山中真正险峻者不过五六座大峰**——多数是缓岭密林，适合僧道结庵长居。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

衡山在唐人的回望里，是一座**走过两段大变化的山**——汉武代祀霍山、隋开皇还祀衡山。

最早一笔在《尚书·禹贡》——"荆及衡阳惟荆州"。再到《周礼·职方氏》"荆州，其山镇曰衡山"——衡山自先秦已是荆州方镇之山。秦统天下后定五岳，衡山为南岳。**但汉武帝元封五年（前 106）巡狩至盛唐县**——以衡山远在江南、巡狩不便，**改以江北霍山（今安徽天柱山）代祀南岳**——此后五百余年（前 106 → 589）南岳实际指霍山、非衡山。

**隋开皇九年（589）平陈后**——隋文帝下诏定五岳、**南岳复还衡山本山**（《隋书·礼仪志二》）。**自此衡山再为南岳之正**——已五百余年汉代旧账被翻转。隋朝重立南岳庙——是衡山作为国家祀典对象的重启。（注：另有"霍山应圣公"之祀属江北霍山自身、与衡山南岳庙分属两套——本 doc 严守不混。）

唐开国后承隋制——衡山仍为南岳。**玄宗朝再推一程**：先天二年（713）封华岳为"金天王"——西岳得封号最早；**天宝五载（746）下诏册封中、南、北三岳**：中岳为"中天王"，**南岳衡山为"司天王"**，北岳为"安天王"；东岳为"天齐王"在开元年间已封。至此五岳尽得王号——衡山的南岳地位、自此王号永驻。

### 此刻

755 年的衡山归**衡州**管辖——衡州治衡阳县（今湖南衡阳市），在山南约百里。山下湘水东岸的**衡山县**为**南岳庙**所在——盛唐通称；神号自天宝五载（746）起为"司天王"；每岁立夏遣使祭南岳，香火甚旺。

山上是另一界。**衡山自南北朝起便是佛道双圣山**——
- **佛教**：陈光大二年（568）**慧思（515-577）**入南岳建般若寺、立"南岳道场"——慧思是天台宗三祖、智者大师智顗的师父；其在南岳的"般若寺"是天台宗在江南的发祥地之一。**隋唐间南岳佛教自慧思以下传**——盛唐时山中有数十座佛寺。
- **道教**：衡山为**道教第三洞天"朱陵洞天"**（《云笈七签》）——盛唐道教兴盛、山中道士甚多。（注：**司马承祯（647-735）**为玄宗朝上清名道，主要行迹见天台山、王屋山、嵩阳——与南岳之关系待考，本 doc 不预设。）

更晚的是南宗禅：**怀让（677-744）**——慧能弟子，自岭南北上至南岳般若寺、长居三十余年。**怀让传马祖道一**——马祖（709-788）是南宗禅"南岳一脉"开山祖。**755 时怀让已殁 11 年**（744 卒）；马祖在世 46 岁——其 755 时具体驻锡地史源不强定（盛唐曾在衡岳从怀让受法、后于洪州开元寺弘法）。**衡山在 755 时既是天台宗江南祖庭、亦是南宗禅"南岳一脉"祖庭**——一山而兼三流（天台 + 上清道 + 南宗禅）。

### 此刻意味

对天宝十四载的衡州人来说，衡山是**州北那座管辖的国家祀典之山**——日常游赏、岁岁致祭。对长安那一代士人来说，衡山是**南方最远的一座岳**——出长安南行四千余里方至，**多数人一生未必能至**。但衡山是国家祭五岳之一，凡读《周礼》《尚书》者皆知"南岳衡山"——是地理认知层面的常识。

**对盛唐文人来说**，衡山的诗咏密集化在中唐及以后——**杜甫《望岳·南岳配朱鸟》**作于大历四年（769）客衡阳时（755 时尚未作）；李白《与诸公送陈郎将归衡阳》《赠崔秋浦》等盛唐间作——但李白本人未必登过衡山。**衡山的"南宗禅祖庭"地位在中晚唐确立**——755 时怀让方殁、马祖正盛，南岳禅的全国影响尚在酝酿。

唐人尚不知道——再过 30 年（贞元元年/785），南岳庙将由唐德宗诏修；再过 200 年（北宋），衡山将成为南宋朱熹"寿岳"之祖（朱熹《诗集传》考"寿岳即南岳"）；再过千年，**衡山主脊与诸高峰的位置不易**——只是山中佛寺由盛唐数十座变为今数处大刹、道观由盛唐密集化为零星几观；"七十二峰"作为完整数目则是南宋以后系统化。

## 子条目

### 祝融峰（衡山主峰）

衡山东北端最高峰——海拔约 1300 米。**得名"祝融"**——传说上古火神祝融葬于此（《路史·南岳考》）；但"祝融"作为山名先秦未见、最早实证为南北朝间。**祝融峰是南岳的天然标识**——盛唐人凡至衡州、北望祝融峰、便知南岳在眼前。**今祝融殿**为明清重修；盛唐顶峰是否已立祠、史载未详。

### 南岳庙

衡山主脊东南麓、山下湘水东岸的衡山县境——盛唐时称"南岳庙"（天宝五载封司天王后渐改）。**这是国家祀南岳的正坛**——每岁立夏遣使致祭。**唐德宗贞元元年（785）以后改名南岳真君庙、再后改南岳大庙**——名号历代有变；但庙址不易。**今南岳大庙**为清光绪年间重修——是岳庙群中规模最宏的一座；规模仿京师宫城——四面有门、正殿额题、前有牌楼。

### 般若寺（慧思 + 怀让两代祖庭）

衡山主脊半山——**陈光大二年（568）慧思入山建**，是天台宗在江南的发祥地之一。**怀让自岭南北上后亦驻锡般若寺**——长居三十余年（约 713-744）。**两代祖庭在同一寺**——这是衡山佛教史最重的一笔。盛唐时般若寺规模可观、僧侣数百。**今南台寺、福严寺、藏经殿一带**是般若寺旧址的几处分支——主体寺名屡变。

### 七十二峰（南宋《南岳总胜集》系统化定型）

衡山主脊南北长百二十里、东西宽六十里——主体数十峰相次。**"七十二峰"为南宋陈田夫《南岳总胜集》（1163 撰）系统化定型的说法**——盛唐 755 时是否已有完整七十二峰名目，史源不稳。**主要高峰**：祝融、紫盖、芙蓉、天柱、石廪——是盛唐人来衡山时第一波认识的山名。**多数峰是缓岭密林、适合僧道结庵长居**。

### 朱陵洞天（道教第三洞天）

衡山为**道教第三洞天"朱陵洞天"**（《云笈七签》卷二十七）——是十大洞天之一。**唐道教兴盛、山中道士甚多**。**盛唐衡山的道教 + 佛教并立**——是这一座山在 755 时最特殊的"双圣"格局。（注：**司马承祯（647-735）**为玄宗朝上清名道，主要行迹在天台山、王屋山、嵩阳——与南岳关系待考，本 doc 不预设。）

## 历史长河（755 → 今）

**中晚唐**　**怀让一脉传马祖道一（709-788）、马祖再传百丈怀海（720-814）**——**南岳禅自此向天下散**——成中晚唐禅宗最大一支。**贞元元年（785）唐德宗诏修南岳真君庙**——这是衡山在中唐地位重新确立的标志。**杜甫大历四年（769）客衡阳**作《望岳·南岳配朱鸟》"南岳配朱鸟、秩礼自百王"——把衡山写进中唐诗最有名的一首。**柳宗元、刘禹锡**贬永州、连州、过衡州皆有衡山咏。

**两宋**　**北宋初**朝廷诏修南岳庙；**北宋仁宗、神宗朝**南岳为陪都祀岳之首。**南宋朱熹（1130-1200）**考"寿岳"——《诗集传》考衡山为"寿岳"之正——自此南宋人称南岳为"寿岳"。**南宋衡山佛教达鼎盛**——南台寺、福严寺、上封寺、方广寺合称"南岳四大丛林"。

**元明清**　元代南岳庙数次重修；**明洪武三年（1370）**敕修南岳大庙；明嘉靖、清康熙、清光绪四朝皆有大修——**今南岳大庙主体为清光绪年间重修**。**清代衡山为南岳祀典正坛**——岁岁致祭；山中佛寺逐渐定型为今所见数处大刹（南台寺、福严寺、上封寺、藏经殿、祝圣寺、方广寺等）。

**民国至今**　1949 年后衡山列国家级风景名胜区、1982 年列首批；南岳大庙、南台寺、福严寺、藏经殿等列全国重点。**1989 年衡山设立国家级风景名胜区管委会**——主峰祝融峰、磨镜台（怀让传马祖处）、南岳大庙、水帘洞、上封寺等成主要景点。**今游人来南岳**——所见的是清光绪重修的南岳大庙 + 历代山中佛寺道观 + 千年未易的祝融主峰——**衡山主脊与诸高峰的位置一千二百多年不易**（"七十二峰"作为完整数目则是南宋以来系统化）。
