# 终南山

## 一句话

长安南屏，唐人口中的"南山"；太乙近天都，终南成捷径——城南八百里山脊收着盛唐士人的归隐、问道与回望。

## 形貌

终南山倒像一道横在长安肩上的长屏。

自武功县东南起，沿渭南诸县南境一路向东，至蓝田、商州一带方收——东西约八百里（盛唐人语，今测约四百公里）。秦岭东段中部的这一段山脊，北侧陡然落入关中平原，南侧缓缓滑入丹江、汉江流域。长安城自朱雀大街南望，南山就横在那里——晴日里一脉青黛压在城南，雨后云气缠在腰际不下来。唐人讲"南山"几乎不必加字，长安城南便是。

山势并不齐整。主峰太白山远在西端（另有独立 doc），盛唐人去得少；常入眼的几座峰，在长安城南一线由西向东摆开：圭峰、紫阁峰、南五台、翠华山、骊山——再往东就到了辋川。每一峰各有性情：圭峰削如圭玉，紫阁霁后紫气浮顶，南五台五峰并峙若五指。山间多溪谷——樊川自城南南山樊川峪出，潏水自子午谷出，灞水自蓝田东南谷出——这几条水都从南山下来，养着长安东南半城的水气。

山里多寺多观。佛家有草堂、净业、丰德、悟真、香积；道家有楼观台、宗圣宫——两家分山而住，互不相犯。山中物候：三月山桃，五月新笋，七月避暑（士人多上山过伏天），九月红叶，冬日积雪从北坡望去如长城。

## 大唐天宝十四载（755年）

终南山在唐人眼里不是一座新山。

《尚书·禹贡》已记"终南、惇物，至于鸟鼠"——商周文献便有它的名。秦汉立"南山"之号，《史记》载汉宣帝"南山豹"，李广射虎处亦在此区。终南古号繁多：地肺山（道家方志称地之肺叶呼吸处）、太一山（汉武帝祠太一神于山下）、中南山（《诗经·秦风》"终南何有"——古训"终"通"中"）。一座山叠着多套名字，每套属一段史。

道家把终南列为"洞天福地"——上清派列其为"三十六小洞天"之第五"太乙洞天"。山西端楼观台旧传是周大夫尹喜结草为楼以观星气、老子西行过函谷关后于此讲《道德经》之处——这一传说为后世道教所重，北周武帝天和五年（570）正式于其地立"通道观"；唐立国后，李渊以李耳为远祖，**武德初**（一说三年、一说七年，史载小异）改楼观为"宗圣观"，自此为唐宗室祖观。玄宗朝再扩修山中道观体系，**开元二十九年（741）另建会灵观**。"宗圣宫"之名号通行已是元代以后，盛唐时本观仍称宗圣观——但开元天宝间整个山西段已被纳进国家奉道体系。

天宝十四载（755），玄宗已七十一岁，崇道已极。山中道观当时正盛：宗圣宫之外，山中尚有云际、传教、白云、太极诸观——皆开元天宝间敕修或敕额。山中道士不少与朝中往来——开元间司马承祯曾被睿宗、玄宗两朝召至京师问道（玄宗开元九年 721 召至东都），承祯辞归后即住天台山，但他在终南也留下行踪，山中道家由此重；李白《古风》"西上莲花山"借的便是这一脉文人慕道意象。

佛家则另成系统。山中律宗祖庭净业寺由唐初道宣律师立——道宣（596-667）于贞观、永徽间在终南山丰德、净业、白泉诸寺间讲律传戒，成"四分律宗"南山一派，史称"南山律宗"，是唐代汉传佛教三大律宗之一。755 年道宣已圆寂八十余年，但他在终南山立下的律制仍为唐僧团根基——天宝间净业寺、丰德寺依旧是关中受戒中心。山西草堂寺则更早：后秦弘始三年（401）鸠摩罗什自姚兴迎入长安、于逍遥园（即今户县草堂寺址）译经——译经卷数历代目录统计不一：《出三藏集记》作三十五部二百九十四卷，《大唐内典录》作九十八部四百二十五卷，《开元释教录》核定七十四部三百八十四卷。草堂为汉传佛教"三论宗"祖庭，亦中国佛教史上第一座国家译场。罗什圆寂于此（弘始十五年 413），墓塔即在寺西。盛唐时草堂已立三百五十余年。

但终南山对盛唐士人最大的一件事，不是寺也不是观——是**隐**。

### 终南捷径

"终南捷径"这四个字在 755 年时已经是个老典故。

主角是卢藏用（约 664-约 713）。武则天朝末年举进士不第，遂隐于终南。卢藏用心思并不真在隐——他算准了武后、中宗朝以"招隐"为时尚，刻意择终南山以隐——离长安近、消息通畅、朝中动静一日可达，又不必真去江海岭外。果然几年之内"隐士卢藏用"声名远播；中宗景龙年间召为左拾遗，仕至工部侍郎、户部侍郎、尚书右丞——隐既得名，仕又得官。

景云二年（711），睿宗召道士司马承祯入京问道。承祯辞归天台，将出京时与卢藏用同行。出长安东郊，路过终南，卢藏用指着南山对承祯说："此中大有佳处，何必天台。" 承祯答："以仆视之，仕宦之捷径耳。"

这句话被《新唐书·卢藏用传》记下，自此"终南捷径"四字成定典——名义上隐居清修，实则做仕进姿态。这一典故立得极重，因为它一笔点破了盛唐"隐"这件事的虚实之分：真隐者往天台、罗浮去，借隐求名者扎堆终南。对天宝十四载的长安士人来说，"南山"既是诗里写"空山新雨后"的那座山，也是这条"隐而后仕"的暗路——两面都是它。

王维便活在这两面之间。辋川别业（另有独立 doc）即在终南山东段北麓蓝田——王维自况为"半官半隐"，正是终南山下文人最体面的活法。盛唐人讲"南山"，脑子里至少压着三层：诗里的山、道家的洞天、捷径的话柄。

### 此刻意味

对天宝十四载的长安人来说，南山意味着——长安没说出口的另一半。城里讲朝局、讲铨选、讲钱粮；城外那座山讲的是"我也可以不来"。哪怕你一辈子不入山，南山在那儿，就给长安城留了一道精神的余地。

那年冬，南山照例落雪。长安城南的人抬头望山——仍看见一条可退的路。

## 子条目（按位置由西向东列）

### 楼观台 / 宗圣观

终南山西段、周至县东南。山中道观之最古者。

旧传周大夫尹喜结草楼于此望星气，老子西行过此为说《道德经》——此说始见于《史记·老子传》及《列仙传》，后由道家累积成"楼观本纪"。北周武帝天和五年（570），于其地正式立"通道观"——这是国家级建观之始。**唐武德改观**：李渊以李耳为远祖，**武德初**（武德三年 620、武德七年 624 两说并存）改楼观为"宗圣观"，从此为李唐宗室的祖观。**玄宗朝扩修**：开元天宝间宫观格制再升，**开元二十九年（741）另建会灵观**——盛唐时本观仍称宗圣观。"宗圣宫"之名号见于元以后碑刻沿用，其遗址沿革需按《楼观本纪》及历代碑刻另列异文。宫中有说经台、炼丹峰、化女泉——皆传与老子尹喜故事相关。唐代楼观为道教经籍重镇，藏书甚富。

### 草堂寺

终南山西段北麓、户县（盛唐时鄠县）城南三十里。终南山下最古之佛寺。

后秦弘始三年（401），姚兴迎鸠摩罗什自姑臧入长安——奉为国师，于逍遥园别立译场，名"草堂"（取庵庐草顶之意）。罗什于此译《大品般若》《妙法莲华》《维摩诘》《阿弥陀经》《中论》《百论》《十二门论》诸经——译经卷数历代目录统计不一：《出三藏集记》作三十五部二百九十四卷，《大唐内典录》作九十八部四百二十五卷，《开元释教录》核定七十四部三百八十四卷。无论以何目录为准，皆是汉传佛教经典最重的一批译场遗产。弘始十五年（413）罗什圆寂于此，墓塔（"鸠摩罗什舍利塔"）立寺西，至今犹存。隋唐间草堂寺为"三论宗"祖庭——吉藏（549-623）一脉所宗。755 年此寺已立三百五十余年。

### 净业寺

终南山中段、长安城南丰德乡之南、丰峪口内。律宗"南山宗"祖庭。

唐初武德、贞观间立。道宣律师（596-667）于此寺习律、讲律、著作——所撰《四分律删繁补阙行事钞》《四分律比丘含注戒本》《四分律拾毗尼义钞》三书，奠定汉传佛教戒律根本——后世称"南山三大部"。道宣居净业凡数十年，圆寂亦于此寺。"南山律宗"由此得名——"南山"即终南。755 年道宣已圆寂八十八年，净业寺仍为关中受戒之地。

### 翠华山

终南山中段、长安城南五十里、樊川东。

唐人不甚提"翠华"二字（其名清以后始盛）；盛唐时此山以"太乙湫"知名——山中有湫池，传说太乙真人居处，故唐时山亦称太乙山支脉。湫畔后世发掘出唐至宋数处道教遗物。翠华山岩崩湖（即"太乙湫"）实为远古地震山崩堰塞湖，盛唐人不识其成因，只视为神迹。

### 紫阁峰

终南山中段、户县东南、距长安约六十里。

紫阁形如重楼叠阁，霁后日光斜照山顶常浮紫气——故名"紫阁"。盛唐道家以此峰为修真胜地。杜甫《自京赴奉先县咏怀五百字》"南登紫阁峰"——天宝十三载（754）秋杜甫曾登此峰，作《九日蓝田崔氏庄》《望紫阁峰》等。岑参、孟郊后皆有紫阁峰诗。中唐以后峰下立紫阁寺。

### 南五台

终南山中段、长安城南五十里、子午谷东。

五峰并峙：观音台、文殊台、清凉台、灵应台、舍身台。山名"五台"是与佛教北五台（山西五台山）对应而言。盛唐时山中已立诸寺——但规模远不及晚唐至宋。后世（宋元间）南五台佛教大兴。盛唐人讲"五台"二字多指山西五台；终南南五台之名在中晚唐才渐起。

### 圭峰

终南山中段、户县东南，与紫阁峰邻近。

形削如圭玉，故名。盛唐时山中有草堂寺一脉延伸出来的诸支寺。中唐华严宗五祖宗密（780-841）后住圭峰，世称"圭峰大师"——圭峰之名由宗密始大著，盛唐时尚不显。

### 太白山（另有独立 doc）

终南山西端最高峰，海拔三千七百余米——秦岭主峰，山顶八月犹积雪。"南登太白岭，南望武功城"（李白），太白雪是盛唐边塞诗里反复出现的意象。盛唐人去得不多——山高路险，山顶冰川至七月犹有雪——多半远望。本山虽属终南山系西端，因体量与文学分量自成一独立 POI，另见 [太白山](./太白山.md)。

### 辋川别业（另有独立 doc）

终南山东段北麓、蓝田县东南，盛唐文人园。王维经营二十年。盛唐"长安士人入终南"的现象，辋川是最具体的一个 case——本山系下最完整保存到 doc 一级的子地名。另见 [辋川别业](../relics/辋川别业.md)。

## 历史长河（755 → 今）

**中晚唐**　至德元载（756）六月长安失守，叛军遁入山中者甚众；山中道观寺院多遭兵燹，但僻处深谷者得全。两京收复后，山中诸寺观陆续复修。中唐至晚唐，终南山隐风未减——韦应物贞元间曾隐山中（虽韦氏更与栎阳相关），贾岛少时为僧、与无可、姚合互酬唱多在终南诸寺，刘禹锡、白居易、卢仝皆有终南诗。圭峰宗密（780-841）住圭峰，使华严宗与禅宗的"教禅一致"在此山中立。终南山诗，自王维之后又有一大堆。会昌五年（845）武宗灭佛，山中寺院多被毁——但律宗、密宗的源头牌位还在终南，平息几年后又复。

**五代至宋**　唐末黄巢之乱、朱温拆长安，关中残破——但终南山因隐僻反而保存了一批寺观传承。北宋陕西路凤翔府、京兆府之下，终南山仍是文人入秦必访之地。司马光、苏轼、张载（关学创始人，凤翔郿县人，活动于终南山西端）皆有终南诗文。**金代全真道**：王重阳（1112-1170）为陕西咸阳人，于终南山附近的刘蒋村、活死人墓修道——金大定七年（1167）东游至山东宁海，传道收徒，全真道由此成势；终南山附近的早期修道遗迹由是被纳入全真谱系。全真七子之一丘处机后入终南问道——终南由唐之"奉道祖观"型，转入金元之"全真祖庭"型。元代敕封楼观台为道教"东华派"祖庭之一。

**明清**　终南山道教继续承传。明万历间楼观台、宗圣宫各有修葺；草堂寺亦有重修。清乾隆年间陕西巡抚毕沅（即立"王维隐居处"碑那位）巡视陕西诸名胜，亦至终南——所撰《关中胜迹图志》于楼观台、草堂寺、净业寺各有条目，距道宣已逾千一百年。同代考据派学者依《续高僧传》《广弘明集》比定唐代寺观位置，部分可考、部分已说不清了。

**民国至今**　民国间山中道观多荒，僧道散去。1950 年代后，楼观台、草堂寺、净业寺先后被列为国家级或省级文物保护单位；楼观台 1982 年列全国重点文物保护单位，草堂寺、净业寺亦相继列入。**1980 年代以来，"终南山隐士"成现代现象**——据 2003 年美国学者比尔·波特（Bill Porter）《空谷幽兰》一书所记，山中尚有数千僧道隐居者（数字至今争议较大，但确有规模）；近年学界、媒体亦持续关注。距道宣立净业寺已逾一千三百年，距卢藏用"终南捷径"亦逾一千二百八十年——这条山脊上的隐者并没有断。

> 兴来每独往，胜事空自知。
> 行到水穷处，坐看云起时。
> 偶然值林叟，谈笑无还期。

李白《下终南山过斛斯山人宿置酒》：

> 暮从碧山下，山月随人归。
> 却顾所来径，苍苍横翠微。
> 相携及田家，童稚开荆扉。
> 绿竹入幽径，青萝拂行衣。

杜甫《自京赴奉先县咏怀五百字》（节，天宝十四载冬作）：

> 南登紫阁峰，北揖昆明池。
> 老夫情怀恶，呕气不能食。

《新唐书·卷一二三·卢藏用传》：

> 司马承祯尝召至阙下，将还山，藏用指终南谓曰："此中大有佳处。" 承祯徐曰："以仆视之，仕宦之捷径耳。" 藏用惭。

## 额外章节：盛唐终南诗人一览

| 诗人 | 主要终南踪迹 | 代表作 |
|---|---|---|
| 王维 | 蓝田辋川（属终南东段），半官半隐 | 《终南山》《终南别业》《辋川集》 |
| 李白 | 数次入山访道，至紫阁、太白 | 《下终南山过斛斯山人宿置酒》《古风·西上莲花山》 |
| 杜甫 | 天宝十三载（754）登紫阁峰；居樊川时常望南山 | 《望紫阁峰》《自京赴奉先县咏怀五百字》（"南登紫阁峰"） |
| 孟浩然 | 应试落第后曾欲隐终南，未成行；诗中常念 | 《岁暮归南山》 |
| 储光羲 | 田园派，多写终南北麓农事 | 《同王十三维偶然作十首》 |
| 卢藏用 | "终南捷径"主角，景龙间居山 | （无诗存，只留典故） |
| 司马承祯 | 道家，曾入山；后归天台 | 《坐忘论》《天隐子》 |

盛唐诗人中，未到过终南山的几乎数得过来。"南山"在盛唐诗里是个高频词——白居易后来作《长恨歌》"骊宫高处入青云"——骊宫骊山，亦属终南山系东端。
