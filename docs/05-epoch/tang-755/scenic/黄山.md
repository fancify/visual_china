# 黄山

## 一句话

歙州黟县境内的一座大山——**唐天宝六载（747）玄宗诏改"黟山"为"黄山"**，因轩辕黄帝传说；755 时是江南尚不显的一座山、改名才 8 年；唯李白一首《送温处士》留诗——**真正显于明清以后**。

## 形貌

黄山在江南西道歙州新安郡**黟县**境内（今安徽省黄山市黄山区、黟县、歙县三县接界）——东西约 40 公里、南北约 30 公里，主峰**莲花峰**海拔 1864 米。

地势：黄山主体由花岗岩组成——独成一片山系、不与他山相连。**整片山系峰峦密集**——"七十二峰"（晚出概念，盛唐未必有此数；本 doc 不预支）。主要高峰有莲花、天都、光明顶、玉屏等——海拔皆超 1800 米。**山脚下**是徽州 / 黟县盆地——盛唐时是新安江上游一片相对偏僻的山区。

**唐时黄山的物理特征对盛唐人尚不显**——主要因黄山位于江南腹地、距长安五千余里、山中无大寺无名道观、亦非五岳。**改名前的"黟山"** 是江南山中一处普通山——黟县因"黟山"得名（黟字本义"乌黑"——黟山因主体山石乌黑得名）。

**黄山的真正物理特征**——花岗岩石峰、奇松、云海、温泉、佛光——**多在明清以来才被系统记录与游历**。盛唐时人来黄山者少、山中游道未通、绝大多数高峰盛唐前无人登。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

黄山在唐人的回望里，是一座**刚得新名的山**。

**先秦至两汉**——黟山名见《山海经》（一说《海内东经》之"三天子都山"即此，史源略异）；《汉书》《后汉书》皆不详。**两晋至南北朝**——黟山在江南山中默默无闻、文献零散。**隋开皇间**——歙州境内置黟县，黟山为县内一山。

**唐开国后**——黟山仍是江南腹地一座山、史载零散。**转折在玄宗朝**——**天宝六载（747）六月十七日**，玄宗诏改"黟山"为"黄山"（《唐会要》《太平御览》引唐代地志）。**改名缘由**：道经《周书异记》《续仙传》等记轩辕黄帝在此山炼丹、飞升（道教传说）——玄宗朝崇道达鼎盛、改地名以应道教神话，是玄宗朝崇道叙事的一环（**至于"为塑造李唐与轩辕政治联系"则为后人推测**）。**这是黄山名号在历史上的起点**——之前千余年皆称"黟山"。

**改名后 8 年至 755**——黄山作为山名渐入唐代地志、诗咏。但**实际游历者甚少**——李白《送温处士归黄山白鹅峰旧居》（天宝间作）"黄山四千仞、三十二莲峰"是唐前期黄山诗的代表（"四千仞"是夸张数，按唐尺一仞约 1.8 米计 7200 米，远超实际海拔；属诗中常见夸张）。李白此诗系送友人温处士归隐黄山——并非李白本人登过黄山。

### 此刻

755 年的黄山归**歙州**（治歙县、今安徽歙县）管辖；山主体在歙州属下**黟县**境内——县城北约百里。**改名才 8 年**——盛唐文人嘴里"黄山"二字尚是新名、不少人仍习惯称"黟山"或"黟黄之山"。

**山中状态**：**盛唐黄山中佛寺、道观皆稀**——史源记盛唐黄山中有道士隐修传说、但具体规模不详；亦无大型佛寺。**山下黟县境** 是徽州相对偏僻一隅——盛唐徽州（歙州）作为江南西道之东南州、户口与文化分量远不及苏州、洪州、扬州等大州。

**755 时盛唐人对黄山的实际认知**：
- **名字** 改自黟山才 8 年——尚是新名。
- **轩辕黄帝传说** ——道教属性、玄宗朝官方支持。
- **李白诗** 已传——但李白本人未必登过黄山。
- **实际游历者** 极少——盛唐黄山几无成熟登山道。

天宝十四载的黄山——**是江南山中一座"刚得名、未显诗坛、无大寺、少游者"的山**。这与今天黄山"国家级风景名胜区"、"世界自然与文化双重遗产"的地位形成巨大反差。

### 此刻意味

对天宝十四载的黟县人来说，黄山是**县北那座祖辈说过的旧黟山、近些年被朝廷改了名**——日常采药、樵伐、偶有道士入山隐修。对长安那一代士人来说，黄山是**江南腹地一座道教传说之山**——玄宗改名后入官方记录、但实际未入主流文学。对盛唐文人来说——黄山的诗咏密度远不及华山、嵩山、终南、庐山——李白一首是孤峰。

唐人尚不知道——再过 870 年（明万历四十四年/1616 起），徐霞客 (1587-1641) 两次游黄山（1616、1618 各一次）、留两篇《游黄山日记》、自此黄山"奇松、怪石、云海、温泉"四绝传天下；再过 900 年（明末清初），渐江、石涛、梅清等画家形成"黄山画派"——把黄山推为中国山水画的灵感源泉之一；再过千年——**黄山会成为"中国第一山"的现代旅游 anchor**。**755 时盛唐人嘴里的"黄山"，与今天人嘴里的"黄山"，文化分量天差地远**。

## 子条目

### 黟山改黄山（天宝六载/747）

**唐天宝六载（747）六月十七日**——玄宗诏改"黟山"为"黄山"（《唐会要》《太平御览》引唐代地志；正史《新唐书》《旧唐书》本纪未明记此事）。**改名缘由**：道经《周书异记》《续仙传》记轩辕黄帝在此山炼丹、飞升——道教传说。**玄宗朝改名背景**：李唐皇室追祖老子、合轩辕；天宝间玄宗崇道达鼎盛、改地名以应道教神话、是玄宗朝政治叙事的一环。**此改名后 8 年至 755**——黄山新名渐入唐代地志诗咏；但旧名"黟山"仍广为盛唐人沿用。

### 莲花峰 + 主峰群

黄山主体由数十座花岗岩石峰组成——盛唐时具体峰名尚未系统化（"七十二峰"是晚出概念）。**唐宋后定型的主要高峰**：
- **莲花峰**：海拔 1864 米——黄山最高峰；得名"莲花"因九峰相聚如莲——但此名晚出。
- **天都峰**：海拔 1810 米——以"天都"得名意为"天上都会"；此名亦后世。
- **光明顶**：海拔 1860 米——黄山中部一处大平台。

**盛唐时这些具体峰名是否已有**——史载零散、本 doc 不预设。**李白《送温处士归黄山白鹅峰旧居》中"白鹅峰"**——是否即今黄山的白鹅岭尚有异说；本 doc 不强定。

### 李白《送温处士归黄山白鹅峰旧居》

**李白唯一一首明确写黄山的诗**——天宝间作（具体年代史源略异）。**诗云**："黄山四千仞、三十二莲峰；丹崖夹石柱、菡萏金芙蓉……"——是李白送友人温处士归隐黄山白鹅峰的赠别诗。**"黄山四千仞"是夸张**——按唐尺一仞约 1.8 米计 7200 米、远超实际海拔 1864 米；属唐诗常见夸张笔法。**"三十二莲峰"** 亦未必精确——是李白对黄山峰群印象的诗化表达。**李白本人未必登过黄山**——此诗是送行诗、不是登山诗。

### 山中道士与轩辕黄帝传说

黄山改名所依据的传说——**轩辕黄帝在此山炼丹、飞升**。**道经《周书异记》《续仙传》等记**此说——但这些道经的年代多在南北朝至唐间陆续成书。**盛唐黄山中有道士隐修**——但具体哪些道士、何时入山、几人为众——史载零散。**轩辕黄帝传说在黄山的物理 anchor**——后世指"轩辕峰"、"丹井"、"飞升峰"等；这些具体地名多是宋元后陆续附会。

### "黄山"vs 五岳（一山之政治分量差）

**黄山虽得唐玄宗诏改名号**——但**未列五岳**。**五岳**：东岳泰山、南岳衡山、西岳华山、北岳恒山、中岳嵩山——皆有"王号"（天宝五载/746 玄宗封）。**黄山仅得新名、未获岳号**——这是 755 时黄山在国家祀典中的位置：**有名、无礼**。这一点与黄山在今天作为"中国第一山"的地位形成巨大反差——**今黄山的地位多由明清以来旅游 / 文学 / 绘画累积形成**，与盛唐 755 时人的认知完全不同时空。

## 历史长河（755 → 今）

**中晚唐**　755 后黄山几无文献——中晚唐诗人鲜有黄山咏；黄山作为"轩辕道教山"的官方身份延续、但实际游历者极少。**唐末黄山几乎退出文学视野**。

**两宋**　**北宋初**黄山仍是江南山中一处"虚名"——朝廷未敕修山中佛寺、亦无大型敕祀。**南宋**——黄山在江南宋金对峙中处于宋境腹地、未受兵燹；但仍非文学游历核心。**南宋绍兴间**始有黄山纪游诗（如曾几等）——但密度仍低。

**金元**　元代黄山中始有较成规模的佛寺（如祥符寺前身），但游历仍稀。

**明清**　**明万历至清初**——黄山地位发生根本变化。**徐霞客（1587-1641）两次游黄山**（万历四十四年/1616 + 万历四十六年/1618）——留两篇《游黄山日记》——"五岳归来不看山、黄山归来不看岳"（此谚相传徐霞客所言、但《游黄山日记》原文未明见此句、属后世附会）。**明末清初渐江、石涛、梅清等画家形成"黄山画派"**——黄山自此进入中国山水画的核心。**清康熙间**朝廷诏修黄山祥符寺；乾隆朝多次南巡未亲至但有遣使。**清代黄山的文学游历密度终于超过盛唐百倍以上**——但物理上的黄山——花岗岩石峰、奇松、云海——盛唐 755 时所立的还是这些石峰。

**民国至今**　1949 年后黄山先列省级风景名胜区、1982 年列国家级首批；**1990 年列联合国教科文组织世界文化与自然双重遗产**。**今黄山每年游客数千万人次**——是中国旅游 / 摄影 / 山岳画的核心 anchor。**唐人在 755 所见的那座"刚改名 8 年、无大寺无名道观"的山**——与今游人所见的"黄山"，地理实体同一座，文化分量天差地远。
