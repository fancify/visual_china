# 潼关

## 一句话

**关中东大门、长安东出第一关**——黄河南岸 + 秦岭北麓夹峙处；**至德元载（756）六月哥舒翰潼关之败**——失关西破、玄宗自此 西奔入蜀；**755 时仍立、是长安东向最重要的物理 anchor**。

## 形貌

潼关在唐关内道**华州华阴郡境**——具体在**黄河南岸 + 秦岭北麓夹峙之处**（今陕西省渭南市潼关县境）。**地势**：**黄河自陕北南流、在潼关一带转向东**——与**秦岭北麓**夹成约 5-10 公里宽的咽喉地带；**关城**立咽喉中央——南依秦岭、北临黄河、西望华阴县城（华州治）、东出函谷关一带。

**潼关的地理战略价值**：**关中盆地东出 / 中原西入的唯一咽喉**——南面是秦岭主脊（峻峭难越）、北面是黄河天堑（无桥不可渡）；**潼关在中间狭窄通道上扼守**——一夫当关、万夫莫开。**这是中国地理上最有名的"关塞"之一**——与函谷关、剑门关、玉门关等并列。

**关城形制**：盛唐潼关的具体形制 + 周长 + 门数史载零散——本 doc 不预设细节（今所见关楼 + 关墙多为明清重修主体）。**潼关西门外**——是关中平原与东境的过渡地带（黄河南岸沿岸的滩涂 + 麦田）；**潼关东门外**——是函谷古道、向洛阳东出的通衢。

**潼关与函谷关的关系**：**函谷关**是**战国秦关**——西汉武帝元鼎三年（前 114）东徙至新安（今河南新安）；**潼关**自东汉建安元年（196）曹操弃函谷新关、于今潼关一带（黄潼会合处）筑潼关——自此潼关取代函谷关成关中东大门。**唐时函谷关属虢州、潼关属华州**——两关相距约 80 公里、皆是关中东冲；但**潼关分量远超函谷**。

**物候**：黄河南岸 + 秦岭北麓——温带季风气候；夏多雨、冬寒；黄河泛滥在唐时不大、但风沙 + 雪较多。**关城内外多有驿馆 + 戍兵 + 关吏** —— 是盛唐边塞 + 商旅 + 使节最频繁的关塞之一。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

潼关在唐人的回望里——是**关中东大门 + 长安东出第一关 + 千年战略要塞**。

**潼关的设立**：**东汉建安元年（196）曹操弃函谷新关、于今潼关一带筑潼关**——自此潼关取代函谷关成关中东冲。**自东汉至隋唐 + 唐**——潼关一直是关中东出的咽喉；**北朝至隋唐间历代帝王（北周武帝平北齐、隋文帝灭陈、唐高祖入长安）**——皆以潼关为东西大军调动的物理 anchor。

**唐潼关的政治军事地位**：**唐承隋制、置潼关县属华州**；**潼关守军由唐廷直接指挥**——是关中防御体系的核心。**自唐初至盛唐**潼关无大战；关塞按例修缮、关城严整；具体常额守军、防御工程史载零散，本 doc 不写无源数字。

**潼关与长安的关系**：**长安东行 + 经新丰县 + 至华州 + 入潼关**；**潼关东出 + 经函谷新关 + 至洛阳**——是盛唐两京驿道的核心一段（具体里程史源略异）。

**安史前夕**（即 755 这一刻）——再过半年安禄山起兵于范阳；**至德元载（756）安禄山西攻 + 围攻洛阳 + 西向潼关**；**六月哥舒翰潼关之败**——失潼关、关中无险可守；**七月玄宗西奔入蜀、过散关 + 入剑门 + 至成都**。**潼关之失是安史之乱的转折点**——**755 时潼关仍是关中东大门、是长安最稳的物理屏障**；半年后即失。

### 此刻

天宝十四载这一年的潼关——**关塞严整、守军 1 万、扼守关中东大门**。

**关城状态**：盛唐 755 时潼关主体仍立、有驿馆 + 戍兵营 + 关吏；具体形制 + 守军规模史载零散。**关城外**：东、西两道驿路——西通长安、东通洛阳。**潼关使**为唐廷直接派遣的官员。

**755 时潼关上的政治氛围**：唐廷对潼关的安全意识在玄宗朝相对放松——**因关中无战 138 年**；安禄山起兵前夜、玄宗朝仍未在潼关大规模增兵备战。**这是 755 时长安对边境的盲点**——半年后潼关失、关中告急。

**755 末 + 756 初的潼关命运**：**安禄山十一月起兵 → 十二月陷洛阳 → 756 西向潼关攻击**；**唐玄宗紧急调封常清 + 高仙芝守潼关、皆兵败**（756 春）；**改派哥舒翰守潼关**；**至德元载六月哥舒翰兵败潼关**——主要是玄宗强令出关而败；**潼关失守、关中无险可恃**；**七月玄宗西奔**。**这一段是潼关在盛唐唯一一段真正经历大战的时间——但发生在 755 之后**。

### 此刻意味

对天宝十四载的潼关 / 华州人来说——潼关是**自家境内的关塞**：日常驿路过往、戍兵守关、商旅入关纳税。对长安那一代士人来说——潼关是**关中东大门**：东出 / 入京必经；**盛唐诗中"潼关"二字偶有出现**（高适、岑参等边塞诗或涉），但**潼关的诗咏密集化在 755 后**（杜甫《潼关吏》《北征》等多作于乾元 / 至德间）——是中唐战乱诗的核心地标。

唐人尚不知道——再过半年、潼关将经历盛唐 138 年和平的终结：哥舒翰之败 + 失潼关 + 玄宗西奔；**杜甫将作《潼关吏》《北征》等诗**——把潼关写进中唐诗最沉重的一页。**今潼关古关已不复唐时形制**：**清咸丰、同治间多次大修**；**1959 年因三门峡水库工程、潼关古城被拆迁/迁建**（具体淹没/拆除范围需据潼关县地方志 + 文保报告细化）；今游人来潼关、所见的是**潼关古关址纪念碑 + 部分迁建关楼**——**唐人在 755 所见的那座潼关古关**：千二百多年后**主体已不存**——这是潼关与其他唐六陵不同的物理终点。

## 子条目

### 黄河 + 秦岭夹峙的咽喉

**潼关之险在地理**——**北面黄河、南面秦岭**：黄河自陕北南流、至潼关转向东；秦岭主脊在潼关南面峻峭——**夹成约 5-10 公里宽的咽喉地带**。**关城就立咽喉中央**——南北约 5 公里、东西约 1 公里。**这种"两山夹一水、一关守咽喉" 的地形**——是中国地理最显的关塞形制；**剑门关（秦岭主脊东端剑门栈道）、嘉峪关（祁连山-黑山夹河西走廊）等亦为类似形制**。

### 函谷关（旧关，西汉东徙后已退冲要）

**函谷关**：战国秦关、原在**今河南灵宝境**——是先秦至秦汉的关中东大门。**西汉武帝元鼎三年（前 114）东徙函谷关至新安**（今河南新安东）——成"新函谷关"；**东汉建安元年（196）曹操弃函谷新关、筑潼关**——自此函谷关退冲要、潼关成关中东大门。**唐时函谷关属虢州**——是黄河南岸的一处历史地标、但已非交通要塞。**盛唐人对函谷关的认知**：是史书 + 地理常识、不是日常驿路过往的关塞——与潼关地位差异极大。

### 潼关之败（至德元载/756 六月）

**安史之乱的转折点**——**至德元载（756）六月**哥舒翰兵败潼关：
- **背景**：**755 年十一月安禄山起兵于范阳**；**十二月叛军陷洛阳**；**756 年正月安禄山在洛阳称帝、号大燕**；**叛军继续西向、围攻潼关**。
- **唐军守潼关**：**755 末玄宗派封常清守洛阳、高仙芝守潼关 + 陕郡**——皆失利；**封 + 高皆被诛**（756 春）。**改派哥舒翰为兵马副元帅、守潼关**——率十万军 + 兼河西陇右两镇的精兵。
- **败因**：**玄宗强令哥舒翰出关迎战**——哥舒翰反对（认为应据潼关守险、待敌内乱）；但玄宗朝廷被杨国忠等催促；**哥舒翰被迫出关**。
- **败战**：**756 年六月初**——哥舒翰率军出关、在**灵宝**遭叛军伏击——**唐军大败**、十万军损失大半；**哥舒翰被叛将所掳、降叛**。
- **失潼关后果**：**潼关失、关中无险可恃**；**七月玄宗自长安出延秋门西奔**——经金牛道入蜀；**马嵬驿杨贵妃死**；**至德元载/756 八月玄宗至成都**——蜀中流寓十余月。

**潼关之败是中国军事史最有名的"皇帝强令出关而败"案例之一**——亦是盛唐转衰的物理 anchor。

### 杜甫《潼关吏》（中唐战乱诗的核心）

**杜甫《潼关吏》**——**乾元二年（759）作**；**755 时未作**。诗记杜甫自洛阳西归长安、过潼关、与潼关吏对话；潼关吏自夸"潼关百万师"、"修关还备胡"；杜甫"哀哉桃林战、百万化为鱼" 怀念哥舒翰之败。**这首是中唐战乱诗的核心**——把潼关写成"安史之乱物理 anchor + 唐廷战略失误的代名词"。**自此潼关在唐诗中的位置骤升**——比盛唐时高一档。

### 西门 + 东门（两道开放方向）

**潼关西门**：**向关中开**——是长安东出第一关的西出口；**西门外**沿黄河南岸 + 渭水南岸有驿路至华阴县城（华州治）再至长安。**潼关东门**：**向中原开**——是关中东出的东入口；**东门外**沿黄河南岸有驿路至函谷新关 + 至洛阳。**两门是潼关的物理双面**——西面对关中、东面对中原；**盛唐过潼关者必经此两门**。

### 潼关的水关（黄河南岸入潼关）

**潼关北面临黄河、有"水关"**——是黄河漕运的关塞。盛唐黄河漕运：自洛阳含嘉仓溯黄河西上、过三门峡（陕州境）险段、入潼关水关、再西达长安。**潼关水关的工程**：黄河南岸有夯土堤 + 水门 + 缓流设施；**漕运船只入关后停泊关城旁仓库**——是关中粮食安全的物理 anchor。**安史前夕（755 末）**——三门峡 + 潼关水关漕粮仍按例运至长安；半年后失。

## 历史长河（755 → 今）

**中晚唐**　**至德元载（756）六月哥舒翰潼关之败**——失关西破；**七月玄宗西奔入蜀**；**至德二载（757）九月唐军郭子仪 + 李光弼 + 回纥兵东出潼关、复长安 + 洛阳**——潼关失而复得。**中晚唐潼关常被战争反复争夺**：藩镇之乱、黄巢之乱等皆涉及潼关。**杜甫《潼关吏》《北征》等诗**——把潼关写进中唐诗最沉重的一页。

**唐末黄巢 + 五代**　**广明元年（880）黄巢入潼关、陷长安**——潼关失守是黄巢之乱的关键一步；**五代间潼关多次易主**。

**两宋**　**北宋初潼关重修**——作为关中东大门继续；**北宋元丰间潼关属陕州**。

**金元明清**　**明初洪武七年（1374）大规模修筑潼关**——形成今所谓"潼关古城"的明清主体格局；**清咸丰、同治间多次大修**——含**潼关古关楼**、城墙等。**清代潼关是关中东大门 + 黄河漕运咽喉**。

**民国至今**　**1949 年后潼关古关初列省级文物保护**。**1959 年三门峡水库建设**——为修水库蓄水、**潼关古城被拆除**（黄河水位上升、潼关古城淹没区扩大）——**潼关古关址今部分沉于水下、部分迁建**。**今潼关古城遗址**：在新址重修部分关楼 + 立"潼关古关址" 纪念碑。**唐人在 755 所见的那座潼关古关**——千二百多年后**主体已不存**：盛唐潼关的物理终点是 20 世纪三门峡水库工程。这是中国关塞史最大的一次"水库吞古城"事件。
