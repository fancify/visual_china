# 洛阳-江都道

## 一句话

**唐廷漕东南粮米的主线** + **江南赴京使节路**——自洛阳东南行、沿通济渠经汴州 + 宋州 + 泗州 + 楚州、再过邗沟入江都郡治扬州城；**水陆并行**：水路是隋大业元年开的通济渠 + 重浚的邗沟、陆路是沿渠两岸的驿馆驰道；**755 时是盛唐财政命脉**——半年后安史起、东都陷而江都未陷、漕运一度断绝。

## 形貌

洛阳-江都道水陆并行——水路里程稍长于陆驿、但行船省力。**自西北向东南沿途要节** + **大致段落**：

**洛阳-汴州段**（洛阳 → 板渚 → 郑州 → 汴州）：
- **洛阳河南府治洛阳县**——道之起点；漕粮抵此入含嘉仓；
- **板渚**（巩县东、河阴县境）——通济渠引黄河水南下处、隋大业元年开渠口；
- **郑州治管城县**（今郑州市）——河南道北部、通济渠西岸；
- **汴州治浚仪县**（今开封）——通济渠中段最大都会、河南道东部漕运枢纽。

**汴州-泗州段**（汴州 → 宋州 → 虹县 / 埇桥 → 泗州）：
- **宋州治宋城县**（今商丘）——通济渠南岸；
- **虹县**（今泗县一带）+ **埇桥**（今宿州市区，755 时属徐州符离县境，紧临通济渠）——通济渠经此入泗水；**注**：宿州元和四年（809）始置、755 时不存在；
- **泗州治临淮县**（今泗洪 / 盱眙之间）——通济渠入淮河口；漕舟自此南转入淮 + 邗沟。

**泗州-江都段**（泗州 → 楚州 → 江都郡治扬州）：
- **楚州治山阳县**（今淮安）——邗沟北端、淮河南岸；漕舟自淮河入邗沟；
- **邗沟**（春秋吴王夫差开、隋大业元年重浚）——自山阳南下经高邮 + 至江都；
- **江都郡治扬州城**（今扬州）——道之南端、淮南节度治、长江北岸第一大都会。

**驿馆制度**：**《唐六典》"三十里一驿"** 是通制——但此道水陆并行、驿馆与渡口 / 水驿夹杂；陆驿沿渠堤设、水驿置于津渡。**《唐六典》载"水驿二百六十"**——通济渠 + 邗沟一线占其相当部分。漕舟自江都至洛阳行月余；急讯陆驿则数日可达。

**地势**：**洛阳出板渚段**——伊洛盆地东缘 + 黄河南岸；**汴州至宋州**——黄淮平原核心、地势平坦、渠水平缓；**泗州一带**——淮河北岸 + 洪泽未成湖（淮水东注入海、洪泽湖至南宋黄河夺淮后始大）；**楚州至江都**——里下河平原 + 邗沟两岸湖荡相连。

物候：温带季风过渡至亚热带、四时分明、春多桃花水、夏多霖、秋高、冬有薄雪不冻渠；**通济渠两岸隋时夹种杨柳**——开元 / 天宝间仍存、是盛唐物质繁华最具象的画面之一。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

洛阳-江都道在唐人的回望里——是**隋炀帝南巡道 + 唐廷漕运第一线**。

**水路的历史**：
- **春秋末**：吴王夫差为北上争霸开**邗沟**——自江都通至淮河；中国最早的人工运河之一；
- **战国 + 秦汉**：邗沟续用、但中原至江淮的水路始终未联通；
- **隋大业元年（605）**：隋炀帝发河南、淮北诸郡民开**通济渠**——自洛阳板渚引黄河水、东南至盱眙入淮；据《隋书》载工程由**阎毗、宇文恺**等主持；**同年重浚邗沟**——至此洛阳与江都水路联通；隋炀帝三下江都皆经此渠；
- **隋大业六年（610）**：开**江南河**——自江都南渡长江、过京口（润州）、至余杭——大运河南段成；
- **隋大业四年（608）**：开**永济渠**——自洛阳引沁水北至涿郡——是大运河北段；但**永济渠是北线**、与洛阳-江都道无关。

**唐承隋制**：唐高祖、太宗朝通济渠 + 邗沟续用、为漕运主线；高宗 / 武则天朝多次"就食洛阳"——皆赖此渠输江淮粮米北上；**开元二十二年（734）裴耀卿改漕运法**——置河阴仓 + 集津仓 + 盐仓 + 永丰仓接驳、**自此关中粮食有保障**——玄宗朝再未东巡 19 年（至 755）。

**江都郡 + 扬州的州名沿革**：**隋大业改江都郡 + 治扬州城**；**唐武德三年（620）复扬州**；**天宝元年（742）改扬州为江都郡**；**乾元元年（758）复扬州**——**755 时正名是江都郡、治扬州城；行政语境"江都"、地理俗名"扬州"或"广陵"**。

**驰道的具体使用者**：**漕舟**（江淮粮米北上 + 关中布帛南下）+ **江南赴京使节**（盐铁使 + 转运使 + 节度奏报）+ **商旅**（盐 + 丝 + 米 + 茶）+ **僧侣**（扬州大明寺 + 鉴真东渡前的本山）+ **文人**（江南士子赴京应举）——是盛唐流量仅次于长安-洛阳驰道的官道。

### 此刻

天宝十四载这一年的洛阳-江都道——**繁忙、按例运行、漕粮按岁北输、即将经历安史浩劫**。

**渠道状态**：盛唐 755 时通济渠 + 邗沟沿渠堤完整、纤路 + 驿馆相次、漕舟接连；渠通漕舟、唐令常制三十里置驿、自洛阳至江都漕舟行月余、急驿数日即至；**沿途府郡**（天宝元年/742 改州为郡）：河南府（洛阳） + 荥阳郡（郑州） + 陈留郡（汴州） + 睢阳郡（宋州） + 临淮郡（泗州） + 淮阴郡（楚州） + 江都郡（扬州）——通济渠主线不经徐州治，但徐州符离县/埇桥临渠；沿途皆有府尹（河南府）/郡守（其他郡）驻、漕官督运、关吏验牒。

**漕运的实际数字**：**裴耀卿改漕运法后江淮粮米北上**——**开元二十五年（737）至天宝元年（742）间累岁运米七百万石**（《新唐书·食货志三》）；**自此关中仰给江淮成为盛唐财政基本面**。**江都郡**——是这条命脉的南端起点；江淮 + 江南诸道粮米先集江都、再溯邗沟北上至楚州 + 入淮 + 转通济渠西北至洛阳含嘉仓 + 再陆运过三门峡至关中太仓。

**通济渠两岸渠柳**：渠两岸植柳，隋开通济时已成定制；唐承之——开元 / 天宝间渠两岸柳荫相连、是盛唐物质繁华的图景之一。（中唐白居易《隋堤柳》咏隋堤渠柳，属后世文学回望，引文见"历史长河"段。）

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九即起）；**天宝十四载十二月丁酉叛军陷东京洛阳**（公历入 756 年 1 月）——通济渠北端起点失；**江淮漕路自此中断**；**江都郡 + 淮南节度未陷**——成为唐廷江南半壁的物理 anchor。**至德元载十二月永王李璘自江陵东下、舟师顺长江欲取广陵（扬州）**、谋自立；**至德二载（757）二月**永王败死、江都未陷；**至德二载（757）十月唐军复东都洛阳**——但通济渠沿线已疮痍、**漕路至广德元年（763）后**才陆续恢复。

### 此刻意味

对天宝十四载的河南 + 淮南人民来说——洛阳-江都道是**自家境内的命脉**：日常漕舟 + 商旅 + 使节往返；汴州城内"舟车辐辏、半天下之财赋出于江淮、皆由此道而北"（《元和郡县图志》卷七）。对洛阳那一代士人来说——道是**江南的物理通道**：东都台省间盛传"江都春潮 + 扬州月色"；江南文人赴京应举、必经此道北上；落第东归、亦经此道南下。

唐人尚不知道——再过半年通济渠北端将断、漕粮停输、关中饥荒；**至德 / 乾元间江淮租赋赖此道苦输支撑唐廷半壁**——但漕路反复中断；**广德元年（763）以后刘晏改漕运法**——立**江船 + 汴船 + 河船 + 渭船**四段接驳——是中唐漕运的恢复 + 此道的再生。**唐末黄巢（879-881）自岭南北上至汴州**——亦经此道；**五代后周显德六年（959）周世宗征南唐**——亦自汴州循通济渠至寿州 + 江北；**北宋汴京（开封）即建于通济渠汴渠段**——大运河中段重心北移至汴京。

**今游人重走洛阳-江都一线**——所见的是**陇海铁路东段 + 京沪铁路南段 + 京沪高速 + 沪陕高速 + G36**的现代驿道——**盛唐人在 755 所走的那条洛阳-江都道**：千二百多年后路线大体在同一地理走廊上、但通济渠**主体已堙**（南宋黄河夺淮 + 元代会通河北移后通济渠淤废、明清间彻底湮没）、**邗沟一段尚通**（即今淮扬运河 + 京杭运河扬州段）；驿馆 + 渠柳几无原物——是中国漕运史最大的一次"物理改道"。**元代会通河 + 通惠河完成后**——形成北至大都南至杭州的连续水道，与隋唐通济渠走向不同；"京杭大运河"作为名词主要为近代以来概念，**755 时尚无"京杭"概念**、唐人称"通济渠 + 邗沟 + 江南河"三段。

## 子条目

### 通济渠（隋大业元年/605 开）

**自洛阳板渚引黄河水东南至盱眙入淮**——隋炀帝大业元年（605）三月开、八月成；据《隋书》载发河南、淮北诸郡民百余万、工程由阎毗、宇文恺等主持；两岸夹种杨柳。深可行龙舟；隋炀帝御舟下江都即循此渠。**唐承之**——通济渠又名**汴渠**或**汴河**（自汴州一段为名）；是唐漕运第一线。**通济渠经过的州县**：**洛阳 + 板渚 + 郑州 + 汴州 + 宋州 + 虹县 + 泗州**——**通济渠主线不经徐州治，但徐州符离县/埇桥临渠**。**南宋绍熙五年（1194）黄河南徙夺淮**——通济渠中下段被黄河水冲毁；**元代会通河北开**——通济渠彻底湮废。**今汴州（开封）+ 泗州（盱眙）一带尚存古河道遗迹**。

### 邗沟（春秋吴王夫差开、隋大业元年重浚）

**自山阳（楚州治；今淮安）至江都（扬州）**——中国最古老的人工运河之一。**春秋末（前 486）吴王夫差为北上争霸开**——自江都向北、引江水入淮；**隋大业元年（605）与通济渠同时重浚**——拓宽 + 加深 + 立水驿。**沿途要节**：山阳 + 宝应 + 高邮 + 江都；两岸植柳，唐承隋制。**唐承之**——邗沟是江南漕路北段；**至德 / 乾元间漕路反复中断**——但邗沟本身始终未湮；**北宋元祐七年（1092）苏轼知扬州**多次行此渠。**今邗沟一段尚为京杭大运河扬州段**——是隋唐三段大运河中**唯一未湮的一段**。

### 漕运（江淮粮米北上）

**盛唐财政基本面**——**关中仰给江淮**。**开元二十二年（734）裴耀卿三门转运体系**——置**河阴仓 + 集津仓 + 盐仓**等三门转运仓接驳，分段输运、避三门峡之险；**开元二十五年（737）至天宝元年（742）累岁运米七百万石**（《新唐书·食货志三》）；**自此关中粮食有保障**——玄宗朝再未东巡 19 年。**漕运是盛唐两京格局的物质基础**——洛阳-江都道是这套体系的中段。

**安史中漕路反复中断**——至**广德元年（763）刘晏始领转运使**重新立 + 恢复部分流量；**大历间刘晏漕运改革**进一步立**江船 + 汴船 + 河船 + 渭船**四段接驳（江船自江南诸州集江都；汴船自江都经邗沟入淮 + 转通济渠西北至河阴仓；河船自河阴仓溯黄河西上至太原仓〔陕州东〕；渭船自太原仓陆运过三门峡 + 再水运渭水入长安太仓）——**此为中唐口径，755 时尚未实施**。**自此中唐唐廷半壁赖江淮租赋苦输支撑**——**漕运史从此进入"江南依赖期"**。

### 江都郡 / 扬州 / 广陵（同一地、三名）

**州名沿革**：**隋大业改江都郡 + 治扬州城**；**唐武德三年（620）复扬州**；**天宝元年（742）改扬州为江都郡**；**乾元元年（758）复扬州**——**755 时正名"江都郡"、治"扬州城"、城又名"广陵"**。**淮南采访处置使治江都郡**——管辖扬、楚、滁、和、寿、舒、庐、光、蕲、申、安等淮南诸州；755 时具体使臣名号兹不强定（避免与中唐邓景山等任职年份混淆）。**江都城规模**：唐六雄之一（与长安、洛阳、汴州、益州、并州并列）；"扬一益二"——**扬州为天下都会第一、益州（成都）第二**。**安史中江都未陷**——是唐廷江南半壁的物理 anchor；永王李璘 757 东巡至江陵欲谋此地、败。**晚唐杜牧"二十四桥明月夜"等扬州诗**——是中唐 / 晚唐回望盛唐扬州的咏怀——**755 时杜牧未生**（杜牧 803 生）；不要将晚唐扬州诗反推至 755 实景。

### 汴州（通济渠中段都会）

**汴州治浚仪县**（今开封）——通济渠中段最大都会、河南道东部漕运枢纽（755 时正名陈留郡）。**《元和郡县图志》卷七**："汴州城内舟车辐辏、半天下之财赋出于江淮、皆由此道而北。" **755 末陈留陷落相关**：陈留郡太守张介然（兼河南节度采访使）等；**节度**：汴州本属河南道、未单立节度；**755 末安史叛军陷洛阳后陈留即危**——**至德元载（756）陈留陷**；**广德元年（763）以后渐复**。**北宋建都汴京（开封）即承通济渠汴渠段之利**——汴京 200 年盛世正是大运河中段重心北移的产物。

### 安史中漕路的中断与恢复（755-763）

**安史之乱中洛阳-江都道的变迁**：
- **天宝十四载十二月丁酉**叛军陷东京洛阳（公历入 756 年 1 月）——通济渠北端起点失；
- **至德元载（756）正月**安禄山在洛阳称帝；陈留 + 通济渠沿线渐陷；
- **至德元载（756）七月**玄宗西奔；**至德元载（756）十二月永王李璘自江陵东下、舟师顺长江欲取广陵（扬州）**；
- **至德二载（757）二月**永王败死——江都未陷；
- **757 年十月**唐军复东都洛阳——通济渠北段渐通；
- **758-762**漕路反复中断——关中饥荒数次；
- **763 年广德元年刘晏始领转运使**——改漕运法、四段接驳重立——**漕路渐复**。

**自此中唐漕运**：**江船 + 汴船 + 河船 + 渭船四段接驳**——是刘晏制；**唐廷半壁赖江淮租赋苦输支撑**——至唐末（黄巢之乱）漕路彻底崩溃。

### 通济渠两岸渠柳（盛唐物质繁华的物理 anchor）

**渠两岸植柳，隋开通济时已成定制；唐承之**——开元 / 天宝间渠两岸柳荫相连、是盛唐物质繁华的图景之一。**两岸驿馆 + 渠中漕舟 + 堤上纤夫 + 柳荫商旅**——是天宝年间江淮平原上的常态景观。**今通济渠主体已湮**——渠柳亦无存；唯邗沟一段尚通、扬州古运河两岸尚有近代补植柳。（中唐白居易《隋堤柳》咏隋堤渠柳为后世文学回望，引文见"引文"段。）

## 历史长河（755 → 今）

**中晚唐**　**755-763 漕路反复中断**；**广德元年（763）刘晏改漕运法**——四段接驳重立；**自此唐廷半壁赖江淮租赋苦输支撑**。**晚唐黄巢（879-881）自岭南北上至汴州**——经此道；**唐末漕路彻底崩溃**——是唐廷财政的终结之一。

**五代**　**后梁、后晋、后汉、后周以开封（汴州）为都；后唐都洛阳**——通济渠汴渠段（汴州一线）仍为五代核心漕道；**后周显德六年（959）周世宗征南唐**——自汴州循通济渠至寿州 + 江北、为北宋统一江南奠基。

**两宋**　**北宋建都汴京（开封）**——通济渠汴渠段是 200 年盛世的物质基础；**张择端《清明上河图》**所绘即汴渠汴京段的繁华——是通济渠在中国画史上的最后高光。**北宋末汴渠渐淤**；**南宋绍熙五年（1194）黄河南徙夺淮**——通济渠中下段被黄河水冲毁；**自此通济渠湮废**——隋唐大运河中段消亡。

**元明清**　**元代重开会通河**（1289 成）——自济宁北至临清、东至须城——**走山东直插北方**——与隋唐通济渠走向不同；会通河 + 通惠河完成后，形成北至大都南至杭州的连续水道（"京杭大运河"作为名词主要为近代以来概念）。**明清间会通河 + 邗沟 + 江南河成主漕路**——通济渠彻底退出。**邗沟一段（淮扬运河）+ 江南河（江南运河）至今仍通**。

**民国至今**　**1949 年后陇海铁路东段 + 京沪铁路南段 + 京沪高速 + 沪陕高速 + G36** 形成今日洛阳-扬州一线的现代交通——大体沿盛唐洛阳-江都道走廊。**今游人重走此道**——所见的是**铁路 + 高速公路 + 京杭大运河（邗沟段）+ 扬州古运河 + 开封古城**等历代遗存。**盛唐人在 755 所走的那条洛阳-江都道**——千二百多年后**通济渠主体已湮**（仅汴州 + 泗州一带存遗迹）、**邗沟一段尚通**；驿馆 + 渠柳几无原物——是中国漕运史最大的一次"物理改道"。
