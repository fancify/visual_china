# 金牛道

## 一句话

汉中至成都的一千一百里官道，唐人口中真正的"蜀道"；秦惠王伪以金牛诱蜀、五丁力士开山入秦的那条路——天宝十五载六月，玄宗循此入蜀。

## 形貌

金牛道不是一条平直驿路——倒像沿嘉陵江、白龙江一路凿出的一段长石栈。

自梁州治所南郑（今汉中）西南行，先过沔水（汉水上游），入勉县——汉中盆地由此收口，山势起。再南，过宁强（唐三泉县）入嘉陵江系——道路自此沿江而行，时凿岩壁、时铺栈木，一程比一程险。沿途江流自北而南：嘉陵江出甘陇南山，与白龙江于昭化（葭萌）合流，再南至广元（利州）——城就立在江口。利州城南，是**剑门**——一道横在川北的灰岩大壁，唐人称"七十二峰相连，刀削斧凿"。剑门一过，山势渐缓，由剑州至绵州、汉州，至成都平原。

全程约**一千一百里**，要翻三道大梁：勉县至宁强一段秦岭余脉；宁强至广元的米仓-龙门山相接处；广元至剑门关的大剑山。沿途有名的"关"七八处——金牛、五丁、七盘、葭萌、剑门、棋盘——皆设于山口或江畔。所谓"栈道"，是凿岩为孔、插木为梁、铺板为道的悬空走廊——遇绝壁则栈，遇深谷则桥，遇险滩则改沿岸开凿。沿江最有名的一段在朝天峡（明月峡）——栈孔密布岩壁，至今石上孔印行行可见。

道上设驿。唐时**三十里一驿**，沿途馆驿计有褒城、武休、长举、三泉、嘉陵、朝天、利州、剑门、剑州、武连、绵州、罗江、汉州——这些是金牛道上有名的驿名，《元和郡县图志》一段一段列得清楚。驿是唐代官道之骨——传旨快马一日二三百里，自长安发文，凉州、益州皆五至七日可到。

物候在金牛道上分两层。北段汉中盆地春耕早、稻一熟；过剑门入剑南，温润转骤——蜀地"少雪多云"自这条道的南口开始。沿途多见的是巴蜀松、红豆杉、油桐、楠木——李白说"上有六龙回日之高标，下有冲波逆折之回川"——写的便是嘉陵江、白龙江之间那一段最险的山水。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

金牛道的来路，比大唐本身长得多。

最早是**传说**。秦惠王后元年间（约前 316 前后），秦欲伐蜀，蜀道险绝、兵不可入。秦惠王乃刻五头石牛，每日在牛尾下置黄金数斗——遣使告蜀王：秦有神牛，能拉黄金粪。蜀王贪之，遣**五丁力士**入秦迎牛——五丁力士拽石牛回蜀，沿途凿山开道，至蜀。秦因之得入蜀路，张仪、司马错随道伐蜀（前 316）——古蜀国由此入中夏。后人遂呼此道为"**金牛道**"——李白《蜀道难》"地崩山摧壮士死，然后天梯石栈相钩连"，写的正是这五丁开山的传说。**这是一则传说，但唐人当作来路的源头**——故事本身是不是史实不再重要。

实际营建始于秦惠王后元至秦昭王间（前 4 世纪末至前 3 世纪中）。张仪、司马错伐蜀之后，秦于沿线置县——汉中、葭萌（昭化）、广汉、雒（汉州）、成都——一条连接关中入巴蜀的官道由此成型。三国时蜀汉据此道，丞相诸葛亮**六出祁山**，南路所恃即金牛道——汉中是大本营，剑门是门户，葭萌、白水、沔阳是粮草中转站。蜀汉灭亡（263）邓艾偷渡阴平、钟会主力攻剑门——金牛道作为蜀地命门的角色，自此固定下来。

魏晋以后，金牛道时通时塞——成汉、北周、隋皆曾在沿线置县驿。**隋开皇间**（约 590s）大规模整修栈道，置驿三十二所；唐贞观、开元间又两次大修——沿江凿孔为栈、依山改道为路。开元六年（718）剑南节度使章仇兼琼修筑剑门关城——剑门作为蜀地咽喉的形制，到 Tang 才完整。

### 此刻

天宝十四载（755）的金牛道，是**蜀道八条**里最重要的一条。

蜀道八条——褒斜道、傥骆道、子午道、陈仓道（故道）、金牛道、米仓道、荔枝道（涪州至长安专送鲜荔枝的支线）、阴平道——前四条皆自关中翻秦岭至汉中盆地，金牛道再自汉中入蜀。**金牛道是后半段唯一的官道**——其他几条入蜀路线（米仓道走巴中、阴平道走平武）皆为辅助。蜀地一切的官府公文、军报、税赋、贡品，**主要都走金牛道**。

道上的驿务此时繁忙。蜀锦、铜铁、井盐自益州起，经汉州、绵州、剑州、利州出剑门，至梁州转褒斜或傥骆入关中，最后至长安——蜀贡每年估约百万匹锦、井盐数千石、铜铁器若干，都是这条道运的。反向南下的是关中产物：北方马匹（剑南军镇用）、北方麦面、关中纸笔。文书更不必说——剑南节度使的公文、南诏战报、雅州黎州的边事，皆经此道日夜传递。

沿途几位关守、几位驿使的姓名留在《元和志》里——多数是再普通不过的小官，但他们守的是这条道。**剑门关**的关令名义上由剑州刺史辖，但实际由军戍——开元后大约有戍卒数百，关楼一座，门钥归朝廷。

那一年也有事。**杨贵妃的鲜荔枝**走的不是金牛道，是涪州至长安的"荔枝道"——专为贵妃辟的一条快递线，每年五至七月，岭南、涪州的鲜荔枝以马匹日夜驰送，七日内可至长安。荔枝道与金牛道一支相接——自涪州、巴东沿大宁河出三峡，转南阳、汉中、再循金牛道北段至关中。这条路用的是金牛道的一部分驿——杜牧《过华清宫》"一骑红尘妃子笑，无人知是荔枝来"，所写正是其末段。

对天宝十四载的金牛道驿吏来说，蜀道还是那条蜀道——只是次年六月会有变化。

### 即将到来

天宝十五载（至德元载，756）六月，潼关失守、玄宗夜出延秋门，第三日（六月十三）至马嵬。禁军杀杨国忠、缢杨贵妃后，玄宗一行继续南行——皇太子李亨北赴朔方，玄宗、贵妃姊妹、陈玄礼、高力士、宗室子弟数百人，循以下路线入蜀：

**长安 → 武功 → 扶风 → 陈仓 → 散关 → 凤州 → 兴州（略阳）→ 利州（广元）→ 剑门关 → 剑州（剑阁）→ 绵州 → 汉州 → 益州（成都）**

——前段自陈仓入散关走的是**故道**（陈仓道），过凤州后接的是嘉陵江沿岸的金牛道。**全程一千七百余里、耗时五十余日**——七月二十八日（公历 8 月 28 日）抵成都。中间剑门关一段最险，玄宗于剑门驻马而望，李白《剑阁赋》据传即为此事作（一说乾元年间作，唐两说并存）。这是金牛道作为唐廷"最后退路"的一次真正使用——金牛道的传说，从五丁迎牛到天子奔蜀，跨了一千年。

## 沿途关隘 / 古驿（doc 内子条目，不另立 POI）

### [剑门关](../relics/剑门关.md)

金牛道咽喉，大剑山七十二峰中横。秦汉以来即为入蜀第一关；三国时诸葛亮即以此为屏，钟会伐蜀正面攻坚于此。开元六年（718）剑南节度使章仇兼琼正式修筑剑门关城——夯土城墙、铁裹关门、东西两山为夹。唐时关属剑州——杜甫乾元二年（759）入蜀过剑门，作《剑门》"惟天有设险，剑门天下壮"——这一首是金牛道最早一首"剑门体"。**剑门关已列为独立 POI doc**，详见链接。

### 葭萌关

利州城北、白龙江汇嘉陵江口——古葭萌县治。战国蜀王封弟苴侯于此，名"葭萌"；秦灭蜀后置葭萌县——是金牛道上最早一个建制县。三国时蜀汉以此为前进基地——刘备入蜀初即驻葭萌，"先主屯葭萌，未即讨张鲁"（《三国志·先主传》）。唐为利州昭化县治——道上自葭萌进剑门，前后约百二十里，江口至关口一程是金牛道最经典的一段。今四川广元昭化古城仍是金牛道线上保存最完整的县城遗址。

### 七盘关

陕川界，嘉陵江畔。在今宁强、广元之间——七盘岭因山形回绕七匝得名。秦汉以来即设关——是关中入蜀道上由"秦地"入"蜀地"的标志性山口。唐时为利州下辖，**杜甫《五盘》**："五盘虽云险，山色佳有余。仰凌栈道细，俯映江木疏。"——"五盘"即七盘的另一名（同山异称）；杜甫 759 年自秦州、同谷一路入蜀过此关。今 108 国道沿七盘岭古道走，原栈孔遗迹仍可见。

### 棋盘关

宁强县境。一名"棋盘岭"——山形如棋枰得名。唐时为三泉县辖的小关——非军事咽喉，却是金牛道上行旅过夜重要节点。明清沿用至民国，是川陕商旅必经处。

### 五丁关

宁强县北。传为**五丁力士开山处**——五丁拽金牛入蜀，至此凿山三日始通。山名"五丁"，关名"五丁关"——一座地名直接挂在传说上的山口。秦汉之间已有此关名见诸记载；唐时为三泉县辖小关，戍卒数十。今宁强北十里仍有"五丁峡"地名——峡口峭壁立刻，传是当年五丁凿山的痕迹。说不清了，但地名独存。

### 朝天驿

广元（利州）城北约二十里，嘉陵江畔朝天峡口。唐时是利州北面第一驿——三十里递达利州城；自利州北行，二十里出朝天峡，再六十里至宁强。朝天驿名由来甚晚（一说唐玄宗入蜀至此，朝天而拜，故名；一说为后世附会），但驿址自汉魏起即为江畔渡口。朝天峡又名**明月峡**——是金牛道上栈孔最密集的一段，岩壁上方栈孔自唐至明清层层叠加，今四川考古所列为"明月峡古栈道遗址"。

### 葭萌驿 / 白水驿

利州境，葭萌县城所在驿与白水关旧址近——是金牛道由广元至剑门一段中点。白水关为古蜀汉地名（蜀汉以白水关御曹魏），唐时关已不存而地名独存，仅为驿与县城。葭萌驿是这一段最大的官驿——驿夫、驿马、驿仓最全；玄宗入蜀过葭萌驿，必歇于此（唐《元和志》补此地）。

### 广汉驿 / 汉州

汉州治所，今广汉。金牛道入成都平原前的最后一站——自汉州南行八十里至成都子城。汉州本汉广汉郡治、隋雒县——县名直至宋元；唐时为汉州刺史治，驿名"广汉"。蜀中入关贡品多在汉州集运、分编，再循金牛道北上——汉州在唐代是金牛道的"南端枢纽"。**三星堆遗址**（商代古蜀都城）即在今广汉境内——这一片地不止是金牛道终点，也是更早一千五百年那一座古蜀都城的所在地；但唐人尚不知三星堆为何物。

### 武连驿

剑州境。剑州治所普安县与绵州之间——是金牛道由剑南北段（山区驿）转中段（丘陵驿）的过渡点。唐时驿大、马多——蜀中士人北上长安多在此整装。明清沿用，今四川剑阁武连镇即驿址旧地。

## 历史长河（755 → 今）

**中晚唐**　至德元载（756）七月二十八日玄宗循金牛道入成都——这是金牛道最高规格的一次使用。乱平后肃宗、代宗朝复以蜀为重——韦皋（785-805）任剑南节度使二十一年间数次整修金牛道，新立褒中、武休、嘉陵新桥——蜀地物资北运至长安的效率，中晚唐反胜盛唐。但晚唐藩镇林立，僖宗中和元年（881）避黄巢入蜀，又一次走的是这条道——第二次"天子奔蜀"。**金牛道在 9 世纪走过两次大唐皇帝**——其他官道无此经历。

**五代两宋**　前蜀（907-925）、后蜀（934-965）以成都为都，金牛道是其与北方对峙的命脉——王建、孟知祥、孟昶皆有金牛道驿务整修记录。北宋平蜀（965）由王全斌循金牛道反向入蜀——四十年间金牛道由"蜀地北御"道改为"宋廷南控"道，**栈道的方向反了一次**。北宋熙宁、元丰间又大修——苏轼《入峡》《竹枝》多写蜀道沿线。陆游南宋乾道八年（1172）入蜀任夔州通判，途中一段乃是循金牛道反向（自梁州至利州）——他写《入蜀记》《剑门道中遇微雨》"衣上征尘杂酒痕，远游无处不消魂；此身合是诗人未？细雨骑驴入剑门"——一首把金牛道写到极致的诗。

**元明清**　元明时金牛道仍是川陕主干道——元设宁羌州（今宁强）、利州路、剑州；明改宁羌、广元、剑州。明洪武年间大修剑门关——今所见剑门关楼实为清同治年间在明基础上重修，又毁于咸丰兵燹与 1935 年公路建设。清代金牛道驿务整齐——四川至京师的"驿传"以此道为主线，与川陕大道（北段）合称"川陕官道"。乾隆《一统志》逐驿列名，剑南、山南西道驿牌大体不动。**两千一百年之间，这条道的总走向几无变化**——这是中国少见的一例。

**民国至今**　1935 年川陕公路（今 108 国道）开通——大体沿金牛道古线，但避开了最险的几段栈道。1956 年宝成铁路（宝鸡—成都）建成，路线在剑门以南与金牛道并行——铁轨与古栈道、夹江而过。1949 年后金牛道许多栈孔被现代爆破抹去；但朝天峡（明月峡）、剑门关、葭萌古城、五丁关一带仍存。2015 年金牛道（蜀道）启动**中国南方丝绸之路 / 蜀道**世界文化遗产联合申报，四川、陕西联合调查——明月峡栈孔六百余处、剑门关三道关墙、葭萌古城西门基址陆续清理。

至于盛唐那条道——栈板早已朽尽，江流仍是当年的江流。岩壁上一排排栈孔——孔在，木不在。沿着 108 国道开车自汉中入广元、过剑门、下成都，约一千一百里——五个钟头就走完了。玄宗那一次走了五十多日。

### 蜀道八条与金牛道的关系

| 道名 | 起讫 | 备注 |
|---|---|---|
| 褒斜道 | 关中眉县斜谷—汉中褒城 | 秦汉主线，玄宗入蜀前段亦走褒斜下汉中转金牛 |
| 傥骆道 | 关中周至骆谷—汉中傥水 | 险峻但近，唐代少用 |
| 子午道 | 关中长安子午谷—汉中 | 唐代杨贵妃荔枝道部分沿用 |
| 陈仓道（故道） | 关中陈仓—汉中嘉陵江 | 沿嘉陵江源，玄宗入蜀走的就是这条 |
| **金牛道** | **汉中—成都** | **关中入蜀真正官道；本 doc 主题** |
| 米仓道 | 汉中—巴中—阆中 | 金牛道的东支辅助线 |
| 荔枝道 | 涪州—长安 | 杨贵妃专线 |
| 阴平道 | 文县—平武—江油 | 邓艾偷渡灭蜀路 |

蜀道八条里，前四条是关中翻秦岭至汉中盆地，**金牛道是后半段唯一的主线**——所以唐人口中"蜀道"狭义即指金牛道，李白《蜀道难》所写主要也是金牛道沿线。

### 金牛传说与五丁开山的文献源流

| 时代 | 文献 | 记录 |
|---|---|---|
| 西汉 | 《蜀王本纪》（扬雄佚） | "秦惠王知蜀王好色，许嫁五女于蜀；蜀遣五丁迎之，至梓潼，见大蛇入山穴中，五丁共引蛇尾，山崩压杀，五人皆死" |
| 东晋 | 《华阳国志·蜀志》（常璩） | "秦惠王作石牛五枚，置金于牛尾下；蜀王使五丁力士拖牛入蜀——道遂通" |
| 唐 | 李白《蜀道难》 | "地崩山摧壮士死，然后天梯石栈相钩连"——融合五丁压蛇死与开山两说 |
| 北宋 | 《太平寰宇记》（乐史） | 录五丁传说地宁强、广元一线，"五丁峡"地名定型 |

**两个版本**：扬雄《蜀王本纪》是"五丁迎五女"被山压杀死；常璩《华阳国志》是"五丁拖石牛开道"。**唐宋以后两说渐合**——李白笔下二者已不分。**这是一则传说，但唐人当作金牛道的来源**——故事是不是史实不再重要。
