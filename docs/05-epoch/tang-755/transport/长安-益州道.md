# 长安-益州道

## 一句话

**长安西南行入蜀的官道总称**——出长安经岐州 / 凤翔 + 散关 + 河池 + 梁州（汉中）+ 利州 + 剑门 + 绵州、至益州（成都）；**至德元载（756）六月十三日玄宗出延秋门、十四日马嵬驿杨贵妃缢死、七月二十八日至成都**——这一段是盛唐转衰的物理 anchor；755 时尚是剑南节度奏报 + 蜀地贡赋 + 文人入蜀的常驿。

## 形貌

长安-益州道贯穿关中 + 秦岭 + 汉中 + 大巴山 + 剑门 + 成都平原数大地理段落。盛唐入蜀有三大主线：

- **故道（陈仓道）**——长安 → 岐州 → 陈仓 → 散关 → 河池（凤州）→ 兴州 → 梁州；
- **褒斜道**——长安 → 岐州 → 眉县 → 斜谷 → 褒城 → 梁州；
- **子午道**——长安 → 子午谷 → 洋州 → 梁州。

三道**在梁州（汉中）合一**——再南行入**金牛道**：**梁州 → 利州 → 剑门关 → 剑州 → 绵州 → 汉州 → 益州**。**长安-益州道**作为复合路名，**默认指故道-金牛道这一支**——即玄宗 756 入蜀走的那条；子午道、褒斜道、金牛道各有独立 doc，本 doc 写"长安至益州的全程通衢"。

**自北向南沿途要塞** + **大致段落**：

**关中段**（长安出延秋门 → 咸阳 → 马嵬驿 → 扶风 → 岐州 → 陈仓 → 散关）：
- **延秋门**——长安西出门；唐宫城禁苑西门、玄宗出奔即由此门；
- **马嵬驿**——京兆府兴平县境、长安西出第一处大驿；
- **岐州治雍县 / 扶风郡**（今宝鸡市凤翔区）——天宝元年改岐州为扶风郡；755 时郡治雍县；至德二载（757）方升凤翔府；
- **陈仓县**（今宝鸡市区一带）——故道入散关前的关中最后一城；
- **散关**——大散关；秦岭主脊北麓、关中西南门户（见 [散关](./散关.md)）。

**山南西道段**（散关 → 河池 / 凤州 → 兴州 → 梁州 / 山南西道治）：
- **凤州治河池县**（今陕西凤县东）——故道入秦岭后第一州；
- **兴州治顺政县**（今陕西略阳）——金牛道入汉中前最后一州；
- **梁州治南郑县**（今陕西汉中）——山南西道节度治；三道（故、褒、子午）在此合一、金牛道在此南入。

**剑南道段**（梁州 → 利州 → 剑门关 → 剑州 → 绵州 → 汉州 → 益州）：
- **利州治绵谷县**（今四川广元）——金牛道入蜀第一州、嘉陵江畔；
- **剑门关**——剑州境、大剑山主脊；金牛道最险一关；
- **剑州治普安县**（今四川剑阁）——剑门关南、栈道所经；
- **绵州治巴西县**（今四川绵阳）——李白故里彰明县属；
- **汉州治雒县**（今四川广汉）——成都北郊；
- **益州（蜀郡）治蜀县 + 成都县**（今四川成都）——天宝元年改益州为蜀郡；755 时郡治蜀县；剑南节度治、终点。

**地势**：**关中段**为渭水南岸平地 + 关中盆地西缘；**秦岭段（散关 + 凤州 + 兴州）**为秦岭主脊 + 嘉陵江上游峡谷——**栈道盘绕、河谷险峻**；**汉中盆地**短暂平坦；**大巴山-米仓山段（梁州 + 利州）**再入山；**剑门一段**为大剑山主脊狭口、千仞峭壁、栈道凿崖而成；**剑南段（绵州 + 汉州 + 益州）**渐入成都平原。

**物候**：自关中向南、由温带季风转副热带湿润——**春多雨、夏湿热、秋阴、冬不甚寒**；**秦岭以南即蜀地**——多云雾少日照、"蜀犬吠日"非虚语。秦岭主脊在散关一带、是中国地理南北分界线最显著的一段——**过散关即换天**。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

长安-益州道在唐人的回望里——是**蜀道 + 西南入蜀的千年通衢 + 李白《蜀道难》笔下的"难于上青天"**。

**蜀道的历史**：
- **战国秦惠王灭蜀（前 316）**——传说"五丁开山" + 凿通秦蜀间故道（《华阳国志·蜀志》记蚕丛、鱼凫、杜宇、开明诸蜀王的早期蜀道传说）；
- **秦汉**——故道（陈仓道）+ 褒斜道初成；汉武帝时褒斜道大修；
- **三国蜀汉**——诸葛亮六出祁山多次用褒斜道、陈仓道；张飞、马超等多次出散关；蜀道栈道形制由此定型；
- **魏晋南北朝至隋**——蜀道沿途州县多次易主；
- **唐承前代**——梁州属山南西道、益州为剑南节度治、长安-益州道是西南军政命脉。

**蜀道的"难"**：**李白《蜀道难》**（约天宝初年即已传咏）——"噫吁嚱、危乎高哉、蜀道之难、难于上青天"；"蚕丛及鱼凫、开国何茫然"；"剑阁峥嵘而崔嵬、一夫当关、万夫莫开"。**这首诗是盛唐蜀道想象的集大成**——把蜀道写成中国最险的山路、把剑阁写成"一夫当关"的关塞代名词。**李白本人乃绵州彰明县青莲乡人**——蜀道是他少年走过的实路、不是想象。

**唐剑南节度的位置**：
- **剑南节度治益州**——统益、蜀、彭、汉、眉、嘉、邛、雅、戎、泸、姚、巂、维、茂等州；
- **山南西道治梁州**——统梁、洋、兴、利、阆、果、剑等州；
- **二镇合管"剑南山南西"** —— 是盛唐西南军政重心、镇兵约 3 万。**杨国忠以剑南节度使遥领**——其势在朝、节度具体由副使代行。

**驰道的具体使用者**：**剑南节度奏报 + 蜀地贡赋（蜀锦 + 蜀纸 + 茶）+ 西南使节 + 文人入蜀 + 戍兵调动**。**蜀地贡赋之"蜀锦"**——是唐廷服御 + 赐赉的主要来源；蜀锦自益州沿金牛道北上、过散关至长安——是这条道上最具象的物资流。

### 此刻

天宝十四载这一年的长安-益州道——**栈道按例修缮、驿马按例轮换、蜀锦北上长安**。

**驰道状态**：盛唐 755 时长安-益州道沿途驿馆完整、栈道每年由所经州县维护、嘉陵江 + 西汉水 + 涪江 等沿途水道有渡口；**长安-益州走完全程约一月**（普通商旅）/ **半月余**（急讯）；**沿途州郡**（天宝间已改州为郡）：京兆府 + 扶风郡（岐州）+ 河池郡（凤州）+ 顺政郡（兴州）+ 汉中郡（梁州）+ 益昌郡（利州）+ 普安郡（剑州）+ 巴西郡（绵州）+ 德阳郡（汉州）+ 蜀郡（益州）——皆有郡守（太守）/府尹驻（京兆府用府尹）、戍兵守关、关吏验牒。

**散关一带的实景**：秦岭主脊咽喉、关城据大散岭、南循嘉陵江上游、栈道凿崖盘绕——是关中西南门户、属故道（陈仓道）系统；与褒水/褒斜道不是一物。**剑门一段的实景**：大剑山主脊狭口、关城据隘、栈道凿岩、千仞峭壁压于头顶——李白"剑阁峥嵘而崔嵬" 非虚语。

**蜀锦北上 + 文人入蜀**：盛唐间蜀地贡蜀锦、贡蜀茶；这条道也是文人入蜀的常路——**李白少年时（约开元初）自绵州青莲乡出蜀北上即经此道（反向）**；**杜甫将在 4 年后（759）自陇右南下入蜀**——但走的是更西的祁山道-金牛道支线、不全是这条主道。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九即起）；起兵后**叛军自范阳南下取洛阳 + 西攻潼关**——长安-益州道在 755 末仍按例运行、并未感受到东面战火；**至德元载（756）六月哥舒翰潼关之败**——失关西破、关中无险可恃；**六月十三日玄宗自长安出延秋门西奔**——经马嵬驿（十四日杨贵妃缢死）+ 扶风 + 散关 + 河池 + 梁州 + 剑州、**七月二十八日至成都**——**自此长安-益州道由日常驿道变成皇舆西奔的物理 anchor**。

### 此刻意味

对天宝十四载的剑南 / 山南西道人民来说——长安-益州道是**自家境内的入京通衢**：日常驿路过往 + 蜀锦北上 + 蜀茶外销 + 戍兵调动。对长安那一代士人来说——蜀道是**李白《蜀道难》笔下的西南想象 + 实路**：知其难、未必亲走；**杜甫尚未入蜀**（759 才入）；**高适将在数年后履任蜀地**（乾元二年/759 任彭州刺史 / 蜀州刺史；宝应元年 / 广德元年前后/762-763 任成都尹兼剑南西川节度使）——把蜀道写进中唐诗。

唐人尚不知道——再过 8 个月、玄宗将自此道西奔；马嵬驿一夜将成中国史最有名的一处驿馆；马嵬意象在后世的经典化与中唐《长恨歌》的回望——见"历史长河"段。**今游人重走长安-成都一线**——所见的是**宝成铁路 + 西成高铁 + G5 京昆高速 + 古蜀道遗存**的现代交通——**盛唐人在 755 所走的那条蜀道**：千二百多年后路线大体在同一地理走廊上、栈道孔洞 + 剑门关址 + 明月峡 + 七盘关 等处尚可访古；但唐代驿馆 + 栈道木构 几无原物——古道及历代后世遗存（翠云廊以明初植柏为主、杂唐宋古柏；马嵬坡为后世重建）仍为今所见者。

## 子条目

### 故道（陈仓道）+ 散关一线

**故道**——又称**陈仓道**——自陈仓南出散关、循嘉陵江上游南行、过凤州、兴州、至梁州。**散关**（大散关）在秦岭主脊北麓、扼故道咽喉——是关中四关之一（潼、散、萧、武、又一说蓝田关）。**故道是长安入蜀最稳的一支**——比褒斜道（直穿太白山东麓、栈道更险）+ 子午道（穿子午谷、最险）皆易行；**玄宗 756 西奔即走此道**。**散关与大震关的区分**：散关在陈仓南、属扶风郡境；大震关在陇州西、是陇右道入关中的西门；二者地理 + 功能皆不同、不可混。详见 [散关](./散关.md)、[陈仓道](./陈仓道.md)。

### 褒斜道 + 子午道（另两支入蜀路）

**褒斜道**——自眉县南出斜谷、循斜水南行、过太白山东麓、出褒谷至褒城（汉中北郊）。是先秦至汉中开通最早的栈道、栈道孔洞至今尚存万余。

**子午道**——自长安南出子午谷、循子午水南行、过宁陕、佛坪、至洋州。是三道中最险的一支。**荔枝道传说**——后世有"一骑红尘妃子笑"（杜牧《过华清宫》、中唐追写）的化用、运荔枝路线有争议（子午道、陈仓-金牛道、金牛道-荔枝道诸说并存）；不宜把子午道实路功能定为"运荔枝专道"。

**三道在梁州合一**——再南入金牛道。**756 玄宗西奔走的是故道（散关）而非子午道、褒斜道**——子午道太险、褒斜道路线略偏东、不及故道顺畅。详见 [褒斜道](./褒斜道.md)、[子午道](./子午道.md)。

### 金牛道（梁州-益州）+ 剑门关

**金牛道**——自梁州（汉中）南行入蜀、过利州、剑门关、剑州、绵州、汉州、至益州。是长安-益州道在汉中以南的全程；**与"褒斜道"不是一物**——褒斜道是关中-汉中段（北段）、金牛道是汉中-益州段（南段）。

**剑门关**——在剑州境（治普安县、今四川剑阁县）、**大剑山主脊狭口**——**关城据隘 + 栈道凿岩 + 千仞峭壁**——是金牛道最险一关。**"剑阁"是地名 + 栈道总称**（剑州境内栈道全段皆可称剑阁）；**"剑门关"是具体关塞**——二者不可混用。**李白《蜀道难》"剑阁峥嵘而崔嵬、一夫当关、万夫莫开"**——是文学描写、不是关城形制的考工记。**今所见剑门关楼**——为明清 + 1985 重建主体。详见 [金牛道](./金牛道.md)。

### 马嵬驿（至德元载六月十四日杨贵妃缢死）

**马嵬驿**——京兆府兴平县境、长安西出第一处大驿。**至德元载（756）六月十四日**——玄宗西奔次日、**六军（左右龙武军）至马嵬驿不发**——**陈玄礼（左龙武大将军）请诛杨国忠**；**国忠 + 杨氏一门皆死乱兵中**；**六军复"贵妃在帝侧、不可恃"——请诛贵妃**；玄宗不得已、命**高力士引贵妃至佛堂、缢死**（正史口径；后世异文有"梨树下"之说）——时年三十八岁。**这一夜是盛唐转衰的物理 anchor + 唐宫廷史最戏剧的一夜**。

**755 时马嵬驿仍是普通驿馆**——8 个月后即成"贵妃缢死处"；自此**马嵬驿在唐诗中的位置骤升**——李益、白居易、刘禹锡、李商隐、罗隐等皆有马嵬诗（中唐回望、详见"历史长河"段）。**今马嵬坡（陕西兴平市西约 25 公里）**——有杨贵妃墓 + 马嵬驿遗址；为后世重建。

### 玄宗西奔入蜀（至德元载六月-七月）

**完整路线 + 时序**（《资治通鉴》卷二一八·至德元载）：
- **六月十三日（乙未）**——玄宗自长安出延秋门、西奔；
- **六月十四日（丙申）**——至马嵬驿、杨贵妃缢死、杨国忠 + 一门皆死；
- **六月十五日至七月**——经扶风 + 陈仓 + 散关 + 河池（凤州）+ 兴州 + 梁州（汉中）；
- **七月**——梁州南下、过利州、剑门关、剑州、绵州、汉州；
- **七月二十八日（己卯）**——至**成都**。

**全程约一月半**——比常旅快、是仓促西奔。**沿途主要事件**：**七月十二日肃宗在灵武即位**（玄宗在路上不知）、改元至德；**玄宗在成都得报**、改尊为太上皇。**玄宗自此在成都流寓至至德二载（757）冬**——前后约一年半；**至德二载冬离蜀北还；十二月还京**——经金牛道、褒斜道（一说仍故道）北归。

**这一段是盛唐转衰的最具象 episode**——长安-益州道由日常驿道变成"皇舆西奔"的物理 anchor。

### 唐入蜀文人（李白 + 杜甫 + 高适）

**李白**——**绵州彰明县青莲乡人**（今四川江油青莲镇）；**约开元十三年（725）二十五岁出蜀**——自绵州沿涪江北上、转入金牛道、过剑门、至梁州、再北出散关至长安——是蜀道反向最有名的一次。**《蜀道难》**（约天宝初已传）+ **《送友人入蜀》**（"见说蚕丛路、崎岖不易行。山从人面起、云傍马头生"）——皆写蜀道。

**杜甫**——**乾元二年（759）自陇右秦州南下入蜀**——走的是**祁山道 + 同谷 + 金牛道**支线、非全程故道-金牛道。**《剑门》《五盘》《飞仙阁》《泥功山》《木皮岭》等陇蜀诗**——皆作于此段、写蜀道沿途的险与穷。**自此杜甫在成都浣花溪草堂寓三年**——蜀道是他中年最熟的物理 anchor。

**高适**——**乾元二年（759）出任彭州刺史 + 蜀州刺史**；**宝应元年 / 广德元年前后（762-763）任成都尹兼剑南西川节度使**——把蜀道写进中唐边塞诗。**高适与杜甫成都唱和**——是盛唐转中唐间文学史最重的一笔。

### 蜀地贡赋（蜀锦 + 蜀纸 + 蜀茶）

**盛唐蜀地三大贡赋**：
- **蜀锦**——益州贡、唐廷服御 + 赐赉的主要来源；自益州沿金牛道北上、过剑门 + 散关 + 至长安；
- **蜀纸**——绵州贡、唐廷文书用纸；"蜀纸"在唐人书帖中常见；
- **蜀茶**——雅州 + 蒙顶山贡、"蒙顶石花" 是唐茶最高品之一；沿金牛道北运。

**这些物资沿长安-益州道北上**——是 755 时这条道上最具象的物流；**蜀锦的繁盛**——是盛唐物质文化在长安的物理 anchor（长安西市的蜀锦铺、宫廷的蜀锦袍）。

## 历史长河（755 → 今）

**中晚唐**　**至德元载（756）玄宗西奔入蜀**——长安-益州道由日常驿道变成皇舆西奔的物理 anchor；**至德二载（757）冬玄宗自成都北还、十二月还京**——经金牛道北归。**广德元年（763）吐蕃陷长安、代宗东奔陕州**——非走蜀道；但蜀地在中晚唐多次为唐廷流亡 + 避乱地。**僖宗广明元年（880）黄巢入长安、僖宗西奔成都**——再次走长安-益州道；**光启元年（885）僖宗再走兴元（梁州）避难**——长安-益州道在唐末成"天子西奔专用路"。

**杜甫《剑门》《五盘》《飞仙阁》《泥功山》**（乾元二年/759 作）、**白居易《长恨歌》**（中唐回望、元和元年/806 作；"六军不发无奈何、宛转蛾眉马前死"）、**李商隐《马嵬》"海外徒闻更九州、他生未卜此生休"**（晚唐）——皆作于这一段时间、写蜀道沿途与马嵬意象。**自此蜀道在唐诗中的位置稳居"西南极地 + 哀挽 anchor"双重身份**。

**两宋**　**北宋初蜀道沿途州县继续作为关中-成都间交通线**——但物流主体由江淮经长江入蜀的水路渐取代；**南宋偏安、关中失、蜀道成宋金边境的西线**——吴玠 + 吴璘兄弟守和尚原 + 仙人关、即在故道一线（散关南）。

**金元明清**　**元代蜀道沿途修缮**——成成都至大都的西线驿路一段；**明洪武间大规模修缮金牛道 + 翠云廊植柏**——今**翠云廊**（剑州至阆州一段、古柏夹道）多为明初所植、亦杂唐宋古柏；**清乾隆间蜀道再修**——清末民初成川陕公路雏形。

**民国至今**　**1956 年宝成铁路通车**——成为长安-成都间的现代陆路主干（与古蜀道大体平行、走故道-金牛道走廊）；**2017 年西成高铁通车**——半日达成都；**G5 京昆高速 + G108 国道**——形成今日长安-成都一线的现代交通。**唐人在 755 所走的那条长安-益州道**——千二百多年后**地理走廊大体未变**；今所见者为**古道及历代后世遗存**——栈道孔洞（散关 + 褒城 + 明月峡 + 七盘关一带）与剑门关址尚有唐代遗痕；翠云廊以明初植柏为主、杂唐宋古柏；马嵬坡为后世重建；驿馆 + 栈道木构 几无原物。**蜀道在 2009 年起申报世界文化遗产**——尚在路上、是中国现存最大尺度的古道文化景观之一。
