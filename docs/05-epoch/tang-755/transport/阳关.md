# 阳关

## 一句话

**河西走廊最西南的关塞**——与玉门关合称"河西二关"；**王维《送元二使安西》"西出阳关无故人"** 是盛唐边塞诗最有名的别离句；**755 时仍立、扼丝路南道、是中国境内最西南的关塞之一**。

## 形貌

阳关在唐**陇右道沙州敦煌郡**——具体在**敦煌西南约 70 公里**（今甘肃省敦煌市西南古董滩遗址）。**地势**：**敦煌绿洲西南边缘**——南接昆仑山北麓余脉、西望塔克拉玛干沙漠东缘、北接玉门关一线（与玉门关南北相距约 80 公里）。**得名"阳关"**——因关在**玉门关之南**、按古制"山南水北为阳"——故称阳关。

**阳关之险**：与玉门关相似、**戈壁茫茫无险**——靠"戈壁无替代路线"的天然封闭；**自阳关向南转东、绕塔克拉玛干南缘**——是丝路南道；**自阳关向东**——回敦煌绿洲、入河西走廊。**一关守一片沙漠的东南入口**。

**关城形制**：**汉阳关原址**：今**古董滩遗址**——考古勘察显示有夯土城墙、附属戍堡、烽燧基址。**盛唐 755 时阳关具体形制**：史载未详；按汉故关延续 + 唐重修推断、应为夯土关楼 + 戍兵营 + 关吏。**今阳关遗址**：地表已大半被风沙覆盖、仅墩台 + 戍堡残基可辨。

**物候**：极干燥、风沙大；**南面是昆仑山北麓 + 党河（玄奘所谓"甘泉水"）**——是阳关唯一的水源补给。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

阳关在唐人的回望里——是**河西二关之南关 + 丝路南道之西出口**。

**汉武立阳关**：**西汉武帝元鼎六年（前 111）**同时立玉门关 + 阳关二关守河西最西。**"阳关"二字得名**：与玉门关 (得名于"和阗玉过关") 不同、**阳关因方位得名**——在玉门关之南、合"山南水北为阳"古制。**自汉至唐**——阳关一直是丝路南道的西端关塞；自阳关 + 玉门关分流后、丝路南道沿塔克拉玛干南缘经鄯善（楼兰故地，755 已废）+ 于阗 → 西达葱岭；丝路北道沿天山南麓经焉耆 + 龟兹 → 西达葱岭。

**盛唐阳关的政治军事地位**：**唐承前代之制**、阳关属沙州（具体为沙州寿昌县）；**驻军、关税、验牒** 等关塞常制；**阳关是唐西域交通界标 + 文学界标**——与玉门关同档。**阳关与玉门关分守北南二道**——盛唐丝路商旅多过此二关之一。

**阳关在文学上的高频化**：**盛唐边塞诗 + 怀古诗 + 送别诗中"阳关"二字出现频繁**：
- **王维《送元二使安西》**："渭城朝雨浥轻尘，客舍青青柳色新。劝君更尽一杯酒，**西出阳关无故人**。"——是盛唐"阳关"诗的最高峰、亦是中国送别诗的代表。
- **王昌龄、岑参等盛唐边塞诗派**多有涉阳关；
- **《阳关三叠》**（后世琴歌）：以王维《送元二使安西》为词、配以琴曲、"三叠" 重复句的传唱形式——具体定型在中晚唐至宋间（琴谱定本更晚）；本 doc 严守"《阳关三叠》是后世琴歌、非盛唐 755 时定型"口径。

**注**：王维《送元二使安西》作于盛唐间——"元二"为元常（一说为元宗简，史源略异）出使安西大都护府；"西出阳关" 是元二的实际行程（自渭城 + 长安 + 西行 + 过阳关 + 入安西）。**自此阳关在唐诗中的位置升至送别意象的最高峰**。

### 此刻

天宝十四载这一年的阳关——**关塞严整、驻军戍守、丝路南道通行**。

**关城状态**：盛唐 755 时阳关主体仍立（汉故关延续 + 唐重修）；夯土关楼 + 戍兵营 + 关吏 + 驿马。**关税收 + 戍军**：阳关收西域南道商旅过关税——是唐西陲关塞常制；具体阳关驻军规模史载零散，本 doc 不预设数字。

**丝路通行**：**755 时阳关仍是丝路南道的西端孔道**——粟特商人、西域诸国使臣、僧侣、汉商等过往；**与玉门关分流**——阳关向南绕塔克拉玛干南缘 + 经于阗 (安西四镇之一) + 西达葱岭。

**安史前夕**（即 755 这一刻）——**751 怛罗斯之战**唐军败于大食、丝路西端进入新平衡；**755 起安史之乱河西陇右兵东调救京、阳关守备日空**。**广德元年（763）后吐蕃陷河西**——阳关一带渐失唐廷直接控制；自此阳关作为唐廷关塞的功能结束。

### 此刻意味

对天宝十四载的沙州人来说——阳关是**自家郡境的西南界关塞**：日常驻守 + 西域南道商旅过关。对长安那一代士人来说——阳关是**送别诗的具象**：**王维"西出阳关无故人" + 《阳关三叠》乐曲**——盛唐文学最具象的西陲离别符号。**盛唐人嘴里的"阳关"**——既是地理实体、也是文学符号；甚至更是文学符号（多数士人未亲至阳关、但通过盛唐边塞诗 + 送别诗熟知）。

唐人尚不知道——再过 8 年（763）吐蕃陷河西、阳关一带渐失唐廷控制；再过千余年（明清间）阳关古关已彻底荒废、地表大半被风沙覆盖。**今阳关遗址（古董滩）列联合国教科文组织世界文化遗产**（2014 年作为"丝绸之路：长安-天山廊道路网"组成部分）。**唐人在 755 所见的那座阳关**——千二百多年后**关城主体已残**、地表风沙覆盖、仅墩台 + 戍堡残基可辨。

## 子条目

### 王维《送元二使安西》（《阳关三叠》原词）

**王维（约 701-761）盛唐间作**——为送友人元二（一说元常、一说元宗简，史源略异）出使安西大都护府所作的离别诗：

> 渭城朝雨浥轻尘，客舍青青柳色新。
> 劝君更尽一杯酒，**西出阳关无故人**。

**这首诗是盛唐送别诗的最高峰**——后被配乐谱成《阳关三叠》：每唱至末句"西出阳关无故人"重复三遍——成为中国乐府最有名的送别曲。**自此阳关在中国文学中**——既是地理实体、亦是送别意象的代名词。

### 阳关与玉门关的分流（南北二道）

**阳关 + 玉门关**——汉武元鼎六年（前 111）同时立、合称"河西二关"：
- **玉门关**：敦煌西北、扼**丝路北道**——沿天山南麓 + 焉耆 + 龟兹 + 西达葱岭。
- **阳关**：敦煌西南、扼**丝路南道**——沿塔克拉玛干南缘 + 经于阗 + 西达葱岭。

**二关分流**：丝路自玉门关 + 阳关分两路、西行至西域诸国后再汇合。**盛唐"出阳关"+"出玉门"为西行二大途径**——王维《送元二使安西》"西出阳关" + 王之涣《凉州词》"春风不度玉门关"——是盛唐两关诗的双璧。

### 古董滩遗址（汉阳关址）

**今阳关遗址在敦煌市西南 70 公里"古董滩"**——因戈壁中常露汉唐文物、得名"古董滩"。**1907 年英国斯坦因 + 1944 年向达、夏鼐 + 1980s 起多次考古勘察**——确认汉阳关位置；出土汉简、铜器、陶器等。**今阳关遗址**：地表风沙覆盖、墩台 + 戍堡残基可辨；**1981 年列国家级重点文物保护单位 + 2014 列联合国教科文组织世界文化遗产**（作为丝路廊道组成部分）。

### 党河（敦煌党水）—— 阳关的水源

**党河**：祁连山西段冰川融水 + 自南向北流经敦煌绿洲——是敦煌绿洲 + 阳关 + 玉门关的唯一水源。**盛唐玄奘**称其为"甘泉水"。**阳关南面**：党河南畔；自阳关西出戈壁 + 走丝路南道——必自党河补给水。**党河至今仍是敦煌的命脉**——是阳关周边戈壁中唯一的活水。

### 《阳关三叠》乐曲（盛唐至中晚唐渐成）

**《阳关三叠》**——以王维《送元二使安西》为词、配以"阳关曲"曲；**乐曲特点**：每唱至末句"西出阳关无故人"重复三遍——形成"三叠"格式。**起源**：盛唐至中晚唐间渐成；**具体定型**：唐宋间、史源略异。**《阳关三叠》成中国乐府最有名的送别曲**——千余年间凡送行者必歌此曲。**今**：**《阳关三叠》乐谱仍存**（明清抄本 + 现代琴谱）——是中国古乐中最有名的离别曲。

## 历史长河（755 → 今）

**中晚唐**　**广德元年（763）后吐蕃陷河西**——阳关一带渐失唐廷直接控制。**自此阳关作为唐廷关塞的功能结束**——但作为丝路南道要塞、商旅仍偶有过往。**中晚唐"阳关"在唐诗中的位置**：随《阳关三叠》乐曲的流行 + 王维诗的传播而升——成"送别 + 西陲" 的双重意象。

**两宋至元**　**北宋初河西已属西夏 + 西州回鹘**——阳关地区为西夏 vs 回鹘的边境；**自此阳关退冲要**。**元代蒙古统一中亚后丝路畅通**——但主要孔道已不在阳关；**元代阳关已彻底退出主要驿路**。

**明清**　**明初置关西七卫**含敦煌一带；**明嘉靖后明廷退守嘉峪关**——**阳关彻底退出明廷边境**。**清乾隆朝平准 + 平回（1755-1759）后新疆纳入清版图**——阳关重新进入中国地理认知，但已是历史遗址、不再是关塞。

**民国至今**　**1907 年英国斯坦因勘察古董滩**；**1944 年向达、夏鼐勘察**；**1980s 起多次考古发掘**——确认汉阳关位置。**1981 年阳关遗址列国家级重点文物保护单位**；**2014 年"丝绸之路：长安-天山廊道路网"列联合国教科文组织世界文化遗产**（注：玉门关 + 悬泉置 等敦煌段遗址明列；阳关是否单独列入需 verify UNESCO 组件名单）。**今游人来阳关**——所见的是**古董滩遗址 + 墩台残基 + 戈壁**——**唐人在 755 嘴里的阳关**：千二百多年后**关城已残**、但地理 anchor 犹在。
