# 褒斜道

## 一句话

蜀道八条中最直的一条，关中通汉中的"近路"；战国秦昭王便已凿栈，三国出此口者赵云邓芝、五丈原前的诸葛亮，唐人沿用如旧。

## 形貌

褒斜道是秦岭里最直的一刀。

自关中眉县城南入斜谷，南行约四百七十里至汉中褒城出褒谷——比走陈仓-散关一线约略短近一半。所谓"褒斜"，便是北口斜水（今石头河）、南口褒水（今褒河）两条河谷的合称：北段沿斜水溯流上太白山东麓，越五里坡分水岭，再顺褒水南下出秦岭。一条路用了两条水的河床，山势却几乎不让人喘息。

栈道是这条路的骨。岩壁陡峭之处无地立足，自战国便在崖上凿方孔——孔内插横木，横木上铺木板，板外另立支柱，**层层叠成悬空一线**。《水经注》载褒斜栈"穿山为孔，架木为梁"，前后凡数百里——是中古最早、也最密的悬空木构系统。木栈不耐雨蚀虫蛀，每数十年必大修一次；唐人沿路所见，已不知是第几代的木桩。

太白山是这条路真正的脊梁。北段越五里坡时，路抬至海拔一千四百米左右——三月山下已有杏花，山口冰雪未融。山林深处多有崖刻，自汉至唐——历代修栈、置驿、改道，多刻石为记。褒谷口南有一处石门——东汉永平六年（63）凿崖通隧的旧道，石壁上累代题刻不绝，后世辑为"石门十三品"，是中国摩崖碑刻一大宗。

至于路上的物候——三月斜谷桃杏、五月留坝新茶、秋来红叶满涧、冬日栈板结冰。商旅多避盛夏，因山雨突至栈板湿滑；也避隆冬，因雪封五里坡。一年里能走的，约只七八个月。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

褒斜栈道在唐人眼里早已是"古道"。

战国时秦昭王（前 306-251）便已凿栈通蜀——《史记·河渠书》明载褒斜之名，是中国正史最早记录的栈道之一。汉武帝元光六年（前 129），有大臣议修褒斜以漕运汉中之粟入关中，发卒数万开凿——后因水急多滩未成漕运，但路通如旧。东汉永平六年（63），汉明帝诏凿石门通隧——褒谷口"鬼斧斫山"的那段隧道便在此年完工。三国时这条路成蜀汉北伐的咽喉之一：**蜀汉建兴六年（228）一次北伐，诸葛亮亲出祁山为主攻，遣赵云、邓芝由斜谷为疑兵**——叙事里那支被曹真识破的"偏师"，便走在褒斜栈板上。**建兴十二年（234）五次北伐，诸葛亮亲率十万出斜谷**——至五丈原扎营、与司马懿隔渭对峙百日，八月病卒军中。姜维数次北伐路线主在祁山、陇右，褒斜亦曾驻兵——但已不是最关键的那条。

魏晋南北朝乱世，栈道屡毁屡修。北魏太武帝太平真君年间（440-450）大修一次，《水经注》郦道元南行所见即此后状态。隋开皇年间（581-600）又一次系统性修治——唐承之，于沿途置驿。

### 此刻

天宝十四载（755），褒斜道是关中通汉中、再下蜀地的官道选项之一——但并非主流。

主流是西边的陈仓-散关-故道一线，因更宽更缓、利大队商旅与官军行进；褒斜短而险，适合轻骑、急递、僧人、文士。沿途自北至南有数处驿站：斜谷驿（眉县）、武休关、武关驿（留坝武关河）、马道驿、青桥驿——驿距多在三十至五十里，皆唐代《元和郡县图志》明载。岁贡之物中，山南西道的茶、漆、药材偶经此道北上长安；蜀地荔枝走的是另一条——子午道改线的"荔枝道"，那是杨贵妃专享，不与褒斜共轨。

褒斜道两端的城此刻各有归属。**北口眉县**属凤翔府（岐州），是开远门西出长安、过咸阳、武功、扶风、岐山再南折入斜谷的最末一程；**南口褒城**属山南西道梁州（汉中），是入蜀前的最后一座大邑。汉中再南，便接金牛道下广元至成都。

### 此刻意味

对天宝十四载在长安的人来说，褒斜道意味着一种"备用通道"——不在日常视线里，但人人知它在那里。万一关中有变，朝廷沿此南奔汉中可达；蜀地有警，急递沿此北上数日可至。

再过半年——这条"备用通道"便要派上用场。至德元载（756）六月潼关失守、玄宗西奔，主道走的是金牛-散关一线；但**消息、密使、零散逃亡的官员**沿褒斜、子午、傥骆诸道日夜兼程——秦岭的几条裂缝里，那一年的人比平日多了百倍。

## 沿途节点（doc 内子条目，不另立 POI）

下列各项均为褒斜道线性节点，作 doc 内独立 `###` 章节锚定；不另立 `.md` 文件。

### 斜谷口

褒斜道北端。眉县城南五里，斜水（今石头河）出秦岭注入渭水之处。两山如门，水从中出——"斜谷"之名即从此。唐置斜谷驿，是关中入山第一站。谷口外有汉武帝元光六年开褒斜时所遗石碑残段，唐人过此犹能辨识。北望，可见秦岭横亘如墙；南行三十里即上太白山东麓。

### 五丈原（诸葛亮卒地）

斜谷北口出山即五丈原——一处南北长约四公里、东西宽不足两公里的台塬，三面临沟，一面接秦岭。蜀汉建兴十二年（234）四月，诸葛亮率十万大军出斜谷，至五丈原与司马懿隔渭水对峙——亮分兵屯田，**作百日相持之计**。八月二十三日，亮病卒于军中，年五十四。蜀军秘不发丧，徐徐退入斜谷——"死诸葛走生仲达"的故事由此而出。后人于原上立"诸葛亮庙"——唐时已有，今所见为元、明、清累代重修；庙旁有"落星石"——传亮卒夜有大星陨于此。一座台塬，被一个人的死压了一千八百年。

### 留坝段

褒斜道腹地。自五里坡分水岭南下约百里至留坝，是全程最险的一段——栈道贴崖走，足下即急涧。留坝今属陕西汉中市，唐时属梁州褒城县辖区。沿路有武休关、武关驿（与关中"武关"同名异地，常被史志混注，《元和郡县图志》专作"武休"以别之）、马道驿——驿名"马道"，因此段山势虽险尚能容马驮通过，比子午、傥骆诸道宽得多。汉张良"明烧栈道"传说，地望多指此段——刘邦入汉中后烧绝栈板以示无东归之意，便是这一带。

### 褒谷口

褒斜道南端。汉中城北约二十里，褒水（今褒河）出秦岭注入汉江之处。东汉永平六年（63）凿崖通隧——即著名的"石门"，是中国最早的人工通车隧道之一。石门两壁累代题刻——东汉《鄐君开通褒斜道刻石》（永平九年）、《石门颂》（建和二年）、北魏《石门铭》（永平二年）、宋代《晏袤释鄐君碑》……自汉至宋约三十种，被誉为"汉字书法摩崖第一谱系"，后世辑为"石门十三品"。1969 年因修石门水库，**十三方原石被切割迁至汉中博物馆**——今所见石门峡谷下游已是水库湖面。

### 三国遗迹（诸葛亮北伐用此道）

褒斜道沿线密布三国遗迹，是后世怀古者最爱踏勘的一条秦岭古道。

**斜谷口**——赵云、邓芝建兴六年（228）出疑兵处。叙事里那场"曹真识破"的伪攻，是两人在斜谷边打了一仗才退。

**五丈原**——诸葛亮卒地。已见上节。

**赤崖** / **木牛流马传说地**——褒斜中段，建兴九年（231）诸葛亮第四次北伐时改用"木牛流马"运粮，史载"运米至赤崖"。具体地望不一，《水经注》系于褒水中游；后世各家考证悬殊。

**樊河渡** / **古虎头桥**——汉中往北的几处渡桥，杨修被斩处、马超墓等三国遗迹散落褒城至勉县一带——严格说不全在褒斜道线上，但唐人沿道北归长安，多会绕道一访。

杜甫《武侯庙》"遗庙丹青古，空山草木长"——写的虽是夔州武侯祠，那种"庙在，人散"的怅然，褒斜道沿线最齐。

## 历史长河（755 → 今）

**中晚唐**　至德元载（756）夏秋玄宗西奔走金牛道一线，褒斜道作为辅道日夜南北通使。德宗建中四年（783）泾原兵变、僖宗中和元年（881）黄巢入长安——两次关中再乱，褒斜道都是士民南下避乱的辅助通道，但主奔蜀的天子皆走金牛。晚唐栈板年久失修，一条路用了一千年，至此已老。

**五代至宋**　五代蜀地三度独立（前蜀、后蜀），关中入蜀几条道反复利用。北宋至和二年（1055）大地震毁褒斜栈道大半，沿用八百年的旧栈系统至此基本崩溃——南宋时主走故道。陆游乾道八年（1172）入南郑（汉中）幕，实地走过的多是故道与子午、傥骆——褒斜在他笔下已是"古道"。一条路的"现役"身份，宋时已结束。

**明清**　明代褒斜道作为汉中至关中的小道仍偶有商旅，但栈道再未系统重建。清乾隆四十四年（1779）陕西巡抚毕沅遍立"古蜀道遗址"碑，褒谷石门旁也立了一块——距栈道初通已两千年。同期考据派学者王昶、阮元等访石门十三品拓片——清代金石学的"石门系列"研究即由此起。至民国，褒斜道作为道路的实体已彻底消失，只余沿线零星栈孔与石刻。

**民国至今**　1936 年宝鸡至汉中公路（川陕公路，今 G108）通车——线路大体沿故道走向，但局部段经过褒斜古道遗址。1969 年石门水库建成，褒水入汉江一段成湖——**石门隧道与十三品原石淹于水下**，原石切割迁至汉中市博物馆专建石门石刻陈列馆收藏。1980 年代起，陕西省考古研究院对沿线栈孔系统调查——眉县斜谷、留坝武休关、褒城石门一带累计登记栈孔遗迹千余处，多数为汉魏至唐宋叠压。2007 年蜀道申报世界文化遗产时，褒斜道列为"金牛-米仓-褒斜"组合的核心线段之一——但**至今未通过联合国教科文组织终审**，列为"预备名录"。2010 年代起，眉县与留坝相继建"褒斜古栈道遗址公园"，复原数十米栈板供游客行走——脚下木板新铺，崖上栈孔是真的。

至于秦岭山中那条贯穿了两千三百年的木栈系统本身——已只剩崖壁上一串一串的方孔，整整齐齐排成一条线。

### 唐代沿途驿站

据《元和郡县图志》并参严耕望《唐代交通图考》卷三，自北向南：斜谷驿（眉县）→ 五丈原亭（岐山）→ 武休关驿（留坝）→ 马道驿 → 青桥驿 → 褒城驿。合计约 470 里——商旅半月可达，急递快马三日可至。

### 石门十三品

褒谷口石门内外历代题刻精选——汉魏至宋累代摩崖的中国书法摩崖第一谱系，包括：东汉《鄐君开通褒斜道刻石》（永平九年/66）、东汉《石门颂》（建和二年/148，**隶书巅峰之一**）、东汉《杨淮表纪》（熹平二年/173）、北魏《石门铭》（永平二年/509，**魏碑名品**），以及《玉盆》《衮雪》《石虎》《李君表》《山河堰落成记》《晏袤释鄐君碑》等。1969 年因石门水库建成原石淹水，十三品**全部切割迁至汉中市博物馆**——今所见原石不在原位，但完整。
