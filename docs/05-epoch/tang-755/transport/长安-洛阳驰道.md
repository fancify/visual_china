# 长安-洛阳驰道

## 一句话

**唐两京间最重要的驿道** + **隋唐第一驰道**——长安出东向、过潼关 + 函谷新关 + 陕州 + 至洛阳；**盛唐天子东巡、漕运、商旅、使节皆赖此**；**755 时是中国最繁忙的驿路、半年后即随安史之乱而崩**。

## 形貌

长安-洛阳驰道沿崤函走廊东出——是唐"水陆驿千余"中的最骨干一条；**唐令常制三十里置驿、十里置亭**。**自西向东沿途要塞** + **大致段落**：

**关中段**（长安出延秋门 / 春明门 → 新丰县 → 渭南县 → 华州治郑县 → 华阴县 → 潼关）：
- **长安出春明门东行**——是唐两京驰道的起点；
- **渡霸桥**——长安东郊著名送别地；
- **新丰县**（今西安临潼东）——汉新丰故地；
- **华阴郡治郑县**（今华县）——华阴郡太守驻；
- **华阴县**（今华阴）——华山下；
- **潼关县**——关中东大门、扼黄河 + 秦岭咽喉。

**河南段**（潼关 → 弘农郡 / 虢州一线 → 陕郡治陕县 → 渑池 → 新安函谷新关 → 洛阳）：
- **出潼关 → 入弘农郡（虢州）一线**——唐时弘农郡太守驻；
- **陕郡治陕县**（今三门峡市陕州区）——黄河漕运咽喉、陕郡太守驻；
- **渑池县**（今渑池）——黄河南岸；
- **新安县函谷新关**（今新安）——汉武帝东徙的函谷新关在河南府新安县境、洛阳西郊；
- **洛阳河南府治洛阳县**——东都终点、河南府尹驻。

**驿馆制度**：**《唐六典》"三十里一驿、十里一亭"**——长安-洛阳驰道沿崤函走廊、按唐令常制三十里置驿。**唐驿马**：每驿配马 8 匹（中等驿）至 75 匹（大驿）；**驿夫**：每驿数十人；**驿吏 / 驿令**：由当地州县官派遣或专职。**长安-洛阳驰道是唐驿政最完善的一条**——驿马轮换、马夫接力、急讯一日数百里、平日商旅五六日抵达。

**地势**：**关中段渭水南岸平地**——夏秋多雨、冬干；**关中盆地东缘渐过渡为黄土丘陵**；**潼关一带**——黄河南岸 + 秦岭北麓夹峙咽喉；**河南段**——黄河南岸 + 崤山一线丘陵；**新安以东**——伊洛盆地东缘。

物候：温带季风、四时分明、春多沙、夏多雨、秋高、冬有薄雪——**长安-洛阳驰道是唐两京间最熟悉的物理空间**。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

长安-洛阳驰道在唐人的回望里——是**两京间的脊骨 + 中国最繁忙的驿路**。

**驰道的历史**：
- **秦始皇驰道**（前 220 起）——以咸阳为中心、辐射四方；含咸阳东出 + 至函谷关 / 洛阳的最早雏形；
- **西汉至东汉**——长安 + 洛阳间作为两都驰道、汉武帝改名"驰道"；
- **隋大业元年（605）**——隋炀帝大规模整修长安-洛阳间驰道；同时**开通济渠**（洛阳至淮水段、南接邗沟至江都）使两京间水陆联通；
- **唐承隋制**——长安-洛阳驰道作为两京间最重要的官道。

**唐两京制下的驰道**：
- **唐高祖、太宗、高宗、武则天朝**——多次往返两京；武则天朝后期改东都洛阳为"神都"、自此玄宗朝开元前期至 736 多次就食洛阳（关中歉收时）；
- **开元二十二年（734）裴耀卿改漕运法**——置河阴仓 + 集津仓 + 盐仓三门转运体系、自此关中粮食有保障；**开元二十四年（736）十月**玄宗最后一次自洛阳还长安——**自此玄宗朝再未东巡 19 年（至 755）**；
- **天宝间**——长安 + 洛阳间的驰道仍是最重要的官道、但两京并行的时代渐过、自此**长安为核心、洛阳为陪都**。

**驰道的具体使用者**：**唐朝廷使节 + 节度使奏报 + 漕粮（关中需求）+ 商旅 + 文人 + 僧侣 + 戍兵调动**等——是中国 8 世纪流量最大的官道。

### 此刻

天宝十四载这一年的长安-洛阳驰道——**繁忙、按例运行、即将经历安史浩劫**。

**驰道状态**：盛唐 755 时驰道沿途驿馆完整、马匹按例轮换、驿夫驻馆；**长安-洛阳间走完驰道约五六日**——比急讯日数百里慢得多、但是商旅 + 文人 + 一般官员的常规。**沿途州府**（天宝元年/742 改州为郡之制下）：京兆府 + 华阴郡 + 弘农郡 + 陕郡 + 河南府——两京特例京兆府、河南府置府尹；余皆郡 + 太守驻；州县官治 + 驿政并行。

**漕运 + 驰道联运**：盛唐两京间漕粮自江淮过通济渠 + 汴渠入河 + 再溯黄河西上抵洛阳含嘉仓 + 部分自陆运过三门峡 + 再至关中——**驰道与水运并行**；**755 时这套漕运体系按例运行**。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九即起）；起兵后叛军自范阳南下取洛阳——**两京驰道在 755 末至 756 初成叛军 + 唐军反复争夺的战场**；**至德元载（756）六月哥舒翰潼关之败**——驰道关中段失守；**至德元载六月十三日玄宗西奔入蜀**——出延秋门西行、经马嵬、扶风、陈仓、散关入蜀（不经潼关）；**至德二载（757）九月、十月**唐军郭子仪 + 李光弼 + 回纥兵反攻——**复东都洛阳、收驰道**。**自 755 末到 757 末两年间**——两京驰道在叛军 + 唐军间反复易主；驿馆 + 驿道毁坏严重。

### 此刻意味

对天宝十四载的关中 + 河南人民来说——长安-洛阳驰道是**最熟悉的两京通衢**：日常商旅 + 漕粮 + 官使过境。对长安那一代士人来说——驰道是**两京一体的物理纪念**：长安出延秋门东行、过潼关、东出函谷新关、至洛阳——这是盛唐"两京并行"的物理体现。**杜甫的《北征》《三吏》《三别》等**——皆作于安史中、写驰道沿途；但**这些诗在 755 时未作**——是中唐战乱诗的核心。

唐人尚不知道——再过半年驰道将成战场；至德 / 乾元间驰道沿途州县反复易主、驿馆烧毁、漕运中断。**自此中唐驰道**——再不复盛唐繁华；漕运改自江淮经汴州、徐州、运河至洛阳——驰道作为唐廷主要的物流通道地位下降。**今游人重走长安-洛阳一线**——所见的是**陇海铁路 + 西宝高速 + 连霍高速**的现代驿道——**盛唐人在 755 所走的那条驰道**：千二百多年后路线大体在 同一地理走廊上、但驿馆 + 关塞 + 沿途景观 几无原物存留。

## 子条目

### 唐驿政（三十里一驿、十里一亭）

**《唐六典》驾部郎中条**：**唐天下水陆驿一千六百三十九**（盛唐前期）；**每驿配马 8-75 匹 + 驿夫 + 驿令**；**三十里一驿、十里一亭**是常制。**长安-洛阳驰道沿途**：唐令常制三十里置驿、沿崤函走廊接力。**唐驿政为"军政急讯 + 漕运 + 商旅 + 朝廷使节"四用**——其中以**急讯日数百里**（最快传递速度、用于军报 + 紧急国事）最显——是中国驿政的高峰。

### 关中段（长安出春明门 + 至潼关）

**长安东出春明门 / 延秋门 + 渡霸桥 + 新丰 + 渭南 + 华州 + 华阴 + 潼关**——常旅五六日程、急讯一二日程；沿崤函走廊、唐令常制三十里置驿接力。**沿途著名地标**：
- **霸桥**：长安东郊送别地；唐诗多咏；
- **新丰县**：汉新丰故地、汉高祖刘邦故里之一；
- **华阴郡 + 华阴县**：华山下、华阴郡太守驻郑县；
- **潼关**：关中东大门、扼黄河 + 秦岭咽喉（见 [潼关](./潼关.md)）。

**关中段是驰道的西段**——是盛唐文人最熟悉的一段：长安东出送别、必到霸桥；过华州 + 入潼关、是长安东出的物理转折点。

### 河南段（潼关 + 弘农 + 陕郡 + 新安函谷新关 + 至洛阳）

**出潼关 + 弘农郡（虢州）一线 + 陕郡治陕县 + 渑池 + 新安函谷新关 + 洛阳**——常旅五六日程；沿崤函走廊、唐令常制三十里置驿接力。**沿途著名地标**：
- **函谷新关**（汉武元鼎三年/前 114 东徙至新安、即河南府新安县境）；
- **陕郡治陕县**（今三门峡市陕州区）——黄河漕运咽喉、三门峡险段、陕郡太守驻；
- **渑池**（今渑池）——黄河南岸古县、"渑池会盟"地（战国秦赵）；
- **新安县**（今新安）——洛阳西郊、函谷新关所在；
- **洛阳河南府治洛阳县**——东都终点、河南府尹驻。

**河南段是驰道的东段**——是黄河漕运 + 驰道并行的一段、亦是盛唐两京漕运的物理 anchor。

### 安史中驰道的失而复得（755 末-757 末）

**至德元载 / 至德二载（756-757）间长安-洛阳驰道的反复易主**：
- **天宝十四载十一月**（农历；公历 755 年 12 月）安禄山起兵于范阳；**十二月丁酉陷东京洛阳**（公历已入 756 年 1 月、驰道东段失）；
- **756 年正月**安禄山在洛阳称帝；
- **756 年六月**哥舒翰潼关之败、关中失（驰道关中段失）；
- **至德元载六月十三日**玄宗西奔——经驰道关中段最后一程西行；
- **757 年九月**唐军 + 回纥兵东出潼关、复长安；
- **757 年十月**复东都洛阳——**驰道全线复**。

**自此驰道两次全线易主**——是中国驿政史最戏剧的两年。**杜甫《北征》《三吏》《三别》等**——皆作于这一段时间、写驰道沿途的废墟与百姓。

### 盛唐两京制下的驰道

**唐两京制**：长安 + 洛阳并立、自唐初至盛唐 130 年。**驰道是两京一体的物理纪念**——历代帝王往返两京、必经驰道。**玄宗朝**：**前期（713-741）开元间玄宗多次"就食洛阳"**（关中歉收时）——经驰道东向；**开元二十四年（736）十月最后一次自洛阳还长安**——**自此玄宗朝再未东巡 19 年**（至 755）。**这一段是盛唐两京制的渐衰**——驰道仍繁忙、但帝王仪仗已不再东向。

### 漕运改革（开元二十二年/734 裴耀卿）

**开元二十二年（734）裴耀卿任江淮、河南转运使**——改漕运法、置**河阴仓 + 集津仓 + 盐仓**三门转运体系（以陆运十八里绕过三门峡险段、节级转输）；**自此关中粮食有保障**——玄宗朝再未"就食洛阳"。**漕运法的物理 anchor**：长安-洛阳驰道沿途的陆运 + 黄河漕运 + 通济渠水运三层。**自此长安成核心、洛阳为陪都的格局正式确立**——是 755 时盛唐两京格局的基础。

## 历史长河（755 → 今）

**中晚唐**　**至德 / 乾元间驰道反复易主**；**至德二载（757）秋唐军收复两京**——但驰道沿途州县已疮痍。**贞元 / 元和间唐廷重置驰道**——但已不复盛唐繁华。**自此中唐驿政的核心**：自江淮经汴州 + 徐州 + 运河至洛阳——驰道沿途州县地位下降。

**唐末 + 五代**　**广明元年（880）黄巢入长安**——驰道再次成战场；**唐末驰道几度毁修**；**五代后梁、后唐皆都洛阳**——驰道地位再次提升、但已是地方割据时代。

**两宋**　**北宋初定都开封**——长安 + 洛阳皆退为陪都；**驰道沿途州县**继续作为关中 + 河南间的交通线、但作为"两京驰道"的核心地位结束。

**金元明清**　元代驰道改建为大都 - 关中线一部分；明清间驰道沿途有"潼关 + 函谷关 + 崤函古道" 等关塞遗存——但已非主要驿路。

**民国至今**　**1934 年陇海铁路通车至西安**——成为长安-洛阳间的现代陆路主干线（与古驰道大体平行）；**1958 年起 + 1990s-2000s 西宝高速 + 连霍高速**——形成今日西安-洛阳间的高速公路。**唐人在 755 所走的那条驰道**——千二百多年后**地理走廊大体未变**；但驿馆 + 关塞 + 沿途景观 几无原物——是中国驿政史最大的一次"物理终结"。
