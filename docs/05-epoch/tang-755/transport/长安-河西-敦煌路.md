# 长安-河西-敦煌路

## 一句话

**唐丝路东段** + **长安出河西至敦煌的官道**——长安 → 新平郡（邠州）→ 安定郡（泾州）→ 天水郡（秦州）→ 金城郡（兰州）→ 武威郡（凉州）→ 张掖郡（甘州）→ 酒泉郡（肃州）→ 晋昌郡（瓜州）→ 敦煌郡（沙州）→ **北道**出玉门关 / **南道**出阳关；**755 时是中国境内最长的官道之一**——里程甚长、驿名与逐段距离须据图经 / 郡县志另列；安史前夕仍是粟特商人 + 西域使节 + 唐使的主要孔道。

## 形貌

长安-河西-敦煌路自长安经陇右、河西诸郡至沙州、里程甚长——是唐时中国境内最长的官道之一；具体里程、驿名与逐段距离须据图经 / 郡县志另列。**自东向西沿途要塞** + **大致段落**：

**关中-陇右段**（长安 → 新平郡 → 安定郡 → 天水郡 → 金城郡）：
- **长安出春明门 / 开远门西北行**——是河西路的起点；
- **新平郡（邠州）+ 安定郡（泾州）**（关内道北部）；
- **天水郡（秦州）**（陇右节度辖境东端、麦积山所在）；
- **金城郡（兰州）**（陇右节度辖境、黄河上游门户）。

**河西走廊段**（金城郡 → 武威郡 → 张掖郡 → 酒泉郡 → 晋昌郡 → 敦煌郡）：
- **武威郡（凉州）**（河西节度治）——河西走廊东端、"七里十万家"；
- **张掖郡（甘州）**；
- **酒泉郡（肃州）**；
- **晋昌郡（瓜州）**——玄奘 629 偷渡玉门关地；
- **敦煌郡（沙州）**——河西走廊最西端、莫高窟所在。

**沙州 / 瓜州西出**：分**北道**与**南道**两支——
- **北道**：自**瓜州（晋昌郡）西出玉门关**、入**莫贺延碛**、至伊吾——是玄奘 629 所行路线；
- **南道**：自**沙州寿昌县西南出阳关**、过**白龙堆**、向鄯善 / 于阗——是丝路南道东端。

**驿馆制度**：盛唐河西路沿途驿馆按《唐六典》"三十里一驿"常制设置——河西诸州人烟稀疏、实际驿馆相距比关中更远（《元和郡县图志》多次记沙州至瓜州间驿馆稀）；具体驿名与处数须据图经 / 郡县志另列。

**物候**：自关中向西、逐渐进入黄土高原 + 河西走廊 + 沙漠戈壁——夏温冬寒、风沙渐大、人烟渐疏；**长安-河西-敦煌路是中国地理最长 / 最复杂 / 物候最多变的一条驿道**。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

长安-河西-敦煌路在唐人的回望里——是**丝路东段 + 唐西陲军政命脉 + 玄奘西行路**。

**驰道的历史**：
- **秦汉初**：自长安西北至河西走廊的官道初成；汉武帝平河西 + 立四郡（武威、张掖、酒泉、敦煌）后、河西路成定制；
- **北朝至隋**：河西走廊多次为中央政权与游牧民族争夺；
- **唐承前代**——河西路是唐安西大都护 + 陇右节度 + 河西节度的物理生命线。

**唐河西节度 + 陇右节度的关系**：
- **河西节度**（治凉州）——统凉、甘、肃、瓜、沙、伊、西七州；
- **陇右节度**（治鄯州）——统鄯、廓、河、洮、岷、秦、渭、临、武、成 等 10 余州；
- **两节度合管"河西陇右"** —— 是盛唐西陲军政重心、兵 10 余万。

**驰道的具体使用者**：**粟特商人** + **西域诸国使臣** + **唐使节赴西域 / 自西域返长安** + **僧侣（如玄奘 629 西行）** + **戍兵调动** + **漕运（小规模、主要靠陆运）**。

**玄奘西行的实际路线**：**贞观三年（629）秋八月玄奘自长安启行**——经秦州、兰州、凉州、甘州、肃州、瓜州、沙州、出玉门关、入莫贺延碛、过伊吾、至高昌——是唐河西路在盛唐前期的最有名记录。

### 此刻

天宝十四载这一年的长安-河西-敦煌路——**繁忙、按例运行、安史前夕的西陲生命线**。

**驰道状态**：盛唐 755 时河西路沿途驿馆完整、驿马按例轮换、驿夫驻馆；**自长安西行至沙州行程甚长**（普通商旅约两月程、急讯则速、具体日程须据图经另核）；**沿途郡治**：京兆府 + 新平郡（邠州）+ 安定郡（泾州）+ 天水郡（秦州）+ 金城郡（兰州）+ 武威郡（凉州）+ 张掖郡（甘州）+ 酒泉郡（肃州）+ 晋昌郡（瓜州）+ 敦煌郡（沙州）——天宝元年 / 742 改州为郡后皆有**太守**驻、戍兵守关、关吏验牒。

**丝路通行**：**755 时河西路仍是丝路东段主要孔道**——粟特商人车马络绎、西域诸国使臣往返、唐使节赴安西大都护府 + 北庭都护 + 西域诸国 / 自西域返长安。**751 怛罗斯之战**唐败后丝路西端进入新平衡——但**755 时河西路本身仍按例通行**。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九）；起兵后**河西陇右兵东调救京**——河西路的西陲军镇守备日空；**至德元载/756**——河西路东段（陇右部分）随安史东西两线变化而反复；**广德元年/763**——吐蕃陷长安、陇右危局加剧——河西诸州自此暴露于吐蕃东进之锋；**765-766 凉州陷**；**766-781 河西诸州陆续失**；**沙州约 781 / 786 前后陷蕃**——自此河西路 70-80 年间为吐蕃辖境；安西大都护府孤悬西域（至贞元六年/790 北庭失）。

**755 时河西路上的实际景象**：**安西使节带文书往返 + 粟特商人车马络绎 + 戍兵 + 僧侣** —— 是盛唐西陲最具象的物理 anchor；逐段里程与驿名须据图经 / 郡县志另列。

### 此刻意味

对天宝十四载的河西人民来说——河西路是**自家境内的西陲生命线**：日常戍守 + 商旅 + 使节往返。对长安那一代士人来说——河西路是**西陲诗 + 丝路想象的物理 anchor**：王维"渭城朝雨" + 王之涣"春风不度玉门关" + 王翰"葡萄美酒夜光杯" + 岑参《白雪歌送武判官归京》——盛唐边塞诗派多写河西路。

唐人尚不知道——再过 10 余年（765-781）河西诸州陆续陷蕃、河西路渐失唐廷控制；**自此河西路 70-80 年间为吐蕃辖境**——直至**大中年间张议潮归义军起义（848）+ 短暂复归唐廷**；但唐廷已无力恢复盛唐时的河西节度。**今游人重走长安-敦煌一线**——所见的是**陇海铁路 + 兰新铁路 + 连霍高速 + G312**的现代驿道——**盛唐人在 755 所走的那条河西路**：千二百多年后路线大体在 同一地理走廊上、但唐代驿馆 + 关塞 几无原物（唯莫高窟 + 玉门关 + 阳关遗址保留）。

## 子条目

### 玄奘西行（贞观三年/629 秋八月）

**玄奘自长安启行的实际路线**——是唐河西路在盛唐前期最完整的具体记录：
- **贞观三年（629）秋八月**自长安启行；
- **同年秋至凉州**——凉州都督府（贞观时河西节度尚未置、节度使制约景云至开元间始定）；
- **再西至甘州 + 肃州 + 瓜州**——在瓜州被瓜州刺史独孤达接见（贞观时州县未改郡）；
- **同年秋末玄奘自瓜州偷渡玉门关**——出河西路；
- **入莫贺延碛 + 至伊吾 + 转折至高昌**。

**玄奘 + 河西路**是唐人对河西路最具体的"个人证词"——《大慈恩寺三藏法师传》卷一记此段甚详；**自玄奘起、河西路在唐人地理认知中具体化**：每一个州 + 每一座关 + 每一段沙漠都有"玄奘曾过"的标记。

### 河西诸州（武威 / 张掖 / 酒泉 / 晋昌 / 敦煌）

**河西走廊上的五大郡**——是河西路的核心节点：
- **武威郡（凉州）**——河西节度治；"七里十万家"——盛唐西陲第一城；
- **张掖郡（甘州）**——河西走廊中段；丝路转运站；
- **酒泉郡（肃州）**——河西走廊西段；
- **晋昌郡（瓜州）**——玄奘 629 偷渡玉门关地；
- **敦煌郡（沙州）**——河西走廊最西；莫高窟所在。

**唐河西节度治凉州 + 统凉、甘、肃、瓜、沙、伊、西七州**——是盛唐西陲军政体系；自武威郡西出至敦煌郡里程甚长、具体里数须据图经另列。

### 河西节度（治凉州）+ 陇右节度（治鄯州）

**盛唐西陲两大节度**：
- **河西节度**（治凉州）——统河西七州；驻兵约 7 万；
- **陇右节度**（治鄯州）——统陇右 10 余州；驻兵约 7 万。

**天宝十二载（753）哥舒翰兼河西、陇右两镇节度使**——一身两镇、辖兵 13 万——是盛唐边军最大势力之一（另一是范阳 + 平卢 + 河东三镇的安禄山）。**755 时哥舒翰兼掌**——是河西路 + 陇右路最稳定的军政状态。**半年后**：**安史之乱起 + 河西陇右兵东调救京**——西陲守备骤空、为 763 吐蕃东扩埋下种子。

### 粟特商人 + 西域使节

**盛唐河西路的实际过往者** + **唐对外贸易的主力**：
- **粟特商人**：自中亚撒马尔罕 + 布哈拉 + 木鹿等"昭武九姓"国出发、过葱岭 + 沙州 + 凉州 + 长安——是丝路东段的主体商旅；
- **西域诸国使臣**：高昌 + 龟兹 + 焉耆 + 于阗 + 疏勒 + 吐蕃 + 大食 等使臣经河西路东达长安朝贡；
- **唐使节**：赴西域 / 自西域返长安；
- **僧侣**：玄奘 + 后续西行求法者。

**这些流量 + 河西路的稳定**——是盛唐西陲国际形象的物理 anchor。

### 安史后河西路的失（756-786）

**安史之乱及其后长安-河西-敦煌路的变迁**：
- **755 末-756 初**：河西陇右兵东调救京——西陲守备日空；
- **757 末**：唐军复两京——但河西陇右兵已损耗严重；
- **763**：**吐蕃陷长安**（短暂）+ 陇右危局加剧——河西诸州自此暴露于吐蕃东进之锋；
- **765-766**：**凉州（武威郡）陷蕃**；
- **766-781**：**河西诸州（甘、肃、瓜）陆续陷蕃**；
- **约 781 / 786**：**沙州（敦煌郡）陷蕃**——河西路彻底为吐蕃所辖；
- **790**：吐蕃陷北庭——安西孤悬西域；
- **848 张议潮归义军起义**——**唐宣宗大中二年张议潮起兵复沙瓜诸州**——但唐廷已无力恢复盛唐河西路。

**自此河西路 70-80 年间为吐蕃辖境、其后归义军接掌**——是盛唐西陲军政体系的彻底解体。

### 莫高窟 + 玉门关 + 阳关（河西路西端文物）

**河西路西端的三大物理遗存**：
- **莫高窟**（沙州境）——盛唐至唐末持续开凿；
- **玉门关**（瓜州西出方向）——盛唐丝路**北道**西端、西出即莫贺延碛；
- **阳关**（沙州寿昌县西南）——盛唐丝路**南道**西端、西出向白龙堆。

**今唐人在 755 所走河西路的物理遗存**：以这三处为核心。**2014 年莫高窟 + 玉门关 + 悬泉置 等敦煌段遗址列联合国教科文组织世界文化遗产**（作为"丝绸之路：长安-天山廊道路网"组成部分）。

## 历史长河（755 → 今）

**中晚唐**　**763 吐蕃陷长安、陇右危局**；**765-786 河西诸州（凉、甘、肃、瓜、沙）陆续陷蕃**——河西路渐失唐控制；**848 张议潮归义军起义**——沙瓜诸州复归唐廷；**唐末归义军继续守河西**——但唐廷已无力恢复盛唐河西路。

**两宋至元**　**北宋河西归西夏（1038 立国）**——河西路成西夏 vs 北宋 vs 西州回鹘的边境；**元代蒙古统一中亚后丝路再通**——但河西路已不再是核心驿路（蒙古重视北方草原路）。

**明清**　**明初置关西七卫 + 嘉峪关为明长城最西关**——河西路彻底退出明廷边境；**清乾隆朝平准 + 平回（1755-1759）后新疆纳入清版图**——河西路重新作为陆路通道、但物流主体已转为茶马互市 + 漠北线。

**民国至今**　**1949 年后兰新铁路 + 连霍高速 + G312** 形成今日长安-敦煌一线的现代交通——大体沿盛唐河西路走廊。**今游人重走河西路**——所见的是**铁路 + 高速公路 + 莫高窟 + 玉门关 + 阳关 + 嘉峪关 + 凉州古城 + 甘州大佛寺**等历代遗存。**盛唐人在 755 所走的那条河西路**——千二百多年后路线大体在同一走廊、但唐代驿馆 + 关塞 几无原物存留（仅莫高窟 + 玉门关 + 阳关遗址保留）。
