# 玉门关

## 一句话

**河西走廊最西北的关塞**——盛唐人嘴里"春风不度玉门关"那一关；**贞观三年（629）玄奘自瓜州偷渡玉门关 + 入莫贺延碛**；**755 时仍立、是丝路东段中国境内的最远关之一**——但唐玉门关具体位置史源略异（瓜沙边境）。

## 形貌

玉门关在唐**陇右道沙州敦煌郡或瓜州晋昌郡西界**——具体位置史源略异：**唐《元和郡县图志》记敦煌西北 140 唐里**（约今甘肃省敦煌市西北 75 公里的**小方盘城遗址**——汉玉门故关址）；**另一说唐初玉门关已东移至瓜州（晋昌郡）西界**——玄奘《大慈恩寺三藏法师传》记其 629 自瓜州偷渡玉门关——是这一说的关键证据。**本 doc 严守"唐玉门关地理 anchor 在瓜沙边境"宽口径**——不预设具体关址。

**地势**：**河西走廊最西端**——敦煌绿洲（沙州治）以西、瓜州（晋昌郡）以北、戈壁与沙漠的交界。**北望马鬃山的余脉**——再北便是天山以南的戈壁；**南望祁连山西段**——党河自祁连冰川北流入敦煌绿洲；**西出关 + 即莫贺延碛**（即唐玄奘所述"八百里流沙"）——是中亚通往中国境内的最远孔道。

**关塞之险**：**戈壁茫茫无险**——玉门关之所以为关塞、不靠山险水险、而是靠"戈壁无替代路线"的天然封闭。**自玉门关 + 阳关向东**是河西走廊（绿洲连接）；**向西**是莫贺延碛（无人沙漠）——一关守一片沙漠的东入口。

**关城形制**：**汉玉门故关（小方盘城遗址）**：方形夯土城墙、边长约 25 米、墙高 9 米、有两门——是汉代最西边塞城堡的典型样本。**唐玉门关位置不一**：若仍在汉址、形制大体延续；若已东移至瓜州境、则形制史载未详。**本 doc 不预设盛唐具体关楼形制**。

**物候**：极干燥、风沙大、夏热冬寒、温差极大；**戈壁无水源**——出关后需自带水 + 取水于稀有泉源点。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

玉门关在唐人的回望里——是**河西走廊最西北的关塞 + 边塞诗的高频地名**。

**汉玉门关的设立**：**西汉武帝元鼎六年（前 111）**——汉武帝平河西后置敦煌郡、立玉门关 + 阳关二关守河西最西。**"玉门"二字之源**：传说"和阗（于阗）玉自西域输入中国必过此关、故名玉门"（《汉书·西域传》注引应劭说）——这是汉代关塞命名的常见模式（以贡物或地理特征命名）。**自汉至西晋玉门关**——是丝路东段的关键节点；汉代敦煌郡西界。

**西晋至北朝间玉门关位置数次东移**：**因河西走廊西端的沙漠化 + 政治变迁 + 商路 shift**——玉门关位置不再固定。**唐初玉门关位置史源略异**——一说仍在汉址（敦煌西北）、一说已东移至瓜州西界。**玄奘《慈恩传》记其 629 自瓜州偷渡玉门关**——若汉址、瓜州距汉玉门关约 260 公里、玄奘记述细节不甚符合；若已东移至瓜州境、玄奘记述则吻合。**本 doc 严守'唐玉门关地理 anchor 在瓜沙边境'宽口径**。

**盛唐玉门关的政治地位**：**唐承前代之制 + 玉门关属沙州或瓜州**；**驻军、关税、验牒** 等关塞常制；**玉门关是唐西域交通界标 + 诗歌界标**——盛唐人嘴里的"出玉门"即"出中国境内、入西域"。

**唐玉门关在文学上的高频化**：**盛唐边塞诗 + 怀古诗中"玉门关"二字出现频繁**：
- **王之涣《凉州词》**："黄河远上白云间，一片孤城万仞山。**羌笛何须怨杨柳，春风不度玉门关**。"——是盛唐"玉门关"诗的最高峰。
- **王昌龄《从军行》《出塞》**：盛唐边塞诗代表（含涉玉门关意象）；
- **岑参《白雪歌送武判官归京》《走马川行》**：盛唐边塞诗派的核心、多次写边塞过往；
- **王翰《凉州词》**：与王之涣《凉州词》并立的盛唐边塞代表。

**注**：**这些诗多作于盛唐 + 755 前后**——是盛唐边塞文学的高峰；755 时盛唐人嘴里的"玉门关" 既是地理实体、也是文学符号。

### 此刻

天宝十四载这一年的玉门关——**关塞严整、驻军戍守、丝路通行**。

**关城状态**：盛唐 755 时玉门关主体仍立（具体关址在瓜沙边境哪一点、史源不一）；夯土关楼 + 戍兵营 + 驿马 + 关吏。**关税收**：玉门关收西域来贡 / 商旅过关税——是唐安西 + 河西节度财政的重要收入。**戍军**：**玉门关属唐陇右节度（或安西大都护）辖境**——具体驻军数史载零散；按盛唐西陲驻军规制、玉门关驻兵约数百至千余。

**丝路通行**：**755 时玉门关仍是丝路东段的主要孔道之一**——粟特商人、西域诸国使臣、僧侣（含玄奘式西行求法者）、汉商等过往不绝。**自玉门关 + 阳关分流**——玉门关向北经天山南麓 + 庭州（北庭大都护治）；阳关向南经塔克拉玛干南缘 + 于阗（安西四镇之一）。

**安史前夕**（即 755 这一刻）——**751 怛罗斯之战**唐军败于大食、丝路西端进入新平衡；**755 年安史之乱起后**——河西陇右兵东调救京、玉门关守备日空。**广德元年（763）后吐蕃陷河西**——**安西大都护府孤悬西域 + 玉门关一带渐失唐廷直接控制**。**755 时玉门关是盛唐西陲的物理 anchor + 即将进入大变局的前夜**。

### 此刻意味

对天宝十四载的沙州 / 瓜州人来说——玉门关是**自家郡境的西界关塞**：日常驻守 + 西域商旅过关 + 出关入碛者偶有。对长安那一代士人来说——玉门关是**边塞诗的具象**：**王之涣"春风不度玉门关"**——盛唐文学最具象的西陲符号（**注**："愿得此身长报国、何须生入玉门关" 实为戴叔伦《塞上曲二首·其二》中唐作；本 doc 严守、不归盛唐王昌龄）。**盛唐人嘴里的"玉门关"**——既是地理实体、也是文学符号；甚至更是文学符号（多数士人未亲至玉门关、但通过盛唐边塞诗熟知）。

唐人尚不知道——再过 8 年（763）吐蕃陷河西、安西孤悬；再过 50 年（约 9 世纪初）玉门关一带渐失唐廷控制；再过千余年（19 世纪末），**英国斯坦因（Aurel Stein）等探险队至小方盘城等遗址勘察 + 出土汉简数千枚**——确认汉玉门关位置在敦煌西北小方盘城遗址；**今玉门关遗址（小方盘城）列联合国教科文组织世界文化遗产**（2014 年作为"丝绸之路：长安-天山廊道路网"组成部分）。**唐人在 755 所见的那座玉门关**——千二百多年后**汉故关址犹存（小方盘城遗址）**；**盛唐唐玉门关具体位置史源不一、相关遗存散布瓜沙边境**——是丝路考古最复杂的关塞之一。

## 子条目

### 汉玉门故关（小方盘城遗址）

**汉武帝元鼎六年（前 111）立的玉门关原址**——今**甘肃省敦煌市西北 75 公里戈壁中的小方盘城遗址**。**形制**：方形夯土城墙、边长约 25 米、墙高 9 米、有两门——是汉代最西边塞城堡的典型样本。**1907 年英国斯坦因首次勘察**；**1944 年向达、夏鼐勘察**；**1979 年起多次考古发掘**——出土汉简数千枚、是研究河西汉边塞的关键实物。**今小方盘城遗址列国家级重点文物保护单位 + 2014 联合国教科文组织世界文化遗产**。

**注**：**盛唐 755 时唐玉门关是否仍在小方盘城遗址**——史源略异；玄奘 629 自瓜州偷渡玉门关或可推断唐玉门关已东移。**本 doc 严守"唐玉门关位置不预设小方盘城" 口径**——但小方盘城作为汉玉门故关的物理 anchor、是盛唐 755 时唐人地理认知中"玉门关"的 historical baseline。

### 玄奘偷渡玉门关（贞观三年/629）

**玄奘（602-664）**——唐初高僧、佛教汉译史最重要人物。**贞观三年（629）秋八月玄奘自长安启行西行**（《大慈恩寺三藏法师传》卷一）；**同年**至凉州 + 甘州 + 瓜州——在瓜州被刺史独孤达接见；**贞观三年秋末玄奘自瓜州偷渡玉门关 + 过五烽 + 入莫贺延碛（八百里流沙）**——是盛唐前最有名的"西行偷渡" 故事。

**偷渡的细节**（《慈恩传》记）：**玄奘自瓜州寻得胡僧石槃陀做向导**；**夜渡葫芦河（疏勒河）**；**过玉门关 + 出五烽**；**入莫贺延碛**——独自一人一马、避烽偷渡。**自玉门关 + 出莫贺延碛 800 里 + 至伊吾**——是玄奘西行最危险的一段路。**这段经历**：玉门关在玄奘心中是"中国境内最西的关塞"；过此关 + 进入"番地"——是盛唐唐人地理认知中"中国 vs 西域"的物理界限。

**注**：**玄奘所过的玉门关位置 + 时间（629）+ 自瓜州出关的细节**——是唐玉门关在瓜州西界一说的关键证据。本 doc 严守。

### 王之涣《凉州词》"春风不度玉门关"

**王之涣（688-742）**——盛唐边塞诗派代表诗人；**生卒年覆盖盛唐前期**；**《凉州词》二首**是盛唐边塞诗的最高峰之一。

**《凉州词·其一》**：
> **黄河远上白云间，一片孤城万仞山。**
> **羌笛何须怨杨柳，春风不度玉门关。**

**注**：王之涣本人是否亲至玉门关——史源未详（王之涣生平多在并州 / 蒲州一带）；**这首诗的"玉门关" 主要是文学想象 + 地理符号**——不一定亲见。**但《凉州词》自盛唐起即成"玉门关诗" 的代表**——后世凡咏玉门关者必引此句。

**自盛唐起、玉门关在文学中的位置**：是"西陲极远 + 春风不到 + 戍卒思乡" 三层意象的凝聚——是中国边塞文学的最高物理 anchor 之一。

### 莫贺延碛（玉门关西的八百里流沙）

**莫贺延碛**——玉门关 + 五烽以西的沙漠戈壁。**自玉门关向西**：先过五座烽燧（汉代敦煌郡西边的烽燧体系），再入莫贺延碛——**800 里无人沙漠**（《慈恩传》语）；**碛中无水**——商旅必自带水 + 在稀有泉源点（含玄奘所记的"野马泉"）取水。**自碛西出至伊吾 + 转折抵高昌**——是丝路东段最艰难的一段。

**盛唐 755 时**——莫贺延碛仍是无人区、丝路商旅 + 西行僧侣的最难关口。**自玉门关 + 西出莫贺延碛 + 至伊吾**约**1500 唐里 / 20 余日程**——比河西走廊（自长安至沙州约 4500 唐里）的最后一段最艰苦。

### 玉门关 + 阳关（"二关"分守河西最西）

**玉门关 + 阳关**——汉武帝元鼎六年（前 111）同时立、合称"河西二关"：
- **玉门关**：敦煌西北、扼**北道**（沿天山南麓的丝路北道）；
- **阳关**：敦煌西南、扼**南道**（沿塔克拉玛干南缘的丝路南道）。

**二关分流**：丝路自玉门关 + 阳关分两路、西行至西域诸国后再汇合。**盛唐"出阳关"+"出玉门"为西行二大途径**——王维《送元二使安西》"西出阳关无故人"、王之涣《凉州词》"春风不度玉门关"——是盛唐两关诗的双璧。

## 历史长河（755 → 今）

**中晚唐**　**广德元年（763）后吐蕃陷河西**——玉门关一带渐失唐廷直接控制；**贞元六年（790）吐蕃陷北庭**——丝路东段彻底进入吐蕃 + 后续回鹘控制时期。**自此玉门关作为唐廷关塞的功能结束**——但作为丝路要塞、商旅仍偶有过往（不再缴唐廷关税）。

**两宋**　**北宋初河西已属西夏（1038 立国）** + **西州回鹘 / 高昌回鹘**——玉门关地区为西夏 vs 回鹘的边境。**北宋丝路通过路线** 主要在西南方向 (河湟 + 西宁 + 茶马互市路)；**北宋玉门关已退冲要**。

**金元**　**蒙古帝国统一中亚后 (1206-1271)**——丝路畅通、但玉门关位置已不在主要通道；**元代玉门关一带为蒙古游牧地**。

**明清**　**明初置关西七卫**——含敦煌一带的"沙州卫"；**但明嘉靖后明廷退守嘉峪关**（在玉门关东 + 嘉峪关山一线、是明长城最西端）——**玉门关彻底退出明廷边境**。**清乾隆朝平准 + 平回（1755-1759）后新疆纳入清版图**——玉门关重新进入中国地理认知，但已是历史遗址、不再是关塞。

**民国至今**　**1907 年英国斯坦因首次勘察小方盘城**（出土汉简等）；**1944 年向达、夏鼐勘察**；**1979 年起多次考古发掘**——确认汉玉门关位置在小方盘城遗址。**1981 年小方盘城列国家级重点文物保护单位**；**2014 年作为"丝绸之路：长安-天山廊道路网"的组成部分列联合国教科文组织世界文化遗产**。**唐玉门关具体位置**：史源 + 考古至今仍有讨论——盛唐 755 时关址可能在汉故关、可能东移至瓜州境。**今游人来玉门关遗址**——所见的是**小方盘城遗址 + 戈壁 + 部分汉烽燧基址**——**唐人在 755 嘴里的玉门关**：千二百多年后**地理 anchor 犹存**、但具体唐关址尚未完全确定。
