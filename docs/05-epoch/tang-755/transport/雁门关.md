# 雁门关

## 一句话

**河东北门 + 太原北翼第一关**——代州雁门县境北的雁门山中；自春秋秦汉以来是关中 / 河东通漠北的塞；**755 时是河东节度辖境的北面屏障**——盛唐边塞诗高频地名；**杨家将故事是宋元后形成、非唐时之事**。

## 形貌

雁门关在唐**河东道代州雁门郡**——具体在**雁门县北的雁门山中**（今山西省忻州市代县北 20 公里境）。**地势**：**句注山脉（雁门山）主脊**——是山西黄土高原与漠南草原的天然界；**雁门关在山口**——海拔约 1500 米；**自此关南行 + 入河东腹地（代州 + 太原府）**；**自此关北行 + 入漠南草原（突厥 / 后回纥）**。

**雁门之名**：传说**雁飞过关 + 必经此口**——故名"雁门"；亦或因"句注山如雁张翼" 得名。**自春秋至两汉**雁门已是关塞——秦汉时雁门郡是河东最北的边郡之一。**北朝至隋唐间**雁门关延续作为边塞——是河东节度防漠北的核心物理 anchor。

**关塞形制**：**唐雁门关**——夯土包石关楼 + 关墙；具体形制盛唐史载零散。**今所见雁门关**：**明洪武七年（1374）大规模重修**、明嘉靖间复修——形成今"明长城外三关之一"格局（与宁武关、偏关并立"外三关"）；**非唐时形制**。

**与雁门关相关的几座山**：
- **句注山（雁门山）**：关所在主脊；
- **太行山西端**：与雁门山连成"晋北长城线"；
- **管涔山**：太原府北的另一山系；与雁门山相距约 100 公里。

**物候**：晋北高寒——夏温冬寒、雪较多；山中林木较盛；**关城内外** 多有戍兵 + 驿马 + 商旅。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

雁门关在唐人的回望里——是**河东北门 + 太原北翼第一关 + 千年边塞**。

**雁门关的历史**：
- **春秋秦汉**：雁门已是关塞；**赵武灵王（前 325-前 299）筑长城自代并阴山下至高阙、置云中、雁门、代郡**（《史记·匈奴列传》原文）——是雁门作为长城关塞的最早记录。
- **西汉**：**雁门郡**（治善无）是河东最北的边郡之一；**李广曾任雁门太守**——"飞将军"称号见于其守右北平时（《史记·李将军列传》）；本 doc 严守"飞将军"是右北平典故、非雁门。
- **东汉至北朝**：雁门多次成为汉匈、北朝突厥间的战场。
- **隋唐**：雁门作为河东北门的关塞地位继续。**唐承前代之制**——置代州雁门郡；雁门关属代州。

**唐雁门的政治军事地位**：**唐承隋制 + 置代州**；**雁门关属代州 + 河东节度辖境**——河东节度治太原府、统代州 + 朔州 + 云州 + 蔚州 + 忻州 等北部诸州。**自唐初至盛唐 138 年（618-755）雁门多次有战事**——主要是与漠北突厥 + 回纥的接触；但**安史前 138 年间**雁门一带相对平静。

**雁门关在文学上的高频化**：**盛唐边塞诗 + 怀古诗中"雁门"二字出现频繁**：
- **王昌龄、岑参、高适**——盛唐边塞诗派多有涉雁门；
- **王昌龄《出塞》"但使龙城飞将在" + 高适《燕歌行》《塞下曲》** 等盛唐边塞诗——涉北塞意象（"龙城" "雁门" "胡马"）；具体某诗是否实写雁门 + 还是泛指北塞、史源略异。

**"雁门"作为盛唐边塞符号**：与凉州、阳关、玉门关、白城 等并立——是盛唐边塞文学的高频地名。**注**：杨家将故事是宋元后形成的民间传说——**唐时雁门关与杨家将无关**。

### 此刻

天宝十四载这一年的雁门关——**关塞严整、驻军戍守、河东北门**。

**关城状态**：盛唐 755 时雁门关主体仍立——夯土包石关楼 + 关墙；具体盛唐形制史载零散。**驻军**：雁门关属河东节度辖境；按盛唐边军规制驻兵约数百至千余。**与漠北突厥 / 回纥的关系**：**744 后东突厥灭、回纥代据漠北**——**755 时唐回纥关系仍属和好**（回纥与唐玄宗朝多有外交）；**雁门关是双方通行的物理 anchor**——回纥使节南下唐 + 唐使节北上回纥多过此关。

**安史前夕**（即 755 这一刻）——**安禄山兼河东节度治太原**（自天宝十载/751 起）；**751-755 间安禄山在河东积蓄军力 + 调动兵粮**；**雁门关作为河东北翼的关塞** 也在安禄山政治军事网络中。**755 末安禄山起兵于范阳**——**自范阳南下取洛阳**（非北上过雁门）——雁门关在安史早期未受直接兵燹；**至德 / 乾元间唐军反攻河东**——雁门关一带反复易主。

### 此刻意味

对天宝十四载的代州人来说——雁门关是**自家州境的北门**：日常戍兵 + 商旅过关 + 回纥使节南下偶有过。对长安那一代士人来说——雁门关是**河东北门 + 边塞诗的高频地名**：盛唐文学最常涉的几座关塞之一。**盛唐人嘴里的"雁门"**——既是地理实体、也是文学符号。

唐人尚不知道——再过 130 余年（885 黄巢之乱）唐边境进一步收缩；再过 300 年（北宋）杨家将故事开始在民间形成——把雁门关写成"宋辽边境的物理 anchor"；再过 600 年（明洪武七年/1374）明廷大规模重修雁门关——形成今所见"明长城外三关之一"格局。**今游人来雁门关**——所见的是**明清重修的关楼 + 关墙 + "杨家将" 民间传说景区**——**唐人在 755 所见的那座唐雁门关**：千二百多年后**唐时形制已不复**——但**关塞位置 + 句注山 + 雁门道**——一千二百多年未易。

## 子条目

### 句注山（雁门山）

**雁门关所在的主脊山**——海拔约 1500 米；**山形如雁张翼**（《尔雅》注引）。**自春秋至隋唐间**——句注山一直是河东北面的天然屏障；雁门关就在句注山的山口。**山中林木较盛**——是太原府北侧的天然生态屏障。

### 飞将军李广（汉雁门守将）

**汉武帝时李广（？-前 119）守雁门**——是雁门最有名的历代守将。**"飞将军"威震匈奴**——是中国军事史最有名的边塞名将之一；李广多次在雁门 + 上谷 + 右北平间转战、对匈奴大破。**自此李广 = 雁门**——成中国文学最有名的"边塞名将与关塞" 配对之一。**唐人 755 时凡读《史记》《汉书》李广传者**——必知雁门 + 李广的故事；**盛唐边塞诗中李广 + 雁门频繁出现**（王昌龄《出塞》"但使龙城飞将在，不教胡马度阴山" 含飞将军意象）。

### 唐与漠北（突厥 / 回纥）经雁门关的关系

**唐初至盛唐**——突厥（东突厥）、铁勒、回纥等漠北势力反复变迁。**雁门关作为唐与漠北的关塞之一**：
- **唐初**：东突厥强、雁门为前线；
- **太宗、高宗朝**：先后破东突厥（630）、突厥诸部内附；
- **武则天朝**：突厥再起（后东突厥）；
- **开元中**：突厥败、回纥渐立；
- **744**：回纥代东突厥据漠北；
- **755 时**：回纥与唐玄宗朝多有外交、雁门关是和好关塞。

**雁门关与漠北的关系**——是盛唐边境政治最稳定的一条线（相比西陲玉门关、阳关一线、雁门 - 漠北线较少大战）。

### 杨家将故事（宋元后形成；唐时与雁门无关）

**杨家将故事**——以**杨业（杨继业，约 928-986）**为核心的北宋抗辽将门家族传说；**主要在宋元间形成 + 明清间发展**。**杨业 + 雁门关的关系**：宋太宗雍熙三年（986）杨业战死陈家谷（在雁门关附近）——是杨家将故事的物理 anchor。

**注**：**唐时雁门关与杨家将无关**——杨家将是 200 年后的事。**今雁门关景区有大量杨家将主题景观**——是明清以来民间传说的累积、不是唐时之物。**本 doc 严守**：唐 755 时雁门关无杨家将。

### 雁门关在边塞诗中的位置

**盛唐边塞诗中"雁门"二字出现频繁**——多与"匈奴"、"飞将军"、"塞外"、"漠北" 等意象并出。**主要诗人涉雁门者**：王昌龄、岑参、高适 等盛唐边塞诗派多有涉；具体每首是否实写雁门或泛指北塞、史源略异——本 doc 不预设。**"雁门" 作为盛唐边塞符号**——与凉州、阳关、玉门关、白城 等并立。**注**：盛唐诗人对雁门的具体诗咏密度——比对玉门关、阳关 略低（因雁门的政治军事意义在西陲西陲较弱、文学密度自然较低）；但"雁门" 在盛唐人地理认知中仍是河东北门的代名词。

### 明长城外三关（雁门 + 宁武 + 偏关）

**明洪武七年（1374）至明嘉靖间**——明廷大规模重修雁门关 + 宁武关 + 偏关、形成"明长城外三关"格局。**外三关位于山西北部 + 北接漠南**——是明代对北边境（蒙古各部）的核心防线。**雁门关在明代地位骤升**——成"内长城" 的核心关塞之一。**今游人所见的雁门关**：**主体为明清重修**——含关楼 + 关墙 + "雁门关六郎庙" 等杨家将景区——非唐时形制。

## 历史长河（755 → 今）

**中晚唐**　**安史中雁门关一带反复易主**——河东节度治太原府失而复得；至德 / 乾元间唐军反攻河东、雁门关随之。**中晚唐雁门关续作边塞**——但唐廷北边收缩、雁门关的战略地位渐稳。**杜甫、白居易 等中晚唐诗人**——对雁门的诗咏少于盛唐。

**唐末 + 五代**　**唐末沙陀人（突厥后裔的一支）** 渐据河东 + 雁门一带；**五代后唐 / 后晋 / 后汉的核心兵源** 即沙陀人；**雁门关在五代成为沙陀军政中心**。

**北宋 + 辽**　**北宋初宋太宗雍熙北伐（986）**——宋军北攻幽云十六州、**杨业战死陈家谷**（雁门关附近）——是杨家将故事的物理 anchor。**北宋 + 辽长期以雁门关一带为分界**；**杨家将传说在北宋后期至元代渐成**。

**金元明清**　**金代雁门为金宋边境**；**元代统一后雁门关地位降**；**明代洪武七年（1374）大规模重修雁门关 + 宁武关 + 偏关、立"外三关"**——是雁门关明代地位骤升的关键时间点。**明嘉靖间复修**——形成今所见关楼 + 关墙主体。**清代雁门关续为关塞、但已退离主要防线**——是清廷"内长城"的纪念性遗存。

**民国至今**　**1949 年后雁门关列山西省重点文物保护单位**、2001 年列全国重点。**今雁门关景区**：**关楼 + 关墙明清主体 + "杨家将"民间传说主题景观 + 太行山雁门段**——是中国"内长城"最有名的关塞之一。**唐人在 755 所见的那座雁门关**——千二百多年后**唐时形制已不复**；但**关塞位置 + 句注山 + 雁门道** 一千二百多年未易。
