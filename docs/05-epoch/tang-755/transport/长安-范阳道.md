# 长安-范阳道

## 一句话

**唐自长安经河东至幽州（范阳）的官道**——长安 → 蒲津渡黄河 → 蒲州 → 太原 → 雁门 → 蔚州 → 妫州 → 幽州；**755 时一身两面**：白日是河东、河北诸州奏报与漕调的通衢、夜里是安禄山自天宝十载/751 兼河东节度使、至 755 兼范阳/平卢/河东三镇约四年所养出的"东北一只巨臂"伸回长安的潜在动脉；**十一月初九（癸亥, 公历 755 年 12 月 16 日）即由这条道反向南下、揭开安史之乱**。

## 形貌

长安-范阳道无单一定线——是唐河东、河北两道纵贯线 + 关内东北出的合称。沿途州县多、地理跨度极大（自渭水南岸至燕山以南），**史源载里程零散；本 doc 不预设虚里程**。

**自西向东大致段落** + **沿途要塞**：

**关内段**（长安出春明门 → 渭南 → 渡渭至同州 → 蒲津渡黄河）：
- **长安东出春明门**——与长安-洛阳驰道共起点；
- **东北行至同州**（治冯翊、今大荔）——关中东渡黄河的渡口州；
- **蒲津渡**——黄河上著名浮桥（开元十二年/724 张说监造铁牛铁人镇桥）；自此入河东道。

**河东段**（蒲州 → 绛州 → 晋州 → 太原 → 雁门 → 云州 / 朔州）：
- **蒲州/河东郡, 治河东县**（今永济）——河东道南端门户；
- **绛州 + 晋州**——汾水中下游、河东道腹地；
- **并州 / 太原府**（治晋阳）——**河东节度治、北都**；唐高祖龙兴之地；
- **代州**（治雁门、今代县）——河东道北端、**雁门关**所在；
- **云州**（治云中、今大同）+ **朔州**——河东节度北界、突厥-唐边境。

**河北段**（自雁门东南折 + 经蔚州 → 妫州 → 幽州；或自太原东折 + 经井陉 → 恒州 → 定州 → 莫州 → 幽州）：
- **蔚州 + 妫州**——河北道西北山地诸州、太行北段；
- **恒州**（治真定、今正定）——河北道腹地、井陉东口；
- **定州**（治安喜、今定州）+ **瀛州 + 莫州**——河北道中段；
- **幽州**（范阳郡治、治蓟县、今北京）——**范阳节度治**、河北道北端、燕山以南门户。

**两条主流走法**：**北线**（关内 → 河东 → 雁门 → 蔚州 → 妫州 → 幽州）走的是河东节度辖境一线；**南线**（关内 → 洛阳 → 河阳 → 卫州 → 相州 → 邢州 → 赵州 → 恒州 → 幽州）走的是河北平原一线、与长安-洛阳驰道前半段重合。**755 时两线并用**——南线更平、北线更险但避两京。

**驿馆制度**：按《唐六典》"三十里一驿、十里一亭"常制铺设；河东、河北诸道沿途驿馆完整、驿马按例轮换。**具体驿名史载零散、本 doc 不杜撰**。

**地势**：自渭水平原 → 渡黄河 → 入汾水谷地 → 越雁门、句注山 → 经恒山、太行山 → 落华北平原。**这是一条南北纵贯三大地理单元（关中盆地 + 山西高原 + 华北平原）的驿路**——比东出洛阳的两京驰道更复杂、更跨域。

物候：自关中温带季风渐入河东半干旱再至河北平原温带季风——夏热冬寒、北段冬有大雪、雁门以北属塞下风物。**沿途州县皆唐廷直辖、驿政按例运行**。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

长安-范阳道在唐人的回望里——是**河东、河北两道的奏报通衢 + 唐廷对东北边境的物理脐带**。

**驰道的历史**：
- **先秦至秦汉**：自咸阳 / 长安东北至燕赵地的官道初成；秦汉间已有蒲津 + 太原 + 雁门 + 蓟县（今北京）一线的驿路雏形；
- **北朝**：**北魏自平城（今大同）迁洛阳前**、平城-邺-洛一线为政治主轴；自此河东、河北南北纵贯线渐稳；
- **隋**：**隋炀帝大业三年（607）巡北塞 + 至榆林郡**——经河东、雁门一线；**大业末**雁门之围（突厥围隋炀帝于雁门）即在此道之上；
- **唐承前代**——长安出河东、河北至幽州一线、是唐对东北边境（突厥旧地、奚、契丹、室韦诸部）的物理生命线。

**唐河北、河东诸节度的格局**（盛唐天宝间）：
- **河东节度**（治太原）——统太原、汾、沁、辽、石、岚、忻、代、云、朔、蔚等十余州；防御突厥旧地、回纥南侵；驻兵约 5.5 万；
- **范阳节度**（治幽州）——统幽、蓟、妫、檀、易、定、恒、莫、沧等州；防御奚、契丹；驻兵约 9 万；
- **平卢节度**（治营州、今辽宁朝阳）——统营、平等州；防御室韦、靺鞨；驻兵约 3.75 万。

**安禄山兼三镇节度**：**天宝十年（751）二月安禄山兼河东节度**——**自此一身兼范阳、平卢、河东三镇**、辖境自燕山北至幽州、再西延至雁门 + 太原；**驻兵合计约 18 万**——是盛唐边军规模最大者（另一极是哥舒翰兼河西陇右、约 13 万）。**自天宝十年至十四载这 4 年**——安禄山借三镇节度往来于长安 + 范阳 + 太原间、奏报、献俘、朝贺频繁。**这条道是他给玄宗朝廷输入"忠顺胡将"形象的物理通道、也是他暗中调度兵力 + 蓄积势力的脐带**。

**驰道的具体使用者**：**唐廷使节** + **节度使奏报** + **戍兵换防 + 镇调** + **岁贡 + 互市**（北段经雁门、幽州与突厥旧部、奚、契丹的边市）+ **僧侣** + **商旅**等。比之长安-洛阳驰道纯粹的"两京文物物流"——长安-范阳道带着更重的**边镇军政气息**。

### 此刻

天宝十四载这一年的长安-范阳道——**表面按例运行、底下张力积满、半年内即由此爆**。

**驰道状态**：盛唐 755 时长安-范阳道沿途驿馆完整、马匹按例轮换、驿夫驻馆；**长安至幽州走完全程约月余**（普通商旅）/ **十数日**（急讯）；沿途州县（天宝元年/742 改州为郡）：京兆府 + 冯翊郡（同州） + 河东郡（蒲州、蒲津渡所在） + 绛郡（绛州） + 平阳郡（晋州） + 太原府（并州） + 雁门郡（代州） + 安边郡（蔚州） + 妫川郡（妫州） + 范阳郡（幽州）（或南线诸河北州郡）——皆有郡守（太守）驻、两府（京兆/太原）则为府尹，驿政与郡县治并行。

**安禄山其人的物理走法**：**安禄山本姓康、营州杂胡（粟特 + 突厥血统）**——出身营州（今辽宁朝阳）、自开元末入唐军、累迁至三镇节度使。**自天宝十年起 4 年间**——他多次自范阳经此道入长安朝觐 + 献俘；**天宝十三载（754）正月最后一次入朝**——上殿献俘 + 玄宗加其官、赐物甚厚；**自此再未入朝**——以"病不能行"为辞、闭门积粮、暗中备兵。**755 这一年安禄山滞范阳**——长安朝廷与范阳间的奏报往来仍走此道、但**实情已不通**。

**起兵前夕**（即 755 这一刻）——**杨国忠数次奏言安禄山必反**；玄宗将信将疑、于六月遣使诣范阳召见、安禄山辞不至；十月遣使再召、依然不至；**十一月初九（癸亥, 公历 755 年 12 月 16 日）**——安禄山在范阳**伪传敕书**、号令所部诸蕃将与汉兵共 15 万、**号召"诛杨国忠、清君侧"**——自范阳南下、揭起反旗。**此即此 doc 的核心 anchor**：**长安-范阳道的方向自此倒转**——昔日由长安发奏报 + 节度使献俘走的官道、此刻成叛军兵锋南下的进军路。

**叛军南下不走北线**（雁门 + 太原一线）——河东节度虽名义上仍属安禄山、但河东实由太原留守王承业守 (素无战才; 至德元载/756 二月李光弼方除河东节度使)、朔方节度郭子仪在西线、北线安禄山不可仰仗；**叛军主力走南线**——自范阳南下经恒州 + 赵州 + 邢州 + 相州 + 卫州 + 渡黄河 + 至洛阳；**天宝十四载十二月丁酉陷东京洛阳**（封常清守洛阳败）。**756 正月初一安禄山在洛阳称帝、国号大燕**——长安-范阳道的南端（关中段）此时尚在唐廷手中、半年后即随潼关之败而失。

### 此刻意味

对天宝十四载的河东、河北诸州人民来说——这条道是**自家境内的官道**：日常戍守、边市互市、奏报使节。对长安那一代士人来说——长安-范阳道是**"东北边镇"的物理 anchor**：高适、岑参、王昌龄、王之涣"蓟北" + "幽燕" + "雁门" 等边塞诗多写此道沿途。

唐人尚不知道——**十一月初九**这条官道将由"奏报通衢"翻转为"叛军南下进军路"；**自此 8 年间（755 末-763 春）**——长安-范阳道沿途州县反复易主、河北诸州几无完土；**至德二载（757）正月安庆绪杀父安禄山于洛阳**；**乾元二年（759）史思明再杀安庆绪自立**；**上元二年（761）史朝义杀父史思明**；**宝应二年（763）正月史朝义自缢于温泉栅** —— 8 年内此道前后经历四代叛军易主、唐廷之收复亦经河南、关中、河北、河东数线反复。

**自此中晚唐河北的命运被此道改写**——**广德元年（763）唐廷以"姑息" 政策授降将以河北诸节度**：李宝臣（成德、治恒州）、田承嗣（魏博、治魏州）、李怀仙（卢龙 / 范阳、治幽州）——**史称"河北三镇"**；**自此至唐末百余年**——河北三镇名属唐廷、实为割据；**长安-范阳道自此再不畅通**。

今游人重走长安-北京一线——所见的是**京广 + 京哈 + 大秦铁路 + 京港澳 + 京昆高速 + G4 + G5**的现代驿道——**盛唐人在 755 所走的长安-范阳道**：千二百多年后**地理走廊大体在同一通道**、但驿馆 + 关塞 + 沿途景观几无原物存留（雁门关明清重修主体；幽州蓟城地表已不可见——压在今北京城下）。

## 子条目

### 安禄山兼三镇（天宝十年/751 起）+ 长安-范阳道为其脐带

**安禄山之三镇**：
- **天宝元年（742）正月**置平卢节度、安禄山兼任；
- **天宝三载（744）三月**安禄山兼范阳节度（旧称幽州长史）；
- **天宝十年（751）二月**安禄山再兼河东节度——**一身三镇、辖境自营州西至太原**。

**长安-范阳道**——是这套三镇体制的物理基础。**安禄山往来于长安 + 范阳间的频次**：天宝初每岁一二次、天宝十年后渐疏；**天宝十三载（754）正月最后一次入长安朝觐**——献奚、契丹俘三千；玄宗大悦、加其官、赐物。**自此他再不入朝**——病辞、潜谋、积粮。**长安-范阳道从此被一种独特的张力贯穿**：奏报、互市、戍兵换防一切如常、但**节度使本人不再过此道入京**——这是 755 一年间长安朝廷未读懂的最大信号。

### 雁门、句注山一线（河东北端边镇）

**雁门关 + 句注山（恒山主脊一段）**——是河东道北界、河北道与突厥旧地（默啜亡后散为回纥东侧诸部 + 奚契丹）的分界线。**唐雁门关**：盛唐时为河东节度北防核心、屯戍兵；**幽燕、蓟北、雁门、代北一线**是盛唐边塞诗的另一重心（与河西凉州/玉门一系并立）——盛唐"雁门-塞下" 是此道北段的诗境物理 anchor（河西诗不入幽燕、不与此道沿途相涉）。

**雁门以北至云州、朔州**——是河东节度北界；**安禄山兼河东后**——此段名义归其辖、但实际驻军主力在范阳、太原仍为河东节度副使节制。**755 起兵时安禄山未取北线**——即是因为河东主力仍在朝廷控制下、不可仰仗。

### 安禄山起兵路线（755 十一月-756 正月）

**755 十一月初九自范阳起兵**——号"诛杨国忠、清君侧"；**起兵后南下路线**（不沿"长安-范阳道"正向、而沿其南线反向）：

- **十一月初九（癸亥, 公历 755 年 12 月 16 日）**：范阳起兵；号兵 15 万（含蕃将精骑 + 河北汉兵）；
- **十一月中下旬**：自范阳南下、经**恒州/常山郡** + **赵州/赵郡** + **邢州/钜鹿郡**——河北诸州皆望风开城；
- **十二月初**：陷**相州/邺郡** + **卫州/汲郡**；
- **天宝十四载十二月丁酉**：**渡黄河、陷东京洛阳**——封常清守洛阳兵败；
- **次日**：洛阳百姓奔走、河南尹达奚珣降；
- **756 正月初一**：**安禄山在洛阳称帝、国号大燕**——是叛军政治符号的转折。

**这段路线全程未入长安-范阳道正向**——但**沿途河北诸州未经一战**——证明此道沿途州县在 755 前安禄山势力已渗透深。**自此"长安-范阳道" 的政治意义彻底翻面**——曾为唐廷"东北脐带"、此刻成叛军"南下进军通道"。

### 河北道州县（南线诸州；755 时尚为唐廷直辖）

**755 时河北道沿途州县**（自南向北、长安-范阳道南线）：
- **怀州 + 卫州**（黄河北岸、河北道南端）；
- **相州**（治邺城、今河南安阳）——曹魏邺城故地；
- **邢州 + 洺州**（太行东麓）；
- **赵州**（治平棘、今赵县）+ **恒州**（治真定、今正定）——河北腹地；
- **定州 + 易州 + 瀛州 + 莫州**（河北中北段）；
- **幽州**（范阳郡治、今北京一带）——河北道北端。

**这些州 755 时皆为唐廷直辖**——名义上分属各郡太守（天宝元年/742 改州为郡）；范阳节度直接辖境为北部诸州（幽、蓟、妫、檀、莫等），相、卫、邢、赵等河北南部州郡并不在范阳节度统一管辖之下、但已被安禄山势力深度渗透。**叛军南下时这些州几无抵抗——证明节度声望 + 州县同情已使唐廷对河北的实际控制名存实亡**。**这是长安-范阳道在 755 时最隐蔽的地缘真相**。

### 安史中长安-范阳道的反复（755 末-763 春）

**8 年间长安-范阳道经历的政治易主**：
- **755 十一月-756 春**：叛军南下取洛阳 + 关中；范阳成叛军大本营、河北诸州皆叛；
- **756-757**：**颜真卿守平原（河北道、唐廷地方抵抗的物理核心）+ 颜杲卿守常山（恒州）**——河北诸州曾短暂群起从唐；但叛军再下、常山陷、颜杲卿被害；
- **至德二载（757）正月**：**安庆绪杀父安禄山于洛阳**——叛军内乱第一回；
- **至德二载（757）十月**：唐军郭子仪 + 李光弼 + 回纥兵复两京——但河北诸州仍叛；
- **乾元二年（759）**：**史思明杀安庆绪自立、号大燕皇帝**；
- **上元二年（761）**：**史朝义杀父史思明自立**；
- **宝应二年（763）正月**：**史朝义自缢于温泉栅**——安史之乱终。

**8 年间长安-范阳道的"南端" + "北端"** 不同步：南端（关中段）在 757 末复唐廷、北端（河北段）至 763 春才名义复唐廷。**这是中国 8 世纪最大的"道路 + 政治"分裂**。

### 河北三镇之始（763 + 之后百余年的"名属唐廷、实为割据"）

**广德元年（763）**——史朝义自缢、安史之乱终。**唐代宗采"姑息"政策**——授降将以河北诸节度：
- **李宝臣**（原名张忠志、史朝义降将）——授**成德节度**、治**恒州**（领恒、定、易、深、赵、冀六州）；
- **田承嗣**（史朝义降将）——授**魏博节度**、治**魏州**（领魏、博、贝、卫、相、磁、洺七州）；
- **李怀仙**（史朝义降将）——授**卢龙 / 范阳节度**、治**幽州**（领幽、营、平、蓟、檀、妫等州）。

**史称"河北三镇"**——是中晚唐藩镇割据最持久的一块。**自 763 至唐末（907）144 年**——三镇多数时期名属唐廷、实为割据；节度使父子相承、多不奉朝命、不输赋税。**长安-范阳道自此再不畅通**——奏报、漕调、戍兵全部停滞；**安史之乱的真正后遗症不是 755-763 这 8 年的战乱、而是 763-907 这 144 年的河北分裂**——而长安-范阳道是这分裂在物理上的最直接体现。

## 历史长河（755 → 今）

**中晚唐**　**763-907 河北三镇割据**——长安-范阳道百余年不畅；**贞元元和间**唐廷数次北伐——**元和七年（812）田弘正归朝、魏博一度内附唐廷 (后又反复)**——道短暂复畅；**长庆元年（821）河北复反**——道再分。**唐末黄巢之乱（880）+ 朱温篡唐（907）**——河北、河东诸节度各自独立化；长安-范阳道作为"长安朝廷与东北边境的脐带"的功能彻底结束。

**五代**　**梁、唐、晋、汉、周五代**——河东（沙陀李氏）与河北（卢龙、镇州、成德诸节度）此起彼伏；后晋石敬瑭割幽云十六州予契丹（936）——**幽州自此入契丹辽**；**长安-范阳道的北端（幽州 / 范阳）落异姓 400 余年**（至明洪武元年/1368 徐达克大都恢复汉地）。

**两宋**　**北宋未复幽云**——长安-范阳道的北段始终在辽 / 金辖境；北宋时长安 / 西京 + 大名府 + 河间府 + 雄州 一线是宋辽边境驿路、与盛唐长安-范阳道走廊有重合但已非同一政治内的驿道。**南宋**则连河北、河东亦不及——驿路完全断裂。

**金元明清**　**金元定都北京（大都）后**——河北纵贯线重新成主干线、但起点已是大都而非长安；元代驿政重建、自大都至西安（奉元路）的驿路是盛唐长安-范阳道的"反向版"。**明清两都皆在北京**——长安 / 西安已退为西陲省会、长安-范阳道作为单一驿路的概念消失。**明长城自雁门、紫荆、居庸一线**——是盛唐河东、河北诸边镇的"物理升级版"。

**民国至今**　**1949 年后**——**京广铁路**（北京-广州）+ **京昆高速 G5**（北京-昆明、过西安）+ **大秦铁路**（大同-秦皇岛）等——形成今日西安-北京间的多条现代陆路通道、走廊与盛唐长安-范阳道大体重合。**盛唐人在 755 所走的那条范阳道**——千二百多年后地理走廊大体未变、但**驿馆 + 关塞 + 沿途景观几无原物**：雁门关是明清重修主体；幽州蓟城地表已被今北京城叠压；范阳节度治幽州的物理位置在今北京西城、宣武一带、地表已无可见唐物。**唯沿线考古遗存**（如河北正定开元寺、定州开元寺塔基、安阳邺城遗址等）零星保留——盛唐这条"东北脐带"的物理终点、与潼关、洛阳一样——基本是后世叠压 + 拆建的结果。
