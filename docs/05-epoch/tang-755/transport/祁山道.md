# 祁山道

## 一句话

蜀道八条里最绕陇右的一条——秦州至梁州约八百里，西汉水、嘉陵江相牵，唐人提起祁山多半为商队与边军；那段千古最熟的"六出祁山"，已是五百年前一位丞相在这条道上往返的旧事。

## 形貌

祁山道不走秦岭正脊。它从渭水谷地南折，绕陇山西麓，再沿西汉水、嘉陵江一线南下，到梁州（汉中）合入金牛道——是蜀道八条里最西、也是绕得最远的一条。

道全程约八百里。北口在**秦州**（治上邽，今天水秦州区），南口在**梁州**（治南郑，今汉中）。自秦州西南出，先经盐官（今礼县盐官镇）下祁山堡，再沿西汉水南折，过西和、成县（唐属成州同谷）；至略阳（唐兴州顺政县）折入嘉陵江，沿江一路南行，过青泥岭、阳平关，便接梁州城北郊。沿途州县共四：秦、成、兴、梁——一道穿陇右、山南西二道，是大唐西南军政两道交界处的一条骨干道。

道分三段。北段秦州至祁山堡约一百二十里，行陇山西麓的丘塬之间，路宽马可并辔，秋日里塬上麦黄、谷底水清。中段祁山堡至略阳约三百里，沿西汉水切入山区——两岸是泥页岩与红砂岩的低山，水势曲折，路常贴崖凿出，称"栈"。最险一段在**青泥岭**——李白《蜀道难》"青泥何盘盘，百步九折萦岩峦"，便是这一段；唐人车马至此须停宿一夜，次日缘崖徒步。南段略阳至梁州约三百八十里，循嘉陵江西岸南下，过阳平关入汉中盆地——出山入川，路便宽了。

比起褒斜道直插秦岭、金牛道走剑门，祁山道有两层特别。一是**绕得远**——多走两三百里，但全程沿水，坡缓、栈少、可行辎重车队。二是**走陇右**——绕开关中正面，从北侧迂回插入汉中，对蜀人是"北上之远路"，对北人是"南下之侧门"。唐时此道商队、屯戍、驿传并行，不甚拥挤，亦不甚清静。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

祁山道在唐人脑中，是**诸葛丞相的旧路**。

蜀汉建兴六年（228）春，丞相诸葛亮第一次北伐——大军不出褒斜、不走故道，反而向西出阳平关、沿嘉陵江北上，循这条最远的路插入陇右，**兵围祁山**。陇右南安、天水、安定三郡望风而应，关中震动。这是蜀汉立国以来最接近成功的一次北伐——若非街亭一败、马谡失守，陇右一时已半归汉。后建兴九年（231）第四次出兵，亮自祁山下出，与魏将司马懿对垒于卤城——粮尽方退，途中射杀魏名将张郃。建兴十二年（234）春第五次出兵，仍出汉中——这一次走的是斜谷而非祁山，但中军帐里挂的还是祁山地图。同年八月，**亮卒于五丈原**。后世传"六出祁山"，实则连斜谷一次共五出汉中，正面祁山仅二次——但"六出祁山"四字传到唐人耳里，已是道与人合一的代称。

魏晋以降，这条道由军用而兼商用。北魏、西魏置仇池镇于西和一带，对氐羌部族屯戍；隋开皇间复整驿路，西汉水沿岸置驿。**唐承隋制**——贞观元年（627）分天下为十道，秦州属陇右道，兴州属山南西道；祁山道恰跨二道交界。开元天宝间，朝廷主驿路重在褒斜与金牛，祁山道降为**辅路**——但西陲屯戍、陇蜀两道之间军需调拨、商队转输，仍多走此线。

### 此刻

天宝十四载这一年，祁山道是**陇右-山南间一条次干道**。

主干仍是金牛道——长安经斜谷或子午入汉中，再经金牛、剑门入成都。祁山道走的多是三类人：一是陇右节度（治鄯州）调往剑南的军马、戍卒，自天水南下走这条远而稳的路；二是关中以西的茶马商队，自岐州、凤翔西出，过陇山入秦州，再南折祁山入蜀；三是陇右一带的少数民族——氐、羌、吐谷浑零部——南来交易盐、马、药材，集于祁山堡、同谷、阳平关诸节点。

道上设驿，每三十里一驿。秦州城南有**祁山驿**，礼县有**盐官驿**（盐井所出，唐为陇右盐场之一），成县有**同谷驿**——杜甫四年后（乾元二年，759）自秦州南奔同谷，所走的正是这一段路。略阳之**顺政驿**、阳平关之**白水驿**——皆是祁山道南段的著名节点。

对天宝十四载的西陲人来说，祁山道意味着**一条相对安稳的退路**。秦州、凉州一线对吐蕃前线吃紧——哥舒翰兼掌河西、陇右两镇，十三万兵驻在临洮、武威。若有不测，陇右的辎重、家眷便循祁山道南下，过嘉陵入汉中，再翻金牛入蜀地——这条路虽绕，但走熟了。十一月安禄山起兵的消息传到长安已是数日之后，传到秦州又得再多十数日。秦州、成州一线的驿马传文，多走的就是这条祁山道——南向汉中、转入剑南，把消息一路送进成都。**再过半年**，玄宗本人也将取金牛道入蜀；那时陇右兵已陆续抽调东归救京，西陲为之一空——这是后话。

## 沿途子条目

下列各节点皆作 doc 内 `###` 锚定，不另立 POI。

### 秦州（上邽）

祁山道北口。陇右道州城，**秦州都督府**治所，下辖上邽、伏羌、清水、长道、成纪、陇城六县。城在渭水南岸、藉水北岸——两河合抱出一块川地，自周秦以来为陇右第一城。秦人在此立**西县**，西汉武帝置**天水郡**——"天水"之名由此而出。三国时为魏陇右屯戍重镇，诸葛亮北伐每兵临此城而不下。唐天宝中户口约二万，城周九里有奇，东、南、西三门外各有驿——南门外即祁山驿，是道之起点。乾元二年（759）秋，杜甫自华州弃官西来秦州，作《秦州杂诗二十首》"满目悲生事，因人作远游"——盛唐西陲第一组流寓诗即出于此城。

### 盐官驿

秦州西南约百二十里，今礼县盐官镇。**陇右盐场之一**——井盐自地下涌出，自先秦以来即为陇右一大物产。唐设盐官（盐铁使下小吏），置驿守之。驿是道之第一站，南行入山前最后一处平川驿站；商队多在此整队备粮，下一段便要入西汉水谷。

### 祁山堡

道之最著名节点。在西汉水北岸——今礼县城东南约二十五里，一处孤立的红土小丘，三面环水，一面临塬。**蜀汉建兴六年（228）诸葛亮第一次北伐，屯兵于此**——堡是天然形胜，丞相据此望陇右三郡。建兴九年（231）第四次北伐再屯祁山，仍以此堡为大本营。"六出祁山"的"祁山"，便是这一处不到一里见方的土山——非山岭，非堡垒，只是一处足以驻数千人、控西汉水谷的小台地。后人讹传为大山——其实自堡上极目所见，西汉水如带、礼县川如棋盘，足矣。Tang 时堡上仍有夯土墙基，相传亮之中军帐故址在堡顶——唐人路过多登一望，士人有题诗者，惜不传。今礼县祁山堡尚存，明清间立**武侯祠**于堡上——是后话。

### 礼县（兰仓 / 长道县）

唐属秦州长道县。地当祁山堡所在川道之中——西汉水自西北来，到此地折南。城不大，约二里见方，是道上中段补给点。礼县川自古产马——秦人发迹于此，《史记·秦本纪》"非子居犬丘，好马及畜，善养息之，犬丘人言之周孝王，孝王召使主马于汧渭之间"——**秦之先祖养马处**便在这一带。唐时仍是陇右马场之一，朝廷岁取战马于此。

### 西和（仇池故地）

唐属成州。城在西汉水谷之南、青泥岭之北——是道入山前最后一处大邑。城东北约三十里有**仇池山**——魏晋间氐人杨氏据此立国，自三国魏黄初元年（220）杨千万、杨飞龙父子据仇池称王，至北魏太和十八年（494）灭，前后凡二百七十余年，史称"前仇池"、"后仇池"。Tang 时仇池古城尚有遗墙，山中氐人后裔零星散居——唐人过此，多想见"仇池雄丽天下无"的旧事。杜甫《秦州杂诗》其十四"万古仇池穴，潜通小有天"——便是此山。

### 同谷（成州治）

唐成州治所，今甘肃成县。城在西汉水中游谷地中，**祁山道中点**——南北各四百里左右。城不大，但是商旅必宿之地。乾元二年（759）冬，杜甫自秦州南奔同谷——所写《乾元中寓居同谷县作歌七首》"有客有客字子美，白头乱发垂过耳"——是杜诗中最寒苦的一组。盛唐人多过此城而不留笔——杜甫是例外。

### 青泥岭

祁山道**最险一段**。在唐兴州长举县境（今略阳北、徽县南交界），西汉水切山而过，路凿崖而立——"百步九折萦岩峦"。唐人车马至此须先在岭北驿宿一夜，次日清晨缘崖徒步、马驮辎重在后，过岭凡三十余里方得喘息。岭名"青泥"——古传雨后崖泥发青，长年不干；亦说山色青、土色泥。李白《蜀道难》"青泥何盘盘"——所写正在此处。

### 略阳（兴州 / 顺政县）

唐兴州治所——山南西道北端州城。城在嘉陵江上游与西汉水交汇之处——**祁山道南段起点**。自此南下转入嘉陵江干流，路稍平。城周不过四五里，但是商旅整队入川前的大邑。城南有**顺政驿**——是道入嘉陵江后第一驿。

### 阳平关（白水关 / 白水驿）

道之南口前最后一关。在嘉陵江与白水（今白水江）交汇处——今陕西宁强阳平关镇。三国时为蜀汉北门，**关上有蜀汉旧戍**——诸葛亮、姜维数次北伐皆由此出。Tang 时关已不为军镇，但仍是驿路咽喉——置白水驿，过关南行三十里便入汉中盆地。

### 梁州（南郑 / 汉中）

道之南口。山南西道州城，治南郑县——汉中盆地北缘的大城。**祁山道、褒斜道、傥骆道、子午道、米仓道——五条蜀道在此合**。城周约十里，户口万余，是关中-蜀地之间最大的中转站。梁州本与金牛道相接——南出汉中过宁强、广元、剑门，便入剑南益州。唐人过梁州多不久留——但凡过梁州，便意味着或刚出蜀北上、或将入蜀南下。至德二载（757）肃宗自灵武南下，曾遣使经此抚剑南；上元元年（760）改梁州为**兴元府**——是后话。

## 历史长河（755 → 今）

**中晚唐**　至德元载至二载（756-757），玄宗、肃宗父子分循子午、金牛道入蜀、北上灵武——祁山道未直接卷入两京收复战，但陇右兵东抽、吐蕃东扩，秦州、成州一线渐成前线。代宗广德元年（763）吐蕃陷长安、又陷秦州——祁山道北段自此沦为蕃汉拉锯地，**道几断**。德宗、宪宗朝几次收复秦州又得而复失——晚唐时祁山道实际通行的，只剩同谷以南至梁州一段。

**五代至宋**　五代后唐、后周间，秦州数易主。北宋初太祖乾德元年（963）平后蜀，置秦凤路——秦州复入版图。**北宋熙宁五年（1072）置熙河路**，王韶开熙河——秦州再为西陲军政中心。祁山道全线复通，南宋时为吴玠、吴璘兄弟据守"和尚原 / 仙人关"抗金的后路——绍兴元年至四年（1131-1134）间，吴氏屡破金兵于仙人关、杀金坪——所恃后勤补给，正是这条祁山道由汉中北送至同谷、礼县。

**明清**　明洪武间复整驿路，置秦州卫、巩昌府——祁山道为陇右-汉中之间的官路。清乾隆四十一年（1776）陕西巡抚毕沅遍立古迹碑——礼县祁山堡上立"汉丞相诸葛武侯祁山故垒"碑；同代在堡上重修武侯祠——殿宇至今尚存。明清以后祁山道渐次让位于陇南新开的商路，但作为**陇蜀间最古的一条线**，老百姓仍走、商队仍过。

**民国至今**　民国二十五年（1936）川陕公路（今 G212）大致沿祁山道南段（略阳-汉中）修筑；中段（同谷-略阳）则因青泥岭险阻，新路改走西汉水东岸——古道渐渐被山林吞没。1958 年宝成铁路通车——铁路自宝鸡循嘉陵江南下，过略阳、广元入成都，是大体沿祁山道南段的走向。1990 年代以来甘肃文物部门数次调查祁山堡、仇池故城、青泥岭古道——堡上残墙、岭上石阶尚有零星可辨。

至于"六出祁山"那段——堡仍在、山仍在、武侯祠仍立；丞相帐前所望的西汉水川道，到了今日，还是那一片麦黄塬绿。诸葛亮卒于建兴十二年（234）——**距今已逾一千八百年**。

祁山道是八条里最绕、最远、最西的一条；走的不是关中正面，而是陇右侧门。

### "六出祁山"考辨

诸葛亮一生北伐凡五次（不是六次），其中**正面出祁山道**者只有第一次（228 春）和第四次（231 春）二次。其余三次：

- 建兴六年冬（228 末）出散关围陈仓（陈仓道）
- 建兴七年（229）取武都、阴平（祁山道西支线）
- 建兴十二年（234）出斜谷至五丈原——亮卒于此

"六出祁山"四字源自元明间罗贯中《三国演义》的小说化合并——把所有北伐都"祁山化"了。唐人尚不知"六出祁山"这个词组——他们读的是陈寿《三国志》：丞相北伐凡五，其中两次出祁山。但**到 755 年时，民间口耳相传的诸葛亮已经是一位"出祁山"的丞相**——这一道与这一人的绑定，盛唐时已经成形。
