# 散关

## 一句话

**关中西南门、陈仓道北口**——凤州河池县境西北的山间关塞；**756 七月玄宗西奔入蜀路过此关 + 入剑南**；**755 时是长安西南向蜀地最直接的物理 anchor**——是关中四关（东潼、南武、西散、北萧）之西关。

## 形貌

散关在唐山南西道**凤州河池郡境西北**——具体在**陈仓道北口**（今陕西省宝鸡市渭滨区南界 + 凤县境一带，海拔约 1000 米）。**地势**：**秦岭主脊北麓与渭水上游南岸夹峙**——是关中盆地西南端、向蜀地（汉中盆地）出秦岭的天然孔道。

**散关之险**：**北面渭水**（向东流入关中）、**南面秦岭主脊**（峻峭难越）；**关城就在秦岭山口**——南依秦岭山势、北临陈仓道（即关中-汉中陈仓古道）；**一关守住关中-汉中两片大地的西南通道**。

**陈仓道**：**关中入汉中的"四道"之一**（陈仓道、褒斜道、傥骆道、子午道）；**陈仓道（故道）是其中最古、最长、最低的一条**：自关中陈仓县（今宝鸡） → 渡渭水 → 入陈仓道（沿渭水支流嘉陵江源头西汉水） → 越秦岭 → 至凤州 + 兴州 + 汉中（梁州）。**散关在陈仓道北口**——是这条道的"关中端入口"；扼住散关 = 扼住陈仓道。

**关城形制**：盛唐散关具体形制 + 周长 + 门数史载零散——本 doc 不预设细节（今所见关楼 + 关墙多为近代重建）。**关楼立秦岭北坡**——可俯瞰渭水 + 关中平原一片；**陈仓道**沿秦岭北麓东西延伸 + 沿嘉陵江源头南行入汉中。

**散关与潼关、武关、萧关并立**——**"关中四关"是后世地理归纳**（潼关东、武关南、散关西、萧关北）——盛唐 755 时唐人是否以此固定四关称呼史源略异；本 doc 取通称作地理 anchor、不预设唐人固定认知。

物候：秦岭北麓山地、温带季风气候——夏温冬寒；山中林木茂盛、雪较多。**关城内外多有戍兵 + 驿马 + 商旅** —— 是盛唐西南方向的主要关塞。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

散关在唐人的回望里——是**关中西南门 + 陈仓道北口 + 关中四关之西关**。

**散关的设立**：**秦代已立**——《史记》记汉文帝幸代过散关，可见秦汉间散关已是关中西南要塞。**东汉建武三年（27）**——光武帝刘秀派吴汉破公孙述（蜀汉割据）、即由陈仓道过散关入蜀；**自此散关在中国军事史上反复出现**——三国蜀汉北伐 + 北朝梁陈对峙 + 隋唐统一 + 南宋抗金 等。

**唐散关的政治军事地位**：**唐承前代之制、置散关为军事关塞**；**散关守军由唐廷直接指挥**——**散关属凤州 + 与梁州（汉中郡）军政体系并属山南西道**。**自唐初至盛唐 138 年（618-755）散关无大战事**——是太平时代；但**散关常驻兵约数百**——是日常守关 + 验商旅 + 收关税。

**散关与长安、汉中的关系**：**长安西行 + 经咸阳 + 扶风 + 陈仓 + 散关 + 河池县**；**散关南出 + 经兴州 + 利州 + 剑州 + 绵州 + 汉州 + 至益州**（具体里程史源略异）——**这是盛唐长安入蜀的最直接、最稳定的驿路**——也是 756 玄宗西奔入蜀的实际路线。

**安史前夕**（即 755 这一刻）——**长安 + 河西 + 陇右兵东调救京、过散关者罕**（多直接东行至潼关）；**散关本身仍是平时关塞**。**散关在 755 前夕的状态**——驻兵约数百、关楼严整、商旅通行不阻、戍兵按日轮值。

### 此刻

天宝十四载这一年的散关——**关塞严整、驻兵数百、扼陈仓道北口**。

**关城状态**：盛唐 755 时散关主体仍立、有戍兵 + 关吏 + 驿馆；具体形制史载零散。**关城外**：陈仓道南北延伸；**陈仓道北口的散关 + 南口的剑门关**——是关中入蜀的两大关塞咽喉。

**755 时散关上的政治氛围**：**散关是平时关塞**——关税收 + 商旅过关 + 戍兵巡逻；**关吏对入蜀的官员、僧侣、商旅** 多有验牒；**南向汉中 + 蜀地的人流密度** 远不如东向洛阳的潼关、但作为入蜀第一关、地位重要。

**755 末 + 756 初的散关命运**：**756 七月玄宗西奔入蜀**——**自长安出延秋门 + 西行经咸阳 + 马嵬 + 扶风 + 陈仓**——**入散关南下**——**经凤州河池 + 兴州（略阳）+ 利州（广元）+ 剑州（剑阁，过剑门关）+ 绵州 + 汉州 + 至益州（成都）**。**这一行约月余**——是中国历史上最有名的"皇帝逃难入蜀" 之路。**散关是这条路上的关中末端关口**——玄宗一行过散关 + 入凤州后、便已离开关中、进入汉中盆地的山道。

**756 七月玄宗过散关的具体经历**：**马嵬驿杨贵妃死后**、玄宗继续西行；**扶风郡（岐州）一带短暂停留**；**过陈仓县 + 入散关 + 至河池**；**自此玄宗朝廷在散关南脉中迅速南下**——这是盛唐天子最狼狈的一段路。

### 此刻意味

对天宝十四载的凤州 / 河池县人来说——散关是**自家境内的关塞**：日常入蜀商旅必过、戍兵守关、关吏验牒。对长安那一代士人来说——散关是**关中西南门**：通往蜀地的物理 anchor；**盛唐诗中"散关"二字偶有出现**（高适、岑参等边塞诗或涉），但**散关的诗咏密集化在 755 后**（玄宗西奔过散关成中唐怀古地）。

唐人尚不知道——再过半年、散关将经历**唐玄宗西奔入蜀**：**关塞从平时驿路转为皇帝逃难必经**；再过 410 年（南宋绍兴年间），**陆游守散关**——**作《书愤》《剑南诗稿》"楼船夜雪瓜洲渡、铁马秋风大散关"**——把散关写进中国文学最有名的"南宋抗金物理 anchor"。**今散关古道遗址**：在宝鸡市渭滨区南界 + 凤县境内——**部分关墙犹存 + 关楼为近代重建**。**唐人在 755 所见的那座散关**——千二百多年后**主体已残**：盛唐散关的物理终点是中国关塞史的常态。

## 子条目

### 陈仓道（散关南通汉中的古道）

**陈仓道**（一名"故道"）——**关中入汉中四道之一**：陈仓道、褒斜道、傥骆道、子午道。**陈仓道是其中最古、最长、最低的一条**：自关中陈仓县（今宝鸡） → 渡渭水 → 入陈仓道（沿嘉陵江源头西汉水） → 越秦岭 → 至凤州 + 兴州 + 汉中。**陈仓道全程约 1200 唐里**——比褒斜道（600 唐里）+ 傥骆道（650 唐里）+ 子午道（700 唐里）都长——但**陈仓道海拔最低、相对最易行**——是大军 + 帝王仪仗 + 商旅的首选。

**散关在陈仓道北口**——是这条道的"关中端入口"。**756 玄宗西奔即取陈仓道**——这是盛唐天子在散关 + 陈仓道的最有名一次过往。

### 玄宗西奔过散关（756 七月）

**至德元载（756）七月**——**唐玄宗自长安出延秋门西奔**：
- **七月初一**：玄宗出长安、西渡渭水；
- **七月初十前后**：至**马嵬驿**（兴平郡境）；禁军杀杨国忠 + 缢杨贵妃；
- **七月中旬**：玄宗经扶风郡（岐州）西行；
- **七月下旬**：**过陈仓 + 入散关 + 至凤州河池**——这是玄宗一行离开关中的关键一步；
- **八月**：经兴州 + 利州 + 剑州（过剑门关）；
- **至德元载八月底至成都**：玄宗朝廷暂驻益州。

**自此"散关 + 剑门关"成中国历史"天子流离地" 的双关物理 anchor**——盛唐人 755 时尚未知；半年后即由历史亲证。

### 关中四关（潼关 + 武关 + 散关 + 萧关）

**关中盆地的天然防御体系**——四关合围：
- **东关：潼关**——黄河南岸 + 秦岭北麓夹峙；扼东向洛阳之冲；
- **南关：武关**——秦岭南麓商山中（商州境）；扼南向荆襄之冲；
- **西关：散关**——秦岭主脊北麓凤州境；扼西南向汉中蜀地之冲；
- **北关：萧关**——六盘山一线（原州境）；扼北向河西陇右之冲。

**四关在中国地理上是最完整的"关中盆地防御体系"**——盛唐 755 时四关皆立、守军按例驻屯。**注**：四关之中、**潼关在 755 后失守 → 关中陷落**；**散关在 756 玄宗过 → 关中天子西奔**——是关中四关中两个"对盛唐有直接政治意义"的关塞。

### 散关税与商旅

**唐散关收商旅过关税**——这是关塞的常规财政职能；**散关税**主要针对入蜀 / 出蜀的商货 + 公私往来。**盛唐散关税收**史载零散；**与潼关、玉门关等比、散关税收规模较小**——因入蜀商旅密度不如东出洛阳。

### 陆游《书愤》"铁马秋风大散关"（南宋）

**陆游（1125-1210）**——南宋抗金诗人。**绍兴三十年代 + 乾道间**陆游曾守散关一带、参与南宋川陕防线；**1186 作《书愤》**"楼船夜雪瓜洲渡、铁马秋风大散关"——是中国文学史最有名的"散关诗"。**注**：陆游诗作于南宋（1186）、与唐 755 时差 430 年——本 doc 作历史长河段引、不入 755 时视角。**陆游让散关在文学中的位置骤升**——成"抗金物理 anchor + 中原沦陷的边境记忆"代名词。

## 历史长河（755 → 今）

**中晚唐**　**756 玄宗西奔过散关**是散关在盛唐唯一一次最有名的过往；**至德二载（757）末玄宗返长安**——亦由原路过散关回京。**中晚唐散关仍为关塞**——但因关中失而复得（吐蕃曾陷长安），散关地位较潼关高。

**唐末黄巢 + 五代**　**广明元年（880）黄巢入长安**——**唐僖宗西奔成都**、亦走散关一路；**这是盛唐"玄宗 → 僖宗" 两次天子过散关入蜀的连续**。**五代间散关多次易主**——为前蜀、后蜀、后晋等政权争夺。

**两宋**　**北宋初散关重修**；**南宋绍兴 - 乾道间**散关为南宋川陕防线核心——**陆游守散关**作《书愤》《剑南诗稿》——**自此散关在文学中的地位达高峰**。**南宋亡（1279）后散关失军事意义**。

**金元明清**　元明清间散关续作关塞、但规模渐降；**明代散关属凤翔府宝鸡县**；**清乾隆四十年（1775）陕西巡抚毕沅立"古散关"碑**——是清代"复古考"的一笔。

**民国至今**　**1949 年后散关古道遗址列陕西省重点文物保护单位**。**1990s-2010s 多次考古勘察 + 部分修缮**——确认散关古关墙、陈仓道路线、烽燧遗迹。**今宝鸡市渭滨区 + 凤县境**——所见的是**散关古道遗址 + 关楼为近代重建 + 部分关墙犹存**。**唐人在 755 所见的那座散关**——千二百多年后**主体已残**；玄宗 756 七月过散关入蜀的那条路——今已成宝成铁路 + 国道 G316 + 川陕公路的现代驿道。
