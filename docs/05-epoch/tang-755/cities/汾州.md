# 汾州

## 一句话

河东道中段一州，治隰城县（今山西汾阳）——汾水自北而南穿境而过，太原南下晋、绛、蒲的官驿道在此过站。

## 形貌

汾州城在汾水中段的西岸河谷里。东去汾水不过数里，再东是太岳山西麓低岭；西倚吕梁山东侧山脚——所谓"东汾西吕"，城正夹在两线之间一片狭长川谷。州治**隰城县**——城周不大，城外是汾水冲积的小盆地，黄土塬面延伸到吕梁脚下便止。城南行经晋州 / 绛州至蒲州约三百唐里，渡黄河入关中；城北去太原约二百唐里，沿汾水谷地一线官道——**汾州正是太原南下蒲州、关中通河东腹心的中段一站**。

物候是河东中部的：春旱多风沙、夏雨集中于七八月、秋早霜、冬干寒——年降水约五百毫米，可旱作粟麦黍豆。城外汾水沿岸有水浇地数顷，沟谷间稀疏河柳。冬季汾水有冻封段（约 12 月至次年 2 月）。**这一片是河东道腹心的农区——南北两端是太原、蒲州两大重镇，汾州夹在中间，做的是过路与积粮的活**。

## 大唐天宝十四载（755年）

唐人眼里汾州不算上古名邑，但沿革清楚——**北魏太和八年（484）始置西河郡**于此，治隰城；北周时移州治、改名屡有；**隋大业三年（607）行州改郡通制，复称西河郡**。**唐武德元年（618）改西河郡为浩州；武德三年（620）改浩州为汾州**——"汾州"二字自此立，因濒汾水而名。**天宝元年（742）**玄宗诏改州为郡，汾州又改回**西河郡**；**乾元元年（758）**肃宗复改郡为州——汾州之名再立。**742 至 758 复州前，汾州官称为"西河郡"，民间口语仍呼"汾州"**——两个名号在唐人嘴里互换不分。

755 年的汾州属**河东道**，州治在隰城县。据《旧唐书·地理志二》记天宝间汾州约 5.9 万户（《旧唐书》59450 户 / 320233 口）——上州门槛之上，辖县在玄宗朝凡五六，**隰城**（州治）、**孝义**、**介休**、**平遥**、**灵石**诸县——皆汾水中段两岸的县治。州中无节度治所，但**汾州都督府**曾置（唐初武德间），玄宗朝降为一般州都督府或省并——具体废置年代史载略有出入。城内有州衙、隰城县治、驿馆——长安出潼关、过蒲州渡河、北循汾水谷地的官驿道，自汾州过境北上太原。**汾州是河东道南北驿道的中转站，也是河东漕粮自太原南运蒲州的中段集散地**。

天宝十四载冬安禄山反范阳——消息沿驿道传到汾州时，城里粮草、驿马的调度也紧起来。汾州不是节度治——河东节度使坐镇太原，汾州是辖境南部的中州，本州无大军；但作太原南下的第一站，朝廷自此调发兵卒、粮饷北上，过路最忙。**755 这一年的汾州，意味着汾水中段的一处驿与仓**——名号还是西河郡，民间口里仍叫汾州。

## 历史长河

**中晚唐**　乾元元年（758）复州名为汾州（西河郡旧名废）。安史乱后河东道地位上升，汾州做太原与蒲州之间的转运枢纽。中晚唐汾州一线产酒——州境内**杏花村**所酿之酒，唐末已有名（具体起年史载有歧；杜牧《清明》"借问酒家何处有，牧童遥指杏花村"作于晚唐，是否专指此地，明清以来争说不一，不预支盛唐）。**乾元元年（758）后郭子仪因平安史功封"汾阳郡王"**——但此"汾阳"为食邑封号、采用汉魏旧称（汉西河郡有汾阳县，治今静乐县境），与盛唐汾州治隰城非直接同地；郭氏后世亦称"汾阳郭氏"——皆 762 以后之事，盛唐汾州人尚不知此。

**宋元明清**　北宋汾州属河东路，治隰城（宋改名西河县）。金元仍为汾州。**明万历二十三年（1595）升汾州为汾州府**，治**汾阳县**——"汾阳"作为本州县名自此始（取古汾阳县名重立于隰城旧治，与汉魏旧汾阳县已不同地）。清承明制，汾州府属山西布政司。明清汾州府以**汾酒**著——明清以来杏花村酒在山西商路上声名鹊起，"借问酒家何处有"的诗句被附会到此地，遂成晋酒第一品牌；但盛唐时汾酒是否已成规模，史载不详——这是后世坐实的名号。

**民国至今**　1913 年废府存县——汾州府正式退出建制，汾阳县为山西省辖。1996 年汾阳撤县设市（县级市），治所仍在唐隰城、明清汾州府旧址上。**今日山西汾阳市治所，仍在唐汾州治隰城县旧址上**——这是河东中段少见的连续一千五百年同一城址。盛唐那座汾州——州城残基已埋入今汾阳市下，隰城官署旧址不可考；汾水仍自城东流过，吕梁山仍在西。
