# 坊州

## 一句话

关内道北线南端那一座上州——治中部县（今陕西延安市黄陵县），桥山在县中，**黄帝陵**即在桥山之上；天宝元年至乾元元年（742-758）官称"中部郡"。

## 形貌

坊州城坐在**葫芦河（沮水）谷地**南口的台地上——州治中部县治取义于"黄帝陵在县中"。城周不大，夯土城墙，开四门；城北便是**桥山**——《史记·五帝本纪》谓"黄帝崩，葬桥山"，山上有桥陵（即黄帝陵），唐时已立祠，自州治北望可见山脊轮廓。州地是陕北黄土高原南缘——塬、梁、峁与沟谷交错，沟里水浇地、塬上旱作粟糜，沮水自西北而来南流入洛水。北接鄜州（治洛交，今陕西富县）、南接同官—京畿、东接丹州、西接宁州；自长安出春明门北行经三原、宜君，过坊州中部，再北便至鄜州——是关中通朔方那条南北纵道上鄜延一线的南门。物候：春旱多风，秋早霜，冬干寒。

## 大唐天宝十四载（755年）

坊州本是隋上郡之内部县——**唐武德二年（619）**高祖分鄜州中部、宜君二县置坊州，州名取自当时筑州城将军**杨思政所筑"坊城"**之号。**天宝元年（742）**玄宗诏天下州改郡，坊州改称**中部郡**；**乾元元年（758）**肃宗复郡为州，中部郡又复称坊州。755 年这一刻，城门匾额上写的还是"中部郡"。郡望属"上"——《旧唐书·地理志》坊州天宝领县四：**中部、宜君、升平、鄜城**。

坊州在 755 年的分量不在本州事务——而在**它所在的那条关中—朔方官道**与**桥山黄帝陵**。自长安北行入鄜延，坊州中部是南起第一站；自鄜州、延州南下入京畿，坊州中部又是南门最后一关。**黄帝陵自唐起即为天子郊祀所重**——盛唐人路过坊州，多有北望桥山、致敬轩辕之笔。这一年州本城照常上缴秋粮、点北线戍卒；他们尚不知道，再过一年安史乱起，鄜延一线将成为关中士庶北逃避地的中转走廊，坊州中部正在这条走廊的南口上。

## 历史长河

**乾元元年（758）**中部郡复称坊州——城匾换回旧名。中晚唐**鄜坊节度使**（大历三年/768 置）辖鄜、坊、丹、延四州，治鄜州——坊州为辖下一州。**北宋**仍置坊州，属永兴军路，后属鄜延路。**金元**坊州属鄜延路 / 延安路。**明洪武二年（1369）**省坊州入鄜州——"坊州"作为州名自此从地名上退去，前后历七百五十年。所辖中部县属鄜州；清沿明制。**民国二年（1913）废府州**——中部县存。**民国三十三年（1944）**以"黄帝陵在县中"之故，改中部县为**黄陵县**。今为陕西延安市黄陵县——县城仍在唐中部县故址、唐坊州州治那块葫芦河谷台地上；桥山黄帝陵自唐至今同址，今为国务院公布第一批全国重点文物保护单位"古墓葬第一号"。

## 引文

> 坊州，中部郡。上。隋上郡之内部县。武德二年，分鄜州之中部、宜君二县置坊州。天宝元年，改为中部郡。乾元元年，复为坊州。天宝领县四，户二万二千四百五十八，口一十二万二百八。

——《旧唐书·卷三十八·地理志一·关内道·坊州（中部郡）》（节录）
