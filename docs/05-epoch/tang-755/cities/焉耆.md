# 焉耆

## 一句话

安西四镇之东镇，龙姓焉耆王城——博斯腾湖西畔一处水网绿洲；玄奘西行称之"阿耆尼"，唐人识它有二：自西州过来的第一站、自龟兹回长安的第一站。

## 形貌

焉耆不在天山下，也不在昆仑下——它在**博斯腾湖西畔**。

塔里木盆地的东北一角，自天山南麓下来的开都河（古名敦薨水）汇入博斯腾湖——湖东西延约百里、南北约五十里，是西域最大的内陆淡水湖之一。湖西、湖北一带是开都河冲积出的水草绿洲：芦苇、水网、泥滩、稻田——一带绿洲在塔里木盆地干燥地貌之中，是难得一片有水汽的去处。焉耆王城便坐在这片湖西绿洲的核心，离湖岸约十数里。

王城规模在塔里木盆地诸城中算中等——不及龟兹之大，也不如于阗之古。**今学界主流将盛唐焉耆王城遗址定位于今焉耆回族自治县西南约十二公里的博格达沁古城**——一座近方形夯土城，城周约三里许，城墙残高数米；东、南、西、北四门遗迹皆可辨。城外另有四十里城、扩湖故城等若干汉唐聚落遗址散布湖西一带——盛唐焉耆是一座**以王城为核、以诸聚落为辅**的小型城邦。

城里街巷与龟兹相似——多依水渠走向，不作长安那种棋盘。王宫与都督府衙署居城中，城内有寺数所，城外湖畔有更大规模的伽蓝。焉耆奉**小乘**佛教——具体派别为"说一切有部"，与龟兹同；玄奘《大唐西域记》记其"伽蓝十余所，僧徒二千余人，习学小乘教说一切有部"——七世纪初焉耆的佛门面相由此可见。比邻的于阗主大乘、龟兹主小乘——**塔里木南北两道由此分作两大佛教谱系**，焉耆与龟兹同属北道小乘一脉。

物候比龟兹稍湿——博斯腾湖的水汽使焉耆夏夜不至于像戈壁那般极冷极旱；湖里产鱼，湖边有芦苇、有水鸟——在西域诸城中是难得一处不只靠雪山融水的绿洲。**焉耆马**自汉以来即名——湖西草场所养——汉武帝时焉耆贡马事见《汉书·西域传》；盛唐焉耆王仍以贡马为常项。城东去过湖南岸走十数日至西州（吐鲁番）；城西去过铁门关八九日至龟兹治所；城南去出**铁门关**（焉耆南山口，今库尔勒铁门关一带，盛唐设守捉）便接库车道；城北越天山南麓诸口可入北庭都护府——焉耆是**安西、北庭两大都护府之间一处咽喉**，亦是**安西东向通沙州的第一站**。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

焉耆是一座**汉、晋、隋、唐数代都未曾真正陌生的西域旧国**——龙氏王统据此地约六百年，是塔里木盆地里王统延续之久仅次于于阗尉迟氏、龟兹白氏的一支。

《汉书·西域传》记西汉时已有"焉耆国"——王治员渠城。汉宣帝时郑吉为西域都护，治乌垒城（在龟兹与焉耆之间，《汉书》记"去焉耆四百里"）——汉人对焉耆的实际治理自此开始。东汉班超经营西域时，焉耆曾起叛——永元六年（94）班超征焉耆，**斩王广**——这是焉耆第一次因叛事见诛于汉。魏晋以下，西域分合无常，焉耆王统亦数易——但"焉耆"二字与王城本身始终落在博斯腾湖西畔。**自魏晋起，王姓"龙"**（《魏书·西域传》《旧唐书·西戎传》均明载）——龙氏延绵至唐，是焉耆王统的主线（不与龟兹白氏相混——白氏是龟兹王姓）。

玄奘西行时过这里——是焉耆在盛唐人眼里的来路上最近的一次记录。**贞观元年（627）秋玄奘自长安出发**——西行年代史料异文较多，《资治通鉴》卷一九五系于"贞观元年（627）八月"，玄奘自己在《大唐西域记》序中作"贞观三年"，《大慈恩寺三藏法师传·卷一》作"贞观三年八月"——本 doc 与瓜州 / 伊州 / 沙州 / 碎叶诸 doc 一致，取**贞观元年（627）说**。一路过秦州、兰州、凉州、瓜州、玉门关、伊吾、高昌——在高昌王麴文泰处停了月余，由麴文泰备齐西行装具——**自高昌出发后遣使送至阿耆尼国——即焉耆**。玄奘在阿耆尼停留**一夜或数日**（《西域记》记"停一宿"，《法师传》记"停数日"——史料异文，本 doc 并列），由焉耆王（《西域记》未记其名，《法师传》记其"出迎慰劳"）礼接。焉耆于玄奘西行而言只是路途之一站——但《大唐西域记·卷一·阿耆尼国》是十二卷西域记的**第一卷第一国**——开篇之笔便落在焉耆。"阿耆尼"梵语 Agni，本义"火"——一名既翻，便作焉耆别号入了汉籍。

唐对焉耆的直接经营自贞观间始。贞观三年焉耆王龙突骑支遣使请通汉道——焉耆道（自焉耆经楼兰、鄯善至沙州的一条西域南道东端）一时复通。**贞观十八年（644）**，焉耆王龙突骑支与西突厥结亲、断汉道——唐廷震怒，**遣安西都护郭孝恪率兵征焉耆**，**八月破焉耆**——擒龙突骑支送长安，立其弟龙栗婆准为王。这是继 640 高昌之后唐征西域第二次攻灭王城——比侯君集 640 灭高昌晚四年。郭孝恪入焉耆城三日后引兵东归——西突厥乘虚再入，焉耆复反；龙栗婆准被杀。这段反复一直延续到**显庆三年（658）**唐平西突厥贺鲁——焉耆事乃定。

**龙朔元年（661）**，唐于焉耆地置**焉耆都督府**——羁縻——由焉耆王世袭都督。这是焉耆作为唐**羁縻属州**的正式建置。同年，唐于龟兹、焉耆、于阗、疏勒**安西四镇**之名乃定（具体顺序史料异文，**显庆三年/658 与龙朔元年/661 之间四镇陆续建立**——通鉴卷二〇〇龙朔元年条记之）——焉耆是其中**东镇**。中间几度反复：咸亨元年（670）四镇陷于吐蕃；长寿二年（692）王孝杰收复——但其后武则天朝至开元初约二十余年间，**碎叶城**（在今吉尔吉斯斯坦楚河流域 Ak-Beshim 遗址，李白出生地一说）一度取代焉耆居四镇之名——四镇为龟兹、于阗、疏勒、碎叶。

**开元七年（719）**——这是焉耆史上的一个关键年份。是年，唐以碎叶远悬碎叶川（楚河）流域、且突骑施（西突厥的一支）压力日增，**乃舍碎叶、复以焉耆为四镇之一**——四镇之东镇自此重归焉耆。自开元七年（719）至贞元末四镇陷蕃（约 790），**焉耆稳为安西四镇之东镇，前后凡七十余年**——755 年焉耆正在这一段最稳固的盛唐羁縻属国时期。

### 此刻

天宝十四载（755）的焉耆，名义上是**焉耆都督府 + 焉耆镇**——王是龙氏一支（**755 年当年焉耆王名史料未详**，故 doc 不强凿；本 doc 沿用"龙氏王统至盛唐仍续"通行说法）。城内日常仍由龙姓王自治，唐于其地驻**焉耆镇**——常驻镇兵数千（具体兵额史料异文：一说三千、一说五千，姑取约数）；镇守使由安西节度使任命，与王宫南北相望。这是一种典型的**羁縻 + 镇戍**复合结构——与于阗的尉迟氏、龟兹的白氏并立为安西四镇的三种**唐 + 本族王统**双重权力示例。

755 年时**安西节度使是封常清**——治所在龟兹（参 [安西大都护](./安西大都护.md)）；封常清那年夏天在轮台、北庭一带巡边，焉耆是他自龟兹东出北上的必经之站。**岑参**那年正在他幕下任判官——**岑参自长安西行至安西、北庭，往返凡两次（天宝八载/749 第一次出塞；天宝十三载/754 第二次出塞）**——两次西行皆**自西州经焉耆入龟兹**，焉耆于他而言是塔里木盆地东北角第一处长歇之地。**《岑嘉州集》中并无独立题为"焉耆"的诗篇，焉耆只见于轮台、走马川、白雪歌诸题中泛指"塞上"的语境**——焉耆于盛唐诗里并非长安人写得最多的一座西域城，但是他们行走西域最熟的一站。

焉耆这一年的城内事：王宫的礼仪、王族的婚丧、城内伽蓝的早晚课、湖畔的渔事、城外的牧马——照常运转。焉耆**贡马**仍是常项——湖西草场养的焉耆马，每年自焉耆经龟兹、伊州、沙州、凉州一路东去长安——是西域诸国贡马里走得最熟的一条道。城南**铁门关守捉**驻盛唐守捉兵数百——是焉耆通龟兹途中的第一处关津，关所设在天山南麓一道险峻峡口里——孔雀河自此峡口出，水声雷鸣。

天宝十四载冬，安禄山反于范阳的消息抵焉耆——已是次年（756）开春前后。封常清自安西东调入关，焉耆境内驻军是否亦在东调之列——史料未见直接证据；《通鉴》明记天宝末年"安西、北庭精兵东调勤王"——焉耆镇兵变动则无史料直载。**王宫与都督府的双重权力网自此渐渐失衡**——755 年这一刻焉耆人尚未察觉的是，这便是焉耆与中原渐渐失去联系的开始。

### 此刻意味

对天宝十四载冬的焉耆人而言，这座湖畔小城意味着——**安西四镇之东镇，自西州东去通沙州的第一段、自龟兹东出通中原的第一站**：城北仰天山、城东傍博斯腾湖、城西通龟兹、城南出铁门关——一处水汽未干的小绿洲，担着安西大都护府东向半边天的咽喉。城里王宫、都督府衙、伽蓝、镇兵营房并陈，是盛唐羁縻属国制度的一处典型。

他们尚不知道，再过三十多年，吐蕃将自南、自西两路压来；再过一个多世纪，回鹘西迁将自高昌而来——龙姓王统将随佛教一道在 9-11 世纪间悄然散尽。

## 城内地标（doc 内子条目，不另立 POI）

下列各项均为焉耆王城内或近郊地标，作 doc 内独立 `###` 章节锚定；不另立 `.md` 文件。

### 焉耆王宫（龙氏王宫）

城中央。**龙氏王统自魏晋以来约六百年**——盛唐时仍由龙姓王居。**王姓为"龙"**（《魏书·西域传》《旧唐书·西戎传》明载——不与龟兹白氏相混；白氏是龟兹王姓）。王宫与都督府衙署对望——羁縻制下王受朝廷册封，唐设都督府仍以王世袭任都督。755 年时王名史料未详——盛唐焉耆诸王名号自龙突骑支（贞观间）、龙栗婆准（贞观十八年/644 短期立）、龙婆伽利（高宗间）一路下来，开元、天宝间诸王名号散见史料而不连贯——本 doc 不强凿当年王名。

### 焉耆都督府衙署

王宫附近。**龙朔元年（661）置焉耆都督府**——这是唐于焉耆的最高羁縻建置。都督府长官以焉耆王世袭兼任都督——衙署办的多是焉耆与安西大都护府之间的文书、税赋（焉耆作为羁縻州不输中原赋税，但有马、毛布、葡萄酒等常贡）、镇兵口粮调度。盛唐安西节度使的诏令、长安朝廷的诏书——经此衙署传达入焉耆王宫；焉耆王对长安的奉表、贡马清单——亦经此衙署西去龟兹再东行。

### 焉耆镇衙署

王宫附近。**焉耆镇是安西四镇之一**——盛唐镇兵数千（具体兵额史料异文）。镇守使由安西节度使任命，由汉人或归唐胡将充任——与都督府衙、王宫并立。这是城内**唐 + 龙氏**双重权力结构的具象。755 年时镇守使姓名史料不详。

### 焉耆大寺

城内。焉耆奉**小乘"说一切有部"**——玄奘《大唐西域记·卷一·阿耆尼国》记此国"伽蓝十余所，僧徒二千余人，并皆习学小乘教说一切有部"。具体寺名史料未详——盛唐焉耆城内大寺名号难考；今博格达沁古城及周边遗址中发掘出过若干佛寺基址，年代上限至 5-6 世纪、下限至 8-9 世纪。**焉耆与龟兹同奉小乘说一切有部、与于阗大乘分庭**——这是塔里木盆地北道与南道佛教谱系分野的一处实例。

### 博斯腾湖（焉耆湖）

城东约二十里。**西域最大的内陆淡水湖之一**——东西延约百里、南北约五十里。开都河（古名敦薨水）自天山南麓下来汇入此湖，孔雀河自此湖南流出谷、过铁门关入塔里木盆地。湖里产鱼——盛唐焉耆人渔业为常项之一；湖畔芦苇成片、水鸟群集——是塔里木盆地一处独有的水汽景象。**焉耆"水国"的面相全靠这一湖**——城北、城西、城东三面水网，是西域诸城里少有的。今属新疆巴音郭楞蒙古自治州博湖县、焉耆县境内。

### 铁门关守捉

城南、天山南麓孔雀河出谷处（今库尔勒市北约十里）。**铁门关**是焉耆通龟兹途中第一处险隘——孔雀河自天山南麓裂出一道峡口南流，峡两侧岩壁陡立、河水雷鸣——盛唐设守捉一处，驻兵数百，是焉耆—龟兹官道上的咽喉。"铁门"二字状关之险——岑参《题铁门关楼》"铁关天西涯，极目少行客；关门一小吏，终日对石壁"——若属同人之作，便是盛唐边塞诗里写焉耆—龟兹道的一束花（按：此诗题"铁门关楼"传本异文较多，作者归属史料异文——一说岑参，一说他人——本 doc 沿用通行说法，hedge 处理）。

### 四十里城（聚落遗址）

城外向东四十里——盛唐时一处汉唐沿用的聚落，盛期 4-8 世纪。今学界考为焉耆都督府所辖一处屯田点或附属城堡。地表夯土残段仍可辨。

### 焉耆马场

城西、湖南草场一带。焉耆马自汉武帝时即贡入中原——《汉书·西域传》"焉耆国近海，多鱼……出马"。盛唐焉耆贡马仍是常项。湖西草场连绵——焉耆人世代以牧马为生计之一，与渔业、农耕并存。

### 阿耆尼故道（焉耆-高昌道）

城东去过博斯腾湖南岸、走十数日至**西州**（高昌、吐鲁番）。是焉耆通中原最东的一条道。玄奘**贞观元年（627）**西行过高昌后即由此道（反向）抵焉耆、随即北上龟兹。**贞观十八年（644）或十九年（645）**玄奘自天竺归途上再过焉耆——一说自于阗东归直入沙州、不经焉耆（《大唐西域记·卷十二》末段记归途仅至于阗一线为详，焉耆段简略）——本 doc 取通行说法：**归途经焉耆与否史料异文**——西行 627 必经，归途 644 / 645 是否亦经焉耆诸说并存。

## 历史长河（755 → 今）

**中晚唐**（755-907）

天宝十四载冬安禄山反——安西、北庭精兵东调勤王（具体焉耆镇是否随调史料未见直接证据，但通鉴载"河、陇、安、北精兵悉征入援"）。自此焉耆与中原音信渐疏。

757 年后，肃宗于灵武自立，朝廷自顾不暇——西陲诸镇音信渐疏。**安西副都护、留守节度使继守**——焉耆镇亦在其中。建中至贞元间（780s-790s），吐蕃自青海北上夹攻——河西、陇右陷蕃后再向西压。**约 790 年前后吐蕃陷焉耆**——具体年份史料异文：旧唐书系约贞元六年（790），新唐书与一些近世考据则系略晚（约 791-792 间）；本 doc 沿用通行说法**约 790 年**。这一年北庭都护府（治庭州）亦陷于吐蕃——同年安西大都护府所辖龟兹、于阗、疏勒、焉耆四镇相继失守。**自开元七年（719）焉耆代碎叶为四镇起，至约 790 年陷蕃止——焉耆居安西四镇之东镇前后凡七十余年**。

吐蕃陷焉耆后，将其纳入吐蕃节儿（驻军长官）治下——但龙氏王统是否随之中断，史料未明。**吐蕃在焉耆的统治延续约六十余年，至会昌、大中间（840s）随吐蕃王朝瓦解而解体**。

**回鹘西迁**　大中十二年（848）前后，漠北回鹘汗国为黠戛斯所灭——其部西迁。**咸通七年（866）**回鹘庞特勤一支据高昌——立**高昌回鹘王国**，势力西及焉耆。**自此焉耆由回鹘据之**——龙氏王统或已断或并入回鹘体系，史料不明。**"焉耆"二字自此自史册渐隐**——回鹘人称此地为**"卡喇沙尔"**（Karashahr，突厥语意"黑城"）——这是焉耆后世通名的来历。"阿耆尼"、"焉耆"、"卡喇沙尔"——三个名字属三种语言、三个时代，落在同一处湖畔绿洲。

**五代至宋**（907-1227）

五代之间，焉耆是高昌回鹘王国西镇——回鹘王廷在高昌、夏天行营在北庭。北宋与高昌回鹘有数次朝贡往来——但这一地区已自外于中原政权达两个世纪。11 世纪后，喀喇汗朝（黑汗王朝）自西而东，疏勒、于阗一线伊斯兰化——**11 世纪末至 12 世纪间，伊斯兰东传至焉耆一带**——焉耆佛教自此渐熄。盛唐焉耆城内大寺、湖畔伽蓝、玄奘讲经过的那座小乘大寺——皆在这一两个世纪里悄然散尽。

**蒙元**　13 世纪初蒙古西征，焉耆一带先后归西辽、察合台汗国。**元代称"喀喇沙尔"**或"哈剌沙"——是西域诸路之一。马可波罗《游记》中提及"卡喇沙尔"——是中世纪西方旅行者笔下此地的一处实证。

**明清**　明代喀喇沙尔归亦力把里（叶尔羌汗国前身）。**清乾隆二十四年（1759）**平大小和卓——喀喇沙尔正式入清版图，置喀喇沙尔办事大臣，属伊犁将军辖。**光绪八年（1882）**清廷设喀喇沙尔直隶厅，光绪十年（1884）新疆建省后改厅。**自盛唐焉耆陷蕃（约 790）至清复入版图（1759），中间整整 969 年**——这一带不属中央王朝。

**民国至今**（1911—）

民国二年（1913）废厅设焉耆县——**"焉耆"二字作为县名重回正式行政**，一千七百余年后再回到原名。今属新疆维吾尔自治区巴音郭楞蒙古自治州，1954 年设焉耆回族自治县——是新疆境内唯一以回族为主体的自治县。盛唐焉耆王城所在的**博格达沁古城**——今焉耆县城西南约十二公里处——20 世纪以来陆续勘探；地表夯土残墙仍可辨，城周三里、四门遗迹可寻。

**博斯腾湖**至今仍为西域最大内陆淡水湖之一——湖西、湖北是芦苇、水网、稻田；湖东、湖南半为咸化盐沼——20 世纪后期因孔雀河上游引水、湖水补给不足，湖面萎缩，部分水域咸化——这是当代水文变化，与盛唐时湖面广阔的景象已不同。

至于盛唐那座**焉耆王城**本身——夯土城墙今存博格达沁古城残段；龙氏王宫、焉耆都督府衙署、焉耆镇衙署的具体位置今不可考——只能由城内地表残基与文献推定。**铁门关**故址今为新疆库尔勒市以北约十公里处的铁门关风景区——孔雀河出谷处的峡口依旧，盛唐守捉的夯土残段已不可寻。

玄奘《大唐西域记·卷一·阿耆尼国》还在书架上摆着——读一读，再到博格达沁古城外的戈壁上走一走，距盛唐那个冬天已逾千二百年；距玄奘西行过宿亦逾千三百年。
