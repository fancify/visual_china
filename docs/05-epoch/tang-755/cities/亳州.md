# 亳州

## 一句话

河南道东南上州，唐人口中"谯郡"——城治谯县是曹魏三曹龙兴之地，城南属县苦县（666 改真源 / 689-690 改仙源 / 705 复真源）则是老子故里，玄宗朝因姓李奉老子至盛，亲祀太清宫便在此郡。

## 形貌

亳州治谯县，城坐涡水北岸——豫东冲积平原南缘，南距淮河约三百里。城周不大，夯土城垣四门各开。城外四望平畴——东去宋州（睢阳）二百里许、西北去汴州（陈留）二百里；城南循涡水东南行可入颍水、再入淮——**亳州是河南道东南角连接江淮的一处水陆节点**，但漕运份量远不及汴州、宋州那两个通济渠正干上的州。城东南九十里许属县真源县（原苦县，742 改名），有**老子故里**之传——太清宫即在县境，是玄宗朝最受恩宠的一处道家祖庙。气候是豫东典型——冬寒夏热、春多沙、秋高，涡水冬不全冻。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

谯郡是中原东南**叠了两段大旧事**的老地方——一段道家祖庭，一段曹魏龙兴。

老子一段最古。**《史记·卷六三·老子韩非列传》**："老子者，楚苦县厉乡曲仁里人也，姓李氏，名耳，字聃。"**苦县**自西汉以来为陈 / 淮阳 / 沛之属县，治在今河南鹿邑——汉唐间一直是**老子故里**的正传所指。东汉桓帝延熹八年（165）遣中常侍管霸至苦县祀老子——中国最早的国家级老子祭祀；唐高祖武德三年（620）以"老子为李氏远祖"诏改苦县老君庙为**紫极宫**——这是李唐皇室"道家系谱"的开端。

第二段是曹魏。**沛国谯（即唐谯县治）是曹操故里**。《三国志·武帝纪》："太祖武皇帝，沛国谯人也，姓曹，讳操，字孟德。"曹嵩、曹腾、曹丕（魏文帝）、曹植（陈思王）——皆谯人。"三曹"是建安文学的中心；曹操**《让县自明本志令》**（建安十五年 / 210，作于谯）以及多次班师归谯的记载，皆把"沛国谯"作他自家精神原乡。

行政沿革：魏黄初二年（221）置**谯郡**——魏文帝以谯为"先人本国"立郡。**北周宣帝大象二年（580）置亳州**于谯——"亳州"二字自此入史；隋大业三年（607）改**谯郡**；**唐武德四年（621）**复亳州。

老子故里苦县的县名沿革——这是 755 时态最易错处，须四节定之：**乾封元年（666）**高宗东封泰山归途，过苦县谒老子庙，下诏**改苦县为真源县**（取"真道之源"）——"真源"二字立于此；**载初年（689-690）**武则天临朝改元，**改真源县为仙源县**（取道家仙真之意，亦合武周改易旧名风格）；**神龙元年（705）**中宗复位，**复改仙源县为真源县**——"真源"二字自此稳定至五代。再到玄宗朝，**天宝元年（742）**诏改州为郡——亳州复称**谯郡**，**改苦县紫极宫为太清宫**（紫极宫名自高祖武德三年立）；**天宝间玄宗朝太清宫亲祀**（《旧唐书·玄宗本纪》《礼仪志》——具体某一次系年史志略异，旧唐书玄宗本纪与礼仪志互参可定大致序列）——这是盛唐道家祭祀的高峰。755 年此郡的官称：**谯郡 + 真源县（自 705 复改稳定五十年）+ 太清宫**。**乾元元年（758）**肃宗复改郡为州，谯郡复称亳州。

### 此刻

天宝十四载（755）——官称"谯郡"，民间常呼"亳州"。河南道东南端。本郡户口据《旧唐书·地理志》记**88960 户、67.5 万口（《旧唐书》）— 上州**——户数远过"上州"四万户门槛，列河南道东南实数大州，与宋州（12 万余户）、汴州（10 万余户）虽差一档但同属上州序列。州内主事：亳州刺史兼谯郡太守、州司马、州司户参军等——一套常规上州官署。亳州非道治（河南道采访使治汴州），但因**太清宫**在境内，朝廷祀典常落此城——道箓、道士、道家典籍自此郡转输入两京。

**玄宗本人亲祀谯郡太清宫**——史志载于天宝间，年份各志略异（旧唐书玄宗本纪与礼仪志互参可定大致序列，最稳的一次系于**天宝三载至天宝七载之间**，玄宗自洛阳东出经汴宋至太清宫祀老子再返洛阳）——这是盛唐道家祭祀的一处高峰。对天宝十四载的谯郡人来说，这座城便意味着两件事——**老子故里 + 玄宗朝道家中心、曹魏三曹故乡**。再过两月，安禄山自范阳起兵；亳州不在叛军主线，但**至德元载（756）**叛军余部经汴宋东南扩张，谯郡终告陷——与睢阳同陷同复。

## 城内地标

### 谯县（州治）

亳州治所，同时为谯郡治、谯县治。**城基与曹魏"沛国谯"旧城叠压**——曹操、曹丕、曹植故宅、曹氏宗庙、夏侯氏宗庙的旧址，皆在唐谯县城的范围之内或近郊。盛唐人访"三曹故里"是寻常事——但确址唐时已不可全寻，多依《三国志》《魏书》文献凭吊。**1970 年代以来此地相继发掘"曹操运兵道"与曹操宗族墓群（出土"曹腾""曹嵩"等砖铭）**——是盛唐人当时所不见、却最能与"沛国谯"印证的实物证据。

### 太清宫（真源县境，老子故里）

谯郡西南约九十里——治真源县境（原苦县，666 改真源 / 689-690 改仙源 / 705 复真源，今河南鹿邑）。**太清宫**自唐高祖武德三年（620）改老君庙为紫极宫以来便是李唐皇室的祖庙性道观；**天宝元年（742）**玄宗诏改名**太清宫**——"太清"取道家三清之一。**玄宗亲祀谯郡太清宫**事见《旧唐书·玄宗本纪》《礼仪志》——这是盛唐道家祭祀的高峰。详见独立 relic doc [龙角山 / 太清宫](../relics/龙角山.md)——本 doc 不重复殿宇年表与碑刻细节。

苦县老子故里说自西汉以来即正传——高宗乾封元年（666）改苦县为真源县后"苦县"二字渐废（中间 689-690 短暂改"仙源"，705 中宗复名"真源"）；五代**后唐同光元年（923）**真源县再改名**鹿邑县**，"鹿邑"自此立。盛唐 755 时仍是"真源县"。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改谯郡为亳州。安史平后地位渐衰——太清宫祀典仍在，但已不复盛唐"皇室祖庙"的待遇。会昌五年（845）武宗灭佛——太清宫属道家，反受其惠；唐末黄巢之乱经此境，宫殿多次焚毁。

**宋元明清**　北宋**大中祥符七年（1014）**真宗亲谒太清宫，重修殿宇，立**《先天太后赞碑》《大宋重修太清宫碑》**——是北宋初的一次大复兴；金元后渐衰，但碑刻保存甚好。元属归德府亳州；明清间数升数降——清雍正二年（1724）升直隶州。清末（19 世纪中）**亳州为捻军重要根据地之一**（张乐行籍亳州）。

**民国至今**　1912 年废州改亳县；2000 年升地级亳州市——"亳州"以市名重立，自 580 年置亳州算来已一千四百二十余年。**鹿邑**仍为县名，属周口市。**今鹿邑县城东四公里有太清宫遗址**——明清重修的前殿、后殿尚存，宋元碑刻数通保存至今；**前殿前的"先天太后赞碑"**（北宋大中祥符七年立）是现存最重要的实物。盛唐玄宗亲祀的那座太清宫——殿宇早不可见，但碑刻、台基、遗址地望与 755 年相去不远。

## 引文

> 老子者，楚苦县厉乡曲仁里人也，姓李氏，名耳，字聃，周守藏室之史也。

——《史记·卷六三·老子韩非列传》。"楚苦县"——战国苦县属楚，秦汉以后属陈 / 淮阳 / 沛 / 谯郡，唐属亳州（谯郡）真源县。

> 太祖武皇帝，沛国谯人也，姓曹，讳操，字孟德。

——《三国志·卷一·魏书·武帝纪》。盛唐人入谯郡访曹操故里所凭便是此条与陈寿《三国志》通篇；曹操《让县自明本志令》（建安十五年 / 210，作于谯）是其自家精神原乡最直接的一段自述。
