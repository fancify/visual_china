# 岳州

## 一句话

江南西道岳州治所，唐人口中『巴陵』——洞庭湖东北岸、长江入湖口南畔的水驿要冲；城西临湖一面立着岳阳楼，开元四年张说改建之后，便成了俯瞰洞庭的天下名楼。

## 形貌

岳州城盖在洞庭湖东北岸、君山东南向的台地上。西门外便是浩荡的洞庭——长江中游最阔的一面水，南北约一百八十里、东西约一百二十里。湖中横着一座**君山**——孤峰浮水，与城隔湖相望约十数里。城北十余里即长江干流，**江湖以一段湖口相吞吐**——洪汛时江水倒灌入湖、枯水时湖水东流入江。**岳阳楼即在城西门正面的临湖突崖上**——北望长江入湖口、西望君山如青螺浮水、南望湘江北来入湖。湖上风帆上行入湘潭、下行抵鄂州——岳州正是这一段水道的转关。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

此地春秋属楚，称巴陵之邑。**东汉建安年间（约 213-217）东吴大将鲁肃**奉孙权命屯军巴丘——这是后世岳阳楼的最早前身一说（《三国志·吴书·鲁肃传》载鲁肃屯巴丘，**鲁肃建安二十二年/217 卒**；后世以巴陵戍楼起于鲁肃，**具体建楼年代不可实证**，本 doc 不强订年）。**隋开皇九年（589）**隋平陈置**岳州**治巴陵县——『岳州』之州名起点。**大业三年（607）**改州为郡，岳州短暂改称**罗州**，旋改**巴陵郡**。**武德六年（623）**唐复置岳州治巴陵县。**天宝元年（742）**玄宗诏天下州一律改郡，岳州改称**巴陵郡**——这是 755 年当年的官称；**乾元元年（758）**肃宗复改郡为州——755 此刻匾额仍写『巴陵郡』。

楼的一段更要紧。**唐玄宗开元四年（716）张说出守岳州**——张说由宰相贬岳州刺史，在任期间**改建巴陵戍楼，自此楼名『岳阳楼』**（《张说集》及后世题楼诗记此事）。鲁肃戍楼本是军事瞭望；张说改建之后，楼才转为登临名楼——盛唐至晚唐数百名家在此题诗，多在张说改建之后。**孟浩然《望洞庭湖赠张丞相》**"气蒸云梦泽，波撼岳阳城"——开元中作（一说开元二十年/732 前后），已是盛唐岳阳楼诗的高标。

### 此刻

天宝十四载（755）的岳州行政上是**江南西道巴陵郡治**——一郡领**巴陵、华容、沅江、湘阴、昌江**五县（按《旧唐书·地理志》《新唐书·地理志》《元和郡县图志》；昌江县天宝五载/746 析湘阴置，755 时已在治内）。**《旧唐书·地理志》载岳州（天宝巴陵郡）『户一万一千七百四十、口五万二百九十八』**——按唐制中州户额约二万、下州一万（《唐六典》），岳州**户口实当下州/中下州之间**，远不及上游江陵、下游鄂州；旧 doc 写『中州』口径偏宽，按本志当**下州/中下州**。岳州的份量不在户口，而在**楼与湖**：一座临湖名楼、一面长江中游最阔的水。755 年冬安禄山起兵的消息沿长江驿道南传至巴陵约二十五日——但岳州距叛军主战场尚远，江汉一带实质卷入要待至德二载（757）以后。

## 子条目

### 巴陵城（岳州治、巴陵县治）

唐岳州治所，亦巴陵郡治、巴陵县治——三套官称同在一城。夯土城墙倚洞庭东北台地，西门外正对洞庭，楼即在西门外临湖突崖。**唐巴陵故城在今湖南岳阳市岳阳楼区一带**——历代屡建屡迁，明清岳州府城在原址上叠建。

### 岳阳楼

**岳阳楼立于唐岳州治巴陵县（今湖南岳阳市）西门外临洞庭突崖上**。**前身为鲁肃屯巴丘旧说所传巴陵戍楼**（《三国志·吴书·鲁肃传》载鲁肃屯巴丘事，**鲁肃 217 卒**；后世以戍楼起于鲁肃，**具体建楼年代不可实证**，本 doc 不强订年），本为军事瞭望，监视洞庭与长江入湖口。**唐玄宗开元四年（716）张说出守岳州时改建——自此楼名『岳阳楼』，由军事转为登临名楼**（《张说集》载此事）。盛唐时楼临湖而立，高数层（具体形制盛唐文献未详记），登楼可西眺君山、北望长江入湖口、南俯洞庭。**孟浩然《望洞庭湖赠张丞相》**"气蒸云梦泽，波撼岳阳城"——开元中作，已是 755 年前盛唐岳阳楼诗的高标。**杜甫《登岳阳楼》**"昔闻洞庭水，今上岳阳楼"——**大历三年（768）杜甫流寓岳州时作，已在 755 后十三年**，列于历史长河段。**范仲淹《岳阳楼记》**——北宋庆历六年（1046）作，**距 755 年近三百年**，更不预支。

### 洞庭湖 + 君山

长江中游最大湖——**岳州、潭州、朗州三州夹之**（盛唐地理：北岸岳州、东南岸潭州、西南岸朗州）。湖南北约一百八十里、东西约一百二十里——盛唐时湖面较今为阔，『八百里洞庭』是唐宋人对盛涨时的概括。湘、资、沅、澧四水皆汇入洞庭——江南西道南半的水量总会；湖东北角即长江入湖口，江湖吞吐之势盛唐人已熟知。湖中**君山**孤峰浮水，与城隔湖相望约十数里——**舜南巡崩于苍梧，二妃娥皇女英追至君山、泪洒竹上成『湘妃竹』、自沉湘水**——盛唐人登岳阳楼西望，君山是必赋之景。

## 历史长河（755 → 今）

**中晚唐**：乾元元年（758）复郡为州，巴陵郡再称岳州。**大历三年（768）杜甫**自夔州顺江而下流寓岳州——《登岳阳楼》"昔闻洞庭水，今上岳阳楼。吴楚东南坼，乾坤日夜浮"——五十七岁的杜甫登楼一首压住盛唐至中唐题楼诗。**大历五年（770）杜甫卒于湘江舟中**，距登岳阳楼仅两年。

**宋**：**庆历六年（1046）滕子京谪守巴陵郡**重修岳阳楼，请同年范仲淹作记。**范仲淹《岳阳楼记》**"先天下之忧而忧，后天下之乐而乐"——中国楼记文学的压卷。范仲淹未亲至岳阳，文中『衔远山，吞长江』等景出于滕子京寄来的《洞庭晚秋图》——著名的『未至而记』。

**元明清**：元置岳州路；明洪武九年（1376）置**岳州府**治巴陵县。岳阳楼数毁数葺，**清光绪六年（1880）岳州知府张德容重建**便是今所见岳阳楼的近代原型。

**民国至今**：1913 年改巴陵县为**岳阳县**——『岳阳』作为正式行政名第一次出现；1983 年升岳阳市。盛唐人称此地为『巴陵』『岳州』，**绝不可预支称『岳阳市』**。

## 引文

——孟浩然《望洞庭湖赠张丞相》，开元中作；**755 年时已盛传**。

> 昔闻洞庭水，今上岳阳楼。吴楚东南坼，乾坤日夜浮。亲朋无一字，老病有孤舟。戎马关山北，凭轩涕泗流。

——杜甫《登岳阳楼》，大历三年（768）作，**距 755 十三年**——历史长河段。

> 先天下之忧而忧，后天下之乐而乐。  ——范仲淹《岳阳楼记》（北宋庆历六年/1046）——历史长河段。
