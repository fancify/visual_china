# 商州

## 一句话

长安东南越秦岭出武关的第一座州——上洛郡治上洛县（今陕西商洛市商州区），武关道中段的山州。

## 形貌

商州城坐在**丹水（今丹江）上游北岸**一处河谷台地上，四面群山——南即秦岭主脊东段，北接蓝田、商於山地。**州治上洛县**——城周不大，按《新唐书·地理志》上洛郡列**"望"**等（唐州县等第八等中第二等，次于"辅"）。户口据《旧唐书·卷三九·地理志二·商州上洛郡》记**户 8,926、口 52,080**；《新唐书·卷四十·地理志四·商州上洛郡》户同、口 **53,080**（两书略异）。城西北越**蓝田关**约二百余里至长安；城东南循丹水谷地经武关、南阳，可下襄阳、入江汉——**这一条由长安出蓝田关、过商州、出武关南下的山间驿道，便是盛唐"武关道"**，与潼关道并列为长安东出二大孔道。

州境多秦岭余脉的深谷高峡——丹水蜿蜒于谷底，两岸塬上种粟、谷间育漆桐茶柞。物候比关中偏南——春来略早、夏多山雨。

## 大唐天宝十四载（755年）

商州的沿革明白：北朝有上洛郡、洛州之名递有。**隋开皇九年（589）平陈后改置商州**——以本地古商於之地得名，治上洛县。**大业三年（607）**炀帝行州改郡制，改商州为**上洛郡**。**唐武德元年（618）**高祖复改郡为州——商州之名再立，属山南道；开元二十一年（733）分山南为东、西二道后，商州属**山南东道**（与关内紧邻、又当武关道，故唐人视之半属关辅）。**天宝元年（742）**玄宗诏天下州改郡——商州又改回**上洛郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州——商州之名第三次立。**755 这一刻官文书上写"上洛郡"，民间口语仍呼"商州"**——两名并行，太守署的本郡长官称"上洛郡太守"。

755 年的上洛郡领县**六**（《新唐书·地理志》记上洛、商洛、洛南、丰阳、上津、乾元）——其中上洛为郡治。本郡 755 当年最显眼的事，是它当**武关道中段**的位置：自长安东南出春明门、过蓝田、越**蓝田关**入商州境，再南行约一百八十唐里抵上洛城；自上洛再南行经商洛县、抵**武关**——出武关南下便入南阳郡（邓州）境，可经南阳、襄阳一路下江汉。盛唐自长安南赴荆襄、岭南，多取此路——这是上洛郡太守在 755 当年的本职：守驿、备粮、巡道、护过境官使。**州东北丹水上游边境近蓝田一段，山间有"蓝桥"驿**（即后世唐传奇裴航蓝桥故事所在地，详见下段历史长河）。

天宝十四载冬安禄山反范阳——长安东出潼关一线吃紧，**武关道的重要性陡然上升**：朝廷自长安调江淮、荆襄租赋入京，多改循武关道——上洛郡的驿务自 755 冬到次年六月长安失守前的半年间，是百年罕见的繁忙。城里人尚未察觉的是：再过半年（756）盛唐转折之际，本州看到的过境官使远多于平日——但具体路线需作分辨：**太子李亨北行灵武走的是关中西北的泾邠路（经奉天、新平、安定一线），并非武关道**；玄宗 756 西巡入蜀亦自陈仓南下，未取此路。武关道在安史乱中的真正分量，是作 **江淮租赋经长江—汉江—武关—长安** 的备用转输通道——玄宗西巡入蜀后，江汉转输部分改循武关道入关中，但具体史源记载有限，详细的运量、班次、过境官使名录皆难考。

## 历史长河

**中晚唐**　至德二载（757）/乾元元年（758）复改上洛郡为商州。**安史乱后江淮租赋经武关道入关中**——商州一线驿务长期繁忙，是中晚唐财赋北运的关键节点之一。**唐传奇《裴航》**（成书晚唐）记裴航过蓝桥驿遇云英、捣药百日、终结为仙侣——"蓝桥"自此成为中晚唐至宋元戏曲文学的常用典故，本事即在商州西北上洛—蓝田交界处的山间驿馆。

**宋元**　北宋商州属永兴军路；金属京兆府路。元属奉元路。商州治所未改。

**明清至今**　明洪武七年（1374）降商州为商县；成化十三年（1477）复升商州，属西安府。清雍正三年（1725）升商州为**直隶州**——领商南、雒南、山阳、镇安四县。民国二年（1913）废府州——商州改商县。1949 年后属陕西省商雒专区；1988 年商县改**商州市**（县级）；2001 年商州市改**商州区**——今陕西商洛市商州区治所，约在唐商州上洛县旧址上。盛唐州城残基已埋入今商州城下，武关道线仅存零星驿堡遗迹于丹江沿岸——"上洛郡"三字，只在《旧唐书》《元和郡县图志》和那一段裴航蓝桥的传奇里仍可见。
