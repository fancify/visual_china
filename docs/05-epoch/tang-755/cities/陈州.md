# 陈州

## 一句话

河南道东南一**上州**，治宛丘县（今河南周口市淮阳区），**742-758 年间名"淮阳郡"**——城坐颍水中游北岸，**春秋陈国故都**所在；城北郊有古地**太昊陵**（伏羲陵），陈州境内最古的一段标签。

## 大唐天宝十四载（755年）

陈州是河南道东南一片平畴上的旧地——治宛丘县。沿革：**隋开皇九年（589）置陈州**于此，**大业三年（607）改淮阳郡**，**唐武德元年（618）复陈州**，**天宝元年（742）再改淮阳郡**，**乾元元年（758）再复陈州**。755 这一刻匾上写"淮阳郡"，太守一员领郡事。《旧唐书·地理志》《新唐书·地理志》皆记陈州淮阳郡为**上州**——户口殷实：**户 66,442 / 口 402,486**，在河南道东南一线大上州中稳居一档。郡境平衍——颍水自西北来、东南流入汝阴郡（颍州），是循颍水北上中原的一处腰节；东北去亳州（谯郡）二百里许，南去蔡州（汝南）二百里，东南去汝阴郡（颍州）二百里。

陈州在唐人心目中最古的一段标签是**陈国故都**。**《史记·卷三六·陈杞世家》**载：周武王灭商，封舜后胡公满于陈，都**宛丘**——陈胡公以下二十余世（前 1046—前 478 年陈为楚惠王所灭，前后五百余年），陈国一直都宛丘。再到战国晚期——**《史记·楚世家》**载楚顷襄王二十一年（前 278）秦将白起拔郢，**楚迁都陈**，号**陈郢**——这是战国末楚国最后一段都邑岁月。城北郊有**太昊陵**（伏羲陵）——太昊伏羲传为陈州始祖，**陈**之名亦多与此牵连；唐时陵墓已古，盛唐人来淮阳郡多就此凭吊上古一段。755 年这一冬安禄山自范阳起兵，淮阳郡不在叛军正面主线，但循颍水南下江淮之路自此过，至德元载（756）以后江淮防线收紧，本郡为唐军江淮转输的一处后方腰节。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改淮阳郡为陈州。安史平后地位渐稳。**宋元明清**　北宋陈州地位骤显——**范仲淹、王安石、苏轼**前后皆曾出知陈州（范仲淹 1041 知陈州、王安石 1066 任陈州通判一段、苏轼曾过陈州留诗甚多），是北宋中前期一处文官常出之地；金元后渐衰，元属归德府陈州；明清间数升数降，清雍正十二年（1734）升**陈州府**。**民国至今**　1912 年废府存县——陈州府改淮阳县（"淮阳"二字作县名自此立，沿用唐 742-758 旧郡名）；2000 年地级**周口市**立，淮阳县属之；2019 年淮阳撤县设区，为周口市淮阳区。盛唐淮阳郡的城圈大体保留在今淮阳区内。**太昊陵**——盛唐人凭吊的那座古陵——明清屡修，今淮阳区北郊太昊陵尚存，是中国最古的几处人文始祖陵之一。

## 引文

> 陈胡公满者，虞帝舜之后也……武王克殷纣，乃复求舜后……得妫满，封之于陈，以奉帝舜祀，是为胡公。

——《史记·卷三六·陈杞世家》。陈州"陈国故都"一段最直接的史源；都宛丘——即唐陈州州治宛丘县所在。
