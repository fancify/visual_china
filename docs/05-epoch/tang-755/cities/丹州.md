# 丹州

## 一句话

关内道东北一州、黄河西岸——治义川县（今陕西延安市宜川县），742-758 年间名"咸宁郡"；**与河东道隔黄河相望；与同州同在黄河西岸，一南一北**，隋代壶口瀑布即在本州东境黄河上下游。

## 形貌

丹州城坐在**陕北黄土高原东南缘**——城外塬、梁、峁与沟谷交错，沟里水浇地、塬上旱作粟糜。州境东抵**黄河**——河对面便是河东道、与同州一线东西对望（同州在西、丹州在北，二州皆当黄河西岸而隔河距数百唐里）。**黄河自北而南穿丹州东境**——本州东境黄河上下游一线是黄河自陕北高原入晋陕峡谷之要段，**壶口一带**——河水自此跌入峡谷成飞瀑——便在本州东境黄河之上。州西北接鄜州（洛交郡）、西南接坊州（中部郡）、北接延州（肤施郡）——是关内道东北鄜延一线偏东一州。物候：春旱多风沙、夏短秋早霜、冬干寒——盛唐丹州一岁两熟难得，旱作粟糜为主。

## 大唐天宝十四载（755年）

丹州在唐人眼里是**关内道东北一下州**——治义川县，742 年改称咸宁郡。沿革：**隋开皇十八年（598）置丹州**——以城东丹阳川（一作丹水）得名；**大业三年（607）罢州**为延安郡属县；**唐武德元年（618）**高祖复置**丹州**——属关内道；**天宝元年（742）**玄宗诏天下州改郡，丹州改称**咸宁郡**——这是 755 年当年的官称；**乾元元年（758）**复改郡为州，"咸宁郡"之名前后用了十六年。**《旧唐书·地理志》载丹州（咸宁郡）为下州**——天宝户口在关内道诸州中靠后，远不及同州、华州、京畿诸郡。755 年这一刻匾上写"咸宁郡"，**太守一人领郡事**（注：唐州一级长官，**742 改州为郡后称"太守"，758 复郡为州后称"刺史"**——755 当年只称"咸宁郡太守"，不称"丹州刺史"）；**天宝十四载咸宁郡太守在任者史志阙考**。

丹州的分量不在本州事务——而在**它当黄河西岸**与**鄜延一线偏东的位置**。自长安北行经坊州、鄜州一线是关中通朔方的官道正干；自鄜州东出经丹州可抵黄河西岸渡口——是关中—河东之间另有一条北段渡河走廊（南段在同州蒲津）。本州东境黄河谷地险峻——上下游间水势湍急，盛唐人路过此地，多有题咏。

## 历史长河

中晚唐丹州仍为关内道下州，**大历三年（768）**置**鄜坊节度使**辖鄜、坊、丹、延四州（治鄜州），丹州为其辖下一州，自此一百余年皆在鄜坊—保塞军—彰义军一系藩镇之下。**北宋**仍置丹州——属永兴军路、后属鄜延路。**金元**丹州属鄜延路 / 延安路。**明洪武二年（1369）省丹州入宜川县**——"丹州"作为州名自此从地名上退去，前后历七百七十年；所辖义川县已先于元至元六年（1269）省入州本身，明初改宜川为治。**清**沿明制，宜川县属延安府。**民国二年（1913）废府存县**——宜川县存（注：**宜川县**今名首见于明初——元代由义川县改名一次，避讳之故；民国沿用，**1914 年并非"宜川"立名之时**）。1949 年后宜川县属延安专区、延安市；今为陕西延安市宜川县——县城仍在唐丹州 / 咸宁郡州治旧址一带。本州东境黄河上的**壶口瀑布**——盛唐唤名见诸地志，今为**国家级风景名胜区**，位置在今陕西宜川县与山西吉县之间黄河干流上，黄河西岸属宜川。

## 引文

> 丹州，咸宁郡。下。隋延安郡之义川县。武德元年，平薛举，置丹州。天宝元年，改为咸宁郡。乾元元年，复为丹州。旧领县四，户一万五千一百五，口八万七千六百二十五。

——《旧唐书·卷三十八·地理志一·关内道·丹州（咸宁郡）》（节录）
