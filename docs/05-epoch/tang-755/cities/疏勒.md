# 疏勒

## 一句话

安西四镇之西镇，**葱岭东口**——塔里木盆地最西的一座绿洲城邦，自此再西出便登帕米尔高原；唐人识它有三：佉沙、玉与玛瑙之乡、玄奘归途上停过约一月的伽蓝之国。

## 形貌

疏勒不在天山下，也不像于阗那样仰昆仑——疏勒卡在**葱岭东口**。

葱岭是唐人对帕米尔高原的称呼——一片由帕米尔、昆仑西段、天山西段、兴都库什北段诸大山合围而成的高原山结。塔里木盆地自东向西收紧、收紧、再收紧，最后收到疏勒这一处——城西、城南、城北三面皆是雪山支脉，唯城东开向塔里木盆地，是一座"袋状"绿洲。城本身坐在喀什噶尔河（唐人称之未有统一名号，今名克孜勒苏河 / Kizilsu，意"红水"）的冲积扇上——河自葱岭山口下来，绕城南、城东而过，最后没入塔里木盆地西缘的沙海。沙、河、山雪三色之外，疏勒比龟兹、于阗多了**第四色**：城西、城北那片葱岭雪线，是塔里木南北两道的西墙。

城是一座椭圆形夯土城——周回史料无明载，姑作"略小于龟兹"理解。今学界主流将盛唐疏勒镇城定位于**喀什市区一带**——具体位置因后世喀什噶尔老城屡建屡迁、夯土多为伊斯兰化后居民活动所毁，**地表轮廓已不可辨**——这一点与于阗约特干遗址相近，而与库车皮朗故城的连绵残墙不同。城外向北、向西，散落着莫尔佛塔、汗诺依等大批佛寺、聚落遗址——盛唐时这一片是疏勒王国的腹地。

城里街巷不作长安棋盘——多是依河渠走向的曲折土巷。王宫居城中，旁有大寺——疏勒自 1 世纪后期传入佛教，至盛唐已是西域大乘佛教与小乘并存的一座绿洲。**疏勒的佛门面相**与龟兹、于阗皆有不同：龟兹奉小乘说一切有部，于阗奉大乘，疏勒则**大小乘并存**——玄奘在《大唐西域记》里特记此分野——"伽蓝数百所，僧徒万余人，习学小乘教说一切有部"。**注：玄奘所记疏勒以小乘为主，与同书所记于阗大乘截然不同；具体的派系组成或在 7-8 世纪间略有变动，盛唐时疏勒佛门派别细节史料 sparse，本 doc 不强凿**。

物候比龟兹、于阗更寒——冬日葱岭风下，城里早霜来得早；夏日盆地南缘的暑气在此为雪山所抑，比塔里木深处略凉。一年降雨极少，水全靠葱岭雪融。城周屯田种麦、粟、瓜、葡萄——西域诸城同制，疏勒亦不例外。城外向东沿盆地北缘东行，过拨换城（今阿克苏），十数日至龟兹；向东南沿盆地南缘，过朱俱波（今叶城），七八日至于阗；向西出葱岭东口，过**葱岭守捉**（在今乌恰县境一带），数日至**护密国、识匿国**（今帕米尔山口一带），再西即入吐火罗、大食辖境；向西北过赤山（今阿赖山）一带，则可接碎叶、怛逻斯一线。**这是塔里木盆地最西的一座城，也是大唐通中亚、通大食、通拜占庭的陆路上离长安最远的一座节度镇**。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

疏勒是一座**汉人见过两次、唐人接手时已是第三遭**的旧西域城——与龟兹相类，但更西、更早进入汉籍。

《汉书·卷九六·西域传上》"疏勒国，王治疏勒城"——这是汉籍最早记疏勒的一笔。东汉永平十六年（73）班超出使西域，至疏勒——废其王兜题，立疏勒故王兄子忠为王——这是疏勒第一次明确归汉。班超经营西域三十年间，疏勒数度反复——但终于成为东汉西域都护府辖下的稳定一州。**自此"疏勒"二字进入中国正史，再未间断**。

魏晋以下，疏勒在前凉、前秦、北凉、嚈哒、突厥的影响之间起落——与龟兹、于阗诸国同一节奏。**5-6 世纪间**疏勒成西域佛教重镇之一——伽蓝、僧舍、佛塔密布；汉地译经里若干种自疏勒携回的写本，便是这一段史的注脚。

玄奘西行归途经此。**贞观十八年（644）**春，玄奘自天竺携经像东归——过葱岭、自吐火罗东行——抵**佉沙国**（即疏勒；"佉沙"是疏勒的梵文音译，玄奘所用）。《大唐西域记·卷一二·佉沙国》记其在佉沙停留——具体停留时长史料未明，**约略一月**（参《大慈恩寺三藏法师传·卷五》归途次第；具体日数史料异文，本 doc 加 hedge）。玄奘记佉沙"**周五千余里，多沙碛，少壤土……土宜稼穑，花果繁茂；出细毡褐、毳布。气序和畅，风雨顺序，人性犷暴，俗多诡诈，礼义轻薄，学艺肤浅……伽蓝数百所，僧徒万余人，习学小乘教说一切有部**"——这是 7 世纪中叶疏勒的实录。自佉沙东行，玄奘再过乌铩（今莎车）、于阗——经一段塔里木南道，方东出阳关回长安。**距 755 年安西四镇这一西镇，正好一百一十一年**。

唐对疏勒的直接经营自贞观间始。**贞观二十二年（648）**，唐将阿史那社尔平龟兹、于阗内附——同年（或翌年间，史料异文）唐于疏勒置**疏勒都督府**（羁縻）——这是"疏勒"二字第一次进入唐官制。**显庆三年（658）**，唐平西突厥贺鲁——疏勒与龟兹、焉耆、于阗并立为**安西四镇**，是塔里木盆地西陲的总根脚。

中间反复数次：咸亨元年（670）四镇陷吐蕃；上元元年（674）一度复置；**武则天朝（约 692-704 间）**一度**以碎叶代焉耆**——四镇构成短期变作"龟兹、碎叶、于阗、疏勒"；**开元七年（719）**焉耆复入四镇——但**疏勒在每一次反复中皆未被碎叶 / 焉耆 替代过四镇位次**，是安西四镇里"最稳"的一镇。长寿二年（692）王孝杰收复四镇——自此至 8 世纪中叶约六十年间，**疏勒稳坐安西四镇之西镇**，唐于其地置**疏勒镇**，常驻镇兵（具体兵额史料异文：一说四千五百、一说三千，姑取约数）；镇下又设守捉若干——但城内日常仍由疏勒王自治，唐"都督"由王世袭兼任，是与于阗相同的**羁縻 + 镇戍**复合结构。

景龙、开元间（700s-720s），疏勒王裴氏（汉名；本族名史料异文）先后袭爵；开元中疏勒王入朝长安，玄宗厚遇——授官、赐姓——疏勒王一支自此渐与长安宫廷往来频密。**开元、天宝间**，疏勒因其地理位置——**唐军西出葱岭、远征中亚的最近一座节度镇**——成为安西方面军的西征基地。

**天宝六载（747）**，安西副大都护**高仙芝**率万余步骑自安西西征——**以疏勒为前进基地**，翻葱岭、穿越坦驹岭天险，降小勃律国（在今巴基斯坦北部吉尔吉特一带）。这一战是盛唐军威西达葱岭之外的最远一仗——疏勒是这一仗的西征大本营。**天宝十载（751）**，高仙芝再率蕃汉兵三万西征——亦以疏勒为前进基地——西行过葱岭、入费尔干纳盆地，与**大食**（阿拔斯王朝）军会战于**怛逻斯**（今哈萨克斯坦塔拉兹附近）。**七月**激战五日，葛逻禄部临阵倒戈，唐军大败，三万之众生还者数千（具体伤亡数史料异文，史家有"几尽""数千还""万人还"诸说，本 doc 沿用通行的"数千还"；具体战役细节、唐军编组等史料 sparse，本 doc 不强凿）。**怛逻斯之后唐势力在中亚河中地区由此退缩**——但安西四镇本身未损，疏勒城下兵力虽折，未被焉耆 / 碎叶 替代过四镇位次，高仙芝亦自疏勒方向退回龟兹。

### 此刻

天宝十四载（755）的疏勒，名义上仍是**疏勒都督府 + 疏勒镇**——王是裴国良（一作裴冷冷；史料异文，**疏勒王 755 年具体在位者史料 sparse**，姑取通行说法）。镇兵编制上属安西节度使，节度大旗下，是**封常清**——封常清本年驻安西治所龟兹，又兼北庭都护、伊西节度使，疏勒为其西线最远一镇。

王治、镇衙署同在疏勒城——王宫居城中，疏勒镇衙署与之南北对望，是与于阗、龟兹相同的**双重权力**格局：唐镇守使管军政、疏勒王管旧俗。城内大寺早晚钟鼓——僧徒万余在伽蓝之间往来；棉、玉、毡褐、瓜果——是疏勒输入长安的常项。**玉与玛瑙**——疏勒之物产，玉自葱岭东麓、城外山中产出，玛瑙自喀什噶尔河（克孜勒苏河，今名）上游山中——**注：和田玉以软玉（透闪石玉）著称，疏勒一带所出与和田玉是否同源、玉质如何，史料只笼统记"出玉"；本 doc 不强凿与和田玉之异同**。玛瑙之名见于《新唐书·西域传·疏勒》"出玉、玛瑙"——是盛唐疏勒物产中最具特色的一项。

天宝十四载冬，安禄山反于范阳的消息抵疏勒——大约已是次年（756）开春前后。安西、北庭精兵东调勤王——封常清、高仙芝十二月十八日（公历 756 年 1 月 24 日）同诛于潼关，消息更晚才到城里。疏勒镇兵当年是否随东调之列——史料未见直接记载；但《通鉴》明记"河、陇精兵悉征入援"、于阗王尉迟胜亲率本国兵五千东出——疏勒方向是否亦有兵抽调，**史料 sparse，本 doc 不强凿**。

755 年这个冬天的疏勒人尚不知道——他们的镇所恃的中央，长安，将自顾不暇；再过三十多年，吐蕃自西、自南两路压来，安西四镇皆将陷蕃；再过两个多世纪，**喀喇汗朝将自西而东，疏勒将先于于阗、先于龟兹，第一个伊斯兰化**——这是后话。

### 此刻意味

对天宝十四载冬的疏勒人而言，这座绿洲城意味着——**安西四镇里离中原最西、离葱岭最近的一座**：城西出葱岭即出大唐辖境，城东去通龟兹接安西治所，城东南去通于阗——是塔里木盆地西陲三道的总交汇。**对唐军而言**，疏勒是西征的最后一座补给基地——高仙芝两次西征皆以疏勒为前进基地、经此集兵出发；**对胡商而言**，疏勒是粟特、吐火罗、大食商旅入塔里木盆地的第一座唐镇——文牒、关税、税务、市易，皆在城内。**对僧人而言**，疏勒是西行求法、东归译经路上的一处大伽蓝——玄奘归途在此挂单约一月，便是这一意义的最具体写照。

他们尚不知道，再过五年，这座城所恃的安西大都护府将开始与中原失去联系；再过三十多年，疏勒亦陷吐蕃；再过两个多世纪，**疏勒的伽蓝、佛塔、写经、僧徒——皆将在喀喇汗朝东进的浪里悄然散尽**。

## 城内地标（doc 内子条目，不另立 POI）

下列各项均为疏勒王城内或近郊地标，作 doc 内独立 `###` 章节锚定；不另立 `.md` 文件。

### 疏勒王宫

城中央。疏勒王统自汉至盛唐延续——具体王族姓氏史料异文（一说裴氏汉名为唐廷所赐；本族名于阗塞语 / 突厥语 / 粟特语并存，**唐文献中以汉名为主**）。盛唐时仍由疏勒王居——王制为羁縻属国制，王受朝廷册封、唐设都督府仍以王世袭任都督。755 年时王宫具体规制史料未明——参龟兹白氏王宫、于阗尉迟氏王宫的同时段记录，姑作"中型木构 + 夯土院落"理解。今喀什市区一带地表早不可辨——盛唐文化层在地下若干米处，新疆考古所在喀什噶尔故城（今喀什老城）周边历年勘察中陆续触及。

### 疏勒镇衙署

王宫附近。唐疏勒镇驻军长官（**镇守使**）的衙署——常驻镇兵（具体兵额史料异文：一说四千五百、一说三千，姑取约数），下设守捉若干。755 年时镇守使姓名史料不详。**疏勒镇是安西四镇里唯一未被焉耆 / 碎叶 替代过四镇位次的一镇**——咸亨陷蕃、长寿复置、武则天朝碎叶代焉耆、开元焉耆复入——每一次反复中疏勒皆在四镇之列，是安西西线最稳的一根桩。衙署与王宫南北对望——是城内**唐 + 疏勒王**双重权力结构的一处具象。

### 疏勒大寺

城中。具体寺名史料未详——玄奘 644 年挂单之寺《大唐西域记》未明指其名，**约为城中或城北一处大乘 / 小乘并存之伽蓝**。盛唐时寺内伽蓝、僧舍、佛塔规模延汉魏旧基，僧徒万余（玄奘所记 7 世纪中叶数；盛唐时是否仍达此数史料 sparse）。**疏勒佛门派别细节**——大小乘并存的具体演变、汉地译经里有无"疏勒本"写本系统——史料 sparse，本 doc 加 hedge。

### 莫尔佛塔（城北郊）

城北约三十里（今喀什市东北约 30 公里恰萨乡境）。一处中亚式覆钵塔为中心的佛寺遗址——主塔残高约 12 米，方形塔基 + 覆钵之上原应有相轮（今已不存）。**始建年代史料异文**——一说约 3-5 世纪，一说更晚；**盛唐时仍盛**。20 世纪初斯坦因首次勘察、记录；今为全国重点文物保护单位。盛唐疏勒人入此礼佛，沿城北一道戈壁路而行——今由喀什市出发，仍可循近同一道路到达。

### 喀什噶尔河（克孜勒苏河，唐人称谓未一定）

城南、城东。自葱岭东麓出谷，绕城南、城东而过，最后没入塔里木盆地西缘。是疏勒绿洲与周边屯田所恃的母亲河——盛唐时引此水灌田，种麦、粟、瓜、葡萄。**克孜勒苏**意"红水"——盖因河水含泥沙呈红褐色，葱岭山中流出后混冲积扇红土而成；唐人对此河有无定名史料未明，盖当时只笼统记"疏勒城南有水自葱岭出"。

### 玉与玛瑙作坊

城外山麓。盛唐疏勒最具特色的一项物产——**玉**（与和田玉是否同源史料未明）、**玛瑙**——皆自葱岭东麓山中产出。城外当有切玉、磨玉、磨玛瑙的作坊——但具体位置、规模史料未明，**本 doc 不强凿**。《新唐书·卷二二一上·西域传上·疏勒》"出玉、玛瑙"——是疏勒物产唯一明确入正史的两项。

### 葱岭守捉（城西）

疏勒西约百里（具体里数史料异文；今乌恰县境一带）。盛唐设守捉一处，驻兵数百——是疏勒西出葱岭、通护密 / 识匿 / 吐火罗的第一站。**自葱岭守捉再西**，便出大唐羁縻州境，是中亚河中、大食、拜占庭一道。

### 高仙芝西征基地（疏勒前进基地）

城东郊（具体位置史料未明）。**天宝六载（747）**高仙芝平小勃律——以疏勒为前进基地、出葱岭、翻坦驹岭天险——是盛唐军威西达葱岭之外的最远一仗的西征起点。**天宝十载（751）**高仙芝伐石国 / 怛逻斯——亦以疏勒为前进基地、西出葱岭。**疏勒是两次西征的西部根脚**——本 doc 此项作 doc 内 section，**怛逻斯本身另立 `../relics/怛逻斯.md`**（与安西大都护 doc 一致）。

## 历史长河（755 → 今）

**中晚唐**（755-907）

天宝十四载冬安禄山反——封常清自安西东调入关、十二月十八日（公历 756 年 1 月 24 日）与高仙芝同诛于潼关。安西镇兵八千随其东出——疏勒城下兵力是否亦在抽调之列史料 sparse；但**安西四镇此后陷入与中原音信渐疏的孤悬境地**，疏勒亦同此命。

757 年后，肃宗于灵武自立——朝廷自顾不暇，西陲诸镇音信渐疏。**安西四镇仍由副都护、留守节度使继守**——李栖筠、郭昕等先后在任。郭昕（郭子仪侄）守安西四十余年——自大历元年（766）起独立支撑——与中原隔绝，仍坚守唐之正朔。吐蕃乘**回纥与唐贞元三年（787）的会盟期**间隙南北围攻——**北庭都护府陷**（治所庭州）、**安西四镇相继陷**。**疏勒陷蕃的具体年份史料异文**——旧唐书系约**贞元六年（790）**前后随安西诸镇而失；新唐书与一些近世考据则系略晚（约 791-792 间）——通行说法采**约 790-791 年间**。**这与于阗、龟兹陷蕃同期**——安西四镇约在两三年内相继失守。这一年前后正是郭昕守安西的第二十五年。

吐蕃陷疏勒后，将其纳入吐蕃节儿（驻军长官）治下——但疏勒王统是否因此中断史料未明。吐蕃在疏勒的统治延续约六十余年，至会昌、大中间（840s）随吐蕃王朝瓦解而解体。

9 世纪中叶吐蕃势衰后，西域诸国格局重整。**漠北回鹘汗国 840 年为黠戛斯所灭**——其部西迁；**约 9 世纪后期**，回鹘一支据疏勒一带——疏勒进入西域回鹘势力范围。同时，西边自费尔干纳、河中一带**喀喇汗朝**（黑汗王朝）兴起——**约 10 世纪中后期**，喀喇汗朝控制疏勒——是疏勒**伊斯兰化的开始**。

**五代至宋**（907-1279）

喀喇汗朝是西域第一个伊斯兰化的突厥语王朝。**约 10 世纪后期**（具体年份史料异文）**喀喇汗朝苏图克·布格拉汗皈依伊斯兰教**——以**喀什噶尔**为东都之一——"喀什噶尔"（突厥语 Kashgar，意未明定）自此成为这座城的新名。**约 11 世纪初**，喀喇汗朝以喀什噶尔为基地东征——**约 1006 年**（一说 1004-1008 间，史料异文）灭于阗、东进至塔里木盆地东缘——西域大乘佛教自此自西向东渐次覆灭。**疏勒先于于阗、先于龟兹完成伊斯兰化**——是西域诸大绿洲城邦里最早的一处。**鸠摩罗什时代以来佉沙的伽蓝、玄奘讲经的那座大寺、莫尔佛塔的香火、城里万余僧徒——皆在这一两个世纪里悄然散尽**。

喀什噶尔在喀喇汗朝时期成为一座重要的伊斯兰学术中心——**11 世纪后期**，喀什噶尔人**马赫穆德·喀什噶里**编纂《突厥语大词典》（约 1072-1077 间成书），是突厥语世界第一部大型辞书；同时期**玉素甫·哈斯·哈吉甫**作《福乐智慧》（约 1069-1070）——两书皆出于喀什噶尔的学术圈——**注：两书写于喀喇汗朝中期，与盛唐疏勒已隔三百多年；只作"后世意义"了解**。

**蒙元至明清**（1227-1911）

13 世纪初，蒙古西征——喀什噶尔归察合台汗国。**元代称"可失哈儿"**（Kashgar 蒙古语音转）。明代喀什噶尔归亦力把里（叶尔羌汗国前身）；**16-17 世纪间**喀什噶尔成为叶尔羌汗国的两个中心之一（另一为叶尔羌 / 莎车）。清乾隆二十四年（1759）平大小和卓——**喀什噶尔正式入清版图**——置喀什噶尔参赞大臣（**仅次于伊犁将军、乌鲁木齐都统的西域三大军政中心之一**），属伊犁将军辖。光绪十年（1884）新疆建省——喀什噶尔设直隶州、设道。**自盛唐疏勒陷蕃（约 790）至清复入版图（1759），中间整整 969 年**——这一带不属中央王朝。

**民国至今**（1911—）

民国二年（1913）废喀什噶尔道；今属新疆维吾尔自治区**喀什地区**，地区行署驻**喀什市**——是南疆第一大城。盛唐疏勒王城具体位置——史界主流认为在今喀什市区一带（一说艾提尕尔清真寺周边、一说喀什老城西侧），**夯土城垣早已散尽，地表已不可辨**。**莫尔佛塔**（城北约 30 公里）今仍为遗址中保存最好的一项——盛唐疏勒佛门的具体物证。喀什噶尔河（今克孜勒苏河）至今仍是这座城的水源——河滩边的玛瑙、玉，自汉至今**两千余年间断续有人捞取**——**注：克孜勒苏河捞玉传统是否完全延续不间断史料未明；至少今仍有玉石、玛瑙交易市场在喀什老城**。

至于盛唐那座**疏勒镇城**本身——**夯土城墙今几不存，雉堞早不可辨**；王宫、镇衙署、大寺的位置只能由文献推定。**喀什老城**的曲折土巷、伊斯兰风格民居与艾提尕尔清真寺，是 10 世纪喀喇汗朝以后逐层覆盖在唐代疏勒之上的——盛唐佉沙、唐疏勒、回鹘喀什噶尔、喀喇汗朝喀什噶尔、清代喀什噶尔、今日喀什——**六层城市叠在同一片绿洲之上**。

玄奘《大唐西域记·卷一二·佉沙国》还在书架上摆着；莫尔佛塔的覆钵塔基还在城北郊外的戈壁上立着——**距盛唐已逾千二百年**。

### 安西四镇里疏勒的"未被焉耆 / 碎叶 替代过四镇位次"特征

**疏勒是安西四镇里唯一在每一次反复中皆未被焉耆 / 碎叶 替代过四镇位次的一镇**——焉耆与碎叶曾互替，龟兹做都护治所所在故格外重要，于阗自治权最大；疏勒之"稳"，盖因其地理位置**葱岭东口**——高仙芝两次西征皆经此 / 多以此为前进基地，朝廷不可弃。

### 玄奘在佉沙的约一月（贞观十八年 / 644 春）

| 阶段 | 事件 |
|---|---|
| 贞观十八年（644）春 | 玄奘自吐火罗东行，过葱岭，抵佉沙——王率众迎入 |
| 同年春 | 玄奘挂单佉沙大寺——与城中小乘说一切有部诸僧论辩 |
| 同年春末 | 玄奘自佉沙东行——经乌铩（今莎车）、于阗 |

**约一月**——是玄奘西行十七年里在一国停留时间中等的一段（远短于于阗之七至八月；具体停留日数史料异文，本 doc 加 hedge）。**《大唐西域记·卷一二》记佉沙独详**——盖因疏勒于盛唐西行人而言，是"出葱岭后归唐的第一座大城"——其重在地理位置而非停留之久。

### 疏勒物产：玉与玛瑙的疑点

- **玉**：疏勒之玉与和田玉是否同源？《新唐书》只笼统记"出玉"——未明指产地、品质、是否软玉（透闪石玉）。和田玉以白玉 / 墨玉 / 青玉为主，自玉龙喀什河、喀拉喀什河两河出；疏勒一带玉产地史料未明——一说葱岭东麓某山中、一说克孜勒苏河上游。**本 doc 不强凿**。
- **玛瑙**：是疏勒物产中最具特色的一项——《新唐书》明确"出玛瑙"，是西域诸国中唯一明记"出玛瑙"者。盛唐玛瑙之贡入长安——多自疏勒、波斯（萨珊朝灭后由大食转输）两路来。
- **盛唐疏勒玉、玛瑙年贡数量、贸易量——史料无明载**。本 doc 不凿。
