# 道州

## 一句话

江南西道道州治所，治弘道县（今湖南永州市道县）——潇水中游、五岭萌渚岭北麓的湘桂边小州，**755 时尚是江华郡寻常中州，元结刺道州尚在八年后**。

## 形貌

道州城坐在**潇水中游西岸**一处河谷台地上——潇水自南五岭萌渚岭北麓出，绕城东北流，下流至永州城北汇入湘水。城周不大，按盛唐州格属**中州**——夯土城垣依潇水西岸地势而立，城南、城西即萌渚岭北麓余脉，林木以樟、楠、栎、竹为主，山间多葛、麻、药草。城北约二百唐里至永州治零陵县，城南越萌渚岭便入岭南道贺州、连州境——**道州正在湘桂边、岭北通衢的最南一段**。物候属湘水流域最南端：春来甚早、雨多、夏热而冬罕霜雪——比永州更近岭南。

## 大唐天宝十四载（755年）

道州沿革几度迁名：本为**梁永阳郡兼营州故地**——梁置营州、永阳郡并存于此潇水中游一带；**隋平陈后废营州、永阳郡，入永州**——隋一代此地只是永州下营道一县。**唐武德四年（621）**唐平萧铣，于营道县**置营州**；**武德五年（622）**因避河北道营州（治柳城，今辽宁朝阳）重名，改**南营州**；**贞观八年（634）**再改**道州**——以本地"营道"之"道"得名。**贞观十七年（643）**省道州入永州——道州一度废而无名。**上元二年（675）**高宗复置道州——这一回的"道州"才是后世沿用之名。**天宝元年（742）**玄宗诏天下州改郡——道州又改**江华郡**——这是 755 年当年的官称；同年改州治营道县为**弘道县**。**乾元元年（758）**肃宗复改郡为州，江华郡再称道州——但 755 年这一刻，城门匾额上写的还是**"江华郡"**。

户口上，《旧唐书·地理志》载道州（江华郡）为**中州**——天宝领**户二万二千五百五十一**、**口一十三万九千六十三**，体量在江南西道诸州中属中等，不及永州二万七千余户但远过寻常下州。《新唐书·地理志》亦作"中"。755 年的江华郡领县**四**——**弘道、江华、永明、营道**，其中弘道为郡治。**大历县乃大历二年（767）后所置**，755 当年尚未有。本郡长官 755 当年称"江华郡太守"。州内官署即郡衙、弘道县治、馆驿数所——一套中州小郡的简单班子。

道州属**江南西道**——江南西道治洪州（今南昌），辖洪、江、鄂、岳、潭、衡、永、道、郴、邵、桂等州。755 年时江南西道未常设节度，由采访使分巡——道州太守归江南西道采访使节度。

755 年冬安禄山反范阳——消息沿驿道南传至道州，约需月余有余。但**道州偏处岭北最南、潇水中游山间，本不在乱事主路上**——次年玄宗西奔入蜀，未经道州境；中原士人南奔者多取永州过岭入桂州一线，道州不当通衢主路，只偶有过境。**这是 755 之后的事，盛唐人并未预知**。

## 历史长河

**中晚唐**　乾元元年（758）复改江华郡为道州。**广德元年（763）至永泰元年（765）**，**元结**（字次山，719-772，河南鲁山人）以道州刺史出任——元结到任时正值安史乱后湘南残破，西原蛮（今广西境内俍人）数年间寇道、永、邵诸州，道州户口由开元间约四千余落至千余户。元结在任作**《舂陵行》《贼退示官吏》《道州刺史厅壁记》《右溪记》**等数十篇——《舂陵行》《贼退示官吏》以散体诗记乱后道州民生凋敝、官府征敛之苦，杜甫读后作《同元使君舂陵行》和之，称元结"知民疾苦";《右溪记》记道州城北潇水之滨小溪——是中唐山水小品的开山一篇。**道州因元结而入中国文学地图，与永州之于柳宗元相先后**——元结刺道州在 763-765，柳宗元贬永州在 806-815，相距四十年。

晚唐五代道州仍属江南西道、楚（马氏）相继。**北宋大中祥符（一说天禧）间**——周敦颐生于道州营道县楼田保（今湖南永州市道县清塘镇楼田村），**周敦颐（字茂叔，1017-1073）**——北宋理学开山，著**《爱莲说》《太极图说》《通书》**等；二程（程颢、程颐）皆从其学，为宋明理学远祖。**道州因此而成理学发源地之一**——但这是 755 之后两百六十年的事。

**宋元明清**　北宋道州属荆湖南路；元置道州路；明洪武初降为道州，属永州府；清沿明制。**明清两代道州属永州府**——府治零陵，下辖零陵、东安、道州、宁远、永明、江华、新田七州县。**清代何绍基**（号子贞，1799-1873）为道州人——晚清书法、诗文大家，与道州周敦颐隔八百年遥相先后。

**民国至今**　民国二年（1913）废府州——**道州改道县**，属湖南省。1949 年后属湖南省零陵专区、零陵地区；1995 年撤地设永州市后属永州市，今为永州市道县——治所约在唐弘道县旧址上。**盛唐州城残基已埋入今道县城下，元结《右溪记》之"右溪"今仍在道县城北潇水之滨**，为湖南省文物保护单位；**周敦颐故里今为道县清塘镇楼田村**，存周敦颐故居、濂溪书院遗迹。"江华郡"三字，只在《旧唐书》《通典》和那一段元结刺道州、周子起濂溪的乡邦记忆里仍可见。

## 引文

> 道州　隋零陵郡之营道县。武德四年，平萧铣，置营州，领营道、永阳、江华、永明四县。其年，又置营浦县。五年，置营山县。贞观八年，改为道州。天宝元年，改为江华郡。乾元元年，复为道州。旧领县四，户二万二千五百五十一，口一十三万九千六十三。

——《旧唐书·卷四〇·地理志三·江南道·道州》
