# 登州

## 一句话

河南道东端胶东半岛北角的**中都督府**，742-758 年间名"东牟郡"——隔黄海与新罗对望，**新罗使臣海路入唐的登陆港**。

## 大唐天宝十四载（755年）

登州治**蓬莱县**（今山东烟台市蓬莱区），属河南道。**唐如意元年（692）分莱州置登州**；神龙三年（707）废，**开元二年（714）复置**——州境在胶东半岛北端，三面临海，南接莱州，北望渤海。**天宝元年（742）改称东牟郡**，乾元元年（758）复登州——755 当年的官号是"东牟郡"，俗称仍呼登州。

**等第为中都督府**——《新唐书·卷三十八·地理志二》登州东牟郡条：「**登州东牟郡，中都督府。本东莱郡之牟州，武德四年以文登县置。如意元年析莱州之黄、牟平、文登置，治牟平。神龙三年徙治蓬莱。土贡：金、石器、水葱席、赀布、海藻、文蛤、牛黄。户二万二千二百九十八，口十万八千九。**」——以户口论，登州户 22,298、口 108,009（《旧唐书·地理志》载天宝户 20,298、口 108,900，互有出入），在河南道诸州之中属**中州偏上**，并非"下州"。州境领蓬莱、黄、牟平、文登四县。城在半岛北岸一处可泊船的湾里——东南是丘陵，北面便是黄海。

登州在 755 年最重的分量，不在户口，而在**海路**。**唐与新罗的官方往来主要走海路**——新罗使臣自其都金城（今韩国庆州）出发，沿半岛西海岸北上，自瓮津或南阳湾横渡黄海，**在登州或楚州登陆**，再陆行经河南道入两京。九世纪日僧圆仁《入唐求法巡礼行记》记 838 年至 847 年间他在登州、文登、赤山往返的细节——唐人和新罗、日本三方海客都以登州为入唐第一站。

蓬莱之名在登州更早——秦皇汉武东巡求仙，"蓬莱、方丈、瀛洲三神山在渤海中"（《史记·封禅书》），徐福东渡的传说亦系于此一带海岸。但**蓬莱阁（北宋嘉祐六年/1061 始建）此时未立**，登州城外只是一座面海的低丘与几处渔村。

## 历史长河（755 → 今）

中晚唐登州为新罗、日本入唐主要海港之一——**唐穆宗长庆四年（824）新罗人张保皋于文登县赤山立法华院**，登州一带成为东亚海路佛教与商货的枢纽（此为 755 之后近七十年事）。北宋登州属京东东路，**嘉祐六年（1061）始建蓬莱阁**——盛唐人未见此阁。金、元属山东东路、般阳路。明洪武九年（1376）设登州卫，**洪武年间筑蓬莱水城（备倭城）**——戚继光生于登州。清雍正十三年（1735）升登州府。民国废府；1983 年蓬莱县属烟台地区；2020 年撤蓬莱市设烟台市蓬莱区——盛唐登州城圈大体在今蓬莱阁山下旧城一带。
