# 钦州

## 一句话

岭南道邕管（一说容管）下辖一座下州，治钦江县——北部湾北岸的海口州城，宁氏世袭俚帅之地，盛唐辖境最南海岸上一处远州。

## 形貌

钦州城在钦江下游近海一段的台地上——州治**钦江县**（故城在今广西钦州市钦南一带，按"钦州市"为 1994 年始设地级市的现代名号——1965 年仅为钦州专区，1994 年钦州地区改置地级钦州市；盛唐尚不存在此称）。城周不大，依江岸铺开；北、西二面是十万大山余脉与岭南山脊相接，南面则是**北部湾**——盛唐人通称"南海"或"涨海"。钦江自西北山中来，一路南下至州治附近分作数支，再东南入海——海口在钦州城东南数十里外。

形势是一城而见三向：**北、西为山，蛮獠诸部世居其间**——西原蛮、黄洞蛮的活动范围伸及钦州西、北面；**东北循陆路过廉州、雷州、可达广州**——是钦州接入岭南腹地的唯一陆路通道；**南面循海可至交州、林邑**——是盛唐海上丝路一处不大不小的中途港。一州而处三界——这是钦州在盛唐的根本形势：海港、陆驿、羁縻俚帅治地三合一。

物候是大唐辖境最南的一类：冬无冰、夏极热、五月至九月一段雨水连绵、瘴疠时作。城内官员多自中原来——岭南诸州素以瘴重著名，钦州尤偏南，贬官南来死于此地者亦不绝；正史载唐时贬至钦州者有数人。

## 大唐天宝十四载（755年）

钦州沿革在唐人眼里清楚。**隋开皇十八年（598）置钦州**于此——治钦江县，"钦州"二字因钦江得名，自此立。**大业三年（607）**炀帝行州改郡通制，改钦州为**宁越郡**——治钦江县。**唐武德五年（622）改宁越郡复称钦州**——治钦江，属岭南道。**天宝元年（742）**玄宗诏改州为郡，钦州又改回**宁越郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州——钦州之名再立。**742 至 758 复州前正名"宁越郡"，民间口语仍呼"钦州"**——两名并行。

755 年的钦州属**岭南道**，岭南五府经略下辖——具体节制属**邕管**还是**容管**，两唐书口径差：**《旧唐书·地理志》载钦州属邕管十州**（邕管治邕州，今南宁一带）；**《新唐书·地理志》载钦州属容管十州**（容管治容州，今广西北流一带）。两书并存，本 doc 加 hedge 不强订——钦州即为邕管/容管所辖南端州治之一。《旧唐书·地理志四》记天宝间钦州**下州**——户口稀少，远州一座；具体户数 codex 提示"户口少"，正史载约数千户上下（《旧唐书》《元和郡县图志》记数互见，本 doc 不取具体数字而以"下州、户口少"为定性）。辖县数：**钦江**（州治）、**安京**、**遵化**、**内亭**等数县（按：**安京**为 755 时态正名；**至德二载（757）改安京为保京**——保京为后起名，755 尚未出现）——皆钦江下游及周边山地县治。城内有宁越郡治、**太守**署、钦江县衙、馆驿——一座下州的标准班子（按：唐州长官正名为"刺史"，742-758 改州为郡间则称"太守"；755 时钦州官称为"宁越郡太守"，本 doc 主体守 755 时态）。

钦州在盛唐另有一桩特殊处——**宁氏世袭钦州刺史数代**。**宁氏**本是钦州、廉州一带俚人豪族——**隋末大业年间宁猛力据钦州一带自雄**（约隋末唐初人，具体生卒年史料异文，本 doc 不强订）；**唐初武德年间（618-626）宁长真归附唐廷**——为高祖朝岭南俚帅实际归附主体，受唐封钦州刺史，唐初参与征林邑之役（具体年代加 hedge：见《新唐书·南蛮传》载，年代史料异文）。**宁长真与宁猛力是否父子或同族异说存在，史源不一**——《新唐书·南蛮传》行文似作父子，但两人活动年代与归附背景各异，后世学者对直系世系存疑。**宁纯、宁原悌、宁道明**等为宁氏后辈人物，世袭钦州刺史至唐中期——具体世系存异说，本 doc 不强订排序。**宁氏属唐廷羁縻州刺史**——名义上是中央任命的下州刺史，实际是世袭俚帅。岭南另一系**冯氏**——以**冯盎**（唐初岭南豪族，本为高凉冯氏 / 高州冯氏，势力远及钦州外围）为代表，与宁氏并称岭南两大俚帅，盛唐前期共撑岭南羁縻—直辖混合体制。**755 年钦州在任太守正史无明载**——可推知者多为宁氏后人或朝廷遥派，史料缺漏；这是盛唐西南海陲下州史料的常态。

钦州的**海上贸易**——盛唐时已有海舶往来钦州港，与**林邑（占婆，今越南中部）、真腊（今柬埔寨）、室利佛逝（苏门答腊）**诸国通商，转运香料、犀角、明珠等南海物产。但钦州在盛唐海上丝路上不是主港——主港是广州、交州、扬州（一说明州）三处；钦州只是一处中途港或辅港，**具体航线与广州、交州的分工，史料语焉不详，本 doc 不强订**。

天宝十四载冬安禄山反范阳——消息沿驿道自长安南传，过商於、襄州、江陵、潭州、衡州、桂州、邕州，再南至钦州——抵钦州约需四十余日。宁越郡太守接到的诏令大约与交州相仿：戒严、守城、备粮，并防境内蛮獠乘隙——但岭南再南距河朔战场过远，朝廷并未抽调钦州兵北上。城里没有大动静，海上的胡舶尚未察觉北方大乱；钦江上的渔船、海口的商舶，一如盛唐前一百年的模样。

## 历史长河

**中晚唐**（755-907）

至德二载（758）肃宗复改宁越郡为钦州——名号回到武德旧称。**宁氏世袭钦州刺史一脉至唐中期渐衰**——具体何时终结史源不一，《新唐书·南蛮传》载宁氏与唐廷间多有反复，至**唐宪宗、文宗朝**间钦州宁氏始不再见于正史。中晚唐钦州随邕管起伏——**咸通元年（860）**南诏军北上攻陷安南、波及岭南西部，钦州境内蛮獠亦时有骚动。**咸通七年（866）高骈收复安南**后，钦廉一线复安。**唐末黄巢之乱**（878-884）波及岭南，钦州随广州、桂州、邕州一线动荡——但作为下州，钦州始终是岭南边缘上的小角色。

**五代两宋**（907-1234）

五代之际钦州随岭南诸道入南汉（917-971）——南汉刘氏据岭南五十余年，钦州为南汉钦州。**北宋开宝四年（971）平南汉**——钦州复入中原王朝直辖。**南宋时钦州升钦州军**（绍兴间），仍治钦江——这一段是钦州在两宋间的地位高点：作为对越（独立后越南李、陈朝）前线州军，钦州与廉州、雷州一线共撑岭南南端。**北宋熙宁九年（1076）**郭逵伐安南，钦州、廉州为北路宋军出发地之一——州城一度承担前线指挥的重责。

**元明清至今**（1271—）

**元至元间废钦州军**——降为**钦州路**，后又有**钦州**、**钦州府**、**钦州直隶州**等名号反复，治所大致沿钦江县旧址。明清两代钦州属广东省（按：自明洪武起钦廉一带划入广东布政司，至清沿袭；钦廉划归广西省是民国后事）。**1912 年废钦州府置钦县**；**1965 年中央批准设钦州专区，1994 年钦州地区改为钦州市（地级市）至今**——"钦州市"作为地级市名号为 1994 年始设（1965 仅置专区），盛唐尚不存在此称。今广西钦州市治所，约在唐钦州钦江县旧址一带。盛唐州城残基已埋入今钦州市城下；钦江仍流，注入北部湾——海口、海港之名一直未改。"宁越郡"三字只在《隋书》《旧唐书》《元和郡县图志》里仍可见；宁氏一族散入岭南俚人后裔，姓氏延续而世系不可详考。

## 引文

> 钦州，宁越。下。隋开皇十八年置。大业三年改为宁越郡。武德五年复为钦州。天宝元年改为宁越郡。乾元元年复为钦州。

——《旧唐书·卷四一·地理志四·岭南道·钦州》（取意节录；具体行文以原书为准）。这是 755 年钦州正名"宁越郡"前后官称沿革的简明实录。

> 宁猛力，钦州人也……世为南越渠帅。唐武德中归附，授钦州刺史。子长真，复继之。

——《新唐书·卷二二二下·南蛮传下》（取意节录）。宁氏世袭钦州刺史至唐中期——是钦州在盛唐羁縻—直辖混合体制下最具特色的一笔。具体年代史料异文，本 doc 不强订。
