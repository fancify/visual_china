# 吉州

## 一句话

江南西道赣江中游一上州——治庐陵县（今江西吉安一带），洪州正南、虔州正北，是赣江干流自洪州南上越大庾岭线上的中段州治。

## 大唐天宝十四载（755年）

吉州治**庐陵县**，在赣江中游东岸。州在洪州正南、虔州正北——自洪州沿赣江南上，过吉州、虔州，至大庾岭舍舟登陆，便是岭南北上的水陆主轴。**吉州正当此轴之中段**——纲船、客舟、岭南北归者皆经此歇舟。州属**江南西道**，**天宝元年（742）随玄宗朝改州为郡通制改称庐陵郡**，755 时正名为"吉州庐陵郡"，两名互换不分；**乾元元年（758）肃宗复改为吉州**——是 755 之后事。

沿革更早：隋开皇十年（590）平陈后置**吉州**，大业三年（607）改为**庐陵郡**；唐**武德五年（622）平萧铣后复置吉州**——"吉州"作为唐州名自此而立。**《新唐书·卷四一·地理志五》载："吉州庐陵郡，上。土贡：丝葛、纻布、陟釐、斑竹。户三万七千七百五十二，口三十三万七千三十二。县五。庐陵、太和、安福、新淦、永新。"** 《旧唐书·卷四十·地理三》同载："旧领县四，户一万五千四十，口五万三千二百八十五。**天宝领县五，户三万七千七百五十二，口二十三万七千三十二**"——两书户数全合（37,752），口数微有出入（旧 23.7 万、新 33.7 万），盖传写之异；以**户 37,752、领县五**为 755 时确数。**唐制以户数定州等：上州户四万以上**——吉州户三万七千余，紧贴上州线，《新》《旧》两书皆列**上州**，是江南西道户口仅次于洪州的一州。

治县庐陵，**汉置庐陵县**（《汉书·地理志》豫章郡下有庐陵县）——县名因境内庐陵山（一说庐水）得名，至 755 时已沿用九百余年；唐人提吉州必带"庐陵"二字，二名互通。州境**赣江中游**——江自南面虔州方向北流，过吉州城外，再北至洪州入鄱阳湖；庐陵县城贴江而立，码头是岭南北上纲船歇舟换粮的中段大埠。

军政上吉州无大兵无大事——江南西道腹地承平已久，军务远不及赣江漕运、土贡（丝葛、纻布、陟釐、斑竹）为重。755 当年的吉州太守姓名于史阙载。

## 历史长河

**中晚唐**　安史乱后江南西道升为重镇，吉州随江南西道节度（治洪州）。唐末黄巢之乱江南西道一度沦兵，吉州亦罹。五代属杨吴、南唐。

**北宋至明清**　北宋庐陵县出**欧阳修**（1007-1072，吉州永丰人）——文坛领袖自称"庐陵欧阳修"；南宋吉州又出**杨万里**（1127-1206，吉水人）、**文天祥**（1236-1283，庐陵人）——皆吉州人。后世归纳"庐陵文化"一说，自欧阳修以下方成气候——盛唐 755 时尚无此概念。元至元十四年（1277）改吉州为**吉州路**，元元贞元年（1295）升为**吉安路**——"吉安"二字自此而立，取"吉泰民安"之义。明洪武元年（1368）改吉安路为**吉安府**——府治庐陵县，明清两代皆然。

**民国至今**　1912 年废府存县，庐陵县改名**吉安县**。今为江西**吉安市**——1979 年置县级吉安市，2000 年升地级市，辖吉州区、青原区诸县。今吉安市区与古庐陵县城方位大体相承——赣江仍自南北流过城外，与盛唐吉州城所对的那段江水是同一段。
