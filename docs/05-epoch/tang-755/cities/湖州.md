# 湖州

## 一句话

江南东道吴兴郡治，治乌程县——太湖南岸的一座旧州，与苏州隔湖相望；春秋谓之菰城，三国孙吴置吴兴，盛唐时是江南诸州中产丝、产茶、出文士的一档上州。

## 形貌

湖州城卡在**太湖南岸、苕霅二溪入湖处**的一片低洼平原上。

城本身不大——周回数里，依**苕溪**（西苕溪、东苕溪在城南合流）北行入太湖的水口而立。州治**乌程县**——城周筑夯土，开门数处通水陆；城外是太湖南岸特有的圩田、桑园、水网，纵横河汊把整片平原切成无数长方田块。**城北十数里便是太湖**——湖面浩渺，北望对岸隐隐是苏州吴郡——两州隔湖相望，舟楫往来不过一日。城南是天目山东北余脉，山不高，但**顾渚山**便在城西南八十里——后来的贡茶产地。

物候是江南偏北的：春雨多、梅季湿、夏热、秋爽、冬不甚寒。城外圩田一年两熟，桑园密布——蚕桑、丝织是州中第一产业。湖州一带的**绫**与**绢**，盛唐时已入贡长安。山中产茶——顾渚一带的茶树盛唐时已为山民所采，是否已成"贡茶"规制，盛唐文献未明（详见历史长河）。

## 大唐天宝十四载（755年）

唐人眼里湖州的来路清楚而不算极古。最早的传说是**春秋楚考烈王封黄歇于此，号"菰城"**——湖中多菰（茭白）故名；但菰城是否即今湖州治所，史料异说，姑存其名。秦置**乌程县**于此——"乌程"二字自此立，相传因春秋时乌氏、程氏两家善酿酒得名，但此说出自后世《吴兴记》一类地志，盛唐人已不能确证。汉因之，乌程属会稽郡，后归吴郡。

**三国吴宝鼎元年（266）孙皓分吴郡、丹阳郡置吴兴郡，治乌程**——"吴兴"二字作为郡名自此立，沿至两晋南朝。**隋开皇九年（589）平陈废吴兴郡**，乌程并入苏州；**仁寿二年（602）于乌程置湖州**——"湖州"二字始见，取太湖南岸之意。**大业三年（607）**炀帝行州改郡通制，湖州又改回**吴兴郡**。**唐武德四年（621）**李渊定江南复改吴兴郡为湖州；**天宝元年（742）**玄宗诏天下州一律改郡，湖州再改**吴兴郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州——湖州之名再立。**742 至 758 复州前，本州正名"吴兴郡"，民间口语仍呼"湖州"**——两名并行。

755 年的湖州属**江南东道**，州治在乌程县，辖**乌程、武康、长城、安吉、德清**等五县（开元、天宝间县名小有出入）。**《旧唐书·地理志三》记天宝吴兴郡户七万三千三百零六、口四十七万七千六百九十八**——上州门槛之上，江南诸州中上之数；户口不及苏州、杭州，但高于同道一般州。州内丝织、米、茶、笔（**湖笔**之乡的诸暨/善琏笔工传说，宋以后方坐实，755 时未必已有名号）皆为名产。江南运河自苏州南行至杭州，从湖州东北擦边而过——湖州本州不在运河主干上，但**苕霅二溪与太湖、运河相通**，州中漕粮、丝绢一样可北运。

755 这一年吴兴郡的太守史载不甚详。**沈约**（441-513）是吴兴武康人——南齐至梁的大文学家、《宋书》作者，吴兴沈氏世家之祖——但沈约是六朝南齐时人，距 755 已二百四十余年；唐时州中沈氏后裔尚存，州人提及沈氏皆为本郡荣光。盛唐之吴兴，是一座**旧得有底子、富得不张扬**的中州——它的全盛要等到中晚唐颜真卿、皎然、陆羽、张志和这一群人聚到这里之后。

天宝十四载冬安禄山反范阳——消息南渡过江、循江南运河经苏州西传至湖州约需一个月。湖州不当兵冲，但作江南腹地的丝米之乡，安史乱后即开始向朝廷大量输送漕粮——盛唐的吴兴，是江南漕运体系的一处后方仓。

## 历史长河（755 → 今）

**中晚唐**　乾元元年（758）复州名为湖州（吴兴郡旧名废）。中唐起湖州始入诗史与茶史：**大历八年（773）颜真卿任湖州刺史**——颜公在湖州任内召集**皎然、陆羽、张志和**等文士结"湖州文会"，刺史厅事修"竹山堂"，编《韵海镜源》——湖州文坛之盛由是天下闻。**陆羽《茶经》**约成书于此期（上元至大历间），书中所记**顾渚紫笋**茶（湖州西南顾渚山所产）入贡始于代宗大历年间——**这是顾渚紫笋作为"贡茶"的最早可考年代；盛唐 755 时顾渚一带是否已采茶入贡，史料未明，故本 doc 主体不预支**。中晚唐顾渚成为朝廷专设的**贡茶院**所在——每岁清明前急递入长安，是唐代最有名的贡茶之一。

**宋元明清**　北宋仍称湖州，属两浙路。南宋因临安（杭州）为行在，湖州为畿辅之一——丝织、湖笔（**善琏湖笔**自宋元间始有名号，明清成天下第一）、湖绫之名于此期定型。元改湖州为湖州路。**明洪武元年（1368）改湖州路为湖州府**，治乌程、归安二县。清承明制，湖州府属浙江布政司。明清两代湖州府以**湖笔、湖丝、菱湖蚕、顾渚茶**著名——文房四宝中的"湖笔"自此成为天下第一。

**民国至今**　1912 年废府，乌程、归安二县合并为**吴兴县**——"吴兴"二字再回到县名（取唐吴兴郡旧名重立）。1981 年吴兴县撤销，置湖州市。今日湖州市治所**仍在唐乌程旧址上**——这是太湖南岸少见的连续一千七百年同一城址（自三国吴宝鼎元年 266 置吴兴郡算起）。盛唐那座吴兴郡——州城残基已埋入今湖州市下，乌程县衙旧址不可确指；但**苕溪仍北流入太湖，顾渚山仍在城西南**，两州（湖州、苏州）隔湖相望的格局，与天宝十四载那一年大体未变。
