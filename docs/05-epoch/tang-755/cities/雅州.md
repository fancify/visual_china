# 雅州

## 一句话

剑南道西南边角的一座下州，742-758 年间名"卢山郡"——青衣江穿城而过、蒙顶山在城北、西出便接吐蕃东境与南诏北界，是唐人入剑南最西南的一处官治。

## 形貌

雅州城坐在邛崃山东麓、青衣江畔的一片盆地里。

州治在**严道县**（今四川雅安市雨城区一带），城周不大——比成都、梓州小得多，约三四里许。**青衣江**自城西北的邛崃山间流来，绕城东南折，再东去经眉州境入岷江——城就摊在江北岸的一片台地上。城外四望：北有蒙顶山一带的丘陵（蒙山主脊海拔约一千四百米，城北二十里许可望），西北是邛崃山主脊（再过去便是吐蕃东境），西南越大相岭可入嶲州（今四川西昌一线）、再西南便是南诏的北部边境。盆地中央是青衣江冲积出的稻畦——气候比成都湿、比梁州冷，**云雾常合在山腰**——蒙顶山一带"雅雨、雅鱼、雅女"的传称是后世积出来的，唐时只见雨多、雾重、茶树青。

雅州物产，茶最为人称——蒙顶山一带自西汉起便有植茶之记（旧志传汉甘露年间僧人吴理真植七株于蒙顶五峰之顶，是为蒙顶茶始祖；按：吴理真其人为后世传说人物，此说出自宋元方志，唐时记载尚略，盛唐人入此见茶园而已，未必知此源）。雅州茶在唐时已贡——《元和郡县图志》明记雅州土贡茶。盆地物产之外，邛崃山段的麝香、漆、药材，亦循青衣江入眉州、再入成都。

## 大唐天宝十四载（755年）

雅州在唐人眼里是**剑南道西南边角的一座下州**——户口不多，但位置紧。

沿革上：隋开皇十三年改蒙山郡；仁寿四年（604）置雅州；大业三年（607）改**临邛郡**——按隋制以郡领县，雅州一度并入临邛郡的辖区，治严道。唐武德元年（618）复**雅州**，置都督府辖周边羁縻数州。天宝元年（742）从全国诸州改郡之例，改作**卢山郡**——"卢山"二字取自城西**卢山**（青衣江上游山名，今四川芦山一带）。乾元元年（758）肃宗复郡为州，卢山郡复称雅州——**742 至 758 这十七年间，雅州官号是"卢山郡"**，史志、碑刻、文书同。

户口上，《旧唐书·地理志》载雅州（卢山郡）**天宝间约一万户出头**——下州 / 中下州门槛（唐下州门槛 2 万户以下），雅州户口仅及其半。郡领严道、卢山、百丈、名山、荥经、依政六县（按：列县唐代有微调，755 当年以严道为治县无疑；荥经、名山亦确属雅州，余县据《元和志》《通典》参校）。一郡之治，**州刺史、州司马、州司户参军**一套官署叠在严道城里。雅州为剑南道治下的**边州**——西出便是羁縻州（雅州都督府辖羁縻州数十，多为生羌、生獠及邛部各部落），再西便是吐蕃东境。

755 年的雅州在唐人眼里的紧要，**不在户口，而在地理**：

其一，雅州**正西**循青衣江上游、过卢山、再西越邛崃山——便是吐蕃东境（吐蕃在松州 / 维州 / 黎州一线与唐军对峙）。雅州虽不在松州、维州那一线最前沿（剑南道最前沿是松州、维州、保州），但作为剑南道西南边的次前沿，州兵亦时有警备。

其二，雅州**西南**越大相岭、过嶲州——便是南诏的北部边境。**天宝十载至十三载（751-754）唐与南诏两次大败**：天宝十载（751）剑南节度使鲜于仲通伐南诏，败于西洱河；天宝十三载（754）剑南节度使李宓再伐，全军覆没——这两次大败主战线在嶲州 / 姚州 / 西洱河（今云南大理）一带，雅州作为剑南道入嶲州的中转，是这条战线的**后方补给所**。城里这几年驿马、军粮、伤兵北送的事不少——城里的人对南诏战事的耗费、对剑南节度的怨气，比成都、梓州的人更切身。

其三，雅州**正南**循青衣江出眉州境入成都——是成都西南面的屏障；雅州若失，成都便露出西南半边。**唐人称"雅州为益州西南门户"**便指此。

至于"**茶马古道**"——这是后世（尤其明清之后）通称的概念。盛唐 755 这一刻，**自雅州西出循青衣江—邛崃山道**通吐蕃东境的小道确已开通，亦有零星的茶、马交换记载——但唐人尚未把这一线唤作"茶马古道"。**雅州为茶马互市起点**之说，要到宋元以后**茶马司**正式立制方逐渐成型。本条仅记：755 年这一刻，雅州西去吐蕃东境的小道在，城里产茶、贡茶亦在；至于"茶马古道"这四个字，则是从盛唐这一刻算起约五百年之后才有的称谓。

## 历史长河

中晚唐：广德元年（763）吐蕃东进取河陇之后，雅州西境一带屡为吐蕃、生羌侵扰——剑南西山诸羁縻州在此后百年间反复易手；雅州本城未陷，仍为剑南道治下州。咸通十年（869）南诏世隆攻入成都府，雅州亦遭兵燹——但城未失。

唐宋之间：五代前蜀、后蜀皆置雅州。**宋初置雅州**，治严道——北宋于雅州、黎州一线置**茶马司**，专管"以茶易马"事——雅州自此成为西南茶马互市的正式官治起点。元置雅州属吐蕃宣慰司之一翼；明洪武六年（1373）复置雅州。**清雍正七年（1729）升雅州为府**——治严道县（后世改名雅安县），辖名山、芦山、荥经、天全、清溪、宝兴六县。

民国至今：1913 年废雅州府，置雅安县；1939 年设西康省，雅安一度为省会；1955 年西康省撤并入四川，雅安改属四川省；2000 年地区改市——今**雅安市**，辖雨城区、名山区等八县区。蒙顶山的茶今仍植——蒙顶甘露、蒙顶黄芽是名茶。盛唐"卢山郡"的城址在雨城区下叠压——考古所见唐城遗迹零星，今地表所见多为明清雅州府城残基。**"雅州"二字唐末至清仍用、民国改"雅安"后渐隐**——而"雅安"二字至盛唐 755 时尚不存在，本 doc 不以此名预支。

## 引文

《元和郡县图志·卷三二·雅州》（节）：

> 雅州，卢山。下都督府。开元户一万八百九十三。乡三十。元和户三百三十一。……青衣水在州南二里。蒙山在州北十里。土贡茶。

——唐人对雅州城形最简的一笔：水在南、山在北、贡茶。
