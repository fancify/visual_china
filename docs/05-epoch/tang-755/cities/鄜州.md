# 鄜州

## 一句话

关内道鄜延北部那一座上州——治洛交县（今陕西富县），长安东北约五百唐里（《旧唐书》），沿洛水谷地北上是关中通陇朔之要；天宝元年至乾元元年（742-758）官称"洛交郡"。

## 形貌

鄜州城坐在**洛水（北洛河）上游的河谷台地**上——州治在**洛交县**（今陕西富县城）。城周不大，夯土城墙，开四门；四周是黄土丘陵塬地，洛水自西北而来，绕城西南而下，再南流入同州境内汇入渭水。

州地不是平原——而是**陕北黄土高原南缘**：城外塬、梁、峁、沟错落，沟谷里有水浇地、塬面上是旱作的粟与糜子。北接延州（治肤施，今陕西延安），西接坊州（治中部，今陕西黄陵），南去坊州、再南循官道一日可达三原、再南便入京畿。**鄜州、坊州、延州三州**——盛唐人合称"鄜延"或"鄜延北部"——是关中以北、陇山以东那条南北窄长走廊上的几个州治。

地势上鄜州是**关中—朔方道**的中段。自长安出春明门北行，过泾阳、三原、宜君，过坊州中部县，再北便至鄜州洛交——再北过甘泉、抵延州肤施，再北越芦子关入夏州、入朔方。**这条南北纵道**——盛唐时是关中与朔方节度（治灵州）之间最直的一线，755 年安史乱起后，太子李亨自马嵬北上灵武，走的也是这条道的北半段。

物候是黄土高原南缘的样子：春旱多风、夏短而热、秋早霜、冬干寒。**洛水**——这里指北洛河，不是河南的伊洛河——自鄜州一段水量不大但常年不断，是黄土沟谷间难得的一条常年河。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

鄜州地有古传。秦文公十年（前 756）作**鄜畤**——《史记·封禅书》载文公梦黄蛇自天下属地，命史官祠之于鄜衍，立**鄜畤**祭白帝——这是"鄜"字最早入正史的一笔。鄜畤的具体位置后世说法不一，大致在今陕西富县、洛川一带的洛水北岸塬地上。秦汉雍五畤之一便是鄜畤——盛唐人写到鄜州，常以"鄜畤旧地"为典。

行政沿革上，**西魏废帝三年（554）始置敷州**——治杏城（今陕西黄陵附近），辖境包括后来的鄜、坊一带。北周保定二年（562）于洛交县置**鄜城防**。**隋开皇三年（583）罢郡——置鄜州**，治洛交县——"鄜州"之名自此立。大业三年（607）罢州为**鄜城郡**。**唐武德元年（618）**高祖复置鄜州——治洛交。**天宝元年（742）**玄宗诏天下州改郡——鄜州改称**洛交郡**。**乾元元年（758）**肃宗复郡为州——洛交郡又复称鄜州。755 年这一刻，城门匾额上写的还是"洛交郡"。

唐人若在 755 年驻足回望：鄜畤之名自秦文公算起已五百一十一年；鄜州之名自隋开皇立算已一百七十二年；本朝武德置鄜州亦已一百三十七年。是一座关中以北黄土塬上的老上州。

### 此刻

天宝十四载（755）的鄜州，行政上是**关内道洛交郡治**——据《旧唐书·地理志》洛交郡天宝领县五：**洛交、洛川、三川、直罗、甘泉**。郡望属"上"——**天宝户二万三千四百八十三、口十五万三千七百一十四**（《旧唐书》）。

洛交郡是**鄜延北部的中心州**——盛唐"关内道"自南而北一路州郡，京畿（长安周围）—华州、同州、岐州、邠州（拱卫长安）—坊州、鄜州、延州（北线鄜延）—庆州、宁州（陇朔西线）—夏州、丰州、灵州（朔方）。鄜州在这条北线上居中——南去长安约 500 唐里（长安在鄜州西南、鄜州在长安东北约 500 唐里）、北去延州一百二十唐里、再北至朔方治所灵州约六百唐里。**关中通朔方的官道**自洛交穿城而过——这是 755 年鄜州在地图上的位置。

这一年州本城的主事：洛交郡太守、郡丞、录事参军、司功 / 司仓 / 司户 / 司兵 / 司法 / 司士六曹参军——盛唐上州官制的整套班子，规模小于同州、岐州等京畿辅郡。鄜州无节度治所、无大都督府、无监牧使——只是关中以北一座上州。

鄜州在 755 年的实际分量，不在本州事务——而在**它所在的那条南北通道**。安史乱起后的下一年（756 年六月），玄宗西奔金牛道入蜀，太子李亨自马嵬向北分道——经邠州、宁州、庆州、彭原，再东北折向灵武——这条路虽未直穿鄜州，但鄜州、坊州一线是同时被卷入北线避地的几州之一。当年冬，关中士庶北逃避地者多走鄜延，**杜甫便是 756 年八月自长安出奔，把家小寄寓于鄜州羌村**（今陕西富县西北）——杜甫《月夜》"今夜鄜州月，闺中只独看"便是这一年这一城的事，757 年杜甫自鄜州只身南下凤翔投奔肃宗行在，《北征》写"皇帝二载秋，闰八月初吉。杜子将北征"——出鄜州、过邠州、转奉天，再入凤翔，是 757 年安史乱中一条具体而清晰的逃难路线。

对 755 年这一冬的鄜州人来说，城里照常上缴秋粮、点北线戍卒；他们尚不知道，再过一年这座上州将成为关中士庶北逃避地的中转、成为杜甫一家寄居的地方、成为肃宗灵武一系自南而来联络长安的最后几站之一。

## 历史长河（755 → 今）

**中晚唐**　乾元元年（758）洛交郡复称鄜州——城匾换回旧名。**至德二载（757）杜甫自鄜州赴凤翔**、《北征》一诗写尽这一程；杜甫家小留鄜州羌村，至 758 年方接出。中晚唐藩镇并起，**鄜坊节度使**于代宗大历三年（768）置——治鄜州——辖鄜、坊、丹、延四州，是关内道北线一处中等藩镇。鄜坊节度使一职断续存于中晚唐，黄巢之乱（880-881）前后鄜延一带亦被裹入战火。

**五代至宋**　五代鄜州数易其主——后梁、后唐、后晋、后汉、后周相继。**北宋**仍置鄜州——属永兴军路、后属鄜延路；鄜延路是北宋西北边防四路之一，治延州（后升延安府），鄜州为路下属州。范仲淹、韩琦经略陕西时鄜州为后方军镇。

**金元**　**金代鄜州属鄜延路；元代至元六年（1269）改鄜州为鄜城路，旋复为鄜州**——属延安路。明清两朝鄜州先后属陕西布政司延安府——清雍正三年（1725）升鄜州为**鄜州直隶州**，辖洛川、中部、宜君三县。

**民国至今**　民国二年（1913）废府州——鄜州改称鄜县；1964 年因"鄜"字生僻，改作**富县**——"富"是"鄜"的同音改字，**鄜州**这个用了一千三百三十年的州名自此从地名上退去。今为陕西延安市富县——县城仍在唐洛交县故址上，洛水仍自城西南绕过；杜甫寄居的羌村遗址在县城西北约十五里——今属富县茶坊镇羌村——村里至今有"杜公祠"。秦文公的鄜畤——盛唐人写到鄜州时仍以为典——具体位置已不可确指，留在洛水北岸塬地上某一处。

## 引文

《元和郡县图志·卷三·关内道三·鄜州》（李吉甫 813）：

> 鄜州，洛交郡。开元户一万五千四百七十二，乡四十。……禹贡雍州之域。秦文公十年作鄜畤，祠白帝。汉武帝置上郡。后魏置敷州。隋开皇三年罢郡，置鄜州。大业三年罢州为鄜城郡。武德元年改为鄜州。天宝元年改为洛交郡。

——这是 755 年前后洛交郡建置、户口的一手史源；"开元户一万五千四百七十二"是盛唐鄜州户口的标准记载。

杜甫《月夜》（至德元载 756 八月作于长安）：

> 今夜鄜州月，闺中只独看。遥怜小儿女，未解忆长安。

——杜甫家小寄寓鄜州羌村时，自陷京中怀念家室的诗；"鄜州月"三字，是盛唐人写鄜州留下的最有名一笔。
