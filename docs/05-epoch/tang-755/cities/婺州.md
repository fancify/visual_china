# 婺州

## 一句话

江南东道中部一上州——治金华县（今浙江金华），婺江、兰江、钱塘江上游一段汇于州境；州因分野婺女星而得名，州治金华为浙中陆水转运枢纽。

## 大唐天宝十四载（755年）

婺州治**金华县**，在浙江中部——婺江自东而来，与南来的武义江汇于城下，西去与兰溪江合流，下至钱塘江入海。州境为浙中盆地，南北皆山，中为通衢——自越州、杭州陆道南下经此入处州、温州，又可换舟自兰江出钱塘——是浙东向浙南、闽北一道陆水兼用的转运点。州属**江南东道中部**，**天宝元年（742）随玄宗朝"改州为郡"通制改为东阳郡**，755 时正名为"婺州东阳郡"，两名在唐人嘴里互换不分；**乾元元年（758）肃宗复改为婺州**——但这是 755 之后的事。沿革更早：**隋开皇十三年（593）置婺州**，因其地当**婺女星**分野得名；隋大业三年（607）改为**东阳郡**；**唐武德四年（621）**平李子通复置婺州，至天宝元年再改东阳郡。《新唐书·卷四一·地理志五》婺州列为**上州**——开元定制户四万以上为上州，婺州户口远逾此数。**《旧唐书·卷四十·地理三·江南东道·婺州东阳郡》载：**"旧领县五，户三万七千八百一十九，口二十二万八千九百九十。**天宝领县七，户十四万四千八十六，口七十万七千一百五十二**"——755 时婺州在册**户 144,086、口 707,152、领县七**（金华、义乌、永康、东阳、兰溪、武成、浦阳）。其中**浦阳县**为**天宝十三载（754）**析金华、义乌、东阳三县置——是 755 时新出之县。**武成县**原名武义，垂拱四年（688）析永康置武义，天授二年（691）改武成，755 时官称作武成。州境地处浙中要冲，**太守治金华**总浙中七县，户口在江南东道仅次于苏、杭、越、湖数州，是浙中第一大州。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改东阳郡为婺州。中晚唐婺州贡赋以丝、纸、瓷为大宗——婺州青瓷与越窑同系而稍逊。

**两宋**　北宋仍号婺州。**南宋**婺州为江南文风一极——**吕祖谦**（1137-1181）讲学金华、**陈亮**（1143-1194）永康人，与朱熹、陆九渊鼎足而三，开**婺学**（金华学派 / 永康学派）一脉；婺州遂有**"东南邹鲁"**之号。但这是 755 之后四百年事，本 doc 不预支。

**元至明清**　元至元十三年（1276）改婺州为**婺州路**；至正十八年（1358）朱元璋取婺州，改**宁越府**，旋改**金华府**——"金华"二字自此为府名。明清沿金华府之号。

**民国至今**　1912 年废府，析置金华、兰溪、东阳、义乌、永康、武义、浦江诸县。**1985 年撤金华地区设地级金华市**——"金华"作建制名复活。今金华市辖婺城、金东、兰溪、义乌、东阳、永康、武义、浦江、磐安——大体覆盖盛唐东阳郡境。

## 引文

> 婺州东阳郡，上。隋东阳郡。武德四年，平李子通，置婺州。……天宝元年，改婺州为东阳郡。乾元元年，复为婺州。旧领县五，户三万七千八百一十九，口二十二万八千九百九十。天宝领县七，户十四万四千八十六，口七十万七千一百五十二。

——《旧唐书·卷四十·志第二十·地理三·江南东道·婺州东阳郡》
