# 廓州

## 一句话

陇右道最西南那一座小下州——治广威县，黄河自西南峡口下行至此始入河谷，州境南面便是吐蕃，是大唐沿黄河上游所设的最西一州。

## 大唐天宝十四载（755年）

廓州治**广威县**——这县在唐人嘴里换过三个名字：北魏因县内化隆谷置**化隆县**，**先天元年（712）改为化成县，天宝元年（742）又改为广威县**——三名实指一地（今青海化隆群科一带）。州境跨**黄河上游南岸一段河谷**：黄河自唐人所称"吐蕃境"之西南峡口下行至此，始入一段较开阔的河谷，廓州城便扼这一段。**天宝元年（742）随玄宗朝"改州为郡"通制改为宁塞郡**，755 时正名"廓州宁塞郡"——州、郡两名在唐人嘴里互换不分；**乾元元年（758）肃宗复改为廓州**，那是后话。

廓州沿革承隋**澆河郡**——这是吐谷渾的旧城名，隋平吐谷渾置郡；**唐武德二年（619）置廓州**。两《唐书》对廓州着墨极俭——《旧唐书·卷四十·地理志三·陇右道·廓州》：**"舊領縣二，戶二千二十，口九千七百三十二。天寶，縣三，戶四千二百六十一，口二萬四千四百。"**《新唐书·卷四十·地理志四·陇右道·廓州宁塞郡》同载**"戶四千二百六十一，口二萬四千四百。縣三。"**——两书户口完全相符，是天宝十一载（752）籍数。**领县三**：广威（治）、达化、米川。这是陇右道里户口最小的几座下州之一，仅比叠州（1,275 户）、宕州（1,190 户）稍胜，远不及鄯州、河州。

但廓州的分量不在户口——它是**唐与吐蕃前线的最西一锁**。河州在洮河、廓州在黄河；河州扼大夏河谷、廓州扼黄河谷——两州一东一西连成陇右道的南弧。**天宝十三载（754）**——开战前一年——朝廷沿廓州境一连置**六军**：**寧邊軍**（本寧塞軍）、**威勝軍**（治县西八十里宛秀城）、**金天軍**（西南百四十里洪濟橋）、**武寧軍**（百谷城）、**曜武軍**（南二百里黑峽川）、**積石軍**（達化縣西，本儀鳳二年/677 置之靜邊鎮）——六军沿黄河南岸与积石山一线排开，是陇右节度使在最西一段的兵力布置。这六军建置的同一年（754），王忠嗣旧部封常清、哥舒翰诸军正在陇右整训——廓州虽小，是这条防线最西头的一颗钉。

755 时，廓州城内官署按下州制设——**郡太守**（742-758 通制改州为郡，州刺史一律称太守）领长史、司马、录事参军以下数员；城外是六军戍堡与烽燧。州境居民以诸羌、吐谷渾余部为主，唐人编户不多——四千余户、两万四千口里相当部分是边戍卒及其家属，真正的耕户散在黄河南岸窄窄一线河谷地里。

## 历史长河

**中晚唐**　**至德元载（756）后陇右兵八万东调勤王**——廓州一线已无重兵。**至德二载至广德元年间（757-763）**，吐蕃自青海北上逐次拔陇右诸州——《旧唐书》《通鉴》仅记"陇右诸州相继陷蕃"而无确年——廓州当在此七年之内入吐蕃；**广德元年（763）十月吐蕃尚结息率二十万众入长安**时，廓州早已不在唐版图。**乾元元年（758）肃宗复廓州之名**——是名义上的复名，地已不在唐手。

**入吐蕃至宋元**　廓州陷蕃后约九十年——**宣宗大中三年（849）三月沙州张议潮起义**、咸通七年（866）拓跋怀光西破鄯州——河湟一线名义归唐而中央实控不及，廓州一带散在吐蕃余部及党项手中。北宋元符二年（1099）王厚河湟开边后亦未及廓州故地。**金、西夏、元** 之间廓州故地为西宁州、贵德州一带辖境，廓州之名不再立。

**明清至今**　明属西宁卫；清雍正间分置**循化厅、贵德厅**等——皆在廓州故地上。民国后置**化隆县（在治县广威/化隆旧址）、贵德县、循化县**——廓州故境分为三县并存至今。今青海化隆回族自治县群科镇附近一带为唐廓州治县所在；黄河仍自西南谷口下行经此，仍是这地方一千四百年从未动过的一条线。

> 廓州，下，隋澆河郡。武德二年置廓州。天寶元年改為寧塞郡。乾元元年復為廓州。舊領縣二，戶二千二十，口九千七百三十二。天寶，縣三，戶四千二百六十一，口二萬四千四百。

——《旧唐书·卷四十·地理志三·陇右道·廓州》
