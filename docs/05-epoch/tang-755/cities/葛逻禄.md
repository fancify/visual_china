# 葛逻禄

## 一句话

西突厥别部、三姓部族联盟（谋落 / 炽俟 / 踏实力），居北庭都护府西北至中亚草原一线——755 时尚在北庭羁縻名分内（贞观二十三年 649 内属，显庆二年 657 西突厥平后分置阴山、大漠、玄池三都督府；后又分炽俟别置金附州），但天宝十载（751）怛逻斯之战时其中亚一支已倒戈助大食败高仙芝；安史乱后唐西陲崩，葛逻禄独立西迁，9 世纪后融入喀喇汗朝。

## 大唐天宝十四载（755年）

葛逻禄在唐人北庭、安西两都护府的边政档案里，是一个**部族联盟而非一州一城**——故无定治、无一城可指。三姓——**谋落、炽俟、踏实力**——分牧北庭西北至阿尔泰一带，又称"**三姓葛逻禄**"。它的本根是西突厥别部，与回纥、坚昆、突骑施同属铁勒-突厥语族群，但自成一系。

**太宗贞观二十三年（649）**葛逻禄部内属归唐，是它入唐羁縻体系的开端；但具体的州治分置要再过数年。**高宗显庆二年（657）**苏定方平阿史那贺鲁、西突厥汗国瓦解后，唐廷重整西陲羁縻——以三姓葛逻禄分置三都督府：**谋落部置阴山州都督府、炽俟部置大漠州都督府、踏实力部置玄池州都督府**，归北庭（初属安西，702 北庭都护府立后转隶）羁縻系统辖下。这三都督府是"**部落名挂州号**"——无城无郭，无户籍册，每岁酋长入府接受唐廷锦袍、册命、印信，便是一年羁縻之实。其后（具体年待考，约高宗末至武周间）又**分炽俟部别置金附州都督府**，三姓体系扩为四府并立，是唐对葛逻禄羁縻最细密的一段。744-745 间回纥怀仁可汗骨力裴罗灭后突厥之役，**葛逻禄部曾与回纥、拔悉密联手**——这是它在漠北格局重立中的一笔。

但 755 时葛逻禄已三分：**一支仍在北庭西北、阿尔泰山一带羁縻于唐**；**一支东向**，附于刚立的回纥可汗汗国（葛勒可汗朝）；**一支西向**——这一支最关键，已西迁至**七河流域**（楚河、塔拉斯河、伊犁河一带），与黑衣大食（阿拔斯朝）东陲接壤。**天宝十载（751）七月**，高仙芝率三万安西兵西征石国，至**怛逻斯水**（今哈萨克斯坦塔拉兹）与阿拔斯将齐亚德·伊本·萨利赫所部对垒——交锋第五日，**唐军阵中的葛逻禄部突然倒戈，从后掩袭**，唐军大败，高仙芝仅以数千骑遁还。**这是葛逻禄西支的一笔决定性历史动作**——盛唐西陲的军事顶点止于此役，葛逻禄是终结者之一。

755 这一年葛逻禄部的格局是：北庭羁縻名分仍挂着（阴山、大漠、玄池三州），但怛逻斯之后唐与葛逻禄西支已**事实交恶**；北庭节度此时由安西节度封常清兼，对葛逻禄的统御弱化。**户口不强订**——羁縻部族非州县编户。十一月安史乱起、安西北庭精兵东调后，唐对葛逻禄的最后一层名义羁縻也随之松脱。

## 历史长河

中晚唐——755 之后葛逻禄迅速独立。8 世纪末与回纥（已改回鹘）联手攻**突骑施**——突骑施汗国残部至此解体，葛逻禄遂控**七河流域**（楚河、塔拉斯、伊犁三河谷）至北庭以西一线，是后突骑施时代中亚草原最强的一支。**840 年**黠戛斯破回鹘汗国后，葛逻禄收纳西迁的回鹘一支；其本族则于**9 世纪末至 10 世纪初**与样磨、处月诸部共立**喀喇汗朝**（Karakhanid，约 840-1212）——喀喇汗朝以巴拉沙衮（楚河谷虎思斡耳朵旧址，即碎叶旁的布拉纳）和喀什噶尔为两都，皈依伊斯兰（约 960 萨图克·博格拉汗），是中亚突厥化-伊斯兰化的关键政权之一。**葛逻禄之名自此并入"喀喇汗"——作为一个独立部族联盟从史册退去**。

宋元至今——汉地史志中"葛逻禄"二字渐成历史追述，元代蒙古西征过七河流域时偶以"哈剌鲁"作译名指其后裔残部。今日学界以**11 世纪麻赫穆德·喀什噶里《突厥语词典》**所记 Qarluq 三部为葛逻禄之直接续脉。"哈萨克"作为后世波斯语族群名（15 世纪后），非葛逻禄当时所自称——755 时不可用此名。
