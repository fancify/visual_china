# 抚州

## 一句话

江南西道东部抚河（盱江）流域一中州——治临川县（今江西抚州临川区），洪州东南内地一州，盱江北流出鄱阳湖水系入赣江。

## 大唐天宝十四载（755年）

抚州治**临川县**，在抚河（唐人多称盱江、旴江）中游东岸——盱江出武夷山西麓，北流过抚州，再折西北汇入赣江、注鄱阳湖，是江南西道东部一条独立的入湖大水。州当**洪州东南**陆路约二百余里、水路顺盱江而下可至洪州。沿革：**隋为临川郡**，治临川县；**唐武德五年（622）讨平林士弘，置抚州**——"抚"之名自此而立；**天宝元年（742）**玄宗诏天下州改郡，抚州改称**临川郡**，755 时官称即"抚州临川郡"，《新唐书·地理志》载为**上州**（《旧唐书·地理志》列为**中州**——两书异文并存）；**乾元元年（758）**肃宗复改郡为州——但这是 755 之后的事。

**《旧唐书·卷四十·地理三·江南西道·抚州》载：**"抚州，中，隋临川郡。武德五年，讨平林士弘，置抚州……旧领县三，户七千三百五十四，口四万六百八十五。**天宝领县四，户三万六百五，口十七万六千三百九十四**"——**《新唐书·卷四一·地理志五·江南西道·抚州临川郡》载**："抚州临川郡，上。土贡：金丝布、葛、竹箭、朱橘。**户三万六百五，口十七万六千三百九十四**。县四"——两书户口数完全一致：755 时抚州在册**户 30,605、口 176,394、领县四**（临川、南城、崇仁、南丰）。开元至天宝间户口较武德旧籍（户 7,354、口 40,685）翻四倍——是盛唐江南内地州县最显眼的一段增长，与洪州、吉州、虔州同步。

太守姓名于史阙载——755 当年的抚州太守已不可考。州境无大兵无大事，钱粮以纲船顺盱江西北行入赣江、归洪州总集；土贡见《新唐书·地理志》——金丝布、葛布、竹箭、朱橘——是抚河流域的典型物产。

## 历史长河

**中晚唐至五代**　抚州在唐一代始终为江南西道东部内地一中州——文教未盛、科第未显；唐人诗赋中提抚州者甚少。**南唐保大十年（952）**升抚州为**昭武军**节度——州始有军额；是抚州地位首次提升。

**宋元明清**　**北宋抚州属江南西路**——这一段抚州始得"文乡"之名：**临川王氏**（王安石一门）、**晏氏**（晏殊、晏几道父子，临川人）、**曾巩**（南丰人，王安石同乡同辈）皆出于此，是北宋江南西路文学的核心一支；但**这些皆宋人**，755 时尚未为人知。元抚州路。明洪武元年（1368）改**抚州府**——府名自此立。明代抚州临川县出**汤显祖**（1550-1616）——"临川四梦"（《牡丹亭》《紫钗记》《邯郸记》《南柯记》）成于明万历间，是抚州在明清的最高一笔；亦皆 755 之后八百余年事。清抚州府仍属江西布政司。

**民国至今**　1912 年废府存县。今**江西省抚州市**——地级建制立于 2000 年（撤地区设市），治**临川区**，正与唐抚州治县同名同地。今抚州市区盱江北流过城——与唐临川县城方位大体相承；唐抚州城圈遗迹今已不见。
