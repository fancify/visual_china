# 黔州

## 一句话

黔中道一道之治、黔州都督府所在——治彭水县（今重庆市彭水苗族土家族自治县），乌江中游一座山深州薄的下州，**统辖周边五十余羁縻州**，是大唐西南向"溪洞蛮""徭僚"诸部投放秩序的那一只手。

## 形貌

黔州城在**乌江中游一处河谷台地**上——州治在**彭水县**（今重庆市彭水苗族土家族自治县），城东、城南皆贴乌江，城西、城北抬头便是武陵山西南支脉的山岭。海拔自江面三百米直上谷外两千米——这是巴山东南尾、武陵山西北角、云贵高原东北沿三股山势交叠的一处坳口。州境东接施州（今湖北恩施）、南接思州（今贵州务川）、西接涪州（今重庆涪陵）、北接夔州（今重庆奉节）。山岭密、谷涧深，沿江一线之外，多数地方人迹罕至——**当地的人，唐人称'溪洞蛮''徭僚'，是日后苗、土家、仡佬诸族的先民**。

谷里的州治薄而紧——夯土城墙不大，城内有黔州刺史衙（742-758 时官称太守）、彭水县衙、馆驿一所、戍所一处；城外乌江有渡口与小型市集。城里管的是城里事，城外便归各部羁縻——汉官管登记的羁縻州、羁縻县、羁縻乡，部酋管谷涧之内的实际部众，两套秩序分层叠在一处。物产以**朱砂**、漆、葛、麻、药材为主——黔中朱砂自汉以来便是名货，唐时仍贡。**乌江自西南来、自城下东北出**——下行约六百里出涪州入大江（长江），上行可通思州、播州境，是黔州境唯一一条对外通道。山深路狭，斥候走一趟要五七日。物候偏湿热，瘴疠时作，是唐人眼里的'南瘴'地之一。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

黔州沿革不算短，但唐人视野中的'黔州'起得偏晚。**隋开皇九年（589）**文帝平陈后置**黔州**——治彭水，'黔'之名从此立。**大业三年（607）**改黔州为**黔安郡**。**唐武德元年（618）**高祖复置黔州——并**置黔州都督府**，治彭水县——这是黔州在大唐版图上的起头一笔。**天宝元年（742）**玄宗诏天下州一律改称郡，黔州改称**黔中郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州，黔中郡再称黔州——但 755 这一刻，城门匾额上写的还是'黔中郡'，民间口语仍称'黔州'，两个名号并行。

黔中道的形制晚出。盛唐初年此地随江南道而辖；**开元二十一年（733）**玄宗分江南为东、西二道、又**别置黔中道**——黔中道治便定在黔州，辖**黔、施、思、费、夷、锦、溱、播、珍、南**等正州十数、**羁縻州五十余**——是大唐边裔诸道里羁縻州最密的一道（《新唐书·地理志七下》载黔州都督府所领羁縻州凡五十）。**开元中又置黔中经略使**——治亦在黔州，掌一道军政按察兼羁縻部落事——是为黔州在盛唐西南诸道里的真正分量。

### 此刻

天宝十四载（755），黔州官称**黔中郡**——是**黔中道一道之治**、**黔州都督府治**、**黔中经略使治**。三套官署叠在彭水一城。

户口上，《新唐书·卷四十一·地理志五·黔中道·黔州黔中郡》载黔州**'户四千二百七十，口二万四千二百四'**（即户 4,270 / 口 24,204，按唐志通例为天宝间数）；《元和郡县图志·卷三〇》另载黔州**'开元户四千二百三十四，乡一十二'**——两数差近，可互为参订。按唐制户不满两万为下州，黔州正卡在**下州之最下**一线——而这只是登记在册的汉编户，**谷涧之内羁縻部众数倍于此，多不入版籍**。**此刻的黔州太守**（按：742-758 郡制刺史官称'太守'；758 复州后才再称刺史——本 doc 严守 755 时态）以**黔中郡太守**之职兼**黔中经略使**——一身两任；至于'兼都督府长史'一条，新旧《唐书》及《通典》《元和志》于黔州都督府长史的具体官 setup 各有歧异，此处不强订，姑列经略使为确。

实际管事分两层：**汉编户**按下州常制——户籍、徭役、租调、贡赋（朱砂、葛布、药材为主）由州县衙按律办；**羁縻州**所领五十余多由部酋世袭其官、受都督府授敕，中央只两条要求：一不反叛、二岁有贡使。**经略使的兵也薄**——常驻戍卒不过千把人，分驻沿乌江几处戍堡与烽燧；真要弹压一处部酋反叛，须自剑南或江南西道征兵协剿。长安诏令南下经襄州、江陵、夔州、涪州一线再溯乌江上至黔州，约两月余始至。十一月初九安禄山起兵于范阳——消息传到黔州，约已是这一带最远的一段静寂里。朱砂照贡、谷涧照静、乌江照流——大约也以为这一年与往年没什么两样。

## 历史长河（755 → 今）

**中晚唐至五代**　乾元元年（758）肃宗复郡为州——黔州之名重归。安史乱后剑南、江南诸道兵东调，黔中戍卒益薄、羁縻部酋自治更甚。**贞元、元和间**黔中道改置**黔中观察使**（一说乾元已改观察，《新唐书·方镇表》互有异文），仍治黔州——但终唐之世，黔中道未升节度使，**始终是一道之中羁縻浓厚的边道**。元和户与天宝户大体未跌（《元和志》'户四千二百三十四'），**反映黔州本以登记户少为常态、并非乱后骤降**。僖宗中和间黄巢入蜀，黔中戍备更虚；唐末归王建前蜀；五代之间先后属前蜀、后唐、后蜀，管辖虚浮，部酋世袭。

**宋元明清**　**南宋庆元五年（1199）**改黔州为**绍庆府**，治仍在彭水；元至元二十二年（1285）置**绍庆路**。**明洪武四年（1371）**改绍庆路为**彭水县**——黔州、绍庆诸名号自此罢，彭水县直行至清末，明清两代属**四川行省重庆府**。**至于'贵州'**——元至元十六年（1279）置八番顺元宣慰司为西南省级名号雏形，**明永乐十一年（1413）**始置贵州布政司，'贵州'作为正式省级名号自此而立——**但贵州省治在贵阳，与唐黔州治彭水绝非同地**。'唐之黔州 ≠ 后世之贵州'，这一条须严守。

**平播之役**　明万历二十八年（1600）李化龙督师讨**播州宣慰使杨应龙**——是为这一带历史长河里最重的一笔。**播州（今贵州遵义）在唐属黔中道辖境之一州，杨氏自唐末'守播州'起世袭其地七百余年**，至此方平。平播事在彭水县境西南数百里，**与彭水县（唐黔州治）非同地**——但彭水县是平播大军溯乌江西上的一处转运点。

**民国至今**　民国二年（1913）废府存县；1949 年后属四川省涪陵专区、涪陵地区。**1983 年改设彭水苗族土家族自治县**——'苗''土家'入县名，是这一带千五百年部族史的一次正式承认。**1997 年重庆直辖**后划归重庆市，今为**重庆市彭水苗族土家族自治县**，县治汉葭街道，承唐彭水县治旧址。

唐黔州故城的地表遗迹今已不存——彭水县城叠压在唐宋元明清五代县城之上。乌江仍自城下东北流过，武陵山西支仍向天，朱砂矿仍在县境西的山里——只是'黔州''黔中郡''绍庆府'这些名号，**都早已让位给'彭水'两个字**。
