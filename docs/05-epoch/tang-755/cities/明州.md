# 明州

## 一句话

江南东道余姚郡治，治鄮县（今浙江宁波）——**开元二十六年（738）析越州东境新立的一座州**，越州的东面海门、四明山下的青瓷与佛寺之乡。

## 形貌

明州城在四明山东北麓、甬江入海口的内陆一段。

州治鄮县城周回不大——压在四明山北麓与杭州湾南岸之间的一片冲积台地上。**四明山**自西南而来，到州城西南约数十里处止——山势不甚高（主峰金钟峰一千余米），但层峦叠嶂、雨水充沛，是浙东青瓷与名茶的产山。城北、城东是水网平原——奉化江、姚江两流自西南、西北合于州城东北，名"三江口"（这三江口的合流形态在盛唐已成，但"宁波三江口"作为城市地标的写法是宋以后事）；自合流处东注入海，即**甬江**——一州之水皆归此一江入东海。海岸线在城东北约数十里，是杭州湾南岸最深的一段港湾。

物候是江南偏东南的：**春雨绵绵、梅雨长、夏热伴东南风、八月台风偶至、冬不甚寒**。沿海一带咸雾偏多。州中山地出茶（四明山）、青瓷（上虞、慈溪、余姚一线的越窑窑场，盛唐时窑火正盛——其中**上林湖窑址在今慈溪境内，唐时属余姚县**），平原种稻麦、间桑麻；近海产盐、产鱼蚌、产海产珍贝。州中户口据《元和郡县图志》《通典》所记开元—天宝间**约四万户**——**刚跨过盛唐"上州"门槛（开元二十一年定制：四万户以上为上州）**，在江南东道诸郡中算中下等——比同道越州（约九万）、杭州（约八万六）单薄一半，但已不是下州。

## 大唐天宝十四载（755年）

唐人眼里的明州**是一座新州**——盛唐玄宗朝才立的州，不到 20 年。

最早可追溯的是秦时的**鄮县**——秦始皇灭楚（前 223）后置会稽郡，鄮县是郡下一县，治所即今宁波一带，"鄮"字取"贸"音，因鄞东海岸有海贸之集而得名。汉承秦制，鄮县属会稽郡。东汉、三国吴、东晋南朝相沿——这一带千年间一直是**会稽郡东境的一个海滨小县**，治所微移而县名不废。**隋开皇九年（589）**平陈，废鄮入句章；**唐武德四年（621）**析句章复立鄮县，属越州。

转折在盛唐——**开元二十六年（738）闰八月，玄宗诏析越州东境置明州**（《旧唐书·玄宗本纪》《新唐书·地理志》）。州名取**四明山**之"明"——四明山主峰有**"四窗岩"** 之传（山顶岩壁有四方天窗似的洞，月光穿照如四明），故山名"四明"，州名"明"由此出。**新立的明州领鄮、奉化、慈溪、翁山四县**——鄮县为治所；奉化、慈溪由鄮县析出，翁山在海中（今舟山群岛）。**至天宝元年（742）**玄宗诏天下州一律改郡，**明州改称余姚郡**——是为 755 年的官称；**乾元元年（758）**肃宗复改郡为州，余姚郡复称明州。

——这是一桩要分清的事：**明州不是"汉魏旧州"**。鄮县是旧县，但**作为一州的明州是 738 才有的事**。755 年的明州人提起本州，提的是"新立不过十七年"的州——而非千年古郡。

755 年的明州属**江南东道**，下辖**鄮、奉化、慈溪、翁山**四县（《新唐书·地理志》余姚郡四县口径；翁山县在海中诸岛——开元二十六年置州时所立；**翁山县后于大历六年 771 因海寇袭扰而废**，755 时尚在）。户**约四万**——是盛唐**上州**门槛之上的一档，但在江南东道诸郡中比重不大。城中有州衙、郡衙、鄮县治、市肆——这是一座**初立不过两旬的州城**，城圈、衙署、坊市格局都还在草创。

明州的两大底子：**一在山，一在海**。**山**——四明山下出茶、出青瓷、出佛寺。**天童寺**（晋永康元年/300 立，西晋僧义兴所立，号天童玲珑岩寺）在四明山东麓；**阿育王寺**（西晋太康三年/282 立，号"育王古寺"，相传寺中藏有阿育王所造佛舍利塔）在城东约三十里——这两座是江南最古的佛寺中的两座，盛唐时已逾四百年。**海**——明州面对杭州湾、对马海峡、东海——是**江南面海最深处之一州**。盛唐时已有海贸之实——但**日本遣唐使以明州为主登陆港，是中晚唐（9 世纪起）渐盛的事**：早期遣唐使（630-733 年间共十余次）多自难波出发取北路或南岛路登陆苏州、扬州、登州、楚州一带；明州港作为遣唐使主要登陆港的盛况，要到**承和五年（838）小野篁那次后期才完全确立**——755 年这一刻，明州港虽已通海，但远未到中晚唐"四明海舶岁岁来"的盛境。**这一笔要分清：盛唐 755 时的明州，已是一处通海之州，但不是日本遣唐使的主港**。

天宝十四载冬安禄山起兵的消息传到明州大约是十二月初——比传到越州还晚几日。明州地处江南腹地东南、距前线四千里——朝廷与州中皆未感切肤；这场乱将把江南推到唐廷漕赋命脉的位置，明州作为江南东道新州、海港之州，承漕、出粮、出青瓷绫绢之责将日重——但这一切尚在前面。对天宝十四载的明州人来说，这座城是**一座新立不到二十年的州、四明山下的青瓷与佛寺之乡、面海一道海塘**——一座江南东南角的中等州城。

## 历史长河

**中晚唐**　乾元元年（758）复改余姚郡为明州——州名"明"自此定。**大历六年（771）**翁山县因海寇袭扰而废，明州自此领三县（鄮、奉化、慈溪），至晚唐永贞元年（805）析鄮县置**鄞县**——州治移驻鄞县（今宁波市区一带）。**长庆元年（821）**州城自小溪（今鄞江镇）迁至**三江口**（即今宁波老城）——这是明州城作为"三江口之城"的肇始；自此一千二百余年州治再未大迁。**晚唐至五代**，明州港随海贸之盛而起——**日本遣唐使后期（承和五年/838 小野篁等）以明州为主登陆港**；**新罗、日本商舶岁岁来**——明州自此成中晚唐至五代东海贸易的第一州。**上林湖越窑**（今慈溪上林湖窑址）在晚唐五代烧出**秘色瓷**——与法门寺地宫 1987 年出土的咸通十五年（874）秘色瓷恰能对应；这是明州境内最重要的盛唐至晚唐物质遗存。

**五代至宋**　**五代**明州属吴越国（907-978）——钱氏定都杭州，明州为东境海港重镇；**北宋**仍号明州，**南宋绍熙五年（1194）升为庆元府**——南宋间庆元府是**市舶司所在**之一，与广州、泉州、华亭并列为四大对外贸易港。**元**改庆元路；**明洪武十四年（1381）改庆元路为宁波府**——"宁波"二字自此立（取"海定则波宁"意），"明州"之名自洪武十四年正式退场——共立 643 年（738-1381）。本 doc 严守 755 时态——**绝不预支"宁波"二字**。

**明清**　明清沿宁波府之号。明嘉靖间倭乱沿海，宁波府是首当其冲——府城海塘屡修；明万历间宁波府人口逾百万。清沿之，鸦片战争后宁波为**1842 年《南京条约》五口通商口岸之一**——开埠岁月由此起。

**民国至今**　1912 年废府置县。1949 年后改设宁波专区，**1983 年地市合并设地级宁波市**。今**宁波市**，浙江省辖**副省级、计划单列市**，面积九千八百余平方公里、人口约九百四十万。市辖海曙、江北、北仑、镇海、鄞州、奉化、余姚、慈溪、宁海、象山——大体覆盖盛唐余姚郡境、外加翁山旧县（今舟山，1987 年自宁波析出为单设地级舟山市）。**天童寺、阿育王寺今皆仍存**——皆历代屡毁屡兴，今寺基本格局是明清以后所立，但寺址与盛唐同地；**上林湖越窑遗址**（1957 年起多次发掘）2015 年于上林湖**后司岙**出土晚唐五代秘色瓷实物，是明州境内最重要的盛唐至晚唐考古成果。**北仑港**是今中国大陆第一大港、世界第一大港之一（货物吞吐量计）——这一处海港的盛况，恰是盛唐 755 年那座面海新州的远后延续。
