# 华州

## 一句话

关中东屏，华阴郡治；华山立于城南、潼关守于城东——长安东去约一百八十唐里那一座京畿辅郡，西岳脚下的州城。

## 形貌

华州城坐在关中平原东端，渭水南岸的台地上，南望华山、东接潼关。

州治在**郑县**——城在渭水之南、华山西北约二十唐里许。城周不大，夯土城墙，开四门；不及长安、洛阳之气象。城西去长安约一百八十唐里——自长安春明门东行，过灞桥、新丰、临潼即至，是长安东出第一座郡治。城东去潼关约八十唐里——州境最东端便是潼关塞，黄河与渭水合流处之南，关塞踞在崤函古道与黄河之间，扼东出河洛、西归关中的咽喉。

华州**最显的山是华山**——五峰拔地、纯花岗岩、海拔南峰二一五四米，关中盆地东缘第一座神山。山北脚下是西岳庙——汉武帝集灵宫旧址，盛唐国家祀西岳之正坛，每岁立秋遣使致祭。州境的"南屏一线"便是华山——城南望去，崖壁千仞如一面石屏。**华山地理细节见独立 doc**，详见 [华山](../scenic/华山.md)——本 doc 不重复五峰、上山道、西岳祀典诸细节。

物候是关中东端的样子：夏热冬寒，春秋多风；渭水自西而东贴城北流，北岸是同州冯翊一线。秋深时华山顶云海漫至崖腰——华州城里仰头便见此景。

## 大唐天宝十四载（755年）

华州在唐人眼里是一座**自西魏立州、至盛唐已二百一十余年**的京畿东辅。

汉时此地属京兆尹下辖之郑县——郑县之名追到春秋郑国东迁前的姬姓郑邑（虢国之西、华山之北），秦始皇置县，汉因之。**西魏废帝二年（553）始置华州**——以境内华山得名，治李润堡（在今华州区西北），辖郑县、华阴诸地；"华州"这一名号自此而立。**北周明帝武成元年（559）**移州治至郑县——自此一千四百年间，华州治几度迁徙但主体始终在郑县一带（郑县即今陕西渭南市华州区治所）。

**隋大业三年（607）**罢州为郡——华州废，并入京兆 / 冯翊；**唐武德元年（618）**高祖复置**华州**——这是唐开国后关内道州制的第一笔。**天宝元年（742）**玄宗诏天下州改称郡——华州改称**华阴郡**。**乾元元年（758）**肃宗复改郡为州——华阴郡又复称华州。755 这一刻，城门匾额上写的还是"华阴郡"。

755 这一年华州属**关内道**——是盛唐**关内京畿近辅**之一（与同、岐、邠并列，长安外围拱卫京师的几郡）。郡望按《通典》《元和郡县图志》记天宝间华阴郡户约三万五千余、口约二十万——略低于同州冯翊郡（户五万八千余），高于岐州扶风郡某些散数据；**正在"上州"门槛附近**——盛唐州郡分等以四万户为上州门槛，华阴郡数年间可能踩线浮动。

郡治郑县城内的官署是盛唐州郡通制——华阴郡太守、郡丞、录事参军、司功 / 司仓 / 司户 / 司兵 / 司法 / 司士六曹参军一套。**京畿辅郡的特点是**：本郡刺史（郡太守）一职朝廷颇重，本郡仓廪、驿马、关防与长安直接挂钩——华阴郡尤其是潼关的"后勤背靠"，潼关守军的粮秣、军马，多自华阴郡调发。

对当时人而言，华州的分量大致如此：**华阴郡是长安的东大门、西岳华山的脚下州、潼关塞的依托郡**——并非节度治所、并非大都督府，但盛唐凡涉及关中东出、潼关防务、华山祀典三事的官文书，多要过华阴郡署。州内最显的两个地标——华山与潼关——分别系着国家祀典与京畿防务，是华州在盛唐版图上**南祀北防**的两面。

755 年冬安禄山起兵——消息沿驿道传到华州时，潼关一线最先紧起来。城里也开始备粮、整军、修关——但城里人尚不知道，再过半年这条防线将怎样塌下来。

## 历史长河

**756 潼关之败**　至德元载（756）六月，安史叛军逼近潼关。玄宗诏哥舒翰率河西、陇右、朔方诸镇兵共**二十万**屯潼关——这是盛唐关中防线最后一次大集结。哥舒翰本欲扼险拒守，朝廷催战；六月初四出关与崔乾祐战于灵宝西原，**全军覆没**，哥舒翰被俘投降。潼关失守——**华州亦随之陷叛军**。十日后玄宗夜出延秋门西奔入蜀。**这场仗的战场不在华州城内，但华州境上的潼关一塌，关中尽失**——华州在 755 之后的命运，自此一变。

**杜甫华州司功**　乾元元年（758）肃宗复置华州。**杜甫于乾元元年六月由左拾遗贬为华州司功参军**——掌教化与文翰；时年四十七岁。在华州约一年（758 夏-759 秋），作《华州留别》《题郑县亭子》等，并目睹安史乱后关中残破——本年作《新安吏》《潼关吏》《石壕吏》（"三吏"）即在自华州出洛阳途中所作。乾元二年（759）秋杜甫弃官西去秦州——华州一段是杜甫由朝官转为民间诗人的转捩点。**盛唐华州在唐史上最重的一笔，便是 758-759 这一年杜甫在此**。

**中晚唐**　乾元元年（758）肃宗复改华阴郡为华州。中晚唐藩镇并立，华州几度成关中东路的军镇——德宗建中四年（783）泾原兵变后，奉天、华州一线皆成行在外围；晚唐黄巢之乱（880-881），起义军自洛阳西攻潼关、入长安，华州亦数度易手。**唐末**朱温取关中，华州落入梁手。

**五代至宋**　五代华州数易其主。后周显德间一度升华州为**镇国军节度**——以其当潼关冲要。**宋初**仍置华州——属永兴军路；元丰间为**华阴郡军事**州。宋代华州著名一笔是**北宋寇准** 太平兴国年间（约 980 前后）任华州通判——是寇准初仕。

**金元明清**　金代华州属京兆府路；元至元间改属奉元路，省郑县入华州本州直辖——这是郑县作为县名第一次消失。明清华州皆属西安府——为散州，仍治郑县旧城；雍正三年（1725）同州升府之改，华州未升，仍作西安府辖之散州，但因华山祀典之故，地位仍重。

**民国至今**　民国二年（1913）废州存县，**华州改为华县**——"华州"作为千年州名退出现行行政地名。**1958 年**华县与渭南县合并；1961 年复分。**2015 年**国务院批准撤华县设渭南市**华州区**——"华州"两字以"华州区"形式重新回到地图上，距 1913 废州已一百零二年；自 553 年西魏置华州起算，本名号断断续续使用了一千四百六十余年。

至于盛唐那座华州——州城残基已埋入今华州区下，郑县官署旧址不可考；唯一一处可以与唐华州人共看的，是华山——五峰仍在，西岳庙的主体建筑虽是明清规制，但庙址未动；潼关旧塞已被三门峡水库淹没，唯**杜甫"三吏"那几行诗**——在华州一年里写下的——一千二百多年没动过。

## 引文

《元和郡县图志·卷二·关内道二·华州》（李吉甫 813）：

> 华州，华阴。开元户三万一千八百六十二，乡七十六。……禹贡雍州之域。春秋时秦地。西魏废帝二年置华州。隋大业三年废。皇朝武德元年又置。

——这是 755 年前后华州建置、户口的一手史源。

杜甫《题郑县亭子》（乾元元年 758 任华州司功参军时作）：

> 郑县亭子涧之滨，户牖凭高发兴新。
> 云断岳莲临大路，天晴宫柳暗长春。
> 巢边野雀群欺燕，花底山蜂远趁人。
> 更欲题诗满青竹，晚来幽独恐伤神。

——杜甫在华州州治郑县亭中作。"云断岳莲临大路"——"岳莲"即华山五峰之西峰（莲花峰）——华州城南望华山，是杜甫此诗的实景。
