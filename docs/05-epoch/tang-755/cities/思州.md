# 思州

## 一句话

黔中道治下、乌江上游一座山深谷狭的下州——治务川县（今贵州铜仁市沿河土家族自治县东、乌江东岸；唐务川县治所，宋后迁治今务川县城），742-758 年间官称"宁夷郡"，是黔州都督府西南向苗、仡佬诸部投放秩序的偏边一站。

## 形貌

思州在**乌江上游一处山谷里**——州治在务川县（今贵州铜仁市沿河土家族自治县东、乌江东岸；唐务川县治所，宋后迁治今务川县城），城下是乌江一支洪渡河，城外四面是武陵山西南支脉的山岭。海拔自河谷六百米直上山脊一千五百米——巴山尾、武陵山、苗岭三股山势在此交叠。州境东接施州（今湖北恩施）、北接黔州（今重庆彭水）、西邻费州、南接夷州，山高谷深，沿河一线之外多数地方人迹罕至——**当地的人，唐人称"溪洞蛮""徭僚"，是日后苗、仡佬族的先民**。

## 大唐天宝十四载（755年）

思州沿革起得偏晚。**隋开皇十九年（599）**置**务川县**——'务川'之名从此立。**唐武德四年（621）**置**务州**——治务川县。**贞观四年（630）**改务州为**思州**——此为'思州'之始。**天宝元年（742）**玄宗诏天下州一律改称郡，思州改称**宁夷郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州，宁夷郡再称思州——但 755 这一刻，城门匾额上写的还是'宁夷郡'，民间口语仍称'思州'。

天宝十四载（755），思州官称**宁夷郡**——隶**黔中道**、属**黔州都督府**所辖正州之一。《新唐书·地理志五·黔中道·思州宁夷郡》载思州**'下州'**——户口数与黔州相若或更薄，是黔中道羁縻浓厚边裔诸州里的偏边一员。**此刻的思州太守**（按 742-758 郡制刺史官称'太守'）一身领郡事，下辖务川、思王、思邛三县——汉编户按下州常制办租调贡赋；谷涧之内的羁縻部众由部酋世袭其官、受黔州都督府授敕。长安诏令南下经襄州、江陵、夔州、涪州、黔州一线再溯乌江上至思州，约两月余始至。十一月初九安禄山起兵于范阳——消息传到思州，约已是这一带最远的一段静寂里。

## 历史长河

**中晚唐至五代**　乾元元年（758）复州——思州之名重归。安史乱后黔中戍卒更薄、羁縻部酋自治更甚；唐末入前蜀，五代之间先后属前蜀、后唐、后蜀，管辖虚浮，**思氏部酋自唐末世袭其地**。**朱砂、水银**虽自汉以来为务川一带名物、唐时入贡，**但'思州朱砂'之名大盛在宋以降**——唐时仍以黔中朱砂统称。

**宋元明清**　北宋仍称思州；**元置思州军民安抚司**，治务川；**明永乐十一年（1413）**朝廷平思州、思南二宣慰使田氏内讧，**改思州、思南诸地为八府**，同年置**贵州布政司**——'贵州'作为正式省级名号自此而立。**但贵州省治在贵阳，与唐思州治务川绝非同地**——'唐之思州 ≠ 后世贵州省'，此条须严守。明清思州府治后移今贵州岑巩，**唐思州治务川县另置务川县属思南府**。

**民国至今**　民国废府存县；1949 年后属贵州省遵义专区。**1986 年改设务川仡佬族苗族自治县**——'仡佬''苗'入县名，是这一带千五百年部族史的一次正式承认。今为**贵州省遵义市务川仡佬族苗族自治县**，承唐务川县治旧址。

## 引文

> 思州，宁夷郡。下。本巴郡涪陵县地。隋开皇十九年置务川县，属庸州。唐武德四年于县置务州。贞观四年改为思州。天宝元年改为宁夷郡。乾元元年复为思州。
