# 绵州

## 一句话

剑南道东北部涪江上游一上州——治巴西县（今四川绵阳一带），金牛道自剑州南下经此可至汉州、益州；州境昌明县青莲乡是李白童年所居。

## 大唐天宝十四载（755年）

绵州治**巴西县**，在涪江上游南岸——金牛道自北面剑州南下，经绵州过汉州（雒县）即至益州（成都），是入蜀干道在剑南腹地的一段。州属**剑南道东北部**，**天宝元年（742）随玄宗朝"改州为郡"通制改为巴西郡**，755 时正名作"绵州巴西郡"，两名在唐人嘴里互换不分；**乾元元年（758）肃宗复改为绵州**——但这是 755 之后的事。沿革：**隋大业时为金山郡**；**唐武德元年（618）改为绵州**，领巴西、昌隆（玄宗朝避讳改昌明）、涪城、魏城、金山、万安、神泉七县；武德三年分置显武、龙安、文义、盐泉四县；七年省金山县；贞观元年又省文义县。**《旧唐书·卷四一·地理四·剑南道·绵州巴西郡》列为上州**，载：**"旧领县九，户四万三千九百四，口十九万五千五百六十三。天宝领县九，户六万五千六十六，口二十六万三千三百五十二"**——755 时绵州在册**户 65,066、口 263,352、领县九**（巴西、涪城、昌明、魏城、罗江、神泉、盐泉、龙安、西昌），户口较武德旧籍增近五成，是剑南东部仅次于梓州、益州的繁庶州治之一。州境涪江自西北山地（龙门山东麓）下来，绵州城下江面渐阔，可行舟楫顺流至遂州合涪江—嘉陵江干流；**昌明县境青莲乡**（在绵州东北）——**李白幼年随父自西域碎叶迁入，于此读书长成至开元间出蜀**，是 755 时绵州人最津津乐道的乡邦旧事。

## 历史长河

**中唐**　杜甫宝应至广德间（762-765）流寓剑南，自梓州、阆州转徙间数度经金牛道，或过绵州境，留诗散见集中——已是安史乱后事。

**五代至明清**　前蜀、后蜀据绵州为剑南东北重镇。北宋属梓州路（后利州路），州治仍在巴西。明清绵州属四川承宣布政使司 / 四川省，为成都府属直隶州。

**民国至今**　1913 年废州改绵阳县属四川省——"绵阳"二字始为建制名。1976 年析绵阳县东境置江油市；**1985 年撤绵阳地区设地级绵阳市**——"绵阳市"乃近代建制名，严守 755 正文不预支。今绵阳市辖境约略相当唐绵州故壤，市区即唐巴西县城方位之承；江油青莲镇尚存李白故里诸迹。

## 引文

> 绵州巴西郡，上。隋金山郡。武德元年，改为绵州……旧领县九，户四万三千九百四，口十九万五千五百六十三。天宝领县九，户六万五千六十六，口二十六万三千三百五十二。

——《旧唐书·卷四一·志第二十一·地理四·剑南道·绵州巴西郡》
