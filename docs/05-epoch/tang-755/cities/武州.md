# 武州

## 一句话

山南西道西北端一座下州——治将利县（今甘肃陇南武都老城一带），白龙江上游山间小城；西接吐蕃、北邻党项、境内氐羌杂居，是大唐西陲山地一隅。

## 大唐天宝十四载（755年）

武州治**将利县**，在白龙江上游北岸——江水自西而东切入秦岭余脉与岷山东延的夹谷，城在两山一江之间的一片狭窄台地上。州属**山南西道西北端**，越境向北便是陇右道的成州、秦州，向西即吐蕃、党项、氐羌的活动地带——本州境内汉化已久，但山谷里仍多氐人、白马氐遗种与新附的党项部落杂居。沿革在大唐人眼里清楚而短：汉武帝元鼎六年（前 111）平氐置**武都郡**——"武都"二字自此立，是氐人故地的旧名；隋大业三年（607）州改郡通制，**隋武都郡**领将利、覆津、盘堤诸县；**唐武德元年（618）改武都郡为武州**——治将利，属山南道；开元二十一年（733）分山南为东西二道后属**山南西道**。**天宝元年（742）**玄宗诏改州为郡，武州复改**武都郡**——这是 755 当年的官称。**乾元元年（758）**肃宗复改郡为州——武州之名再立。**742 至 758 复州前正名"武都郡"，民间口语仍呼"武州"**，两名并行。

755 年的武州在册**领县三：将利（州治）、覆津、盘堤**——皆白龙江沿岸或山间小县。《新唐书·卷四十·地理志四·山南西道·阶州武都郡》后来追述本州户口（即天宝十一载/752 籍）：

> **阶州武都郡，下。本武州，因没吐蕃，废，大历二年复置为行州，咸通中始得故地……户二千九百二十三，口万五千三百一十三。县三。**

——755 时武都郡在册**户 2,923、口 15,313**，是山南西道户口最少的几州之一，与兴州（4,756 户）、文州、扶州相侔——典型的西陲山地小州。武都郡太守署、将利县衙、馆驿俱在城内，规模都小；州境无折冲府兵之大数，无节度治所，亦无大驿——驿道止于城下白龙江畔。城东沿白龙江而下，可入兴州境接金牛道；城东北越山可至凤州接陈仓道；城南越山可入剑南道；城西则是吐蕃、党项地界——是蜀道北翼几条山间小路在白龙江上游的一处汇点，但远不及兴、利、凤诸州的驿务繁忙。

地势是本州的命。白龙江自西北而东南切谷而下，两岸是岷山东延与秦岭余脉的高山——城内人抬头便见雪线。气候上属秦巴山地北缘——年降水七百毫米左右，冬寒夏凉，山田多产麦、粟、漆、蜂蜜，江岸狭窄不宜舟楫大行。本州在大唐人眼里是"西陲下州"——名虽列于册，事则不闻——天宝十四载冬安禄山反范阳，消息沿陈仓道一线南传，到武州时已是末梢，太守是这一隅最远的一位听讯者。

## 历史长河

**中唐**　**宝应、广德间（762-763）吐蕃乘安史之乱东出，河陇渐陷**——武州、岷州、宕州、洮州相次陷蕃（《资治通鉴·卷二二二》宝应元年至广德元年记吐蕃陷河陇诸州事）。武州自此**没于吐蕃近百年**——大历二年（767）唐曾"复置为行州"，但只是侨置一虚名于他境，州治将利仍在蕃手；至**咸通（860-874）间始收复故地**——此为《新唐书》"本武州，因没吐蕃，废，大历二年复置为行州，咸通中始得故地"一段所记。755 当年武州尚在唐手，城里人未知再过七八年这座下州便会沉入吐蕃版图——这是 755 之后的事，本 doc 主体不预支。

**五代两宋**　**后周显德二年（955）**改武州为**阶州**——"阶州"二字自此立（"显德"为后周年号 954-960，与北宋无涉），"武州"之名自唐末废，本朝再不复用。入宋后阶州属利州西路（一作秦凤路），治福津——治所自将利东迁，与唐武州治不再同址。南宋时阶州一度为宋金交界，吴玠、吴璘抗金亦及此境之西。

**元明清至今**　元仍置阶州属巩昌路。明洪武中降为阶州直隶州。清雍正七年（1729）升为阶州直隶州属甘肃省。1913 年废州存县——后世几度并析，今为陇南市治下。唐武州治将利县城旧址，约在今甘肃陇南境内白龙江北岸老城一带——盛唐州城残基已不可辨；"武州"两字，只在《旧唐书》《新唐书》《元和郡县图志》和后世文人偶拾的几句山行诗里留存。
