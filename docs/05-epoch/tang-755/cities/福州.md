# 福州

## 一句话

江南东道长乐郡治，唐人口中的"福州"——闽中山间一片冲积小平原上的州城，闽江入海前最后一段台地之畔；**福建经略使**所驻，自此南去是泉、漳、潮诸州，北上越仙霞岭便入两浙。

## 形貌

福州不在江南腹地——是江南再南，岭外山中一处难得的平地。

自长安南下，过商於、襄州、洪州，再过虔州、汀州，越武夷山脉之东缘——便入闽中。另一道自洪州东出信州、衢州、处州，转南越仙霞岭，过建州、剑州——亦可抵福州。两道皆隔山——闽中**七山二水一分田**之地，自古便与中原、江南各自一域。

**闽江**自西北山中三源汇流而下——一路东南穿建州、剑州山谷，至今福州西郊一带切出一段冲积平原，再折东南入海。福州城就盖在闽江北岸的一小片台地上——**屏山**在北、**于山**在东、**乌山**在西南，三山鼎立，城就嵌在三山之间的一处洼地。城东南面是闽江本支——汛期江面阔达数里；东去三十余里便至**鼓山**——闽江入海前最后一座高峰，海舶北来南去皆以此山为望。再东至海口约百余里——这是福州与海的距离，比广州、交州都远些，海舶不直入城。

物候是江南中偏南——冬无严霜、夏热湿；五月至九月一段多雨多台风（唐人尚未有"台风"之称，但《元和志》《通典》皆载闽中"夏秋多飓"），秋冬转旱。城外阡陌不广——闽中山田为主，稻一年一熟或勉强两熟；山间多种荔枝、龙眼、橄榄、橘柚——盛唐贡品中"福州荔枝、龙眼"已入《新唐书·地理志》土贡条。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

福州在唐人眼里不算新地——但建置数易，名号也最乱。

**汉武帝建元三年（前 138）东瓯王内附、元封元年（前 110）灭闽越**——闽越王无诸故都**东冶**（即今福州一带）被废为县，徙其民于江淮间——闽中一度近乎空地。东汉以下复置冶县、东侯官；三国吴永安三年（260）置建安郡——治建安（今福建建瓯），福州一带在其辖下。**隋开皇九年（589）平陈**，闽中复入中原王朝；**开皇十二年（592）置泉州**——州治在今福州——**这是"泉州"二字最早的官称，指今福州，而不是后世闽南那座泉州**。大业初（605 前后）废泉州，改属建安郡。

入唐之后名号几度更易。**武德五年（622）**改建安郡置**建州**——治建安。**武德六年（623）**自建州析出建州东南、即今福州一带，**仍称"泉州"**——**这是隋至唐武德间"泉州 = 今福州"之号，与后世闽南泉州非同地**。武德八年（625）置都督府。**唐景云二年（711）**——为区别新析出的闽南泉州（即今福建泉州），将此处之"泉州"改名**闽州**；同年从闽州南境析出新州，仍称"泉州"——自此"泉州"二字南移至闽南，今福州专作"闽州"。**开元十三年（725）**改闽州为**福州**——"福州"之名自此入官府正式文书。**天宝元年（742）**玄宗诏天下州改称郡，福州改称**长乐郡**。**乾元元年（758）**肃宗复改郡为州，长乐郡复称福州——此乃 758 年以后的事。

唐人若在 755 年驻足回望：自汉武灭闽越（前 110）至此已八百六十五年，自隋置泉州（592）至此一百六十三年，自玄宗改名福州（725）至此三十年——这片闽江口的小平原，名号在一百五十年里转了一圈又一圈：东冶 → 建安 → 泉州（隋唐武德）→ 闽州（711）→ 福州（725）→ 长乐郡（742）——这一年城门匾额上写的是"长乐郡"，里头的人多半还是叫"福州"或仍记得旧"闽州"之号。"**福州**"二字 755 年时入官府文书才整三十年，是个相对新的名号。

### 此刻

天宝十四载（755）的福州，行政上是**江南东道长乐郡治、闽县治**——郡治县治同在一城。长乐郡领**闽、侯官、长溪、长乐、福唐、连江、古田、尤溪**等县（盛唐户口建置略有出入，《元和志》以闽县为附郭）——一郡八县之地，是闽中诸郡里建置最稳的一处。**长乐郡太守**多兼**福建经略使**——这是盛唐玄宗朝在闽中设置的边方军政长官：唐前期"福建经略使"治福州，统辖福、建、泉（即闽南新泉州）、漳、汀诸州——是大唐对闽中山地与海岸的一套节度前奏体制。755 年福建经略治福州（长乐郡），由本郡太守兼摄；正式独立的"福建节度使"要到晚唐**乾宁三年（896）王潮**奏置威武军节度使时才出现——755 当年还没有这个官名。

人口据《元和郡县图志·卷二九》载福州**开元户三万一千六十七、元和户一万九千四百五十五**——本 doc 此数明取自**元和志**（《旧唐书》《新唐书》地理志另载户 34,084、口 75,876，与元和志数略异，两组数不可混用、不可张冠李戴）。开元户三万一千余口是**中州门槛**之上——盛唐州县按户口分上、中、下，福州正在中州一档；元和户一万九千四百余则反映安史之后、元和初年（813 前后）闽中户口的收缩。闽中山地里这是一个不小的数字——但比起江南诸大州（润、苏、杭、越、洪）仍显单薄。城内人口多为汉人移民后裔——自三国吴、晋永嘉之后中原南渡，闽中遂有汉人州县；城外山间仍多越人遗族，唐人通称"**山越**"或"**畬民**"——多在山中刀耕火种，与城内汉人州县体制平行而少相属。

755 年冬，安禄山起兵的消息沿驿道自长安南传——过商於、襄州、洪州、虔州、汀州，**抵福州大约需三十余日**。福建经略接到的诏令大约是：戒严、守城、备粮，并防境内山越乘隙——但闽中山深路远，朝廷并未抽调福州兵北上。次年（756）玄宗入蜀、肃宗灵武即位，江南诸道亦各自为政；福建经略在此期间仍奉唐正朔，但实际上自管自的——这是盛唐边远州郡在安史之乱中的普遍处境。**至德至乾元间（756-758）**江南连年漕粮北运援唐军——福州虽近海，但因不在主漕道（主漕道是江南运河 → 淮南 → 河南），福州的贡赋多走山路经洪州北上，量也有限。

对盛唐而言，福州是**江南东道之尽南、闽中山间一处难得的州治、福建经略所驻、海舶北上明州（宁波）途中的一处可泊之港**——但还不是后世意义上的"福建省会"，更不是大港。盛唐三大对外港口仍是广州、交州、扬州（或明州）——福州港尚未在《广州通海夷道》《市舶使》等盛唐官方海贸记载里独立成名。它的崛起要等到**晚唐五代王审知治闽**（898-925）以后——那是一百四十年后的事。755 年的福州港，是一处地方性海口，胡舶偶来，主要还是为闽中山货、海产与江南诸州之间的转口。

## 子条目

### 福州城（闽县治）

唐长乐郡治、闽县治——郡治县治同在一城。城在闽江北岸台地，**三山**（屏山、于山、乌山）环抱——"三山"自此入福州城地理。城周史料未明确——南宋《三山志》追溯唐子城周约九里许，但 755 时城制更小；唐**贞元间**（785-805）观察使柳冕筑罗城——这是福州城形制大扩之始；755 时距柳冕筑罗城尚有三十年，城规模约止于子城一围。城内有长乐郡署、闽县治、馆驿、佛寺数所、市场。**开元寺**（今福州开元寺，前身为南朝梁太清三年/549 始建之"灵山寺"）在 755 时已建有四百余年——盛唐玄宗朝**开元二十六年（738）**诏天下诸州置开元寺，福州灵山寺改名开元寺；755 时正是开元寺立名第十八年。

### 闽江

闽中第一大江——自西北三源（建溪、富屯溪、沙溪）汇流而下，过建州、剑州山谷，至今福州西郊一带切出冲积平原，再折东南入海。江面在福州城外宽阔——汛期可至数里。唐人航行可自福州沿闽江上溯至剑州（今南平）、建州（今建瓯）——再北越仙霞岭可入两浙衢州。下行可循海岸南至泉、漳，北至温、台、明诸州。**闽江入海口**在今福州东南约百余里处，海舶北来南去皆经此——但海舶多不直入福州城下，而泊于江口外港。

### 鼓山

福州城东三十余里，闽江入海前最后一座高峰——海舶北来南去皆以此山为望。海拔近千米，山势峻拔——盛唐时山中已有寺院；后世**涌泉寺**（唐建中四年 / 783 始建）名重东南，但 755 时尚未立——此刻鼓山只是一座地理坐标式的山，未成宗教名山。山顶巨石（"鼓"形）传说风雨时鸣如鼓——"鼓山"之名由此。

### 屏山·于山·乌山（三山）

福州城内三山——**屏山**在城北、**于山**在城东、**乌山**在城西南。三山自唐至今未变——"三山"自此入福州城地理与诗文。屏山高不过五十丈、于山乌山更低——但鼎立环抱城中洼地，是福州城最显著的地形标识。后世福州因此有"三山"别号——但 755 年时尚未入正式官称。

### 闽越国故都·东冶（汉之遗事）

福州城所在地，**汉武帝灭闽越前的闽越王无诸故都"东冶"**——汉武帝建元三年（前 138）东瓯内附、**元封元年（前 110）灭闽越**——徙其民于江淮间，闽中一度近乎空地。东冶旧址在今福州城北屏山一带——南宋时尚可指认"冶山"遗址，今**冶山遗址**（福州市鼓楼区）尚存——盛唐 755 年时距灭闽越已八百六十五年，**冶城**实物已无人见过，仅作传说存于本地士人辞典。**无诸**则在闽中世代被祀为城隍式祖神——盛唐时福州城内已有"闽越王庙"奉祀无诸（确切建年史料未稳，可能晚至宋初）。

### 福建经略使

盛唐玄宗朝在闽中设置的边方军政长官——**治福州（长乐郡）**，统辖福、建、泉（闽南新泉州）、漳、汀诸州——是大唐对闽中山地与海岸的一套节度前奏体制。**福建经略使**之名始见于盛唐《唐六典》及《元和志》——具体设置年代史源略异，可考者**开元二十一年（733）**置十五道采访使时已有福建经略之名号。经略使例由本郡（长乐郡）太守兼摄——下设经略判官、推官——一套小型的边方军政幕府。岭南五府经略治广州，统辖广、桂、邕、容、安南五管——福建经略与之并列，但规模、辖境、品秩皆远逊。**福建节度使**之独立设置要到晚唐**乾宁三年（896）王潮**奏置威武军节度使时——755 当年还没有这个官名。

### 山越·畬民

福州城外、闽中诸山间——盛唐时仍有大量越人遗族，唐人通称"**山越**"或"**畬民**"。"山越"之名自三国吴时入文——**孙权**屡讨山越，置闽中诸县；唐人沿用此名指闽、浙、赣山间未入州县体制的越人后裔。**畬民**则是后世（南宋以后）对此类人群的专称——盛唐时尚未成正式人群名号。山越居山间刀耕火种、以稻、粟、芋为食——与城内汉人州县体制平行而少相属。盛唐至中唐闽中山越屡有起兵——但规模皆小，不入正史；地方州县多以羁縻、招抚为务。755 年时福州治下山越的具体动向史无明载——但福建经略的日常要务里，必含此一项。

## 历史长河（755 → 今）

**中晚唐**　乾元元年（758）长乐郡复称福州。**贞元间**（785-805）福建观察使**柳冕**大筑福州罗城——这是福州城形制大扩之始。**元和初**（806 前后）福建观察使**裴肃**经营有方。中晚唐闽中渐繁——海贸渐盛，福州港始有胡舶往来；但仍不及广州、明州。**唐昭宗乾宁三年（896）王潮**自汀州起兵，克福州——朝廷封王潮为**威武军节度使**，治福州——**福建节度自此独立**。王潮卒（897）后弟**王审知**继位——**唐昭宗光化元年（898）至后梁开平三年（909）**王审知治闽——以福州为治、轻徭薄赋、招纳中原士人南迁、开发海贸——福州自此从一座中州小郡跃为东南大州。**后梁开平三年（909）**朱温封王审知为**闽王**——**闽国**自此立国，仍都福州。

**五代**　**闽国（909-945）**——王审知至王延政四代——皆都福州。闽国时期是福州历史上第一段"自立"岁月——城制大扩（王审知再筑外城）、海贸大盛（大食、波斯、新罗、日本海舶皆有至福州者）、佛教大兴（雪峰寺、鼓山涌泉寺皆此期立或扩建）。**后晋开运二年（945）**南唐军灭闽——闽国亡，福州短暂为南唐辖；旋为吴越所取——**吴越（947-978）**辖福州凡三十年。**北宋太平兴国三年（978）**吴越纳土归宋——福州入北宋。

**宋元**　北宋福州为**福建路治**——"福建"二字作路名自此始；福州为路治、安抚使治。**南宋淳熙九年（1182）**梁克家撰**《三山志》**——福州城最早现存方志，记唐子城、罗城、夹城、外城沿革；今所知 755 时福州城形制，多自此志追溯。**南宋祥兴元年（1278）**元军南下，福州陷——南宋帝昰、帝昺在福州、泉州、潮州一带流亡——**祥兴二年（1279）厓山海战**南宋亡。元至元十五年（1278）设**福州路**——治福州，辖境略同唐长乐郡。

**明清**　明洪武元年（1368）改福州路为**福州府**——"福州府"之名自此入官府正式文书，自明至清五百余年未变。明嘉靖间（1522-1566）倭寇屡扰闽海——福州亦在其列；**戚继光**于嘉靖四十一年（1562）平闽海倭——福州得安。**清初郑成功**据闽海——顺治十六年（1659）攻福州不克；**清康熙二十二年（1683）**台湾内附，闽海平。**清光绪十年（1884）中法马江海战**——法军舰队袭福州马尾港、毁福建水师——这是福州近代最惨痛一役。

**民国至今**　1913 年废福州府，置闽侯县——1946 年置福州市。1949 年福州为中华人民共和国福建省省会至今。今福州市为福建省省会，闽江下游冲积平原腹地——人口逾八百万；旧唐子城、罗城遗址在今鼓楼区一带，**冶山遗址**（汉闽越王无诸冶城所在）尚存，2013 年列为全国重点文物保护单位；福州**三坊七巷**历史文化街区（明清里坊格局）2009 年列入中国十大历史文化名街——但这是明清层，盛唐长乐郡的城基址叠压在其下。

唐人称"福州"的那座城——开元十三年改名整三十年的一处中州——如今地名"福州"独存；从盛唐至今一千二百七十年间，城名号几度更易：东冶 → 建安 → 泉州（隋唐武德）→ 闽州（711）→ 福州（725）→ 长乐郡（742）→ 复福州（758）→ 福州府（1368）→ 福州市（1946）——"福州"二字从开元十三年（725）一直用到今天，是唐玄宗朝那一笔的延续。

## 引文

> 福州长乐郡，中都督府。本闽州，开元十三年更名。土贡：蕉布、葛布、海蛤、鲛革、文皱布、橄榄、荔枝、龙眼。户三万四千八十四，口七万五千八百七十六。

——《新唐书·卷四十一·地理志五·江南道·福州长乐郡》（节录 verbatim）。"开元十三年更名"是 755 年距此整三十年前的事；土贡条以**布、革、果**三类为主——闽中山地产蕉布葛布（纤维织物）、海中产蛤与鲛革（鲨鱼皮）、山间产橄榄荔枝龙眼（南方果品）——这是盛唐福州贡入长安的本地物产。"户口"取盛唐数据（旧、新《唐书》同载户 34,084、口 75,876）——长乐郡是中州门槛之上。《元和郡县图志》另载"开元户三万一千六十七、元和户一万九千四百五十五"，开元数略异、元和数则反映安史之后户口收缩，详下条。

> 福州，隋建安郡之闽县。武德六年置泉州，八年置都督府。景云二年改为闽州。开元十三年改为福州。天宝元年改为长乐郡。

——《旧唐书·卷四十·地理志三·江南东道·福州》。沿革之繁，跨隋至盛唐一百五十年——"福州"在 755 时是个用了三十年的新名号。

> 福州长乐郡，开元户三万一千六十七，元和户一万九千四百五十五。东南至大海二百三十里，西北至建州七百里。

——《元和郡县图志·卷二九·江南道五·福州》。这是 755 年前后福州境内可课于中原州县体制者的户数与方位——李吉甫成书于元和八年（813），故书中**开元户**为盛唐之数、**元和户**为成书当时之数；元和户较开元户折损近四成，反映安史之乱后闽中亦未幸免。
