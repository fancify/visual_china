# 扬州

## 一句话

天宝十四载冬，**长江北岸最富的一座城**——三段运河在此交汇，胡舶系缆，鉴真一年前自此第六次扬帆东渡。

## 形貌

扬州不是一座单层的城——是子城、罗城前后相叠，骑着蜀岗一路铺到长江北岸。

子城在北，蹲在蜀岗东端的台地上，是隋唐两代的官城——周长大约五里余，城内为大都督府衙、淮南节度使治所、仓廪与官署。墙是夯土包砖，南门正对一条直通罗城的官街。子城登高便望得见南面——蜀岗以南是长江冲积的低地，秋冬水退时一望无际，春夏漕船云集，桅杆几乎数不清。

罗城在南，是百姓与商贾的住居市廛——隋唐两代逐步扩成，包到江岸边的运河码头。罗城东西约三公里，南北约四公里，城周二十余里——比之长安罗城的"百八坊"棋盘要散得多。罗城里街巷不严格按坊网铺开，更多是顺着河道与官街自然生长的网络——这是江南水城的本格：先有水，后有路，再有屋。坊也有，但坊墙的拘束远不及长安——夜禁的鼓在扬州历来执行得松，《唐会要》记元和年间扬州夜市才被官方追认，实则盛唐已是事实。

城里最热的，是运河两岸。邗沟自城北引入，一路穿罗城而南，在城南汇入长江——这是隋大业元年（605）重凿邗沟与通济渠定下来的格局，**洛阳到江都，由这两段运河一气贯通**。江南河自城南渡江后接续——扬州因此成为大运河北段与南段的咬合点：来自洛阳的漕米北转长安，来自江南的盐、茶、布、瓷在此装船北上，来自岭南、安南乃至大食波斯的海舶溯长江至瓜洲渡，在城南换装。整座城像运河上一个巨大的转盘——三段水路在此交汇，物资在此换轴。

蕃坊在城南近水处。波斯、大食、新罗、日本、昆仑奴杂处，几座清真院（"大食法寺"）、波斯邸店、新罗坊散落在城东南一带——具体方位文献不传，但《旧唐书·田神功传》记宝应元年（762）刘展之乱平定后，"大食波斯贾胡死者数千"，可见盛唐扬州的胡商密度仅次于广州、长安。

蜀岗西端有大明寺，隋开皇间立——城里第一名刹，鉴真曾驻锡讲律。城北蜀岗与江之间，有雷塘——隋炀帝最后葬身之处，三十余年前的旧事，扬州人走过雷塘时还都指得出方向。城南二十里隔江相望，是瓜洲渡——南来北往的客人，凡涉江入扬，第一脚踏在瓜洲，第二脚踏在扬州罗城。

春日的扬州尤甚——长江北岸的暖湿与运河带来的舟楫一齐到，岸上柳色与城里灯火相照。"烟花三月下扬州"——这是李白送孟浩然时随口写下的一句，自此便成了这座城的代名词。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

扬州在唐人眼里不是新城——是一处至少经了八九百年的旧地。

吴王夫差在周敬王三十四年（前 486）筑邗城——这是扬州城址最早的一次定基，也是邗沟开凿之始。秦置广陵县；西汉吴王刘濞建广陵国都，景帝三年（前 154）"七国之乱"刘濞起兵于此而败——汉广陵故城的台基在唐人眼里还露在田野里。东汉、三国、东晋、南朝宋齐梁陈——扬州历经"广陵""江都""南徐州""南兖州"诸名，城址几度南迁北移，但蜀岗东端这一段台地始终被择作官城。

隋开皇九年（589）平陈，置扬州总管府于江都——这是"扬州"作为治所专名第一次稳定下来。**真正改变这座城命运的，是大业元年（605）的运河工程**：隋炀帝命尚书右丞皇甫议征丁百余万，开通济渠（洛阳-淮安）、续凿邗沟（淮安-江都）——一段半新一段旧，两条水路接通，使江都到东都一千余里水程贯通。江南河（江都-余杭）于大业六年（610）再开——大运河四段中三段以江都为节点。隋炀帝自大业元年起三幸江都，至大业十四年（618）三月被宇文化及缢杀于江都宫——"江都之变"。炀帝本葬吴公台下，唐高祖武德五年（622）迁葬雷塘。

唐武德三年（620）置兖州，七年（624）改邗州，九年（626）复为扬州。贞观元年（627）分天下为十道，扬州属淮南道——下辖江都、广陵、高邮、六合、海陵、扬子、天长七县。武德七年（624）置大都督府——唐初十大都督府之一，先后由太宗诸子（吴王李恪等）遥领。开元十年（722）改置淮南节度使，治扬州——节度使衙署即在子城。**到天宝十四载，扬州已是"大都督府 + 淮南节度使治 + 江南漕运总枢"三重身份合一**。

### 此刻

天宝十四载（755），扬州刺史兼淮南节度使是陈希烈——朝中宰相兼地方节度的旧制（唐前期常见），实际镇守由副使主持。这一年扬州城里的事务，大致是漕运、互市、僧家这三宗。

漕运第一宗。开元二十二年（734）裴耀卿主持改革江淮漕运，建"江淮河漕"分段转运法——江淮漕米先集扬州，由邗沟北运至淮安转通济渠，至洛阳含嘉仓，再经渭水入长安。**天宝中年扬州年转米约二百万石**（《通典·食货十二》），是关中粮仓的命脉。扬州城南的运河码头与城内的官仓、转运院，是整个大唐漕运体系的咽喉。如果说洛阳是漕运的西端枢纽，扬州便是东端的起点。

互市第二宗。**"扬一益二"——盛唐至中晚唐天下第一富庶之城是扬州，益州（成都）次之**——这一谚自李肇《唐国史补》始见，反映的就是天宝前后的实情。扬州的富，富在三处：江淮盐铁、江南布帛、海外蕃货。盐铁是国家专营，扬州为江淮盐运使治；江南布帛集中在罗城东南诸市；海外蕃货来自长江口的瓜洲渡——大食、波斯、昆仑、新罗、日本之舶溯江而上，停泊于扬州城南运河口，在城内蕃坊换易丝绸、瓷器、茶叶后再回航。城里酒肆胡姬当垆，徐凝晚唐写"天下三分明月夜，二分无赖是扬州"——盛唐扬州的繁华之底，已在此年。

僧家第三宗。**鉴真和尚自天宝二载（743）起，五度尝试东渡日本均告失败**——天宝二载、三载、七载、九载、十一载（按《唐大和上东征传》记载），或被官府截，或遇风涛，或抱病暂止——其中一次失败漂至海南振州（今三亚）。天宝十二载（753）十一月，鉴真随日本遣唐使藤原清河、副使吉备真备、阿倍仲麻吕一行第六次东渡——自扬州龙兴寺（一说大明寺、文殊院诸说并存，《东征传》原文为"扬州"，未明寺名）出发，至天宝十三载（754）正月抵达日本萨摩国阿多郡秋妻屋浦，二月入太宰府，三月入奈良——授戒太上皇圣武、孝谦天皇及僧俗四百余人——日本律宗自此立。**天宝十四载冬，鉴真已在奈良东大寺戒坛院近一年余**，扬州大明寺中曾经驻锡的方丈，已成异国传法之祖。

### 此刻意味

对天宝十四载的扬州人来说，这座城意味着——**世界的另一端**。长安是天下中心，扬州是天下转盘。漕米、盐铁、瓷器、海舶、胡商、东渡僧——一切要走的，都从这里走；一切要进来的，都从这里进。

对江南的诗人来说，扬州又是另一重意味——是"下"的方向。李白送孟浩然"故人西辞黄鹤楼，烟花三月下扬州"，写于开元十六年（728）三月，那时孟浩然由襄阳东游扬州——"下"字暗合长江水势，也暗合时人对扬州的想象：那是要顺流而下、烟花作底色的去处。盛唐文人凡远游江南，扬州几乎必经——王昌龄、张若虚、孟浩然、李白、刘长卿、皇甫冉、戴叔伦、韦应物——每个人都到过。

这种繁华，他们尚未察觉，再过六七年就要第一次被血洗。

## 城内地标（doc 内子条目，不另立 POI）

下列各项均为扬州城内或紧邻地标，作 doc 内独立 `###` 章节锚定；不另立 `.md`。

### 子城

蜀岗东端台地上的官城——大都督府衙、淮南节度使衙、淮南转运院、扬州刺史府皆在此。城周约五里余，城墙夯土包砖，南门正对一条直街通罗城北门。1985 年起扬州考古所联合中国社科院考古所对子城进行勘探与发掘，确认了夯土城墙基址、东南角的水门，并清理出唐代衙署区的础石、铺地砖与"开元通宝"窖藏。子城西侧的台地上，正是后来南朝宋孝武帝起的"广陵旧宫"——隋唐为节度使别院。盛唐扬州的官权重心，便压在这片高地上。

### 罗城

子城以南，由蜀岗下铺到江岸边的居民商业城——东西约三公里，南北约四公里，城周二十余里，是唐扬州的本体。罗城里街巷不严守坊网，沿运河自然生长。坊有，但坊墙松，夜禁松，临运河一线全是商铺、邸店、酒肆、车马行——盛唐扬州的"市"已经溢出官定的坊市制度，是后来宋代"坊市合一"的先声。考古勘探出罗城南门、东门遗址；2013 年发现的"扬州城遗址东门"基址，为现存唐城东门的最完整一处。

### 邗沟

自城北引入、纵贯罗城、南汇长江——大运河四段中"邗沟"的城内段。最早为吴王夫差前 486 年开凿，隋大业元年（605）皇甫议征丁重凿。城内段宽十丈左右，南北直贯——两岸是扬州最密的码头、邸店、仓廪。漕船自此北上至淮安转通济渠，海舶自此南达瓜洲渡转长江——**这条水道是扬州之所以为扬州的根本**。

### 邗沟入江口

罗城南门外、近瓜洲渡的位置——邗沟在此汇入长江。盛唐时入江口位于今扬州市区东南、瓜洲古镇以西一带（具体坐标因长江北岸沙线南移千余年已不可详考）。漕船过入江口时要换轴：北来船降帆候潮，南去舶系缆候风。**唐人记"瓜步候潮"的瓜步即指此处**。

### 大明寺

蜀岗中峰，扬州城北。隋开皇间（**或作大明年间南朝宋初创**——《嘉庆扬州府志》两说并存）立，初名"大明"，唐代仍沿用。寺势踞蜀岗顶，俯瞰扬州全城。**鉴真和尚在此驻锡讲律多年**（《东征传》系统中作扬州大明寺、龙兴寺、白塔寺、文殊院诸说并存——史料对鉴真主寺有异文，通行作"大明寺"），是律宗江淮一脉的根本道场。天宝二载（743），日本僧荣叡、普照入唐求戒师，自洛阳追访至扬州大明寺——鉴真应请，自此一生六次东渡始。

### 龙兴寺

罗城内、近运河——鉴真东渡前后曾驻锡之寺。诸说之一以为鉴真第六次东渡 753 年自此寺启行——《东征传》原文未明确寺名，后人考据各执一词。龙兴寺、白塔寺、文殊院、大明寺四处皆有鉴真曾驻锡之说——盛唐扬州的佛门，便是这样以鉴真为中心串起来的。

### 法云寺

罗城内。盛唐扬州著名律宗道场之一。寺院有双桧——晚唐李廓诗"扬州法云寺双桧"，写的就是此寺。具体寺址今已不可考——扬州城内唐寺自宋以降几度被毁，明清重建多易址。

### 蕃坊

罗城东南、近运河南口。波斯、大食、新罗、日本、昆仑诸蕃商人聚居区——清真院（"大食法寺"）、波斯邸店、新罗坊、日本馆散落其间。具体方位文献不传，但**宝应元年（762）田神功平刘展之乱后，"大食波斯贾胡死者数千"**（《旧唐书·田神功传》《资治通鉴·卷二二二》）——盛唐扬州的蕃坊人口至少以万计，仅次于广州、长安。这段血史尚未发生时，蕃坊里的胡人正每日往返于运河码头与城里官署之间，做着大唐最远的生意。

### 淮南节度使衙

子城内。开元十年（722）置淮南节度使——治扬州，统淮南道八州。衙署在子城西南。淮南节度使首要职责并非边防（淮南腹地无重兵），而是**漕运护卫与盐铁专营**——是江淮财赋系统的最高军政长官。盛唐淮南节度使常以朝中宰相遥领（陈希烈、韦坚、杨慎矜等）。

### 江都驿

罗城北门外。淮南驿道的总驿——洛阳南来、江南北上的官员、使节，第一站皆在江都驿。驿道由扬州北经楚州（淮安）入通济渠，西北抵汴州（开封）；东南渡江入江南河，达润州（镇江）、苏州、杭州。日本遣唐使、新罗使、安南使凡北上长安洛阳者，皆经此驿。

### 转运院

罗城南门内、近运河——开元二十二年（734）裴耀卿"江淮河三段漕运"改革后所建。江淮漕米先集此院过秤、换装，再北运至淮安转通济渠。**院内官仓规模为唐天下转运院之最**——年转米二百万石的物理体现，全在这里。

### 二十四桥

罗城内运河上的桥群——"二十四"是个虚数还是实数，后人考据不一：沈括《梦溪笔谈》主张是实指城内二十四座桥；李斗《扬州画舫录》则以为是单座"二十四桥"。**晚唐杜牧"二十四桥明月夜，玉人何处教吹箫"**所写，已是会昌、大中年间（840s）的扬州——盛唐时这些桥皆存，只是没人把它写得这样有名。

### 雷塘

城北蜀岗与江之间——隋炀帝最后葬身处。武德五年（622）唐高祖迁葬于此——三十余年前的旧事。盛唐扬州人走过雷塘时还都指得出方向，但帝陵已无地面标识——直到 2013 年现代考古才在雷塘重新发现炀帝与萧后合葬墓。详见 [隋炀帝陵](../relics/sui-yang-mu.md)。

### 瓜洲渡

罗城南二十里，长江北岸——南来北往的客人涉江入扬州的第一渡。盛唐时瓜洲已是大江北岸最重要的渡口——海舶溯江至此换装、漕船候潮于此入邗沟。**张祜"潮落夜江斜月里，两三星火是瓜洲"**写的便是此渡。详见 [瓜洲渡](../transport/guazhou-du.md)。

### 江都郡

扬州的旧名 / 别名——隋大业三年（607）改州为郡，扬州改"江都郡"；唐武德三年（620）复改扬州；天宝元年（742）改郡名再次出现"江都郡"——天宝至德间唐廷曾短暂将州改郡，扬州又称江都郡。**所以天宝十四载这一年，扬州官方文书上写的名字是"江都郡"**，但民间与诗人笔下仍称"扬州"。乾元元年（758）复改州。

## 历史长河（755 → 今）

**中晚唐**　天宝十四载十一月安禄山反，扬州因远离河北未即陷——但江淮漕路即受冲击。至德元载（756）唐廷迁江淮漕路绕道东南，扬州地位反而上升——成为南方诸道供给关中的总后勤。乾元元年（758）扬州刺史邓景山因军务苛敛激变，第二年（759）平卢节度使刘展自宋州起兵反唐，史称"刘展之乱"——上元元年（760）刘展兵入扬州。**宝应元年（762），淮南节度使田神功奉命平叛——大军入扬州后纵兵掳掠，"大食波斯贾胡死者数千"**（《旧唐书·田神功传》卷一一〇）。这是盛唐扬州蕃坊的第一次血洗。胡商或死或散，扬州的"世界港"色彩自此褪去一截。

中晚唐扬州的繁华并未即止——德宗朝杜佑（《通典》撰者）任淮南节度使十数年，扬州仍是江淮第一城；至会昌、大中年间，杜牧两度入扬州，**"十年一觉扬州梦"、"二十四桥明月夜"——晚唐扬州的传奇是杜牧一手写出来的**。但僖宗乾符二年（875）王仙芝、黄巢起兵，江淮震动；中和三年（883）淮南节度使高骈幕府叛乱，毕师铎杀高骈，扬州陷于乱。光启三年（887）孙儒、杨行密争扬州，城屡易主——**唐末扬州的元气，是在这段诸藩混战里散尽的**。天复二年（902）唐廷封杨行密为吴王，扬州为吴国都——此时唐廷已是空名。

**五代至宋**　杨行密于天祐二年（905）卒于扬州，其子杨渥继位——这是杨吴政权之始。天祐十六年（919）杨吴改元，扬州为吴都江都府。南唐李昪升元元年（937）受禅，迁都金陵（江宁府），扬州降为江都府——但仍是南唐的"东都"。北宋开宝八年（975）平南唐，扬州归宋，称扬州军，后改广陵郡——**唐宋之交的扬州，已不再是天下第一城，但仍是江淮转运重镇**。北宋时漕运改走汴渠，扬州由"漕运总枢"降为"江淮节点"。仁宗庆历八年（1048）欧阳修知扬州——他在蜀岗大明寺旁建"平山堂"，**"远山来与此堂平"**，自此平山堂为扬州第一胜迹。神宗熙宁年间苏轼两次过扬州，至大明寺凭吊欧阳修——**唐扬州的繁华，已成宋人的怀古对象**。

南宋扬州为宋金前线——高宗建炎三年（1129）金兵入扬州，城残破；绍兴和议后扬州为南宋淮南东路治所。**南宋扬州的诗人是姜夔——淳熙三年（1176）冬过扬州写《扬州慢》"过春风十里，尽荠麦青青"**——距杜牧"春风十里扬州路"已三百二十年，距盛唐则已四百二十年。元至元十三年（1276）伯颜下扬州，置淮东宣慰司——蒙古时代扬州为江淮转运驻所，再次承担漕运职能。意大利威尼斯人马可·波罗或言其曾任扬州长官三年——史源存疑，但扬州在元代的国际地位由此可见。

**明清**　明洪武元年（1368）扬州府设——府治江都县。明扬州城是新筑的"旧城"（大致在唐子城罗城之间），周长九里——比唐罗城小得多，唐子城已废为田野。嘉靖三十五年（1556）增筑"新城"于旧城东——明扬州的"双城"格局，与唐"子城罗城"无关。明万历至清乾隆间扬州盐商鼎盛——两淮盐运使治在扬州，徽商盐商在城内建私园数十处，**"扬州瘦马"、"扬州画舫"、"扬州八怪"**皆此时事。康熙南巡六次、乾隆南巡六次，皆驻跸扬州——天宁寺、高旻寺、瘦西湖、平山堂俱在此时改扩。**清扬州的繁华是盐商的繁华，与唐运河的繁华隔了千年——但城里的水网、巷弄、寺观格局，仍能看到唐城的底纹**。咸丰三年（1853）太平军三入扬州——清军反复争夺，扬州城内宋元以来旧迹大半毁于此役。

**民国至今**　清末民国扬州渐衰——津浦铁路（1912 通车）走徐州、滁州一线，扬州被铁路绕开；运河也因黄河夺淮（1194 至 1855 期间黄河南徙）以来淤塞渐重，漕运地位移给镇江、南京。1949 年扬州专署设——后历经几次区划调整，1983 年地市合并为扬州市。1985 年起扬州考古所与中国社科院考古所联合发掘扬州城遗址——**子城、罗城基址首次走出文献回到地面**：南门、东门、瓦砾窖藏、唐代街道、唐代寺院础石陆续清理出来。1995-2013 罗城东门遗址完整发掘，2013 年雷塘考古发现隋炀帝与萧后合葬墓——这是唐高祖武德五年迁葬以来，炀帝陵第一次被现代考古确证。今天的扬州市区，明清旧城在中央，唐罗城在四周——子城遗址在蜀岗顶上辟为"唐子城遗址公园"，大明寺仍在蜀岗中峰（明代以后多次重修），平山堂、瘦西湖、瓜洲古镇分散在城外。

至于盛唐那座城本身——邗沟一线运河仍在（今扬州古运河），大明寺仍在原址，瓜洲渡因长江北岸沙线南移已退守一段——其他的，便都在脚下的层里。
