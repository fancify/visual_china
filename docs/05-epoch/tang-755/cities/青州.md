# 青州

## 一句话

齐地旧都的层叠之上、海岱之间那一上州——西去洛阳一千五百里、东北望渤海，新罗渤海使臣自登莱上岸西行入京，多经此城。

## 形貌

青州治**益都县**，城坐弥河西岸台地，南依**云门山、驼山**，北望渤海。**《尚书·禹贡》"海岱惟青州"**——古所谓青州，自泰山以东至渤海一片，唐青州正卡在这一片的腹心。城周不大，唐制属上州；夯土城垣开四门，城内有刺史署、北海郡太守衙、州学。城南云门、驼山间崖壁，自北朝起凿龛造像数处——是齐地佛教的小一处遗存，规模远不及龙门、麦积。物候属海岱东缘，比中原温润，比江淮干燥；冬日弥河不全冻，春深城南诸山有红叶。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

青州境是**两千五百年旧土**。《尚书·禹贡》九州之一，"海岱惟青州"——禹贡九州里东海一州即此。先秦时为**齐国**核心境：齐桓公九合诸侯、管仲相齐皆在此境之内；齐都**临淄**在唐青州境内**西北约百里**（今淄博市临淄区），是春秋战国第一大都，齐人列国称雄四百余年的根脚——临淄当独立 relic doc，本 doc 仅 cross-ref（详见 inventory）。西汉**齐郡**治临淄；东汉为齐国，三国魏置**青州刺史部**，治临淄。**北魏皇兴三年（469）**自临淄徙青州治**东阳城**（今青州市东北），州治始离临淄、移至今青州市一带——自此唐青州治益都县（北齐改名）之地始为青州主城，临淄退为属县。**隋大业三年（607）**改青州为**北海郡**；**唐武德四年（621）**复青州；**天宝元年（742）**玄宗诏改州为郡，青州复称**北海郡**；**乾元元年（758）**肃宗复改郡为州——青州之名再立，沿用至宋元。

### 此刻

天宝十四载（755）——官称"北海郡"，民间口语仍呼"青州"。河南道东部一上州，据《旧唐书·卷三八·地理志》青州条"天宝领县七"——辖益都、临淄、千乘、博昌、寿光、临朐、北海七县（属县沿革小异，《新唐书》《元和郡县图志》所列略有出入）。本郡户口据《旧唐书·地理志》记**天宝间约 7 万余户**——河南道东部一**上州**，比汴、宋那种漕运中段的十几万户大郡小一档，但比一般中州体面。青州不是节度治——河南道采访使治汴州（据《舆地广记》"河南采访使治汴州"）；但青州因海岱所在，**海东朝贡与转运分量都重**。州内常驻州兵若干、府兵折冲府数处。

755 年的青州，行政上是治、军事上是节点、外交上是**海东门户**。盛唐**新罗、渤海使臣**自海路东来：新罗自金城（今庆州）渡黄海登**登州**（蓬莱）；渤海自上京龙泉府南下经**辽东、营州**或自鸭绿江口渡海登登州——再西行入京，多经青州。青州**不是海港本身**——海港是登州蓬莱（青州东北二百余里，独立 doc 待立）——但青州是登莱内陆西去洛阳长安一线的**第一上州**，使节、商人多在此换驿、补给、停留。州内驿馆偶闻**新罗、渤海**之译语，是盛唐河南道东部一处少见的小国际化场面。

755 年冬，安禄山自范阳起兵，叛军主力南下河北、西取洛阳——青州虽属河南道，但偏东远离安史主战场，**未直接经历至德元年（756）至乾元二年（759）的洛阳-潼关-邺城核心战事**。但青州的精兵东调入援、海东使臣的往来一度受阻、漕粮的东向支线（自登莱进口新罗-渤海特产）短促中断——是这一年青州人能感到的盛唐之变。

## 城内地标

### 益都（州治）

青州治所。城在今山东省青州市——弥河西岸。北齐天保七年（556）改东阳城南益县为益都县，唐沿之。唐城周回不大，城内有刺史署、北海郡太守衙、长史司马、州学、佛寺数所。城南云门山下有官道——长安、洛阳东出至此一线、东北经潍州登州的官道自此分。

### 云门山 / 驼山（城南）

州治南五里、十里。两山不高，但崖壁自北朝起凿窟造像数处——北周-隋-唐相续，存盛唐窟龛若干。齐地佛教的一处遗存，规模远不及龙门、麦积。

### 临淄县（独立 relic doc 待立）

唐青州属县之一，州治益都西北约百里。**齐都旧址**——春秋战国第一大都，齐桓公至齐王建四百余年都此。盛唐时临淄已退为青州属县；但**齐故城**遗址周回约六十里的城圈、宫城基址、稷下学宫旧地、田齐王陵——是盛唐青州人凭吊春秋齐之处。**详见 [临淄](../relics/临淄.md)**（⏳ 待立）——本 doc 不重复齐都年表。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改北海郡为青州。**开元二十六年（738）李邕任北海太守**——李邕（678-747），书法名家，"北海"之号即由此任而得，世称"李北海"；李邕北海任内，杜甫、高适皆曾至青州相访，是盛唐青州一段文坛雅集。中晚唐青州为**淄青平卢节度使**辖境核心——李正己、李纳、李师古父子三代据淄青五十余年（765-819），半独立于中央，是河朔三镇之外、河南道东端的一处**藩镇**。**元和十四年（819）**宪宗削平淄青、李师道伏诛，青州复归中央。

**宋元明清**　北宋初年青州为京东东路治——是河南道东部一时第一大州。**范仲淹**皇祐三年（1051）任青州知州——范氏晚年任此，作《青州谢上表》。北宋庆历后青州为京东东路重镇。金属山东东路益都府，元为益都路。明洪武九年（1376）属山东布政司青州府；清沿之，青州府辖益都、临淄、临朐、安丘、诸城、寿光、博兴、高苑、乐安、昌乐、博山十一县。云门山、驼山诸窟自五代以后渐冷落，但崖面盛唐窟龛保存远比江南完整。

**民国至今**　1913 年废府改益都县；1986 年撤县设青州市（县级）——"青州"二字以市名重立，自乾元元年（758）肃宗复名起算用了一千二百三十余年。盛唐那座青州——益都县城旧址叠在今青州市主城下，弥河仍自城东流过。**唯一可与盛唐青州人共看的，是云门山、驼山的崖壁**——盛唐窟龛、佛龛造像，与 755 年相去不远。

## 引文

——《尚书·禹贡》（青州自禹贡九州以来便是海岱之间的一州；齐地海盐、丝织、海物之贡，自先秦至盛唐未断）
