# 泗州

## 一句话

河南道东南一州，治**临淮县**（今江苏淮安市盱眙县境东、后陷洪泽湖中；**唐泗州 755 治临淮，非治宿预、非今安徽泗县**）——**742-758 年间官名"临淮郡"**；**汴渠（通济渠）东南端入淮、泗水自北南下入淮**的双河枢纽，盛唐江淮漕路咽喉之一。

## 大唐天宝十四载（755年）

泗州在河南道东南端、淮水北岸的一处水道汇口上——治**临淮县**（按：盛唐泗州治曾两徙——隋开皇九年 / 589 置泗州初治宿预；**开元二十三年 / 735 徙治宿预 → 临淮**；755 时治临淮已二十年。临淮县今地约当江苏淮安市盱眙县境东，后世清康熙十九年 / 1680 城陷洪泽湖中）。沿革：**隋开皇九年（589）置泗州**于宿预；**大业三年（607）改下邳郡**；**唐武德四年（621）复泗州**仍治宿预；**开元二十三年（735）徙治宿预 → 临淮**——是为 755 治县；**天宝元年（742）改临淮郡**——是为 755 年当年的官称；**乾元元年（758）复泗州**。郡境的根本，是**汴渠（通济渠）东南段入淮口 + 泗水自北南下入淮口**这两条干流的交汇——盛唐江淮漕路自扬州邗沟入淮（南端楚州末口），西北经盱眙、泗州境转入汴渠西北上汴州、洛阳——**泗州正坐在这咬合点上**，与楚州、汴州、宋州一线共担漕政。州境东接楚州（淮阴郡）、北接徐州（彭城郡）、西接亳州（谯郡）、西南越淮接濠州（钟离郡）、南越淮接寿州一线。**州等第双源 hedge**：《旧唐书·地理志》记**中州**；《新唐书·地理志》记**上州**——两书并存，本项目采"中-上之间"读，与本州运河漕运的分量相称。**天宝户口**：户 **37,526**、口 **205,959**（旧唐书泗州条本数）——在河南道东南诸州中游一档。郡领**临淮（州治）**、宿预、徐城、涟水、虹、**盱眙**——按**盱眙县先天元年（712）自楚州来属泗州**（《新唐书·地理志二》泗州条；与本项目 [楚州](./楚州.md) doc 之 hedge 一致：旧唐书楚州条犹列盱眙，新唐书正为四县），755 时盱眙已属泗州凡四十三年。

唐人眼里泗州境内最大的一段标签，不是古迹也不是名宦，而是**泗州大圣僧伽和尚**（628-710）——西域何国人，**唐高宗龙朔初年（661 前后）来泗州**，于**临淮县**信义坊建**普照寺**（即后世泗州大圣寺）——按 735 治徙临淮，正是承僧伽和尚在临淮的香火一线；中宗景龙四年（710）僧伽卒于长安荐福寺，归葬泗州——盛唐人自玄宗朝起即崇泗州大圣，香火甚盛；至中晚唐及宋，泗州大圣信仰随漕路传遍江南。755 这一刻僧伽去世已四十五年，普照寺香火正盛——普照寺与州治同在临淮一城，州治、漕路、信仰三事一处汇合。

755 年冬安禄山自范阳起兵，泗州不在叛军正面主线，但**江淮漕路自此年起立刻紧绷**——临淮郡作为汴渠 + 泗水入淮的双河枢纽，是江淮漕米北上长安最要紧的一节；至德二载（757）尹子奇围睢阳（宋州），张巡、许远死守十月——**正是泗州、楚州、汴州一线漕路维系两京最危急的时刻**。755 这一冬城里的换防、漕船改道、粮草调发，已与一年半后的睢阳之围远遥相关。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改临淮郡为泗州，治仍临淮。**德宗建中四年（783）淮西李希烈乱**波及泗州；**贞元、元和间漕路紧要不减**——元和郡县图志（813）泗州条记得详。唐末杨行密据淮南立吴，泗州为吴、唐边境一线。

**宋元明清**　**北宋泗州属淮南东路，治仍临淮**——盛唐 735 徙治临淮一脉相承，宋元明清七百余年皆治此城，无再迁。金、元间属泗州；**明清属凤阳府泗州**，治临淮。**明万历六年（1578）以后黄河夺淮**、泗州境水患日深——**清康熙十九年（1680）黄河夺淮、泗州城（临淮）整体陷入洪泽湖**——这是清初一桩著名水患事件，自盛唐 735 徙治、至清初陷湖，**临淮作泗州治凡九百四十五年**，城没湖底，至今为洪泽湖底之"水下泗州城"遗址。**乾隆四十二年（1777）泗州治移至虹县**——"今安徽泗县"作泗州治之始，与唐泗州治临淮已**地理上相差百余里**。

**民国至今**　1912 年泗州废为泗县——治旧虹县城（今安徽泗县）；**唐 735 后泗州治临淮县**所在则陷于今**洪泽湖底**（江苏淮安市盱眙县境东）；唐 735 前旧治宿预县所在则演为今**江苏宿迁市宿豫区**。**唐泗州（临淮，今洪泽湖底）≠ 今泗县 ≠ 今泗洪 ≠ 今宿迁**——四处地名虽相涉，地理上各为一处：唐 735 后治临淮（今洪泽湖底）、唐 735 前旧治宿预（今宿迁市宿豫区）、清乾隆后泗州治（今安徽泗县）、今江苏泗洪县（明清泗州境之一部，1949 后置县）。盛唐**僧伽和尚普照寺**故址即在洪泽湖底（临淮县境，与州治同城），随泗州城一并沉没；今洪泽湖周边数处寺庙皆为后世重建，与唐普照寺非同址。

## 引文

> 泗州，中。隋下邳郡之宿预县。武德四年，平王世充，置泗州。开元二十三年，徙治临淮。天宝元年，改为临淮郡。乾元元年，复为泗州。

——《旧唐书·卷三八·地理志一·河南道·泗州》（节录；735 徙治临淮、742 改临淮郡、758 复泗州三事皆据本条；旧志此处记"中"与新志"上"两书并存不厘合，户口节"天宝户三万七千五百二十六、口二十万五千九百五十九"另见本条末）

> 释僧伽者，葱岭北何国人也……至唐龙朔初年，来游北土……于泗州临淮县信义坊乞地施标，将建伽蓝。

——《宋高僧传·卷十八·感通篇·僧伽传》（赞宁；僧伽 628-710 龙朔初来泗州临淮县建普照寺的史源——盛唐"泗州大圣"信仰自此立；按 735 泗州治徙临淮，正与僧伽普照寺一城）
