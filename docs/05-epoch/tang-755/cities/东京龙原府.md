# 东京龙原府

## 一句话

**后世五京之一**；755 时或已为龙原府 / 栅城府雏形，日本道已通——自此城东出图们江口，便是渤海通日本的"日本道"起点。

## 大唐天宝十四载（755年）

东京龙原府是渤海国**后世"五京"**（上京龙泉府 / 中京显德府 / **东京龙原府** / 西京鸭绿府 / 南京南海府）之一；755 时或已为龙原府 / 栅城府雏形，日本道已通。**主流考古定其治所于今吉林延边珲春八连城**——城在图们江下游北岸的山间平地，东距东海（日本海）不足百里；早期或亦称"**栅城府**"。

755 当年的位次需加 hedge：渤海"**五京**"制度的真正成熟与定型，主流认为在**文王大钦茂朝中后期**（约 785 前后）；755 这一节，龙原府或已立府治雏形，但"东京"之号是否已与上京、中京并称为五京之一，史料未明——本 doc 取"已为府、未必并称五京"的过渡态。**东京龙原府约 785 后才短期为都**（大钦茂 785-794 间一度自上京徙居东京，其卒后王廷迁回上京）——755 当年远未为都。文王在位（737-793）期间渤海三度迁都；**天宝末 / 755 前后王廷自中京迁上京，具体月日不强订**（《新唐书·渤海传》"天宝末，钦茂徙上京"，仅约略系于天宝末年）——本 doc 散文中言"755 当年王廷仍在中京、正向上京过渡"，取过渡态以避免精确到月日的伪定。

龙原府之要害在"**东出**"——城东南循图们江下游约百里入海，自此渡日本海便至日本本州西岸。渤海**首次遣使日本是圣武天皇神龟四年（727）**——使节高仁义、高斋德等自渤海出发至日本出羽国登陆；自此终渤海一朝，遣日本使凡三十余次，多自此道出入——史称"**日本道**"。755 时这条海道已通近三十年。龙原府西南陆界与**新罗**北境（浿江一线）相接——渤海与新罗陆道亦自此城分出。

755 年此时长安方向，安禄山十一月初九于范阳起兵——消息传至渤海东境至少要再过月余。文王此后保持藩属之礼、不与叛军接触——七年后宝应元年（762）唐廷升其"渤海郡王"为"**渤海国王**"。

## 历史长河（755 → 今）

文王晚年（**约 785-793 / 文王晚年**，大钦茂 793 卒）自上京徙居东京龙原府——其卒后王廷迁回上京。9 世纪渤海日盛，五京并存，东京为其东海方面之要冲。**辽天显元年（926）契丹灭渤海**，改东丹国——东京旧城渐废。辽金元此地属东北边僻女真之境。明置卫所，清初属宁古塔将军辖境东南端——**雍正七年（1729）始设珲春协领**，光绪七年（1881）升副都统衙门——"珲春"之名乃清代满语转译，非渤海或唐之旧。1909 后设珲春县，1988 设珲春市。八连城遗址 1980 年代起由吉林省文物考古研究所多次发掘，2004-2009 年正式确认为渤海东京龙原府治所——**2001 年列全国重点文物保护单位**。
