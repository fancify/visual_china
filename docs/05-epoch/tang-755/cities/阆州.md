# 阆州

## 一句话

剑南道东部嘉陵江中游一上州——治阆中县（今四川阆中市），嘉陵江盘曲三面绕城；金牛道自利州南下经此可换舟，州境为古巴西郡地。

## 大唐天宝十四载（755年）

阆州治**阆中县**，在嘉陵江中游西岸——江水自北而来，到城下盘曲三面绕城，是巴蜀少见的水绕孤城之势。州属**剑南道东部**，**天宝元年（742）随玄宗朝"改州为郡"通制改为阆中郡**，755 时正名为"阆州阆中郡"，两名在唐人嘴里互换不分；**乾元元年（758）肃宗复改为阆州**——但这是 755 之后的事。沿革更早：**隋开皇三年（583）置隆州**，大业三年（607）改为**巴西郡**；唐武德元年（618）复称隆州；**先天元年（712）改隆州为阆州**——"阆州"二字自此而立。《新唐书·卷四十·地理志四》阆州列为**上州**（《旧唐书·地理志》同列"上"，两书州等差略有出入但于阆州一致）——是嘉陵江上自利州、合州之间一段中游的重要州治。**《旧唐书·卷四一·地理四·剑南道·阆州阆中郡》载：**"旧领县八，户三万八千九百四十九，口二十七万三千五百四十三。**今领县九，户二万五千五百八十八，口十三万二千一百九十二**"——"今"即天宝十一载（752）籍——755 时阆州在册**户 25,588、口 132,192、领县九**（阆中、南部、苍溪、西水、奉国、新井、晋安、新政、岐坪），户口较武德旧籍 (户 38,949、口 273,543) 几折其半，是隋末蜀乱与初唐州境四析（南充、相如划果州，仪陇、大寅划蓬州）双重折减的实数——但在剑南东部诸州中仍稳居中等，与利州、巴州相侔，仅次于梓州、益州。

州境是**古巴西地**——战国巴国都一度迁阆中（旧说，史载不甚明），秦惠文王灭巴置巴郡，汉为阆中县属巴郡，三国蜀汉以**张飞领巴西太守**治阆中——是阆州城里盛传的旧事。755 时阆州无大兵无大事——金牛道北自利州南下，可在阆州换嘉陵江舟楫顺流至合州、渝州，是水陆兼用的转运点之一；唐人提阆州必带嘉陵江盘城与古巴西郡这两笔。

## 历史长河

**中唐**　**杜甫宝应元年（762）至广德二年（764）流寓阆州**——自梓州东行入阆，先后两入阆州，于此作**《阆水歌》《阆山歌》《阆州东楼筵奉送十一舅往青城》**等数十首；这是阆州在唐人诗里最浓的一笔，但已是 755 之后七年事。

**五代至明清**　前蜀、后蜀据阆州为剑南东部重镇。北宋属利州路。明蜀王分封后属保宁府——明末张献忠扰巴蜀，阆中亦罹兵燹；清代阆中仍为保宁府治。

**民国至今**　1913 年废府存县称阆中县属四川省。1991 年撤县设**阆中市**（县级，南充市代管）——"阆中"作建制名复活，已是民国后事。今阆中古城尚存——明清城垣、张飞庙、贡院诸迹犹在；嘉陵江仍盘曲三面绕城，与唐阆中县城方位大体相承。

## 引文

> 阆州阆中郡，上。武德元年置隆州，先天元年改为阆州。天宝元年改为阆中郡，乾元元年复为阆州。

——《旧唐书·卷四一·志第二十一·地理四·剑南道·阆州阆中郡》
