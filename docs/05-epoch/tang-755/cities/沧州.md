# 沧州

## 一句话

河北道沧州，742-757 年间名"景城郡"——渤海西岸的下口州治，治清池县；**盛唐时黄河自滑、汴东北流，经此一带入渤海**，沿海煮盐自汉以来便兴，是盛唐河朔东部的盐课重镇。

## 形貌

沧州城在河北平原东端、渤海西岸的低平海积地上——州治清池县城（今河北沧州市）距海岸约百里。城东尽是盐碱卤泊与沿海煮盐场，**城东北一线即黄河下游入海故道**（盛唐黄河自滑州、汴州、相州东北流，经德、博、棣、沧诸州境近海入渤海）；城西过永济渠是河北腹地，城南过御河可下棣州、青州，城北过御河上游通幽、燕一线。**冀东海积平原与渤海西岸盐场的咬合处**——非腹地大州、亦非边塞军镇，是河北道东部一隅的中州。

物候属冀东沿海——春多盐风、夏湿热、秋早霜、冬寒少雪；土多盐碱，腹地农耕弱于河北诸州，但**沿海煮盐与水陆转运是本州的根本**。户口据《旧唐书·地理志》开元、天宝间**约 12 万户**（《旧唐书》载 124024 户 / 825705 口）——按唐州县等第已逾上州门槛，盐课在河北道东部诸州中颇重。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

沧州境是**汉以来旧地**。先秦此地属燕、齐二国之间的渤海西岸；秦置县属上谷、巨鹿一线；西汉置渤海郡（治浮阳，今沧州东南），辖清池、浮阳、章武等县——"渤海郡"之名自此立，取临渤海之意，是西汉冀东沿海大郡。东汉、魏晋渤海郡屡有省并。

**北魏熙平二年（517）置沧州**——治饶安城（在今河北盐山西南）；"沧州"之名自此立，取临沧海（渤海古称沧海）之意。**隋大业三年（607）**炀帝改州为郡，沧州改**渤海郡**（按：隋以本州沿用汉郡名"渤海郡"，唐人地志多记此沿革）。**唐武德元年（618）复置沧州**——治清池县（州治自北魏饶安城南徙至清池一线，今河北沧州市旧城）。**天宝元年（742）**玄宗诏天下州改郡，沧州改**景城郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州——沧州之名再立，沿用至宋元明清。

### 此刻

天宝十四载（755）——官称"景城郡"，民间口语仍呼"沧州"。河北道东端，据《旧唐书·地理志》沧州**领县十一 / 户约 12 万余**（《旧唐书·地理志》）：清池（州治）、饶安、无棣、盐山、长芦、鲁城、临津、景城、弓高等（按：乾符县名为乾符元年〔874〕后改鲁城而来，755 时无此县）——州境东西约二百里、南北跨御河下游与渤海西岸近百里。户数已逾十万——上州（按唐制户口分等），在河北道东部诸州中位居前列，但仍不及幽、定、魏那种河北腹地节度治所一级。沧州不是节度治也不是都督府——**河北道由采访使巡历**，治所在魏州；沧州只是治下中州之一。

盐课第一宗。**渤海西岸自汉以来便是冀东重要盐场**——沧州沿海盐池、煮盐场分布数十处；**境内有长芦县（取永济渠岸"长芦"之名）**——后世（宋金以下）所谓"长芦盐场"通称即始自此县地名，**但"长芦盐场"作为独立盐场建制名号是后世事，盛唐时只称"沧州海盐"、"景城海盐"**。新唐书·食货志记沧、棣等州海煮盐课——盛唐盐铁专卖未及天宝（盐铁使由唐肃宗乾元年间始置），但沧州海盐已是河北盐运的一支——经永济渠西去河北腹地、北上幽燕、南下黄河。

水路第二宗。**永济渠**——隋大业四年（608）开，自洛阳东北至涿郡（今北京西南），经沧州西境过——盛唐永济渠是河北漕运的主干。**黄河下游**——盛唐黄河自滑州北流，过濮州、博州、贝州、沧州境近海入渤海（按 manifest 现行 polyline：濮阳-滑县-茌平-博平-高唐-平原-德州-沧州一线）；按：黄河自春秋至汉、汉至唐主流多次改道，但**755 时黄河确以北流为主、经冀东入渤海**——并非如后世（北宋庆历八年 / 1048 黄河决商胡埽以后）改南流入淮、入海。沧州因此既得永济渠之利、又当黄河下游近海一段——水陆转运在河北道东部诸州中颇重。

河朔藩镇第三宗——**这是 755 年时尚未成形的事**，但已伏。十一月初九安禄山自范阳起兵——消息沿驿道传到沧州时，城里粮草、盐课、河防的调度也紧起来。**沧州当河朔东南、距叛军范阳本据约六百里**——叛军主力南下经河北腹地（恒、定、赵、魏一线）西攻洛阳；**沧州城在叛军东侧远端，755 当年并未直接陷落**。至德、乾元以后河朔三镇（成德、魏博、卢龙）渐成——沧州先后属横海军（沧景节度使）治下，**这是 755 一年以后的事**。

### 唐人眼里的山河

城外往东出五十里入盐碱海积地、再东百里见渤海——海风带盐味、沿岸煮盐场的烟火夜里能望见。城外往西出二三十里过永济渠——河上漕船自南来、北上幽州。城外往北过御河上游、再过沧州北界即瀛州、莫州一线，盛唐河北平原。城外往南过御河、棣州界——再南即黄河下游近海故道，**盛唐黄河自此东北入渤海**。沧州境内盐场为盛唐 sea-salt 重要来源，唐前期未设独立衙署管理（盐铁院、盐铁使为肃宗乾元以后中晚唐建制；天宝间沧州盐课由州署直管）——城内有州署、市易司、常平仓，一套中州常规官署。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改景城郡为沧州。**乾符元年（874）僖宗改鲁城县为乾符县**——是本州县名最晚一次大改，"乾符"之名沿用至五代。唐后期沧州地位渐重——**横海军节度使**（亦称沧景节度使、义昌军节度使）治沧州，是河朔三镇外侧的一支牙军，**乾元以后置**；本州盐课随之成为河朔节度自用与漕往京师之大宗。中晚唐刘晏改革盐法（乾元、上元间）后，沧州海盐课在江淮以北诸盐场中位居前列。**黄河北流入渤海的形势**贯穿整个唐代——直至北宋庆历八年（1048）商胡埽决口、黄河南徙夺淮入海前，沧州一带始终是黄河近海一段的州治之一。

**宋元明清**　北宋仍置沧州属河北东路——黄河北流入海格局于北宋庆历八年（1048）起大变，南徙夺淮，沧州遂失"黄河近海州治"之地位，仅余渤海西岸盐场之利。**北宋以下"长芦盐场"始为独立盐场建制名号**——元置长芦都转运盐使司、明清沿之，**长芦盐场遂为中国北方第一大盐场**（清光绪、民国间长芦盐课冠北方）。金、元属河间路沧州；明初属北平府、洪武中改属河间府；清雍正七年（1729）升沧州直隶州（属直隶布政司）。

**民国至今**　1913 年废沧州为沧县；1947 年置沧市；今**河北省沧州市**即唐沧州城旧址范围——盛唐清池县城旧址叠在今沧州市运河区一带。**长芦盐场仍在**——但已不在沧州市辖境内（今主体在天津、唐山、黄骅一线）；**京杭大运河沧州段**沿用永济渠故道——盛唐永济渠至今仍可指认其大致走向。**黄河早已不经沧州入海**——南宋以下黄河南徙入淮入海，至清咸丰五年（1855）铜瓦厢决口黄河复北徙、改由山东利津入渤海，沧州亦不再当黄河下游。
