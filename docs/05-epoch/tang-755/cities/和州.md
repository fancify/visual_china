# 和州

## 一句话

淮南道最南端的州治、长江北岸渡江门户，742-757 年间名"历阳郡"——城在历阳县（今安徽马鞍山市和县），与江南宣州当涂、润州对岸的牛渚一线隔江相望；李白《横江词》六首即作于此一段江北。

## 形貌

和州城在长江北岸一片低平台地上，**南距长江岸约十里**——这是淮南道最南的一州，过江便是江南。

城所在历阳县——县治、州治同城。城西、城南是江岸冲积平原与陂塘水网，城北接巢湖以南的浅丘，城东沿江一线是横江渡的腹地。盛唐人称这一段长江为"横江"——**江北的渡口在和州境内（横江浦），江南的渡口便是牛渚（今马鞍山采石矶）**——一水之隔，南北对望，江面阔处达七八里，风急时浪头甚高。**牛渚矶在江南，三国时孙吴在此立水军要塞拒曹魏**——盛唐人沿江一望即指得出；和州城本身则是江北这一渡口的腹地州治，城周不大、户口中等，**《旧唐书·地理志》载和州天宝领县三（历阳、乌江、含山），户二万四千七百九十四、口一十二万一千一十三**——**和州按户口当中州门槛（户 24,794 / 口 121,013），史志或作上州/上郡**（《旧唐书·地理志》本条作"上"），户口不及同道寿州、楚州，但因控渡江故位置紧要。

物候上和州是淮南最南端、长江北岸：四月柳色、五月梅雨、六七月江风甚劲——与对岸润州、宣州当涂的物候几无二致，只是行政上一属淮南、一属江南。

## 大唐天宝十四载（755年）

和州在唐人眼里是一座**自三国以来便扼江北渡口**的旧地。

秦置历阳县——属九江郡，**"历阳"之名汉代以下相承不替**。汉时一度改为**当涂县**（与今安徽当涂县同字异地，不可混），后复历阳。三国时为孙吴江北前哨，与江南牛渚一江对应——孙吴**横江将军**之号即由此江段得名。**北齐天保六年（555）始置和州**——"和州"之名自此立。隋开皇九年（589）平陈，废和州入扬州（隋开皇时扬州治丹阳；江都郡名要至大业三年/607 始正式启用）；唐**武德三年（620）复置和州**，治历阳县——属淮南道；**天宝元年（742）**玄宗诏天下州一律改郡，和州改称**历阳郡**——这是 755 年当年的官称；**乾元元年（758）**复改郡为州，"历阳郡"之名前后用了十六年。

755 年的和州属**淮南道**——更准确地说，此时州已改郡，官称作**历阳郡**。淮南节度使一职**至德二载（757）方置**——755 年当年淮南尚无节度使，仅有玄宗开元二十一年（733）所立**淮南采访使**（治扬州）统淮南诸郡。历阳郡在淮南采访使辖境最南、临江一隅。**天宝十四载历阳郡太守在任者史志阙考**（注：唐州一级长官官称，**天宝元年（742）改州为郡后称"太守"，乾元元年（758）复郡为州后称"刺史"**——755 年当年只称"历阳郡太守"，不称"和州刺史"，是同一职而不同名）。

历史回望——和州人若在 755 年驻足西望江岸，所见仍是那同一条横江。**秦末**项羽自江东率八千子弟北上灭秦——《史记·项羽本纪》载其**自江东渡淮北上**的路径正经此一带江岸；垓下败后项羽南奔欲渡乌江，**乌江县恰是天宝和州辖下三县之一**——项羽自刎处的乌江亭，盛唐和州人指得出大致方位（在今安徽和县东北长江北岸乌江镇）。**三国孙吴**横江将军镇此江、与江南牛渚水寨成犄角——这些都是城外江岸唐人尚指得出的"前朝旧迹"。

文人方面——**李白《横江词》六首**作于天宝十三载（754）前后，正在 755 年的前一年。李白此时往来于宣州当涂与和州横江之间，**"横江欲渡风波恶，一水牵愁万里长"**写的便是这一段渡江风波之险；六首中"海神来过恶风回，浪打天门石壁开"、"郎今欲渡缘何事，如此风波不可行"诸句，皆是江北横江渡口的实景。**李白此时已五十余岁**，距其至德二载（757）陷永王案、流夜郎尚有三年——盛唐和州在李白笔下，是一段风浪甚恶、却仍要渡的江北岸。

## 历史长河

中晚唐和州因控横江渡故，**漕路、军路两线皆经此**——但盛唐和州不立都督府、不置节度治所，755 当年仅为淮南采访使辖下中州；**至德二载（757）淮南节度使始置**（治扬州），其后和州一直为淮南节度治下中州。**安史乱起后**和州因远离河北未即陷，但江淮震动；广德元年（763）以后淮南藩镇相互攻伐，和州数易其主。**僖宗中和年间**黄巢起兵后，淮南诸州尽乱，和州亦在其中。**杨吴、南唐**间和州为江北门户，与江南润州一江相对——**北宋开宝八年（975）**平南唐后归宋。

**和州历史上一段最为后世传诵的吏事**——**中唐刘禹锡**于**穆宗长庆四年至敬宗宝历二年（824-826）任和州刺史**。刘禹锡此前因"二王八司马"事被贬，辗转朗、连、夔诸州，至和州任已是第四次贬所。**相传《陋室铭》"山不在高，有仙则名；水不在深，有龙则灵。斯是陋室，惟吾德馨"即作于和州刺史任内**——此说见宋元以来诸笔记，唐人本传未明载、文献从严尚有异说，**但"和州陋室"作为后世文化记忆已甚为牢固**。**注：刘禹锡任和州刺史时距盛唐天宝十四载已六十九年——属中唐元和后期至长庆间事，与本 doc 所写盛唐 755 时和州并非同一时代**。

**宋元明清**——北宋和州属淮南西路，**南宋绍兴和议**后和州为南宋沿江军镇之一，与对岸采石矶一线为长江防线咽喉；**南宋绍兴三十一年（1161）虞允文采石之战**即在和州对岸江面上大破完颜亮南侵舟师——这是和州对岸的历史名战，距盛唐已四百年。元置和州；明洪武初为和州直隶州，属南直隶。**清雍正二年（1724）和州为直隶州**，属安徽布政司。**民国元年（1912）废和州为和县**——"和县"之名自此始用；本 doc 所写盛唐 755 时和州人，**自然不知道这座城千余年后会以"和县"为名**。

1949 年后和县属安徽省，先后属巢湖专区、巢湖市；**2011 年和县划归马鞍山市**。今和县人口约六十万，县城所在地仍是唐和州 / 历阳郡州治旧址范围；**乌江县已不存独立建制**，乌江镇属今和县。**唐和州城墙地表无存**——具体子城罗城的城圈线考古尚未完整复原。盛唐李白笔下"横江欲渡风波恶"的那段江面——今马鞍山长江大桥即横跨其上，**一千二百多年前的恶风险浪，如今是一桥之过**。

## 引文

《旧唐书·卷四十·地理志三·和州历阳郡》：

> 上。北齐置和州……武德三年复置和州。天宝元年改为历阳郡。乾元元年复为和州。旧领县三，户六千六百，口三万三千八百四十二。天宝领县三，户二万四千七百九十四，口一十二万一千一十三。在京师东南二千八百六十里，至东都二千四十里。历阳、乌江、含山。

——本 doc"形貌""大唐天宝十四载（755年）"两节户口、沿革、辖县诸事，悉本此条；新唐书地理志另载口数 122,013（与旧唐书略有出入），姑两存。

李白《横江词》六首节录：

> 人道横江好，侬道横江恶。
> 一风三日吹倒山，白浪高于瓦官阁。

> 横江欲渡风波恶，一水牵愁万里长。
> 海神来过恶风回，浪打天门石壁开。

——天宝十三载（754）前后作，《全唐诗》卷一百六十六。**"横江"即和州境内长江北岸渡口一带**——江南对岸便是牛渚（采石矶）。

《史记·卷七·项羽本纪》节录：

> 于是项王乃欲东渡乌江。乌江亭长檥船待，谓项王曰："江东虽小，地方千里，众数十万人，亦足王也。愿大王急渡。"

——项羽自刎处乌江亭，在天宝和州辖下乌江县境内（今安徽和县乌江镇）。
