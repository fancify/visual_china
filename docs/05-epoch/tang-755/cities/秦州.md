# 秦州

## 一句话

陇山以西第一州，渭水上游的麦积山下——长安出陇州再西便至此城，自此分两道：北循渭水入陇右、南越青泥岭入梁州。

## 形貌

秦州治上邽县，城坐渭水南岸台地。北临渭水、南依秦岭余脉——城周回不大，唐制属中州偏上。地势上正卡在三向交汇处：东去陇州、长安沿渭水谷地一线；北循渭水上行可入兰州、鄯州方向；南越青泥岭、河池便接梁州——**陇蜀通道与陇右驿道在此交叉**。城外东南六十里便是**麦积山**——孤峰拔起百余米，崖壁自后秦以来凿龛造像数百窟（详见独立 doc）。物候属陇右东缘——比鄯凉温润，比长安湿凉；渭水冬日不全冻，秋深陇山红叶。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

秦州境是**两千年旧土**。先秦秦人的早期都邑在此境之**西**（西犬丘，今礼县大堡子山一带 2000 年代秦公大墓发掘可佐）——秦非子封邑、秦庄公徙都皆在天水以西诸谷，"秦"这一国号的根脚出于此地。**先秦秦国的"秦"与唐州名"秦州"字面同源，但秦国都邑在天水郡西境，距唐秦州治上邽尚有百余里**——地缘相通而非同址。汉武帝**元鼎三年（前 114）**析陇西郡北部置**天水郡**——"天水"二字始落此地。东汉一度改名汉阳郡，曹魏复名天水郡，治屡迁。**北魏太和十一年（487）置秦州**，治上邽——"秦州"之名自此立，取古秦国故地之意。隋开皇三年（583）罢郡，大业三年（607）复改天水郡。**唐武德二年（619）复置秦州**，治上邽；同年置**秦州都督府**督秦、渭、陇、岷等数州军事——是初唐西陲一军事中枢，贞观间罢废（具体年代史略）。**天宝元年（742）**玄宗诏改州为郡，秦州复称**天水郡**；**乾元元年（758）**肃宗复改郡为州——秦州之名再立，沿用至宋元。

### 此刻

天宝十四载（755）——官称"天水郡"，民间口语仍呼"秦州"。陇右道东南端，据《旧唐书·卷四〇·地理志·秦州》"天宝领县五"——辖上邽、伏羌、清水、陇城、成纪五县（长道当时属成州，非秦州）。本郡户口据《旧唐书·地理志》记**天宝间约 2.5 万户（24827 户）— 中州偏上**，比鄯凉那种节度治十几万户的西陲大州小一档，但比一般内地中州体面。秦州不是节度治——陇右节度使坐镇鄯州，秦州是陇右道东端**转运与屯戍并重的州**。州内常驻州兵若干、府兵折冲府数处。粮草、军械自长安经陇州西运至此，于此分发——往北沿渭水入陇右节度辖境，往南越秦岭支线入剑南。**秦州的繁忙不在自己境内，而在过路**——这与岐州相似：京畿西门户是岐州，陇右东门户是秦州。

寺院方面，**麦积山石窟**（北朝—唐代多代凿造）是境内最盛的佛教圣地——崖壁上自后秦始凿，北朝隋唐相续；至盛唐时崖面已有窟龛百余处，崖壁砂岩不宜雕刻，故以**泥塑**为主。**详见独立 relic doc**。对天宝十四载的秦州人来说，这地方意味着**陇右的东门户、陇蜀的北门户**。他们尚不知道——半年之后玄宗西奔过岐州、入散关南下蜀道；陇右节度使哥舒翰自鄯州率八万兵东调入潼关，是这一年秦州最后一次见盛唐边军大队东调。再过五六年，吐蕃北上陷河湟——**宝应元年（762）陷秦、渭等州**（《新唐书》），广德元年（763）陷河、兰、岷、廓诸州。

## 城内地标

### 上邽（州治）

秦州治所。城在今甘肃天水市秦州区——渭水南岸。秦时已置上邽县（属陇西郡），汉魏晋南北朝隋唐俱沿用此名。唐城周回不大，城内有刺史署、长史司马、武德间天水郡都督佐衙旧址。城西渭水上有官桥——长安西去陇右、南越秦岭入梁州的官道自此分。

### 麦积山（详见独立 relic doc）

州东南六十里，秦岭余脉一座孤峰。崖壁砂岩，自后秦（约 4 世纪末）始凿窟，北朝隋唐相续。755 年崖面已凿窟一百余处——窟内**泥塑**造像数千身，是中国最大的泥塑石窟。**详见 [麦积山](../relics/麦积山.md)**——本 doc 不重复窟龛年表与造像细节。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改天水郡为秦州。宝应元年（762）吐蕃北上陷秦、渭等州（《新唐书》），广德元年（763）继陷河、兰、岷、廓诸州——秦州自此归吐蕃多麦节度（mDo-smad）辖境。咸通七年（866）归义军势盛时未及秦州，渭水上游一带先后为吐蕃余部、党项、嗢末诸族所据。

**宋元明清**　北宋初年复置秦州，属秦凤路。神宗熙宁五年（1072）王韶熙河开边以秦州为后方——河湟一线重入北宋版图。金属凤翔路，元改巩昌路秦州。明洪武二年（1369）属陕西布政司巩昌府；清雍正七年（1729）升秦州直隶州（1666 陕甘分省后归甘肃）。麦积山石窟五代以后逐渐冷落，但崖面窟龛保存远比敦煌完整——一千二百年未经大乱。

**民国至今**　1913 年废州改天水县；1985 年撤县设天水市；2004 年设秦州区——"秦州"二字以区名重立，自乾元元年（758）肃宗复名起算用了一千二百四十余年。盛唐那座秦州——上邽县城旧址叠在今天水市秦州区主城下，渭水仍自城北流过。**唯一可与盛唐秦州人共看的，是麦积山顶**——崖壁上的盛唐窟龛（135、142、123、43 诸窟）、泥塑造像、壁画残迹，与 755 年相去不远。
