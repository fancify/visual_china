# 兴州

## 一句话

山南西道下州一座，治顺政县——嘉陵江与故道水（八渡河）交汇处的一座山城，盛唐"陈仓—金牛"两条蜀道的换乘节点。

## 形貌

兴州城在嘉陵江与沮水（今八渡河）汇合处的台地上——两江夹角，三面环山，一面临江。州治**顺政县**（故城在今陕西略阳县城一带，按"略阳"为元至元二十年/1283 改置的县名，盛唐尚不存在此称）——城周不大，依山势铺开。北倚秦岭余脉嶂峰，南俯嘉陵江——盛唐这一州是典型的山区下州，户口稀、道里险、官署小。

地势是这一州的根本。嘉陵江自北而南从凤州境流入兴州——**陈仓道（亦称故道）的中段沿江而行**，自凤州、河池、长举一路南下至此；沮水自西北汇入嘉陵江，向西北谷地通往陇南、武都一带的羌氐故地。城东南去金牛驿、宁强方向，是**金牛道**接入梁州（汉中）的入口；城正南继续沿嘉陵江下行，则通利州、剑州、益州。一城而见两路三向——**这是兴州在盛唐的根本形势：陈仓道南端、金牛道北端、嘉陵江中游的一处枢纽**。

物候上是典型秦巴山地：四月春迟、五六月梅雨、年降水约九百毫米——常绿阔叶林、竹、漆、桐、茶皆产。盛唐自陈仓南来过兴州的人，眼里气候已不是关中——这是入巴蜀前的第一片"南方"。

## 大唐天宝十四载（755年）

兴州的沿革在唐人眼里清楚但不长。**北魏正始年间（504-507 间）置兴州**于此——治顺政县，"兴州"二字自此立。**西魏、北周递有**。**隋大业三年（607）行州改郡通制，改兴州为顺政郡**——治顺政县，属梁州道。**唐武德元年（618）改顺政郡复称兴州**——治顺政，属山南道；开元二十一年（733）分山南为东、西二道后，兴州属**山南西道**。**天宝元年（742）**玄宗诏改州为郡，兴州又改回**顺政郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州——兴州之名再立。**742 至 758 复州前，正名"顺政郡"，民间口语仍呼"兴州"**——两名并行。

755 年的兴州属**山南西道**，州治在**顺政县**。《旧唐书·地理志二》记天宝间兴州**下州**——户口仅数千户（《旧唐书》引天宝十一载顺政郡 4756 户 / 24006 口），是山南西道户口最少的几州之一。辖县三：**顺政**（州治）、**长举**（北界，今徽县东南一带）、**鸣水**（南界，今宁强县境一带）——皆嘉陵江中游谷地县治。城内有顺政郡治 / 太守署、顺政县衙、州司马、馆驿——一座下州的标准班子。本州无折冲府兵之大数，亦无节度治所——但作为陈仓道、金牛道交汇的驿镇，**州内驿务远比户口所示为重**。城外北面嘉陵江上有渡口、城东南往金牛驿方向有山间栈阁——后世杜甫《白沙渡》《水会渡》诸篇所写，即兴州境内嘉陵江上诸渡。

天宝十四载冬安禄山反范阳——消息沿陈仓道驿传南下，自长安过陈仓、散关、凤州、河池至兴州，再分两路：东折金牛道入梁州、南循嘉陵江入利州剑州益州——**顺政郡太守是这一支驿务的把口人**。城里没有大动静，但驿马更换、文牒过境，比平日翻了倍。州内的人尚未察觉的是：**再过半年**——至德元载（756）七月，玄宗自陈仓道一路南来，于此过境再转金牛道入蜀（见下"历史长河"段）；又过三年——乾元二年（759）冬，杜甫自秦州赴同谷再入蜀，亦自兴州境取道南下——这一座下州在 755 之后短短四年间，将先后接驾玄宗、过路杜甫，是盛唐转折之际最忙碌的一段山驿。

## 历史长河

**中晚唐**（755-907）

**至德元载（756）七月**，玄宗自长安西奔——先北过渭水入凤翔境，再循**陈仓道**南越散关，过凤州、河池、长举一路南下至**兴州**——抵兴州后稍驻，转**金牛道**东折，经金牛驿、五丁关至利州（今广元）、剑州、绵州、汉州，最后入益州（成都）。**这一段路线见《资治通鉴·卷二一八·至德元载》**——玄宗入蜀经凤州 → 兴州，兴州为山南西道中段重镇，亦是"陈仓—金牛"换接的关键节点。盛唐皇舆经此，顺政郡太守与顺政县令率属吏接驾——是这一郡治境内罕见的"正驾过境"。

至德二载（757）肃宗复改顺政郡为兴州——名号回到武德旧称。**乾元二年（759）冬**，杜甫自秦州赴同谷再入蜀——自同谷南下经木皮岭、白沙渡、水会渡、飞仙阁一线约在兴州境西北边缘穿过（杜甫《木皮岭》《白沙渡》《水会渡》《飞仙阁》《五盘》诸篇即此段途中作），是盛唐人留下的兴州沿线最详实的旅行实录。中晚唐兴州随山南西道起伏；**僖宗中和元年（881）**避黄巢入蜀亦循陈仓—金牛线，与玄宗当年同道——兴州境内栈阁、驿馆又一次承担"接驾"之劳。

**五代两宋**（907-1234）

五代之间兴州随山南西道入前蜀、后唐、后蜀——名号未改。北宋仍置兴州，治顺政。**南宋时兴州地位陡升**：建炎四年（1130）后宋金以秦岭—大散关一线为前线，**兴州（顺政）作为陈仓道南端的军事要地、川陕宣抚使节制下的前沿据点**——吴玠、吴璘兄弟驻守关陕四十年，兴州是与河池、和尚原、仙人关诸关一线的后方供应基地。绍兴间兴州升**沔州**（南宋绍兴十四年/1144 改），后又复为兴州——名号几度反复。**这一段抗金前沿的地位，是盛唐下州的兴州所不能预见的**。

**元明清至今**（1271—）

**元至元二十年（1283）废兴州**——降为**略阳县**，"略阳"二字自此始（取近旁略阳川地名为县名），属沔州、后属汉中府。明清两代略阳县属汉中府——"兴州"两字自此罢，州治旧址降为一县治。1913 年废府存县；1949 年后属陕西省汉中专区，1996 年汉中撤地设市，**今陕西汉中市略阳县治所，约在唐兴州顺政县旧址上**。2008 年汶川大地震略阳县城受灾严重，震后重建老城面貌大改。盛唐州城残基已埋入今略阳县城下，故道（陈仓道）线仅存零星栈阁遗迹于嘉陵江两岸悬崖间——"兴州"两字，只在《旧唐书》《元和郡县图志》和那一段杜甫诗里仍可见。

## 引文

> 兴州，顺政。下。开元户六千八百四十一。乡一十六。元和户……州境东接梁州褒城县，西接成州同谷县，南接利州绵谷县，北接凤州河池县。

——《元和郡县图志·卷二二 山南西道·兴州》（取意节录；具体行文以原书为准）

杜甫《白沙渡》（乾元二年/759 冬，自同谷入蜀途中作，时经兴州境嘉陵江上渡口）：

> 畏途随长江，渡口下绝岸。差池上舟楫，杳窕入云汉。
> 天寒荒野外，日暮中流半。我马向北嘶，山猿饮相唤。

——盛唐入蜀途中嘉陵江沿岸渡口实景。"白沙"渡在兴州、利州交界一段——此诗是盛唐人留下的兴州沿线最直白的一段旅行笔记。
