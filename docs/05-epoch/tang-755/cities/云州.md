# 云州

## 一句话

河东道北缘一州，治云中县（今山西大同）——雁门关北、阴山南麓的一座边州，城即**北魏平城故地**；后突厥、回纥前线，安史乱中 757 年史思明军一度攻陷。

> **disambig**　唐"云中"二字两地两用，须分清：本州**云州·云中郡**治云中县（今山西大同），即北魏平城故地；与之相对，**单于大都护府**所治"云中故城"在汉云中郡故地（今内蒙古和林格尔北、托克托南土城子古城遗址），是汉云中郡治云中县旧址、北魏盛乐城叠压区——**两座"云中故城"南北相距约二百五十唐里、跨阴山一线，同名异地**，详见 [单于府](./单于府.md)。

## 形貌

云州城坐在大同盆地中央。北望阴山（今大青山一线）山脊，南去**雁门关**约二百唐里。盆地海拔约一千米，是雁门以北少数一片可耕川地——桑干河（古名㶟水）自城南流过，再东经飞狐口入河北。城外是黄土与玄武岩台地交错带；春迟风沙重、冬严而长——比代州雁门更冷。**云州是河东节度使北翼第二防线**——南依代州、雁门关，北直接漠南突厥、铁勒诸羁縻部，与单于府隔阴山相望。

## 大唐天宝十四载（755年）

唐人眼里云州是**北魏旧都之地**。**北魏道武帝天兴元年（398）始迁都平城**，至**孝文帝太和十八年（494）迁都洛阳**——平城作魏都凡九十六年；其址即今山西大同。平城北郊**云冈石窟**自北魏文成帝兴安二年（453）开凿——是中国佛教石窟北朝阶段的代表作之一。**贞观十四年（640）**唐以隋云内县置**云州**，治云中县——"云州"之名自此立；后旋废，**永淳元年（682）**复；又废，**开元二十年（732）**唐玄宗再复云州——这次稳定延续。**天宝元年（742）**改州为**云中郡**；**乾元元年（758）**肃宗复改云州。**742 至 758 这十七年间，云州官号是"云中郡"**——民间口语仍呼"云州"。据《旧唐书·卷三九·地理志二·河东道·云州》记云州**下州**，**领县一，户七十三，口五百六十一**——此数极少，是《旧唐书》取乾元后期、安史乱中云州残破之口径（云州本即下州，又适逢 757 史思明北犯一线之后）；而《新唐书·卷三九·地理志三·河东道·云州云中郡》载**户三千一百六十九，口七千九百三十**——此数较多，是《新唐书》取盛唐天宝间之口径，更接近 755 时本貌。**旧、新唐书两套户口数相去近四十倍**——或因**两书取样年份不同**（《旧唐书》或为乾元后期、安史乱后云州残破之残数，《新唐书》或为天宝盛期之数），或为传写之异；史源歧出，本 doc 不强订。揆其大势：盛唐云州约三千余户、近八千口，已是河东北翼**下州**门槛之内的边州（**下州**通制即不足两万户）；安史乱后云州残破至七十余户、五六百口，几近废地。本 doc 主体取 755 时态，**依《新唐书》口径**。云州辖云中一县。755 时云州属**河东道**，节度使治太原；云州为其北翼第二防线，本州兵不多，但作雁门关北第一州，前哨**后突厥（已亡于天宝三载/744）旧地、回纥**南界——盛唐边塞诗里那种"白草连天"的塞外地理，云州正是一处典型。

## 历史长河

**安史乱中**　**至德二载（757）冬，史思明引兵北犯云中**——一度攻陷云州一线（《资治通鉴·卷二二〇》）；不久郭子仪、李光弼复河东，云州还唐。乾元元年（758）复州名。中晚唐云州渐紧贴回纥、沙陀前线，僖宗以后沙陀李氏据云州、代州一带——李克用之父**李国昌乾符五年（878）据云州反唐**，旋为唐军所逐，父子北走鞑靼；中和年间（881-885）李克用复起兵勤王，受唐朝廷招抚，最终以河东节度使据有云、代一带；唐末沙陀立国，云州为其北翼根脚。

**五代至清**　**后晋天福元年（936）石敬瑭割燕云十六州予契丹**——**云州为十六州之一**，自此入辽；辽于云州置**西京大同府**（重熙十三年/1044 升）——"大同"作府名自此始，与北魏平城旧址同地。金、元仍为西京大同府路。明洪武七年（1374）改为大同府——**洪武五年（1372）大将军徐达筑大同砖城**（今大同古城基址），明长城外重镇之一；清承明制。盛唐云州自 936 入辽以后，行政上即与中原王朝分离凡四百余年，至明初方归。

**民国至今**　1913 年废府存县，大同县属山西省。今**山西大同市**治所，仍叠在唐云州治云中县、北魏平城旧址上——**云冈石窟**距市区西约十六公里，1961 年第一批全国重点文物保护单位，2001 年世界文化遗产。盛唐云州那座城——州城残基已埋入今大同古城（明洪武筑）之下，云中县官署旧址不可考；桑干河仍自城南流过，云冈的北魏佛像，与盛唐云州人所见无别。

## 引文
