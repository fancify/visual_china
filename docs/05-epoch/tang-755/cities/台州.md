# 台州

## 一句话

江南东道台州，742-758 年间名"临海郡"——治临海县，灵江入海口上溯之地；州南八十里即天台山，**天台宗祖庭**所在——这一处佛门的分量，已盖过州城本身。

## 形貌

台州城在灵江北岸的一处河谷盆地里。

灵江自西而来，南汇始丰溪、北纳永宁溪，至临海县城前已是宽江——再下行约百里便是河口、入东海。城就盖在江北的一片台地上——南面临江、北背括苍山支脉，是一座"半山半江"的州城。论形貌——州城本身不大、官署不显，是一座倚山靠江的边远州治；但论户口——《旧唐书·地理志》载台州户**八万三千八百六十八**、口**四十八万九千零一十五**，是江南东道的**上州**。这一户口与形貌之间的反差颇值一辨：台州州境八九成是天台山系与括苍山系的山地丘陵——并无大平原、亦无大漕赋；其户口之繁，**全靠山区聚落代代繁衍而成**——不是依平原大田、不是依水陆通衢，是山地自养的人口。这是浙东南山区的一个共同特点——温州、处州亦类似。州南八十里**天台山**——主峰华顶高约一千一百米——是江南东南海滨第一山系；山南可通永嘉、山西可通婺州、山北是临海郡腹地。城中以州衙、临海县治为脊，市肆沿江而开——海舶可上溯至州城下卸货，是这一带"江海互通"的便利。

物候是浙东偏南：冬不甚寒、夏长而潮、春秋多梅雨。山中云雾常年缭绕——这是天台山"佛国"意象的物候底色。州境物产：山中出茶（天台云雾茶之祖型）、纸（剡藤纸近邻）、竹器；江海之间出盐、鱼、海产。

## 大唐天宝十四载（755年）

台州在唐人眼里是一座**佛门重于政治**的州城。

最早是越国地——秦属会稽郡；**三国吴黄武三年（224）析会稽郡置临海郡**，治临海县——"临海郡"之名自此立，自孙吴以来已五百三十年。两晋南朝沿之，皆为临海郡。**隋大业三年（607）**杨广改州为郡之制下，临海郡续存，但辖境屡有省并。**唐武德四年（621）**于此置**海州**——但仅一年，**武德五年（622）改名台州**——州名取**天台山**之"台"字（"天台"在唐时读 tāi，故"台州"亦读 tāi-zhōu）。"台州"之名自此定。**天宝元年（742）**玄宗诏改州为郡，台州复改作**临海郡**；**乾元元年（758）**肃宗再复台州——742 至 758 这十六年间，公文上写的是"临海郡"。

755 年的台州属**江南东道临海郡**，《旧唐书·地理志三·台州》载**领县六**：临海、唐兴、黄岩、乐安、宁海、象山——其中**唐兴县**即今天台县（天台山所在）、**乐安县**即今仙居县、**象山县**于**景龙二年（708）**析鄮县置（隶台州，唐末或属明州异说，本 doc 取《旧唐书》原属台州说）。户口据《旧唐书·地理志》——**户八万三千八百六十八，口四十八万九千零一十五**——以唐户三万为上州门槛计，台州户已逾八万、是江南东道诸郡中实打实的**上州**。**这一上州地位与州城本身的边远偏小并不冲突**——台州户口之繁，是山地聚落自养的果，不是因为有大都会、不是因为有大漕赋；州治偏远而漕赋不重——但**天台宗**这一座祖庭立在州境内，使台州在唐人眼里的分量又远超同等户口之州。

至于天台宗——**陈隋间智者大师智顗（538-597）**于陈太建七年（575）入天台山，立止观法门，开**天台宗**——是中国佛教第一个真正成体系的宗派（早于法相、华严、禅诸宗之立宗）。**隋开皇十八年（598）**智顗卒，同年杨广（时为晋王）奉旨于天台山下建寺，**国清寺**自此立——是天台宗祖庭。755 年时国清寺已立一百五十七年——盛唐间为江南东南最大寺院之一，住持代代相承智者法脉。**唐人入台州者，十之七八是为天台山而来**——这是台州在盛唐人地图上的位置。

至于**寒山**——传为唐前期至中期天台山国清寺一带的隐居者，与**拾得**（传国清寺厨僧）并称——以白话诗著称，今存《寒山诗集》三百余首。但寒山具体生卒、籍贯、僧俗身份史籍异文甚多——有以为初唐人、有以为盛唐至中唐人；唐道宣《续高僧传》未载，闾丘胤《寒山子诗集序》（题贞观中作，学界多以为后人托名）已不可尽信。**755 年的台州是否已有寒山其人在天台山中**，史无确据——本 doc 仅作 hedge 备一笔。

天宝十四载（755）冬安禄山起兵——消息传至台州约在十二月。台州在江南腹地东南、距前线四千余里——朝廷与州中皆未感切肤。但江南漕赋自此一战后将成唐廷半壁江山之命脉，台州亦在出粮、出盐、出茶之列。这座以天台宗祖庭立名的州城，此后的命运便与江南诸州一同被卷入唐廷的漕赋体系。

## 历史长河

至德二载（758）肃宗复改临海郡为台州——州名定于"台"，自此一千余年再未大变。中晚唐天台宗高僧辈出——湛然（711-782，台州人）于盛唐末至中唐间在国清寺弘**摩诃止观**，是天台宗第九祖、中唐天台中兴之祖。**唐德宗贞元二十年（804）**日本入唐请益僧**最澄**至天台山国清寺求法——归日本后于比叡山立**日本天台宗**，自此国清寺成日本天台宗祖庭，至 21 世纪仍有日韩僧侣定期来台礼祖。五代吴越国时国清寺再修；北宋四明知礼（960-1028）于宁波延庆寺中兴天台山外教观，史称"四明尊者"。南宋绍兴间台州仍为临海郡治——理学家**朱熹**曾任台州崇道观主管（祠官）。明清沿之，**民国元年（1912）**废府置县，临海县承之；**1949 年后**置临海县，1981 年改临海市；**1994 年**国务院批准撤地区设地级**台州市**——市治椒江区（在临海县东南约六十里的海滨新城），临海县仍为台州市辖县级市，是今台州境内唯一直接承唐"临海郡治"之地。

**天台山国清寺**今为天台宗祖庭、全国重点寺院——现存大殿为清雍正十二年（1734）重建，2005-2010 年间又大修；但寺基与寺周山形仍与盛唐相去不远。**寒山诗集**自宋编定以来流传不衰——20 世纪 50 年代经美国诗人加里·斯奈德（Gary Snyder）译介，**寒山诗在欧美形成"Cold Mountain"现象**，是中国古诗在欧美最受欢迎的体系之一。盛唐 755 年那一座临海郡——州城叠在今临海市区下、国清寺叠在今天台县国清寺旧址上，仍是这一带最重的两处文化坐标。

## 引文

《旧唐书·卷四十·地理志三·江南道·台州临海郡》：

> 上。隋永嘉郡之临海县。武德四年置海州，五年改为台州。天宝元年改为临海郡。乾元元年复为台州。**旧领县四，户三千八百十六，口一万九千三百一**。天宝领县六，**户八万三千八百六十八，口四十八万九千十五**。在京师东南**五千五十里**，至东都**四千一百九十里**。临海……唐兴……黄岩……乐安（704 年置）……宁海（621 年置）……象山（708 年置，武则天景龙二年析鄮县置）。

——这一段是 755 年时态台州的地理志原文：**领县六、户 83,868、口 489,015**——以唐户三万为上州门槛计，台州天宝年间是**上州**，志书顶格亦明书"上"字。比照"旧领县四、户三千八百一十六、口一万九千三百一"——这是更早期（很可能武德/贞观间的）旧额；天宝间一州户口由三千余户飙至八万余户、约二十二倍之多——这一倍数远超人口自然繁殖速度，更可能反映的是隋末户口荒残后唐代山区开发与人口登记复苏的双重叠加。距京师"五千五十里"、至东都"四千一百九十里"——这是唐地理志一贯的官道里程，自长安经洛阳南下江淮、沿运河至润州、再东南越浙东而抵台州，是当时入台的主路。志末列县名兼置县年：乐安 704 年（武则天长安四年）析临海置；象山 708 年（中宗景龙二年）析鄮县置（鄮县属明州，故象山初置时入台州、唐末或一度改隶明州，本 doc 取《旧唐书》原属台州说）；宁海 621 年（武德四年）置，是台州六县中除临海外最早的一个。这些置县年也佐证：台州山区聚落自盛唐前夜起方进入大规模州县化登记，与"户口二十二倍"之激增吻合。

《元和郡县图志·江南道·台州》（唐 李吉甫 813）：

> 台州，临海郡。下州。……西北至上都（长安）四千一百一十五里，西北至东都（洛阳）三千二百九十里。**天台山在县北一百里**。

——元和志为 813 年志书，已在安史之乱后五十余年，江南东南诸州户口大减、台州由天宝上州一度回落至下州——故李吉甫志中作"下州"。但 755 时态台州是**上州**，与元和志载并非同一时段事。盛唐人对台州的第一印象——志书将天台山列入州条目首要地理项——这是天台山在唐人心中超过州城本身的口风。
