# 维州

## 一句话

剑南道最西那一座小下州——薛城（今四川阿坝州理县薛城镇）卡在岷江上游一处高山深谷里，唐与吐蕃东境在此反复争夺百余年。

## 形貌

维州城坐在**岷江上游一处河谷台地**上——州治在**薛城县**（今四川省阿坝州理县薛城镇）。城周不大，城东临杂谷脑河（岷江上游一支），城西、城南皆是高山——海拔自谷底一千五百米直上四千米，城里抬头便能看见雪线。州境东接茂州，南越鹧鸪山接益州西山诸羌部落，西、北两面便是**吐蕃东境**——沿岷山 / 邛崃山 / 西山诸羌一线通吐蕃东境。这一带山岭密如鱼鳞，谷间唯有沿江一线可行。

谷里以农牧并行为主，但农事极薄——梯田只在台地上有零星几片，种青稞、苦荞、燕麦；山坡之上是高山草甸，氐、羌、党项杂居其间，以放牧牛羊、采药、伐木为生。城里则是另一番面貌：刺史署按下州制设，但维州是**羁縻性质浓厚的边州**——城外戍堡星散，烽燧沿江排列；汉官管城里事，羌酋管谷里部落事，两套秩序并行。

物候上属高寒——冬日江上结冰，五月夜里仍要盖被，十月便见霜雪。**杂谷脑河自西北而下贴城东流过**，下行约二百里汇入岷江干流——这条河谷是维州唯一一条对外通道：东出至茂州、汶川，再东南便接益州西山诸戍。山深路险，斥候走一趟要三五日。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

维州地很老，但唐人接手得晚。隋开皇九年（589），文帝平岷山诸羌后置**薛州**——治薛城，"薛"之名从此立。隋末乱起，吐蕃尚未东出，这一带仍归氐羌部落自治。**唐武德七年（624）置维州**——取"维系"诸羌之意。**天宝元年（742）玄宗诏天下州改郡——维州改曰维川郡；乾元元年（758）肃宗复郡为州**。所以天宝十四载（755）的官文书上，这地方多写"维川郡"——民间口语仍称"维州"，两个名号并行。

唐对维州的实际管辖一直浮浅。仪凤三年（678）后，吐蕃东进，与唐 / 吐谷浑余部在西山一带交锋，吐蕃势力自青海高原沿黄河上游、雅砻江一线向东推进——开元、天宝间，吐蕃东境已逼至岷江上游，与维州**仅一山之隔**。维州自此一直是**剑南道的最西一州**，与剑南节度治所益州（成都）之间隔着茂州、汶川两道屏障；其本身则是抵挡吐蕃东扩的第一道——也是最薄的一道——防线。

### 此刻

天宝十四载（755），维州是**维川郡**治。户口据《旧唐书·地理志》载约**二千户余**——是剑南道、乃至全唐户数最少的几座下州之一。按唐制，户不满二万为下州，维州正卡在这一线的最下端。城里官署按下州制设，但因户口极薄，长史司马诸僚多由邻州兼摄；陇右、剑南两大节度使分管西陲，剑南节度使治在益州（成都），维州只是节度下的一处羁縻边州——常驻戍卒不过数百，倚周边戍堡与烽燧分驻应敌。

维州还是**唐与吐蕃东境的互市点之一**。盛唐自开元间便于剑南西山一线设若干互市——维州、松州、茂州各有市，吐蕃东境部落以马匹、牦牛毛、麝香、贝母换唐之绢帛、茶叶、铁器。维州市规模不大，但因地最近——出城西门便是边境——是吐蕃东道商旅过岷山入唐境最早一站。

对天宝十四载的维州人来说，这座城意味着——**唐土的西南端、吐蕃的东限**。城外杂谷脑河谷之西，越过一道山岭便是吐蕃东境戍堡；山深路狭，斥候要先一拨派到西面去，秋深霜降之后，吐蕃骑兵便要试探着来——这一带七十年来年年如此。他们大约也以为，这一年与往年没什么两样。只有他们尚不知道，再过八年，这座城将自此入吐蕃，**七十年不复唐土**（763-765 间陷蕃 → 大和五年 831 收复，凡约七十年）。

## 历史长河（755 → 今）

**中晚唐**　乾元元年（758）肃宗复郡为州——维州之名重归。但安史乱后陇右、剑南兵东调，西山一线已无重兵。**广德元年（763）十月，吐蕃尚结息率二十万众入长安**——同年前后，吐蕃自青海北上、自雅砻江东下，连陷河、洮、岷、廓、鄯、维、松诸州。**广德元年（763）维州没吐蕃**（《新唐书》明载），自此入吐蕃近七十年，直至**大和五年（831）收复**始议归还（详下）。

文宗大和五年（831）九月，吐蕃维州副使**悉怛谋**举城降唐——剑南西川节度使李德裕受其降，遣兵入城。但宰相牛僧孺以"修好不可背盟"为由力主退还——朝议反复，文宗终从牛议，将维州城与悉怛谋等送还吐蕃。**悉怛谋及部众数百人皆为吐蕃所杀**。这一事件在中晚唐政争中震荡甚远——牛李党争自此愈烈，**李德裕作《论维州投降事疏》《代刘沔与回鹘宰相书》以鸣其冤**——后世史家如司马光、王夫之论唐衰，常溯此一举。这是历史长河里维州留下的最重一笔。

宣宗大中三年（849）后归义军起，沙、瓜、凉、鄯诸州陆续归唐——但西山一线偏南，**维州不在归义军势力所及**，仍归吐蕃多麦节度。咸通、乾符间吐蕃内乱，西山诸州落入氐羌部落自治；五代之间维州先后属王建前蜀、孟知祥后蜀——但管辖虚浮，实际由当地羌酋世袭。

**宋金至清**　**北宋景祐三年（1036）改维州为威州**，仍治薛城——这是维州旧名归入宋制的主线一笔。南宋偏安，威州地复落羌部，宋廷管辖虚浮。明代置**保县**——治在薛城旧址周近。清雍正间改设**理番厅**——"理番"取治理番部之意——这是"理县"两字的起头。

**民国至今**　民国二年（1913）废厅改**理番县**；1946 年改**理县**至今。今为四川省阿坝藏族羌族自治州下辖县，县治杂谷脑镇（自清后期起县治自薛城迁至杂谷脑，相距十余里）。**薛城镇仍存**，是理县下辖一镇——唐维州城残基已不可考，仅薛城后山有一处明清残墙断垒，民间传为"维州古城"，未经考证。杂谷脑河仍自城东贴流而下，仍是这地方过去一千五百年从未动过的一条线。

## 引文

> 维州，下，本汉蜀郡之徼外地。隋开皇九年置薛州；唐初置维州，取"维系"诸羌之意。天宝元年改为维川郡，乾元元年复为维州。

——综《新唐书·卷四十二·地理志六·剑南道·维州维川郡》及《元和郡县图志·卷三十二》参订（新旧《唐书》于武德置州年份互有异文，此处取诸书共识之沿革主线，不强标一卷）

> 我哀蜀人之困穷，仰天叹息泪泗下。

——杜甫《不见》（约 761 年作于成都）——杜甫客蜀八年（759-765），亲见吐蕃寇蜀。维州一线的吐蕃压境，在他的诗里只是这样一句叹。
