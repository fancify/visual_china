# 洮州

## 一句话

陇右道最南那一座小下州——治临潭县（今甘肃临潭一带），坐在**洮河上游谷地**上，正卡在唐与吐蕃的西南前线；天宝元年至乾元元年（742-758）官称"临洮郡"。

## 形貌

洮州城坐在**洮河上游**——黄河支流——南岸一处河谷台地上，州治在**临潭县**（今甘肃临潭一带）。城周不大，夯土城垣，是边州常制。州境东接秦州、北接渭州与兰州、东北邻河州、东南接岷州，西、南皆与**吐蕃**接——洮河自西南而来，向东北汇入黄河。谷里以农牧并行：河两岸有薄水浇地种青稞、粟、麦，台地草滩上散居党项、吐谷浑余部诸羌的部落，放牛羊。海拔逾两千五百米，冬日河上结冰，五六月夜里仍要盖被。

## 大唐天宝十四载（755年）

洮州沿革：**隋开皇九年（589）**置洮州；**大业三年（607）**罢州为临洮郡；**唐武德二年（619）**复置洮州；**开元十七年（729）废入岷州，开元二十七年（739）复洮州**；**天宝元年（742）**玄宗诏天下州改郡，洮州改称**临洮郡**；**乾元元年（758）**肃宗复郡为州。**755 年这一刻官文书上写"临洮郡"**——民间仍称洮州。郡望据《旧唐书·地理志》列为**下州**——边州户口本不多。

洮州在 755 年的分量全在**边**上：开元至天宝间，**王忠嗣（738-746 陇右节度）**与**哥舒翰（747 起继任陇右节度）**与吐蕃在洮河、青海湖、九曲一线连岁鏖战——洮州城外的洮河上游谷地，正是河西、陇右两节度伸向吐蕃东境最近的一段。城里常驻戍卒不过千余，倚周边戍堡与烽燧分驻应敌；秋深霜降之后，吐蕃骑兵便要试探着来。他们尚不知道，再过八年，洮州将自此入吐蕃。

## 历史长河

**中晚唐**　**广德元年（763）**前后，吐蕃自青海北上，连陷河、洮、岷、廓、鄯诸州——洮州陷蕃**约在广德至大历间（763-766）**，史无确年（《通鉴》仅记"陇右诸州相继陷蕃"）。自此入吐蕃**逾八十年**；宣大间归义军一度西进未及洮州，五代之间洮州一带先后落入吐蕃余部、党项、吐谷浑后裔之手。**宋金元明**　北宋熙宁中王韶熙河开边，**熙宁六年（1073）**前后宋复置洮州——治临潭旧址；金属临洮路；元置洮州元帅府；**明洪武十二年（1379）**置洮州卫——卫城在唐临潭旧址略移；清雍正间废卫，仍称洮州厅。**民国至今**　民国二年（1913）废府州厅，改设**临潭县**——"临潭"取临洮潭谷之意，承唐县名。1949 年后属甘肃；唐洮州城残基已不可考——只有洮河仍自城西南而下、向东北汇入黄河，仍是这地方过去一千五百年从未动过的一条线。

## 引文

> 洮州，临洮郡，下。隋置洮州，大业初改为临洮郡。武德二年，复为洮州。天宝元年，改为临洮郡。乾元元年，复为洮州。

——《旧唐书·卷四十·地理志三·陇右道·洮州》（盛唐洮州沿革的标准记载，节录）
