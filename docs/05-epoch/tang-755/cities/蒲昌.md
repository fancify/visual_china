# 蒲昌

## 一句话

唐西州（742-758 间官称交河郡）下辖五县最东的一县——治在今新疆吐鲁番市鄯善县一带，吐鲁番盆地东端的一处绿洲驿站，伊州西路上自伊吾入西州所经之地。

## 大唐天宝十四载（755年）

蒲昌是**西州交河郡领县五**（高昌、交河、柳中、蒲昌、天山）中位置最东的一县，治在吐鲁番盆地东端的一处绿洲——具体县治史家有两说：**一说在今鄯善县鲁克沁/吐峪沟一带**，**一说即今鄯善县城所在**（鲁克沁另说属柳中县）。地在火焰山东端南麓，东出便是戈壁与噶顺沙漠，西去州治高昌县约二百里。755 这一刻州名作"交河郡"，蒲昌仍称"蒲昌县"，县令一员领县事。

县名"蒲昌"取自**蒲昌海**——即《汉书·地理志》所载罗布泊古名。但**蒲昌县治不在湖滨**——罗布泊在塔里木盆地东端，去本县东南数百里外；县只是借此海方位为名。**唐蒲昌县不可与后世"鄯善县"等同**——清光绪二十八年（1902）借汉鄯善国旧名于此立鄯善县，与唐蒲昌县沿革无直接接续；同样，"鄯善"作古地名指向罗布泊东南的汉鄯善国（今若羌一带），不是唐蒲昌。

蒲昌的分量不在户口——是**道里**。地当西州东境，往东出盆地的官道中，**伊吾道**自伊州（伊吾）西行入西州，蒲昌正处此路所经；另有**大海道**之名，史家或与伊吾道相关、或别为一线，定位有异说，本 doc 不强订。玄奘《大唐西域记》归途自瓜州西经此带——吐鲁番阿斯塔那墓群（地在西州治高昌县北郊，与蒲昌县同盆地）出土的过所、契约、户籍中，蒲昌县文书数量极丰——盛唐西州一县基层的田亩、税米、戍丁、判词，今日皆可逐字读到。

## 历史长河

中晚唐西州陷蕃后（约贞元六年 / 790 前后），蒲昌县建制随西州一并废弃。九世纪中叶高昌回鹘据盆地后，蒲昌一带渐易他名——"蒲昌"二字作政区名自此退场。元明清盆地为高昌回鹘、东察合台、叶尔羌、准噶尔诸政权相继控制；清乾隆二十四年（1759）平准噶尔，盆地入清；**光绪二十八年（1902）于此地立鄯善县**——借汉鄯善国旧名，与唐蒲昌县无承袭关系；唐蒲昌县治所在的鲁克沁/县城一带遗迹散见今鄯善县境，未有完整考古复原。

## 引文

《新唐书·地理志四下·陇右道·西州交河郡》：

> 西州交河郡……领县五：高昌、交河、柳中、蒲昌、天山。

——蒲昌作西州属县最简的史志原文。
