# 平壤

## 一句话

高句丽长寿王 427 年迁来的旧都，至总章元年（668）唐灭高句丽止凡 241 年——755 时已非唐辖，归新罗近二十年；唐人记忆里只剩**长寿王旧都 + 668 旧战场**。

## 大唐天宝十四载（755年）

平壤在 755 年的**唐人地理常识**里，是一座**已不属唐**的故都。

此地古名**王险城**——汉武帝元封三年（前 108）灭卫满朝鲜，置乐浪郡于此一带数百年；高句丽长寿王十五年（427）自国内城（今吉林集安）南迁，立都平壤，并自名其都为"**长安城**"——这是高句丽史上为时最久的一座都城，凡 241 年。**总章元年（668），李勣、薛仁贵率唐军灭高句丽**——同年唐高宗诏立**安东都护府于平壤**，薛仁贵为第一任都护，欲以此直辖朝鲜半岛。

但治平壤未久。**咸亨元年（670）起新罗与唐在半岛交兵八年**（罗唐战争），高句丽遗民屡反——**仪凤元年（676），安东都护府自平壤北徙辽东郡**（今辽阳）；**调露元年（679）再徙新城**（今辽宁抚顺）；其后随形势数次南北迁徙，**755 年时安东都护府寄治辽西故城**（今辽宁义县）——已远离平壤约一千五百里。

**开元二十三年（735）正月，玄宗诏赐新罗"浿江（大同江）以南地"**——平壤一带正式入新罗北界。自此至 755 年凡二十年，平壤已是**新罗最北的故都遗墟**——城圈犹存，王宫尽废；驻守的是新罗北边戍兵，不再是唐安东都护府的军政。

755 年的长安人若忆起平壤——所记多是 668 旧事：李勣、薛仁贵破高句丽都的捷报，唐军入"长安城"（高句丽自名）受降的一刻。**这是唐对平壤所留下的最后一笔实迹**——之后是八十年的退兵、撤府、北徙、最终弃地。城里如今走动的人，说的是新罗话，不是唐话。

## 历史长河（755 → 今）

中晚唐至五代，平壤始终在新罗北界、后入高丽——**高丽太祖王建于天授十八年（935）受新罗以国归之，平壤复为高丽北方重镇**，史称"**西京**"，与开京（开城）并立。蒙古入高丽（13 世纪）时平壤数被攻陷。朝鲜李成桂洪武二十五年（1392）建国，都汉阳（今首尔），平壤为西北方第一大邑，置平安道观察使。万历二十年（1592）日本丰臣秀吉攻朝鲜，**明军李如松万历二十一年正月（1593）于平壤大破小西行长**——这是平壤在明代东亚史的最重一笔。清末甲午（1894）日清平壤之战，清军溃败。1910 日韩合并后属日本朝鲜总督府。1945 苏军入北纬 38 度以北，平壤为朝鲜半岛北部行政中心；1948 朝鲜民主主义人民共和国立国都于此——但**这是 1300 年后的事**，与 755 年那座新罗北界的故都遗墟无涉。
