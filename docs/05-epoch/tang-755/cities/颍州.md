# 颍州

## 一句话

河南道东南一中州，治汝阴县（今安徽阜阳市），**742-758 年间名"汝阴郡"**——城坐颍水中游北岸，南距颍水入淮口（颍上一段）百里许，是江淮之间循颍水北上中原的一处水陆中转，亦《史记》所载春秋齐相**管仲"颍上人"**故里（同游之鲍叔牙籍贯则属地方传说，详下）所在。

## 形貌

颍州治汝阴县，城在颍水北岸——豫东南淮北平原南缘的一片低平台地。城周不大，夯土城垣四门各开。城西北有泉河自西来汇颍水，城南即颍水正干东南流——再行百里许至属县**颍上**境内入淮（颍口，亦称颍尾，是淮河中游北岸最重要的入淮口之一）。城外四望平畴桑麻，西去蔡州（汝南）二百里许，北去陈州（淮阳）二百里，东北去亳州（谯郡）二百余里。气候是江淮之间过渡型——冬寒不及河北、夏湿不及江南，颍水冬不全冻，可行小漕。

## 大唐天宝十四载（755年）

汝阴郡是唐人眼里**循颍水串江淮与中原的一段腰节**——治汝阴县，**天宝元年（742）改称汝阴郡**，沿革几度迁名：北魏置颍州、隋大业三年（607）改汝阴郡、唐武德四年（621）复颍州、天宝元年（742）再改汝阴郡、乾元元年（758）再复颍州。**等第两书并载有别**——《旧唐书·地理志》作"**中**"州，《新唐书·地理志》作"**上**"州，盛唐户口实在河南道诸州中游一档，远不及宋、汴、亳那一档大上州。755 年这一刻匾上写"汝阴郡"，太守一人领郡事，下辖**汝阴、颍上、沈丘**等县。

颍州在唐人心目中另一重身份是**《史记》所载春秋齐相管仲故里**。**《史记·卷六二·管晏列传》**："管仲夷吾者，颍上人也。少时常与鲍叔牙游，鲍叔知其贤。"——"颍上"自汉以来或解作颍水之上、或落到具体的颍上县（颍水入淮口一带，唐时为颍州属县），盛唐人来汝阴郡多取后说，凭吊管仲旧地。**鲍叔牙籍贯则《史记》本文未明言**——只言其与管仲少时同游；齐国本籍（临淄一带）与颍上同里两说并行，颍上同里一说多本明清地方志与管鲍祠传统，唐人凭吊时已与管仲并提，但就严格文献而言属**地方传统**而非《史记》原文。城南颍水之滨亦相传有管仲墓、鲍叔牙墓——确址唐时已不可全寻，多依文献与地方传凭吊。755 年这一冬安禄山自范阳起兵，汝阴郡不在叛军正面主线，但循颍水南下江淮之路恰自此过；至德元载（756）以后江淮防线收紧，颍州遂为唐军江淮转输的一处后方腰节。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改汝阴郡为颍州。安史平后地位渐稳——颍水中游漕运不及汴水正干，但江淮转输的辅道之一。唐末黄巢之乱经此境，城邑数易其手。

**宋元明清**　北宋颍州地位骤显——**仁宗皇祐元年（1049）欧阳修出知颍州**（1007-1072；任在皇祐元年至二年、再于熙宁四年退老此地），主持开浚州城西北一片低洼水泽为湖，**颍州西湖**自此立——与杭州西湖、扬州瘦西湖并称，是欧阳修晚年精神所托之地；**元祐六年至七年（1091-1092）苏轼出知颍州**，续浚西湖、留诗甚多。这是北宋颍州的两大节——盛唐汝阴郡的城圈到了北宋已是**欧苏二公接踵主政之地**。金元后渐衰；元属汝宁府颍州；明初降为颍州，弘治、嘉靖间数升数降；**清雍正二年（1724）升直隶州、雍正十三年（1735）升府**——立**颍州府**，并于同年（1735）升府时增设附郭县**阜阳县**（"阜阳"二字即立于此，为颍州府新置附郭县名）；府辖阜阳、颍上、霍邱、亳州、太和、蒙城等州县。咸丰、同治间**捻军纵横河南、安徽东北**（张乐行籍亳州、颍上一线）——颍州府境为捻军最重要的活动腹地之一，1853 太平天国北伐军过境、1864 捻军在颍州府境与僧格林沁数番大战。

**民国至今**　1912 年废府存县——颍州府改阜阳县。1996 年升地级**阜阳市**——盛唐颍州的城圈大体保留在今阜阳市颍州区内（区名沿用"颍州"古名）。**颍州西湖**——欧阳修、苏轼浚治的那一片水泽——清末以来渐淤，今阜阳市西郊尚有"颍州西湖"景区，是清后期至今数番疏浚的旧迹。盛唐人凭吊管鲍之交的颍水之滨——今阜阳市颍上县仍有**管鲍祠**（明清屡修），是中国最古的几处管仲、鲍叔牙祭祀之一。

## 引文

> 管仲夷吾者，颍上人也。少时常与鲍叔牙游，鲍叔知其贤。管仲贫困，常欺鲍叔，鲍叔终善遇之，不以为言。

——《史记·卷六二·管晏列传》。**注**：《史记》原文只称管仲为"颍上人"，鲍叔牙籍贯本文未明言；"管鲍同里"主要依明清以来的地方志与颍上管鲍祠传统并提，属地方传，非《史记》直接所载。"颍上"自汉以来或解作颍水之上、或落到具体的颍上县（唐颍州属县）——盛唐人来汝阴郡多取后说凭吊管仲旧地。

> 颍州，汝阴郡。中。隋汝阴郡。武德四年改为信州，六年改为颍州。天宝元年改为汝阴郡。乾元元年复为颍州。

——《旧唐书·卷三八·地理志一·河南道·颍州（汝阴郡）》（节录）。盛唐 755 时匾名"汝阴郡"。**注**：等第《旧唐书》作"中"，《新唐书·地理志》同条作"**上**"（"颍州汝阴郡，上"）——两书并存，本 doc 取并提 hedge，不偏一书。
