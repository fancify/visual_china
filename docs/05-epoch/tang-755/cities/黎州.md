# 黎州

## 一句话

剑南道西南边角的一座下州，742-758 年间名"洪源郡"——治汉源县（今四川汉源县老城一带），大渡河上游南岸，西出便接吐蕃东境与西山诸羁縻部落，领羁縻五十四州，是唐人入剑南最西南的一处官治、亦是茶入西山的起点之一。

## 大唐天宝十四载（755年）

黎州治**汉源县**，在大渡河上游南岸的一片山间台地上。州属**剑南道西南边角**——东北循青衣江出雅州可至成都，正西越大相岭 / 邛崃山段便是吐蕃东境与西山诸羁縻部落（东蛮、西山八国等生羌之地），正南越大渡河、过越巂郡（嶲州）便是南诏的北部边境。沿革上：**隋置汉源县**，地属越巂郡，入唐改属雅州——唐前期此一带为雅州汉源县地，黎州尚未立。**武则天大足元年（701）割雅州之汉源、飞越二县及嶲州之阳山县置黎州**——以蜀南"沈黎"故地为名（《旧唐书·地理四》："取蜀南沈黎地为名"）——这是黎州建置的起点。**神龙三年（707）废黎州**，汉源、飞越复属雅州；**开元三年（715）复置黎州，又置都督府**——黎州自此为剑南道西南边角的羁縻枢纽。**天宝元年（742）随玄宗朝"改州为郡"通制改作洪源郡**，**乾元元年（758）肃宗复改为黎州**——755 这一刻州官号是"黎州洪源郡"，两名互换不分。

黎州为**下州**——这是州等里最末一等。**《旧唐书·卷四一·地理四·剑南道·黎州洪源郡》载："黎州，下，雅州之汉源县。大足元年，割汉源、飞越二县及嶲州之阳山置黎州。天宝元年，改为洪源郡。乾元元年，复为黎州，领羁縻五十四州也。领县三，户一千七百三十一，口七千六百七十八。"**——755 时黎州在册**户 1,731、口 7,678、领县三**：**汉源**（治县，越巂郡之地，隋汉源县）、**飞越**（分汉源县于飞越水所置）、**通望**（旧阳山县，原属嶲州）。一州户口才一千七百余，比邻州雅州（卢山郡）的一万八百户尚不及其六分之一——这是边州山地的实数，城里官吏、戍卒、汉源稻畦上的几百户编户齐民，便是黎州正额的全部。

**但黎州的紧要不在户口，而在它身后那五十四州羁縻**：黎州都督府西、西北、西南方向的生羌、邛部、东蛮、西山八国诸部落，皆以羁縻州形式挂在黎州名下——名义上唐土，实则各有部酋、不输赋税、不入版籍。**领羁縻五十四州**这一笔，是黎州在剑南西南诸州里最重的分量。

755 年的黎州在唐人眼里的紧要，**全在地理三面接边**：

其一，黎州**正西**越大相岭、过沈黎以西——便是吐蕃东境与西山诸羁縻部落。吐蕃自高宗、武后朝起东进剑南西山一线，**松州、维州、保州、黎州**自北向南连成剑南道的西部边界，黎州为这条边界最南的一处官治。开元、天宝间剑南节度（西山三十二羁縻州属之）与吐蕃在维州、黎州一线的攻防，是州人的日常。

其二，黎州**正南**越大渡河、过越巂郡（嶲州）——便是南诏的北部边境。**天宝十载至十三载（751-754）唐与南诏两次大败**——天宝十载（751）剑南节度使鲜于仲通伐南诏，败于西洱河；天宝十三载（754）剑南节度使李宓再伐，全军覆没——主战线在嶲州 / 姚州 / 西洱河（今云南大理），黎州作为剑南道入嶲州的中转一线，是这条战线的**侧翼后方**。城里这几年驿马、军粮、汉源稻的征调，比承平年要紧。

其三，黎州**正东北**循青衣江上游、过大相岭可达雅州（卢山郡），再东北便是益州（成都）——是黎州接入剑南腹地的唯一通道。**唐时已有自黎州、雅州西入吐蕃东境、西山诸部的小道**，零星的茶、马交换记载亦在——但"茶马古道"这四个字、"茶马互市"这套官制，要到**北宋于黎州、雅州置茶马司**以后方逐渐成型，盛唐 755 这一刻只见小道在、茶在、马在，尚无"茶马"之名。

## 历史长河

中晚唐：**广德元年（763）吐蕃东进取河陇之后**，剑南西山诸羁縻州在此后百年间反复易手——黎州西境屡为吐蕃、生羌侵扰，**唐德宗贞元年间（780 年代）韦皋为剑南西川节度使，于黎州、嶲州一线整饬边备**，黎州都督府辖羁縻州数次重订。**咸通十年（869）南诏世隆攻入成都府**，黎州亦遭兵燹——但城未失。

宋元：**北宋于黎州、雅州一线置茶马司**，专管"以茶易马"事——黎州自此成为西南茶马互市的官治起点之一，与雅州相为表里。元置黎州属吐蕃宣慰司之一翼。

明清至今：**明洪武八年（1375）降黎州为黎州长官司**，属雅州；清雍正七年（1729）废黎州长官司，改设**清溪县**，属雅州府。**民国三年（1914）改清溪县为汉源县**——"汉源"二字至此从唐黎州的治县名复活为县级建制。1950 年后属四川省雅安专区 / 雅安市汉源县。**今为四川省雅安市汉源县**——县治已迁至富林镇（瀑布沟水库蓄水后，唐黎州 / 汉源旧城多没入库底）。"黎州"二字明清渐隐——而"汉源县"为 1914 年建制名，**严禁预支**到 755 年正文。
