# 徐州

## 一句话

河南道东端上州，742-757 年间名"彭城郡"——古传彭祖封邑、西楚霸王项羽建都之地；淮北平原上四方辐辏之城，西接汴宋、东出海州、北望兖鲁、南趋淮泗。

## 形貌

徐州城在淮北一片低丘起伏的平原间——城周虽是华北大平原南缘，但城南九里山、云龙山、子房山诸座小山合抱，**是豫东冲积平原一路东来，淮北最先见到山形的一处**。城内有泗水自西北贯东南过——泗水北承济水、南入淮河，是徐州得为"淮泗咽喉"的根本。州治在彭城县——城基沿用秦汉以来旧址，城周不大，规模在唐河南道东部诸州中算中等：东去海州五百余里出海，西去汴州四百余里接通济渠，北去兖州二百里入鲁地，南去泗州、楚州二百余里入江淮——四向都通而无一向独大，正是这一州在唐人地理里"淮泗咽喉"四字的实意。

物候属淮北——比汴州略暖，比江淮略凉；春秋两季短而干燥，夏热湿、冬干寒。城外四野种麦、种粟、种豆——豫东淮北自先秦以来便是中原粮仓南缘。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

徐州境是**叠了三千年旧事的一片淮泗老地**。

最早的来路要回到**彭祖封邑**那一段古老传说——古传徐为彭祖之封，故名"彭城"。彭祖传说杂出《楚辞》《庄子》《列仙传》——盛唐人提"彭城"二字时，往最远一回望便是这一段，但所怀的是传说而非信史。商周之际，徐为东夷淮夷的一支——周代有徐国，与中原诸侯时和时叛，至春秋后期为吴所灭。**秦置彭城县**，属泗水郡——这是"彭城"作为正式县名之始。

**秦末楚汉之际**，秦亡（前 207 子婴降）后，**项羽于汉元年（前 206）四月自立西楚霸王，都彭城**（《史记·项羽本纪》）——彭城自此一跃为天下最显眼的一座城，是楚汉相争的主战场之一。汉高祖二年（前 205）刘邦率诸侯兵五十六万袭彭城，项羽自齐还师反击——**彭城之战**汉军大败，刘邦仅以数十骑南逃。汉代彭城为楚国都（高祖六年封刘交为楚王）、东汉为彭城国——西汉以来"彭城"二字一直作为郡国名号沿用。

**北周大象二年（580）置徐州**（《旧唐书·地理志》据）——沿用前代徐州州名（按：上古《尚书·禹贡》九州之一的"徐州"是泛指淮泗一带，并非州县建制；彭城为徐州治所早自北魏即称徐州，北周乃沿前代之名重置）。**隋开皇九年（589）罢彭城郡置徐州**，大业三年（607）炀帝改州为郡——徐州复称**彭城郡**。**唐武德四年（621）复置徐州**——治彭城县；**天宝元年（742）**玄宗诏天下州改郡，徐州改**彭城郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州——徐州之名再立。

### 此刻

天宝十四载（755）的徐州，行政上是**河南道彭城郡治**——一郡领彭城、萧、丰、沛、滕、符离、蕲等县（按：唐徐州辖境多有沿革，755 时大致七县；其中沛县是汉高祖刘邦故里、丰县是汉高祖出生地——汉家"丰沛"二字便指此），治彭城县。

户口上，**《旧唐书·地理志》载彭城郡（徐州）65170 户、47.8 万口——上州**（"徐州上"），属河南道东部第二线大州，不及汴、宋、相，但在淮北诸州中位居前列。城内主事：徐州刺史、州司马、州司户参军——一套常规上州官署。**755 时徐州并无节度使建制**——河南道采访使按察、本州刺史一级直管。

这一年城里的实务，仍是**淮泗水陆转运**：泗水北承济水、南入淮河的中段过境运量、汴州东南二百里通济渠折转入泗水后东行至泗州的水路（按：通济渠主干东去入淮口在泗州盱眙；徐州不直接当通济渠主线，但泗水自徐州东南入通济渠汇流处距徐州不远，故唐人言"徐州当淮泗"实包此水路网整体）——徐州刺史署里堆着的是漕运文牒、各县常平仓的报表、淮北兵戍的换防牒。

十一月初九安禄山自范阳起兵——消息沿驿道南传，约月余抵徐州。徐州东去山东河北方向、西接汴宋一线——叛军主力南下经河北、过黄河西攻洛阳；**徐州城在洛阳东南远端，755 当年并未直接陷落**。至德二载（757）正月叛军尹子奇围睢阳（宋州），张巡、许远死守十月——徐州作为睢阳东邻补给地，城内紧绷的程度仅次于睢阳本郡。**这是 755 年时尚未发生的事**——但 755 这一冬城里的兵戍换防、粮草调发，已与一年半后的睢阳之围远遥相关。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改彭城郡为徐州。唐后期徐州地位渐重——大历十四年（779）以后徐州常为节度使治：**徐泗濠节度使**（亦称徐州节度使、武宁军节度使）——治徐州。中晚唐徐州因当淮泗、河朔与江淮之中转节点，成为河南方面的军事重镇之一。

**宋元明清**　北宋徐州属京东西路（一度京东东路），仍治彭城。**北宋熙宁十年（1077）苏轼为徐州知州**——任内适逢黄河决口、洪水围城，苏轼率民筑长堤护城（黄楼之事），是徐州城建史上一段名事。金属山东西路徐州；元属归德府徐州；明初属南京（南直隶），洪武四年（1371）改属凤阳府，弘治年间升直隶州；清雍正十一年（1733）升**徐州府**——领铜山、丰、沛、萧、砀山等七县一州，自此徐州由州升府。**京杭大运河至徐州段**是元明清之后的事——元代废通济渠（汴水）旧道，开会通河直贯山东；明清京杭运河经徐州城东、过吕梁洪、入淮——**这是元明以下的事，盛唐 755 时尚无京杭运河；唐时徐州当淮泗，所凭借的是泗水与汴水东折一线**。

**民国至今**　1913 年废徐州府，置铜山县；1945 年抗战胜利后设徐州市；1948 年淮海战役以徐州为中心战场——徐州城在二十世纪中段又一度成为南北争夺的战略要冲。今徐州为江苏省地级市——市区即唐徐州彭城城基所在，城南九里山、云龙山、子房山仍在。**唯一可与盛唐徐州人共看的，是城南诸山的轮廓与泗水（今因黄河南徙改流，泗水古道已不存原貌，但水势南去淮河的方向未变）**。

## 引文

> 项王故里，沛公旧乡——一时同起，分土并王。

——《史记·项羽本纪》《高祖本纪》合论意。彭城（西楚霸王都）、丰沛（汉高祖故乡）同在唐徐州境——这是徐州在唐人地理记忆里"楚汉之争发源地"的具体实意。
