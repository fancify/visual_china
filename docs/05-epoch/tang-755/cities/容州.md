# 容州

## 一句话

岭南道容州，742-757 年间名"普宁郡"——**岭南五管之一容管经略使治**，城在北流江北岸，北出鬼门关接桂江—西江入广州，南循北流江—南流江通钦廉出南海，是岭南腹地至南海的中段一站。

## 形貌

容州城在北流江北岸的一片河谷盆地里。

四面山合：北有都峤山，唐时已为岭南名山，山中有道观佛寺数所；东、西、南三面亦皆有岭——岭南所谓"群峰拱卫"的地势，容州城是个典型。**北流江**自西北山中来——一路东南流，绕城南而过，再东南汇入桂江南支与南流江上游分水的低地。容州盆地东西不过数十里，南北亦近——城就在盆地中央偏北。州治在普宁县（今广西容县），辖境约今容县、北流、岑溪、藤县一带。

容州城周不大——下州至中州之间。《旧唐书·地理志》载容州普宁郡**天宝户四千九百七十、口一万七千八十七**（天宝后户口已经历安史之乱及岭南动荡之削减）；以之对照《元和郡县图志》所载容州**开元户一万一千一百五十六、口五万八千一百九十**——盛唐峰值户口约为天宝末两倍有余，比邕、桂偏小。本 doc 以元和志开元户作 755 年盛唐峰值之参照、以旧唐志天宝户作天宝末战乱后之参照，两数并存。但位置极紧：北出**鬼门关**（在容州北、北流—郁林之间古道隘口，唐人通称"鬼门关"，盛唐畏其瘴疠为甚），过郁林、藤州，便接桂江—西江一线入广州；南循北流江、再转**南流江**南下，可至钦州、廉州——出南海。容州这一城，扼**北流—玉林古道**与**北流江—南流江通海道**两条岭南腹地通南海的要路。物候是岭南内陆的：冬不冻、夏极湿、五月至九月雨水连绵，瘴疠时作——盛唐贬官入容州者，史志可考者数十人。

## 大唐天宝十四载（755年）

容州在唐人眼里是一座**岭南五管之一**的边疆都督府所在。

秦时此地属象郡。汉武元鼎六年（前 111）平南越，置合浦郡——容州境属合浦。三国吴、两晋、南朝宋齐梁陈，建置数易，皆隶交、广二州之合浦、宁浦诸郡。隋开皇九年（589）置**铜州**——"铜州"之名取此地素产铜——州治北流县。仁寿二年（602）改铜州为**封州**（与今广东封开之封州同名，史志两州异地）。隋末废。**唐武德四年（621）**复置铜州——治北流县；**贞观八年（634）**改铜州为**容州**——"容"取容山（即都峤山一名异说，史源未尽合）之名。**天宝元年（742）**玄宗诏天下州改郡，容州改称**普宁郡**——"普宁"取治所普宁县之名；**乾元元年（758）**复改容州。**742 至 758 这十七年间，容州官号是"普宁郡"**——史志、碑刻、文书亦同。

755 年的容州属**岭南道**，州治在普宁县。**容州都督府**自中宗朝（约 704）置——**开元中升为容管经略使治**，统辖**容、辨、白、牢、钦、禺、汤、瀼、古、绣**等州——是**岭南五管**（广、桂、邕、容、安南）之一。**容管经略使**例由本郡太守兼，秩四品——下设判官、推官、巡官；本郡（普宁郡）太守为容管首长。**注：唐于州一级本称"刺史"，"太守"为汉魏旧称——盛唐文书唯天宝-至德间改郡时短期复"太守"之名，余皆称刺史**——容州 755 年的长官官称是**普宁郡太守兼容管经略使**（742-758 改郡期内）。岭南五府经略治广州，755 时容管在五府经略统辖之下；岭南节度使要乾元元年（758）才置，755 当年还没有"岭南节度使"这个官名。

755 年冬安史起兵的消息沿驿道南传——过长安、襄州、江陵、潭州、桂州、藤州、抵容州约**三十余日**。容管接到的诏令大约是：戒严、备粮、防内属诸蛮獠乘隙——但岭南腹地距河朔战场过远，朝廷未抽容管兵北上。次年（756）玄宗入蜀、肃宗灵武即位，容管亦在岭南各自为政之列——名义奉唐正朔，实际自管自的。

## 历史长河

至德二载（757）普宁郡复称容州。中晚唐容管屡有蛮獠之乱，但容州城本身建置稳。**唐宋之际**，容州先后为南汉、北宋所据——北宋仍置容州，属广南西路；**容管之名沿用至北宋**，为五管旧制最后一段尾声。元置容州属广西行省梧州路；**元后容州行政地位逐渐缩水**，渐沦为下州。明洪武十年（1377）**降容州为容县**，属梧州府——容州六百四十余年的州级建置自此终。清沿明制。**民国元年（1912）**废府存县，"容县"之名自此入官府文书——盛唐"容州"之号，至此已七百余年仅存于史册。今日容县为广西玉林市辖县，人口约七十余万；唐容州城遗址在今容县县城一带，城北都峤山仍为容县名山。**自唐贞观八年（634）置容州至明洪武十年（1377）降为容县，凡七百四十三年——容州"州"级建置一段。**

## 杨贵妃容州说（必要 hedge）

后世野史有"杨贵妃容州人"之说，但**《旧唐书·杨贵妃传》明载"父玄琰，蜀州司户，贵妃生于蜀；其家本蒲州永乐人也"**——主流正史以**蒲州永乐**（今山西永济）为贵妃本籍、**蜀州**（今四川崇州）为生地。**"容州说"出元代后野史**（元、明、清地方志与文人笔记零星可见，主要见明代以下之《容县志》等地方文献），并非《旧唐书》《新唐书》正载。盛唐 755 年的容州人若听人说本州出了杨家女做贵妃——会以为是奇闻而非旧事；这一说要再过五六百年才入民间口耳，又再过几百年才入地方志。本 doc 仅录此说为后世传说之一脉，**正史所主仍取蒲州永乐**。

## 引文

《元和郡县图志·卷三八·容州》：

> 容州，普宁。中都督府。管州十：容、辨、白、牢、钦、禺、汤、瀼、古、绣……开元户一万一千一百五十六，元和户三百六十二。……州城北有都峤山。

——这是唐人对容州城形与辖境最简的一笔：城北都峤山、所管十州，便是容管经略的全部。
