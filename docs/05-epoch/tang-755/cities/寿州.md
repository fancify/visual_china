# 寿州

## 一句话

淮南道淮水中游一州，治寿春县（今安徽寿县）——楚考烈王二十二载迁都之地，淝水西陲；唐人称"寿春"或"寿春郡"，控淮水南北舟楫之冲。

## 形貌

寿州在淮水南岸的一片缓岗上。州治寿春县城就盖在淮水与淝水交汇之南——城北濒淮，城西临淝；东南去合肥（庐州治所）一百八十里，西去光州（光山）三百余里，北过淮便入颍州、亳州境。这片地势的根本特点是**水陆之冲**——淮水东西大动脉自此过，南北舟楫由淝水入淮、出淝抵江——盛唐人言"江淮人走崤函，合肥、寿春为中路"，便是此意。

城北八公山起伏一片，山不甚高、林木苍翠——西汉淮南王刘安都寿春时尝游此山，《淮南子》一书便是这一带门客集编的产物。城西十余里淝水自南来，东晋太元八年（383）秦晋淝水之战即在此一段水滨——"草木皆兵"四字典出此战。城南六十里入安丰县境，**芍陂**横亘——春秋楚相孙叔敖于楚庄王朝（约前 597 前后）筑此陂束水蓄灌，"灌田万顷"，至唐仍是淮南最大的人工陂塘。**自孙叔敖至唐，一千三百余年，此陂稻畦从未荒过**。

物候上属淮南过渡带——春来不算早，夏不甚酷，秋雨连绵，冬偶有薄雪。州贡以稻、麻、蒲、鱼见名；淮鱼自此东运的传统起自汉魏。

## 大唐天宝十四载（755年）

寿州沿革极早。**春秋时蔡昭侯自上蔡迁此**，号州来——故寿春一名亦称"州来"，《左传》屡见。**楚考烈王二十二年（前 241）自陈徙都寿春，命曰郢**——自此至秦灭楚（前 223）凡十八年，楚顷襄王之后的考烈王、幽王、哀王、负刍四王皆都此。城北八公山下的诸楚王陵中，**武王墩一号墓**（2024 年考古确认为楚考烈王墓）即其一——盛唐人来此凭吊，犹是寻常事，只是当年人未必能指认具体哪一座坟属哪一王。汉文帝十六年（前 164）刘安袭父刘长之嗣**封淮南王、都寿春**；至**武帝元狩元年（前 122）刘安谋反案发、淮南国除**——但**《淮南子》一书自寿春传出**——盛唐文士每至此，便会想起八公山下那一批门客。

建置上：**隋开皇九年（589）平陈后置寿州**，治寿春县；大业三年（607）罢州改置**淮南郡**——这是"淮南郡"作为正式郡名第一次入隋制。唐**武德三年（620）杜伏威归唐**，于此地复置寿州（**按：旧唐书系武德三年；异文《太平寰宇记》作武德二年，本 doc 从旧唐书**）；武德七年置都督府督寿、蓼二州，贞观元年罢都督府。**天宝元年（742）**玄宗诏改州为郡，寿州改称**寿春郡**——是为 755 年当年的官称；**乾元元年（758）**肃宗复改郡为州，寿春郡复称寿州。盛唐寿州属**淮南道**——淮南道治扬州（江都郡），辖扬、楚、滁、和、舒、庐、寿、光、蕲、安、黄等州，寿州在其辖境西北角，扼淮水中游。

户口上，《旧唐书·地理志》载寿州"**旧领县四，户二千九百九十六，口一万四千七百一十八**"（贞观初）；至天宝，"**领县五，户三万五千五百八十二，口一十八万七千五百八十七**"——一百二十年间户数增逾十倍，是盛唐淮南腹地开发的硬证。寿春郡领**寿春、安丰、霍山、盛唐、霍丘**五县——城南六十里安丰县境内即芍陂；西南霍山、盛唐二县在大别山东北麓；东北霍丘县在淮水沿岸。州等第为**中州**——按唐制"户四万以上为上州、二万以上为中州、不满二万为下州"，寿州近中州上限。

755 年的寿州，行政上是**淮南道寿春郡治**。城内主事为寿春郡太守（按 742-758 改州为郡间，州刺史一律改称太守，至德二载/757 末又复称刺史）、郡丞、司马、司户参军、司仓参军——一套中州小郡的官员系统。本州军事不属节度——淮南道至天宝末尚未置节度使，州兵由州自办，受淮南采访使节制。这一年寿州在战略上是**淮水中游联运枢纽**：自荆州、襄州、江陵北上汴洛的漕运须由淝水入淮，自江都、楚州西去关中的漕舟亦由淮水过寿春；州城周围既无大兵驻屯，亦无大将出镇——盛唐淮南腹地的州，多半如此。

天宝十四载冬安禄山反范阳。消息沿驿道南传——过陈州、颍州，**抵寿州约二十日**。寿州太守接到的诏令大约是：戒严、整漕、备粮——淮水漕运在乱起后立刻成为南北接续中原的命脉，寿春正坐在这条命脉的腰上。次年（756）玄宗西奔入蜀，肃宗北赴灵武即位，关中漕粮断绝——淮南诸州的漕舟、稻米、蒲茭、铁器，**几乎是这一年此后撑起朝廷半边天的物资**。寿州虽未亲历战火，淮水漕路的转输负担一年比一年重。

至于**淝水**——城西十余里那条小水，在 755 年的寿州人眼里大约只是一条寻常溪流。但自此再过二百年，宋初柴荣的兵锋将再扫此城；再过千余年，"草木皆兵"四字仍是中文里讲风声鹤唳第一个引的典——盛唐人不会预知这些，他们只知道这是城西的水，每年涨秋两季，淝水入淮，舟行甚利。

## 历史长河

**中晚唐**（758-907）　乾元元年（758）复州名为寿州。中晚唐寿州因淮水漕运成为淮南最重要的转运节点——元和、长庆间庐寿转运使（驻寿州）调度淮南诸州漕粮入汴入洛，是这一时期寿州的财政重要性所在。唐末黄巢之乱（875-884）波及淮南，寿州数度易手；天复二年（902）杨行密据淮南立吴，寿州入吴国版图——唐廷至此再未管到这一片淮水南岸。

**五代两宋**（907-1279）　寿州先属杨吴、后属南唐。**后周显德四年（957）寿州之战**——周世宗柴荣亲率大军围寿春，南唐节度使**刘仁赡**主守，城中坚守逾年，刘仁赡疽发背卒于围中，城终破——这一战是后周取淮南十四州的关键一役，《资治通鉴》卷二九三-二九四记之最详。宋初寿州为淮南西路治所，但州治后移合肥（庐州）；金宋对峙时寿州在淮水南岸为南宋边州——南宋初年屡有金兵渡淮南下经此。

**明清民国**（1368-1949）　明洪武初降寿州为散州，属凤阳府；清雍正二年（1724）升直隶州，旋复散州，仍属凤阳府。1912 年废州存县，改寿州为**寿县**，属安徽省。盛唐"寿州"二字至此罢，让位于一个单字县名。

**民国至今**　寿县城至今仍立淮水南岸——县治承明清寿州城旧址，**今寿县古城墙是宋代重修的实物**，周七里二百步、四门四瓮城——这是中国保存最完整的宋代州城城垣之一。1986 年寿县列为国家历史文化名城，2001 年寿县古城墙列为全国重点文物保护单位。盛唐寿州城的夯土城基便叠在宋城墙之下——考古钻探可达，但完整的唐城轮廓至今未发掘清楚。城南六十里安丰塘（即唐人所称芍陂），2015 年列入"世界灌溉工程遗产"——**自孙叔敖至今二千六百余年，灌溉未断**。
