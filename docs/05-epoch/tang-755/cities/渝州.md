# 渝州

## 一句话

嘉陵江入长江口一下州——治巴县（今重庆老城渝中半岛），州名"渝"取自嘉陵江下游古名渝水；**两《唐书》归道有出入：旧志列山南西道末、新志归剑南采访使**，是巴蜀通黔中、走荆襄的水上节点。

## 大唐天宝十四载（755年）

渝州治**巴县**，在嘉陵江自北而南汇入长江的夹角半岛上——两江抱城，是巴蜀东南最锐利的一处水口。州名"渝"二字本自**渝水**（嘉陵江下游古名），魏晋以来一脉相承。**隋为巴郡**，**唐武德元年（618）置渝州**，因开皇旧名；**天宝元年（742）随玄宗朝"改州为郡"通制改为南平郡**，755 时正名为"渝州南平郡"，两名在唐人嘴里互换不分；**乾元元年（758）肃宗复改为渝州**——但这是 755 之后的事。州属归口在两《唐书》间有出入：**《旧唐书·卷三九·地理二》将渝州列于山南西道之末**（位于陵州、荣州、瀘州、昌州、保宁都护府之前的边角位置），**《新唐书·卷四二·地理志六》改隶剑南采访使**——755 时正值玄宗采访使分道盛行，渝州在制度上的"剑南/山南"两可，是其地居二道交界、又介黔中之北的实况。《新唐书》载渝州为**下州**——领县虽少，户口亦薄，在剑南诸州中属末等。**《旧唐书·卷三九·地理二·山南西道·渝州》载：**"隋之巴郡。武德元年，置渝州，因开皇旧名……天宝元年，改为南平郡。乾元初，复为渝州。**旧领县四，户一万二千七百一十，口五万七百一十三。天宝，户六千九百九十五，口二万七千六百八十五**。在京师西南二千七百四十八里，至东都三千四百三十里。"——755 时渝州在册**户 6,995、口 27,685、领县四**（巴、江津、万寿、南平），户口较武德旧籍 (户 12,710、口 50,713) 折去几近一半——这是隋末蜀乱后入唐百余年间州境西析（南平州→霸州→并入渝州）与人口未复的实数。州境是**古巴子国地**——汉为江州县属巴郡，梁置楚州，**隋改为渝州，以水（渝水）为名**；唐人提渝州必带"两江夹城、巴子旧都"两笔，但唐时渝州在剑南诸州中并不显赫，无大兵无大事，只是巴蜀通黔中（去黔州、思州、播州）、走荆襄（顺长江下涪州、夔州）的一处水上转运点。

## 历史长河

**北宋**　**熙宁五年（1072）取消南平军建制**，渝州地仍属夔州路。**元符元年（1098）**因"渝"与赵谂之名同音忌讳，**改渝州为恭州**。**南宋淳熙十六年（1189）**，宋光宗赵惇先封恭王、后即帝位，"双重喜庆"取义升恭州为**重庆府**——"重庆"地名自此而立，已是渝州废名 91 年后事。

**元明清**　**元至元十六年（1276）置重庆路**属四川行省；**明洪武四年（1371）改重庆府**；清因之。**清末光绪二十一年（1895）《马关条约》开重庆为商埠**——是巴蜀近代化起点。**1929 年正式设重庆市**；**1939-1945 抗战陪都**；**1997 年 6 月升直辖市**——皆唐渝州废后千年之事，本 doc 严守 755 时态不预支。

## 引文
