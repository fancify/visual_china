# 江州

## 一句话

江南西道一处中州，**唐人口中"浔阳"的那个浔阳**——长江南岸、鄱阳湖湖口、庐山下；东林寺已立四百年，山下柴桑还留着陶潜旧宅几间。

## 形貌

江州的城就盖在长江与鄱阳湖相接的那个口子上——这是它的根本。

长江自荆州、鄂州东流而下，过武昌江口再行三百余里，到江州城北便开始与鄱阳湖**湖口**相通——古称"湓口"，因有湓水（湓浦）自城西注入大江而得名。城西的湓浦口是江州最重要的码头，也是后世白居易**送客闻琵琶**的那一处水边。江北便是淮南道蕲、黄二州的地界；江南便是这一片浔阳平原，再南二十余里便是庐山——一山一江一湖三事相接，**江州是这三者交汇的点**。

唐江州州治在**浔阳县**——一州只领浔阳、彭泽、都昌三县。州城在今九江市的位置，背枕庐山余脉，面临长江——城周约六里，唐人记得不甚详；夯土城墙开东、西、南、北四门。城内有刺史衙、司马厅、州学、佛寺数所，外有湓浦码头与江岸驿——驿道自此东去洪州、西去鄂州、南渡鄱阳入赣江、北过江入蕲州。

物候是江南西道的：春多梅雨，夏热湿，秋有"浔阳江头夜送客，枫叶荻花秋瑟瑟"的那种秋意，冬偶有薄雪。一江之水在此分两道——主流东去金陵入海，支流南折入鄱阳——使江州成为长江中下游漕运、商旅、贬官、僧侣的天然中转站。城北望江一江白水，城南望山一片青黛——盛唐人从船头看这座城，第一眼便看见庐山。

至于山——**庐山的事不在这一篇 doc 里讲**（参 [庐山](../scenic/庐山.md)）。江州 doc 里只记一句：唐时江州州境东南即庐山诸峰，州治距庐山北麓东林寺约二十余里——盛唐人下了江船入江州城，再雇马或步行半日，便到了慧远当年驻锡的山下。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

江州在唐人眼里**已是一处叠了七百多年州治的旧城**。

晋元康元年（291），晋惠帝分荆、扬二州十郡置**江州**——这是"江州"作为一州名号的初立。初治豫章（今南昌），后徙寻阳——"寻阳"二字是江州治所自晋以来的核心地名。东晋至南朝二百余年间，江州一直是建康（今南京）西面的门户、长江中游的军政重镇——南朝宋、齐、梁、陈历代皆于此置重兵。**晋宋之际江州刺史**这一职位极重——桓伊、王凝之、刘毅、王弘等皆曾任江州刺史。便是这位江州刺史**桓伊**，于东晋太元六年（381）助慧远入庐山——后又助建东林寺。

隋开皇九年（589）平陈，**置江州**——此时已不像晋宋那样统数郡，仅辖湓城、彭泽诸县。大业三年（607）罢州置**九江郡**——"九江"作此地郡名首见于此（"九江"之名古已有之，《尚书·禹贡》"九江孔殷"即指；秦汉亦有九江郡，但治寿春，统今安徽淮南至江西九江一带，非此地专名）。**唐武德四年（621）**讨平林士弘，复置江州。武德五年（622）置**江州总管府**，后改都督府；贞观初省都督府，江州仍为下州。

**天宝元年（742）**玄宗诏天下州一律改称郡——江州改称**浔阳郡**。**至德二载（757）**肃宗复改郡为州——但 755 年这一刻，城门匾额上写的是"浔阳郡"。**乾元元年（758）**复称江州，自此稳定至唐末。

唐人若 755 年在江州城里回望——七百年前是孙吴的柴桑水军、三百多年前是慧远东林、三百二十多年前是陶潜《归去来》、一百六十多年前是隋平陈的兵船——这片江湖之地的层叠之厚，比北方某些边州深得多。

### 此刻

天宝十四载（755）的江州，行政上是**江南西道浔阳郡治**——领浔阳、彭泽、都昌三县。一州三县，是江南西道里规模偏小的州——所辖远不如洪州（豫章郡，治今南昌，江南西道首州）、鄂州（江夏郡）。但江州的分量**不在户口而在位**：长江中游入鄱阳的湖口、东林白莲社所在、陶渊明故里——三笔加在一起，是它在文化版图上的地位。

户口据《通典》《元和郡县图志》记浔阳郡天宝间约一万七千余户——按**开元十八年（730）后口径**（四万户以上上州 / 二万至四万中州 / 二万以下下州），江州一万七千余户严格论已落入**下州门槛**，惟接近中州下限，故唐代地志多以"中州偏下"或"中下州"概称。在江南西道里属二线州——介于洪州（户口约五万，上州）与一般下州之间。本州本身**未置节度使**——江南西道至天宝末年尚未单立节度，755 年时归**江南西道采访使**节制（治洪州），军事调度则随事临时置使。江州城本身无大军镇，本州常驻团结兵数百至千许——这是一座**文州**而非武州。

这一年江州城里的主事：刺史一员（具体姓名敦煌写本之外无系统记录，且诸地方志多缺这一年人选）、司马一员、司户参军、司仓参军——一套标准的中州官员。城里有州学，城北有湓浦驿——驿夫马匹常备，是长江中游驿道的一个节点。

755 年冬，安禄山起兵的消息沿驿道自长安南传——过荆襄、过江至江州，约需十余日。江州刺史接到的诏令大约是：戒严、备粮、整团结兵、待调。江州本无重兵——但作为长江中游漕粮转运的一个节点，江州的米粮、江州的船工、江州的水手，在接下来八年的战事里都将派上用场。**江州的重要性此后不增不减——它始终是一州中州**——但它真正在文学版图上发光，是再过六十年的事——

那时一位名叫白居易的诗人，将自长安贬来江州做司马，在城北湓浦口送一位客人——客船边坐着一个琵琶女。

那是元和十一年（816）秋——离 755 年这一刻还有六十一年。

### 此刻的意味

对盛唐人而言，江州是**南来北往的长江中游一站**——李白游历江南时多次过此（开元至天宝间凡数过浔阳、登庐山——李白《登庐山五老峰》《望庐山五老峰》皆出此期），孟浩然、王昌龄、刘长卿、戴叔伦等亦多有浔阳诗。**唐人对江州的标准印象是三件事**：一是庐山东林，二是陶渊明故里，三是大江与鄱阳交汇的湖口风物。755 年的江州城里，士人来谒东林、访栗里、登庐山——还和东晋陶潜归隐时的格局差不多。

至于陶渊明本人——他已殁三百二十八年（约 427-755）。柴桑县境内的旧宅遗迹，盛唐人仍可寻访——"栗里""上京里""南村"诸名口耳相传。陶诗"采菊东篱下、悠然见南山"那座南山，便是城南庐山——755 年的浔阳人仍每日望见。

## 子条目

### 浔阳县（江州州治）

唐江州治所。城在长江南岸、湓水（湓浦）入江口附近——今九江市区一带。州城与县城同址，刺史衙与浔阳县衙同在一城。城周约六里，夯土城墙。城西门外有**湓浦码头**——是江州最重要的码头，长江漕舟、商旅、贬官、僧侣多自此上岸。城东门外驿道东去洪州——过都昌、入鄱阳湖东岸。城南门外驿道直抵庐山北麓东林寺、西林寺一带。城北门外便是长江——江北是淮南道蕲州、黄州地界。唐浔阳县治所至宋以后仍为江州（南宋改江州为定江军、元改江州路、明改九江府）治所——但**城址数迁**，唐浔阳故城精确位置今已不可指，约在今九江市区西部一带。

### 彭泽县

江州下辖第二县——治彭泽城（今江西彭泽县西北），在江州城东约一百八十里，临长江南岸。彭泽县境内有彭蠡湖（即鄱阳湖古称之一——盛唐时"彭蠡"与"鄱阳"两名并存）东北口、马当矶（长江三大矶之一）等。这一县最有名的旧事——**陶渊明义熙元年（405）八月任彭泽令**——这是陶渊明仕途的最后一站。**为令八十余日**，闻郡督邮将至，叹"吾不能为五斗米折腰向乡里小儿"，**赋《归去来兮辞》解印去**——唐时彭泽县境内尚有传说中的陶令故宅遗迹。盛唐李白《赠崔秋浦》"陶令日日醉，不知五柳春"即用此典。

### 都昌县

江州下辖第三县——治都昌城（今江西都昌县），在江州城东南约一百二十里，鄱阳湖东岸。这一县是江州伸入鄱阳湖东岸的一只触角——本是汉鄡阳县（已沉入鄱阳湖底）地，唐武德五年（622）置都昌县，"都昌"取"都氏昌盛"之意（一说"鄱昌"音转）。都昌县境内有南山——后世周敦颐《爱莲说》"濯清涟而不妖"之莲，相传是都昌南山之莲；但这是北宋之事，盛唐不预支。

### 湓浦码头（湓口）

江州城西门外，湓水（亦称湓浦、湓浦水）入长江之口——亦称**湓口**。**汉建安十四年（209）**孙权将周瑜屯柴桑、刘备见周瑜于柴桑——便是这一片江口。唐时湓浦是长江中游最重要的码头之一——漕舟、客船、商旅、僧侣多自此上岸入江州城。湓浦水自城西山岭流来，绕城而入江——水浅不通大船，但岸边码头与江中船相接。**白居易元和十一年（816）秋于湓浦口送客**——船边闻琵琶——作《琵琶行》："浔阳江头夜送客，枫叶荻花秋瑟瑟。主人下马客在船，举酒欲饮无管弦……"——这首诗的开头四句把湓浦码头的秋夜江景写定了千年。但白居易这一刻还在长安——离他贬江州还有六十年。

### 东林寺（东晋慧远立，唐时仍盛）

庐山北麓——江州境内最重要的佛寺。**东晋太元六年（381）**慧远入庐山驻锡——江州刺史**桓伊**助建东林寺。寺成纪年异文并存（一作太元九年 384，一作太元十一年 386）。**元兴元年（402）**慧远集刘遗民、雷次宗等沙门居士百二十三人于东林寺无量寿佛前发愿——同求往生西方净土。这便是**白莲社**——后世推为净土宗祖庭/发祥地（"净土宗"作为宗派建构是中古以后才完整定型的，盛唐时尚未正名为"祖庭"）。**义熙十二年（416）**慧远卒于东林——驻锡庐山三十余年。

唐时东林仍为南方佛门重镇之一。盛唐高僧法照（后世推为净土宗五祖）大历二年（767）受感而入庐山——但 755 年这一刻尚未到。755 年的东林寺仍循其旧规——僧众数百，寺产田业绵延庐山北麓数里。盛唐人到江州，多先访东林——这是江州城外第一访点。

详见庐山 doc。

### 西林寺

庐山北麓——东林寺之西。**东晋太元二年（377）**慧远兄长**慧永**（332-414）先建西林寺——比东林早四年。慧永卒后寺由后人住持，规模始终不及东林。盛唐犹存。苏轼元丰七年（1084）题诗西林壁——"不识庐山真面目"——是此寺最广为人知的一次出场，但已是三百二十九年后的事了。

详见庐山 doc。

### 陶渊明旧宅（柴桑栗里）

江州浔阳县境内——汉柴桑县地（柴桑县唐初已并入浔阳县）。**陶渊明（约 365-427）**籍贯浔阳柴桑——故里在庐山西北麓平地。**栗里**在柴桑南，是陶渊明晚年居所之一；**上京里**、**南村**亦传为陶居。陶诗"结庐在人境""归去来兮""采菊东篱下"诸句的实景，便在这一片浔阳柴桑的稻田溪水间。

盛唐时陶渊明已殁三百二十八年（约 427-755）——旧宅遗迹半存。**虎溪三笑**故事附会陶渊明曾来访慧远、共陆修静过虎溪而笑——三笑之事系后人造（陆修静卒于刘宋元徽五年 477，慧远义熙十二年 416 已卒——三人异时，不能同笑），但"陶在山下、慧远在山上、虎溪居中"的空间格局，盛唐人皆识。

唐时陶渊明声名渐起——盛唐李白、王维多有受陶影响。**李白《赠崔秋浦》《九日登山》《田园言怀》**皆涉陶；**王维《偶然作六首·其四》**"陶潜任天真，其性颇耽酒"——直引陶事。**孟浩然《仲夏归汉南园寄京邑旧游》**亦学陶之归田。755 年的盛唐士人提起陶渊明——已渐成一种**文学共识**：南方隐逸文学的开山祖。中晚唐白居易、苏轼以下千年的"慕陶"传统——这一脉的根脚便扎在浔阳柴桑这片平地里。

### 庐山诸寺（江州境内）

江州境内庐山诸寺甚多——东林、西林、大林、栖贤、归宗等晋宋以来古刹皆在江州地界。这些寺院的本传不在江州 doc 里讲（参 [庐山](../scenic/庐山.md)）——但它们都向江州州衙缴纳寺产田赋、登记僧籍。江州州学与庐山诸寺之间，是盛唐江州地方文化两套并行的网络——一套官学、一套佛门。

### 鄱阳湖湖口

江州城东数十里——长江与鄱阳湖（盛唐时鄱阳湖远比今日为小，但已有相当水面）相接的湖口。这一带是江州境内的水网枢纽——长江漕舟自此南折入鄱阳，过赣江上达虔州（今赣州）；自鄱阳东岸沿赣江支流亦可往洪州（今南昌）。**唐人称"湖口"已是地名**——湖口戍是江州境内的一处镇戍点。鄱阳湖的本传不在江州 doc 里讲，但江州与鄱阳的地理关系是江州**漕运中转站**这一身份的根本——长江—鄱阳—赣江这条水路，是盛唐岭南向长安输贡的命脉之一。

## 历史长河（755 → 今）

**中晚唐**（755-907）

至德二载（757）江州复名江州（自浔阳郡复），乾元元年（758）后稳定。安史之乱及以后江南未遭兵燹——江州反成北方士人南奔的一处避地。**元和十年（815）**白居易因上书议武元衡被刺事件被贬——**元和十一年（816）秋至元和十三年（818）冬**任**江州司马**——这是江州在唐文学史上最高光的一段。元和十一年秋于湓浦口送客作《琵琶行》——"浔阳江头夜送客，枫叶荻花秋瑟瑟"；元和十二年（817）四月游庐山大林寺作《大林寺桃花》——"人间四月芳菲尽，山寺桃花始盛开"；又作《庐山草堂记》——为唐代山水记之范本。白居易江州三年——把江州这座城写成了中唐第一文学州。

中晚唐皇甫湜、孟郊、刘禹锡、顾况、韦应物等亦多有浔阳诗。**江州司马**这一职位自白居易后成中唐贬官文化的一个符号——"江州司马青衫湿"的典故便从此起。

**五代至宋**（907-1279）

南唐李氏立国金陵——江州入版图。南唐保大年间（943-957）于庐山南立**白鹿洞**前身（初为李渤兄弟读书处）。北宋初年定为官学——南宋淳熙六年（1179）朱熹任**南康军**（治所即在庐山下星子县）知军时重修白鹿洞书院。江州州治宋时仍为江州，南宋初**改江州为定江军**，旋复为江州——绍兴二年（1132）置江州都督府以镇江防。北宋熙宁五年（1072）陈舜俞作《庐山记》三卷——所记之"江州"是这一段。**南宋抗金、抗蒙**期间江州是长江中下游的战略要冲——德祐元年（1275）伯颜攻江州，江州陷——这一陷比临安陷还早一年。

**明清**（1368-1911）

**元改江州路**，**明洪武九年（1376）改江州路为九江府**——"九江"二字自此作为正式府名定下，沿用至清。明清江州（九江府）治所稳定在今九江市区一带——但**唐浔阳故城**位置已数迁不可指。明末清初九江为长江中游商埠——徽商、晋商在此设号——江州自唐以来的水路中转站身份延续。**1858 年**第二次鸦片战争后《天津条约》——九江被列为**通商口岸**——1861 年正式开埠，英、日、俄等国设立租界于浔阳门外江边——这一时段九江成长江中游对外贸易的重要节点。1894 年甲午战后租界扩展，1927 年北伐军收回。

**民国至今**（1911—）

1912 年废九江府，置浔阳道——"浔阳"作为道名重回正式行政，但只持续到 1927 年。其后历次行政区划更易——1949 年后设九江市（专员区/地区），1983 年九江撤地区设地级市——今**江西省九江市**，辖二区三市八县，人口约四百五十万。城区中心今仍在长江与鄱阳湖湖口处——浔阳区即为今九江市主城区。**九江长江大桥**1992 年通车——长江中游水陆要冲的盛唐遗韵于此可窥一斑。

**白居易江州司马旧址**：今九江市浔阳区**琵琶亭**——位于长江大桥南端江边，1987 年重建——是后世对"湓浦口送客"那一夜的纪念。**琵琶亭**唐时本无，**北宋初**始有人于湓浦口附近立亭——以后历代屡毁屡建，今亭是 1980 年代复建。盛唐天宝十四载这一刻——湓浦口尚无亭，只有码头与江水。

**东林寺**今仍在庐山北麓——1980 年代起为净土宗复兴的重要道场，日本、韩国净土宗信徒朝山者不绝。**陶渊明纪念馆**在今九江市西南柴桑区（2010 年代由九江县改设——专以"柴桑"二字立名），有陶渊明墓园、栗里故宅复建——盛唐人若 755 年寻访的栗里，今天还可在地图上找得到。

至于盛唐那座方圆六里、城门匾额上写着"浔阳郡"的小城——夯土城墙在今九江市区的层叠下早已埋没，**考古至今也没碰到过完整的城圈**。**江州司马**白居易当年在城中所居之"司马厅"——具体位置已不可指。一千二百多年过去，那座城名换了几番——江州→浔阳郡→江州→定江军→江州→江州路→九江府→浔阳道→九江市——但**长江、湖口、庐山**三件事一件没变。

## 引文

李白《望庐山瀑布二首·其二》（约 760-761，去 755 时五至六年）：

> 日照香炉生紫烟，遥看瀑布挂前川。飞流直下三千尺，疑是银河落九天。

——李白晚年三上庐山——皆自江州湓浦上岸。

白居易《琵琶行》（元和十一年 816 秋，于江州湓浦口）开篇：

> 浔阳江头夜送客，枫叶荻花秋瑟瑟。主人下马客在船，举酒欲饮无管弦。

——白居易江州司马任上送客作。**距 755 年这一刻六十一年。**

白居易《大林寺桃花》（元和十二年 817 四月，于庐山大林寺）：

> 人间四月芳菲尽，山寺桃花始盛开。
> 长恨春归无觅处，不知转入此中来。

——白居易江州司马三年间登庐山多次——大林寺花径草堂之作。

陶渊明《饮酒·其五》（约义熙年间，约 416 前后，去 755 时三百三十余年）：

> 结庐在人境，而无车马喧。问君何能尔，心远地自偏。
> 采菊东篱下，悠然见南山。山气日夕佳，飞鸟相与还。
> 此中有真意，欲辨已忘言。

——诗中"南山"即庐山，"东篱"即栗里宅院。**盛唐江州人**仍每日望见这座南山。

《元和郡县图志·卷二八·江南道·江州》（李吉甫 813）：

> 江州，浔阳郡。开元户一万七千六百九十九，乡四十六。……州城在长江之南，背枕庐山，前临大江。东至洪州五百里，西至鄂州四百二十里。

——江州在 755 年前后户口、方位的一手史源。
