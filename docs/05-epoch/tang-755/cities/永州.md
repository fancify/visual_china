# 永州

## 一句话

江南西道永州治所，治零陵县（今湖南永州市零陵区）——湘水与潇水合流之处，南楚边境一座中州小城，**755 时尚是州县小郡，柳宗元贬此尚在五十年后**。

## 形貌

永州城在湘水中上游南岸——湘水自南来，到零陵县城下一段，与西南来的潇水合流：水色一清一浊，合流之处称『潇湘』。**『潇湘』二字自此而入诗——盛唐之前，已是江南山水的一个代名词**。城周不大，夯土城垣依地势而立——东傍湘水、北望潇水入湘口；城南、城西是连片的低丘与稻畦，更南便是越城岭余脉、南岭山系——南岭以南便是岭南道桂州境。

零陵地势是湘水中上游一片开阔的小盆地，比下游潭州、衡州狭窄，比再南的桂州、贺州开朗。物候上，永州属湘水中上游——春来早、雨多，夏不极热，秋日江风渐凉，冬偶有薄霜、罕雪。林木以樟、楠、栎、竹为主；山间多药草、葛、麻；水边稻田连片，但产量较江南膏腴远不及。**地僻、人稀、税轻——但也意味着远离中原**。

唐人提『永州』、提『零陵』，最先想到的是两件事：一是**舜南巡崩于苍梧、葬于九疑**的古传——零陵县名相传即由舜陵而来（『零陵』之『零』，古字与『陵』近义，相传为舜陵之异称，此说见于汉以来诸书，确否史家有疑，本 doc 作传说处理）。二是**南楚边境的偏远**——零陵入正史，自秦置长沙郡、汉置零陵郡始，便是南楚最南的一片地。盛唐人入永州，便有『至此已是天下南鄙』之感。

## 大唐天宝十四载（755年）

永州沿革久远。**秦置长沙郡**辖此一带；**汉武帝元鼎六年（前 111）置零陵郡**——这是『零陵郡』作为正式郡名第一次入史，治所最初在零陵县（今广西全州境，与唐零陵县非同地），后东汉徙治泉陵县（今湖南永州市零陵区）。汉零陵郡辖湘水中上游一带广大地域，唐时已分置数州。三国时零陵郡属吴；两晋南朝相承。**隋开皇九年（589）**隋平陈，置**永州**——州治始定于零陵县（即今湖南永州市零陵区）；这是『永州』作为正式州名的隋制起点。**大业三年（607）**改永州为**零陵郡**。

入唐之后。**武德四年（621）**唐平萧铣，复置永州——治零陵县。**天宝元年（742）**玄宗诏天下州一律改称郡，永州改称**零陵郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州，零陵郡再称永州——但 755 年这一刻，城门匾额上写的还是『零陵郡』。

户口上，《通典》《元和郡县图志》载永州天宝间约 **二万七千余户**（《元和郡县图志》『开元户二万七千四百九十四』），元和年间户数大跌至约六千余——按唐州等第：**唐上州四万户、中州二万户、下州不足二万户**，永州天宝间约 2.7 万户**属中州下等**，但元和间已退为下州。755 年这一刻，永州正在它中晚唐人口骤降之前的最后一段相对充实的时光。

永州属**江南西道**——江南西道治洪州（今南昌），辖洪、江、鄂、岳、潭、衡、永、道、郴、邵、桂等州。永州在其辖境西南——再南便是岭南道，再西南便是桂州、岭南通衢。**755 年时江南西道未常设节度，由采访使分巡——永州太守（时官称，742-758 郡制下州长官称太守）归江南西道采访使节度**。州内官署即州衙、零陵县治、馆驿数所——一套中州小郡的简单班子。

这一州在盛唐尚无后世那般文学分量——永州的『出名』，要等到中唐**柳宗元 805-815 贬永州司马**之后（详见〔历史长河〕段）。**755 年这一刻，柳宗元尚未出生**（柳宗元生于大历八年/773）——后世人凭《永州八记》认得的小石潭、西山、钴鉧潭、袁家渴、石渠、石涧、小石城山——755 年时都还在城西郊的山林溪谷里默默无名。盛唐入永州者，多为南下贬官、宦游、过岭赴桂州、广州的官人——但分量都不及柳宗元那一笔。

物产上，永州贡葛布、麻布、药材——是江南西道众州中较寻常的一份。城内市井不盛，街巷简朴；州城外湘水上有渡口与商船往来——上行可达桂州、下行可至衡州、潭州、洪州。永州正在『湘水中上游—南岭—桂州』这一条南北通衢的中段。

755 年冬安禄山反范阳——消息沿驿道南传至永州，约需月余。但**永州偏处岭南北面、湘水中上游，本不在乱事主路上**——次年玄宗西奔入蜀（参 [益州·玄宗入蜀路线](./益州.md#玄宗入蜀路线-756-年六月至七月)），一行循金牛道入蜀，未经永州境；但乱中士人自洛阳、长安南奔者，多有沿湘水南下、过永州入岭南避乱者——**永州遂在安史之乱期间成为江南西道至岭南道的一处避乱通道**。这是 755 之后的事，盛唐人并未预知。

## 子条目

### 零陵城（永州治、零陵县治）

唐永州治所，亦零陵郡治、零陵县治——三套官称同在一城。城周较中州小，夯土城墙依湘水南岸地势而立——东门贴江，可俯瞰湘水、潇水合流之处。城内有永州太守衙（742-758 郡制下州长官称太守；常名仍呼『刺史衙』）、零陵县衙、馆驿、佛寺数所，城外湘水有渡口。**唐零陵故城在今湖南永州市零陵区，今地表已无完整城圈遗存**——历代零陵城屡建屡迁，宋元明清零陵县城在原址附近叠建，今所见城垣多已不存，仅地名『零陵』承之至今。

### 潇湘合流口

城北、湘水与潇水合流之处——**唐人称此为『潇湘』，盛唐之前已成江南山水代名词**。湘水自南、自零陵县境西南绕城而北，潇水自西南来——两水于城北合流，水色一清一浊，须再行一段才相融。**屈原《离骚》《九歌》中『潇湘』之地望大致即此一带**——但屈骚之『潇湘』多指广义湘水流域，与唐时零陵之潇湘合流处非一一对应。盛唐杜甫《湘夫人》、李白《远别离》皆用『潇湘』之意象——所指即此水汽弥漫、烟波渺渺的一段湘水。

### 九疑山（南望）

永州城南、湘水上游、零陵县境之南——**九疑山（即苍梧山）相传为舜南巡崩葬之处**。《史记·五帝本纪》『（舜）践帝位三十九年，南巡狩，崩于苍梧之野，葬于江南九疑，是为零陵』；《山海经·海内南经》『苍梧之山，帝舜葬于阳，帝丹朱葬于阴』——『零陵』县名相传由此舜陵而来。**此为古传——舜南巡是否实至九疑、九疑是否即舜葬，史家多有疑义**（先秦舜之传说本身存疑，《史记》所记亦为汉人成说）；本 doc 作传说处理。盛唐人入永州，必询舜陵旧迹——后世明清两代在九疑山立舜帝陵庙，今为全国重点文物保护单位，但盛唐时具体祠祀形制不详。九疑山九峰相似，故名『九疑』——盛唐人或登此山，望九峰云雾，怀帝舜南巡之事。

### 小石潭 · 西山 · 钴鉧潭等（柳宗元《永州八记》八景）

**755 年这一刻，这些山水尚未入文学史**——它们要等到中唐**柳宗元元和四年（809）以后**贬永州司马、漫游零陵城西郊的山林溪谷时才被一一记下。小石潭、西山宴游、钴鉧潭、钴鉧潭西小丘、至小丘西小石潭、袁家渴、石渠、石涧、小石城山——这八处都在零陵城西、湘水西岸的丘陵溪谷之间。**今永州市零陵区西郊仍有『柳子庙』及柳宗元相关遗迹**——但盛唐 755 年这一刻，这些山水都还只是城西郊的寻常山溪。本 doc 列此条目作为后世文学地理的一处坐标；具体诗文与游踪详见〔历史长河·中晚唐〕段。

### 永州州学 / 龙兴寺

唐贞观间设州县学制后，永州州学在零陵县城内，例祀孔子。盛唐时永州州学规模较小——州地僻、士子少——贡举考生由此出者不多。**开元二十六年（738）玄宗诏天下州各置龙兴寺**——永州龙兴寺当时亦立，柳宗元贬永州后期居此寺数年，作《永州龙兴寺东丘记》《永州龙兴寺西轩记》等多篇——755 年时寺已立，但尚无柳氏所记的那一段文学因缘。

### 湘江渡 / 潇水渡

城东、城北各有湘水、潇水渡口——南北过往商船、贬官、戍卒皆在此渡。**长安至桂州、广州的官道，过潭州、衡州、永州一线**——永州正在这条岭北通衢的中段。盛唐过永州者多在此渡留宿一夜，再南行越岭入桂州。

## 历史长河（755 → 今）

**中晚唐**（755-907）

乾元元年（758）肃宗诏复郡为州，零陵郡再称永州。**永州在中唐之所以成名，全在柳宗元一人之笔**。

**柳宗元（773-819）**字子厚，河东人——贞元九年（793）进士，贞元二十一年（805）参与**王叔文永贞革新**；同年八月顺宗内禅，宪宗即位，永贞集团尽被贬——柳宗元贬**邵州刺史**（未到任），途中再贬**永州司马**——元和元年（806）抵永州，元和十年（815）召还。**柳宗元贬永州前后凡十年**。在永州十年间，柳氏起初居龙兴寺，后移居城西愚溪畔——漫游零陵城西的小石潭、西山、钴鉧潭、袁家渴、石渠、石涧、小石城山八处，作**《永州八记》**——这是中国山水游记的奠基之作。同期作《捕蛇者说》《永州龙兴寺东丘记》《与韩愈论史官书》《种树郭橐驼传》《天说》《非国语》等数十篇——半生著述皆成于永州。**柳宗元因此而成中唐古文八大家之一，永州也因此而入中国文学地图**。

柳宗元元和十年（815）召还长安，未几再贬柳州刺史——元和十四年（819）卒于柳州，年四十七。**柳氏在永州十年所识山水、所写文章，反成其身后最大遗产**——盛唐之永州本是寻常中州，至此则成『柳州』之外另一处『柳子』之地。

晚唐至五代间，永州仍属江南西道、楚（马氏）相继。永州虽不复有柳宗元那般大手笔，但中晚唐过永州的诗人甚多——刘禹锡贬朗州司马时（805-815）与柳宗元同贬南方，二人有诗信往还；元稹、白居易、贾岛、姚合等亦皆有过湘潇之诗。

**五代两宋**（907-1276）

五代时永州属南楚（马氏）。**南唐李煜尚未灭国前，南楚已亡**——后周显德元年（954）马氏内乱，南唐取楚地——但永州一带最终归于南汉、又入宋。**宋初平江南，永州归宋**——宋承唐制仍称永州，治零陵县。**宋人在永州，最重柳子之迹**——零陵城西的『柳子庙』在北宋时已立（一说始建于北宋至和年间/1054-1056，确否方志异说），自此沿至今。**南宋陆游、范成大、杨万里**等皆过永州——杨万里《诚斋集》多湘潇之诗，于柳子遗迹甚有所感。

**元明清**（1276-1911）

元置永州路——治零陵县。明洪武初降为永州府——治零陵县，沿至清。明清两代永州府辖零陵、东安、道州、宁远、永明、江华、新田七州县——府治零陵。**明清永州府『柳子文化』继续发酵**——柳子庙屡修，《永州八记》成科举与文人必读。**清代何绍基**（号子贞，1799-1873）为道州（今永州道县）人——晚清书法、诗文大家。**清末左宗棠（1812-1885）**虽为长沙人，但其早年游学多在湘潇一带，对柳子文有深感。

**民国至今**（1911-）

1913 年废府存县，永州府辖区改设零陵县、道县等——『永州』之名一度退出。1949 年后属湖南省零陵专区、零陵地区；**1995 年撤地设永州市**——『永州』二字时隔八十余年后复归正式行政名，治今永州市冷水滩区（非唐零陵县治旧址），零陵区为其市辖区，承唐零陵县治旧址。

**柳子庙**今仍在永州市零陵区柳子街——为全国重点文物保护单位，现存建筑为清光绪三年（1877）重修。**九疑山舜帝陵**亦经历代修复，今为全国重点文物保护单位。**小石潭、西山、钴鉧潭**等《永州八记》八景，地表多已变迁——小石潭今已不可确指其址，西山尚在零陵城西，钴鉧潭遗址多说不一。**今永州人最以柳宗元、何绍基、周敦颐（道州人，理学开山，1017-1073）三位文人为乡先贤**——盛唐 755 年的零陵尚未拥有这三位，但盛唐那一座湘水之南的小城，正是日后这三位文气所自出之地。

至于盛唐人入零陵所见的那一江潇湘、那一片九疑、那一座小城——一千二百年来，山未改名，水未改流，但城已数易其形。盛唐零陵的夯土城墙下，叠压着秦汉零陵郡的旧址；其上叠压着宋元明清永州府的城圈；再其上是今永州市零陵区的街市。**755 年那一刻的零陵——柳宗元尚未生、《永州八记》尚未作、柳子庙尚未立——只是江南西道辖境西南的一座中州小城**。

——柳宗元《捕蛇者说》（元和间作于永州司马任上；755 年时柳宗元尚未生，此文是后世为永州留下的最有分量一笔之一）

> 自西山道口径北，逾黄茅岭而下，有二道：其一西出，寻之无所得；其一少北而东，不过四十丈，土断而川分，有积石横当其垠……

——柳宗元《永州八记·小石城山记》（同上，作于元和间）

> （舜）践帝位三十九年，南巡狩，崩于苍梧之野，葬于江南九疑，是为零陵。

——《史记·五帝本纪》（汉司马迁；舜南巡九疑之传说，史家有疑义，本 doc 作古传处理）
