# 振州

## 一句话

岭南道振州，742-758 年间名"延德郡"——治宁远县（今海南三亚崖州区，**非今三亚市区**），盛唐海南岛五州之**南州**、大唐辖境最南端的州治；天宝七载（748）鉴真第五次东渡漂海至此，留振州约一年方北返。

## 形貌

振州城在海南岛南端、五指山南麓的一片沿海平地上。

州治在**宁远县**——治所在今海南三亚市**崖州区**（旧称"崖城"，盛唐振州城遗址即在此一带），**非今三亚市区**——三亚市区为 1954 年设三亚镇、1987 年升地级市后所成，盛唐振州城距今三亚市区西约四十公里。岛中黎母山（今五指山）一脉自南而北横亘，振州正在其南麓与南海之间一线狭长平地上。州境东西约百余里，南北仅二三十里——再南即南海。物候是热带的：冬不见霜、夏极热、雨季六月至十月台风极盛——比岛北之琼州更"南"，瘴疠之重在岭南诸州中亦数一数二。

## 大唐天宝十四载（755年）

振州在唐人眼里是**海南岛上五州中的南州**——岭南道之尽南、大唐辖境最南的一座州治。

**唐高祖武德五年（622）**析崖州南境置**振州**——治宁远县；这是振州建置之始（《旧唐书·卷四一·地理志四·振州》："武德五年置振州"）。**天宝元年（742）**玄宗诏天下州改郡，振州改郡——**《旧唐书》作"临振郡"，《新唐书》作"延德郡"，两书并存**；**乾元元年（758）**肃宗复改郡为州。**742 至 758 这十七年间，振州官号为"临振郡/延德郡"**（两源同存，本 doc frontmatter 与 alias 仍循《新唐书》延德之名以求一致）。755 年的振州属**岭南道**——岭南五府经略治广州，琼、崖、振、儋、万安五州皆属之；岭南节度使要乾元元年（758）才置。州治宁远县。**户口据《新唐书·地理志》载——下州，户 819、口 2,821**——比同岛之琼州、崖州更少；海南五州皆下州，其中振州因僻在岛南，户口最少、瘴疠最重。

**鉴真和尚第五次东渡**与振州关联极深：**天宝七载（748）冬**鉴真自扬州第五次启航赴日，海上遇风漂流——**漂至振州**。《唐大和上东征传》（淡海三船 779 撰）载，振州别驾冯崇债迎入官舍，**留振州约一年**——其间鉴真目疾加剧、弟子病死数人；约天宝八载（749）由振州北上，沿岛东海岸经万安、崖州，过琼州海峡至雷州，再循江北归扬州。**第六次东渡终成行**：天宝十二载（753）冬启航，天宝十三载（754）春抵奈良——755 年时鉴真已驻奈良一年余。**振州是鉴真在海南所留最久的一州**——盛唐振州城下海湾，是中日佛教史上一段绕不开的水路。

天宝十四载冬安禄山反范阳——消息沿驿道自长安南传，过商於、襄州、江陵、潭州、桂州、广州，再过海经琼州南下至振州，约**五十余日**。振州距河朔战场过远，朝廷未抽岛南兵北上——岛州官民这一冬听到的消息已是隔年的事。

## 历史长河

乾元元年（758）延德郡复称振州。中晚唐振州黎獠（俚人）反复，城治时盛时衰。**北宋开宝五年（972）**改振州为**崖州**——与岛北唐代崖州（治舍城）异地同名，史志两崖州不可混。元、明、清沿崖州之名；明洪武元年（1368）降崖州为崖县；清光绪三十一年（1905）复升崖州。**民国元年（1912）**废州存县，名"崖县"——1954 年崖县治迁三亚镇，**1984 年撤崖县改设县级三亚市**，**1987 年三亚升地级市**——"振州"之名自此入史册七百余年。今日盛唐振州城遗址在三亚市崖州区崖城镇一带，明清崖州故城墙残段尚存——但唐振州州署的夯土，考古至今未到那一层。

## 引文

《旧唐书·卷四一·地理志四·岭南道·振州》：

> 振州，下。隋珠崖郡之临振县地。武德五年置振州，治宁远县……天宝元年改为**临振郡**，乾元元年复为振州。领县四：宁远、延德、吉阳、临川。

《新唐书·卷四三上·地理志七上·岭南道·振州延德郡》：

> 振州延德郡，下。本临振郡，天宝元年更名。土贡……领县五：宁远、吉阳、延德、临川、**落屯**。

——下州、岛南。旧唐书四县（宁远、延德、吉阳、临川），新唐书五县（增"**落屯**"）；郡名旧志作"临振"、新志作"延德"——两书并存。盛唐振州一州，史志数笔即尽。
