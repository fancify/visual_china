# 唐州

## 一句话

河南道南端一上州，治比阳县（今河南驻马店市泌阳县一带，**非今唐河县**），**742-758 年间名"淮安郡"**——城在桐柏山西麓、唐河上游北岸；州名"唐"取自境内**唐河**，**与江苏淮安（唐为楚州）同名"淮安郡"而完全不同地**。（**至德二载(757)后割属山南东道**——755 这一刻仍隶河南道。）

## 大唐天宝十四载（755年）

唐州是河南道南端、邓州正东一片浅山平原间的旧地——治**比阳县**。沿革：**隋开皇九年（589）置显州**于此（治比阳），**大业三年（607）改淮安郡**；**唐武德四年（621）改唐州**——"唐州"之名自此立，取自境内**唐河**（唐河源出桐柏山西麓，南入汉水之白河，州治所在比阳即唐河上游北岸）；**天宝元年（742）再改淮安郡**，**乾元元年（758）再复唐州**。755 这一刻匾上写"淮安郡"，太守一员领郡事；本郡此时仍隶**河南道**——**至德二载（757）后方割属山南东道**，本 doc 主体严守 755 时态。《旧唐书·地理志》《新唐书·地理志》均记**上州**——天宝**户 42,643 / 口 约 183,000**，体量在河南道南端诸州中算一档实州，与邓、襄一线邻郡可呼应。郡境东倚**桐柏山**——**淮水正源出桐柏山主峰太白顶下**，是唐人地理常识里"淮渎"的来路；西南去邓州二百里许，南去申州（义阳郡）二百里，北去汝州（临汝郡）二百余里，东去蔡州（汝南郡）二百里，是邓、襄一线与蔡、汝一线之间的腰节。**州治考异**——唐淮安郡治比阳，今多考在今河南驻马店市泌阳县境（隋以来比阳县治所考异说有泌阳东、泌阳北两说），**绝非今河南南阳市唐河县**——"唐河县"作县名自明清以后方立，因唐河水所经而得名，与盛唐唐州州治比阳县相去数十里、非同地。

唐州在唐人心目中最古的一段标签是**桐柏山-淮渎**。**桐柏山在唐州东境**——主峰太白顶下为**淮水正源**，是上古"四渎"之一淮水的发源处；盛唐人来淮安郡多就此踏看淮源一段，唐州境内"东倚桐柏、西循唐河"是其形势。755 这一冬安禄山自范阳起兵，淮安郡不在叛军正面主线，但循邓、唐、申、蔡一线江汉转输的腰节自此过，至德元载（756）以后江淮防线收紧，本郡为唐军江汉转输的一处后方腰节。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改淮安郡为唐州。**大历十二年（777）山南东道节度使梁崇义据襄阳叛唐**，唐州一度受迫于襄阳兵锋；**建中四年（783）淮西节度使李希烈据蔡州叛唐**，唐州再陷淮西之手——李希烈陷唐、邓、襄诸州，"唐州"是淮西叛军西扩的腰节之一；贞元二年（786）李希烈卒，唐州归唐。**宋元明清**　北宋仍置唐州，属京西南路；金属南京路；元省入南阳府裕州。明清"唐州"建制不复——**唐州州治比阳县**于宋时改名**泌阳县**（取唐河支流泌水为名），明清沿之；**今河南南阳市唐河县**则为明清后另置之县，因唐河水所经得名，**与盛唐唐州治所比阳县（今泌阳一带）不可径直划等号**。**民国至今**　今河南驻马店市泌阳县大体承唐唐州州治比阳县之地；今河南南阳市唐河县则与唐唐州本治非同地——盛唐唐州的"唐"字今分摊在两处现代县名上（泌阳承治所、唐河承州名），是**地名分流**的一段典型。

## 同名 disambig

**唐州淮安郡**（755 时隶河南道，至德后割山南东道，治比阳，今河南泌阳一带）**与江苏淮安完全不同地**——盛唐时今江苏淮安市一带为**淮南道楚州**，治山阳县（今江苏淮安市楚州区），**非"淮安郡"**；"淮安"作府名归今江苏淮安一线，自南宋宝庆三年（1227）改楚州为淮安军始、元至元二十年（1283）立淮安路、明清淮安府定型——**与唐"淮安郡"（742-758 唐州官称）非同地、非同源**，唐州本郡之"淮安"源出"淮水之安"（淮源所在）、楚州之"淮安"源出宋元淮南漕运重镇之安——两层"淮安"在唐世并无重叠。

## 引文

> 唐州，淮安郡，上。本显州，隋大业三年改为淮安郡，武德四年讨平王世充，改为显州……贞观九年，改为唐州。天宝元年，改为淮安郡。乾元元年，复为唐州。

——《旧唐书·卷三九·地理志二·山南东道·唐州》。唐州沿革"显州→淮安郡→显州→唐州→淮安郡→唐州"一段最直接的史源（沿革数易，本 doc 以《旧唐书·地理志》为准）。
