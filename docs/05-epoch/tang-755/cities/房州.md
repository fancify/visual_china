# 房州

## 一句话

山南东道西陲一州，治房陵县（今湖北十堰房县）——汉江上游、群山环抱，盛唐"中州流放罪臣"之地，**中宗李显废为庐陵王后在此居十四年**。

## 形貌

房州在秦岭余脉与大巴山之间的一片山地里。州治房陵县在汉江南岸一道东西向的小盆地——城北过堵河可达汉江，再北便是均州（今湖北丹江口）境的武当山；城南是神农架主脉，林深谷险，唐人记为"山高林密、虎豹出没"——汉江上游一段最难行的山区便在此。城周不大、户口稀——盆地里勉强容下几片粟麦稻畦，城外便是层层青山，云气常驻。**这一州地势的根本特点是"封闭"——四面皆山，自长安南来须翻秦岭至商州、再越武当山入房州境，自江陵北上须沿汉江西行过襄州、均州方至——一旦入此境，再出便难**。

物候上，房州属汉江上游过渡带——比襄州、江陵冷，比梁州、利州潮；春来早、秋雨多，夏不酷热，冬偶有雪。林木以栎、栗、漆、桐、楠为主，山间多药草——神农尝百草的旧传便落在城南神农架一带，唐时已有"神农"祠祀。这一片山中产黄连、当归、贝母、漆、茶——州贡以漆、茶、药材见名。**山深林密之利，恰是流放地之利：罪臣入此，无路可逃、无信可通**。

## 大唐天宝十四载（755年）

房州沿革久远。**北周保定中（约 561-565）置房州**——治房陵县，因境内房山得名。隋大业三年（607）罢州改置**房陵郡**——这是"房陵郡"作为正式郡名第一次入史。**唐武德元年（618）复置房州**，仍治房陵县；**天宝元年（742）**玄宗诏改州为郡，房州改称**房陵郡**——是为 755 年当年的官称；**乾元元年（758）**肃宗复改郡为州，房陵郡复称房州。盛唐房州属**山南东道**——山南东道治襄州（今襄阳），辖襄、邓、唐、随、郢、复、安、申、光、黄、蕲、沔、房、均、忠、夔、归、峡、硖等州，房州在其辖境西陲。

户口上，《通典》《元和郡县图志》载房州天宝间约 **一万四千户**——按唐州等第：**唐上州四万户、中州二万户、下州不足二万户**，房州约 1.4 万户**属下州**。郡内丁口估计五六万，撑起一郡行政而已。城内官署即郡衙、房陵县治、馆驿数所——一套**下州小郡**的简单班子。

这一州在盛唐最有名的一段身份，不在户口、不在物产、不在驿务——而在它是**唐前期宗室叙事中"宗室与大臣流放地"之首**。山高林深、四面皆山——是天然的软禁牢笼。**嗣圣元年（684）二月**，武则天废**唐中宗李显**为**庐陵王**，先迁均州、四月再迁房州——李显在房州前后居十四年（684 二月废、698 召还，凡十四年），是其复位前唯一一处长居之地。《旧唐书·中宗本纪》、《资治通鉴》卷二〇三皆详记此事。庐陵王居房州期间，闻使者将至辄惶恐欲自杀——韦氏（后之韦皇后）相劝"祸福无常，宁失一死"——夫妇相依度此十数年。圣历元年（698）武则天召还东都，立为皇太子；神龙元年（705）张柬之等政变，中宗复位——此前的十四年废居，便落在房州城里。

755 年的房州，中宗早已去世（710 卒）四十余年，但城里的旧人尚有口耳相传——盛唐人入房州，必访"庐陵王故宅"（在房陵县城西，今已不可考其确址），是这一州最有分量的旧迹。除中宗外，唐前期流房州者尚多：**罗艺（武德间反叛伏诛后家属流房州）**、**裴炎家族（武后朝坐裴炎案者）**、**李重福（韦氏党羽案）**等——唐人凡言"流房州"，便知其分量。

天宝十四载冬安禄山反范阳——消息沿驿道西传，房州偏处山中，本不在乱事主路上；但作为山南东道辖境西陲、汉江上游门户，**这一州的山林便是逃乱者南奔的一处藏身处**。次年玄宗西奔入蜀（参 [益州·玄宗入蜀路线](./益州.md#玄宗入蜀路线-756-年六月至七月)）——一行循金牛道入蜀，未经房州境；但乱中士人自洛阳、长安南奔者，多有循汉江西上、入房州境避乱者。**山高林密在乱世，便由"流放牢笼"翻转为"避乱桃源"**——这是房州在 755 之后最被叙写的一面。

## 历史长河

**中晚唐**　乾元元年（758）复州名为房州。中晚唐房州仍多置宗室流放案——德宗朝、宪宗朝皆有宗室、官员流房州的记录。元和、长庆间藩镇林立，山南东道节度使屡换其人——房州处其辖境西陲，多任由州刺史自治。唐末黄巢之乱（875-884）波及山南——房州山深，又一度成避乱所。

**五代两宋**　五代时房州属荆南高氏。北宋初仍置房州——北宋的房州继续是流放地：**宋太祖、太宗朝皆有宗室、降王流房州的记录**——最著名一段是**南唐后主李煜之弟李从善**、**吴越王钱俶之子钱惟濬**等。北宋的房州还接收过一位极有分量的旧客——**宋太宗第八子赵元俨**（即"八贤王"原型之一）虽未流此，但宋皇室流房州者远多于唐。南宋时房州属京西南路；元属襄阳路。

**明清民国**　明洪武十年（1377）降房州为**房县**——"房州"二字至此罢，盛唐州名让位于一个单字县名。明清两代房县属襄阳府、郧阳府。**明成化年间（约 1465-1487）荆襄流民大起义**——百万流民入鄂西北山区，房县为流民窝点之一，明廷于此设郧阳抚治、置郧阳府以镇之（1476）。清承明制。1913 年废府存县，房县属湖北襄阳道；1949 年后属湖北省郧阳专区、襄阳专区、十堰市。**今房县属湖北省十堰市，县治承明清房县城**——盛唐房州治房陵县城旧址，便叠在今十堰市房县城关镇之下。

城南六十里入神农架——1970 年自房、兴山、巴东三县划地置神农架林区，是中国唯一以"林区"为名的省级直辖行政单位。**唐人传说中"神农尝百草"的山，至今仍是一片国家级自然保护区**——金丝猴、野生珙桐、铁坚杉、华南虎（最后一次目击 1980 年代）皆在此境。盛唐房州人未识金丝猴之名，但其先祖采的黄连、当归、贝母——一千二百年来未改其山。
