# 安州

## 一句话

淮南道安州，742-758 年间名"安陆郡"——治安陆县（今湖北安陆市），涢水绕城、白兆山在西；李白开元十五年（727）出蜀后入赘前宰相许圉师孙女家，**"酒隐安陆，蹉跎十年"**——是盛唐诗人在长江以北、汉水以东最重的一段寓所。

## 形貌

安州城在涢水南岸一片低平台地上——涢水自北而南绕城东东南流，再南入长江支流汉水水系。城西约二十里有**白兆山**，山间有桃花岩——李白寓安陆十年间常往来此山。州治在安陆县（今湖北孝感市安陆市旧城范围），城周不大、户口中等偏下。**《旧唐书·地理志》载安州天宝领县六（安陆、云梦、应山、应城、孝昌、吉阳），户二万二千二百二十一、口一十七万一千二百二**——按户口当下州门槛之上、中州之下，史志或作"中"。物候是江汉北缘的：四月麦黄、五月梅雨、七八月江风南来——比关中温润，比江南干凉。

## 大唐天宝十四载（755年）

安州在唐人眼里是一座**李白"蹉跎十年"**的旧寓之州。

行政沿革——隋开皇九年（589）置安州，大业三年（607）改安陆郡；**唐武德四年（621）复安州**——治安陆县，属淮南道；**天宝元年（742）**玄宗诏天下州改郡，安州改称**安陆郡**——这是 755 年当年的官称；**乾元元年（758）**复改郡为州，"安陆郡"之名前后用了十六年。**天宝十四载安陆郡太守在任者史志阙考**（注：唐州一级长官，**742 改州为郡后称"太守"，758 复郡为州后称"刺史"**——755 当年只称"安陆郡太守"，不称"安州刺史"）。

但安州在盛唐文学史上的真正分量，是**李白**。**开元十五年（727）**李白二十七岁出蜀，自江陵东下、过洞庭、北上至安陆，**入赘前宰相许圉师孙女家**——居安陆十年，自称"**酒隐安陆，蹉跎十年**"（《代寿山答孟少府移文书》）。其间开元十八年（730）首入长安干谒未成，复返安陆；开元二十二年（734）前后再入长安亦未成；**开元二十四年（736）前后**李白携家自安陆移居兖州瑕丘（参 [兖州](./兖州.md)），"寓家东鲁"——但安陆十年是其前期诗作最集中的一段。城西白兆山桃花岩、城东涢水岸边——李白《安陆白兆山桃花岩寄刘侍御绾》"桃花岩上春风暖"诸句，写的便是这一山一水。**注：李白与汪伦"桃花潭水深千尺"事在皖南宣州泾县桃花潭**（参 [宣州](./宣州.md) 一带），**与安陆白兆山桃花岩并非一处**——盛唐时即不混。

755 当年李白五十五岁，在江南宣州、当涂一带——**安州的"李白寓所"已是二十年前的旧事**；但安陆城内许氏旧族尚在、白兆山桃花岩仍是过往诗人指点的地方。

## 历史长河

中晚唐安州仍为淮南道下州，**广德元年（763）以后淮西、淮南诸藩镇渐成**，安州属之。**唐宪宗元和年间**裴度平淮西吴元济、李愬雪夜入蔡州——**安州地近淮西**，是元和讨蔡时淮南道的转运州之一。**五代**安州数易其主，**北宋开宝元年（968）**升安州为德安军、宣和元年（1119）升德安府；**南宋**德安府为江北抗金前线之一，与对岸鄂州一线呼应。元置德安府属荆湖北道，明清因之。

**民国二年（1913）**废德安府，安陆县直属湖北省；1949 年后属孝感专区、孝感市；**1987 年安陆撤县设市**（县级），属孝感市代管。**白兆山桃花岩**今为安陆李白文化纪念地（白兆山李白文化风景区），盛唐李白寓安陆十年的那一片山水——白兆山主峰、涢水北岸、安陆旧城——大体在今安陆市城区与城西二十里白兆山一带，**"安州""安陆郡"两个唐时官名，在地方志的沿革表里偶见**。

## 引文

李白《代寿山答孟少府移文书》（开元间作于安陆）：

> 酒隐安陆，蹉跎十年。

——李白自述安陆十年之语，是这一段寓所最直白的一笔。其《安陆白兆山桃花岩寄刘侍御绾》一首，诗题中的"白兆山桃花岩"即安州城西二十里之白兆山——是李白十年寓安陆间常去的隐居处。**注："桃花潭水深千尺，不及汪伦送我情"另一名句出自《赠汪伦》，事在宣州泾县桃花潭（参 [宣州](./宣州.md)），与安陆白兆山桃花岩异地异事——盛唐时即分清，不可混**。
