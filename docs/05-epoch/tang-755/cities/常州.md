# 常州

## 一句话

江南东道晋陵郡治，太湖北岸的一座旧城——江南运河自润州南下、过此再东南通苏杭，是润州与苏州之间的中段大州。

## 形貌

常州在太湖北、长江南——一片低平水乡里的州城。

城周不及苏、杭之大，约十数里——夯土城墙带砖包，门数处。城外四望尽是水网：北一带通向长江南岸的孟渎、利港，东南一带连入太湖与江南运河，西邻丹阳、东接无锡，皆水路一日可达。**江南运河自润州京口南下，过丹阳、入常州境，再东南向苏州**——常州城正卡在这条运河中段最繁的一线上。城南运河码头泊船终岁不绝，南来北往的漕米、丝绢、盐铁、瓷器皆在此换船、交税、宿驿。

子城（晋陵郡州衙）居全城正中——太守治、长史、司马、诸曹参军公署、州学、武库、州狱并列其内。罗城外护城河绕城一周，与江南运河相通——这是江南水城的本相。城内主街数条，傍水而行——河上小桥架接坊与坊之间，与苏州的水巷格局相承。

物候是太湖以北、长江以南的：**春深多雨、梅季湿气重、秋日明净、冬有薄霜**。郊外四时种作：春课田、夏催赋、秋收漕、冬考课——一州官吏一年的常事，与江南诸大州无别。州境西北通长江——城北四十里许便是江口的孟渎口，是常州接长江漕运的一个支岔。州境东南抵太湖——湖中渔舟、湖畔桑园，是常州民生的另一面。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

常州在唐人眼里是一座旧地——但作为州一级建制不算早。

最早的根脚是春秋吴地——属吴国延陵邑。吴公子**季札**所封即在此（一说在今丹阳、一说在今常州，史家两说并存）——季札"挂剑"的故事便托于这一带。秦统一后置**毗陵县**，属会稽郡。汉承秦制——东汉永建四年（129）析会稽郡置吴郡，毗陵县入吴郡。三国孙吴改毗陵为**毗陵典农校尉**——治屯田事；西晋太康二年（281）置**毗陵郡**，治丹徒，"毗陵郡"之名自此起。东晋永嘉南渡之后，毗陵郡屡有侨置流民。**南朝梁太清三年（549）**侯景之乱中毗陵郡治屡有迁徙；**陈永定元年（557）**改毗陵郡为**晋陵郡**——治晋陵县（在今常州市区），"晋陵"作为郡名自此起。

**隋开皇九年（589）平陈**——废晋陵郡，于此置**常州**，州名取常熟县（一说取常山以北郡望，史家两说并存）。"常州"二字作为正式州名自此始。**大业三年（607）**炀帝改州为郡——常州改作**毗陵郡**，回用旧名。**唐武德三年（620）**李渊定江南，复改毗陵郡为**常州**。**贞观元年（627）**属江南道；开元二十一年（733）江南道分东西，**常州属江南东道**——下辖晋陵、武进、义兴、无锡、江阴五县（依《旧唐书·地理志》）。**天宝元年（742）**玄宗诏天下州一律改称郡——常州改作**晋陵郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州，晋陵郡复称常州——晋陵郡之名前后用了十六年。

唐人若在 755 年驻足回望：自春秋吴公子季札所封算起，已逾一千二百余年；自西晋置毗陵郡（281）算起，已逾四百七十年；自陈改晋陵郡（557）算起，已近二百年。郡名换过三轮——毗陵、晋陵、常州——皆在这片地上，城圈虽迁，骨架未离太湖之北、运河之中。

### 此刻

天宝十四载（755）的常州，行政上是**江南东道晋陵郡治**——领晋陵、武进、义兴、无锡、江阴**五县**。**《旧唐书·卷四〇·地理志三·江南道·常州》载：**"常州，**上**。隋毗陵郡。武德三年，平李子通，置常州，领晋陵、义兴、无锡三县。其年，又分置武进县……天宝领县五，户**十万二千六百三十一**，口**六十九万六百七十三**。"——按唐州县等第制度，常州为**上州**，户口在江南东道诸州中**与润州、苏州、湖州并列腹心一档**：略少于苏州（开元户七万六千、天宝约 7 万 6 千户），与润州（天宝户十万二千余）几乎相当，亦近于湖州。

军政上，常州不是边镇——无节度使治所、无大军屯驻。一州兵不过千余，郡太守多由文臣出任（按 742 改州为郡之制，755 时官称晋陵郡太守；758 复郡为州后又改称常州刺史）。盛唐晋陵郡太守的世系不如苏州、扬州那样名臣赫赫，多为中级地方官——史志阙考者居多。**张籍**（约 766-约 830）任水部员外郎、国子司业，**晚年曾出为江南诸州**——一说曾历常州刺史，史无明文，唐人笔记或追记；张籍生于代宗大历元年前后，距 755 已十年——755 这一刻他尚未出生。**韦应物**（约 737-792）755 年时尚是十八九岁的少年——他后期任苏州刺史是德宗贞元年间（约 788-791）的事，并非常州。**白居易、刘禹锡**任苏杭刺史皆是穆宗、敬宗朝事——亦非常州。**故 755 年的常州，是一座有等而无名诗人的上州**——它的诗史地位远不如苏、杭、扬。

漕运是常州一年里最重的一桩。**江南运河自润州京口南下、过丹阳入常州境，再东南至无锡、至苏州**——这一段是江南运河中段最繁的水路。常州城南运河码头是这一线上的中间停泊点：南来漕船在此过夜、换浆手、交州关税；北上漕船在此候风、补给。**到唐玄宗朝，江南运河已运作一百四十余年**——隋大业六年（610）炀帝命凿江南河自京口至余杭，**至 755 年已一百四十五年**。盛唐江淮漕米年二百万石，由江南运河北运至润州京口，过江入瓜洲，再循江淮运河、汴渠至洛阳——常州是这条漕路上"润州 → 苏州"之间不可或缺的一环。

对天宝十四载的常州人来说，长安在三千余里之外——隔着江、隔着淮、隔着大半个中原。安禄山十一月初九在范阳起兵的消息，沿驿道经汴州、宋州、徐州、扬州，南渡过江至润州、过常州至苏州——前后约一月。这一年的常州城里，市坊如常——运河船工、州中胥吏、码头脚力、城中商户——皆未察觉这场乱将怎样改写江南。再过半年，朝廷将循江南运河南调江淮租赋——常州作为运河中段必经之州，**自此从"江南腹心州"渐转向"漕运第一线州"**。中晚唐至北宋常州户口与漕额逐增——其底子，正是 755 这一年定下的。

## 子条目

### 子城（晋陵郡州衙）

全城正中——晋陵郡太守治、晋陵县治 / 武进县治之外的诸曹官署皆在此。自陈永定元年（557）置晋陵郡以来，州衙所在大体相承——隋唐之际或有微移。子城外有夹城与护城河——与江南运河相通。**今址叠在常州市旧城中部一带**——常州市考古所近代有零星发掘，确认夯土残迹，但盛唐子城整体格局尚不可辨。

### 江南运河（常州段）

隋大业六年（610）凿——自京口至余杭八百余里。**常州段**是江南运河中段最繁的一段：自西北丹阳入州境，经常州城南，东南至无锡——一州境内运河约百余里。城南码头泊船终岁不绝——南来北往的漕船、商船、官旅皆在此换班、交税、宿驿。**这一段唐宋两代屡修——河身今犹存**，是常州城南的主要水道；今常州古运河（南运河）即此一线。

### 孟渎口（运河接长江一支）

常州城北约四十里——江南运河接长江南岸的支岔。**孟渎自西晋陈敏所凿（一说更早）**——晚唐五代屡疏。这是常州对江北的水道接口——州境漕米支岔自此出江，与扬州、楚州一线长江北岸的诸渎相对。盛唐时孟渎尚不及后来扬州一段的大运河北口那样繁——但已是常州北面的重要水口。

### 太湖（常州东南一隅）

常州东南—— 一州之地有约十数里临太湖。太湖唐时多葑草、鱼汛—— 湖畔多渔家、桑园。州中渔税、湖鲜，是常州民生一项。**太湖正脉以南属苏州、湖州、长州**——常州只占太湖北一隅，但城与湖之间不过数十里，舟楫可达。

### 阳湖

常州东郊一小湖——与太湖相通。盛唐时阳湖湖面比今日略宽，是常州城东郊出游的一处。**"常州学派"中的"阳湖派"**——清乾嘉间常州学人所立之名号，指此湖，755 年此名未出。

### 季子祠（季札庙）

季子祠相传立于春秋——祭吴公子季札。**季札之墓**一说在今江苏丹阳延陵镇九里村（属唐润州延陵县境），一说在常州城外武进、晋陵交界——两说唐时已并行。**唐贞观以来季子祠列入州祭**——州刺史岁时致祭，是常州一郡最早的纪念。**"挂剑台"**之名亦在唐时已传——但具体在何处，唐人记之亦不一。

### 武进县

常州属县——治在常州城近郊（与晋陵县同城分治，**晋陵县治偏东、武进县治偏西**，唐时两县并设于常州州城内，是少见的"一城两县"格局）。**武进**之名自西晋太康二年（281）已立——取"以武而进"之意。隋开皇九年（589）省入晋陵县；**唐武德三年（620）复置武进县**——与晋陵县同治常州。这种一城两县的格局沿至宋元明清，民国元年（1912）两县合并为武进县——今天的常州市区，便叠在唐时的晋陵 + 武进二县同城之上。

### 义兴县

常州属县——治义兴城（今江苏宜兴市）。**春秋时已有阳羡邑**——秦置阳羡县。**晋永嘉之乱**后周玘以乡兵讨陈敏，**晋元帝以其功置义兴郡**（**永嘉四年 / 310**）——治阳羡。隋开皇九年（589）废义兴郡，省入晋陵县；**唐武德三年（620）复析晋陵置义兴县**——属常州。义兴县境多陶土，**陶瓷器自汉以来已著**——盛唐至中晚唐间义兴所产青瓷已运销江南诸州；后世"宜兴紫砂"之名要再过六七百年到明中后期才起来——但**义兴陶**作为一种地方器物，盛唐时已可见。

### 无锡县

常州属县——治无锡县（今江苏无锡市）。**春秋时为吴地**，秦置无锡县——属会稽郡。汉因之。隋开皇九年（589）废无锡入晋陵县；**唐武德三年（620）复析置无锡县**——属常州。无锡县境跨太湖北岸——**惠山**（即"九龙山"）在城西，山下有**惠山寺**（南朝梁大同三年 / 537 立）——是太湖北岸佛寺之一。盛唐至中晚唐间，无锡作为常州属县无大事可记——它的全盛要等到北宋以后，至明清成江南漕粮第一县。**755 年的无锡仍只是常州一属县**——城周不及常州州城之半。

### 江阴县

常州属县——治江阴城（今江苏江阴市）。**地处长江南岸**——是常州境内唯一直接临长江的县，与长江北岸的扬州境隔江相望。**南朝梁绍泰元年（555）**置江阴郡，治江阴城；**隋开皇九年（589）**废江阴郡为江阴县，属常州。江阴县境扼长江下游南岸——盛唐时设有渡口与江口烽堠，是常州接长江漕运、备江盗的前沿。

## 历史长河（755 → 今）

**中晚唐**　乾元元年（758）肃宗复改晋陵郡为常州——"常州"之名自此通行至清。**贞元间**（约 785-805）常州始入诗史——韦应物、刘长卿、戴叔伦诸人过常州皆有诗。中唐起常州户口与漕额逐增——**元和八年（813）前后**李吉甫《元和郡县图志》所记常州户已逾十一万，比开元、天宝时仍略增。**张籍**（约 766-约 830）晚年是否任常州刺史，唐人笔记两说并存——史志阙考。

**五代至宋**　唐末黄巢之乱（880）兵未深入江南腹地，常州未遭兵火。乾宁三年（896）杨行密据淮南建吴国——常州一度归之；**天复二年（902）钱镠据吴越国**——常州归吴越；至太平兴国三年（978）钱俶纳土归宋——吴越七十六年间常州为吴越国西府之一，钱氏保境安民，盛唐至中晚唐的城市底子由是保全过五代。**北宋承之置常州**——属两浙路。**南宋**绍兴间常州随江南半壁腹心，富庶居天下前列——**咸淳毗陵志**（南宋 史能之 1268 修）成稿，是常州现存最早的方志。

**明清**　元改常州为常州路，明洪武元年（1368）改路为**常州府**——府治承元路旧城，下辖武进、无锡、宜兴、江阴、靖江五县。**明清两代常州是江南文教重地**——府学、私塾、书坊林立。**清乾嘉间常州学派**（庄存与、刘逢禄等）以**今文经学**与**阳湖派古文**著称——是清中后期江南学术的一脉。常州地方书院、藏书楼自宋至清屡兴——**与苏、杭、扬并列江南文教四府**，但规模略次于苏、杭。

**民国至今**　1912 年废府——常州析为武进、无锡、宜兴、江阴、靖江诸县。1949 年后置常州市，今属江苏省。武进、无锡早已各自独立成市——常州市辖天宁、钟楼、新北、武进、金坛五区。**常州古运河**（即唐江南运河常州段）今仍存——南运河、北运河两线穿城而过，是常州城内主要水道。**子城遗址**今叠在常州市旧城中部一带——考古零星发掘，整体格局尚不可辨。

至于盛唐那一座晋陵郡——子城衙署、城南运河码头、城外的孟渎口、季子祠——其址或叠或失。可走的，是常州古运河那一段、武进与晋陵旧县共治的旧城轮廓、季子墓（延陵 / 武进二说并存）——这几处的位置、走向、用法，**与天宝十四载那一年大体相承**。
