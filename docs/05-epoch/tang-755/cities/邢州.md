# 邢州

## 一句话

河北道邢州，742-757 年间名"钜鹿郡"——治龙冈县；古战场"巨鹿"在境内（前 207 项羽破釜沉舟之地），唐时邢窑白瓷已盛——所谓"南青北白"的北白便指此一窑系。

## 形貌

邢州城在太行山东麓的洪积扇与华北大平原相接之处——州治龙冈县（今河北邢台市旧城区）。城西仰太行山脊，过山即潞、泽诸州；城东一片大平原直铺至沧、瀛海岸，平整无碍。**七里河（古澧水支流）自城南绕东**——是邢州城的内河水脉；境内沙、洺诸水皆自太行东流，汇入永济渠以东的低地。城北去赵州（赵郡）一百三十里，南下相州（邺郡）二百里，东接冀州，西出井陉道可入并州（太原）——正坐在河北道中段南北纵轴上。

物候属华北腹地——春旱、夏热、秋燥、冬寒；城外种粟、种麦、种桑，太行东麓的坡地上盛产瓷土与煤——这两样恰是**邢窑**的本钱。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

邢州境是**叠了三层旧事的一片华北老地**。

最古一段是商代。**《史记·殷本纪》载祖乙迁于邢**——旧注以"邢"即此境一带（古地望有殷邢、卫邢两说，前者地望即今邢台一线，后者在豫北温县一带；史家多以邢台一线为殷祖乙所迁之邢）。盛唐人来邢州，凭吊"商祖乙迁邢"是寻常一笔，但所凭依的只是《史记》一条与古史传说，确址唐时已不可全寻。周代封姬氏为邢国，春秋时为狄所灭、为卫所并；战国属赵，秦置信都县——汉因之。

第二段是**巨鹿之战**——这是邢州境内最重的一段大旧事。**秦二世三年（前 207）**项羽率楚军救赵——赵王歇与张耳为秦将章邯、王离围于巨鹿（今河北平乡西南，唐邢州东南境）。**项羽渡漳水、破釜沉舟、烧庐舍，持三日粮——示士卒必死、无还心**；九战九胜，大破王离军，俘王离，斩苏角，章邯退保棘原。诸侯将入辕门皆膝行——项羽自此为诸侯上将军。这是秦亡前夜的决战、楚汉相争的开端——盛唐人入邢州，最远一回望便是这一段（《史记·项羽本纪》）。**按：唐巨鹿县（即古战场所在）属邢州——但具体战场遗址唐时已不能确指**，多依《史记》《汉书》文献凭吊。

第三段是**邢窑**。隋唐间太行东麓的邢州境内（今邢台市内丘县、临城县一带）兴起一处大窑系——以白瓷见长，与南方越窑青瓷并称"南青北白"。**陆羽《茶经》载"邢瓷类银、邢瓷类雪"**（陆羽《茶经》成书于建中前后，约 760-780 间，距 755 年时代略晚一二十年——但《茶经》所述邢窑之盛，正是回看盛唐之事）；盛唐时邢窑白瓷已是河北道、两京、扬州、广州皆有销路的名品。**按：邢窑具体唐代窑址 1980 年代以后方相继发掘——内丘县、临城县境内多处窑址，但何处为盛唐 755 时正盛、何处稍后兴起，考古序列仍在细化中**。盛唐 755 时邢州境内白瓷产销已成体系——这是文献与今考古都可确证的；但每处具体窑址年代须 hedge。

行政沿革：**隋开皇十六年（596）析洺州置邢州**——"邢州"二字自此立；隋大业三年（607）改州为郡——邢州复称**钜鹿郡**。**唐武德元年（618）复置邢州**，治龙冈县。**天宝元年（742）**玄宗诏天下州改郡——邢州再改**钜鹿郡**；**乾元元年（758）**肃宗复改郡为州——邢州之名再立，沿用至宋。

### 此刻

天宝十四载（755）——官称"钜鹿郡"，民间口语仍呼"邢州"。河北道中段。本郡领县**八**：龙冈（州治）、钜鹿、内丘、平乡、南和、沙河、任、尧山——州境东西约三百里、南北约二百里。户口据**《旧唐书·地理志》载邢州 70189 户 / 382798 口——上州**，在河北道诸州中位居前列，仅次于幽、魏、相、定、恒、冀诸大州。城内主事：邢州刺史兼钜鹿郡太守、州司马、州司户参军——一套常规上州官署。**755 时邢州并无节度使建制**——河北道采访使巡历、本州刺史一级直管。

这一年城里的实务：太行东麓诸县常平仓的报表、邢窑白瓷的征贡（盛唐邢窑白瓷岁有贡入两京之例，具体岁贡数志阙）、沿太行东麓南北驿道的人马调发。**邢州东南去巨鹿县故战场约百余里**——唐时已立巨鹿县，但盛唐人入巨鹿县访"项羽破釜沉舟"故地，多依《史记》凭吊，确址不可全寻。

十一月初九安禄山自范阳起兵——消息沿太行东麓南北驿道急传，约半月抵邢州。**邢州正在叛军南下的主线上**：自范阳南下河北诸州一路经赵、邢、洺、相、卫——叛军主力十一月底至十二月初过邢州南下。**至德元载（756）邢州陷于叛军**；至德二载（757）唐军北复——河北诸州几度易手，邢州在这一线上反复焚毁。但 755 年冬这一刻，邢州城里"钜鹿郡"四字尚悬于谯楼——半个月后，这座城将进入八年战火。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改钜鹿郡为邢州。河朔三镇（成德、魏博、卢龙）割据中段——邢州属**昭义节度**（一度属成德、魏博），是河朔与河东之间的转轴。**会昌三年（843）昭义节度使刘从谏死，子刘稹拒命——武宗诏伐**：邢州为讨刘稹的主战场之一。中晚唐邢窑白瓷仍盛——但产销已不复盛唐"贡入两京"的体面。

**宋元明清**　北宋改邢州为**信德府**（宋徽宗宣和元年 / 1119 升府），治龙冈县；金属河北西路邢州；元至元二年（1265）改邢州为**顺德府**——"顺德"二字自此立，沿用至明清。明清顺德府治邢台县（元至元二年改龙冈县为邢台县——"邢台"二字自此立）。明清间邢窑早衰——白瓷产业自北宋以后渐转移至定窑（曲阳）、磁州窑（磁县）一线；邢窑旧址被层层埋没，明清方志已不能确指。**今内丘县、临城县境内邢窑遗址 1980 年代以后陆续发掘**——盛唐 755 时正盛的窑系，地表上湮没千年方再被认出。

**民国至今**　1913 年废顺德府，复置邢台县；1945 年抗战胜利后设邢台市；今**河北省邢台市**——市区即唐邢州龙冈县（元改邢台县）旧址范围。**巨鹿之战古战场**在今邢台市东南平乡县、巨鹿县一带——具体战场地望仍有学术分歧，多依《史记》"巨鹿"二字与漳水流向推定。**邢窑遗址**在今邢台市内丘县、临城县境内——主要窑址已立为全国重点文物保护单位。盛唐邢州人若得归来，城南的太行轮廓、城东的平原走势、城西的井陉道方向皆未大变——但他口中"钜鹿郡"四字、邢窑炉火、巨鹿古战场的方位记忆，皆须从《元和郡县图志》《史记》一卷一卷找回。

## 引文

> 项羽乃悉引兵渡河，皆沉船，破釜甑，烧庐舍，持三日粮，以示士卒必死，无一还心。于是至则围王离，与秦军遇，九战，绝其甬道，大破之，杀苏角，虏王离。

——《史记·卷七·项羽本纪》。**前 207 巨鹿之战**——破釜沉舟典出此条；战场即唐邢州境东南巨鹿县一带。盛唐人入邢州访古战场所凭便是此条。

> 邢瓷类银，邢瓷类雪。

——陆羽《茶经》（约 760-780 间）。盛唐邢窑白瓷之盛，《茶经》是最直接的同时代回望——成书略晚于 755，但所述正是盛唐时的窑业格局。
