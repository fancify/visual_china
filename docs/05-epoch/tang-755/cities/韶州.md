# 韶州

## 一句话

岭南道一处下州，治曲江县（今广东韶关）——大庾岭路南口、曹溪禅宗祖庭、开元宰相张九龄的故里，三事系于一州。

## 形貌

韶州城卧在**北江**上游、武水与浈水交汇处——这是它的位置根本。两水自北、自东北南来，于曲江城下合流为北江——北江再南去广州入珠江口。城北约百余里便是**大庾岭**（即梅岭），翻过此岭即入江南西道虔州境——这是岭南通中原最便捷的一条陆路。城南、城东低山连绵，岭南北段丹霞地貌起伏可见。

物候是岭南北段的：春夏多雨潮湿，夏热稍逊于广州，冬偶有薄霜——比真正的岭南腹地凉而旱。州境内以两水冲积平地与低丘为主——可作稻田、产茶，山中有竹有杉。韶州在唐人岭南道版图里是**北门**——再北翻大庾岭便出岭，再南顺北江而下二三百里即广州。盛唐人由长安南下岭南、岭南士人北上长安，几乎人人都要在韶州城下歇一夜——这是岭南五府的北襟。

至于曹溪一山一寺，**山水细节不在这一篇 doc 里讲**（参 [曹溪](../scenic/曹溪.md) 待立）。本 doc 只记一句：韶州曲江城东南约六十余里，曹溪之畔，有一座唐初建立的山寺——755 这一刻，寺名为"中兴寺"或"法泉寺"（武则天神龙元年 705 敕赐"中兴寺"，旋改"法泉寺"），北宋开宝元年（968）方改今名"南华寺"。

## 大唐天宝十四载（755年）

唐人眼里韶州的沿革清楚——**隋开皇九年（589）平陈，于汉南海郡曲江县地置韶州**，治曲江。隋末韶州一度并入广州。**唐武德四年（621）**平萧铣后复置韶州，仍治曲江。**天宝元年（742）**玄宗诏天下州改郡——韶州改称**始兴郡**。**乾元元年（758）**肃宗复改郡为州——但 755 这一刻，城门匾额上写的是"始兴郡"。民间口语仍呼"韶州"——两个名号在唐人嘴里互换不分。

755 年的韶州属**岭南道始兴郡治**——领县六：曲江（州治）、始兴、乐昌、翁源、仁化、浈昌（《旧唐书·地理志四·韶州》"领县六"为唐代定制；县名诸志小有出入，以《旧唐书》为常说）。州等据《新唐书·地理志七上·岭南道·韶州始兴郡》明载为"**下**"——**户三万一千、口十六万八千九百四十八**。唐代州等划分并非单凭户口（户三万余在开元户等口径上原近中州下段），而是综合政治、军事、地理分量——韶州地处岭外，远悬五岭之南，不当军镇要冲、不近京畿、不为节度治所，因而《新唐书》列为**下州**。话虽如此，岭南道州多下州小户，韶州在岭南北段仍居二线州格——次于广州、桂州、容州一线的大州，**全凭大庾岭路这一条命脉**撑起声名。

韶州的分量**不在户口而在三件事**——

**一是大庾岭路**。大庾岭古名塞上、台岭，秦汉时已有小径——但岭高路险，岭南北通中原向以走湘水越骑田岭入郴州一线（湘粤水路）为主。**开元四年（716）**，时任左拾遗的韶州曲江人**张九龄**上书玄宗，请凿开大庾岭新道——诏许之，九龄亲行督役，凿通梅岭山隘。此后大庾岭路宽可通车舆，岭南贡赋、士人往来、海舶舶来品（香药、犀角、珍珠、玻璃）皆走此路北上。**韶州正卡在大庾岭南口**——岭南向中原的吞吐量，过半由韶州过手。这是韶州盛唐分量的真正根本。张九龄《开凿大庾岭路序》自序此事："开元四载，冬十有一月，俾使臣左拾遗内供奉张九龄……缘磴道，披灌丛，相其山谷之宜，革其坂险之故"——这是韶州人写韶州事的盛唐一手文献。

**二是曹溪禅宗祖庭**。**六祖慧能（638-713）**，新州（今广东新兴）人，得黄梅五祖弘忍衣钵后，南归岭南，**居韶州曹溪宝林寺三十余年**——开演"顿悟"一脉，门下神会、怀让、行思诸大弟子由此散于天下，禅宗"南宗"由此立。**慧能于唐先天二年（713）圆寂于韶州曹溪**——755 这一刻，**六祖已圆寂四十二年**，曹溪寺塔下有其全身——但慧能开出的南宗禅，正在中唐天下风起。神会北上洛阳辩"南北宗"事在开元二十二年（734），离 755 仅二十一年。**755 时的韶州，是天下禅僧南来朝祖的一处圣地**——但盛唐人嘴里的"曹溪"，更多是个**地名兼宗门符号**，《六祖坛经》尚未广行天下（坛经的广传与定型是中晚唐至宋的事）。本 doc 此处止步——禅宗细节参曹溪 doc。

**三是张九龄的故里**。**张九龄（673-740）**，韶州曲江人，进士出身，开元二十一年（733）至开元二十四年（736）任中书令——盛唐最后一位以纯文臣身份拜相、且以诗名传世的宰相。**九龄于开元二十八年（740）卒**——755 这一刻，曲江公已卒十五年。但九龄在韶州曲江的故居、家庙、亲族仍在——韶州城外的张氏，是岭南唐代头一号士族。九龄诗"海上生明月，天涯共此时"（《望月怀远》），及其《感遇》十二首，是盛唐五言古的高标。九龄被罢相后退居韶州曲江，至卒——"曲江公"之号即取此地。

755 年冬安禄山反范阳——消息沿驿道南传，过荆襄、过湘水、翻大庾岭入韶州，约需半月余。**755 时韶州官称始兴郡，长官为太守**（742-758 郡制时期，州皆改郡、刺史皆改太守）——始兴郡太守接到的诏令大约是：戒严、备粮、护大庾岭路。韶州本无大军镇——岭南军事节制在广州五府经略——但**大庾岭路是岭南向北输贡的命脉**，乱世里这条路一旦断，岭南便成飞地。韶州在接下来八年战事里没有大战——但岭南向朝廷输送的粮、香药、犀角、珍珠，几乎件件过韶州城下。

## 历史长河

**中晚唐**　至德二载（757）肃宗复改郡为州，乾元元年（758）后稳定。中晚唐韶州是贬官南下、北归必经之地——**韩愈元和十四年（819）正月贬潮州刺史**，自长安南下翻大庾岭入韶州，作《左迁至蓝关示侄孙湘》"云横秦岭家何在，雪拥蓝关马不前"虽作于秦岭，但其潮州之贬，韶州大庾岭是关键一段。**刘禹锡贞元二十一年（805）贬连州**、**柳宗元元和十年（815）自永州量移柳州**亦过韶州——这一时期岭南北段是中唐贬官诗的舞台。曹溪禅在中晚唐渐盛——**唐宪宗元和十年（815）追谥慧能为"大鉴禅师"**，曹溪一寺由是更显。

**两宋**　北宋开宝元年（968）曹溪法泉寺改名"南华寺"，此名沿用至今。**绍圣元年（1094）至元符三年（1100），苏轼贬惠州、儋州，往返皆过曹溪南华寺**——苏轼《南华寺》"亭亭塔中人，问我何所见"。东坡两过曹溪，礼六祖真身——是宋代士大夫与曹溪交涉最深的一段。北宋韶州属广南东路。

**元明清**　明洪武初改韶州路为韶州府——治曲江县。明清韶州府为岭南北门，大庾岭路仍为粤北通赣的主道——直至清末粤汉铁路通车（1916 始通至韶州），大庾岭路才退出历史主干。**南华寺**明万历二十八年（1600）憨山德清禅师重修曹溪，是禅宗祖庭的一次中兴。

**民国至今**　**"韶关"之名起于民国**——粤汉铁路通车后，韶州城北设"韶关"火车站，因城北原有清代"韶关税关"得名，铁路时代后"韶关"逐渐取代"韶州"成为城名。今**广东省韶关市**——城区中心仍在武水、浈水交汇处，与盛唐韶州州治曲江县同址，是粤北一千四百余年同一城址的连续。**曹溪南华寺**今仍在韶关市曲江区马坝镇东南，六祖慧能真身坐像仍存。**大庾岭古道**今为中国现存古驿道中保存最完好的一段——梅岭关楼下张九龄开凿的石阶仍可登临。**张九龄墓**在今韶关市武江区田螺冲。盛唐那座方圆数里、城门匾额上写着"始兴郡"的小城——夯土城墙在今韶关市区的层叠下早已埋没。一千二百多年过去，那座城名换了几番——但**大庾岭、北江、曹溪**三件事一件没变。

## 引文

> 韶州，下。隋南海郡之曲江县。武德四年，平萧铣，置韶州。天宝元年，改为始兴郡。乾元元年，复为韶州。

——《旧唐书·卷四一·地理志四·岭南道·韶州》（节录；755 时官称"始兴郡"为天宝元年改；韶州州等明书"下"）

> 韶州始兴郡，下。本东衡州，武德四年更名……户三万一千，口十六万八千九百四十八。

——《新唐书·卷四十三上·地理志七上·岭南道·韶州始兴郡》（节录；州等"下"、户口数 verbatim）

张九龄《开凿大庾岭路序》（开元四年 716，于韶州大庾岭工次自序此事）：

> 开元四载，冬十有一月，俾使臣左拾遗内供奉张九龄……缘磴道，披灌丛，相其山谷之宜，革其坂险之故。

——这是韶州人写韶州事的盛唐一手文献；岭路自此一变。

韩愈《左迁至蓝关示侄孙湘》（元和十四年 819 春贬潮州途次蓝关作，是岁春末过大庾岭入韶州，去 755 六十四年）：

> 云横秦岭家何在，雪拥蓝关马不前。知汝远来应有意，好收吾骨瘴江边。

——韩愈贬潮州，韶州大庾岭是其南下中段；中晚唐贬官诗中关于韶州一线的最高一处。
