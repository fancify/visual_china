# 河州

## 一句话

陇右道最南那一座小下州——枹罕（今临夏市）位于大夏河谷，洮河自南向北汇入黄河；黄河在州域北缘流过。

## 形貌

河州城坐在**大夏河南岸**一处河谷地上——州治在**枹罕县**（今甘肃临夏），城周不大，城北临大夏河、城南依山。**黄河不贴城**——它在州域北缘自西向东流过，距州治尚有数十里；大夏河自南向北汇入黄河。州境东接兰州，西北沿黄河上溯三百里到鄯州，南越积石山一带便逼近吐蕃东境——河州正卡在唐与吐蕃之间最近的一段河谷上。

谷里以农牧并行为主：**大夏河两岸**有水浇地，种粟、麦、青稞；台地之上是干旱草滩，党项、吐谷浑余部诸羌的部落散居其间，放牛羊。城里是另一番面貌——州刺史署、长史司马诸僚按下州制设，城外有戍堡与烽燧若干，斥候每年三月开始向南方边境一拨一拨派出。

物候属高寒——海拔逾一千八百米，冬日河上结冰，五月夜里仍要盖被。**大夏河自城西南而下贴城北流**，汇入州域北缘的黄河；黄河自此向东北去，绕过兰州西出陇山——河州是黄河自青海高原下行后所经的第一座唐州治。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

河州地很老。汉昭帝置金城郡，治枹罕县——这是"枹罕"两字的起头。两汉之间羌人屡反，城几度易主。西魏大统十二年（546）置**河州**——"河州"这一名号自西魏起立，取黄河之意。隋开皇三年（583）罢郡为州，仍曰河州；大业三年（607）再罢州置郡。唐武德二年（619）平薛举割据势力后置河州——治枹罕。**天宝元年（742）玄宗诏天下州改郡——河州改曰安乡郡；乾元元年（758）肃宗复郡为州**。所以天宝十四载（755）的官文书上，这地方多写"安乡郡"——民间口语仍称"河州"，两个名号并行不分。

唐对河州的实际管辖不长。仪凤三年（678）吐蕃挟吐谷浑余部内附，湟水谷以南成为唐与吐蕃直接对峙的前沿。河州自此一直是**陇右道的最南一州**——与鄯州的"前线总司令部"地位不同，河州是边境上的一座小州，户口不旺，但**位置极要**：吐蕃自青海高原东出，最近的一条道便是过黄河上游入河州。

### 此刻

天宝十四载（755），河州是**安乡郡**治。户口据《旧唐书·地理志》载约**五千七百八十二户**——是陇右道里最小的几座下州之一（《元和志》户数因元和年间数与盛唐异文而有出入，盛唐时口数大略相当）。城里官署按下州制设；陇右节度使治在鄯州，河州只是节度下的一处边州，无大兵驻——常驻戍卒不过千余，倚周边戍堡与烽燧分驻应敌。

河州还是**唐与吐蕃的互市点之一**。盛唐自开元间便于陇右沿边设若干互市——河州、鄯州、兰州各有市，吐蕃以马匹、毛皮、麝香换唐之绢帛、茶叶、铁器。河州市规模不大，但因地近，是吐蕃北道商旅过陇山入唐境最早一站。

对天宝十四载的河州人来说，这座城意味着——**唐土的南端、吐蕃的北限**。城外大夏河谷之南，越过积石便是吐蕃东境；州域北缘的黄河是另一道边线。春末河开，斥候要先一拨派到河南去；秋深霜降之后，吐蕃骑兵便要试探着来。他们尚不知道，再过八年，这座城将自此入吐蕃。

## 历史长河（755 → 今）

**中晚唐**　乾元元年（758）肃宗复郡为州——河州之名重归。但安史乱后陇右兵八万东调，河州一线已无重兵。**广德元年（763）十月，吐蕃尚结息率二十万众入长安**——同年前后，吐蕃自青海北上、连陷河、洮、岷、廓、鄯诸州。河州陷蕃**约在广德元年至大历元年间（763-766）**，史无明确陷年（《旧唐书》《通鉴》载笔互有异文，仅记"陇右诸州相继陷蕃"）。**自此入吐蕃八十余年**——湟水以南的河湟谷地俱归吐蕃多麦节度。

宣宗大中三年（849）三月沙州张议潮起义；咸通七年（866）归义军部将拓跋怀光西破鄯州、斩论恐热，**河州一带亦在此前后归唐**——但归义军势力旋衰，五代之间河州再失中央管辖，落入吐蕃余部及党项手中。

**宋金至清**　北宋元符二年（1099）王厚河湟开边——破青唐复设鄯州；河州一带亦在此前后入宋（**熙宁六年/1073 王韶收复河州**，是宋熙河开边的关键一战）。宋廷于此置河州——治枹罕，与唐河州同址。金代仍称河州。元置河州路；明洪武六年（1373）改河州卫——卫城在唐枹罕旧址上向南略移。清雍正间废卫，复置河州——这一名号自西魏立至清末，存名一千四百年。

**民国至今**　民国二年（1913）废府州、改河州为导河县；1929 年改临夏县——"临夏"取临近大夏河之意。今为甘肃省临夏回族自治州治所，人口约一百三十万。城里清代以来的回族、东乡族聚落格局是这地方的近代面貌；唐河州城残基已不可考——只有大夏河仍自城西贴流而下，州域北缘的黄河仍向东北去，仍是这地方过去一千五百年从未动过的两条线。

## 引文

> 河州，本汉枹罕县，属金城郡。后魏大统十二年置河州。隋大业三年罢州置枹罕郡。武德二年置河州。天宝元年改为安乡郡。乾元元年复为河州。

——《旧唐书·卷四十·地理志三·陇右道·河州》（盛唐河州沿革的标准记载）
