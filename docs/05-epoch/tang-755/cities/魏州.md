# 魏州

## 一句话

河北道腹心一州——黄河北岸冲积平原上的最大州，**户十六万**冠河北一道；安史平后此城便是"魏博节度使"的治所——"长安天子，魏府牙军"由此立。

## 形貌

魏州治贵乡县，城坐河北平原南腹——黄河北岸约百里许（盛唐黄河自滑州东北经濮州、博州、贝州一线下沧州入海，魏州距河颇近）。城周不大但厚实，外环夯土，内有州刺史署、长史司马诸僚衙、州学。城南、城东俱平畴——河北道最厚一片冲积平原即在此一带，麦、粟、桑、麻并产，户口稠密。北循驿道经贝州、冀州可至定州；西经相州、邺郡至河东太行东麓；东南越河至河南道濮、滑、汴一线——**魏州是河北道南半"出河北入河南"驿道的总枢纽**。物候为华北平原中段——春旱、夏溽热雨集中、秋高、冬寒有薄雪。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

魏地是**先秦战国魏国故壤**——这是"魏州"二字最远的根脚（魏文侯都安邑、迁大梁，皆在战国前期；唐魏州治贵乡虽非战国魏都本城，地属魏地腹心）。汉以来此地为魏郡——治邺，与唐魏州治贵乡相距百里；唐相州治安阳，邺城遗址在安阳北漳水南，与唐相州治隔漳水相望（非一城）。

行政沿革须分别两县两州。州一级：**北周宣帝大象二年（580）置魏州**——"魏州"二字自此入史，治贵乡县；隋大业三年（607）诏改州为郡，魏州改称**武阳郡**（隋制非沿北朝旧名）；**唐武德四年（621）**平窦建德、刘黑闼后复置魏州，治贵乡——"魏州"复立；**天宝元年（742）**玄宗诏天下州一律改称郡——魏州改称**魏郡**（取汉魏郡旧名，但治所在贵乡而非汉魏郡治邺）——这是 755 年当年的官称；**乾元元年（758）**肃宗复改郡为州，魏郡复称魏州。**贵乡县**为州治附郭——隋开皇六年（586）析置贵乡县，唐武德四年（621）后稳定为魏州（魏郡）附郭县，盛唐 755 时即此。

### 此刻

天宝十四载（755）——官称"魏郡"，民间口语仍呼"魏州"。河北道腹心一**上州**。据《旧唐书·地理志二·魏郡》记**户约十六万、口约百一十万**——**户数为河北一道最厚、亦冠盛唐天下大州之一**（户数超关内道、河南道大半数州，与扬州、宋州、汴州同级，仅次于长安京兆府、洛阳河南府等几处京畿）。按《旧唐书·地理志二·魏郡》魏郡天宝领县 10：贵乡、元城、莘县、临河、馆陶、冠氏、朝城、昌乐、洹水、成安（贵乡为附郭州治；武圣未独立置县，临清、经城属贝州，不入魏郡）。州内官署按上州制设，魏郡太守由朝廷直除。

魏州非节度治，但**河北道采访使治魏州**（道一级监察长官驻此；一说幽州，史料系年略异）；幽州（范阳郡）则为范阳节度治，距魏州陆路约九百唐里。**755 这一刻魏州的最大分量是户口与漕路**——城南循永济渠（隋大业四年 / 608 开，自洛阳东北经汴、滑、魏、贝、沧至涿郡）过境，**魏州正是永济渠中段的最大停泊州**；河北道贡赋粮米经此南运洛阳、长安，是关东漕运东支的命脉。十一月九日安禄山自范阳起兵，叛军主力自幽州南下，**魏州在叛军南进必经之线**——同月底前后即陷（与河北诸州一道，仅常山颜杲卿、平原颜真卿一带短暂抗节）。盛唐魏州城里那十六万户人家，从此十年陷于河朔藩镇的根脚——"魏博"二字将由此立。

## 历史长河（755 → 今）

**中晚唐**　乾元元年（758）肃宗复魏郡为魏州。**广德元年（763）**史朝义败、安史平定，朝廷以**田承嗣**为**魏博节度使**——治魏州（贵乡）——辖魏、博、贝、卫、澶、相六州——"魏博"二字自此立，**魏州为河朔三镇（成德、魏博、卢龙）之一的治所**。魏博自田承嗣以下，田氏父子三代相承，至**田弘正元和七年（812）以六州归朝**为一段；后再叛、再服，**罗绍威光启二年（886）—天复六年（906）**间数变，皆以魏州为根脚。中晚唐史志多以"魏府"称之——**魏府牙军**（节度使的亲兵牙兵）以骄横著称，《资治通鉴·卷二五五》中和元年（881）记"长安天子，魏府牙军"——这是河朔藩镇与朝廷关系的一句口语总结。盛唐 755 时魏州尚是朝廷直辖之上州，"魏博"与"牙军"皆是后话；本 doc 严守 755 时态。

**五代至宋**　五代后梁、后唐皆视魏州为河北根本——后唐庄宗李存勖灭后梁前以魏州为本镇。**后晋开运二年（945）**升魏州为**广晋府**；**后汉乾祐元年（948）**改广晋府为**大名府**——"大名"二字自此立。**北宋庆历二年（1042）**仁宗以契丹南犯，升大名府为**北京大名府**——北宋"四京"之一（东京汴梁、西京洛阳、南京应天、北京大名）——治所即唐魏州贵乡旧址。这是 755 年后魏州城的最盛一段。

**金元明清至今**　金代大名府属大名府路，元属中书省。明洪武三十四年（1401）漳卫河大水**毁北宋大名府旧城**——府治南迁约八里至今河北大名县城（明代新筑大名府城）——**盛唐魏州、北宋北京大名府的旧城自此沉于地下**。1990 年代以来河北省文物考古多次勘探"大名府故城"遗址（在今大名县城东北八里营镇一带），城圈轮廓、夯土城基、宋代宫殿基址相继出土——**这是盛唐魏州城所唯一可与今人共看的实物**。清代大名府属直隶省，民国 1913 废府存县，1949 后属河北省邯郸专区、邯郸市，今为河北省**邯郸市大名县**——盛唐魏州十六万户的大城，今为一座普通县城，旧城遗址列**全国重点文物保护单位**（"大名府故城"，2006 年公布第六批）。

## 引文

> 长安天子，魏府牙军。

——《资治通鉴·卷二五五》中和元年（881）记河朔人语。"魏府牙军"即魏博节度使的亲军——自田承嗣立军以下，至晚唐已成左右节度使废立、骄横不可制的一支。这句话是中晚唐人对河朔与朝廷关系的一句口语总结——盛唐 755 时尚无此分。
