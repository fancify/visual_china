# 涪州

## 一句话

山南东道西南端长江南岸一州——治涪陵县（今重庆市涪陵区），乌江自南而来于城下北入长江，是**荔枝道**自蜀入秦的起点；州因涪陵水（乌江）而名，州境为古巴郡涪陵县地。

## 大唐天宝十四载（755年）

涪州治**涪陵县**，在**长江南岸**——乌江（唐人称涪陵水、内江）自南而来，于城下北入大江（长江）；两江交汇成一处水口，州城便扼这处水口的南岸。州属**山南东道西南端**——东接忠州、北望万州、西接渝州、南接黔州，东南折入山岭便是黔中道境。**隋开皇三年（583）置涪州**——"因涪陵水（乌江）得名"。**唐武德元年（618）以渝州之涪陵镇置涪州**（《新唐书·地理志四》载："涪州涪陵郡，下。武德元年以渝州之涪陵镇置。"）。**天宝元年（742）**随玄宗朝"改州为郡"通制改为**涪陵郡**——755 时正名"涪州涪陵郡"，两名在唐人嘴里互换不分；**乾元元年（758）**肃宗复改为涪州——但这是 755 之后的事。**新唐书·地理志四·山南东道**列涪州于忠州之后、万州之前——隶**山南道东道採访使，治襄州**；**州等为"下"**——是山南东道靠西南内陆的下州。**《新唐书》载涪州天宝籍"户九千四百，口四万四千七百二十二，县五"——领涪陵、宾化、武龙、乐温、温山五县**：涪陵为郡治；宾化"本隆化，贞观十一年置，先天元年更名"；武龙武德二年与涪陵同置；乐温武德二年析巴县地置，初隶南潾州，九年来属；温山本隶南潾州，后来属。州贡**麸金、文刀、獠布、蜡**——皆乌江流域山地之产；境内獠人（仡佬先民）杂居，是山南东道西南向黔中道过渡的一处缓冲。**《元和郡县图志·卷三〇》得名释作："在蜀江之南，涪江之西，故为名"——所言"涪江"即涪陵水即乌江，非今四川境另一条涪江。**

涪州在 755 这一刻的真正分量，不在户口而在**荔枝**。**杨贵妃嗜生荔枝**——唐人盛传玄宗为之置驿自涪陵北上、过梁州、洋州、入子午谷至长安——是为**荔枝道**。《唐国史补》载"置驿自涪陵"，《新唐书·杨贵妃传》亦记"妃嗜荔枝，必欲生致之，乃置骑传送，走数千里，味未变已至京师"；苏轼《荔枝叹》追述"天宝岁贡取之涪"——北宋人犹记涪州为荔枝道起点。**驿骑昼夜兼行，约六七日至长安**——民间"三日"之说（晚唐杜牧"一骑红尘妃子笑"渲染所致）系夸辞，史地学者多持七日七夜之论。**这一年的荔枝**——五六月间贡入，长安尚不知北方有事；至冬十一月初九安禄山起兵于范阳，玄宗仓皇西幸；明年（756）六月马嵬之变，贵妃缢死——**涪州的荔枝驿，到这一年也算走到了尽头**。

## 历史长河

**中唐至五代**　乾元元年（758）复为涪州。安史乱后山南东道诸州兵东调，涪州戍备渐薄。**《元和郡县图志·卷三〇》将涪州列入"江南道六·黔府观察使"——上元二年（761）因黄莩硤獠贼聚，江陵节度请涪州来隶江陵，元和三年（808）李吉甫奏复隶黔府**——故元和年间涪州归道已变，非 755 山南东道旧属。元和户数据《元和志》仅"三百五，乡二十一"——较开元盛时"六千九百九，乡二十六" 折损极重，反映安史至元和间峡江诸州户口大溃。荔枝贡随玄宗朝去而罢——白居易《荔枝图序》追记岭南荔枝事而不再提涪州——盖荔枝道盛贡只在天宝十数年间。

**两宋元明清**　**北宋元符二年至元符三年（1099-1100）黄庭坚谪居涪州**——号"涪翁"，作《北窗诗》《竹枝词九首》，是涪州在中古文人志业里最浓的一笔。宋属夔州路。元至元二十年（1283）废涪陵县入州，翌年改属重庆路。明清两代属**四川重庆府**。

**民国至今**　民国二年（1913）改涪州为**涪陵县**。1949 年后置**涪陵专区/涪陵地区**，1996 年撤涪陵地区，1997 年重庆直辖后划归重庆市，今为**重庆市涪陵区**——区治崇义街道，承唐涪陵县治旧址。乌江仍自南而来、于城下北入长江，与唐涪陵县城方位大体相承。
