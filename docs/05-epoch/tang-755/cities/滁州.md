# 滁州

## 一句话

淮南道江北一小州、扬州西去陆路二百余里的一处腰节——治清流县（今安徽滁州市），742-758 年间名"永阳郡"；城南琅琊山林壑深秀，盛唐时已有山名，唯尚未因后世欧阳修《醉翁亭记》而著称天下。

## 形貌

滁州城在长江北岸丘陵间——南距大江约百里，**不直接临江**，与南面对岸江南西道宣州（治宣城、辖当涂）隔江而望。城所在清流县——县治、州治同城；西邻寿州，东接扬州、和州，同属淮南道江北一带。**琅琊山在滁州城西南数里**，林壑幽深、泉石错落，唐时已名"琅琊"，但盛唐尚未有亭、未有铭。**《旧唐书·地理志》载滁州（永阳郡）天宝领县三**（清流、全椒、永阳），户口在淮南诸州中靠后——按户口大致当下州门槛。物候是淮南江北的典型：春雨连绵、夏热湿、秋凉早、冬有霜。

## 大唐天宝十四载（755年）

滁州沿革——**隋开皇九年（589）平陈，始置滁州**，治新昌县（旋改清流县）；隋大业初废滁州入江都郡。**唐武德三年（620）复置滁州**，治清流县——属淮南道；**天宝元年（742）**玄宗诏天下州一律改郡，滁州改称**永阳郡**——这是 755 年当年的官称；**乾元元年（758）**复改郡为州，"永阳郡"之名前后用了十六年。

755 年时滁州属**淮南道**——更准确地说，此时州已改郡，官称作**永阳郡**。**淮南节度使一职至德二载（757）方置**——755 年当年淮南尚无节度使，仅有玄宗开元二十一年（733）所立**淮南采访使**（治扬州）统淮南诸郡。永阳郡在淮南采访使辖境西北一隅，与扬州、和州、庐州、寿州同道。**天宝十四载永阳郡太守在任者史志阙考**（注：天宝元年改州为郡后郡之长官称"太守"，乾元元年复郡为州后称"刺史"——755 年当年只称"永阳郡太守"，不称"滁州刺史"，是同一职而不同名）。城周不大、不立都督府、不置军镇——盛唐滁州是淮南江北一座安静的中下州。

## 历史长河

中晚唐至德二载（757）淮南节度使置于扬州后，滁州归其辖；安史乱起后江淮震动，但滁州因偏处淮南西北丘陵、远离主战场，未受大扰。

**北宋庆历五年（1045）欧阳修以右正言、知制诰落职贬知滁州**——在滁二年余，著**《醉翁亭记》**："环滁皆山也。其西南诸峰，林壑尤美，望之蔚然而深秀者，琅琊也"——"醉翁之意不在酒，在乎山水之间也"——自此琅琊山、醉翁亭、滁州三字始与一段宋人文心紧紧绑定。**注：欧阳修任滁州知州距盛唐天宝十四载已近三百年——属北宋中期事，与本 doc 所写盛唐 755 时滁州并非同一时代；盛唐滁州无亭、无铭、无欧公**。

**元末濠州（今安徽凤阳）人朱元璋**起兵反元，**至正十四年（1354）取滁州**——以滁州为江淮间一处早期据点，旋南渡破集庆（今南京）立基业。后世明洪武初滁州为直隶州，属南直隶；清沿明制，雍正间为直隶州，属安徽布政司。民国元年（1912）废滁州为滁县。1949 年后属安徽省，先后属滁县专区、滁县地区；**1992 年滁县地区改设地级滁州市**——"滁州市"作为正式行政区名是民国以后、20 世纪末始用。盛唐唐永阳郡人，自然不知千余年后此城会以"滁州市"为名、以"醉翁亭"为天下文人凭吊处。

## 引文

> 滁州，永阳郡。下。隋开皇九年置滁州。大业初废。武德三年复置滁州。天宝元年改为永阳郡。乾元元年复为滁州。领县三：清流、全椒、永阳。

——《旧唐书·地理志三·淮南道·滁州（永阳郡）》（节录）
