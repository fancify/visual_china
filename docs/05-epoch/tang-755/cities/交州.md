# 交州

## 一句话

安南都护府治所，唐人口中"交趾"——岭南道之尽南，红河三角洲上一座汉以来九百年的郡治；自此往南便是日南、林邑，再无大唐州县。

## 形貌

交州不在岭南腹地，而在岭南再南——红河入海前最后一段冲积平原上。

自长安南下，过商於、襄州、江陵、洞庭、潭州，再过衡州、永州、桂州，自桂州西南行越静江、出岭，便入安南境——一路水陆相接约七千余里。若取另一道，自长安南出蓝关、武关，过襄州，转沿江而东至洪州，再南出大庾岭、过韶州、广州，自广州西南循海岸或循内河至交州——亦约七千余里。两道皆远，唐人以为天涯。

红河（唐人称**富良江**）自西北山中来，一路南下至安南境内分作数支，于宋平县（交州治）一带汇为三角洲，再东南入海。三角洲东西约百余里、南北亦近百里——平地肥沃，稻可两熟，是岭南最南的一片大田。三角洲西、北二面是山——西望是后世越南北部群山，北望是十万大山余脉与岭南山脊相连，东南面则是一片汪洋（北部湾）。冬日北风过岭而南，到此已弱；夏日西南季风自海上推入，雨水极盛——一年里热月长冷月短，瘴疠时作。

交州城就盖在红河南岸的台地上——城周史料未明载，唐人通称"宋平城"或径称"交州城"。城内有安南都护府衙、交州刺史治、宋平县治——三套官署同在一城。城西、北、南三面有水绕——红河本支为北面天然护城河，城东则系陆路出口，东去循海可达钦州、廉州，再东去广州。城外四面阡陌，稻田之间散落村落，多是汉魏以来便已立籍的越人聚落与汉人移民混居之处——汉、唐间所谓"交趾旧族"，多在此地。

物候是大唐辖境最南的：冬无冰、夏极热，五月至九月一段雨水连绵，秋冬转旱。城内官员多自中原来——岭南诸州素以瘴疠著名，交州尤甚——盛唐至中唐，贬官南来死于此地者不绝。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

交州在唐人眼里不是新地——**自汉武帝元鼎六年（前 111）灭南越国置交趾、九真、日南三郡至唐天宝（742），交州一带为中原王朝直辖已逾八百五十年**。

汉武帝元鼎六年（前 111）灭南越王赵建德、吕嘉，分其地置南海、苍梧、郁林、合浦、交趾、九真、日南、珠崖、儋耳九郡——其中**交趾郡**治羸𨻻县（在今越南河内西北），辖红河三角洲一带。**"交趾"二字**——《汉书·地理志》颜师古注引张晏曰"足相向，故曰交趾"，又引应劭说作南方蛮夷土俗——是中原王朝对此地最早的正式名号。汉光武建武十六年（40），交趾女子**征侧、征贰**起兵反汉——攻掠岭南六十五城；建武十九年（43），**伏波将军马援**率军南讨，平征氏姊妹于禁谿、麊泠——**立铜柱以表汉之南界**（"铜柱折，交趾灭"传说自此始；铜柱地点说法异，《后汉书·马援传》载"立铜柱，为汉之极界"，唐时已仅存传说而无实物）。马援回军时尝叹"男儿要当死于边野，以马革裹尸还耳"——此语自交州传回中原，遂入中古武人辞典。

东汉以下，交州历代为中原王朝南端。三国吴黄武五年（226）分交州置广州——交州治所自此专指安南一带；西晋承之；南朝宋、齐、梁、陈交州刺史时设时废；隋开皇九年（589）平陈，复置交州——大业三年（607）改交州为**交趾郡**。

入唐之后建置数易。**武德五年（622）**置交州——州治宋平县；**调露元年（679）**改交州都督府为**安南都护府**——这是大唐六大都护府之一，专辖岭南再南的红河三角洲及周边羁縻州县；治所即在交州城（宋平县）。**天宝元年（742）**玄宗诏天下州一律改称郡，交州改称**宋平郡**——同年安南都护府是否一并改名"镇南都护府"，《新唐书·地理志》载有"镇南"之名，《旧唐书》《元和郡县图志》记载略异，史家两说并存；正文从《新唐书》取"镇南"名号备录，但 755 当年通称仍多作"安南"。**至德二载（757）**肃宗复改郡为州，宋平郡复称交州；安南都护府复名安南——这是 757 年以后的事。755 这一年，城门匾额上写的是"宋平郡"或"安南都护府"，里头的人多半还是叫"交州"。

唐人若在 755 年驻足回望：自汉武置郡（前 111）至此已八百六十六年，自马援立铜柱（43）至此已七百一十二年——这片三角洲在中原王朝直辖名义下，已是这个长度。距长安七千余里、地最南、瘴最重——但建置最久。唐玄宗朝盛时安南境内并不太平——开元十年（722）梅叔鸾（梅黑帝）起兵据安南，被广州都督杨思勖率岭南诸军平定；天宝中亦时有蛮獠骚动。"安南"二字在长安朝堂上，是疆界最南而麻烦最多的一处。

### 此刻

天宝十四载（755）的交州，行政上是**岭南道安南都护府治、宋平郡治、宋平县治**——三套官称同在一城。安南都护府辖交、爱、驩、峰、陆、长、福禄、汤、武安等内属正州十余州，以及羁縻州数十——是大唐对岭南再南红河流域的整套羁縻—直辖混合体制。**都护**例由朝廷直派，秩三品；下设副都护、长史、司马、判官——一套完整的边疆都护府幕府。本州（宋平郡）刺史多兼都护府长官。

755 年时安南都护是谁，正史无明载——天宝十一载（752）以后至至德初年间的都护人选，《旧唐书》《新唐书·安南都护府》皆有缺漏；可推知者多为遥领或短任。**岭南五府经略**则治广州，统辖广、桂、邕、容、安南五管——755 在任经略使史源异文，可考者有**何履光**之名（《旧唐书》《新唐书》互见）。岭南节度使要到**乾元元年（758）**才置——755 当年还没有"岭南节度使"这个官名。安南都护府名义上是五府经略下辖之一，但实际军事调度自有体系。

人口据《元和郡县图志·卷三八》载安南都护府**开元户**"二万四千二百三十、口九万九千六百五十二"——本 doc 取**元和志开元户数**为准（《旧唐书·地理志》另载天宝户数略异，本 doc 不混用）。这是开元年间安南境内可课于中原州县体制者；红河三角洲上的汉、越居民并非全数在册。

755 年冬，安禄山起兵的消息沿驿道自长安南传——过商於、襄州、江陵、潭州、衡州、桂州、邕州，**抵交州大约需四十余日**。安南都护府接到的诏令大约是：戒严、守城、备粮，并防内属诸蛮獠乘隙——但岭南再南距河朔战场过远，朝廷并未抽调安南兵北上。次年（756）玄宗入蜀避乱、肃宗灵武即位，岭南诸道亦各自为政；安南都护府在这场北方大乱里，名义上仍奉唐正朔，实际上自管自的——这是 755 至 757 间盛唐边远都护府的普遍处境。

对盛唐而言，交州是**大唐辖境最南端、岭南道之尽南、安南都护府治所、马援铜柱所在的那片土地、海上丝路之南行最远的中原港口之一**——再南便是林邑（占婆，今越南中部）、扶南遗地（今柬埔寨）；再东南便是诃陵（爪哇）、室利佛逝（苏门答腊）——皆为羁縻或朝贡之外邦。自交州循海西南行可至天竺、波斯——盛唐**海上丝路**自广州、交州、明州（宁波）三大港出发，交州是其中最南的一处。这一年自交州出海的胡舶——大食、波斯、昆仑、林邑诸船——尚未察觉北方大乱；它们带来香料、犀角、明珠，运走丝、瓷、铁器。直到次年这一线贸易才略受影响——但交州港的胡舶往来一直未断。

至于**鉴真和尚**——与交州的地理关联在**第五次东渡**：天宝七载（748）鉴真自扬州第五次启程，海上遇风漂流至海南**振州**（今三亚）——振州在交州东南海上四五百里。鉴真自振州沿岸北返，过雷州、岭南、循江北归扬州——并未入交州。但交州人若知此事，当知大唐辖境之南至振州、之南再南至林邑——海道之远尽于此。鉴真**第六次东渡终成行**：天宝十二载（753）冬自扬州登舟，**天宝十三载（754）**抵奈良东大寺。**鉴真 754 抵日，755 年已驻奈良一年余**——这与 755 年交州的命运并无直接关联，只是同一片大唐海岸线上同一段岁月里的事（与[广州](./广州.md) doc 时序统一）。

## 子条目

### 交州城（宋平县治）

唐安南都护府治、宋平郡治、宋平县治——三套官称同在一城。城在红河南岸台地，周长史料未明确（一说约九里许，史源未稳）；城北临红河本支，城西、南有支流绕——东出陆路。城内有都护府衙、刺史治、县治、馆驿、佛寺数所、市场。盛唐至中唐交州城屡有修葺——至**咸通七年（866）**唐安南都护**高骈**收复交州后大筑**大罗城**，安南都城形制方告完备；755 时距高骈筑大罗城尚有百一十一年——此刻的交州城是旧城，尚非后世所称"大罗"。**大罗城**遗址在今越南河内市旧城西北一带——但 755 年这座城，址在何处至今考古不能完全确指。

### 安南都护府

调露元年（679）改交州都护府置——大唐六大都护府之一（安西、安北、安东、安南、单于、北庭）。**专辖红河三角洲及周边羁縻州县**——据《元和郡县图志·卷三八》载安南都护府辖**正州十三**：**交、陆、峰、爱、驩、长、福禄、汤、芝、武峨、演、武安、贵**——以及羁縻州数十。府署设都护（秩三品）、副都护、长史、司马、判官、录事参军等——一套完整的边疆都护府幕府。**天宝元年（742）安南都护府是否一并改名"镇南都护府"**，《新唐书·地理志》载有"镇南"之名，《旧唐书》《元和郡县图志》记载略异——史家两说并存；今多取"安南"为正名，"镇南"为天宝-至德间短时官称备录。**至德二载（757）**复名安南。府署即在交州城（宋平县治）内——衙署、库房、兵营皆集中于此。

### 宋平县

交州治、宋平郡治、安南都护府治——县治同郡治同都护府治同在一城。**"宋平"之名**始于南朝刘宋——刘宋孝武帝大明四年（460）于交州治羸𨻻县境析置宋平县，自此沿用。隋大业三年（607）废。唐武德四年（621）复置宋平县——为交州治。"宋平"二字含"刘宋（朝代名）平之"之意，是南朝交州治所的旧名，在唐时已成纯地名而非政治标识。县境约红河三角洲北部一带——南至日南、西至峰州、东至陆州。

### 马援铜柱（传说之所，地点异说）

东汉建武十九年（43）伏波将军马援平征氏姊妹后所立——**"立铜柱，为汉之极界"**（《后汉书·马援传》）。铜柱具体地点说法不一：一说在今越南北部钦州交界，一说更南至日南郡境（今越南中部），一说在林邑境内——三说自唐至清并存。唐人提"马援铜柱"——多作汉之南界 marker，不一定指实物地点；755 年时此柱实物已无人见过，仅作传说存于士人辞典。**"铜柱折，交趾灭"**——这则谶语是马援回军前对当地汉人移民说的，意为铜柱不倒、交趾不失——一句话立了一个七百年的预言。后世杜甫《诸将》"汉朝陵墓对南山，胡虏千秋尚入关。昨日玉鱼蒙葬地，早时金碗出人间……"——盛唐人写"金人"、"铜柱"等汉物意象，多取此一脉典故。马援铜柱的实物今不可考——但"铜柱"作为南界象征，自东汉至清入诗入文未绝。

### 富良江（红河）

唐人称红河为**富良江**——自西北山中来，一路东南流经峰州、交州境，于宋平县一带分作数支，于交州东南面入海（北部湾）。江面宽阔——汛期可至数里。唐人航行可自交州沿富良江北上至峰州（今越南富寿一带），再上至唐朱鸢州、汤州；下行可循海岸至日南、林邑。富良江名号自唐至宋未变——**北宋熙宁九年（1076）郭逵伐安南，宋越两军会战于富良江**——是中原与独立后越南最大一次正面会战。**白藤江**则在富良江入海口附近的另一条支流——后晋天福四年（939）吴权于白藤江败南汉刘弘操军，越南自此脱离中原王朝独立——但这是另一段历史。755 年时富良江、白藤江都还在大唐安南都护府境内。

### 安南境内的蛮獠

交州、安南都护府境内的非汉居民——唐人通称"蛮獠"或"南蛮"。**李氏**：宋平、爱州一带汉化已深的越人大族，唐时多入官；**杜氏**：交州本地豪族；**冯氏**：钦州、廉州一带——盛唐冯盎（唐初岭南豪族冯智戴之父）一系——本为高凉冯氏，势力远及交州外围。安南境内的蛮獠—部落首领多受唐廷羁縻州封号，世袭酋长——但不入正赋。开元十年（722）梅叔鸾（"梅黑帝"，自称"黑帝"）起兵据安南——攻陷交州城——广州都督杨思勖率岭南诸军、并联合岭南蛮獠首领冯氏诸部南讨——擒梅叔鸾，事平。755 年时距梅叔鸾之乱已三十三年——但安南境内地方蛮獠的不稳，仍是都护府日常要务。

### 海上丝路·交州端

盛唐三大对外港口——广州、交州、扬州（一说明州）——交州为最南。自交州循海西南行，过日南、林邑、扶南、室利佛逝、诃陵——可达天竺、师子国（斯里兰卡）；自此再西可至大食。**大食、波斯、昆仑（南海诸国）、林邑、扶南**诸国商舶皆有至交州者——卸货于城外港埠，胡商在城内有专属住居区。唐贾耽《广州通海夷道》（约 800 年）所记航路上，交州是出广州后西南行的第一大港。755 年时这一海道正盛——城内胡商、僧人、使节络绎；瓷、丝、铁器自此南去，香料、犀角、明珠、宝石自此入唐。

## 历史长河（755 → 今）

**中晚唐**　至德二载（757）安南都护府复名安南。755 年安史之乱起后，岭南诸道虽奉唐正朔但实际上各自为政——安南都护府亦在此列。**贞元六年（790）**安南都护**赵昌**任内安抚有方，境内稍安；**贞元七年（791）**安南**杜英翰**之子杜英策起兵——自称大胜王，反唐——后被赵昌平定。**元和四年（809）**安南都护**张舟**——这是中晚唐安南都护中较有作为的一任。中唐安南境内多事——本地豪族与中原任命之都护时有冲突；蛮獠时叛时附。**咸通元年（860）**南诏军北上攻陷安南——交州落于南诏之手；**咸通七年（866）安南都护高骈率唐军反攻**——克交州、灭南诏驻军、收复安南全境。同年高骈大筑**大罗城**——这是交州城形制最完备的一次扩建，自此交州城又称"大罗"。咸通九年（868）唐廷以高骈功，析安南都护府升**静海军节度使**——交州自此为静海军节度治所。**唐昭宗天祐四年（907）唐亡**，静海军节度使**曲承裕**自立——但仍奉中原王朝（后梁、后唐）正朔。

**五代**　五代之际中原政权更迭频繁，静海军节度使**曲、杨、矫、吴**诸家相继自立。**后晋天福四年（939）**——**吴权**于**白藤江**大败南汉刘䶮军，杀刘弘操——遂自立为王，**越南正式脱离中原王朝独立**。自前 111 年汉武置交趾郡至 939 年吴权独立——**红河三角洲为中原王朝直辖凡一千零五十年**——这是一个长尺度的终结。757 年安南都护府复名时无人能料；755 年宋平郡之城门匾额下走过的胡商使节，更料不到。

**宋元**　吴权立国后越南历经吴、丁、前黎、李、陈、后黎、阮诸朝。**北宋开宝八年（975）**宋廷封丁部领为"交趾郡王"——这是中原王朝对独立后越南的第一次正式册封；自此中越关系由"郡县"转为"宗藩"。**北宋熙宁九年（1076）**宋越交战于富良江——宋军前锋郭逵渡江攻击越南李朝军——双方互有胜负，最终议和——宋廷未能恢复对安南的直辖。**李朝顺天元年（1010）**李公蕴（李太祖）自华闾迁都至大罗城——**改名昇龙**——"河内"之名要再过八百年（19 世纪初阮朝置河内省，1831 年明命帝设河内省，"河内"之名始正式入官府文书）才出现。755 年的宋平郡，1010 年的昇龙，19 世纪的河内——是同一片三角洲上同一座城（大致）的三个名号。**元至元二十二年（1285）、二十四年（1287）**忽必烈两次南征陈朝越南——皆败于越人陈兴道。"白藤江"再次成为越人击败北来大军的传说之地——这一次更深一层。

**明清**　**明永乐五年（1407）**朱棣以胡氏簒陈、安南内乱为由——遣张辅率明军南下——平胡朝，**复置交趾布政司**——直辖凡二十年。**宣德三年（1428）**黎利起兵驱明军——明廷遂废交趾布政司——越南再次独立。明清两代越南奉中原王朝为宗主——清乾隆五十三年（1788）孙士毅伐越南、清光绪十一年（1885）中法越南战争——中越关系自此入近代格局。1831 年阮朝**明命帝**设**河内省**——"河内"之名自此入官府正式文书。**自唐宋平郡至阮河内省凡一千零七十六年**——这片三角洲上的城名号几度更易：宋平 → 安南 → 大罗 → 昇龙 → 东都 → 东京 → 河内。

**民国至今**　19 世纪后半法属印度支那时期，河内为法属总督驻地之一。1945 年越南民主共和国成立，定都河内。1954 年法越停战、1976 年越南统一——河内为统一后越南首都至今。今河内市为越南首都，红河三角洲腹地——人口逾八百万；旧大罗城（黄城）遗址在 2010 年被列入联合国教科文组织世界文化遗产，称"昇龙皇城中心区"。盛唐那座宋平郡城——遗址在今河内市西北一带，具体位置考古尚未确指；今河内市内残存的"昇龙皇城"以李朝-阮朝层为主，**唐安南都护府之层在地层中或仍存**，但实物可指认者寥寥。

唐人称"交趾"的那座城——盛唐辖境最南的一座郡治——如今地名"河内"独存；昇龙皇城遗址下或许还埋着大罗城和那座更早的宋平郡署的夯土——考古至今未到那一层。马援铜柱的实物自始未曾找到——一柱立于汉、传于唐、典于诗，最终只剩在文字里。

——《后汉书·卷二四·马援传》（建武十九年/43 春平征氏姊妹之役）。马援自此立铜柱表汉之南界——铜柱实物已不可考，"铜柱"自唐入诗。

> 安南都护府，开元户二万四千二百三十，口九万九千六百五十二。东至大海三百里。西南至林邑国一千五百里。

——《元和郡县图志·卷三八·岭南道五·安南都护府》。这是 755 年前后安南境内可课于中原州县体制者的户口与方位——李吉甫成书于元和八年（813）。
