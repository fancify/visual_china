# 雅隆

## 一句话

吐蕃赞普世系的发祥地——雅鲁藏布江南岸一道狭谷里的祖陵心脏；755 这一冬，赤德祖赞的新坟正要在琼结山下夯起。

## 形貌

雅隆并非一城，而是一道河谷。

雅鲁藏布江——蕃人自称"藏布"，意为"清净大水"——自西向东横贯藏南。江至山南一段，自南岸接入一支由南向北汇入的支流，蕃人称"雅隆藏布"（Yarlung Tsangpo，又作"雅砻河"），唐人文献作"雅鲁"。河谷自南而北展开约一百二十里，南起穷结（藏文 Chongye，今琼结）、北至雅鲁藏布江汇口（今泽当），是藏南少见的一处冲积熟地。海拔约三千五百米，比逻些（拉萨）的吉曲河谷稍低，**农耕远早于逻些**——藏地最早的青稞田、最早的水磨、最早的牛拉木犁，蕃文史源皆系于雅隆。

谷地有三处核心。

最北是**泽当**（Tsetang，意为"游戏之地"），坐雅隆藏布与雅鲁藏布江汇口南岸。蕃人传聂赤赞普自天而降即在此谷立国，谷东**贡布日山**（Gangpo Ri）山腹一处石窟，是传说中蕃人始祖猕猴与罗刹女所居之处——蕃地"猕猴变人"的创世神话即由此而出。山下河谷东岸缓坡上立**昌珠寺**（藏文 Tradruk）——松赞干布朝（641 后）立，是松赞干布"镇地十二寺"中的一处，藏地最早的佛寺之一；至 755 年时已逾百年，殿内主供文成公主自长安携去的一尊小像。

泽当之南约二十里，谷东台地上立**雍布拉康**（Yumbulagang，意为"母子宫"）——一座立于尖峰之上的小石塔式宫堡。蕃文史源记其为聂赤赞普所筑——若此说成立，则是西藏地面所存最古之宫，时已逾八百年。755 年时此处仍立——红石窄墙、木梁横架、塔身高不过数丈，与中原任何一座宫殿都不像。塔后山脊一路向南延展，是雅隆河谷的东屏。

最南是**穷结**（敦煌古藏文写本作 Phying-ba，今琼结），距泽当约六十里。穷结河谷较泽当窄，两山逼近——东曰 Mura Ri，西曰 Phying-ba Stag-rtse（"穷结虎峰"，蕃文史源记此处有一座赞普旧居）。河谷中段一片缓坡，自松赞干布卒（约 650）后，便陆续起夯土封土的赞普陵——蕃文史源记此处先后立陵九座，是为**琼结藏王墓**（蕃文 Chongye btsan po'i bang so）。755 年这一冬，第十座陵正要在此地夯起——便是赤德祖赞之陵。

整道河谷的形貌，远不如逻些有"都"的样子——没有大寺、没有市集、没有外郭城墙；倒像一道沿江狭谷里铺开的农耕带，几座旧宫、几处佛寺、几列封土的赞普陵——是蕃人记忆里"祖宗起的地方"。

## 大唐天宝十四载（755 年）

### 唐人眼里的来路

雅隆这一道河谷，在唐人眼里几乎无来路——唐文献于吐蕃只详至逻些与诸大相，至于赞普祖陵所在的山南雅隆，《旧唐书》《新唐书》皆未单著。蕃地之事入唐史，是自松赞干布贞观八年（634）遣使始；而松赞干布之前，蕃在唐人眼里只是"羌之别种"。

但在蕃人自记里，雅隆是一切的起点。

敦煌古藏文卷子 P.T.1287《赞普传记》开篇即记："从天而降之神子聂赤赞普，立国于雅砻泽当贡布日山之南"——蕃人创世，自此地起。聂赤赞普以下，**雅隆悉补野部**（Yarlung Pugyel）历四十余代赞普，皆居山南雅隆。**至第三十二代（敦煌写本作三十二，藏地后世史书或作三十三）松赞干布约贞观七年（633）徙都逻些**——此前蕃国都城近千年皆在雅隆穷结一带。徙都之后，雅隆并未弃——它成了**祖陵之地与宗教礼拜的心脏**：赞普卒，皆归葬穷结；松赞干布朝立的几座"镇地寺"——昌珠、雍布拉康——也都仍在山南。

蕃人的政治中心在逻些，但蕃人的祖宗在雅隆——这一格局，自松赞干布迁都之后百余年未变。

### 此刻

天宝十四载（755）的雅隆河谷，是一处正待为新坟动土的地方。

第三十六任赞普**赤德祖赞**（藏文 Tridé Tsuktsen，唐文献作"弃隶蹜赞"）于天宝十三载（**754**）卒，**次年（755）葬于琼结藏王陵**——藏文史源记其卒因为臣下毒害（《贤者喜宴》），唐文献不言。

> **史源注**：卒年记载略有歧出——《旧唐书·吐蕃传上》系于天宝十三载（754），《新唐书·吐蕃传上》系于天宝十四载（755）；敦煌 P.T.1288《大事纪年》记其卒于"狗年"，王尧、陈践译本系为公元 755。本文主线取**754 卒 / 755 葬**之口径——蕃地葬制，赞普卒后需经数月乃至年余方夯陵下葬，754 卒、755 葬于体例相合。

**755 这一冬，赤德祖赞的陵正在穷结河谷中段封土**：选址自陵列东端起，封土向南延，规模未及松赞干布陵之大，但仍是雅隆百余年间起夯的第十座赞普陵。

旧赞普未远，新赞普幼冲。

**赤松德赞**（藏文 Trisong Detsen，唐文献作"乞黎苏笼猎赞"）生于公元 742 年（敦煌 P.T.1287 作牛年生），父丧时年仅十三四岁。蕃文史源记国政由数位大相共持，最重一位是**玛祥仲巴杰**（藏文 Mashang Drompa Kye）——本教（蕃地原信仰，藏文 Bön）出身，权倾朝野，主"抑佛崇本"。玛祥执政之事，敦煌 P.T.1288《大事纪年》直载："狗年，赞普父崩；玛祥仲巴杰摄政"——是 755-756 蕃地国政实情。雅隆河谷此时的诸赞普祖陵旁，仍是本教祭祀的场所——血祭、牛羊牲献，并无后来佛门的影子。**距赤松德赞翦除玛祥、迎莲花生入藏、桑耶寺落成（779）始有规模化的吐蕃僧团，要再等二十四年**——那时雅隆将由"祖陵之地"渐让位给桑耶（在雅鲁藏布江北岸），但 755 这一年的雅隆，仍是蕃地最重的祖宗中枢。

至于唐——

雅隆距长安直线约三千二百里，比逻些更南、更深入山地。唐使入蕃，正使皆止于逻些觐赞普；雅隆之祖陵与诸旧宫，唐使罕至。755 年这一冬的雅隆河谷里——穷结新坟的夯土声起，昌珠寺主殿香烟未断，雍布拉康塔顶旗幡在高原风里翻飞——他们与长安隔着青藏高原东缘的崇山、河西走廊、关中渭水，不知安禄山十一月初九已在范阳起兵；也不知再过八年，吐蕃军会自这一道雅隆走出，攻破长安。**这一冬的雅隆，尚是一座纯然属于蕃人自记的祖陵河谷**。

## 谷内地标（doc 内子条目，不另立 POI）

下列各项皆为雅隆河谷内地标，作 doc 内独立 `###` 章节锚定（Obsidian 全文可搜）；不另立 `.md` 文件。

### 琼结藏王墓（穷结陵区）

穷结河谷中段一片缓坡，自松赞干布卒（约 650）后陆续起的赞普陵列。蕃文史源（《贤者喜宴》《王统世系明鉴》）与今地面遗存共记**九座大型封土陵**——其中可对应至赞普者，最重为**松赞干布陵**（藏文 bang so dmar po，意为"红陵"，居陵列中央偏北，封土方形夯土，边长约一百三十米）、芒松芒赞陵、都松芒波杰陵、赤德祖赞陵。陵前立**石狮、石碑、神道**——其中松赞干布陵前石狮为吐蕃石刻之最早一对，至今犹存（霍巍《西藏古代墓葬制度史》考定）。755 年这一冬，赤德祖赞之陵正在动土夯起；陵旁本教祭师主祭，血祭牛羊。

### 雍布拉康（藏文 Yumbulagang）

泽当南二十里，立于一处尖峰之顶。蕃文史源记其为聂赤赞普所筑，是蕃地"地面所存最古之宫"——若此说成立，时已逾八百年。塔身红石窄墙、三层木梁顶，高不过数丈——既非中原意义上的宫，也非寺。755 年时仍作蕃地早期赞普神迹之地受礼拜，塔下立祭石数枚。

### 昌珠寺（藏文 Tradruk）

泽当东南河谷东岸缓坡。松赞干布朝（约 641 后）立——是松赞干布"镇地十二寺"之一，藏地最早的佛寺之一；蕃文史源记其压一龙池而立。755 年时殿内主供文成公主自长安携去的一尊小像（藏文称 Chöje Mikyö Dorje），主殿木构梁柱多为 7 世纪原件——后世藏地考古辨识与逻些祖拉康主殿同期。755 年时寺规模不大，僧众数十——蕃地僧团尚未规模化，昌珠仍是宫廷家寺性质。

### 贡布日山（藏文 Gangpo Ri）

泽当东侧一座孤峰。蕃文创世神话载此山腹一处石窟为蕃人始祖**猕猴**与**罗刹女**结合之地——蕃人之祖由此而出。山上多本教祭石，山腰有早期赞普祭天所。755 年时仍是雅隆河谷一处重要的本教神山。

### 泽当（藏文 Tsetang，"游戏之地"）

雅隆藏布与雅鲁藏布江汇口南岸缓地。蕃文史源记聂赤赞普自天而降即立于此处——其后近千年雅隆悉补野部赞普之"夏宫"或聚会之地。755 年时是河谷北端的一处聚落，规模不大——几座祖宫遗址、一座佛寺（昌珠）、几处贵族府第；不是"城"，更像一处礼仪节点。

### 穷结河谷农耕带

穷结河两岸狭长冲积带。蕃地最早的青稞田、最早的水磨、最早的牛拉木犁——蕃文史源皆系于此。755 年时此一带仍是雅隆河谷农耕之心——比逻些吉曲河谷的农耕熟得多。

## 历史长河（755 → 今）

**中晚唐**　755 年赤德祖赞葬于穷结、赤松德赞继位——之后四十余年蕃地达极盛，但雅隆作为政治中枢的角色不再回返。德宗朝（780s）吐蕃东扩，沙州、瓜州、河西陇右尽入蕃手；雅隆河谷的祖陵自赤松德赞一代起，新陵规模渐增——赤松德赞陵、牟尼赞普陵、赤德松赞陵、赤祖德赞（热巴坚）陵相继立于穷结陵列。会昌二年（842）末代赞普朗达玛被刺，蕃地大乱——雅隆陵区随之荒废，部分陵被盗掘。**蕃地"分裂时代"自此开始**——雅隆河谷散为数支地方势力分据。

**宋元明**　北宋至明，雅隆河谷虽不再是政治心脏，仍是藏地"乃东"势力的根据。**14 世纪帕木竹巴政权据藏地**——以山南乃东（在泽当近郊）为中心，重启雅隆作为政治节点的地位约百年。但其后格鲁派兴起、雪域中心北移至逻些与日喀则，雅隆再退为礼拜与农业之地。穷结陵区在元明间多被本地居民开垦或盗掘——封土多平、神道散佚；至清初地面所见已远非吐蕃盛期之貌。

**清至民国**　雅隆河谷诸地（泽当、琼结、雍布拉康、昌珠寺）皆归西藏地方政府辖。1959 年改设西藏自治区筹委会后，山南专区立治泽当——雅隆河谷再次成为地区中心，但已与吐蕃赞普之政事无关。1980 年代以来，琼结藏王墓陵区作大规模文物调查与保护——松赞干布陵、芒松芒赞陵、赤松德赞陵等可对应陵位陆续确认；雍布拉康、昌珠寺亦作修缮。1996 年琼结藏王墓列入全国重点文物保护单位。

**今**　雅隆河谷已是西藏自治区山南市辖境——泽当为山南市治，琼结为县治。穷结河谷中段那一列封土陵，是吐蕃赞普世系在地面留存最完整的物证。**自赤德祖赞 755 葬此，至今一千二百七十年**——封土虽平、神道虽散，但雅隆作为"蕃人祖宗起的地方"，这一身份未改。
