# 庐州

## 一句话

淮南道巢湖北岸一州，治合肥县——742-758 间官称庐江郡；曹魏旧合肥城、张辽逍遥津之地，唐人言"合肥、寿春为中路"的合肥便是此城。

## 形貌

庐州城在巢湖北岸约三十里的一片缓岗上——南瞰巢湖，北望江淮分水岭。城所在合肥县——县治、州治同城。州治合肥县城周不大，**淝水**自西北肥西诸山东南来，绕城北、东两面入巢湖；这条小水便是城以"合肥"得名的根脚——汉应劭《地理风俗记》"夏水暴长，施水会于此",今淝、施二水合于城北。

合肥的地势是**江淮中分**的腰点：北过寿州境二百里入淮水，南渡巢湖、过和州境入长江——盛唐人言"江淮人走崤函，合肥、寿春为中路"，便是说**自江南北上，过合肥则趋寿春入淮，过寿春则趋汴洛入关**，与东路扬楚漕路、西路荆襄陆路并行。城东南二十里巢湖一片烟波——东西长八十里、南北宽四十里，盛唐已是淮南最大的湖；湖鱼、湖菱、湖蒲，州贡见名。物候上属江淮过渡——春迟、夏溽、秋早、冬偶薄雪。

至于**淝水之战**——盛唐人若提"淝水"二字，所指多非合肥城下这一段，而是**寿州西**那一段（即淝水北流注淮处）。东晋太元八年（383）秦晋淝水之战古战场在寿州硖石山一带、非合肥——**两座城都临淝水，但战在上游不在下游**。这一节本 doc 不与寿州混。

## 大唐天宝十四载（755年）

庐州沿革自隋唐定型。**隋开皇元年（581）置庐州**，治合肥县；大业三年（607）改置**庐江郡**——这是"庐江郡"作为正式郡名的隋制根脚。唐**武德三年（620）复置庐州**，领合肥、庐江、慎三县；**武德七年（624）废巢州**，巢县来属——自此庐州得领四县。**天宝元年（742）**玄宗诏天下州一律改郡，庐州改称**庐江郡**——这是 755 年当年的官称；**乾元元年（758）**肃宗复改郡为州，**庐江郡复称庐州，自中州升为上州**——这一升等便发生在 755 年安史乱起之后的三年里，唐廷因淮南漕路战略加重而提升此一州的等第。

户口上，**《旧唐书·卷四十·地理志三·淮南道·庐州》载："旧领县四，户五千三百五十八，口二万七千五百一十三。天宝领县五，户四万三千三百二十三，口二十万五千三百九十六。"**——一百二十年间户数增至约八倍、口数增至约七倍半，是盛唐淮南腹地开发的硬证。**天宝领五县**：合肥、慎、巢、庐江、舒城；本州未置都督府、未列都督府都领，按唐州等第为**中州**（按 758 年方升上州）；地理上，《旧唐书》同条载"**在京师东南二千三百八十七里，至东都一千五百六十九里**"——这是 755 年长安、洛阳两京到合肥的官道里数。

行政上，755 年庐州属**淮南道庐江郡**——淮南节度使治扬州，辖扬、楚、滁、和、舒、庐、寿、光、蕲、安、黄等十余州；庐州位于淮南节度辖境之中，扼巢湖北岸、合肥地利。城内主事为庐江郡太守（按 742-758 改州为郡间，州刺史一律改称太守）、郡丞、司马、司户参军、司仓参军——一套中州官员系统。本州军事不属节度大镇，州兵由州自办，受淮南采访使节制。

历史回望——合肥人若在 755 年驻足城外，所见仍是淝水东流入巢湖那同一片水道。**东汉末三国时合肥已是孙吴北线、曹魏南线的咽喉**：建安五年（200）曹操初据中原后即遣刘馥为扬州刺史，**馥单骑赴合肥空城、修立州治、聚诸生、招军屯垦**——这是"曹魏合肥城"的起点。**建安二十年（215）**孙权率十万众围合肥，魏将**张辽**以八百死士夜冒孙权军——"披甲持戟，先登陷阵"——**孙权军披靡，权乘骏马上津桥乃免**——这便是后世传为"**张辽止啼**"的逍遥津之战。**逍遥津**在合肥城东郊（今合肥市逍遥津公园一带），盛唐人来此凭吊犹是寻常事；至于城内**教弩台**（俗称曹操点将台，相传为曹魏屯兵教射处），盛唐时已是城中一座旧土台，州人指认大致方位——是否曹魏原物则不可确考。

755 年冬安禄山反范阳。消息沿驿道南传——过陈州、颍州、寿州，**抵庐州约二十余日**。庐江郡太守接到的诏令大约是：戒严、备粮、严防淝水巢湖一线——次年（756）玄宗西奔入蜀、肃宗北赴灵武即位，淮南诸州的漕舟、稻米、铁器一时成为朝廷半边天的物资来源。庐州虽未亲历战火，**乾元元年（758）庐江郡复称庐州、自中州升为上州的等第调整，便是这一段战时财政战略的官方注脚**。

## 历史长河

中晚唐庐州因控合肥渡而成为江淮中段陆路腰点——**元和、长庆间**淮南漕运吃紧，庐州州治受朝廷重视。**唐末黄巢之乱（875-884）**波及淮南，庐州亦在其中；天复二年（902）杨行密据淮南立吴，庐州入吴版图——杨行密本人即合肥人，**自合肥起家、以淮南立国**——这是合肥本城出身的人在中古史上第一次执掌一方政权。

**五代两宋**——庐州先属杨吴、后属南唐；**北宋开宝八年（975）**南唐平后归宋。宋庐州属淮南西路，**南宋绍兴和议**后北宋"淮南西路"析为庐州、舒州、和州诸路；庐州因控江淮中道为南宋沿淮防线咽喉。**北宋仁宗朝**庐州出了一位本城人——**包拯（999-1062，字希仁）**——天圣五年（1027）进士、累官知开封府、御史中丞、枢密副使，谥孝肃；后世戏曲小说"包青天"诸事即附会其上。**注：包拯属北宋而非盛唐，与本 doc 所写 755 时庐州相距二百四十余年；盛唐合肥人不知三百年后此城会出此一人**。

**明清民国**——明洪武初置庐州府，下辖合肥、庐江、舒城、巢、无为五州县；清雍正二年（1724）庐州府属安徽布政司；**民国元年（1912）废庐州府，存合肥县**——"庐州"二字作为正式行政名至此罢，让位于一个单字县名。1949 年后合肥县先属皖北行署区，**1952 年安徽省立、定省会于合肥**——自此合肥由一座古庐州治所升为全省政治中心。

**1949 年至今**——合肥市今为安徽省省会，人口逾九百万，城区已远逾盛唐庐州城周；唐合肥县城具体位置在今合肥老城区一带，**城圈遗迹地表无存**，但**逍遥津**与**教弩台**两处地名相承一千八百年——**今合肥逍遥津公园即唐时州人指认的逍遥津故地**，公园中尚立张辽衣冠冢与"威震逍遥津"碑（碑系民国后所立）；教弩台今在合肥城内明教寺旁，相承的台基与三国遗存的对应关系考古界仍有讨论。**至于巢湖**——盛唐"东西八十里"那片烟波至今仍在，是中国五大淡水湖之一；二〇一一年原巢湖市废并入合肥，**唐天宝庐江郡所辖巢县至此千余年后又回到合肥版图**。

——《旧唐书·卷四十·地理志三·淮南道·庐州》

> 太祖既征孙权还，使辽与乐进、李典等将七千馀人屯合肥。……权率十万众围合肥，辽夜募敢从之士，得八百人，椎牛飨将士，明日大战。平旦，辽被甲持戟，先登陷阵，杀数十人，斩二将……权人马皆披靡。权乘骏马上津桥，桥南已见彻，丈馀无版……辽复还突围，拔出馀众。

——《三国志·卷十七·张辽传》取节录——逍遥津之战，建安二十年（215）八月，在合肥城东郊；至盛唐已五百四十年。
