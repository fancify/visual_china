# 郴州

## 一句话

江南西道最南一上州，治郴县（今湖南郴州城区），耒水上游谷地一座小城——城南即五岭北麓，骑田岭一道山口通往岭南韶州、连州，是大唐江南西道与岭南五府经略接界的山间隘口之一。

## 大唐天宝十四载（755年）

郴州治**郴县**，在耒水上游一处山间谷地，州境正南便是**五岭北麓**——西有骑田岭，东有大庾岭，骑田岭一道山口便是江南西道南越五岭进岭南最便捷的陆路通道之一。自潭州、衡州沿湘水南下，至衡州转耒水东南上行，过耒阳便入郴州；自郴州城南行越骑田岭，南面便是岭南道韶州、连州——这是盛唐人下岭南的一条次要通道（主道在江南西道东侧自洪州、虔州下大庾岭至韶州；郴州道是西侧一支）。沿革清楚——**隋为桂阳郡**，**唐武德四年（621）平萧铣置郴州**，初领六县；**天宝元年（742）**玄宗诏改州为郡——郴州改称**桂阳郡**，755 时城门匾额上写的便是这三字；**乾元元年（758）**肃宗复改郡为州——但这是 755 之后的事。《新唐书·卷四一·地理志五·江南西道》列郴州为**上州**，载："郴州桂阳郡，上。土贡：赤钱、纻布、丝布。有桂阳监钱官。户三万三千一百七十五。县八"——领郴、义章、平阳、资兴、高亭、义昌、临武、蓝山八县。**桂阳监钱官**一笔尤要紧——郴州境内古有铜矿（义章县下《新唐书》注"有银、有铜、有铅"），盛唐于此置桂阳监督铸钱，是江南西道少有的一处官营钱监；郴州对长安的分量，泰半在这处铸钱监上。**《旧唐书·卷四十·地理志三·江南西道·郴州》载："旧领县五，户八千六百四十六，口四万九千三百五十五。天宝领县八，户三万一千三百三。在京师东南三千三百里，至东都三千五十七里"**——755 时郴州在册户 31,303（《旧唐书》天宝条原文未给口数；据《新唐书》天宝时期户口为 33,175 户，二书略有出入，盖籍年口径之差），在岭南北缘诸州中属中等，与永州、道州相侔。755 时郴州长官为**桂阳郡太守**（742-758 郡制时期，州皆改郡、长官称太守）——本州兵备甚轻，本地团结兵不过数百，真正可数的事是**桂阳监**的铸钱与**骑田岭**的过岭驿路。

## 历史长河

**中唐**　**贞元二十一年（805）韩愈自阳山量移江陵法曹，途中过郴州待命**——曾留滞郴州近三月，作《同冠峡》《郴口又赠二首》《郴州祈雨》《题郴州龙兴寺》诸诗；这是郴州在唐人诗里最浓的一笔——但已是 755 之后五十年事。**柳宗元元和四年（809）自永州寄诗友刘禹锡多次提及郴道**——永州、郴州、道州一线是中唐贬官诗的湘南串珠。**秦观绍圣四年（1097）贬郴州**作《踏莎行·郴州旅舍》"雾失楼台，月迷津渡"——是郴州在词史上的高光，但这已是宋哲宗朝事。

**五代至明清**　南唐马氏楚国据湖南——郴州入版图，桂阳监仍铸钱。北宋郴州属荆湖南路，**升郴州军**（北宋军州一级）。元属湖广行省郴州路。明洪武初降为郴州直隶州属湖广布政司，清承明制为**郴州直隶州**——直至清末。

**民国至今**　1913 年废州存县称郴县属湖南省。1959 年置郴州市（县级），1995 年撤地区设地级**郴州市**——"郴州"作地级市建制名复活，已是改革开放后事。今郴州市治所在唐郴县治附近，五岭北麓骑田岭一道山口仍在——京广铁路、京港澳高速正循盛唐过岭路线之故道穿岭而过。
