# 泉州

## 一句话

江南东道清源郡治——治**晋江县**（今福建泉州鲤城区），闽中东南山海之州；晋江入海口北岸，长乐郡（福州）西南五百余里，唐时是闽中山海间一处中下小州，远未及宋元后那番"东方第一大港"光景。

## 大唐天宝十四载（755年）

泉州治**晋江县**，在闽中东南、晋江入海口北岸——一处倚山面海的小平原。州属**江南东道**南端，长乐郡（福州）西南五百余里；**天宝元年（742）随玄宗朝"改州为郡"通制改为清源郡**，755 时正名为"泉州清源郡"，两名在唐人嘴里互换不分；**乾元元年（758）肃宗复改为泉州**——这是 755 之后的事。

沿革须细辨——**"泉州"二字在隋唐之际几度移名，前后指两个地方**，本 doc 一刻不可混。**第一阶段（武德至景云间，"泉州" = 今福州）**：唐武德五年（622）于今福州一带置**丰州**——这"丰州"的州治在今福州、不在今泉州；武德八年（625）废丰州，其地并入彼时的**泉州**——而**这个"泉州"州治也在今福州，并非今闽南泉州**（详见福州条）。**第二阶段（景云后，"泉州" = 今闽南泉州）**：今闽南泉州一带，**圣历二年（699）**自前述泉州（即今福州）析南安、莆田、龙溪三县置**武荣州**——治南安县（在今闽南南安），这是今泉州地置州之始；**久视元年（700）武荣州一度复废**，其地仍归泉州（=今福州）；不久又复置；**景云二年（711）改武荣州为泉州**——**"泉州"之名自此南移至闽中东南今地，自此定下来不再移**；与此同步，**原指今福州的旧"泉州"于开元十三年（725）改名福州**——两州名分定，互不相涉。**开元八年（720）**分南安县置**晋江县**，今泉州州治移此——755 时清源郡治晋江县已三十五年。

清源郡领县四：**晋江（附郭）、南安、莆田、仙游**——仙游本名清源县（圣历二年自莆田析置），**天宝元年（742）**改名仙游，正与"清源郡"郡名同年——一郡之名让给郡，旧县名让贤改作"仙游"。**《旧唐书·卷四十·志第二十·地理三·江南东道·泉州清源郡》载："天宝领县四，户二万三千八百六，口十六万二百九十五"**——755 时清源郡在册**户 23,806、口 160,295、领县四**。州等：**旧唐书列"中"，新唐书列"上"**——盛唐州县按户口分上、中、下，泉州约在中州门槛上下，两书差一档；本 doc 以"中下小州"概之，更近实情。比之同道大州（润、苏、杭、越户皆十万级）小一个量级，比邻州长乐郡（福州，户 34,084）亦差近三分之一——**唐时泉州是远比江南腹地、亦远比长乐郡更小的州**，远不是宋元后那座"东方第一大港"。

**晋江**自西北山中出，至晋江县切出一段冲积平原，再东南入海——海口外洋面通往海舶。**《新唐书·卷四一·地理志五》**于泉州条下记一条罕见的远洋航路："自州正东海行二日至高华嶼，又二日至𪓟鼊嶼，又一日至流求国"——这是盛唐官方文献里所记自泉州东渡至流求的最早路程。但此时所谓"泉州港"尚未在《广州通海夷道》等盛唐海贸正记里独立成名；盛唐三大对外港口仍是广州、交州、扬州（或明州）。泉州城下偶有胡舶来——但量与名皆远不及岭南。土贡据《新唐书》载："绵、丝、蕉、葛"——丝绵之物属华夏腹地常贡，蕉葛则是闽中山地纤维织物，与长乐郡（福州）同类。

## 历史长河

**晚唐五代**　**唐昭宗光启二年（886）王潮**自汀州起兵，先克泉州；后克福州——王氏据闽，泉州为其根本之一。**五代闽国（909-945）**——王审知封闽王，泉州为闽国南部重镇。闽国亡后**留从效、陈洪进**据泉、漳，**后周显德二年（955）**置**清源军节度使**治泉州——"清源"二字作军名复活（与唐天宝郡名同字）。**北宋太平兴国三年（978）**陈洪进纳土归宋——泉州入北宋。

**宋元**　北宋泉州属**福建路**，户口大增。**北宋元祐二年（1087）**置**福建路市舶司**于泉州——泉州港自此跃居东南海贸大港。**南宋至元**泉州港号"**刺桐**"（Zayton/Zaitun，阿拉伯波斯商人之译名）——大食、波斯、印度、东南亚、高丽、日本商舶云集；**元至元间**马可波罗记泉州为"光明之城"，与亚历山大港并列为世界两大港。元代泉州设**福建行省**治此一度。这是泉州"东方第一大港"的极盛——但已在 755 之后三百五十年。

**明清**　明洪武初海禁；嘉靖间倭患——泉州港日衰；漳州月港、广州、宁波接替之。清初郑成功据闽海——泉州为反清据点之一；康熙二十二年（1683）台湾内附后泉州人大规模渡台垦殖——闽南语、闽南文化经泉州扩布台湾及东南亚。

**民国至今**　1913 年废府置县。今**福建省泉州市**——闽南三市（泉、漳、厦）之一；旧晋江县城在今鲤城区；宋元海贸遗迹（开元寺、清净寺、伊斯兰圣墓、洛阳桥等）甚多，2021 年"泉州：宋元中国的世界海洋商贸中心"列入世界文化遗产——但这一切的世遗叙事，是宋元层，不是盛唐 755 这一刻。

## 引文

> 泉州清源郡，中。隋建安郡。武德五年置丰州，领南安一县。八年废丰州，南安属泉州。圣历二年，分泉州之南安、莆田、龙溪三县置武荣州。久视元年又以三县置武荣州。景云二年改为泉州。天宝元年改为清源郡。乾元元年复为泉州。天宝领县四，户二万三千八百六，口十六万二百九十五。

——《旧唐书·卷四十·志第二十·地理三·江南东道·泉州清源郡》（户口数与沿革 verbatim）。**沿革读法须注意**：引文"武德八年废丰州，南安属泉州"中的"泉州"指彼时州治在今福州的旧泉州，非本州；至"景云二年改为泉州"一句，"泉州"之名方南移至本州（今闽南泉州）。"泉州"二字 755 年时入官府文书已四十四年（自景云二年/711 改武荣州为泉州起，指今地）——比起兄长福州（开元十三年/725 始名福州）尚早十四年，但其州体远小。

> 泉州清源郡，上。土贡：绵、丝、蕉、葛。户二万三千八百六，口十六万二百九十五。自州正东海行二日至高华嶼，又二日至𪓟鼊嶼，又一日至流求国。

——《新唐书·卷四一·地理志五·江南道·泉州清源郡》（节录 verbatim）。州等"上"与旧唐书"中"略异——盛唐州县州等两书互有出入，泉州为典型一例；土贡"绵丝蕉葛"四物，"绵丝"属华夏腹地常贡、"蕉葛"为闽中山地纤维织物；末一句"自州东海行至流求国"的航路是盛唐官方文献里所记泉州东渡流求的最早记录——但此时泉州尚不是真正意义的海贸大港。
