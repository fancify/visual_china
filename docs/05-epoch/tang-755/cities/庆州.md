# 庆州

## 一句话

关内道北部一州，治安化县（今甘肃庆阳）——长安正北约六百唐里，沿子午岭东麓驿道北上灵州、朔方的中段一站。

> **disambig**　此为关内道庆州（陇东庆阳）。同名/近名异地有二，须分清：一指**新罗都金城**（高丽 940 改称庆州，今韩国庆州市，佛国寺所在地），与本州同字异地，详见 [金城](./金城.md)；二指**兰州金城郡**——唐兰州古郡名"金城郡"，与陇东庆州及新罗金城同涉"金/庆"字，地各不同，详见 [兰州](./兰州.md)。北周大象二年（580）所置庆州治合水，为本州前身沿革，非另一同名，详见下文"大唐天宝十四载"段。

## 形貌

庆州城在陇东黄土高原上。子午岭自南向北分关中盆地与陇东塬区——岭东是鄜延、岭西便是庆州、泾原一带。城在马莲河（古名马岭水）与柔远河交汇处的小盆地里——州治安化县城周不大，城外是一片片黄土塬：南有董志塬、北有早胜塬、东有大塬，塬面平坦适农、塬间深沟切割——所谓"塬、墚、峁"的陇东典型地貌，唐人虽无此辞，所见所行已是此景。城东南去长安沿马岭水南下，过宁州、邠州，约六百唐里至京；城正北经马岭、华池一线，循子午岭东麓北上，至盐州、灵州、朔方节度治所。**庆州是关内道沿子午岭走廊北段的州治枢纽——长安通灵武的官驿道在此过境**。

物候是陇东塬区的：春旱多风、夏热雨集中于七八月、秋早霜、冬干寒——年降水四百毫米上下，仅可旱作粟麦，无水稻。城外塬面种粟黍麦豆，沟谷间稀疏河柳。冬季马莲河有冻封段（约 12 月至次年 2 月）。**这一片是盛唐关内道与陇右道之间一段半农半牧的边缘——南向农耕、北向接草原**——朔方党项、吐谷浑余部、回纥诸部出入于此一线。

## 大唐天宝十四载（755年）

唐人眼里庆州不算古城——西魏大统十六年（550）尝置朔州于此（与塞北朔州同名异地），北周大象二年（580）改置**庆州**，治合水县；隋大业初（约 607）废庆州省入弘化郡。**唐武德元年（618）复置庆州**，治安化县——"庆州"之名自此立。**天宝元年（742）**玄宗诏改州为郡，庆州改称**安化郡**；**至德元载（756）**因避安禄山"安"字讳，改安化郡为**顺化郡**（《新唐书·地理志一》载此异名；《旧唐书》仅记 742 改安化、758 复庆州，未及顺化）；**乾元元年（758）**肃宗复改郡为州——庆州之名再立。**742 至 758 这十七年间，庆州官称先后为"安化郡"（742-756）、"顺化郡"（756-758）**，民间口语始终呼"庆州"。

755 年的庆州属**关内道**，州治在安化县。据《旧唐书·地理志一》记天宝间庆州约 **2 万户**——大致在中州门槛上下；辖县在玄宗朝凡七八，安化、合水、马岭、华池、同川、洛源、延庆诸县（与晚唐及《新唐书》所记略有出入）。**玄宗朝置庆州都督府**——管沿子午岭一线州兵与驿戍，并节制本州境内的折冲府兵。城内有郡衙、安化县治、驿馆——长安西北出便桥、过咸阳，沿泾水西北、再北循马岭水，过邠州、宁州，至庆州——这是**关内道驿道中"长安—庆州—灵州"一线的中段**，朔方节度使的粮草、军报、轮戍兵卒，多自此过。

天宝十四载冬安禄山反范阳——消息沿驿道传到庆州时，城里粮草、驿马的调度也紧起来。庆州不是节度治——朔方节度使坐镇灵州，庆州是辖境东南的中州，本州无大军。次年（至德元载 756）玄宗西奔幸蜀，太子李亨自马嵬北上、过邠州、再北过庆州、最终抵灵武即位——**庆州是肃宗灵武即位路上必经的一站**（《旧唐书·肃宗本纪》、《资治通鉴》卷二一八）。盛唐庆州人这一冬目送过的，是后来重建大唐的那一行人。

## 历史长河

**中晚唐**　乾元元年（758）复州名为庆州（顺化郡 / 安化郡旧名废）。安史乱后吐蕃北上，广德元年（763）陷河陇——庆州一度紧贴吐蕃、党项前线，唐廷于此屯兵、修城、置盐州行营。建中四年（783）泾原兵变，朱泚据长安、德宗奔奉天——庆州兵自子午岭东麓南下勤王。中晚唐沿边党项部落渐次内徙至庆州、夏州、盐州一带——这是日后**西夏拓跋氏出于此境**的根脚。

**宋元明清**　北宋庆州属永兴军路（后陕西路）——范仲淹庆历元年（1041）知庆州，筑大顺城（在今华池县境）防西夏，"塞下秋来风景异"句作于此地。北宋政和七年（1117）升庆州为**庆阳府**——"庆阳"之名自此始。金、元属巩昌路、奉元路。明洪武二年（1369）庆阳府属陕西布政司；清雍正七年（1729）随陕甘分省归甘肃布政司——**庆阳府辖境延续至清末**，治安化县（今庆城县）。

**民国至今**　1913 年废府存县，安化县因避同名（直隶安化、湖南安化）改名**庆阳县**——盛唐"安化"二字至此终。2002 年撤庆阳地区设庆阳市——治所迁至西峰区（董志塬上），盛唐庆州治所旧址叠在今**庆城县城**（旧庆阳县城）。马莲河仍自塬下流过，子午岭仍在东。**唯一可与盛唐庆州人共看的，是城外的董志塬**——陇东最大一片完整黄土塬，唐时塬面种粟黍、今日种冬麦玉米，地貌一千二百年来未改。

## 引文

> 庆州，安化。中都督府。开元户二万一千一百六十三。乡六十一。元和户八千二百三十三。……州城西北八十里有马岭，岭出马岭水（今马莲河），南流入泾。

——《元和郡县图志·卷三·关内道三·庆州》（取意节录；具体行文以原书为准）
