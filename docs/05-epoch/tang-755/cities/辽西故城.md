# 辽西故城

## 一句话

安东都护府的最后一处寄治，**唐人口中"六都护"东端的那一处**——大凌河西岸医巫闾山东麓的一座边城，治所自平壤一路南徙、西徙、内徙，到天宝二载（743）落在这里。

## 形貌

辽西故城落在医巫闾山东麓、大凌河西岸的一块台地上——今辽宁锦州义县城内。

地势上，这是辽西走廊的中段腰节。**医巫闾山**南北横亘，是华北平原与辽东、辽西的分水脊；大凌河自西北山口出，绕城东南而东南去，注入渤海。城在山东、河西，刚好卡在走廊由北向南折出渤海湾的那道弯口——西去渝关（今山海关一带）约三百里，东南去辽东故城（今辽阳）约四百里，东北循大凌河上溯可入契丹、奚的游牧地。这是一座**腰节城**，不在哪一头，却把哪一头都连着。

唐代的辽西故城周长据后世推度约五里许，夯土城垣，四面开门——但 755 年时安东都护府寄治此城，城内地标主要是**都护衙、副大都护署、僚佐廨、戍兵营房、市集与几座旧寺**。这是一座**军政小城**，不是州府治所——所谓"寄治"二字便点透了它的性格：本来这里只是营州都督府属下的一个戍点（隋曰燕郡，唐曾置渝州、燕郡守捉等名号在此一带——具体县治名号唐前期屡有更易，史志记载有出入），安东都护府治所一路南徙、西徙，到天宝二载（743）借这座小城落脚——城本不为它而立，它却最终在此安置。

城外的物候是辽西的：**四月解冻、五月草青、九月霜白、十一月封河**——大凌河每年都要冻三四个月，封冻期间冰可走车马，这是辽西走廊冬日的天然官道。山中盛产柞木、松木、铁矿——辽东以来的传统兵器产地之一。城内常住人口推测在数千之间——汉人、高句丽遗民、契丹、奚、靺鞨杂处，胡风极重；说官话的人少，说杂胡话的多。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

安东都护府这个名号，在 755 年的唐人嘴里已经是一个"**屡迁屡徙**"的名号了。

**总章元年（668）九月**，唐将李勣、薛仁贵平高句丽——尽收高句丽五部一百七十六城——于其都城平壤立**安东都护府**，以薛仁贵检校安东都护，统辖原高句丽全境与靺鞨诸部。这是安东都护府的初置，治平壤。然而新罗与高句丽遗民的反抗持续不断——**仪凤元年（676）二月**，安东都护府自平壤西徙**辽东故城**（今辽宁辽阳，唐辽东州治、原高句丽辽东城旧基）。**仪凤二年（677）**再徙**新城**（今辽宁抚顺新城子区一带）——前后只隔了一年。

接下来一段史料较稀，治所推测在新城一带维持了约二十年。**圣历元年（698）**前后——契丹李尽忠万岁通天之乱（696-697）甫平、营州都督府正在恢复中——安东都护府再徙至**渔阳**。**渔阳具体地址史家异说**：一说今天津蓟州（即渔阳郡治所）、一说在范阳幽州东北一带——《新唐书·地理志》与《唐会要》记载略有出入；这一段史料较稀，今学界亦无定说。**开元二年（714）**安东都护府自渔阳一带再徙**平州**（今河北卢龙）——平州是范阳节度使治下州、营州一线的退守点。**天宝二载（743）**——便是 755 年这一刻往前推十二年——安东都护府自平州东徙**辽西故城**：自平壤算起的第六处治所，自此安顿到 755 年这一刻。

唐人若在 755 年驻足回望，**自高句丽亡国至今八十七年**（668-755）——安东都护府的治所在这八十七年间走了一条"**自东向西、自南向北、再东折回辽西**"的折线，平壤、辽东、新城、渔阳、平州、辽西故城，前后六次南徙、西徙、内徙、再外迁——**这是大唐东北边境的真实写照**：高句丽是平了，但辽东再没有真正回到中原州县序列；安东都护府这一名号便随着边境的退缩一路西移。

### 此刻

天宝十四载（755）的安东都护府，**治所在辽西故城已逾十二年**。

行政上，安东都护府属**河北道**——唐玄宗开元二十一年（733）分天下十五道时安东都护府归河北道，但实际军政受**营州（平卢节度使）**节制——平卢节度使治营州（今辽宁朝阳），辖营、平两州及安东都护府，统府兵、戍兵、羁縻府州——**安东都护府是平卢节度使下的最东一支**。755 年时平卢节度使正是**安禄山**——他自天宝元年（742）兼平卢节度，至此已十三年；范阳节度使治幽州、平卢节度使治营州、河东节度使治太原——**三镇皆在他一人手中**。安东都护府这位边境都护使、副大都护、长史、司马诸员的任免，名义上来自长安朝廷，实际上多走范阳节度使府报。

755 年的安东都护府辖境，已与初置时的"**统高句丽五部一百七十六城**"不可同日而语。东边的**新罗**早已取了原高句丽南境（自唐高宗末至武则天朝一路北扩，今平壤一线已属新罗北界）；东北的**渤海国**自圣历元年（698）大祚荣立国、先天二年（713）入唐册封、至 755 年文王大钦茂正自中京迁往上京龙泉府——渤海是名义上唐藩属、实际上自治的政权，安东都护府不能干涉。**实际辖境**只剩辽东、辽西一带的羁縻府州——靺鞨、契丹、奚诸部的酋长在此领唐廷颁的"刺史""都督"印号，每岁朝贡，但内政自管。安东都护府的功能也从初置时的"军事征服 + 行政统治"退缩为"**羁縻怀柔 + 朝贡通使**"——是一个名号大、实权小的边境衙署。

755 年这一冬，辽西故城里的安东都护——史志未载具体姓名，《新唐书·地理志》此处缺名——是一位三品至四品的都护使。他案头的事务：契丹某部酋长换代、奚某部入秋朝贡、渤海过境使节文牒、新罗边界小冲突——这些是辽西故城都护衙的日常。十一月初九范阳那一头消息传来——安禄山起兵——**安东都护府的反应史料极简**：旧唐书与资治通鉴均未明确记载安东都护府在 755 年冬的立场。学界一般推度，安东都护府作为平卢节度使府属辖，**事实上随平卢一并归安禄山部署**——至德二载（757）史思明降唐时辽西一线方有所反复。

对盛唐而言，辽西故城是**大唐东北边境的最后一处都护治所**——再东出便是高句丽旧地、新罗北界、渤海西陲，俱不在大唐辖境之中。**唐人讲"六都护"——安西、北庭、安东、安南、单于、安北——而安东是这疆域东北的一角**；安东都护府在六都护里地位最弱、辖境最虚，但名号还在，便是大唐对东北边境的最后一笔象征。

至于这座城在 755 年的命运——再过三年（758）安东都护府便要再徙——这一段属"历史长河"段。

## 子条目

### 安东都护衙

辽西故城内最重要的官署——大都护、副大都护、长史、司马、录事参军一套完整僚佐。建制依《唐六典》卷三十"安东大都护府"条：都护一员（正三品；中宗朝降为大都护一员从二品、副大都护二员正四品上），副大都护二员，长史一员，司马一员——但 755 年时实际员额据史志多有缺空，许多职位长期不补。都护衙位置据后世推度，应在辽西故城北部高地——但因唐代辽西故城遗址未经系统发掘，**具体方位至今未定**。

### 戍兵营房

安东都护府辖戍兵——按唐前期边军建制，安东都护府额兵不详，但据《通典·兵典》与《新唐书·兵志》推度，玄宗朝安东都护府常驻兵力大约在数千之数——多为戍兵（**镇兵 + 防丁**）与羁縻部族派出的少量蕃兵混编。营房在城内东南角，靠近大凌河取水方便。

### 羁縻府州印信库

安东都护府辖**靺鞨、契丹、奚、室韦**诸部羁縻府州——府州酋长得唐廷颁"都督"、"刺史"印号；每岁朝贡时呈印、领赏。府州印信副册存于都护衙——这是辽西故城都护衙的核心文书工作。**渤海国**先天二年（713）以来虽亦"入藩"，但因渤海自治程度高、且自圣历后渐成大国，渤海事务多由长安鸿胪寺直接对接，安东都护府仅作中转。

### 高句丽遗民坊

辽西故城内有一片**高句丽遗民聚居**的坊区——自高句丽亡国（668）后，唐廷将部分遗民徙至辽东、辽西一带（一说徙至中原内地者另有数十万）——历八十七年至 755 年，遗民已传三四代。坊里仍有高句丽人的衣冠、丧葬习俗与汉化掺杂——他们或为戍兵、或为工匠、或为驿吏。安东都护府的日常运转，相当一部分系于这一群人。

### 大凌河渡

辽西故城东南门外，大凌河渡口——是辽西走廊南北向官道的咽喉。冬季封河时冰渡，春秋平水时舟渡，夏季涨水时绕道上游渡口。河西渡过便入医巫闾山东麓道，可一路南下入幽州境；河东渡过则东北可达辽东故城。安东都护府的兵卒、文书、使节往返，皆走此渡。

### 医巫闾山

辽西故城西侧主峰——**医巫闾山**自隋唐起便是北镇之山。隋开皇十四年（594）封北镇医巫闾山为"广宁公"，唐玄宗开元间例祀北镇——州县每岁春秋两祭。山中有北镇庙（唐时祭仪场所，**位置在医巫闾山南麓，今北镇市境内**——距辽西故城约九十里），是这一带最大的国祀礼场。山中并产铁矿、林木——辽西的兵器铸造与营建用材，半出此山。

## 历史长河（755 → 今）

**中晚唐**（758-907）

至德三载 / 乾元元年（758），安东都护府自辽西故城再徙——史载或徙至幽州境，或徙至平州——具体地点学者意见分歧。安史之乱后期辽西一带屡为战场，安东都护府的实际职能渐废。**贞元年间（785-805）**前后安东都护府名号尚在档案，但已无治所——府名实亡。

辽西故城本身在中晚唐渐成营州都督府辖下的一处戍点。**咸通年间（860-873）**前后大凌河流域奚、契丹诸部势力大涨，辽西故城与营州、平州一线渐脱中原统治。

**五代至辽**（907-1125）

**契丹**（后改国号"辽"）自唐末崛起。**916 年**耶律阿保机立辽，其后取**渤海**（926）、夺幽州（936 后晋石敬瑭割让燕云十六州）——辽西故城自此入辽境。**辽设宜州**——治在唐辽西故城旧址，属上京道。这是"宜州"地名的起点——自此一千年间这座城便以"宜州"为正名。

**辽开泰九年（1020）**前后，辽圣宗于宜州城内立**奉国寺**——一座规模宏大的辽代佛寺，正殿七间，内塑过去七佛（迦叶、毗婆尸、尸弃、毗舍婆、拘留孙、拘那含、释迦牟尼）泥塑彩像——这是辽代佛教雕塑的代表作。**奉国寺至今犹存**——是中国现存辽代单体木构最大的一处。但 755 年时奉国寺尚未立——这是**唐治所旧址在辽代翻出的新一层**。

**金元**（1125-1368）

金灭辽（1125）后宜州沿置；**金元间宜州一度改名义州**——"义"字始入这座城的地名。元代义州属辽阳行省。

**明清**（1368-1911）

明洪武年间废义州为义州卫——为辽东都司辖下边卫。清入关后改义州卫为义州；清雍正十二年（1734）裁府改州——直至民国二年（1913）依"州改县"全国通例，义州改为**义县**——"义县"二字始为正式地名。这便是今"辽宁锦州义县"的"义县"由来——**清后名**，不是唐时本名；唐时此地或称燕郡、或称渝州、或归营州都督府属——具体县治名号史志记载有出入，本 doc 用"辽西故城"作 epoch 主名。

**民国至今**（1911—）

义县今属辽宁锦州市辖。城内**奉国寺大雄殿**1961 年首批列入全国重点文物保护单位——是义县最重要的地面文物。**唐安东都护府辽西故城遗址**——按地理方位推度应在今义县老城区一带，但唐代城垣**至今未经系统考古发掘**——所谓"辽西故城"作为唐代具体城址的边界、规模、布局，许多细节仍属推度。考古上能看到的，主要是辽宜州城遗址的若干夯土残段、明清义州城的城墙基址、以及奉国寺这一座辽代地面建筑——**唐安东都护府那一层，至今没真正翻出来**。

唐人口中"六都护"——安西、北庭、安东、安南、单于、安北——这六个边角名号里，安东最早衰、最早废、最少被诗人写。盛唐边塞诗里"凉州""陇右""安西"满纸都是；唐玄宗朝边塞诗里写到辽东、辽西、安东都护府的，**统共数得过来**。后人读边塞诗，记得王翰、王之涣、岑参的西陲——记不得辽西故城。但 755 年这一刻，它确实在那里——医巫闾山东、大凌河西、一座小城里坐着一位都护——管着一片名义上属于大唐的东北边塞。

## 引文

《资治通鉴·卷二〇一·唐纪十七》总章元年（668）九月：

> 上以高丽既平，分其地置都督府九、州四十二、县一百，置安东都护府于平壤以统之，擢其酋帅有功者授都督、刺史，令各仍其旧俗。

——安东都护府置于平壤之始。

《资治通鉴·卷二〇二·唐纪十八》仪凤元年（676）二月：

> 徙安东都护府于辽东故城。

——自平壤西徙辽东故城（今辽阳），系**安东都护府第一次徙治**。

《新唐书·卷四三下·地理志七下·羁縻州·安东都护府》：

> 总章元年置于平壤，仪凤元年徙辽东，明年又徙新城，圣历元年徙渔阳，开元二年徙平州，天宝二载徙辽西故城。

——一段记尽 668-743 安东都护府六次徙治。（按：渔阳具体地址《新唐书》此处简记，《唐会要·卷七三》亦载，但**今学者对"渔阳"地址有今天津蓟州说、河北蓟县东南说**——史源略有出入。）

## 相关 POI

- [幽州](./幽州.md) — 范阳节度使治所；安东都护府名义上归河北道，实际军政受平卢（治营州）节制，平卢与范阳天宝间同在安禄山一人手中
- [上京龙泉府](./上京龙泉府.md) — 渤海国新都；自辽西故城东北循大凌河上溯过黑水靺鞨地，便是渤海地——755 年时渤海与安东都护府以名义朝贡相属、实际自治
- [金城](./金城.md) — 新罗都城；755 年时新罗已据高句丽南境、自唐高宗末北扩到平壤一线，与安东都护府辖境隔大同江为界
- [太原](./太原.md) — 河东节度使治所；与范阳、平卢同在安禄山一人手中——三镇并兼是 755 年北方边境的最大异象
- [北庭大都护](./北庭大都护.md) — 唐"六都护"之一，与安东都护府对称分立东西
- [安西大都护](./安西大都护.md) — 唐"六都护"之西端，盛唐西陲军政中心；与安东都护府是盛唐边境诸都护中辖境最实与最虚的一对
