# 碎叶

## 一句话

楚河南岸、天山西陲外的一座旧四镇城——曾是安西最远的一支驻军（692-719），719 后唐戍兵撤，三说并存（突骑施 / 唐羁縻挂名 / 西突厥后继）；玄奘 627 西行经此称"素叶水城"，唐人后世又附会作李白出生地——但**李白生于碎叶系一说，并非定论**。

## 形貌

碎叶城在楚河（唐曰"素叶水"）南岸的一片冲积台地上——今吉尔吉斯斯坦楚河州托克马克附近的 **Ak-Beshim 遗址**。

地势上，这是天山山脉西端北侧的一处出山口冲积扇。**南面**是阿拉套山（吉尔吉斯阿拉套山脉）的雪峰，雪线四千米以上；**北面**是楚河谷的草原，楚河自东向西流，最终消失于穆因库姆沙漠（哈萨克斯坦）。城在山北河南之间的台地上——东去伊塞克湖（唐曰"热海"）约二百里，西去怛逻斯（今哈萨克斯坦塔拉兹）约六百里。这是一座**山外的城**——出了葱岭以西、天山以北，便不再属于唐人熟悉的西域地理，但碎叶恰恰是这片"葱岭外"地里唯一被唐人作为四镇驻屯过的点。

唐代碎叶城周长据考古推测约六公里（Ak-Beshim 遗址外城残段），内有夯土与土坯城垣、坊巷、佛寺、市集——是一座**典型的丝路绿洲城**，城内主要居民是粟特商团、突厥诸部、回鹘、葛逻禄、唐汉戍兵、西域佛僧——多语并行，胡风极重。物候是中亚草原的：四月雪解、夏短而凉、九月霜降、十一月封河。城外的楚河每岁封冻三四个月——冬日冰面可走车马，是楚河谷东西向的天然冬道。

## 大唐天宝十四载（755年）

755 年的碎叶，**已经不再是大唐的城——但唐人仍记得它的名字**。

碎叶第一次进入汉地史册，是**玄奘**。贞观元年（627）秋玄奘自凉州偷越出境，过沙州、伊吾、高昌、龟兹，再越凌山（今木扎尔特冰川）西出天山——便至素叶水城。《大唐西域记·卷一》载："**素叶水城，周六七里，诸国商胡杂居也**"——西突厥统叶护可汗时驻于此（玄奘与之相见）。"素叶"二字便是 Suyab 的玄奘音译——后唐人或音转作"碎叶"。

碎叶进入**安西四镇**体系，在**长寿元年（692）**——武则天朝王孝杰复安西四镇之时。原四镇为龟兹、焉耆、于阗、疏勒（贞观二十二年置）；高宗末武则天初一度尽失，长寿元年王孝杰收复，**以碎叶代焉耆**入四镇之列——四镇遂为龟兹、碎叶、于阗、疏勒。这是碎叶军事地位的最高点：作为安西大都护府辖下**最西的一镇**，戍兵直驻天山以北、楚河之南，是大唐边境军事力量的最远投射点。**自 692 至 719，碎叶在四镇序列里凡二十七年**。

**开元七年（719）**——这是碎叶在大唐军事序列里的终点。这一年安西副大都护汤嘉惠以**碎叶距安西治所（龟兹）过远、戍兵供给艰难、突骑施部势力日盛**为由，请求"**以焉耆代碎叶**"——朝廷允之，碎叶遂自四镇除名，焉耆复入。719 后唐戍兵撤；后续控制三说并存（突骑施 / 唐羁縻挂名 / 西突厥后继诸部）。

但**碎叶降为非唐地之后的归属**，史书所记不一：
- 一说碎叶在**突骑施**汗国（开元间苏禄可汗一时强盛）治下——这是开元至天宝间的主流说法；
- 一说碎叶名义上仍属唐**羁縻**——大唐在西突厥旧地置都督府十六、刺史府七十二，碎叶在档案上或仍挂名一府；
- 一说碎叶在**西突厥后继**诸部（突骑施、葛逻禄、处月、处密）之间几度易手——开元二十八年（740）苏禄死后突骑施部分裂，碎叶一度归葛逻禄、一度归处月——具体年份史源异文。

**三说并存，本 doc 不强择一**。755 年这一刻的碎叶，比较稳妥的判断是：城仍在、人仍住、商道仍通——但已不在大唐戍兵的辖境里。**怛逻斯之战**（天宝十载 / 751 高仙芝败于阿拔斯）前后突骑施势力为主（一说）；天宝十四载冬安史乱起时，碎叶在唐人眼里只是一个"**还存名号、不存戍兵**"的旧四镇城。

至于**李白与碎叶**。新唐书李白传载"其先隋末以罪徙西域，神龙初遁还，客巴西"——这是李白祖籍碎叶（或西域）说的史源。**1971 年郭沫若《李白与杜甫》主张李白生于碎叶**——其论据有三：李白祖徙西域、家世经商有粟特商人特征、神龙元年（705）李白父举家东归正与碎叶突骑施势力崛起的时间吻合。但**李白生地另有蜀中江油说**——史源各有所据，至今学界两说并存，**碎叶说是一说，并非定论**。

碎叶进入唐人文学心象，恰恰不靠它的军事或政治分量，而靠这一段说不清的祖籍传闻——以及更早玄奘的那一笔"素叶水城"。**它在 755 年是一座边外的旧城，但在后世唐诗与汉地记忆里是一处文学坐标**。

## 历史长河

天宝以后碎叶名号在汉地史志中渐少出现。**安西四镇陷蕃主在贞元前后（约 791），非安史乱中**——碎叶本已不在四镇，但楚河流域大势随之西突厥诸部洗牌。**九世纪起**葛逻禄西迁占楚河，**十世纪**喀喇汗王朝在楚河谷立国，**Ak-Beshim 遗址城**渐废，城里的建制中心移至附近的巴拉沙衮（今布拉纳遗址）——便是后世"虎思斡耳朵"之始。**蒙元**（13 世纪）西征过楚河谷时碎叶旧城已荒；至**帖木儿朝**（14-15 世纪）汉地史志中"碎叶"作为一个具体地名彻底消失，仅作历史追述时偶被提起。

近代考古以**苏联中亚考古学家 А.Н. Бернштам（伯恩什塔姆）1939、1947 年探察 Ak-Beshim** 始，确定其为唐碎叶旧址。1953-1954 年苏联与吉尔吉斯科学院联合发掘，在 Ak-Beshim 中部发现**佛寺基址 1-2 处 + 一处景教堂遗迹**——证实玄奘所记"诸国商胡杂居、佛教多元"之实。1960 年代以来俄、吉、中、日多轮调查持续推进，唐城轮廓、城内坊巷、佛寺布局逐渐复原——但唐戍兵营房、安西副镇衙署等具体位置至今未明。

今日 **Ak-Beshim 遗址**在吉尔吉斯斯坦楚河州托克马克市东南约八公里处，为**联合国教科文世界遗产"丝绸之路：长安—天山廊道路网"**33 处遗产点之一——这是碎叶在二十一世纪的位格：一处国际遗产点、一片夯土残址、一座**李白可能、但远非必然**出生的边城。

## 引文

《大唐西域记·卷一·素叶水城》（玄奘述·辩机撰，贞观二十年 646）：

> 大清池西北行五百余里，至素叶水城。城周六七里，诸国商胡杂居也。土宜糜麦、蒲萄。林树稀疏，气序风寒，人衣毡褐。素叶以西数十孤城，城皆立长，虽不相禀命，然皆役属突厥。

——这是汉地史册对碎叶最早的一笔实地记述：商胡杂居、风寒衣毡褐、城各立长而役属突厥——一座盛唐之前已成形的丝路绿洲城。

《新唐书·卷二〇二·李白传》：

> 李白字太白，兴圣皇帝九世孙。其先隋末以罪徙西域，神龙初遁还，客巴西。

——一段二十六字，是后世"李白生碎叶"诸说的文献起点——但"西域"未明指碎叶，"神龙初"亦未明指生年——**此处所容纳的歧说，远多于此处给出的确证**。
