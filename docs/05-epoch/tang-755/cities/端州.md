# 端州

## 一句话

岭南道端州，742-758 年间名"高要郡"——**西江中游北岸**的一座下州，城东**羚羊峡**为西江三峡之一、城北**七星岩**唐时已名，**端砚**自唐于斧柯山下始产；亦是盛唐贬流岭南、自广州西溯的第一站。

## 形貌

端州城在西江中游北岸的一片台地上。

**西江**自西北苍梧—梧州一线来，东南流绕城南而过——出城东，江忽折入两山相夹的一段峡谷，**羚羊峡**：南岸烂柯山、北岸羚羊山——两山逼江、江束如槽——为**西江三峡**之东端一峡（自西而东：三榕峡、大鼎峡、羚羊峡）。江出峡，再东南数十里即接四会、三水之口，再东南便入广州珠江口。**城北是七星岩**——一片石灰岩孤峰，唐时已名（端州七星岩之"七星"得名于七峰排列如北斗），峰间湖泽、岩洞错落，是岭南内陆罕见的喀斯特小景。城东南远望是**鼎湖山**——盛唐时尚未成佛教圣地（鼎湖庆云寺要明万历间才立），但山名唐时已用。

端州城周不大——下州之列：《旧唐书·卷四十一·地理志四·端州高要郡》《新唐书·卷四十三上·地理志七上·端州高要郡》皆明作"**下**"；天宝领县户口**户九千五百，口二万一千一百二十**，按唐州县等第户三万以下为下州，正合。州治**高要县**（今广东肇庆市端州区一带），辖**二县：高要、平兴**（《新唐书·卷四十三上·地理志七上·端州高要郡》明作"县二"；《元和郡县图志·卷三十四》亦作管县二）。城在江北岸略隆起的台地——城南门下江即是西江码头，岭南腹地经梧州、苍梧而下广州的西江航道，过端州必停一夜。物候是岭南内陆的：四时如夏、五月雨季、瘴疠时作——但比之更南的钦廉、雷琼诸州，端州瘴气较轻；盛唐贬官走广州西路入岭南者，过梧州、抵端州即觉"南方风物已半"。**斧柯山在城东南西江南岸**——山中产砚石，自唐于此始凿——**端砚**之名由此而立。

## 大唐天宝十四载（755年）

端州在唐人眼里是一座**广州上游、岭南腹地通广州第一段**的下州。

秦时此地属南海郡。汉武元鼎六年（前 111）平南越后属苍梧郡——汉、三国吴、两晋皆相沿，治所与建置数易。**南朝梁置高要郡**——"高要"取治所高要县之名（取自西江北岸"高峡要冲"之意）；陈、隋初皆相沿。**隋开皇九年（589）**平陈，置**端州**——"端州"之名首见，治高要县；隋大业三年（607）废端州，改为信安郡。**唐武德四年（621）**复置端州——治高要县；贞观初置端州都督府（旋废）。**天宝元年（742）**玄宗诏天下州改郡，端州改称**高要郡**；**乾元元年（758）**复改端州。**742 至 758 这十七年间，端州官号是"高要郡"**——史志、碑刻、文书亦同。

755 年的端州属**岭南道**，州治在高要县。**岭南五府经略治广州**——端州在五府经略统辖之下，由**广州都督府**节制；端州本身不为五管治所，是广州西上游一座**下州**。**755 年端州长官的官称是"高要郡太守"**（742-758 改郡期内复"太守"之名；余时唐于州一级正名"刺史"）。具体在任太守姓名史志阙记。755 年冬安史起兵的消息沿驿道南传——过长安、襄州、江陵、潭州、桂州、梧州，再循西江东下，抵端州约**四十余日**——比抵广州早三五日；端州地方军政未直接卷入河朔战事。

端砚之事：盛唐时高要县斧柯山下已开始采石制砚——**"端砚"作为砚名首见于唐**，但传世端砚的最早实物多为中晚唐至五代物；李贺、刘禹锡、李商隐、皮日休诸晚唐诗中"端溪""端砚""紫端"始入诗——**755 年此刻，端砚产地已成，但是否已贡入宫廷、是否已为天下士人所识——史源不能断言**；按制度时序推之，盛唐端州贡品中或已有砚石，但"端砚"成为文房四宝之冠的过程要到中晚唐才完成。本 doc 不录"玄宗朝端砚已贡内廷"为 verified 事，仅录砚石产地此时已立。

**鉴真过端州**：天宝七载（748）六月鉴真第五次东渡——船自扬州出，遭风暴漂至海南振州（今三亚）——途中弟子荣叡卒，鉴真本人目疾因暑热加重——**北返途中过端州时目盲**（《唐大和上东征传》载之），自端州继续北行至广州大云寺暂止。**端州是鉴真目盲之地**——这是中日佛教史上一处具体地名。距 755 年此刻七年——盛唐端州人若听过这一段，大约只当一桩异闻；千年后日本奈良唐招提寺礼拜鉴真坐像的人，未必都知道这位老和尚是在岭南这座小州城里彻底失明的。

## 历史长河

至德二载（757）高要郡复称端州；乾元元年（758）正式行用端州之号。中晚唐端州渐以**端砚**名于天下——李贺《杨生青花紫石砚歌》、刘禹锡《唐秀才赠端州紫石砚》等诗皆以端砚入笔。**北宋庆历七年（1047）至皇祐三年（1051）**包拯任**端州知州**三年——疏浚星湖、修筑包公祠井（今肇庆包公祠所祀），离任时"不持一砚归"——这是后世包公故事中最有名的一段。距 755 年盛唐此刻两百九十二年——盛唐的端州人若得知后世会有这样一位"清官"以"不持端砚"立名，大约只能笑这名声立得太迂。**北宋徽宗重和元年（1118）**端州升为**肇庆府**——"肇庆"取宋徽宗潜邸"肇庆军"之名（徽宗为端王时领端州；登基后端州升府改名肇庆，"肇庆"意"始庆"，避端王旧名之讳）。**自唐武德四年（621）复置端州至北宋重和元年（1118）升肇庆府，凡四百九十七年——端州"州"级建置一段**。元置肇庆路，明置肇庆府，清沿明制。明嘉靖以后两广总督一度驻肇庆（万历至崇祯朝），肇庆为两广军政重心。**民国元年（1912）**废府，肇庆改名高要县。1988 年置肇庆市。今肇庆市端州区即唐端州故城所在——西江仍东流绕城南、羚羊峡仍夹江、七星岩仍在城北——这是岭南诸唐州中地貌延续较完整的一处。端砚自唐至今一千二百年未断产——斧柯山下的砚坑（老坑、麻子坑、坑仔岩、宋坑）至今仍是文房四宝之冠。

## 引文

《旧唐书·卷四十一·地理志四·岭南道·端州高要郡》：

> 端州 隋信安郡。武德四年，平萧铣，置端州。天宝元年，改为高要郡。乾元元年，复为端州。旧领县四，户九千五百，口二万一千一百二十。下。

《新唐书·卷四十三上·地理志七上·岭南道·端州高要郡》：

> 端州高要郡，下。本信安郡，武德四年更名。土贡：石砚。户九千五百，口二万一千一百二十。

《元和郡县图志·卷三四·岭南道一·端州》：

> 端州，高要郡。……领县四：高要、平兴、乐城、悦城。州西南有羚羊峡，长八里；州北有岩石壁立，名七星岩。

——两《唐书》一笔点明端州**下州**、领县四、户九千五百口二万一千一百二十；《元和郡县图志》补景物——西江峡、北岩石——便是这座岭南腹地小州的全部地理。
