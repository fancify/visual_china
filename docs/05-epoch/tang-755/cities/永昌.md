# 永昌

## 一句话

南诏西陲军镇，哀牢故地——汉永昌郡治在此，755 时为**南诏永昌节度新置（天宝末-至德间）**所辖，扼**西南丝路（永昌道）**东段；西略寻传、骠国、抵腾冲一线为大历后异牟寻朝才完成。是大唐西南方向最远的一处汉名旧城。

## 大唐天宝十四载（755年）

永昌在唐人眼里是**汉名旧城**——东汉明帝**永平十二年（69 年）哀牢王柳貌率众内附**，朝廷以哀牢故地置**永昌郡**，治不韦县（今云南保山一带），属益州。"永昌"二字自此立——这是云南最西、最早入中原版籍的一笔。东汉至两晋间永昌郡领八县，西尽于今滇缅边——"哀牢九十九部"散居高黎贡山、怒江、澜沧江诸河谷。南北朝以下中原势衰，永昌郡时设时废；爨氏据滇东西时，永昌一带由当地哀牢族裔（后世彝、白、傣等族之祖）部落自治。

唐高宗**仪凤元年（676）一度置永昌府**——是大唐对哀牢故地的最远直辖，但维持不久即归地方。开元末六诏归一后，永昌之地随姚州都督府所及之范围浮沉；天宝九载（750）姚州陷、唐势退出云南西部之后，**南诏王阁罗凤始经略哀牢故地**——755 时**永昌节度新置（天宝末-至德间）**，南诏沿用唐"永昌"旧名作节度治。**南诏西略寻传、攻骠国、并诸蛮以抵腾冲一线，是大历后异牟寻朝（778 立）才逐步完成**——寻传部归南诏在 778 之后，骠国通使更晚至 802；755 当年永昌节度只是**南诏新置的西陲军镇雏形**，西陲尚未抵腾冲，更未及寻传、骠国。

**永昌道**——汉时已通——是**西南丝绸之路东端的主干**：益州 → 嶲州 → 姚州 → 太和 → 永昌 → 腾越（后世腾冲）→ 骠国 → 天竺，自蜀至印度的陆路。755 年这条道在大唐与南诏交恶（751、754 两役）之后已半断——但商旅、僧侣、使节仍走，未尽绝；永昌以西段（腾越、寻传、骠国一带）则待南诏后世（异牟寻、劝丰祐朝）经略后才真正纳入南诏掌控。

755 时永昌城内并不繁盛——南诏取自哀牢与汉旧城的格局，城周不大，多木构草顶；高黎贡山的雪线在城西可见，怒江自北向南流过城东不远，瘴疠盛夏多发。**户口不强订**——南诏永昌节度的户籍系统与唐州县不同，文献无完整数。

## 历史长河（755 → 今）

中晚唐永昌节度续为南诏西陲，至 902 大长和国仍领之。大理国（937-1253）改设**永昌府**于此——"永昌"自此为府名延续。元改永昌州、再改金齿宣慰司（永昌一带哀牢族裔以"金齿"为饰）。**明洪武十五年（1382）平云南，立永昌府**——明清两代永昌府治保山县（今云南保山市），下辖永平、腾越（今腾冲）等州县；徐霞客《滇游日记》（崇祯十二年 / 1639）详记永昌、腾越一线。**1913 年民国废府改保山县**——"永昌"二字自此退出行政名，地名沉入"保山"。今保山市隆阳区一带即唐至明清永昌城脉，腾冲市即唐至明清腾越州治——盛唐 755 时的永昌节度遗迹今地表所存极少，但**汉永昌郡 → 唐永昌府 → 南诏永昌节度 → 明清永昌府**这条名脉自 69 至 1913 共一千八百四十四年，是西南汉名延续最长的一处。
