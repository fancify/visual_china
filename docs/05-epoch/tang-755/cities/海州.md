# 海州

## 一句话

河南道海州，742-757 年间名"东海郡"——黄海西岸的港门州治，治朐山县；通新罗、日本的海路主端在登州与扬州、明州，海州只是黄海西岸的辅助节点，城外有东汉孔望山摩崖造像。

## 形貌

海州城在黄海西岸的低平海积平原上——州治朐山县城（今江苏连云港市海州区）距海岸数十里。城东南有朐山（今锦屏山一线），城东海上有**云台山主峰——盛唐时尚为海中孤岛**，与陆地之间隔一道浅海（明清以后泥沙淤积方与陆相连）。城西过沭水、沂水是河南道腹地低平农耕区，城南越淮水即淮南道楚州境，城北过沂沭诸水是密州、莒州的山东丘陵南麓。**淮北沂沭平原与黄海港门的咬合处**——既非腹地大州、亦非边塞军镇，河南道东南一隅的中州。

户口据《旧唐书·地理志》天宝间**约两万户**——按唐州县等第刚过中州门槛、未及上州。海路上地位却较高：盛唐通新罗、日本者，主路由**登州（蓬莱）出海**，扬州、明州各为南北远洋出口，**海州居其中、是通新罗西海岸的辅道**。物候是淮北沿海的——春海雾、夏东南风、秋早霜、冬寒少雪；东海岸盐场自汉以来便兴，唐时海州盐课在淮北诸州中颇重。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

海州境是**汉以来旧地**。先秦此地属东夷郯国、邳国一线，秦置朐县（"朐"取自境内朐山）属东海郡——汉东海郡治郯县（今山东郯城），辖朐县等数十县，是西汉淮泗北端的大郡。东汉至魏晋东海郡屡有省并。**孔望山摩崖造像**——城南约二十里孔望山西南崖壁，自后汉建武中至灵帝间凿造佛、道、儒三教杂糅造像数十身，被认为是中国现存最早的佛教石刻造像之一（具体断代仍有考古争议，主流意见东汉中晚期，但亦有六朝至唐说）；按主流断代约 2 世纪为主，**若此说成立则至唐时已六百年旧物**，盛唐海州人尚可指认。

**北魏孝昌四年（528）置海州**，治龙沮城（在今连云港西南）——"海州"之名自此立，取临海之意。**隋大业三年（607）**改海州为**东海郡**。**唐武德四年（621）复置海州**，治朐山县——州治自北魏龙沮城南移至朐山一线。**天宝元年（742）**玄宗诏改州为郡，海州复称**东海郡**；**乾元元年（758）**肃宗复改郡为州——海州之名再立，沿用至宋元。

### 此刻

天宝十四载（755）——官称"东海郡"，民间口语仍呼"海州"。河南道东南端，据《旧唐书·地理志》海州**天宝领县四**：朐山（州治）、东海、沭阳、怀仁——州境东西约百余里、南北跨沂沭下游与黄海西岸。本郡户口据《旧唐书》载**28549 户 — 中州门槛之上**，不及苏、扬、楚那种十万户的运河大州，但比河南道腹地诸小州稍大。海州不是节度治也不是都督府——河南道由采访使巡历，治所在汴州（陈留郡）；海州只是治下中州之一。盛唐河南道无大边防，但海州临海故，向来配置海防小营——以备海盗、巡海岸。

海路第一宗。盛唐时新罗与唐之海路**主道在登州（今山东蓬莱）出海**——沿庙岛群岛北上辽东半岛南端、再东行至新罗西海岸；另一辅道**自海州东出黄海中部、直接东抵新罗西海岸**——风季顺时可省千余里。新罗、渤海使节东来朝贡，登州为主、海州为辅；日本遣唐使南线（明州、扬州出入）则少经海州。盛唐海州东海港口具体位置史志阙详——或在朐山港、或在云台山下海港，今云台山一带（盛唐尚为海中孤岛）的港口位置仍待考古确认。

盐课第二宗。东海岸自汉以来便是淮北重要盐场——海州沿海盐池、煮盐场分布数十处。盛唐盐铁专卖未及天宝（盐铁使由唐肃宗乾元年间始置），但海州盐产已是江淮盐运的一支——经沭水、淮水入大运河北上。**注：隋大业六年（610）所开邗沟、江南河、通济渠未直接通海州**——海州的盐运与漕运须经楚州、淮安方入运河主干。

文物第三宗。**孔望山摩崖造像**——城南约二十里孔望山西南崖壁，**主流断代东汉中晚期（约 2 世纪，亦有六朝乃至唐说）**凿造佛、道、儒三教杂糅造像数十身：佛像、礼佛图、仙人骑兽、君臣百戏——被认为是中国现存最早的佛教石刻造像之一。**若主流断代成立则至盛唐时已六百年旧物**——盛唐海州人尚可指认其名 / 方位，但唐人不以为奇：盛唐崇佛多向龙门、敦煌、麦积，孔望山东汉旧造像文献亦少著录。

天宝十四载冬安禄山反范阳——消息沿驿道传到海州时，城里粮草、海防的调度也紧起来。但海州远在河南道东南、距河北叛军主战场尚有千里——这一年海州未即陷，但漕、盐两路皆受冲击。

## 历史长河

**中晚唐**　乾元元年（758）复改东海郡为海州。中晚唐海上交通逐渐加重——**新罗张保皋于唐文宗大和年间（827-835）经营海上贸易**，以海州、登州、扬州为西端三大港；中晚唐新罗坊广布黄海西岸诸州（扬州、楚州、登州皆见明确记载），海州境内是否亦立新罗坊史料未明，姑存疑。僖宗乾符年间黄巢起兵后海州随河南淮北震动；唐亡后海州先后入南唐、北宋。

**宋元明清**　北宋初复置海州属淮南东路；金属山东东路海州；元置海宁州属淮安路；明洪武初废海宁州、复设海州属南直隶淮安府；清雍正二年（1724）升海州直隶州（属江苏布政司）。云台山一线明清以后逐渐与陆地相连——盛唐"海中孤岛"的形势已无。

**民国至今**　1912 年废州为东海县；1935 年陇海铁路通车至大浦港、沿海一带新设连云港市（取"云台山港 / 连岛港"之意）；今**连云港市海州区**即唐海州城旧址范围——盛唐朐山县城旧址叠在今海州区主城下。**孔望山摩崖造像仍在原址**——东汉至盛唐至今二千年未离。**云台山仍在**——但已与陆地相连。
