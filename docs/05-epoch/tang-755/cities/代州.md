# 代州

## 一句话

河东道北门一州，治雁门县（今山西代县）——城在雁门关之南，太原北上出塞的必经一站；北接朔方、单于诸羁縻府，是河东节度使北翼的咽喉。

## 形貌

代州城坐在滹沱河上游一片山间盆地里。北倚**勾注山**（即雁门山），南瞰滹沱；东西两面是太行余脉与吕梁北段的山口余脊，盆地南北约百里、东西数十里，是河东北部少有的一块可耕川谷。州治**雁门县**——城周不大，城外是滹沱冲积的小平畈，沿河有水浇地数顷，再远便是黄土塬与山脚石坡。

城北约六十唐里便是**雁门关**——勾注山主脊上的关口，海拔约一千五百米，是河东与塞外的天然分界。**雁门"东陉关"、"西陉关"两道（后世合称雁门关）**在唐时已是太原北上的必经，唐人由长安北行至太原、再北越雁门即入云中、朔方诸羁縻州境——这一线官驿道由代州过站。所谓"河东北门"——便是代州夹在太原与雁门之间这一段，南去太原约三百唐里，北越雁门关百余里即至云州、朔州一线。

物候是河东北段的：春迟而风沙重、夏短而雨集中于七八月、秋早霜、冬严而长——年降水约四百毫米，可旱作粟黍菽豆，少水稻；勾注山以北便是塞外牧地，代州境内尚以耕为主，但城外山脚已可见羊群。冬季滹沱河十一月底封冻，城北雁门关一带十月即雪——"九月雁飞过雁门"是当地老话，代州之名一半在雁、一半在关。

## 大唐天宝十四载（755年）

唐人眼里代州不算极上古之邑——但它的沿革在北朝以下脉络清楚。

**北周武帝建德六年（577）**平北齐之后，在原北齐肆州之地置**肆州**——治雁门县，是为本州肇始；**肆**之名取古肆卢县（汉雁门郡属县）。**隋开皇五年（585）**文帝改肆州为**代州**——"代"字取汉代郡（旧治今河北蔚县一带）之名旁推而来，自此"代州"二字立。**隋大业三年（607）**炀帝行州改郡通制，改代州为**雁门郡**——治仍在雁门县。**唐武德元年（618）**高祖复改郡为州，复称代州；**天宝元年（742）**玄宗诏改州为郡，代州又改回**雁门郡**；**乾元元年（758）**肃宗复改郡为州——代州之名再立。**742 至 758 复州前，代州官称为"雁门郡"，民间口语仍呼"代州"或径呼"雁门"**——两个名号在唐人嘴里互换不分。

755 年的代州属**河东道**，州治在雁门县。据《旧唐书·地理志二》记天宝间代州约 **2.1 万户**（21282 户 / 100350 口）——中州门槛，辖县在玄宗朝凡五：**雁门**（州治）、**五台**、**繁畤**、**崞县**、**唐林**——皆滹沱上游与五台山一线山谷间的县治。州中无节度治所——河东节度使坐镇太原，代州属其辖境，是北翼边州；但州境内常驻**代州都督府**（天宝时仍为代州都督府，掌北边军政）——以及朝廷于雁门关南北所置**戍**、**烽**、**铺**若干，是河东节度使**北防体系**的前哨。

**雁门关**在唐时为河东出塞第一关——盛唐玄宗朝多次北幸、册命漠南诸羁縻部，使臣往返皆过此关；代州城内有州衙、雁门县治、驿馆，是过往使节、戍兵、商旅最集中的所在。**五台山**则在代州境东南——五台县治即在山西麓，开元、天宝间五台山佛教**文殊道场**地位已立（华严宗、密宗皆奉为圣山），京师与河北、河南诸寺僧人朝山者络绎，多自代州过站。这条"代州—五台"的礼山路线，是盛唐河东宗教地理上一笔分量极重的事——五台山详见 [五台山](../scenic/五台山.md)（doc 待立）。

天宝十四载冬安禄山反范阳——消息沿驿道传到代州时已是十一月中下旬。代州是河东北门，本州兵不多，但作太原北上之第一关下大城，朝廷自此调发戍兵、点检烽铺最忙。**七年之后**（763 年前后）回纥兵马助唐复京、北归经此；**两百年之后**（979 年前后）宋太宗北伐至此城下——代州之分量自盛唐至北宋一直在"北门"二字。**755 这一年的代州**，意味着雁门关南那一座驿与戍——名号还是雁门郡，民间口里仍叫代州，山顶的雪与关下的霜，与盛唐数十年来无别。

## 历史长河

**中晚唐**　乾元元年（758）复州名为代州（雁门郡旧名废）。安史乱后河东道地位上升，代州做太原与雁门关之间的转运枢纽与北防前哨。中唐回纥助唐复京、自北而南过雁门关南下太原，过代州者甚众；晚唐沙陀部李克用以**代北**为根据地——**乾符五年（878）李克用据代州反唐**，旋败走鞑靼；**中和三年（883）**李克用以平黄巢之功封河东节度使，治太原而根脚仍在代北——代州成为沙陀立国的最初据点。**李克用之子李存勖即后唐庄宗**——后唐起家之地，便是代州、雁门、朔州这一线。

**宋金元**　北宋以代州属河东路，治雁门县；**雁门关为北宋北防之第一线**——景德元年（1004）澶渊之盟前，宋辽屡在雁门交兵。**北宋雍熙三年（986）杨业战死陈家谷口**（今宁武县与代县之间）——杨业为北宋雁门一线主将，**杨家将故事即起此一线**；明清以来"杨家将"小说与戏曲俱以代州雁门为舞台，"杨业血战雁门"、"佘太君百岁挂帅"等情节皆系明清以后小说家言，**盛唐时代州人尚无此故事，亦无杨氏一族**——本 doc 历史长河段始得提及。金、元仍为代州。**元末"雁门十八隘"**——元末以后才成称谓，唐时所谓"雁门"仅东陉、西陉两道关口；明清以后于雁门主关之外加列十八隘（白草口、广武口、太和岭口等）合称，是后世防制之名。

**明清**　明洪武七年（1374）改代州为代县，旋复为州，属太原府；**明初置雁门巡检司**——雁门关下夯土与砖石加修，今所见雁门关城为**明洪武七年（1374）至嘉靖年间**所修，是明长城外三关之一（与宁武关、偏关并称"外三关"）。**雁门关在明清以来的"杨家将"叙事中地位达到顶峰**——戏曲、小说、说书皆以此关为背景。清承明制，代州属山西布政司太原府；清雍正二年（1724）升代州为**直隶州**，辖五台、繁畤、崞县——直隶州地位与汾州府、平阳府并列。

**民国至今**　1913 年废州存县——代州正式退出建制，**代县**为山西省辖。**雁门关**自民国以来逐渐由军事关隘转为名胜古迹，1992 年起代县政府陆续修复关城与"雁门关"匾额，2001 年公布为全国重点文物保护单位。**今日山西忻州市代县治所，仍在唐代州治雁门县旧址上**——这是河东北段少见的连续一千五百年同一城址。盛唐那座代州——州城残基已埋入今代县城下，雁门县官署旧址不可考；滹沱河仍自城南流过，勾注山仍在北面横亘，山顶的雪与关下的霜——千年没变过的，是雁门关那一线。
