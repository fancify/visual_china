# 温州

## 一句话

江南东道温州，742-758 年间名"永嘉郡"——治永嘉县（今温州鹿城），瓯江入海口上一座山海之间的小州；**武则天上元二年（675）始析括州永嘉、安固二县置温州**——盛唐里这是一座相对偏远新设的边海州治。

## 大唐天宝十四载（755年）

温州治**永嘉县**，在瓯江下游北岸——江自西括苍山中来，至永嘉县城前已成宽江，再下行约三十里即出**瓯江口**入东海。城南临江、北背山，是浙东南山海相接的一座小州城。

沿革较晚——这一带秦汉为闽中、东瓯地，东瓯王驺摇汉惠帝三年（前 192）受封于此，故**"瓯"**之名古；汉建武中徙东瓯于江淮，地遂虚。两晋南朝间属临海郡、永嘉郡——**东晋明帝太宁元年（323）**析临海郡置**永嘉郡**，治永宁县（即今永嘉县），"永嘉"作郡名自此立、自东晋以来已四百三十余年。隋开皇九年（589）废永嘉郡入处州，旋改括州；大业三年（607）改括州为永嘉郡——但治在括苍（今丽水），不在瓯江口。**武则天上元二年（675）**始**析括州之永嘉、安固二县置温州**——州治移至瓯江口的永嘉县，自此瓯江口才有独立州治；"温州"二字自此立。州名之由——一说**东瓯地暖少寒**、一说**境内有温泉**，史无确据，唐人即已两说并存。**天宝元年（742）**玄宗诏改州为郡，温州改作**永嘉郡**；**乾元元年（758）**肃宗复为温州——742 至 758 这十六年间，公文上写的是"永嘉郡"。

755 年的温州属**江南东道永嘉郡**，《旧唐书·卷四十·地理三·江南东道·温州永嘉郡》载：**"温州，上。……上元二年，分括州之永嘉、安固二县置温州。天宝元年，改为永嘉郡。乾元元年，复为温州。天宝领县四，户四万二千八百一十四，口二十四万一千六百九十四。在京师东南四千七百三十七里，至东都三千九百四十里。"**——领县四：**永嘉、安固、横阳、乐成**；志中州等列为"上"，但户口实数（户 42,814、口 241,694）以唐户三万为上州门槛计仅勉强逾线——较其北邻台州（户 83,868）实为减半，是浙东南山海间的一座**上州下限**之州。

唐人眼里的温州不重——州小、户口寥、距京东南四千七百余里，是江南东道最东南的边海州之一。州境八九成是括苍山系南段与雁荡山系的山地丘陵——并无大平原；瓯江出海一段虽可通海舶，但盛唐海上贸易未盛，温州非大港。州中物产以山中茶、漆、纸，海中盐、鱼为主——这是一座靠山靠海、户口自养的小州，**唐时温州远不是宋元后那座商业大港**。755 年安禄山起兵于范阳，消息传至温州约在十二月——州中无兵无事，于江南腹地东南最远一隅。

## 历史长河

乾元元年（758）肃宗复改永嘉郡为温州——州名定于"温"，自此一千余年再未大变。中晚唐温州仍为偏远小州，无大事入史。**唐末五代**温州先后属闽国、吴越国——吴越王钱镠据浙东南，温州为吴越属州。**北宋至南宋**温州渐兴——**南宋永嘉学派**叶适（1150-1223）等于温州倡事功之学，与朱熹理学、陆九渊心学三分浙学；同期温州瓯江口成**南宋海上贸易**重要节点之一——但这些已是 755 后四百余年事。**明清**置温州府，治永嘉县；**民国元年（1912）**废府存县，置永嘉县；**1949 年后**置温州市（市治在唐永嘉县治旧地，即今鹿城区）。**1980 年代改革开放后**温州以民营经济兴起、温州商人遍布全国乃至海外——这是 20 世纪末才有的事，与盛唐温州的山海小州面貌相去甚远。今**浙江省温州市**鹿城区，仍是直接承唐永嘉县治之地——州城叠在今鹿城区下，瓯江仍自西而东流过城前入海，与盛唐方位大体相承。

## 引文

《旧唐书·卷四十·地理三·江南东道·温州永嘉郡》：

> 温州，上。上元二年，分括州之永嘉、安固二县置温州。天宝元年，改为永嘉郡。乾元元年，复为温州。天宝领县四，户四万二千八百一十四，口二十四万一千六百九十四。在京师东南四千七百三十七里，至东都三千九百四十里。

——这一段是 755 年时态温州的地理志原文：**领县四、户 42,814、口 241,694**——以唐户三万为上州门槛计，温州天宝年间勉强列上州，志书亦明书"上"字，但户口实数已是上州下限。距京师"四千七百三十七里"、至东都"三千九百四十里"——这是唐地理志一贯的官道里程，自长安经洛阳南下江淮、沿运河至润州、再东南越浙东台州而抵温州，是当时入温的主路。志中"上元二年分括州之永嘉、安固二县置温州"一句——是温州建制最早的明文，置州年武则天上元二年（675）距 755 仅八十年，在唐二百九十年里温州是设州较晚的一州，这与其户口寥、地偏远的事实互为印证。
