# 蒲类军

> **史源校勘**：唐《旧唐书》《新唐书》载北庭大都护府**三县**（金满、轮台、**蒲类**）+ **三军**（瀚海、天山、伊吾），并无"蒲类军"之正式军号。本 doc 题名"蒲类军"系沿用项目 inventory 旧标，实指"**蒲类海军镇**"——即驻于蒲类海（巴里坤湖）北的北庭辖军戍据点，属**地理性军镇**而非正式"某军"军号。文件名暂保留作 alias，正文以"蒲类海军镇"为准。

## 一句话

北庭大都护府蒲类县境内、**蒲类海**（汉名，今巴里坤湖）北岸的军戍据点——史称**蒲类海军镇**，戍庭州东境、压伊州方向的牧部，常额戍卒约数千；非唐正式"某某军"军号，是地理性军镇。

## 形貌

蒲类海军镇在天山东段北麓的**蒲类海**岸——湖是一片高原咸水湖，盐花结于岸缘，湖周一圈低草滩、再外是戈壁。镇治依**蒲类县**（北庭三县之一；**唐高宗永徽元年（650）置蒲类府**——后裁府存县，并入北庭大都护府辖境），距北庭府治庭州（金满县城）东约三百唐里——沿天山北麓东行三日驿程。湖南是天山东段诸雪峰，湖北是准噶尔盆地东南缘的戈壁——再东北便是回纥诸部牧地，再东南三日是伊州（哈密）。镇城规小——夯土一周里许，戍卒营房、马厩、兵库、粮仓而已，不及州治之制。

## 大唐天宝十四载（755年）

**北庭大都护府辖境内的湖岸军戍据点**——按《新唐书·地理志》，北庭正式建制为**三县**（金满、轮台、蒲类）+ **三军**（**瀚海军**驻庭州金满附郭、**天山军**驻西州、**伊吾军**驻伊州境），并无"蒲类军"之正式军号；蒲类海北岸所驻的戍卒、烽燧、屯所，史料统称"蒲类海军镇"或归入蒲类县下军事条目，属**地理性军镇**，节制于北庭都护府瀚海军。常额戍卒数千，与三正军合计北庭兵两万二千之数。755 年北庭节度由安西节度封常清**兼**——程千里于天宝十一载（752）任北庭节度、天宝十三载（754）冬献俘留京后东调河东，已不在西陲；蒲类海军镇日常戍务由军使、副使代行，归庭州内城衙署节制。

"蒲类"二字出**汉宣帝甘露二年（前 52）**——汉破车师、屯田蒲类，汉宣帝时设**蒲类国**治此湖东。汉之后历八百年，魏晋、十六国、车师、高昌、柔然、突厥相继据此地——唐高宗永徽元年（650）置蒲类府，是**汉名重入中原版图**的一笔。755 年蒲类海军镇戍卒，每日所巡的湖岸，便是汉宣帝时屯田卒所开的同一片草滩。

军镇日常：每年三月草青开操，沿天山北麓东巡至伊州境、北巡至准噶尔南缘——查游牧部落动静、护商队、传驿；九月霜至收兵入镇过冬。冬日湖面结冰、戈壁雪封，驿马自蒲类海军镇至庭州常须七八日。十一月初九（公历 12 月 16 日）安禄山反范阳——消息传至蒲类海军镇，已是**天宝十五载（756）正月或更晚**；湖岸戍卒不知，这是他们在唐旗下最后一个完整的冬天。

## 历史长河

安史乱后蒲类海军镇随北庭一体孤悬西域——**贞元六年（790）吐蕃破北庭**，蒲类同年陷。吐蕃据天山北道约半世纪，至会昌、大中间（840s-850s）转入西州回鹘（高昌回鹘）势力——蒲类海一带为其牧地，"蒲类"名号亦自此渐废，回鹘人改称湖名**巴里坤**（突厥-蒙古语，意"虎前湖"）。元代为别失八里宣慰司辖境。**清乾隆二十二年（1757）**平准噶尔后置**巴里坤厅**——"巴里坤"自此为正式行政名号，蒲类古名遂入史册。今**新疆哈密市巴里坤哈萨克自治县**驻军遗址距县城西约数里，唐蒲类海军镇的夯土残段在 20 世纪文物普查中曾见——但完整考古发掘至今未启。

## 引文

《新唐书·卷四十·地理志四·北庭大都护府》：

> 北庭大都护府，本庭州……领县三：**金满、轮台、蒲类**。……瀚海军，在府城内；**天山军**，在西州；**伊吾军**，在伊州西北。

——北庭正式建制：**三县**（金满、轮台、蒲类）+ **三军**（瀚海、天山、伊吾），并立而无"蒲类军"之名。"蒲类"是县名（治所在蒲类海北岸），其湖岸军事据点史称**蒲类海军镇**，归瀚海军节制——而非北庭三军之一。本 doc 沿用项目 inventory 旧题"蒲类军"作 alias，正文实指此军镇。
