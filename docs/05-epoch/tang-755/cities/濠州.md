# 濠州

## 一句话

淮南道淮水中游南岸一下州，治钟离县（今安徽滁州市凤阳县临淮关一带，近蚌埠市区）——西邻寿州，扼淮水南岸；742-758 年间官名"钟离郡"。

## 大唐天宝十四载（755年）

濠州治**钟离县**，属**淮南道**。城在淮水中游南岸——西去寿州（寿春郡治）逾百里，东南接和州、滁州。**钟离**之名出春秋——本楚邑，《史记·楚世家》屡见；秦汉为九江郡钟离县；**隋开皇二年（582）改西楚州置濠州**，治钟离；大业三年（607）改置**钟离郡**。**唐武德三年（620）复置濠州**；**天宝元年（742）改钟离郡**——是为 755 年当年的官称；**乾元元年（758）复濠州**。

户口上，《旧唐书·地理志》载濠州为**下州**——天宝户 21,864（《旧唐书》户数），户额已逾两万、偏中州门槛（唐制户二万以上为中州），然《旧唐书》本文仍系"下州"；两源 hedge 取下州。郡领钟离、定远、招义三县。州境的根本特点是**淮水中游南岸的过渡带**：北临淮水，与颍州、泗州隔水相望；西邻寿州，淮南漕路的中段自此过；东南去淮南道治所扬州尚远。盛唐淮南腹地的中州，多半如此——无大军屯驻、无大将出镇，州事在太守、郡丞、司户、司仓之间。

755 年冬安禄山反范阳，淮水漕路立刻成为南北接续中原的命脉——濠州坐在这条漕路上，与寿州、楚州、泗州共担转输。这一年的钟离郡，城内事务平常如旧，只是诏令一道接一道南来。

## 历史长河（755 → 今）

中晚唐濠州仍为淮南漕路一节点；唐末杨行密据淮南立吴，濠州入吴版图。五代两宋濠州先属杨吴、南唐，后入北宋淮南西路。**元末至正十二年（1352）朱元璋于濠州钟离起兵**，投郭子兴红巾军——朱氏即濠州钟离人（1328-1398）；**明洪武二年（1369）于濠州立中都**；**洪武七年（1374）改濠州为凤阳府**——盛唐"濠州"二字至此罢，让位于"凤阳"。清因明制属凤阳府；民国废府存县；今治为安徽省滁州市凤阳县（近蚌埠市区），盛唐钟离县城旧址在临淮关一带——蚌埠则系 20 世纪因津浦铁路而起的新兴城市，非盛唐濠州本城。
