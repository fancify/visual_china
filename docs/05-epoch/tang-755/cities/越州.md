# 越州

## 一句话

江南东道会稽郡治，山阴县所在；春秋越国故都，王羲之兰亭、马臻鉴湖、贺知章故里——唐人提"越"，脑中先是水，再是会稽山，再是那一篇《兰亭》。

## 形貌

越州城在会稽山北、鉴湖之上——南屏会稽，北面平原直入杭州湾。

州治山阴县城周回不甚大——比扬州一州数县合城远小。城南五里便是会稽山的北麓余脉，山势不高，自南而北一道一道压过来，到州城脚下方止；城北则是水网平原，纵横河汊一直摊到杭州湾南岸。城西南是**鉴湖**——东汉永和五年（140）会稽太守马臻凿——周回三百里许（《水经注》语），是江南诸湖中最大的人工灌溉湖；城就盖在湖与山之间这道窄长的台地上。城里以山阴道为脊——东西一条主干，沿道两侧便是州衙、县衙、市肆、寺观。

水是越州的底子。一州境内水道密如丝网——大者为鉴湖、若耶溪、浙江下游一段，小者无数支汊穿城绕坊。唐时州城有水门——舟楫直入城内市肆，载薪柴、瓷器、绫绢、丝茶。山阴县外乡村，**乘船比乘马便**——这一带的农户家家有舟，至今未改。

物候是江南偏东的：春雨绵绵，三月梅雨，五月入夏，八月仍闷热，冬日不甚寒但湿冷入骨。会稽山下出茶，鉴湖中产鱼蚌，山阴诸窑出青瓷——盛唐时**越窑**已为南方青瓷最重要的一家，胎薄釉青，行销江淮、入贡长安。"秘色"之名要到晚唐至五代才正式见于文献（如陆龟蒙《秘色越器》），盛唐尚未如此称——但越窑青瓷的烧造工艺，755 年时已近成熟期。

至于会稽山南麓——那一带是**大禹陵**所在。传大禹葬会稽，《史记·夏本纪》《越绝书》皆载之。唐时禹陵已立祠，岁时官祭——盛唐人入越州，必入会稽山一访。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

越州在唐人眼里不是新城，而是一座**春秋故都**。

最早是越国——**春秋越王句践**自允常起便都山阴（一说原都诸暨，至允常或句践迁山阴），勾践卧薪尝胆、败夫差、北上称霸，便从这座城起。秦置**会稽郡**，治吴县（今苏州）；东汉永建四年（129），析会稽郡置吴郡——会稽郡治自此南移**山阴县**（即今越州城所在），自此一千六百余年，山阴一直是会稽郡 / 越州 / 绍兴府治所。**隋开皇九年（589）**平陈后置吴州，治会稽——这是吴会两州合一时期；**唐武德四年（621）**改吴州为**越州**，自此"越州"之名定。

至于城内古迹，最负盛名的是城西南的**兰亭**。东晋永和九年（353）三月三日上巳节，时任会稽内史**王羲之**与谢安、孙绰、支遁等四十一人会于会稽山阴之兰亭，**修禊事**——临流赋诗，王羲之乘兴作《兰亭集序》，行书三百二十四字。这一篇序在书学史上的位置——书圣手书、行书第一——不必赘述。盛唐人提起兰亭，脑中浮现的是这一篇序与这一处水。

兰亭原迹的下落，至 755 年时已有"陪葬昭陵"之传——**唐太宗酷爱《兰亭》**，《新唐书·禇遂良传》《尚书故实》及张彦远《法书要录》等唐宋史料均记太宗以《兰亭》原本陪葬昭陵；冯承素、虞世南、禇遂良、欧阳询诸家奉敕摹本传世——其中**冯承素摹本**（神龙本）今藏故宫博物院，是各家公认最近原迹的一本。但原本是否真陪葬昭陵、是否后世昭陵被发掘时已亡失，唐时已是悬案，至今未有定论——此处加一道 hedge。

755 年的盛唐人到兰亭——可访的是**永和兰亭的旧地**（兰渚山下、鹅池畔），并非永和原构。东晋以来兰亭屡毁屡建——隋唐时已是地方名胜，州县岁修。

### 此刻

天宝十四载（755）的越州是**江南东道会稽郡治**——唐天宝元年（742）玄宗诏改州为郡，越州改称**会稽郡**；至德二载（757）肃宗复改郡为州，会稽郡再称越州——但 755 年这一刻，公文上写的是"会稽郡"。

会稽郡天宝十四载领县——史源略有出入，两说并存：
- **《旧唐书·地理志》载越州天宝领县六**：**会稽、山阴、上虞、馀姚、剡、诸暨**；
- **《新唐书·地理志》及《元和郡县图志》载越州（会稽郡）领县七**：在上述六县基础上**增萧山**——即**会稽、山阴、上虞、馀姚、剡、诸暨、萧山**。

两书差异即"萧山一县"——**萧山县**即隋开皇九年（589）所置永兴县；**唐天宝元年（742）改永兴为萧山**——755 年这一刻官称已是萧山县，非"永兴县"，贺知章故里此时县名作萧山。旧书未列萧山系沿用早期文书，新书 / 元和志补入萧山——本 doc 取七县说（与 742 改名以后实际辖境一致）。**会稽县**与**山阴县**唐时同城分治——城以纵街为界，一城两县（这是越州唯一一处"一城两县"格局）。本郡户口据《通典》《元和郡县图志》记开元 / 元和户数约在九万户上下——是**唐上州**，江南诸州户口排前。唐州县分级**上、中、下**视户口而定（开元二十一年定制：上州四万户以上），会稽郡九万户稳居上州，且在江南东道仅次于浙西的余杭郡 / 吴郡一档。

这一年朝中事——年初京中传来一则消息：**贺知章已殁十余年**。**贺知章**（659—744）越州永兴县（742 改名萧山县）人，少以文章知名；武则天证圣元年（695）擢进士第，玄宗朝官至秘书监、银青光禄大夫、太子宾客——盛唐文坛地位极高，与张旭、包融、张若虚并称"吴中四士"。**天宝三载（744）正月**，贺知章上疏请度为道士、归乡——玄宗诏许，赐宅为千秋观（一名贺监祠），并亲制《送贺知章归四明诗序》；李白时在长安，作**《送贺宾客归越》**："镜湖流水漾清波，狂客归舟逸兴多；山阴道士如相见，应写黄庭换白鹅。"——贺归越数月后卒，**卒年传统记天宝三载**（《旧唐书·贺知章传》），但具体月份史籍异文并列（一说同年八月；新唐书未明记月份），享年八十六。**鉴湖**——唐时人多称"贺监湖"或"镜湖"——便是因贺知章归乡时玄宗赐镜湖一曲为放生池之故。755 年的越州人提起贺监，是十一年前的事。

至于剡溪、若耶溪一带——**李白**天宝六载（747）后两次南游剡中，《梦游天姥吟留别》"我欲因之梦吴越，一夜飞渡镜湖月"——盛唐诗人入越，多走这条山水路线。**孟浩然**前此（开元天宝间）亦曾游越州。盛唐诗人笔下的越州，是一处**山水加古典**的双重叠合地——既可访兰亭、禹陵、若耶，又可登天姥、入剡溪。

755 年冬，安禄山起兵的消息传至越州大约是十一月底十二月初。会稽郡在江南腹地、距前线三千里——朝廷与民间皆未感切肤；但江南漕赋自此一战后将成为支撑唐廷半壁江山的命脉，越州作为江南东道大郡，承漕、出粮、调兵之责将日重——这一切尚在前面。对天宝十四载的越州人来说，这座城是**山水、古典、贺监故里**——一座江南文气最浓的州。

## 子条目

### 山阴县 / 会稽县（一城两县）

唐越州（会稽郡）治所——州治、郡治、县治三套同在一城。山阴县名取诸会稽山之阴（山北曰阴）；会稽县与山阴县**唐时同城分治**——城以纵街为界，街东属会稽县，街西属山阴县，一城两县。这种"一城两县"格局在江南诸大州中并不少见（苏州、杭州亦类此），盛唐人入越州，公文上分得清，街市上分不清。城周回不大，州衙、郡守、县衙、学官、市肆、寺观皆聚此一城。今**绍兴市越城区**即唐山阴 / 会稽两县旧址。

### 兰亭

会稽山阴之兰亭——州城西南约十二里，兰渚山下。**东晋永和九年（353）三月三日上巳**，王羲之与谢安、孙绰、支遁等四十一人修禊于此，曲水流觞，得诗三十七首，王羲之书序——《兰亭集序》行书三百二十四字，后世奉为行书第一。盛唐时兰亭已为州县岁修之名胜——可访者是后人重建之亭与古鹅池（传为王羲之养鹅处）；永和原构早已不存。

太宗朝（627-649）酷爱《兰亭》——奉敕摹本传世者有冯承素（神龙本）、虞世南、禇遂良、欧阳询诸家；冯承素摹本今藏故宫博物院，公认最近原迹。**原本陪葬昭陵之传**——见《新唐书·禇遂良传》、张彦远《法书要录》、五代《尚书故实》等——但是否真葬昭陵、是否五代温韬盗昭陵后亡失，自唐至今未有定论。755 年的越州人来兰亭，凭吊的是永和九年那场雅集——三百年前的事。

### 鉴湖（镜湖 / 贺监湖）

州城南。**东汉永和五年（140）**会稽太守**马臻**主持开凿——蓄会稽山三十六源之水，灌山阴、会稽两县八千余顷田。湖周回三百里许（《水经注·渐江水》语），是江南最大的人工灌溉湖。唐人称镜湖、贺监湖——后者因玄宗天宝三载（744）赐贺知章镜湖一曲为放生池而得名。**鉴湖**作为正式名见于盛唐文献——李白《送贺宾客归越》"镜湖流水漾清波"即此。

鉴湖至盛唐已运行六百余年——堤堰、水门、放水道历代修葺。湖中盛产菰、莼、菱、芡、鱼蚌——越州贡赋中"菱芡"一项即出此湖。湖北岸有马太守庙——州人岁祭马臻——这是越州地方信仰的一支。湖南岸便是会稽山北麓——湖与山一上一下，是越州最为人称道的一处景。

宋以后鉴湖渐被围垦——南宋时已半淤为田；至明清时鉴湖三百里之制几乎尽失，今所谓"鉴湖"仅存零星水面与河汊。**今日所见的鉴湖**与盛唐马臻所凿的鉴湖，不可同日而语。

### 若耶溪

州城南、会稽山中——自南而北流入鉴湖。传**西施浣纱**于此溪——《吴越春秋》《越绝书》皆有记。盛唐诗人入越者必访——李白《采莲曲》"若耶溪傍采莲女，笑隔荷花共人语"、王昌龄《采莲曲》皆涉此溪。溪两岸竹林深竹、清流见底——是越州山水画里最常入镜的一处。

### 山阴道

州城西南通诸暨、东阳道——沿会稽山北麓而行。《世说新语·言语》载王献之"从山阴道上行，山川自相映发，使人应接不暇"——这一句自南朝起便是越州山水的总评。盛唐人入越州，自山阴县出西南门走山阴道，左侧是会稽山岭、右侧是鉴湖与平原——一日之程，山光水色互为映发，便能体王献之之意。

### 会稽山

州城南，主峰**香炉峰**。会稽山自浙东而来，南北延展数百里，至山阴县城南达北麓。传**大禹**会诸侯于会稽——"会计"即"会稽"二字所出，意"会而稽功"。山中有**禹陵、禹庙**——传大禹葬地，唐时已立祠，岁时官祭；又有**大禹寺**——盛唐间为会稽佛寺之最大。山中还有诸多东晋以来高士隐居遗迹——王羲之故宅、谢安东山（在会稽山南麓上虞县境，谢安隐居处）皆在其中。

### 大禹陵 / 禹庙

会稽山南麓——传大禹葬地。《史记·夏本纪》"或言禹会诸侯江南、计功而崩，因葬焉、命曰会稽"。唐时禹陵已立祠——前殿祭禹，后陵封土。盛唐人入越州，必经禹陵一拜——这是华夏古帝的一处实存陵寝。**唐玄宗朝祭禹**作为国家级祭祀的常制，禹陵亦在岁祭之列。

### 王羲之故宅 / 戒珠寺

州城内西街——传王羲之故宅遗址。**戒珠寺**立于唐时（具体立寺年代史籍异文：一说唐初立，一说大历年间立；盛唐时是否已立寺有 hedge）——寺名取"戒珠"二字（一说取王羲之"以鹅易经"换《黄庭经》典故，一说取佛家戒为珍珠之意）。寺内有**墨池**——传王羲之临池学书、洗砚成黑之处。

### 越窑

越州境内**上虞、余姚、慈溪**一带——这是越窑窑场的主体分布区域，越州本州亦有。唐时为南方青瓷最重要的窑系——胎骨细薄、釉色青翠如玉，行销江淮、入贡长安。盛唐越窑产品在长安、洛阳、扬州、明州（今宁波）的唐代墓葬与遗址中常见出土。**"秘色瓷"**之名见于晚唐至五代——晚唐陆龟蒙《秘色越器》、宋初《吴越备史》、徐夤《贡余秘色茶盏》诗等始正式记"秘色"；755 年盛唐时越窑青瓷已近成熟期，但"秘色"尚未作为名号定型——此处加 hedge。

### 千秋观（贺监祠）

州城南、鉴湖之畔——贺知章归乡后玄宗赐宅为千秋观，亦称贺监祠。**天宝三载（744）正月**贺知章上疏请度为道士归乡，玄宗赐宅、赐镜湖一曲为放生池，亲制送行诗序。贺归乡数月后卒——此宅亦改为观。755 年盛唐人入越，可访千秋观一拜——贺知章是越州本朝最负盛名的人物。

## 历史长河（755 → 今）

**中晚唐**　至德二载（757）肃宗复改郡为州——会稽郡复称越州。安史之乱期间江南未陷——越州反成朝廷漕赋、兵源、文人避乱地。**元稹**长庆三年至大和三年（823-829）出任**越州刺史、浙东观察使**——治越七年，与白居易书信往来频密（"元白唱和"中越州一段是大宗），整顿鉴湖水利、修学校。中唐至晚唐**越州贡赋**在江南诸州中比重渐重——蜀锦衰、越锦起，越州绫罗与越窑青瓷成两大宗。**贺知章故乡**之名在中晚唐已成越州的文化招牌——白居易、刘禹锡、李商隐、温庭筠诸家诗中皆有越州题咏。

**五代至宋**　唐末乾宁三年（896）钱镠取越州——五代时越州属**吴越国**（907-978）—— 吴越王钱氏奉中原正朔，越州作为吴越国境内重镇，承中唐贡赋之盛、文风之久。**北宋**仍号越州——治会稽县（山阴县在熙宁九年 1076 并入会稽县后又复分，反复几次）。**南宋建炎三年（1129）**冬，金兵南下，宋高宗**赵构**自扬州渡江、过临安、避难至越州——**建炎四年（1130）正月**改元"绍兴"、并升越州为**绍兴府**——"绍兴"二字取"绍奕世之宏休、兴百年之丕绪"。**越州**之名自此退场，**绍兴府**之名自此立——这一年距 755 年已 375 年。

鉴湖在北宋仍有水利之实——但围湖造田渐多；至南宋鉴湖已半淤为田，三百里之制名存实亡。

**明清**　明清沿绍兴府之号——南宋以后陆游、徐渭、张岱诸家相继出绍兴，绍兴师爷之号、绍兴黄酒之兴皆此期事，但皆 755 之后，不入本 doc 详述。明清绍兴府兰亭屡修——**康熙三十七年（1698）**康熙南巡亲书"兰亭"二字立碑，**乾隆十六年（1751）**乾隆南巡又题诗——今兰亭"父子碑"康乾题字并存；距永和九年的兰亭雅集已近一千四百年。

**民国至今**　民国元年（1912）废府置县——绍兴府废，会稽县、山阴县合并为**绍兴县**。1949 年后改隶绍兴专区，**1983 年撤绍兴地区改设地级绍兴市**（绍兴县建制此前于 1981 年已有调整，后于 2013 年正式撤县设柯桥区）——今**绍兴市**，浙江省辖地级市，面积八千二百七十余平方公里、人口约五百三十万。市辖越城、柯桥、上虞、嵊州、新昌、诸暨——大体覆盖盛唐会稽郡境，但盛唐萧山县（今萧山区）已划归杭州。

**兰亭**今为绍兴市西南兰渚山下的兰亭风景区——亭、池、流觞曲水皆为历代重建（最近一次大规模重建是 1980 年代），永和九年原构早已不存。但"曲水流觞"的形制仍能体永和雅集大意——后人来此，凭吊的是一篇序、一段山水、一位书圣。

**鉴湖**今仅存东湖一段水面与若干河汊——三百里之制完全不存。绍兴老城内河道纵横、乘船穿巷的格局至 21 世纪仍可见——这一处水城肌理，是盛唐越州留下来的实物。

**越窑**窑址 20 世纪起陆续发掘——上林湖（今宁波慈溪）越窑遗址 1957 年起多次发掘，确认为唐五代越窑的主要窑场；2015 年起的**上林湖后司岙**发掘出晚唐五代**秘色瓷**实物——与法门寺地宫 1987 年出土的咸通十五年（874）秘色瓷恰能对应。盛唐 755 年的越窑器，今出土于长安、洛阳、扬州、明州诸地的唐代遗址 / 墓葬中常见。

至于贺知章——他写"少小离家老大回，乡音无改鬓毛衰，儿童相见不相识，笑问客从何处来"（《回乡偶书》），那是 744 年归乡之作。一千二百八十年过去——绍兴乡音几经更替，那座千秋观早已不存（具体湮废年代史籍未明记），但**"少小离家老大回"**那一句，至今是中国人写归乡最近本能的一句。

## 引文

王羲之《兰亭集序》（永和九年三月三日）：

> 永和九年，岁在癸丑，暮春之初，会于会稽山阴之兰亭，修禊事也。群贤毕至，少长咸集。此地有崇山峻岭，茂林修竹，又有清流激湍，映带左右，引以为流觞曲水，列坐其次。

——东晋永和九年（公历 353 年）三月三日上巳节，王羲之与四十一人雅集兰亭——盛唐人入越州必读此序、必访此地。

李白《送贺宾客归越》（天宝三载 744 春，长安作）：

> 镜湖流水漾清波，狂客归舟逸兴多。
> 山阴道士如相见，应写黄庭换白鹅。

——李白送贺知章归越州之作。"狂客"是贺知章自号"四明狂客"。"山阴道士"与"换白鹅"用王羲之以《黄庭经》换鹅典故——一篇里把贺知章、王羲之、山阴道、鉴湖四件越州事都装进去了。

贺知章《回乡偶书二首·其一》（天宝三载 744 归越作）：

> 少小离家老大回，乡音无改鬓毛衰。
> 儿童相见不相识，笑问客从何处来。

——贺知章离越州永兴县（742 改名萧山）少年游学、长安任职近五十年，八十六岁归乡作。这一首在唐人写归乡诗里位置无可替代。

《世说新语·言语》（南朝宋 刘义庆）：

> 王子敬云：从山阴道上行，山川自相映发，使人应接不暇。

——王献之（子敬）行山阴道之总评——盛唐越州之山水印象，自这一句定调。

《元和郡县图志·江南道·越州》（李吉甫 813）：

> 越州，会稽郡。开元户……天宝户……乡……西北至上都（长安）三千五百四十五里，西北至东都（洛阳）二千七百二十里。会稽山在县东南一十三里。镜湖在县南三里，周回三百一十里。

——这是 755 年前后越州户口、方位、山水的一手史源。具体户数 813 年时数据，755 年盛唐时大致同档（开元天宝间江南户口大体稳增）。
