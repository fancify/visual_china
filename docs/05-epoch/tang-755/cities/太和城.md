# 太和城

## 一句话

南诏王都，唐人口中"云南王"的居处；夹苍山洱海立城，是大唐西南方向最大邻邦的心脏——四年前（751）鲜于仲通伐南诏 6 万军覆于西洱河、去年（754）李宓 7 万再覆，两役累计 13 万死伤大半，骨陈洱海与城外。

## 形貌

太和城不在平地，倚山而立。

城址在点苍山佛顶峰南麓的缓坡上——西负苍山十九峰横列如屏，东瞰洱海一汪如镜；城基自山麓斜铺向湖滨，南北长约三公里，是一座狭长的山城。城墙夯土，依山势分内外两道：外为南北二城墙，自山腰一路下行直抵洱海岸；内城居中偏西、上倚陡坡，是宫殿区，称"金刚城"——南诏王廷正在此一处。这种"山为西墙、水为东堑"的格局，是太和城与中原任何一座方城都不同的地方——长安、成都的城墙四面齐筑，太和城却让一座山、一片湖代两面墙站着。

苍山十九峰自北向南横列，峰间有十八条溪水东流入洱海——南诏人称"苍洱"，自春至秋雪线退在峰顶，冬末复白。洱海古名**叶榆水**，又名西洱河——南北长约四十公里，状如人耳，故名"洱"。湖水自北向南绕太和城东缘而行，至城南纳点苍诸溪。湖中产鱼，岸植稻——叶榆河谷自汉以来已是西南夷里少有的农耕熟地。

城内的居处不如中原城密集。王宫与贵族第宅集中在金刚城及其下坡的缓地，外郭多为军营、市集、工坊、僧院。南诏自皮逻阁立国（738）以来不过十七年，建筑多木构，瓦少而草顶多；夯土城墙之外，木栅、藤栏处处可见——是一座新兴山国的样貌，而非千年沉淀的旧都。

城南三里有**德源城**，是阁罗凤兄弟一系早年的居处之一；城北二十里另有**大厘城**（今大理喜洲），为南诏六诏统一过程中重要城堡。太和城居中，是政令所出。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

太和城在唐人眼里是一座**新城**——新得只比唐立国（618）晚了一百二十年，比兴庆宫还要年轻。但它脚下这片土地，唐人记得更早。

汉武帝元封二年（前 109）在此置**叶榆县**，属益州郡——苍山、洱海一带正式入中原版籍的第一笔。东汉以下，叶榆县时设时废，**爨氏**在云南崛起，两爨（东爨乌蛮、西爨白蛮）分据滇东滇西数百年——这是六朝至隋唐之间云南的格局。唐武德四年（621）在洱海一带置**姚州**，贞观二十二年（648）析置**姚州都督府**，统辖洱海以东三十二州——这是大唐对洱海地区直接管辖的高点。

洱海西岸彼时不属姚州，而是**"六诏"**分立——"诏"在乌蛮、白蛮语中为"王"之意，六诏即六个相互独立的部落小国：蒙巂诏、越析诏、浪穹诏、邆赕诏、施浪诏，最南一支是**蒙舍诏**——居于巍山一带，因在六诏最南，故又称"南诏"。

南诏的崛起，是 8 世纪上半叶最迅速的一件事。开元中（约 720s 末-730s 初），蒙舍诏王**皮逻阁**继位——他借唐廷之力，先平定洱海以东的爨氏内乱，再北并越析、邆赕、施浪三诏，西并浪穹、蒙巂——开元二十六年（738），六诏归一，皮逻阁称王。**当年正月，唐玄宗诏册皮逻阁为"云南王"，赐名蒙归义**——南诏自此与唐建立正式的册封关系。开元二十七年（739）皮逻阁徙都太和（旧唐书载），至此太和城正式立为南诏王都——距 755 年仅十六年。

天宝七载（748）皮逻阁卒，子**阁罗凤**继位——唐再册阁罗凤为云南王。盛唐与南诏自 738 至 750 年的十二年间，是唇齿之欢——南诏借唐威定洱海，唐借南诏拒吐蕃东南面。

转折在天宝九载（750）。

### 此刻

天宝十四载（755），太和城的王位上是阁罗凤——在位已七年，年约四十出头。但他与大唐的关系，已在最近五年内由"册封藩属"翻为"敌国"。

事起于**云南太守张虔陀**。天宝九载（750），张虔陀向阁罗凤索贿不遂，反将阁罗凤之过密奏长安，且对南诏夫人无礼——阁罗凤大怒，发兵攻陷姚州，杀张虔陀，掠子女金帛——这是南诏与唐的第一次正面战事。

朝中执此事者是**杨国忠**——其时刚继李林甫为右相、兼遥领剑南节度使。杨国忠决定征讨南诏，以建武功。

**第一次南征**：天宝十载（751）四月，剑南节度使**鲜于仲通**率剑南、山南西道兵八万，自姚州南下伐南诏。阁罗凤先遣使请降谢罪，并以"吐蕃强我西北"为辞——鲜于仲通不许，进兵至西洱河（洱海）。阁罗凤求救于吐蕃。两军会于西洱河——**唐军大败，死六万人**（《资治通鉴》卷二一六 天宝十载夏四月条；《新唐书·南蛮传》《旧唐书·南诏传》均载）。鲜于仲通仅以身免——杨国忠扣压败报，反以战功上奏，赐鲜于仲通京职。这一战之后，阁罗凤北面与吐蕃赞普通好；同年冬，吐蕃册阁罗凤为"赞普钟"（义弟），号"东帝"——南诏正式叛唐附蕃。

**第二次南征**：天宝十三载（754）六月，杨国忠再遣**剑南留后侍御史李宓**率兵七万（一说十万），自姚州、戎州两路并进，意在一举灭南诏。阁罗凤诱敌深入，引唐军至太和城下西洱河畔——预设伏兵，又值瘴疠盛夏，士卒病疫者过半。两军接战，**唐军全军覆没，李宓殁阵**（一说被擒，旧唐书与新唐书并载，史家多取殁阵说）。前后两役，唐军损失累计十三万至十八万——是**天宝中后期唐对外用兵最惨重的一次失败**，至安史之乱前为巅峰。蜀中、剑南诸州户口为之一空——杜甫《兵车行》"边庭流血成海水，武皇开边意未已"，"君不见青海头，古来白骨无人收"——所指虽含河西陇右诸征，西洱河两役为其中分量最重的一笔。

至天宝十四载冬，西洱河岸边——太和城南数里——白骨未尽。

此刻的太和城，是一座刚刚以血印重立的王都。阁罗凤已下令**收唐军战死者骸骨**，于太和城西二十里筑**"京观"**（垒尸为山的纪念冢；一说在城南，史料异文），后世称"万人冢"，又称"大唐天宝战士冢"——这一冢，是阁罗凤一面纪念战胜、一面也存留与唐复和的伏笔。

阁罗凤本人此刻的态度并不全然反唐。十一年后大历元年（766），他在太和城西北立**南诏德化碑**，碑文 3800 字——明文写：

> 我上世奉中国，累封赏，后因奸臣离间，遂至失和……星霜屡周，未尝忘也。

——这是他自己留下的辩词。755 年时此碑尚未立，但碑文所记的内心立场，可作 755 年太和城王廷的一份注脚：阁罗凤与唐为敌，自认是被迫——他要给后人留下复和的余地。

755 年时阁罗凤朝中重臣有**清平官**段忠国、宰相**郑回**——郑回原是唐西泸县令，西洱河战中被俘，阁罗凤命教王子（异牟寻一辈）汉学——这一脉文教，是日后南诏与唐复合的种子。

755 年时太和城外的局势：北面与吐蕃通和已五年，南面六诏旧地已尽并；东面姚州都督府已废，唐姚州兵马于第二次南征中尽殁；东北面剑南节度使的边将仍据雅州、黎州、嶲州——但已无攻势。整个云南，**自洱海至滇池、自滇西至滇东南**，名义上皆是南诏王土。

### 远在河朔

天宝十四载十一月初九，安禄山在远在河朔的范阳起兵——四五千里外的消息传到太和城已是月余之后，经成都、经姚州、辗转入山。对阁罗凤而言，这是个不坏的消息：大唐自顾不暇，西南方向再无大规模征伐之力。次年（756）玄宗入蜀避乱，南诏与唐之间隔着剑南节度的残兵——但阁罗凤并未乘机北上，只蚕食蜀边数州——他在等。

等到阁罗凤之孙**异牟寻**继位（778），在汉学**郑回**的劝导之下，贞元十年（794）南诏与唐于**苍山神祠**会盟——盟于太和城西的苍山中。**南诏与唐之间四十三年的敌国关系，由此终结**。这是 755 年这座太和城埋下的最远一笔伏笔。

### 金刚城

太和城内城，居全城正中偏西，上倚苍山佛顶峰南麓陡坡——南诏王宫与王廷所在。城名"金刚"，取佛教"金刚不坏"之意——南诏自皮逻阁、阁罗凤起便奉佛，王廷称号、宫殿命名多取佛义。考古勘探显示金刚城范围约东西 300 米、南北 200 米，有大型夯土台基若干——疑为王殿、议政堂、寝殿基址。755 年时阁罗凤的朝政正在此一处推行。

### 南北城墙

太和城最显著的遗存。南北二墙自苍山中段山腰起，分别向南、向北斜下，至洱海岸停止——两墙间南北长约三公里，东西宽则视山势变化。墙体夯土，残高今仍存二至五米，宽十米——是云南境内现存最完整的唐代城墙遗迹。山为西墙、洱海为东堑——南北二墙之间，即太和城外郭范围。

### 苍山（点苍山）

太和城西的依靠。十九峰自北向南横列约四十公里，主峰马龙峰海拔约 4122 米——夏秋雪线退至峰顶，冬末复白。峰间十八溪东流入洱海——"苍山十九峰、十八溪"为南诏地理的核心口诀。南诏王廷视苍山为**神山**——苍山神祠（一说在马龙峰下）是南诏王每岁亲祭的国家级祠庙。贞元十年（794）唐使袁滋与南诏王异牟寻就在苍山神祠会盟，盟文写"上请天地水三官、五岳四渎、及管川谷诸神灵同请降临，永为证据"——苍山作神在南诏宗教中分量最重。

### 洱海（叶榆水 / 西洱河）

太和城东的天然护城湖。古名叶榆水，汉时已著于《史记》《汉书·地理志》——"叶榆"为乌蛮、白蛮古语转译。湖南北长约四十公里、东西宽三至九公里，状如人耳，故唐人始称"洱海"。湖中产鲫、鲤、弓鱼（一种洱海特有冷水鱼）；岸植稻、麻，渔耕之业自汉至今未绝。**西洱河两役（751、754）即发生在洱海岸边——古战场遗迹在太和城南至下关（洱海南口）一带**。

### 南诏德化碑（766 立，碑址在城西北）

太和城西北一里许（今大理市太和村西）。大历元年（766）阁罗凤命刻——立碑时距 755 年正好十一年，距阁罗凤即位（748）十八年。碑高约 3.97 米、宽 2.46 米、厚 0.6 米，**碑文阴阳两面共约 3800 字**——记述自皮逻阁立国（738）至 766 年立碑时南诏与唐的关系、自辩天宝战争之由、宣告南诏与吐蕃通和的政治立场。

碑文为唐人**郑回**（原西泸县令，西洱河战中被俘留南诏，任清平官）所撰，书法笔意近盛唐——是云南境内唯一一通**唐代当代书法实物碑**，金石学价值极高。明清以来碑面多有风化、缺字，今存可识者约 800 字（清乾隆至今历代摹拓本可对照）。

碑现存大理市太和村原址——立碑之处即立碑之所，唐物在唐地，**自 766 年起未曾移动一次**。

> 755 年时此碑尚未立——但碑文所记的政治立场，可作 755 年阁罗凤心境的一份注脚。

### 万人冢（大唐天宝战士冢）

太和城南、西洱河岸边。**天宝战死唐军骸骨之埋处**——751、754 两役死者，阁罗凤命收骨垒葬，立京观以警后人；后世从中原汉文角度则称"万人冢"。今墓址在大理市下关将军洞附近——历代屡有修葺，碑刻多有。明清以来云南汉文化日盛，万人冢渐被视为"客死他乡的唐军英灵"——清季已有汉人士人岁岁致祭。755 年时此冢新立，骨上之土犹松。

### 德源城（城南三里）

蒙舍诏王族早年居处之一——皮逻阁、阁罗凤一系起自德源。城规模较小，今址在大理弥渡一带（一说在太和城南近郊，史料异文未定）。755 年时仍是阁罗凤宗室别业。

### 大厘城（城北二十里）

今大理喜洲。南诏六诏统一过程中重要城堡——皮逻阁攻浪穹、邆赕诏时尝立此为基地。755 年时仍为南诏北面军事重镇——后世异牟寻迁都至**羊苴咩城**（即喜洲附近的阳苴咩，今大理古城西），便是顺着这一带的旧基础。

### 阳苴咩城（说明：755 年时尚未立为都）

太和城北十里许（今大理古城一带）。**大历十四年（779）异牟寻迁都于阳苴咩**——太和城作为南诏王都自 739 至 779 年止，凡四十年。**755 年时太和城仍为正都，阳苴咩尚未升格为都城**。后世大理国（937-1253）都阳苴咩，今大理古城即承此地脉——但这是另一段故事了。

## 历史长河（755 → 今）

**中晚唐**　太和城作为南诏王都自 739 至 779 年凡四十年。大历元年（766）阁罗凤立南诏德化碑——这是太和城在历史上最郑重的一刻。大历十四年（779）阁罗凤卒，孙异牟寻继位，即迁都阳苴咩城——太和城自此降为陪都，但金刚城仍是南诏王廷重要的祭祀与避暑之地。贞元十年（794）异牟寻与唐使袁滋盟于苍山——南诏与唐复和。从此南诏自吐蕃转回大唐藩属。会昌至大中（840s-850s）南诏国势再盛，西破骠国（缅甸），东抚滇黔；唐宣宗大中十三年（859）南诏改国号"大礼"，此后又改"大封民"——但太和城始终在苍洱之间，未废。

**唐末五代**　唐昭宗天复二年（902），南诏权臣郑买嗣篡位灭蒙氏，立"大长和国"——南诏 164 年（738-902）至此结束。其后再历"大天兴"（928-929）、"大义宁"（929-937）两个短命政权——三十五年三易其国。**后晋天福二年（937），段思平起兵灭大义宁，建大理国，都阳苴咩城**——大理国于太和城旧址不再设都，太和城自此完全退出王都之列。但金刚城内的祭祀仍在——大理国主每岁亲祭苍山神祠，仍以太和城为出发点。

**宋元**　大理国（937-1253）三百一十六年，太和城渐为农居。北宋开宝二年（969）宋太祖**玉斧划大渡河为界**——"此外非吾有也"——宋廷自此放弃对云南直辖；大理国与北宋之间隔着一道大渡河，太和城遂在中原视野外淡出近三百年。元宪宗三年（1253）**忽必烈率蒙古军南下灭大理**，自宁夏六盘山经吐蕃道入滇——太和城在大理城西十里，被攻破时已是农田废墟。至元十一年（1274）赛典赤·赡思丁立云南行省于昆明——云南政治中心自此自大理东移至昆明，太和城旧址再无政权依附。元代以下，太和城遗址只剩"南诏故都"之名，城墙尚高，地里耕作。

**明清**　明洪武十五年（1382）汤和、傅友德平云南，大理路改大理府，太和县名（注：明清"太和县"为大理府附郭县，沿用古城之名，与今太和城遗址有地理关联但不完全重合）沿用至清。明清两代屡有文士访太和城遗址——杨慎（1488-1559，明状元，谪戍云南三十年）撰《云南山川志》《滇载记》，详记南诏遗迹；清李元阳（明嘉靖滇人）《大理府志》、王崧《云南备征志》（清道光）皆有"太和故城"条。万人冢、南诏德化碑历代均有重修——明嘉靖、清乾隆、道光皆立过保护性碑。**碑面字迹自唐至清屡经摹拓，是中国西南唯一一处唐代金石实物的延续保护**。

**民国至今**　1956 年云南省公布太和城遗址为省级文物保护单位。1961 年南诏德化碑被列为**第一批全国重点文物保护单位**——是云南境内仅有的两处唐代石刻（另一处为剑川石钟山石窟）之一。1990s 至 2010s 之间，云南省文物考古研究所对太和城遗址多次发掘——金刚城内大型夯土台基、南北城墙、城内坊里布局陆续清理；2010 年代后陆续辟为遗址公园。今日自大理古城（阳苴咩故址）沿苍山东麓向南走十里，便见太和村——南北二城墙的夯土残段仍立在田野间，最高处约五米；南诏德化碑被覆以碑亭保护，碑文残字隔玻璃可辨。

至于盛唐那座太和城——金刚城上的木构殿宇早已无存，西洱河岸的京观早被农事削平，苍山十八溪的水仍在每岁春夏汇入洱海。**只有那块碑，立在它 766 年立的那个地方未曾挪过。**

## 额外章节

### 南诏王世系（738-755 间）

| 在位 | 王 | 唐廷册封 | 关键事 |
|---|---|---|---|
| 728-748 | 皮逻阁 | 开元二十六年（738）册"云南王"，赐名蒙归义 | 并六诏，738 立国，开元末天宝初徙都太和 |
| 748-779 | 阁罗凤 | 天宝七载（748）袭"云南王" | 750 攻姚州起战，751 / 754 两役破唐军，751 受吐蕃册"赞普钟"，766 立南诏德化碑 |

755 年时阁罗凤正在位第八年。其下设清平官（相当于宰相）六人——文献可考者有**段忠国**、**赵铎**、汉人**郑回**等。郑回此时正以汉学教王孙异牟寻一辈——这一支文教，是日后南诏与唐复合的种子。

### 天宝战争两役对照（751、754）

| 项 | 第一役（751） | 第二役（754） |
|---|---|---|
| 唐方主帅 | 剑南节度使鲜于仲通 | 剑南留后侍御史李宓 |
| 唐军兵力 | 八万 | 七万（一说十万） |
| 战地 | 西洱河（洱海一带） | 太和城下西洱河畔 |
| 结果 | 死六万，鲜于仲通仅以身免 | 全军覆没，李宓殁阵（一说被擒） |
| 南诏方 | 阁罗凤求救吐蕃 | 阁罗凤诱敌深入，伏兵 + 瘴疠 |
| 朝中处置 | 杨国忠扣败报，反叙战功 | 杨国忠再奏请征 |
| 唐方后果 | 蜀中户口锐减 | 蜀中再空，至安史前积怨已深 |

前后两役损失合计十三至十八万——盛唐对外用兵单地区损失最大的一笔，且是**安史之乱前唐军兵力最大的一次外耗**。安禄山十一月起兵时，唐廷一边已无可调之西南机动兵——杜甫《兵车行》所指，半实在这一处。

### 南诏德化碑文要点（766 立，可作 755 年王廷心态之回望注脚）

碑文阴阳两面共约 3800 字，今存可识约 800 字。要点：

1. **追述六诏归一**——皮逻阁开元二十六年（738）受唐册"云南王"，南诏与唐之友始
2. **自辩天宝战争之由**——归咎于"奸臣"张虔陀（云南太守）与杨国忠（剑南节度遥领），强调阁罗凤"先请降谢罪不许"
3. **宣告南诏与吐蕃通和**——751 年受吐蕃册"赞普钟"为"东帝"，与唐为敌国
4. **保留复和余地**——"我上世奉中国，累封赏……星霜屡周，未尝忘也"——明示有朝一日仍愿归唐
5. **嘱后世**——"叛汝者，吾雠也"，即对将来与唐为敌者的警告

——这五条，是 755 年阁罗凤王廷已有的政治立场。十一年后 766 年立碑时，只是把这立场镌进石头里。**贞元十年（794）异牟寻与唐复和，正是兑现碑文第四条**。一块碑，铺路四十年——这是中古中国西南最长的一次政治伏笔。

### 太和城与中原方城的对照

| 项 | 太和城 | 长安 / 益州（同时代中原方城） |
|---|---|---|
| 平面 | 狭长山城，南北约 3 公里 | 方城，棋盘格 |
| 城墙 | 二道（南北墙；西用山、东用湖代墙） | 四面齐筑 |
| 内城 | 金刚城，倚陡坡 | 宫城居北、皇城居中 |
| 建筑 | 木构为主，瓦少草顶多 | 夯土 + 瓦顶 + 砖石 |
| 立都年龄 | 738 立国 / 739 徙太和，755 时仅十六年 | 长安自隋大兴（582）算起约 170 年；益州自张仪（前 311）算起千年 |
| 与自然关系 | 苍山为屏、洱海为堑——把地形拉进城防 | 把地形挡在城外、城内自成秩序 |

——太和城是 8 世纪中叶西南山国第一次按照山水做城的代表。**它不模仿长安**——这一点与新罗庆州（部分仿长安）颇不相同。太和城就是太和城，苍山与洱海决定了它的形。
