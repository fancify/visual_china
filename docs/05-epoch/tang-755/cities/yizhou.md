# 益州

## 一句话

天宝十四载冬，西南半个天下唯一的大城——再过半年，将成为玄宗仓皇南奔的终点。

## 形貌

益州城在岷江冲积出来的成都平原正中，一座方城摊在平地上。

外郭周长唐时约二十余里，城墙夯土，比长安低、比长安窄，却比长安活——城里没有那么森严的坊门夜禁。两条江穿城而过：北面是郫江（一名内江），南面是检江（一名外江、流江）——两江本出岷山一源，秦时蜀守李冰开成都两江，从此分作两支，绕城而过，又在城东南合流。江上诸桥之中最有名者，是城南的**万里桥**——三国蜀汉建兴年间，丞相诸葛亮在此送费祎使吴，临行叹"万里之行，始于此桥"，桥名自此而来。

子城（蜀郡府衙、益州大都督府治所）居全城正中，是历代蜀守、刺史、节度使衙署所在——秦张仪筑成都城时即立此基，汉蜀郡治、蜀汉都城、成汉、梁益州、隋蜀郡，皆在此一处。子城西南数里，浣花溪自西郊流入——溪水清浅，岸柳依依，是成都西郊踏青最盛之处。子城东南，**锦官城**别立——本汉蜀郡专设的官营织锦工区，蜀锦自此出，行天下；唐时锦官城已与大城连成一片，"锦城""锦江""锦里"诸名皆由此出。

城里物候比长安温润得多。冬不甚寒、夏多阴雨，蜀人称之"少雪多云"。春日浣花溪畔海棠盛开，秋日满城芙蓉——孟蜀以后才有"芙蓉城（蓉城）"之名，盛唐时只是芙蓉自荣。城外西北五十里岷江出峡处，秦李冰所开的**都江堰**至 755 年已运行近千年——分江水以灌成都平原，使其"水旱从人，不知饥馑"（《华阳国志》语）。蜀地之所以为蜀地，江山形势之外，全在这一处水利。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

益州在唐人眼里不是一座新城，而是一座旧城——旧得几乎能跟长安、洛阳争长。

最早是**古蜀国**——传说中蚕丛、柏灌、鱼凫、杜宇、开明五代蜀王，建都在郫邑、广都、瞿上一带。唐人读《华阳国志》、读《蜀王本纪》，知道这一片地从夏商以前便已有国——只是"不通中国"，秦惠王命司马错伐蜀（前 316），蜀始入中夏。秦惠王后元十一年（前 311）派蜀守张仪、张若按秦都咸阳之制筑成都城——周回十二里、高七丈，分大城（少城）两重——这是益州城最早的骨架。**唐时的成都子城，跟张仪那座城仍是同一处**。

汉武帝时设蜀郡，益州刺史部之治。蜀郡守**文翁**于景帝末年（约前 141）在城南立学官，遣郡吏赴京师从博士受业——蜀地由是大化，"蜀学比于齐鲁"（《汉书》语）。文翁石室是中国第一所地方官立学校，到唐时仍在城南旧址讲学，唐人入蜀者，必访石室一观。

三国时蜀汉昭烈帝刘备以此为都，章武元年（221）即帝位。诸葛丞相秉政期间，蜀汉都城所恃者亦在此一城。蜀汉亡（263）、成汉（304-347）、东晋梁益州、北周益州、隋蜀郡——魏晋以后这地方几次易主，但**城始终在**，是西南唯一未曾废弃的旧都。唐人登城南**武侯祠**——祠原与汉昭烈庙合并，相传南北朝时已立，唐时香火甚盛——望柏森森，便能想到那段三国旧事。盛唐人写"出师未捷身先死"，未必非杜甫不可——是一城千年沉淀下来的共同记忆。

唐武德元年（618）立益州大都督府——开元后定为大都督府、剑南节度使治所。盛唐人称"扬一益二"：天下繁华之都，扬州第一，益州第二——长安洛阳为政治京都，不入此谚。

### 此刻

天宝十四载（755），益州是**剑南节度使**的治所——开元七年（719）始设节度使于此，统辖剑南一道及西南羁縻夷州，控南诏、抚吐蕃东南面、护蜀道。这一岗位的轻重远超内地一般节度——蜀地富、地形险、与南诏前线，三事于一身。

但 755 年这一年，剑南节度使的政事乱得很。

天宝十年（751）四月，前任节度使**鲜于仲通**率兵八万南征南诏，于西洱河（今洱海一带）大败，士卒死六万人——这是天宝间唐对南方第一次惨败。败讯报至长安，杨国忠正继李林甫为右相、兼遥领剑南节度使——他扣压败报，反以战功上奏。天宝十三载（754）六月，杨国忠再遣**剑南留后侍御史李宓**率兵七万再征南诏，又败于西洱河——七万之众，全军覆没，李宓本人殁阵（一说被擒）。两年两次大败，蜀中十数州户口为之一空——杜甫《兵车行》里"边庭流血成海水，武皇开边意未已"，所指即此。

至天宝十四载冬，剑南节度使仍由杨国忠在京遥领。蜀中实务，名义上由副大使知节度事打理；西南南诏与吐蕃前线，则由几位边将分守雅州、黎州、嶲州一带——但朝廷威信已伤。

成都城本身仍是繁华。**扬一益二**的"益二"，是这一年的实况：西市的胡商不让长安，蜀锦的商船经岷江、长江出三峡至江陵，再下扬州，远销海外；浣花溪畔的别业、城西的射圃、子城东南的酒肆——李白《登锦城散花楼》"日照锦城头，朝光散花楼"，写的便是这种繁丽。

对天宝十四载的成都人来说，长安离他们一千七百里——隔着秦岭、隔着大巴山、隔着蜀道八条。安禄山十一月初九在范阳起兵的消息，传到成都得再多一个月。他们尚不知道，再过半年，整个长安朝廷将循金牛道翻越剑门——把"益州"这两个字暂时改写成"行在"。

### 即将到来

天宝十五载（756）六月初九潼关失守，玄宗夜出延秋门西奔。十三日宿马嵬驿——禁军杀杨国忠，缢杨贵妃。皇太子李亨北赴朔方，玄宗一行继续南行入蜀。自长安经金光门外大路、武功、扶风、陈仓、入散关、过凤州、兴州、利州、剑州、绵州、汉州——七月二十八日（公历 8 月 28 日）抵成都。**前后行程一千七百余里，耗时五十余日**。这是 755 年这一座城最后一次以"益州"之名见史——次年至德二载（757），玄宗、肃宗父子先后以"南京""成都府"二诏改易其名（《旧唐书·肃宗本纪》两载并存，史家通行的折中说法是 757 年十二月先称"南京"、758 年三月正式定名"成都府"）。

蜀中半年的玄宗朝廷里，已无杨国忠——成都成了一个无主的避难所、又是一个安史乱里第二个权力中心。同一时间在灵武称帝的肃宗，与在成都的太上皇之间，隔着千余里。

杜甫此时还在长安（实则正困于沦陷区）；李白则在江南、于至德元载（756）冬投永王李璘幕——三位最大的盛唐诗人在 755 年冬未及一见，未来三年内，他们的轨迹都会与这座益州城以不同方式连上。

### 子城（蜀郡府衙 / 益州大都督府）

全城正中，自秦张仪（前 311）筑城起便是衙署所在。汉蜀郡守、蜀汉皇宫、成汉宫城、北周益州刺史府、隋蜀郡治、唐益州大都督府——千年之间，这一处衙址未曾大迁。**子城**这两个字，在 Tang 文献里几乎与"成都"互训。756 年玄宗入蜀，下榻处即子城——子城的正堂临时改作"行宫"。

### 文翁石室

子城东南。汉景帝末（约前 141）蜀郡守文翁所立学官——**中国第一所地方官立学校**。文翁选郡中俊士赴京从博士受业，归而教化蜀人，"由是大化"。东汉光武中元年间（约 56-57）官学重修——传说有石室遗构，石室因此得名。唐时文翁石室仍在原址讲学——剑南诸州贡举生、官学生，多在此就读。蜀中"文章之地"的名声，文翁是源头。

### 武侯祠

子城南郊，与汉昭烈庙（刘备庙）合在一处。诸葛亮卒于建兴十二年（234）——蜀汉景耀六年（263）方在沔阳立庙；成都的武侯祠相传南北朝至隋间已立，唐时已是城南最大祠庙之一。祠前古柏数十株，相传为蜀汉旧植——杜甫至德二载（757）后入成都，作《蜀相》"丞相祠堂何处寻，锦官城外柏森森"——所写即此祠。755 年时祠在，柏在；那首诗未在。

### 浣花溪

子城西南数里。锦江支流，自西郊岷江余水汇入。溪水清浅，岸柳成行——蜀中士女三月浣花、四月浴溪，为成都西郊春景第一。755 年时浣花溪畔尚是寻常溪谷，亭台稀。乾元二年（759）冬，杜甫自秦州、同谷一路入蜀，过年关至成都，次年春于浣花溪畔卜居，自建草堂——**浣花溪自此成为唐代诗史第一现场**。但 755 年时，这一节尚未开始。

### 杜甫草堂（759 后立）

浣花溪畔。乾元二年十二月，杜甫举家入蜀，依友人剑南节度使副大使**裴冕**、彭州刺史**高适**、成都尹**严武**周济，于上元元年（760）春于浣花溪西畔卜地立屋——"诛茅初一亩，广地方连延"。草堂前后凡居四年（760-765），其间作诗二百四十余首——《茅屋为秋风所破歌》《蜀相》《春夜喜雨》《江畔独步寻花》《登楼》《丹青引赠曹将军霸》皆此地作。永泰元年（765）严武卒，杜甫失依，**遂离草堂顺岷江、长江东下**，未再回。755 年时草堂尚未立——但既写益州，不能不及。详见 [杜甫与成都](#杜甫与成都-758-770)。

### 大慈寺（敕额 757 后）

子城东南。寺址相传后赵宝掌禅师（5 世纪传说）始营，至德二载（757）玄宗驻跸期间敕赐"大圣慈寺"匾——是为盛唐至中晚唐蜀地第一大寺。寺内最盛时有院九十六、塔楼阁堂共八千五百四十二间，玄奘亦曾在此求戒（一说幼年居成都净慧寺，唐两说并存）。寺内壁画为蜀中第一——晚唐五代孙位、张玄、贯休皆有壁画在此寺，史称"大圣慈寺壁画甲于天下"。755 年时寺已有，但**敕额"大圣慈寺"在 757 年才下**——这一点要分清。

### 万里桥（七星桥南桥）

子城南，跨检江（南河）。三国蜀汉建兴年间，诸葛亮在此送费祎使吴："万里之行，始于此桥"——桥名自此而来。Tang 时为石墩木梁桥，是成都南门外最重要的渡口——蜀中商船自岷江出锦江、过万里桥至岷江，再下乐山、嘉州，入长江。杜甫《狂夫》"万里桥西一草堂，百花潭水即沧浪"——草堂即在万里桥西不远。

### 锦官城

子城东南。本汉蜀郡专设的官营织锦工区——蜀守命濯锦于此，水绿丝光，蜀锦自此出。唐时锦官城已与大城连成一片，作为蜀锦集散区与官营织造司所在——丝织业之盛，长安、扬州皆向蜀中购锦。"锦官""锦城""锦江""锦里"诸名皆由此出。杜甫"锦官城外柏森森"，李商隐《杜工部蜀中离席》"美酒成都堪送老，当垆仍是卓文君"——蜀地形象自唐起便与"锦"字相系。

### 锦江

绕城南而过的检江别名。一名濯锦江——蜀人濯锦其中，水绿如蓝。锦江与城北郫江合流于城东南，是成都对外水运第一道——蜀中纲粮、蜀锦、铜铁，皆自此出。Tang 末韦庄入蜀，写"满街杨柳绿丝烟，画出清明二月天。好是隔帘花树动，女郎撩乱送秋千"——锦江上的二月。

### 散花楼

子城东南角，本隋蜀王杨秀立，名"散花楼"——李白开元年间游蜀，作《登锦城散花楼》："日照锦城头，朝光散花楼。金窗夹绣户，珠箔悬银钩。飞梯绿云中，极目散我忧。"——这首是 24 岁的李白第一次写成都。Tang 末楼毁，今址不可考。

### 草堂寺（蜀地）

子城西，浣花溪附近。后秦时鸠摩罗什译经地——这一处草堂寺与长安户县（鄠县）那一处草堂寺为两处同名寺；蜀人多以浣花溪畔后来杜甫草堂的诗意命名混作"草堂"，文献里多有混淆。唐时寺尚在，规模不大。

### 净众寺 / 净慧寺

子城北。一说玄奘幼年（约 614-622 间）随其兄长捷在此学经——此说见唐《大唐故三藏玄奘法师行状》，但玄奘故里实在洛阳，蜀中习业说与洛阳少年说并存于唐宋史料。如取蜀中说，则净慧寺是玄奘出家前最后一所学寺。

### 都江堰（彭州导江县 / 灌县地，城西北约 50 里）

岷江出峡进入成都平原处。秦昭王末年（约前 256-前 251）蜀郡守**李冰**率众开凿——分岷江为内江外江，内江入郫江、检江灌成都平原 数百万亩，外江入岷江主流。**水旱从人，不知饥馑**——《华阳国志·蜀志》记此堰功最详。Tang 时归彭州导江县（开元 18 年/730 立彭州，导江县先后归彭州、蜀州数易其属），堰仍以李冰旧法岁修：每年冬季截流淘滩、立都江鱼嘴。一座秦代水利工程到 Tang 已运行近千年，到今再千二百余年——这是中国仍在使用的最古水利。**作为益州近郊地标列在此 doc 内；考虑独立性原则下，若 S4b 后期细化古迹分册，可拆出独立 doc。**

### 青城山（蜀州青城县，城西北约 100 里）

都江堰西南三十里处。道教十大洞天之第五，五岳之外的道教祖山。东汉张道陵晚年隐此创天师道——青城山是中国本土宗教的诞生地之一。Tang 时山中有常道观、上清宫、天师洞、朝阳洞等。玄宗朝奉道至盛，青城山香火甚旺。**列在此 doc 内，亦可拆为独立山水 POI doc。**

## 历史长河（755 → 今）

**中晚唐**　至德元载（756）七月二十八日，玄宗抵成都——这一座蜀城第一次成为唐朝廷的"行在"。至德二载（757），玄宗、肃宗父子先后两诏改益州为"南京""成都府"——`南京` 名号短暂，`成都府` 之名自此通行至五代。乾元二年（759）冬，杜甫携家入蜀——四年间在成都浣花溪畔筑草堂、与高适、严武交往，作诗二百四十余首。永泰元年（765）严武卒，杜甫离蜀。这四年是益州城最浓的一笔诗史。中晚唐之间，剑南节度使屡易其人——韦皋（785-805）守蜀二十一年，南破吐蕃、收维州，是蜀地史上最长久的节度使；蜀地由此稳定半个世纪。

**晚唐五代**　唐末黄巢之乱（880）兵未入蜀；僖宗中和元年（881）冬，避乱入蜀至成都，居二年余——又是唐廷第二次以蜀为行在。乾符末王建以"忠武八都"军入蜀，景福二年（893）任西川节度使，天复三年（903）封蜀王。天祐四年（907）唐亡，王建于成都称帝，国号**蜀**（史称前蜀），都成都——这是益州城自蜀汉以来第二次成为皇都。前蜀（907-925）二代二十年而亡；后唐灭蜀后，孟知祥再据蜀，明德元年（934）建**后蜀**——后蜀（934-965）三十一年，孟昶时蜀中"宫娥花蕊夫人"赋"芙蓉城"——"蓉城"之名自此而传。乾德三年（965）宋灭后蜀。

**宋元**　北宋于蜀地置益州路（咸平四年/1001 改为益州、利州、梓州、夔州四路），治成都府。北宋成都富庶过于盛唐——交子（世界上最早的纸币）在成都印行（约 1023）。蒙古入蜀（13 世纪中期）极惨——四川人口自南宋时一千二百万降至元末不足一百万，成都数次屠城。元世祖至元二十三年（1286）置四川行中书省，治成都——"四川"之名由此立，沿至今。

**明清**　明洪武四年（1371）置成都府——府治承宋元旧城。明末张献忠据蜀（1644-1646）建"大西国"，再屠成都。清顺治四年（1647）入蜀，康熙年间陆续从湖广、江西等省迁民入川（"湖广填四川"）——成都人口由是再增。康熙至乾隆间陆续修葺武侯祠、杜甫草堂——杜甫草堂自宋元以来屡经废兴，清康熙年间重立今址；武侯祠在明代与汉昭烈庙正式合并为"君臣合祠"，乾隆间再修。盛唐之"益州"，到清时已化为"成都府附郭县（成都县、华阳县）"。

**民国至今**　1928 年正式设成都市。1949 年后为四川省省会。1958 年起进入大规模工业化——但因山形所限，旧城仍多保留。今日成都的旧城核心，仍大体在唐宋成都子城的轮廓内。20 世纪 50 年代以来陆续发掘武侯祠汉昭烈庙、杜甫草堂、大慈寺、文翁石室遗址——尤其 2003 年发现的**江南馆街唐街遗址**，揭示了唐五代成都城东南的街道格局与排水系统，是迄今最完整的唐代南方城市街区考古实证。都江堰至今仍灌溉成都平原约 1000 万亩——一座秦代水利，**距李冰之筑已两千二百余年**。

至于盛唐那一座益州——子城衙署、文翁石室、武侯柏林、浣花溪畔——其址或叠或失。可走的，只有都江堰一处——分江的鱼嘴每岁仍在水里。

### 杜甫与成都（758-770）

杜甫与成都的关系，是中国诗史最重的一段。

759 年冬，他四十八岁——自长安、华州、秦州、同谷一路南行，腊月入成都。760 年春，于浣花溪西畔卜地立草堂——"诛茅初一亩，广地方连延，杜门不复出，久与世情疏"。四年间作诗：

| 年份 | 作 | 备注 |
|---|---|---|
| 760 春 | 《卜居》《蜀相》《为农》《狂夫》《江畔独步寻花》 | 入草堂初定，初游武侯祠 |
| 760 秋-761 春 | 《茅屋为秋风所破歌》《春夜喜雨》《江村》 | 茅屋被秋风掀，作第一长篇 |
| 762 | 《奉送严公入朝》《喜雨》《登楼》（一说 764） | 严武入朝、徐知道之乱，避走梓州 |
| 763-764 | 《闻官军收河南河北》《登楼》《丹青引赠曹将军霸》 | 安史平、回成都 |
| 765 | 《去蜀》《旅夜书怀》 | 严武卒，杜甫离草堂顺岷江、长江东下 |

四年里他写下二百四十余首——草堂时期是他一生写作密度最高、心境最平的一段。永泰元年（765）五月严武卒，他失依，乘舟出岷江、过嘉州、戎州、渝州，下三峡——离开益州后再未回。大历五年（770）冬卒于湘江舟中。

后人到成都浣花溪，访的不是杜甫本人住过的茅屋——那间屋他离开后渐废，宋代屡有重修，今址是清康熙年间所立。但**那一带的溪、桥、柏、海棠**——唐时怎么生长，今日仍怎么生长。
