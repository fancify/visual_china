# 衢州

## 一句话

江南东道东南山区一上州——治信安县（今衢州市），处钱塘江最上游谷地、武夷仙霞两山之间，自婺州西部析出未及七十年的新州。

## 大唐天宝十四载（755年）

衢州治**信安县**，在钱塘江最上游的谷地——城南为仙霞岭余脉，西去武夷山，北连金华盆地南缘。州属**江南东道东南**，**天宝元年（742）随玄宗朝"改州为郡"通制改为信安郡**，755 时正名为"衢州信安郡"，两名在唐人嘴里互换不分；**乾元元年（758）肃宗复改为衢州**——但这是 755 之后的事。沿革：**武则天垂拱二年（686）析婺州之信安、龙丘置衢州**——"取武德废州名"（武德四年曾置衢州，旋废），"衢州"二字自此再立，是江南东道里成州最晚的一批之一。**《新唐书·卷四十一·地理志五》衢州信安郡列为上州**。**《旧唐书·卷四十·地理三·江南东道·衢州信安郡》载：**"垂拱二年，分婺州之信安、龙丘置衢州，取武德废州名。天宝元年，改为信安郡。乾元元年，复为衢州，又割常山入信州……**领县五，户六万八千四百七十二，口四十四万四百一十一**"——755 时衢州在册**户 68,472、口 440,411**、领县五：**信安**（州治）、**龙丘**、**须江**、**盈川**、**常山**。户口与江南东道诸州比，已稳在上州之列；唯地处山区，城邑不甚大，是钱塘江上游一处转输节点。

州境是**山州**——州城北面婺州金华盆地，南面仙霞岭一道一道压下来；钱塘江干流（唐人称"浙江上游"）自城南合衢江入州境，向东北流去婺州、出杭州入海。自州城南行越仙霞岭可入建州（今建瓯），是江南东道与岭南、闽中往来一处旱口；东行经婺州、越州可至杭州入海。州境出茶、出柑橘，山间多竹木——是江南东道里一处不甚显赫但实在的腹地。

## 历史长河

**中晚唐至宋**　乾元元年（758）肃宗复改为衢州，又割常山一县入信州（今江西上饶一带）——衢州自此领县四。中晚唐衢州未出大事，亦未出大人物——是江南东道里安静的一州。**北宋宣和三年（1121）**改西安县（即唐信安县；咸通中信安改名西安）为衢州治，州号不改。

**宋孔氏南宗居衢**　**南宋建炎二年（1128）**，金兵南下、北宋亡——孔子第四十八代孙、衍圣公**孔端友**奉先圣木主自曲阜南渡，扈从宋高宗驻跸临安；**建炎三年（1129）**赐家衢州，立家庙——孔氏自此分为**南北两宗**，南宗居衢州至今。元明清三朝衢州因孔氏南宗而文气一脉不绝。**明清衢州府**沿宋制，治西安县，领西安、龙游（即唐龙丘）、江山（即唐须江）、常山、开化五县——大体覆盖盛唐衢州境，仅盈川一县已并入龙游。

**民国至今**　民国元年（1912）废府存县，西安县复名衢县。1949 年后改隶衢州专区；**1985 年撤地区设地级衢州市**——"衢州"作建制名经过一千二百九十九年（垂拱二年 686 至 1985 年）后复现。今**衢州市**辖柯城、衢江两区与江山、常山、开化、龙游四县（市）——大体覆盖盛唐衢州五县境。**南孔家庙**今在衢州市柯城区，岁时仍祭；这一脉自南宋至今八百九十余年未断，是衢州自盛唐以来未有的文化分量——但 755 时无此事。

## 引文

> 衢州信安郡，垂拱二年，分婺州之信安、龙丘置衢州，取武德废州名。天宝元年，改为信安郡。乾元元年，复为衢州，又割常山入信州……领县五，户六万八千四百七十二，口四十四万四百一十一。

——《旧唐书·卷四十·志第二十·地理三·江南东道·衢州信安郡》
