# 苏州

## 一句话

江南东道腹心州，唐人口中的"吴郡"——春秋阖闾故城、江南运河中段，盛唐尚未到它最盛的时候。

## 形貌

苏州在太湖东岸一片平畴里——城外四望，水网密如蛛丝。

外郭周回不足二十里，远比长安、益州小一圈，**也不比扬州大**——但这是江南东道腹心的一座旧城。城墙夯土带砖包，开八门：阊、胥、盘、葑、娄、齐、平、相——皆春秋阖闾旧名。八门之中**阊门**最盛——城西北的水陆双门，扼江南运河北口，自此舟楫北上至润州、扬州，南下至杭州、越州。**盘门**居城西南，**水陆并行**的城门——一道城门里，舟与车分两路而入，这种双门制度在江南诸州里独此一处。胥门在西，相传伍子胥忠谏不纳被赐死后曾悬其首于此门——故名。

城里的格局，是水道先于街巷。整座城被纵横河道切成长方棋格——河上架小桥数百，每坊一河、每河一街——舟与步并行。子城（吴郡州衙）居全城正中偏南，**自春秋阖闾建城以来位置大体未变（或仍在原址）**——历代衙署多次重建，骨架与方位相承。子城外向北一条主水巷，唐人谓之"南北直河"——后世名"卧龙街"再到清代的"护龙街"，便是今日观前街一线。城外护城河绕城一周，与运河相通——这是江南水城的本相，唐时已成。

物候是长江以南、太湖东岸的：四时分明，春深多雨，梅季湿气重，秋日明净，冬不甚寒。城外北郊有**虎丘**——一座海拔不足四十米的小山，山下却葬着春秋吴王阖闾——传说剑池下藏其墓与三千名剑。城西南有**寒山寺**所在的妙利普明塔院（详见子条目），城外阊门外便是江南运河——舟人夜行，常以塔灯为志。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

苏州在唐人眼里是一座极古的旧城——但**它的"最盛"还未到**。

最早的根脚是**春秋吴国**。阖闾元年（前 514），吴王阖闾命**伍子胥**相土尝水，筑大城于此——周回四十七里，开陆门八、水门八——这是中国早期城建史上少见的"水陆双门"格局。**唐时的苏州子城，与阖闾那座城仍是同一处**——城圈虽屡有缩展，骨架未变。阖闾葬虎丘——剑池传说自此而起。

秦灭楚（前 223），于此地置**会稽郡**，治吴县——这是中国正史里"吴县"二字第一次作为县名。**汉因之**，会稽郡治仍在吴县。东汉永建四年（129）析会稽郡置吴郡——治所亦在吴县。三国吴黄武元年（222）孙权称王——都在建业（今南京），但吴郡仍是其根本州郡，孙策、孙权、陆逊、顾雍诸大族世居此地。两晋南朝因之——**"吴郡"作为郡名一直沿用到隋平陈（589）**。

**隋开皇九年（589），平陈后改吴郡为苏州**——取城西南姑苏山之名。"苏州"二字作为正式州名自此起。**大业三年（607）**炀帝复改州为郡——苏州又复称吴郡。**唐武德四年（621）**李渊定江南后复改吴郡为苏州。**天宝元年（742）**玄宗诏天下州一律改称郡，苏州再改吴郡——这是 755 年当年的官称。**至德二载（757）**——一说**乾元元年（758）**，史家两说并存——肃宗复改郡为州，敦煌郡复称沙州、吴郡复称苏州。但 755 年这一刻，城门匾额上写的还是"吴郡"。

唐人若在 755 年驻足回望：从春秋阖闾筑城（前 514）算起，到这一年已是**一千二百六十九年**——城在原址未曾大迁；同样属于这片平原的旧都吴大城、越王勾践灭吴后的废墟、孙吴顾陆世家的旧宅，都在城内外几十里之内。"姑苏"这个旧名也在唐人嘴边——李白《登金陵凤凰台》"晋代衣冠成古丘"那样的句子写南朝旧事，吴郡人写自己城——所写的更早。

### 此刻

天宝十四载（755）的苏州，行政上是**江南东道吴郡治**——领吴、长洲、嘉兴、昆山、常熟、海盐、华亭七县。本州户口据《通典·州郡典》《元和郡县图志》记**开元户七万六千余**——是江南诸州第一档的大州，**唐制属上州**。整个江南东道（开元二十一年自江南道析出，治所在吴郡）此时**正在悄然起势**——盛唐户口数字在这一带还未追上中原，但江南运河打开之后，盐铁、漕粮、丝、米、纸、笔、茶——大半天下用度自这一道出。

755 年的苏州刺史，史载不甚详——天宝中后期江南东道诸州刺史更替频繁，史多失记。州内实务由刺史、长史、司马、各曹参军组成的一套州县官员系统打理。一年的常事是：春课田、夏催赋、秋收漕、冬考课——江南漕米经由**江南运河**北运至润州京口，过江入瓜洲，再循江淮运河、汴渠至洛阳，最后入河、入渭抵长安——盛唐天下漕运的命脉，半数仰仗这条水路。

**江南运河**这一条水路的关键节点：苏州城阊门外便是运河——隋大业六年（610）炀帝命凿江南河自京口至余杭，八百余里，沿途经润州、常州、苏州、湖州、杭州——这条河打通了从淮南至浙东的整套水运。**到唐玄宗朝，江南运河已运作一百四十余年**——苏州正在它的中段最盛处。城阊门外的码头泊船多时数百艘，南北行旅在此换船、买货、宿驿。

755 年苏州的诗人——史料留下来的不多。**张继**（生卒不详，进士及第天宝十二载 / 753）是其中一位——他天宝末以乡贡进士及第，安史乱起后曾过吴中，传世名作《枫桥夜泊》"姑苏城外寒山寺，夜半钟声到客船"即其安史中（约 756-763 间）作——具体作期史料未明。张继至 755 这一年的行止亦未详——可能在长安、可能在乡——但他与这座城的诗史绑定，是几年后的事。**韦应物**（约 737-792）755 年时尚是十八九岁的少年——他后期任苏州刺史是德宗贞元年间（约 788-791）的事，距 755 年还有三十余年。**白居易、刘禹锡**任苏州刺史是中晚唐穆宗、敬宗朝的事——更晚。

对天宝十四载的苏州人来说，长安在三千余里之外——隔着江、隔着淮、隔着大半个中原。安禄山十一月初九在范阳起兵的消息，沿驿道经汴州、宋州、徐州、扬州，南渡过江至苏州，**前后约要一个月**。这一年的苏州城里，市坊如常——阊门外的运河船工不知道：再过半年，朝廷将循江南运河南调江淮租赋，**苏州自此从"江南腹心州"开始向"江南漕运第一城"转身**——中晚唐的繁盛由乱后才真正起来。盛唐的吴郡，是一座旧得发亮、却尚未达极盛的城。

## 子条目

### 子城（吴郡州衙）

全城正中偏南——自春秋阖闾筑城（前 514）以来便是衙署所在（或仍在原址）。秦会稽郡治、汉吴郡治、三国吴郡治、晋宋齐梁陈吴郡治、隋苏州 / 吴郡治、唐吴郡治——一千二百余年间衙署多有重建迁改，但所在大体相承。唐时子城内有刺史衙、长史、司马、诸曹参军公署、州学、武库、州狱——一套完整的上州官衙。**子城遗址今在苏州市旧城南部锦帆路一带**——苏州市考古所近代多次发掘，确认夯土基础与唐宋叠压层。

### 阊门

城西北水陆双门——**江南运河北口在此入城**。这是唐宋以来苏州最盛的门面：胡商、漕船、官旅、举子、文人——出入苏州者十有八九自阊门进出。门内是商铺密集的"南濠"市，门外是江南运河的泊舟码头。唐人写诗以"阊门"代指苏州一城——是后来宋元明清沿袭的传统。

### 盘门

城西南，**水陆并行**双门——舟与车分两路而入。"盘"取其门道盘曲——这种双门制度在江南诸州里独此一处。盘门外是大运河南行至杭州的起点。今苏州城西南所存的**盘门**——城门主体是元末张士诚据吴时（1356-1367）重修，明清续修——但门址、双门格局、水陆并行结构，**自春秋阖闾起一直未移**。

### 胥门

城西门——相传伍子胥忠谏不纳，被吴王夫差赐死，悬其首于此门。"胥"即子胥之胥——这是这座城里最早、也最直接的纪念。唐时门尚开，门内有伍员祠——州人岁时致祭。

### 虎丘

城西北七里——海拔三十六米的小山。**春秋吴王阖闾**葬此——"葬经三日，白虎踞其上，故曰虎丘"（《史记·吴太伯世家》司马贞《索隐》引《吴越春秋》）。山上自南朝起即有寺——山顶剑池相传藏阖闾墓与名剑三千。东晋王珣、王珉兄弟舍宅为寺，名"虎丘山寺"——南朝至唐一直是吴郡第一名山寺。**白居易**（825-826 任苏州刺史，距 755 已七十年）筑山塘，自阊门外通虎丘——"七里山塘"自此为吴中名景；但 755 年时山塘尚未筑，自阊门到虎丘走的还是寻常田陌。剑池真伪——唐宋皆有人下池探查，传说存焉。

### 妙利普明塔院（寒山寺前身）

城西阊门外枫桥畔——**南朝梁天监年间（502-519）始立**，唐时称"妙利普明塔院"或"普明禅院"。寺前临运河，舟人夜行常闻钟声——这是张继《枫桥夜泊》"姑苏城外寒山寺，夜半钟声到客船"所写的那座寺。**"寒山寺"之名在唐时已有**——张继诗（约 756-763 间作）已用此名，宋人笔记亦多以"寒山寺"称之；相传唐贞观间高僧寒山、拾得曾至此寺，后人以寒山之名号寺。故唐时此寺有"妙利普明塔院"（旧名 / 正式塔院称）与"寒山寺"（俗称 / 后渐通行）两名并存，本 doc 写 755 时即写两名——但匾额所题之正式名仍当是"妙利普明塔院"。

### 江南运河（城阊门外段）

隋大业六年（610）凿——自京口至余杭八百余里，过苏州城阊门外。苏州段是江南运河中段最繁的一段：阊门外码头停舟数百，南北漕米、丝、盐、茶、纸皆经此中转。这一段唐宋两代屡修——河身今犹存，是苏州城西的主要水道。

### 八门旧址

阖闾筑城时开八门：**阊、胥、盘、葑、娄、齐、平、相**——皆春秋旧名。唐时八门部分**或仍存**——阊门、胥门、盘门、葑门、娄门大体在原址；齐门、平门、相门据宋人追记或废或改，唐时具体存废史料未尽明。这八个门名一直沿用到明清——是中国城建史上少见的"门名两千年不易"案例。

## 历史长河（755 → 今）

**中晚唐**　至德二载 / 乾元元年（757-758）肃宗复改吴郡为苏州——"苏州"之名自此通行至宋。中唐起苏州始入诗史：**韦应物**贞元间（约 788-791）任苏州刺史——"今朝郡斋冷，忽念山中客"即此时作。其后**白居易**（825-826）、**刘禹锡**（837 前后）相继任苏州刺史——白筑七里山塘自阊门通虎丘，苏州任内作《登阊门闲望》"阊门四望郁苍苍，始觉州雄土俗强"——以刺史之眼写吴郡阊门一带的繁盛；后日忆江南而作《忆江南》——"江南好，风景旧曾谙……能不忆江南"。（白氏名作《钱塘湖春行》实作于杭州刺史任内 822-824，写的是杭州西湖，非苏州事。）中晚唐的苏州，由江南腹心州走向**江南漕运第一城**——晚唐谚"扬一益二"之外，苏州常列其后。

**五代至宋**　唐末黄巢之乱（880）兵未深入江南。乾宁三年（896）杨行密据淮南建吴国——苏州一度归之；天复二年（902）钱镠据吴越国，苏州归吴越，一直到太平兴国三年（978）钱俶纳土归宋——**吴越七十六年间**苏州为吴越国西府，钱氏奉中朝、保境安民——盛唐至中晚唐的城市底子，由是基本保全过五代。北宋承之置苏州。**政和三年（1113）宋徽宗诏升苏州为平江府**——"平江"之名由此起，沿至元代。**注意**："平江府"是北宋后名号，755 时不可用——盛唐诗文里若见"平江"皆是后人追书。北宋苏州人朱长文作《吴郡图经续记》（1084）——以宋人之笔回望唐宋苏州城制，与唐《元和》相参证。**南宋**苏州随江南半壁腹心，富庶居天下前列——《吴郡志》（南宋范成大 1192）成稿，是宋代州郡志最详者之一。

**明清**　元改平江府为平江路，至正十六年（1356）张士诚据平江称王，至正二十七年（1367）败于朱元璋——盘门、阊门今所见之主体即张士诚据吴时所修。**明洪武元年（1368）改平江路为苏州府**——"苏州"作为府名再立。明清两代苏州是**江南文教与经济的双重首府**——府城内官学、私塾、书坊、园林林立。明中后期至清中叶，苏州**沧浪亭**（北宋庆历四年 / 1044 苏舜钦自宋廷被贬后于此筑亭，取屈原"沧浪之水"——755 时不存）、**拙政园**（明正德八年 / 1513 王献臣筑）、**留园**（明万历间）、**狮子林**（元至正二年 / 1342 始立）相继成形——"苏州园林"之名由是天下知。但这些园林**全是 755 年以后的事**——盛唐吴郡尚无此景。

**民国至今**　1912 年废府，苏州改吴县。1949 年后置苏州市，今属江苏省。1982 年苏州古城被列为国家第一批历史文化名城——古城面积十四点二平方公里、河道总长三十五公里——城内河街并行的格局，**与唐宋苏州大体一致**。1997 年苏州古典园林（拙政园、留园、网师园、环秀山庄）列入联合国教科文组织世界文化遗产，2000 年补入沧浪亭、狮子林、艺圃、退思园。盘门（今盘门景区）的双门水陆格局保存最完整——是中国仅存的水陆双门城门。

至于盛唐那一座吴郡——子城衙署、八门旧址、阊门外的运河、城西北的虎丘剑池——其址或叠或失。可走的，是盘门的水陆双门、虎丘山顶的剑池一汪、阊门外那一段大运河——这几处的位置、走向、用法，**与天宝十四载那一年大体未变**。

——张继《枫桥夜泊》（《全唐诗》卷二四二；张继天宝十二载 / 753 进士；此诗作于安史中约 756-763 间，具体年份史料未明）。**注**："寒山寺"之名张继诗已用，唐时与正式塔院名"妙利普明塔院"两名并存——755 这一刻取其同时之状。

> 江南好，风景旧曾谙。日出江花红胜火，春来江水绿如蓝。能不忆江南？

——白居易《忆江南·其一》（约 837 前后作，距 755 年八十年；白曾任苏州刺史 825-826 年）

> 苏州之繁雄，固为浙右第一。

——朱长文《吴郡图经续记·卷上》（北宋元丰七年 / 1084；朱长文以宋人之笔记唐宋苏州，是 755 年后三百三十年的回望）
