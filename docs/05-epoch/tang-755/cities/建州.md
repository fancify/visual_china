# 建州

## 一句话

江南东道东南端闽中山区一州——治建安县（今建瓯），闽江上游建溪盘城而过，州境西据武夷山脉、东出仙霞岭与衢州接壤，是闽北山中第一个有汉人州县体制的治所。

## 大唐天宝十四载（755年）

建州治**建安县**，在闽江上游**建溪**之畔——建溪自武夷山脉东麓发源，南流至建安城下盘城而过，再南汇富屯溪、沙溪于剑州一带成闽江本支，是闽中"三源汇流"中的最东一源。州属**江南东道**最东南，**天宝元年（742）随玄宗朝"改州为郡"通制改为建安郡**，755 时正名为"建州建安郡"，**乾元元年（758）肃宗复改为建州**——但这是 755 之后的事。沿革更早：**东汉献帝建安初年（196 前后）析侯官县置建安县**——县名取自当朝年号；**三国吴永安三年（260）置建安郡**治建安——这是闽中第一个郡级建置，盛唐"建州"二字即承此而来。隋大业初罢州郡复称建安郡。**唐武德四年（621）置建州**——治建安，领绥城、唐兴、建阳、沙、将乐、邵武等县。《新唐书·卷四一·地理志五》建州建安郡列为**上州**（《旧唐书·地理志》同列"中"，两书州等略异——本 doc 以新唐书"上"为准、旧志"中"并存）。**《旧唐书·卷四十·志第二十·地理三·江南东道·建州建安郡》载：**"旧领县二，户一万五千三百三十六，口二万二千八百二十。**天宝领县六，户二万七千二百七十，口一十四万三千七百七十四**"——"天宝"即天宝十一载（752）籍——755 时建州在册**户 27,270、口 143,774、领县六**（建安、邵武、浦城、建阳、将乐、沙）。**《新唐书》另载户 22,770、口 142,774、领县五**（无沙县）——两数同源略异，新志或较旧志少录沙县一项。在闽中三州（福州长乐郡、建州建安郡、泉州清源郡）里建州户口居中——比福州（户 34,084）略少、比泉州为多；闽北山中一州能聚十四万余口，在盛唐已属难得。

州境是**闽北山中**——西据**武夷山脉**东麓（建阳县境内"有武夷山"，《新唐书》明载），东出**仙霞岭**与江南东道衢州、信州接壤——自洪州东出信州、衢州，过仙霞岭便入建州；自建州顺建溪南下可至剑州、再循闽江至福州。这是中原士庶入闽的主要陆道——比自虔州、汀州一线更繁。755 时建州无大兵无大事，州境贡品以**蕉布、花练（彩丝织物）、竹綀（竹纤维布）**为主（《新唐书》土贡条），皆闽北山地特产；建安县境**有银、有铜**——是闽中少数有金属矿之州。

## 历史长河

**晚唐五代**　唐末**黄巢乱**（878-884）波及闽中——建州亦在其列；**乾宁三年（896）王潮**克闽中，朝廷封王潮为威武军节度使治福州——建州归其辖。**后梁开平三年（909）王审知**封闽王立**闽国**——都福州，建州为闽国北境一州。闽国后期内乱，建州一度为王延政据地称帝（943-945），国号"殷"——这是建州历史上唯一一段称都岁月。**后晋开运二年（945）**南唐军灭闽——建州亦陷。

**北宋**　建州属福建路；北宋**北苑**（建州建安县境）置贡茶院造贡茶——这是建州在中国茶史上最浓的一笔，但已是北宋事，**755 年时建州尚无以茶名世**。**南宋绍兴三十二年（1162）**升建州为**建宁府**——"建宁"之名自此入官府文书。

**元明清**　元为建宁路。明洪武元年（1368）复为建宁府——治建安、瓯宁两县（瓯宁自宋元符元年 / 1098 析建安西境置）。明清建宁府历五百余年未变。

**民国至今**　1913 年废府，建安与瓯宁合并为**建瓯县**——"建瓯"二字自此始（取建安、瓯宁各一字）。**1992 年撤建瓯县设建瓯市**（县级，南平市代管）——这是 755 年的建州城治、建安县地一千二百三十七年后的近代建制名，**严禁预支至盛唐正文**。今建瓯市为福建省南平市辖县级市，闽北山中一处地方城市；建溪仍盘城而过，与唐建安县城方位大体相承。

## 引文

> 建州　中。隋建安郡之建安县。武德四年，置建州，领绥城、唐兴、建阳、沙、将乐、邵武等县。天宝元年，改为建安郡。乾元元年，复为建州。旧领县二，户一万五千三百三十六，口二万二千八百二十。天宝领县六，户二万七千二百七十，口一十四万三千七百七十四。

——《旧唐书·卷四十·志第二十·地理三·江南东道·建州建安郡》

> 建州建安郡，上。武德四年置。土贡：蕉、花练、竹綀。户二万二千七百七十，口十四万二千七百七十四。

——《新唐书·卷四一·地理志五·江南道·建州建安郡》（节录 verbatim）。**州等"中"（旧志）"上"（新志）两书略异**——本 doc 以新志"上"为主、旧志"中"并存；**户口两数（旧 27,270/143,774 与新 22,770/142,774）亦略异**——同源异传，盛唐建州户口大致在二万二千至二万七千户、十四万二千至十四万三千口之间。
