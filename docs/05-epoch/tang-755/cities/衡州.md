# 衡州

## 一句话

江南西道湘水中游一处中州，治衡阳县（今湖南衡阳）——城正北二十余里便是南岳衡山，七十二峰自此起脚。

## 形貌

衡州城卧在湘水中游西岸，**南岳衡山**南麓平地上——这是它的位置根本。

湘水自南方永州、零陵一路北流，过衡州城东，再北去潭州（长沙）入洞庭——衡州正当湘水中游的一处转折。城东临湘水，水面已宽，可通漕舟与客船；城西倚衡山余脉低岭；城北二十余里便是**南岳衡山**主体——七十二峰自城北视野里铺开，远看祝融、芙蓉、紫盖诸峰起伏。城南有耒水自东南来，于衡州城南数十里入湘——耒水谷是衡州通往郴州、岭南的水路一支。**一山一水一谷三事相接，衡州正是这三者交汇的点**。

物候是江南西道南段的：春雨绵长（湘中梅雨）、夏极热（湘江河谷盆地酷暑）、秋多晴爽、冬偶有薄雪。一州之地以湘水冲积平原与衡山东南麓低丘为主——可作稻田，山麓产茶与药材。衡州在唐人江南西道的版图里靠南——再南便是郴州、连州，过岭便是岭南广州。盛唐人由长江南来、自岳州入湘水、过潭州，到衡州时已是湘中——下一站便要望见五岭。

至于山——**衡山的事不在这一篇 doc 里讲**（参 [衡山](../scenic/衡山.md)）。衡州 doc 里只记一句：唐时衡州州境正北即南岳衡山主体，州治距衡山南麓约二十余里——盛唐人下了湘江船入衡州城，再北行半日，便到了**南岳大庙**前。

## 大唐天宝十四载（755年）

唐人眼里衡州的沿革清楚——**隋开皇九年（589）平陈，于汉长沙郡衡山以南旧地置衡州**，治衡阳县。**大业三年（607）**炀帝行州改郡通制——衡州改称**衡山郡**。**唐武德四年（621）**复置衡州，仍治衡阳。**天宝元年（742）**玄宗诏天下州一律改称郡——衡州改称**衡阳郡**。**乾元元年（758）**肃宗复改郡为州——但 755 这一刻，城门匾额上写的是"衡阳郡"。民间口语仍呼"衡州"——两个名号在唐人嘴里互换不分。

755 年的衡州属**江南西道衡阳郡治**——领**六县**：衡阳（州治）、攸、茶陵、耒阳、常宁、衡山（《旧唐书·地理志三·衡州》"领县六"为唐代定制；《新唐书》《元和郡县图志》县名略有出入，以《旧唐书》六县为常说）。据《旧唐书·地理志三·衡州》载：**"户三万三千六百八十八，口一十九万九千二百二十八"**——折今计**户 33,688、口 199,228**。按开元十八年（730）后户等口径，**中州门槛为 25,000-60,000 户**——33,688 户落在中州中段，故衡州在唐代州格里是一**实打实的中州**，非"中州偏下"。在江南西道里属二线州——介于潭州（长沙郡，江南西道湘水流域首州）与一般下州之间。

衡州的分量**不在户口而在两件事**：一是**南岳衡山**——五岳之一、唐玄宗朝南岳大庙已盛；二是**湘水中游漕运中转**——长江—洞庭—湘水—耒水这条南北水路，是盛唐岭南向长安输贡、士人南下北归的命脉之一。本州本身**未置节度使**——江南西道至天宝末年尚未单立节度，755 年时归江南西道采访使节制（治洪州），军事调度则随事临时置使。衡州城本身无大军镇，本州常驻团结兵数百——这是一座**山水之州**而非武州。

南岳衡山在唐代礼制里地位极重。**开元年间**玄宗诏天下五岳皆封王——衡山封为"司天王"（《唐会要》开元十三年 725 封五岳为王）。**南岳大庙**自隋以来即立于衡山南麓朱陵洞前——唐玄宗朝屡有扩修。盛唐人到衡州，多先北谒南岳大庙——这是衡州城外第一访点，亦是历代南方士人出仕、还乡必经的一处礼场。

755 年冬安禄山反范阳——消息沿驿道南传，过荆襄、过岳州入湘水、至衡州，约需半月余。**755 时衡州官称衡阳郡，长官为太守**（742-758 郡制时期，州皆改郡、刺史皆改太守）——衡阳郡太守接到的诏令大约是：戒严、备粮、整团结兵、待调。衡州本无重兵——但作为湘水中游漕粮、岭南贡赋转运的一个节点，衡州的米、衡州的船工，在接下来八年的战事里都将派上用场。**衡州的重要性此后不增不减——它始终是一州中州**——它真正在文学版图上发光，是再过四十余年的事——

那时一位名叫韩愈的诗人，将自潮州量移袁州，途经衡州登南岳，作《谒衡岳庙遂宿岳寺题门楼》——但这已是元和十四年（819）的事，离 755 这一刻还有六十四年。

## 历史长河

**中晚唐**　**乾元元年（758）**肃宗复改郡为州，此后稳定。中晚唐衡州为南方贬官与游历士人多过之地——**杜甫大历四年（769）冬自潭州南下衡州**，作《望岳》（"南岳配朱鸟"——三望岳之南岳一首），后又有《岳麓山道林二寺行》等湘中诗。**韩愈元和十四年（819）量移袁州途中过衡州**，登南岳作《谒衡岳庙》"五岳祭秩皆三公，四方环镇嵩当中"——是衡岳一山在中唐文学中的高光时刻。**柳宗元元和十年（815）自永州量移柳州**亦过衡州——湘水一线由南而北，永州、衡州、潭州三地是中唐贬官诗的串珠。

**五代至宋**　南唐马氏楚国据湖南——衡州入版图。北宋衡州属荆湖南路。**南宋淳熙年间（1174-1189）朱熹两至衡山**——讲学于衡岳书院（南宋时石鼓书院已立衡州城北石鼓山，为宋代四大书院之一）。**石鼓书院**北宋至道三年（997）由李宽初创于湘水、蒸水交汇处的石鼓山——南宋朱熹、张栻、吕祖谦相继讲学于此，是宋代理学南传的重镇。元代衡州属湖广行省。

**明清**　明洪武初改衡州路为衡州府——治衡阳县。明清衡州府为湘南重镇，南岳大庙历代屡修。**明末清初顾炎武、王船山（王夫之，1619-1692）**皆衡州本籍人——**王夫之是衡州府衡阳县人**，生于明万历四十七年（1619），明亡后隐居衡阳石船山下著书四十载——《读通鉴论》《宋论》《船山遗书》皆成于此。"船山先生"之号即取衡阳石船山。但**王夫之是明清之际人，离盛唐 755 已 864 年——本 doc 不预支**。清承明制，衡州府属湖南布政司。1852 年太平军攻衡州——城陷数月。1898 年湘鄂铁路议起，衡州始为湘南铁路节点。

**民国至今**　1913 年废府存县——衡阳县为湖南省辖。1942 年衡阳设市。**1944 年衡阳保卫战**——抗战末期国军第十军方先觉部坚守衡阳 47 日，为抗战大型城防战中坚守最久者之一，城终陷，但拖住日军南下广西的战略时间。1949 年后衡阳市为湖南省辖地级市——今**湖南省衡阳市**，辖五区五县代管二市，人口约六百六十万。城区中心今仍在湘水西岸——盛唐衡州州治衡阳县治所，与今衡阳市主城区基本同址。**今日湖南衡阳市治所，仍在唐衡州治衡阳县旧址上**——这是湘中少见的连续一千四百余年同一城址。

**南岳大庙**今仍在衡山南麓——历代屡修，现存建筑以清光绪八年（1882）大火后重建为主，是南方现存最大的古建筑群之一。**石鼓书院**抗战中毁于日军炮火，2007 年于原址复建。**王船山故居**在今衡阳市衡阳县曲兰镇湘西村——清代石船山隐居处遗址。盛唐那座方圆数里、城门匾额上写着"衡阳郡"的小城——夯土城墙在今衡阳市区的层叠下早已埋没，**考古至今也没碰到过完整的城圈**。一千二百多年过去，那座城名换了几番——衡州→衡山郡→衡州→衡阳郡→衡州→衡州路→衡州府→衡阳县→衡阳市——但**衡山、湘水、耒水**三件事一件没变。

——《元和郡县图志·卷二九·江南道·衡州》（取意节录；元和志录开元户数为 21,482，比天宝盛时 33,688 已降——盖中唐战乱后湘中户口锐减；以《旧唐书》天宝户为 755 当时态值）

> 南岳配朱鸟，秩礼自百王。欻吸领地灵，鸿洞半炎方。

> 喷云泄雾藏半腹，虽有绝顶谁能穷。

——韩愈量移袁州途中过衡州，谒南岳大庙作。
