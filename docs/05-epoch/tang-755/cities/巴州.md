# 巴州

## 一句话

米仓道南段腰心的山州——清化郡治化城县（今四川巴中市），与梁州、兴州同属山南西道，是大唐自汉中翻米仓山入巴东诸州的脊梁锁扣。

## 形貌

巴州城坐在**巴江北岸**——巴江源自米仓山南麓，绕城东南而下，再南汇入渠江。州治**化城县**——城周不大，按盛唐山南西道东半的常制。州东南循巴江下行可达壁州、通州；西去越米仓山一道脊梁便接梁州（汉中）境；北面是大巴山脊，深谷连山。物候是典型川东北山地——春迟、夏雨连旬、年降水近一千二百毫米，山中产竹、漆、桐、茶、丹砂；巴江两岸是巴米、巴绢的核心产带。

## 大唐天宝十四载（755年）

巴州的沿革几度反复：**北魏一度置巴州，旋废**；**西魏废帝二年（553）复置巴州**——治化城县，"巴州"二字自此立。**隋开皇九年（589）废巴州**入州县通制；**唐武德元年（618）高祖复置巴州**，属山南道；开元二十一年（733）分山南为东、西二道后，**巴州属山南西道**。**天宝元年（742）**玄宗诏天下州改郡——巴州改为**清化郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州——巴州之名再立。**742 至 758 复州前正名"清化郡"，民间口语仍呼"巴州"**——两名并行。

755 年的清化郡**领县十**（《旧唐书·地理志》巴州清化郡条），化城为郡治。**《旧唐书·地理志》记天宝户三万二百一十、口九万一千零五十一（户 30,210，口 91,051）**——在山南西道东半诸州中户口居中偏上，米仓道辐辏之利已显，胜于邻近壁州、集州。本郡 755 当年的本职在**米仓道**：自梁州南郑南门出，循浕水翻米仓山主脊，过集州（今南江），南下至本郡治化城——再东南循巴江过清化、壁州（今通江）、通州（今达州），整条道穿州而过。**米仓道与金牛道并列为蜀道八条之二，金牛道为"正"，米仓道为"陪"**——清化郡太守的本职便是守这一条陪道的腰节：备粮、护驿、巡道、迎送过境官使。城南有**南龛**摩崖造像（详见下"巴中石窟"段），州城东南巴江上有官桥渡口。

天宝十四载冬安禄山反范阳——消息自长安经子午道或洋州、梁州转米仓道南下传至巴州，比传到成都晚不了几日；山南西道全境告急。再过半年（756 七月）玄宗西奔入蜀经凤州、兴州、利州、剑州至成都——**取的是金牛道，未取米仓道**；米仓道因山深险曲，叛军终唐之世未尝至此——这条道在安史乱中的角色是巴东诸州的**后方北门**，非战线。清化郡在 755 当年的安静与紧绷并存——城里日常如旧，州署文书已开始增厚。

## 巴中石窟（南龛 / 北龛 / 西龛 / 水宁寺）

巴州城近郊**南龛**山崖——盛唐至中晚唐间陆续开凿的摩崖造像群，是**巴蜀北道现存唐造像最密的一处之一**，与广元千佛崖、剑阁觉苑寺、绵阳碧水寺并称。**南龛石窟**今存窟龛百余、造像约二千余尊——主体造像分期据巴中石窟考古简报为**初唐至中晚唐间陆续开凿**，盛唐天宝至大历间一段为造像高峰；具体最早开凿年代历来异说，**或上溯隋末初唐、或断为盛唐**——本 doc 取保守表述，不坐实最早年代。州境另有**北龛**（城北一里许）、**西龛**（城西二里许）、**水宁寺**（州东恩阳河上游）——三处造像规模略小于南龛，但盛唐造像延续不断。755 当年化城县城南、城北、城西山崖上叮叮当当的开凿声尚未停歇——这是清化郡在盛唐留给后世最重的一笔。

## 历史长河

**中晚唐至五代**　乾元元年（758）复改清化郡为巴州。中晚唐巴州为山南西道东半重镇，米仓道驿务长期繁忙；南龛、北龛、西龛、水宁寺造像延续至会昌灭法（845）前后渐止。五代之间巴州随山南西道入前蜀、后唐、后蜀——名号未改。

**两宋至明清**　北宋仍置巴州，治化城。元改属四川行省。**明末张献忠转战川北**，巴州几经兵燹；明清两代巴州属保宁府，州治化城县。

**民国至今**　民国二年（1913）废州存县——改为巴中县（"巴中"一名作为正式县级地名自此始，**755 时此名尚未出现**）。1949 年后属四川省，1993 年设巴中地区，2000 年撤地设巴中市（地级）——下辖巴州区。今四川巴中市巴州区治所，约在唐巴州化城县旧址上。南龛、北龛、西龛、水宁寺石窟皆列今全国重点文物保护单位——盛唐州城残基已埋入今巴中城下，米仓道线仅存零星驿堡、栈阁遗迹于巴江、米仓山一线。"清化郡"三字，只在《旧唐书》《元和郡县图志》和那一段南龛造像题记里仍可见。
