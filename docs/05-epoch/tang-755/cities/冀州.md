# 冀州

## 一句话

河北道腹心一上州，742-758 年间名"信都郡"——治信都县（今河北衡水市冀州区）；古名"冀"上承《禹贡》九州之首（但唐冀州只居其一隅，不可混），战国属赵，与赵、邢、沧、景、定诸州交错共撑河北中段。

## 形貌

冀州在河北大平原的腹心——州治信都县（今河北衡水市冀州区），城西北仰太行余势已远、城南越过漳水入魏博，城东沿低洼一直铺向沧、景之海岸。漳、滹沱、滱、衡漳诸水自西、自北汇入境内的低地泽薮——古"信都"二字便取义于此一片平原上的水脉网络。**北接赵州，西邻邢州，东去沧州、景州，南连魏州**——四向皆同道、皆上下州，正坐在河北道中段的纵横十字上。物候属华北腹地——春旱风沙、夏伏雨涝、秋燥、冬寒；城外种粟、种麦、种桑，是河北腹心的赋税大州。

## 大唐天宝十四载（755年）

冀州境是禹贡冀州的一隅——但唐人写"冀州"二字，已是河北道中段的一上州（**按：《尚书·禹贡》九州之首"冀州"古义指今河北中南至山西全境一大片，唐冀州只是其中一州、一隅而已——盛唐人读《禹贡》凭吊"冀州"是大义，治信都的这座城与禹贡冀州的全境不能混**）。战国时此境属赵，秦置信都县，汉因之；**隋开皇六年（586）置冀州**，大业三年（607）改**信都郡**；**唐武德四年（621）复冀州**，**天宝元年（742）**玄宗诏改郡——冀州再为信都郡；**乾元元年（758）**肃宗复为冀州。755 这一刻匾上写"信都郡"，民间口语仍呼"冀州"。**《旧唐书·地理志二·河北道·冀州（信都郡）》载天宝领县九、户一十万三千八百八十五、口八十三万五百二十**（旧领六县时户一万六千二十三、口七万二千七百三十三），盛唐时与赵、邢、定、沧诸州同居河北道腹心一线。**742-758 郡制下**冀州正名"信都郡"，长官即**信都郡太守**（一人）领郡事——不再"刺史兼"——州司马、司户参军一套常规上州官署；755 时冀州（信都郡）并无节度建制——河北道采访使巡历、本郡太守直管。

十一月初九安禄山自范阳起兵，**消息自幽州南下沿太行东麓的官道急传**——冀州在赵、邢与沧、景之间的纵轴上，叛军主力先经赵、邢南下取相、卫扑东都，而**史思明一路沿河北诸州反复焚毁**：**至德、乾元间冀州数度易手**——尤以**乾元二年（759）**史思明再起、河北诸州大乱时，冀州为河北腹心战场之一。755 年冬这一刻冀州城里"信都郡"四字尚悬于谯楼——半月后将进入八年战火。

## 历史长河

**中晚唐**　乾元元年（758）复改信都郡为冀州。河朔三镇中冀州属**成德节度**辖（治恒州 / 镇州），是成德镇的腹心州之一；中晚唐河朔割据格局中，冀州长期不入唐廷掌握。

**宋元明清**　北宋仍冀州，属河北东路；金属河北西路冀州；**元改冀州属真定路**；明清冀州属**真定府**（一度直隶州）。明清间冀州治信都县（民国元年废信都县入冀县），仍是真定 / 保定一带的腹心上州。**"衡水"二字为民国后市制方立**——盛唐 755 时无"衡水"地名，不可预支。

**民国至今**　1913 年废府，冀州改冀县（民国元年信都县并入）；今**河北省衡水市冀州区**——市区即唐冀州信都县旧址范围。盛唐冀州人若得归来，城外的平原走势、漳滹诸水的方位皆未大变——但他口中"信都郡"四字、河北道腹心的地位记忆，皆须从《元和郡县图志》《旧唐书·地理志》一卷一卷找回。

## 引文

> 冀州，信都郡。望。隋开皇六年置冀州，大业三年改为信都郡。武德四年复为冀州，天宝元年改为信都郡。乾元元年复为冀州。

——《旧唐书·地理志二·河北道·冀州（信都郡）》（节录）
