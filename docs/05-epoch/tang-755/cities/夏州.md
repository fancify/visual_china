# 夏州（朔方郡）

## 一句话

关内道北线一州，治朔方县——城即十六国大夏赫连勃勃所筑**统万城**，长安以北约一千二百唐里，朔方节度使辖境东翼、扼河套南口的旧夷都。

## 形貌

夏州城在鄂尔多斯高原南缘——今陕西靖边县北约六十里的白城子。

城西临**奢延水**（无定河上游），北望毛乌素沙地——这一带在盛唐已是半农半牧的塞外塬区，城外是一片高低起伏的沙、草、塬交错带。城正南循延州道、过鄜延一线南下，约一千二百唐里至长安；正西循官道、过盐州，可至**灵州**（朔方节度治所，cross-ref [./灵州.md]）凡四百余唐里；正北过红柳河、横山，便是河套南口、与突厥旧地相接。**夏州是朔方节度使辖境的东翼——与灵州一西一东，共扼河套南面**。

最特别的是城本身。**统万城**——赫连勃勃十六国大夏国都——城墙以蒸土版筑，史载"蒸土筑城，锥入一寸即斩工匠"（《晋书·赫连勃勃载记》）。盛唐时去其建城已三百余年，城垣依然坚固高峻，唐人沿用——州治朔方县城便是这座旧夷都。**城周四面**：东西两城相连，外郭周回约十里，内城更密——西南角的"角楼"残基至今可见（白城子遗址实测）。城外朔方县境内有**朔方县城外驿、长泽县、宁朔县**——皆夏州属县；境内军戍以**朔方节度下属诸军**（如朔方军、天柱军等）有戍线经此。

物候是鄂尔多斯南缘的：春旱多沙、夏短而雨集中、秋凉而风、冬严而干。年降水三百余毫米——勉强可种粟黍，不可水稻；沙地多生**柠条、沙蒿、沙柳**，奢延水沿岸有河柳与杨。冬季奢延水十一月底封冻，唐人冰上往来。

## 大唐天宝十四载（755年）

唐人眼里夏州是塞上旧城。

**东晋义熙九年（413）**匈奴铁弗部首领赫连勃勃起建统万城为大夏国都——是为本城肇始。**北魏始光四年（427）**太武帝拓跋焘破统万、夏国亡——魏改其地置统万镇；**北周保定三年（563）**改设夏州——"夏州"之名自此立，因赫连勃勃国号"大夏"故。隋开皇三年（583）沿夏州、大业三年（607）改置朔方郡；唐武德元年（618）复夏州——治朔方县（即统万旧城）。

**天宝元年（742）**玄宗诏改州为郡，夏州官称**朔方郡**——与朔方**节度使**之"朔方"同名而非同物：节度使治灵州、辖整片河套；朔方郡只是夏州一州一郡。**乾元元年（758）**肃宗复改郡为州——夏州之名再立。**742 至 758 这十七年间夏州正名"朔方郡"**，民间仍呼"夏州"。

755 年夏州属**关内道**、治朔方县——据《旧唐书·地理志一》记天宝间夏州约 **9,213 户**（一说万户上下）——虽户口仅九千余（在下州门槛之下），但**官品列中都督府**（《唐六典》《元和郡县图志》并载）。辖县朔方、德静、宁朔、长泽四县（与《新唐书》所记略有出入）。夏州**隶朔方节度使辖境**——节度使治灵州（经略军即驻灵州城内），夏州为其东翼军政据点；境内尚有**朔方军 / 天柱军**等朔方节度下属军号的戍线分布。城内有州衙、朔方县治、驿馆与本州戍兵；**州城本身即赫连旧统万城**——盛唐人住在十六国国都的城墙里，是这座城最大的奇观。

**这一带还是拓拔鲜卑（拓跋）的早期活动地**——拓跋部自北魏分裂后一支南迁，散居夏州、银州、绥州一线；盛唐夏州境内**党项**拓拔氏部落渐次内徙、占据沙塬。安史乱后这股党项力量在夏州一带积聚——**两百年后李继迁、李元昊于此境起而立西夏**，其族出便是这一片拓拔故地。

天宝十四载冬安禄山反范阳——夏州在关内道北翼、远离东都战火；本州兵随朔方节度使调度南下勤王。次年（至德元载 756）肃宗自马嵬北上抵灵武即位——夏州非肃宗必经，但作为朔方节度辖境内的东翼重镇，本州兵自此次第汇入郭子仪、李光弼的复京大军中。

## 历史长河

**中晚唐**　乾元元年（758）复夏州。安史乱后党项内徙加剧——拓拔思恭一支在夏州、银州渐成气候。**僖宗中和元年（881）黄巢入长安**，党项首领**拓拔思恭**自夏州率部勤王、败而复战、终复京——僖宗以功授**定难军节度使**，治夏州，辖夏、绥、银、宥、静五州——**这便是西夏立国的雏形**。自此夏州拓拔氏自唐末经五代至宋初凡两百年世袭节镇。

**宋元明清**　北宋初仍以李氏（拓拔之冒姓）镇夏州。**太平兴国七年（982）**李继捧献夏、绥、银、宥、静五州于宋；其族弟**李继迁**不附宋、走入沙漠、起兵反——**宋淳化五年（994）宋太宗诏毁统万城**、迁其民——盛唐夏州城至此结束建城千余年的州治生涯，化为废墟。**李继迁之孙李元昊宝元元年（1038）于兴庆府称帝立西夏**——西夏国祚百九十年，但其族根脚的拓拔故地夏州，自宋初毁城后已废。

**元明清至今**　元明清间统万城遗址在塞外沙地中渐为沙埋，明人始呼"白城子"——因夯土墙白灰所筑、近看泛白故。清陕西志记其址在靖边县北。**1956 年陕西考古队首次实测**——白城子即统万城；1996 年起陕西考古所多次发掘，确认东西两城、城垣残高数米、城周四面格局基本保存——**这是中国境内现存最完整的十六国国都遗址**。2012 年公布为全国重点文物保护单位，2023 年起申请世界文化遗产中。**今白城子无人居**，沙地与遗址相伴——千六百年前赫连勃勃所"取统一天下君临万邦"之意的统万城，盛唐人住过、宋人毁过、今为废墟一座。

## 引文

> 赫连勃勃……以叱干阿利领将作大匠，发岭北夷夏十万人，于朔方水北、黑水之南营起都城。勃勃自言：朕方统一天下，君临万邦，可以"统万"为名。城高十仞，基厚三十步，上广十步，宫墙五仞，其坚可以砺刀斧。……蒸土筑城，锥入一寸即杀作者并筑之。

——《晋书·卷一三〇·赫连勃勃载记》
