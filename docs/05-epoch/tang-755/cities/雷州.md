# 雷州

## 一句话

岭南道雷州下州一座，治海康县——雷州半岛南端伸入南海的一座边远郡治，城南徐闻一线是大唐渡琼州海峡入海南诸州的唯一津口。

## 形貌

雷州城在雷州半岛中段东岸，州治**海康县**（今广东湛江市雷州市旧城一带）——城周不大，依坡而立。半岛三面环海：西临北部湾，东俯雷州湾、出海便是南海，南端徐闻县隔**琼州海峡**与海南崖、儋、振、琼、万安五州对望——海峡最窄处约三十里，是大唐辖境内最南的一道内海津渡。

半岛地形低缓——岛中无高山，最高仅二三百米；地表多玄武岩台地与红土平原，间杂热带季风林与稻田。物候是大唐境内最南的一等：冬无霜、夏极长、五月至九月台风、雨季连绵；瘴疠时作，是岭南诸州里出名的"瘴乡"——盛唐至中唐，贬官南来死于雷州境者不绝。城西、南是徐闻县——汉武帝**元鼎六年（前 111）平南越后**始置之古县；自徐闻渡琼州海峡入海南，是这条海道的唐人津口。城外田畴间散落俚人、僚人聚落与汉人移民杂居——盛唐"蛮獠"之一支。

## 大唐天宝十四载（755年）

雷州的沿革几经更名。**隋开皇九年（589）平陈后，于此置合州**——治海康县；**隋大业初（约 605）废合州**，并入合浦郡。**唐武德四年（621）置南合州**——仍治海康；**贞观元年（627）改东合州**；**贞观八年（634）以本州西有'擎雷水'、又传雷神显灵之事——改名雷州**，自此"雷州"之名正式立于岭南。**天宝元年（742）**玄宗诏改州为郡，雷州改作**海康郡**——以州治海康县取名；**乾元元年（758）**肃宗复改郡为州，海康郡复称雷州——**742 至 758 这十六年间，本州正名"海康郡"**，史志、碑刻、文书亦同；民间口语仍多呼"雷州"——两名并行。

755 年的雷州属**岭南道**，州治在**海康县**。《旧唐书·地理志四》《元和郡县图志·卷三八》载本州为**下州**——户口仅数千户（《旧唐书》记天宝间海康郡户约 4320、口 20572 上下），是岭南道户口最少的几州之一。辖县三：**海康**（州治）、**遂溪**（北界，今湛江市遂溪县一带）、**徐闻**（南端，雷州半岛南尽头）——皆雷州半岛地。城内有海康郡治 / 太守署、海康县衙、馆驿——一座边远下州的最简班子。岭南节度使要到**乾元元年（758）**才置——755 当年本州在军事上属**岭南五府经略**辖下广管，**岭南五府经略**治广州，755 在任经略使可考者有何履光之名（《旧唐书》《新唐书》互见）。雷州无折冲府兵之大数，亦无军镇——但作为渡琼州海峡入海南诸州的唯一津口，**徐闻渡**的驿务是这一郡日常要务。

盛唐之雷州，是岭南**典型贬流地之一**——长安朝堂上提到"雷州"二字，多半带着流放的意味。徐闻渡口外便是琼州海峡——再南便是海南岛的崖、儋、振、琼、万安五州；自此入海南者，多为贬官、流人、戍卒、奉调的远征军——以及偶尔自天竺、师子、波斯、大食循海北上漂至本州境的胡舶。本地俚人首领**陈文玉**——隋唐之际人，《雷州府志》及方志载其为雷祖（雷神）化身、为本地神祇——755 时陈文玉已逝百余年，其墓与祠在海康县境，**雷祖陈文玉信仰**自唐入宋遂为本地最大宗神祠；这是雷州境内俚汉杂处之地最深的一脉地方信仰。

天宝十四载冬安禄山反范阳——消息沿驿道南传，自长安过商於、襄州、江陵、潭州、衡州、桂州、广州——再过端、康、新、勤、罗诸州——抵雷州海康郡治，**约需五十余日**。海康郡太守接到的诏令大约是：戒严、防内属俚僚乘隙、备粮——但雷州距河朔战场万里之外，朝廷并未抽调岭南再南的兵北上。城内的人这一冬照常过：海康市集上俚人、汉商往来如常，徐闻渡的盐船、海舶照常出入。盛唐再南之地的安静，恰是它在那个冬天最显眼的特征。

## 历史长河

广德元年（763）吐蕃北上、河陇陷蕃——这是河西之事；岭南再南的雷州未受波及。中晚唐雷州一直为岭南下州，**咸通元年（860）**南诏军北上攻陷安南——交州落于南诏；雷州在交州东北二千余里、未直接受兵——但岭南五府经略东南诸州的兵力调度此时全集中于安南方向，雷州的边远更显。**唐末五代**雷州随岭南入南汉（917-971），南汉置雷州属琼州管——这是雷州第一次官署与海南直接并管的时期。**北宋开宝四年（971）**宋灭南汉，雷州复入中原王朝——属广南西路。**北宋时雷州遂为南来贬官的著名收容地**——**寇准**（宋真宗朝宰相，乾兴元年/1022 贬雷州司户参军，次年卒于雷州贬所，是北宋名臣死于雷州者第一人）、**苏轼**（绍圣四年/1097 自惠州再贬儋州，南渡海峡前于雷州短驻，与弟苏辙在海康相会，自此再别）、**苏辙**（绍圣四年/1097 同时贬雷州，居海康至元符三年/1100 始北归）——三人皆北宋名宦，于雷州留下大量诗文，本州遂有"宋三贤祠"之祀。**这一段贬官北宋史，是雷州在历史长河里最被后世记诵的一段**——但**全部都是北宋事**，盛唐 755 年时寇、苏、苏俱未生。

元置雷州路。明洪武元年（1368）改**雷州府**——治海康县。明清两代雷州府辖海康、遂溪、徐闻三县（清因之），徐闻是渡海赴琼的唯一津口——府治海康为半岛行政中心。清亡后 1912 年废府存县，雷州境内三县直属广东。**1899 年法国租借广州湾**——遂溪县西部沿海一带划为法租借地，**"湛江"**之名要再过半个世纪（1945 年抗战胜利后法租借收回，设湛江市）才出现——盛唐的雷州、明清的雷州府，与"湛江"二字皆无关。1994 年原海康县撤县设市，改称**雷州市**——"雷州"之名自唐至今未断，是岭南这一线少有的千年地名。今雷州市为湛江市辖县级市，盛唐海康郡治旧址约在今雷州市旧城一带；徐闻渡的津口至今仍在——徐闻港今为粤琼海运的主要班轮码头之一。

## 引文

> 雷州，海康。下。开元户四千三百二十。乡一十二。元和户……州境东南至大海。南至徐闻县渡琼州海峡入海南崖州境。本汉合浦郡徐闻县地。

——《元和郡县图志·卷三八·岭南道五·雷州》（取意节录；具体行文以原书为准）。这是唐人对本州最简的一笔：东南至海、南至徐闻渡，自此入海南。

《汉书·卷二八下·地理志下·合浦郡徐闻县》注：

> 自日南障塞、徐闻、合浦船行可五月，有都元国；又船行可四月，有邑卢没国；又船行可二十余日，有谌离国；……自夫甘都卢国船行可二月余，有黄支国。

——这是西汉自徐闻港出海至南洋诸国的最早海道——徐闻在前 111 年置县时便是中原王朝面向南洋的对外贸易港；盛唐**徐闻渡**接的是这条九百年的旧航道。
