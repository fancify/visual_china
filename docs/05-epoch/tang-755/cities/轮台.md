# 轮台

## 一句话

唐北庭大都护府治庭州下辖三县（金满、轮台、蒲类）之一——治在庭州西南、天山北麓一处绿洲（今新疆乌鲁木齐市一带），是天山北道自庭州西去碎叶的官道首站；岑参北庭诗中"轮台"二字多泛指北庭节度辖境而非确指此县治。

## 大唐天宝十四载（755年）

轮台县是**北庭大都护府治庭州领县三**（金满倚郭、轮台、蒲类）中位置最西的一县，治在庭州西南数十里、天山北麓一处绿洲——县治史家两说：**一说在今新疆乌鲁木齐市米东区一带，一说在昌吉市以东**，本 doc 不强订。地当天山北麓自庭州（今吉木萨尔县北庭故城）西行、过清海、双河、接碎叶水那条官道的首站——盛唐与突骑施、葛逻禄诸部往来皆经此地，分量在道里不在户口。

县名"轮台"——**与汉武帝太初四年（前 101）所置塔里木盆地北缘的"轮台屯田"同名而异地**：汉轮台在今新疆轮台县（库车以东），唐轮台在天山以北的乌鲁木齐一带，相距千余里，沿革无承袭。**唐轮台县置年史家两说**：**一说贞观十四年（640）平高昌之役后于庭州初置**，**一说武则天长安二年（702）分置北庭都护府时方正式置县**——本 doc 不强订。轮台县作为庭州（北庭大都护府治所）下辖三县之一，至天宝十四载（755）建制稳定。

755 这一刻轮台县的分量，主要在**岑参诗里**。**岑参（715-770）天宝十三载（754）第二次出塞**——是**安西节度使封常清**（兼北庭都护、伊西节度）幕下判官；自天宝十三载至至德二载（754-757）岑参三年在西域，多次自安西龟兹东北行至北庭、至轮台。**《白雪歌送武判官归京》《走马川行奉送出师西征》《轮台歌奉送封大夫出师西征》《天山雪歌》**诸篇皆作于此三年间——"轮台九月风夜吼，一川碎石大如斗，随风满地石乱走"（《走马川行》）、"北风卷地白草折，胡天八月即飞雪"（《白雪歌》）、"上将拥旄西出征，平明吹笛大军行"（《轮台歌》）——盛唐边塞诗最高的一束花，多自封常清北庭幕中写出。**惟需注意：岑参诗中"轮台"二字多泛指北庭节度辖境（北庭/庭州一带）而非具体县治**——盛唐诗用"轮台"作西域戍守之代称已成惯例（自汉武帝《轮台诏》、班超经营西域以来"轮台"即西域戍边符号），不能据诗题坐实为唐轮台县治所在。755 年冬岑参仍在北庭幕中度过，安史乱起的消息要再过数月才能传到天山北麓。

## 历史长河

中晚唐西域陷蕃后（贞元六年/790 前后北庭陷于吐蕃），庭州、轮台县建制随北庭一并废弃。九世纪中叶以后回鹘西迁、葛逻禄、喀喇汗、西辽、蒙古、察合台诸政权相继据有天山北道——"轮台"作政区名自此退场。元代蒙古、清代准噶尔皆以游牧为主，未在此另立县。**清乾隆二十二年（1757）平准噶尔**后天山北麓重入中央版图——清廷另在塔里木盆地北缘汉轮台旧地立**轮台县**（光绪二十八年/1902 正式置县），地在今新疆轮台县，**承汉轮台之名而非唐轮台之地**。今乌鲁木齐市米东区、昌吉市一带的唐轮台县故址，至今未有完整考古复原；岑参诗中那座县城的具体位置，史家仍存异说。
