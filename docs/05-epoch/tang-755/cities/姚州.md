# 姚州

## 一句话

剑南道西南最外的一座下州，唐姚州都督府治——天宝九载（750）阁罗凤攻陷、751 与 754 两次南征唐军覆师其后，755 这一刻名义仍在大唐版图，实际已是南诏阁罗凤的占领地。

## 形貌

姚州城在金沙江南岸、龙川江畔的一片小盆地里——治**姚城县**（今云南楚雄彝族自治州姚安县城西北一带）。

城周不大——比成都、戎州小得多，约二三里许。城外四望：北面金沙江一线，自西北向东南折，过姚州后东去为戎州境；南面越过一段山地接洱海东岸（去太和城约三百里）；西面循澜沧江上游可至吐蕃东南境；东面循金沙江下行入戎州、再下渝州。盆地中央是龙川江冲积的稻畦——气候比剑南腹地暖、雨季长、瘴气重，盛夏五月至九月唐人称"瘴乡"——长安、剑南来此戍的官兵，能熬过一个瘴季的不多。"楚雄"二字盛唐时尚不存，本 doc 不以此名预支。

## 大唐天宝十四载（755年）

姚州在唐人眼里是**剑南道西南最外的一座下州**——但它的紧要远过其户口所示。

### 沿革与官号

唐高祖武德四年（621）平蜀之后，于洱海东岸一带置**姚州**；唐高宗麟德元年（664）置**姚州都督府**，治姚城县——辖羁縻州三十二（《旧唐书·地理志》载），是大唐西南羁縻系统的中枢，亦**大唐对洱海以东直接施治的高点**。洱海以西尚属六诏分立时代，由姚州都督府节度。

天宝元年（742）全国诸州改郡——按《新唐书·卷四十三上·地理志七上》明载"**姚州云南郡，下**"，姚州 742 同改郡名为**云南郡**（羁縻州亦在改郡之列）；长官称**云南郡太守**。乾元元年（758）肃宗复郡为州——姚州官号在 742-758 之间史志正书"云南郡"，惟通鉴等史志记事中常以"姚州"旧称行文，本 doc 行政官称统一作"云南郡 / 云南郡太守"，地理俗称则随史志作"姚州"。户口上，《元和郡县图志》姚州条载唐前期**户三百余、乡二**——下州中的下州。

### 此刻的紧要：唐失控的边境州

755 年的姚州，紧要不在户口，**而在它已经丢了**。

事起于**云南郡太守张虔陀**——742 改郡之后，姚州 = 云南郡，张虔陀 750 所任正是此一郡守职（《资治通鉴》卷二一六亦书"云南太守"，即云南郡守的省称）。天宝九载（750），张虔陀向南诏阁罗凤索贿不遂，反将阁罗凤之过密奏长安，且对南诏夫人无礼——阁罗凤大怒，**发兵攻陷云南郡（姚州）**，杀张虔陀，掠子女金帛（《资治通鉴》卷二一六 天宝九载条）。**这是姚州第一次失陷**——都督府名义仍存，治所被毁，唐廷在洱海以东三十二羁縻州的控制随之崩溃。

随后**两次南征**皆覆师于西洱河：

**天宝十载（751）四月**——剑南节度使**鲜于仲通**率剑南、山南西道兵八万伐南诏，**分二道出兵：一出戎州，一出嶲州**（《资治通鉴》卷二一六天宝十载条："**仲通率兵八万，分道出戎、嶲州**"）。阁罗凤遣使请降谢罪不许，进兵至西洱河（今洱海）。阁罗凤求救于吐蕃，两军会于西洱河——**唐军大败，死六万人**（《资治通鉴》卷二一六；《新唐书·南蛮传》《旧唐书·南诏传》并载）。鲜于仲通仅以身免——杨国忠扣压败报，反以战功上奏。同年冬，吐蕃册阁罗凤为"赞普钟"——南诏正式叛唐附蕃。

**天宝十三载（754）六月**——杨国忠再遣**剑南留后侍御史李宓**率兵七万（一说十万）攻南诏（《资治通鉴》卷二一七作"将兵七万击南诏"）。阁罗凤诱敌深入，引唐军至太和城下西洱河畔——预设伏兵，又值瘴疠盛夏，士卒病疫者过半。两军接战，**唐军全军覆没，李宓殁阵**（一说被擒，旧唐书与新唐书并载，史家多取殁阵说）。前后两役，唐军损失累计十三万至十八万——蜀中、剑南诸州户口为之一空。**杜甫《兵车行》"边庭流血成海水，武皇开边意未已"** —— 西洱河两役是其中分量最重的一笔。

两次南征之间，**唐军此役于姚州故境短暂据守**——但 750 沦陷之后姚州都督府是否名义复置、唐军是否以姚州故城为兵站基地，史源不一，本 doc 不强订。754 李宓覆师之后，**姚州都督府事实上为南诏阁罗凤所有**。太和城 doc 已记：755 年时"东面姚州都督府已废，唐姚州兵马于第二次南征中尽殁"。

至 755 这一刻，姚州**名义上**仍列大唐版籍，《通典》《元和志》皆载其辖羁縻州数十；**实际上**州廷已不能视事——南诏阁罗凤据洱海以东直至金沙江一线，姚州故地羁縻部落（生獠、生爨、乌蛮、白蛮诸种）尽附南诏。阁罗凤虽据其地，但**尚未在姚州故地立纪德碑**——十一年后大历元年（766）他于太和城西北立**南诏德化碑**，碑文 3800 字自辩天宝战争之由，755 时此碑**尚未立**。

至于安禄山十一月初九于范阳起兵——这消息传到姚州故地已是 756 年正月间。**姚州，是这场盛唐最后劫难里，唐廷最早丢失的一块边州**——比安禄山起兵早了五年。

## 历史长河

**中晚唐**　姚州故地为南诏所据近五十年，至贞元十年（794）异牟寻与唐苍山神祠会盟复唐后，姚州都督府名义短暂恢复——但治所、户口、辖境均无法回到 750 年前的样貌。咸通中南诏世隆复叛，姚州再失。整个中晚唐至宋初，**洱海以东这一片地，再未真正回到中原王朝的直辖**。

**宋元明清**　北宋归大理国；元至元十一年（1274）立**姚州**于此（云南行省下），治姚城——"姚州"二字在元代复起。明洪武十五年（1382）改**姚安军民府**——"楚雄"之名自此立；清雍正七年（1729）改**姚州**直隶州——"姚州"末次以州名行政。

**民国至今**　1913 年废姚州、改**姚安县**——"姚州"二字至此与行政区划脱钩。今**云南省楚雄彝族自治州姚安县**光禄镇一带保留唐宋至明清的城址遗存——20 世纪 80 年代以来零星发掘见唐故城夯土与瓦当碎片。光禄古镇至今以"姚州古城"为旅游标牌——但盛唐 755 那一座姚州，**只剩地下的夯土与瓦砾**。

## 引文

《资治通鉴·卷二一六》天宝九载条（节）：

> 云南太守张虔陀贪暴，与阁罗凤有隙，遣使奏其罪。阁罗凤忿怨，发兵反，攻陷云南、姚州，杀虔陀，取夷州三十二。

——这一笔，是姚州 750 失陷的源头，也是天宝战争的导火索。
