# 苏毗

## 一句话

吐蕃赞普麾下"五茹"之一的**苏毗茹**——年楚河谷里那群古藏部族的旧土，755 这一冬正酝酿着一场被赞普镇压下去的反叛。

## 形貌

苏毗不是一座单一的城，而是吐蕃东北方一片旧部族故地的统称——藏文作 **Sumpa**（孙波），唐文献作"苏毗"。其地理重心，盛唐时**已自青藏高原东北的那曲、玉树一线（古苏毗早期活动区）西南移至年楚河谷与后藏一带**——年楚河（藏文 Nyangchu）自喜马拉雅北麓的康马一线向北流，至日喀则（藏文 Shigatse）一带汇入雅鲁藏布江干流。这一线的河谷宽展平阔、海拔降至三千八百米上下，是吐蕃境内少有的农业腹地——青稞、油菜可成片，比逻些谷地更宜耕。

苏毗茹之下不只一处中心。盛唐时藏文史源所记的"茹府"（茹之治所）在何处，学界尚有异说——多归于今**江孜、白朗、日喀则**一带的几座要冲，亦有说更东的那曲、玉树仍是苏毗故地的一部分（**藏文史源对苏毗腹地的范围向有异说**，并存以待考）。其聚落形态与逻些类似——并无中原意义上的城墙夯土，倒像河谷里一处接一处的牧场、青稞田、夯土碉房与小寺庙，由部族大人（藏文 dpon）与赞普所遣茹长（藏文 ru dpon）共理。

物候上，年楚河谷干冷少雨——七月雨水短促，冬日风硬而无大雪；河谷两侧抬眼便是雪山，南面是喜马拉雅主脊，北面是冈底斯余脉。

## 大唐天宝十四载（755年）

### 吐蕃眼里的来路

苏毗这一支古藏部族，藏文史源记其在赞普世系立国之前便已据有高原东北方诸谷。**松赞干布之前，苏毗与雅隆悉补野（吐蕃王室所出之部）并立为古藏诸部之一**——藏文史源《柱间史》《王统世系明鉴》载苏毗一度强盛，与象雄（藏文 Zhangzhung，今阿里高原）、雅隆三足鼎立。**松赞干布（约 629-650）在位时并苏毗，将其编为吐蕃"五茹"（藏文 ru，意为"翼/部"——四茹之外加一茹）之一，是为苏毗茹（Sumpa-Ru）**——半自治，岁贡兵马于赞普牙帐。

唐文献对苏毗本身着墨甚少，多附见于吐蕃传末。但《隋书·女国传》《旧唐书·东女国传》载隋唐之际青藏高原东北方有"女国"——以女子为王、母系承嗣、贵女贱男——**唐文献的"东女国"指何处，学界异说颇多**：一说即苏毗北部某一支部族；一说在今川西金川一带的氐羌部落；亦有说与苏毗本身重叠不分。**苏毗本部是否实行母系王制、抑或仅其一支部族如此，史源未一**——加 hedge，不径作断言。

### 此刻

天宝十四载（755）这一年的苏毗茹，恰陷一场吐蕃内部的政治震荡。

敦煌古藏文卷子 P.T.1288《大事纪年》载：赤德祖赞（**藏文 Tridé Tsuktsen**，唐文献作"弃隶蹜赞"）朝末年——藏历"狗年"，王尧、陈践译本系为**公元 755 年**——**苏毗茹王 没陵赞 谋叛吐蕃事被发觉**，时执国政者为大相**玛祥仲巴杰**（藏文 Mazhang Trompakye），镇压之。这一年逢赤德祖赞之卒、赤松德赞继位之前（详 [逻些](./逻些.md)）——赞普新丧、新王幼冲，苏毗茹这一动是吐蕃 755 内部不稳的一处明显标志，亦是后来赤松德赞朝整顿诸茹、收紧中央的伏笔。

至于唐人，对这一场发生在年楚河谷里的吐蕃宫廷血案一无所知——逻些尚且远在三千余里之外，苏毗茹更在其后数百里。在唐文献的视野里，755 这一年的西陲只是哥舒翰陇右、剑南一线对吐蕃的常态对峙——苏毗茹之乱，是吐蕃自家的事。

对天宝十四载的苏毗茹人来说，这一冬的年楚河谷不平静。**没陵赞之谋败露，部族大人或亡或降，茹长换人——河谷里的青稞还在仓里，但帐外的风已经变了**。

## 历史长河（755 → 今）

**中晚唐**　755 苏毗茹之乱被镇之后，赤松德赞朝（755-797）对苏毗茹整顿尤切——茹长尽以王室亲信代之，苏毗骑兵自此作"苏毗千户"长期供赞普中央军。八九世纪吐蕃极盛时，苏毗兵随赞普西征中亚、东破长安（763）、南略南诏，是吐蕃军中最能战的一部。会昌二年（842）末代赞普朗达玛被刺、吐蕃分裂——苏毗茹随之解体，年楚河谷各部族复归自治。

**宋元至清**　北宋至元，年楚河谷由数个地方政权迭据——**萨迦派**根基所在的**萨迦寺**（建于宋熙宁六年/1073）即在苏毗茹故地南境，元世祖至元年间萨迦派被尊为帝师，由此年楚河谷一带成为后藏政教中枢。明帕木竹巴朝以后，**江孜**（藏文 Gyantse）在年楚河谷崛起为后藏一大宗——**江孜白居寺**（始建于明永乐十六年/1418）与十万见闻佛塔，是这一带保存最完整的明代藏式建筑。万历后**藏巴汗**据后藏；清五世达赖入主全藏，**后藏自此以日喀则（藏文 Shigatse）扎什伦布寺（始建于明正统二年/1437，宗喀巴弟子根敦珠巴所立，后为历世班禅驻锡）为政教中心**——苏毗故地由是归班禅系统辖。

**民国至今**　1951 年中央人民政府与西藏地方达成《十七条协议》；1965 年立西藏自治区。今年楚河谷为西藏自治区**日喀则市**辖地，下辖江孜县、白朗县、康马县等——皆唐时苏毗茹故地。**萨迦寺、江孜白居寺、扎什伦布寺**先后列入全国重点文物保护单位；萨迦寺南寺所藏 8 千余函古藏文典籍，是研究吐蕃苏毗茹时期最重要的一批一手文献。**苏毗作为部族之名，至今仍偶见于藏文史地著作；现代藏区已无以此自称的部族**——这一支古藏血脉，早已融入"后藏"两字之下。

## 引文

——《旧唐书·卷一九七·南蛮西南蛮·东女国传》（**东女国与苏毗本部之关系，史源异说，并存**）
