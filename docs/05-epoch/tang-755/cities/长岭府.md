# 长岭府

## 一句话

渤海**后世**"五京十五府"中的**长岭府**——治瑕州，主流定于今吉林通化一带，扼上京 / 西京鸭绿府之间的山岭通道；755 时或仅为府辖境之雏形，独立府治之号未必已立。

## 大唐天宝十四载（755年）

长岭府是渤海国**后世**"五京 + 十府 = 十五府"体系（《新唐书·渤海传》所载）中的一府，治**瑕州**——治所主流定于**今吉林通化市一带**，少数主张在邻近集安境内；本 doc 从主流（通化），但 hedge：唐时本无"通化"之名，更无"通化市"（民国后政区），doc 仅在"今地"对照时使用。

**755 时态加 hedge**：渤海十五府体系的**完整定型**，主流认为在**文王大钦茂朝中后期至宣王大仁秀朝**（约 8 世纪末至 9 世纪初），与五京制度之成熟同步；天宝十四载（755）这一节，文王大钦茂在位（737-793）已近二十年，王廷正自中京显德府向上京龙泉府过渡——长岭府辖境（鸭绿江中游北侧山区，西京鸭绿府以北）虽已是渤海腹地之一部，但"**长岭府**"之号是否已与他府并立、是否已有府治之实，史料未明。本 doc 取"已为府辖境，未必已立府治"的过渡态。

长岭府之要害在"**山岭**"——府境跨长白山脉西南余脉，南为西京鸭绿府（治神州，今吉林集安鸭绿江中游一带）所辖鸭绿江中游川谷，北通上京龙泉府的山间官道由此过——是渤海**鸭绿道**（自上京经长岭至西京鸭绿府、再下鸭绿江入唐境营州方向）之中段。境内人口以**高句丽遗族与靺鞨**为主，文化上承高句丽山城传统。

755 年此时长安方向，安禄山十一月初九于范阳起兵——消息传至渤海西南边府至少要再过月余。文王此后保持藩属之礼、不与叛军接触——七年后宝应元年（762）唐廷升其"渤海郡王"为"**渤海国王**"。

## 历史长河（755 → 今）

**辽天显元年（926）**契丹灭渤海，改东丹国——渤海十五府体系尽废，长岭府辖境融入辽东京道女真各部之间。辽金元此片山地属边僻女真之境；明置卫所，清初为**长白山以南封禁地**之南端；光绪年间渐开禁，**民国二年（1913）**始设通化县（取自更早设的通化局），1985 年设通化市。鸭绿江中游北岸的群山与川谷间，今仍多有高句丽 / 渤海山城遗迹（如**集安洞沟古墓群**等），1980 年代起陆续发掘——长岭府的具体府治遗址尚无定论，与西京鸭绿府治神州的归属问题缠在一起，是渤海考古的悬案之一。
