# 潭州

## 一句话

江南西道湘水中游一州，治长沙县（今湖南长沙）——**屈贾故地**：屈原沉汨罗、贾谊贬长沙王太傅，两人之名压住盛唐人提起这一片湘中之地的所有想象。

## 形貌

潭州城坐在湘水东岸的一段台地上。湘水自南而北纵贯此境——南上至衡州（衡山郡）、再南至郴州；北下注洞庭，入岳州（巴陵郡）。州治**长沙县**——城周不甚大，夯土城墙，开四门；城西门外便是湘江。城东背一支低岭（后世所称岳麓山乃湘水西岸一山，**与城东这支低岭非一山**——岳麓山在江西，长沙城在江东）。城南、城北皆为湘水冲积谷地，宜稻——盛唐人称之为"长沙之野"。

物候是江南中游：春多阴雨，江雾常起；夏湿热，湘水盛涨；秋日江风渐凉；冬季短而少雪——较江夏（鄂州）更南，霜信稍迟。湘水上下风帆往来——上行入衡岳、再入岭南灵渠一线，下行抵洞庭、入江——潭州正在这条南北水道的中段。

## 大唐天宝十四载（755年）

潭州按户口仅为中州——但**它脚下这一片地，是楚地最古的一块**。

楚之故壤，秦置长沙郡——汉高祖五年（前 202）封吴芮为长沙王，立**长沙国**，治**临湘县**（即今长沙）——这是"长沙"作为正式郡国名第一次见于史。**屈原沉汨罗**——汨罗江在长沙之北、岳州之南，《史记·屈原列传》记屈原放逐江南、自沉汨罗（前 278 一说，年代史家两说并存）；**贾谊**——汉文帝前元六年（前 174 一说，《史记·贾生列传》系年异说间一二年）贬为**长沙王太傅**，作《吊屈原赋》《鵩鸟赋》——"屈贾文章"四字，自此压住此地。

沿革上：**隋开皇九年（589）**平陈，置**潭州**——州治始定于长沙县；这是"潭州"作为正式州名的隋制起点。**大业三年（607）**改潭州为**长沙郡**。**唐武德四年（621）**复置潭州——治长沙县。**天宝元年（742）**玄宗诏天下州改称郡——潭州改称**长沙郡**；**乾元元年（758）**肃宗复改郡为州——长沙郡复称潭州。755 年这一刻，城门匾额上写的是"长沙郡"。

755 年的潭州属**江南西道长沙郡治**——《旧唐书·地理志》记潭州长沙郡 "**天宝领县六，户三万二千二百七十二，口一十九万二千六百五十七**"。郡望以户口论属**中州**（唐制户四万以上为上州、二万以上为中州、不满二万为下州——潭州户三万二千，按户为中州）；然潭州为**都督府**（统潭、衡、永、道、邵、连、郴七州军事），都督府例升一阶，故实际地位高于一般中州。**江南西道户口前列：鄂州约一万九千余、潭州三万二千二百七十二、洪州约五万五千余**——三州为江南西道沿江湘三大城。所辖**六县**——长沙、浏阳、湘乡、湘潭、益阳、醴陵（浏阳为唐景龙二年/708 析浏水县置，755 时已属潭州；**衡山县唐属衡州（衡山郡），不属潭州**——前稿误列衡山入潭州，已订正；**宁乡为北宋太平兴国二年 977 始置，唐时为长沙县地，本 doc 755 时态不入宁乡之名**）。

这一年长沙城内的主事：长沙郡太守、长沙郡司马、长沙县令——一套完整的郡县官系统（742-758 改州为郡期间，州刺史例称太守）。江南西道当时并未常设节度——天宝初江南西道为采访使分巡，长沙郡太守归江南西道采访使节度。**江南西道军政重心并不在潭州一城**：江西的洪州、湖北的江夏、湖南的潭州——三镇分立。盛唐人提"扬一益二"时不算潭州；潭州的份量不在繁华一线——**它在文学地理上是"屈贾故地"的一个坐标**，唐人贬谪南行多过此地，作怀古之诗。

755 年冬安禄山反范阳——消息沿长江驿道南传至江夏、再南渡入潭州时，大约已是十二月底或次年初。**但潭州距叛军主战场尚远**——河北、河南、关中三处先燃；江南西道一带的实质卷入要等到至德、乾元间安史余烬南下、江汉一线吃紧之后。755 年冬的长沙城，仍是一座湘水中游的普通州城——只是城南汨罗一线，已沉了屈子近千年。

## 历史长河

**中晚唐**　乾元元年（758）复州名为潭州。中晚唐之间，**大历三-五年（768-770）杜甫晚年流寓湘中**——自夔州出三峡、过江陵、入洞庭、上下湘水，潭州、衡州两地往返凡数次，最后大历五年冬卒于湘江舟中（一说卒于潭州、一说卒于耒阳，史家两说并存）。杜甫晚年湘中诗几十首——《登岳阳楼》《入衡州》《回棹》《长沙送李十一衔》——皆作于此一段。**但杜甫流寓潭州已在 755 后十三年，本 755 doc 主体不入此段，仅作历史长河志一笔。** 中晚唐韩愈贞元末贬阳山、元和初过潭州，柳宗元贬永州十年（永州在潭州西南）——皆与潭州州学官员有诗酒往还。

**五代至宋**　五代时潭州为**南楚**（马氏）国都——马殷自唐末据潭州，后梁开平元年（907）封楚王，立国，国都正在潭州——这是潭州历史上唯一一次作为正式王国都城。后周显德元年（951）南楚亡。宋平江南，潭州归宋。**北宋开宝九年（976）**潘美攻南唐过此。北宋潭州属荆湖南路——治长沙县。**北宋开宝间（968-976）**潭州城南岳麓山下立**岳麓书院**——后与白鹿、应天、嵩阳并称北宋四大书院，南宋朱熹、张栻于此讲学，"朱张会讲"为南宋理学一大盛事。**但岳麓书院之立已在 755 后二百余年——盛唐人提潭州时，岳麓山下尚无书院，山中仅有寺观数所。** 南宋绍兴间岳飞屯军潭州——抗金中线南段，与鄂州大本营遥相呼应。

**元明清**　元置**潭州路**，后改**天临路**——治长沙县。明洪武五年（1372）置**长沙府**——治长沙县，自此"长沙府"之名延续至清末。明清长沙府是湖南布政使司所在（清雍正后湖广分省，湖南治长沙）——**湘中第一军政中心**。清同治、光绪间湘军起于长沙一带——曾国藩、左宗棠、胡林翼诸湘军要人多出此城及其周邻。

**民国至今**　1913 年废府存县——长沙县仍存。1933 年置长沙市。**1938 年文夕大火**——长沙城在抗战焦土政策中几被焚尽；盛唐潭州古城地表遗存至此一绝，今所见长沙古城墙等多为后世复建或考古回填。1949 后长沙为湖南省会至今。盛唐潭州治所旧址叠在今**长沙市天心区、芙蓉区**老城范围内——湘水仍自城西流过，**汨罗江仍在城北流入洞庭，屈原沉江处后世立有屈子祠（始建年代史家异说，唐时已有屈原祠见诸记载但具体方位 755 doc 不强填），贾谊故宅传址在城西**（"贾谊故宅"之传承自南北朝至明清屡有迁建，今所见太平街贾谊故居乃明清复建于清同治年间再修，与汉代贾谊本宅之关系**仅属传承附会而非实证**）。**与盛唐潭州人共看的地物**：湘水中段的河面、岳麓山的山影、长沙之野的麦秋——这一千二百年间未改。至于"屈贾"——今长沙以"屈贾之乡"自署，城中屈贾祠、贾谊故居、汨罗屈子祠俱在——但盛唐潭州人若复活，会发现今人把那两位古人的纪念立得比他们当年更实了。

## 引文

> 潭州，长沙郡。中。都督府。隋长沙郡。武德四年，平萧铣，置潭州总管府，管潭、衡、永、郴、连、南梁、南云七州。……天宝元年，改为长沙郡。乾元元年，复为潭州。旧领县五，户一万三千一百三十一，口五万二千五百八十三。天宝领县六，户三万二千二百七十二，口一十九万二千六百五十七。长沙、浏阳、湘乡、湘潭、益阳、醴陵。

——《旧唐书·卷四〇·地理志三·江南道·潭州》（取意节录；具体行文以原书为准。**三万二千二百七十二为"天宝领县六"之户，非"开元户"——前稿标签误，已订正**。潭州本州户三万二千余按唐制户口标准为中州，都督府升一阶。**六县为长沙、浏阳、湘乡、湘潭、益阳、醴陵——衡山县唐属衡州不属潭州**）

司马迁《史记·屈原列传》：

> 屈原至于江滨，被发行吟泽畔……乃作《怀沙》之赋……于是怀石遂自沉汨罗以死。

——屈原沉汨罗具体年史家两说并存（前 278 一说较通行，但《史记》本传未明系年；前人多据《楚辞》及秦楚交兵年代推考）。

司马迁《史记·贾生列传》：

> 贾生既辞往行，闻长沙卑湿，自以寿不得长，又以谪去，意不自得。及渡湘水，为赋以吊屈原。

——贾谊为长沙王太傅约在汉文帝前元六年（前 174 一说），《吊屈原赋》《鵩鸟赋》皆作于长沙任内。
