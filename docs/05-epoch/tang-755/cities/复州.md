# 复州

## 一句话

山南东道江汉平原南岸那一座小州——治沔阳县（今湖北仙桃市沔城回族镇），汉水中下游右岸湖网间的小郡（旧书不标州望、新书作上；本项目按户口口径降读，户 8,210 偏下州）；天宝元年至乾元元年（742-758）官称"竟陵郡"。

## 形貌

复州城坐在**汉水中下游南岸**的低平台地上——四周尽是江汉平原的湖荡与圩田。城周不大，夯土城墙；州境无大山，地势自西北向东南缓倾，云梦故泽的余水散为大小数十湖泊，雨季漫衍、旱季退为低洼草甸。北接郢州（治长寿，今湖北钟祥）、东接安州（治安陆）、南隔汉水望江陵之东境、西接襄州之南。汉水自西北而来，在州境东南折向东注入长江——州治正在汉水自北转东的那段右岸。从复州渡汉北上接郢、襄一线官道，南渡汉江入江陵东境的长江干流；本地货以鱼盐、葛布、菰米、莲藕为主。物候：夏热多雨水，江湖蚊蚋盛；冬不甚寒，常有湿雾。

## 大唐天宝十四载（755年）

复州本是汉沔阳县地，**隋开皇三年（583）**置**复州**——州名取自境内复池（一说沔水复合之义）；**大业三年（607）**改州为郡，复州改称**沔阳郡**。**唐武德四年（621）**平王世充后，唐复置**复州**治沔阳县。**天宝元年（742）**玄宗诏天下州改郡，复州改称**竟陵郡**——州名取自境内古竟陵之号（春秋楚有竟陵邑，秦置竟陵县，在汉水中游一带）；**乾元元年（758）**肃宗复郡为州，竟陵郡又复称复州。755 年这一刻，城门匾额上写的是"竟陵郡"。州望两志不一——**《旧唐书·地理志》复州条不标州望**（旧书卷三十九复州条仅列天宝户 8,210、口 44,885 而无"上/中/下"字样），**《新唐书·地理志》作"上"**；本项目按天宝户口口径（户 8,210）降读为偏下州一级。《旧唐书·地理志》《新唐书·地理志》载复州（竟陵郡）天宝领县三：**沔阳、竟陵、监利**；天宝户口verbatim作**户八千二百一十、口四万四千八百八十五**（旧唐书）。

复州在 755 年的分量不在本州事务——而在**江汉平原南半那张水网的一个不大节点**。它北望郢、襄，东望安、黄，南越汉江便是江陵东境——四面皆是更大的州，复州夹在其中，户口、漕运、戍卒都不当一面。这一年州本城照常上缴秋粮、点汉水水驿。十一月安禄山起兵于范阳，消息沿汉水驿道南传至沔阳约二十余日——江汉一带实质卷入要等到至德二载（757）以后山南东道节度使的转战。

## 历史长河

**乾元元年（758）**竟陵郡复称复州——州名照旧。中晚唐复州属山南东道节度使（治襄州）辖下。**陆羽**（733-804）——**复州竟陵人**，盛唐至中唐之间生于本州，**建中元年（780）前后于湖州著《茶经》三卷**，奠定中国茶文化典籍之祖；陆羽虽自署"复州竟陵人"，但《茶经》成书已在 755 后二十余年，**列于历史长河段**。**北宋**仍置**复州**，属荆湖北路；**元**置**复州路**，旋改沔阳府。**明洪武九年（1376）**改为**沔阳州**，属湖广承宣布政使司——"复州"作为州名自此从地名上退去，前后历七百余年。**清**沿明制为沔阳州，属荆州府。**民国元年（1912）**改沔阳州为沔阳县。**1986 年**撤沔阳县设**仙桃市**——县级市，今属湖北省直辖。盛唐复州治沔阳县故城在今**仙桃市沔城回族镇**——唐州治、明清沔阳州治、民国沔阳县治同址相承一千四百年，1949 后县治始东移至今仙桃市区。

## 引文
