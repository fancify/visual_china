# 中京显德府

## 一句话

渤海五京之一，文王大钦茂自旧国敖东城迁来的第二都——755 当年正是它把"都"的身份让给上京龙泉府的过渡之年。

## 大唐天宝十四载（755年）

中京显德府是渤海国"**五京**"（上京龙泉府 / 中京显德府 / 东京龙原府 / 西京鸭绿府 / 南京南海府）之一。**主流考古定其治所于今吉林延边和龙市西古城**——敦化敖东城是更早的"**旧国**"建国地，非中京；学界亦有少数主张中京即敖东，本 doc 从主流（西古城）。

755 当年，渤海在位之王是**文王大钦茂**（在位 737-793）——唐玄宗开元二十六年（738）册为"渤海郡王"。文王在位三十余年中渤海经历**两次迁都，三处都城序列**：旧国（敖东城）→ 中京显德府 → **上京龙泉府**。《新唐书·渤海传》记"**天宝末，钦茂徙上京**"——天宝末即天宝十四载（755）至十五载（756）之窗——**依迁都序列推断，755 这一节中京显德府当是渤海王廷将走未走、上京尚未建成的过渡之都**（hedge：史料只载迁出时点，未明确 755 当年朝堂实坐何城）。朝堂、宗庙、府库尚多在中京显德府办公；王本人则常往返中京与上京之间督建新都。

显德府仿唐州府之制——下辖六州（卢州 / 显州 / 铁州 / 汤州 / 荣州 / 兴州，《新唐书·渤海传》载），是渤海腹地之东南都会。城在图们江北、海兰河南的山间平地——南可循图们江入东海，北可循牡丹江至上京、循鸭绿江至西京。中京之所以被弃，史料未明因——三上次男推测与北面靺鞨各部安抚、与新都仿长安格局的规模需求有关。

755 年此时长安方向，安禄山十一月初九于范阳起兵——消息传至渤海中京至少要再过月余。文王此后保持藩属之礼、不与叛军接触——七年后宝应元年（762）唐廷升其"渤海郡王"为"**渤海国王**"。

## 历史长河（755 → 今）

迁都上京之后中京显德府降为五京之一的陪都，仍设六州之治。9 世纪渤海日盛，史称"海东盛国"——五京并存。**辽天显元年（926）契丹灭渤海**，改东丹国——中京旧城渐废。辽金元此地属边僻女真地。明置卫所、清初为宁古塔将军辖境之南端。**民国后此片设和龙县**（1909 设和龙峪局，1913 设和龙县），1965 年延边朝鲜族自治州下设和龙县，1993 年设和龙市——城西原野中所余夯土方城与建筑基址，1980 年代起由吉林省文物考古研究所多次发掘，2000-2002 年正式定为渤海中京显德府治所，**2001 年列全国重点文物保护单位**。**敦化敖东城**则被定为渤海"旧国"建国地，与六顶山墓群（王族墓）一并列为全国重点保护——两者皆是渤海国遗迹，但身份不同。
