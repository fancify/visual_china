# 肃州

## 一句话

河西走廊西段中转一州——东接甘州、西邻瓜州，南屏祁连，自此再西经瓜州至沙州约 600 余里。

## 形貌

肃州是河西走廊中段偏西的一座小州。

南屏祁连——雪岭终年不化，冰川融水北流灌入北山戈壁前的酒泉绿洲；北望马鬃山余脉与一片戈壁；东去甘州、西出便至瓜州，再过瓜州一段便是沙州。州治在酒泉县城——城周约六里许，夯土城墙开四门，城内有刺史衙、酒泉县衙、州学、佛寺数所。绿洲东西不过数十里，但水土足以养活一州——这是河西走廊上"一州一绿洲"的典型样态。

物候是河西的：四月解冻，五月雪水盛，六七月最热，八月霜降，冬日漫长。城外戈壁是常态——出城向北、向西，便没有树。

## 大唐天宝十四载（755年）

肃州在唐人眼里是河西**走廊西段的一处中转之州**——不如东头的凉州大，也不如西头的沙州显，但卡在甘州、沙州之间，是河西七州一条线上少不掉的一环。

唐人若回望肃州的来路：**汉武帝元狩二年（前 121）**霍去病破匈奴、取河西，于此置**酒泉郡**——汉初河西四郡之一（武威、张掖、酒泉、敦煌）。"酒泉"之名，民间传"霍去病以美酒倒入泉中与将士共饮"（按：此为民间传说，正史无确证；《汉书》仅以"金泉"训其名意）。东汉魏晋因之。十六国时期此地数易其手——前凉、前秦、后凉、北凉相继，西凉李暠义熙元年（405）一度自敦煌徙都酒泉，三世二十一年（参 [沙州](./沙州.md) 西凉故宫条）。**隋仁寿二年（602）**置**肃州**——"肃州"之名自此立下。

入唐之后：**武德二年（619）**置肃州；**天宝元年（742）**玄宗诏天下州一律改称郡，肃州改称**酒泉郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州，酒泉郡复称肃州——但 755 年这一刻，城门匾额上写的还是"酒泉郡"。

行政上，755 年的肃州是**陇右道酒泉郡治**——领酒泉、福禄等县（《新唐书·地理志》载肃州下二县）。军镇上归**河西节度使**节度——治所凉州，统凉、甘、肃、瓜、沙、伊、西七州。本州户籍据《旧唐书·地理志》天宝约**2330 户、8476 口**——下州门槛，远不及凉州的二万户，户数亦远少于沙州（约 2.7 倍差距）。

755 年冬，安禄山起兵的消息沿驿道自长安西传——过陇山、入秦州，再过兰州、凉州、甘州，**抵肃州沿驿道旬日可至**，再西沿丝路经瓜州数日便至沙州。这条西传的驿道，便是河西走廊本身。肃州刺史接到的诏令大约是：戒严、备粮、待调。河西精兵东调的命令亦自此过——肃州境内的兵卒东行，与瓜州、沙州、甘州的兵卒在驿道上一站接一站汇向东。这便是 755 年冬肃州人眼里看到的画面：和河西诸州一样的画面。

至于**祁连山**——肃州城南远望即是。冰川融水自南向北灌入酒泉绿洲，是这一州赖以存活的水脉。

## 历史长河

**中晚唐至吐蕃**（755-848）——天宝末河西精兵东调后，吐蕃自青海北上逐步取河西。肃州陷蕃时间据《资治通鉴》《新唐书·吐蕃传》约在**大历元年（766）前后**——较沙州陷蕃（约 786/787）早二十年。自此入吐蕃手八十余年。

**归义军**（848-1036）——大中二年（848）张议潮自沙州起兵，大中五年（851）遣兄张议潭奉沙、瓜、伊、西、甘、**肃**、兰、鄯、河、岷、廓十一州图籍入长安——肃州在十一州之列，正式归唐。归义军延及五代，张氏之后曹氏继任。

**西夏至元明**（1036-1644）——景祐三年（1036）西夏李元昊取肃州。蒙古灭夏（1227）后入元。明洪武五年（1372）冯胜西征至此，立**嘉峪关**于肃州西南——嘉峪关是明长城最西端关隘，**与盛唐肃州无涉**，但自此嘉峪关与酒泉相伴六百年。嘉靖三年（1524）明廷闭嘉峪关、徙关西七卫内附——肃州自此成为明朝实际西界。

**清至今**（1644—）——清承明制，仍设肃州。1913 年民国改肃州为酒泉县；1985 年置**酒泉市**——今甘肃酒泉市，下辖肃州区即唐肃州故地，城址数迁后唐城已难寻完整城圈。
