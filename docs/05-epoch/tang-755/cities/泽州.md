# 泽州

## 一句话

河东道东南南端一上州——治晋城县（今山西晋城市城区），742-758 年间官名"高平郡"；城据太行八陉之"太行陉"北口，与潞州共扼太行—怀庆盆地通道；境内长平为战国长平之战（前 260）故地。

## 大唐天宝十四载（755年）

泽州治**晋城县**，属**河东道**南端。城在太行山西麓的丹河谷地——北接潞州（上党郡）约二百里，南越**太行陉**下太行入河内、怀州（怀庆盆地），西通绛州、东出太行接邢州、洺州。沿革：**隋开皇三年（583）置泽州**，**大业三年（607）改长平郡**；**唐武德元年（618）复置泽州**；**天宝元年（742）改高平郡**——是为 755 年当年的官称；**乾元元年（758）复泽州**。

户口上，**《旧唐书·地理志》载泽州（高平郡）为上州（旧唐书载）**——天宝间**户 27,822 / 口 257,090**（《新唐书》同列上州一档；唐人户口次序在潞州之下，但远在河东道下州之上）。按：户口实在 T3 体量（户 > 25,000），**升 T3**。郡领晋城、高平、阳城、沁水、端氏、陵川等县——州境覆盖今山西晋城市全境，地势是太行西麓的山间台原与丹河、沁水两河谷地。

泽州在唐人战略地图上最堪一提的，是**太行陉**——"太行八陉"中靠南的一陉，自泽州晋城南出，越太行山脊下河内（今焦作-沁阳一带），是河东道东南端入河南道的咽喉之一。州城南去约四十里即陉口，陉道蜿蜒七十余里出山。**与潞州的天井关（南扼河南道）、太行陉（东南扼河内）一道，泽州与潞州共扼河东东南通向河南的两条山口**——这是河东南端 vs 河南北部物资军旅往来的两条主线。

泽州境内**高平县**——即战国**长平**故地。**周赧王五十五年 / 秦昭襄王四十七年（前 260）**，秦将白起大破赵括于长平，坑赵卒四十余万——《史记·白起王翦列传》详载之，为战国史上最惨烈一役。盛唐文人入高平县，自城外野望，**长平古战场的丘陇尚可指认**——已是一千零十五年前旧事。"高平"二字自汉以下县名延续，唐高平县即治今山西高平市旧址。

755 年冬安禄山反范阳，叛军主力沿河北平原南下扑洛阳——泽州虽不在第一线，但太行陉一旦失守则河东南端动摇；次年（756）河东节度使李光弼自太原南下，泽、潞一线即为河东军南向勤王的依托。**安史七年（755-762）间，泽州与潞州、相州、邢州、洺州、磁州一带反复为战区**——昭义军节镇日后即由此一线成形。

## 历史长河（755 → 今）

中晚唐泽州属**昭义军节度使**辖（治潞州，领泽、潞、邢、洺、磁五州；大历元年 / 766 始置）——是中唐倚重的"忠藩"之一。**会昌四年（844）**武宗讨刘稹之役，泽州随昭义军反正归朝，是会昌平泽潞的核心战区之一。**唐末**李克用据河东，泽州入河东军势力范围。**五代**泽州数易其主——**后晋天福间**（约 936-944）仍置泽州；**后周显德间**柴荣亲征北汉于泽州一线（高平之战 / 显德元年 / 954——周世宗大破北汉，关河东南门户）——此役与战国长平之战同地相隔一千二百年。**北宋**泽州属河东路；金代属河东南路；元属泽州司侯司、泽州路。**明洪武二年（1369）**泽州属山西布政司平阳府，后属泽州；**清雍正六年（1728）**升**泽州府**，治凤台县（雍正间析晋城析置）。**民国二年（1913）**废府存县——晋城县承之，"泽州"千年州名退出现行行政地名（按：晋城自唐至清为州治县名，"晋城"作为市名是 **1985 年撤晋东南地区析置晋城市**以后事，盛唐 755 时不可用作主名）。今晋城市辖区主城在唐泽州晋城县、清泽州府凤台县旧址上。

——《旧唐书·地理志二·河东道·泽州（高平郡）》（节录）
