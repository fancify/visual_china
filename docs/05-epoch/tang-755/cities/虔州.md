# 虔州

## 一句话

江南西道最南一上州——治赣县（今江西赣州），章贡二水于城下合流为赣江南源，州南翻大庾岭即入岭南韶州，是开元四年（716）张九龄凿通大庾岭新道之后，江南向岭南南北通衢的岭北门户。

## 大唐天宝十四载（755年）

虔州治**赣县**，在章江、贡江交汇之处——两水自西、自东南来，于赣县城下合流北去（按盛唐人地理认识，这一段水即赣江南源；今人称章贡合流以下方为赣江干流）。州属**江南西道**，地处一道最南——再南便是**大庾岭**，翻过岭即入岭南道韶州始兴郡。沿革清楚：**隋为南康郡**，**唐武德五年（622）平江左置虔州**；**天宝元年（742）**随玄宗朝"改州为郡"通制改为**南康郡**——755 这一刻官称即"南康郡"，民间口语仍呼"虔州"，两名互换不分；**乾元元年（758）**肃宗复改郡为州——但这是 755 之后的事。"虔"字得名旧说取自州境内**虔化山、虔化县**——县在今江西宁都境内。州境领县据《新唐书》为七，《旧唐书》载天宝六县：赣、虔化、南康、雩都、信丰、大庾、安远（《旧唐书》六县、《新唐书》七县——两书小有出入，七县为开元后增置之数）。

《**新唐书·卷四十一·地理志五·江南西道·虔州南康郡**》明列虔州为**上州**——**户三万七千六百四十七，口二十七万五千四百一十**；土贡丝布、紵布、竹綀、石蜜、梅、桂子、斑竹。《**旧唐书·卷四十·地理三·江南西道·虔州**》同载："**旧领县四，户八千九百九十四，口三万九千九百一。天宝领县六，户三万七千六百四十七，口二十七万五千四百一十**"——武德旧籍至天宝百三十余年间，户增四倍有余、口增近七倍，是盛唐江南西道南端开发最显著的一州。这数字背后是**开元四年（716）张九龄主持开凿大庾岭新道**——岭路一通，虔州自江南西道边鄙一州，跃为南北通衢之上的转运重镇。岭南贡赋、海舶舶来品（香药、犀角、珍珠）自韶州翻岭北上，至虔州大庾县下舟入章江，顺流过赣县合贡江为赣江南源，再北去吉州、洪州，由洪州入江——这是盛唐岭南向中原最便捷的一条水陆联运道，**虔州恰卡在岭北换舟一端**。州治赣县城下的水次码头，盛唐渐成江南西道南段一处大集——州户口骤增之实，半在此。755 时虔州无大兵无大事，**州正名南康郡，长官为太守**（742-758 郡制时期，州皆改郡、长官称太守）——岭路畅通、章贡舟楫繁忙，是这一刻虔州的常态。

## 历史长河

**中晚唐**　乾元元年（758）复称虔州。中晚唐贬岭南者多经此——长安南下，过荆襄、过洪吉，至虔州大庾县翻大庾岭入岭南韶州，是中唐贬官诗里反复出现的一条南去之路。

**两宋**　北宋虔州属江南西路。**南宋绍兴二十三年（1153）**，因"虔"字带兵戈不祥之议，改虔州为**赣州**——"赣州"之名自此立，沿用至今；"虔州"之号至此正式退出建制。

**元明清**　元为赣州路，明清为赣州府——治赣县。明清赣州府控江西最南，向岭南、向闽西、向湘东的陆路要冲，是江西一省南门。王阳明正德末年提督南赣汀漳军务即驻赣州。

**民国至今**　1913 年废府存县称赣县。今**江西省赣州市**——市区中心仍在章江、贡江交汇处的老城（今章贡区），与盛唐虔州州治赣县同址，是一千四百余年同一城址的连续。**大庾岭古道**（梅关古道）今为中国保存最完整的古驿道之一，岭北一端即今江西大余县、岭南一端即今广东南雄市——盛唐虔州大庾县与岭南韶州在岭脊相望，今仍如旧。

## 引文

> 虔州，隋南康郡。武德五年，平江左，置虔州。天宝元年，改为南康郡。乾元元年，复为虔州。旧领县四，户八千九百九十四，口三万九千九百一。天宝领县六，户三万七千六百四十七，口二十七万五千四百一十。

——《旧唐书·卷四十·志第二十·地理三·江南西道·虔州》（节录；755 时官称"南康郡"为天宝元年改）

> 虔州南康郡，上。土贡：丝布、紵布、竹綀、石蜜、梅、桂子、斑竹。户三万七千六百四十七，口二十七万五千四百一十。县七。

——《新唐书·卷四十一·志第三十一·地理志五·江南西道·虔州南康郡》（节录；上州州等、户口、土贡 verbatim）
