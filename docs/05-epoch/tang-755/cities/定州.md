# 定州

## 一句话

河北道中北部一州，治安喜县（今河北定州）——太行山东麓南北驿道的中段一站，**博陵崔氏的郡望所在**。

## 形貌

定州城在华北平原西缘、太行山东麓的山前平地上。西望太行——自西向北一线山脊连绵，山口处出滱水（今唐河）东南流，自城西北绕过，注入大陆泽方向的低洼平野。城所在是一片缓坡冲积扇——西高东低、土壤略沙、宜粟麦桑——盛唐河北道腹地最殷实的一段农耕区。**定州正卡在"太行山东麓南北驿道"的中段——南去赵州、邢州、相州、洛阳，北上恒州、易州、幽州——是河北道纵向官驿的关键节点之一**。

物候是华北的：春旱多风、夏热雨集中于七八月、秋爽霜早、冬干寒。城外塬田种粟麦豆，沟谷水边间桑——河北桑蚕在盛唐已盛，定州一带的绢帛是贡品之一。冬季滱水北段有薄冰封冻段（约 12 月至次年 2 月）。**这一片是盛唐河北道的"农耕核心带"——人口稠密、田畴连阡、士族郡望林立**——与岭北的幽州军镇、关内的灵夏边地，气象大不同。

## 大唐天宝十四载（755年）

唐人眼里定州不算新城——战国中山国故地，秦汉为中山郡；**北魏皇始三年（398）置安州，天兴三年（400）改定州**（《新唐书·地理志》），"定州"之名自此始（取"安定北疆"意）。北魏、北齐、北周相沿。隋开皇三年（583）罢郡存州，**隋开皇九年（589）复**；大业初（约 607）改为博陵郡。**唐武德四年（621）复置定州**，治安喜县——州名重立。**天宝元年（742）**玄宗诏改州为郡，定州改称**博陵郡**；**乾元元年（758）**肃宗复改郡为州——定州之名再立。**742 至 758 这十六年间，定州官称"博陵郡"**，民间口语仍呼"定州"。

755 年的定州属**河北道**，州治在安喜县。据《旧唐书·地理志二》记天宝间定州约 **7-8 万户**——稳稳的**上州**门槛之上（盛唐州郡分等以四万户为上州，定州在河北道核心农耕带，户口远超此线）。辖县**本 doc 采《新唐书·地理志》博陵郡 10 县口径**——安喜、北平、唐昌（即义丰，天宝元年改名）、新乐、恒阳、唐县、望都、深泽、无极、鼓城——多沿滱水、唐河、滋水（今木刀沟）一带（安平先天二年 713 还属深州，755 时不在定州辖下）。**口径说明**：《旧唐书·地理志》载天宝口径含鼓城而别记深泽他属，与《新唐书》10 县口径互有出入；本 doc 从《新唐书》更新口径，含深泽。城内有郡衙、安喜县治、驿馆。

**定州最重的一笔，是"博陵崔氏"的郡望**——河北五姓七望之一。崔氏出自姜姓齐国丁公，汉魏以来分清河崔、博陵崔两大支；博陵崔氏自汉末崔寔、北魏崔楷以下，世代显宦，至唐已逾五百年门第。**盛唐前期博陵崔氏在朝者代有人在，但具体到天宝十四载（755）哪几位崔姓宰臣、刺史出于博陵房支、哪几位实际居住定州——史料散在两《唐书》列传中，本 doc 不一一坐实**（hedge：唐时士族居籍与郡望已分离，博陵崔氏多数支系并不实际住在定州，而是散居洛阳、长安）。所确者，**"博陵"二字在盛唐人口中即是崔氏门第的代名词**，定州城里的崔氏祠庙、族田、谱牒仍在。

天宝十四载冬安禄山反范阳——定州地处幽州（范阳节度治）至洛阳的驿道正中段，是叛军南下河北的必经之地。安禄山十一月初九自范阳起兵——十一月、十二月间，叛军南下席卷河北诸州，**定州（博陵郡）守备一时崩溃，旋陷于叛军**（《资治通鉴》卷二一七）。次年（756）颜真卿、颜杲卿兄弟在常山、平原起兵反正，河北十七郡一度归唐——博陵郡亦在其中，但旋复陷。**博陵郡这一冬目送的，是河北道由农耕腹地翻转为兵燹战场的一刻**。

## 历史长河

**中晚唐**　乾元元年（758）复州名为定州（博陵郡旧名废）。安史乱后定州属**成德节度使**辖（治恒州，即今正定）——河朔三镇之一，唐廷只能羁縻，定州随之半离朝廷直辖近一百五十年。元和年间宪宗削藩，定州一度归朝；穆宗朝复入河朔。**博陵崔氏在中晚唐仍出宰相数员**（崔群、崔铉、崔慎由、崔胤等，多属博陵房或其分支）——但实际居籍多在长安、洛阳。

**宋元明清**　北宋定州属河北西路，仍为州治——契丹南下的前沿防线，北宋曾筑定州军城，名将李允则、韩琦皆任过定州。**北宋咸平至至和间（1001-1055）于定州城内建开元寺塔（料敌塔）**——通高 84 米，砖塔，为现存中国最高的砖塔，至今仍立于定州城中。金、元属真定路。明洪武初定州属真定府；清雍正二年（1724）升直隶州，辖深泽、束鹿、安国、曲阳、阜平、行唐六县——清代**直隶定州**辖境延续至清末。

**民国至今**　1913 年废州改县，**定州**降为**定县**——盛唐"定州"之名一度断绝。1986 年撤定县复设**定州市**——"定州"之名重立，沿用至今（今河北省定州市，省直辖）。**北魏置州以来一千五百九十一年，"定州"二字三度立而不废**。开元寺塔仍在；汉中山靖王刘胜墓（金缕玉衣出土地）在满城（今河北保定满城区），唐定州境内不直接相关。**唯一可与盛唐定州人共看的，是太行山东麓那条南北驿道的走向**——自定州正南去赵州、北上恒州的官路，今 107 国道与之大致重叠；山形与水脉，一千二百年未改。
