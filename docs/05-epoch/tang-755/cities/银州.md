# 银州

## 一句话

关内道北线一座下州——治儒林县，**与今宁夏银川市无涉**；城在无定河中游（今陕西榆林市横山、米脂一带），742-758 年间名"银川郡"，是夏州东南邻、党项拓拔氏内徙后的一处根据地。

## 形貌

银州城在鄂尔多斯高原南缘——无定河自西北而东南穿城下而过，城东南去绥州约百余里，西北去夏州（统万城/白城子）二百余里。州治儒林县——城据河谷台地，南北两面是黄土沟塬，城外即半农半牧之地。城西北循官道过夏州、再循朔方一线可入河套；东南循无定河下游至绥州，再南折可入鄜延、过延州南下长安。物候是黄土塬北缘：春旱多沙、夏短雨集、秋早霜、冬严寒——勉强种粟黍，多生柠条、沙柳与河柳。

## 大唐天宝十四载（755年）

银州沿革——据《元和郡县图志·卷四》：**北周武帝保定二年（562）分置银州**；**隋大业二年（606）废银州**，县属雕阴郡；**唐贞观二年（628）平梁师都，重置银州**（《旧唐书·地理志一》同此年），治儒林县，属关内道。**天宝元年（742）**玄宗诏天下州一律改郡，银州改称**银川郡**——这是 755 年当年的官称；**乾元元年（758）**肃宗复改郡为州，"银川郡"之名前后用了十六年。

755 年的银州属**关内道**、治儒林县——《旧唐书·地理志一》记为**下州**（边州），辖县四：**儒林**（州治）、**抚宁**、**真乡**、**开光**——皆无定河中游谷地县治。州境户口据《旧唐书·地理志》载：**户 7,602，口 45,527**。城周不大，无折冲府之大数，亦不立都督府——但作为夏州东南、绥州西北之间的一处腰节，本州驿务关连朔方节度使辖境东缘的军政线。**银州——银川郡——治儒林县所在，约在今陕西榆林市米脂县至横山区党岔镇南一带，与今宁夏回族自治区银川市相距八百里以上、毫无沿革关系**（今银川市为明清宁夏卫升宁夏府之地，唐时为灵州所辖；本 doc 所写"银川郡"专指 742-758 年间银州所改之郡名，并非地名"银川"）。

银州境的本色是**党项拓拔氏故地**——党项是羌系一支，原居青藏高原东北缘的赐支河（今黄河上游）一带；**唐高祖武德七年（624）前后**拓拔氏自吐谷浑南迁压力下始内徙，散居银、夏、绥、宥诸州一线。**至中宗朝（705-710）以后**党项各部进一步内徙——盛唐银州境内已是党项拓拔部落与汉人州县杂处的局面，城内是唐人州治、城外塬地多见党项部落营盘。755 年这一刻，**党项尚是唐朝边州羁縻部落**，未成大势——这是这一州后来千年命运的伏笔。

## 历史长河

**中晚唐**　乾元元年（758）复银州。安史乱后党项内徙加剧——拓拔氏一支在夏、银、绥一线渐成气候。**僖宗广明元年（880）黄巢入长安**，党项首领**拓拔思恭**自夏州率部勤王——僖宗**中和二年（882）以功授定难军节度使**，治夏州，辖**夏、绥、银、宥、静五州**——银州自此入定难军辖下，**党项李氏（拓拔思恭赐姓李）自此世袭节镇**，前后两百年。

**宋元**　北宋初仍以李氏镇夏州。**太平兴国七年（982）**李继捧献夏、绥、银、宥、静五州于宋；其族弟**李继迁**不附宋、走入沙漠、起兵反——银州一带是其早期根据。**宋淳化五年（994）**宋毁统万城——夏州、银州一线作为唐五代两百年的党项根据地至此瓦解。**宝元元年（1038）李元昊于兴庆府（今宁夏银川市）称帝立西夏**——党项李氏立国，国祚百九十年，至蒙古成吉思汗（1227）灭之。**注：西夏立国都城兴庆府（今宁夏银川市）与唐银州（银川郡，今陕西榆林一带）名号相近而非同地，沿革无承袭关系——西夏 1038 年立国距盛唐 755 已近三百年，本 doc 主体所写不预支**。

**元明清至今**　元废银州，地入米脂、绥德；明清两代米脂、横山属延安府或榆林卫。1949 年后属陕西省榆林专区/榆林地区，2000 年榆林撤地设市；**今唐银州治儒林县所在，约在陕西榆林市米脂县至横山区党岔镇南一带——非今宁夏银川市**。盛唐"银川郡"三字，今在《旧唐书》《元和郡县图志》和无定河谷塬上的一段零星残迹里仍可寻。

## 引文

> 銀州　下，隋雕陰郡之儒林縣。貞觀二年，平梁師都置銀州，隋舊名。天寶元年，改為銀川郡。乾元元年，復為銀州。舊領縣四，戶一千四百九十五，口七千七百二。天寶，戶七千六百二，口四萬五千五百二十七。

——《旧唐书·卷三八·地理志一·关内道·银州》（据 zh.wikisource 校录 verbatim）

> 周武帝保定二年，分置銀州。隋大業二年廢銀州，縣屬雕陰郡。貞觀二年平梁師都，於此重置銀州。

——《元和郡县图志·卷四·关内道四·银州》（据 zh.wikisource 校录 verbatim；补银州建置最早年代）
