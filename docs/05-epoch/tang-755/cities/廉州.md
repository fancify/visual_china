# 廉州

## 一句话

岭南道邕管（一说容管）下辖一座下州，治合浦县——北部湾东岸的边远海港，**汉以来九百年的合浦古港**之上，南珠之乡。

## 大唐天宝十四载（755年）

廉州治**合浦县**（今广西北海市合浦县；按"北海市"为民国后所立之名，盛唐尚不存在）。沿革几经更名：**汉武帝元鼎六年（前 111）平南越置合浦郡**——治合浦县，**南海郡西邻**；自此九百年间，合浦县治大体未移。**唐贞观八年（634）于合浦县置廉州**；**天宝元年（742）**玄宗诏改州为郡，廉州改作**合浦郡**——以汉郡旧名；**乾元元年（758）**肃宗复改郡为州，复称廉州——**742 至 758 这十六年间，本州正名"合浦郡"，长官称太守**（非"刺史"），民间口语仍多呼"廉州"。

755 年的廉州属**岭南道**——岭南五府经略下辖。具体节制属**邕管**还是**容管**，两唐书口径差：**《旧唐书·地理志四》归邕管十州之一**（邕管治邕州，今南宁一带）；**《新唐书·地理志》载归容管**（容管治容州，今广西北流一带）——两书并存，本 doc 加 hedge 不强订。《旧唐书·地理志四》载本州为**下州**——天宝间户口约数千户上下（《旧唐书》《元和郡县图志》载数互见，本 doc 不取确数而以"下州、户口稀少"为定性）。辖县三四：**合浦**（州治）、**封山**、**蔡龙**、**大廉**（数县）——皆北部湾东岸沿海与山地县治。城内有合浦郡治、太守署、合浦县衙、馆驿——边远下州的最简班子。

廉州在盛唐的分量，不在户口，而在**两件旧物事**。其一是**汉合浦古港**——**汉武元鼎六年（前 111）置合浦郡时，合浦县便是中原王朝面向南洋的对外港口之一**；《汉书·地理志》载汉船自**徐闻、合浦**出，"船行可五月，有都元国……船行可二月余，有黄支国"——这是中原文献里最早的一条远洋海道，**汉海上丝绸之路东端起点之一**（与雷州半岛南尽之徐闻并称）。盛唐廉州港承的便是这条九百年的旧航道——与林邑、真腊、师子、波斯诸国通商——但盛唐海上丝路的主港早已北移至广州、交州，廉州只是一处中途港、辅港。其二是**南珠**——合浦海域出产珍珠，自汉以来即为贡品，**"合浦珠还"**典故出《后汉书·循吏列传·孟尝传》：东汉顺帝朝合浦太守孟尝至郡，先时官吏贪暴、珠遂渐徙交趾界——孟尝革除前弊、珠复归廉。盛唐合浦郡仍以南珠为贡——这是一州在户口之外，朝廷必记一笔的资源。

天宝十四载冬安禄山反范阳——消息沿驿道南传，自长安过商於、襄州、江陵、潭州、桂州、邕州、再南循陆路过钦州至廉州——抵合浦郡治约需四十余日。合浦郡太守接到的诏令大约与钦州、雷州相仿：戒严、备粮、防境内蛮獠——但岭南再南距河朔战场万里，朝廷未抽调廉州兵北上。海口的渔船、采珠舶照常出入；这一州在那个冬天，是大唐最南的安静里的一处。

## 历史长河

中晚唐廉州随岭南起伏——**咸通元年（860）**南诏军北上攻陷安南，钦廉一线受波及；**咸通七年（866）高骈收复安南**后复安。五代入南汉（917-971）。**北宋开宝四年（971）平南汉**复入中原，属广南西路。**北宋熙宁九年（1076）**郭逵伐安南，钦州、廉州为北路宋军出发地之一。元置廉州路。明洪武元年（1368）改**廉州府**——治合浦县，明清两代廉州府辖合浦、灵山、钦州一线，是岭南南端府治之一。清末民初**廉州府废**；1912 年存合浦县。**1949 年后合浦县属广东省**——**1965 年钦廉一线划归广西，1951 年北海设市、1983 年北海升地级市辖合浦县**——"北海市"作为地级市名号为现代所立，盛唐尚不存在此称。今广西北海市合浦县治，约在唐廉州合浦县旧址一带。**合浦汉墓群**——东郊一带分布汉代墓葬数千座，1957 年起陆续发掘，出土玛瑙、琥珀、玻璃器、波斯陶等舶来品——印证汉以来合浦作为对外港的地位；但**此为后世考古发现，非盛唐人所知**，盛唐合浦郡太守见到的，只是城外田畴间隆起的旧冢累累。
