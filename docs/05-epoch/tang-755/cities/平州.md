# 平州

## 一句话

河北道最东北、唐境出入关外的咽喉——治卢龙县（今河北秦皇岛卢龙县），742-758 年间名"北平郡"；东去渝关（即后世山海关地，755 时只有渝关 / 卢龙塞之名）一日程，是幽州东出营州 / 平卢路的中段一站。

## 形貌

平州夹在燕山东端南麓与渤海北岸之间——南距海约百里，北倚卢龙塞群山，东出渝关便是辽西走廊的西头，西循官道经渔阳（今天津蓟县）二百余里至幽州。城在卢龙县（今河北卢龙）的一片台地上——青龙河、滦河从西北汇入，城下不远即古孤竹国故地，殷商伯夷、叔齐相传即葬于此（《史记·伯夷列传》），唐人路过多有凭吊。物候是河北最北的：四月才解冻，霜冻早，秋深风沙起自塞外。

## 大唐天宝十四载（755年）

平州是唐人眼里**幽州东出营州必过的那一关**——治卢龙县，**天宝元年（742）改称北平郡**，沿革几度迁名：隋开皇九年（589）置平州、大业三年（607）改北平郡、唐武德二年（619）复平州、天宝元年（742）再改北平郡、乾元元年（758）再复平州。**《旧唐书·地理志》载平州（北平郡）为下州**——天宝户口在河北道诸州中靠后，远不及幽州六万七千余户的体量。755 年这一刻匾上写"北平郡"，太守一人领郡事，下辖卢龙、石城、马城三县。

755 年时**山海关之名尚未出现**——明洪武十四年（1381）徐达东巡，相中此地"两山一海"之势，立山海卫筑山海关，自此"山海关"始为关名；755 年时此处是**渝关**（亦作榆关），城东百里许，《通典》《元和郡县图志》所载即此；汉名又称**卢龙塞**——河北通辽西的咽喉，**范阳与平卢两镇之间调兵必经此道**。安禄山自天宝三载（744）兼领平卢节度、天宝十载（751）再兼河东，**起兵前**（755 十一月之前）已将平卢本镇兵自营州沿渝关-卢龙塞一线西调至范阳集中；**十一月初九起兵于范阳，主力则南下河北、河南**，下博陵、常山、邢洺以扑东都，并不东去营州。渝关与平州城下的这道官道，是反叛前夕平卢兵西并范阳的腰节，而非起兵后的进军路线。

## 历史长河

**乾元元年（758）**复改平州——同年置**卢龙节度使**于幽州，平州自此为卢龙镇辖下。中晚唐河朔三镇中卢龙一镇之名即取自此地的汉古地名"卢龙塞"，但节度治在幽州，平州只是辖州。**辽**改平州为辽兴军节度，治卢龙；**金**承辽制；**元**置永平路；**明洪武初**改永平府，**洪武十四年（1381）**徐达自永平府东出立**山海关**——自此卢龙旧塞、唐渝关一段山海合抱处有了"山海关"之名。清沿明制，仍永平府，治卢龙；民国废府存县；今河北卢龙县属秦皇岛市，"秦皇岛"市名取自秦始皇东巡入海求仙的传说处（在今秦皇岛海港区），但作为正式地名是民国以后的事——盛唐时此海岸尚无市邑。唐平州治卢龙县的那块台地，今仍是卢龙县城所在。

## 引文

> 平州，北平郡。下。隋开皇九年置平州，大业三年改为北平郡。武德二年复为平州，天宝元年改为北平郡。乾元元年复为平州。领县三：卢龙、石城、马城。

——《旧唐书·地理志二·河北道·平州（北平郡）》（节录）
