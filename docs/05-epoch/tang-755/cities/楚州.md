# 楚州

## 一句话

淮南道楚州，742-758 年间名"淮阴郡"——治山阳县，邗沟北端与泗水入淮处之间的运河节点；古淮阴侯韩信故里。

## 形貌

楚州城在淮水南岸——治山阳县（今江苏淮安市淮安区），城北二三里便临淮河，城西不远是泗水入淮口，城南则邗沟北端**末口**自此入淮。**淮河 + 泗水 + 邗沟 三水交汇的咽喉，便是这一州的根本所在**：自江南北上的漕船经邗沟至末口入淮，再溯泗水北上、或西转通济渠至汴州——楚州当其中转节点。城周不大，唐州县规模中的中州；但运河上位置之要、漕政分量之重，远超本州本身的户口体量。州境东接海州（黄海西岸），北越淮入河南道徐州、泗州，南渡邗沟至扬州，西沿淮上行至寿州——四向皆通而尤以南北漕路最重。

物候属淮南北端——比扬州凉、比徐州暖；冬有雪而不深，夏热湿；春秋两季运河上漕船最盛。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

楚州境是**周秦以来的旧地**。先秦此地属淮夷一支，吴王夫差**周敬王三十四年（前 486）凿邗沟、通江淮**——邗沟北端入淮之口便在今楚州境，这是邗沟与淮泗交汇格局的最早定基。**秦置淮阴县**——"淮阴"取淮水之南之意，属泗水郡；秦末**淮阴人韩信**起于此地——《史记·淮阴侯列传》记韩信少时贫无行、钓城下，漂母数十日饭之，后封淮阴侯——这是楚州境内最大的一段古事。**西汉枚乘**亦为淮阴人——《七发》传世，是汉初辞赋一脉的早期作家。东汉末至魏晋，淮阴一带先后为广陵郡、临淮郡所属；东晋以来江淮战事不断——刘宋元嘉二十七年（450）魏太武帝南伐至瓜步，江淮一片荒残；南齐、梁、陈相继据守。

**隋开皇九年（589）平陈、置楚州**——以山阳县为治；"楚州"之名取古楚地之意（按：先秦此地实为吴、越、楚先后所争，至战国后期入楚，故名）。隋大业三年（607）炀帝改州为郡，**楚州境拆并入江都郡 / 邗江县**（《隋书·地理志》），楚州短暂被废。唐**武德四年（621）置东楚州**——治山阳县；**武德八年（625）改楚州**——"东"字去之，自此正式称楚州，治所稳定在山阳。**天宝元年（742）**玄宗诏天下州改郡，楚州改**淮阴郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州——楚州之名再立，沿用至宋。

### 此刻

天宝十四载（755）——官称"淮阴郡"，民间口语仍呼"楚州"。淮南道东北端，**领县四**（按新唐书地理志五·楚州淮阴郡）：山阳（州治）、盐城、**安宜**（宝应元年/762 因获定国宝改名"宝应"，755 时仍称安宜）、淮阴。**按**：旧唐书地理志三·楚州条作"天宝领县五"，于上四县之外尚列盱眙——然盱眙县先天元年（712）已改属泗州（参《新唐书·地理志五》泗州条），旧唐书楚州条犹列盱眙，盖沿用武德八年至先天元年间旧记未及厘清，新唐书改正为四县。本 doc 从新唐书四县说。本郡户口据《旧唐书·地理志三·楚州》载**户二万六千六十二，口十五万三千**——州等第旧唐书作"中"（与户口口径相称），新唐书作"紧"（按运河漕政分量定）；本 doc 取"中州（新唐书作紧）"双引为准。城内主事：**淮阴郡太守**（742-758 行郡制时本州长官称太守，乾元元年复州后方称刺史）、郡司马、郡司仓参军、郡司户参军——一套完整的州县官员系统。**755 时楚州无节度使治**——淮南道时由**淮南采访使**总察道境（治扬州，参 [扬州](./扬州.md)）；淮南节度使要到至德二载（757）方置，755 当年尚未存在。楚州为采访使治下中州（紧）。

这一年城里的实务有三宗。**漕运第一宗**：江淮漕米**经扬州邗沟北运至楚州**末口入淮、再转通济渠（泗水入淮口在盱眙）西北上汴州——开元二十二年（734）裴耀卿改革江淮漕运后建"江淮河三段转运法"，楚州正是**第一段邗沟转运的北端、第二段汴渠转运的南端**——两段在此咬合。城南末口、城西泗水入淮口附近常年泊船——官仓、转运院、驿馆分布两岸。**互市第二宗**：江北一线**新罗坊**——盛唐黄海西岸有新罗坊明确记载者，扬州、楚州、登州为最著（《入唐求法巡礼行记》圆仁记 9 世纪楚州新罗坊事甚详；按 755 时已有新罗使、新罗商人留居楚州，史志未直接载具体坊名，但其后期繁盛溯源至盛唐已可推）。**古迹第三宗**：城西约十里有**韩信钓台**——传韩信少时垂钓处；台旁有**漂母祠**（后世所建，盛唐时具体形制存疑）——盛唐楚州人指认这两处遗迹时，距韩信前 196 年被诛已九百五十余年。

十一月初九安禄山自范阳起兵——消息沿驿道南传，约月余抵楚州。叛军主力南下经河北、过黄河西攻洛阳；**楚州城在洛阳东南远端，755 当年并未直接陷落**。但江淮漕路自此年起紧绷——至德二载（757）正月叛军尹子奇围睢阳（宋州），张巡、许远死守十月，**正是楚州、扬州、寿州一线江淮漕粮维系长安最危急的时刻**。这是 755 年时尚未发生的事——但 755 这一冬城里的兵戍换防、漕船改道、粮草调发，已与一年半后的睢阳之围远遥相关。

## 历史长河

**中晚唐**　乾元元年（758）复改淮阴郡为楚州。代宗宝应元年（762）刘展之乱波及江淮——田神功平乱时大食、波斯贾胡死者数千（参 [扬州](./扬州.md)），楚州一带亦遭兵祸。**开成四年（839）日本入唐请益僧圆仁经楚州赴长安**——《入唐求法巡礼行记》详记楚州新罗坊、新罗译语金正南、楚州淮河口形势——是 9 世纪楚州运河节点最详的实物记录。**乾符五年（878）黄巢军南下**——楚州震动；唐亡后楚州先后入吴、南唐。

**宋元明清**　**北宋楚州**属淮南东路，**绍兴五年（1135）金兵南侵，韩世忠率部驻楚州十余年**——筑山阳城拒金；楚州为南宋淮东三大军镇之一（其余为扬州、盱眙军）。**金属山东东路；元属淮安路**——楚州之名渐被"淮安"压过。**明洪武元年（1368）改淮安府**——治山阳县；**明清两代漕运总督、河道总督驻淮安**——大运河漕政、黄河（明清黄河南徙夺淮入海）河政两大要务皆集此城；淮安、扬州、苏州、杭州并称运河四大都会。**漕运总督公署、河道总督衙署**至今犹有遗存——但这是**明清两代之事，755 时无此制**：唐代漕政集中于扬州淮南节度使府与汴州转运院。**清咸丰五年（1855）黄河北徙复归山东入海**——淮安运河重要性顿减；漕运总督于光绪三十年（1904）裁撤。

**民国至今**　**1914 年置淮安县**——"淮安"二字作为县级行政名号正式定下，承续明清淮安府旧称；1948 年置淮安专区；1983 年淮阴市；2001 年**改淮安市**——今江苏省地级淮安市，市辖淮安区即唐山阳县城旧址范围。**韩信钓台、漂母祠**在今淮安区故城西北——历代屡毁屡建，今所见为明清以来重建格局。**末口故址**在今淮安区北辰坊一带——夯土与河道遗迹尚可指认。**唯一可与盛唐楚州人共看的，是淮河南岸的城基方位与运河南北纵贯的总走向**——河道经过千年改易，水势南去入海或北归山东几番反复，但邗沟入淮、泗水入淮这两个交汇节点的地理坐标始终在楚州境内。

## 引文

> 楚州，中。隋江都郡之山阳县。武德四年，臧君相归附，立为东楚州，领山阳、安且、盐城三县。八年，废西楚州，以盱眙来属，仍去「东」字。天宝元年，改为淮阴郡。乾元元年，复为楚州。旧领县四，户三千三百五十七，口一万六千二百六十二。天宝领县五，户二万六千六十二，口十五万三千。在京师西南二千五百一里，至东都一千六百六十里。

——《旧唐书·卷四十·地理志三·淮南道·楚州淮阴郡》（verbatim；按：本条作"天宝领县五"含盱眙，然盱眙先天元年/712 已改属泗州，新唐书地理志五正为四县。州等第本条作"中"，新唐书作"紧"，二说并存）

> 楚州淮阴郡，紧。本江都郡之山阳、安宜县地，臧君相据之，号东楚州。武德四年，君相降，因之，八年更名。土贡：赀布、纻布。户二万六千六十二，口十五万三千。县四。山阳……盐城……宝应，本安宜……上元三年以获定国宝更名……淮阴……

——《新唐书·卷四十一·地理志五·淮南道·楚州淮阴郡》（verbatim 节录）

> 信钓于城下，诸母漂，有一母见信饥，饭信，竟漂数十日。信喜，谓漂母曰："吾必有以重报母。"母怒曰："大丈夫不能自食，吾哀王孙而进食，岂望报乎！"

——《史记·卷九十二·淮阴侯列传》。盛唐楚州人指认城西韩信钓台、漂母祠时，所凭的是这一段史记本传——距韩信前 196 年被诛已九百五十余年。
