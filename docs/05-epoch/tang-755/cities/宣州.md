# 宣州

## 一句话

江南西道东部一州，治宣城县（今安徽宣城）——长江南岸丘陵地带、青弋江畔的"谢公旧治"，谢朓楼下、敬亭山前，是盛唐李白晚年最常停的一座江南小城。

> **disambig**　唐"宣州 / 宣城郡"治宣城县（今安徽宣城市），属**江南西道**——与同期江南东道润州（丹阳郡治丹徒，今江苏镇江）不可混。润州.md 内已点出："汉丹阳郡治宛陵（今安徽宣城），与唐丹阳郡治丹徒并非一地"——汉丹阳郡的旧治宛陵，正是本州治宣城县的前身；**唐丹阳郡（润州）与汉丹阳郡（宛陵/宣州）名同地异**，盛唐人多以"丹阳"称润州、以"宣城"或"宣州"称本州，不至混。

## 形貌

宣州不在长江干流上，而在长江南岸的丘陵之间。城贴**青弋江**东岸——青弋江自南面的黄山北麓发源、曲折北流过宣城、再西北至芜湖（唐属宣州芜湖县）注入长江。城在江岸一片低台地上，城周不大，南门外有渡口与码头——南来的茶、纸、笔、铜锡沿青弋江下宣城，北去江口（芜湖）的纲船与盐船也在此装卸。**城北约五里便是敬亭山**——非高山，海拔仅三百余米，林麓宜步，盛唐人登宣城常先上敬亭。城东南郊低丘连绵是宣州茶园、桑园所在；城西则是稻田与水塘。

物候是江南北端的：四月柳色、五月梅雨、六七月暑湿、九月秋爽——比长江北岸的扬州、寿州都稍温润。青弋江冬季不冻、水位有汛。**宣州在长江南岸丘陵带上的位置——既不当大江干道、又不当大运河——是江南西道东端一座"半山半水"的州治**。城东去润州、苏州一线，城西循长江南岸西去至池州、江州，城北越江至和州。

## 大唐天宝十四载（755年）

宣州在唐人眼里是一座旧城——汉为**丹阳郡**之治宛陵（西汉丹阳郡治此），三国孙吴属丹阳郡，东晋南朝皆为宛陵旧县。**南齐建武年间（约 495-498）大诗人谢朓除宣城太守**——《南齐书·谢朓传》记其事。谢朓在宣城太守任上前后约二年，作《晚登三山还望京邑》《之宣城郡出新林浦向板桥》等名篇，并相传**于城北建（或题）一楼**，后世遂称"**谢朓楼**"（亦称"谢公楼""北楼"）——这是宣州在中古文学史上一次最重要的留名。

隋开皇九年（589）平陈后置**宣州**——治宣城县；**大业三年（607）**炀帝罢州为郡，改宣州为**宣城郡**。**唐武德三年（620）**复置宣州，治宣城县。**贞观元年（627）**分天下为十道，宣州属江南道；**开元二十一年（733）**江南道分东、西二道——**宣州属江南西道**（与洪、江、岳、潭、衡诸州同道；东邻润、苏、湖、杭属江南东道）。**天宝元年（742）**玄宗诏改州为郡，宣州改称**宣城郡**——这是 755 年当年的官称。**乾元元年（758）**肃宗复改郡为州——"宣城郡"前后用十六年。**注**：本州 dao 归属以《新唐书·地理志五·江南西道·宣州宣城郡》及《元和郡县图志·卷二十八·江南道四》为准，属**江南西道**——本目录 POI-INVENTORY 此前曾误置江南东表段，2026-05-13 codex r1 review 已校正归江南西。

755 年的宣州属**江南西道宣城郡**，州治宣城县。据《旧唐书·卷四十·地理志三·江南道·宣州》原载："宣州……天宝元年改为宣城郡。乾元元年复为宣州。**领县九：宣城、当涂、泾、溧水、溧阳、南陵、绥安、广德、宁国**。**户一十二万一千二百四，口八十八万四千九百八十五**"——按唐州县等第（《唐六典·尚书户部》"户满四万为上州"），宣州户口逾十二万，**远超上州门槛**，**在江南西道居首**，远高于洪州（约 5.5 万户）与潭州（约 3.2 万户）。**江南西道采访使治洪州**，宣州为江南西道东端最重要的一座州城——长江南岸丘陵带与江南河南端之间的咬合点。

> **户口异文**　《元和郡县图志·卷二十八·宣州》载"开元户八万八千余、乡一百八十"——开元（713-741）数；本 doc 引文段亦保留此数。**《旧唐书·地理志三》载天宝盈数为户121,204、口884,985**——天宝（742-756）数，时点更贴近 755。两数皆 verified，分别对应开元末与天宝中的两次户口统计。

但宣州在 755 年这一刻最有分量的事，不在户口、不在漕运，而在**李白**。

天宝十二载（753）秋，**李白自梁苑、东鲁一路南下，至宣州——登谢朓楼，登敬亭山**。同年秋作《宣州谢朓楼饯别校书叔云》（"弃我去者，昨日之日不可留；乱我心者，今日之日多烦忧。……蓬莱文章建安骨，中间小谢又清发"）——为族叔李云饯别于谢朓楼上。又作《独坐敬亭山》（"众鸟高飞尽，孤云独去闲；相看两不厌，只有敬亭山"），《秋登宣城谢朓北楼》（"江城如画里，山晚望晴空；两水夹明镜，双桥落彩虹"）。**李白此时五十三岁，离长安已七年，一度欲终老宣州**——是他晚年漫游中停留时间最长的江南州治之一。**天宝十四载（755）这一刻，李白人在宣州一带、当涂之间，时年五十五岁**——年底安史之乱起后他辗转皖南、入永王军、获罪、流夜郎、遇赦——晚年最后一次回宣州、当涂，是宝应元年（762）卒于当涂叔父李阳冰处。

所以**宣州在 755 这一年，是李白人在的地方**——这件事，比户口与漕运都更要紧。

宣州物产盛唐已小有名：泾县产纸（《新唐书·地理志》宣州贡纸——唐人已用"宣州纸"作书写），宣城一带产笔（白居易《紫毫笔》"宣城之人采为笔"是元和间事），南陵县利国山产铜银。**但"宣笔""宣纸"作为与湖笔、徽墨并称的天下名物，是宋元以后的事**——755 当年宣州笔纸已产，未定型为全国名号；本 doc 严守 755 时态，不预支后世盛名。

## 历史长河

**中晚唐**　乾元元年（758）复名宣州。**晚唐宣州置宣歙观察使**——治宣州，与江南西、江南东并列为江南三大方镇之一；杜牧会昌中（约 842-844）任宣州团练判官，治宣城——刘禹锡、白居易、杜牧、皮日休、陆龟蒙皆留宣州题咏。光启间黄巢乱后，天复二年（902）宣州入杨吴。

**宋元至今**　南唐属江南军镇；北宋归宋后**乾道二年（1166）升宣州为宁国府**——"宁国"作为府名自此立，盛唐"宣州/宣城郡"二名作为正式州名至此终。北宋宣笔、宣纸渐为天下名物——苏轼、梅尧臣（宣城本邑人）皆有题咏。元宁国路、明清宁国府皆治宣城。**1912 年废府存县**，宣城县沿盛唐"宣城"二字以下未改；2000 年撤县设宣州区、属宣城地级市——治所即唐宣州州治旧址叠在今宣城市中心。**敬亭山仍在城北**（今敬亭山国家森林公园），**谢朓楼经历代重修**（南齐谢朓所题旧楼早佚，**今楼为 1997 年复建**，址在旧基附近）；青弋江仍自城西流过。盛唐李白笔下"两水夹明镜，双桥落彩虹"的宣州城外水景——青弋江与水阳江在此分汇——**水道千余年来略有改易，两水夹城的大势未变**。

## 引文

李白《独坐敬亭山》（天宝十二载至十三载 753-754 间作于宣州城北敬亭山）：

> 众鸟高飞尽，孤云独去闲。
> 相看两不厌，只有敬亭山。

李白《宣州谢朓楼饯别校书叔云》节录（753 秋作于谢朓楼）："弃我去者，昨日之日不可留；乱我心者，今日之日多烦忧。……蓬莱文章建安骨，中间小谢又清发。俱怀逸兴壮思飞，欲上青天揽明月。"——"小谢"即谢朓，盛唐李白与南齐谢朓隔二百五十年同登一楼。

《元和郡县图志·卷二十八·宣州》节录："宣州，宣城郡。开元户八万八千余，乡一百八十。州城本汉宛陵县——丹阳郡治。……南至池州二百里，北至和州三百里，东至湖州三百里，西至池州二百里。"——**汉丹阳郡治宛陵，即唐宣州州治宣城县**；唐丹阳郡（润州）治丹徒，名同地异。
