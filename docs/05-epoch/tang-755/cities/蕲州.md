# 蕲州

## 一句话

淮南道长江北岸一州、大别山南麓——治蕲春县（今湖北黄冈市蕲春县），742-758 年间名"蕲春郡"；蕲水自北而南入江，与江南江州、鄂州一线隔江对望。

## 形貌

蕲州城在长江北岸一片低台地上——**蕲水自大别山南麓南流绕城入江**，城北二三十里便是大别山的南缘浅丘，城南十余里即长江北岸。这是淮南道沿江一线最西的几州之一——**西邻黄州 / 齐安郡（今湖北黄冈一带），再西过武昌郡（鄂州，属江南西道）**。物候是江北的：四月麦熟、五月梅雨、六七月江风南来；冬日较江南稍冷，但比河北温润得多。蕲水两岸药材出名，**山间野生药草自唐以前便为江北土贡之一**——蕲艾、蕲竹、蕲蛇之名后世更甚。

## 大唐天宝十四载（755年）

蕲州在唐人眼里是**江北沿江一州、不甚显赫**——治蕲春县，742 年改称蕲春郡。沿革：隋开皇九年（589）置蕲州，大业三年（607）改蕲春郡；**唐武德四年（621）复蕲州**——属淮南道；**天宝元年（742）**玄宗诏天下州改郡，蕲州改称**蕲春郡**——这是 755 年当年的官称；**乾元元年（758）**复改郡为州，"蕲春郡"之名前后用了十六年。**《旧唐书·地理志》载蕲州（蕲春郡）为中州**——天宝户口在淮南道诸州中靠中后，远不及扬州、寿州。755 年这一刻匾上写"蕲春郡"，**太守一人领郡事**（注：唐州一级长官，**742 改州为郡后称"太守"，758 复郡为州后称"刺史"**——755 当年只称"蕲春郡太守"，不称"蕲州刺史"）；**天宝十四载蕲春郡太守在任者史志阙考**。

755 年的蕲春郡属**淮南道**——淮南节度使一职至德二载（757）方置；755 当年淮南尚无节度使，仅有玄宗开元二十一年（733）所立**淮南采访使**（治扬州）统淮南诸郡。蕲春郡在淮南采访使辖境最西南、临江一隅，与对岸江南西道江州、鄂州一江相望。

## 历史长河

中晚唐蕲州仍为淮南下中州，**广德元年（763）以后淮南藩镇渐成**，蕲州属之。**僖宗中和年间**黄巢起兵后江淮诸州尽乱，蕲州亦在其中。**杨吴、南唐**间蕲州为江北沿江诸州之一，与对岸江州一线江防呼应；**北宋开宝八年（975）**平南唐后归宋，属淮南西路。**元末至正十一年（1351）**——**徐寿辉、邹普胜在蕲州起兵，立"天完"红巾军**，奉弥勒下生之说，蕲水流域一时为江南红巾军初起之地；后陈友谅杀徐寿辉自立——这是蕲州在江南红巾军史上独立一笔，距盛唐已近六百年。

**蕲州历史上最为后世传诵的本地名人**——**明代李时珍（1518-1593）即蕲州蕲春县人**，著《本草纲目》五十二卷，为中国本草学集大成之作。李时珍之父李言闻亦为蕲州医家——蕲州自唐以来便以药材出名（蕲艾、蕲竹、蕲蛇之名后世并立"四蕲"），至明而出李时珍。**注：李时珍距盛唐天宝十四载已七百六十余年——属明代中后期事，与本 doc 所写盛唐 755 时蕲州并非同一时代；盛唐蕲州人自然不知道这座城八百年后将出此一位本草大家**。

明清蕲州属黄州府；**民国二年（1913）废府存县**，蕲州改蕲春县，属湖北省。1949 年后属黄冈专区、黄冈市；今蕲春县属湖北省黄冈市，县城所在地仍是唐蕲州 / 蕲春郡州治旧址范围。**"黄冈"作为府名始于明初，作为市名是 1995 年地区改市后事**——盛唐 755 时此名不立。蕲春县城西南二十里许有李时珍墓园（明代墓），今为李时珍纪念地。

## 引文

> 蕲州，蕲春郡。中。隋开皇九年置蕲州，大业三年改为蕲春郡。武德四年复为蕲州，天宝元年改为蕲春郡，乾元元年复为蕲州。

——《旧唐书·卷四十·地理志三·淮南道·蕲州（蕲春郡）》（节录）
