# 琼州

## 一句话

岭南道琼州，742-758 年间名"琼山郡"——治琼山县（今海南海口琼山区），盛唐海南岛五州之北州；岛中五指山远望，下州户口寥落，曾一度失于黎獠。

## 形貌

琼州城在海南岛北端、琼州海峡南岸约一二十里内陆的一片冲积平原上。

海南岛是大唐辖境最南的一块大岛——南北约八百里、东西约六百里，岛中一脉黎母山（今五指山）自南向北横亘，主峰拔地近一千九百米，盛唐人自岛北琼州城南望，晴日可见远山一线。岛北平原是全岛唯一较开阔的一片汉人耕地——琼州城就立在这片平地中央，城北过琼州海峡（唐人或称"南海"一段）可达雷州半岛之雷州、廉州，是岛与岭南本土之间的唯一水路。城南、城东、城西三面，是黎獠（俚人）盘踞的丘陵山林——盛唐人在此立"州"，**辖境实际只及岛北平原一隅**。

物候是岛屿热带的：冬不见霜、夏极热而长，五月至十月一段台风、雨水极盛——岛北平原春夏稻可两熟。城外水道纵横入海——南渡江（唐时未必此名）自岛中黎母山北麓出，绕琼州城东而北入海。瘴疠之重，仅次于安南。

## 大唐天宝十四载（755年）

琼州在唐人眼里是**海南岛上五州中的北州**——岭南道之尽南、过海之地。

**汉武帝元封元年（前 110）**伏波将军路博德平南越后，于海南岛置**珠崖、儋耳二郡**——这是中原王朝首次正式经略海南。但岛上俚人反复，汉元帝初元三年（前 46）罢珠崖郡——海南遂脱中原直辖凡六百余年。至**隋大业六年（610）**复置珠崖、儋耳、临振三郡——海南再入中原。唐武德五年（622）于汉珠崖郡故地置**崖州**，治舍城县（今海口东南）。

**贞观五年（631）**——析崖州之琼山地置**琼州**，初领**琼山、万安**二县（《旧唐书·地理志》："分崖州之琼山地置琼州，领琼山、万安二县"）；州治琼山县——"琼"字取琼山附近石山形如"琼台"之意（一说取琼山县名而州名同）。这是琼州建置之始。**显庆五年（660）**寻复置琼州，领县扩为**琼山、容琼、曾口、乐会、颜罗**五县（《旧唐书·地理志》："显庆五年寻复置琼州，领琼山、容琼、曾口、乐会、颜罗五县"）——可知唐前期琼州一州建置反复、辖县几经增并。755 天宝年间琼州具体领县，旧唐书地理志载（核源以《旧唐书·卷四一·地理志四·岭南道·琼州（琼山郡）》一条为准）。自此海南岛上唐前期渐成**琼、崖、振、儋、万安**五州格局：**琼州**（治琼山，今海口琼山区）居岛北；**崖州**（治舍城，亦在岛北，琼州东）；**儋州**（治义伦，今海南儋州市，岛西北）；**振州**（治宁远，今海南三亚，岛南）；**万安州**（治万安，今海南万宁，岛东南）——五州环岛而列，州治皆在沿海平地，岛中黎母山一片为羁縻或无州县之地。

**天宝元年（742）**玄宗诏天下州一律改郡，琼州改称**琼山郡**——同年崖州改珠崖郡、儋州改昌化郡、振州改延德郡、万安州改万安郡。**乾元元年（758）**肃宗复改郡为州，琼山郡复称琼州。755 当年城门匾额下写的是"琼山郡"，岛人多半还叫"琼州"。

755 年的琼州属**岭南道**——岭南五府经略治广州，琼州、崖州、儋州、振州、万安州五州皆属之；岭南节度使要到乾元元年（758）才置。州治琼山县。**户口据《旧唐书·地理志》载——琼州户六百四十九**（口数旧唐书未明载；《元和郡县图志》另立元和户口径，与天宝户不可径混）——下州，比扬州一坊不到，比同岛之崖州（约三千户）、振州亦少。盛唐海南五州皆下州，户口寥落——岛上黎獠（俚人）部落不入正赋，朝廷所能直辖者仅沿海平地数县。

**但琼州一州唐前期并不安稳**——《旧唐书·高宗本纪》、《资治通鉴》载，**总章二年至咸亨（669-674）间黎獠（俚人）反，琼州一度失守**——唐廷遣岭南诸军讨平，州治曾迁，后复归。**贞元五年（789）**岭南节度使李复奏请重置琼州——可知中晚唐间琼州亦曾废而复立。755 年这一刻，琼州虽在册，但州治实际控制范围仅琼山平原一隅，城外即"生黎"之境。

**鉴真和尚**与海南岛的关联在**第五次东渡**：天宝二载（743）至天宝七载（748）间鉴真前四次东渡日本皆失败；**天宝七载（748）冬第五次自扬州启航**，海上遇风漂流——**天宝七载至九载（748-750）**漂至海南**振州**（今三亚），州刺史冯崇债迎入官舍，留振州约一年，再北上经万安、崖州、雷州、岭南、循江归扬州——一路弟子病死、目疾加剧，**鉴真至是双目失明**。**鉴真未入琼州城**——自振州北归循岛东海岸经万安、崖州过海至雷州，未取道琼州；但琼州人若知有此事，当知岛南振州亦是大唐之州，海南五州一体。**第六次东渡终成行**：天宝十二载（753）冬自扬州登舟，天宝十三载（754）春抵奈良东大寺——755 年时鉴真已驻奈良一年余（与[交州](./交州.md)、[广州](./广州.md) doc 时序统一）。

天宝十四载冬安禄山反范阳——消息沿驿道自长安南传，过商於、襄州、江陵、潭州、桂州、广州——再过海，**抵琼州大约需五十余日**。岭南五府经略接到的诏令大约是戒严、备粮、防黎獠乘隙；但海南五州距河朔战场过远，朝廷未抽调岛上兵北上。岛州官民这一冬听到的消息已是隔年的事，安史之乱期间海南五州名义上奉唐正朔，实际自管自的——这是 755 至 763 年间盛唐边陲岛州的普遍处境。

## 历史长河

**贞元七年（791）**——岭南节度使李复奏请，割崖州之**临机县（即临高）**来属琼州——琼州自此始有临高。可知唐 755 这一年琼州治下尚无临高县，临高入琼州已是后话。中晚唐黎獠反复——咸通五年（864）岭南节度高骈讨南诏前，岛上诸州又一度告急。唐亡后五代琼州先后属南汉、北宋。**北宋熙宁六年（1073）**废琼州为琼管安抚司，统岛上诸州；**元至元十五年（1278）**置**琼州路安抚司**——海南五州格局至元仍存。**北宋绍圣四年至元符三年（1097-1100）苏轼贬儋州**——苏轼自惠州再贬，**渡琼州海峡至琼州登岸，再西行至儋州**——居儋三年，作《儋耳山》《纵笔三首》等；**元符三年（1100）北归途中过琼州、渡海至雷州**——苏轼《澄迈驿通潮阁》"杳杳天低鹘没处，青山一发是中原"即作于琼州澄迈驿（今海南澄迈）渡海北归之际。盛唐 755 这一年距苏轼渡海北归尚有 345 年——但 1100 那一渡，是中国文学史上海南岛最有名的一次出场。**明洪武三年（1370）**改琼州路为**琼州府**——明清两朝海南由府辖琼山、澄迈、临高、定安、文昌、会同、乐会、儋、万、崖、感恩、昌化、陵水十三县（州）；琼州府治琼山县，府署一直在琼山——这是从唐贞观五年（631）至清末**琼山为岛北州治、府治凡 1280 年**的延续。

民国废府存县，琼山县仍存；1950 年海南岛解放，1988 年海南建省、琼山县后并入海口市，今为海口市琼山区——盛唐那座琼州城遗址在今琼山区府城镇一带，府城至今犹存明清城墙残段——但唐琼州州署的夯土，考古至今未到那一层。岛中五指山，盛唐人遥望"黎母山"的那一脉，今为海南五指山市与黎族苗族聚居自治区——黎人，盛唐"黎獠"之裔，至今仍居岛中山林。

## 引文

《元和郡县图志·卷三八·琼州》：

> 琼州，琼山。下。开元户六百四十八。乡七。元和户……州城南有黎母山。

——李吉甫所录琼州一笔：户六百余、乡仅七、城南即黎母山。下州、岛州、边州——三字而已。
