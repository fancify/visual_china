# 兰州

## 一句话

陇右道兰州，742-757 年间名"金城郡"——黄河北贴城外约一里、城在河南阶地上的州治；东出陇山入秦，西去乌鞘岭入凉州，是唐人由关中赴河西的中段一站。

## 形貌

兰州城在黄河南岸的一片狭长河谷里。

南北两山合抱——南有皋兰山，北有白塔山（唐时未有此名，山形依旧），中间夹着黄河一条。河谷东西长不过数十里，南北宽处仅二三里——城就摊在水南这点窄地上。州治在五泉县（今兰州市区），城东南五泉山是唐时已有的名山——五眼泉水自山间涌出，灌城外的菜畦与官苑。城北过河，便是河西走廊东端的入口；城南越皋兰，是陇右内地，再往南是青藏高原的边缘。**陇右道与河西节度沿线，兰州是唯一以黄河贴城而过的州治**——黄河北贴城外约一里，城在黄河南岸阶地上；再上溯，便没有可称城的地方；再下行，至灵州、丰州才又见黄河边的大城。

兰州城周不大，户口也不多——元和郡县图志载兰州开元间户约**四千余**、口二万出头，至天宝约 **2889 户、14226 口**（《旧唐书·地理志》）——下州 / 中州门槛之下。比起凉州七里十万家、长安百万口，兰州只是过路的中等城。但位置紧——东出陇山入关中、西过乌鞘入河西、北渡黄河接灵武、南越皋兰入陇右诸州，四向辐辏都在这一城之转。物候是河谷的：春旱、夏热而短、秋早霜、冬干寒。城外杏花三月开，比关中略迟；冬季黄河有封冻段（约 11 月至次年 2 月，且非每年都封冻完整）。

## 大唐天宝十四载（755年）

兰州在唐人眼里是一座**已经做了八百年城**的西陲过路所。

秦时此地属陇西郡榆中县——榆中是兰州最早的县名。汉武帝元狩二年（前 121）霍去病破匈奴，**始置金城县**——"金城"二字，《汉书·地理志》注引应劭说"初筑城得金，故曰金城"，颜师古则以为"取其坚固如金"之意——两说并存。西晋初分凉州置**金城郡**，治此。隋开皇三年（583）废金城郡，**置兰州**——"兰州"之名自此立。唐武德二年（619）复置兰州；天宝元年（742）从全国诸州改郡之例，改作**金城郡**；乾元元年（758）再复兰州——**742 至 758 这十七年间，兰州官号是"金城郡"**，史志、碑刻、文书亦同。

755 年的兰州属**陇右道**，州治在五泉县。陇右道治在鄯州（今青海乐都），陇右节度使亦治鄯州——天宝十二载（753）哥舒翰兼河西、陇右二节度，节度使府在鄯州，兰州只是辖下中州。但兰州军事上紧贴河西节度——黄河北岸的渡口、烽燧、屯田，多与河西的赤水军、白亭军一线呼应。**行政归陇右，军事接河西**——这是兰州在 755 年的双重身份。城内有金城郡衙、五泉县治、驿馆——唐两京至凉州的官驿大道（一称"陇右驿道"）自长安西出，过陇山、秦州、渭州，再过兰州，便至乌鞘岭——兰州是这条驿道上**关中至河西的中段最大节点**。

天宝十四载冬安禄山反范阳——消息沿驿道传到兰州时，城里粮草、驿马的调度也紧起来。但兰州不像凉州、鄯州那样有节度使府兵，只是过境之地——河西、陇右精兵东去勤王时，多自兰州过河南下，经秦州、陇州入关中。兰州人这一冬目送过的，是后来再也没回来的一批批河西陇右兵。

## 历史长河

广德元年（763）吐蕃北上，先取河陇——兰州陷蕃，与凉州、鄯州、河西诸州一并入吐蕃**约八十年**。大中五年（851）张议潮收复河陇，兰州归唐，归义军辖。唐亡后兰州先后为吐蕃六谷部、西夏所据——西夏据兰州近百年，于此立兰州监军司。元置兰州属巩昌路。明洪武年间复设兰州，属陕西布政司临洮府——明清两朝逐渐成为甘肃布政司的咽喉。清乾隆三年（1738）设**兰州府**，自此兰州由县级跃为府级。民国废府存县；1941 年设兰州市。今日兰州为甘肃省会，人口已逾四百万——黄河北侧绕城北流过——直至明清始有渐次跨河展拓，盛唐时五泉山下涌出的那五眼泉，至今仍在城南皋兰山麓。

## 引文

《元和郡县图志·卷三十九·兰州》：

> 兰州，金城。下都督府。开元户四千四百二十六。乡一十六。元和户二百八十六。……州城西北有皋兰山，黄河在州北一里。

——这是唐人对兰州城形最简的一笔记述：山在西北、河在城北一里。
