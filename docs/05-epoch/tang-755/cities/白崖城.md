# 白崖城

## 一句话

南诏王族蒙氏起家的故里——洱海以南、弥渡盆地东侧的一座石崖城堡，蒙舍诏（六诏中最南一支）的早期根据地；755 时虽不再是政令中心（国都已迁太和城十六年），却仍是大蒙国王廷礼制上的"祖庭"，**六诏归一后唯一保留蒙舍诏旧名的故地**。

## 形貌

白崖城不在湖滨，而在山口。

城址在今云南大理白族自治州弥渡县红崖一带——洱海以南约六十里，苍山余脉东南向延出的一道石崖之上。崖体灰白临谷而立，"白崖"之名即取此色。崖下是**弥渡盆地**（古称"白崖川"），南北长约二十里、东西宽五六里，是苍洱地区南面最大的一片熟地；毗雌江（今弥苴河上源支流之一）自南向北流经盆地，再北汇入洱海。城基依崖而筑，城墙以**石块垒砌为主、辅以夯土**——与太和城纯夯土的山城做法不同，白崖城更近一座小型石崖堡垒，周不过一二里许。

城外的盆地是蒙舍诏祖业的核心：稻、麦、麻、豆皆熟，畜马牛羊。盆地西南口出去便是**巍山**——蒙舍诏更早的发祥地（细奴逻、逻盛炎、盛逻皮、皮逻阁四代王早年皆居"蒙舍川"），白崖城则是蒙舍诏向北扩张、跨入洱海盆地之前的前哨与据点。城内建筑多木构草顶，仅王族宅院与祭祀台基用瓦——是一座典型 7-8 世纪西南山国的小型石崖城。

## 大唐天宝十四载（755年）

白崖城在唐人眼里是"**六诏旧地**"之一。"诏"在乌蛮、白蛮语中为"王"之意；7 世纪末洱海地区有六个相互独立的小王国——蒙嶲诏（漾濞）、越析诏（宾川）、浪穹诏（洱源）、邆赕诏（邓川）、施浪诏（洱源东），最南一支即**蒙舍诏**（巍山 / 弥渡）——因居最南故又**单称"南诏"**，这是"南诏"二字最初的来历，**不指国号，只指方位**。白崖城便是蒙舍诏在弥渡盆地的核心据点，与南面更靠山的**巍山垅圩图山城**（细奴逻一代王城旧址）一同构成蒙舍诏的早期双核心。

开元二十六年（738），蒙舍诏王皮逻阁借唐廷之力**并六诏为一**——北灭蒙嶲、越析，再北并浪穹、邆赕、施浪。当年正月唐玄宗即册皮逻阁为"云南王"，赐名蒙归义。**开元二十七年（739）皮逻阁徙都太和城**——白崖城自此不再是政令中心，降为大蒙国祖庭与南面军镇。天宝七载（748）皮逻阁卒，子**阁罗凤**继位；再过三年便是 751 西洱河第一战、754 西洱河第二战——这是阁罗凤一生的转折（详见 [太和城](./太和城.md)）。

755 时阁罗凤在位已七年。天宝九载（750）攻陷姚州、杀云南太守张虔陀；天宝十载（751）鲜于仲通率八万唐军伐之，大败于西洱河；天宝十一载（752）阁罗凤投吐蕃，受赞普赤德祖赞册"**赞普钟**"（义弟）号"东帝"——大蒙国已由"唐册云南王"翻为"吐蕃赞普弟"；天宝十三载（754）李宓再伐，复败殁阵。这一新身份并不抹去蒙氏自巍山、白崖一路打上来的旧账——**南诏德化碑**（766 立）便从蒙舍诏起家、六诏归一一路写下。755 这一刻白崖城的实际功能有三：

其一，**祖庭与祭祀地**。蒙舍诏自细奴逻（约 7 世纪中）起家至皮逻阁共五代——前四代王的祖墓与祭祀台基集中在巍山与白崖一带，阁罗凤每岁仍亲往致祭。

其二，**大蒙国南面军镇**。白崖城向东南越云南驿便接姚州故地——天宝九载（750）阁罗凤攻陷姚州、杀云南太守张虔陀，正是自白崖、云南驿一线北出的；次年（751）鲜于仲通八万唐军南下伐之，大败于西洱河，又是从姚州、云南驿一线进退。755 时姚州都督府已废，南面再无唐军压力，戍兵已大半移至太和城与北面浪穹一线（防吐蕃方向）。

其三，**六诏故地的礼制锚点**。皮逻阁并六诏后，对其他五诏旧地多以毁城、徙民、改属为手段——五诏之"诏"号皆废，**唯蒙舍诏自家的"白崖"二字与城仍在**——这是六诏归一后唯一一处仍以蒙舍诏旧称示人的地名。对阁罗凤而言，留下白崖城便是留下"我们从这里来"的一处实物。

至于远在河朔的安禄山起兵——十一月初九的消息辗转入云南，要到次年才到太和，至白崖城更晚。对白崖城的人而言，755 年的冬至和往年并无不同：苍山余脉的雪线推到崖顶，盆地里收完晚稻，崖下毗雌江水落，城内王族祭台前的香火不绝。

## 历史长河

**中晚唐**　白崖城自 739 太和立都以下，作为大蒙国祖庭延续。贞元十年（794）异牟寻与唐使袁滋苍山神祠会盟，南诏与唐复和——白崖城地位未变。咸通十年（869）南诏世隆攻入成都府，白崖城作为后方补给地之一城仍在用。

**唐末五代**　唐昭宗天复二年（902）权臣郑买嗣篡蒙氏立"大长和国"——蒙氏 164 年王统终结，白崖城作为蒙氏祖庭的礼制意义随之中止，转为地方军镇。

**宋元**　后晋天福二年（937）段思平灭大义宁建大理国（937-1253），都阳苴咩城——白崖城归大理国白崖赕（"赕"为大理国一级行政单位，相当于州）。元宪宗三年（1253）忽必烈灭大理，元改白崖赕为白崖甸，属大理路。

**明清**　明洪武十五年（1382）平云南，改白崖甸为**红崖驿**——"白崖"之名自此被"红崖"取代（明人取崖色再观谓崖体经风化呈红，遂改名）。**"红崖"是明改之名，本 doc 以"白崖"为 755 时态正名，不以"红崖"预支盛唐**。清雍正七年（1729）置赵州，红崖驿属之；清末民初置弥渡县，红崖归属弥渡。

**民国至今**　今属云南省大理白族自治州弥渡县。白崖城遗址在弥渡县红崖一带——城墙石基零星可辨，崖体仍立，盆地仍熟。1990s-2010s 间云南省文物考古研究所多次踏查——石垒墙基、王族宅院夯土台、祭祀台等遗迹陆续清理；今为云南省级文物保护单位。蒙舍诏起家的崖体仍在弥渡盆地之东——盛唐 755 时阁罗凤亲祭的祭台基址，今地表上残存几方夯土。

## 引文

> 蛮谓王为诏。先有六诏：蒙巂诏 / 越析诏 / 浪穹诏 / 邆赕诏 / 施浪诏 / 蒙舍诏。蒙舍诏在诸部之南，故称南诏也。

——《新唐书·卷二二二上·南蛮上·南诏传》

> 白崖城在云南之南，距太和六十里。皮逻阁起此而王，至阁罗凤迁太和后，城犹存。

——樊绰《云南志》（《蛮书》，咸通三年 / 862 著）卷六"云南城镇"。按：樊绰书成于盛唐 755 之后约百年，本条引文作中唐 retrospective 用，755 这一刻此书尚未成。
