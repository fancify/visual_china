# 许州

## 一句话

河南道颍水中游一上州——治长社县（今河南许昌市），742-758 年间官名"颍川郡"；城地即东汉末年献帝许都旧址，曹操在此屯田二十四年。

## 大唐天宝十四载（755年）

许州治**长社县**，属**河南道**。城在颍水中下游北岸——西北约一百八十里至洛阳，东北约一百五十里至汴州，东南接陈州、蔡州。沿革：**隋开皇九年（589）置许州**，大业三年（607）改**颍川郡**；**唐武德四年（621）复许州**；**天宝元年（742）改颍川郡**——是为 755 年当年的官称；**乾元元年（758）复许州**。

户口上，**《旧唐书·地理志》载许州（颍川郡）为上州，天宝领县七，户七万三千二百四十七，口四十八万七千八百六十四**——颍水两岸是河南道中部老熟麦区，户口在洛阳、汴州、宋州一线之下，亦不下于陈州、蔡州。**旧唐书天宝许州七县**为**长社（州治）、长葛、鄢陵、扶沟、临颍、舞阳、许昌**（按：唐许昌为属县名，非州城；州治在长社）——**注**：**阳翟县唐属许州或洛州异说并存**——《旧唐书·地理志一·河南道》许州本传所列七县中**不含阳翟**（旧唐书阳翟入洛州/河南府一系），本 doc 主从旧唐书口径，**阳翟不入许州七县**；《新唐书·地理志》许州颍川郡作领县九、户七万三千三百四十七（数字与旧唐书略异，新旧异说并存）。

许州地最堪一提的，是**东汉末年的许都旧迹**。**建安元年（196）九月**曹操迎汉献帝自洛阳东迁，"车驾入许"——此即"许下"、"许都"之始（《三国志·魏书·武帝纪》、《后汉书·献帝纪》）；操于许下置屯田——枣祗、韩浩典之，募流民屯田许下，"得谷百万斛"，这是建安屯田之始，亦是曹魏开国底子。汉献帝在许凡二十五年——至**延康元年（220）十月**曹丕受禅、迁都洛阳，许都之名才罢。盛唐文人入许州，自城外望，**许下故宫、汉魏屯田旧渠的土垄在田中犹可指认**——已是五百三十余年前旧事。

755 年冬安禄山反范阳，叛军主力南下经河北过黄河、十二月十三日陷洛阳——许州在洛阳东南、汴州西南，正当叛军南扑的腹背一带。次年（756）许州一度陷叛军，至德二载（757）随东京收复。

## 历史长河（755 → 今）

中晚唐许州仍为河南道颍水一线上州；五代后周显德元年（954）周世宗柴荣自此发迹——周世宗微时曾为澶州、许州一带商旅，后周复都开封。**北宋升许州为颍昌府**（元丰三年 / 1080）——为京西北路重镇之一。金改许州；元属汴梁路；明清复**许州**——清雍正二年（1724）升许州直隶州，十二年（1734）复为府，治今许昌市。**1913 年废许州，改为许昌县**——"许昌"二字始为城名（按：唐许昌仅为许州属县名，非州城名；许昌之为城名是 1913 年以后事，盛唐 755 时不可用作主名）；1947 年置许昌市。今许昌市辖区大致承明清许州旧城，市区文峰塔、春秋楼诸景为元明以后旧迹；汉魏许都故城遗址在今许昌市东张潘镇一带——已为田野遗址，地面所见的许昌是一座明清以下的城。

## 引文

> 许州，颍川郡。上。隋开皇九年置许州，大业三年改为颍川郡。武德四年复为许州，天宝元年改为颍川郡。乾元元年复为许州。……天宝领县七，户七万三千二百四十七，口四十八万七千八百六十四。

——《旧唐书·地理志一·河南道·许州（颍川郡）》（节录）

> 建安元年……秋七月……天子拜太祖（曹操）大将军，封武平侯。……九月，车驾出轘辕而东，以太祖为大将军，封武平侯。自天子之西迁，朝廷日乱，至是宗庙社稷制度始立。……是岁用枣祗、韩浩等议，始兴屯田。

——《三国志·魏书一·武帝纪》建安元年（节录；汉献帝东迁入许、许下屯田之始）
