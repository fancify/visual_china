# 泸州

## 一句话

剑南道南端长江北岸一下州都督府——治泸川县（今四川泸州老城江阳一带），长江与沱江于城下交汇，南通戎州、姚州与南诏之路在此换舟，州境与黔中道相接、内多"夷獠"羁縻。

## 大唐天宝十四载（755年）

泸州治**泸川县**，在**长江北岸、沱江入长江口**——大江自戎州顺流而下至此，沱水自北汇入，两江合襟为剑南道南端一津要。州属**剑南道南端**，**天宝元年（742）随玄宗朝"改州为郡"通制改为泸川郡**，755 时正名为"泸州泸川郡"，两名在唐人嘴里互换不分；**乾元元年（758）肃宗复改为泸州**——但这是 755 之后的事。沿革更早：**隋置泸川郡**；**唐武德元年（618）改泸川郡为泸州**；高宗、武后间于州境又陆续析置**晏州、纳州、奉州、浙州、巩州、薛州**诸羁縻州，皆隶泸州都督府节度，是唐廷于剑南南端经略"夷獠"的前沿。《旧唐书·卷四一·地理四》泸州列为**下州**、兼**都督府**——州等不高而事权颇重。**《旧唐书·卷四一·地理四·剑南道·泸州泸川郡》载：**"旧领县六，户一万九千一百一十六，口六万六千八百二十八。**天宝，户一万六千五百九十四，口六万五千七百一十一**"——755 时泸州在册**户 16,594、口 65,711**。"旧领县六"乃武德初籍 (富世、江安、绵水、合江、来凤、和义) 之初唐 snapshot；**来凤县已于武德九年（626）省并入富义县**（《新唐书·卷四二·地理志》富义县下："本富世，武德九年省来凤县入焉"，《旧唐书》亦载"武德九年省来凤"），故 755 时**实领五县：泸川、富义（旧富世）、江安、合江、绵水**——其余高宗、武后间所析晏、纳、奉、浙、巩、薛诸州皆为羁縻州，不入泸州本州县册。户口较武德旧籍 (户 19,116、口 66,828) 略减——剑南南端开发本浅，籍内编户素少，而州境羁縻夷獠之口不入此册。

州境是**隋唐间夷獠杂居地**——长江南岸即入"南中"，循江西上至戎州可换陆路南趋姚州、入南诏（大蒙国），是唐廷与南诏交通的一条偏径；沱江上溯则可通梓州、益州。755 时泸州无大兵无大事——以**都督府**身份统泸州本州及晏州、纳州、蔺州等羁縻州，是剑南道南端控扼"夷獠"与南通要津的一只手。唐人提泸州必带江沱合襟与羁縻边州这两笔。

## 历史长河

**中唐至五代**　安史乱后剑南道分东西，泸州归剑南东川。前蜀、后蜀据巴蜀，泸州仍为剑南南端都督府重地。

**宋至明清**　北宋初置**泸州**属梓州路，后改潼川府路；徽宗朝升泸州军。**南宋抗蒙泸州合江神臂城据守 36 年（1243-1277）**——余玠筑山城于合江县长江南岸石壁，凭险拒蒙古铁骑，至元至元十四年（1277）始陷，是宋季蜀地最坚一节，去 755 已逾五百年。明洪武改泸州军民府，旋复为泸州，属四川布政司；清属四川省叙永厅、永宁道，泸州仍为州治。

**民国至今**　1913 年废州存县称**泸县**属四川省。1949 后析泸县城区设泸州市，今为四川省辖地级市，州境老城江阳区一带与唐泸川县治大体相承。州内出**泸州老窖**白酒——城南营沟头明窖池连烧近四百年，是今泸州最盛之名，已是民国之后事。

## 引文

> 泸州泸川郡，下，都督府。隋泸川郡。武德元年，改为泸州。……旧领县六，户一万九千一百一十六，口六万六千八百二十八。天宝，户一万六千五百九十四，口六万五千七百一十一。

——《旧唐书·卷四一·志第二十一·地理四·剑南道·泸州泸川郡》
