# 朔州

## 一句话

河东道北翼下州，治善阳县（今山西朔州市）——雁门关北六十唐里，代州正北越关第一州；秦汉**马邑**故地，唐河东节度北防边镇。

## 形貌

朔州城在雁门关以北、桑干河上游南侧的山间盆地里。南倚勾注山（雁门关所在）北麓，北望阴山余脉与漠南草地——朔州正夹在两者之间，是河东出塞之后第一片可以筑城屯田的川地。州治**善阳县**——城周不大，城外是桑干河上游一片冲积平畈，再远便是黄土塬与草滩相接。城南越雁门关六十唐里至代州，城北过马邑故塞便接单于府羁縻诸州——朔州是雁门关外的第一站。物候是塞下的：春迟而风沙重、夏短、九月即霜、冬严而长——城外山脚十月已可见羊群，盛唐朔州境内耕牧并行。

## 大唐天宝十四载（755年）

朔州在唐人眼里是一座**秦汉马邑故地**之州。**秦汉马邑县**——汉武帝元光二年（前 133）**王恢马邑之谋**于此设伏击匈奴单于，事败而开汉匈百年大战之端（《史记·韩长孺传》）——**马邑**之名自此入战国秦汉史册。**隋开皇五年（585）始置朔州**，大业三年（607）改**马邑郡**；**唐武德四年（621）复朔州**；**天宝元年（742）**玄宗诏改州为郡，朔州改作**马邑郡**；**乾元元年（758）**肃宗复州——**742 至 758 这十七年间，朔州官称为"马邑郡"**，民间口语仍呼"朔州"或径呼"马邑"。

755 年的朔州属**河东道**，州治在善阳县。《旧唐书·地理志二》载朔州**马邑郡天宝户五千四百九十三、口二万四千五百三十三**——**下州**门槛，是河东北翼边州。**雁门关南为代州、关北为朔州**——两州合护河东北门；朔州属河东节度（治太原）辖境，州境内常驻戍兵、烽铺若干，是雁门关外的前哨。**742-758 改郡期内本州长官官称为"马邑郡太守"**（注：唐州本称刺史，唯此期短改太守）。天宝十四载冬安禄山反范阳，消息过雁门关传至朔州约旬日；朔州兵不多，但作关外戍守第一线，烽铺与戍堡的点检最忙。

## 历史长河

中晚唐朔州做沙陀部李克用代北起家之地之一（与代州、云州同线）；五代后唐起家于此。**后晋天福元年（936）石敬瑭割燕云十六州予契丹，朔州为十六州之一**——自此朔州入辽境三百年，至北宋始终未能收复。金元仍为朔州。明洪武年间复设朔州，属山西大同府。清沿明制。民国废州存县——"朔县"为山西省辖；今日山西朔州市治所，仍在唐善阳县旧址一线。

## 引文

《旧唐书·卷三九·地理志二·河东道·朔州》：

> 朔州，马邑。下。天宝领县二，户五千四百九十三，口二万四千五百三十三。

——这是唐人对盛唐朔州户口最简的一笔记述：下州、领县二、户五千余。
