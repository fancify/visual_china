# 莫州

## 一句话

河北道莫州，**742-758 年间名"文安郡"**——河北平原中段、大清河上游近白洋淀一片低洼地的一处上州，治莫县（今河北沧州市任丘市鄚州镇）；**州名"莫"乃开元十三年（725）由"鄚"改字而来**，前后只差一个偏旁。

## 大唐天宝十四载（755年）

莫州是河北平原中段一处不大的州——治莫县。沿革：**唐景云二年（711）**睿宗析瀛州莫县置**鄚州**——州名取治县"鄚"字（"鄚"字本汉以来县名旧字）；**开元十三年（725）**玄宗诏改**"鄚"为"莫"**——县名、州名一并改字，仅减一偏旁，自此本州、本县皆作"莫"。**天宝元年（742）**改**文安郡**——以郡境内文安县（汉以来旧县）取名；**乾元元年（758）**肃宗复莫州。755 这一刻匾上写"文安郡"，太守一员领郡事。《旧唐书·地理志》记**上州**——天宝**户 53,493 / 口 339,972**（《旧唐书》《新唐书》地理志同），户口在河北道诸州中游偏上一档，不及幽、定、魏那一线大州，但已入上州之列。

郡境在河北平原中段——东接瀛州（河间郡）、北接易州（上谷郡）、西北接涿州一线、南接沧州（景城郡）。**大清河上游**自西山东流入境——州境多低洼湖泊，今白洋淀一带湖泽即此一区——本州水陆兼有：陆道循永济渠东岸南北通幽、沧，水道循大清河自西山下平原。755 这一冬安禄山自范阳起兵——叛军主力南下经河北腹地（恒、定、赵、魏一线）西攻洛阳，**莫州在范阳本据正南二百里许，叛军循驿道南下时本州当其侧翼**，州城调度紧起来——但本州非叛军主力直冲之地，安史乱中（755-763）一线反复，**763 年史朝义自杀、安史方平**，本州渐稳。

## 历史长河

**中晚唐**　乾元元年（758）肃宗复改文安郡为莫州。安史平（763）后属义武军一线——本州地小，不为节度治。**宋元明清**　**北宋初仍置莫州**于此——治莫县；属河北东路。**金升莫州**为防御州一档；**元后并入河间府**——莫州建制遂废，仅余莫县（后改任丘县）——州县合并后"莫州"二字便不再作州名。明清属河间府任丘县。**民国至今**　1913 年废府存县——任丘县属河北省；**1986 年任丘撤县设县级市**——今**河北省沧州市任丘市**即唐莫州一段旧境。莫州故城在今任丘市东北约二十里**鄚州镇**——盛唐"鄚州 / 莫州"地名延续至今、镇名仍写**旧字"鄚"**（自开元十三年改"莫"后，州县皆改作"莫"，唯镇名乡名屡有沿用旧字"鄚"者，今鄚州镇地名即此一线遗存）。

## 引文

> 莫州，文安郡。上。景云二年，析瀛州之莫县置鄚州，治莫县。开元十三年，改"鄚"为"莫"。天宝元年，改为文安郡。乾元元年，复为莫州。旧领县三，户五万三千四百九十三，口三十三万九千九百七十二。

——《旧唐书·卷三十九·地理志二·河北道·莫州》大意（节录非全文；按本州州名"鄚→莫"改字与"莫州→文安郡→莫州"郡州反复一段最直接的史源）
