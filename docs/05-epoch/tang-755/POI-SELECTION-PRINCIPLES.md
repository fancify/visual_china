## 1. 总体筛选哲学

**收什么**：唐 755 时**确实存在且对当时人有意义**的地理 / 文化实体，OR 当时已废但**唐人在其上凭吊 / 投以诗文**的遗迹。

**不收**：755 后才出现的实体；755 时人不知道、不凭吊、不入诗文的地方；纯现代景点；近现代旅游打包概念（"四大名楼" 等）。

---

## 1.5 规则优先级层次（**2026-05-14 codex r1 加**）

当多条规则同时适用时，按以下优先级：

```
T1-T3 时态硬约束  >  R3 文化独立白名单  >  R2 城市主轴绑定  >  R1 子条目去重  >  R4 补段
```

**示例**：
- 莫高窟在沙州境，沙州.md 多段提 → 表面看 R1 应归城市；但 **R3 "莫高窟原则"压倒 R1** → 独立
- 滕王阁在洪州境，洪州.md 24 次提 → 表面 R1 + R2 都倾向归城市；但 **R3 文学符号脱离城市**（王勃 675 序）→ 独立
- 岳阳楼在岳州境，岳州.md 20 次提 → **R2 "符号绑定整体" 压倒 R3**（岳阳楼仍紧绑洞庭-巴陵）→ 归城市
- 兰亭在越州境，越州.md `### 兰亭` 已展开 → R3 不强（书法符号但 spec 未列白名单）→ **R1 应用** → 归城市

## 2. 去重规则（基于 spec "避免双重出现"）

### R1 子条目原则
地标若已在某城市 doc 内做了 `###` 独立段落（不是 frontmatter / tags / brief mention），不再单独立 POI。

| ✅ 应用例 | ❌ 反例 |
|---|---|
| 兰亭已在 越州.md `### 兰亭` 大段 → 移除独立 | 鹳雀楼在 河中府.md 仅 frontmatter tag，未独立段 → 保留独立 |
| 武侯祠 在 益州.md `### 武侯祠` → 移除独立 | 雁门关在 代州.md provenance + frontmatter，但未 `###` 独立段 → 保留独立 |
| 大慈恩寺 在 长安.md 多段展开 → 不再立 | 同左 |

### R2 主轴绑定原则
若 POI 是城市灵魂（"楼即城的别名"），即使可独立，归城市 doc 不另立。

| ✅ 应用例 | ❌ 反例 |
|---|---|
| 岳阳楼 716 张说改建始得名"岳阳楼"，与岳州/巴陵共生 → 归岳州.md | 滕王阁 王勃 675 序已让"滕王阁"成为脱离洪州的文学符号 → 独立 |
| 西湖唐时刚被李泌、白居易疏凿，与杭州一城两水 → 归杭州.md | 黄鹤楼 三国吴 223 建、崔颢 730s 诗已传开 → 独立 |

### R3 莫高窟原则（spec）
文化分量 >> 城市附庸 时，即使在某城市境内，独立成 POI。

| 标杆例 |
|---|
| 莫高窟（沙州境）/ 龙门石窟（洛阳南郊）/ 云冈石窟（云州境）— spec 直接拍板 |
| 推广：滕王阁 / 黄鹤楼 / 鹳雀楼 / 武当山（房州西邻均州境，山名 >> 州名）|

### R4 城市子条目可补不可漏
若某地标按 R1-R3 归城市 doc，**目标城市 doc 未必都已有该 ### 段**——若未有则**补段而非立独立 doc**。

| 例 |
|---|
| 灞桥 / 鸿门 应在 长安.md 补 `### 灞桥` / `### 鸿门` 子条目，不另立 POI |
| 五丈原（属郿县，岐州西部）岐州.md 未提 → 立独立 POI（因 5 五丈原跨"岐州 + 武功" 县级且独立诗境）|

## 3. 增补规则（用户拍板 2026-05-14）

### A1 物理湮没增补（2026-05-14 codex r1 收紧）
唐时重要 + 今已**物理湮没**（或仅余夯土残基）的实体，独立 POI。
**双重叠加**：唐人当年凭吊 + 今人凭吊不到 = 最佳载体。

**应用 6 条**（缩窄；今犹存的归 A4）：
- 水域：汉昆明池（唐末干涸）/ 渼陂湖（已缩为小水塘）
- 离宫遗：九成宫（仅碑残基）/ 玉华宫（山林遗址）/ 汉甘泉宫（残土堆）
- 帝宫：武周天堂（695 焚后再未建；今仅基址）

### A4 唐态断裂但今存（2026-05-14 codex r1 新增）
今天物理犹存，但**唐时格局已散** / **唐时主体已废后世重建** 的，独立 POI。
**核心**："看得见 ≠ 看得到唐"，唐人那种规模 / 格局 / 香火 / 文化分量已散，今所见是后世遗痕。

**应用 6 条**：
- 古寺：灵岩寺（济州长清；今存千佛殿宋塑罗汉；但唐时"四大名刹"格局已散）
- 离宫：武周明堂（开元 25 拆上层后已非原貌）
- 凝碧池（洛阳禁苑；今地名已无；王维诗作于 756 — 正文须严守"755 时尚非亡国符号"）
- 古寺：凉州大云寺（仅塔遗痕）
- 帝陵：黄帝陵（明清以来历代修；唐时仅民间祭、官祀未立）
- 古战场：五丈原（明清重建武侯祠；唐时仅台原荒坡）

### A2 朝代均衡增补
原 inventory 偏 秦/汉/唐 三朝陵；十六国 / 南北朝 230 年 + 三国 / 两晋 + 道教祖庭 / 文学纪念地 都要均衡覆盖。

**应用 16 条**（codex r1 + user 2026-05-14 "多文化纪念古迹"加码）：

**十六国 / 南北朝 11**：
- 陵群 4：北魏邙山 / 北齐磁县 / 北周咸阳原 / 南朝陵墓石刻群
- 都城 3：统万城（大夏）/ 龙城（北燕+营州故）/ 盛乐（北魏前期都）
- 石窟 3：响堂山（北齐）/ 炳灵寺（西秦-唐）/ 天龙山（北朝-唐）
- 边镇 1：武川镇/怀朔镇（唐皇族祖籍）

**三国 / 两晋 文化纪念 3**：
- **铜雀台 + 邺城三台**（建安文学中心；曹魏宫苑；杜牧《赤壁》"东风不与周郎便"之文化原点）
- **逍遥津**（合肥；张辽威震逍遥津；庐州境）
- **隆中**（襄阳南；诸葛亮隐居 + 隆中对策地；唐人凭吊地）

**唐道教祖庭 / 文学纪念 2**：
- **楼观台**（盩厔；老子说经台；李渊朝立"宗圣观"——唐皇室道教祖庙）
- **昭君墓**（汉昭君葬地；唐诗高频；杜甫《咏怀古迹》主角）

**综合 1**：
- 历代长城（秦/汉/北齐/隋/唐代边墙 五子段）

### A3 周边文明 + 都护边缘文化覆盖（2026-05-14 codex r1 改名）
**bounds 内** 的吐蕃 / 西域 / 南诏 / 新罗 / 突厥-回纥 / 渤海 古迹要有。
> 西域 4 条实属唐安西大都护辖境，原 A3 名"疆域外"不准确——改为"周边文明 + 都护边缘"，强调"非汉唐核心文化"。

**应用 11 条**（扩；codex 指出原 8 条 5/8 偏佛教 + 突厥/渤海缺）：
- 吐蕃 3：大昭寺+红山宫 / 雍布拉康+藏王陵 / **赤岭碑 唐蕃界碑**（rāstra-grub；733 立）
- 西域 4：克孜尔 / 苏巴什 / 楼兰 / 七个星
- 南诏 1：巍山祖庭
- 新罗 2：庆州佛国寺+石窟庵 / **月城+雁鸭池**（新罗景德王朝离宫园林）
- 突厥-回纥 1：**阙特勤碑 + 毗伽可汗碑**（鄂尔浑；731-735 立；755 时回纥代突厥据其地）
- 渤海 1：**敖东故都**（敦化；旧国 698-742 都；文王 742 迁中京、755 后再迁上京——755 时是"刚迁出十年的旧都"）

## 8. 实际操作流程（已执行）

1. 读 POI-INVENTORY.md 自然类原始 165 条
2. 对每条候选执行：
   - 跑 `grep "<候选地标>" cities/*.md` 检查城市 doc 是否已 `###` 展开（R1）
   - 检查 frontmatter / tags / 主轴叙事密度（R2 / R4）
   - 检查 spec 的 莫高窟原则适用性（R3）
3. 按 R1-R4 分流：保留 / 移除 / 补段
4. 按 A1-A3 增补
5. 按 M1-M4 决定合 / 拆
6. 输出 POI-INVENTORY-NATURAL-V2.md

## Codex review 关注点

1. **R1-R4 原则有无内部矛盾 / 漏洞**：是否存在边界 case 无法落到任一规则？
2. **A1-A3 增补的 29 条**：与原 165 条对照，文化分量是否均衡？
3. **朝代覆盖（§7）**：有无朝代严重不足（譬如：三国 / 两晋 仅 5 条，是否过少？）
4. **类别覆盖**：宗教类 / 妇女史 / 矿冶 / 物产 / 园林 / 古墓 等纵向类别是否有遗漏？
5. **判断边界**：岳阳楼归城市、滕王阁/黄鹤楼独立——基于 716 改建时差 + 文学符号成熟度——这条判断 codex 是否赞同？
6. **湮没类筛选的"今状态"标准**：例如"灵岩寺今存千佛殿宋塑罗汉群"——这种"今犹存但唐时格局已散"是否符合 A1 标准？
7. **周边文明 8 条选择**：吐蕃 / 西域 / 南诏 / 新罗 是否还有遗漏？是否过分偏佛教景点（5/8 是寺窟）？
