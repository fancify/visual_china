# 九成宫

## 一句话

麟游山中的隋仁寿宫——唐太宗改名"九成"、魏徵撰文欧阳询书《醴泉铭》立此一碑；高宗朝水灾后渐离、玄宗朝不至——**盛唐人提起"九成"，已是六十年前的旧事**。

## 形貌

九成宫在岐州扶风郡麟游县城东南——南依杜水（杜河）、北倚天台山，是秦岭余脉一片山间盆地。

宫址海拔约 1100 米——比长安城高出 700 余米，夏日凉爽，是隋唐两朝最早确定的"避暑离宫"选址。整宫依山势南北铺开——南端临杜水南岸的台阶式宫门，北端达天台山南麓的后宫高台。**全宫南北长约 1100 米、东西宽约 350 米**，殿宇凡数十座——主要包括南面的**正门 + 朝堂 + 寝宫 + 后园**四大区。

宫名"九成"——《尚书·益稷》"箫韶九成，凤皇来仪"——"九成"言乐曲九变、亦言宫高九层之意。隋人称仁寿宫——唐太宗改名，是要把这座旧隋宫从"杨坚的离宫"重新写成"贞观的离宫"。

宫西约一里有**醴泉**——**贞观六年（632）四月十六日己亥**太宗、长孙皇后游九成宫西，**以杖刺地、流泉涌出**（《九成宫醴泉铭》本文开篇即明文"维贞观六年孟夏之月"）。泉水甘洌——古人以为祥瑞。当月魏徵奉敕撰文，欧阳询奉敕书丹——立**《九成宫醴泉铭》**碑于宫西醴泉之畔。**碑高约二米四，文凡 1108-1109 字**（碑面残蚀致字数略有口径异文）——是初唐楷书"欧体"的最重要范本之一。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

九成宫的来路要从**隋仁寿宫**算起。

**隋开皇十三年（593）二月**——隋文帝杨坚命**杨素**为营宫监、**宇文恺**为副总管，于岐州杜阳县（今麟游县境）始建**仁寿宫**。**杨素施工严酷**——"丁夫无算，死者万计；役者死，倒尸壕堑，以土埋之，营造监修不许问"——这是隋史里的一个污点。**开皇十五年（595）三月**——仁寿宫成，隋文帝避暑于此。

仁寿宫之后**作为隋两朝主要避暑离宫**——文帝晚年凡七次幸此，仁寿四年（604）七月，**文帝崩于仁寿宫**——遗诏在此发出。炀帝继位后改用东都，仁寿宫渐稀至。隋末群雄起，仁寿宫荒废。**唐武德年间**仁寿宫长期废置——城墙坍塌、殿宇朽蠹。

**唐贞观五年（631）正月**——太宗诏修复仁寿宫，**改名九成宫**。这是太宗主政以来第一次大型宫殿工程——但**用的是隋仁寿旧基**，并未大兴土木：旧殿能修则修，不另立新院。**改名"九成"以避开"仁寿"二字**——隋文帝杨坚崩于此宫的旧名，唐太宗不愿沿用。

**贞观六年（632）四月十六日己亥**——太宗与长孙皇后初幸九成宫、得醴泉于宫西。**同年六月**魏徵奉敕撰《九成宫醴泉铭》、欧阳询书丹，立碑于醴泉旁。**自此九成宫与"醴泉铭"绑成一体**——千年后凡练欧楷者，必摹《九成宫醴泉铭》；千年后凡读魏徵者，必读其在九成宫醴泉铭中那段"居安思危、戒奢以俭"。

**贞观、永徽两朝皆为九成宫鼎盛期**。太宗在位间凡五幸九成宫（贞观六、七、八、九、十一年）；高宗即位后**永徽二年（651）九月改九成宫为"万年宫"**（《唐会要》卷三十）——取"万年长存"之意，非镇水。永徽间高宗多次幸万年宫——**永徽五年（654）五月**夜，**山水暴涨自天台山下来、冲玄武门入宫**——《资治通鉴》卷一九九记宿卫多溺死；**《旧唐书·高宗本纪上》记"水溺麟游居民及当番卫士、死者三千余人"**——这是两宫并立后唐宫殿史的最重大一次水灾。乾封二年（667）复名"九成"。

### 此刻状态

天宝十四载这一年的九成宫——**空宫，无主已三十余年**。

玄宗朝避暑首选骊山温泉（华清宫）——这处宫在长安近郊东三十里，朝政可日还。麟游九成宫距长安四百余里、山路盘行不易——**玄宗在位四十四年（712-756），仅 开元二年（714）一次幸九成宫**（《唐会要》卷三十）——之后再未至。

**自开元二年（714）后**九成宫的命运便是逐年荒废——殿宇虽未坍，但岁修日少；朝廷有宫监、宫使典守，但典守职责名义大、实务少（具体官职名待考）。天宝间宫内只有少量寺人、宫人守户——殿宇间多积尘灰。**755 年这一刻的九成宫是空宫**——南门、朝堂、寝宫一线仍立，但厅事无人；后园芳草过膝。**宫西的醴泉铭碑仍立**——一百二十三年前的字迹尚清。麟游县在 755 时为扶风郡岐州属下州属县——州县岁有报呈，但九成宫本身已不与现役朝廷事务挂钩。

**这一年的麟游县境**没有兵事——再过半年安禄山起兵于范阳；再过一年玄宗西奔入蜀，麟游不在西奔路线上（玄宗循扶风、陈仓、入散关，绕九成宫南面而过）；这座空宫在安史中未受兵燹直接破坏。

### 此刻意味

对天宝十四载的麟游人来说，九成宫是**县城东南山中那座空着的隋唐旧宫**——日子如常、宫如常。对长安那一代士人来说，九成宫是**初唐辉煌的一处遗物**——尤其是西门外那一通《九成宫醴泉铭》碑——盛唐人学楷书者，无人不知"皇甫诞、九成宫"的双璧（欧阳询另书《皇甫诞碑》，与《九成宫》并列）。对朝廷来说——九成宫只是一处"先帝旧宫"，名义保留、实务空转。

唐人尚不知道——晚唐间九成宫将渐废、殿宇坼毁（具体废止年份诸史无明文）；再过三百余年，宋元间九成宫遗址全为农田覆盖。**碑则一直立着**——《九成宫醴泉铭》在田间立到清乾隆四十年（1775）才入麟游县学碑亭——千年间风化、剥蚀，已断成数段。

## 子条目

### 九成宫醴泉铭

宫西约一里、醴泉之畔立。**贞观六年（632）六月立**（同年四月十六日己亥得泉、六月立碑——故宫藏品口径）——魏徵撰文、欧阳询书丹。碑通高 246 厘米，碑额"九成宫醴泉铭"六字篆书；碑文凡 1108-1109 字、楷书 24 行、每行 50 字（字数与行字均因碑面残蚀有口径异文）。**文起首"维贞观六年孟夏之月、皇帝避暑乎九成之宫……"**

碑文凡 1108-1109 字（字数因碑面残蚀略有口径异文），核心是**魏徵借醴泉颂太宗、又借颂转谏**——颂太宗"承天景命、抚有万邦"之绩；接转"虽休勿休、居安思危、戒奢以俭"之劝。**这通碑在初唐 632 年立时是政治文献；至盛唐 755 年时已是楷书范本；至宋元后是欧阳询书法的最重要遗品**。

碑现存麟游县博物馆（碑亭内）——千年间露天立于宫西田中，至清乾隆四十年（1775）始入碑亭；又经清末民国数次震动、断成三大段。今碑文凡完好可读者约 800 字、半泐者约 200 字、全失者约百余字。

### 醴泉

宫西约一里——**贞观六年（632）四月十六日己亥**太宗与长孙皇后游宫西、太宗以杖刺地，泉水涌出（《九成宫醴泉铭》开篇"维贞观六年孟夏之月、皇帝避暑乎九成之宫"明文）。泉水甘洌——魏徵铭文称"清若镜，甘如醴"。古人以"醴泉"应天瑞，太宗以此为治世祥瑞；魏徵则借此颂转谏——这是初唐政治文化的一个典型笔法。**醴泉至唐末仍有水——但宋元后泉脉日渐稀**。今九成宫遗址区内尚有标识为"醴泉故址"的水井残基，泉源已不见。

### 玄武门 + 后园（永徽五年水灾地）

宫北端，依天台山南麓。**永徽五年（654）五月**夜——山水暴涨自天台山下来、**冲玄武门入宫**，水深数丈。当时宫名已是"万年宫"——永徽二年（651）九月已改名（非水灾后）。《资治通鉴》卷一九九记宿卫多溺死；**《旧唐书·高宗本纪上》明文"水溺麟游居民及当番卫士、死者三千余人"**——是唐宫殿史最重大的一次水灾。高宗夜起避水——一州生灵在一夜间损失三千。**乾封二年（667）复名"九成"**。**这次水灾是九成宫从隆盛转衰的转折点**——高宗朝此后幸宫的次数减、避暑改去他处。

### 天台山（宫北靠山）

九成宫北面的靠山——海拔约 1300 米。**山非五岳、非道教祖庭**，是普通秦岭余脉——但因山势紧临宫后、夏日云雾常裹宫殿，九成宫得"山中气清、暑气不入"的避暑底子。山中夏雷雨频——这便是永徽五年水灾的源头。盛唐至中晚唐，山中无名刹、无名道观，是一座**靠山而非主角**的山。

## 历史长河（755 → 今）

**中晚唐**　天宝十四载后宫空如故。**晚唐间九成宫渐废**——具体废止年份诸史无明文记载，唐末战乱中殿宇坼毁、木材移作他用。**自此千年间九成宫只是一片山中废墟**。

**五代至宋**　北宋元丰间程颐过麟游游九成宫故址——所记"唯余夯土十数堆、墙垣尽倒"。北宋人来麟游主要是看醴泉铭碑——欧阳询楷书千余字仍在田间露天立。**宋人称欧体为"楷法之极则"**——苏轼、米芾、黄庭坚、蔡襄各有摹本——《九成宫醴泉铭》自此入宋人楷书学习的核心。

**金元**　元李好文《长安志图》卷上·麟游县·九成宫条记"宫址已为田、碑立田中"——元代农人耕作于宫基之上、夯土堆间。

**明清**　明代麟游县志记九成宫遗址"东西一里、南北一里余、夯土堆数十"；清乾隆四十年（1775）麟游县令初为醴泉铭碑立碑亭——碑亭今仍在县城内。**清代欧楷"九成宫"已是天下临池者的标准教材**——但临者多不曾来麟游、临的是拓本。

**民国至今**　1949 年后九成宫遗址列为陕西省重点文物保护单位。**1978-1995 年陕西省考古研究院多次发掘**——确认正门、朝堂、寝宫基址、天台山入水口、永徽五年水灾的洪积层（夯土上 60 厘米厚的灰白色淤泥层，年代约合永徽五年五月）。**2000 年代麟游县始建九成宫遗址公园**——夯土殿基已可在地表辨识、入口处立标识。**今游人来麟游，所见的九成宫只是几堆夯土与一通断碑**——盛唐人当年所见的那座山中宫殿，**考古至今也没碰到完整的一座殿**。
