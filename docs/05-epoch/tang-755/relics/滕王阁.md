# 滕王阁

## 一句话

唐高祖第二十二子滕王李元婴 **永徽四年（653）任洪州都督时所立**——立于豫章县西郊赣江与抚河汇口处；**上元二年（675）王勃过洪州即席作《秋日登洪府滕王阁饯别序》**——自此滕王阁不只是一座江南郡阁、是中国文学最有名的一座楼；755 时建阁已 102 年、王勃序传世已 80 年——是**江南正在使用中的名楼**。

## 形貌

滕王阁立**洪州 / 豫章郡治豫章县**（今江西南昌）西郊——**赣江与抚河汇口东岸**；自阁上西望——**赣江自南向北浩荡、远接鄱阳湖；远山一线即西山**（南昌西郊洪崖山一带）。**阁本身为江南郡阁形制**——三重檐、木构、台基高约数丈；具体唐制细节史源未详，**屡毁屡建、唐时即已数次重修**（《唐才子传》《文苑英华》等所记 755 时见的并非永徽四年原构）。

阁名"滕王"——以**永徽四年（653）李元婴任洪州都督时所建**得名；滕王元婴 是唐高祖李渊第二十二子、太宗李世民异母弟、**贞观十三年（639）封滕王**——封号取自春秋滕国（今山东滕州）旧名、与江南无地理关系。元婴生性好治宫室、骄纵失礼——**永徽年间在洪州大兴土木立此阁**；高宗对此屡有谴责（《旧唐书·滕王传》"骄纵失礼、好治宫室——高宗屡有玺书谴之"）——**阁在某种意义上是唐宗室骄奢的物理纪念**、但因王勃一序、自此意义彻底翻转。

阁前有平台、可远眺；**阁下即赣江舟楫**——洪州是江南西道的水运枢纽、自鄱阳湖南通赣江、北接长江——滕王阁正立此水陆要冲。755 时阁殿宇完整、为洪州官方迎宾设宴之所、亦为文人登临咏怀之地——**是江南三大名楼**（注：**"三大名楼"是明清后世并称、755 时唐人无此固定 grouping**）中最东南的一座、与北方蒲州**鹳雀楼**形成"南北名楼"的隐约对照——但盛唐人尚不作此对举。

物候：江南西道亚热带季风、四时分明——春多雨、夏多潮、秋高、冬阴；**阁前赣江夏秋常涨水、舟楫繁忙**——是 755 时唐人登阁所见的实物景。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

滕王阁在唐人的回望里——是**两段叠加的来历**：建阁人滕王元婴 + 作序人王勃——一座阁、两个名号、互为表里。

**第一段：滕王李元婴 + 永徽建阁（639-653）**。**贞观十三年（639）**唐太宗封异母弟元婴为滕王、食邑山东滕州——但元婴本人始终未就藩滕州。**永徽四年（653）**——元婴自苏州 / 山东等任迁**洪州都督**；**到任后在豫章县西郊赣江畔大兴宫室**——立此阁、以封号"滕王"为名。**阁立时元婴 25 岁上下**——洪州都督的一座官方郡阁、本不为留名后世；但**元婴本人骄纵失礼、屡为高宗所谴**——《旧唐书·滕王传》"好治宫室、骄纵失礼——高宗屡有玺书谴之"——可见此阁在唐廷视角下并非美谈。**文明元年（684）**元婴卒于隆州（今四川阆中）刺史任上——年 56；自此**滕王阁与建阁人脱钩**、成洪州一座郡阁。

**第二段：王勃 + 上元作序（675）**。**上元二年（675）**——王勃自洛阳赴交趾（今越南北部）省父途中、过洪州；**九月九日重阳**——洪州都督**阎伯屿**（按王勃序"阎公"、《唐才子传》明记"阎都督名伯屿"）在滕王阁设宴饯别群僚 / 客；席间阎都督本意请其婿吴子章作序、**王勃即席请笔、立成《秋日登洪府滕王阁饯别序》**——**"豫章故郡、洪都新府……落霞与孤鹜齐飞、秋水共长天一色……"**——一座江南郡阁自此成中国文学最有名的一座楼。**王勃 675 / 676 间（异说）赴岭南、渡海或行海路至交趾、溺水后受惊而卒**（《旧唐书·王勃传》《唐才子传》）——年 27（一说 28）。**王勃自此再未过滕王阁**——只一次、却一序定千古。

**第三段：盛唐 80 年——序定典范（675-755）**。王勃《滕王阁序》自上元二年传出以后——**在初唐至盛唐 80 年间渐成"江南登临赋"的最高典范**。**唐人文集 / 传抄**之间此序流布甚广——盛唐文人多读多诵；阁在洪州本就是江南名胜、加王勃序加持——**755 时已是盛唐文人入江南西道必登的一座阁**。

### 此刻状态

天宝十四载这一年的滕王阁——**建成 102 年、王勃序传世 80 年、阁殿宇完整、正在使用中**。

**物理状态**：阁在豫章县西郊赣江畔；具体 755 时所见的形制——**应是永徽原构 + 多次小修的累积**——史源未详具体哪一次重修；《唐才子传》言滕王阁"屡毁屡建"——盛唐 100 余年间或已重修数次、但**755 时阁殿宇完整、官方维持使用**。

**功能状态**：滕王阁 755 时是**豫章郡 / 江南西道官府的官方迎宾设宴之所**——**豫章郡太守 + 江南西道采访使**过境、皆于此设宴；文人过境、亦多登临咏怀。**江南西道采访使**驻洪州、是 755 时阁的主要使用者之一。**洪州 / 豫章郡治豫章县**——**天宝元年（742）改州为郡**之制下、洪州改豫章郡；755 时长官以"豫章郡太守"称之、"洪州都督"之号已不再用（详 子条目"阎都督伯屿"段）。

**安史前夕**（即 755 这一刻）——再过一月（11 月初九）安禄山起兵于范阳——但**洪州地处江南、距河朔遥远**；**755 时滕王阁感受不到北方战云**——是江南最稳的一段时光。**至德元载 / 二载（756-757）**——**安史中江南未陷**；**滕王阁所在的洪州亦未陷**；**756 年玄宗入蜀、肃宗在灵武即位、洪州 + 江南西道仍为唐廷的财赋后方**——**滕王阁在安史中物理上未受冲击**。**乾元元年（758）以后**——洪州为江南西道节度使治、地位再升；**滕王阁在中晚唐的核心叙事自此展开**（详 历史长河段）。

### 此刻意味

对天宝十四载的洪州人来说——滕王阁是**州城西郊那座官府郡阁**：日常洪州 + 江南西道官府的迎宾之所、有时民间登临；既是城市纪念建筑、又是江南名胜。对盛唐文人来说——滕王阁是**江南入境的第一座名楼**：自岭北 / 江北南下经鄱阳湖入赣江、抵洪州——必登滕王阁、必诵王勃序——"豫章故郡、洪都新府"——这是 755 时盛唐文人对江南西道的物理 anchor。

唐人尚不知道——再过 65 年（元和十五年/820）韩愈会作《新修滕王阁记》——把这座楼的"文学纪念"地位再推一层；再过 200 余年（北宋）王安石 + 苏轼 + 黄庭坚等会续咏滕王阁；**再过千年、阁屡毁屡建、史载有 29 次重建之说**（明清地方志统计、非唐人语）——**每一次重建都因王勃那一序**。**今游人来南昌见的滕王阁**：是 1989 年于古阁基址附近、按宋画式样新建的钢筋混凝土仿木九重楼——**与盛唐 755 时所见 木构数重 江南郡阁 形制不同**——但**地理位置 + 文化记忆 在赣江与抚河汇口处 一千三百多年 未易**。

## 子条目

### 滕王李元婴（628-684；建阁人）

**唐高祖李渊第二十二子、唐太宗李世民异母弟**——母**柳宝林**；**贞观十三年（639）**封滕王、食邑山东滕州——但未就藩。**永徽间**先任金州刺史、苏州刺史、洪州都督——**永徽四年（653）任洪州都督时立滕王阁**。**性好治宫室、骄纵失礼**——《旧唐书·滕王传》"高宗屡有玺书谴之"；**文明元年（684）**卒于**隆州刺史**任上（一说梁州、本 doc 取《新唐书·宗室世系表》隆州说）——年 56。**滕王元婴在唐宗室谱系中并非显赫人物**——但因这一座阁、名字得以与王勃序同传后世。**注**：滕王元婴的"滕"字——封号取自春秋**滕国**（今山东滕州）——与江南无地理关联；阁所以名"滕王"——只因建阁人封号为滕王、不因阁在滕地。

### 王勃（650-675/676）与《滕王阁序》

**王勃 字子安、绛州龙门（今山西河津）人**——**初唐四杰之首**（与杨炯、卢照邻、骆宾王并称）。**麟德元年（664）应举**——上元二年（675）赴交趾省父途中过洪州；**九月九日重阳**洪州都督阎伯屿在滕王阁设宴——本意请婿吴子章作序、王勃即席请笔、**立成《秋日登洪府滕王阁饯别序》（即《滕王阁序》）**——"豫章故郡、洪都新府……落霞与孤鹜齐飞、秋水共长天一色……"——盛唐文人皆习此序。**同年八月或九月王勃过岭南**——**渡海或行海路至交趾、溺水后受惊而卒**（《旧唐书·王勃传》《唐才子传》）——**年 27**（《新唐书》《唐才子传》"年二十八"、《旧唐书》"年二十九"——本 doc 取 27 一说为主）。**王勃只一次过滕王阁、只一次作序、自此再未来**——但这一序传世一千三百多年。**注**：序末附《滕王阁诗》"滕王高阁临江渚、佩玉鸣鸾罢歌舞……闲云潭影日悠悠、物换星移几度秋；阁中帝子今何在、槛外长江空自流"——是序后所附七言诗、与序文并传。

### 阎都督伯屿（675 设宴主人）

**上元二年（675）洪州都督**——王勃序中"阎公"即此人。**《唐才子传》明记"阎都督名伯屿"**——盛唐 + 中晚唐文集中亦多见"阎公"称谓。**阎伯屿本意请婿吴子章作序、不料王勃即席请笔、立成此序**——阎公先怒、后服、终拍案称"此真天才"（《唐才子传》本传节）。**注**：本 doc 守"阎都督"——**不写"洪州刺史"**：洪州在 675 时为都督府治、长官称都督；天宝改郡后改"豫章郡太守"——755 时则不再有"都督"称谓的洪州长官、但 675 时是阎都督。

### 阁制与历次重修

**滕王阁屡毁屡建——明清地方志统计有 29 次重建之说**（具体次数史源不一、本 doc 仅言"多次"）。**永徽四年（653）原构**——元婴所立、木构数重；**唐间至少数次重修**——具体年代史源未详；**755 时所见**：应为某次唐重修后的形态、殿宇完整、官方使用。**中晚唐**——韩愈《新修滕王阁记》（元和十五年/820）记**洪州刺史王仲舒新修阁** + 韩愈作记纪事；**北宋至明清**——屡毁屡建、形制屡变；**今所见**（1989 年新建）——按宋人**郭熙《早春图》**风格的宋画式样仿木九重楼——**非唐制、非宋制原物、非任何一次古阁的复原**。

### 江南名楼并称问题

**唐人未作"江南三大名楼"称**——**"滕王阁（洪州）+ 黄鹤楼（鄂州）+ 岳阳楼（岳州）"三大名楼并称是明清后世总结**、非 755 时唐人语。**北方"鹳雀楼"（蒲州）**——王之涣"白日依山尽"作于盛唐、在 755 时已传 30 余年——但 755 时唐人未作"南北名楼对举"。**755 时唐人对滕王阁的认知**：是江南西道洪州的一座著名郡阁、因王勃序而文学价值居高——但**未与黄鹤楼 / 岳阳楼并立为"江南三大"**——这是后世文学史的回溯整理、不入 755 时态。

### 滕王阁与洪州 / 豫章郡（行政背景）

**滕王阁所在的洪州 / 豫章郡**——**天宝元年（742）**唐玄宗改州为郡之制下、洪州改"豫章郡"；**至德二载（757）**肃宗复改郡为州、豫章郡复称"洪州"（与 [洪州.md](../cities/洪州.md) 统一口径）；**乾元元年（758）**洪州升为江南西道节度使治。**755 时滕王阁所在**——豫章郡治豫章县、太守驻；江南西道采访使亦驻洪州——是江南西道的政治 + 文化 + 经济中心。**阁立此城西郊**——是江南西道最有象征性的一座官阁。**注**：**755 时正名豫章郡 / 治豫章县**；"南昌"作汉初古县名早已存在（**西汉豫章郡治南昌县**、唐高祖**武德五年/622 废南昌县**、之后多次反复）；**元至元二十一年/1284 改南昌路**（路级名号、前一年至元十四年/1277 已先改隆兴府为隆兴路）——**自此"南昌"重新成为主要地名**；755 时虽"南昌"古县名久废、洪州 / 豫章郡才是正名——本 doc 严守唐口径、不写"南昌滕王阁"。

## 历史长河（755 → 今）

**中晚唐**　**安史中（755-763）江南未陷**——滕王阁物理上未受冲击。**乾元元年（758）洪州升江南西道节度使治**——阁地位再升。**元和十五年（820）韩愈作《新修滕王阁记》**——记**洪州刺史王仲舒新修阁**+ 韩愈作记纪事——**这是滕王阁继王勃序之后第二段最重要的文学纪念**。**晚唐杜牧、温庭筠、皮日休、陆龟蒙**等续咏滕王阁——但都以"读王勃序"为底色、自身咏诗未成大作。**唐末黄巢（878-884）扰江南**——洪州为战乱波及、滕王阁是否毁于此期间史源未详；**905 年钱镠割据浙东、洪州为杨吴所据**——滕王阁入十国杨吴格局。

**两宋**　**北宋初期**——洪州为江南西路治、滕王阁屡修；**北宋诗人欧阳修、王安石、苏轼、黄庭坚、辛弃疾**等多有滕王阁咏——**黄庭坚《登快阁》**虽不专写滕王阁、但属洪州登临文学的延续。**北宋至和年间（1054-1055）**滕王阁重修——按宋画式样、形制有所变；**南宋**洪州仍为江南西路治、滕王阁续作官府郡阁。**南宋绍兴 / 隆兴 / 嘉定 / 端平**等年间多次重修——是滕王阁最稳定的二百年。

**金元**　**元代洪州先称隆兴府、至元十四年/1277 改隆兴路、至元二十一年/1284 改南昌路**——**自此"南昌"（作路级名号）重新成为主要地名**（"南昌"原为汉初古县名、西汉豫章郡治南昌县、唐高祖 622 废、元至此重启）。**元代滕王阁**屡毁屡建——元末战乱中或毁。

**明清**　**明清间南昌府治南昌县**——滕王阁继作官府郡阁。**明洪武初年**重修；**明清间统计滕王阁重建 28 次、加上初唐永徽建阁一次共 29 次重建之说**——是中国楼阁中重建次数最多的一座。**清同治十一年（1872）**毁于火；后未再建——**自此滕王阁缺席江西凡 117 年**。

**民国至今**　**1942 年梁思成、莫宗江**绘《重建滕王阁计划草图》——拟按宋人**郭熙《早春图》**风格仿建——计划未行；**1985 年江西省政府启动重建**——**1989 年 10 月 8 日**——**今所见 滕王阁 落成开放**——按梁思成 / 莫宗江草图基础、新建钢筋混凝土仿木九重楼、高 57.5 米、立古阁基址附近（赣江与抚河汇口处略东、即今南昌赣江东岸）。**今游人所见**：是 1989 年新建的现代仿古楼——**与盛唐 755 时所见 木构数重 江南郡阁 形制不同**——但**地理位置 + 文化记忆 在赣江与抚河汇口处 一千三百多年 未易**——**这一点上、阁与"滕王阁"这个名字、是唐宋元明清民共和六代的连续物**。
