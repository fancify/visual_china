# 瓜洲渡

## 一句话

**江北最重要的长江渡江津口**——瓜洲在长江北岸、江都县南郊三十余唐里；是江北 ↔ 江南的渡江点（对岸为京口），**与"邗沟入江口"毗邻但不等同**：邗沟入江口是邗沟水道与长江交接的水道分界点，瓜洲渡则是江岸上的渡江津口，二者相距约二十唐里、共同构成大运河南端 ↔ 长江的中转枢纽。与江南岸**京口**（润州治）隔一江相望，**江都 + 京口 + 瓜洲三地一江一渡**形成"江口三镇"；**755 时**江南诸道粮米先集江都、再经邗沟入江口入江、于瓜洲渡一带换江船，是盛唐漕运南端的物理 anchor；天宝十四载十二月丁酉叛军陷东京洛阳（公历 756 年 1 月）、通济渠北端失、漕路中断、瓜洲一带的繁忙骤断。

## 形貌

瓜洲在 755 时——**已不是江中沙洲、而是与江北岸相连的半岛 + 渡口**。

**地形沿革**：
- **春秋至秦汉**——瓜洲为江中沙洲、形如瓜、故名"瓜洲"；其时长江口尚远在今扬州东南、瓜洲处大江主泓；
- **六朝至隋**——长江主泓南移、北汊渐淤、瓜洲沙洲与江北岸沙嘴相接；至隋大业元年（605）重浚邗沟时、入江口已近瓜洲北缘；
- **唐初至盛唐**——瓜洲与江北岸完全相连成半岛、北接江都县南郊、南临长江、东西宽十余唐里；**邗沟自江都南下入江、入江口在瓜洲北侧约二十唐里**——瓜洲渡是邻近的渡江津口、与邗沟入江口共同构成江南漕米北上 + 江北客商南下的总枢纽。

**地理位置**：**广陵郡江都县南郊**——自江都城南行三十至四十唐里至瓜洲；瓜洲东西略长 + 南北较窄、形如半岛伸入长江；**瓜洲南面长江**——江面在此约七至八唐里宽（盛唐时江流较今宽、北岸线约今扬州市邗江区瓜洲镇一带）；**江南岸**即**润州治京口县**（今镇江市）——南北直线对岸；**渡江**约半个时辰（顺风顺流）至一个时辰（逆风）。

**渡口设施**：盛唐 755 时瓜洲渡——**官设水驿 + 民间渡舟**并立。官设水驿属**淮南道**辖、**江都尉**或**江都丞**督；津吏 + 驿卒 + 渡舟 + 纤夫；漕舟自江南来、于此换江船入邗沟北上；客商往返、于此换江舟过京口。**津渡费用**：唐律津渡有公私两途——官差持牒免费；商旅按舟纳钱；**《唐六典》卷七·水部郎中员外郎条**载津渡水驿之制——三十里一水驿、漕路 + 津要并设。

**周边水道**：
- **邗沟**——自江都城南门外出、南行三十余里入长江；入江口在瓜洲北侧约二十唐里（与瓜洲渡毗邻、是水道分界点而非渡江津口）；**春秋吴王夫差（前 486）开** + **隋大业元年（605）重浚**；两岸植柳；
- **长江**——江面宽七八唐里；江南岸京口在隔江相望；江中有沙洲若干、唐时已成航道礁滩；
- **江北沿岸沙嘴**——瓜洲东西各有沙嘴伸入江中、是江舟避风停泊处。

**物候**：盛唐时瓜洲一带——**亚热带季风、四时分明**；春多桃花水 + 渡口柳烟；夏多霖 + 江风急；秋高气爽 + 漕舟密集；冬有薄霜 + 江口结薄冰不冻渠。**江口风涛**——长江下游近海口、潮汐影响明显（盛唐时江潮可上溯至江都南郊）；渡江须候潮 + 候风、是江口渡口的常态。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

瓜洲渡在唐人的回望里——**是江中沙洲变半岛的地理史 + 隋唐大运河南端起点的漕运史**。

**地形史的回望**：
- **春秋**——瓜洲已是江中沙洲、形如瓜；《水经注·卷三十·淮水》载江中沙洲与北岸沙嘴的古貌；
- **战国 + 秦汉**——长江主泓渐南移、瓜洲北汊渐淤；
- **六朝**——南朝梁陈间瓜洲与江北岸已大致相连、成半岛雏形；
- **隋大业元年（605）**——隋炀帝重浚邗沟（春秋夫差前 486 已开此渠）、入江口即在瓜洲附近、半岛 + 渡口格局初成；
- **唐初至盛唐**——瓜洲彻底成为江北岸半岛、渡口格局完备。

**漕运史的回望**：
- **春秋末（前 486）**——吴王夫差为北上争霸开邗沟、自江都通至淮河、邗沟入江口在瓜洲一带；
- **隋大业元年（605）**——隋炀帝重浚邗沟 + 开通济渠、洛阳与江都水路联通；**邗沟入江口自此成为隋唐大运河南端的水道接点、瓜洲渡作为毗邻的渡江津口（换船渡江节点）同时定型**；隋炀帝三下江都皆经此渡；
- **隋大业六年（610）**——开江南河、自江都南渡长江、过京口至余杭——大运河南段成；**瓜洲 + 京口两岸渡口成为大运河南段与中段的物理接点**；
- **唐承隋制**——开元 / 天宝间瓜洲渡漕舟接连、商旅往返、是盛唐财政命脉的南端 anchor。

**广陵郡 + 江都县的州名沿革**：**隋大业三年（607）改扬州为江都郡**；**唐武德三年（620）复扬州**；**天宝元年（742）改扬州为广陵郡**（汉旧名）；**乾元元年（758）复扬州**——**755 时正名"广陵郡"、治江都县、瓜洲在江都县南郊**；行政文书写"广陵郡江都县瓜洲渡"，民间俗称"扬州瓜洲""江都瓜洲"皆可。

**渡口的具体使用者**——**漕舟**（江南诸道粮米经此北上 + 关中布帛南下经此入江）+ **江南赴京使节**（盐铁使 + 转运使 + 江南诸节度奏报）+ **商旅**（盐 + 丝 + 米 + 茶 + 瓷器）+ **僧侣**（江南名山赴中原 + 扬州大明寺与江南天台 / 牛头宗的来往）+ **文人**（江南士子赴京应举 + 落第东归）——是盛唐江北最繁忙的渡口之一。

### 此刻

天宝十四载这一年的瓜洲渡——**繁忙、按例运行、漕舟接连、商旅往返、即将经历安史浩劫**。

**渡口状态**：盛唐 755 时瓜洲渡——官设水驿完备、民间渡舟数十；江中漕舟自江南来、于瓜洲一带换江船入邗沟北上（邗沟入江口在瓜洲北侧约二十唐里）；江北客商南下、于瓜洲换江舟过京口；**邗沟自江都城南门出、南行入长江、两岸植柳、纤路与官道并行**。**唐令常制三十里置驿**——江都至瓜洲一线水驿 + 陆驿夹杂；津吏验牒、漕官督运、渡舟应差。

**漕运的实际数字**：**盛唐漕运南端起点**——江南诸道粮米先集江都、再经邗沟入江口入江、于瓜洲渡一带换江船北上至楚州 + 入淮 + 转通济渠西北至洛阳含嘉仓 + 再陆运过三门峡至关中太仓；**开元二十五年（737）至天宝元年（742）累岁运米七百万石**（《新唐书·食货志三》）——这七百万石的相当部分须经瓜洲渡一带换船北上；**瓜洲渡是盛唐财政命脉南端的渡江津口 anchor**——与邗沟入江口共同构成南端起点。

**江口三镇**——**江都 + 京口 + 瓜洲**：江都在江北、是淮南采访处置使治、唐六雄之一（"扬一益二"之扬）；京口在江南、是润州治、镇江中下游的咽喉；瓜洲在江北南郊、是江都的南门户 + 渡江津口（**邗沟入江口在瓜洲北侧约二十唐里、是毗邻的水道分界点而非同一处**）；三地一江一渡、构成长江下游北段的核心节点。**渡江路线**：瓜洲渡↔京口渡——七八唐里江面、半个时辰至一个时辰可达；**江中无桥**（长江下游有桥要到 1968 年南京长江大桥）；**渡舟为唯一通道**。

**唐人对瓜洲的认知**：瓜洲是漕运 + 商旅 + 使节的主要中转地；江北赴江南必经瓜洲渡过京口；江南赴江北亦必经京口渡至瓜洲。**俗谚"京口瓜洲一水间"**——但此为北宋王安石《泊船瓜洲》之句（1075 作）、不能 755 现场；**张祜《题金陵渡》"潮落夜江斜月里、两三星火是瓜洲"**——是中唐作（张祜 785-849）、亦非 755 现场；755 时瓜洲尚未入诗、是纯然的漕运 + 商旅渡口。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九即起）；**天宝十四载十二月丁酉陷东京洛阳**（公历 756 年 1 月）——通济渠北端起点失；**江淮漕路自此中断**——瓜洲渡虽存、但北上漕舟无处可去；**广陵郡 + 淮南节度未陷**——成为唐廷江南半壁的物理 anchor。**至德元载（756）十二月永王李璘自江陵东下、舟师顺长江欲取广陵（扬州）**——长江下游一带、瓜洲 + 京口 + 江都皆为战略要地；**至德二载（757）二月**永王败死——瓜洲一带未陷；**至德二载（757）十月唐军复东都洛阳**——但通济渠沿线已疮痍、漕路至广德元年（763）后才陆续恢复——瓜洲渡的繁忙亦自此重启。

### 此刻意味

对天宝十四载的广陵郡江都县人民来说——瓜洲渡是**自家南郊的命脉**：日常漕舟出入 + 商旅往返 + 使节渡江；江都城南门外舟车辐辏、皆为入瓜洲渡之用。对江南诸道的人民来说——瓜洲渡是**北上的第一站**：江南粮米经此入邗沟、北上至洛阳；江南士子赴京应举、经此过京口北上。对盛唐唐廷来说——瓜洲渡是**财政命脉的物理 anchor**——南端起点；缺此一渡、江淮粮米无以北输、关中无以支撑。

唐人尚不知道——天宝十四载十二月丁酉（公历 756 年 1 月）通济渠北端将断、漕粮停输、关中饥荒；瓜洲渡虽未陷、但**北上之路已断**；**至德 / 乾元间江淮租赋赖此道苦输支撑唐廷半壁**——但漕路反复中断、瓜洲渡的繁忙骤减；**广德元年（763）以后刘晏改漕运法**——立**江船 + 汴船 + 河船 + 渭船**四段接驳——**江船自江南诸州集江都；汴船自江都经邗沟入淮 + 转通济渠西北至河阴仓**——瓜洲渡是江船 ↔ 汴船的接驳点、刘晏制下中唐漕运的物理枢纽再生。

**今游人重访瓜洲一带**——所见的是**扬州市邗江区瓜洲镇** + **古运河南端 + 长江北岸**——盛唐时的瓜洲半岛因后世江流变迁有所迁移、但大致在同一地理位置；**邗沟入江口**今为京杭大运河扬州段的南端、仍通；**京口对岸**为今镇江市、隔江相望；**渡舟早已停用**——1968 年南京长江大桥通车后、长江下游渡口陆续退出；**瓜洲渡的物理形态早已不在**、但作为漕运史地名仍在。

## 子条目

### 瓜洲（沙洲变半岛的地形史）

**春秋至唐初瓜洲的演变**：
- **春秋**——瓜洲为江中沙洲、形如瓜、故名；
- **战国 + 秦汉**——长江主泓南移、瓜洲北汊渐淤；
- **六朝**——南朝梁陈间瓜洲与江北岸大致相连、成半岛雏形；
- **隋大业元年（605）**——隋炀帝重浚邗沟（春秋夫差前 486 已开此渠）、入江口即在瓜洲附近；
- **唐初至盛唐**——瓜洲彻底成为江北岸半岛、东西宽十余唐里。

**地理史的意义**：瓜洲是中国地形史上**沙洲与陆地连接**的典型案例——长江下游主泓持续南移、北岸沙嘴持续延伸、原本江中的沙洲逐渐与北岸相连。**今瓜洲位置**——较盛唐时略北、因后世江流变迁、今瓜洲镇位于扬州市邗江区南郊、长江北岸；与盛唐瓜洲半岛在同一地理走廊上、但细部位置略有迁移。

### 瓜洲渡（毗邻邗沟入江口的渡江津口）

**渡口的形成**：**春秋末（前 486）吴王夫差开邗沟**——邗沟入江口在瓜洲北侧一带、瓜洲渡作为江北渡江津口与之毗邻；**隋大业元年（605）隋炀帝重浚邗沟**——瓜洲渡格局初成；**隋炀帝三下江都皆经此渡**；**唐承隋制**——瓜洲渡是漕运 + 商旅 + 使节的总枢纽。

**渡口的设施**：盛唐 755 时——官设水驿 + 民间渡舟并立；津吏 + 驿卒 + 渡舟 + 纤夫；漕舟自江南来、于瓜洲换江船入邗沟北上；客商往返、于瓜洲换江舟过京口。**渡口费用**：**《唐六典》卷七·水部郎中员外郎条**载津渡水驿之制——三十里一水驿、漕路 + 津要并设。

**渡口的地位**：瓜洲渡是**毗邻邗沟入江口的渡江津口**（与邗沟入江口相距约二十唐里、二者共同构成大运河 ↔ 长江的中转枢纽——入江口承担漕舟入邗沟的水道动作、瓜洲渡承担换船渡江的津口动作）——江南诸道粮米经此一带换船北上、是盛唐财政命脉南端渡江节点的物理 anchor；**755 一刻**仍按例运行、天宝十四载十二月丁酉（公历 756 年 1 月）随通济渠北断而骤减。

### 邗沟（春秋吴王夫差开、隋大业元年重浚）

**自江都（广陵郡治）南行入长江、北行至山阳（楚州治；今淮安）**——中国最古老的人工运河之一。**春秋末（前 486）吴王夫差为北上争霸开**——自江都向北、引江水入淮；**隋大业元年（605）与通济渠同时重浚**——拓宽 + 加深 + 立水驿；两岸植柳。**沿途要节**：江都 + 邗沟入江口（瓜洲北侧）+ 高邮 + 宝应 + 山阳；**唐承隋制**——邗沟是江南漕路北段；**至德 / 乾元间漕路反复中断**——但邗沟本身始终未湮；**北宋元祐七年（1092）苏轼知扬州**多次行此渠；**今邗沟一段尚为京杭大运河扬州段**——是隋唐三段大运河中**唯一未湮的一段**。**邗沟入江口**——位于瓜洲北侧约二十唐里、与瓜洲渡毗邻但不等同（一为水道分界点、一为渡江津口）；**邗沟入江口作为独立 POI 另立 doc**——本文聚焦瓜洲渡本身。

### 京口（润州治；今镇江）

**江南岸隔江相望**——京口在长江南岸、是润州治、镇江中下游的咽喉；**755 时正名丹阳郡**（天宝元年/742 改润州为丹阳郡；乾元元年/758 复润州）；**京口 + 瓜洲一江相隔**——七八唐里江面、半个时辰至一个时辰可达。**京口的渡口**——京口渡与瓜洲渡对开、是长江下游北段的主要渡口对；江南漕米经京口渡过瓜洲入邗沟北上；江北客商经瓜洲渡过京口入江南河南下至余杭。**江口三镇**——江都 + 京口 + 瓜洲、一江一渡、构成长江下游北段的核心节点。

### 漕运（江南粮米北上经瓜洲渡）

**盛唐财政基本面**——**关中仰给江淮**。**瓜洲渡是漕运南端的渡江津口、毗邻邗沟入江口**：江南诸道粮米先集江都 → 经邗沟入江口入江、于瓜洲一带换江船 → 北上至楚州入淮 → 转通济渠西北至洛阳含嘉仓 → 陆运过三门峡至关中太仓。**开元二十五年（737）至天宝元年（742）累岁运米七百万石**（《新唐书·食货志三》）——这七百万石的相当部分须经瓜洲一带北上；**瓜洲渡是盛唐财政命脉的物理 anchor**——南端渡江津口。

**安史中瓜洲渡的命运**——通济渠北断、漕路停输、瓜洲渡的繁忙骤减；**广德元年（763）以后刘晏改漕运法**——立**江船 + 汴船 + 河船 + 渭船**四段接驳；**江船自江南诸州集江都；汴船自江都经邗沟入淮 + 转通济渠西北至河阴仓**——瓜洲渡是江船 ↔ 汴船的接驳点；**自此中唐唐廷半壁赖江淮租赋苦输支撑**——瓜洲渡再次成为漕运的物理枢纽。

### 永王李璘东下事（756-757 长江下游战事）

**至德元载（756）十二月**永王李璘自江陵东下、舟师顺长江欲取广陵（扬州）、谋自立；**长江下游一带、瓜洲 + 京口 + 江都皆为战略要地**——若永王取广陵、则唐廷江南半壁危；**至德二载（757）二月**永王败死——为地方官所执；**江都未陷、瓜洲渡一带未陷**——唐廷江南半壁得以保全。**永王事是安史中长江下游唯一一次大规模军事冲突**——瓜洲渡作为江北最重要的渡口、是战事的物理 anchor 之一；但文献未明载瓜洲渡在永王事中的具体角色、当为后方枢纽。

## 历史长河（755 → 今）

**中晚唐**　**755-763 漕路反复中断**——瓜洲渡的繁忙骤减；**广德元年（763）刘晏改漕运法**——四段接驳重立、瓜洲渡作为江船 ↔ 汴船接驳点重新繁忙；**中唐张祜《题金陵渡》"潮落夜江斜月里、两三星火是瓜洲"**——瓜洲始入唐诗、成为江口意象的代表；**晚唐杜牧赴扬州 / 离扬州**多次经瓜洲渡——但杜牧诗中瓜洲未单咏。

**五代**　**杨吴 + 南唐**以广陵 / 江都为东都——瓜洲渡仍为江北南端核心渡口；**南唐后主李煜**渡瓜洲北降宋——是瓜洲渡在中国政治史上的一个 dramatic moment。

**两宋**　**北宋**——瓜洲渡是江南漕米北上汴京的总枢纽、与隋唐时地位无异；**北宋熙宁八年（1075）王安石《泊船瓜洲》**：**"京口瓜洲一水间、钟山只隔数重山。春风又绿江南岸、明月何时照我还。"**——瓜洲渡自此成为中国诗史上最著名的江口意象之一。**南宋绍兴间宋金对峙**——瓜洲渡成为南宋江北防线的最前沿；**绍兴三十一年（1161）虞允文采石矶大捷**——瓜洲一带亦为战场。**南宋绍熙五年（1194）黄河南徙夺淮**——通济渠中下段被冲毁、但邗沟与瓜洲渡仍通。

**元明清**　**元代会通河北开**——大运河走向变化、但邗沟 + 瓜洲渡仍为南北水道的核心；**明清间瓜洲为长江下游的繁华渡口**——商船商旅密集；**清乾隆间瓜洲城**已成江北重镇；**清道光以后江流变迁**——瓜洲城屡有崩塌、咸丰间瓜洲老城大部入江、镇市北移至今址。

**民国至今**　**1949 年后**——瓜洲渡逐渐被现代交通取代；**1968 年南京长江大桥通车**——长江下游渡口陆续退出；**今瓜洲镇**位于扬州市邗江区南郊、长江北岸、京杭大运河扬州段南端；**邗沟入江口**今仍通、为京杭大运河南端入江处之一；**京口对岸**为今镇江市、隔江相望；**渡舟早已停用**。**今游人重访瓜洲一带**——所见的是古运河 + 长江北岸 + 瓜洲镇遗存（部分明清城垣）+ 王安石《泊船瓜洲》纪念碑——盛唐时的瓜洲渡作为物理形态早已不在、但作为漕运史 + 诗史地名仍在中国文化记忆中。

## 引文

据《元和郡县图志·卷九·淮南道·扬州（广陵郡）·江都县》大意——邗沟自江都南三十里入大江；江中原有沙洲名瓜洲、与北岸沙嘴相连成半岛、为江北津要；漕舟自江南来、于此一带换江船入邗沟北上（瓜洲沙洲沿革 + 邗沟入江口）。

> 大业元年三月、发河南诸郡男女百余万、开通济渠……自江都宫入于海。
> ——《通典·卷十·食货十·漕运》（隋重浚邗沟 + 开通济渠之始）

> （开元二十五年至天宝元年）累岁运米七百万石、自江淮抵东都；自东都抵关中、岁百余万石。
> ——《新唐书·卷五十三·食货志三》（盛唐漕运的实际数字）

> 潮落夜江斜月里、两三星火是瓜洲。
> ——张祜《题金陵渡》（中唐作；瓜洲始入唐诗，属 755 之后的文学回响）

> 京口瓜洲一水间、钟山只隔数重山。
> ——王安石《泊船瓜洲》（北宋熙宁八年/1075；瓜洲入诗的最高峰，属后世文学回望）

> （至德二载二月）肃宗命高适、来瑱、韦陟等讨之；永王李璘东下、败走、为地方官所执而死。
> ——《资治通鉴》卷二一九·至德二载（参《旧唐书·永王李璘传》）
