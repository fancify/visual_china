# 剑门关

## 一句话

**蜀道金牛道咽喉关、大剑山主脊狭口的一道关塞**——剑州普安郡普安县境（今四川剑阁县剑门关镇）；**李白"剑阁峥嵘而崔嵬、一夫当关、万夫莫开"在 755 时已是流传的蜀道想象**；**至德元载（756）玄宗西奔入蜀经此关 + 至德二载（757）自蜀北还复经此关**——剑门关由此从蜀道一关塞、变成盛唐转衰的物理 anchor。

## 形貌

剑门关在唐剑南道**剑州（普安郡）普安县境**——**大剑山主脊狭口**（今四川省广元市剑阁县剑门关镇境，海拔约 1000 米）。**地势**：**大剑山自西北而东南横亘川北**——主脊东西连绵约七十里、峰峦相接如刀削；**剑门关就在大剑山主脊中段的狭口**——南北两侧是千仞峭壁、东西两山相夹如门、关城正立其中。**自普安县（剑州治）北行二、三十里至剑门关下**——是金牛道由汉中入蜀的最险一关。

**剑门关与大剑山、小剑山**：**大剑山在北、小剑山在南**——二山主脊相连、皆属龙门山脉东南余脉；**剑门关在大剑山主脊**——不在小剑山。**唐人称"剑门"** 多指大剑山主脊狭口的关塞；**称"剑阁"** 则是剑州境内栈道全段的总名——**剑阁是地名 + 栈道总称、剑门关是具体关塞**——二者不可混用。**李白《蜀道难》"剑阁峥嵘而崔嵬"** 写的是剑州境内栈道的整体山势、不专指关城形制。

**金牛道与剑门关**：**金牛道自梁州（汉中）南行 + 经利州（绵谷）+ 入剑门关 + 至剑州（普安）+ 绵州 + 汉州 + 至益州（成都）**——是关中入蜀道在汉中以南的全程；**剑门关扼大剑山主脊狭口**——是这条道上**最险一关**。**注**：金牛道是**汉中-益州段**——与**关中-汉中段**的褒斜道、子午道、故道（陈仓道）不同；后世通称"蜀道"广义包含八条、狭义即指金牛道。**剑门关不在褒斜道线上**——褒斜道是关中-汉中段；剑门关在汉中以南的金牛道上。

**关城形制**：盛唐剑门关的具体形制 + 周长 + 门数史载零散——本 doc 不预设细节。**今所见剑门关楼**：以**明洪武年间 + 明万历 + 清代多次重修**为主体；**1935 川陕公路开通**时关楼大部毁；**1985 年关楼重建** —— 都不是唐代原物。**唐剑门关的形制不可坐实**——不要写"唐时如此"。**可知者**：**关城据狭口而立、关楼立大剑山主脊**——南依大剑山势、北临嘉陵江系入剑门一段的栈道；**关城内外多戍兵 + 关吏 + 商旅** —— 是金牛道沿线最重要的关塞。

**物候**：大剑山北麓、温带季风偏副热带湿润过渡；夏湿热多雾、冬不甚寒；山中林木茂密、多巴蜀松 + 楠木 + 古柏。**剑门关北面 + 嘉陵江上游沿岸**——是金牛道栈道最密集的一段（明月峡 + 七盘关一带栈孔今犹存）；**剑门关南面 + 普安县至剑州治**——山势渐缓、栈道由凿岩转沿坡。**剑门关至剑州治普安县约二、三十里**——是这条道上"关塞 + 州治"分立的典型布局。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

剑门关在唐人的回望里——是**蜀道千年咽喉 + 三国蜀汉北御故事 + 李白《蜀道难》笔下"一夫当关、万夫莫开"的关塞代名词**。

**剑门关与蜀道栈道的来路**：
- **传说期**——古蜀王时即有蜀道（蚕丛、鱼凫、杜宇、开明诸蜀王传说）；战国秦惠王灭蜀（前 316）传说"五丁开山"——开秦蜀间故道；
- **秦汉**——秦汉间金牛道（汉中-益州段）渐成；汉武帝时栈道大修；
- **三国蜀汉**——**诸葛亮北伐时筑栈道 + 整修剑阁**——《华阳国志》载诸葛亮"凿石架空、为飞梁阁道、以通行旅"；剑阁栈道形制由此定型；钟会、邓艾灭蜀（263）正面攻坚剑阁；
- **西晋**——**张载《剑阁铭》**："一夫荷戟、万夫趑趄、形胜之地、匪亲勿居"——是剑阁地形最有名的早期描写、立铭于剑阁道侧；
- **南北朝至隋**——剑阁沿线州县多次易主；
- **唐承前代**——剑门关属剑州（普安郡）、是金牛道咽喉。

**剑门关在唐诗中的位置**：**李白《蜀道难》**（约天宝初已传）——"剑阁峥嵘而崔嵬、一夫当关、万夫莫开、所守或匪亲、化为狼与豺"——是 755 时已流传的剑阁想象。**注**：**李白本人乃绵州彰明县青莲乡人**——蜀道是他少年走过的实路、剑门是他出蜀北上必经之关；《蜀道难》中的"剑阁"描写非纯文学想象、带实地经验。**但"一夫当关、万夫莫开" 是文学描写、不是关城形制的考工记**——不可据此推剑门关具体周长门数。

**唐剑州 + 普安郡的政治地位**：**天宝元年（742）改剑州为普安郡**——治普安县；**755 时是普安郡治普安县**；**剑门关在普安县境大剑山主脊**——距治普安约二、三十里；**乾元元年（758）复改普安郡为剑州**。**剑州 + 普安郡属剑南道**——剑南节度治益州、统益、蜀、彭、汉、眉、嘉、邛、雅、戎、泸、姚、巂、维、茂、剑等州；**剑门关由普安郡太守所辖、实际由军戍**——常驻戍兵约数百（具体数史载零散）。

### 此刻

天宝十四载这一年的剑门关——**关塞严整、戍兵守关、关吏验牒、栈道按例修缮**。

**关城状态**：盛唐 755 时剑门关主体仍立、有戍兵 + 关吏 + 驿馆；具体形制史载零散。**关城外**：金牛道南北延伸；**剑门关北面**——出关至利州（绵谷县）约百二十里、过明月峡 + 七盘关一带栈孔最密的一段；**剑门关南面**——出关至剑州治普安县约二、三十里、山势渐缓。

**755 时剑门关上的政治氛围**：**剑门关是平时关塞**——蜀地贡赋 + 文人入蜀 + 戍兵调动 + 公文传递皆过此关；**关吏对入蜀的官员、僧侣、商旅** 多有验牒；**北向汉中 + 长安方向的人流密度** 在金牛道上最大——蜀锦、蜀纸、蜀茶北上皆过剑门。**蜀锦北上长安**——自益州沿金牛道、过剑门关 + 散关、至长安；这是 755 时剑门关上最具象的物资流。

**755 时长安士人眼中的剑门**：李白《蜀道难》"剑阁峥嵘而崔嵬"已传咏长安、剑阁意象在盛唐诗中渐成蜀道代名词；但**长安士人多未亲走剑门**——剑门是想象中的"西南极险"；**李白本人是从剑门走出来的人**——他笔下的剑阁是实景。**杜甫尚未入蜀**——他将在四年后（乾元二年/759）自陇右南下入蜀、过剑门作《剑门》"惟天有设险、剑门天下壮"——但这是 755 之后的事。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（十一月初九即起）；**剑门关在 755 末仍按例运行 + 戍兵守关**——并未感受到东面战火。**剑南节度使杨国忠以宰相遥领、节度具体由副使代行**——剑门关的关务由普安郡太守日常监督、由军戍执守。**755 时剑门关的政治地位**：金牛道咽喉、剑南军政命脉的物理 anchor——但尚未承担"皇帝逃难必经"的历史角色。

### 此刻意味

对天宝十四载的剑州 / 普安郡人民来说——剑门关是**自家境内的关塞**：日常入蜀商旅必过、戍兵守关、关吏验牒；过往不绝、关税正征。对长安那一代士人来说——剑门关是**蜀道最险的物理 anchor + 李白《蜀道难》笔下的关塞代名词**：知其"一夫当关、万夫莫开"、未必亲走；剑门在诗中的位置——比在路上的位置更熟。

唐人尚不知道——再过 8 个月、**至德元载（756）七月玄宗西奔入蜀**——自长安出延秋门 + 经马嵬 + 扶风 + 散关 + 河池 + 梁州 + 利州 + **过剑门关** + 至剑州 + 绵州 + 汉州 + 至益州（成都）——**剑门关由日常关塞变成天子流离地的物理 anchor**；再过一年、**至德二载（757）冬玄宗自蜀北还**——复经剑门关北归；自此**剑门关在中国历史上的位置**——从蜀道一关塞、升为"盛唐衰转之关"、"天子西奔与北还的双向门户"。**今游人重走金牛道**——所见的是**明清重建关楼 + 1985 修复 + 翠云廊古柏 + 现代 108 国道 + 西成高铁**的复合景观——**盛唐人在 755 所见的那座剑门关**：千二百多年后**唐代原物几无**；明清关楼亦多次毁建；唯**大剑山主脊** + **狭口地势** + **栈孔遗痕** 仍是 755 时的同一道关塞所在。

## 子条目

### 大剑山 + 小剑山 + 剑门关的地理

**大剑山**——剑州境内主脊、自西北而东南横亘约七十里；峰峦相接、刀削斧凿；**剑门关在大剑山主脊中段狭口**——是这条山脊唯一的天然通道。**小剑山**——大剑山以南、二山主脊相连；**剑门关不在小剑山**——常被混淆。**大、小剑山合称"剑山"**——是剑州 + 剑阁地名的来源；**山自唐至今未改名**。

**大剑山主脊狭口的地形**：**南北两侧是千仞峭壁**——东西两山相夹如门——故名"剑门"；**关城正立狭口中央**——一道关墙横绝两山。**李白"剑阁峥嵘而崔嵬"** 写的是大剑山主脊的山势；**张载《剑阁铭》"一夫荷戟、万夫趑趄"** 写的是关塞的地形优势——皆非关城形制的考工记。

### 剑门关 vs 剑阁 vs 剑州（三个易混词）

唐人地名中"剑门 / 剑阁 / 剑州"三者层级清晰、不可混用：
- **剑州**——剑南道一州、治普安县（今四川剑阁县）；天宝元年（742）改剑州为普安郡、乾元元年（758）复改剑州；
- **剑阁**——**剑州境内栈道的总名**——自梁州（汉中）南入剑州、出剑州南至绵州一段栈道皆可称剑阁；亦作地名（指剑州境）；**剑阁不是某一具体关塞**；
- **剑门关**——**大剑山主脊狭口的具体关塞** —— 在普安县境、距治普安约二、三十里；是金牛道咽喉的物理 anchor。

**李白《蜀道难》"剑阁峥嵘而崔嵬、一夫当关、万夫莫开"** —— "剑阁" 是栈道总名 + 包含剑门关；"一夫当关" 是写剑门关的形势；二者在唐诗中常合用、但本 doc 分清三者层级。**今四川剑阁县** —— 是**剑州 + 剑阁** 的延续地名；**剑阁县境内的剑门关镇** —— 是**剑门关** 的延续地点。

### 三国蜀汉与剑门栈道

**剑门关的栈道形制始于三国蜀汉**：**诸葛亮北伐时筑栈道 + 整修剑阁**——《华阳国志·汉中志》载诸葛亮"凿石架空、为飞梁阁道、以通行旅"——是剑阁栈道形制最早系统记述。**蜀汉**：**剑阁是蜀汉北御曹魏的最后防线**——丞相诸葛亮多次出汉中北伐皆经此道；**蜀汉灭亡（263）**——钟会主力攻剑阁、邓艾偷渡阴平绕剑阁入蜀——剑阁正面不可破、由侧后阴平道破之；自此剑阁 / 剑门作为"蜀地北御门户"的角色固化下来。**唐人凭吊剑门** 多带三国蜀汉的余绪——这是剑门关在唐诗中的另一层历史厚度。

### 至德元载玄宗西奔过剑门关（756 七月）

**至德元载（756）七月**——**唐玄宗自长安出延秋门西奔**、经马嵬 + 扶风 + 散关 + 河池（凤州）+ 兴州 + 梁州（汉中）+ 利州（绵谷）；**七月**——**入剑门关 + 至剑州（普安）+ 绵州 + 汉州 + 至益州（成都）**。

**玄宗过剑门关的具体经历**：
- **路线**——自利州（绵谷）南行 + 过明月峡 + 七盘关 + 至剑门关；
- **过关日期**——史载零散（《资治通鉴》卷二一八记至德元载七月玄宗自梁州南下、月底至成都）；
- **此时玄宗的处境**——长安已陷、潼关已破、马嵬一夜杨贵妃缢死、杨国忠 + 一门皆死、皇太子李亨在马嵬北赴朔方分朝；**玄宗一行约千人**（六军 + 宗室 + 高力士等）；
- **剑门关上**——地方官与戍兵迎送, 具体细节史载未详；**自此玄宗一行入剑南** —— 自普安郡南下至成都约月余抵达。

**这是中国历史上最有名的"天子过剑门"** —— 剑门关由日常关塞变成"皇舆入蜀"的物理 anchor。**李白《蜀道难》"所守或匪亲、化为狼与豺"** —— 在 755 前作时尚是 文学描写、在 756 玄宗过剑门后即成 政治预言（中晚唐诗人多有此意）。

### 至德二载玄宗自蜀北还过剑门关（757 冬）

**至德二载（757）冬**——**玄宗自成都北还、复经剑门关**：
- **背景**——至德二载九月广平王李俶（即代宗）+ 郭子仪、李光弼复东京洛阳；十月复西京长安；**肃宗诏玄宗自蜀还京**；
- **路线**——自益州（成都）+ 汉州 + 绵州 + 剑州 + **过剑门关** + 利州 + 梁州 + 兴州 + 凤州（河池）+ 散关 + 扶风 + 至长安；**与 756 西奔路线大体相同、方向相反**；
- **过关日期**——十一月、十二月间过剑门 + 散关；**十二月底玄宗至长安**——肃宗在咸阳望贤宫迎驾；玄宗由太上皇身份还京；
- **自此**——**玄宗在长安兴庆宫流寓** 至宝应元年（762）崩；剑门关由"天子西奔关" 转为 "天子还京关"。

**玄宗两过剑门关**（756 西奔 + 757 北还）—— **剑门关在盛唐转衰一段中是唯一一座 双向亲历皇帝的关塞**。**散关亦为玄宗双向过**（西奔 + 北还皆经散关）——剑门 + 散关 二者在 蜀道一线 是 "皇舆双关" 的物理 anchor；本 doc 与 [散关](../transport/散关.md) 共构这一段叙事。

### 李白《蜀道难》与剑阁意象

**李白《蜀道难》**（约天宝前期已传咏长安）——**755 时已是流传的蜀道想象**：
- **"剑阁峥嵘而崔嵬、一夫当关、万夫莫开"** —— 写大剑山主脊狭口 + 剑门关地势；
- **"所守或匪亲、化为狼与豺"** —— 化用张载《剑阁铭》"形胜之地、匪亲勿居"；
- **"朝避猛虎、夕避长蛇"** —— 写蜀道沿途的险与异。

**李白本人是绵州彰明县青莲乡人**——蜀道是他少年走过的实路、**约开元十三年（725）二十五岁出蜀**——自绵州沿涪江北上 + 转入金牛道 + 过剑门 + 至梁州 + 再北出散关至长安；**剑门关是他出蜀北上的实路一关**——《蜀道难》中的剑阁描写非纯文学想象、带实地经验。**但"一夫当关、万夫莫开" 是文学描写、不是关城形制的考工记**——不可据此推剑门关具体周长、门数、戍卒数。**755 时长安士人传咏《蜀道难》**——剑阁在诗中比在路上更熟。

### 张载《剑阁铭》与"形胜匪亲"

**张载（西晋）《剑阁铭》**——立铭于剑阁道侧：
- **"一夫荷戟、万夫趑趄"** —— 后世化为李白"一夫当关、万夫莫开"；
- **"形胜之地、匪亲勿居"** —— 是 西晋人对剑阁地形的政治判断（地形如此险要、不可托非亲信）；
- **"惟蜀之门、作固作镇"** —— 写剑阁为蜀地北门户。

**张载《剑阁铭》是剑阁地形最早的系统铭文**——比李白《蜀道难》早约 450 年；**李白《蜀道难》"所守或匪亲、化为狼与豺"** 明显化用《剑阁铭》"匪亲勿居"——是唐诗化用西晋铭文的典型一例。**唐人凭吊剑门**——多带《剑阁铭》的余绪。

## 历史长河（755 → 今）

**中晚唐**　**756 玄宗西奔过剑门关** + **757 北还复过剑门关**——是剑门关在盛唐最高规格的两次使用；**乾元元年（758）复改普安郡为剑州**——剑州地名延续至今。**杜甫《剑门》**（乾元二年/759 冬自陇右入蜀途中作）——**"惟天有设险、剑门天下壮、连山抱西南、石角皆北向"**——是杜甫亲过剑门后所作；**自此"剑门天下壮" 成中国地理诗中的"剑门体" 之始**。**广明元年（880）黄巢入长安**——**僖宗西奔成都、亦经剑门关**——这是盛唐"玄宗 → 僖宗" 两次天子过剑门入蜀的连续；剑门关在唐末再次成"天子流离地"。

**五代两宋**　**前蜀（907-925）、后蜀（934-965）以成都为都**——剑门关是其与北方对峙的命脉；**北宋平蜀（965）**——王全斌循金牛道反向入蜀过剑门；**南宋偏安、关中失**——剑门关成宋金边境次要关塞（吴玠 + 吴璘兄弟主力守和尚原 + 仙人关、是故道一线、不在剑门）；**陆游南宋乾道八年（1172）入蜀**——途中过剑门作**《剑门道中遇微雨》"此身合是诗人未？细雨骑驴入剑门"**——是宋诗中剑门意象的高峰。

**金元明清**　**元代剑门关沿用**；**明洪武年间大修剑门关 + 翠云廊植柏**——今**翠云廊**（剑州至阆州一段、古柏夹道、约 24 公里、古柏八千余株）多为明初所植、亦杂唐宋古柏；**明万历 + 清代多次重修关楼**；**清乾隆间剑门关续修**——清末民初成川陕公路雏形。**注**：**今所见剑门关楼**——以明清重建为主体、不是唐代原物。

**民国至今**　**1935 年川陕公路（今 108 国道）开通**——大体沿金牛道古线、但避开了最险的几段栈道；**剑门关楼于公路建设中大部毁**；**1949 年后剑门关古关址列四川省重点文物保护单位**；**1985 年关楼重建**——以明清形制为蓝本；**1956 年宝成铁路通车** + **2017 年西成高铁通车**——半日达成都；**2009 年起金牛道（蜀道）启动 中国南方丝绸之路 / 蜀道 世界文化遗产联合申报**——尚在路上。**翠云廊古柏夹道**——是今金牛道沿线最完整的古道景观；**剑门关镇** + **翠云廊** + **明月峡栈孔** 三处构成今"剑门-翠云廊景区"。**唐人在 755 所见的那座剑门关**——千二百多年后**唐代原物几无**；唯**大剑山主脊** + **狭口地势** + **栈孔遗痕** 仍是 755 时的同一道关塞所在；玄宗 756 七月过剑门入蜀的那条路——今已成 108 国道 + 西成高铁 + 古蜀道遗存的现代复合走廊。
