# 井陉关

## 一句话

**太行八陉之第五陉、井陉口咽喉、河北 ↔ 山西天然孔道**——太行山中段一道狭长隘口、东通恒州 + 河北平原、西通太原 + 山西高原；**Tang 755 时属恒州/常山郡 井陉县境（西关）+ 获鹿县境（东关土门）**；**汉高祖三年（BC 204）韩信背水破赵于此**——中国战争史最深 anchor 之一；**755 这一刻仍是和平年代的塞名 + 驿路关塞**——半年后即将成河北-山西拉锯的物理通道。

## 形貌

井陉关在唐河北道**恒州/常山郡境**——具体在**井陉县**（今河北井陉旧治、井陉旧关一带）西界 + **获鹿县**（今河北鹿泉一带）东界；**距恒州治真定县（今河北正定）约九十里**。**地势**：**太行山脉自南而北横贯于此**——东面是华北大平原（自井陉口出山即落平原）；西面是山西高原（越井陉西入太原盆地东缘）；**关塞立太行山中一道狭长峡谷**——东西约百里、南北仅数里、两侧山岭夹峙；**这道峡谷即太行八陉第五陉之"井陉"**——井陉关即扼此陉咽喉。

**井陉关的地理战略价值**：**河北平原 ↔ 山西高原最直接的天然孔道**——自太原向东出承天军 (河东道太原府辖) / 井陉西口 + 入井陉，下东口即落恒州 + 河北平原；自恒州 / 获鹿向西入土门 + 走井陉，西出即至太原盆地。**唐时这是河东节度与河北方向之间最常用一线**——比北面的雁门-蔚州一线更平、比南面的滏口陉 + 太行陉一线更直；**自先秦至明清**——井陉一直是中原王朝 + 北方政权之间军政信使 + 商旅 + 调兵的核心通道。

**关名"井陉"的来历**：**"陉"** 即太行山脉中断之孔——太行八陉皆以此命名；**"井陉"** 名意见于《吕氏春秋》及《水经注》——一说**关沟形如井**（两侧山岭高峙、中间狭长低陷）；一说**地势如井底**。**关又名"土门关"**——这是井陉东口（今河北鹿泉境）的别称；**"井陉口"** 则是关沟入山处的统称；井陉关、土门关、井陉口三名同物异称、唐人皆用之。

**关城形制**：盛唐井陉关的具体形制 + 周长 + 门数史载零散——本 doc 不预设细节（**今所见井陉旧关 + 娘子关楼皆为后世重建、不是唐关原物**）。唐井陉关大体应有**关墙 + 关门 + 戍兵营 + 关吏馆**——但具体规模未见可信细节。**关塞外**（东口）：通获鹿 → 恒州治真定 → 定州 + 赵州 + 河北诸郡；**关塞内**（西口）：通承天军（即后世娘子关一带）→ 太原府 + 河东诸州。

**太行八陉中的位置**：太行山脉自南而北横贯关内道东缘 + 河东道东缘 + 河北道西缘——共有**八条**穿越太行的天然孔道（"陉"），自南而北为：**轵关陉**（济源境）—**太行陉**（沁阳-晋城）—**白陉**（陵川境）—**滏口陉**（邯郸-壶关）—**井陉**（鹿泉-井陉）—**飞狐陉**（涞源-蔚县）—**蒲阴陉**（易县-灵丘）—**军都陉**（昌平-延庆）。**井陉关即第五陉之咽喉**——位太行八陉中段、是河北 ↔ 山西最直接一陉。

**物候**：太行山中段——温带季风气候；冬寒夏温、关沟内常年风行；**绵蔓水（即今绵河、滹沱河支流）自西而东穿井陉口出山**——是关沟内的天然水源，亦是韩信背水之战的"水"。盛唐过关者循绵蔓水谷东出 + 西入——溪畔行旅的山色不息至今。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

井陉关在唐人的回望里——是**塞名 + 战名 + 千年关塞 三者叠合**。

**井陉的塞名最早见于《吕氏春秋·有始览》**——所载"天下九塞"中"井陉"为其一；与大汾、冥阨、荆阮、方城、殽阪、令疵、句注、居庸 八塞并立——**这是中国最古老的塞名序列之一**；井陉在战国时已是赵国西陲与山西高原之间的天然分界。

**汉高祖三年（BC 204）韩信背水之战**——这是井陉关最深的历史 anchor。**当时形势**：汉高祖刘邦命韩信、张耳率军东向破代 + 赵；**已破魏豹**（BC 205）+ **韩信先破代地、擒代相夏说于阏与**（BC 205 秋）——代亡后陈余退守赵境与赵王歇合军；**汉三年十月（BC 204 初）**——韩信 + 张耳率军（数万、一说三万）东出井陉口攻赵；**赵方**赵王歇 + 成安君陈余屯赵军二十万于井陉口西侧 + 设防严密。**陈余之子谋臣李左车献策**：**断汉军粮道 + 困汉军于井陉狭道**——陈余不从、谓"义兵不用诈谋"。**韩信探知陈余不从李左车计**——夜遣轻骑二千、人持一赤帜、绕至赵营侧后伏；又**自率万人出井陉口、背绵蔓水列阵**——汉军见水为阵、皆笑韩信不识兵法。

**次日决战**：陈余率赵军全军出击攻汉军 + 汉军前部佯败退入水阵；**赵军见水阵而轻视、倾巢追击**——**韩信"陷之死地而后生"**、汉军背水死战、赵军不能克。**与此同时**汉伏兵二千轻骑趁赵营空虚 + 拔赵旗 + 立汉赤帜二千于赵营；赵军回望、见自家营垒尽汉赤帜——**军心大溃**；**韩信前后夹击、大破赵军**——杀成安君陈余于泜水之上、俘赵王歇、广武君李左车被俘后受韩信厚礼问策。**自此韩信** + **张耳**定赵——汉四年韩信封为齐王（BC 203）。

**"背水一战"** 自此入中国战争史 + 兵法常言——"陷之死地而后生、置之亡地而后存"（《孙子·九地》语 + 韩信引以解此战）；**井陉口、绵蔓水、土门** 三名地随之 enshrined 千年。

**汉至北朝的井陉**：
- **东汉建武元年（25）**——**光武帝刘秀自河北南下入山西、再东向河北**——多次经井陉道；
- **三国曹魏**——井陉道是邺城（今安阳-临漳）通晋阳的西北线；
- **十六国 + 北朝**——后赵 / 前燕 / 后燕 / 北魏 / 北齐 / 北周 之间多次以井陉关易主；**北魏曾在井陉东口设戍**——但具体名目史载零散；
- **隋开皇 + 大业**——井陉道为隋东都至太原的官道之一；隋时已立**承天军**于井陉西口（今山西平定境、即后世娘子关一带）；隋末李渊自太原起兵南下入关中、未走井陉道——但李世民统军定河北则多次经此关。

**唐承隋制**——置恒州、领井陉县 + 获鹿县；**天宝元年（742）改恒州为常山郡**——治真定县；**井陉关属常山郡**、**井陉县**（西关）+ **获鹿县**（东关土门）合护一陉。**唐前期井陉关的政治军事地位**：是河东节度与河北方向之间的关塞——**驿马** + **军报** + **商旅** 来往不息；**承天军**则在井陉关西、属河东道太原府辖境——与井陉关合护此陉东西两端。**井陉关的守军**：由恒州地方军 + 唐廷派遣的关使共同守关；具体常额戍兵 + 防御工程史载零散。

**安史前夕**（即 755 这一刻）——再过半年安禄山起兵于范阳；**叛军南下河北一路陷郡**——常山郡（颜杲卿、颜真卿兄弟分守常山 + 平原）；**井陉关将成为河北 + 河东两道军事调度的关键节点**——但发生在 755 之后。**755 时井陉关仍是和平年代的关塞**——半年后即成安史拉锯战场之一。

### 此刻

天宝十四载这一年的井陉关——**关塞严整、守军按例屯戍、车马驿东通获鹿 + 恒州、西通承天军 + 太原**。

**关城状态**：盛唐 755 时井陉关主体仍立、有关门 + 戍兵营 + 关吏；具体形制 + 守军规模史载零散。**关沟内外**：东门通土门（今河北鹿泉一带）+ 获鹿县 + 恒州治真定县 + 河北平原；西门通承天军 + 太原府 + 河东诸州。**关使**为唐廷或常山郡太守派遣——具体在 755 这一刻谁任井陉关使、史无确载。

**755 时井陉关的政治氛围**：唐廷对河北 + 河东方向的安全意识在玄宗朝相对放松——**北疆已无大战二十余年**；安禄山节度使专制范阳已十余年；**唐廷未察觉安禄山有反意**——井陉关一带亦无特别戒备。

**755 末 + 757 的井陉关命运**：**安禄山十一月起兵于范阳**——主力沿幽州 + 河北 + 河南方向南下攻取洛阳 + 西攻潼关；**井陉关并非主攻方向**——但当时**常山太守颜杲卿**（颜真卿堂兄）在井陉东口的常山郡起兵反正、断叛军后路——**叛军必反扑常山 + 控制井陉 + 切断河北 ↔ 河东通讯**。**至德二载（757）**——史思明 + 安庆绪在井陉一带与唐军（李光弼、郭子仪 自太原东出井陉）反复争夺；**李光弼 嘉山之战** + **常山之战** 皆与井陉道直接相关。**井陉关因此成为安史之乱中河北河东战场的核心通道**——这是关塞唯一一段直接关涉天下大势的时间、但发生在 755 之后。

### 此刻意味

对天宝十四载的恒州人 / 井陉县人 / 获鹿县人来说——井陉关是**自家境内的关塞**：日常驿路过往、戍兵守关、商旅入关纳税。对太原 + 河东一带的士人来说——井陉关是**东门户**：东出 / 西入皆经此关；**盛唐诗中"井陉"偶有出现**（高适、岑参等边塞诗 + 怀古诗或涉），**但唐人提"井陉"主要在怀古韩信背水之战的语境**——是一个被战名所凝定的关塞。

唐人尚不知道——再过半年、安禄山将自范阳起兵、河北一路糜烂；井陉关将成为河北 + 河东之间军事调度的核心通道；常山太守颜杲卿将在关东起兵反正 + 旋被叛军反扑而死；**这道太行山中的狭长关沟**——半年后将见证盛唐最沉重的转折之一。**今井陉旧关 + 土门关** 古关皆不复唐时形制——**井陉旧关** 关楼主体为明清重建、**娘子关楼**（井陉西口、即唐承天军一带）为明清重建；**土门关**（井陉东口、今河北鹿泉境）今存有清代重修关楼遗迹。今人来井陉、所见的是**明清关楼 + 现代石太铁路 + 京昆高速** 共构一道**两千年关沟交通史的物理叠层**——**唐人在 755 所见的那座井陉关**：千二百多年后**形制已不可复原**。

## 子条目

### 太行八陉之井陉（第五陉）

**太行八陉**——自南而北八条穿越太行山脉的天然孔道：轵关陉、太行陉、白陉、滏口陉、**井陉**、飞狐陉、蒲阴陉、军都陉。**井陉为第五陉**——位太行八陉中段（中部偏北）；**地形**：自东口土门（今河北鹿泉境）入山——沿绵蔓水谷向西约百里——至西口承天军（今山西平定境、即后世娘子关一带）出山；**井陉关即扼此陉中段咽喉**、立东口与西口之间。**井陉自东而西穿太行山中段——是河北平原通往山西高原最直接的通道**——盛唐时是河北 ↔ 河东驿路的核心一段。

### 井陉关 ≠ 紫荆关 ≠ 倒马关（位置不同、各属一陉）

**唐人 + 后人常将井陉关 / 紫荆关 / 倒马关 / 居庸关并称为"太行四关"**——但四关位置各异、各属太行八陉中不同一陉：
- **井陉关** = 第五陉井陉之咽喉（河北鹿泉/井陉境）；
- **紫荆关** = 第七陉蒲阴陉之咽喉（河北易县境）；
- **倒马关** = 第六陉飞狐陉之咽喉（河北唐县境）；
- **居庸关** = 第八陉军都陉之咽喉（北京昌平境）；

**四关合护幽燕 + 河北平原西界**——明长城"内三关"（紫荆 + 倒马 + 居庸）即沿此格局；井陉关则因位置偏南、属"娘子关防线"另一段。**盛唐 755 时四关并存**——但唐时尚不以"四关"为成语。

### 井陉关 = 土门关 = 井陉口（同物异称）

**井陉关、土门关、井陉口** 三名在唐文献 + 后世皆指同一关塞，只是侧重不同：
- **井陉关**——以关塞行政名命名、是正式名号；
- **土门关**——以东口地名命名、是民间常用名（"土门"指东口低陷如门）；
- **井陉口**——以关沟入山处命名、是地理通称；

**唐人三名混用**——史志多用"井陉关" / "井陉口"、地方民间多用"土门"。**今河北鹿泉土门关** 即唐井陉东口、河北井陉旧关 一带为唐井陉西口至关塞段。

### 韩信背水之战与井陉口（BC 204）

**汉高祖三年（BC 204）**——**韩信、张耳** 率汉军数万东出井陉口攻赵；**赵王歇** + **成安君陈余** 屯军二十万于井陉口西阻汉军；**陈余之客谋臣李左车** 献"断汉军粮道" 计 + 陈余不从；**韩信**夜遣二千轻骑伏赵营侧后 + 自率万人**背绵蔓水列阵**；次日决战、**陷之死地而后生** + 汉军死战不退、伏兵拔赵旗 + 立汉赤帜二千、赵军军心溃、**汉军前后夹击、大破赵军**——杀陈余于泜水之上、俘赵王歇、广武君李左车被俘后受韩信厚礼问策。**"背水一战" 自此成中国兵法常言**——是井陉关历史上最深的一次战名。**绵蔓水**（即今绵河、滹沱河支流）即背水之战的"水"——**今河北井陉境绵河沿岸**有"韩信祠"+"背水阵遗址"+"千古战场"碑等纪念物（皆后世立、不是唐时）。

**注**：背水之战发生于**汉高祖三年（BC 204）**——非 BC 205（BC 205 是破魏豹 + 韩信破代地、擒代相夏说于阏与之年；BC 204 才是破赵之年、即背水之战）；**赵方**主帅是**成安君陈余**（赵相、摄赵政）+ **赵王歇**——非"陈余 = 赵王"（陈余实为赵相成安君、张耳之死敌；阏与所擒者为代相夏说、非陈余）；**李左车**是赵广武君 / 谋臣——战后受韩信厚礼问策；**张耳**是韩信副将（汉初常山王、与陈余早年刎颈交、后为汉破赵助韩信）。

### 承天军 + 娘子关——井陉关西端的另一道关塞

**井陉关西口**——即关沟西出太行山处；**唐时此处设承天军**（隶河东道太原府）——是井陉西端的军事据点；**承天军 + 井陉关 合护井陉一陉**——一关一军、东西两端。**承天军遗址**在今山西平定县东南、即后世娘子关一带；**"娘子关"**之名最早见于明代——传源唐代平阳公主（李世民妹、唐高祖之女）率"娘子军"屯此守关（**注**：此说见于晚出地方志、不见唐代文献、非可坐实之史实；本 doc 不预设李世民妹平阳公主必在此关屯军）。**盛唐 755 时尚不存在"娘子关"之名**——西口处的关塞名为承天军。

### 安史之乱与井陉关（755 末 - 757）

**天宝十四载十一月（755 末）**——安禄山起兵于范阳；**常山太守颜杲卿**（颜真卿堂兄）在常山郡（井陉东口）起兵反正、**断叛军后路**——是河北最早起兵反正的太守之一。**叛军反扑常山**——史思明率叛军围常山；**至德元载（756）正月**——**颜杲卿城破被执、不屈死于洛阳**（即"颜杲卿之死" + 颜真卿《祭侄文稿》之背景）；**井陉关一度落入叛军**。**至德二载（757）**——**李光弼 + 郭子仪**自太原东出井陉、**嘉山之战** + **常山之战** 复定河北、**井陉关复归唐廷**。**这是井陉关在盛唐唯一一段直接关涉天下大势的时间**——但战事密集在 755 之后、不在 755 这一刻。

### 关沟内的水（绵蔓水）

井陉关所在的太行八陉第五陉——**沿绵蔓水谷展开**；绵蔓水自西而东穿关沟、出土门入华北平原、汇入滹沱河；**这条水即韩信背水之战的"水"**——汉军背水列阵 + 死战不退、赵军倾巢追击 + 落败——**绵蔓水从此成为兵法 anchor 之水**。**盛唐过关者**——必沿此水行走；溪水冬不冻（关沟内地温较高）、夏不涸——是关沟驿道的物理 anchor。**今关沟内仍有此水**（今名绵河）——但水量较唐时减少；溪两岸有明清关楼 + 民国石太铁路（1907 通车）+ 京昆高速 + 现代高铁——共构一道**两千年关沟交通史的物理叠层**。

## 历史长河（755 → 今）

**中晚唐**　**安史之乱（755-763）期间**——井陉关是河北 + 河东两道军事调度的核心通道；**757 复归唐廷**、763 安史之乱结束。**中晚唐井陉关**——常被河朔藩镇与唐廷反复争夺；具体战事史载零散。

**五代**　**五代间井陉关多次易主**——后梁、后唐、后晋、后汉、后周皆控井陉关；**石敬瑭 938 割燕云十六州予契丹**——燕云不含井陉一带、井陉关随中原政权流转。

**北宋**　**北宋初**——井陉关属真定府（恒州升府）；**真定府** 是北宋河北西路治所、北门户。**北宋 + 辽** 在井陉关方向少有直接战事（澶渊之盟后北疆稳定）。

**金元**　**金时井陉关属真定路**；**金末 1213 年成吉思汗大军南下**——曾经井陉道破金；**蒙元定鼎后**——井陉关归元；**元至元二年（1265）真定路** 沿用。

**明**　**洪武元年（1368）明军北伐**——收复井陉关；**明洪武 + 嘉靖年间**——重修井陉旧关 + 娘子关楼（明嘉靖二十一年/1542 重修娘子关）；**明井陉关 + 娘子关合护一陉**——属山西镇辖境（西关）+ 真定府辖境（东关）。

**清**　**清初井陉关沿明制**；**清中后期渐失军事意义**——清廷与蒙古结盟、塞外不再是边境；井陉关、娘子关渐成驿路而非边关。

**民国至今**　**1907 年正太铁路通车**——自正定经井陉至太原、即今石太铁路前身、沿绵河 / 井陉关沟铺设；**井陉旧关 + 土门关 + 娘子关楼** 分立国保 / 省保单位；**1980s 起娘子关 + 井陉旧关 开放为旅游景区**；**今石太铁路 + 京昆高速 + 石太高铁** 皆经井陉关沟——延续两千年关沟通道传统。**今游人来井陉**——所见的是**明清关楼 + 现代铁路 + 高速 + 高铁** 共构一道关沟、千年层叠。**唐人在 755 所见的那座井陉关**——千二百多年后**形制已不可复原**：盛唐井陉关的物理终点是明清重修工程与现代交通的层叠。
