# 须弥山石窟

## 一句话

**萧关道侧的西陲佛窟**——原州平凉郡平高县西北五十里、六盘山北段的赭红色丹霞崖上凿出的石窟群；**北魏太和年间（477-499）始凿、历西魏、北周、隋、唐五代续凿**；**唐高宗、武则天朝（650-705）开凿盛、留下二十米六的弥勒倚坐大佛（今第 5 窟）**——755 时是萧关西出河西、陇右间的佛教 anchor；**再过 7 年（762-763）吐蕃东扩、原州陷蕃、须弥山随之入蕃**。

## 形貌

须弥山在唐**关内道原州平凉郡境**——具体在**平高县治西北约五十里**（今宁夏回族自治区固原市西北原州区须弥山乡，距固原城约 55 公里）。**地势**：**六盘山北段余脉的一片红砂岩丘**——南连六盘山主脊（向南连陇山），北接清水河上源谷地；山势不高、海拔约 1700-2000 米，但**赭红色砂岩崖壁壁立**——是石窟开凿的天然崖面；山形圆浑、远望若一座座小堆叠的"山"——**唐人因山形似佛经"须弥"得名**（梵文 Sumeru 音译；须弥是佛教世界中心山名）。

**石窟分布**：石窟开凿在山间数处崖壁上——**今知者 162 个洞窟 + 千余尊造像**（含残）、**沿山岭五公里崖面分布**。**今分八区**：大佛楼区、子孙宫区、圆光寺区、相国寺区、桃花洞区、松树洼区、三个窑区、黑石沟区——盛唐 755 时具体分区名史载零散、本 doc 不预设盛唐分区名；但**石窟群规模 + 主要崖面位置** 自北魏-唐已基本定型。

**第 5 窟大佛**：**今须弥山最大窟**——窟外为一木构窟檐（明清重建）；窟内**一尊弥勒倚坐大佛**——通高约 **20.6 米**（一说 19 米）；佛身石胎泥塑 + 施彩；**唐高宗、武则天朝（650-705）造**（宿白 + 文物出版社《须弥山石窟内容总录》通说）；具体年月无题记、本 doc 不预设单年。**这尊大佛盛唐 755 时已立约半个世纪** —— 是须弥山唐代造像的代表作 + 与莫高窟北大像（武则天朝 695 起、约中宗睿宗朝 705-710 间成）、龙门奉先寺卢舍那（上元二年/675 成）大略同期。

**北魏窟与北周窟**：须弥山现存北魏窟数量较少 + 散布数区；**北周窟为高峰** —— 第 51 窟（北周）等是中国北周石窟的精品；**唐窟数量最多** —— 与北周窟一同构成须弥山的核心。本 doc 不细列窟号（非盛唐唐人所用编号；今编号为 1980s 之后定）；但盛唐人 755 来到须弥山所见——**北魏 + 北周 + 唐 三代造像并立、唐代仍在续凿**。

**形制**：北魏 + 北周窟多为**中心塔柱窟**（沿袭印度旧制 + 中央留塔柱、四面开龛供像）；隋唐窟多为**殿堂窟**（方形覆斗顶 + 正壁开龛 + 三壁绘经变）；唐代第 5 窟大佛窟为**大像窟**（窟体随佛身高度凿至崖顶 + 外覆木构窟檐）。**造像材质**：与莫高窟同——红砂岩可凿不可雕、故造像多为**石胎泥塑**（先在岩芯上粗凿轮廓 + 敷泥层细塑 + 最后施彩）；壁画亦先抹一层细泥地仗、再绘。

**物候**：六盘山北段红砂岩丘——**夏温较高、冬寒；春秋多风**；山间林木相对较盛（盛唐时较今为盛）、崖面外是半干旱黄土草原。**寺院**：盛唐石窟外多有寺院 + 僧寮 + 客舍——须弥山为原州西北重要佛教中心、当时寺院规模史载零散、本 doc 不预设具体寺名（"圆光寺"之名为明清后续称）。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

须弥山在唐人的回望里——是**北魏开凿的约 278 年佛山 + 北周 + 隋唐三百年续凿的西陲圣地**。

**须弥山的开凿史**：

- **北魏太和年间（477-499）**——孝文帝时期始凿。《魏书·释老志》载北魏太和年间各地凿窟造像之盛——须弥山虽未直名、但其北魏窟的造像风格与云冈、龙门北魏窟同源、本 doc 取通说"太和始凿"。具体始凿年月无题记、不预设单年。**北魏窟数量较少 + 散布数区**——是须弥山开凿之始。
- **西魏 + 北周（535-581）**——**北周为须弥山开凿第一高峰**：北周武帝、宣帝朝（560s-580s）于须弥山大开窟龛；**第 51 窟（北周）等是中国北周石窟代表作**——造像体量大、面容方圆、衣纹厚重；这一段须弥山与同期麦积山、炳灵寺一道、构成北周时期的西陲佛窟群。**北周末武帝灭佛（建德三年/574）一度波及**——但须弥山位于西陲、灭佛之厄较中原略缓。
- **隋（581-618）**——开皇、大业年间续凿；隋窟数量较北周为少。
- **唐初至盛唐（618-755）**——**唐高宗、武则天朝（650-705）为须弥山开凿第二高峰**：**第 5 窟二十米六弥勒倚坐大佛**即此期所造；**武则天崇弥勒 + 好造大像**——盛唐西陲诸窟（莫高窟北大像、须弥山第 5 窟大佛、麦积山部分唐窟）皆此风气下的产物。**玄宗朝（712-756）须弥山续凿、数量较武则天朝略减**；**到 755 时须弥山唐窟已立** ——是西陲与河西、陇右诸窟同盛的格局。

**唐人对须弥山的认知**：**这是原州 + 平凉郡境内的佛教 anchor**——盛唐唐人往来于长安 + 灵州 + 凉州 + 河西诸州者，经萧关道过原州时，多于须弥山停驻礼佛。须弥山虽不及莫高窟之名盛、亦不及龙门之距京近、但**作为萧关道沿线的佛窟**——它是盛唐西陲佛教网络的重要节点。

**与同期西陲诸窟的关系**：盛唐西陲佛窟一线——**莫高窟（沙州）+ 榆林窟（瓜州）+ 河州/安乡郡炳灵寺 + 麦积山（秦州）+ 须弥山（原州）**——是河西陇右一带的佛窟主轴。**须弥山在这条主轴最东端**——是中原方向（长安）通河西陇右最先遇到的西陲佛窟。盛唐 755 时这五窟皆在续凿（莫高、麦积、须弥皆有盛唐窟）——五窟连成一线、是盛唐西陲佛教视觉体系的核心。

**萧关道关系**：**须弥山在萧关道沿线**——自长安西北行经邠州 + 泾州 + 入原州境 + 过平凉县 + 至平高县 + 西北行约 50 唐里 + 至须弥山；自须弥山再西北行 + 接萧关 / 灵州 / 凉州方向。**盛唐使节 + 商旅过萧关道者**——多于平高县或须弥山停驻；**回纥使节南下长安者 + 唐使节北上回纥者**——途中礼佛或留题、是萧关道沿线佛教文化的物理 anchor。

### 此刻

天宝十四载这一年的须弥山——**石窟群整、北魏 + 北周 + 唐三代造像并立、寺院 + 僧寮 + 客舍俱在、香火盛**。

**石窟状态**：盛唐 755 时须弥山——**北魏 + 西魏 + 北周 + 隋窟皆立**（约二百余年到三百年前所凿）+ **唐代第 5 窟二十米六弥勒倚坐大佛已立约半个世纪**（武则天朝 650-705 间造）+ **玄宗朝唐窟仍在续凿**。**香客 + 僧人 + 信徒** ——多自原州城（平高）+ 平凉县 + 长安方向来。

**寺院状态**：须弥山诸窟外**多有寺院** + 僧寮 + 客舍——盛唐时寺院具体名史载零散、本 doc 不预设盛唐寺名（"圆光寺"之名为明清后续称）；但须弥山为原州境内佛教中心——**寺僧数量较多 + 受原州州署 + 平高县署 + 当地望族供养**。

**755 时的政治氛围**：**须弥山是萧关道沿线 anchor**——盛唐使节 + 商旅 + 戍兵 + 香客往来不绝。**与回纥的关系**：744 后回纥代东突厥据漠南；**755 时唐回纥关系仍属和好**——回纥使节南下长安、唐使节北上回纥的部分客流过萧关道、亦或停驻须弥山。**安西、北庭都护府方向**——经河西走廊补给的部分使节亦走萧关道、过须弥山西北行。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳（在今北京）**——**自范阳南下取洛阳**；须弥山方向未受直接兵燹；但**756 后河西陇右兵东调救京**——朔方军（治灵州）+ 河西军 + 部分安西军过萧关道入关中救京——须弥山由此进入安史战争的物理网络周边、但未受直接冲击。

### 此刻意味

对天宝十四载的原州 / 平高县人来说——须弥山是**自家州境西北的佛山**：日常香火 + 礼佛 + 寺僧讲经 + 唐窟续凿。对长安那一代士人来说——须弥山是**萧关道沿线的西陲佛窟**：盛唐 长安士人对须弥山的诗咏较莫高、麦积为少（本 doc 不预设具体盛唐人题咏须弥山的存例）；但**萧关道沿线诸窟 + 萧关 + 河西陇右**——这套地理体系是盛唐边塞诗 + 使节诗 + 西游记体的核心物理网络之一。

须弥山 755 时的状态——**北魏 + 北周 + 唐三代造像并立 + 唐窟续凿 + 寺院香火盛**。**这是须弥山的盛期**——也是西陲佛教整体的盛期。

唐人尚不知道——再过 7 年（762-763），**吐蕃东扩 + 大举攻唐 + 陷原州 + 平凉 + 萧关一带**；**广德元年（763）十月吐蕃 20 万骑入长安**——之前已经先取关中西北诸州、须弥山随原州陷蕃；自此**须弥山一带成唐蕃边境 + 入吐蕃辖境**。**唐廷再无力收复**——须弥山的盛唐开凿期由此终结；**中晚唐须弥山渐荒废**——寺院香火大减、唐窟续凿戛然而止。盛唐人 755 年所见的那座须弥山——**七年后即将成"敌境"、十余年后即将成"荒山"**。

## 子条目

### 须弥之名（梵文 Sumeru 与佛山命名）

**"须弥"——梵文 Sumeru 音译**；佛教世界观中，须弥山是世界中心之山——山高八万四千由旬、四宝构成、日月环绕、四大部洲围绕。**汉译佛经自后汉至魏晋已译"须弥"为山名**——东晋佛驮跋陀罗译《华严经》、姚秦鸠摩罗什译《大智度论》 等皆载须弥山。

**固原须弥山之得名**——**因山形似佛经须弥而得名**：山势圆浑、一层一层堆叠如须弥四宝层叠；**北魏 + 北周开凿者** 选此山凿窟——既因山形契合佛山想象、亦因红砂岩崖面适宜开窟。**本 doc 取此通说**——固原须弥山因山形似须弥得名（非反向：非须弥经名取自此山）。**唐人 755 时所称的"须弥山"**——既是地理实体、也是佛山符号；与"五台" "峨眉" 等并立、是中国佛教山名体系的一员。

### 北魏太和年间始凿（477-499）

**北魏太和年间（孝文帝时 477-499）始凿**——是须弥山开凿之始。《魏书·释老志》载北魏太和年间各地凿窟造像之盛——**云冈石窟 + 龙门石窟 + 须弥山石窟**——皆此期开端或承前续凿。须弥山虽未直名于《释老志》——但其北魏窟的造像风格 + 衣纹 + 面容——与云冈、龙门北魏窟同源、本 doc 取通说"太和始凿"。

**北魏窟特点**：**中心塔柱窟为主**（沿袭印度旧制 + 中央留塔柱、四面开龛供像）；造像面容方圆、衣纹厚重、身体比例略瘦长——是北魏中后期"秀骨清像" 前的风格。**北魏窟数量较少**——散布须弥山数区；**这是须弥山的开凿之始**——比莫高窟（前秦建元二年/366 始凿）晚约百年、比云冈（北魏文成帝和平元年/460 始凿）晚约二三十年。

**注**：**"太和" 是北魏孝文帝年号 + 477-499**——不要与同名其他朝代年号混（如东汉桓帝太和、东吴会稽王太和、北周 / 隋亦无"太和" 年号；本 doc 严守北魏太和 = 孝文帝）。**北魏始凿不要写"北周始凿"**——北周续凿、非始凿；本条所记是"始凿" 这一节。

### 北周开凿高峰（6 世纪后期）

**北周（557-581）为须弥山开凿第一高峰**——**北周武帝 + 宣帝朝（560s-580s）于须弥山大开窟龛**；**第 51 窟（北周）等是中国北周石窟代表作**——窟内大佛三尊立像 + 体量大 + 面容方圆 + 衣纹厚重；与同期麦积山北周窟 + 炳灵寺北周窟一道、构成北周时期的西陲佛窟群。

**北周开凿的政治背景**：**北周帝室宇文氏崇佛 + 兴佛事**；**北周武帝建德三年（574）灭佛**（"周武毁佛")——一度波及须弥山；但须弥山位于西陲、灭佛之厄较中原略缓 + 部分窟龛仍续完工；**宣帝、静帝朝（579-581）复佛**——须弥山续凿。

**北周开凿在须弥山的位置**：北周窟散布须弥山数区——非集中于一处；这与同期麦积山、炳灵寺的北周窟开凿模式相近——是西陲佛窟在北周时期的一种"分散开窟、集中信奉"格局。

### 唐代第 5 窟大佛（武则天朝、20.6 米弥勒倚坐）

**第 5 窟是须弥山最大窟**——窟内**一尊弥勒倚坐大佛**——通高约 **20.6 米**（一说 19 米；本 doc 取宿白 + 文物出版社《总录》通说）。**断代**：**唐高宗、武则天朝（650-705）造**（宿白 + 文物出版社《须弥山石窟内容总录》通说）；具体年月无题记、本 doc 不预设单年。

**与同期诸大像的关系**：

- **莫高窟北大像（第 96 窟）**：通高 35.6 米；武则天证圣 / 延载二年（695）前后发愿、约中宗、睿宗朝（705-710 间）成；
- **龙门奉先寺卢舍那大佛**：通高 17.14 米；上元二年（675）成；
- **须弥山第 5 窟大佛**：通高约 20.6 米；高宗、武则天朝（650-705）造；
- **乐山大佛**：通高 71 米；开元元年（713）始凿、贞元十九年（803）成；755 时尚在凿——本 doc 不入本条。

**这些大佛**——皆**武则天崇弥勒、好造大像**风气下的产物（武则天自托弥勒下生、690 改国号为周）；**或与此风气余韵相接**（莫高北大像 + 乐山大佛皆此余韵）。**第 5 窟大佛盛唐 755 时已立约半个世纪**——是须弥山唐代造像的代表作 + 是萧关道沿线最大的佛像。

**注**：**本 doc 取"高宗武则天朝"通说**——不预设具体年月；**第 5 窟具体凿造年代**——史源无明确题记、宿白基于造像风格断代；**唐代大佛 = 武则天朝 + 须弥山第 5 窟**——不要把须弥山与麦积山、炳灵寺三窟混（三窟在不同地：麦积山在秦州、炳灵寺在河州、须弥山在原州；三窟皆有唐窟、但各自独立、唐代大佛在第 5 窟须弥山而非他处）。

### 萧关道沿线的佛教 anchor（与莫高、麦积、炳灵寺、克孜尔的体系）

**盛唐西陲佛窟主轴**——自西向东：

- **克孜尔石窟（龟兹 / 安西大都护府境）**——西陲最远、龟兹文化中心；
- **莫高窟（沙州 / 河西节度境）**——河西西端、距长安最远的"汉地佛窟"；
- **榆林窟（瓜州 / 河西节度境）**——莫高之东、与莫高姐妹窟；
- **河州/安乡郡炳灵寺（陇右节度境）**——黄河上游、佛教 + 道教并存；
- **麦积山（秦州 / 陇右节度境）**——天水东南、北朝隋唐五代续凿；
- **须弥山（原州 / 关内道境）**——萧关道沿线、西陲最东端、连入长安方向；
- **龙门（洛阳郊）**——东都郊外、中原佛窟巅峰；
- **云冈（云州 + 朔州）**——北朝旧都郊外、北魏佛窟巅峰。

**须弥山在此体系中的位置**——**最东端的"西陲佛窟"**：再向东即关中、即长安、即龙门——已属"中原"。**须弥山是萧关道客流过原州时所见的第一座西陲式佛窟**——也是出长安西北行的客流最先遇到的、与西陲佛教视觉体系相同语汇的石窟。**这是须弥山在盛唐佛教地理上的独特位置**——既西陲、又近关中；既萧关道 anchor、又长安通灵凉河西的物理节点。

### 与平高县 + 原州 + 萧关的距离

**须弥山距诸城**：

- **平高县治（今宁夏固原市原州区）西北约 50 唐里**——是原州州治境内最重要的佛教点；
- **原州城（即平高县城）**——盛唐时为原州 / 平凉郡治、关内道辖；
- **萧关**——盛唐通认在平高县境六盘山东麓（今固原西南三关口一带），距须弥山约 80-100 唐里（南向）；**须弥山与萧关同属平高县境** + 同在六盘山一线、是萧关道沿线两处地标。
- **长安东南通约 600 唐里**——经邠州 + 泾州 + 原州萧关一线；
- **灵州（朔方节度治）西北通约 400 唐里**——萧关道北端；
- **凉州（河西节度治）西北通约 1200 唐里**——经萧关 + 河西走廊东段。

**唐人在须弥山行旅经历**——多为：**自长安出 + 西北行 + 经邠泾原 + 至平高 + 西北行 50 唐里 + 至须弥山礼佛 / 停宿 + 再西北行 + 接萧关 + 入灵凉河西方向**。**这是萧关道与须弥山的物理关系**——须弥山是萧关道在原州境内最重要的佛教节点。

### 762-763 吐蕃陷原州 + 须弥山入蕃

**安史 + 后续年间**——河西陇右兵东调救京 + 西陲守备日空 + **吐蕃东扩**：

- **756 起**：陇右、河西守备日空；
- **762-763 间**：**吐蕃陷原州 + 平凉 + 萧关一带**；**须弥山随原州陷蕃**；
- **广德元年（763）十月**：**吐蕃 20 万骑入长安**（《资治通鉴》卷二二三）；
- **此后**：须弥山一带为吐蕃辖境约百年；**直至宣宗大中年间（847-859）唐廷部分收复陇右诸州**——但须弥山一带的寺院 + 香火 + 唐窟续凿已断绝盛唐脉络。

**入蕃后的须弥山**：吐蕃亦信佛——但**吐蕃佛教自有其源流**（自尼泊尔 + 印度 + 唐传入）；**入蕃后的须弥山** 是否续凿、何时续凿——史源零散、本 doc 不预设细节。但**盛唐 755 时的那座须弥山** —— 7 年后即失唐控、自此 "脱离盛唐物理网络"；唐人 755 时尚不知。

## 历史长河（755 → 今）

**中晚唐**　**762-763 间须弥山随原州入蕃**；**763 后约百年为吐蕃辖境**；**大中年间（847-859）唐廷部分收复陇右诸州**——但须弥山一带的寺院 / 唐窟续凿已断绝盛唐脉络。**晚唐 + 五代须弥山已不复盛唐之 importance**——香火大减、唐窟续凿戛然而止；中晚唐至五代间须弥山渐荒废。

**北宋 + 西夏**　**北宋初须弥山一带为宋夏边境** —— 党项李继迁 + 元昊建立西夏（1038）后，须弥山一带长期为宋夏拉锯前线；**西夏崇佛 + 西夏文化在须弥山的影响史源略异**——西夏在须弥山是否大修石窟、本 doc 不预设细节。**北宋仁宗朝在原州 + 镇戎军（今固原）一带大筑堡寨**——须弥山附近边事日繁、寺院香火渐复。

**金元明清**　**金代须弥山属金境**；**元代设开成路 + 镇戎州**——须弥山续作佛教点；**元代蒙古崇藏传佛**——须弥山是否受藏传佛教影响 史源零散、本 doc 不预设。**明景泰三年（1452）立固原州**——是今"固原"地名之始；**明嘉靖年间在固原一带筑长城（明长城西段）**——须弥山一带成明长城关塞 + 佛教点的并立格局；**明清在须弥山续修寺院**——**"圆光寺" "相国寺" 等名为明清后续称**（盛唐时无此名）；明清续凿石窟较少 + 多为修缮 + 重塑像 + 重彩。

**民国至今**　**1949 年后宁夏回族自治区 + 固原市设立**；**1982 年 2 月 23 日国务院公布第二批全国重点文物保护单位**——须弥山石窟列入。**1980s-2010s 多次考古勘察 + 部分窟修缮**——**今所见须弥山** ：**162 个洞窟 + 千余尊造像** + **赭红色丹霞崖五公里崖面分布** + **第 5 窟二十米六大佛立、明清木构窟檐覆**。**唐人在 755 所见的那座须弥山**——千二百多年后**唐窟主体仍立** + **第 5 窟大佛仍立**；但**寺院 + 香火 + 续凿** 已与盛唐迥异。**须弥山的位置 + 红砂岩崖 + 第 5 窟大佛**——一千二百多年未易；**萧关道的物理路径**——一千二百多年未易；**唐人 755 时所礼的那尊弥勒倚坐大佛**——今仍在崖间静坐。

**注**：**1982 年 = 第二批全国重点文物保护单位**——不要写"1961 第一批"；**1961 第一批** 含莫高窟、龙门、云冈、麦积山等；**须弥山列 1982 第二批**——本 doc 严守。
