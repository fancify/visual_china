# 建康

## 一句话

**六朝古都的废墟**——孙吴建业（229 称帝）→ 东晋建康（建邺 313/314 因避愍帝司马邺讳改建康；317 司马睿称晋王 / 318 即皇帝、都建康）→ 宋齐梁陈承之——**六朝凡 333 年（229-589）江左王业的物理 anchor**；**隋开皇九年（589）灭陈、隋文帝诏毁建康城邑宫室、并平荡耕垦**——**至 755 时已废 166 年**；**755 时正名润州丹阳郡辖江宁县（润州/丹阳郡治丹徒县）**——城垣余夯土、台城 + 石头城 + 乌衣巷遗踪散于田野——是唐人凭吊最深的"前朝旧都"。

## 形貌

建康废墟在唐江南东道**润州（丹阳郡）下辖江宁县**境内——具体在今**南京市秦淮河北岸的台塬地**上。**地势**：长江南岸、紫金山西、秦淮河绕南而过——东、北以紫金山 + 覆舟山 + 鸡笼山为屏；西以石头山（清凉山）扼长江；南以秦淮河绕——是孙吴 229 选址时所谓"龙盘虎踞"的天然形势。

**755 时所见的建康旧城**——主体已**毁 166 年**、城垣多为夯土残基；但六朝宫城 + 礼制建筑 + 街坊遗迹尚可在田野中辨识：

- **台城**（六朝宫城；东晋至陈四代续筑）——在今南京玄武湖南、覆舟山东南；**南北约 2 公里、东西约 1.5 公里**；755 时台城宫墙残基仍在、内中宫殿基址多为农田与树林覆盖；
- **石头城**（孙吴建于 212 年、城西军事要塞）——在今南京清凉山一线；**扼长江、控建康西门**；755 时城墙残基仍可见、内中已无戍守；
- **乌衣巷**（东晋王导、谢安等高门宅区）——在今南京秦淮河北、夫子庙一带；755 时已 4 世纪、王谢宅早不存、街巷依稀；
- **覆舟山 + 鸡笼山 + 青溪一线**——六朝皇家苑囿（华林园等）背景；755 时山仍在、苑园基址散没于林莽；
- **秦淮河**——六朝建康繁华核心、河上桨声夜唱所在；755 时水道仍流、但两岸已是江宁县的乡野；
- **朱雀桥 / 朱雀航**（秦淮河上的浮桥、六朝建康正南门外的咽喉）——755 时旧址尚可辨、浮桥已撤；中晚唐刘禹锡"朱雀桥边野草花"即此地物理 anchor。

**唐江宁县新城**——隋开皇十年（590）移江宁县治于**丹阳故地**（建康旧城西南数里、今南京白下区一带）；**755 时江宁县治在新县城内（隶润州/丹阳郡，郡治丹徒县）**；旧建康城是新县城的近郊废墟、相距数里。**唐人来江宁**——多自新县城出城、北行数里即至旧建康台城遗址。

物候：江南东道腹地、亚热带季风气候——四时分明、夏热冬温、雨多、秦淮春雨、紫金山秋枫。**建康废墟在 166 年间风化**——夯土城墙渐为草木覆盖、宫城基址多为农田、石头城仍是江岸最显的物理标识。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

建康在唐人的回望里——是**江左王业 333 年的物理纪念 + 六朝兴亡的现场 + 中国凭吊文化的核心 anchor**。

**六朝建都建康 333 年（229-589）**：

- **孙吴（229-280）**——**孙权于黄龙元年（229）自武昌徙都建业**——是建康作都的开端；**孙吴 52 年四帝（孙权 / 孙亮 / 孙休 / 孙皓）**；**280 西晋灭吴**——建业并入西晋（建兴元年/313 因避愍帝司马邺讳改名建康）。
- **东晋（317/318-420）**——**永嘉之乱（311）后晋室南渡**、**晋元帝建武元年（317）称晋王于建康、太兴元年（318）即皇帝位**（建邺已于建兴元年/313 因避愍帝司马邺讳改名建康）；东晋 11 帝 103 年皆都建康——**这是江左衣冠南渡的物理 anchor**、**王导、谢安、桓温、刘裕等门阀皆居此**。
- **南朝宋（420-479）**——刘裕篡晋立宋；**宋 8 帝 59 年**都建康。
- **南朝齐（479-502）**——萧道成篡宋立齐；**齐 7 帝 23 年**都建康。
- **南朝梁（502-557）**——萧衍篡齐立梁；**梁武帝在位 48 年是南朝最长**；**梁 4 帝 55 年**都建康；**侯景之乱（548-552）摧毁建康**——是建康作都期间最大一次浩劫、台城被围 130 余日、梁武帝饿死台城。
- **南朝陈（557-589）**——陈霸先（南朝唯一寒族出身的开国皇帝）立陈；**陈 5 帝 32 年**都建康；**589 隋灭陈、统一南北**。

**建康（建业）六朝累计建都约三百年**——是中国南方最长的一段建都史；**长安 + 洛阳之外的第三都**；**280 西晋灭吴至 317/318 东晋建都之间约 37 年（西晋时期）建邺非都、降为西晋扬州治所**——此一中断之外、六朝累计三百年建都的厚度、是它在唐人凭吊里"前朝旧都"分量最重的来源。

**隋灭陈（589）+ 毁建康**：

- **隋开皇九年（589）正月**——隋将贺若弼 + 韩擒虎渡江、夜入建康朱雀门、擒陈后主于景阳井（"胭脂井"传说所在）；**陈亡**。
- **隋文帝诏**："**建康城邑宫室、并平荡耕垦**"（《隋书·地理志》原文）——以建康为南朝四代都城、易导致后世割据、命悉毁城邑宫室、平为农田。
- **隋开皇十年（590）**——移江宁县治于丹阳故地（旧建康城西南数里）；**自此建康作为城市消失、降为江宁县境内的一片废墟**。

**自隋唐两朝起建康废**：

- **隋**——置蒋州（治江宁）于江宁县；蒋州短期存在；
- **唐高祖武德三年（620）**——改江宁县为归化县 + 立蒋州；
- **唐高祖武德九年（626）**——复改归化为江宁县；
- **唐太宗贞观七年（633）**——江宁县隶润州；
- **天宝元年（742）**——州改郡之制下润州改丹阳郡（治丹徒）；江宁县仍属之；
- **天宝十四载（755）此刻**——**江宁县在丹阳郡（润州）境内**；郡治丹徒、县治新江宁城（旧建康废墟西南数里）。

**注**：**唐肃宗至德元载（756）后江宁县沿革有变化**——**至德二载（757）置江宁郡**（自润州析江宁、句容等四县另立、治新江宁城）；**乾元元年（758）改江宁郡为昇州**；**上元二年（761）废昇州、江宁县复隶润州**。**755 时这一刻**——**江宁县仍在润州/丹阳郡境内、江宁郡 + 昇州皆未立**——本 doc 严守 755 时态、不用"昇州""上元""江宁郡(析立)" 等 755 后名号。

**注**：上文 frontmatter `tang_admin` 字段标"runzhou-danyang-jun-jiangning-xian"——**755 此刻正名是润州（天宝元年/742 改丹阳郡），州/郡治丹徒县，下辖江宁县**；**不应理解为 755 时已有"江宁郡"建制**（江宁郡 757 才置，详见后文）。S4 epoch schema 立时统一 rename。

**唐人对建康的认知**（755 时盛唐）——**前朝旧都**：
- 江左衣冠的物理 anchor、六朝兴亡的现场；
- 隋平陈毁城已 166 年、废墟在江宁县境内；
- 凭吊情绪强、但盛唐前期对建康的具体诗咏密度较低；
- **怀古文学的高峰在中晚唐及以后**——刘禹锡《金陵五题》（827 作）、杜牧《江南春》《泊秦淮》（晚唐作）、韦庄《台城》（晚唐作）等皆 755 后作。

**注**：**李白《登金陵凤凰台》**——通行口径作于至德二载/757 流寓金陵时（或天宝间另说），**本 doc 历史长河段引用、不作 755 现场**。**杜牧"南朝四百八十寺、多少楼台烟雨中"**（《江南春》）——晚唐约 838 杜牧任池州刺史时作；**不能作为 755 现场唐人语**。**刘禹锡《乌衣巷》"朱雀桥边野草花、乌衣巷口夕阳斜、旧时王谢堂前燕、飞入寻常百姓家"**——中晚唐宝历二年/826 至大和元年/827 间作；**亦 755 后**。

### 此刻

天宝十四载这一年的建康废墟——**已废 166 年（589 隋平陈、毁建康 → 755）**；以**江宁县境内废墟**身份存在。

**废墟状态**：
- **台城**：755 时宫墙残基仍可辨；殿宇基址多为农田、树林覆盖；旧池苑（华林园等）多为林莽与水塘；
- **石头城**：755 时城墙夯土残基仍立、扼长江西岸；唐人多自此观大江；
- **乌衣巷**：755 时街区为江宁县城北郊的乡野；东晋王导、谢安等高门宅址早已为田畴覆盖；旧名"乌衣巷"或仍存民间口传、但物理已无遗痕；
- **朱雀桥 / 朱雀航**：755 时浮桥早撤、桥址在秦淮河北岸残存夯土墩；
- **覆舟山 + 鸡笼山**：山仍在、苑园基址散没于林莽；
- **秦淮河**：水道仍流、两岸是江宁县乡野与若干村落。

**江宁县的政治布局**：江宁县治在新县城（隋开皇十年/590 移江宁县治于丹阳故地）；县令掌江宁县军政、属润州（丹阳郡）；**唐廷未在江宁设特殊岁修保护建康废墟**——任其在田野中风化。**地方民间**：江宁县乡民耕作于台城旧址 + 朱雀桥旁——视建康废墟为"南朝旧城"、不再有政治意义。

**唐人凡过江南东道润州境者**——多有过江宁、观建康废墟、登石头城望江、过乌衣巷遗踪。**这种"凭吊"是江南东道润州境内最特殊的物理景观**——与丹阳齐梁陵区 + 句容南朝陵墓 + 江宁陈陵 等同构江南东道润州境的"南朝物理纪念"网络。

**安史前夕**（即 755 这一刻）——再过几个月安禄山起兵于范阳；**安史中江南未受直接兵燹**——建康废墟在乱中未受额外破坏。**中晚唐江南渐成繁荣区**——但建康废墟未被重建、继续在江宁县境内田野中风化。

### 此刻意味

对天宝十四载的江宁人来说——建康废墟就是**自家县境的近郊废墟**：日常耕作时可见的台城夯土、登石头城观江、过秦淮河北岸的朱雀桥旧址——这些是地理日常、不再有政治意义。对长安那一代士人来说——建康是**江左王业 333 年的物理纪念 + 六朝兴亡的现场**：孙权称帝建业、晋室南渡建康、王谢门阀、侯景之乱、隋平陈毁城——这一长段历史皆有此地为物理 anchor。

**盛唐 755 时对建康的怀古**——**密度较低**：盛唐人多在边塞 / 长安洛阳两京 / 蜀道 / 江左等地写诗、对建康的具体咏怀史源略零散。**怀古文学的密集化在中晚唐及以后**——刘禹锡、杜牧、韦庄等让"金陵""建康""六朝"成为中国文学最有名的怀古意象、让"朱雀桥""乌衣巷""台城""石头城"成为千古凭吊的物理 anchor。

**注**：**李白《登金陵凤凰台》**"凤凰台上凤凰游、凤去台空江自流——吴宫花草埋幽径、晋代衣冠成古丘"——通行口径作于至德二载/757 流寓金陵时（或天宝间另说）；**本 doc 不作 755 现场**。

唐人尚不知道——再过 2 年李白将流寓金陵、登凤凰台作"晋代衣冠成古丘"；再过 70 年刘禹锡将作《金陵五题》"朱雀桥边野草花、乌衣巷口夕阳斜"——使建康废墟成中国凭吊文学最深的一座物理 anchor；再过 80 年杜牧将作《江南春》"南朝四百八十寺、多少楼台烟雨中"；**再过 613 年（明洪武元年/1368）朱元璋以此地为应天府、定为明初京师**；**再过 666 年（永乐十九年/1421）明成祖迁都北京、以应天府为南京（府名仍为应天府，南京是都城名号）**——**建康旧地重为帝都**；再过 1100 余年（民国 + 现代）——南京继续作为中国南方最重要的城市之一。**今游人来南京**——所见的是**台城遗址（玄武湖南）+ 石头城公园（清凉山）+ 乌衣巷（夫子庙旁）+ 朱雀桥（秦淮河上）+ 覆舟山**——**多为后世重修或现代复建**、**六朝原物已极少**；**唐人在 755 所见的那片夯土废墟与田野遗踪**——大半早已为后代叠加的城市覆盖。**建康作为六朝古都的物理 anchor**——是中国凭吊文化最深的一处"前朝旧都"。

## 子条目

### 台城（六朝宫城；东晋至陈四代续筑）

**台城**是六朝建康宫城——**东晋成帝咸康年间（335-342）始筑**、**宋孝武帝、齐武帝、梁武帝等续修扩建**。**位置**：今南京玄武湖南、覆舟山东南——**南北约 2 公里、东西约 1.5 公里**；**南门朱雀门、北门玄武门**等。**宫城内含**：太极殿（前朝大殿）+ 显阳殿（后宫）+ 华林园（皇家苑囿）+ 景阳楼（陈后主与张丽华、孔贵嫔等所居）等。

**台城最戏剧的一刻**——**梁武帝太清二年至三年（548-549）侯景之乱**：**侯景围台城 130 余日**——城内粮尽、梁武帝萧衍饿死台城内、年八十六；**这是建康作都期间最大一次浩劫**、是六朝最阴暗的一段。

**台城最末一刻**——**陈祯明三年（589）正月**：**隋将贺若弼 + 韩擒虎渡江、夜入建康朱雀门、夺台城**——**陈后主投景阳井**（即"胭脂井"传说所在；后世称"辱井"）——陈亡、六朝结束。

**隋开皇九年（589）后**——隋文帝诏毁建康城、台城宫殿宫墙皆平荡耕垦；**至 755 时台城已废 166 年**、宫墙残基尚可辨、殿宇基址多为农田与树林覆盖。**中晚唐韦庄《台城》"江雨霏霏江草齐、六朝如梦鸟空啼——无情最是台城柳、依旧烟笼十里堤"**（晚唐作；755 时未作）——是台城作为凭吊 anchor 的文学高峰。

### 石头城（孙吴 212 年筑；六朝建康西门要塞）

**石头城**——**孙权建安十七年（212）筑**——是孙吴定都建业前的军事要塞；**位置**：今南京清凉山一线、长江南岸的山势险要处——扼长江、控建康西门。**城周约 3000 米、依山势筑**——是六朝建康最重要的军事屏障。

**石头城是六朝建康的西大门**——**敌兵自长江上游而来必先克石头城、再进建康台城**：东晋、宋齐梁陈历次南北对峙皆以石头城为前线。**晋元帝渡江后初居石头城**——再迁建康；**苏峻、桓玄等内乱皆以石头城为据点**。

**隋开皇九年（589）后**——石头城作为军事要塞的功能消失、城墙残基保留；**至 755 时石头城仍立**、城墙夯土残基可见、内中已无戍守。**唐人多登石头城观大江**——是江南东道润州境内最显的"望江凭吊"地。**刘禹锡《石头城》"山围故国周遭在、潮打空城寂寞回——淮水东边旧时月、夜深还过女墙来"**（中晚唐 827 作；755 时未作）——是石头城作为凭吊 anchor 的文学高峰。

### 乌衣巷（东晋王导、谢安等高门宅区）

**乌衣巷**——**东晋时建康城内秦淮河北岸的高门宅区**；**居者多王导、谢安、谢玄等家族**——是东晋至南朝门阀政治的物理 anchor。**位置**：今南京秦淮河北、夫子庙一带；**"乌衣"二字之来**——一说东晋戍兵着乌衣戍此巷、一说居者着乌衣（玄色士服）。

**乌衣巷在六朝**——**东晋王导、王羲之、王献之 + 谢安、谢玄、谢灵运 等皆居此**；王、谢两大家族是东晋至南朝门阀政治的核心；**"王与马、共天下"**（东晋初语）所指的"王"即琅琊王氏、居乌衣巷。

**隋开皇九年（589）后**——王谢宅、乌衣巷皆毁；**至 755 时已 4 世纪、王谢宅早不存**、街巷依稀；旧名"乌衣巷"或仍存江宁县民间口传、但物理无遗痕。**中晚唐刘禹锡《乌衣巷》"朱雀桥边野草花、乌衣巷口夕阳斜——旧时王谢堂前燕、飞入寻常百姓家"**（中晚唐 826-827 间作；755 时未作）——是乌衣巷作为凭吊 anchor 的文学高峰。**注**：**755 时唐人嘴里的"乌衣巷"**——已无原物；刘禹锡所写的"夕阳斜""野草花"——是中唐的浮想、非 755 现场。

### 朱雀桥 + 朱雀航（秦淮河上的浮桥；六朝建康正南门外咽喉）

**朱雀桥 / 朱雀航**——**秦淮河上的浮桥**；**位置**：今南京秦淮河上、镇淮桥附近；**六朝建康正南门朱雀门外的咽喉**——是建康台城南向的物理入口。**"航"在六朝建康语境中通常指浮桥**——以舟船连缀而成、可解可立、是建康城南二十四航之一（朱雀航是最显的一座）。

**朱雀桥在六朝**——**孙吴起设、东晋至陈承之**；**六朝建康自台城朱雀门出 + 过朱雀航 + 至秦淮河南岸**——是建康城最繁华的一段。**朱雀桥旁两岸**——是六朝建康繁华核心；桥北通宫城 + 乌衣巷、桥南通秦淮河南岸的市井。

**隋开皇九年（589）后**——朱雀航撤、桥址夯土墩残存；**至 755 时**——浮桥早撤、桥址在秦淮河北岸残存夯土墩。**中晚唐刘禹锡"朱雀桥边野草花"**（《乌衣巷》826-827 间作；755 时未作）——是朱雀桥作为凭吊 anchor 的文学高峰。

### 隋灭陈 + 隋文帝诏毁建康（589）

**隋开皇九年（589）正月**——**隋将贺若弼 + 韩擒虎渡江、夜入建康朱雀门、擒陈后主**——陈亡、隋统一南北。**隋文帝诏**："**建康城邑宫室、并平荡耕垦**"（《隋书·地理志》原文）——理由：建康为南朝四代都城、易导致后世割据、命悉毁城邑宫室、平为农田。**隋开皇十年（590）**——移江宁县治于**丹阳故地**（旧建康城西南数里）；**自此建康作为城市消失、降为江宁县境内的一片废墟**。

**这是中国历史上最大规模的一次"政治性毁城"**——以平天下之心毁前朝之都；**与项羽烧咸阳秦宫（前 206）+ 黄巢入长安（880）后破坏 + 北宋淳化五年/994 毁统万城**等并列"政治性毁城" 的几大事例。**注**：相比项羽烧秦宫为一时之愤、隋文帝毁建康是经过明诏的、系统性的"夷为农田"——这是它在中国都城史最特殊的一笔。

**自此建康废**：隋唐两朝皆未在建康旧址重置都城；唐高祖、太宗、高宗、武则天、玄宗朝皆以长安 + 洛阳为两京——**建康在 755 时已废 166 年、是"前朝旧都"的物理纪念**；要再过 800 年（明洪武元年/1368）朱元璋以此地为应天府、定为明初京师——建康旧地才再为帝都。

### 江宁县沿革（隋唐间复杂的县名变化）

**江宁县是 755 时建康废墟所属的行政县**——其沿革在隋唐间反复变化：

- **隋开皇十年（590）**——移江宁县治于丹阳故地、为蒋州治；
- **唐高祖武德三年（620）**——改江宁县为**归化县**、立蒋州；
- **唐高祖武德九年（626）**——复改归化为**江宁县**；
- **唐太宗贞观七年（633）**——江宁县隶**润州**（治丹徒）；
- **唐玄宗天宝元年（742）**——州改郡之制下润州改**丹阳郡**（治丹徒）；江宁县仍属之；
- **天宝十四载（755）此刻**——**江宁县在丹阳郡（润州）境内**；
- **至德二载（757）**——析润州的江宁、句容等四县另立**江宁郡**（治新江宁城）——**755 时尚未立**；
- **乾元元年（758）**——改江宁郡为**昇州**——**755 时尚未立**；
- **上元二年（761）**——废昇州、江宁县复隶润州；
- **唐末光启三年（887）**——再立昇州；
- **南唐 + 北宋间**——昇州升江宁府；
- **明洪武元年（1368）**——朱元璋以此为应天府、定明初京师；
- **永乐十九年（1421）**——明成祖迁都北京、以应天府为南京（府名未改、南京应天府是都城名号）。

**755 此刻正名**——**润州治丹徒、丹阳郡（即润州）下辖江宁县；县治新江宁城（旧建康废墟西南数里）**。**唐人在 755 来此**——多说"过江宁""至润州"或"游建康（旧名雅称）"——**不说"昇州""江宁府""南京"等 755 后名号**。

### 六朝建康的礼制建筑与佛寺

**六朝建康除台城 + 石头城外**——尚有大量礼制建筑 + 佛寺：

- **南郊圜丘 + 北郊方泽**：六朝祭天 + 祭地之所、在台城南北郊；
- **太庙 + 社稷坛**：六朝宗庙、在台城东南；
- **国子学 + 太学**：六朝儒学中心、在台城东；
- **同泰寺**（梁武帝萧衍 527 立、舍身寺所在）+ **瓦官寺**（东晋兴宁年间 363-365 立、顾恺之画维摩诘像所在）+ **栖霞寺**（南齐永明七年/489 立、栖霞山下）+ **鸡鸣寺**（旧名同泰寺、鸡笼山下）等——**六朝建康佛寺数百所**；**梁武帝萧衍崇佛、四次舍身同泰寺**。

**晚唐杜牧《江南春》"千里莺啼绿映红、水村山郭酒旗风——南朝四百八十寺、多少楼台烟雨中"**（晚唐约 838 作；755 时未作）——是六朝建康佛寺作为凭吊 anchor 的文学高峰。**注**：杜牧此诗作于晚唐池州 / 江南行走时、写江南诸州佛寺残迹与烟雨——**"南朝四百八十寺"是诗语虚指、非考实**；**755 时盛唐人嘴里没有这句**。

**755 时建康废墟的佛寺**——多数已毁或衰废；少数（如栖霞寺、瓦官寺等）仍在江宁县境延续——但已非六朝盛况。

## 历史长河（755 → 今）

**中晚唐**　**至德二载（757）析润州立江宁郡**——治新江宁城（旧建康废墟西南数里）；**乾元元年（758）改江宁郡为昇州**；**上元二年（761）废昇州、江宁县复隶润州**——唐廷在江宁的行政设置反复。**李白至德二载/757 流寓金陵作《登金陵凤凰台》**"吴宫花草埋幽径、晋代衣冠成古丘"——是盛唐至中唐之交对建康最直接的怀古笔法。**中晚唐刘禹锡 826-827 任和州刺史、过金陵作《金陵五题》（《石头城》《乌衣巷》《台城》《生公讲堂》《江令宅》）**——是中晚唐建康凭吊文学的高峰。**晚唐杜牧《江南春》《泊秦淮》《赤壁》等**——续中晚唐凭吊江南的文学传统。**唐末光启三年（887）再立昇州**——江宁开始重新作为城市发展。

**两宋**　**南唐（937-975）建都昇州（改名江宁府、再改金陵府）**——是建康在六朝后第一次重为帝都、虽仅 38 年；李煜词"四十年来家国、三千里地山河——凤阁龙楼连霄汉、玉树琼枝作烟萝"等皆作于此。**北宋平江南（975）**——降金陵府为昇州、再升江宁府；**北宋王安石、苏轼、欧阳修等多次过江宁凭吊**；**南宋失中原后（1127）**——建康（江宁府再改建康府）为南宋陪都之一、抗金前沿；岳飞 + 韩世忠 + 辛弃疾等皆驻建康；**辛弃疾《永遇乐·京口北固亭怀古》**等南宋怀古词皆以建康 + 京口为物理 anchor。

**元明清**　**元代建康改集庆路**——降为路；**明洪武元年（1368）朱元璋以集庆为应天府、定明初京师**——**建康旧地重为帝都、距 589 隋灭陈毁建康已 779 年**；**应天府筑明城墙 + 明故宫等**——这是建康旧地最大的一次复兴。**永乐十九年（1421）明成祖迁都北京、以应天府为南京（府名仍为应天府，南京是陪都名号）**——南京应天府作为陪都延续。**清顺治二年（1645）清军入南京**——南京改江宁府、为两江总督驻地、清代江南最重要的城市之一。

**民国至今**　**1912 年中华民国临时政府定都南京**；**1927 年国民政府迁都南京**；**1937 年南京沦陷**（日军侵略 + 南京大屠杀）——这是南京在 20 世纪最阴暗的一段。**1949 年人民解放军渡江入南京**——南京作为江苏省省会延续至今。**1949 年后多次考古**——**南京市博物馆 + 南京大学历史学院 1950s-2010s 多次勘察建康城遗址**（台城 + 石头城 + 乌衣巷 + 朱雀桥 + 覆舟山）——**今台城遗址 + 石头城公园 + 乌衣巷 + 朱雀桥 + 鸡鸣寺**等列国家级 / 省级文物保护单位、为南京旅游核心。**今游人来南京**——所见的是**多为后世重修或现代复建**的台城 + 石头城 + 乌衣巷 + 朱雀桥；**六朝原物已极少**——**唐人在 755 所见的那片夯土废墟与田野遗踪**——大半早已为后代叠加的城市覆盖。**建康作为六朝古都的物理 anchor**——是中国凭吊文化最深的一处"前朝旧都"。
