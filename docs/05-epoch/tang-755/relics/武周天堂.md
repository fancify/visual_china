# 武周天堂

## 一句话

武则天在洛阳紫微城明堂西北所造的礼佛阁——五层、文献记"高数百尺"、内立一尊**夹纻大像**（漆塑非铜）——**建成不到七年，证圣元年正月丙申夜一把火烧光**，自此再未重建；盛唐至 755 时，天堂遗址只是明堂西北那一片台基。

## 形貌

天堂在洛阳东都紫微城内、**明堂西北约 155 米**——位于宫城轴线区域西侧（非正北；近年考古勘探确认）。

**始建年代**：天堂之造与明堂略同时，按《通鉴》《唐会要》《旧唐书·则天本纪》合参——明堂垂拱四年（688）成、天堂略后于明堂、约**永昌元年至载初元年间（689-690）造**，完成于天授至延载之间（约 690-694）。**文献记天堂"高数百尺"**——按唐尺约 30 cm 计为 90 米、按唐大尺 36 cm 计约 110 米；俗称"高约 150 米"是近代误传，**唐宋史源未有此具体数字，本 doc 严守"文献口径"**。

**形制**——五层木构方阁，每层都内开。**像甚大——可"自第三层平望其腹"**（《通鉴》载）；按《唐会要》同记**"夹纻大像，其小指中犹容数十人"**——像之大小指可容数十人立、可估全像规模。**像的材质是夹纻（漆塑）非铜**——唐宋史源明文，本 doc 严守。**铸造方式**：以麻布、漆、骨灰、木骨依次成像，是初唐造像史最大的一尊夹纻像。

天堂的建筑用途——**武则天的私家礼佛堂**。武周朝崇佛——女皇以佛教法王地位自居，在紫微城北建一座专属礼佛阁，与南面议政的明堂并立——一前一后，一政一教，是武周朝政治建筑的核心叙事。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

天堂在唐人的回望里是**一夜之间烬于火的"武周高阁"**。

武则天**光宅元年（684）九月**改东都为"神都"——意在君权神授；**天授元年（690）九月**正式即帝位、国号周。武周朝在洛阳的大兴土木以**垂拱四年（688）明堂**为先——"万象神宫"建成于紫微城内贞观殿旧址、高三百尺、四方层楼上下三层；其后**约 689-690 年间始造天堂**——立明堂西北、规制更高，是女皇的私家礼佛阁。**建成于天授至延载间（约 690-694）**——盛唐两京"最高木构二阁"自此成立。

**营建主持**：**薛怀义**——本是市井卖药者冯小宝，得幸于武后，**度为僧、白马寺主**（《旧唐书·薛怀义传》明文）；**垂拱四年（688）武则天命薛怀义充使督作明堂、并于明堂北起天堂**——他既是这两座大殿的实际营建主持者、又是日常法会的主持人。

**长寿元年（692）至证圣元年（695）**——是天堂存世的极盛期。武周朝大佛事在此举行：天堂内立夹纻大像，每月朔望，女皇率群臣礼佛于此；天堂上下楼皆有佛像、经卷、灯火——文献记每佛事日"夜然烛万枝、光照洛城"。

**证圣元年（695）正月**——这是天堂的最后一个月：
- **正月辛巳（朔，元日）**——改元"证圣"；
- **乙未（十五日）**——大开无遮会于明堂；
- **丙申（十六日）**——张大像于天堂；**是夕——薛怀义因失宠纵火天堂、烈焰自天堂蹿出、风助火势——火延明堂、夜至晨——明堂、天堂二殿俱烬**（《资治通鉴》卷二〇五原文）。

**自证圣元年（695）至 755 年**——计 60 整年。期间：**明堂**在 696 年由武则天命匠师重建、规制略减；玄宗朝几度改名——**开元五年（717）改为"乾元殿"；开元十年（722）复题"明堂"；开元二十七年（739）拆去上层、去柱心木、依旧为"乾元殿"**（《唐会要》卷十一）——755 时这座旧明堂的官称便是"乾元殿"（**非长安大明宫的"含元殿"**，二者勿混）。**天堂自烬后再未重建**。**唐人见到的天堂遗址，只是明堂（乾元殿）西北那一片台基**——夯土方堆、四面有阶——夯土堆上长着草、有几株槐——便是天堂在 755 时的全貌。

### 此刻状态

天宝十四载这一年的天堂遗址——**已经空了整整六十年**。

紫微城内的格局：**南有应天门**（仍立、是洛阳大典之处）→ **天街** → **武周明堂遗痕——755 时官称"乾元殿"** （已是开元二十七年/739 改名后的样子；非武周原貌；与长安大明宫"含元殿"是两座殿，勿混）→ **天堂遗址**（在乾元殿西北、夯土台基 + 四面阶基）→ 再北是寝殿。60 年间这一带是宫城北部一片"半废墟半新殿"——明堂虽有但已经过 696 重建 + 717 改乾元 + 722 复明堂 + 739 再改乾元殿四次反复、天堂只是台基。

紫微城内的"乾元殿"在玄宗朝主要作大朝会之用——但盛唐玄宗常居长安大明宫，开元二十四年（736）十月最后一次自洛阳还长安后再未东巡——**自此东都紫微城逐渐"无主"**。755 时洛阳宫城无帝、无大事——日常由东都留守、河南府、河南尹一线管辖。**乾元殿仍立、台基仍在、夯土堆仍是夯土堆**——是一座空着的禁苑核心。

**755 年这一刻的洛阳人——尚不知道再过一年东都将沦陷**。**至德元载正月（756）安禄山在洛阳称帝**——以应天门受朝、号"大燕"、年号圣武；明堂、天堂的残基便成了伪燕朝寝的"前台"。**这一段历史给天堂的废墟添了一层意味**——它先是武周高阁之烬、再成伪燕称帝之地。

### 此刻意味

对天宝十四载的洛阳人来说，天堂遗址是**紫微城西北那片"祖辈说过的旧高楼"地**——日子如常、夯土堆如常。对长安那一代士人来说，天堂是**武周一朝最具象的"建筑神话"**——盛唐人凡涉及"高木构"、"焚于火"、"武周时代"——皆点天堂；薛怀义"自烧自殿"是中晚唐谈资。

唐人尚不知道——再过一年明堂亦将经安史之乱毁；再过千年，紫微城内夯土全为农田覆盖；再过千二百年，**洛阳考古队会在天堂遗址里挖出一处略方台基** + **中心石砌圆坑**——便是天堂"五层方阁、夹纻大像"那座木构的根基。

## 子条目

### 夹纻大像

天堂内立的一尊夹纻大像——**像之大可"自第三层平望其腹"**（《资治通鉴》载）；**其小指中犹容数十人**（《唐会要》明文）。**像材质**：夹纻（汉魏以来佛像古法之一）——以麻布、漆、骨灰、木骨层层成像、内空——这种方法可成大型佛像，远轻于铜铸、亦不必铸坊大工程。**这是初唐造像史最大的一尊夹纻像**——比龙门石窟卢舍那大佛（高 17.14 米，唐高宗咸亨四年/673 凿成）大近一倍。**证圣元年（695）正月丙申夜一夜俱烬于火**——夹纻是漆塑，遇火即燃；像主体在天堂内同步焚毁。**像残体后世未有出土**——夹纻属易燃材料，火后无金属残痕可寻。

### 五层方阁

天堂主体——五层方形木阁、每层都内开（可上下楼）、四面飞檐——是初唐木构最高的一座阁。**建造监督是薛怀义**（《旧唐书·薛怀义传》明文；他充使督作明堂、并起天堂——一身两职），长寿元年（692）至证圣元年（695）三年间是天堂的实际营建主持者。**695 年正月丙申夜火即是他纵的**——是建造者亲手烧毁了自己的工程。**今天堂遗址的考古基础**：略方台基（南北约 77.7 米、东西约 69.15 米）+ 中心石砌圆坑（直径约 14.8 米）+ 五重圆形承重基础——这种"略方台基外圈 + 圆形承重内圈"的结构是中古高层木构的典型基础形制。

### 明堂遗址（同夜俱毁、玄宗朝几度改名）

天堂东南约 155 米——武则天垂拱四年（688）建、号"万象神宫"——三层方阁、高三百尺、四面三百尺。证圣元年正月丙申夜与天堂同烬；同年（696）武则天命匠师重建、规制减；玄宗朝先后**开元五年（717）改"乾元殿"；开元十年（722）复题"明堂"；开元二十七年（739）再拆下层、改名"乾元殿"**——名虽存、已非武周之旧；**与长安大明宫前殿"含元殿"是两座殿，请勿混**。755 时此殿为玄宗朝"乾元殿"，仍立、空着、用于大朝会备用。**明堂的"今状态"**：1986-1988 年发掘、2012 年明堂遗址保护展示馆开放——今游客可入夯土殿基上的复原结构内俯仰。

### 紫微城南北中轴

应天门 → 天街 → 旧明堂（755 时已改为"乾元殿"）→ 天堂遗址（在乾元殿西北、不在正北）→ 后寝殿——这是武周朝紫微城的南北中轴 + 西北分支，盛唐玄宗朝部分保留。755 时这一带**应天门仍立**（756 安禄山即用此门受朝）、**乾元殿仍立**（已是开元 27 年改名后的样子）、**天堂只剩台基**——三件物事在中轴及其西北：一座新门、一座几度改名的旧殿、一片高台基址——便是天堂在 755 时所属的物理背景。

## 历史长河（755 → 今）

**中晚唐**　至德元载（756）正月——安禄山在洛阳称帝、以应天门受朝、号"大燕"——乾元殿（旧明堂）、天堂的残基成伪燕"前台"。**至德二载（757）九月**——洛阳乾元殿在陕郡战前后被叛军焚毁——**乾元殿一线连同天堂遗址一片，自此彻底成废墟**。中晚唐洛阳作为陪都地位下降——乾元殿不再重建、天堂遗址仍是夯土。

**五代北宋**　**唐亡（907）后洛阳不再作都**——五代后梁、后唐、后晋、后汉、后周间皆设洛阳为"西京"、但宫城已不再大修；**北宋初定都开封后**——洛阳作为陪都的物理实存进一步收缩；**北宋洛阳宫城内**——明堂、天堂遗址应已为林木所覆。**北宋诗人多游洛阳——但游的是龙门、白马寺、隋唐运河**——紫微城内的明堂、天堂遗址在宋诗里几无声息。

**金元**　北宋亡（1127）后洛阳屡受兵燹——金元间紫微城内一带逐渐成农田与民居——**明堂、天堂遗址被农人耕作、覆于地下**。元代洛阳路改河南府路——紫微城旧址完全失却作为"宫城"的地位。

**明清**　明清两朝紫微城旧址成洛阳老城北部——民居、街巷覆盖之下，夯土基础埋于地下数尺。**清初顾炎武游洛阳——所见的"隋唐故宫"已只是地名**。

**民国至今**　1959 年中国社会科学院考古研究所洛阳工作队开始系统勘探**隋唐洛阳城**——明堂、天堂遗址位置初步确认。**2007-2010 年正式发掘明堂遗址 + 天堂遗址**——明堂遗址出土夯土基址、阶基、瓦当；**天堂遗址出土略方台基（南北 77.7m × 东西 69.15m）+ 中心石砌圆坑（直径约 14.8m）**——确认是"略方台基 + 圆形承重内圈"的木构基础，与文献"五层方阁"形制相合。**2012 年明堂遗址保护展示馆开放**（含复原观景结构）；**2013 年天堂遗址博物馆开放**——内部有玻璃栈道、可俯瞰夯土基础。**今游人来洛阳应天门一线，可看见武周朝两座最高木构的根基**——一座立着复原结构（明堂）、一座只是玻璃下的夯土（天堂）。**盛唐人当年所见的那座夯土台基本身，至此终于回到了游人视野**——但**那五层方阁、那尊夹纻大像、那一夜的火——考古至今也复原不出**。
