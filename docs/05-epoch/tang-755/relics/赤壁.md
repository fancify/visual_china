# 赤壁

## 一句话

**长江南岸的三国古战场**——东汉建安十三年（208）冬周瑜 + 刘备五万水陆联军于此火攻曹操、奠定三分天下的物理 anchor；**唐 755 时属江南西道鄂州（江夏郡）蒲圻县西一百二十里**、长江南岸赤色崖壁犹存——是唐人沿江下行最显的怀古地之一。

## 形貌

赤壁在唐江南西道**鄂州（天宝元年/742 改江夏郡）蒲圻县西约一百二十里**——长江南岸临江崖壁（今湖北省赤壁市西北西梁湖与长江交汇处南岸）。**地势**：长江中游南岸、自西向东流的大江在此微北曲、南岸为一段赤褐色砂岩崖壁、北岸为江北平畴（乌林一带）；崖壁高约数十米、临江峭立、自江上望去其色赤褐、故名"赤壁"。**自鄂州治江夏县（今武汉武昌）沿江上行约 200 唐里可至**——唐人由江夏溯江西上、过金口、入蒲圻县界、再行可见此崖。

**对岸即乌林**——长江北岸乌林矶（唐代所属另核；今湖北洪湖市乌林镇一带）；**乌林是曹操水军屯泊之地**、与赤壁隔江相对；**赤壁之战的真正决胜处在乌林**（曹军舰队所泊）——黄盖火船自南岸赤壁出发、乘东南风纵火直冲乌林、烧曹军船舰；故《水经注》《元和郡县图志》皆载"赤壁山……北岸即乌林、与赤壁相对"。**唐人对此战物理地理的认知**——以《三国志》《水经注》《元和郡县图志》为骨——"南岸赤壁 + 北岸乌林 + 江心火船"三点构成一战的物理 anchor。

**蒲圻县**——唐鄂州下辖县之一、治在今湖北赤壁市蒲圻老县治；**注**：今"赤壁市"（1986 年蒲圻县改名赤壁市）是现代地名、**755 时此地名"蒲圻县"**、城在赤壁山东约一百二十里、与赤壁山为县与县境西陲一处野外崖壁的关系。**蒲圻县治不在赤壁山**——赤壁山只是蒲圻县西部长江沿岸的一段崖壁、并无大型聚落。

**唐 755 时所见**——**赤褐色崖壁临江峭立 + 江水东流 + 北岸乌林隔江相望 + 江上往来商船渔舟**；崖上无大型建筑、或有地方祠庙传说、755 形制不可坐实（详今考古）；**赤壁山以南山势渐低、丘陵向南延伸至幕阜山余脉**。

物候：长江中游、亚热带季风——夏湿热、冬温湿、四时不甚分明；**冬季多东南风 / 偏南风**——这是赤壁之战火攻的关键气候条件（详子条目）；雨集中夏季、长江汛在夏秋；**唐人沿江下行多在春秋两季**。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

赤壁在唐人的回望里——是**三国奠基的物理战场 + 中国军事史最有名的火攻案例**。**建安十三年（208）冬这一战 + 火攻 + 三分定局**——构成唐人对赤壁三层认知：

**建安十三年（208）的局势**。**建安十三年七月**——曹操自邺出兵南征荆州；**八月**荆州牧刘表卒、次子刘琮立；**九月**曹操军至新野、刘琮举荆州降；**刘备自樊城南奔**——长坂坡（今湖北当阳）败于曹操虎豹骑、自夏口（今武汉汉口）东向、入江夏太守刘琦境；**鲁肃自江东来吊刘表丧、于当阳遇刘备**——劝结盟拒曹。**自此孙刘联盟成立**：**孙权遣周瑜、程普、鲁肃将水军三万**自江东西上、**与刘备（约两万）合**——总兵力约五万；**曹操军号八十万**（实际不超二十万——含降伏荆州水军七八万、自北方率领的步骑约十余万；**"八十万"是曹操在赤壁前致孙权书所自称的数字、不是实数**——这是唐人读《三国志·吴书·周瑜传》"今治水军八十万众、方与将军会猎于吴"裴注所知的史实）。

**建安十三年（208）冬十二月赤壁之战**（《三国志·武帝纪》载十二月；《后汉书·献帝纪》载十月——本 doc 取《武帝纪》十二月口径）。**曹军自江陵东下、与孙刘联军遇于赤壁**——**初战曹军不利**（《武帝纪》"公至赤壁、与备战、不利"）；**曹军退泊乌林、北岸**——**孙刘联军泊赤壁、南岸**——两军隔江相望。**曹军北人多不习水战、连舟舰首尾相接以平稳**；**黄盖建策火攻**（《三国志·黄盖传》原文）——"操军方连船舰、首尾相接、可烧而走也"。

**黄盖诈降 + 火攻 + 东南风**。**黄盖书诈降于操**（《三国志·周瑜传》裴注引《江表传》原文）——选蒙冲斗舰十艘、载燥荻枯柴、灌油其中、裹以帷幕、上建旌旗——**约定降日东南风急**——**火船自南岸赤壁顺东南风纵火直冲乌林曹军舰队**——曹军舰队相连无法分散、火势蔓延、岸上营寨亦受波及——**曹军大溃**。**注**：**"诸葛亮借东风"是元末《三国演义》小说情节、不是史实**——史载是**黄盖诈降 + 利用冬季时令东南风**（长江中游冬季多有东南风 / 偏南风、约一周期出现一次）；**唐人 755 时所知的赤壁火攻** 是黄盖之策、风向是天时、**与诸葛亮无关**（诸葛亮在赤壁之战中的角色主要是"促成孙刘联盟"——见《三国志·蜀书·诸葛亮传》"先主遣亮使孙权"）。

**曹操败走华容道**——**自乌林北遁、循江北华容道（今湖北监利、洪湖一带湿地）西北还江陵**；**华容道泥泞、曹军徒步践踏、死伤甚众**（《三国志·武帝纪》裴注引《山阳公载记》）；**自此曹军损失水军主力 + 北返**——**孙刘联军乘胜追击 + 周瑜进取江陵**——**至建安十四年（209）冬周瑜陷江陵、曹仁北退**——**赤壁之战 + 江陵之战合为孙刘联军的两段胜利**。

**三分定局**。**赤壁之战后**——**曹操退守北方 + 孙权据有江东 + 刘备得荆州南四郡（武陵 + 长沙 + 桂阳 + 零陵）**——**三国鼎立的物理基础在此奠**。**自 208 至 222 间**孙刘曾因荆州归属再起纷争（详《三国志·关羽传》《吕蒙传》：建安二十四年/219 吕蒙袭荆州、关羽败死）——但**赤壁之战是三分天下的起点**——盛唐人读《三国志》《后汉书》及裴注皆以此一战为汉末三国分立的物理 anchor。

**自三国至唐**——赤壁作为战场仅一战；战后蒲圻一带渐入孙吴、晋、宋齐梁陈、隋、唐辖境；**北朝至隋唐间或有地方祠庙传说**（755 形制不可坐实）——是这片崖壁作为"古战场"在地方信仰中可能的物理痕迹。

### 此刻

天宝十四载这一年的赤壁——**已是战后 547 年**（208 → 755）；属唐江南西道**鄂州蒲圻县**境内长江南岸一处崖壁。

**地表所见**——**赤褐色崖壁临江峭立、高数十米、绵延约里许**；**江水自西向东、北岸乌林隔江相望**；**崖上无大型纪念建筑**——或有地方祠庙传说、755 形制不可坐实、即便有亦香火不盛；**江上往来商船渔舟**——长江中游自唐初已是江南西道与江南东道、岭南、剑南间最重要的水运动脉、蒲圻段是江夏（鄂州治）西上岳州、潭州（今长沙）的必经一程。

**蒲圻县治**——在赤壁山东约一百二十里、唐县城较小、户口数千；**鄂州治江夏县**（今武汉武昌）距赤壁约 200 唐里（沿江上行）——是唐江南西道的重镇之一、唐人由江夏西上 + 入蒲圻县界 + 至赤壁是一条常走的怀古路线。**鄂州境内的赤壁山** 是**江南西道凭吊三国最显的物理 anchor**——与对岸（江北鄂州境外、即北岸乌林（唐代所属另核））合为完整战场。

**唐人对赤壁的认知**——主要通过：
1. **史书**：**《三国志》**（西晋陈寿撰、裴松之注 429）+ **《后汉书》**（南朝宋范晔撰）+ **《晋书》**（唐房玄龄等贞观二十二年/648 撰）——盛唐已成书广流；裴注引《江表传》《山阳公载记》补充诸多细节；
2. **地理书**：**《水经注》**（北魏郦道元 6 世纪初撰）卷三十五·江水三明载"江水又东径赤壁山北、昔周瑜与黄盖诈魏武大军处所"——是唐人定位蒲圻赤壁为战址的核心 textual 来源；**《元和郡县图志》**（唐李吉甫元和八年/813 撰）虽为 755 后约 60 年成书、但其所依据的 8-9 世纪江南西道地志资料延续魏晋以来对蒲圻赤壁的认定；
3. **物理凭吊**：唐人沿江下行至蒲圻 + 江上望赤壁崖 + 隔江望乌林——是盛唐凭吊三国最浓厚的一站；**李白、杜甫等盛唐诗人有过赤壁者** 多见其凭吊诗篇（具体诗篇史源略歧、本 doc 不强引）。

**这一刻最值得说的事**——**唐人 755 时所见的赤壁、是"蒲圻赤壁"**（即真战场）；**而后世文学最有名的"赤壁"凭吊地——黄州赤壁——尚未成为凭吊重镇**：**杜牧《赤壁》作于会昌二年至四年/842-844 黄州刺史任上、距 755 约 90 年**；**苏轼《赤壁赋》《念奴娇·赤壁怀古》作于元丰五年/1082 黄州团练副使任上、距 755 约 327 年**——**这两组传世名作在 755 时皆尚未发生**；**盛唐人凭吊赤壁、是凭吊真战场（蒲圻）、不是凭吊文学景观（黄州）**——这是 755 时态最有意思的一点。

**安史前夕**（即 755 这一刻）——再过半月安禄山起兵于范阳；江南西道远离北方战场、但**安史之乱中长江中游一线仍是漕运 + 兵员的关键通道**——再过两年（**至德二载/757**）**永王李璘自江陵东下、谋取广陵（今扬州）**——经长江一线下行、过鄂州、过赤壁、过浔阳——李璘事败后被肃宗诏讨平；**李白曾入李璘幕府**——长流夜郎途中经过长江中游赤壁一线（具体行程史源略阙）。**755 时的赤壁** 是江南腹地的一处怀古崖壁、未被即将到来的战乱直接波及。

### 此刻意味

对天宝十四载的蒲圻县人来说——赤壁是**自家县境西陲长江岸边的一段赤色崖壁**：江上商船岁岁过、崖上小庙香火淡——是日常风物、非政治舞台。对鄂州 + 江夏的士人来说——赤壁是**汉末三分定局的物理战场**：周瑜火攻 + 黄盖诈降 + 曹操败走华容道——一战定三国——**是江南西道境内最浓厚的怀古点之一**。对盛唐自长安南来沿江下行的旅人来说——**赤壁与岳阳楼、洞庭、湘水、衡山** 并列为江南怀古路线上的核心 anchor——但**与岳阳楼后世的"楼"性纪念物不同、赤壁始终是一段裸露的崖壁 + 一片江水 + 一处北岸的乌林**——**物理状态最朴素、最贴近原战场**。

**盛唐人对赤壁的凭吊** 集中于以下几点：
- **周瑜 + 黄盖 + 火攻**——是建安年间最有戏剧性的军事胜利；
- **曹操 + 华容道 + 北返**——是建安年间最有戏剧性的军事失败；
- **三分天下的起点**——是中国历史最有名的"以少胜多"案例之一；
- **长江中游的物理地理**——南岸赤壁 + 北岸乌林 + 江水东流——是江南西道版图上最有"故事密度"的一段崖壁。

唐人尚不知道——**再过约 90 年**（**会昌二年至四年/842-844**）杜牧任黄州刺史、过黄州城西北赤鼻矶、误以为是赤壁古战场、作《赤壁》"折戟沉沙铁未销、自将磨洗认前朝、东风不与周郎便、铜雀春深锁二乔"——**这是中国文学史上最有名的赤壁凭吊诗、却作于错的赤壁（黄州赤鼻矶、即文赤壁）**；唐人更不知道——**再过约 327 年**（**元丰五年/1082**）苏轼贬黄州团练副使、二游赤壁（黄州赤鼻矶）、作前后《赤壁赋》+《念奴娇·赤壁怀古》"大江东去、浪淘尽、千古风流人物、故垒西边、人道是、三国周郎赤壁"——**苏轼自知此非真战场**（《念奴娇》"人道是"三字暗含存疑）、**但他选择以文学之名认定此地为赤壁**——**自此"文赤壁"（黄州赤壁）以文学之名压过"武赤壁"（蒲圻赤壁）成为中国文化记忆中的赤壁主形象**。

**这是唐人 755 时见不到的一个反讽**——**真正的战场（蒲圻）将以"武赤壁"为名、退居怀古史的一角；错的赤壁（黄州）将以"文赤壁"为名、登上中国文学记忆的最高位**——**赤壁的双重身份**：**蒲圻是史的赤壁、黄州是诗的赤壁**——755 时态下尚未分裂、唐人所见的是单一的"赤壁"（蒲圻战场）；中晚唐 + 北宋以后才二分。

## 子条目

### 建安十三年（208）冬赤壁之战

**建安十三年（208）冬十二月**（《三国志·武帝纪》口径）——**曹军自江陵东下、与孙刘联军遇于赤壁**。**双方兵力**：曹军号八十万、实际约二十万（含降伏荆州水军七八万、自北方率领的步骑约十余万）——是裴注《江表传》载曹操致孙权书自称数字、不是实数；孙刘联军约五万（周瑜 3 万 + 刘备 2 万）。**初战**——**曹军不利**（《三国志·武帝纪》"公至赤壁、与备战、不利"）——曹军北人多不习水战、长江风浪所颠、士卒疫疾流行（《武帝纪》裴注引"是岁、大疫"）。**曹军退泊乌林、连舟舰首尾相接以平稳**——**孙刘联军泊赤壁、南岸**——隔江相望。

**黄盖建策**（《三国志·黄盖传》原文）——"操军方连船舰、首尾相接、可烧而走也"——**周瑜从之**。**黄盖书诈降于操**（《三国志·周瑜传》裴注引《江表传》原文）——**约定降日东南风急**——**选蒙冲斗舰十艘、载燥荻枯柴、灌油其中、裹以帷幕、上建旌旗、预备走舸系于舰尾**——**风急船速、火烈风猛、船往如箭**——**直冲曹军舰队 + 火势蔓延 + 岸上营寨亦受波及**——**曹军大溃**。**曹操弃舟而陆走、自乌林北遁、循华容道西北还江陵**——**华容道泥泞、曹军徒步践踏、死伤甚众**（《三国志·武帝纪》裴注引《山阳公载记》）。

**注**：**"诸葛亮借东风"是元末《三国演义》小说情节、不是史实**；史载赤壁之战中**风向是黄盖利用的冬季时令东南风**（长江中游冬季多有东南风 / 偏南风、约一周期出现一次）——**是天时、不是人为**。**诸葛亮在赤壁之战中的角色** 主要是**促成孙刘联盟**（《三国志·蜀书·诸葛亮传》"先主遣亮使孙权、亮与孙权战于赤壁"）；**唐人 755 时所知的赤壁火攻** 是黄盖之策、与诸葛亮无关——这是盛唐人读《三国志》清楚的史实。

### 武赤壁 vs 文赤壁（蒲圻 vs 黄州）

**赤壁之战的真实地点**——**自魏晋至唐**史地之书皆以**蒲圻县西一百二十里长江南岸赤壁山**为战址（《水经注》卷三十五江水三 + 《元和郡县图志》卷二十七鄂州蒲圻县）——是**"武赤壁"**——史的赤壁。

**异说**——历代有**蒲圻 / 黄州 / 蕲春 / 武昌 / 嘉鱼**五处异说；其中**蒲圻赤壁**是学界主流、亦是史地之书的定说；**黄州赤壁**（今湖北黄冈市西北赤鼻矶、长江北岸）是**唐杜牧 + 宋苏轼凭吊的赤壁**——**"文赤壁"**——诗的赤壁；**蕲春赤壁、武昌赤壁、嘉鱼赤壁**是地方传说、史源较弱。**本 doc 取主流口径**：**蒲圻赤壁为真战场（武赤壁）**、**黄州赤壁为后世文学景观（文赤壁）**——**二分明确**。

**755 时态下**——**唐人所见的"赤壁"** 仅指**蒲圻赤壁**（武赤壁、真战场）；**黄州赤壁尚未成为凭吊地**——杜牧《赤壁》、苏轼《赤壁赋》皆作于 755 之后（842-844 / 1082）。**自杜牧、苏轼以后**——黄州赤壁以文学之名后来居上、压过蒲圻赤壁成为中国文化记忆中的赤壁主形象；**但 755 这一刻、"赤壁"仍是单一的（蒲圻）、未分裂**。

### 长江南岸赤壁山的地理

**赤壁山** 在唐鄂州蒲圻县西一百二十里——**长江南岸临江崖壁**（今湖北赤壁市西北西梁湖与长江交汇处南岸）。**长江于此微北曲**——**自西向东流**——**南岸为赤褐色砂岩崖壁、高约数十米、绵延约里许**——其色赤褐、自江上望去如赤色屏障、故名"赤壁"。**北岸为乌林矶**（今湖北洪湖市乌林镇一带）——是江北平畴、曹军水军所泊。

**南岸赤壁 + 北岸乌林 + 江水东流** 是赤壁之战的物理三角——**这是唐 755 时唐人所见的物理地理、与今地表景观大体一致**（今赤壁山仍立、乌林矶尚在、长江流向未大变）。**赤壁山以南山势渐低**——丘陵向南延伸至幕阜山余脉；**赤壁山东去蒲圻县治约一百二十里**——是县境西陲的一处野外崖壁。

### 火攻 + 东南风 + 黄盖诈降

**赤壁之战最有名的物理事件**——**火攻 + 东南风 + 黄盖诈降**——三位一体。**黄盖建策**：曹军船舰首尾相接、可烧而走——这是黄盖看到曹军物理弱点的诊断。**黄盖诈降**：**遣使致书曹操、陈言孙权强弱不敌、愿率部归附**——**约定降日**；曹操中计、令水军前部张帆迎接降船。**东南风急**：**长江中游冬季多有东南风 / 偏南风**——**约一周期出现一次**——黄盖据时令判断、选定降日。**火攻**：**蒙冲斗舰十艘**（蒙冲是汉魏间小型快速战船、可冲撞 + 易行驶）——**载燥荻枯柴、灌油其中、裹以帷幕、上建旌旗、预备走舸系于舰尾**——**风急船速、火烈风猛、船往如箭**——**直冲曹军舰队、火势蔓延、岸上营寨亦受波及**——**曹军大溃**。

**注**：**"诸葛亮借东风"是元末罗贯中《三国演义》小说情节**——**不是史实**；**史载赤壁之战的风向是黄盖利用的冬季时令东南风**——**是自然现象、不是人为**。**唐 755 时所知的赤壁火攻** 是**黄盖之策 + 时令东南风**——**与诸葛亮无关**（诸葛亮在赤壁之战中的角色主要是促成孙刘联盟）。

### 三分天下的起点

**赤壁之战是中国历史最有名的"以少胜多"案例之一**——孙刘联军五万破曹军二十万（号八十万）；**自此曹操退守北方 + 孙权据有江东 + 刘备得荆州南四郡**——**三国鼎立的物理基础在此奠**。**赤壁之战 + 建安十四年（209）冬周瑜陷江陵 + 建安二十四年（219）吕蒙袭荆州 + 章武元年（221）刘备称帝 + 黄初元年（220）曹丕代汉 + 黄龙元年（229）孙权称帝** 共同构成三国分立的连续过程——**赤壁是这一过程的起点**。

**盛唐人读《三国志》《后汉书》及裴注皆以此一战为汉末三国分立的物理 anchor**——是**江南西道境内历史密度最高的一段崖壁**。

### 北朝至隋唐的赤壁庙

**赤壁山或有地方祠庙传说、755 形制不可坐实**——据北朝《水经注》"赤壁山……昔周瑜与黄盖诈魏武大军处所"的记述可推知北朝间已有凭吊地的地位；至于隋唐间崖上是否实有祠庙、供奉对象为何（周瑜 / 黄盖 / 关公等说皆出后世），史源不足、本 doc 不强坐实。**755 时即便有亦香火不盛**——非国家祀典。**唐宋以后赤壁庙渐立**——但物理形制小、不及汉魏 + 隋唐都城遗址（如邺城 + 汉魏洛阳故城）的物理规模。**今赤壁古战场遗址**——以**翼江亭、拜风台、武侯宫** 等明清间所立纪念建筑为主——皆非 755 时所见、亦非汉魏间原物。

## 历史长河（755 → 今）

**中晚唐**　**唐 755 至中晚唐**蒲圻赤壁仍为长江南岸怀古地、有诗人过赤壁凭吊；**中晚唐诗人李端、温庭筠、罗隐等**有过赤壁的诗作（具体诗篇史源歧出、本 doc 不强引）。**最大的一笔**——**会昌二年至四年（842-844）杜牧任黄州刺史**——**过黄州城西北赤鼻矶、误以为是赤壁古战场**——作《赤壁》：

> 折戟沉沙铁未销、自将磨洗认前朝。
> 东风不与周郎便、铜雀春深锁二乔。

——**这是中国文学史上最有名的赤壁凭吊诗、却作于错的赤壁（黄州赤鼻矶、即文赤壁）**；自此**黄州赤壁开始以文学之名取得凭吊地位**——但**蒲圻赤壁仍是史地之书的定说**——二分尚未明朗。

**五代 + 北宋**　**五代南唐间** 蒲圻赤壁仍为江南腹地的一处怀古地——但因不在主要兵争路线上、几无大事载于史。**北宋初定都开封**——长江中游为荆湖南北路（今湖北 + 湖南）；**蒲圻县属荆湖北路鄂州**——是宋代最稳定的内陆腹地之一。**北宋元丰五年（1082）苏轼贬黄州团练副使**——**七月既望与友人游赤壁（黄州赤鼻矶）**——作《赤壁赋》：

> 客有吹洞箫者……其声呜呜然、如怨如慕、如泣如诉……苏子曰：客亦知夫水与月乎、逝者如斯、而未尝往也、盈虚者如彼、而卒莫消长也。

——**十月又游赤壁、作《后赤壁赋》**；**同年作《念奴娇·赤壁怀古》**：

> 大江东去、浪淘尽、千古风流人物。
> 故垒西边、人道是、三国周郎赤壁……

——**"人道是"三字暗含存疑、苏轼自知此非真战场**——但**他选择以文学之名认定此地为赤壁**——**自此"文赤壁"（黄州赤壁）以文学之名压过"武赤壁"（蒲圻赤壁）成为中国文化记忆中的赤壁主形象**。**北宋以后**——黄州赤壁的文学声誉日隆、蒲圻赤壁的史地声誉渐退；**赤壁的双重身份成立**：**蒲圻是史的赤壁、黄州是诗的赤壁**。

**南宋 + 元**　**南宋绍兴间**鄂州（武昌）是抗金前线重镇——蒲圻赤壁地处南宋腹地、未受兵祸；**南宋诗人陆游、范成大、辛弃疾等** 过赤壁有诗——多依苏轼路线、凭吊黄州赤壁多于蒲圻赤壁。**元代** 长江中游战事零星、蒲圻赤壁渐成田野中的怀古点。

**明清**　**明嘉靖间**蒲圻县在赤壁山上立**翼江亭、拜风台、武侯宫**等纪念建筑——但这些纪念物**沿袭《三国演义》情节**（如"拜风台"对应小说中诸葛亮借东风）、**不是史实纪念**；**清代**赤壁山上小庙渐多、但物理形制小。**黄州赤壁则被苏轼后人 + 历代文人不断修葺**——**明清间黄州赤壁形成"东坡赤壁"建筑群**——成为中国文化记忆中的赤壁主形象。**自明至清**——**"文赤壁"（黄州）压过"武赤壁"（蒲圻）的局面已不可逆**。

**民国至今**　**民国间**蒲圻县 + 黄冈县皆有赤壁纪念物；**1949 年后**蒲圻县列重点文物保护单位；**1986 年蒲圻县改名"赤壁市"**——以行政地名固定其"武赤壁"地位；**今"赤壁市"** 即唐 755 时之"蒲圻县"——是中国少数以古战场命名的县级市。**1980s 起赤壁市建赤壁古战场遗址公园**——以翼江亭 + 拜风台 + 周瑜雕像 + 三国蜡像馆为主；**长江南岸赤壁崖壁** 仍立、其色赤褐、与 755 唐人所见的物理景观在大轮廓上一致。**今游人来赤壁市**——所见的是**长江南岸赤色崖壁 + 明清间纪念建筑 + 现代复建景观**——**真正与 755 连续的物理 anchor**：**崖壁本身 + 长江流向 + 北岸乌林**——其余皆后世所加。

**注**：**今"赤壁市"是 1986 年蒲圻县改名而来的现代地名**——**唐 755 时此地名"蒲圻县"**。**今"黄州赤壁"** 在今**湖北黄冈市西北、长江北岸赤鼻矶**——**与蒲圻赤壁分别属今咸宁市与黄冈市**——**距离约 200 公里**——**二者非同地**。

- [岳州（巴陵郡）](../cities/岳州.md) — 唐江南西道岳州治巴陵县（今岳阳）；自鄂州沿江上行过赤壁后下一站；岳阳楼凭吊与赤壁凭吊同为长江中游怀古路线两站（待立）
- [华容道](./华容道.md) — 曹操败走华容道的物理路线、北岸乌林西北至江陵的湿地泥泞段；与赤壁是赤壁之战的"战 + 退"两段（待立）
- [乌林](./乌林.md) — 长江北岸乌林矶（唐代所属另核）、曹军水军所泊；与赤壁是赤壁之战的"南岸 + 北岸"两段（待立）
- [黄州赤壁](./黄州赤壁.md) — 长江北岸黄州赤鼻矶、杜牧 + 苏轼凭吊的"文赤壁"；与蒲圻赤壁是"武 + 文"二分（待立）
