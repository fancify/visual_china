# 曹溪 / 南华寺

## 一句话

韶州曲江东南六十里曹溪畔的禅宗祖庭——六祖慧能在此居三十余年、开演南宗顿悟；先天二年示寂于新州国恩寺、同年迁神龛归曹溪安奉；**755 时寺名"法泉寺"、距慧能圆寂已四十二年**——天下禅僧南来朝祖之圣地。

## 形貌

曹溪寺在韶州曲江县城东南约六十里——**双峰山下、曹溪之畔**，是岭南道一处山中佛寺。

寺址在曹溪南岸——曹溪是浈水（北江上游）东岸一条支流，水自双峰山中下来，沿山脚南行，绕寺而过。**寺所在地是一片山间小盆地**——东西约半里、南北一里余，四面青山。寺门朝南，门外是曹溪渡口；寺内主体三进——天王殿、大雄宝殿、六祖殿——南北轴线。**六祖殿在寺最北高台**——是慧能当年讲经、晚年居住的核心殿（慧能示寂于新州国恩寺、神龛后迁归曹溪安奉于殿后真身塔）；殿后即**真身塔**——慧能肉身舍利所在。

寺名经几度更易。**唐仪凤二年（677）慧能至此时寺名"宝林寺"**——南朝梁天监元年（502）智药三藏过岭至此立、因山形似西天宝林山而得名。**神龙元年（705）武则天敕赐"中兴寺"**——旋改"法泉寺"（异文：或为同年再改、或为景龙间改，史源略异，本 doc 取 705 神龙元年至景龙年间口径）。755 时寺名是"法泉寺"。**北宋开宝元年（968）始改"南华寺"**——以"南方华夏禅宗祖庭"取名——此名沿用至今。

曹溪一带是岭南道**北江中段最潮湿一片**——夏日溽热、冬日少雪、四时常雨。寺周南国植物茂——榕、樟、菩提树为多——这是岭南山寺的物候底色。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

曹溪在唐人的回望里是**南宗禅的祖庭**——一段从黄梅五祖到岭南六祖的衣钵传承。

**六祖慧能（638-713）**——新州（今广东新兴）人，俗姓卢。**唐高宗龙朔元年（661）二十四岁**至湖北黄梅东禅寺、入五祖**弘忍**门下——为僧团内一普通行者，主舂米杂役。弘忍命门下作偈以试根器——上座神秀作"身是菩提树，心如明镜台；时时勤拂拭，勿使惹尘埃"；慧能作（一作令人代书）"菩提本无树，明镜亦非台；本来无一物，何处惹尘埃"——五祖夜召慧能传衣钵，密授《金刚经》、令南归避祸。

**慧能南归约 15 年**——避居岭南山中，间或为猎户营烧饭、或在四会、怀集一带韬晦——具体行迹史源略有出入。**仪凤二年（677）**——慧能至韶州曹溪宝林寺，**自此长居曹溪三十余年**；先天二年（713）八月初三示寂于新州国恩寺、旋迁神龛归曹溪。在曹溪期间慧能开演"顿悟"一脉——**门下出神会、怀让、行思、玄觉、永嘉、本净、智常等十大弟子**——禅宗"南宗"由此立。

**先天二年（713）八月初三**——慧能**示寂于新州国恩寺**（新州即慧能故乡、今广东新兴；非曹溪），世寿七十六（按 638 生 / 713 卒）。**同年十一月迁神龛归曹溪安奉**（王维《六祖能禅师碑铭》"某月日、迁神于曹溪"——是慧能身后归曹溪之明文）。**遗体不腐**——门人以漆布裹之、置于六祖殿后塔内，**至今犹存**（南华寺真身塔内的慧能肉身像）。

慧能圆寂后**北宗 vs 南宗** 之争渐起——神秀（？-706）一系在洛阳、长安间盛、武则天朝至中宗朝为"两京法主"；慧能一系在岭南、湖南、江西间渐播。**开元二十/二十二年（732/734，两说并存）**慧能弟子**神会**于**滑台（今河南滑县）大云寺举无遮大会**、公开攻北宗"师承旁出、传衣不正"——这是南北宗论辩的转折事件。**天宝四载（745）神会自滑台入洛阳荷泽寺驻锡弘法**——**确立慧能为禅宗六祖正统**——这一年距 755 仅 10 年。

### 此刻状态

天宝十四载这一年的曹溪法泉寺——**六祖已圆寂四十二年；真身仍在；天下禅僧南来朝祖**。

寺主持是慧能门下第几代弟子待考（《传灯录》《祖堂集》对曹溪天宝间住持记载零散），但**六祖殿与真身塔仍是寺的核心**——殿前柏树成林、塔后香火不断。**自神会 734 滑台辩后**，南宗禅在两京渐为正统；**安史前夕**（即 755 这一刻）——南宗禅僧在长安、洛阳已不在少数，北宗渐衰之势已现。

**神会请托王维撰《六祖能禅师碑铭》**——王维碑铭自述"弟子曰神会……谓余知道、以颂见托"——史源记此事在天宝间（具体年份史源略异，本 doc 取"天宝间作"口径；**非"奉敕"作**）——此碑文是慧能首次入正式文人撰碑文体的关键文献。碑文流传至中唐——但**碑原立曹溪何处、是否存至 755**史载未明，本 doc 不预设。

天宝十四载的曹溪法泉寺——香火依旧、真身依旧、塔仍在；但**寺外的天下**——再过几个月，安禄山起兵于范阳；再过半年潼关失守、玄宗西奔；岭南远在战火之外，曹溪几乎不受兵燹影响。**这是曹溪在中唐前最后的一段平静**。

### 此刻意味

对天宝十四载的韶州人来说，曹溪法泉寺是**州城东南那座最有名的山寺**——日子如常、香火如常。对长安那一代禅僧来说，曹溪是**南宗祖庭、慧能真身所在**——盛唐凡习南宗者，必欲一谒曹溪；但岭南远、路险，能至者少。**盛唐人嘴里的"曹溪"，更多是个地名兼宗门符号**——比实地远，比传说近。

唐人尚不知道——再过 60 年（元和十年/815），唐宪宗追谥慧能为"大鉴禅师"——柳宗元、刘禹锡先后撰碑、立于曹溪；再过两百余年（开宝元年/968），北宋将法泉寺改名"南华寺"——此名沿用至今；再过千年，慧能的肉身仍坐在六祖殿后的真身塔内——**唐人在 755 看见的那尊肉身像，与今人看见的是同一尊**。

## 子条目

### 六祖殿（慧能讲经处 + 神龛归曹溪后安奉处）

寺最北高台之上——慧能于曹溪三十余年间在此讲经、晚年居住；先天二年（713）八月初三示寂于新州国恩寺（非曹溪），同年十一月神龛迁归曹溪安奉于此。**六祖殿是曹溪寺的物理核心**——其前为大雄宝殿、其后为真身塔。盛唐殿宇规模史载未详——今所见六祖殿为明万历重修。**殿内主像是慧能塑像**——但**慧能的真身（漆布肉身像）不在六祖殿内，而在殿后真身塔**——两处分而存。

### 真身塔（慧能肉身舍利所在）

六祖殿后高台之上——慧能圆寂后门人以漆布裹之、置于此塔。**塔内的慧能肉身像至今犹存**——是南华寺最重要的一件遗物，亦是中国佛教史上**最早保存至今的肉身像**（早于本山的智药三藏百余年）。盛唐塔形史载未详——今所见真身塔为明清重修；但塔内的肉身像主体未易、**与唐人在 755 所见为同一尊**。

### 双峰山

寺所在的山——双峰相对，山势如门。智药三藏南朝梁天监元年（502）过岭至此，以山形似西天宝林山而立宝林寺。**双峰山在唐时是曹溪的天然标识**——山下即寺、寺下即溪、溪下即渡。盛唐人若自韶州城东南行六十里，远见双峰相对，便知曹溪已近。

### 神会滑台 + 入洛阳事

**开元二十/二十二年（732/734，两说并存）**——慧能弟子**神会（684-758）**于**滑台（今河南滑县）大云寺举无遮大会**、公开攻北宗"师承旁出、传衣不正"——这是禅宗史"南北宗分裂"之标志事件。**天宝四载（745）神会自滑台入洛阳荷泽寺驻锡弘法**——荷泽宗自此立、神会被尊为荷泽宗祖。**神会卒于乾元元年（758）**——755 这一刻神会尚在世、住洛阳荷泽寺、已七十一岁。神会的"南宗正统论"在 755 时已是两京禅门的主流见解。

### 王维《六祖能禅师碑铭》

天宝间作（具体年份史源略异，本 doc 取"天宝间"口径）——**神会请托王维撰文**（王维碑铭自述"弟子曰神会……谓余知道、以颂见托"——非"奉敕"作）。**这是慧能首次入正式文人撰碑文体的关键文献**——王维是盛唐第一流诗人 + 长期奉佛、与神会交游密——此碑既是碑、亦是文学。**碑文流传至中唐**——柳宗元、刘禹锡 815 撰大鉴禅师碑时即承王维碑而发挥。**碑原是否立曹溪 + 是否存至 755**——史源未明、本 doc 不预设。

## 历史长河（755 → 今）

**中晚唐**　安史乱中曹溪几无兵燹——仍是岭南山中静寺。**贞元末元和初（800s）**——韩愈、柳宗元、刘禹锡等贬岭南、过曹溪——韩愈元和十四年（819）正月贬潮州过岭、未必入曹溪寺；柳宗元、刘禹锡元和间 815 撰**《大鉴禅师碑》**——曹溪由是再显。**元和十年（815）**宪宗追谥慧能为**"大鉴禅师"**——这是慧能身后第一次入官修朝廷之文。柳宗元撰**《赐谥大鉴禅师碑》**、刘禹锡撰**《大鉴禅师第二碑》**——两碑皆立于曹溪、至北宋仍立。

**两宋**　**北宋开宝元年（968）**法泉寺改名**"南华寺"**——此名沿用至今。**南宋绍兴四年（1134）**宋高宗诏修六祖殿、敕赐"南华禅寺"——这是南华寺第二次得朝廷正式赐额。**绍圣元年（1094）至元符三年（1100）**——苏轼贬惠州、儋州，往返六年两过曹溪南华寺——**苏轼留《南华寺》《六祖塔功德疏》等数篇**："亭亭塔中人，问我何所见"——苏轼两过曹溪、礼六祖真身，是宋代士大夫与曹溪交涉最深的一段。

**元明清**　明洪武初南华寺一度衰；明万历二十八年（1600）**憨山德清禅师重修曹溪**——德清自岭南来南华、住持十余年、重修六祖殿与真身塔——是禅宗祖庭明代的一次中兴。清初南华寺续修——康熙、雍正、乾隆三朝俱有敕修记录。**清代南华寺为岭南第一禅寺**——士大夫贬岭南、贾客入岭南、官吏赴任，皆以至曹溪礼六祖为人生一事。

**民国至今**　民国十年代以来南华寺续有修缮。**1934-1936 年虚云老和尚重修曹溪南华寺**——虚云（1840-1959）是中国近现代禅宗最重要的高僧之一、自江西云居山来曹溪、住持三年、彻底重修六祖殿与真身塔——是南华寺第三次大中兴。**1981 年中国佛教协会列南华寺为全国汉传佛教重点寺院**；**1983 年国务院列南华寺为全国重点文物保护单位**。**今南华寺仍在韶关市曲江区马坝镇东南**——六祖慧能肉身仍坐于真身塔内。**唐人在 755 所见的那尊肉身像，是今天游人所见的同一尊**——一千二百四十二年，未易。
