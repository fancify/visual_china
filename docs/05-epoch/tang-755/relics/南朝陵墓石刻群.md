# 南朝陵墓石刻群

## 一句话

**建康-丹阳-句容**三地分布的**南朝（宋齐梁陈）四代帝陵 + 石刻群**——420-589 年间陆续葬入；**755 时已立 160-330 年、废陵已废 166 年**（589 隋灭陈）；**麒麟、天禄、辟邪石兽千年立田野——是中国南朝雕塑唯一大规模遗存**。

## 形貌

南朝陵墓石刻群在唐**江南东道润州境**——具体分布**三地**：
- **建康（江宁）境**：今**南京市**江宁区、栖霞区 等——主要含**陈武帝陈霸先万安陵、陈宣帝显宁陵、陈文帝永宁陵**等。
- **丹阳境**：今**江苏省丹阳市** 周边——**齐宣帝萧承之永安陵 + 齐景帝萧道生修安陵 + 齐武帝萧赜景安陵 + 梁武帝萧衍修陵 + 梁文帝萧顺之建陵 + 梁简文帝萧纲庄陵** 等齐梁帝陵集中地。
- **句容境**：今**江苏省镇江市**句容市——含部分石兽组（具体陵主对应学界仍有争议；本 doc 不预设主陵）。**宋武帝刘裕初宁陵**：通行口径在**今南京麒麟门 / 麒麟铺一带**（非句容），且石兽归属仍有学界争议——不挂句容。

**这三地相距约 50-100 公里**——构成南朝皇室陵墓的核心区。**南朝四代 168 年（420-589）十几位帝陵 + 数十王公墓的石刻群**——是中国南朝唯一大规模雕塑遗存。

**南朝陵墓石刻的形制**：南朝陵墓神道两侧立**石兽 + 石柱（华表）+ 石龟趺**等：
- **石麒麟、石天禄**：帝陵常用（角数、命名在朱偰以来研究中有分歧；通行口径"独角麒麟、双角天禄"，但学界异说较多）；
- **石辟邪**：王公墓多用（规模略小）；
- **石柱（华表）**：神道入口；
- **石龟趺**：碑座。

（注：南朝石兽的具体命名 + 角数对应制度，朱偰、罗宗真等学者有不同口径，本 doc 不绝对化。）

**南朝石兽风格**：与北朝石兽（多为石虎、石犀、石马等具象动物）不同——**南朝石兽多为神兽（麒麟、天禄）**、风格雄健、肌肉饱满、姿态前扑——是中国雕塑史最优秀的"动物造像"之一。

**现存数据**：现行保护清单口径**南京约 17 处、镇江（含丹阳、句容）约 12 处、共 29 处**（南京镇江协同立法宣传口径；百科 + 旧清单 33 处等差异——具体数字随统计口径有差异）——分布在南京、丹阳、句容三地的田野中、村庄旁、河岸边。

物候：江南东道腹地、亚热带季风气候——四时分明、夏热冬温、雨多。**石兽千年间露天立于田野**——多有风化、苔藓、裂痕；但主体形态犹存。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

南朝陵墓石刻群在唐人的回望里——是**南朝（宋齐梁陈）四代帝陵的物理遗存 + 江左繁华已尽的物理纪念**。

**南朝四代 168 年（420-589）**：
- **宋（420-479）**：刘裕篡晋立宋——是南朝第一代；**8 帝、59 年**。
- **齐（479-502）**：萧道成篡宋立齐——是南朝第二代；**7 帝、23 年**。
- **梁（502-557）**：萧衍篡齐立梁——梁武帝在位 48 年是南朝最长；**4 帝、55 年**；**侯景之乱（548-552）**摧毁建康，梁亡。
- **陈（557-589）**：陈霸先（南朝唯一寒族出身的开国皇帝）立陈——**5 帝、32 年**；**589 隋灭陈、统一南北**。

**现存石刻相关的重要南朝陵墓**（非四代帝陵全表；齐明帝兴安陵、宋文帝长宁陵等亦在邻近区、本 doc 仅列与现存石刻紧密相关者）：
- **宋武帝刘裕初宁陵**：通行口径在**今南京麒麟门 / 麒麟铺**一带（具体石兽归属学界仍有争议）；
- **齐宣帝萧承之永安陵 + 齐景帝萧道生修安陵 + 齐武帝萧赜景安陵**：**丹阳齐陵区**（即"齐陵"——今丹阳市西北郊）；
- **梁武帝萧衍修陵 + 梁文帝萧顺之建陵 + 梁简文帝萧纲庄陵**：**丹阳梁陵区**（与齐陵相距数公里）；
- **陈武帝陈霸先万安陵 + 陈宣帝显宁陵 + 陈文帝永宁陵**：建康附近（南京）。

**南朝陵墓石刻的特殊性**：
- **南朝礼制相对简约**——石刻数量少于北朝同期；
- **石兽 + 石柱 + 石龟趺三件套**为南朝帝陵神道标配；
- **石兽风格雄健**——与汉魏石兽厚重朴拙不同、与北朝石兽具象写实不同。

**自隋灭陈（589）后**——南朝帝陵成废陵；**隋唐两朝皆未在此设特殊岁修**——任田野中风化。**唐人在 755 时所见的南朝陵墓石刻**——是 166 年废陵后的田野遗物。

**唐人对南朝陵墓石刻的认知**：盛唐人凡至润州境者多有过陵区——**唐诗中"金陵""建康""江左""六朝"等词所指的物理 anchor 即此**。**盛唐诗咏南朝的诗虽多**——但具体描写**南朝陵墓石刻的诗** 较少；**主要凭吊集中在中晚唐及以后**：
- **刘禹锡《乌衣巷》《石头城》《金陵五题》**（827 作）——是中晚唐凭吊建康 + 南朝最有名的组诗；
- **李白《登金陵凤凰台》《登瓦官阁》**（盛唐间作）——"晋代衣冠成古丘" 等金陵怀古笔法（**注**："南朝四百八十寺"句是杜牧《江南春》、非李白）；
- **杜牧《江南春》《泊秦淮》《赤壁》**——中晚唐凭吊南朝的代表。

**注**：**李白《登金陵凤凰台》约 754 作（盛唐间）**；**杜牧南朝诸诗多作于晚唐**——盛唐 755 前对南朝陵墓石刻的具体诗咏密度较低；**南朝陵墓石刻的文学密集化在中晚唐及以后**。

### 此刻

天宝十四载这一年的南朝陵墓石刻群——**已废 166 年（589 隋灭陈→755）**；以**润州境内田野遗物**身份存在。

**石刻状态**：盛唐 755 时南朝石兽群应**主体犹存**——160-330 年间露天立于田野；石麒麟、天禄、辟邪、石柱等多在原位、未大破坏。**陵冢**：南朝帝陵封土多在 755 时仍可辨——但已 160-330 年未维护、夯土残败。**唐人凡过润州境者**——可见**麒麟石兽立田中、华表柱孤立田边**——是江南乡野最特殊的物理景观。

**唐润州的政治布局**：**润州（江南东道；治丹徒/京口、即今镇江）辖境内的南朝陵墓**——分属润州属下数县（丹阳县、句容县、江宁县、上元县 等）。**唐廷未在此设特殊纪念**——任陵区为田野。**地方民间**：南朝陵墓石刻周边的乡民多以石麒麟、天禄等为风水、土地神——这是民间信仰对古迹的接纳。

**安史前夕**（即 755 这一刻）——再过几个月安禄山起兵于范阳；**安史中江南未受直接兵燹**——润州境内南朝陵墓石刻群在乱中保存完好。**中晚唐润州渐成江南繁荣区** ——南朝陵墓石刻群继续在田野中风化。

### 此刻意味

对天宝十四载的润州人来说——南朝陵墓石刻是**境内田野中千年来累积的废陵 + 神兽石刻**：日常耕作时可见、不再有政治意义。对长安那一代士人来说——南朝陵墓石刻群是**江左繁华已尽的物理纪念**：宋齐梁陈 168 年的帝王长眠之地、石麒麟立田野中风化。**李白盛唐间作《登金陵凤凰台》**：**"晋代衣冠成古丘"**——是盛唐对金陵 / 六朝最有名的文学怀古，**不可证明亲见南朝石兽或直咏陵刻；属金陵怀古文学**。

唐人尚不知道——再过百年（中晚唐）刘禹锡、杜牧等会让"南朝"二字成中国文学最有名的怀古意象；再过 800 年（明清间）南朝陵墓石刻渐为农田覆盖、部分被风化或盗损；再过千二百年（21 世纪），**南朝陵墓石刻群列国家级重点文物保护单位**（1988 年起多批次列入）——**今散布南京、丹阳、句容三地田野中的 现行口径约 29 处石刻**——是**中国南朝雕塑唯一大规模遗存**。**唐人在 755 所见的那些石麒麟、天禄、辟邪**——一千二百多年后大半犹存、立田野中风化、苔藓覆盖、姿态雄健不变。

## 子条目

### 石麒麟 + 天禄 + 辟邪（南朝石兽三种）

**南朝帝陵神道两侧立石兽**——核心三种：
- **麒麟**：帝陵专用（独角神兽）——头有独角、身有翼羽、姿态前扑；
- **天禄**：帝陵专用（双角神兽）——头有双角、身有翼羽、姿态前扑；
- **辟邪**：王公墓用（独角或双角、规模略小）——形与麒麟天禄类、但更小。

**南朝石兽的风格**：**肌肉饱满、姿态前扑、神态雄健**——与汉代石兽（厚重朴拙）+ 北朝石兽（具象写实）不同。**南朝石兽是"神兽"造像**——非具象动物（如石虎、石马）、而是想象中的瑞兽。

**南朝石兽的高度**：**帝陵石麒麟 + 天禄通常高 2.5-3.5 米、长 3-4 米**——比唐陵石兽小、但比例更优雅。

**现存著名石兽**：
- **梁武帝萧衍修陵神道石麒麟**：丹阳——是南朝石兽的代表；
- **齐武帝萧赜景安陵神道石麒麟**：丹阳；
- **陈武帝陈霸先万安陵神道石麒麟**：南京；
- **梁文帝萧顺之建陵神道石麒麟**：丹阳。

### 丹阳齐 + 梁陵区（南朝陵墓最密集地）

**丹阳市西北郊**——是齐、梁两代帝陵最密集的一片**陵区**。**齐陵区（齐宣帝、景帝、武帝）+ 梁陵区（梁武帝、文帝、简文帝）** 在丹阳境内相距数公里、合占丹阳西北郊约 30 平方公里。

**丹阳为何成南朝陵区**：（1）**齐梁两代是萧氏皇室**——萧氏祖籍兰陵（南兰陵在丹阳境内）；（2）建康（南京）作为都城、自然吸引王公墓；（3）萧氏将祖籍兰陵的旁边作家族陵墓区。

**今丹阳齐梁陵区**：列国家级重点文物保护单位、含多组神道石麒麟 + 石天禄 + 石柱 + 石龟趺；地表陵冢多已为农田 + 村庄 + 现代道路覆盖、但石刻群基本未易。

### 句容 + 江宁（南京）的南朝陵墓

**句容**：今镇江市辖；含**宋武帝刘裕初宁陵**（一说在南京境）等。
**江宁（南京）**：今南京市辖；含**陈武帝陈霸先万安陵 + 陈宣帝显宁陵 + 陈文帝永宁陵** 等陈代帝陵 + **梁安成康王萧秀墓** 等王公墓。

**南京境内南朝石刻较散**——不像丹阳齐梁陵区那样集中、但单组石刻精度高（**萧秀墓神道石辟邪是南朝石刻雕塑的最高水准之一**）。

### 唐人对南朝的怀古文学

**唐诗中"南朝""金陵""建康""江左""六朝"** 等词所指的物理 anchor——多在润州境内的南朝陵墓石刻群 + 建康废城旧址（升州/江宁县境）。

**盛唐诗（755 前后）**：**李白《登金陵凤凰台》**（约 754 作）："吴宫花草埋幽径，晋代衣冠成古丘"——是盛唐对南朝最深的怀古笔法。**李白《登瓦官阁》**等亦涉。**注**：李白的金陵诗是否亲见南朝陵墓石刻——史源未详；可能主要是地理文学想象。

**中晚唐诗（800-900s）**：**刘禹锡《乌衣巷》《石头城》《金陵五题》**（827 作）——是中晚唐凭吊建康 + 南朝最有名的组诗；**杜牧《江南春》《泊秦淮》**——"南朝四百八十寺、多少楼台烟雨中"。**这些诗多作于 755 后**——是中晚唐凭吊南朝的代表、不在盛唐 755 前作。

### 隋灭陈（589）+ 建康废城

**隋开皇九年（589）**——隋灭陈、统一南北；**隋文帝命毁建康城**——以建康为南朝四代都城、易导致后世割据；**自此建康从都城退为废城**。**隋后建康降为蒋州 + 江宁县**；唐承之、置润州境内的江宁县。

**南朝陵墓 + 建康废城的双重废**——是 5-6 世纪江南文明最大转折。**唐润州境内**——既有建康废城（江宁县）、又有南朝陵墓石刻群（丹阳 + 句容 + 江宁）——是江南文明从"南朝陵阙"到"唐润州" 的物理过渡。

## 历史长河（755 → 今）

**中晚唐**　**安史中江南未受直接兵燹**——南朝陵墓石刻群在乱中保存完好。**中晚唐润州渐成江南繁荣区** ——刘禹锡、白居易、杜牧 等多次过润州凭吊南朝；**南朝陵墓石刻群在中晚唐进入唐诗高峰期**。

**两宋**　**北宋初江南为宋都附近的繁华地**；**北宋金石学家欧阳修、赵明诚《金石录》记南朝陵墓石刻 + 碑刻**——是宋代南朝石刻研究的高峰。**南宋失中原后江南为南宋核心**——但**南朝陵墓石刻群在南宋间多有损坏**（兵燹 + 盗扰）。

**元明清**　元明清间南朝陵墓石刻多在田野中——部分被盗或风化损毁；**明末清初江南农田扩张** 占用了部分陵区。**清乾隆 + 嘉庆间地方官员开始记录南朝石刻**——是金石学进入清代的延续。

**民国至今**　**1936-1942 年朱希祖（中央研究院）+ 朱偰（南京博物院）等先后开展南朝陵墓石刻调查**——是南朝石刻群第一次系统化勘察。**1949 年后多次保护**：**1956 年江苏省列南朝陵墓石刻为省级保护单位**；**1988 年起多批次列国家级重点文物保护单位**——现行口径约 29 处。**2000s-2010s 多次考古勘察 + 局部修缮**。**今游人来南京 / 丹阳 / 句容**——所见的是**散布田野中、村庄旁、河岸边的石麒麟 + 天禄 + 辟邪 + 石柱**——**中国南朝雕塑唯一大规模遗存**。**唐人在 755 所见的那些石兽**——一千二百多年后大半犹存、立田野中风化、苔藓覆盖、姿态雄健不变。
