# 响堂山石窟

## 一句话

**邺都 ↔ 晋阳道上的北齐皇家石窟**——北齐文宣帝高洋一朝（550-559）始凿、武成帝高湛 + 后主高纬续凿，至北齐末（577）历三十年；位磁州/邯郸郡治滏阳县西南鼓山、距邺都遗址约六十里；755 时是磁州境内的北齐遗产 + 邺都北郊礼佛之所；再过四年（759）即将成邺城之败的战场背景。

## 形貌

响堂山石窟在唐河北道**磁州/邯郸郡**（治滏阳县）西南——具体在**滏阳县西南鼓山**之上；**鼓山**自西向东横亘磁州西境、是太行山东麓余脉伸入华北平原的一段石灰岩山脊（今河北邯郸市峰峰矿区 + 武安市交界一带）。**地势**：太行山东麓最后一道山脊、东望华北平原 + 西望太行腹地；滏水（漳水南支、源于鼓山西侧）自山下东流入华北平原——鼓山是滏水的发源 + 邺都西北约六十里的天然屏障。**响堂山** 因山岩中空、人语山响、击石如鼓而得名——"响堂"即"鼓响之堂"。

**响堂山石窟分南北响堂两组**——这是响堂山最核心的物理事实，盛唐人凡至响堂必先认这两处：

- **北响堂**：在滏阳县西南鼓山西麓——**鼓山主峰北侧山腰**；**北齐文宣帝高洋（550-559 在位）一朝始凿**——是响堂山系开窟最早、规模最大的一组；现存**九大窟**（包括大佛洞 + 释迦洞 + 刻经洞 等）+ 数十小龛 + 数百造像。**大佛洞**——是北响堂规模最宏者、高五丈余（约 12 米）、内立大佛三尊（中尊高三丈 / 约 7 米）；窟顶覆斗藻井 + 四壁千佛龛——是北齐皇家石窟最盛之作。
- **南响堂**：在滏阳县西南鼓山东南麓——距北响堂约**十五里**（约 7.5 公里）；**北齐武成帝高湛（561-565 在位）至后主高纬（565-577 在位）间续凿**——略晚于北响堂；现存**七大窟**（华严洞 + 般若洞 + 空洞 + 释迦洞 等）+ 数十小龛 + 数百造像。南响堂规模略小于北、但雕刻精度更高——尤其是**法华经雕塑** + **维摩诘经变**——是北齐晚期石窟艺术的代表。

**南北两响堂同属响堂山系**——皆在鼓山岩壁之上、相距数公里、皆北齐皇家所凿；**合称"响堂山石窟"**——是北齐高氏皇室在邺-晋阳道上最主要的开窟造像工程。

**响堂山的物理位置**：磁州治滏阳县（今河北邯郸市磁县境）西南三十里；邺都遗址在响堂山东南约六十里——即**响堂山在邺都西北约六十里**（按唐里约 25-30 公里、与今地理量距相合）；从邺都北上晋阳的官道经磁州、绕鼓山东麓北行——**响堂山正在邺-晋阳道边的山岩之上**，是北齐皇室往返两都途中礼佛 + 营葬的物理 anchor。

755 时响堂山石窟**保存大体完好**——北响堂大佛洞 + 南响堂华严洞等北齐主窟皆仍存；窟前木构寺院（北齐时所建、隋唐间多次重修）尚在；**北齐高氏皇室的功德石窟在邺都废墟之外得以独立留存**——这是与邺都焚毁形成强烈对比的物理事实。

物候：太行东麓、暖温带季风——四时分明、春多沙、夏多雨、秋高、冬寒；鼓山岩壁朝阳避风、窟前松柏环绕——是华北平原西缘最早的一组成熟石窟群之一。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

响堂山石窟在唐人的回望里——是**北齐皇家在邺-晋阳道上留下的最大佛教遗产**。**北齐两都制 + 高氏皇室开窟 + 皇家功德三重叠加**构成唐人对响堂山三层认知：

**第一层：北齐两都制（534-577）**。**东魏天平元年（534）十月**——高欢迫孝静帝自洛迁邺；高欢自己以**晋阳（今山西太原）** 为霸府、坐镇晋阳遥控邺都朝廷；**自此邺都 + 晋阳成北齐两都**——邺都是法理上的都城（皇帝在邺）、晋阳是实际的权力中心（高氏霸府所在）。**北齐天保元年（550）**高洋受东魏禅、立北齐——延续两都制：邺都为正都、晋阳为陪都；**北齐皇室常年往返于邺-晋阳两都之间**——途中经磁州 + 鼓山响堂山一段，是两都道上最显的山岩地段。**这一两都制是响堂山石窟得以兴起的政治地理基础**——皇室往返、礼佛 + 功德、自然在两都道边的山岩上营窟。

**第二层：北齐高氏皇室开窟（约 550-577）**。响堂山石窟的开凿**具体年份史源不详**——一般认为始于北齐文宣帝高洋天保元年（550）前后、续凿至北齐末年（577）、历三十年间。**北响堂大佛洞**——一般认为是**文宣帝高洋（550-559 在位）** 朝营造、是北齐最早 + 最盛的皇家石窟。**有一说**——**北响堂大佛洞实为高欢（高洋父、北齐高祖、卒于 547）的"瘗骨之所"**：高欢卒于东魏武定五年（547）、葬于何处史源有歧，**一说葬鼓山响堂山的"高欢陵"** —— 这一说出自**唐司马贞《史记索隐》**所引《北齐书》逸文 + 后世**《磁县志》** 等——但**今考古发掘北响堂大佛洞 + 周边并未确认高欢墓**，此说存疑、本 doc 取"北齐皇家功德窟"通说、不取"高欢瘗骨"专说。**北齐文宣帝高洋（529-559）** —— 是北齐开国皇帝、高欢次子、550 年受东魏禅而立北齐；文宣帝一朝（550-559）是北齐国力最盛 + 佛教最兴的时期、亦是响堂山石窟开凿最盛的时期。**注**：文宣帝是**北齐**高洋、不是**南齐**齐武帝（南齐高帝萧道成、武帝萧赜）——南齐是南朝齐（479-502）、与北齐（550-577）相隔半世纪、政权完全不同；这是中古史源最容易混淆的一组朝代名、本 doc 严守北齐高氏口径。

**第三层：北齐文化中心（550-577）**。**北齐 27 年建都邺都**——是北朝后期最盛的文化中心：**佛教兴盛**（响堂山石窟 + 邺都诸寺 + 河北诸州石窟皆北齐遗产）+ **胡汉文化交融**（宫廷胡风浓厚、北齐胡乐胡舞）+ **律令制度**（《北齐律》为唐律先声）。响堂山石窟正是**北齐佛教兴盛在邺都北郊的物理 anchor**——是北齐皇室 + 邺都僧团 + 鼓山岩壁三者共同造就的功德工程。**北齐承光元年（577）正月**——北周武帝宇文邕灭北齐、邺归北周；**北周建德三年至六年（574-577）武帝灭佛运动**——是响堂山石窟受冲击最重的一段：窟内造像多被毁坏 + 部分石经被凿去 + 窟前寺院多遭破坏。**北周大象二年（580）杨坚焚邺**——邺都殿宇尽毁；**但响堂山石窟在邺都西北六十里的山岩之上、未直接遭焚**——是北齐遗产中保存最完好的一组。

**隋唐间**——响堂山石窟受三次重要冲击 + 三次重要修复：
- **北周武帝灭佛（574-577）**——窟内造像多毁；
- **隋开皇间（581-600）** 隋文帝复佛 + 重修响堂山诸寺——窟前木构寺院多在此时重建；
- **唐贞观至天宝间（627-755）**——响堂山是磁州 + 邯郸郡境内的佛教圣地、地方僧团续修；**响堂山亦续凿一些唐代小龛**（北响堂 + 南响堂今存唐代龛多在北齐主窟旁的山岩上、史源不一）；
- **唐武宗会昌灭佛（841-846）**——是响堂山在唐代受冲击最重的一段、但 755 时尚未发生；
- **本 doc 严守 755 时态**——会昌灭佛属"755 后"事件、归历史长河段。

### 此刻

天宝十四载这一年的响堂山——属唐河北道**磁州/邯郸郡（治滏阳县）境内**、距治滏阳约三十里。**注**：磁州**天宝元年（742）正月改磁州为邯郸郡**、天宝十四载（755）十月时仍称邯郸郡；**乾元元年（758）二月**才复称磁州——本 doc 取 755 邯郸郡口径、行文兼用"磁州/邯郸郡"以利唐人识别。

**北响堂**：鼓山西麓——大佛洞 + 释迦洞 + 刻经洞 等九大窟皆在；**大佛洞**中尊大佛（高三丈 / 约 7 米）虽经北周灭佛冲击 + 隋唐补修、面相已非全然北齐原貌、但形制大致存；**窟前木构寺院**——隋开皇间重建、唐贞观至天宝间多次修缮、是磁州境内最大的几座佛寺之一（俗称"北响堂寺"）。**南响堂**：鼓山东南麓——华严洞 + 般若洞 + 释迦洞 等七大窟皆在；**华严洞窟顶八角藻井 + 四壁法华经变** —— 是北齐石窟艺术的代表、唐人来南响堂多观此窟；**窟前寺院** 较北响堂略小（俗称"南响堂寺"）。

**响堂山的物理位置**：**磁州治滏阳县西南三十里**；**邺城遗址在响堂山东南六十里**——755 时**邺城已废 175 年**（580 杨坚焚邺→755）、是相州 + 邺县境内的一片夯土残墟；响堂山则是邺都北郊唯一仍活的北齐遗产 + 礼佛之所。**邺-晋阳道**：自邺都北上、经磁州治滏阳、过鼓山响堂山一段、再北行抵晋阳——755 时这条官道仍是河北道连接太原府的主干道；响堂山正在道边的山岩之上、是过路商旅 + 僧人 + 官员的常驻停留点。

**唐人对响堂山的认知**——**北齐遗产 + 磁州佛教圣地 + 邺都余响**三重叠加：
- **北齐皇家功德的物理 anchor**——盛唐文人凡至磁州、邯郸郡境多知响堂山是北齐皇室所凿；
- **磁州境内最大的佛教圣地**——磁州人 + 邯郸郡居民日常礼佛之所；
- **邺都废墟的"活遗产"**——邺都本身已废、响堂山是邺都北齐文化在物理上唯一仍立的延续。

**安史前夕**（即 755 这一刻）——再过半月（农历十一月初九）安禄山起兵于范阳；自范阳南下取河北诸州、**磁州 + 邯郸郡 + 邺县皆在叛军南下必经之路上**。**响堂山自身是山中石窟、不直接成战场**——但**邺都旧址 + 磁州治** 即将成为安史之乱中后期的关键战场。

### 此刻意味

对天宝十四载的磁州/邯郸郡人来说——响堂山是**州西三十里那座"北齐皇家石窟"**：岁岁礼佛、月月有商旅过、年年有外州僧人来访。对盛唐文人来说——响堂山是**北齐文化在邺都废墟之外保存最完好的一组物理 anchor**：邺都已废 175 年、北齐文宣帝高洋的宫殿尽毁、唯有鼓山岩壁上的大佛洞 + 华严洞——把北齐皇室的功德 + 信仰 + 艺术 凝固在石头里、留与盛唐人凭吊。

唐人尚不知道——**再过四年（乾元二年/759）三月**，邺城旧址周围（即磁州 + 邯郸郡 + 相州一带）将爆发安史之乱中唐军最大的一场败仗：**邺城之败 / 相州大败**——郭子仪、李光弼等九节度使奉肃宗诏围安庆绪于邺、约二十万唐军（号称六十万）围邺四个月、城中粮尽人相食；史思明自范阳率十三万救邺——会战于邺城北、唐军大溃。**响堂山一带是这场大战的物理背景**——磁州 + 邯郸郡境内多有唐军 + 叛军过境、虽响堂山自身未直接受战火、但邺都旧址的北郊在 758-759 年间是中国中古最戏剧的一场战场。**注**：本 doc 严守 755 时态、邺城之败属"755 后"事件、归历史长河段。

唐人更不知道——再过近一千二百年（1961），**响堂山石窟列第一批全国重点文物保护单位**（国家文物局 1961 年公布、第一批 4 号）；再过近一千三百年（1990s-2010s），邯郸市文物研究所 + 北京大学考古文博学院系统调查响堂山——确认北响堂 9 大窟 + 南响堂 7 大窟 + 数百小龛 + 千余造像——是北朝石窟保存最完整的一组之一。**今游人来响堂山**——所见的是鼓山岩壁上的北齐石窟 + 大佛洞中尊大佛 + 华严洞四壁经变——**与 755 唐人所见的物理景观在大轮廓上是连续的**：北齐皇室的功德 + 鼓山的岩壁 + 千四百多年后仍立的造像——未易。

## 子条目

### 北响堂（鼓山西麓 / 北齐文宣帝一朝始凿）

**位鼓山西麓主峰北侧山腰**——是响堂山系开窟最早 + 规模最大的一组；**北齐文宣帝高洋（550-559 在位）一朝始凿**——具体始凿年史源不详、一般认为天保元年（550）前后；续凿至北齐末（577）、历三十年。**现存九大窟**：自南向北依次为**大佛洞**（中窟、最盛）+ **释迦洞** + **刻经洞**（北齐刻经的最早实例之一）等；窟内造像 + 千佛龛 + 刻经皆北齐皇家精品。**大佛洞**——是北响堂规模最宏者：窟高五丈余（约 12 米）、内立大佛三尊（中尊高三丈 / 约 7 米）、覆斗藻井 + 四壁千佛龛——是北齐皇家石窟最盛之作。**有一说**——**大佛洞下层石室为高欢"瘗骨之所"**（高欢卒于东魏武定五年/547、葬地史源有歧）；此说出自唐司马贞《史记索隐》引《北齐书》逸文 + 后世《磁县志》——但今考古发掘并未确认高欢墓；本 doc 取"北齐皇家功德窟"通说、不取"高欢瘗骨"专说。**北响堂窟前寺院**——隋开皇间重建、唐贞观至天宝间多次修缮、俗称"北响堂寺"；755 时是磁州 + 邯郸郡境内最大的几座佛寺之一。

### 南响堂（鼓山东南麓 / 北齐武成帝至后主续凿）

**位鼓山东南麓**——距北响堂约**十五里**（约 7.5 公里）；**北齐武成帝高湛（561-565 在位）至后主高纬（565-577 在位）间续凿**——略晚于北响堂、是响堂山系第二组。**现存七大窟**：**华严洞** + **般若洞** + **空洞** + **释迦洞** + **千佛洞** 等；窟内造像 + 法华经变 + 维摩诘经变 + 千佛龛——是北齐晚期石窟艺术的代表。**华严洞**——窟顶八角藻井 + 四壁雕华严经七处九会变相 + 飞天 + 伎乐天 + 化生童子——是南响堂规模最宏 + 雕刻最精的一窟。**南响堂雕刻精度高于北响堂**——但规模略小；二者合起来构成响堂山系的完整面貌：北响堂以规模 + 北齐早期皇家气象胜、南响堂以精度 + 北齐晚期艺术成熟胜。**南响堂窟前寺院**——隋唐间重建、俗称"南响堂寺"；规模较北响堂寺略小。

### 北齐两都制（邺-晋阳道）

**响堂山石窟得以兴起的政治地理基础**——是北齐独特的两都制：
- **邺都**（今河北临漳邺城遗址）——北齐法理上的都城、皇帝所在；
- **晋阳**（今山西太原晋阳古城）——北齐实际的权力中心、高氏霸府所在；自高欢 534 入邺以来即坐镇晋阳遥控邺都朝廷。

**北齐皇室常年往返邺-晋阳两都之间**——途中经**磁州治滏阳 + 鼓山响堂山** 一段、是两都道上最显的山岩地段。**响堂山正在邺-晋阳道边的山岩之上**——是北齐皇室往返两都途中礼佛 + 营葬 + 营窟的物理 anchor。**鼓山响堂山的"响堂"二字**——一说源于山岩中空 + 人语山响 + 击石如鼓；一说源于北齐皇室在邺-晋阳道上闻山响而开窟——本 doc 取通行的"山岩中空"说。

### 文宣帝高洋（北齐开国 / 响堂山开凿之主）

**文宣帝高洋（529-559）**——是北齐开国皇帝、高欢次子、高澄之弟；**北齐天保元年（550）五月**——高洋受东魏孝静帝禅、自立为帝、改元天保、立北齐——是北齐第一位皇帝。**文宣帝一朝（550-559）**——是北齐国力最盛 + 佛教最兴的时期：**北征柔然 + 西击北周（前身西魏）**、巩固北方版图；**邺都 + 晋阳两都营建大兴**——邺南城在东魏末已建成、文宣帝朝续扩 + 营建北齐宫城；**响堂山石窟在此朝始凿**——一般认为大佛洞即文宣帝朝功德窟。**文宣帝晚年（556-559）多酒醉狂悖** —— 是北齐皇室"狂帝"传统的开端；天保十年（559）十月卒、葬武宁陵。**注**：文宣帝是**北齐**高洋——不要与**南齐**齐武帝（南齐武帝萧赜、464-493）混淆；南齐是南朝齐（479-502）、北齐是北朝齐（550-577）、二者相隔半世纪、政权完全不同；这是中古史源最容易混淆的一组朝代名。**亦不要将高洋与高欢（高洋父、东魏权臣）或高澄（高洋兄、东魏文襄帝追谥）混淆**——高欢卒于东魏武定五年（547、未称帝）、高澄卒于东魏武定七年（549、被刺、追谥文襄帝）、高洋是真正建立北齐 + 称帝的人。

### 北齐皇家石窟传统（响堂山 + 邺西诸窟）

**北齐皇室在邺-晋阳道沿线营窟造像**——是北朝石窟史上独立的一段：
- **响堂山石窟**（南北响堂 / 磁州 / 邯郸郡）——北齐皇家最盛的功德窟；
- **小响堂石窟**（邺西林虑山一带 / 相州境）——北齐皇室次级功德窟；
- **天龙山石窟**（晋阳西南天龙山 / 太原府境）——北齐文宣帝朝在晋阳近郊营造（部分窟为东魏始凿）；
- **义县万佛堂**（辽西 / 北齐辖境之外 但受北齐石窟艺术影响）；
- **邺都诸寺**（邺南城内 / 已随邺都焚毁）——多以塑像而非石窟为主。

**响堂山石窟是北齐皇家石窟传统的代表 + 现存最完好的一组**——是研究北朝晚期石窟艺术 + 北齐宫廷美学 + 北齐佛教制度的最重要物理资料。**响堂山雕刻特色**——**北齐皇家气象 + 胡汉文化交融 + 法华经维摩诘经变盛行 + 大像传统（受云冈影响 + 启隋唐大像之先）**——是北朝晚期石窟艺术的成熟之作。

### 北周武帝灭佛对响堂山的冲击（574-577）

**北周建德三年至六年（574-577）武帝宇文邕灭佛**——是响堂山石窟在北齐末年（577 邺归北周）受冲击最重的一段：**窟内造像多被毁坏**（北响堂大佛洞中尊大佛 + 胁侍像皆有损 + 部分千佛龛被凿去）+ **部分石经被凿去**（北响堂刻经洞的北齐刻经多有损）+ **窟前寺院多遭破坏**（北齐时北响堂寺 + 南响堂寺皆受冲击）。**这一次灭佛是中国佛教史"三武灭佛"之第二武**（前为北魏太武帝、后为唐武宗）——是响堂山在中古受冲击最重的两次之一（另一次是唐武宗会昌灭佛 841-846、属"755 后"事件归长河）。**注**：北周武帝宇文邕（543-578、560-578 在位）的"邕"字与高欢之孙的"虎"讳无关——是中古最容易混淆的一组讳点（详 邺城 doc 中关于后赵石虎讳金虎台改金凤台的口径）；本 doc 不引"虎"讳话题。

### 响堂山的考古调查（1990s 起）

**自 1990s 起**——邯郸市文物研究所 + 北京大学考古文博学院 + 河北省文物局多次系统调查响堂山石窟：
- **1991-1995 北响堂调查**：确认 9 大窟 + 数十小龛 + 北齐主窟形制；
- **1996-2000 南响堂调查**：确认 7 大窟 + 数十小龛 + 北齐晚期雕刻精度；
- **2001-2010s 综合调查 + 数字化**：建立响堂山石窟数字档案 + 出版多卷调查报告 + 与海外流失造像的回归对话（响堂山在 1920s-1930s 间多有造像被盗凿流失海外、美国弗利尔美术馆 + 大都会艺术博物馆藏有相关响堂山造像）。

**今响堂山是中国北朝石窟保存最完整的一组之一**——与云冈 + 龙门 + 麦积山 + 敦煌等大窟并称"中国十大石窟"之一；是北齐皇家石窟唯一保存完整的一组。

## 历史长河（755 → 今）

**中晚唐**　**乾元二年（759）三月相州大败**——郭子仪 + 李光弼等九节度使围安庆绪于邺、史思明南救、唐军大溃——这是安史之乱中唐军最大的一场败仗（详 [邺城](./邺城.md) doc）；**响堂山一带是这场大战的物理背景**——磁州 + 邯郸郡境内多有唐军 + 叛军过境、虽响堂山自身未直接受战火、但邺都旧址北郊在 758-759 间是中国中古最戏剧的一场战场。**唐武宗会昌五年（845）灭佛**——是响堂山在唐代受冲击最重的一段：**窟前寺院（北响堂寺 + 南响堂寺）多遭拆毁** + **窟内造像 + 经版多被凿去**；这一次灭佛是"三武灭佛"之第三武、与北周武帝灭佛 + 北魏太武帝灭佛并列。**唐宣宗大中元年（847）复佛**——响堂山诸寺多在此后修复。

**五代 + 两宋**　**五代后梁后唐间**磁州在魏博节度使辖境——响堂山诸寺多次毁修；**北宋初**磁州在河北西路、响堂山是磁州境内最大的佛教圣地之一。**北宋庆历八年（1048）黄河北流改道**——黄河自澶州（今河南濮阳）北决、改东北向流入海、漳水下游水系大变——但响堂山所在的鼓山是滏水（漳水南支）发源地、与黄河改道关系较远、响堂山自身未受冲击。**南宋初**响堂山在金国辖境（1127 靖康之难后河北归金）；**金代响堂山续有重修** —— 金人对北朝佛教遗产相对宽容、响堂山诸寺多在金代续存。

**金元**　**金元间响堂山诸寺多次毁修** —— 金大定间（1161-1189）+ 元至元间（1264-1294）多有重修；**元代响堂山是磁州（元彰德路磁州）境内最大的佛教圣地之一**——多有藏传佛教元素加入（藏传佛教在元代北传影响北方诸寺）。**元末明初**响堂山诸寺受元末战乱影响略有损坏。

**明清**　**明嘉靖间**响堂山诸寺大修——立"响堂山石窟"碑（嘉靖三十六年/1557 立、与邺城曹魏故址碑同年立、是明代华北佛教考古的两通重要碑）；**清乾隆四十年（1775）**陕西巡抚毕沅过磁州考察响堂山（毕沅作《中州金石记》收录响堂山碑刻 + 北齐造像题记）；**清末响堂山** 是磁州 + 河南北部 + 山西东南交界一带最重要的佛教石窟群 + 文人凭吊之所。**1920s-1930s 间响堂山多有造像被盗凿流失海外**——美国弗利尔美术馆 + 大都会艺术博物馆 + 部分欧洲博物馆藏有响堂山造像——是中国近代石窟流失最严重的几组之一。

**民国至今**　1949 年后响堂山石窟列河北省重点文物保护单位、**1961 年响堂山石窟列第一批全国重点文物保护单位**（国家文物局 1961 年 3 月 4 日公布、第一批 4 号；与云冈 + 龙门 + 麦积山 + 敦煌等同批列入）。**1990s 起邯郸市文物研究所 + 北京大学考古文博学院系统调查**——确认北响堂 9 大窟 + 南响堂 7 大窟 + 数百小龛 + 千余造像；建立响堂山石窟数字档案 + 出版多卷调查报告；**与海外流失造像的回归对话**——部分响堂山造像在 21 世纪初通过数字化复原回归 + 海外博物馆合作展览的方式 与原窟址在虚拟与物理上重新连接。**2010s 响堂山石窟保护 + 展示工程**——窟前栈道修缮 + 防风化处理 + 数字化展厅建成；响堂山是华北平原最重要的北朝石窟群、与云冈 + 龙门 + 麦积山 + 敦煌等大窟并称"中国十大石窟"之一。**今游人来响堂山**——所见的是鼓山岩壁上的北齐石窟 + 北响堂大佛洞中尊大佛 + 南响堂华严洞八角藻井 + 千余北齐造像——**与 755 唐人所见的物理景观在大轮廓上是连续的**：北齐皇室的功德 + 鼓山的岩壁 + 千四百多年后仍立的造像——未易。
