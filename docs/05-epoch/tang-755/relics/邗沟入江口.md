# 邗沟入江口

## 一句话

**邗沟水道与长江交汇的漕运南端启动点**——春秋吴王夫差前 486 开邗沟通江淮于此入江、隋炀帝大业元年（605）重浚拓宽、唐 755 时是淮南道广陵郡江都县南三十唐里长江北岸的水道分界点——是盛唐江淮粮米北上洛阳的物理起点、亦是江都 + 瓜洲 + 入江口"江北水路三镇"中职能最专的一处。

## 形貌

邗沟入江口在唐淮南道**广陵郡（天宝元年/742 改扬州为广陵郡，治江都县）江都县南约三十唐里**长江北岸——具体位置在江都县治（即今扬州老城区）南郊、长江北岸；按今制约合 15 公里、邗沟南段自江都城南行至此入长江。**地势**：长江下游北岸冲积平原边缘、长江北岸一带的低洼地；**邗沟**——是自江都向北通山阳（楚州治；今淮安）入淮的人工运河、长约二百余唐里（约 110 公里）；**入江口处**——邗沟水道与长江水道交汇、是江北漕舟自邗沟入长江南行、或自长江入邗沟北上的水道分界点。

**入江口的物理形态**——是一片江北沙洲与水道相错的湿地：**江口处有沙洲**（江北边滩 + 江中沙洲、随江水涨落出没）；**沙洲与江北岸已基本相连**（至唐 江岸南退、沙洲与北岸渐成半岛）；**邗沟南段经此入江**——水道宽数十丈、深可行漕舟与江船。**入江口正西约二十唐里为瓜洲渡**（长江南北两岸之间的渡口；江北 ↔ 江南）；**入江口与瓜洲渡功能不同**——**瓜洲渡是渡口**（两岸往来）、**邗沟入江口是水道分界点**（漕运起讫）；两地地理近、但职能截然——**这是 755 时唐人对江都南郊水陆体系的核心区分**。

**入江口周边水系**——是江都南郊最密的一段：**北约三十里为江都县治**（即今扬州老城区、广陵郡治所在）；**正西约二十里为瓜洲渡**（长江南北两岸渡口）；**正南为长江北岸主流**（自西向东流、至此江面宽阔约十数里）；**东南方向约六十里为江都县东境海陵县**（今泰州；唐扬州属县、距长江入海口尚近百里）。**入江口正位于这片水网交汇处**——南可入长江顺流东下入海、北可循邗沟上溯入淮、西可循江南河（隋大业六年/610 开）南渡入江南——是江北水路的总枢之一。

**唐 755 时入江口**——是江都县南郊一处漕舟商旅辐辏的水道节点。**邗沟入江口本身**——是水道交汇处、不设独立县治；**江都县治在今扬州老城区**（距入江口约三十里北）；**瓜洲渡在入江口正西约二十里**——两处常被唐人合称"江都南口"或"广陵江口"。**入江口的漕政常制**——属淮南采访处置使 + 广陵郡 + 江都县三级辖治（节度使为安史之后置）；漕舟过此 + 转运使督运 + 关吏验牒——是盛唐江淮漕路最南端的一道行政关口。

物候：长江下游北岸、亚热带季风——四时分明、春多桃花水、夏有江水暴涨 + 邗沟水位上升、秋高、冬有薄雪不冻江——盛唐人秋日自江都南行至入江口多记"江口风潮、漕舟接连"。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

邗沟入江口在唐人的回望里——是**中国人工运河史最古的物理 anchor 之一 + 隋唐漕运体系的南端启动点**。**整段邗沟的历史从春秋末至 755 已 1240 年**（前 486 → 755，无公元 0 年）、由四次重大节点构成、是唐人通过《左传》《史记》《隋书》等正史得以系统知晓的一段水道史。

**春秋末：吴王夫差开邗沟（前 486）**。**鲁哀公九年（前 486）**——吴王夫差为北上中原争霸、命掘邗沟、自江都（吴 时称邗、邗城）通至淮河（在末口、今淮安东北）；**《左传·哀公九年》**："吴城邗、沟通江淮"——是邗沟开凿的最早记载。**夫差开邗沟**——非纯为漕运、而是为北上军事调兵 + 运粮；**邗沟开成后**——吴军循邗沟北上、艾陵之战（前 484）败齐、是吴霸的物理基础之一。**注**：**夫差所开邗沟**——自江都向北、引江水入淮、长约二百余里、是中国最早的人工运河之一（与战国时代魏惠王开鸿沟约同期、邗沟略早）；**夫差所开之"江"**——即今长江、入江口即今邗沟入江口的物理前身——但 1240 年间江岸南退、入江口具体位置已数度迁移、755 时已非夫差时的原点。

**战国 + 秦汉 + 魏晋南北朝**——邗沟续用、但中原至江淮的水路始终未联通；**两汉间邗沟为江淮漕路**——荆扬粮米北上汴洛皆赖此；**魏晋间多次浚治**——曹魏邗沟续用、晋陈敏 + 邓艾等先后浚邗沟入江口段；**南北朝间南朝以江都为江北重镇**——邗沟入江口是建康（南朝都）西北方向的水路通道之一。**注**：**邗沟在春秋至隋间凡千余年**——名称略变（邗沟 = 邗江 = 邗水、同名异写）、但物理走向基本稳定；本 doc 统作"邗沟"。

**隋大业元年（605）：隋炀帝重浚邗沟**。**隋大业元年（605）三月**——隋炀帝发河南、淮北诸郡民百余万开**通济渠**（自洛阳板渚引黄河水东南至盱眙入淮）；**同年（605）**——隋炀帝**重浚邗沟**（拓宽 + 加深 + 立水驿）——非新开、夫差前 486 已开；**自此洛阳与江都水路联通**——隋炀帝御舟下江都即循通济渠 + 邗沟此渠。**注**：**隋炀帝重浚邗沟**——是 1090 年来邗沟最大的一次工程改造（前 486 → 605，无公元 0 年）；浚后邗沟可行隋炀帝御龙舟、深可数丈、两岸植柳成行——是中国人工运河史上 "邗沟 2.0"。**入江口处亦经重浚**——拓宽水道 + 整治沙洲 + 立水驿；隋炀帝三下江都皆经此口。

**隋大业六年（610）：开江南河**——自江都南渡长江、过京口（润州）、至余杭——大运河南段成。**邗沟入江口自此正式成为"邗沟 + 江南河"水道转接的枢纽**——江南河北端即在长江南岸京口（润州；今镇江）、与邗沟入江口隔江相望；江南漕舟自余杭循江南河北上至京口、渡江至邗沟入江口、再循邗沟北上至山阳入淮——**这是隋唐大运河体系下江南至洛阳的完整水路**。**注**：**"江南河"是隋开的南段、不是邗沟**——这是 755 时唐人对大运河三段（永济渠 + 通济渠 + 邗沟 + 江南河）最易混的一笔；本 doc 严守史源、邗沟与江南河分立。

**唐承隋制（618-755）**。**唐高祖、太宗朝**——通济渠 + 邗沟 + 江南河续用、为漕运主线；**高宗 / 武则天朝**——多次"就食洛阳"皆赖此渠输江淮粮米北上；**开元二十二年（734）裴耀卿改漕运法**——三门转运置河阴仓 + 集津仓 + 盐仓三仓分段接驳；**开元二十五年（737）至天宝元年（742）累岁运米七百万石**（《新唐书·食货志三》）——**这七百万石的物理起点即在邗沟入江口**：江南诸道粮米先集江都、再循邗沟北上至山阳入淮、转通济渠西北至洛阳含嘉仓、再陆运过三门峡至关中太仓。**注**：**广陵郡 + 扬州 + 江都州名沿革**——隋大业三年（607）改扬州为江都郡；唐武德三年（620）复扬州；**天宝元年（742）改扬州为广陵郡**（汉旧名）；乾元元年（758）复扬州——**755 时正名是广陵郡、治江都县、城又名"广陵"**；行政文书写"广陵郡"，民间俗称"扬州""江都""广陵"皆可。

### 此刻

天宝十四载这一年的邗沟入江口——**自夫差开邗沟已 1240 年**（前 486→755）；**自隋炀帝重浚已 150 年**（605→755）；属唐淮南道**广陵郡江都县**境内、在江都县治南约三十唐里长江北岸。

**邗沟水道**：自江都城南南流至入江口、长约三十唐里；755 时邗沟水道完整、两岸植柳（隋开邗沟时已成定制、唐承之）、水驿与漕馆相次、漕舟接连——是盛唐江北最稳的水路之一。**入江口的沙洲与岸**：邗沟入江口所在沙洲，唐时已与江北岸相连成半岛（即瓜洲）；江面更南尚有零星江中沙洲随江水涨落出没；**江岸南退**——自春秋至唐凡千余年江岸渐南退、入江口具体位置已数度南移、755 时已比夫差时南移约数十里；这是江岸自然变迁的常态。**漕政常制**：入江口属广陵郡江都县辖、淮南采访处置使督漕（节度建制安史后方设）；漕舟过此 + 转运使督运 + 关吏验牒；漕舟自江都至洛阳行月余、急驿数日即至。

**唐淮南采访处置使 + 广陵郡治**：**淮南采访处置使治广陵郡**（管辖扬、楚、滁、和、寿、舒、庐、光、蕲、申、安等淮南诸州）；**广陵郡治江都县**（即今扬州老城区、距入江口约三十里北）——是江北漕运的总枢；755 时具体使臣名号兹不强定（避免与中唐邓景山等任职年份混淆）。**江都城规模**：唐六雄之一（与长安、洛阳、汴州、益州、并州并列）；"扬一益二"——扬州为天下都会第一、益州（成都）第二；**邗沟入江口是江都城南郊最具象的水路 anchor**——江都城的繁华、最直接的物质源头即在这片江口的漕舟与商旅。

**唐人对邗沟入江口的认知**——**漕运南端启动点 + 江北水路总枢 + 春秋夫差物理遗存**三重叠加：
- **漕运史的物理起点**——盛唐文人凡论江淮漕运、必至邗沟入江口；这里是江南粮米入淮 + 入洛 + 入关中的第一道水门；
- **江北水路三镇**——江都 + 瓜洲 + 入江口构成 江北水路的"三镇"：江都是州治 + 都会；瓜洲是江南北渡口；入江口是漕运分界点——三者地理近、职能分明；
- **春秋夫差遗存**——盛唐文人凡过江都至入江口、必引《左传·哀公九年》"吴城邗、沟通江淮"为典；**邗沟 1240 年的物理 anchor**——是别处难见的时间深度。

**安史前夕**（即 755 这一刻）——再过半月（农历十一月初九）安禄山起兵于范阳；**自范阳南下取河北诸州 + 渡黄河 + 取河南**——**通济渠北端（汴州 + 洛阳）即将失**；**邗沟入江口本身不在叛军南下路线上**——但其所连的漕路即将断绝；**至德元载十二月（756 年 12 月）永王李璘自江陵东下、舟师顺长江欲取广陵（扬州）**——长江下游战事所在；**至德二载（757）二月**——永王败死于长江下游；舟师是否经邗沟入江口未见明载（《资治通鉴》记其至当涂、未及广陵）。**永王败后**——江都未陷、邗沟入江口亦未失；**至德 / 乾元间**——江淮租赋赖此口苦输支撑唐廷半壁；但漕路反复中断——**邗沟入江口此一刻**虽尚承平、再过半年即将成为唐廷江南漕路苦撑的物理起点。

### 此刻意味

对天宝十四载的江都人来说——邗沟入江口是**县南三十里那片"邗沟入江"古口**：漕舟岁岁、商旅时来；邗沟水南流入江 + 江口沙洲相连——构成江都南郊日常生活背景里的水路画面。对盛唐文人来说——邗沟入江口是**中国人工运河史最深的一处时间 anchor**：春秋夫差前 486 开邗沟、至天宝十四载已 1240 年、隋炀帝 605 重浚至今亦 150 年——**1240 年的物理连续**、是别处难见的时间密度。**盛唐文人凡过江都至入江口**——必引《左传》"吴城邗、沟通江淮"；**盛唐文人凡论江淮漕运**——必至入江口为漕路南端起讫。

唐人尚不知道——**再过半年（756 正月东都陷；同年七月肃宗即位后改元至德元载）**，安史叛军将攻陷东都洛阳、通济渠北端断；**邗沟入江口虽未陷**——但漕路自此反复中断；**至德二载（757）二月**永王李璘舟师败覆于长江下游；是否经邗沟入江口附近未明载；**至德 / 乾元间**唐廷半壁赖江淮租赋苦输支撑——**邗沟入江口此一刻**正是这"苦撑"的物理起点。**唐人更不知道**——**南宋以后**（黄河夺淮 + 江岸南退 + 江中沙洲变迁），邗沟入江口的物理位置将彻底改变；至**宋元间**——入江口已不在 755 唐人所见的物理位置；**今扬州南郊江口处**——是后世数百年江岸南退 + 沙洲变迁的产物、与 755 唐人所见的入江口在物理位置上已不连续——是中国人工运河史 + 长江下游岸线变迁史最大的一次"物理改道"。

## 子条目

### 邗沟（春秋夫差开 + 隋大业 605 重浚）

**邗沟 = 自江都（江北）至山阳（淮河南岸；今淮安）的人工运河**——长约二百余唐里（约 110 公里）；是中国最古老的人工运河之一。**春秋末（前 486）**——吴王夫差为北上中原争霸开；《左传·哀公九年》"吴城邗、沟通江淮"——是邗沟开凿的最早记载。**隋大业元年（605）**——隋炀帝与通济渠同时**重浚**邗沟（非新开、拓宽 + 加深 + 立水驿、两岸植柳成行）；自此邗沟可行隋炀帝御龙舟、是中国人工运河史上 "邗沟 2.0"。**唐承之**——邗沟是江南漕路北段；**沿途要节**：山阳（楚州治；今淮安） + 宝应 + 高邮 + 江都（广陵郡治；今扬州）——两岸植柳。**至德 / 乾元间漕路反复中断**——但邗沟本身始终未湮；**北宋元祐七年（1092）苏轼知扬州**多次行此渠；**今邗沟一段尚为京杭大运河扬州段**——是隋唐四段水路系统（永济渠 + 通济渠 + 邗沟 + 江南河）中**唯一未湮的一段**。

### 入江口与瓜洲渡的区分（地理近、职能分明）

**邗沟入江口 ≠ 瓜洲渡**——这是 755 时唐人对江都南郊水陆体系最易混的一笔；二者地理近（相距约二十里）、但职能截然：

- **邗沟入江口**：邗沟水道与长江水道交汇处——是**水道分界点**；漕舟自邗沟入长江南行、或自长江入邗沟北上、皆经此口；属漕运范畴。
- **瓜洲渡**：长江南北两岸之间的渡口——是**两岸往来的渡口**；江北 ↔ 江南、人员 + 商旅 + 使节渡江皆经此；属交通范畴。

**两地的物理关系**——**入江口在瓜洲渡正东约二十里**、皆在长江北岸；**瓜洲渡所对的江南岸**是京口（润州治；今镇江）；**入江口所对的江南岸**是江南河北端（隋大业六年/610 开、自京口至余杭）。**唐人凡论江都南郊水陆**——皆作"江都 + 瓜洲 + 入江口"江北水路三镇并称、各专其职；本 doc 严守此分别。

### 隋炀帝重浚（605）与通济渠 + 江南河的水路系统

**隋大业元年（605）**——隋炀帝发河南、淮北诸郡民百余万开**通济渠**（自洛阳板渚引黄河水东南至盱眙入淮）+ **同年重浚邗沟**——自此洛阳与江都水路联通；**隋大业六年（610）**——开**江南河**（自江都南渡长江、过京口、至余杭）——大运河南段成。**自此隋唐大运河四段水路系统形成**：

- **永济渠**（大业四年/608 开）：自洛阳引沁水北至涿郡——大运河北段；
- **通济渠**（大业元年/605 开）：自洛阳板渚东南至盱眙入淮——大运河中段北线；
- **邗沟**（春秋夫差开、大业元年/605 重浚）：自山阳至江都——大运河中段南线；
- **江南河**（大业六年/610 开）：自江都南渡长江过京口至余杭——大运河南段。

**邗沟入江口**——是通济渠 + 邗沟（北线）与江南河（南线）的水路交接点；隋炀帝三下江都皆经此口；唐承隋制、入江口续作漕运南端启动点。**注**：**"京杭大运河"作为名词**——主要为元代会通河（1289 成）+ 通惠河完成后形成北至大都南至杭州的连续水道、是近代以来概念；**755 时尚无"京杭"概念**——唐人称"通济渠 + 邗沟 + 江南河"三段。

### 漕运（江淮粮米北上的南端起点）

**邗沟入江口是盛唐漕运的物理起点之一**——**江南诸道粮米先集江都、再循邗沟北上至山阳入淮、转通济渠西北至洛阳含嘉仓、再陆运过三门峡至关中太仓**。**开元二十二年（734）裴耀卿三门转运体系**——置河阴仓 + 集津仓 + 盐仓等三门转运仓接驳，分段输运、避三门峡之险；**开元二十五年（737）至天宝元年（742）累岁运米七百万石**（《新唐书·食货志三》）；**这七百万石的物理起点即在邗沟入江口**——是盛唐财政的"南端水门"。

**安史中漕路反复中断**——**天宝十四载十二月丁酉**叛军陷东京洛阳（公历入 756 年 1 月）——通济渠北端起点失；**江淮漕路自此中断**；**广陵郡 + 淮南采访处置使（安史后改置淮南节度）未陷**——成为唐廷江南半壁的物理 anchor；**邗沟入江口此一刻**虽尚承平、再过半年即将成为唐廷江南漕路苦撑的物理起点。**刘晏大历五年（770）起再任漕运改漕运法**——立**江船 + 汴船 + 河船 + 渭船**四段接驳（江船自江南诸州集江都；汴船自江都经邗沟入淮 + 转通济渠西北至河阴仓；河船自河阴仓溯黄河西上、三门险段陆运转载；渭船自渭水入长安太仓）——**此为中唐口径，755 时尚未实施**。**邗沟入江口在刘晏漕运改革中**——续为江船 + 汴船的水路交接点、职能未变。

### 永王李璘东巡（756-757）与邗沟入江口

**至德元载十二月（756 年 12 月）**——永王李璘自江陵东下、舟师顺长江欲取广陵（扬州）、谋自立。**永王舟师东下路线**——自江陵东出过武昌、循长江下流；**《资治通鉴》记其至当涂**（即今安徽马鞍山一带、距广陵尚有数百里）；**舟师是否曾东抵邗沟入江口附近水域、史无明载**。**至德二载（757）二月**——肃宗命**高适、来瑱、韦陟**等讨之；**永王败死**、舟师覆于长江下游；**江都未陷**——邗沟入江口与广陵郡治皆守住、唐廷江南半壁的物理 anchor 得保。

**注**：**永王李璘事**——李白《永王东巡歌》十一首、为永王作；后李白因此事系狱、流夜郎、是李白晚年最大的政治事件。**邗沟入江口与永王舟师**——是 755-757 三年间江都南郊水域上最戏剧的一段史；本 doc 严守史源、永王败死于至德二载二月、舟师覆于长江下游；是否曾抵邗沟入江口附近水域未见明载。

### 江岸南退与入江口位置变迁（春秋至唐）

**邗沟入江口的物理位置**——千余年来数度迁移、非一处不变：

- **春秋末（前 486）夫差开邗沟时**——入江口在江都（吴 时邗城）城南、距江都城约数里；
- **两汉魏晋间**——江岸南退约数十里、入江口随之南移；
- **隋大业元年（605）重浚邗沟时**——入江口在江都南郊约二十里、隋炀帝御舟即在此入江；
- **唐 755 时**——入江口在江都南郊约三十唐里、本 doc 所记的位置；
- **南宋以后**——江岸继续南退 + 江中沙洲变迁、入江口物理位置彻底改变；至元代——入江口已不在 755 唐人所见的物理位置；
- **今扬州南郊江口处**——是后世数百年江岸南退 + 沙洲变迁的产物、与 755 唐人所见的入江口在物理位置上已不连续。

**注**：**江岸南退**——是长江下游岸线变迁的常态、非邗沟入江口独有；瓜洲渡的位置亦在千余年间数度迁移——皆是长江下游"江岸自然南退"+"江中沙洲淤积"两重物理作用的产物。**今扬州南郊江口与 755 唐人所见的入江口在物理位置上已不连续**——这是中国人工运河史 + 长江下游岸线变迁史最大的一次"物理改道"。

## 历史长河（755 → 今）

**中晚唐**　**755-763 漕路反复中断**——但邗沟入江口本身始终未湮；**至德二载（757）二月**永王李璘舟师败覆于长江下游（是否经邗沟入江口附近水域未见明载）；**广德元年（763）刘晏改漕运法**——四段接驳重立、邗沟入江口续作江船 + 汴船的水路交接点；**自此中唐唐廷半壁赖江淮租赋苦输支撑**——邗沟入江口是这套体系的物理起点。**晚唐杜牧（803-852）任扬州淮南节度掌书记**（833-835）——杜牧"二十四桥明月夜"等扬州诗作晚唐扬州；**杜牧诗中常提江口 + 邗沟**——是晚唐文人对盛唐扬州的回望。**唐末黄巢（879-881）自岭南北上**——曾经扬州；**唐末漕路彻底崩溃**——是唐廷财政的终结之一；**邗沟入江口在唐末**仍是江北水路的物理起点、但漕舟稀疏。

**五代 + 两宋**　**五代后梁、后晋、后汉、后周以开封（汴州）为都；后唐都洛阳**——通济渠汴渠段（汴州一线）仍为五代核心漕道；**邗沟入江口在五代**——续作江北水路起点；**北宋建都汴京（开封）**——通济渠汴渠段是 200 年盛世的物质基础；**邗沟入江口在北宋**——是江南漕运至汴京的物理起点；**北宋元祐七年（1092）苏轼知扬州**多次行此渠、过入江口。**南宋绍熙五年（1194）黄河南徙夺淮**——通济渠中下段被黄河水冲毁；**自此通济渠湮废**——但邗沟续通、**邗沟入江口续作江南漕运至临安（南宋都）的物理起点**——只是漕路改循江南河 + 邗沟 + 京口、不再经通济渠。

**金元**　**金元间黄河多次改道 + 江岸南退**——邗沟入江口的物理位置继续南移；**元代会通河**（1289 成）——自济宁北至临清、东至须城——走山东直插北方、与隋唐通济渠走向不同；会通河 + 通惠河完成后形成北至大都南至杭州的连续水道——**邗沟入江口在元代**——是会通河南段（即邗沟段）的入江点；**入江口物理位置已与 755 唐人所见不再连续**。**注**："京杭大运河"作为名词主要为近代以来概念、元代尚作"通济河"或"漕河"。

**明清**　**明清间会通河 + 邗沟 + 江南河成主漕路**——通济渠彻底退出；**邗沟一段（淮扬运河）+ 江南河（江南运河）至今仍通**；**明嘉靖 + 清乾隆间江岸继续南退 + 江中沙洲变迁**——入江口物理位置数度迁移；**清末扬州古运河南端**（即今扬州南郊江口）——是后世数百年江岸南退 + 沙洲变迁的产物。**注**：**瓜洲渡在明清间**多次随江中沙洲变迁而迁移、**至民国 + 现代瓜洲仍在长江北岸（今扬州邗江区瓜洲镇）**，但江岸南退使瓜洲与盛唐物理位置略有迁移——这是江口区域整体南退的最直观证据。

**民国至今**　**1949 年后京杭大运河扬州段（即邗沟段）续通**——是隋唐三段大运河中唯一未湮的一段；**今扬州古运河 + 京杭大运河扬州段两岸**——尚有近代补植柳；**邗沟入江口的现代位置**——在今扬州南郊瓜洲一带、与 755 唐人所见的入江口在物理位置上已不连续；**2014 年大运河（含邗沟段）列联合国教科文组织世界文化遗产**——是中国人工运河史的全球性认证。**今游人重走邗沟入江口**——所见的是**现代扬州古运河 + 京杭大运河扬州段 + 长江北岸今貌**——盛唐人在 755 所走的那片入江口、千二百多年后**物理位置已迁移**（江岸南退数十里）、**水道职能则连续**（漕运 + 商旅起讫至今）；驿馆 + 渠柳几无原物——但**邗沟入江口作为中国人工运河史最深的一处时间 anchor**（自夫差前 486 已 2511 年至今）——是别处难见的时间深度。
