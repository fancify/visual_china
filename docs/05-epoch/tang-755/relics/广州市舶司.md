# 广州市舶使驻所

## 一句话

**中国第一个专管海外贸易的官职** + **海上丝路东端的财政咽喉**——**开元二年（714）史源首见为安南市舶使（周庆立）；广州为后起主设地，具体年份阙**；755 时由宦官以"中使"身份充任，登舶验货、抽分进奉；**是盛唐财政、外交、海舶贸易三股线在岭南海岸打的一个结**。

## 形貌

须先辨一桩名号——**唐时是"市舶使"，不是"市舶司"**。

"市舶使"是**官名 + 差遣**——皇帝派遣的一种使职，以"中使"（宦官）身份临时出岭南，监收海舶、为内廷采办珍奇；**没有衙署常设的正式机构**，市舶使在广州时寄治于南海郡太守衙或广州都督府衙之偏厅，差遣完毕便回长安复命。**北宋开宝四年（971）广州正式置市舶司**——是市舶作为常设衙署、有专职印信、定额属吏的开端；**元丰三年（1080）元丰市舶改制**——宋神宗厘定《元丰市舶条例》，由转运司或路一级文臣兼领、制度化定型。今人写"唐广州市舶司"——是后世从北宋反推上去的称呼；755 这一刻广州城里只有"市舶使"，没有"市舶司"。

但即便如此，市舶使所驻之地——**就在广州城内**。具体位置盛唐文献阙记，按制大致在广州都督府衙署左近、近珠江北岸——便于登舶验货。海舶系缆于城南门外珠江岸边，市舶使下衙登船——先验货、再抽分、最后开市。一座没有自家衙署的"使"，因此把整个广州城最热闹的一段江岸变成了它的临时官署。

市舶使的职掌，《唐会要·卷六十六》"市舶使"条与《通典·食货六》大致可拼出三事：

- **登舶验货**——海舶到岸，市舶使先登船清点货品；珍奇之物**先尽进奉**（送内廷），其余才入市；
- **抽分征课**——按船货种类税率不一，**香药、宝石、犀角、象牙、珠贝、玳瑁、玻璃**等为大宗，由市舶使收"舶脚"（关税）与"下碇税"（停泊税）入官；
- **收市进奉**——以官钱"和买"舶商高价珍品（实际多是低价强购），其余货品准入广州市易。

蕃货的具体种类，盛唐人见得最多的——**大食的乳香、沉香、苏合香、龙涎；波斯的瑟瑟（青金石）、玛瑙、玻璃器；师子国（斯里兰卡）的宝石、珊瑚；天竺的胡椒、檀香；占婆、林邑的犀角、沉香；诃陵（爪哇）的象牙、玳瑁**——一船所载，往往值钱数万贯，**等同关中一州一年的赋入**。这是盛唐市舶使在唐廷财政里份量奇重的根本——一艘海舶一抽分，所得便顶得过一州赋税。

市舶使本人**是宦官**——这是盛唐市舶使的另一个特点。**唐前期内廷宦官以"中使"身份出外差遣**：盐铁、漕运、市舶、监军——是玄宗朝以后宦官势力渐起的标志之一。市舶使的人选由皇帝亲点、**直接对内廷负责**——这意味着海舶抽分的财货很大一部分直接入内库（皇室私库），而**非户部国库**。盛唐市舶使因此既是国家的财源、又是皇家的私库——这一身二任，是后世市舶制度反复改革的根因。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

广州市舶使在唐人眼里——是**中国海外贸易制度的第一份正式名册**。

自汉武帝元鼎六年（前 111）灭南越置南海郡起——南海一线的海舶贸易便由**南海郡太守 + 番禺县令**兼管；这一段近八百年里，海舶岁岁来集，但**无专使、无定法**：太守征课、舶商纳贡，往往随心所定——史称"郡守自专"。两汉、孙吴、晋宋齐梁陈、隋——**对外贸易由地方官兼理，没有中央层面的专设机构**。

入唐——**武德、贞观、永徽间**，广州海舶贸易已颇为发达：《唐会要》《新唐书·南海传》皆载贞观永徽间大食、波斯、师子国、诃陵之舶岁至广州——但市舶仍由广州刺史兼管。**变化在玄宗朝**：

- **开元二年（714）**——史源所见唐廷首次正式以"市舶使"为名遣官出外：《新唐书·卷六·玄宗本纪》记是年"**右威卫中郎将周庆立为安南市舶使**，与波斯僧广造奇器以进"——这是中国史源中**最早一次"市舶使"作为正式官名的记载**。**须辨一桩**：周庆立此次设于**安南**，**非广州**；"广州为首设地"是后世史家通说，**史源异文不能断言**；但开元中后期市舶使转设广州、由广州长官旁兼或由内廷遣中使专任——**这一段过渡是确定的**，只是具体年份史志阙记。

- **开元中后期至天宝间**——市舶使常设广州，由宦官中使充任；偶亦遣外朝武臣（如周庆立一类）出任。这一段近四十年，是中国海外贸易制度从"地方官兼管"过渡到"中央专使监管"的关键时期。

- **天宝元年（742）**——玄宗改州为郡：广州改称**南海郡**——755 年这一刻，市舶使所驻地已随之改称南海郡，史源中并未见"南海郡市舶使"作为正式衔名出现；按制度可称"**驻广州 / 南海郡一线的市舶使**"，唐人口语与诗文里则仍多称"广州市舶使"。

唐人若在 755 年驻足回望——**市舶使制度此时已立四十一年**：从开元二年（714）到天宝十四载（755）。**一座制度的少年期**——其间海舶贸易稳步扩张、蕃坊渐成规模、抽分收入入内库充羡余。**而盛唐人尚不知道**：再过三年（758），市舶使制度便要在乾元元年九月那场"大食波斯舶围广州"的事件里第一次撞上自己的反面——舶商的不满与武装冲突。

### 此刻

天宝十四载（755）的广州市舶使——制度上**已运作四十一年**，行政上属**广州都督府 + 岭南五府经略**双层节制，但实际上**直接对内廷负责**。

**市舶使在任者**——755 年这一刻的具体人选**史志阙记**。开元末天宝间数任市舶使姓名见诸断片：**开元初周庆立**（安南，714）、**开元中"中使吕太一"**（《唐会要·卷六十六》载吕太一为岭南选补使、监海舶；但具体年份与是否兼市舶使史源异文）——**755 在任者无明确记载**。这是盛唐市舶使作为"差遣使职"的史料局限：宦官中使非正式品官，吏部铨选不入正史百官志详录，往往只在事件发生时被偶然提及。

**市舶使衙署所在**——盛唐文献阙记，按制大致在广州城内、近珠江北岸、与广州都督府衙署同区。**今广州市越秀区中心一带**（盛唐广州故城范围）——具体位置已不可考。**唐时市舶使既非常设衙署，亦无独立印信**——只是一处临时寄治的偏厅；待差遣毕、中使北返长安、衙署随之撤去。这是"市舶使"与"市舶司"在物理形态上的根本差别——**唐时是"人在则署在、人去则署去"**；宋以后才是"署常在、人轮换"。

**抽分入官的规模**——盛唐文献阙具体数字，但可作一处横向参照：**唐玄宗朝开元天宝间内库岁入约百万贯**——其中市舶进奉所占份额，史源未明。中晚唐至 9 世纪：《唐会要》载贞元间岭南市舶岁入约**四十至七十万贯**——**等同当时国家两税总收入的十分之一**。盛唐市舶规模虽逊于中晚唐（因海舶贸易在中晚唐持续上升）——但已是国家财政的一份显见的份额。

**蕃坊与市舶使的关系**——蕃坊（城西南、大食波斯天竺师子国诃陵占婆诸国商人聚居区）行**蕃法自治**：蕃长由蕃人自推、唐廷加以册封；坊内日常事务（婚丧、贸易、宗教）行蕃法；刑事大案交唐官。但**蕃商的海舶进出港、货物登岸**——一概由市舶使监收。市舶使与蕃长之间——是**国家与异俗社团**之间的一道接合。**755 年此刻蕃坊规模**：史料阙记，按推论以**数千至万计**蕃商住广州（参见 [广州](../cities/广州.md) doc 详论）；其中以大食、波斯商人为最盛。

**755 年冬安禄山反**——**消息沿驿道南传**：自范阳起兵、过河北、入河南、走江汉、过五岭——抵广州大约需四十余日。岭南远在天下之极南——**755 末至 756 头几个月，广州市舶照常运作**：海舶岁岁来集、蕃坊岁岁开市、市舶使岁岁抽分入官——只是抽分入官的那一笔财货，**自此不再能稳定北运长安**：江淮漕路东移、潼关失守、玄宗西奔入蜀——盛唐财政的中央调度链在这一年彻底断裂。**广州市舶反成肃宗朝的一项重要财源**——这是后话。

### 此刻意味

对天宝十四载的广州市舶使来说——这一职差遣的份量，**在 755 这一年比开元初要重得多**。

四十一年前（开元二年/714）周庆立任安南市舶使时——市舶是一种"为内廷采办珍奇"的差遣，**地位偏轻、份量偏小**。**四十一年后**（天宝十四载/755）——海舶规模、蕃坊人数、抽分收入皆已数倍于开元初；市舶使的差遣**也由"采办珍奇"扩展为"国家财政 + 内廷羡余 + 朝贡外交"三位一体**。盛唐"开元盛世"的物质繁华——除关中粮、江淮漕、河北丝之外——**广州市舶的香药宝石**也是一项构成。**唐人不会在 755 这一刻意识到**：再过五年、十年——这条海舶线将成中晚唐财政的命脉之一；自此往后五百年（唐宋元）海上丝路的盛极，**底子是 755 这一刻打下的**。

对舶商而言——**广州市舶使是大唐最远端的一道关、亦是最远端的一道门**：海舶北上抵广州、市舶使登船验货、抽分入官；商人交完税便可入蕃坊住下、岁岁待信风南返。盛唐市舶使对蕃商的**抽分率，史源未明确**——大致按货品种类轻重不一；但**抽分之后即可入市自由贸易**——这一点是盛唐市舶制度对蕃商的吸引力所在。**北宋以后**市舶司渐生"禁榷"（独占）之制——某些贵货须先尽官买、商人不得私市——盛唐尚无此禁。

对朝廷而言——市舶使是**对岭南海岸的一种远端治理**：不派重兵（岭南只设经略不设节度，盛唐如是）、不立重镇（广州军政简略），但通过宦官中使遣使一年一巡，**把海舶贸易纳入皇权直辖**。这是大唐对"远而富之地"的一种独特处置——**轻兵镇之、重商护之、专使收之**。**755 年此刻**，这套处置仍按例运作；自此往后三百年（中晚唐至宋初），广州市舶使（北宋以后改为市舶司）一直是国家财政的一份显见的支柱——只是制度形态在不断改进。

## 子条目

### 市舶使衙署（临时寄治）

广州城内——具体位置盛唐文献阙记，按制大致在广州都督府衙署左近、近珠江北岸。**唐时市舶使是差遣使职、非常设衙署**——以"中使"身份临时出岭南，寄治于南海郡太守衙或广州都督府衙之偏厅；差遣完毕便回长安复命。**衙署的具体形貌**——史料不能断言；按常制大约是一座二三进的院落、有验货厅、印信房、库藏室。**755 年此年衙署所在**——史志阙记。

### 抽分与进奉

**抽分**——市舶使按船货种类抽取关税入官，是市舶制度的核心。**抽分率盛唐文献阙具体数字**——按《唐会要·卷六十六》所记大致按货品轻重不一；某些贵货如沉香、龙涎、瑟瑟须先尽抽分入内库。**进奉**——抽分之后剩余的珍奇之物，市舶使须**先尽进奉**（送内廷）；其余才入市。**进奉与抽分的区别**——抽分是"税"（按比例收入官），进奉是"贡"（择最珍奇者尽数送内廷）。盛唐市舶使的差遣实际上是**国家征税 + 内廷采办**双轨合一——这一身二任在中晚唐成为反复改革的对象。

### 宦官充任的传统

**唐市舶使以宦官充任**——这是盛唐市舶制度的另一个特点。**唐玄宗朝以后内廷宦官以"中使"身份出外差遣**渐多：盐铁使、漕运使、市舶使、监军使——是宦官势力渐起的标志之一。**史源所见唐市舶使列名**：开元二年周庆立（安南；外朝武臣，非宦官——是市舶使初置时的过渡）；开元中后期至天宝间多任市舶使姓名史志阙记，但**多为宦官**（《旧唐书·宦官传》及《唐会要》断片可推）。**755 年在任者无明确记载**。**这一传统延续至中晚唐**——北宋开宝四年（971）广州正式置市舶司、由文臣兼领，宦官充任之制始革；元丰三年（1080）元丰市舶改制后由转运司或路一级文臣兼领定型，宦官充任之制彻底结束。

### 蕃舶与抽分货品

盛唐广州市舶使所抽分的主要货品——**香药 + 宝石 + 象牙 + 犀角 + 玳瑁 + 珠贝 + 玻璃器**为大宗：

- **香药**——大食的乳香、沉香、苏合香、龙涎香；天竺的檀香、胡椒；占婆林邑的沉香、犀角；
- **宝石**——波斯的瑟瑟（青金石）、玛瑙、玻璃；师子国的红宝石、珊瑚；
- **动物制品**——象牙（天竺、占婆、诃陵）、犀角（占婆、林邑）、玳瑁（南海诸岛）、珠贝（合浦珠池亦在此线）；
- **其他**——胡椒、苏合香、丁香（南海诸国）、奴隶（诃陵——史源载诃陵贡奴）。

**回返货**——大唐输出**丝、绢、锦、瓷、茶、铜钱**为大宗；铜钱外流是中晚唐至宋元长期的一桩贸易逆差，**盛唐已开此端**。

### 蕃坊与市舶使

**蕃坊**（城西南珠江北岸）——大食、波斯、天竺、师子国、诃陵、占婆诸国商人聚居区；蕃长由蕃人自推、唐廷册封；坊内行蕃法。**蕃坊与市舶使的接合**：蕃商的海舶进出港、货物登岸——一概由市舶使监收；但**蕃商在岸上的日常生活（婚丧、贸易、宗教）行蕃法**——蕃长自治。这是大唐对外人的一种处置：**让你按你的规矩活在我的城里，但出了港门，就是我的天下**。详见 [广州](../cities/广州.md) 蕃坊条。

### 海上丝路东端起点

**广州市舶使**——海上丝路东端的财政 anchor。**自巴士拉（今伊拉克）出波斯湾**、经阿曼、印度西海岸、师子国、马来半岛、占婆——**抵广州大约 70-90 日航程**；**五月乘东南信风入港、九、十月乘东北信风离港**——这是盛唐海舶的一岁一巡。**广州是这条线的东端**——海舶北上更远（如至扬州、登州、明州）虽偶有记载，但**主流海舶贸易在广州登岸**——市舶使在此抽分入官。**755 年此刻**——这条线已稳定运行近一世纪（自唐高祖、太宗朝起），并将在中晚唐至宋元持续繁荣五百年。

## 历史长河

**中晚唐**（755-907）

天宝十四载冬安禄山反——岭南未即卷入，**广州市舶照常运作**。**乾元元年（758）九月**——大食、波斯舶**反**，焚劫广州城：《旧唐书·肃宗本纪》乾元元年九月癸巳条："广州奏：大食国、波斯国兵围州城，刺史韦利见弃城而遁，劫库藏，焚庐舍，浮海而去"——**这是中外史源中最早一次记载南海诸国与唐廷的武装冲突**。原因史源未明，可能与肃宗朝乾元元年改革税法、抽分率提高有关；**亦可能与同年广州刺史韦利见处置失当有关**。**这是市舶使制度立四十四年来第一次撞上自己的反面**——舶商的不满与武装冲突。**距 755 年仅三年**。

中晚唐市舶使制度持续兴盛。**德宗贞元（785-805）至宪宗元和（806-820）间**——岭南市舶岁入达**四十至七十万贯**，是国家两税总收入的十分之一；广州刺史多由名臣出任：**杜佑**（《通典》撰者，贞元中任岭南节度使兼市舶使）、**孔戣**等。**贞元间海舶贸易的繁荣**——是中晚唐财政在安史之乱后能维持的物质基础之一。

**中和元年（881）**黄巢南下入广州——史载"杀回回、犹太、基督教徒、波斯教徒共十二万"（《新唐书》或二十万说）——**广州蕃坊在此年遭到致命打击**；市舶使制度因之受重创。**这是市舶使制度自开元二年（714）立以来第一次的重大断裂**——但唐廷在中和后期至昭宗朝仍设市舶使。**距 755 年一百二十六年**。

**五代**（907-960）

唐末刘氏起于岭南——**南汉**（917-971）立国五十五年定都广州（改称**兴王府**），**市舶制度由南汉自设**——其抽分入南汉国库，不再北运中原。这是市舶使制度在岭南的一次区域化。

**北宋**（960-1127）

宋太祖开宝四年（971）灭南汉、广州归宋——**同年广州正式置市舶司**，是市舶作为常设衙署、独立机构化的开端；**唐至五代是"市舶使"（差遣官名）、北宋开宝四年以后是"市舶司"（常设衙署）**——这是制度史上的一次根本变化。**距 755 年二百一十六年**。**元丰三年（1080）**——宋神宗厘定《元丰市舶条例》（"元丰市舶改制"）：明定抽分则例、由转运司或路一级文臣兼领、定额属吏、专职印信——**市舶司制度自此完全定型**。**距 755 年三百二十五年**。

北宋元丰至徽宗宣和间——**广州市舶司岁入达百万贯以上**，是中国海外贸易的高峰；**泉州市舶司**（北宋元祐二年/1087 置）兴起，与广州、明州、密州（板桥镇）并列四大市舶司。**自此广州不再独占海舶贸易**——南方多港分立。

**南宋至元**（1127-1368）

南宋广州市舶司继续运作——但**因泉州兴盛、海舶北上多由泉州登岸**——广州市舶份额下降。元代广州市舶司继置，但**元廷主要海贸口岸已转至泉州**——广州为辅。

**明清**（1368-1911）

明洪武初年置"市舶提举司"——**明初海禁严**，市舶贸易萎缩。**清乾隆二十二年（1757）**——清廷下"一口通商"令——**广州成为大清唯一对欧贸易港**，十三行（粤海关）取代了原市舶司的功能；**自此"市舶司"作为衙署名号正式废止**——海关制度兴起。这是**市舶使 / 市舶司**作为一项制度，**自开元二年（714）立、至乾隆二十二年（1757）废**——前后**一千零四十三年**。**距 755 年一千零零二年**。

**民国至今**（1911—）

清末民初海关制度西化——市舶司彻底退入历史。**1949 年以后**广州海关沿设；**1978 年改革开放**以后广州重新成为中国对外贸易的重要门户。**今广州海关大厦在沿江西路一带**——与盛唐市舶使衙署所在地的位置大致同区，**一座制度的物理 anchor 千年未离**。

至于盛唐那座没有自家衙署的"市舶使"——**其衙署旧址早已无寻**，唐时寄治的偏厅压在今越秀区繁华街区之下。**唐时市舶使本人**——以宦官充任、岁岁北返长安复命——**这一形态的市舶使制度已彻底消失**。今人读《唐会要·卷六十六》"市舶使"条时——只当作笔记里的奇事；那时其实**是这座城里某一日、某一处、某一艘海舶系缆、某一个中使下衙登船**——盛唐广州市舶的一岁一巡，凝在这十六个字里。

## 引文

> 凡海舶至广州者，市舶使先验之，进奉者上闻；其有所市，给值。

——《唐会要·卷六十六》"市舶使"条（散文概括，非完全直引；条文具体见原书）。**这是市舶使制度的核心条文**——验货、进奉、收市三事，盛唐至中晚唐市舶制度皆以此为基。

> 右威卫中郎将周庆立为安南市舶使，与波斯僧广造奇器以进。

——《新唐书·卷六·玄宗本纪》开元二年（714）条。**这是中国史源中最早一次"市舶使"作为正式官名的记载**——周庆立此次设于安南、非广州；"广州为首设地"是后世通说，史源异文。**距 755 年四十一年**。

> 有大食国、波斯国兵围州城，刺史韦利见弃城而遁，劫库藏，焚庐舍，浮海而去。

——《旧唐书·肃宗本纪》乾元元年（758）九月癸巳条。**这是中外史源中最早一次记载南海诸国与唐廷的武装冲突**——亦是市舶使制度立四十四年来第一次撞上自己的反面。**距 755 年三年**。
