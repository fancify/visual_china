# 蒲津渡

## 一句话

蒲州城西门外黄河上铁索浮桥——**战国秦晋故渡、北周斛律羌定木舟形制**；**开元十二年/724 唐玄宗诏张说监铸 8 铁牛 + 4 铁人 + 2 铁山 + 铁柱 + 铁索维之**——是大唐最大规模铁器铸造工程、亦是关中 ↔ 河东 物理生命线；唐 755 时为**关中入河东第一渡**（蒲津渡向北 / 东北 入河东方向；长安东出洛阳方向另走潼关）。

## 形貌

蒲津渡在蒲州城西门外——**黄河之上**、距州治河东县约一二唐里、出蒲州西门即抵桥头。

蒲州坐黄河大转弯东岸——黄河自龙门峡谷出、南下经河津、龙门、再东折至潼关——蒲州正在黄河"几"字形末段东转之前的最后一段。**此段黄河宽约六百步**（约今三百米），水势平、河床稳、自古便是秦晋之间最易立渡之处。**蒲津渡跨黄河、东岸蒲州、西岸同州**（治冯翊；同州属关内道、与蒲州隔河相望）——一桥连两州、一渡通两道、是关中入河东第一咽喉。

**桥之形制**——以铁索维之、木舟为面、铁牛锚之。**铁索四条**（《蒲津桥赞》载）自东岸铁牛贯过黄河、系于西岸铁牛；**铁索之上铺木板木舟**、连缀成长桥；**舟随水浮、索维其位**——这是中古"舟桥"（亦称"浮桥"）的成熟形态。桥长**约六百步**（一说更长、史源略异）——东西两端各置桥头堡、两岸守桥兵卒、过桥行人车马络绎不绝。

**铁牛为锚**——这是开元十二年（724）的核心工程。**东西两岸各立 4 铁牛、合计 8 头**——每头铁牛重约**数万斤**（《蒲津桥赞》"牛逾岸、铁百万"——一说每牛重 5 万至 7 万斤、即 30 至 45 吨左右；具体重量近世以 1989 出土实物称量定）；**牛身嵌入岸边铁柱深桩、与浮桥铁索系连**——铁牛蹲伏于岸、铁索自牛颈出、跨河系西岸牛——是浮桥最关键的锚定结构。**铁牛之外、配 4 铁人 + 2 铁山 + 7 铁柱**（一说更多；以 1989 东岸出土数为准）——铁人立铁牛旁、形如执缰之人；铁山立桥头、状如假山、亦作锚配重；铁柱则深入河岸、维铁索之根。

**蒲津渡上下景观**——桥东即蒲州城西门、出门即抵桥头；桥下即黄河、舟楫北自龙门下、南至潼关上、过桥者皆停舟桥头候过桥车马。**桥之北侧不远**即**鹳雀楼**——蒲州城西南角的三层瞭望楼、自楼上俯视即蒲津渡舟桥；楼上观渡、渡下观楼、是蒲州城西最完整的一处地理景观。**桥头西望**——黄河西岸群山方向、远眺可至潼关、长安在西南千里之外。**桥头东望**——蒲州城墙、中条山横列于东南。

**注**：**"津"在古汉语中即渡口之义**——蒲津渡即"蒲州津渡"、不可解为"金"或他字。**蒲津渡 ≠ 风陵渡**——风陵渡在蒲州南约百里、黄河大转折处（今山西芮城风陵渡镇）、是**黄河、渭河、洛河** 汇流区（晋陕豫三省交界处、黄河急转东向）之另一渡；蒲津渡则在蒲州西门外、黄河直流段。**蒲津渡 ≠ 大阳津、茅津**——大阳津在陕州（今山西平陆）、茅津在陕州治南；蒲津、风陵、大阳、茅津为黄河下游四大唐津渡、本 doc 严格只论蒲津。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

蒲津渡的来路可追到**战国秦晋时代**——比开元铸铁牛早九百余年。

**春秋秦穆公时（BC 7 世纪）**——蒲津渡已是秦晋两国主要渡口。**秦穆公十五年（BC 645）韩原之战**——秦伐晋、晋惠公夷吾兵败被俘、即自蒲津渡过黄河押解入秦——是史载蒲津渡最早的具体事件。**战国魏国时**——蒲坂（即后蒲州治河东县）属魏、蒲津渡是魏国西出渡河入秦的主道；魏惠王 + 秦孝公间秦魏屡战、蒲津渡几度易主。**秦统一后**——蒲津渡为关中东出河东第一渡、咸阳 ↔ 河东 (汾水谷地) 主路所经。

**汉代**——**汉武帝元封间（BC 110 前后）**重修蒲津桥、用木船 + 缆绳连缀；**王莽天凤间（AD 14-19）**再修；**东汉、三国、两晋**间渡桥屡毁屡修、形制基本沿汉武故制——皆为**木舟 + 缆绳**的临时性浮桥、风雨季节常断、非常设。

**南北朝**——黄河中段是诸国疆界、蒲津渡战略地位空前。**北魏太武帝太延五年（439）灭北凉、统一北方**——蒲津渡入北魏；**北魏分裂（534）后东魏 / 西魏隔黄河对峙**——蒲州在西魏境、蒲津渡是西魏东出渡河攻东魏的前线渡口。**北周武帝保定元年（561）期间（实由宇文护执政）命大将斛律羌建蒲津浮桥**——**自此蒲津渡首次有常设形制**：**铁索 + 木舟 + 两岸石牛锚之**（注：**北周时期是石牛、非铁牛**——铁牛是开元十二年/724 才铸；本 doc 严格区分）。北周斛律羌的浮桥**用了约 30 年**——至隋代渐废。**隋开皇 + 大业间（581-618）多次修浮桥**——但形制仍是木舟 + 缆绳、未铸大型金属锚件。**隋大业七年（611）炀帝东巡涉蒲津**——史载其时桥犹是临时形制。

**唐初武德 + 贞观**——蒲津渡仍用木舟浮桥；**唐高宗 + 武后间**渐感不便——黄河水势在此段虽稳、然每岁汛期木舟仍易毁、缆绳易朽、岁岁修桥、岁岁劳民。

**开元十二年（724）—— 这是蒲津渡历史上的一个 epoch**。**唐玄宗诏兵部尚书张说监督蒲津桥重修工程**——**铸 8 铁牛 + 4 铁人 + 2 铁山 + 铁柱 + 铁索**、以铁锚维浮桥。**张说（667-731）**是开元前期玄宗朝重臣——历任中书令、兵部尚书、燕国公、典谟册命多出其手；时年五十七、监督蒲津工程是其晚年的一桩大事。**铸造规模**——《蒲津桥赞》载"用铁 五十万斤、用工 不可胜数"（具体数字近世以 1989 出土铁件称量考实——东岸 4 铁牛合计约 200 吨、配套铁人铁山铁柱合计约 100 吨、合计 300 吨；加西岸尚未发掘部分、全工程铁料应在 600 吨上下）——这是中国古代单一工程规模最大的铁器铸造之一、其规模在开元盛世前后无出其右者。

**张说《蒲津桥赞》（开元十二年/724 作）**——张说亲自为铁牛工程作赞、是工程的官方记录文书。**赞文今存《全唐文》卷二二三**——四言铭体、记工程缘起、铸造规模、监督官员、桥之形制——是蒲津渡铁牛工程最直接的一手史源。**赞文开篇**："维大唐开元十二年、岁在甲子、青阳司辰、月旅大簇——粤十有一日、皇帝赫然诏曰：'河中之间、广济之要、是为津梁、宜以铁牛维浮桥'"——是工程的官方诏旨记述。

**开元定名"大庆桥"**——开元工程竣工后、玄宗诏定蒲津桥别名"大庆桥"——以志开元盛世之大庆；但民间仍多称"蒲津桥"或"蒲津渡"。

**开元十二年至天宝十四载（724-755）— 共 31 年**——铁牛维桥、桥不再岁岁修、舟楫日过千百、商旅络绎、远比汉武 + 北周 + 隋间任何时期都繁盛。**这是蒲津渡作为关中 ↔ 河东 物理生命线的全盛 31 年**。

### 此刻状态

天宝十四载（755）这一年的蒲津渡——**已铸成 31 年、是大唐黄河第一渡**。

**8 铁牛仍蹲东西两岸**——锈色斑驳、牛身嵌入铁柱深桩、与铁索维浮桥；牛旁 4 铁人立、状如执缰；牛后 2 铁山、铁柱 7 根（东岸）扎根河岸——31 年来未经大灾、铁件完整。**铁索四条**横贯黄河、铺木板木舟为桥面——舟随水浮、索维其位、桥长约六百步。**桥头守兵**——蒲州州兵 + 河东节度使派的兵卒、东西两岸各置堡——查过往车马、记往来人数（《唐六典》卷三·户部记"凡天下津梁、皆有税"——蒲津作为河中第一津、税收应不轻；但蒲津 properly speaking 是 渡 / 桥 而非 关、不直接征关税）。

**蒲津渡日行**——**自长安北 / 东北出、北渡渭水、经同州（治冯翊）、即抵蒲津渡西岸**；过桥即入蒲州、东行可至河东（治河东县、即蒲州治）、再北上抵太原（北都）方向、或东折经平定关入河北道。**长安东至洛阳另走潼关线**——蒲津线 ≠ 潼关线、二者出长安后分道：潼关线东出经华州、过潼关、南折入河南道至洛阳；蒲津线北 / 东北出、不经潼关、不经华州、径达同州蒲津渡。**蒲津渡 vs 风陵渡**——蒲津是长安 ↔ 河东 主道、风陵是长安 ↔ 河南陕州 道；二渡分担黄河下游两个走向、蒲津更重。**蒲津渡 vs 潼关黄河渡**——潼关本身是关 / 兼渡、过潼关后又南折、不直接通蒲州；**蒲津渡是关中入蒲州、入太原方向的最直径**。

**桥下黄河**——这一段水流不急、河床稳、舟楫南北通行皆经过桥头。**桥畔商市**——蒲津渡两岸自然形成市集、过路商旅停脚、铸铁工匠 + 木舟修匠 + 旅店酒肆皆备。**桥北鹳雀楼**——三层木构、自楼上俯视即蒲津渡舟桥铁牛；**王之涣《登鹳雀楼》在 755 时已传 20 余年**——盛唐文人凡至蒲州必登楼诵诗、楼下即蒲津渡铁牛黄河——是盛唐文化与工程两大 anchor 的物理叠合。

**755 此刻的政治位置**——**安禄山在范阳（治幽州、今北京）**——河北、河东三镇节度使、是 755 时大唐军力最重的边镇之一。**长安 ↔ 范阳道**——蒲津线作 长安 → 同州 → 蒲津渡 → 蒲州 → 太原 → 范阳（蒲津渡正在此道上）；另有潼关线作 长安 → 潼关 → 洛阳 → 范阳（不经蒲津渡）——二线分担关中 ↔ 河北交通。**安禄山的奏疏、贡物、使节** 数十年来皆经蒲津渡入长安；玄宗 + 杨贵妃所赐的礼物 + 诏书亦经蒲津渡北还范阳——**蒲津渡的铁牛、是安禄山与长安朝廷物理联结的中枢**。

### 此刻意味

对天宝十四载的蒲州人来说，蒲津渡是**城西门外那座有铁牛的浮桥**——日常过桥赴同州、走亲访友、贩米贩盐、皆经此渡；岁岁汛期渡上虽水势稍急、但铁牛锚之、桥不毁；**铁牛是日常的、是城西门外那两排黑黝黝的铸铁形象**——孩童在牛身上爬、过路客在牛旁歇脚、铁牛已成蒲州景观的一部分。

对盛唐文人来说——蒲津渡是**关中东出第一桥**、过此桥才入河东之地。**鹳雀楼上诵王之涣"白日依山尽、黄河入海流"**——楼下即此渡之桥、铁牛在黄河之上 31 年——**诗境与工程合于一处**。**李白、王维、杜甫等盛唐诗人多次经蒲津渡**（杜甫尤其、其乾元年间避乱涉河经蒲津之诗多——但 755 此刻杜甫尚在长安未南奔，本 doc 不预支）。

对朝廷来说——蒲津渡是**关中 ↔ 河东 物理生命线**、是长安 ↔ 范阳道关键一段、亦是大唐最大规模铁器工程的样板。**开元十二年（724）的张说监督铸牛、是开元盛世国力的物理证据**——31 年来铁牛未毁、铁索未断、桥不再岁岁修——这是工程力的胜利。**对蒲津渡铁牛、玄宗朝廷一直有自豪感**——开元定名"大庆桥"、张说赞文入《全唐文》、铁牛工程为后世史志反复引述。

唐人尚不知道——再过半年余（至德元载六月 / 公历 756 年 7 月，《资治通鉴》卷二一八）安史叛军南下、潼关失守、玄宗西奔；蒲州为河东方面唐军黄河防线重镇、**蒲津渡作为军事咽喉一时险象环生**——但叛军未直接渡蒲津、桥与铁牛未毁。**至德元载（756）十二月肃宗下诏改蒲州为河中府**——蒲津渡此后正式名号转入"河中府蒲津渡"、但物理仍立。再过千余年（明清间）黄河多次改道、铁牛随河岸塌陷渐入河床淤泥；再过 1234 年（1989-1991）—— **山西省永济市蒲津渡遗址考古发掘出土 4 铁牛 + 4 铁人 + 2 铁山 + 7 铁柱**（东岸部分；西岸尚未发掘）——这是开元十二年铸造物 1265 年后最完整的物理存证、亦是中国现存最古老的桥梁工程实物。**铁牛仍在**——这是工程的另一种 anchor——比浮桥更耐久的 anchor：木舟铁索皆朽、铁牛仍蹲；千载之后任黄河改道、任铁牛入泥、铁牛终能出土、铁牛终能立于今日之蒲津渡遗址博物馆——告知千年后来者：开元十二年、张说曾在此监督铸了 8 头铁牛、维一座桥、连两州。

## 子条目

### 开元十二年（724）铁牛工程

**铸造规模**——8 铁牛 + 4 铁人 + 2 铁山 + 铁柱 + 铁索；东西两岸各立 4 铁牛、铁人铁山铁柱分配两岸；用铁约 50 万斤（张说《蒲津桥赞》原文；近世以 1989 东岸出土实物 200 吨倒推全工程约 300-600 吨）。

**铸造工艺**——铁牛 单体重 数万斤（约 30-45 吨）、当为**分范铸造 / 就地浇铸**（蒲州周边产铁、运河运铁皆便）；**连接方式待考**（"焊接"系现代术语、唐时无此工艺；分段件之间的连接当用榫接 / 铆合 / 铁箍等古法、具体形制以 1989 出土实物为准、需金相 + 结构进一步研究）。**铸造监督**——兵部尚书张说（667-731）；监督官员中应有铁工监 + 河东道铁官 + 蒲州地方官；具体名单史源未详。**铸造时间**——开元十二年（724）一年内完成主体工程（《蒲津桥赞》开篇即云"维大唐开元十二年"——是当年事）；铁索 + 木舟桥面 续后续工。

**铸造目的**——**取代北周 / 隋间木舟 + 缆绳的临时浮桥**——岁岁汛毁、岁岁修桥、劳民耗财；铁牛锚定后浮桥可常设、不再岁修。**这是工程经济学的胜利**——一次性大投入（50 万斤铁）、换取数百年免岁修——开元盛世国库充实、敢于做此长期投资。

**与同期工程对比**——开元间另有**洛阳天津桥重修**（开元十六年/728）+ **长安灞桥** + 其他大型桥渡工程；但蒲津铁牛规模独大、亦最有名。**张说《蒲津桥赞》** 是中国工程史一份珍贵的官方铭文。

### 张说（667-731）

**蒲津渡铁牛工程的监督者**——开元十二年（724）兵部尚书任上督此工。

张说字道济、河南洛阳人——历仕武则天、唐中宗、睿宗、玄宗四朝；**开元元年（713）拜中书令**、为玄宗朝早期宰相之一；**开元六年（718）封燕国公**；**开元十一年（723）改兵部尚书 + 同中书门下三品**——次年（724）即监督蒲津工程。**开元十八年（730）卒、年六十四**——蒲津铁牛已铸成 6 年、张说亲见工程成果。

**张说的文学地位**——盛唐前期"燕许大手笔"之"燕"（与许国公苏颋并称）——朝廷典谟册命多出张说之手；**蒲津桥赞** 即其晚年代表作之一、四言铭体、用典严密、是唐代工程铭文的范例。**《全唐文》卷二二三**收张说所作铭赞十余篇、《蒲津桥赞》居其中。

**注**：**张说 ≠ 张九龄（678-740）**——张九龄是开元二十一年（733）继张说之后的宰相、亦是盛唐重臣；但蒲津工程是 724 年事、张九龄当时尚未拜相、与此工程无直接关系。**张说 读音注**——张说 之"说" 读 yuè（同"悦"）、不读 shuō；《说文》"说"通"悦"、是其名；史源多注音。

### 蒲州 / 河东郡

**蒲津渡所在州**——755 时正官名为**蒲州 / 河东郡**（天宝元年/742 玄宗诏改州为郡、蒲州称河东郡；**至德元载/756 蒲州升河中府**——一说乾元元年/758 升府，但通说作 756）。**755 时是蒲州 / 河东郡、不是河中府**——这是本 doc 严格守的 755 时态。

蒲州治**河东县**（今山西永济市西约 10 公里黄河东岸）——县城距黄河约一二唐里、蒲津渡即在西门外。**蒲州为河东道（治太原）南部首州**、属京畿外围最近的州之一、过蒲津渡即抵关中。**州户**——天宝间蒲州 / 河东郡 户约 7 万、口约 40 万（《通典》卷一七九·州郡九 数）——是河东道大州之一。

### 鹳雀楼（楼下之渡）

**蒲津渡北侧不远即鹳雀楼**——三层木构、立蒲州城西南角的城墙之上、自楼上俯视即蒲津渡舟桥铁牛。**北周宇文护建说**——一说鹳雀楼始建于北周（557-581）、初为黄河前哨瞭望楼、与蒲津渡军事功能相辅。**王之涣（688-742）《登鹳雀楼》**——开元年间作、至 755 已传 20 余年——"白日依山尽、黄河入海流、欲穷千里目、更上一层楼"——楼下即蒲津渡铁牛黄河、是盛唐文化与工程两大 anchor 的物理叠合。**鹳雀楼是蒲津渡的瞭望塔、亦是其文学 anchor**——楼上观渡、渡下观楼、二者构成蒲州城西南最完整的一处地理景观。

### 蒲津渡 vs 风陵渡 vs 大阳津 vs 茅津

**蒲津渡 ≠ 风陵渡**——风陵渡在蒲州南约百里（今山西芮城风陵渡镇）、黄河大转折处、是**黄河、渭河、洛河** 汇流区（晋陕豫三省交界处、黄河急转东向）之另一渡。**蒲津渡 ≠ 大阳津**——大阳津在陕州（今山西平陆）、黄河上、是陕州东出河东主渡。**蒲津渡 ≠ 茅津**——茅津在陕州治南、亦黄河上、是陕州 ↔ 河南府 道之一渡。**唐代黄河下游四大津**——蒲津、风陵、大阳、茅津——蒲津最北、最重、唯一有铁牛工程；其余三津皆木舟 + 缆绳临时形制。

### 1989-1991 出土铁牛（千载后存证）

**山西省永济市蒲津渡遗址考古发掘**——1989 年起、至 1991 年大致完成东岸部分发掘。**出土实物**：**4 铁牛**（每头长约 3 米、高约 2 米、单重约 50-70 吨）+ **4 铁人**（每人高约 1.9 米、立铁牛旁）+ **2 铁山**（铁铸假山形、桥头配重）+ **7 铁柱**（深入河岸、锚铁索之根）。**出土位置**——东岸即蒲州城西门外、原黄河东岸河床中（千年来黄河改道、铁牛入泥）。**西岸部分尚未发掘**——按张说赞文载全工程 8 铁牛 + 4 铁人、东岸 1989 出土 4 铁牛 + 4 铁人 + 2 铁山、则西岸应另有 4 铁牛 + 配套铁柱（铁人总数已与张说载吻合于东岸、不外推西岸亦有铁人）；但因今河道改变、西岸具体位置较难定。

**铁牛今藏**——出土后就地展陈、**永济市蒲津渡遗址博物馆**专为此建（1999 年开馆）——铁牛、铁人、铁山、铁柱皆陈列原位、是中国现存最古老的桥梁工程实物。**铁牛物理状态**——经千年河水浸泡、铁件表面锈蚀严重但形态完整；近年做了防锈处理。

**注**：**蒲津渡铁牛 ≠ 沧州铁狮子**——沧州铁狮子是后周广顺三年（953）铸、独立铁兽、与桥渡无关。**蒲津渡铁牛 ≠ 黄河铁牛（泛称）**——黄河沿线多处有"铁牛"传说（如河南 + 山东诸地）、但实物存证仅蒲津渡一处、是中国黄河铁牛的孤本。

## 历史长河（755 → 今）

**中晚唐**　**至德元载（756）十二月**——肃宗下诏改蒲州为**河中府**——蒲津渡此后正式名号为"河中府蒲津渡"、但物理仍立。**安史乱中（755-763）蒲津渡 为唐军黄河防线重镇**——叛军未直接渡蒲津、铁牛与浮桥未毁。**乾元元年（758）杜甫经蒲津渡**——其《蒲津桥》（一作《蒲州渡》、史源略异）记当时桥况：铁牛仍蹲、铁索仍维、舟楫如常——是杜甫亲见的开元工程最直接记录之一。**中唐至晚唐**——蒲津渡是河中府主要财政与军事交通节点；**德宗 + 宪宗 + 文宗朝**间多次修缮浮桥木舟、但铁牛工程基本无需大修。**晚唐**——黄巢之乱（880s）经河中、蒲津渡或一时为叛军所控、但桥未毁；**唐末藩镇间河中府数易其主**——蒲津渡铁牛仍蹲于岸。

**五代两宋**　**五代（907-960）**蒲津渡所在的河中府数易其主、铁牛偶有损但未毁；**后晋开运三年（946）契丹南侵、河中府陷**——蒲津渡一时被契丹兵占领、但浮桥未毁。**北宋（960-1127）**蒲津渡仍是河中府主桥、铁牛仍蹲；**北宋间黄河水势渐变**——河床抬升、铁牛或部分被淤泥掩埋、但浮桥仍通行。**北宋元丰间（1078-1085）司马光奉敕修《资治通鉴》**——其卷二一二记开元十二年张说铸铁牛事、是宋人对此工程的官方记录。**靖康（1126-1127）后金兵南下、河中府入金境**——蒲津浮桥木舟在战乱中毁、铁牛仍立。**金代（1115-1234）**蒲津渡为金境内黄河重渡、金人重修浮桥但形制大不如唐；**金末蒙金战争**间蒲津渡一带数次易主。

**元明清**　**元代（1271-1368）**——河中府改"中兴府"（一说仍称河中府、史源略异）；蒲津渡浮桥渐废、铁牛部分入河床淤泥；**元代地方志记蒲津铁牛"间或可见"**——已非常设浮桥的锚件、而是河岸遗物。**明代（1368-1644）**——蒲州属山西布政司、蒲津渡有简易木船渡而无浮桥；**明嘉靖三十四年（1556）关中大地震**（震中在华县、史称"嘉靖大地震"）——蒲州亦受影响、铁牛或在此次地震后入河更深。**明末清初黄河多次改道**——蒲津渡原桥位渐入河床、铁牛全部为淤泥所掩、地表已不可见。**清代（1644-1911）**——蒲津渡仅作"古迹"记于《蒲州府志》《永济县志》——蒲津渡铁牛已隐于河床数百年、地表无任何工程遗存；过往渡黄河者改用临时木船。**清末** 蒲津渡作为渡口的功能彻底终结——铁路（陇海线 1932 修至潼关、过黄河铁桥）替代了黄河浮桥时代。

**民国至今**　**民国间**蒲津渡铁牛位置据老乡口传 + 地方志、仍知大致位置、但无人发掘。**1949 年后蒲州 / 永济县归山西省**——蒲津渡作为"开元铁牛"传说继续在地方流传。**1989 年**——**山西省考古所联合永济市启动蒲津渡遗址正式发掘**——**1989-1991 年陆续出土 4 铁牛 + 4 铁人 + 2 铁山 + 7 铁柱**（东岸部分）——是开元十二年（724）铸造物 1265 年后最完整的物理存证。**1999 年蒲津渡遗址博物馆开馆**——铁牛就地陈列、是中国现存最古老的桥梁工程实物。**2000 年代以后**——蒲津渡遗址博物馆为山西重点文物保护单位、亦是永济市与盛唐工程文化的核心 anchor 之一。

**铁牛今物理状态**——出土后做了防锈与防腐处理、陈列原位、但黄河改道后的新河道距铁牛遗址已较远；铁牛不再泡水。**西岸 4 铁牛尚未发掘**——因今黄河西岸（陕西大荔县境）河床变化与勘察条件限制、近年未启动系统发掘；学界期待将来出土、与东岸合璧。

**鹳雀楼复建**——1997-2002 年永济市新建鹳雀楼（与唐楼非物理连续；详 [鹳雀楼](./鹳雀楼.md) doc）——新楼与蒲津渡遗址博物馆同在永济市西郊、相距约一二唐里、与古蒲州城西南角的形胜大致对应；游人来此可同游鹳雀楼新楼 + 蒲津渡铁牛遗址、是盛唐文化与工程两大 anchor 的现代物理叠合。
