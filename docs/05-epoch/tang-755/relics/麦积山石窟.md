# 麦积山石窟

## 一句话

**陇右秦州第一佛山、西陲泥塑石窟之冠**——天水郡上邽县东南六十里麦积山中、后秦弘始间（399-416）始凿、北魏 + 西魏 + 北周三朝高峰、隋唐续凿；**Tang 755 时已 350 年、是秦州境内最大佛教圣地**；**再过七年（762）吐蕃陷秦州、麦积山随陇右失唐控制约百年**——盛唐西陲泥塑高峰的物理终点。

## 形貌

麦积山在唐陇右道**秦州天水郡境**——具体在**上邽县东南六十里**（今甘肃省天水市麦积区麦积山乡）；距秦州治**上邽县**约六十唐里。**地势**：**麦积山立陇右黄土塬南缘 + 西秦岭北麓的过渡带**——东接陇山西麓、南连西秦岭北坡、北望渭水谷地、西接秦州盆地；**山体孤峰突起**——**山形似农家积麦之垛**、故名"麦积"。

**麦积山之独**：**山高约一百四十二米**（今实测）——**山体呈孤立的圆锥状砂砾岩柱**；**山西南面 + 东南面是悬崖峭壁**——**悬崖高约八十米**、几乎垂直；**石窟即开凿在这两面悬崖之上**——**自地面至山顶分层而开**、自下而上层层叠叠；**唐人称之"麦积崖"**——**"崖"字扣住麦积山悬崖石窟的物理本质**。

**麦积山岩石为砂砾岩**——**这是麦积山石窟形制的关键 anchor**：**砂砾岩疏松、不宜雕刻**——故麦积山石窟**主体不是石雕、而是泥塑**；**工匠在崖壁开凿洞窟 + 内置木骨 + 外敷草泥 + 再施彩绘**——**麦积山泥塑的工艺源此**。**这一点区别于云冈（砂岩石雕）+ 龙门（石灰岩石雕）+ 敦煌（砾岩 + 草泥塑像）**——**麦积山以泥塑见长**、**故后世称"东方雕塑馆"**。

**石窟分布**：**石窟开凿在山的西南面 + 东南面两面悬崖**——**自地面至山顶分层而开**、**层层叠叠如蜂房**；**洞窟之间以栈道相连**——**栈道凿崖而建**、**自下而上"S"形盘旋**；**唐人 + 北朝人攀登石窟**——**必经栈道**。**栈道高悬数十米**——**唐人有"麦积凌空"之喻**（《麦积山石窟内容总录》引唐宋诗记）。

**周边地理**：**麦积山西北约六十里即秦州治上邽县**——**秦州 ↔ 麦积山 一日可达**；**麦积山北约二十里即渭水**——**渭水自西向东流过秦州盆地**；**麦积山东即陇山西麓**——**陇山东接关中、西接陇右**；**麦积山南即西秦岭北坡**——**南通祁山道 + 入蜀**。**麦积山在物理上**——**是陇右南境 + 关中西出 + 入蜀北口三大方向的交汇 anchor**。

物候：麦积山一带——温带半湿润 + 季风过渡气候；**冬寒夏凉、夏秋多雨**；**山中常有云雾**——**唐人称"麦积烟雨"**（北宋《方舆胜览》引唐宋诗记）；**雨季崖壁湿润**——**这是麦积山泥塑保护的最大隐忧**。**山中有泉水 + 山溪**——**是石窟僧侣与香客的天然水源**。**春秋两季是麦积山香火最盛的季节**——**秦州 + 陇右诸州的信众多于此时上山礼佛**。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

麦积山在唐人的回望里——是**陇右佛教圣地 + 后秦至北周连续开凿 350 年的西陲泥塑高峰**。

**麦积山的开凿**：**麦积山石窟始凿于十六国后秦时期（384-417）**——**一般认为始凿于姚兴弘始年间（399-416）**；**具体始凿年代不可考**——**后秦姚兴崇佛、迎鸠摩罗什入长安译经**（弘始三年/401）；**麦积山开凿与姚秦佛教兴盛同期**。**注：后秦 = 姚秦（姚萇 + 姚兴 + 姚泓、384-417）**——**不是苻坚的前秦（351-394）**；**唐人对后秦麦积山的认知**——**多承北朝石窟史的记载**。

**北朝开凿高峰**：**北魏宣武帝（500-515）+ 孝明帝（515-528）+ 西魏文帝（535-551）+ 北周武帝（560-578）是麦积山开凿的高峰期**——**约 5-6 世纪、一百余年**；**北魏宣武帝景明 + 正始年间（500-507）**——**麦积山大规模开凿**；**北魏孝明帝 + 北魏末**——**继续开凿**；**西魏文帝时麦积山为西魏皇室造像之地**——**西魏文皇后乙弗氏大统六年/540 被废 + 出家 + 同年三月赐死 + 同年葬麦积山**（《北史·后妃传上·文皇后乙弗氏》）；**北周武帝灭佛（574-578）期间麦积山一度停凿**——**但北周宣帝 + 静帝（578-581）复兴佛法**；**隋文帝 + 隋炀帝间麦积山续凿**——**规模渐降**。

**唐前期的麦积山**：**唐高祖 + 太宗 + 高宗 + 武则天 + 玄宗五朝**——**麦积山续凿、但规模远不如北朝高峰**；**唐人在麦积山主要是续凿小龛 + 重修旧窟 + 补绘壁画**；**唐窟在麦积山现存窟龛中所占比例较小**——**约占总数的一两成**。**755 时麦积山**——**主体仍是北朝遗存**；**唐人所见的麦积山**——**是后秦至北周 350 年累积的西陲泥塑高峰**。

**麦积山的开凿断代关键 anchor**：**后秦弘始间（399-416）始凿**——**麦积山现存最早窟龛**；**北魏宣武帝景明 + 正始（500-507）+ 孝明帝正光（520-525）**——**麦积山北魏大规模开凿期**；**西魏文帝大统（535-551）**——**乙弗皇后葬麦积山 + 西魏皇室造像**；**北周武帝（560-578）+ 北周宣帝 + 静帝（578-581）**——**北周开凿 + 灭佛波及 + 复兴**；**隋开皇 + 仁寿（581-604）**——**隋文帝崇佛、麦积山续凿**；**唐高宗 + 武周间**——**唐前期麦积山续凿**；**唐玄宗开元 + 天宝间**——**麦积山续凿 + 重修旧像**；**755 时盛唐麦积山**——**主体仍是 5-6 世纪北朝遗存**。

**麦积山与秦州佛教**：**秦州自北朝以来即陇右佛教重镇**——**麦积山 + 秦州城内诸寺 + 周边山寺**构成秦州佛教的物理 anchor；**麦积山为秦州第一佛山**——**州人 + 陇右诸州信众多上麦积礼佛**。**755 时秦州刺史**——**史无明载者** + 待考；**麦积山住持僧 + 寺院主**——**亦史载零散**。

**麦积山在丝绸之路上的位置**：**丝绸之路东段自长安西行**——**经凤翔 + 陇州 + 大震关 + 秦州 + 兰州 + 凉州 + 甘州 + 肃州 + 沙州 + 西州**；**麦积山在秦州境**——**是丝绸之路东段第一处大型石窟**；**自麦积山继续西行**——**炳灵寺石窟（河州、北魏始凿）+ 敦煌莫高窟（沙州、前秦始凿）+ 克孜尔石窟（龟兹、3-9 世纪）**——**沿丝路一线分布**。**麦积山在 755 时**——**不仅是秦州境内的佛山**、**更是中原 + 西域佛教交流的物理 anchor**——**僧侣 + 商旅 + 朝贡使**多在麦积山留题记 + 造像 + 抄经；**麦积山的北朝 + 隋唐遗存**——**呈现中原汉式 + 西域犍陀罗 + 中亚粟特三系佛教艺术的融合**——**这是丝绸之路文化交流的物理证据**。

### 此刻

天宝十四载这一年的麦积山——**石窟严整、僧侣有度、栈道盘旋、香火不绝**。

**石窟状态**：盛唐 755 时麦积山主体仍立——**北朝 + 隋唐窟龛分布西南面 + 东南面悬崖**；**栈道凿崖盘旋、自下而上层层相连**；**山中寺院 + 僧舍 + 经楼 + 钟楼一应俱全**；**具体规模史载零散**——**本 doc 不预设具体数字**。**山下 + 山中常驻僧侣**——**应有数十至百余**；**春秋香期则信众数千上山**。

**755 时麦积山上的佛教氛围**：**麦积山是秦州 + 陇右诸州的佛教圣地**——**香火不绝**；**唐廷于秦州置都督府 + 麦积山在都督府境**；**唐前期麦积山曾有皇室或贵戚造像 + 寺院题记**——**但 755 这一刻具体施主、史载零散**。**麦积山的法会 + 节庆 + 浴佛 + 盂兰盆**——**皆是 755 这一年的日常**。

**755 时麦积山的物理细节**：**栈道盘旋 + 崖壁洞窟层层相连**——**香客攀登 + 僧侣往返**；**山中泉水 + 山溪供应僧舍**；**山下村落 + 寺田 + 香客驿站**；**麦积山与秦州城（上邽县）日常往还**——**僧人下山化缘、香客上山礼佛、商旅过麦积山进入西秦岭山道**。

**755 时麦积山周边的佛教生态**：**秦州城内 + 城外有数十所寺院**——**麦积山为其中规模最大、最古老者**；**秦州东南六十里 + 上邽县境**——**麦积山附近另有仙人崖 + 石门山 + 净土寺等佛教遗迹**（皆隋唐已有 + 后世续修）；**这些山寺与麦积山共同构成秦州东南山区的佛教 anchor**。**陇右诸州 + 河西诸州的僧侣 + 信众**——**多有专程至麦积山礼佛 + 抄经 + 造像题记者**；**麦积山在 755 时**——**是丝绸之路东段（长安 ↔ 凉州 ↔ 敦煌）上的一处重要佛教节点**。

**755 末 + 756 的麦积山命运**：**安禄山十一月起兵于范阳**——**主力南下沿河北 + 河南方向**；**麦积山未涉直接战事**。**756 七月玄宗西奔入蜀**——**玄宗一行经岐州 + 散关入蜀、未走秦州一线**；**麦积山在 756 这一年——并非主战场**；**但 757 之后陇右兵东调救京、秦州兵力空虚**——**为吐蕃东扩埋下伏笔**。

### 此刻意味

对天宝十四载的秦州人 / 上邽人来说——麦积山是**自家境内的佛山**：春秋上山礼佛、求子求寿求平安；**山中僧侣 + 北朝旧像 + 唐窟新龛**——**是秦州人日常宗教生活的物理 anchor**。对长安那一代士人来说——麦积山是**陇右佛教圣地**：**与敦煌 + 凉州 + 西州诸窟同为西陲佛教的物理 anchor**；**盛唐诗中麦积山虽不如龙门 + 敦煌密集**、**但作为陇右秦州第一佛山**——**是盛唐西陲佛教地理的核心一节**。

唐人尚不知道——再过四年（759）杜甫将弃华州司功参军、西赴秦州流寓三月——**在秦州杂诗中多次涉麦积一带山寺**（《山寺》等）；亦尚不知道——再过七年（762/763）吐蕃东扩、陷陇右诸州、**麦积山将随秦州失去唐控制**——**自此约百年**间麦积山在吐蕃手中（约 762-849）；**中晚唐麦积山一度荒废**——**盛唐那座香火不绝的麦积山**、在中晚唐基本残破。**注：陇右收复与河西收复是两脉络、不可合并**——**大中三年/849 唐廷收复秦、原、安乐等州（陇右收复一脉络、《资治通鉴》卷二四八·大中三年）**；**张议潮起于沙州、收复瓜沙及河西诸州（河西收复一脉络、《新唐书·卷二一六·吐蕃传下》《新唐书·张承奉传》）**；**麦积山是否随秦州恢复、另作待考**。**盛唐那条秦州 ↔ 麦积山 ↔ 长安的物理通道已断**——**麦积山在中晚唐基本失去开凿的施主与香客密度**。

## 子条目

### 后秦 ≠ 前秦（麦积山始凿姚秦弘始间）

**后秦 = 姚秦（姚萇 + 姚兴 + 姚泓、384-417）**——**羌族姚氏所建**、**都长安**；**不是苻坚的前秦（氐族苻氏、351-394、亦都长安）**——**两秦不可混**。**麦积山始凿于姚秦弘始年间（399-416）**——**姚兴在位时**；**姚兴崇佛**——**弘始三年（401）迎鸠摩罗什入长安、置逍遥园译经**；**麦积山开凿与姚秦佛教兴盛同期**——**虽具体施主、年代史载零散**、但**学界一般定姚秦为麦积山始凿时代**。**唐人 755 时的认知**——**多承北朝石窟史的记载**；**麦积山在 755 时已 350 年**——**是秦州境内最古老的佛教遗存之一**。

### 麦积山泥塑（不是石雕）— 砂砾岩的物理约束

**麦积山岩石为砂砾岩**——**疏松、不宜雕刻**；**这是麦积山石窟形制最大的物理 anchor**：**主体不是石雕、而是泥塑**。**工匠在崖壁开凿洞窟 + 内置木骨 + 外敷草泥 + 再施彩绘**——**麦积山泥塑的工艺源此**。**这一点区别于云冈（砂岩石雕）+ 龙门（石灰岩石雕）**——**两窟皆以石雕见长**；**麦积山以泥塑见长**——**故后世称"东方雕塑馆"**。**敦煌莫高窟亦以泥塑 + 壁画见长**（砾岩亦不宜雕刻）——**麦积山 + 敦煌在岩石 + 工艺上同一类**；**唐人 755 时所见的麦积山泥塑**——**多是北魏 + 西魏 + 北周遗存**、**唐塑相对较少**。

### 中国四大石窟（敦煌 / 云冈 / 龙门 / 麦积山）

**中国四大石窟**——**敦煌莫高窟（沙州）+ 云冈石窟（云州、北朝、北魏前期）+ 龙门石窟（河南府、北朝至唐）+ 麦积山石窟（秦州、后秦至唐）**；**四者皆是北朝至唐的佛教石窟高峰**——**地理上分守河西 + 山西北 + 河南 + 陇右四大方向**。**755 时四窟皆立**——**敦煌 + 龙门 + 麦积山仍开凿、云冈已基本停凿**；**长安 ↔ 四窟的距离**——**龙门最近（约 750 唐里）+ 麦积山次（约 1000 唐里）+ 云冈次（约 1500 唐里）+ 敦煌最远（约 3200 唐里）**；**麦积山在四窟中位居西陲泥塑高峰**——**地理上扼陇右佛教 anchor**。

### 杜甫《秦州杂诗》与麦积山（759 流寓秦州）

**乾元二年（759）杜甫弃华州司功参军、西赴秦州流寓约三月**——**作秦州杂诗二十首**；**其中《山寺》等涉麦积一带山寺**——**注：杜诗具体所指未必皆麦积山**，**但秦州境内之山寺多与麦积山相关**。**杜甫流寓秦州时麦积山仍在唐境**（吐蕃 762/763 才陷秦州）——**杜甫或亲访麦积山**、**史载零散**。**杜甫《秦州杂诗》是盛唐 → 中唐过渡的陇右诗踪**——**麦积山在杜诗的回声中、是盛唐西陲的最后一段佛教山色**。**注：杜甫秦州杂诗作于 759、不是 755 现场**——**本 doc 历史长河段方可引、不宜入 755 此刻**。

### 西魏文皇后乙弗氏葬麦积山（大统六年/540）

**西魏文帝皇后乙弗氏**——**文帝元宝炬之原配**；**大统六年（540）乙弗氏被废 + 出家为尼**——**同年三月赐死**——**同年葬麦积山**（《北史·卷一三·后妃传上·文皇后乙弗氏》）；**麦积山遂有"皇后崖"之号**。**三事同年发生、当拆开时序读**：**先废后 + 出家**——**继赐死**——**再葬麦积山**——**而非"出家即葬"**。**这是麦积山在北朝皇室记忆中的最显赫一刻**——**麦积山遂成西魏皇室造像之地**；**北周武帝灭佛（574-578）期间麦积山一度受波及**——**但乙弗皇后崖窟一直保留**。**唐人 755 时所见的麦积山**——**仍有"皇后崖"的旧痕**；**这是 350 年前西陲皇室遗骸与麦积山泥塑同处一山的物理 anchor**。

### 麦积山栈道与"凌空"——盛唐人攀登石窟的方式

**麦积山石窟自西南面 + 东南面悬崖自下而上分层而开**——**洞窟之间唯一通道是栈道**；**栈道凿崖而建**——**自下而上"S"形盘旋**；**栈道高悬数十米**——**唐人 + 北朝人攀登石窟必经栈道**。**唐人有"麦积凌空"之喻**——**指栈道盘旋 + 洞窟高悬的物理感**；**今所见栈道为现代重建**——**唐栈道原物早已不存**；**但"麦积凌空"的物理体验**——**自北朝至唐至今、千余年未变**。**755 时麦积山的栈道**——**应是木栈 + 部分凿石阶**；**香客 + 僧侣 + 工匠**——**皆需攀栈道上下**。

### 麦积山现存 194 窟（北朝主体 + 隋唐续凿）

**今麦积山石窟艺术研究所档案**——**现存洞窟 194 个 + 雕塑 7000 余尊 + 壁画 1300 余平方米**；**洞窟分布**——**西南面 + 东南面悬崖**；**自下而上分层而开**；**主体为北魏 + 西魏 + 北周 + 隋唐遗存**。**北朝窟约占总数六七成**——**是麦积山泥塑高峰的物理主体**；**唐窟约占一两成**——**主要在中下层 + 较新崖面**；**宋金以下窟数渐少**。**755 时唐人所见的麦积山**——**已有北朝旧窟约一百余 + 隋唐新窟若干**；**具体数字史载零散**——**本 doc 不预设盛唐 755 时的窟数**；**但麦积山在 755 时已是"千龛万像"的规模**——**唐人在山中可见崖壁层层、佛龛累累**。

### 麦积山 ↔ 敦煌：两窟同属砾岩泥塑

**麦积山 + 敦煌莫高窟**——**两窟在岩石性质 + 工艺上属同一类**：**砾岩 + 砂砾岩皆不宜雕刻**——**主体为泥塑 + 壁画**；**两窟皆开凿于悬崖**——**麦积山西南 + 东南面悬崖、敦煌鸣沙山东麓崖壁**；**两窟皆始凿于十六国时期**——**麦积山后秦弘始间（399-416）+ 敦煌前秦建元二年（366）**；**敦煌略早麦积山约 30 年**。**两窟同属丝绸之路北线 + 河西走廊的物理 anchor**——**敦煌在沙州（河西走廊西端）+ 麦积山在秦州（陇右南境）**；**两窟距长安**——**敦煌约 3200 唐里 + 麦积山约 1000 唐里**；**两窟皆于 2014 列联合国教科文组织世界文化遗产《丝绸之路：长安-天山廊道路网》**。

### 762/763 吐蕃陷秦州与麦积山失唐控制

**广德元年（763）吐蕃东扩大举 + 陇右诸州陆续陷蕃**——**秦州具体陷年待考**（一说宝应元年/762、一说广德元年/763）；**自此约百年**间麦积山失唐控制（陷蕃—约 849）。**中晚唐麦积山一度荒废**——**香火渐衰、施主断绝、唐窟续凿基本停止**；**但麦积山泥塑主体未毁**——**北朝旧像 + 唐窟新龛在吐蕃统治期得以保留**。**注：陇右收复与河西收复是两脉络、不可合并**——**大中三年/849 唐廷收复秦、原、安乐等州（陇右收复一脉络、《资治通鉴》卷二四八·大中三年）**；**张议潮起于沙州、收复瓜沙及河西诸州（河西收复一脉络、《新唐书·卷二一六·吐蕃传下》《新唐书·张承奉传》）**；**麦积山是否随秦州恢复、另作待考**。**盛唐那条秦州 ↔ 麦积山 ↔ 长安的物理通道已断**——**麦积山在中晚唐基本失去开凿的施主与香客密度**。**这是麦积山在 755 之后最沉重的转折**——**盛唐西陲泥塑高峰的物理终点是吐蕃东扩**。

## 历史长河（755 → 今）

**中晚唐**　**至德二载（757）唐军收复长安**——麦积山仍在唐境；**乾元二年（759）杜甫流寓秦州约三月**——**作秦州杂诗二十首、涉麦积一带山寺**；**这是盛唐 → 中唐过渡的一段陇右诗踪 + 麦积山的最后一段盛唐回声**。**广德元年（763）吐蕃东扩大举 + 陇右诸州陆续陷蕃**——**秦州具体陷年待考**（一说宝应元年/762、一说广德元年/763）；**自此约百年**间麦积山失唐控制。**中晚唐麦积山一度荒废**——**香火渐衰、唐窟续凿基本停止**；**注：陇右收复与河西收复是两脉络、不可合并**——**大中三年/849 唐廷收复秦、原、安乐等州（陇右收复一脉络、《资治通鉴》卷二四八·大中三年）**；**张议潮起于沙州、收复瓜沙及河西诸州（河西收复一脉络、《新唐书·卷二一六·吐蕃传下》《新唐书·张承奉传》）**；**麦积山是否随秦州恢复、另作待考**。**盛唐那条秦州 ↔ 麦积山 ↔ 长安的物理通道已断**。

**五代**　**五代间陇右一带战乱频仍**——**麦积山多次易主**；**后梁 + 后唐 + 后晋 + 后汉 + 后周**——**陇右名义上属中原王朝、但麦积山实际归吐蕃 + 党项诸部交错控制**；**麦积山五代续凿规模渐降**——**主要是补绘 + 小龛**。

**两宋**　**北宋初麦积山归秦凤路秦州**——**北宋开凿规模有限**；**北宋仁宗 + 神宗间麦积山为宋与西夏对峙的边境山**；**北宋末 + 南宋初麦积山归金**（北宋亡 1127 后）；**金代麦积山续凿 + 重修**——**金代规模较北宋略大**；**南宋亡后麦积山归元**。

**元明清**　**元代麦积山续作佛教圣地、但规模渐降**；**明代麦积山属陕西布政司巩昌府秦州**——**明代续凿规模有限、主要是重修**；**明代麦积山曾遭地震破坏**——**部分崖壁 + 栈道坍塌**；**清代麦积山重修栈道 + 部分窟龛**——**今所见栈道为现代重建（基于清代基础）+ 部分窟龛为清代补塑**；**清中后期麦积山渐失开凿活动**——**主要是寺院与香火**。

**民国至今**　**1949 年后麦积山石窟列国家重点文物保护单位（1961 第一批）**；**1953 年成立麦积山石窟艺术研究所**——**开始系统编号 + 测绘 + 保护**；**现存 194 个洞窟 + 7000 余尊雕塑 + 1300 余平方米壁画**——**主体为北魏 + 西魏 + 北周 + 隋唐遗存**；**1986-1990 年代多次抢救性加固崖壁 + 栈道**；**2014 年麦积山石窟列联合国教科文组织世界文化遗产《丝绸之路：长安-天山廊道路网》**（33 处遗产点之一）。**唐人在 755 所见的那座麦积山**——**千二百多年后**主体仍立**；**盛唐那条秦州 ↔ 麦积山 ↔ 长安的物理通道**——**今已成宝兰客专 + 国道 G310 的现代驿道**。**唐人在 755 所见的那座麦积山 + 后秦至北周 350 年累积的西陲泥塑高峰**——**盛唐西陲佛教的物理终点是吐蕃东扩与百年陷蕃**；**而麦积山泥塑本身**——**穿越后秦 + 北朝 + 隋唐 + 吐蕃 + 五代 + 宋金 + 元明清 + 民国 + 当代 1600 余年**——**仍是中国四大石窟中以泥塑见长者的物理 anchor**。
