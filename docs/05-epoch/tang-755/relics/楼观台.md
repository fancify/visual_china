# 楼观台 / 宗圣宫

## 一句话

京兆府盩厔县终南山北麓的道教祖庭——传说老子在此说《道德经》传关令尹喜；**李唐皇室追祖老子、武德间立"宗圣观"奉为道祖庙**——是唐皇家道教中心，玄宗朝多次亲祀。

## 形貌

楼观台在京兆府盩厔县东南三十里——**终南山北麓**，一片山间台塬。

台址在**就峪口**外的台原——南依终南山主脊、北望渭水关中平原、西侧界涝峪（与渼陂同一谷系）、东侧界田峪。**整个区域分两部分**：山下平地的**宗圣宫**（朝廷敕建、皇家祭祀场所）+ 山中崖壁上的**说经台**（道教传说老子说经处）。两处直线相距约二里、由石阶相连。

**宗圣宫在台原南端**——朝廷敕建的道宫，正门朝南、面对终南；唐时殿宇、斋堂、道士院凡数十座（具体殿名盛唐史源未详，今所见殿宇多为明清后重建）。**说经台在山中一处崖壁上**——传说老子讲《道德经》、关令尹喜执经而听的台。台前有古柏数株、传为"老子手植"——盛唐人来观时尚立。

楼观一带的物候——背靠秦岭主脊，但已在塬上、不入深山。夏日山间云气常起、冬日终南雪线可见——是终南北麓众多道观中**唯一进入皇家祭祀编制**的一处。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

楼观的来路两层——**先秦传说 + 唐皇室追祖**。

**先秦传说**：《史记·老子列传》记老子"见周之衰，乃遂去，至关、关令尹喜曰：子将隐矣，强为我著书"——《史记》仅言"至关"，并未明指地点为楼观台。**南朝梁、北周间的《楼观本起内传》**始把老子说经地点定于盩厔楼观——这是道教传说渐固化成具体地点的过程。**自南北朝起楼观便是关中道教祖庭之一**——北朝周武帝时楼观道士尹喜后人（号尹通）颇得朝廷供奉。

**唐皇室追祖**：李唐皇室号称"陇西成纪人"、实际祖出武川（北魏六镇之一）——但李唐为塑造"天命所在"的正统叙事，**自高祖即位起便正式追老子为先祖**、立道教为国教之一。**武德三年（620）四月**——高祖李渊亲幸楼观、祠老子；**武德间**朝廷敕修楼观，赐名"宗圣观"（一作"宗圣宫"）——自此楼观便是**李唐皇室的道教祖庭**。老子的"皇帝"尊号自高宗乾封元年（666）起正式系于册命（见下"老子官号链"段）。

**太宗、高宗、武则天、玄宗**——四朝皆有楼观致祭记录。**老子的官号链**（《旧唐书·高宗本纪》《唐会要·尊崇道教》）：
- **乾封元年（666）**——高宗追尊老子为**"太上玄元皇帝"**（这是道教史上第一次帝王正式封老子为皇帝、非玄宗朝）
- **天宝二载（743）**——玄宗加号**"大圣祖玄元皇帝"**
- **天宝八载（749）**——玄宗加号**"大圣祖大道玄元皇帝"**
- **天宝十三载（754）**——玄宗加号**"大圣高上大道金阙玄元皇帝"**（一作"大圣高上大道金阙玄元天皇大帝"——史源有异文）——**这是 755 这一刻、楼观所祀老子的全官号**。

玄宗自身崇道极深——亲注《道德经》、命天下立玄元皇帝庙；楼观作为唐皇室道教祖庭、自然是首要祭祀场所。

### 此刻状态

天宝十四载这一年的楼观台 / 宗圣宫——**是李唐皇室道教祭祀的巅峰一刻**。

**宗圣宫**在 755 时是关中最大的道观——殿宇数十、道士数百。**主殿祀老子（"大圣高上大道金阙玄元皇帝"——一作"金阙玄元天皇大帝"，异文）**——是天宝十三载（754）新加官号定后的祀奉；**主殿具名"老君殿"是后世通称；盛唐殿名史源未详**，本 doc 不预设。**说经台**则是宗圣宫之外山中的核心朝礼地——盛唐凡入楼观者必登说经台、观古柏、礼老子。

**盛唐文人来终南访道者甚众**——但**专写楼观台的盛唐前诗作不密集、亦无确切作年的代表作**；楼观作为道教祖庭的诗咏多在中唐及以后。李白崇道深、曾在终南山中多次访道；王维隐辋川、与终南各道观相去不远——但具体某诗系于楼观、史源不稳。

天宝十四载的楼观——**朝廷玄元皇帝大祀仍按例举行**；安史前夕（即 755 这一刻）——玄宗已 71 岁、仍在长安兴庆宫；老子在楼观仍是"大圣高上大道金阙玄元天皇大帝"。**这是唐皇室道教最辉煌的一年**。再过半年——安禄山起兵于范阳；玄宗西奔入蜀；朝廷道教大祀中断。

### 此刻意味

对天宝十四载的盩厔人来说，楼观台是**县东南山中那座皇家敕修的道宫**——日常香火、岁岁朝礼。对长安那一代士人来说，楼观是**老子说经的传说圣地**——离长安百余里、可日还、是访道短游的首选。对朝廷来说——楼观是**李唐皇室祖庙级别的道观**——其分量等于汉昆明池之于汉武、九成宫之于太宗。

唐人尚不知道——再过一百四十余年，唐末战乱中宗圣宫殿宇坼毁过半；再过四百余年（北宋至金），楼观地位渐降、退为地方道观；再过千年，**说经台前的古柏仍立**——明清以来历代修缮、今为陕西省周至县楼观台森林公园核心。**老子手植柏的传说一千二百多年不易**——这是楼观的物理 anchor。

## 子条目

### 老君殿 / 宗圣宫主殿

**宗圣宫的核心祀殿——立老子像之处**。武德间始建、玄宗朝多次扩修。**755 时殿内祀奉的是"大圣高上大道金阙玄元皇帝"**（一作"大圣高上大道金阙玄元天皇大帝"——史源异文）——这是天宝十三载（754）新加官号定后的祀奉。**像之具体规模、衣冠形制盛唐史源未详**——今所见楼观宗圣宫遗址内的老君殿与像为明清重修，盛唐原貌已不可考。

### 说经台

**山中崖壁上的传说老子讲经地**——距宗圣宫约二里、由石阶相连。台上有**老子说经像**（不同时代造像不一）、台前有**古柏数株**——传为老子手植。盛唐人凡入楼观者必登台、观柏、礼老子。**今说经台仍在**——为楼观台森林公园核心、台前古柏经现代植物年龄测定有数株确为千余年（部分可能上溯至隋唐间）。

### 古柏（"老子手植"）

说经台前数株古柏——传为老子手植、亦或为关令尹喜手植，道教传说不一。**现代植物学测定**显示其中数株树龄约 1500-2500 年（视具体株位）——若上限属实，可上溯至秦汉甚至更早，但**老子手植说仍是传说**。盛唐人在 755 所见的古柏，与今游人所见的，**至少有数株是同一株树**——这是楼观最重的一个物理 anchor。

### 周代尹喜结草为楼观（地名来源）

**楼观之名**——传说周代关令尹喜在此结草为楼、观星望气、得遇老子。"楼观" 二字即"楼上观星"之意。**这是地名的传说起源**——但先秦至两汉无明确史源、最早记录楼观与老子说经直接挂钩的是南朝《楼观本起内传》。**盛唐人接受"周代尹喜立楼观"说**——但本 doc 视之为**道教传说**而非 verified history。

## 历史长河（755 → 今）

**中晚唐**　安史乱中楼观未受直接兵燹——盩厔县在玄宗西奔路线之外。**自德宗朝起朝廷道教祀渐减**——楼观岁修日少；但唐宪宗、武宗朝因崇道有新致祭记录。**武宗朝（840-846）**为崇道而灭佛——这一时期楼观地位再升；但武宗崩、宣宗复佛后楼观地位渐降。**唐末战乱（黄巢之乱 880s）**——宗圣宫殿宇被毁过半、道士散尽。

**五代北宋**　唐亡（907）后楼观失去"李唐皇室祖庙"的政治地位——五代间宗圣宫荒。**北宋初道教仍兴**（宋真宗朝崇道）——大中祥符二年（1009）真宗诏修楼观、敕赐"顺天兴国之观"匾——这是北宋楼观地位的一次短暂提升。**但宋代道教中心已渐转至武当、龙虎山等地**——楼观从盛唐"全国道教中心"退为关中地方道观。

**金元**　**金大定七年（1167）王重阳（1112-1170）在终南山中创全真道**——全真道兴起在终南山附近、与楼观地理相近、但**全真不以楼观为祖庭**（全真奉钟离权、吕洞宾等八仙为祖）。金元间全真兴、楼观渐衰；**元代楼观尚存数殿，但已不复盛唐**。

**明清**　明洪武初敕修楼观；明万历间多次重修；清康熙、雍正、乾隆三朝俱有修缮记录。**清代楼观为陕西道教重要场所**——但全国地位远不及武当、龙虎。

**民国至今**　1949 年后楼观台先列陕西省重点文物保护单位、后列全国重点。**1980 年代起**周至县始建**楼观台国家森林公园**——以说经台为核心、古柏群为亮点。**2008 年陕西省考古研究院发掘宗圣宫遗址**——确认唐代道宫规模、出土武德至天宝间器物。**今游人来楼观台，可登说经台、观老子手植柏（数株至少千年）**——盛唐人在 755 所登的台、所观的柏，与今人所见在物理层面有真实连续。这是楼观区别于汉昆明池 / 九成宫 / 玉华宫的关键——**它的物理 anchor 未断**。
