# 唐乾陵

## 一句话

唐高宗李治 + 武则天合葬陵——**双帝合葬**：高宗葬 684、武则天葬 706；**唐六陵中唯一未被五代温韬盗发的一座**（《新五代史》"唯乾陵风雨不可发"）；**755 时已立 71 年、陵阙石刻俱在**——梁山主峰为穴、神道有无字碑 + 述圣纪碑。

## 形貌

乾陵在京兆府乾县城北约 6 公里——**梁山主峰**（今陕西省咸阳市乾县城北梁山）。**梁山**：海拔约 1047 米——是渭北高原的一座大山，**"双乳峰"**（北峰主穴、南峰为门）形似女体——后世附会武则天为女主葬此的天意。**乾陵"因山为陵"**——以梁山主峰为穴、地宫凿入山体；陵园依山势南北铺开。

**陵区规模**——周回约 40 公里；陵园四面有夯土围墙；南北中轴自南向北 **朱雀门→神道→献殿→陵主穴**——是唐"因山为陵"陵制的成熟样本。

**神道石刻**：自南向北依次有**石柱 1 对 → 翼马 1 对 → 鸵鸟 1 对 → 仗马 5 对 → 翁仲（石人）10 对 → 述圣纪碑 + 无字碑（东西分立） → 蕃酋像 61 座 → 石狮 1 对 → 朱雀门**——是唐陵神道石刻规制的标杆。**翁仲、蕃酋像、石狮**等多为盛唐 7-8 世纪原物。

**著名碑石**：
- **无字碑**：神道西侧——**武则天自立、无字**；高 7.53 米、宽 2.1 米、重约 100 吨——是中国陵墓石刻史最神秘的一块碑。
- **述圣纪碑**：神道东侧——武则天**为高宗述功**所立、由武则天撰文、中宗书丹、共约 8000 字（今多湮）；高约 7 米——是中国陵墓"妇人为夫立碑"的少见之例。

**陪葬墓**：**17 座**（《长安志》记；今乾陵陪葬墓主要 17 座）——含**章怀太子李贤墓**（武则天次子）、**懿德太子李重润墓**（中宗长子、武则天孙、701 卒）、**永泰公主李仙蕙墓**（懿德妹、701 卒）、**薛元超墓**（宰相）、**泽王李上金、许王李素节、邠王李守礼、王及善、刘仁轨、李谨行、高侃**等。**陪葬墓多在陵东、东南**——成大型陪葬区。**注**：狄仁杰葬地非乾陵陪葬——《唐会要·陪陵名位》乾陵陪葬名氏中无狄仁杰，本 doc 不入。

物候：梁山一带为关中渭北、暖温带季风气候——夏热冬寒、雨集中夏季；陵区今为台塬农田、果园（多种苹果、葡萄）。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

乾陵在唐人的回望里——是**唐"双帝合葬"成熟样本** + **武周时代的物理纪念**。

**唐高宗李治（628-683）**：太宗第九子、贞观十七年（643）立为太子（原立李承乾废、再立李泰废、终立李治）；贞观二十三年（649）六月即位。**永徽（650-655）至显庆（656-661）**间在长孙无忌、褚遂良、李勣等辅政下治；**显庆五年（660）后渐失健康**——风疾发作、视物不明；**自此武皇后（武则天）渐参政**。**弘道元年（683）十二月丁巳**——高宗崩于贞观殿、年五十六；遗诏太子李显（中宗）即位；**文明元年（684）八月**——葬高宗于乾陵。**高宗在位 34 年、世寿五十六**。

**武则天（624-705）**：唐高宗皇后 → 武周皇帝；**显庆五年（660）后参政**；**麟德元年（664）后已"二圣临朝"**；**上元元年（674）改称"天后"**；**永徽六年（655）废王皇后立武后**——是武则天政治生涯的核心节点。**高宗崩后武后临朝称制**（684-690）；**天授元年（690）武则天即帝位、改国号"周"**——立"武周"15 年（690-705）。**神龙元年（705）正月神龙革命**——张柬之等迫武则天禅位中宗；**同年十一月武则天崩**——遗诏"去帝号、称则天大圣皇后"。**神龙二年（706）五月**——武则天与高宗合葬于乾陵——**唐陵中唯一"真正二帝合葬"的一座**（武则天曾自立为帝、即位为武周帝 690-705；昭陵太宗 + 长孙皇后是"帝后合葬"、皇后非帝；乾陵的特殊性在此）。

**乾陵的营建**——**文明元年（684）八月葬高宗时已立陵园主体**；**神龙二年（706）五月武则天葬入时开穴合葬**——具体地宫是否分穴并立或合穴未详。**乾陵作为武周时代终结的物理 anchor**——是中国陵墓史最特殊的一座。

### 此刻

天宝十四载这一年的乾陵——**已立 71 年**。

**陵阙、献殿、神道、石刻**：盛唐 755 时主体仍立；**乾陵未被任何盗扰**——是唐六陵中唯一一座 755 时完整无损者。**梁山地宫**：高宗 + 武则天合葬、深凿入山体；具体内部形制盛唐时人即不知，史载未详。

**无字碑 + 述圣纪碑**：神道东西两碑仍立；**无字碑的"无字"**——盛唐时已是大谜：武则天为何立此碑而不刻一字、唐人有多种推测（功不可纪 / 自评待后人 / 中宗朝政治避讳 等），但盛唐文献中具体说法零散。**述圣纪碑**：武则天撰文 + 中宗书丹，**字面已部分湮**（盛唐 755 时是何情况、史载未详；估计大部分仍可读）。

**蕃酋像 61 座**：神道北端——立**61 个西域诸国 + 各蕃使头领像**，是唐高宗 + 武则天朝接受万邦朝贺的物理纪念。**今 61 像中部分已残（无头）**——但盛唐 755 时应主体完整。

**陪葬墓 17 座**：含章怀太子、懿德太子、永泰公主等武则天宗族 + 高宗朝大臣。**著名的"血色合葬"**：**永泰公主李仙蕙（684-701）+ 懿德太子李重润（682-701）**——皆武则天孙、701 年被武则天处死（永泰公主夫君武延基与懿德太子等密议武则天朝事被武则天发觉）；**神龙元年（705）武则天禅位后中宗为追悼**——706 与乾陵同期开穴陪葬。**盛唐人凡至乾陵者必知此"血色合葬"**——是武则天朝最有名的家族悲剧。

**国家祭祀**：唐玄宗朝乾陵的祭祀按例行——每岁春秋遣陵令行祭；**玄宗对武则天的态度**——表面尊"则天大圣皇后"号、保留乾陵；但实际推动"开元中兴"以正"贞观盛世"为镜——开元朝武则天的政治遗产是被有选择保留的。

**安史前夕**（即 755 这一刻）——再过半年安禄山起兵；再过一年潼关失、玄宗西奔；**关中诸唐陵在安史中均有兵燹**（具体乾陵损毁程度史载未详）；**至德 / 乾元间唐军反攻关中**——乾陵地表石刻应有局部损毁；但**地宫主体未被盗**——是乾陵的运气。

### 此刻意味

对天宝十四载的乾县人来说——乾陵是**县北梁山的双帝陵**：高宗与武则天的合葬、岁岁有官祭、17 座陪葬墓中含武则天孙辈三人（章怀、懿德、永泰）。对长安那一代士人来说——乾陵是**武周时代的物理纪念**：李唐皇室如何处理"国号一度被改为周"+ "祖母为帝"这一历史段的物理样本。**乾陵的无字碑** 是 755 时盛唐人凭吊武则天的具象点。

唐人尚不知道——再过 130 余年（885 黄巢之乱）唐六陵将被陆续盗掘；**五代温韬大规模盗发唐陵**（《新五代史·温韬传》）——**乾陵独全**：《新五代史·温韬传》明文 **"唐诸陵，惟乾陵风雨不可发"**——传说温韬数次率众攻乾陵、皆遇大风雨而退；**乾陵自此 1300 年未被盗**——是唐 18 帝陵（含让皇帝惠陵）中**唯一未被盗的一座**。再过千二百年（1958-1972），**章怀太子墓、懿德太子墓、永泰公主墓陆续发掘**——出土壁画、墓志、器物震撼世界；**但梁山地宫主体至今未发掘**——是中国考古学界保留态度的代表。**唐人在 755 所见的那座双帝陵**——一千二百多年后陵冢、神道石刻、陪葬墓主体犹存；**地宫的高宗与武则天遗体、千年未扰**——这是乾陵在唐六陵中独一无二的物理 anchor。

## 子条目

### 梁山地宫（高宗 + 武则天合葬，未发掘）

**梁山主峰**——海拔约 1047 米；地宫凿入山体南坡。**文明元年（684）八月高宗先葬**；**神龙二年（706）五月武则天合葬入**——唐陵中**唯一"真正二帝合葬"**（武则天曾即位武周帝 690-705；昭陵太宗 + 长孙皇后是"帝后合葬"、皇后非帝）。**地宫具体内部形制**——千二百多年未发掘、未被盗扰；**1958-1960 现代考古勘察确定地宫位置**——位于梁山主峰南坡、深约 50 米；**未深探**。**中国考古学界 1970s 以来保持"乾陵地宫不主动发掘" 的政策**——是对千年未扰之地宫的最大尊重。

### 无字碑（武则天自立、无字）

神道西侧——**武则天自立、无字**。**碑材**：单块整石；**碑身**：高 7.53 米、宽 2.1 米、厚 1.49 米、重约 100 吨——是中国陵墓石刻史最大的一块单石碑。**"无字"的含义**——盛唐至清代有多种解释：功不可纪（武则天功太大、无法记尽）/ 自评待后人 / 中宗朝政治避讳 / 武则天遗愿等——本 doc 不预设。**今无字碑仍立乾陵神道西侧**——一千二百多年未易；碑面上后世（金元至明清）有人在碑上刻字（约 13 处后世题刻）——但**武周朝武则天本人未刻一字**。

### 述圣纪碑（武则天为高宗述功）

神道东侧——与无字碑东西分立。**武则天撰文、中宗李显书丹**——记高宗一生功绩、共约 8000 字（今多湮）。**碑高约 7 米**——是中国陵墓"妇人为夫立碑"的最大一座。**碑文今多湮**——风化、盗损、磨蚀使大部分字面已不可读；**盛唐 755 时是何情况、史载未详**——估计大部分仍可读。

### 蕃酋像 61 座

神道北端——立**61 个西域诸国 + 各蕃使头领像**。**这是高宗 + 武则天朝接受万邦朝贺的物理纪念**——展示唐 7-8 世纪国际地位的高峰。**像背原刻有该蕃头领国号 + 姓名**——今多湮（含西域诸国如吐蕃、突厥、龟兹等使臣或王子）。**今 61 像中绝大多数已无头**——历代盗损与风化所致；具体头是何时被毁，史载未详。**盛唐 755 时 61 像主体应完整**——是来乾陵的盛唐人最直观的"万邦朝贺"具象。

### 陪葬墓 17 座（章怀、懿德、永泰为最显）

乾陵陪葬墓 17 座（《长安志》记）。**著名陪葬墓**：
- **章怀太子李贤墓**（武则天次子、被武则天逼自尽、684 卒；706 改葬乾陵）——出土壁画《狩猎出行图》《马球图》等
- **懿德太子李重润墓**（中宗长子、武则天孙、701 卒）——出土壁画《阙楼仪仗图》等；**701 与永泰公主同期被武则天处死**
- **永泰公主李仙蕙墓**（懿德妹、武则天孙、701 卒）——出土壁画《宫女图》等
- **薛元超墓**（宰相，683 卒）
- **泽王李上金墓**（高宗子）、**许王李素节墓**（高宗子）
- **邠王李守礼墓**（章怀太子之子）
- **王及善、刘仁轨、李谨行、高侃**等宰相 / 大将

**陪葬墓在乾陵东、东南**——成大型陪葬区。**1960s-1970s 章怀、懿德、永泰三墓陆续发掘**——出土壁画约 600 平米——是唐墓壁画的代表；今陈列于**乾陵博物馆**。

### "唯乾陵风雨不可发"（五代温韬盗陵传说）

**五代后梁/后唐间温韬（约 880-925）盗发唐陵**——《新五代史·温韬传》记温韬**"在镇七年、唐诸陵在境内者悉发"**——但明文："**唐诸陵，惟乾陵风雨不可发**"——传说温韬数次率众攻乾陵、皆遇大风雨而退。**乾陵自此 1300 年未被盗**——是唐六陵中唯一未被温韬盗发的一座。**这一传说在唐宋间已传**——是乾陵在中国陵墓史地位的核心。

### 武则天的双重身份（皇后 + 皇帝）

**武则天（624-705）**——**唐高宗皇后**（永徽六年/655 立）→ **唐武周皇帝**（天授元年/690 即位）→ **则天大圣皇后**（神龙元年/705 禅位、遗诏改号）→ **与高宗合葬乾陵**（神龙二年/706）。**这是中国历史上女性政治地位最高的一段**。**乾陵的合葬安排**——既承认武则天的"皇后"身份（与高宗同陵），又默认其曾为"皇帝"（无字碑的存在）——是中宗朝对武则天的政治拿捏。**唐玄宗朝继续保留乾陵的合葬规制**——是李唐皇室对武则天政治遗产的接受。

## 历史长河（755 → 今）

**中晚唐**　**至德 / 乾元间唐军反攻关中**——乾陵一带反复易主、陵区岁修中断、地表石刻应有局部损毁。**贞元 / 元和间唐廷尝试恢复唐陵岁修**——乾陵列入。**中晚唐乾陵的祭祀**：高宗 + 武则天双帝陵——历代帝王致祭。

**唐末黄巢 + 五代**　**广明元年（880）黄巢入长安**——唐陵在乱中受波及；**五代温韬大规模盗发唐陵**（《新五代史·温韬传》）——**乾陵独全**："唯乾陵风雨不可发"。**自此乾陵 1300 年未被盗**——是中国陵墓史的奇迹。

**两宋**　**北宋初宋太祖诏修唐陵**——乾陵列入。**北宋元丰间宋敏求《长安志》**记乾陵规模 + 陪葬墓 17 座。**北宋金石学家欧阳修、赵明诚等多有乾陵题跋**——但乾陵实物未被破坏。

**金元明清**　金元明清间乾陵地表石刻历代有损毁（部分蕃酋像、翁仲被毁）——但梁山主峰未易、地宫未被盗。**清乾隆四十年（1775）陕西巡抚毕沅立"唐高宗乾陵"碑**。

**民国至今**　**1949 年后乾陵列陕西省重点文物保护单位、1961 年列首批全国重点**。**1958-1960 现代考古勘察确定地宫位置**；**1960s-1970s 章怀、懿德、永泰三陪葬墓陆续发掘**——出土壁画约 600 平米；**1971 年起乾陵地表勘察连续**——但**梁山地宫至今未发掘**（中国考古学界保留态度）。**1978 年乾陵博物馆开放**。**今游人来乾陵**——所见的是无字碑、述圣纪碑、蕃酋像、神道石刻、陪葬墓博物馆——**梁山主峰下高宗与武则天的合葬地宫，千年未扰、千年未见**。**唐人在 755 所见的那座双帝陵**——一千二百多年后地宫仍闭、神道仍立、无字碑仍无字。
