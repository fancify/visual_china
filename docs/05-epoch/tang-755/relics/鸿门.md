# 鸿门

## 一句话

唐京兆府新丰县东十七里（《元和郡县图志》卷一）骊山北麓一道土坂——**楚汉前 206 项羽四十万大军屯此、鸿门宴上放刘邦走的那一夜**——一个杯酒之间、中国历史走向被改写的瞬间；盛唐 755 时鸿门已远 960 年、坂上无营无垒、只有过往骊山温泉宫的车马道；唐人凭吊鸿门，凭吊的是**史记中那一夜的剑、那一杯酒、那一个未拔的玉玦**。

## 形貌

鸿门故址在唐京兆府新丰县东十七里（《元和郡县图志》卷一·关内道·京兆府·新丰县条；今西安市临潼区新丰镇东、鸿门堡村一带），骊山北麓一道东西横亘的黄土坂——南依骊山主脉、北俯渭水南岸冲积平原、西去长安外郭城约六十余唐里、东去潼关约二百唐里。坂体本身是骊山北麓向渭水冲积扇过渡的一道**黄土梁**——东西约数里、南北约一里余、坂顶平展、坂北缓坡入渭南平畴——是楚汉时大军屯营的天然形胜：**南屏骊山、北瞰冲积平原、东西可控关中通道**。

唐人到此凭吊——所见的是**一片缓坂耕原**：坂上麦黍参差、几条骊山下温泉道穿坂而过；坂南是骊山林麓、温泉宫华清池（玄宗每年十月驻跸）即在坂西南数里；坂北望渭水滩涂、远见渭北诸陵苍灰一线。**坂上无营壘遗迹**——楚汉前 206 项羽四十万军（号称、实约七八万）屯此不过数月、营垒为土木临时结构、汉以来千余年农耕已尽数夷平；唯坂体本身、骊山北麓那道黄土梁——**是鸿门 marker**。

坂周聚落唐时已成——**新丰县治在坂西十七里**（汉高祖七年/前 200 自丰邑徙父老于此、依丰邑形制立县、故名新丰）；县治与鸿门坂在同一渭南骊山北麓农耕区内；坂东南去临潼骊山温泉、坂西去新丰县市集、坂北通渭水北岸。**鸿门坂北侧汉以来有"项王营"地名**——唐人沿用此名指鸿门项羽屯军处；今鸿门堡村即此地名千余年延续。

物候：**关中渭南春多沙、夏多雨、秋高、冬寒**——坂体苍黄、骊山青翠（南望）、渭水细流（北望）。**这片土坂的视觉气质不在它"曾经如何"——而在它"骊山下一道平展的黄土"**：杯酒之间、刘项分野——是中国政治史最戏剧的一夜在地理上的最朴素 anchor。

## 大唐天宝十四载（755 年）

### 唐人眼里的来路

鸿门在唐人的回望里是**楚汉之际最戏剧的一夜的物理坐标**。

楚汉前 207 年——刘邦从武关入关中、十月至霸上（今西安东郊灞水西岸、距长安东约三十里）；秦王子婴素车白马降于轵道旁；刘邦封秦宫府库、还军霸上、与关中父老约法三章。**前 206 年十二月**（秦历，约前 206 年初；史家通称前 206）——项羽率诸侯兵四十万（号称；实约七八万）破函谷关、屯**鸿门**；刘邦十万军（号称；实约二三万）屯**霸上**——两军相距约四十余唐里。

**《史记·项羽本纪》**记此夜——曹无伤密告项羽：沛公欲王关中、使子婴为相；项羽大怒、飨士卒、明日击破沛公军。**范增劝项羽**——沛公居山东时贪财好色、今入关财物无所取、妇女无所幸——其志不在小；急击勿失。**项伯（项羽季父、张良旧识）夜驰沛公军告张良**——张良劝刘邦不可不来谢项羽。**次日清晨刘邦从百余骑驱至鸿门**——见项羽、谢曰："臣与将军戮力而攻秦——将军战河北、臣战河南——不自意能先入关破秦、得复见将军于此。" **项羽留刘邦与饮**——这是鸿门宴的开始。

**席间**——范增数目项羽、举所佩玉玦以示之者三、项羽默然不应；范增遂出召**项庄**入、令拔剑舞坐中、因击沛公于坐。**项庄拔剑起舞**——**项伯亦拔剑起舞、常以身翼蔽沛公、庄不得击**。张良急出至军门见樊哙、告之"今者项庄拔剑舞、其意常在沛公也"——**樊哙带剑拥盾闯入军门**、卫士仆地、披帷西向立、瞋目视项王、头发上指、目眦尽裂；项王按剑而跽问"客何为者"——张良答"沛公之参乘樊哙者也"——项王曰"壮士、赐之卮酒"——樊哙立而饮之、又赐生彘肩、樊哙覆其盾于地、加彘肩上、拔剑切而啖之。

**少顷**——刘邦借如厕之机、招樊哙出；不辞而别、留张良谢项羽；与樊哙、夏侯婴、靳彊、纪信四人持剑盾步走、从骊山下道芷阳间行回霸上。**张良入谢项羽**——奉白璧一双献项王、玉斗一双献亚父范增。**项王受璧置之坐上**——**亚父受玉斗置之地、拔剑撞而破之**——曰："**唉、竖子不足与谋——夺项王天下者必沛公也——吾属今为之虏矣。**" 这是鸿门宴的全幕。

**鸿门宴的历史后果**——四年后（前 202）刘邦垓下败项羽、项羽自刎乌江；汉立国——中国政治史走向被那一夜改写。**唐人对鸿门宴的认知**——**《史记》项羽本纪 + 高祖本纪 + 樊哙列传**是主要来源；**《汉书·高帝纪》**沿《史记》而稍简；**汉魏六朝间鸿门已成楚汉怀古第一地**——与霸上、广武、垓下、乌江并列为楚汉五大现场。**鸿门 + 霸上 + 灞桥三地几何**——鸿门在长安东六十余里、霸上在长安东三十里、灞桥在长安东三十里（霸上是塬名、灞桥是桥名、二地相近）——三地皆灞水流域、皆楚汉之际刘项分野的物理坐标；唐人时常并提、但需分辨。

**唐承隋制**——鸿门故址属京兆府新丰县境郊原；《元和郡县图志》卷一明记"鸿门坂在县东十七里"——是鸿门坂在唐时方位的正式记载。后世宋《长安志》（宋敏求，1080）卷一五新丰县条复沿其说、是鸿门坂在宋以后地志中的正式收录。

### 此刻

天宝十四载这一年的鸿门——**是骊山北麓一道郊原黄土坂**。

**坂体状态**：东西数里、南北里余的黄土梁仍在；坂顶麦黍参差、坂南骊山林麓、坂北渭南冲积平畴；楚汉时营垒早夷为耕地。**坂上无标识**——盛唐人凭吊鸿门、凭吊的是史记文字与坂体本身的叠加；坂上既无碑、亦无寺；唐人到此须自携《史记》在心中铺《项羽本纪》一段文字、与脚下黄土对照。

**鸿门与骊山温泉宫的几何**——天宝十四载之时玄宗每年十月驻华清宫（在骊山西北麓温泉宫、距鸿门坂西南约五里）——**驾幸温泉的车马道自长安东出、过霸上、经鸿门坂北侧、入华清宫**；**这一年（天宝十四载）十月玄宗曾幸华清宫**（按惯例八月至十月、回銮具体日期据编年史另核）——**玄宗西归长安的车驾即从鸿门坂北过**；**十一月初九安禄山起兵范阳**——再过一月长安震动、玄宗马嵬西奔——**但西奔之路出长安西去、不经鸿门**（鸿门在长安东、向东去潼关之路才经鸿门）。**鸿门坂下的车马道**——这一年是盛唐天子每年驻跸的官道、却也是 962 年前楚汉两军屯营对峙的同一道土坂。

**唐人对鸿门的认知**——**《史记》项羽本纪是必读底本**；**李白《鸿门会》"项王气盖世——紫电明双瞳——呼吸八千人——横行起江东"** + **《飞龙篇》中亦及鸿门意象**；**王维等盛唐诗人在长安东郊出行时多有鸿门凭吊之作**——皆把鸿门作为"楚汉之际最戏剧的一夜"的物理坐标。**盛唐人凭吊鸿门**——凭吊的核心是"**那一杯酒之间、项羽放刘邦走、自此天下归汉**"——这是中国政治史最戏剧的一瞬；项羽的妇人之仁、范增的悲怒玉斗、项伯的私情救亲、项庄的舞剑、樊哙的闯帐——五个人物在这片黄土坂上把一个王朝的兴废演完了。

**周边相关**——鸿门坂西南数里是骊山秦始皇陵（封土仍在、唐时已成关中怀古第一陵）；坂西南五里是华清宫温泉；坂西十七里是新丰县治（汉高祖七年立、唐为京兆府辖县）；坂西北六十余里是长安外郭城——这一片新丰县郊原是关中史迹密度最高的一块土地：秦陵 + 鸿门 + 霸上 + 灞桥 + 华清宫——五处史迹方圆数十里之内。

### 此刻意味

对天宝十四载的长安人来说，鸿门意味着**长安东郊一片最具戏剧张力的怀古地**。**对学士与文人来说**——鸿门是**"项羽放刘邦走的那一夜"的物理纪念**：从这片土坂上回望前 206 年那个十二月之夜——是中国政治史最具偶然性的一瞬；项羽一念之间、刘项分野、楚汉胜负、汉立国、中国此后两千余年走向皆从这一夜开端。**对朝廷来说**——鸿门故址不在禁苑范围之内——是开放的郊原、京兆府新丰县辖；岁修无份、碑立无人；**这一片黄土坂在唐时只属凭吊文人、不属朝廷工程**。

**对盛唐"天宝盛世"这一时刻而言**——鸿门提供的是一道**最戏剧的史鉴**：玄宗朝沉于天宝末政、安禄山起兵于范阳已成必然（11 月初九即起）；玄宗与杨贵妃驻华清宫的车马道、与楚汉两军对峙的鸿门坂、在地理上重合于同一片骊山北麓——**盛唐天子歌舞之地、就是楚汉杯酒决胜之地**；项羽一念放刘邦走、玄宗一念用安禄山——两念之间、两个王朝走向被改写。**755 年这一刻**——天宝十四载十月玄宗刚自华清宫驻跸归长安、车驾从鸿门坂北过；**再过一月**安禄山起兵、再过半年长安失陷、玄宗西奔马嵬——盛唐由"汉武+周文"急转为"楚汉+暴秦"的速亡。**但天宝十四载凭吊鸿门的长安人**——尚不知自己脚下的盛唐也将在两年内成另一个"楚汉之际"。

唐人尚不知道——一千二百余年后（2026）鸿门坂仍在；坂上仍无营垒、仍是骊山下一道平展的黄土；史记中那一杯酒、那一把未拔的剑、那一个未点的玉玦——仍是中国政治史最戏剧的一夜。**但在 755 年这一刻**——鸿门只是骊山北麓一道苍凉的黄土坂；坂南华清宫的玄宗刚回长安；坂北的车马道上是秋末的尘土；坂顶麦茬未拔、坂体寂寂、骊山在南、渭水在北。

## 子条目

### 鸿门坂（"鸿门坂在县东十七里"）

**《元和郡县图志》卷一·关内道·京兆府·新丰县条**："**鸿门坂在县东十七里——汉高祖既入秦——项羽率诸侯兵屯于此——是为鸿门之会。**" 这是鸿门坂在唐时方位的正式记载——以新丰县治为坐标、东十七里、骊山北麓黄土梁。**汉高祖七年（前 200）刘邦自丰邑徙父老于此、依丰邑形制立县、故名新丰**——新丰县治本身就是楚汉之际的物理纪念（刘邦因父亲思乡而立）；鸿门坂在新丰县东十七里——是新丰县境内最具楚汉记忆的一处坐标。**今西安市临潼区新丰镇东、鸿门堡村一带**——坂体仍存、千余年地名延续；2010 年代鸿门遗址公园建成、有"鸿门宴"主题雕塑与展示。

### 项王营 + 沛公营（坂上 vs 霸上）

**鸿门坂北侧汉以来有"项王营"地名**——唐人沿用此名指鸿门项羽屯军处。**项羽四十万大军（号称；实约七八万）屯鸿门坂**——以坂体为天然营垒、南依骊山、北俯渭南平畴、东西控关中通道。**刘邦十万军（号称；实约二三万）屯霸上**——霸上是灞水西岸塬地（今西安市灞桥区一带）、距长安东三十里、距鸿门西约四十余唐里。**两军相距约四十里**——鸿门宴当日刘邦从百余骑自霸上驱至鸿门、骊山下道芷阳脱身回霸上时**仅二十里**（《史记·项羽本纪》"沛公已去——间至军中——道芷阳间行——沛公谓张良曰：从此道至吾军、不过二十里耳"）——**这二十里是抄骊山北麓近道、非来时的鸿门 - 霸上正道**。**鸿门 vs 霸上 vs 灞桥三地不可混**：鸿门在长安东六十余里、霸上在长安东三十里、灞桥在长安东三十里（霸上是塬名、灞桥是桥名、二地相近但非一物）。

### 鸿门宴的几个细节（A 重大史实张力）

**鸿门宴的时间**——**前 206 年十二月**（秦历正月为十月、十二月即秦历末月；约前 206 年初）——史家通称"前 206 年"。**鸿门宴的兵力**——**项羽四十万、刘邦十万**是《史记》原文记载、但学者多认为是号称——实际项羽约七八万、刘邦约二三万；**四十万是楚汉之际诸侯联军总称、十万是关中所有秦降卒加楚军本部**。**鸿门宴的座次**——**《史记·项羽本纪》**："项王、项伯东向坐——亚父南向坐——沛公北向坐——张良西向侍。" **东向为尊**（秦汉室内座次：东向 > 南向 > 北向 > 西向）——项羽自坐东向、亚父范增南向（次尊）、沛公北向（再次）、张良西向（最末，且"侍"非"坐"——侍立或斜坐）——**这套座次本身已表明项羽视刘邦为下属、不视为同等诸侯**。**项庄舞剑、意在沛公**——出自《史记》"项庄拔剑起舞——其意常在沛公也"（张良语樊哙）——是中国成语史最经典的一句、源于此夜。**樊哙闯帐**——项羽赐之卮酒、生彘肩——"生彘肩"或为"半熟彘肩"（汉代肉食烹法争议）——但《史记》原文"生"字——唐人沿之；樊哙以盾覆地、加彘肩上、拔剑切而啖之——这是中国武人形象史上最早的定格。

### 鸿门 vs 霸上 vs 灞桥（地理三辨）

**鸿门**：京兆府新丰县东十七里、骊山北麓黄土坂、长安东六十余唐里——楚汉前 206 项羽屯军处、鸿门宴现场。**霸上**：长安东三十里、灞水西岸塬地——楚汉前 207-206 刘邦屯军处、与鸿门相距四十余里。**灞桥**：长安东三十里、灞水之上桥梁、与霸上相近但不同（霸上是塬名、灞桥是桥名）——唐时是长安东出第一桥、折柳赠别之地（"年年柳色、灞陵伤别"——李白《忆秦娥》）。**三地皆灞水流域、皆楚汉之际刘项分野的物理坐标**——但分属新丰县（鸿门）与万年县（霸上、灞桥）；唐人时常并提、但需分辨。**杜甫陇蜀诗多写关中地理、偶涉灞桥 / 渭水、不专写鸿门**——**杜甫至德二载（757）自鄜州赴凤翔行在路过此片郊原**——这是中晚唐对楚汉怀古的相关笔触、但作于 755 后两年、不入 755 现场。

### 鸿门坂与骊山温泉宫的几何（盛唐 755 现场）

**鸿门坂西南五里是骊山温泉宫华清池**——玄宗每年八月至十月驻跸（《资治通鉴》卷二一七天宝十载十月条 + 二一八天宝十四载十月条）。**长安东出至华清宫的车马道**——出长安春明门、东行三十里至霸上、过灞桥、东行二十余里至鸿门坂北、东南折入骊山温泉宫——**鸿门坂北就是盛唐天子每年驻跸温泉宫的必经之地**。**天宝十四载十月**——玄宗一行按惯例曾幸华清宫（具体起讫日据编年史另核）——回銮长安日期待考；**安禄山起兵在十一月初九**——玄宗回长安时尚不知范阳已起。**鸿门坂下的车马道**——这一年既是天子驾幸的官道、又是 962 年前楚汉两军对峙的同一道土坂——**盛唐天子驻跸之地、就是楚汉决胜之地**；这一层地理重合、是鸿门坂在天宝十四载这一刻最具诗学张力的一面。

## 历史长河（755 → 今）

**中晚唐**　**安史之乱中鸿门坂未直接为战场**——但长安东郊农耕区破坏严重；新丰县治受波及；坂周聚落逃散。**至德元载（756）六月哥舒翰潼关战败**——叛军入关、长安失陷；安军主力沿潼关 - 华阴 - 新丰 - 长安一线西进——**鸿门坂北的车马道这一次是叛军进军道、不是天子驻跸道**。**至德二载（757）九月唐军收复长安**——郭子仪、李嗣业自凤翔出、潼关道收复、鸿门坂复归唐境。**贞元、元和间长安渐复**——鸿门坂周农耕渐恢复；**杜牧、李商隐、温庭筠等中晚唐诗人到长安东郊凭吊鸿门**——皆延楚汉怀古之旧调。**晚唐李山甫《项羽庙》"为虏为王尽偶然"** + **胡曾《咏史诗·鸿门》"项籍鹰扬六合晨——鸿门开宴贺亡秦——樽前若取谋臣计——岂作阴陵失路人"**——是晚唐对鸿门宴的诗学定格。

**五代至宋**　**唐亡（907）后长安衰为西京小邦城**——鸿门坂周农耕继续；新丰县仍存。**北宋元丰三年（1080）宋敏求《长安志》卷一五新丰县条**："鸿门坂在县东十七里——项羽屯兵处。" 延《元和郡县图志》方位、补楚汉旧说——是鸿门坂故址在宋以后地志中最完整的收录。**北宋苏轼《鸿门》诗**"项羽何为不杀刘——区区亚父亦良谋——天教笠泽残宵雪——化作西风万古秋"——是宋人对鸿门宴的咏史定格。**金元间新丰县废兴几度**——但鸿门坂黄土一道始终未动。

**明清**　**明清诸朝鸿门故址成关中农耕区一部分**——坂南有鸿门堡、项王营等聚落；坂体黄土仍存、但已无任何后人重修。**明末清初顾炎武《历代帝王宅京记》《肇域志》关中卷**复考鸿门坂方位、沿《元和》《长安志》旧说。**清乾隆四十年（1775）陕西巡抚毕沅**——在关中诸古迹立碑标识、鸿门坂亦在其中（"鸿门项王营" 石标）。**毕沅碑文延《史记》项羽本纪原文**——是清代金石学家 / 方志学者对鸿门坂认知的最高水位。

**民国至今**　**1956 年鸿门坂列入陕西省第一批省级文物保护单位**。**1960-1980 年代陕西省考古所对鸿门坂周边进行多次勘探**——确认坂体范围、楚汉时期文化层（陶片、瓦砾）；但**未发现大型营垒遗迹**——印证《史记》"项羽屯鸿门"为临时大军屯营、非长期固定工程。**1980 年代鸿门堡村保留"项王营"地名**——千余年地名不改。**2007 年鸿门宴遗址公园建成**——保护坂体、立"鸿门宴" 主题雕塑（项羽、刘邦、范增、樊哙、项庄五人物）、修步道与展览馆。**2018-2024 年鸿门宴遗址公园进一步完善**——是中国楚汉文化的标杆性主题遗址之一。**今游人登坂凭吊**——所对的是一片**骊山北麓的黄土平展坂体**：坂上无营垒、坂南骊山青翠、坂北渭南平畴；脚下是 2220 年前那一夜杯酒之间的剑、酒、玉玦——**与唐人 755 时所凭吊的同一片土坂、同一段史记文字**——两套鸿门、两种凭吊、却都是同一夜。
