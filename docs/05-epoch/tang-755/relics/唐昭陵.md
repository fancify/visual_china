# 唐昭陵

## 一句话

唐太宗李世民陵——**贞观十年六月长孙皇后崩、葬昭陵**（昭陵营建之始）；**贞观二十三年（649）八月太宗葬此**——**唐"因山为陵"首座**、九嵕山主峰为穴；**755 时已 119 年、陪葬墓近 200 座**——唐六陵规模最大者。

## 形貌

昭陵在京兆府醴泉县东北约 22.5 公里——**九嵕山主峰**（今陕西省咸阳市礼泉县烟霞镇九嵕山）。**九嵕山**：海拔约 1188 米——是渭北高原与陕北高原交界的一座大山。**昭陵"因山为陵"**——以九嵕山主峰为穴、地宫凿入山体；陵园依山势铺开。

**陵区规模**——**周回约 60 公里**（《长安志》"周回一百二十里"，按唐里约 60 公里）；是唐六陵中规模最大者。**陵园**：四面有夯土围墙；南有献殿（祭祀用、今基址仅存）；神道北起献殿、南至**朱雀门**——约长 1.5 公里、两侧立**石人、石虎、石马、石犀、华表**等石刻。

**昭陵六骏**——神道北端的**六块石屏**：太宗为纪念其六匹战马（飒露紫、拳毛䯄、白蹄乌、特勒骠、青骓、什伐赤）所凿；每块石屏高约 1.7 米、宽 2 米、厚 0.4 米；浮雕一骏——**是唐代石刻艺术的巅峰之一**。

**陪葬墓**：**《唐会要·陪陵名位》列昭陵陪葬名氏甚详**；**现代调查与考古勘察**称昭陵陪葬墓**近 200 座**（昭陵博物馆口径"近 200"；近年研究截至 2022 年称 203 座，本 doc 不落死数）——含贞观朝大半功臣 + 宗室。**著名陪葬墓**：**长孙皇后**（陵主皇后）、**长孙无忌**（皇后兄）、**房玄龄**、**杜如晦**、**李靖**、**李勣（徐茂公）**、**程咬金**、**尉迟敬德**、**魏徵**、**温彦博**、**段志玄**、**萧瑀** 等；**太子李承乾**（贞观十七年/643 废、贞观十九年/645 卒，葬昭陵）；**新城公主、长乐公主、城阳公主** 等太宗诸女。**陪葬墓主体在陵东、东南**——成大型陪葬区。

物候：九嵕山一带为关中渭北、暖温带季风气候——夏热冬寒、雨集中夏季；山中林木在唐时较盛，盛唐文献记九嵕山有古松柏数千株。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

昭陵在唐人的回望里——是**唐"因山为陵"首座**、**贞观盛世的物理 anchor**。

**唐太宗李世民（598-649）**：北周武川镇出身贵族（陇西成纪人系自我塑造）；隋大业末从父李渊起兵于太原、立唐有大功；**武德九年（626）玄武门之变**——诛太子建成、齐王元吉；同年八月即位为唐太宗。**贞观（627-649）23 年间**——太宗与文德皇后长孙氏并治、纳魏徵谏、平突厥、定吐谷浑、通西域——**唐贞观盛世立**。

**昭陵的营建**——**贞观十年（636）六月长孙皇后崩**、太宗为后营山陵于九嵕山；**同年十一月葬长孙皇后于昭陵**——这是昭陵营建之始。太宗诏"**因山为陵、不藏金玉**"（《唐会要》卷二十）——**自此唐五陵皆因山为陵**（献陵除外）；同时"不藏金玉" 是太宗对汉魏厚葬的反对。

**贞观二十三年（649）五月己巳**——太宗崩于翠微宫含风殿（终南山中离宫、非昭陵、非玉华宫——本 doc 严守）；**八月庚寅**——太宗葬于昭陵、与文德皇后长孙氏合穴。**自此昭陵成唐"帝后合葬"首座**（长孙皇后非帝、与太宗合葬属"帝后"；高宗 + 武则天合葬乾陵则是唐陵中唯一"二帝合葬"——见 [唐乾陵](./唐乾陵.md)）。

**昭陵六骏**：**贞观十年（636）至贞观二十三年间陆续凿**——太宗为纪念其六匹战马（**白蹄乌-破薛仁杲；特勒骠-破宋金刚；拳毛䯄-破刘黑闼；青骓-破窦建德；飒露紫、什伐赤-与王世充洛阳战事相关**——皆隋末至武德间太宗骑乘的六匹战马；具体战役映射史源略有异说，本 doc 取主流口径）；六骏石屏立神道北端、是唐代石刻艺术的巅峰。**铭文**：六骏石屏上原有太宗御制铭、由欧阳询书丹——今铭已多湮灭。

### 此刻

天宝十四载这一年的昭陵——**已立 119 年**。

**陵阙、献殿、神道、石刻**：盛唐 755 时主体仍立；**陵园四面夯土围墙**当时尚完。**九嵕山顶的地宫**：太宗 + 长孙皇后合穴、深凿入山体——具体盛唐时是否有盗扰、史载未详（一般认为安史前唐陵未被大规模盗扰）。**昭陵六骏**：仍立神道北端、原位、未流失。

**陪葬墓**：近 200 座中绝大多数应在 755 时已葬入（贞观至高宗朝陆续）。**陪葬墓的祭祀**——是唐皇室对开国功臣家族的政治照应；具体盛唐时陪葬墓家族是否仍享公祭、史载零散。

**国家祭祀**：唐玄宗朝昭陵的祭祀按例行——每岁春秋遣陵令行祭；**开元、天宝间玄宗多次亲祀昭陵**（开元二十三年/735、天宝间数次，但未明记具体年份）。**陵令**驻陵旁、典守陵区。**昭陵在唐人心中的地位**：**是李唐皇室祖宗的最重者**——太宗的政治遗产、贞观盛世的物理纪念。

**安史前夕**（即 755 这一刻）——再过半年、安禄山起兵于范阳；再过一年潼关失、玄宗西奔入蜀——**关中诸唐陵在安史中均有兵燹**（具体昭陵损毁程度史载未详）；**至德 / 乾元间唐军反攻关中**——昭陵一带反复易主、陵区岁修中断。**昭陵在 8 年的安史动荡中**——具体物理损毁程度史载零散。

### 此刻意味

对天宝十四载的醴泉人来说——昭陵是**县东北 22.5 公里的祖陵**：太宗的陵、岁岁有官祭、近 200 座陪葬墓中含贞观朝大半功臣。对长安那一代士人来说——昭陵是**唐贞观盛世的物理纪念**：太宗的"因山为陵"+ "不藏金玉" 立国规制；昭陵六骏是太宗自身战功的具象。**杜甫《北征》《饮中八仙歌》等盛唐诗中"贞观"二字所指的物理 anchor 即昭陵**。

唐人尚不知道——再过 130 余年（885 黄巢之乱）唐六陵将被陆续盗掘；**五代温韬大规模盗发唐陵**（《新五代史·温韬传》）——**昭陵被温韬盗发**、太宗陪葬器物大半被掠（含传说的王羲之《兰亭集序》原本——若昭陵确有此本陪葬、《新五代史》记温韬掠"上下钟王笔迹纸墨皆漆"）。再过 1300 年（1914），**昭陵六骏中飒露紫 + 拳毛䯄被盗运至美国**——今藏宾夕法尼亚大学博物馆；**陕西碑林博物馆藏其余 4 骏**——**1300 年后的六骏分散两国**。

## 子条目

### 九嵕山地宫（太宗 + 长孙皇后合葬）

**九嵕山主峰**——海拔约 1188 米；地宫凿入山体南坡。**贞观十年（636）十一月长孙皇后先葬**（开穴时间）；**贞观二十三年（649）八月太宗后葬入**——是唐"帝后合葬"首座（高宗 + 武则天合葬乾陵是其后）。**地宫具体内部形制**——盛唐至五代间被温韬盗发后，详细规制史载不全；**今地宫位置已确认**（1972 后多次考古勘察）但**地宫主体未发掘**——是中国考古学界保留态度的代表。**地宫之深、之大**——盛唐人记"凿山七十丈深"（《长安志》引《唐会要》）；具体数字史源略异。

### 昭陵六骏（神道石刻）

**贞观间凿**——太宗为纪念其六匹战马所凿。**六骏名 + 战功**（主流口径；具体战役映射史源略有异说）：
- **白蹄乌**：破薛仁杲（武德元年/618）；
- **特勒骠**：破宋金刚（武德三年/620）；
- **拳毛䯄**：破刘黑闼（武德四年/621）；
- **青骓**：破窦建德（武德四年/621）；
- **飒露紫 + 什伐赤**：与王世充洛阳战事相关（武德四年/621）。

**六骏石屏**——每块高约 1.7 米、宽 2 米、厚 0.4 米；浮雕一骏；铭文为太宗御制、欧阳询书丹（今铭多湮）。**1914 年飒露紫 + 拳毛䯄被毕士博 + 卢芹斋盗运至美国**——今藏**宾夕法尼亚大学博物馆**；**其余 4 骏（白蹄乌、特勒骠、青骓、什伐赤）今藏陕西碑林博物馆**。**唐人在 755 所见的那六匹石马**——**今分两国博物馆、相距万里**。

### 陪葬墓（《唐会要》列名 + 近 200 座）

**《唐会要·陪陵名位》列昭陵陪葬名氏甚详**；**现代调查与考古勘察称昭陵陪葬墓近 200 座**（昭陵博物馆口径；近年研究截至 2022 年称 203 座，本 doc 不落死数）——**含贞观朝大半功臣 + 宗室 + 公主**。**著名陪葬墓**：
- **长孙皇后**（陵主皇后、合葬地宫）
- **长孙无忌**（皇后兄、太宗朝首席宰相、显庆四年/659 被武则天逼死）
- **房玄龄**（太宗朝首席宰相、贞观二十二年/648 卒）
- **杜如晦**（太宗朝宰相、贞观四年/630 卒）
- **李靖**（太宗朝大将、贞观二十三年/649 卒）
- **李勣（徐茂公）**（太宗朝大将、总章二年/669 卒；陪葬太宗陵）
- **程咬金**（太宗朝大将、麟德二年/665 卒）
- **尉迟敬德**（太宗朝大将、显庆三年/658 卒）
- **魏徵**（太宗朝直谏名臣、贞观十七年/643 卒）
- **温彦博、段志玄、萧瑀** 等贞观朝功臣
- **李承乾**（太宗废太子、贞观十九年/645 卒）
- **新城公主、长乐公主、城阳公主**等太宗诸女

**陪葬墓多在陵东、东南**——成大型陪葬区。**陪葬制度**：太宗诏立的"陪葬陵"制度——是唐皇室对开国功臣家族的最高礼遇。

### "因山为陵"立制

**贞观十年（636）太宗诏"因山为陵、不藏金玉"**——是唐皇室陵制的关键转折。**自此唐五陵皆因山为陵**：昭陵（九嵕山）、乾陵（梁山）、桥陵（丰山）、定陵（龙泉山）、惠陵（蒲城）；**献陵除外**（贞观九年/635 太宗依旧礼"积土为陵")。**"因山为陵"的好处**：节省营建 + 不易盗扰（地宫深凿山体）+ 山势壮观。**"不藏金玉" 的好处**：反对汉魏厚葬、节省国库。

### 长孙皇后（陵主皇后）

**长孙皇后（601-636）**——太宗皇后；隋洛阳人；与太宗结发夫妻、贤德著称；贞观间多次劝太宗纳谏（尤护魏徵）。**贞观十年（636）六月崩、年三十六**；同年十一月葬昭陵——是昭陵营建之始。**长孙皇后是昭陵的实际开启者**：地宫先开为后穴、十三年后太宗葬入；这是唐皇室"先后葬"的首座。

## 历史长河（755 → 今）

**中晚唐**　**至德 / 乾元间唐军反攻关中**——昭陵一带反复易主、陵区岁修中断、应有局部毁损。**贞元 / 元和间唐廷尝试恢复唐陵岁修**——但昭陵规模大、岁修负担最重。**中晚唐昭陵的祭祀渐稀**——但作为太宗陵、政治地位仍是诸帝陵中最高。

**唐末黄巢 + 五代**　**广明元年（880）黄巢入长安**——唐陵在乱中受波及；**五代温韬大规模盗发唐陵**（《新五代史·温韬传》）——**昭陵被温韬盗发**：**温韬入昭陵地宫**、取陪葬器物大半（含传说的王羲之《兰亭集序》原本——若昭陵确有此本陪葬、《新五代史》记温韬掠"上下钟王笔迹纸墨皆漆"）。**这是昭陵地宫第一次被人为破坏**。

**两宋**　**北宋初宋太祖诏修唐陵**——昭陵列入岁修；**北宋元丰间宋敏求《长安志》**记昭陵规模 + 陪葬墓近 200 座——是宋代昭陵的关键文献。**北宋昭陵地表石刻在场**——六骏、石人、石虎、华表等仍立。

**金元**　金元间昭陵岁修渐废；**元代陵区为农田覆盖、地表石刻多有埋没**。

**明清**　**明清间昭陵地表大部已成荒野**——但九嵕山主峰未易；陵前石刻露天立于山下。**清乾隆四十年（1775）陕西巡抚毕沅立"唐太宗昭陵"碑**。**1914 年昭陵六骏中飒露紫 + 拳毛䯄被毕士博 + 卢芹斋盗运至美国**——是中国 20 世纪文物流失的代表案例。

**民国至今**　1949 年后昭陵列陕西省重点文物保护单位、1961 年列首批全国重点。**1972 年起多次考古勘察**——确认陵冢、献殿基址、神道、近 200 座陪葬墓位置；**陪葬墓发掘**：长乐公主墓、新城公主墓、李勣墓、尉迟敬德墓等先后发掘——出土大量唐代壁画、墓志、器物。**1978 年昭陵博物馆开放**——以陪葬墓出土文物 + 4 件六骏石屏为核心。**陕西碑林博物馆藏其余 4 骏 + 美国宾大藏 2 骏**——今游人来昭陵博物馆所见六骏多为复制品。**唐人在 755 所见的那座九嵕山主峰**——一千二百多年未易；但献殿、石刻群、六骏、围墙——多已不复盛唐原貌。
