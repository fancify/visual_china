# 唐桥陵

## 一句话

唐睿宗李旦陵——**开元四年（716）六月睿宗崩、十月葬桥陵**；蒲城县丰山为穴、是**玄宗为父建造的陵**——开元朝盛唐石刻艺术的代表；**755 时已 39 年、陵前石刻完整、是诸唐陵中艺术水准最高的一座**。

## 形貌

桥陵在京兆府蒲城县城西北约 15 公里——**丰山**（一名"金粟山"；今陕西省渭南市蒲城县城西北桥陵镇）。**丰山**：海拔约 751 米——是渭北高原一座中等高度的山。**桥陵"因山为陵"**——以丰山主峰为穴、地宫凿入山体；陵园依山势南北铺开。

**陵区规模**——周回约 20 公里；陵园四面有夯土围墙；**南北中轴**自南向北：朱雀门 → 神道 → 献殿 → 陵主穴。

**神道石刻**：桥陵神道石刻**是开元朝盛唐石刻艺术的代表**——精度、写实、气势俱在唐六陵中名列前茅。自南向北依次有：
- **石柱（华表）**1 对
- **獬豸**（独角神兽）1 对——是桥陵独有的石兽（其他唐陵神道少见）
- **鸵鸟**1 对——浮雕在屏石上
- **石马**5 对
- **翁仲（石人）**10 对
- **石狮**1 对
- **朱雀门**

**桥陵神道石刻的特点**：**翁仲高约 3.6 米**——比昭陵翁仲更大、更写实；**石狮**形态雄健、是盛唐石狮的标杆。**今桥陵神道石刻多数仍在原位**——是唐六陵中神道石刻保存最完整、艺术水准最高的一座。

**陪葬墓**：**《唐会要·陪陵名位》列桥陵陪葬名氏 7 人**——**惠宣太子李业、惠庄太子李㧑、惠文太子李范、金仙公主、梁国公主、鄎国公主、李思训**；**蒲城地方志 + 现代考古调查记至少十余座陪葬墓**（具体总数史源不一）。**陪葬墓主要在陵东、东南**——成大型陪葬区。

物候：丰山一带为关中渭北、暖温带季风气候——夏热冬寒、雨集中夏季；陵区今为台塬农田、果园（多种苹果、葡萄）。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

桥陵在唐人的回望里——是**唐睿宗李旦的陵 + 玄宗为父建造的盛唐陵**。

**唐睿宗李旦（662-716）**：高宗 + 武则天第四子；**两度即位的特殊经历**：
- **第一次即位**：嗣圣元年（684）二月——武则天废中宗李显后立李旦为帝（年 23 岁）；但实际武则天临朝称制、李旦只是傀儡；天授元年（690）九月李旦让位、武则天即帝位、立"武周"。
- **退为皇嗣**：690-705 间李旦为"皇嗣"——名义上是武则天的继承人、实际地位低；圣历元年（698）武则天召中宗回长安立为太子、李旦改封相王。
- **第二次即位**：景云元年（710）六月——韦后毒杀中宗、立温王李重茂为帝；不久李隆基（李旦第三子、后为玄宗）与太平公主（李旦妹）联手发动唐隆政变、诛韦后；六月李重茂禅位、**李旦即位为唐睿宗**（年 49 岁）。
- **让位玄宗**：先天元年（712）八月——睿宗禅位给太子李隆基（唐玄宗）、自称"太上皇"；至先天二年（713）"先天政变"后睿宗完全退出政治。
- **太上皇 4 年**：开元元年（713）至开元四年（716）——睿宗居百福殿、不预政事；**开元四年（716）六月**——睿宗崩于百福殿、年五十五。

**桥陵的营建**——**开元四年六月睿宗崩**；玄宗命营桥陵于蒲城丰山；**同年十月葬睿宗于桥陵**——是玄宗朝营建的第一座唐皇室帝陵。**桥陵选址在蒲城**——非礼泉、非乾县（昭陵、乾陵所在）——是开元朝唐陵向东扩散的物理起点。

**桥陵神道石刻**：开元四年（716）至开元五年（717）间陆续凿成——是**开元朝盛唐石刻艺术的代表**。玄宗朝石刻艺术达盛唐高峰——桥陵的翁仲、石狮、獬豸 等石刻精度、气势、写实均超昭陵。**这是玄宗对父亲的最高致敬**。

### 此刻

天宝十四载这一年的桥陵——**已立 39 年**。

**陵阙、献殿、神道、石刻**：盛唐 755 时主体仍立、完整无损；**神道石刻**——开元朝原物、未受任何损坏。**地宫**：睿宗合穴（睿宗皇后刘氏 + 窦氏先葬入丰山——《新唐书》记），具体地宫规制盛唐时人即不知、史载未详。

**陪葬墓**：《唐会要》列 7 名氏 + 至少十余座——多在 716-742 间陆续葬入（睿宗诸子诸女）；具体陪葬区状态盛唐 755 时应主体完整。

**国家祭祀**：唐玄宗朝桥陵的祭祀按例行——**桥陵是玄宗本人父亲的陵、地位最高**；开元、天宝间玄宗多次亲祀桥陵（《唐会要》《资治通鉴》皆有载）。**陵令**驻陵旁、典守陵区。

**安史前夕**（即 755 这一刻）——再过半年安禄山起兵；再过一年潼关失、玄宗西奔——**关中诸唐陵在安史中均有兵燹**（具体桥陵损毁程度史载未详）；**至德 / 乾元间唐军反攻关中**——桥陵一带反复易主、陵区岁修中断。**桥陵在 8 年的安史动荡中**——具体物理损毁程度史载零散；但今桥陵神道石刻多数仍在原位、推测主体未大毁。

### 此刻意味

对天宝十四载的蒲城人来说——桥陵是**县西北的睿宗陵**：玄宗父亲的陵、岁岁有官祭、陪葬墓（《唐会要》列 7 名氏 + 至少十余座）中含睿宗诸子诸女。对长安那一代士人来说——桥陵是**玄宗朝政治源头的物理纪念**：睿宗让位 + 玄宗即位的故事系于此陵。**桥陵神道石刻** 是 755 时盛唐人来蒲城见到的最直观的"开元朝石刻艺术"。

唐人尚不知道——再过 130 余年（885 黄巢之乱）唐六陵将被陆续盗掘；**五代温韬大规模盗发关中唐陵**——《新五代史·温韬传》"唐诸陵在境内者悉发"、"唯乾陵风雨不可发"；**桥陵据此推定亦在被盗之列**（具体细节史载未详）；再过千二百年（21 世纪），**桥陵神道石刻仍立**——多数为原位原物；**桥陵地宫至今未发掘**。**唐人在 755 所见的那 17 对 / 数十座神道石刻**——千二百多年后**绝大多数仍立桥陵神道**——是唐六陵中神道石刻保存最完整的一座、是**盛唐人留给后世的最直观的开元石刻**。

## 子条目

### 丰山地宫（睿宗 + 皇后合葬）

**丰山主峰**——海拔约 751 米；地宫凿入山体南坡。**开元四年（716）十月葬睿宗**——同时合葬睿宗已先葬的皇后刘氏、窦氏（窦氏是玄宗生母、被武则天处死、追谥皇后）。**地宫具体内部形制**——盛唐时人即不知、史载未详；**至今未发掘**。

### 桥陵神道石刻（开元朝盛唐石刻代表）

桥陵神道石刻**是开元朝盛唐石刻艺术的代表**——精度、写实、气势俱在唐六陵中名列前茅。**主要石刻**：
- **石柱（华表）**1 对——立神道南端
- **獬豸**（独角神兽）1 对——桥陵独有；象征"辨正邪"
- **鸵鸟**1 对——浮雕在屏石上、是盛唐对西域来贡物的纪念
- **石马**5 对——配马夫立像
- **翁仲（石人）**10 对——文武各 5 对；**高约 3.6 米**——是唐陵翁仲中最大者
- **石狮**1 对——立朱雀门外、形态雄健、是盛唐石狮标杆

**今桥陵神道石刻多数仍在原位**——是唐六陵中神道石刻保存最完整的一座。**1960s-2010s 多次考古勘察**——确认各石刻位置；现立保护亭或露天保护。**盛唐 755 时所见的这片石刻**——一千二百多年后游人所见的多为同一件、同一位。

### 陪葬墓（《唐会要》列 7 名氏 + 至少十余座；睿宗诸子诸女为主）

**《唐会要·陪陵名位》列桥陵陪葬名氏 7 人**——主要为**睿宗子女**：
- **惠庄太子李㧑**（睿宗第二子）
- **惠文太子李范**（睿宗第四子、玄宗朝相王、730 卒）
- **惠宣太子李业**（睿宗第五子、734 卒）
- **金仙公主**（睿宗女、玄宗胞姊）
- **梁国公主**（睿宗女）
- **鄎国公主**（睿宗女）
- **李思训**（开元朝画家、宗室；陪葬桥陵）

**蒲城地方志 + 现代考古调查**记至少十余座陪葬墓（具体总数史源不一）。**陪葬墓主体在陵东、东南**——成大型陪葬区。**惠庄、惠文、惠宣三太子陪葬**——是玄宗对兄弟（睿宗诸子）的礼遇；**这种"皇兄弟陪葬"格局**——为开元朝皇室"兄弟和睦"的政治叙事的物理体现。

### 让皇帝李宪（玄宗兄）与桥陵

**李宪（679-742）**——睿宗长子、玄宗胞兄、原立太子；景云元年（710）后让太子位给三弟李隆基（玄宗）。**李宪葬于桥陵附近的惠陵**（见 [唐惠陵](./唐惠陵.md)）——与桥陵同县不同陵。**李宪与桥陵的关系**：睿宗长子、本应陪葬桥陵；但玄宗追封李宪为"让皇帝"、另立惠陵——是开元至天宝朝皇室"让位之美"的物理 anchor。

### 玄宗朝石刻艺术与桥陵

**开元朝（713-741）是盛唐石刻艺术的高峰**——桥陵神道石刻是这一高峰的代表。**与昭陵神道石刻比**：昭陵石刻（贞观间凿）写实但稍朴；桥陵石刻（开元间凿）写实 + 气势 + 比例俱佳。**与乾陵神道石刻比**：乾陵石刻（高宗末至神龙间凿）规模宏大；桥陵石刻规模略小、但精度更高。**桥陵的"獬豸"独角神兽**——是唐陵神道石刻中独有的一种、表"辨正邪"——是玄宗朝对父亲睿宗"明辨忠奸"政治品格的纪念。

## 历史长河（755 → 今）

**中晚唐**　**至德 / 乾元间唐军反攻关中**——桥陵一带反复易主、陵区岁修中断。**贞元 / 元和间唐廷尝试恢复唐陵岁修**——桥陵列入。**中晚唐桥陵的祭祀**：玄宗朝高祭、肃宗朝以后渐稀；但作为玄宗父陵、政治地位仍是中晚唐诸帝陵中较高。

**唐末黄巢 + 五代**　**广明元年（880）黄巢入长安**——唐陵在乱中受波及；**五代温韬大规模盗发关中唐陵**——《新五代史·温韬传》"唐诸陵在其境内者悉发"、"唯乾陵风雨不可发"；**桥陵据此推定亦在被盗之列**（具体细节史载未详）：陪葬器物大半被掠；但神道石刻多在原位、未大毁。

**两宋**　**北宋初宋太祖诏修唐陵**——桥陵列入岁修；**北宋元丰间宋敏求《长安志》**追述桥陵规模 + 陪葬墓（具体数目史源不一）。

**金元明清**　金元明清间桥陵地表石刻部分有损毁（一些翁仲被毁或埋没）——但主峰丰山未易、神道石刻多在原位。**清乾隆四十年（1775）陕西巡抚毕沅立"唐睿宗桥陵"碑**。

**民国至今**　**1949 年后桥陵列陕西省重点文物保护单位、1988 年列全国重点**。**1960s-2010s 多次考古勘察**——确认陵冢、献殿基址、神道石刻、陪葬墓（《唐会要》列 7 名氏 + 至少十余座）位置。**部分陪葬墓发掘**：惠庄太子墓、惠文太子墓、惠宣太子墓等先后发掘——出土壁画、墓志。**今桥陵神道石刻多数仍在原位**——是唐六陵中保存最好的神道；**1990s 后桥陵神道露天保护**——成著名石刻 + 帝陵景区。**唐人在 755 所见的那 17 对 / 数十座神道石刻**——千二百多年后绝大多数仍立丰山神道。
