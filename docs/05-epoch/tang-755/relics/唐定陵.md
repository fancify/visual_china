# 唐定陵

## 一句话

唐中宗李显陵——**景龙四年（710）六月中宗被韦后毒杀、景云元年（710）十一月葬定陵**；富平县龙泉山为穴；**755 时已 45 年、陵阙石刻俱在**——是中宗"二度即位、被韦后毒杀" 的悲剧陵。

## 形貌

定陵在京兆府富平县城西北约 13 公里——**龙泉山**（又称凤凰山；今陕西省渭南市富平县宫里镇凤凰山）。**龙泉山**：海拔约 751 米——是渭北高原一座中等高度的山。**定陵"因山为陵"**——以龙泉山主峰为穴、地宫凿入山体；陵园依山势南北铺开。

**陵区规模**——周回约 10 公里（较桥陵小）；陵园四面有夯土围墙；**南北中轴**：朱雀门 → 神道 → 献殿 → 陵主穴。

**神道石刻**：自南向北有华表、翼马、鸵鸟、石马、翁仲（石人）、石狮等——是唐"因山为陵"陵制的标准神道；具体石刻数量较桥陵略简（中宗政治地位介于睿宗与玄宗之间、定陵规制相应有别）。**今定陵神道石刻多有损毁或埋没**——保存程度不及桥陵。

**陪葬墓**：定陵陪葬墓**较其他唐陵少**——主要含**节愍太子李重俊墓**（中宗第三子、神龙政变后立太子、景龙元年/707 发动"重俊政变"诛韦后失败被杀）等。**注**：**温王李重茂**（中宗第四子、景龙四年/710 短暂为帝 17 天）**葬于武功西原、非定陵陪葬**——本 doc 严守。**陪葬墓数少**——反映中宗政治势力的相对衰弱。

物候：富平一带为关中渭北、暖温带季风气候——夏热冬寒、雨集中夏季；陵区今为台塬农田、果园（多种石榴、柿子——富平柿饼是当地物产）。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

定陵在唐人的回望里——是**唐中宗李显的陵 + 韦后毒杀帝王的政治悲剧**。

**唐中宗李显（656-710）**：高宗 + 武则天第三子；**两度即位的特殊经历**：
- **第一次即位**：嗣圣元年（684）一月——高宗崩、李显即位为中宗（年 28 岁）；不到两个月（同年二月）被武则天废为庐陵王；自此**流放均州（湖北十堰）、房州（湖北房县）14 年**——中宗第一次帝位 仅 55 天。
- **流放期**：684-698 间、14 年——中宗与韦皇后（嫁中宗为太子妃、共流放）相依为命；这一段是中宗的"政治死亡期"、亦是与韦后情感最深的一段。
- **召回长安立太子**：圣历元年（698）武则天召中宗回洛阳立为皇太子（实际等于召回内定继承人）。
- **第二次即位**：神龙元年（705）正月——张柬之等发动"神龙革命"、迫武则天禅位；中宗复辟即位（年 49 岁）——这是中国历史上少有的"流放 + 复辟"皇帝。
- **韦后干政**：705-710 间——韦皇后效仿武则天干政；与女儿安乐公主谋权；中宗朝政治日乱。
- **被韦后毒杀**：**景龙四年（710）六月**——韦后与安乐公主合谋、在饼中下毒；**中宗于神龙殿食毒饼而崩**——年 55；**韦后立温王李重茂（中宗第四子，年仅 16 岁）为帝**、自己临朝称制；不及一月（六月二十日）**李隆基（睿宗第三子、后为玄宗）与太平公主**发动"**唐隆政变**"——诛韦后、安乐公主、温王让位、睿宗李旦复立。

**定陵的营建**——**景云元年（710）秋八月**睿宗即位后命营定陵于富平龙泉山；**同年十一月**葬中宗于定陵——是睿宗朝营建的第一座唐皇室帝陵。**定陵规制**：因山为陵、规模略小——是中宗政治地位（被废、流放、复辟、被毒杀）的物理反映。

### 此刻

天宝十四载这一年的定陵——**已立 45 年**。

**陵阙、献殿、神道、石刻**：盛唐 755 时主体仍立；但因中宗政治地位 + 韦后乱政的关系——定陵的祭祀分量在唐六陵中较低。**龙泉山地宫**：中宗 + 后葬入（具体合葬何后、史源略异——韦后被诛、未葬定陵；中宗原配赵皇后早卒、葬陵不详）；**地宫具体内部形制盛唐时人即不知、史载未详**。

**陪葬墓**：8 座中**节愍太子李重俊墓**（神龙政变后立太子、景龙元年/707 反韦后失败被杀）是最显的一座；李重俊与其父中宗皆是韦后政治悲剧的牺牲品——**父子同葬一陵区**。

**国家祭祀**：唐玄宗朝定陵的祭祀按例行——但**玄宗对中宗（其伯父）的态度复杂**：中宗是玄宗父睿宗的胞兄、又是被韦后毒杀者；玄宗即位后追谥中宗"大和大圣大昭孝皇帝"——名号上礼遇极高，但实际祭祀分量不及桥陵（玄宗自己父亲陵）。

**安史前夕**（即 755 这一刻）——再过半年安禄山起兵；再过一年潼关失、玄宗西奔——**关中诸唐陵在安史中均有兵燹**（具体定陵损毁程度史载未详）；**至德 / 乾元间唐军反攻关中**——定陵一带反复易主、陵区岁修中断。

### 此刻意味

对天宝十四载的富平人来说——定陵是**县西北的中宗陵**：被韦后毒杀的中宗、神龙革命复辟皇帝的陵、岁岁有官祭。对长安那一代士人来说——定陵是**中宗朝政治悲剧的物理纪念**：流放 14 年、复辟 5 年、被韦后毒杀——这是唐皇室"被废 + 复辟 + 被毒杀"的极端样本。**定陵的政治分量**：在唐六陵中是较低的一座——但作为玄宗朝唐皇室家族的重要纪念地、玄宗仍维持其岁祭礼。

唐人尚不知道——再过 130 余年（885 黄巢之乱）唐六陵将被陆续盗掘；**五代温韬大规模盗发关中唐陵**——《新五代史·温韬传》"唐诸陵在境内者悉发"、"唯乾陵风雨不可发"；**定陵据此推定亦在被盗之列**（具体细节史载未详）；再过千二百年（21 世纪），**定陵神道石刻多有损毁**——保存程度不及桥陵；**地宫未发掘**。**唐人在 755 所见的那座"被韦后毒杀皇帝的陵"**——一千二百多年后陵冢仍立、神道石刻部分犹存、地宫主体未被发掘。

## 子条目

### 龙泉山地宫（中宗合葬）

**龙泉山主峰**——海拔约 751 米；地宫凿入山体南坡。**景云元年（710）十一月葬中宗**；**合葬皇后**——具体合葬何后史源略异（韦后被诛后被废为庶人、未葬定陵；中宗原配赵皇后早卒；可能有其他妃葬入）。**地宫具体内部形制**——盛唐时人即不知、史载未详；**至今未发掘**。

### 神道石刻（多有损毁）

定陵神道石刻自南向北有华表、翼马、鸵鸟、石马、翁仲、石狮等——较桥陵略简。**今定陵神道石刻保存程度较低**——多数已残损或被埋没；部分石人、石狮存于**富平县文物管理所**或原位。**1990s-2010s 多次考古勘察**——确认神道布局；但因地表破坏严重，复原困难。

### 节愍太子李重俊（陪葬定陵）

**李重俊（？-707）**——中宗第三子；神龙革命复辟后被立为皇太子（神龙三年/707 立）。**重俊政变**：因不满韦后、武三思（武则天侄）干政——景龙元年（707）七月李重俊与左羽林大将军李多祚等发动政变、诛武三思、武崇训父子；进攻肃章门、为韦后中宗所阻——李重俊出奔终南山被害；年约 23-24 岁。**711 年睿宗朝追谥"节愍太子"、改葬定陵**——成为定陵陪葬墓的核心。**李重俊与父中宗在韦后政治悲剧中先后被害**——是定陵最沉重的一笔。

### 韦后毒杀中宗事

**景龙四年（710）六月辛未**——中宗于神龙殿食毒饼而崩。**毒源**：韦后 + 安乐公主合谋（《资治通鉴》《旧唐书》《新唐书》皆载）；具体下毒动机：韦后欲效武则天临朝称制、安乐公主欲被立为"皇太女"。**毒杀方法**：传说在饼中下毒（《旧唐书》记韦后"与安乐公主合谋、内毒于饼以献"）；中宗食后崩。**这是中国历史上少有的"皇后毒杀皇帝"事件**。**事后**：韦后立李重茂为帝、自己临朝称制；不及一月被李隆基、太平公主发动"唐隆政变"诛——韦后、安乐公主皆被杀。

### 温王李重茂（葬武功西原，非定陵）

**李重茂（约 695-714）**——中宗第四子；景龙四年（710）中宗被毒杀后被韦后立为帝、年 16；在位仅 17 天即被李隆基迫退位（"唐隆政变"）；改封温王。**先天二年（713）改封襄王**；**开元二年（714）卒、年约 19**——具体死因史源不一。**李重茂葬于武功西原、非定陵陪葬**——本 doc 严守。**这是中国历史上少有的"在位 17 天 + 19 岁卒" 的悲剧帝**。

### 玄宗与定陵（伯父陵）

**唐玄宗李隆基**——是中宗李显的胞弟睿宗李旦之子；中宗对玄宗是伯父辈。**玄宗即位后追谥中宗"大和大圣大昭孝皇帝"**——名号尊崇。**但实际祭祀**：定陵不及桥陵（玄宗父）、亦不及乾陵（玄宗祖父母）——是中宗政治地位的反映。**玄宗朝定陵岁修按例行**——开元、天宝间未明记玄宗亲祀定陵；具体祭祀规格史载零散。

## 历史长河（755 → 今）

**中晚唐**　**至德 / 乾元间唐军反攻关中**——定陵一带反复易主、陵区岁修中断。**贞元 / 元和间唐廷尝试恢复唐陵岁修**——定陵列入。**中晚唐定陵的祭祀**：因中宗政治地位较低、定陵祭祀分量在诸帝陵中较小。

**唐末黄巢 + 五代**　**广明元年（880）黄巢入长安**——唐陵在乱中受波及；**五代温韬大规模盗发关中唐陵**——《新五代史·温韬传》"唐诸陵在其境内者悉发"、"唯乾陵风雨不可发"；**定陵据此推定亦在被盗之列**（具体细节史载未详）：陪葬器物可能被掠；地表石刻损毁。

**两宋**　**北宋初宋太祖诏修唐陵**——定陵列入岁修；**北宋元丰间宋敏求《长安志》**记定陵规模 + 陪葬墓 8 座。

**金元明清**　金元明清间定陵地表石刻继续损毁——但主峰龙泉山未易；陵冢仍立。**清乾隆四十年（1775）陕西巡抚毕沅立"唐中宗定陵"碑**。

**民国至今**　1949 年后定陵列陕西省重点文物保护单位、2001 年列全国重点。**1990s-2010s 多次考古勘察**——确认陵冢、献殿基址、神道石刻、8 座陪葬墓位置。**今游人来富平定陵**——所见的是龙泉山主峰 + 残存神道石刻（多有损毁）+ 部分陪葬墓。**节愍太子墓**作为最显的陪葬墓——已多次勘察、但未大规模发掘。**唐人在 755 所见的那座"被韦后毒杀皇帝的陵"**——一千二百多年后陵冢仍立；地宫未扰；神道石刻多损但部分仍存——是定陵作为唐六陵中"悲剧帝陵"的物理 anchor。
