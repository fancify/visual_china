# 灞桥

## 一句话

京兆府 万年县 东郊 灞水 之上的一座大桥——**秦汉以来已有桥、王莽始建国元年/9 改名"长存桥"而终复"灞桥"旧名**；Tang 755 时是**长安东出第一渡**——出春明门东三十唐里即至桥、过桥即出关中东向；**唐人折柳送别成例、"灞桥折柳"是中国送别文化的物理 anchor**。

## 形貌

灞桥在**长安城东三十唐里**——出**春明门**（长安东面三门之中门）东向、过浐水、再至灞水东岸——桥即在两河之间或灞水之上（唐代灞水有数渡、主桥即长安东出大道所经者）。

**灞水**——源出秦岭东北麓、流经京兆府境东部、北注**渭水**；上游有蓝田、辋川一带山溪汇入，下游与浐水合流后注渭。**灞水本名"滋水"**——见《水经注·渭水》："滋水……秦穆公更名以章霸功、自是改称霸水。"——秦穆公伐西戎得霸、以此功改"滋"为"霸"；后世"霸"字渐讹作"灞"（加水旁、专为水名）——**唐人通行"灞水"写法**、但史源间或仍作"霸水"、二字相通。

**灞桥之形**——史源未详记 755 时桥的具体形制（木桥 / 石桥 / 桥孔几座皆未明记）；秦汉间灞桥为**木构梁桥**——立木桩为墩、横梁铺板；汉武帝间曾大修；隋唐间形制屡有更易（隋大业年间据传一度改石桥、唐间又屡修）——**Tang 755 时灞桥具体形制本 doc 不预设**；可确定的是：桥宽足容车马并行、是国家级驰道（**两京驰道经长安东出 → 灞桥 → 新丰 → 渭南 → 华阴 → 潼关 → 至洛阳**）必经的第一座大桥。

**灞桥两岸**——东岸为新丰县境（属京兆府）方向、西岸为万年县东郊；**桥东不远即古"灞陵"故址**（汉文帝陵；唐人多指为"凤凰嘴"、详子条目）；**桥北十余里即渭灞合流口**——灞水北注渭、再东行至潼关；**桥南即蓝田道分叉**（南向蓝田、过蓝关、入商州武关方向）。**这是关中东郊最密集的地理交会点**：长安 → 灞桥 → 新丰 → 鸿门 → 戏下 → 骊山温泉宫——一道东向 60 里的故事线。

**桥畔之柳**——**汉代起、灞桥两岸植柳成林**——是国家级驰道的行道树；至唐已 800 余年、林木荫翳；**送别折柳之习自汉代起**——**"折柳"二字同"留"音通**、寓挽留之意；唐人送别至灞桥、必折柳赠别——**这是灞桥成为"送别 anchor"的物理基础**：行道柳林 + 国家级出关大桥 + 长安东出第一渡。

**物候**——关中暖温带季风气候——夏多雨、冬寒多雪、春风扬尘、秋柳早衰；**"风雪灞桥"** 是唐宋以来唐诗咏灞桥的常景之一——寒冬灞桥 + 柳枝萧瑟 + 行人折柳。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

灞桥在唐人的回望里——是**送别地 + 国家级渡桥**，二重身份在长安东郊重叠了千余年。

**先秦至秦汉**——**灞水本名"滋水"、秦穆公（前 7 世纪）以伐戎得霸更名"霸水"**（《水经注》）——这是中国地名史最早一次"以武功改水名"的记录。**秦汉之际灞水东岸地名 anchor 渐多**：**秦王朝末（前 206）刘邦由武关入秦、至灞上**（即灞水西岸坂地、长安东郊）、**秦王子婴素车白马、奉天子玺降于轵道旁**（《史记·高祖本纪》）；**项羽率诸侯兵 40 万至戏下**（戏水之滨、灞水东约 30 里、新丰县境）、**闻刘邦已定关中、怒**——**鸿门宴**即在此一带（戏水之滨、鸿门坂；不在灞桥本身、但近）。**自此"灞上"、"鸿门"、"戏下" 三名连为关中东郊一线**——灞桥居其首、是西汉关中东大门最近的一座桥。

**西汉文帝霸陵（前 157 葬）**——**汉文帝刘恒以灞水东岸为陵**、号"霸陵"。**汉文帝"以山为陵、不起坟茔"** 的简葬理念使霸陵成中国帝陵史的一个特例——**唐人多认为霸陵在"凤凰嘴"** （灞桥东、白鹿原西北角的一处土塬突起）——**但 2017-2021 陕西省考古研究院 江村大墓 多季发掘 + 2021 年 12 月 14 日国家文物局发布会**——**确认 江村大墓**（凤凰嘴南约 2 公里、白鹿原西麓）**才是真正的汉文帝霸陵**——**凤凰嘴是误指**（详子条目）。**Tang 755 时唐人不知此真相**——他们眼中的"灞陵"是凤凰嘴上的那座空台、是凭吊汉文帝简德的怀古地。

**王莽始建国元年/9 改"霸桥"为"长存桥"**——《汉书·王莽传中》记：王莽 9 年改长安诸门、桥名为新莽风格名号、灞桥改名"长存桥"。**这是中国桥史最早一次进入正史的"皇帝改桥名"记录**——但王莽之政 14 年而亡、新莽改名皆复旧；**东汉建武初 灞桥复名**——其后历魏晋南北朝隋唐、桥名定为"灞桥"（或"霸桥"、字异同名）。

**汉代起灞桥折柳送别成例**——**汉人送行至灞桥、折柳赠别**——这是"折柳"民俗的最早 anchor；**至盛唐已 800 余年、习俗未坠**。**唐人凡自长安东出（赴洛阳、赴东都、赴东方、赴塞外）必经灞桥**——**送行者亦多送至灞桥即返**——这是唐代长安东郊最频繁的送别场面。**"灞桥折柳"** 因此成中国送别文化的核心 anchor：**长安 = 出发点；灞桥 = 送别点；柳 = 挽留意象；折柳 = 送别仪式**——四者构成一套完整的文化空间。

**盛唐灞桥诗咏**——王维（701-761）、李白（701-762）、岑参（约 715-770）、王昌龄（约 698-756）等盛唐诗人**皆有送别诗**——**但具体某首诗明确系于灞桥者、史源未必稳**——本 doc 不预设单一"灞桥诗"代表。可确定的是：**送别 + 柳 + 长安东郊** 一线的诗意象、在盛唐已极成熟。**王维《送元二使安西》"渭城朝雨浥轻尘、客舍青青柳色新；劝君更尽一杯酒、西出阳关无故人"** ——**这首诗送行至"渭城"（咸阳郡渭城县、长安西约 30 里、非灞桥）**——**是长安西出 → 西域方向的送别**；但**"客舍青青柳色新"的"柳" 与灞桥折柳同源** —— **盛唐送别意象的两个空间端点：东出灞桥 + 西出渭城**——本 doc 严格区分二者。

### 此刻

天宝十四载这一年的灞桥——**桥在、柳成、习俗盛**。

**桥仍立**——具体形制本 doc 不预设、但桥宽足容车马并行、是两京驰道东出第一桥；**自春明门东三十唐里**——日出后骑行约一时辰即至桥、马车则需一时辰多；**桥上日过千百车马**——朝廷使者、藩镇使节、商旅、士人、举子、僧道——凡自长安东出者必过此桥。

**桥畔柳林依旧**——汉代以来 800 余年的行道柳、唐玄宗朝多有补植；**柳荫连绵自桥东西延数里**——春则绿意盈眼、秋则黄叶飘落、冬则枝条萧瑟。**折柳送别成例**——**自长安出仕、赴任、远行者**——送行者陪行至灞桥、桥畔置酒、折柳赠别、相别而返。**长安人对灞桥的日常感知**——是"出城东 30 里、那座送别的大桥"。

**桥东不远的灞陵**——唐人凭吊汉文帝简德的怀古地、凤凰嘴上的那座空台、岁岁有人登临；**唐人怀古意象中"灞陵"与"灞桥"几乎并称**——李白（传作）《忆秦娥》："**年年柳色、灞陵伤别**"——"灞陵" 在此与"灞桥" 通指、皆送别地（**本 doc 严格注：此词传作李白、学界自宋以来即多疑非李白真作；不作 755 此刻 verified 史源**——详子条目）。

**桥北十余里**——渭灞合流口、再东即两京驰道主干；**桥南分蓝田道**——南向蓝田、过蓝关、入商州方向。**整个灞桥地段**——是关中东南两向出关的物理交会。

**755 时唐廷对灞桥的常规事务**——驰道驿道维护（属万年县衙 + 京兆府）；桥本身的修缮（朝廷工部辖）；桥两岸的禁伐令（行道柳林为国家驰道树、私自伐取者罚）；桥畔送别活动无任何禁约——是民间日常文化活动。

**755 末安史之变前夕**——**桥与朝廷无任何异常**；**十一月安禄山起兵范阳**、十二月叛军南下、陷洛阳；**756 六月叛军西犯**——**潼关失守**（哥舒翰兵败）——**玄宗六月十三日西奔**——**自长安西门出、经马嵬驿（扶风郡兴平县东）、马嵬驿之变 → 入蜀**——**玄宗西奔不经灞桥**（灞桥是长安东出方向、玄宗向西逃蜀）。**但 756 六月叛军入长安、灞桥成战略要冲**——**756 七月安禄山部将孙孝哲入长安、屯兵灞桥东**——灞桥成叛军东西联络的咽喉；**757 九月唐军郭子仪 + 回纥兵东出潼关复长安**——**亦经灞桥而入**——灞桥见证安史之乱中长安的两次易手。这些是 755 之后的事——本 doc 仅作"未来视角"提及、严守 755 此刻时态。

### 此刻意味

对天宝十四载的万年县人来说——灞桥是**家门口的大桥**：每日车马不绝、桥畔有人折柳、河水东去；柳荫如盖、夏日凉爽；春日柳絮飞如雪、是关中最有名的春景之一。对长安城里的士人来说——灞桥是**送别的物理 anchor**：朋友赴任、举子下第回乡、官员贬谪东出、僧道云游——皆送至灞桥、折柳为别；归来时亦在灞桥重逢、桥上桥下、人事代谢。对朝廷来说——灞桥是**长安东出第一桥**：两京驰道的咽喉、关中东冲的物理 anchor、工部岁岁修缮的国家级桥。

唐人尚不知道——**半年之后**（756 六月）**潼关失守**、**叛军入长安**、**灞桥成战略要冲**；**两年之后**（757 九月）**唐军复长安、亦经灞桥而入**——这座桥将在安史之乱中见证长安的两次易手、城东郊柳林或被砍伐为军用、桥栏或为兵燹所损。唐人更不知道——**1100 余年之后**（清末民国）灞桥已数次重建、形制屡变；**1949 年后多次修桥、今西安东郊灞桥区有数代灞桥并存**（明清桥基 + 民国桥 + 现代公路桥）——**唐人所过的那座灞桥早已不存**、但**"灞桥"二字仍在**、桥址相去不远——是中国地名史中千年延续最完整的一例。**"灞桥折柳"四字** 穿越千载、仍是中国送别文化的最深 anchor——比桥本身、比柳本身、比唐人的具体送别更耐久。

## 子条目

### 灞桥 vs 灞陵——两个相邻、易混的 anchor

**灞桥**——**长安东出大桥**、跨灞水、自春明门东三十唐里；**灞陵**——**汉文帝刘恒陵**（前 157 葬）、在灞水东岸、距灞桥东数里。**两者地名相邻、易混**：

- **灞桥**：万年县东郊、灞水之上的国家级桥、送别 anchor。
- **灞陵**：万年县东 + 新丰县境交界、白鹿原西北角的一座汉帝陵、怀古 anchor。

**唐人多以"灞陵"指代灞桥附近送别地**——**李白（传作）《忆秦娥》"年年柳色、灞陵伤别"** 中的"灞陵"实指**灞桥送别地**（与汉文帝陵无关）——这是诗中的地理通指、不是史地学意义上的灞陵。**本 doc 严格分**：作地名实指时、灞桥 = 桥、灞陵 = 文帝陵；作诗意象时、二者通用、皆送别地。

### 灞陵真位置：凤凰嘴 vs 江村大墓（2021 考古确认）

**汉文帝霸陵**——**前 157 葬于"灞水东岸"**（《史记·孝文本纪》）——**具体位置自唐以来即有争议**。

**唐人通行说**：**凤凰嘴说**——白鹿原西北角的一处土塬突起、自唐元和年间起即被指为霸陵；**Tang 755 时唐人凭吊霸陵、登的就是凤凰嘴**；元和八年（813）李吉甫《元和郡县图志》卷一·京兆府·万年县记霸陵"在县东二十五里"——具体方位即指凤凰嘴一带。

**江村大墓发掘**——**2017-2021 年陕西省考古研究院 + 西安市文物保护考古研究院**多季发掘**江村大墓**（凤凰嘴南约 2 公里、白鹿原西麓、距灞桥东约 5 公里）——出土"凶宫司丞"等大量陶官印 + 帝陵规格的椁室结构 + 与汉景帝阳陵相符的帝陵建制；**2021 年 12 月 14 日国家文物局发布会**——**正式确认江村大墓即汉文帝霸陵**——凤凰嘴只是一处自然土塬突起、并非陵墓。

**Tang 755 时唐人所登的"灞陵"——是凤凰嘴误指**——但唐人不知；他们在那座空土塬上凭吊汉文帝简德、与真实陵墓相距 2 公里——是中国陵墓史一个延续千余年的地理误指。本 doc 严格区分：**唐人视野中的灞陵 = 凤凰嘴**（误）；**真灞陵 = 江村大墓**（2021 考古确认）——**755 时唐人不知**。

### 灞水 = 古滋水 + 秦穆公更名

**灞水本名"滋水"**——见《水经注·渭水下》："滋水……秦穆公更名以章霸功、自是改称霸水。"——**秦穆公（前 7 世纪、春秋秦君）伐西戎得霸**、以此武功更"滋"为"霸"。**这是中国地名史最早一次"以武功改水名"的明确记录**——比"渭水"（古名渭、未改）、"洛水"（古名洛、未改）等都更晚得名。

**"霸"字加水旁作"灞"**——是后世为专表水名而加偏旁的字形演变；**汉魏间二字通用**；唐代通行"灞水"——但史源间或仍作"霸水"。**桥名亦同**：唐前期史志多作"霸桥"、唐后期 + 宋代渐定"灞桥"——二字本同一名。

### 折柳送别：习俗源流

**"折柳"二字**——**"柳"与"留"古音通**、寓挽留之意。**折柳赠别**——**自汉代起即有**——汉乐府《折杨柳》是最早的折柳送别诗题；**至唐已 800 余年、习俗成熟**。**唐人折柳的具体仪式**：送行者陪行至灞桥、桥畔置酒（送行酒）、攀桥畔柳枝、折一节柳条赠行者、相别而返——行者持柳东去、至下一驿或途中弃之。**这一套仪式在盛唐已极成熟**——**长安城东郊的灞桥、是中国"折柳送别"民俗的物理 anchor**。

**与"折柳"并行的送别诗题**——**《阳关三叠》（王维《送元二使安西》衍曲；中唐成型、755 时尚未成"三叠"乐曲、仅是王维诗）**、**《折杨柳》（汉乐府古题、唐人多有拟作）**、**《杨柳枝词》（白居易等唐人新创、中唐后流行）** —— 这些都是与"灞桥折柳"同源的送别音乐与诗题。

### 灞桥 vs 渭城：长安东西两端送别

**长安出仕、远行者**——分**东出**与**西出**两路：

- **东出**：自春明门 → 灞桥 → 新丰 → 渭南 → 华阴 → 潼关 → 至洛阳、东都、东方、东南——**送别 anchor = 灞桥**；折柳习俗、灞桥诗、灞陵伤别皆系于此。
- **西出**：自金光门 → 渭城（咸阳郡渭城县）→ 武功 → 凤翔 → 陇州 → 大震关 / 兰州 → 西域；**送别 anchor = 渭城**（咸阳郡渭城县、长安西约 30 里、渭水南岸）；**王维《送元二使安西》"渭城朝雨浥轻尘、客舍青青柳色新"** ——是西出送别的代表诗。

**两套送别空间**——**东出灞桥 + 西出渭城**——是盛唐长安最完整的"出发／送别地理"。**柳意象贯通二者**——灞桥折柳 + 渭城柳色——是长安城东西郊柳林共有的春景。**本 doc 严格区分二者**：王维《送元二使安西》系**渭城**、不系灞桥；但"柳"的意象同源。

### 李白《忆秦娥》"灞陵伤别"——传作存疑

**《忆秦娥》"箫声咽、秦娥梦断秦楼月。秦楼月、年年柳色、灞陵伤别。"** ——传世题作李白；**但学界自宋以来即多疑非李白真作**：

- **早期编集未收**——李白文集（如宋蜀刻本《李太白文集》）未收此词；
- **始见于宋《草堂诗余》**——南宋后期始系李白名下；
- **词体形制**——《忆秦娥》词牌定型在中唐至晚唐间（约 8-9 世纪）、李白（701-762）盛唐时期词体尚未成熟、李白现存"词"皆有争议；
- **明清以下争议未息**——胡仔《苕溪渔隐丛话》、王世贞、王国维等历代学者多有怀疑。

**本 doc 取**："**传作李白《忆秦娥》、学界存疑**"——**不作 755 此刻 verified 史源**；**但其中"灞陵伤别"四字** 已成中国送别文化的固定意象、**自宋以下千年传诵**——**这是文化 anchor 的一种形态**：**作品本身的真伪退居其次、意象与文化记忆压倒考据**。**Tang 755 时此词尚未出现**（无论真伪）——**严守 755 时态、本 doc 不作此刻引证**。

### "灞桥风雪" / "灞桥诗思"——后世意象的早期种子

**"灞桥风雪"** —— 寒冬灞桥 + 柳枝萧瑟 + 行人折柳 + 雪覆桥栏——是中国诗中最有名的"凄美送别"意象。**这一意象的成熟在宋元**——**苏轼《和林子中待制》"何如此地学长安、 灞桥风雪 行人多"**（北宋）、**元曲多以"灞桥风雪"喻人生苦寒**——**Tang 755 时此意象尚未成熟**；**但唐人灞桥送别诗中已有"寒夜灞桥 + 柳枝萧瑟" 的早期素材**——是后世"灞桥风雪"的种子。

**"灞桥诗思"** —— 宋代郑綮（一作其他人、史源歧异）问"近有诗否"、答曰："**诗思在灞桥风雪中驴子上**"——这是宋代以来定型的"灞桥 = 诗思 anchor" 之说；**唐 755 时此典故尚未成**——但其精神 anchor 已在唐人送别诗中初具：**寒冷 + 离别 + 长路 + 柳枝**——四元素的盛唐组合、为后世意象 anchor 提供素材。

## 历史长河（755 → 今）

**中晚唐**　**756 六月潼关失守 → 玄宗西奔（不经灞桥、向西入蜀）**；**756 七月安禄山部将孙孝哲入长安、屯兵灞桥东**——灞桥成叛军东西联络咽喉；**757 九月唐军郭子仪 + 回纥兵东出潼关复长安、亦经灞桥而入**——灞桥见证安史之乱中长安的两次易手。**中晚唐灞桥送别诗咏渐密**——**白居易（772-846）《赋得古原草送别》"又送王孙去、萋萋满别情"**（虽未明系于灞桥、但唐人送别的共同语境）；**李益（748-829）**、**许浑（约 791-约 858）**、**李商隐（约 813-约 858）**等中晚唐诗人皆有送别诗咏柳 + 长安东郊、灞桥意象渐凝。**唐末黄巢之乱（880s）** —— 黄巢军入长安、长安残破、灞桥或受兵燹之损——具体史源未详；但桥址未弃、屡修屡过。

**两宋**　**北宋长安降为永兴军路京兆府**（一度作陕西路），**地位远不及唐都**；**但灞桥仍存**——是关中东郊大桥。**北宋灞桥诗咏继唐而盛**——**苏轼 + 黄庭坚 + 陆游**等皆有灞桥诗；**"灞桥风雪"意象在北宋成熟**——**苏轼《和林子中待制》"灞桥风雪行人多"**（北宋）；**郑綮"诗思在灞桥风雪中驴子上"** 之典亦在此期定型。**金人占长安后**（1127 后）**灞桥仍在金境**——**金代屡修桥**；**南宋与金以淮河—大散关为界、灞桥在金境**、远离南宋视野。

**金元明清**　**元代灞桥屡修**——**元至元年间（13 世纪后期）灞桥曾大修、改桥为石桥**（一说至元间元世祖敕修；具体年份史源略异）——**自此灞桥由木桥改石桥的形制定型**。**明清间灞桥仍是关中东出大桥、屡有重修**——**明嘉靖** + **清雍正** + **清光绪** 多次修桥；**清代灞桥形制**：石桥多孔、桥宽足容车马并行——**这是清末民国所见的灞桥**。**明清灞桥诗咏不绝**——但"灞桥"已渐成**怀古意象**多于实景描写、桥本身的物理意义在明清不复唐时之重——**长安东出的国家级地位已让位于其他驿路 + 漕道**。

**民国至今**　**民国间灞桥屡修**——**1949 年后多次修桥**——**1957 年灞桥水泥公路桥建成**（古石桥旁新立）、**1994 年灞桥大桥再次重修**——**2003-2004 年灞桥镇 + 灞桥区考古发掘**——**清代灞桥桥基出土**（古桥基址在现代公路桥南侧 + 灞河中段、长约 400 米、多孔石墩、是明清桥的物理存证）；**唐宋灞桥桥基尚未发掘出土**——但桥址相去不远（灞桥镇灞河两岸）。**今西安市灞桥区**——**有数代灞桥并存**：清代古桥基（水下 / 河滩残存）+ 民国旧桥 + 1957 年水泥公路桥 + 现代立交桥；**"灞桥"二字仍在**——是中国地名史中千年延续最完整的一例。**唐人所过的那座灞桥早已不存**——但**"灞桥折柳"四字** 穿越千载、仍是中国送别文化的最深 anchor。**今西安东郊灞桥区**——**有灞桥遗址公园、桥畔仍植柳成林**（虽是 20 世纪末重栽、非汉唐旧林）——**这是文化 anchor 在物理 anchor 终结千年后的一次现代续接**。**自秦穆公更名"霸水" 至今 2700 年、自汉代灞桥折柳至今 2000 年、自盛唐灞桥折柳意象至今 1300 年** —— **灞桥与折柳的意象 anchor、比任何一座具体的桥活得都更久**。
