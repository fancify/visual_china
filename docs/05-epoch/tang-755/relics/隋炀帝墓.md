# 隋炀帝墓

## 一句话

**隋亡帝身处所**——大业十四年（618）三月炀帝被弑于江都宫、萧后草葬流珠堂；**唐武德五年（622）改葬江都西吴公台下、贞观元年（627）说改葬雷塘**——**755 时炀帝陵已在江都北郊雷塘约 130 年**；唐淮南道广陵郡江都县北十里、湖塘草冢一座，是唐人凭吊隋亡的物理 anchor。

## 形貌

隋炀帝墓在唐淮南道**广陵郡江都县北郊雷塘**——具体在**江都县城北约十里的雷塘北岸**（今江苏省扬州市邗江区槐泗镇雷塘）。**地势**：江淮平原中段、长江北岸冲积平原；雷塘是江都城北的一片低洼湖塘、与北蜀岗丘陵相邻——海拔不及 10 米；**陵处雷塘北岸的一片高地**——是江都郊外农田与水网混合的低丘地带。

**陵冢形制**：唐贞观元年改葬时所筑——**封土为坟**（非依山为陵；隋唐帝陵礼制并行）；底边约 30-50 米、高约 5-8 米——**比隋文帝泰陵（陕西杨陵）小一截、亦不及唐昭陵 / 乾陵规模**。**改葬礼制相对简朴**——以亡国君主之礼降等而葬、神道石刻不密集、陵前未立大型石翁仲 / 石马。**陵区范围约 200 米见方**——围以低矮土垣 + 数株柏树；陵令所驻屋数间。

**雷塘的地形**：江都北郊一片大湖塘、东西约 3 里、南北约 2 里——湖塘内水草丛生、芦苇蔓延；**雷塘北岸是隋炀帝陵所在**；湖塘南岸再向南即江都罗城北门。**雷塘水**：源自蜀岗北麓——经雷塘汇入江都北郊水网、再东入运河（邗沟段）。**唐人来雷塘**：自江都北门出、行十里至雷塘北岸——可见炀帝陵土堆 + 雷塘水波 + 远处蜀岗。

**与吴公台 / 流珠堂的关系**：
- **流珠堂**：江都宫西院的一处堂室——大业十四年（618）萧后草葬炀帝于此；位置在江都宫内（今扬州市子城遗址区）。
- **吴公台**：江都西南郊的一处南朝旧台（陈代吴明彻所筑、故名"吴公台"）——武德五年（622）唐高祖诏迁炀帝葬此；位置距江都城约 5 里。
- **雷塘**：江都北郊——贞观元年（627）说太宗再改葬；本 doc 主写之地。**三处葬地之变** 是炀帝身后最戏剧的故事之一。

物候：江淮平原、亚热带季风气候——夏热多雨、冬温湿润；雷塘水域常年不涸、芦苇丛生、水鸟栖息——是江都郊外最有"湖塘 + 废陵"意境的一处野地。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

隋炀帝墓在唐人的回望里——是**隋亡的物理 anchor + "通济渠 + 三征高丽 + 江都之弑" 整段隋史悲剧的最终凝结**。

**隋炀帝杨广（569-618）**——**隋文帝杨坚次子**、**仁寿四年（604）继位**——年三十六；年号"大业"（605-618）。**炀帝即位的功业**：
- **大业元年（605）开通济渠**——洛阳至淮安、贯通黄河 + 淮水；
- **大业四年（608）开永济渠**——黄河至涿郡（今北京西南）；
- **大业六年（610）开江南河**——江都至余杭；**三段运河合成隋唐大运河四千余里**——是中国古代最大单项土木工程。
- **大业三年（607）始建东都洛阳**——隋洛阳城（含含嘉仓 + 含嘉门 + 应天门 + 永通桥）；
- **大业三年（607）颁《大业律》** + 完善科举（开"进士科"——一说炀帝时正式立、一说沿文帝旧制）。

**炀帝的滥用国力**：
- **大业八年（612）+ 大业九年（613）+ 大业十年（614）三征高丽**——三次大举出兵辽东、皆败；耗损民力约二百万；
- **大业元年（605）+ 大业六年（610）+ 大业十二年（616）三次南巡江都**——龙舟万艘、随从十余万、沿运河南下；**第三次南巡（大业十二年/616）是炀帝最后一次离开洛阳**——再未北还。
- **修长城 + 修长安 + 修洛阳 + 修西苑** 等大型工程并发——民力穷竭。

**大业末年的天下大乱**：**大业七年（611）起山东王薄起兵**——隋末农民大起义自此起。**大业十二年（616）翟让 + 李密瓦岗军起 + 窦建德河北起兵 + 杜伏威江淮起兵 + 萧铣江陵自立 + 李渊太原起兵**（大业十三年/617）——天下分崩。**炀帝困守江都**：大业十二年（616）三幸江都后、留居江都未还洛阳——朝中权臣宇文化及 + 司马德戡等酝酿政变。

**江都之变（大业十四年/618 三月）**：**大业十四年三月十一日（公历 618 年 4 月 11 日）**——**宇文化及 + 司马德戡率骁果军于江都宫发动政变**；**炀帝被令狐行达缢死**——年五十；**萧皇后与宫人撤漆床板为棺、殡于西院流珠堂**（《隋书·炀帝纪下》原文）。**同年五月唐高祖李渊于长安即位**——大业年号止、隋亡。

**炀帝身后的三次葬地之变**：
1. **大业十四年（618）三月草葬流珠堂**——萧后于江都宫西院草葬；
2. **唐武德五年（622）八月**——唐高祖李渊诏改葬隋炀帝（《旧唐书·高祖纪》原文："改葬隋炀帝于雷塘"）；但**学界多认为此次改葬地实际为江都西南郊吴公台下**（南朝陈吴明彻所筑旧台）——"雷塘"系唐人书写中对江都北郊一带的概称、与贞观元年再改葬之"雷塘"非同一具体葬址；这是第二次葬地；
3. **唐贞观元年（627）**——一说唐太宗李世民诏再改葬于**江都北郊雷塘**；这是第三次葬地。**第三次改葬的史源主要在《旧唐书·太宗纪》系前朝帝陵礼制整顿诏 + 《元和郡县图志》等唐宋地志（"江都县北十里、隋炀帝陵在焉"）；具体改葬月份史源略异**——本 doc 取贞观元年（627）说。

**萧皇后（567 前后-647）**——**梁明帝萧岿之女**、**隋炀帝皇后**；江都之变后被宇文化及挟北逃、后辗转入窦建德营 + 突厥；**贞观四年（630）李靖灭东突厥**——萧后被迎归长安；**贞观二十一年（647）卒、年约八十一**；**与炀帝合葬于雷塘**（按《北史》系记载——本 doc 取之）。

### 此刻

天宝十四载这一年的隋炀帝墓——**雷塘陵已立约 128 年（贞观元年/627 改葬雷塘 → 755 = 128 年）**；以**前朝亡国君主帝陵 + 雷塘湖畔废陵**身份存在。

**陵冢仍在雷塘北岸**——封土主体犹存、围墙土垣多已坍；**陵令所**——唐廷按"前朝帝陵令"制设、岁有少量看护吏员、香火按官定规模岁奠两次（清明 + 冬至）。**唐廷对炀帝陵的态度**：（1）按"前朝鉴" 礼制低规格维护；（2）唐玄宗本人未巡江都——故未亲祭炀帝陵；（3）天宝中盛唐对炀帝的评价：在《贞观政要》《隋书》（贞观间唐人魏徵等编撰、贞观十年/636 修成）等官修史中已定性为"亡国之君 + 滥用民力" 典型——但同时承认通济渠 + 大业律 + 隋唐大运河的功业。

**唐人对隋炀帝陵的认知**——主要通过：
1. **史书记载**：《隋书》（贞观十年/636 唐人魏徵等编撰）+ 《北史》（唐李延寿撰、唐高宗显庆四年/659 成书）——盛唐已成书 + 广流；
2. **物理凭吊**：盛唐文人出江都北行、必过雷塘——可见炀帝陵土堆 + 雷塘水波；
3. **唐诗**：盛唐 + 中晚唐多有"雷塘"凭吊诗——李商隐、罗隐、皮日休 等多有作（多在 755 后）；盛唐前期对雷塘的具体诗咏史源略零散。

**这一年的雷塘陵** 处于**前朝废陵 + 唐人凭吊 + 江都郊野湖塘** 的稳定状态——再过半年（天宝十四载十一月）安禄山起兵于范阳；**至德元载（756）唐廷迁江淮漕路绕道东南**——扬州地位反而上升；**乾元二年（759）刘展之乱起 + 上元元年（760）刘展兵入扬州 + 宝应元年（762）田神功平叛大军入扬州纵兵掳掠**——雷塘陵区是否有波及、史载未详。

### 此刻意味

对天宝十四载的江都人来说——隋炀帝陵就是**自家城北十里的湖塘废陵**：雷塘水波、芦苇丛生、土堆一座——城里小儿都能指得出方向。对长安那一代士人来说——雷塘是**隋亡的物理纪念**：通济渠 + 永济渠 + 江南河 + 大业律 + 含嘉仓 + 三征高丽 + 龙舟南巡 + 江都之弑——整段大业末年悲剧的最终凝结地。**唐人凡过江都北行者多有怀古**——盛唐文人在雷塘前的"前朝鉴" 情绪、是盛唐到中晚唐凭吊文学的延续主题。

**唐皇室对炀帝陵的态度**——既"前朝鉴"（《贞观政要》引太宗"以隋为鉴"屡作戒）、又有距离感（唐玄宗未亲祭、改葬规格降等）。**炀帝陵作为"亡国君主葬地" 的典型**——与汉武茂陵、唐昭陵、乾陵的"盛朝帝陵" 形成强烈对比；唐人在雷塘前所感的"兴亡之念"——是中国陵墓文学"前朝鉴" 系最重要的一处实景。

唐人尚不知道——再过 150 年（约 9 世纪末），**唐末黄巢之乱（880-884）+ 五代杨吴 / 南唐据江都**——雷塘陵在唐末五代乱中是否有盗扰、史载零散。再过千余年（清嘉庆十二年/1807），**清扬州知府伊秉绶在雷塘西重立"隋炀帝陵"碑**——但伊秉绶所定位置实是清代追认、未必是贞观改葬原址。再过两百年（2013 年），**南京博物院 + 扬州市考古研究所发掘扬州曹庄隋炀帝墓 + 萧皇后墓**——**出土十三环金玉带、鎏金铜铺首、墓志砖（明确"隨故煬帝墓誌")**——这才是炀帝真陵；**与传统雷塘"隋炀帝陵"非同地**。**唐人在 755 所见的那座雷塘湖畔土堆**——按传统雷塘说仍是凭吊地；但 2013 后学界对"真陵在曹庄 vs 传统雷塘陵" 已有新认识——这是后人之事。**唐 755 时雷塘陵确按贞观元年改葬之地存在**——是这段三葬故事中唐人所知的最后一站。

## 子条目

### 流珠堂（江都宫内、618 草葬第一站）

**江都宫西院的一处堂室**——大业十四年（618）三月十一日炀帝被弑后、**萧皇后与宫人撤漆床板为棺、殡于西院流珠堂**（《隋书·炀帝纪下》原文）。**位置**：江都宫内（今扬州市子城遗址区）。**草葬礼制极简**——以漆床板为棺、未筑陵冢、未立神道——是亡国君主初葬最低规格。

**流珠堂作为炀帝身后第一站葬地**——唐武德五年（622）迁葬吴公台前在此停灵 4 年；**唐时流珠堂早已不存**——江都宫在隋末乱中已废、流珠堂遗址在 755 时已是江都子城遗址区的一片荒地。**这是炀帝"三葬之始"** —— 唐人凭吊雷塘时所追忆的"江都之变" 起点。

### 吴公台下（622 改葬第二站）

**江都西南郊的一处南朝旧台**——**陈代吴明彻所筑、故名"吴公台"**（陈太建年间，约 570s）。**位置**：江都城西南约 5 里（今扬州市西郊蜀岗西端）。**唐武德五年（622）八月**——唐高祖李渊**诏改葬隋炀帝**（《旧唐书·高祖纪》原文："改葬隋炀帝于雷塘"）——但**学界多认为此次改葬地实际为吴公台下**（"雷塘"系唐人书写概称、与贞观再改葬之具体雷塘葬址非同地）；故本 doc 采吴公台中转说，即 **622 改葬吴公台 + 627 太宗再改葬雷塘** 两段说。

**武德五年改葬的政治背景**：（1）唐立国后第三年 + 唐廷为前朝君主整顿陵制；（2）江都已归唐 + 杜伏威平定江淮；（3）唐高祖按"前朝帝礼" 降等改葬——以亡国之君之礼。**改葬规模相对简朴**——封土为坟、未立大型神道石刻。

**吴公台作为炀帝第二站葬地**——存续约 5 年（622-627）；**唐贞观元年（627）再改葬雷塘**——吴公台陵址废。**唐时吴公台下的炀帝旧穴**——755 时已废 128 年、地表未必有标识；唐人凭吊江都西郊时偶有提及"吴公台" 二字——多作为"南朝陈代旧台" 史地参考。

### 雷塘（627 改葬第三站、755 时所在）

**江都北郊一片大湖塘**——**雷塘水自蜀岗北麓 + 经雷塘汇入江都北郊水网、再东入运河（邗沟段）**；东西约 3 里、南北约 2 里。**雷塘北岸**——**唐贞观元年（627）改葬隋炀帝于此**；是炀帝身后第三站葬地、亦是 755 时所在。

**唐贞观改葬的政治背景**：（1）贞观元年是唐太宗即位首年；（2）太宗以"以隋为鉴" 屡作戒于贞观群臣（《贞观政要》系）——改葬炀帝至雷塘是"前朝鉴" 礼制的物理表达；（3）萧后此时仍在长安（贞观二十一年/647 卒后亦合葬雷塘）——按礼制夫妻合葬之制完成。**改葬月份史源略异**——本 doc 取贞观元年（627）说、未细订月份。

**雷塘陵的物理形貌**：封土为坟、底边约 30-50 米、高约 5-8 米——比隋文帝泰陵（陕西杨陵）小一截。**陵区周围**：低矮土垣 + 数株柏树 + 陵令所驻屋数间 + 雷塘水波。**陵前**：未立大型神道石刻——按亡国君主降等礼制。**唐廷对雷塘陵的维护**：按"前朝帝陵令" 制岁奠两次（清明 + 冬至）——盛唐 755 时仍按此制。

**雷塘的湖塘景观**：芦苇丛生 + 水鸟栖息 + 远处蜀岗 + 江都罗城北门遥望——是江都郊外最有"前朝废陵" 意境的一处野地。**唐人凭吊雷塘**——多自江都北门出、行十里至雷塘北岸——是盛唐到中晚唐凭吊文学的高频地。

### 隋炀帝杨广（569-618）—— 通济渠 + 三征高丽 + 江都之弑

**隋炀帝杨广（569-618）**——**隋文帝杨坚次子**、**仁寿四年（604）继位**——年三十六；年号"大业"（605-618）。**炀帝在位 14 年**：

**功业**：
- **大业元年（605）开通济渠**——洛阳至淮安、贯通黄河 + 淮水（征丁百余万）；
- **大业四年（608）开永济渠**——黄河至涿郡（征丁百余万）；
- **大业六年（610）开江南河**——江都至余杭；**三段运河合成隋唐大运河四千余里**；
- **大业三年（607）始建东都洛阳**——隋洛阳城（含含嘉仓 + 应天门 + 永通桥）；
- **大业三年（607）颁《大业律》** + 完善科举（"进士科" 制度化—一说炀帝时正式立）。

**滥用国力**：
- **大业八年（612）+ 大业九年（613）+ 大业十年（614）三征高丽**——三次大举出兵辽东、皆败；耗损民力约二百万；
- **大业元年（605）+ 大业六年（610）+ 大业十二年（616）三次南巡江都**——龙舟万艘、随从十余万；
- **修长城 + 修长安 + 修洛阳 + 修西苑** 等大型工程并发。

**江都之弑（大业十四年/618 三月十一日）**：**宇文化及 + 司马德戡率骁果军于江都宫发动政变**——**炀帝被令狐行达缢死**、年五十；**萧后草葬流珠堂**——隋亡。

**唐人对炀帝的史评**：在《隋书》（贞观十年/636 修成）+ 《北史》（唐高宗显庆四年/659 成书）+ 《贞观政要》（盛唐间集太宗贞观年间君臣言论）等盛唐前官修文献中已定性为"亡国之君 + 滥用民力" 典型——但同时承认通济渠 + 大业律 + 大运河的功业。**盛唐到中晚唐对炀帝的认知** 是矛盾的——既鞭挞其"无道"、又承认其"创业" 的物理遗产（运河 + 大业律 + 含嘉仓）至唐而广用。

### 萧皇后（567 前后-647）—— 江都之变 + 北逃突厥 + 归唐 + 合葬雷塘

**萧皇后**——**梁明帝萧岿之女**、**约 567 年生**（《北史》系）；**开皇二年（582）十六岁嫁晋王杨广为妃**；**大业元年（605）立为皇后**。**江都之变（618）后**：被宇文化及挟北逃；**化及败 + 萧后入窦建德营**（618-621）；**620 萧后又转入突厥处罗可汗营**——其侄孙女已嫁突厥可汗、故被迎；**贞观四年（630）李靖灭东突厥**——萧后被迎归长安；**贞观二十一年（647）卒于长安、年约八十一**——**与炀帝合葬于雷塘**（按《北史》系记载）。

**萧后的特殊地位**：（1）经历四朝（梁 + 隋 + 唐 + 突厥）——是中古史中最长寿且经历最戏剧的女性之一；（2）入唐归长安后、唐廷待之以前朝皇后之礼——居西内（太极宫）之后宫；（3）合葬雷塘是按夫妻合葬之制——但具体合葬墓的物理位置在 755 时唐人所知的雷塘陵；2013 曹庄发掘出土萧后墓与炀帝墓相距甚近、为现代考古确认。

### 三葬之变（618 流珠堂 → 622 吴公台 → 627 雷塘）

**炀帝身后三次葬地之变**——是中国陵墓史最戏剧的故事之一：

| 次第 | 年份 | 葬地 | 主导 | 礼制 |
| --- | --- | --- | --- | --- |
| 一 | 大业十四年（618）三月 | 江都宫西院**流珠堂** | 萧皇后 + 宫人 | 漆床板为棺、草葬 |
| 二 | 唐武德五年（622）八月 | 江都西南郊**吴公台下** | 唐高祖李渊 | 改葬、降等帝礼 |
| 三 | 唐贞观元年（627） | 江都北郊**雷塘** | 唐太宗李世民（一说） | 再改葬、合葬萧后处（647 后） |

**第一次葬（流珠堂）**：江都之变后即日萧后所行——草葬以漆床板为棺、是亡国君主初葬最低规格。**第二次葬（吴公台下）**：唐高祖改葬以"前朝帝礼" 降等——是唐廷整顿前朝陵制的一笔。**第三次葬（雷塘）**：太宗贞观元年所行——以"以隋为鉴" 政治叙事 + 完成前朝帝陵礼制；亦是萧后合葬之地（647 后）。

**史料异文**：《旧唐书·高祖纪》武德五年条原文作"改葬隋炀帝于雷塘"——但学界多认为此次改葬地实际为吴公台（参《隋书·炀帝纪》《北史》+ 《元和郡县图志》系唐宋地志考据 + 《旧唐书·太宗纪》系贞观帝陵礼制整顿条）、贞观元年太宗再改葬雷塘——本 doc 采"武德五年葬吴公台 + 贞观元年改葬雷塘"两段说。

### 江都之变（大业十四年/618 三月）

**大业十四年（618）三月十一日**——**宇文化及 + 司马德戡 + 裴虔通 + 元礼 + 唐奉义 等率骁果军于江都宫发动政变**：

**政变起因**：（1）大业十二年（616）炀帝三幸江都后久留未还、骁果军（炀帝直属禁卫军）思归关中；（2）天下大乱 + 关中已归李渊（大业十三年/617）；（3）骁果军郁愤起、宇文化及等借机发动政变。

**政变经过**：三月十一日凌晨、骁果军入江都宫；炀帝被裴虔通 + 司马德戡等围于寝殿——**炀帝令缢于身、令狐行达执之**（《隋书·炀帝纪下》原文）——年五十。**同时被弑者**：齐王杨暕（炀帝次子）+ 赵王杨杲 + 宇文皛 等。**萧皇后与宫人撤漆床板为棺、殡于西院流珠堂**——是为草葬。

**政变后**：**宇文化及自立为大丞相**——挟萧后 + 余宗室北上、企图回关中；中途遇李密瓦岗军 + 化及败走魏县（今河北大名）；**武德二年（619）宇文化及被窦建德所杀**——隋宗室与化及一并诛灭。

**江都之变作为"隋亡" 的最终事件**——是大业末年大乱的终点 + 唐立国的转折；**唐人对江都之变的记忆**：（1）在《隋书》《北史》等官修史中详载；（2）盛唐文人凭吊雷塘陵时所追忆的核心情节；（3）"江都之变" 与"通济渠" + "三征高丽" 合为唐人"前朝鉴" 系叙事的三大主轴。

### 大运河 + 通济渠 + 永济渠 + 江南河（炀帝功业 / 唐廷漕命）

**隋唐大运河四段**（炀帝在位时完成三段、唐贞观间补疏）：
- **通济渠**（大业元年/605）：洛阳—淮安、长约 1,300 公里；
- **永济渠**（大业四年/608）：黄河—涿郡（今北京西南）、长约 1,000 公里；
- **江南河**（大业六年/610）：江都—余杭、长约 800 公里；
- **邗沟**（春秋吴王夫差始凿、大业元年/605 炀帝整修）：江都—淮安、长约 150 公里。

**四段合成隋唐大运河**——四千余里、贯通海河 + 黄河 + 淮河 + 长江 + 钱塘江五大水系。**炀帝开运河的政治目的**：（1）军事——便于南北调兵（永济渠通辽东、为征高丽用）；（2）漕运——江南漕米北运洛阳与长安；（3）巡幸——炀帝三幸江都皆走通济渠 + 邗沟 + 江南河南下。

**炀帝运河至唐而广用**——盛唐天宝间 **江淮漕米年约二百万石经通济渠 + 邗沟 + 渭水入长安**（《通典·食货十二》）；**没有大业三段运河、没有盛唐长安洛阳两京的供给**。这是炀帝"亡国之君兼运河之祖" 的最大政治悖论——唐人在雷塘前所感的"前朝鉴" 情绪里、必然含此一层。

## 历史长河（755 → 今）

**中晚唐**　**至德元载（756）唐廷迁江淮漕路绕道东南**——扬州地位反而上升；**乾元二年（759）刘展之乱起 + 上元元年（760）刘展兵入扬州 + 宝应元年（762）田神功平叛大军入扬州纵兵掳掠**——雷塘陵区是否有波及、史载未详。**中晚唐文人凭吊雷塘**：李商隐（813-858）有《隋宫》"乘兴南游不戒严、九重谁省谏书函"——咏炀帝南巡 + 雷塘之亡；罗隐（833-910）有《炀帝陵》"入郭登桥出郭船、红楼日日柳年年"；皮日休（约 838-883）有《汴河怀古》——咏通济渠 + 炀帝功罪。**中晚唐到唐末的雷塘陵** 是凭吊文学高频地。

**唐末五代**　**广明元年（880）黄巢入长安 + 雷塘陵区在乱中或有波及**——具体史载零散。**五代杨吴 + 南唐据江都**（杨行密 902 起、南唐 937 立）——杨吴在江都立都（金陵未立时）+ 南唐改江都为东都——**雷塘陵在杨吴南唐间是否有重修、史载未详**；按南唐对前朝帝陵重视、可能有零星维护。

**两宋**　**北宋初江都为扬州治**——雷塘陵在城北农田间。**北宋金石学家如赵明诚《金石录》等** 对雷塘陵碑刻有零星记载——但实物多被破毁。**南宋绍兴间扬州为江淮战区**——金兵屡至雷塘、陵区受损；**南宋后期雷塘陵已渐荒芜**。

**金元明清**　**金元明清间雷塘陵地表渐成农田 + 数座土堆**；陵区无大型修葺。**清嘉庆十二年（1807）扬州知府伊秉绶**——以"前朝帝陵不可无识" 立"隋炀帝陵" 碑于雷塘西——但**伊秉绶所定位置实是清代追认、未必是贞观改葬原址**。**清代以来雷塘"隋炀帝陵" 即按伊秉绶碑定位** + 民国 + 1949 后亦循此说。

**民国至今**　1949 后雷塘"隋炀帝陵"列江苏省重点文物保护单位、按清代伊秉绶碑定位；地表为草坡 + 土堆 + 清碑一座。**2013 年扬州西郊曹庄发现真正隋炀帝墓**——**南京博物院 + 扬州市考古研究所发掘**：出土**十三环金玉带、鎏金铜铺首、墓志砖**——墓志砖明确刻"隨故煬帝墓誌"——这才是炀帝真陵；**与传统雷塘"隋炀帝陵" 非同地**（曹庄在扬州西郊邗江区西湖镇、雷塘在扬州北郊邗江区槐泗镇——相距约 8 公里）。**萧皇后墓亦同地发现**——夫妻合葬之制按2013 考古确认。

**2013 后学界对炀帝陵真伪的研究** 重新展开——一说曹庄即贞观元年改葬之地（雷塘说有误）、一说曹庄是更晚（武则天 / 中晚唐）再改葬地——**史源争议尚未盖棺**。**唐人在 755 所见的那座雷塘湖畔土堆**——按传统说仍是凭吊地；但 2013 考古事实使"雷塘炀帝陵" 的物理性质成谜。**今游人来扬州**——可看雷塘清代隋炀帝陵公园（按伊秉绶定位）+ 曹庄真陵遗址公园（按 2013 发掘）——**两处并存**是中国陵墓研究中"传统说 vs 考古实证" 张力最戏剧的案例之一。
