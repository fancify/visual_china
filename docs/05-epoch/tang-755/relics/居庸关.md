# 居庸关

## 一句话

**太行八陉之最东陉、军都陉咽喉、幽燕北门户**——军都山中一道狭长关沟、北通塞外、南控幽州小盆地；**唐 755 时属妫州/妫川郡怀戎县境**（不在范阳郡幽州）；**再过半年安禄山自范阳起兵、叛军主力直接南下河北 + 山西、并不经居庸关**；**居庸关是塞外援军（同罗 / 奚 / 契丹等部）经此南下的通道**——居庸关在 755 这一刻仍是和平年代的关塞、半年后即成安史之乱的物理通道之一。

## 形貌

居庸关在唐河北道**妫州妫川郡境**——具体在**怀戎县东南军都山中**（今北京市昌平区南口镇）；距妫州治**怀戎县**（今河北怀来旧址）约**四十里**。**地势**：**军都山自西向东横亘**——北面是塞外蒙古高原南缘 + 燕山山脉；南面是华北平原北缘 + 幽州小盆地；**关塞立军都山中的一道狭长峡谷里**——南北约二十里、东西约二三里、两侧山势峻拔；**这道峡谷即太行八陉最东一陉之"军都陉"**——居庸关即扼此陉咽喉。

**居庸关的地理战略价值**：**幽州小盆地的北门户**——幽州小盆地（今北京小平原）三面环山：北面军都山、西面太行山、东面燕山；南面唯一开口向华北平原。**居庸关守北门**、**紫荆关守西门**、**渝关（今山海关一带）守东门**——三关共同护幽燕。**755 时各关分立、并不并称**——后世所谓"内三关"（居庸 / 紫荆 / 倒马）是明代命名（明长城九边格局之一段、不含渝关 / 山海关）；唐人不用此名。南北朝至隋唐皆以诸关为塞外与中原的物理边界。

**关城形制**：盛唐居庸关的具体形制 + 周长 + 门数史载零散——本 doc 不预设细节（**今所见居庸关楼 + 八达岭关楼皆为明洪武 / 嘉靖间始建 + 重修、不是唐关原物**）。唐居庸关大体应有**关墙 + 关门 + 戍兵营 + 关吏馆**；具体规模未见可信细节。**关塞外**：北通塞外蒙古高原 + 妫州 + 河东道云州（今大同一带）；**关塞内**：南通幽州 + 蓟县 + 范阳郡治。

**太行八陉中的位置**：太行山脉自南而北横贯关内道东缘 + 河东道东缘 + 河北道西缘——共有**八条**穿越太行的天然孔道（"陉" 即山隘），自南而北：**轵关陉**（济源境）—**太行陉**（沁阳-晋城）—**白陉**（陵川境）—**滏口陉**（邯郸-壶关）—**井陉**（鹿泉-井陉）—**飞狐陉**（涞源-蔚县）—**蒲阴陉**（易县-灵丘）—**军都陉**（昌平-延庆）。**居庸关即军都陉咽喉**——是太行八陉最北最东一陉的关塞。

**物候**：燕山-军都山一带——温带大陆性 + 季风过渡气候；冬寒夏凉、风沙大、雪较多。**关沟内常年有泉水 + 山溪**——是过往兵旅 + 商旅的天然水源；关塞外塞北、关塞内塞南、气候明显有差——这是地理意义的"中原塞外分界"。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

居庸关在唐人的回望里——是**塞外与中原的物理 anchor + 千年关塞**。

**居庸关的设立**：**居庸塞最早见于《吕氏春秋》**——战国燕国时期已是燕国北边的塞名；**秦汉间正式设关**——《汉书·地理志》载上谷郡有"居庸县"，关名所本。**居庸关名意**：一说"居庸"指"居者庸者"——徙役戍守之人居于此；一说秦时徙役士居于此故名。**自汉至北朝**，居庸关一直是中原王朝北疆的关塞——多次易主、多次重修。

**北朝至隋的居庸关**：**北齐天保七年（556）改居庸关名为"纳款关"**——意为"北族归附纳款之处"，反映北齐对北边柔然 + 突厥的羁縻；**隋开皇 / 大业间复名"居庸"**——隋大业三年（607）隋炀帝北巡至怀戎、出居庸关阅塞外；**唐承隋制**——居庸关仍属妫州、为塞北南通幽州的咽喉。

**唐居庸关的政治军事地位**：**唐承隋制、置妫州**——治怀戎县（今河北怀来）；**天宝元年（742）改妫州为妫川郡**——居庸关属妫川郡怀戎县境。**居庸关的守军**：由妫州地方军 + 唐廷派遣的关使共同守关；具体常额戍兵 + 防御工程史载零散，本 doc 不写无源数字。**居庸关在唐前期**——是北疆与塞外（突厥、回纥、契丹、奚等）交通 + 战防的重要关塞。

**与幽州 / 范阳的关系**：**幽州**（治蓟县、即范阳郡治；今北京西南广安门一带）在居庸关**南面约一百里**；**范阳节度使**自开元天宝之际 / 天宝元年（742）正式设置——驻幽州；**节度使辖境包括妫州**——居庸关因此在节度使军政范围内。**这一点很重要**：居庸关在范阳节度辖境西北、邻接妫州境；755 时安禄山身任范阳节度使、幽州 + 妫州 + 居庸关皆在其辖境——半年后他起兵南下、叛军主力沿幽州 + 河北道 + 山西一线南下、不经此关；但塞外援军（同罗 / 奚 / 契丹等部）或经此南下入塞。

**安史前夕**（即 755 这一刻）——再过半年安禄山起兵于范阳；**叛军主力沿幽州南下、河北道一路陷郡**；居庸关作为塞外 + 塞北方向的关塞——并非安禄山主进军路线、但是塞外援军（同罗、奚、契丹等部）经由的通道。**755 时居庸关仍是和平年代的关塞**——半年后即成安史之乱的物理通道之一。

### 此刻

天宝十四载这一年的居庸关——**关塞严整、守军按例屯戍、车马驿往南通幽州、北通塞外**。

**关城状态**：盛唐 755 时居庸关主体仍立、有关门 + 戍兵营 + 关吏；具体形制 + 守军规模史载零散。**关沟内外**：南门通军都陉南口（今南口镇），驿路至**昌平 + 蓟县 + 幽州**；北门通军都陉北口（今八达岭北一带），驿路至**妫州治怀戎县 + 河东道云州**。**关使**为唐廷或节度使派遣——具体在 755 这一刻谁任关使、史无确载。

**755 时居庸关的政治氛围**：唐廷对河北 + 幽燕方向的安全意识在玄宗朝相对放松——**因北疆已无大战二十余年**（自开元末至天宝十四载）；安禄山节度使专制范阳已十余年；**唐廷未察觉安禄山有反意**——居庸关一带亦无特别戒备。**这是 755 时唐廷对幽燕方向的盲点**——半年后即知不可挽回。

**755 末 + 756 的居庸关命运**：**安禄山十一月起兵于范阳**——主力**南下**沿河北 + 河南方向攻取洛阳 + 西攻潼关；居庸关并非主攻方向、未有直接战斗记载。**但 756 春夏间**——叛军一部经居庸关 + 塞外召集同罗 + 奚 + 契丹援军南下；居庸关因此成为安史之乱中的物理通道（不是激战之地）。**这是居庸关在盛唐唯一一段直接关涉天下大势的时间——但发生在 755 之后**。

### 此刻意味

对天宝十四载的妫州人 / 怀戎人来说——居庸关是**自家境内的关塞**：日常驿路过往、戍兵守关、商旅入关纳税。对幽州 + 范阳一带的士人来说——居庸关是**北门户**：北出 / 南入皆经此关；**盛唐诗中"居庸关"偶有出现**（高适、岑参等边塞诗或涉燕地），**但居庸关的诗咏密集化在中晚唐 + 明清**——盛唐时尚不是诗中常见地标。

唐人尚不知道——再过半年、安禄山将自范阳起兵、河北一路糜烂；居庸关将成为塞外援军南下的通道之一；**这道军都山中的狭长关沟**——半年后将见证盛唐最沉重的转折。**今居庸关古关已不复唐时形制**：**现存关楼主体始建于明洪武年间（1368+）**、**成化二年（1466）大规模重修**、**八达岭关楼是明嘉靖以后**；**关沟中的元代云台（1342 建）**——是元代藏传佛教遗物、不属唐时。今游人来居庸关、所见的是**明长城段的居庸关楼 + 元代云台 + 现代游览道**——**唐人在 755 所见的那座居庸关**：千二百多年后**形制已不可复原**——盛唐居庸关的物理终点是明代长城工程。

## 子条目

### 太行八陉之军都陉

**太行八陉**——自南而北八条穿越太行山脉的天然孔道：轵关陉、太行陉、白陉、滏口陉、井陉、飞狐陉、蒲阴陉、**军都陉**。**军都陉**即居庸关所在——位太行八陉最东、最北一陉。**军都陉的地形**：自南口镇（南口）入山——沿一条狭长峡谷向西北约二十里——至北口（今八达岭一带）出山；**居庸关即扼此陉咽喉**、立南口与北口之间。**军都陉自南而北穿军都山——是幽州小盆地通往塞外蒙古高原的最直接通道**——盛唐时是北疆驿路的核心一段。

### 居庸关 ≠ 八达岭（明长城段、不是唐关）

**居庸关本关**在军都陉南段、距南口镇较近；**八达岭**在军都陉北段、距北口较近——两者相距约**二十里**；**八达岭是居庸关北道的另一道关塞**。**唐时八达岭一带或有戍兵 / 烽燧、但不是独立关塞**——居庸关 + 八达岭分立两关、是**明代长城工程**的产物。**今游人所见的"长城八达岭"**——是**明嘉靖以后**修筑的明长城段、不是唐居庸关原物。**盛唐 755 时不存在"八达岭关"**——这一点常被现代游客 / 文献混淆。

### 居庸关 ≠ 范阳郡幽州（在妫州境）

**居庸关在唐 755 时属妫州/妫川郡怀戎县境**——**不在范阳郡幽州境**。**妫州**在幽州**西北约一百里**——治怀戎县（今河北怀来旧址、官厅水库淹没区）；**天宝元年（742）改妫州为妫川郡**；**辖怀戎县 + 妫川县**（两县均在今河北怀来 + 北京延庆一带）。**居庸关虽距幽州（蓟县）较近、但属妫州境**——这是唐代行政地理的细节。范阳节度使辖境包括妫州——居庸关因此在节度使军政范围内、但行政归属是**妫州/妫川郡**。

### 北齐"纳款关"——居庸关名一度变更

**北齐天保七年（556）**——文宣帝高洋改居庸关为"纳款关"，意为"北族归附纳款之处"——反映北齐对北边柔然 + 突厥的羁縻政策。**这一名号仅行约半世纪**——隋开皇 / 大业间复名"居庸"。**唐承隋制**——居庸关名沿用至今。**"纳款关" 是居庸关历史上短暂的别名**——盛唐时已不用、但唐人在史书 + 怀古文中偶提。

### 幽燕咽喉 + 三关格局

**幽州小盆地**（今北京小平原）三面环山——**北面军都山**、**西面太行山**、**东面燕山**；**形成"幽燕咽喉"的三关格局**：**北门居庸关**（军都山中）、**西门紫荆关**（太行山中、易州境）、**东门渝关**（燕山东端、平州境；即明代山海关一带）。**唐时这三关各自分立护幽燕**——居庸关守北、紫荆关守西、渝关守东；755 时不并称、亦无统一名号。**安禄山起兵后**——叛军主力南下、未涉三关；但塞外援军经居庸关南下。**后世所谓"内三关"（居庸 / 紫荆 / 倒马）是明代命名**——属明长城九边格局、含紫荆与倒马（在今河北唐县境）而**不含渝关 / 山海关**；唐人不用此名。三关格局自唐至明清一直是中原王朝防御北疆的核心、但唐 755 时尚是各关分立。

### 安禄山起兵与居庸关（755 末 + 756）

**天宝十四载十一月（755 末）**——安禄山起兵于范阳；**叛军主力沿幽州南下、河北一路陷郡**；**居庸关并非主攻方向、未有直接战斗记载**。但**塞外援军（同罗、奚、契丹等部）**——是安禄山起兵的重要兵源；**塞外援军南下经由居庸关 + 妫州**——居庸关因此是安史之乱的物理通道之一（非激战之地）。**这是居庸关在盛唐唯一一段直接关涉天下大势的时间**——但不发生在 755 这一刻、而在半年之后。

### 关沟内的水（温榆河 / 北沙河水系）

居庸关所在的军都陉——**沿一条山溪展开**；山溪自北而南穿关沟、出南口入华北平原；**这条关沟南流水属温榆河 / 北沙河水系**（沿燕山南麓东南入华北平原）——是关沟内的天然水源。**注意：妫水河（今延庆境妫水河）属永定河水系、是另一条河**——与关沟南流水不同源、本 doc 不混称。**盛唐过关者**——必沿此溪行走；溪水冬不冻（关沟内地温较高）、夏不涸——是关沟驿道的物理 anchor。**今关沟内仍有此溪**——但水量较唐时减少；溪两岸有元代云台 + 明代关楼 + 民国京张铁路（詹天佑 1909 年人字形铁路）+ 现代京张高铁共构一道**两千年关沟交通史的物理叠层**。

## 历史长河（755 → 今）

**中晚唐**　**安史之乱（755-763）期间**——居庸关是塞外援军南下的物理通道；**763 年安史之乱结束**——居庸关回归唐廷。**中晚唐居庸关**——常被河朔藩镇与唐廷反复争夺；具体战事史载零散。

**五代**　**五代间居庸关多次易主**——后梁、后唐、后晋皆控居庸关；**后晋天福三年（938）石敬瑭割燕云十六州予契丹**——**居庸关随幽州归契丹（辽）**——自此居庸关在塞外政权手中约**四百年**（辽 + 金 + 元前期）。

**辽金**　**辽时居庸关**——是辽南京（今北京一带）的北门户；**金时居庸关**——是金中都（今北京一带）的北门户；**金末 1213 年成吉思汗破居庸关**——是蒙古攻金的关键一战。

**元**　**元至正二年（1342）建居庸关云台**——是元代藏传佛教石刻、有六体（梵 + 藏 + 八思巴 + 西夏 + 维吾尔 + 汉）真言；云台原为过街塔基座、塔已不存——**今云台仍立于关沟中**、是居庸关现存最古实物（元代、不属唐时）。

**明**　**洪武元年（1368）明军北伐、收复居庸关**——**洪武年间始建明居庸关楼 + 关城**；**成化二年（1466）大规模重修**；**嘉靖以后扩建八达岭关楼 + 加固关墙**——形成今所见**明长城居庸关-八达岭段**的格局。**明居庸关是九边重镇之一**——属蓟镇辖境、守北京北门。

**清**　**清初居庸关沿明制**；**清中后期渐失军事意义**——清廷与蒙古结盟、塞外不再是边境；居庸关 + 长城渐废。

**民国至今**　**1909 年京张铁路通车**——詹天佑设计的"人字形铁路"穿越关沟、青龙桥车站立关沟中段；**居庸关 + 八达岭关楼 1961 年列全国重点文物保护单位**；**1980s 起八达岭长城开放为旅游景区**——是中国接待外国元首最多的长城段；**居庸关云台 1961 年单独列国保**——是关沟中最古实物。**今居庸关 + 八达岭 + 京张高铁**——共构关沟中**两千年交通史的物理叠层**：唐居庸关 + 元云台 + 明长城 + 清驿路 + 民国铁路 + 现代高铁——一道关沟、千年层叠。**唐人在 755 所见的那座居庸关**——千二百多年后**形制已不可复原**：盛唐居庸关的物理终点是明代长城工程。
