# 云冈石窟

## 一句话

**北魏皇家鲜卑佛教的物理 anchor**——**文成帝和平元年（460）始凿、沙门统昙曜主持、凿昙曜五窟（今 16-20 窟）于平城西武州山**；至**孝文帝太和十八年（494）迁洛前主体完工**——为北魏前期 35 年所造；**755 时距始凿已 295 年、距迁洛 261 年**——以**云州云中郡治云中县**西约 16 里武州山石窟身份存在（属河东节度），主体造像 + 部分寺院犹存、但自孝文帝迁洛后渐冷落、是盛唐北方塞下"北魏皇家佛教"最完整的物理遗存。

## 形貌

云冈石窟在唐河东道**云州云中郡治云中县**西——具体在**云中县西约 16 里武州山南麓**（今山西省大同市西约 16 公里武周山，武州川北岸）。**地势**：武州山是阴山南麓的一段砂岩低山——海拔约 1200 米、相对平城高出 150 米；**南面是武州川**（今十里河，桑干河支流）——**石窟即凿于武州川北岸的砂岩崖壁**；**东南望平城宫城**约 16 公里；**北面是武州山主脊**。

**北魏云冈石窟的窟制**——史载昙曜白文成帝、"于京城西武州塞，凿山石壁，开窟五所，镌建佛像各一"（《释老志》）——**即今编号 16-20 窟（昙曜五窟）**，是云冈最早 + 规模最宏的一组：
- **昙曜五窟（16-20 窟）**：和平元年（460）至文成帝末年（465）凿——五窟主像高 13-17 米；
- **主像题材**：五尊大佛——**一说象征道武、明元、太武、文成、景穆（追尊）五帝**——即"佛即皇帝"的鲜卑皇家政教合一；
- **凿法**：依山为窟、内雕主像 + 二弟子 / 二菩萨；五窟皆椭圆形草庐式平面——是中国早期石窟的典型形制。

**云冈中期 + 晚期窟**：
- **中期窟**（约 465-494）：献文帝 + 孝文帝前期开凿——以**5、6、7、8、9、10、11、12、13 窟**等为主；规模更大、内容更丰富、汉化色彩渐浓；
- **晚期窟**（约 494-525）：孝文帝迁洛后续凿——以**4、14、15 窟 + 21-45 窟（西部小窟群）**为主；规模较小、属民间或地方营造。

**云冈石窟的总规模**：现存**252 个洞窟 + 51000 余尊造像**——主像高达 17 米（19 窟）；是中国最早 + 规模最大的早期石窟群之一（与敦煌莫高窟 + 龙门石窟并列三大石窟）。**北魏前期皇家造像 + 中期汉化造像 + 晚期民间造像**——三段风格在云冈一山之内并陈。

**云冈石窟的配套寺院**：北魏时石窟前建**灵岩寺**（亦称武州山寺）——是云冈石窟的礼制 + 僧团基地。**寺院遗址**在云冈窟前——1980s-2010s 山西省考古研究院 + 云冈石窟研究院多次发掘——确认北魏寺院基址 + 出土北魏陶瓦 + 佛教造像残件。

物候：山西最北、温带大陆性气候——干燥、风大、冬寒夏凉；春秋多沙尘、冬季严寒——是北方塞下的代表气候。武州山砂岩质地中等——千余年风化使部分窟外造像残蚀严重；但窟内造像因避风保存较好。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

云冈石窟在唐人的回望里——是**北魏皇家鲜卑佛教的物理 anchor + 中国早期皇家石窟的代表**。

**始凿的时代背景**——**北魏太武灭佛 + 文成复佛**：
- **太武帝拓跋焘**（408-452）——**太平真君七年（446）三月**因关中盖吴起兵牵涉沙门、下诏诛沙门 + 毁佛像 + 焚佛经——**是中国历史上最早的大规模灭佛运动**；至**正平二年（452）太武帝崩、灭佛终**——前后约 6 年。
- **文成帝拓跋濬**（440-465）——太武帝之孙、景穆帝拓跋晃之子（景穆未即位、为太武帝太子；449 死、追尊景穆）；**兴安元年（452）即位、即诏复佛**；**和平元年（460）沙门统昙曜白帝、奏请凿武州山石窟**——即云冈昙曜五窟。**"佛即皇帝"** 的政教合一——是文成帝复佛 + 凿云冈的核心政治意图：以五尊大佛象征道武以来五帝、表达鲜卑皇家与佛教的合一。

**昙曜其人 + 昙曜五窟**：**昙曜**——北凉沙门、北凉灭后入北魏；太武灭佛中藏于云中；文成复佛后被擢为**沙门统**（北魏佛教最高僧官）。**和平元年（460）昙曜奏请凿武州山石窟**——主持开凿**昙曜五窟**（今 16-20 窟）：
- **16 窟**：主像立佛——一说象征文成帝；
- **17 窟**：主像交脚弥勒——一说象征景穆帝（追尊）；
- **18 窟**：主像立佛 + 千佛——一说象征太武帝；
- **19 窟**：主像坐佛高 16.8 米——云冈最大像之一、一说象征明元帝；
- **20 窟**：主像坐佛高 13.7 米——**云冈最有名的露天大佛**（窟前壁早坍）、一说象征道武帝。

**注**：昙曜五窟的"五窟象征五帝"系一说——史载昙曜白文成帝凿石窟、《魏书·释老志》未明言象征何帝；后世（北齐 + 唐 + 现代学者）多以五窟象征道武以来五帝。**本 doc 严守"一说五窟象征五帝、史载未明言"口径**。

**云冈中期开凿（约 465-494）**——文成帝末年至孝文帝迁洛前**5-13 窟**等开凿；**冯太后摄政 + 孝文帝前期**——汉化渐深、佛像汉化色彩渐浓（如**6 窟**中心塔柱 + 佛传故事浮雕，是汉化的代表）。

**孝文帝迁洛（494）后**——云冈主体工程已完工；**孝文帝迁洛后云冈渐冷落**——大型皇家工程转至洛阳龙门；云冈仅续凿小窟（4、14、15、21-45 窟）至**北魏末（约 525）**。**北齐（550-577）续凿云冈较少**——主体已完成；**北周（557-581）灭北齐后北周末年（574-577）武帝灭佛**——云冈未大破坏（武帝灭佛主要在长安 / 关中）。

**隋 + 唐前期的云冈**——**隋开皇 + 大业**云冈寺院续修；**唐贞观 + 开元** 云州一带屯戍渐稳——云冈仍有僧人居、寺院规模远不及北魏鼎盛期。

### 此刻

天宝十四载这一年的云冈石窟——**距文成帝和平元年（460）始凿已 295 年、距孝文帝迁洛（494）已 261 年**；以**云州云中郡治云中县**西约 16 里武州山石窟身份存在（属河东节度）。

**云州的唐廷建制**：**贞观四年（630）唐廷置云州**；**永淳元年（682）突厥扰边、云州一度废**；**开元十八年（730）复置云州**——天宝元年（742）改云中郡、**乾元元年（758）复称云州**。**755 时云州属河东节度**——河东节度治太原、节度使范围含云州、朔州、代州、忻州等北方边州。云冈石窟在云州治所云中县西 16 里——是云州境内最有名的北朝佛教遗存。

**云冈石窟 755 时的状态**：
- **昙曜五窟（16-20 窟）**：主体造像犹存——主像未大破坏；但**20 窟前壁早坍**（北魏末或更早）——主像（13.7 米坐佛）已为露天；窟外造像因风化已部分残蚀；
- **中期窟（5-13 窟等）**：造像 + 浮雕 + 中心塔柱皆存——汉化色彩浓；
- **晚期窟（西部小窟群）**：规模较小、保存中等；
- **灵岩寺**：北魏寺院已残破——唐时有云冈寺续之、规模远不及北魏鼎盛；唐廷未列云冈为皇家寺、亦未大力修缮；
- **僧人**：唐时仍有僧人居云冈——但人数远不及北魏鼎盛期的"千僧" 之盛；多为云州地方僧团。

**唐人对云冈的认知**——**"北魏皇家石窟"**：唐人凡至云州者多有见云冈、并知其为北魏文成帝时所凿；**盛唐士人通晓的北魏故事中、云冈石窟与孝文帝迁洛 + 太武灭佛 + 太武灭北凉并列**——是北朝最有政治分量的四件大事之一。

**唐人的云冈诗 + 散文**——**盛唐边塞诗中"代北 / 云中"意象高频**，但**直接咏云冈石窟的盛唐诗较少**——主要原因有二：（1）盛唐边塞诗多咏战事 + 思归，非佛教题材；（2）云冈在 755 时已渐冷落、不在盛唐佛教中心叙事中（盛唐佛教中心在长安、洛阳、五台、终南）。**中晚唐边塞诗**（李益、卢纶、马戴等）多咏代北、云中——偶有提及武州山或云冈寺者。

**这一年的云冈** 处于**北魏皇家石窟遗存 + 唐地方寺院 + 边塞冷落**的稳定状态——再过几个月安禄山起兵于范阳；**至德二载（757）唐军反攻**——云州一带反复易主；**广德元年（763）后吐蕃 + 回纥渐扰**——云冈所在武州山一带在唐末沙陀崛起前数十年间为边军 + 民间礼佛之地。

### 此刻意味

对天宝十四载的云州人来说——云冈石窟就是**自家州城西郊的北朝旧迹**：武州山砂岩崖壁上几十座洞窟、主像高三五丈、佛颜静默面对武州川——是北魏 96 年平城都期间"佛即皇帝" 的物理纪念。对长安那一代士人来说——云冈是**北魏文成帝 460 始凿、距今 295 年的皇家石窟**：拓跋鲜卑入主华夏、太武灭佛、文成复佛、昙曜白帝凿石窟、五尊大佛象征五帝、孝文帝迁洛后续凿渐冷——这一段近 70 年（460-525）的皇家佛教史**全部以武州山为舞台**。

**"佛即皇帝" 的物理 anchor**——是云冈最特殊的政治分量。中国历史上以皇家政治意图凿大型石窟、并以佛像直接象征本朝皇帝者——以云冈昙曜五窟为最早 + 最明显；后世洛阳龙门 + 敦煌莫高窟虽规模 + 持续时间皆超云冈、但**"五窟象征五帝" 的皇家政教合一** 此一意图、以云冈为孤例。

但**唐云冈不是"皇家石窟"**——唐云州的云冈是**盛唐边塞下的地方寺院 + 北朝旧迹**；皇家政治的物理记忆已被埋入云州的边防日常。**两重身份的张力**——是云冈在 755 时最深的意味：既是 5 世纪鲜卑皇家"佛即皇帝" 的物理 anchor、又是 8 世纪盛唐边塞下的地方礼佛之处。**这种张力以物理形式存在**：武州山砂岩崖壁 + 露天大佛（20 窟）+ 渐残蚀的中期窟造像 + 唐云州城——四者皆在桑干河北岸不到 20 公里的范围内。

唐人尚不知道——再过千二百年（21 世纪），**云冈石窟列联合国教科文组织世界文化遗产**（2001）；**北魏前期皇家石窟**成为研究 5-6 世纪鲜卑入主华夏 + 佛教汉化的核心实物 anchor。**唐人在 755 所见的云冈石窟**——千二百多年后**主体造像 + 252 窟 + 51000 余尊造像犹存**；但**砂岩风化日深、窟外造像残蚀严重**——21 世纪云冈石窟研究院多年保护工程方稳住风化。**唐人在武州山所见的北魏砂岩崖壁** 千二百年来风蚀渐深——但**昙曜五窟主像（高 13-17 米）** 千二百年未易、是 5 世纪鲜卑皇家"佛即皇帝" 最完整的物理纪念。

## 子条目

### 太武灭佛 + 文成复佛（446-460）

**北魏太武帝拓跋焘**（408-452）——北魏第三代皇帝、最有军事才能的鲜卑君主；**太平真君七年（446）三月**因关中盖吴起兵牵涉沙门、下诏诛沙门 + 毁佛像 + 焚佛经——是中国历史上最早的大规模灭佛运动；**至正平二年（452）太武帝崩、灭佛终**——前后约 6 年。**太武灭佛的政治背景**：（1）寇谦之的天师道在太武帝朝得势——主张排佛；（2）盖吴起兵中有沙门参与——太武帝以"沙门多与盖吴通谋" 为由灭佛；（3）太武帝晚年信道排佛。

**文成帝拓跋濬**（440-465）——太武帝孙、景穆帝之子；**兴安元年（452）十月即位、即诏复佛**——是太武灭佛后的政治反动。**和平元年（460）沙门统昙曜白文成帝、奏请凿武州山石窟**——是文成复佛后最大规模的皇家佛教工程。**自太武灭佛（446）至云冈始凿（460）**——前后 14 年；**云冈是文成复佛的物理 anchor + 北魏皇家佛教的最完整体现**。

### 昙曜五窟（和平元年/460 始凿，今 16-20 窟）

**昙曜**——北凉沙门、北凉灭（439）后入北魏；太武灭佛中藏于云中、保身完整；文成复佛后被擢为**沙门统**（北魏佛教最高僧官）。**和平元年（460）昙曜白文成帝、于京城西武州塞凿山石壁、开窟五所、镌建佛像各一**（《魏书·释老志》）——即**昙曜五窟、今编 16-20 窟**：
- **16 窟**：主像立佛、释迦像、高 13.5 米——一说象征文成帝拓跋濬本人；
- **17 窟**：主像交脚弥勒坐像、高 15.6 米——一说象征景穆帝拓跋晃（追尊太子）；
- **18 窟**：主像立佛、高 15.5 米、身覆千佛袈裟——一说象征太武帝拓跋焘（袈裟千佛是其灭佛后复佛的赎罪意象）；
- **19 窟**：主像坐佛、高 16.8 米——**云冈最大主像**、一说象征明元帝拓跋嗣；
- **20 窟**：主像坐佛、高 13.7 米——**云冈最有名的露天大佛**（窟前壁早坍，主像直接面对武州川）；一说象征道武帝拓跋珪。

**"佛即皇帝" 的政教合一**——是昙曜五窟的核心政治意图；**注**：五窟象征五帝系一说、《魏书·释老志》未明言；本 doc 严守"一说五窟象征五帝、史载未明言"口径。

**昙曜五窟的凿造工艺**——五窟皆椭圆形草庐式平面、依山为窟、内雕主像 + 二弟子 / 二菩萨；造像风格上承凉州模式（昙曜来自北凉）、并融入北魏皇家鲜卑审美——是中国早期石窟艺术的代表 + 凉州 → 平城佛教传播的物理证据。

### 云冈中期窟（约 465-494，5-13 窟等）

**文成帝末年至孝文帝迁洛前**——云冈进入中期开凿；以**5、6、7、8、9、10、11、12、13 窟**等为主：
- **5、6 窟**（孝文帝时凿）：双窟、内雕中心塔柱 + 佛传浮雕；**6 窟中心塔柱** 雕释迦牟尼一生佛传故事——是云冈艺术最精的中期代表；
- **7、8 窟**（献文帝时凿）：双窟、内雕佛 + 菩萨 + 飞天；
- **9、10 窟**（孝文帝早期凿）：双窟、内雕佛 + 维摩 / 文殊对辩；
- **11、12、13 窟**：单窟、内雕佛 + 千佛。

**中期窟的汉化色彩**——较之昙曜五窟（鲜卑皇家审美），中期窟造像 + 浮雕呈现明显汉化：**佛装由半披袒右肩转为汉式褒衣博带、面容由健硕转为清秀、动作由静穆转为生动**——是冯太后摄政 + 孝文帝亲政期间"鲜卑汉化" 的物理表现。**云冈中期窟艺术** 直接影响后来的洛阳龙门石窟 + 巩义石窟。

### 孝文帝迁洛（494）后云冈渐冷落

**孝文帝拓跋宏**（467-499）——北魏第六代皇帝、最有名的汉化改革者；**太和十八年（494）正式迁都洛阳**——云冈主体工程已完工；**自迁洛后云冈渐冷落**——大型皇家工程转至洛阳龙门（**洛阳龙门石窟于太和十九年/495 始凿**，即孝文帝迁洛次年）。**云冈仅续凿小窟（4、14、15、21-45 窟）至北魏末（约 525）**——多为民间或地方营造、规模 + 工艺远不及前期。

**北魏末的六镇之乱（524-525）**——云冈所在云州一带反复争夺；云冈寺院遭一定程度破坏，但石窟造像主体未大损。**东魏 / 北齐 / 北周**（534-581）——云冈续凿渐止；**北周武帝灭佛**（建德三年/574-575）主要在长安 / 关中，云冈未大破坏。**隋 + 唐前期**——云冈寺院续修、但规模远不及北魏鼎盛。

**注**：本 doc 严守"云冈主体 = 北魏前期（460-494, 平城都期）、后续凿延至北魏末（约 525）、北齐至唐续凿较少"口径——不要把"龙门石窟（北魏迁洛后凿，在洛阳）" 与"云冈石窟（北魏前期凿，在平城）" 混。

### 武州山 + 武州川 + 灵岩寺

**武州山**——阴山南麓的一段砂岩低山；古名**武州塞**（北魏时）；《水经注·㶟水》载"㶟水南有武州川水、自西出武州县（北魏置）境东南流"——即云冈所在山川。**武州山砂岩** 质地中等——适合凿窟、但千余年风化使部分窟外造像残蚀严重。

**武州川**——今名十里河、桑干河支流；自武州山南麓东南流入桑干河；北魏时为平城 → 武州山交通水道。

**灵岩寺**（亦称武州山寺、云冈寺）——北魏时配套云冈石窟所建寺院；位于云冈窟前。**1980s-2010s 山西省考古研究院 + 云冈石窟研究院多次发掘窟前寺院遗址**——确认北魏寺院基址（含殿堂、僧房、塔基）+ 出土北魏陶瓦 + 佛教造像残件 + 北朝至唐瓷器。**唐时寺院续之**——但规模 + 僧团远不及北魏鼎盛。

## 历史长河（755 → 今）

**中晚唐**　**至德二载（757）安史中云州一带反复易主**；云冈石窟具体在安史中是否有破坏史载零散——但主体造像未大损。**广德元年（763）后回纥 + 吐蕃渐扰北边**——云州为唐廷北边争夺地；云冈寺院续存。**唐末沙陀崛起**——李克用为代北沙陀首领、根据地即在云州 / 代北一带；沙陀人多信佛——云冈寺院在唐末沙陀治下续存。

**五代 + 辽 + 金**　**后晋天福元年（936）石敬瑭割燕云十六州予契丹**——云州（即平城旧址）入辽；**辽改云州为西京大同府**——**自此"大同"之名始**。**辽 + 金代** 大同为重镇——辽 + 金多次修云冈寺、续凿部分小窟、并修石窟前堡寨"云冈十寺"（辽代）；金代续之。**辽金时云冈寺院规模略复**——但石窟造像主体仍为北魏遗存。

**元 + 明 + 清**　**元代大同路**——云冈寺院续存、但规模渐减；**明代大同府** 是九边重镇之一、设大同总兵镇守；**明代云冈** 因军事修堡——部分窟前堡寨改建；**清代大同府** 渐失军事分量、云冈寺院亦渐衰；**清末民国** 云冈石窟造像保护乏力——窟外造像风化日深。

**民国至今**　**1949 年后**——**云冈石窟列国家级重点文物保护单位**（1961 年第一批）；**1980s-2010s 山西省考古研究院 + 云冈石窟研究院多次发掘 + 保护**——确认窟前寺院基址、稳住砂岩风化、修复部分残蚀造像。**2001 年云冈石窟列联合国教科文组织世界文化遗产**——是中国早期石窟艺术的代表 + 北魏皇家鲜卑佛教的物理 anchor。**今游人来云冈**——所见的是**252 个洞窟 + 51000 余尊北魏 + 北齐 + 隋唐 + 辽金造像 + 昙曜五窟主像（高 13-17 米）+ 露天大佛（20 窟）**——**唐人在 755 所见的云冈石窟主体造像犹存**；**昙曜五窟的"佛即皇帝" 政教合一物理 anchor** 千二百年未易、是 5 世纪鲜卑入主华夏最完整的物理纪念。
