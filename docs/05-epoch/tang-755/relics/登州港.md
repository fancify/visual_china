# 登州港

## 一句话

**渤海湾口的盛唐海港**——登州港在河南道登州（742-758 名"东牟郡"）治蓬莱县海岸；755 时是**唐与新罗、日本北航路的起点**——自登州出黄海 → 朝鲜半岛西海岸 → 新罗 → 日本九州博多；**与南端明州港、苏州港的南航路并立、共同构成盛唐东出海三大海口**；同时是**蓬莱方丈瀛洲三神山的传说之地**——秦皇汉武东巡求仙皆至此岸；755 时港中楼船 + 商舶 + 使节舸 出入有时、是河南道东端面海的物理 anchor。**蓬莱阁（北宋嘉祐六年/1061 始建）此时未立**——盛唐时登州港所见、只是面海低丘与几处渔村码头。

## 形貌

登州港在 755 时——**胶东半岛北角一处面海的天然海湾港 + 渔村码头**，远非后世明清备倭水城的规模。

**地理位置**：**河南道登州东牟郡治蓬莱县**（今山东烟台市蓬莱区）海岸；登州城本身在半岛北岸一处可泊船的湾里——东南是丘陵、北面便是黄海。**蓬莱海岸**——半岛北缘 + 渤海南岸交接处；港湾东西略长、北面对开渤海口 + 东北可入黄海；**与莱州港（今莱州市；登州西南约百余唐里）并立胶东半岛两港**——莱州港在半岛西南面渤海湾、登州港在半岛北角面渤海湾口 + 黄海北端、两港隔半岛分掌渤海两面海运。

**港口设施**：盛唐 755 时登州港——**官设水次（即海岸渡口 / 海口）+ 民间渔村码头**并立；**州治蓬莱县直辖**——津吏 + 海商 + 渔户 + 应差舟丁；**港中泊舟**：江北沿岸海船（多为隋唐式平底沙船 / 鸟船）+ 渤海湾内的近海贸舟 + 跨黄海的新罗 / 日本舶 + 偶有少量大食 / 波斯远海商舶（中晚唐渐多、755 时罕）；**港岸建置**——不立 市舶司（**市舶使开元二年/714 始见于安南, 广州后成主设地; 登州未设**）；州县官以平常州务管之、海事散于诸县。**港口的物理形态**——盛唐时蓬莱海岸尚为低丘 + 沙滩 + 几处天然湾澳；无后世明清水城（明洪武年间始筑蓬莱水城备倭）。

**周边水陆**：
- **海上路线**——**北航路**：登州出港 → 庙岛群岛 → 辽东半岛南端 → 鸭绿江口 / 渤海湾北 → 渤海国（粟末靺鞨；今吉林 + 朝鲜东北部）；**东航路**：登州 → 黄海 → 朝鲜半岛西海岸 → 新罗西海岸瓮津 / 南阳湾 / 唐恩浦 → 新罗都金城（今韩国庆州）；**远航路**：自新罗续航 → 对马海峡 → 九州博多 → 日本京畿（**这是 7 世纪后期至 8 世纪前期日本遣唐使主航线、亦即"北路"**）；
- **陆上路线**——**登州 ↔ 青州 ↔ 兖州 ↔ 洛阳**：自登州西行约百余唐里至莱州、再西约二百唐里至青州（北海郡治；河南道东端大州）、再南行至兖州、再西至洛阳；是新罗 + 日本使臣登陆后入两京的标准陆路；
- **港外水文**——蓬莱海岸面渤海湾口 + 黄海西北角；潮汐落差盛唐时已显（与南端杭州湾 + 长江口潮汐相比偏小）；黄海北部冬季偶有海冰（盛唐 8 世纪正值中世纪暖期 + 海冰较轻）；**风向**——夏多东南风 + 利北上 / 东进；秋多西北风 + 利南返；冬北风急 + 海路停。

**物候**：盛唐时蓬莱一带——**温带季风、四时分明**；春多桃花水 + 海岸薄雾；夏多霖 + 海风急 + 海舶往来盛；秋高气爽 + 商舶南返；冬有薄霜 + 海面薄冰不冻港。**海上能见度**——晴日自蓬莱海岸东望、隐约可见庙岛群岛 + 长山列岛轮廓（即《史记·封禅书》"望蓬莱方丈瀛洲三神山"的实景由来）。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

登州港在唐人的回望里——**是秦皇汉武东巡求仙的海岸 + 隋唐北航路的起点 + 渤海湾口的东出海第一港**。

**秦皇汉武东巡的回望**：
- **秦始皇二十八年（前 219）**——东巡至琅邪 + 望海中三神山；**三十二年（前 215）**——派燕人卢生 + 韩终入海寻三神山；**三十七年（前 210）**——再东巡至琅邪 + 死沙丘；《史记·秦始皇本纪》载"齐人徐市等上书、言海中有三神山、名曰蓬莱、方丈、瀛洲、仙人居之"——这是**蓬莱地名进入中国文化想象的起点**；
- **汉武帝元光二年（前 133）**——东巡海上望蓬莱 + 派方士入海求蓬莱（《汉书·武帝纪》）；元封二年（前 109）+ 太初三年（前 102）+ 天汉三年（前 98）多次东巡 + 派方士入海；**蓬莱县置**——汉武元光二年后于此一带置蓬莱县（《齐乘》异说；正史无明文置县年份，《汉书·地理志》东莱郡未列蓬莱县；蓬莱县名见于唐《元和郡县图志》卷十一·登州蓬莱县条引旧志）；
- **盛唐人的认知**——蓬莱不仅是地名、更是**仙山 + 海上仙境 + 东巡望海**的文化 anchor；登州港即建在这片"望蓬莱"的海岸上、唐人入港有古意。

**州名沿革的回望**：
- **武德四年（621）**——析牟州（北齐旧州）以文登县置；
- **如意元年（692）**——析莱州黄、牟平、文登三县别立**登州**、治牟平县；
- **神龙三年（707）**——徙登州治蓬莱县（自此登州治蓬莱、千余年未再大迁）；
- **天宝元年（742）**——改登州为**东牟郡**；
- **乾元元年（758）**——复登州——**755 当年的官号是"东牟郡"、俗称仍呼登州**；行政文书写"东牟郡蓬莱县登州海口"、民间俗称"登州港""蓬莱港"皆可。

**海路的回望**：
- **南北朝**——刘宋 + 北魏间已有自青州 + 登州出海至高句丽 + 百济 + 倭国的航路；高句丽僧人入华亦多自此登陆；
- **隋大业三年（607）**——隋炀帝遣使裴矩通西域 + 同时遣使经登州出海至倭国（《隋书·东夷传》）；倭国遣隋使小野妹子等亦自登州一带登陆；
- **唐贞观四年（630）**——日本第一次遣唐使犬上御田锹等至唐；早期遣唐使取**北路**（自难波出发 → 北九州 → 对马 → 朝鲜半岛西海岸 → 山东登州 / 莱州登陆）；**贞观至开元间七至八次北路**——登州 + 莱州一带是日本遣唐使的主要登陆港；
- **开元 / 天宝间**——遣唐使航线变迁；**开元二十一年（733）多治比广成第九次遣唐使**为最后一次明确北路；**天宝十二载（753）藤原清河 + 吉备真备 + 鉴真东渡**为南路（自扬州 / 苏州出海 → 东海 → 九州）——**755 这一刻、日本遣唐使刚走完一次南路**；
- **中晚唐**——遣唐使一度回归北路（如承和五年/838 圆仁随藤原常嗣 + 小野篁），亦兼用南路；**圆仁《入唐求法巡礼行记》838-847**记登州 + 文登 + 赤山诸条最详——但属**中唐文献、不能为 755 现场作证**、只可作回看的 framework。

**唐人对登州港的认知**——是河南道东端最重要的海港 + 东出海北航路的起点 + 与新罗 / 日本 / 渤海国的官方往来主要登陆港；自登州登陆的新罗 + 日本使臣经河南道陆行入两京、是盛唐"四夷宾服"图景的东海一角。

### 此刻

天宝十四载这一年的登州港——**按例运行、海路通畅、商舶 + 使节舸 + 渔舟 出入有时、即将经历安史浩劫**。

**港口状态**：盛唐 755 时登州港——州县官以平常州务管之、不立市舶；**官设水次 + 民间渔村码头并立**；港中泊舟数十至百余艘（视季节）；津吏验牒、官差应役、商旅纳钱；**港岸民居数百户**——多为渔户 + 海商 + 应差舟丁 + 少数新罗 + 日本侨客；**登州城（蓬莱县治）**在港湾南岸约二三唐里、城圈在今蓬莱阁山下旧城一带；**州治户口**——《新唐书·地理志二》载登州东牟郡户 22,298 + 口 108,009（《旧唐书》载天宝户 20,298 + 口 108,900、互有出入），在河南道诸州中属**中州偏上**；**土贡**——金、石器、水葱席、赀布、海藻、文蛤、牛黄（《新唐书·地理志二》登州东牟郡条）——**海藻 + 文蛤是面海之州的土贡**、是登州海岸物产的实证。

**海路的实际使用者**：
- **新罗使臣 + 商舶**——盛唐与新罗官方往来岁有定例；新罗使臣自其都金城出发、沿半岛西海岸北上、自瓮津 / 南阳湾 / 唐恩浦横渡黄海、**主在登州或楚州登陆**、再陆行经河南道入两京；新罗商舶载金 + 铁 + 人参 + 海产 + 麻布 + 丝绢往来；
- **日本遣唐使（北路时段）**——盛唐前期日本遣唐使多取北路、自登州 + 莱州登陆；**开元二十一年（733）以前的北路使臣多自此入境**；755 这一刻最近一次遣唐使（753 藤原清河等）取南路、登州港无遣唐使新事、但日本商舶 + 留学僧 + 留学生 往来仍多；
- **渤海国使臣 + 商舶**——**渤海国（698-926；粟末靺鞨大祚荣建国、唐先天二年/713 受册封为渤海郡王）**居唐东北、亦以登州为对唐主港之一；渤海岁有朝贡、自其上京龙泉府（今黑龙江宁安）南下经东京（今吉林珲春）出海、过辽东半岛南端 / 黄海 → 登州登陆；
- **中国海商**——自登州出港至辽东 + 朝鲜半岛 + 日本九州的近海贸；货物为绢帛 + 瓷器 + 茶 + 药材 + 书籍 + 金银器；返货为人参 + 海产 + 金 + 铁 + 麻布；
- **僧侣 + 文人**——汉传佛教自登州输入新罗 + 日本；新罗僧 + 日本僧亦自登州入唐、赴五台山 + 长安诸寺 + 江南名山；圆仁《入唐求法巡礼行记》838-847 记此类细节最详（**中唐文献、回看证据、不能现场指 755**）。

**新罗坊**——**唐时登州 + 楚州 + 泗州 + 莱州 + 扬州 一带的沿海州县出现新罗人聚居区、俗称"新罗坊"**；**新罗坊在唐宋之间渐次形成**——盛唐 755 时是否已成规模、文献无明载、**圆仁 838 巡礼时所见赤山法华院（长庆四年/824 张保皋立）+ 文登县新罗村**为中唐已成的样貌；**755 这一刻**——登州境内或已有新罗人聚居小村 / 散户、但不能预支张保皋 + 赤山法华院 + 大规模新罗坊的中唐之盛。

**张保皋**——**新罗人、9 世纪海贸领袖**——**唐穆宗朝（820-824）任武宁军小将**、后归新罗 + 建清海镇（828 立、在朝鲜半岛西南海岸；今韩国莞岛）、垄断 9 世纪东亚海贸；**长庆四年（824）于文登县赤山立法华院**——成为中晚唐东亚海路佛教 + 商货的枢纽。**张保皋事属 9 世纪、距 755 七十年、不能现场指 755**——但盛唐时登州一带已有新罗商舶 + 海商往来、为张保皋时代的预声。

**唐人对登州港的认知**：登州港在 755 时是河南道东端面海的物理 anchor + 北航路的起点；自登州登陆的新罗 + 日本 + 渤海使节经河南道陆行入两京、是盛唐"四夷宾服"图景的东海一角。**登州人对自家港湾的认知**——半岛北缘一处可泊船的湾里、舟出舟入有时、与隔海的新罗 + 日本 + 渤海国声息相通；**蓬莱仙山的传说**——千年不绝、登州人见海雾涌起 + 海市蜃楼时、犹想"三神山"。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九即起）；**天宝十四载十二月丁酉陷东京洛阳**（公历 756 年 1 月）——河南道核心州县先后陷；**登州 + 莱州 + 青州一带的胶东半岛**——叛军东进未深、登州港暂未陷；**至德 / 乾元间唐廷东方海贸 + 新罗 / 日本往来一度中断**——登州港的繁忙骤减；**广德元年（763）安史平定 + 中晚唐唐廷以江淮 + 河南为半壁**——登州港的海贸逐渐恢复。

### 此刻意味

对天宝十四载的东牟郡蓬莱县人民来说——登州港是**自家的海口**：日常渔舟出入 + 海商往来 + 偶有使节舸停泊；州治蓬莱县城南门外舟车辐辏、皆为入登州港之用。对河南道诸州的人民来说——登州港是**东出海的第一站**：货物经此出海至辽东 + 新罗 + 日本；新罗 + 日本 + 渤海使节经此登陆 + 西行入两京。对盛唐唐廷来说——登州港是**东方海路的物理 anchor**——与南端明州 + 苏州 + 楚州一带的海口共同构成"东海三大口岸";缺此一港、唐廷与新罗 + 渤海国的官方往来无以维系。

唐人尚不知道——天宝十四载十二月丁酉（公历 756 年 1 月）河南道核心州县将陷；登州港虽暂未陷、但**东方海贸自此停顿 + 至德 / 乾元间新罗 / 日本使臣往来锐减**；**至德 / 乾元间登州港的繁忙骤减、但作为唐廷半壁东海口岸仍存**；**中晚唐**——登州港随东亚海贸之盛而再起、**长庆四年（824）张保皋于文登县赤山立法华院**——登州 + 文登一带成为东亚海路佛教 + 商货的枢纽；**圆仁 838-847 巡礼时所见赤山法华院 + 文登县新罗村 + 登州海岸**——是中晚唐登州港之盛的最详细记录（《入唐求法巡礼行记》卷一至卷四）。

**今游人重访登州港遗址**——所见的是**山东烟台市蓬莱区** + **蓬莱阁山下旧城 + 明洪武水城 + 黄海岸线**——盛唐时的登州港物理形态早已不在、但作为海港地名仍在；**蓬莱阁**（北宋嘉祐六年/1061 始建）+ **蓬莱水城**（明洪武年间筑 + 备倭城）——是登州港在唐之后的物质遗存的两个高峰、**盛唐人未见此二者**；**今蓬莱港**——位于烟台市蓬莱区北岸、是渤海湾口的现代港口、与盛唐登州港在同一海岸走廊上、但细部位置略有迁移；**烟台**——明洪武三十一年（1398）于今烟台市芝罘区设奇山所 + 筑烟墩 + 因烟墩得名烟台；烟台市设立于 1937 年——**"烟台"非盛唐地名、不入 755 表述**。

## 子条目

### 蓬莱（三神山传说与县名沿革）

**蓬莱地名的来路**：**秦始皇二十八年（前 219）+ 三十二年（前 215）+ 三十七年（前 210）东巡至琅邪 + 望海中三神山**——蓬莱、方丈、瀛洲；齐人徐市等上书、言海中有仙山、仙人居之；秦始皇遣方士入海求三神山——**这是蓬莱地名进入中国文化想象的起点**（《史记·秦始皇本纪》+ 《史记·封禅书》）；**汉武帝元光二年（前 133）东巡海上望蓬莱**（《汉书·武帝纪》）——汉武帝多次东巡 + 派方士入海求蓬莱；**蓬莱县置**——汉武元光二年后于此一带置蓬莱县（《齐乘》异说；《汉书·地理志》东莱郡未列蓬莱县；《元和郡县图志》卷十一·登州蓬莱县条载置县沿革）。

**唐时蓬莱**——县名而非市名（"蓬莱市"是 1991 撤蓬莱县设市的现代名 + 2020 撤市设区）；**蓬莱县在 755 时是登州东牟郡治**——州治州县同城；县境包括今蓬莱区的大部分 + 海岸；**蓬莱阁**——**北宋嘉祐六年（1061）始建**、盛唐人未见此阁；**蓬莱水城（备倭城）**——明洪武年间筑、盛唐人亦未见。

**三神山的实景**——蓬莱海岸晴日东望、隐约可见庙岛群岛 + 长山列岛轮廓 + 偶有海市蜃楼；唐人入登州港、犹想"三神山"——是登州港的文化 anchor。

### 登州港（北航路起点）

**港口的形成**：**南北朝时**——已有自青州 + 登州出海至高句丽 + 百济 + 倭国的航路；**隋大业三年（607）**——隋炀帝遣使经登州出海至倭国；**唐承隋制**——登州 + 莱州一带是东出海的主要海口；**贞观 / 开元间**——日本遣唐使北路自登州 / 莱州登陆；**755 时**——登州港是新罗 + 日本 + 渤海国对唐往来的主港之一。

**港口的设施**：盛唐 755 时——州县官以平常州务管之、不立市舶（**市舶使开元二年/714 始见于安南, 广州后成主设地; 登州未设**）；官设水次 + 民间渔村码头并立；港中泊舟数十至百余艘；津吏 + 海商 + 渔户 + 应差舟丁。**港岸建置**——盛唐时尚为低丘 + 沙滩 + 几处天然湾澳；无后世明清水城。

**港口的地位**：登州港是**东海北航路的起点**——自登州出黄海 → 朝鲜半岛西海岸 → 新罗 → 日本九州；**与南端明州 + 苏州 + 楚州的南航路并立**——共同构成盛唐"东海三大口岸"；**与莱州港并立胶东半岛两港**——莱州在半岛西南面渤海湾、登州在半岛北角面渤海湾口 + 黄海北端。

### 北航路（登州→新罗→日本）

**北航路的来路**：**南北朝至隋**——自青州 + 登州出海至高句丽 + 百济 + 倭国的航路已开；**唐贞观四年（630）**——日本第一次遣唐使犬上御田锹至唐、取北路；**贞观至开元间七至八次北路**——登州 + 莱州一带是日本遣唐使主要登陆港。

**北航路的具体走法**：**自难波（今大阪）出发 → 北九州博多 → 对马海峡 → 朝鲜半岛西海岸 → 新罗西海岸瓮津 / 南阳湾 / 唐恩浦 → 横渡黄海 → 山东登州 / 莱州登陆 → 陆行经河南道入两京**；全程沿岸航行 + 不大涉远海、相对安全；缺点是经新罗 + 高句丽境、政治风险大（高句丽 668 年灭后此风险消）。

**南北路的转换**：**开元二十一年（733）多治比广成第九次遣唐使**为最后一次明确北路；**开元二十二年（734）以后日本遣唐使转南路**——自难波 → 五岛列岛 → 直渡东海 → 长江口 / 杭州湾登陆（明州 / 苏州 / 扬州一带）；南路风险大但避开新罗 + 高句丽境；**天宝十二载（753）藤原清河 + 吉备真备 + 鉴真东渡**为南路；**755 这一刻**——日本遣唐使刚走完一次南路 + 北路一度沉寂；**中晚唐承和五年（838）圆仁随藤原常嗣第十六次遣唐使**——南北兼用 + 终在登州一带登陆 + 圆仁开始巡礼。

**北航路的盛唐意义**——是登州港的灵魂 + 唐与新罗 + 日本 + 渤海国官方往来的物理通道；缺此一路、唐廷与东海三国的声息断。

### 新罗坊与张保皋（中晚唐预声）

**新罗坊**——**唐时登州 + 楚州 + 泗州 + 莱州 + 扬州 一带的沿海州县出现的新罗人聚居区**；**形成时段**——盛唐至中晚唐渐次形成、**盛唐 755 时尚未成规模 / 文献无明载**；**圆仁 838 巡礼时所见赤山法华院 + 文登县新罗村**为中唐已成的样貌；**新罗坊在唐宋之间渐次形成 + 至中晚唐定型**——是东亚海路民间往来的物理印记。

**张保皋（弓福；?-841/846）**——**新罗人、9 世纪海贸领袖**；**唐穆宗朝（820-824）任武宁军（治徐州）小将**；归新罗 + **828 立清海镇**（朝鲜半岛西南海岸；今韩国莞岛）——垄断 9 世纪东亚海贸；**长庆四年（824）于文登县赤山立法华院**——成为中晚唐东亚海路佛教 + 商货的枢纽；圆仁 838-847 巡礼时多次至赤山法华院（《入唐求法巡礼行记》卷二）。

**张保皋事属 9 世纪、距 755 七十年、不能现场指 755**——但盛唐 755 时登州一带已有新罗商舶 + 海商往来、为张保皋时代的预声；唐人对登州港的"东海枢纽"印象、是盛唐至中晚唐渐次累积的。

### 渤海国与登州（东北航路）

**渤海国（698-926）**——**粟末靺鞨大祚荣建国**；**唐先天二年（713）**唐玄宗册封大祚荣为左骁卫大将军 + 渤海郡王 + 忽汗州都督；**唐开元 / 天宝间**渤海国岁有朝贡 + 海陆兼程；**海路**——自渤海上京龙泉府（今黑龙江宁安渤海镇）南下经东京龙原府（今吉林珲春）出海、过辽东半岛南端 + 黄海北部、至**登州登陆**——再陆行经河南道入两京。

**唐玄宗与渤海**——开元二十年（732）渤海王大武艺遣使越海袭登州 + 杀登州刺史韦俊（《新唐书·渤海传》）——这是**渤海与唐之间唯一一次大规模军事冲突**、登州港正是冲突的物理 anchor；**开元二十一年（733）唐遣大将葛福顺 + 太仆员外卿金思兰（新罗人）讨渤海**；**渤海与唐 736 年后重归宗藩**——755 时渤海岁贡仍以登州为主港。

**登州在唐东北经略中的地位**——是与渤海国官方往来的主港 + 唐东方海防的物理 anchor；**开元二十年的登州袭事**——是登州港史上最戏剧性的军事 moment（与瓜洲渡的永王李璘事、广州的黄巢屠城 类比）。

### 莱州港（胶东两港之另一）

**莱州港**——河南道莱州东莱郡治掖县（今山东莱州市）一带的海港；**与登州港并立胶东半岛两港**——**莱州在半岛西南面渤海湾、登州在半岛北角面渤海湾口 + 黄海北端**；**州治掖县**距海岸约二十唐里、港在掖县北郊海岸；**莱州港 vs 登州港的分工**：**莱州港主面渤海湾内的近海贸 + 河北道沿海州县往来**；**登州港主面黄海 + 朝鲜半岛 + 日本 + 渤海国的远海贸**——分工自然形成、未见 唐廷明诏规定。

**莱州港在 755 时**——按例运行、是河南道东端的副港、与登州港共同构成胶东半岛海运格局；**莱州港作为独立 POI 另立 doc**——本文聚焦登州港本身。

## 历史长河（755 → 今）

**中晚唐**　**755-763 安史中**——登州港暂未陷叛军、但东方海贸停顿 + 新罗 / 日本使节往来锐减；**广德元年（763）后**——东方海贸渐恢复；**长庆四年（824）张保皋于文登县赤山立法华院**——登州 + 文登一带成为东亚海路佛教 + 商货的枢纽；**承和五年（838）圆仁随第十六次遣唐使至唐**——南路至扬州 + 后取北路自登州 / 文登归日；**圆仁《入唐求法巡礼行记》卷一至卷四**记登州 + 文登 + 赤山 + 新罗村 + 海路细节最详——是中晚唐东亚海路的第一手记录。

**五代 + 北宋**　五代时登州属梁 + 唐 + 晋 + 汉 + 周诸朝、登州港的海贸时盛时衰；**北宋登州属京东东路 + 嘉祐六年（1061）始建蓬莱阁**——蓬莱阁是**北宋至清代登州港的文化地标 + 山东四大名楼之首**；**北宋登州港**——与高丽 / 辽 + 后金 (女真) 的海路往来均经此港；**宋宁宗嘉定二年（1209）**登州升为防御州。

**金 + 元**　**金登州属山东东路 + 元属般阳路**；**元代登州港**——是元与高丽 + 日本海运的主港之一；**元代北方海运（自江南至大都）**亦经登州港中转；**元末**登州港随战乱凋敝。

**明 + 清**　**明洪武九年（1376）设登州卫 + 洪武年间筑蓬莱水城（备倭城）**——明代最重要的北方海防要塞；**戚继光（1528-1588）生于登州**——明代抗倭名将；**清雍正十三年（1735）升登州府**；**清代登州港**——是与高丽 + 朝鲜 (李朝) + 日本海运的主港之一；**清末甲午战争（1894-95）**登州一带为战场之一；**蓬莱阁 + 蓬莱水城**经多次重修 + 是清代登州的文化地标。

**民国至今**　民国废登州府；**1949 年后**——登州港逐渐被现代港口取代；**1983 年蓬莱县属烟台地区**；**1991 年撤蓬莱县设蓬莱市**；**2020 年撤蓬莱市设烟台市蓬莱区**；**今蓬莱港**——位于烟台市蓬莱区北岸、是渤海湾口的现代港口 + 与盛唐登州港在同一海岸走廊上但细部位置略有迁移；**蓬莱阁** + **蓬莱水城** + **戚继光故里**——是今游人重访登州港遗址时所见的主要遗存；**烟台**——明洪武三十一年（1398）于今芝罘区设奇山所 + 筑烟墩 + 因烟墩得名 + 烟台市设立于 1937 年——**"烟台"非盛唐地名、不入 755 表述**。

**今游人重访登州港遗址**——所见的是蓬莱阁山下旧城 + 明洪武水城 + 黄海岸线 + 蓬莱阁 + 戚继光故里——盛唐时的登州港物理形态早已不在、**蓬莱阁**（北宋嘉祐六年/1061）+ **蓬莱水城**（明洪武年间）盛唐人皆未见——但作为东海北航路起点 + 三神山传说之地 + 唐与新罗 / 日本 / 渤海国往来的物理 anchor、登州港在中国海洋史 + 东亚文化交流史中的地位千年不衰。
