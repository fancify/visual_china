# 殷墟

## 一句话

**商代晚期王畿遗址**——洹水南岸、相州治安阳县西北一片郊原；**盘庚迁殷（约 BC 1300）至帝辛（纣）亡国（约 BC 1046）约 250 余年间为商王都**（若采《竹书》系文献"273 年"之说、则迁殷年代相应早于 BC 1300、本 doc 不同时坐实两者）；**755 时已废约 1800 年**——但**唐人不知此地即殷墟**：甲骨文 1899 才发现、唐人凭《尚书》《史记》多以为殷之故都在卫州朝歌——盛唐相州人在洹水边耕作、不识脚下是商王陵区。

## 形貌

殷墟在唐河北道**相州邺郡治安阳县**境——具体在**洹水南岸的小屯村一带**（今河南省安阳市殷都区小屯村 + 西北冈一带）。**地势**：太行山东麓冲积平原、洹水自西北向东南流过——海拔约 80 米；周边为黄土淤积平原、易耕作；自商末以下千八百年已成农田 + 村落混合景观、地表无显著城墙或宫阙遗存。

**与相州 / 邺郡治所的相对位置**：
- **相州治安阳县**（今河南安阳市区）——盛唐 755 时邺郡太守治安阳县（天宝元年 742 改州为郡之后只设郡太守、不并列"刺史"）；殷墟在安阳县城西北郊约 5 唐里许（具体里数本 doc 不坐实、唐人无此记载）。
- **邺县**（今河北临漳县邺镇一带）——是相州属县、邺郡郡名所本；**邺县在安阳东北约 30-50 里**——是曹魏邺都、东魏 / 北齐邺都所在；与殷墟不为同一地理点。
- **注**：天宝元年（742）改州为郡之制下、相州 = 邺郡；治所在安阳县、不在邺县。**唐人言"邺"时指向多歧**：或指邺郡 / 相州、或指邺县 / 曹魏邺都 / 北齐邺都——但**几乎无人指向商殷故都**。

**商殷王畿的物理范围**（今考古勘察）：殷墟核心区东西约 6 公里、南北约 4 公里——含**宫殿宗庙区**（小屯村东北、洹水南岸）+ **王陵区**（西北冈、洹水北岸）+ **族邑居住区**（小屯村周边）+ **手工业作坊区**（青铜、骨器、玉石）+ **甲骨窖藏区**。**这一切在 755 时尚未为人知**——唐人见到的是一片洹水冲积平原、间或耕地中出土"古铜器"或"龟甲" 而不识为何物。

物候：太行山东麓平原、温带季风气候——夏热多雨、冬冷干燥；洹水四时有水、是相州一带的重要灌溉源；周边为农田 + 桑林 + 村落、是河北道腹地最熟悉的乡野景观之一。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

殷墟在唐人的回望里——是**一片空白**。这是本 doc 的核心张力。

**商王朝晚期的史实**（盛唐人据《尚书》《史记》《竹书纪年》等可知）：
- **盘庚迁殷**（约 BC 1300、《尚书·盘庚》三篇）——盘庚自奄（一说自亳、史源有异）迁都于殷；自此商王朝以"殷" 为都至亡国约 250 余年（《竹书》系文献作"273 年"——若取此数则迁殷应早于 BC 1300、本 doc 不同时坐实两者）。
- **武丁中兴**（约 BC 1250-1192 在位）——商代中期繁荣顶点；武丁与王后妇好并世；妇好曾领兵征伐羌方、土方等。**唐人据《史记》知武丁** 为商之贤君；不知妇好。
- **帝辛（商纣王、商纣同人）**——商末王、约 BC 1046 武王伐纣 / 牧野之战亡国。**唐人据《史记·殷本纪》《尚书·牧誓》深知帝辛之暴**——酒池肉林、炮烙、剖比干心 等叙事。
- **武王伐纣**——年代史源争议大；今夏商周断代工程定为 **BC 1046**；唐人据《竹书纪年》或《史记》各有所本、未定年。

**唐人对"殷之故都"地望的认识**：
- **《史记·殷本纪》**：盘庚迁殷、自此商至亡 273 年——但《史记》未明指"殷"之具体方位。
- **《竹书纪年》古本残文**：殷自盘庚至帝辛凡七世、八徙——商都迁徙颇繁。
- **《吕氏春秋·慎大览》**：殷之故墟在"邺"、朝歌一带——但"邺"是宽泛地域。
- **盛唐学者通识**：**殷之故都多指卫州卫县朝歌**（今河南淇县）——以《尚书·牧誓》"武王伐纣、戎车三百乘、虎贲三百人、与受战于牧野" + 牧野在朝歌郊外——而朝歌即殷末都。
- **安阳一带在唐人认知中**：是**曹魏邺都 + 东魏 / 北齐邺都**所在——是建安文学的舞台、是六朝故都；**而非商殷故都**。

**这是一个有趣的认知错位**——盛唐人在相州 / 邺郡的怀古，多面向**曹操邺城铜雀台 + 北齐高氏邺宫**；殷之故都在唐人心中是模糊的、指向卫州朝歌的可能性远高于安阳。**洹水南岸的小屯一带**——唐人耕作其上、间或锄出"古铜器"或"龟甲"——但不识为何物。

### 此刻

天宝十四载这一年的殷墟——**已废约 1800 年**（BC 1046 → AD 755、公元无 0 年、合计约 1800 年）；以**完全不被识别的商代王畿遗址**身份存在于相州安阳县境。

**地表景观**：洹水南岸一片农田与村落、间有土丘起伏（即今所知之宫殿基址、王陵封土）——但盛唐人不知这些土丘的来历；多归为"古墓"或"无名土堆"。**间或出土物**：唐时田间偶有锄出商代青铜器 + 龟甲 + 玉器之记录——但史源零散、本 doc 不坐实具体出土事件。

**唐人对此地的认知**：
1. **官方记载**——《元和郡县图志》卷十六·河北道·相州条 详载相州沿革、属县、户口、风俗——但**不载"殷墟"二字**。安阳县下亦无殷都之记。
2. **文人怀古**——盛唐相州 / 邺郡的怀古诗多咏曹魏邺都 + 北齐邺宫 + 铜雀台；**几乎无人凭吊商殷**（凭吊商殷的诗作多投向卫州朝歌方向）。
3. **百姓口传**——相州一带或有"古商邑" 之类的模糊传说；但本 doc 不坐实。

**这一年的相州安阳县** 处于**唐玄宗朝末年河北道腹地州城**的稳定状态——再过半年安禄山起兵于范阳、自范阳南下；**755 末至 763 安史八年间相州反复易主**——是河北道战场之一；**乾元二年（759）邺城之战**（约二十万唐军围邺、号称六十万、败于史思明）即在此地——但唐军 + 叛军间反复争夺的是**邺城（北齐邺都旧址、邺县）**、不是殷墟。**殷墟仍在洹水南岸默默躺着、不被识别**。

### 此刻意味

对天宝十四载的相州 / 安阳人来说——洹水南岸的小屯一带就是**自家村边的耕地**：田垄、桑林、间或起伏的土丘——是熟悉的乡野景观。对长安那一代士人来说——相州 / 邺郡是**曹魏邺都 + 北齐邺都的物理纪念**：建安风骨的舞台、二乔铜雀的故园——是河北怀古最浓厚的几座城之一；**但商殷的物理 anchor 在唐人心中是空缺的**——商代王畿在文献里清晰、在地理上模糊。

这是盛唐怀古文学最有意思的"盲点"之一——唐人能在邺城遗址前抚铜雀台残瓦、能在洛阳邙山前凭吊汉魏帝陵、能在长安咸阳原前凭吊秦汉陵阙——但**走到洹水南岸的小屯村**、不知脚下即是 1800 年前商王武丁、妇好、帝辛的王畿。**这是史官 + 文献 + 物理证据之间的巨大错位**——是中国早期史实物理 anchor 长期失落的一个典型案例。

唐人尚不知道——再过 1144 年（1899）清光绪二十五年、**王懿荣在京师药铺识得"龙骨"上的刻文为商代文字**——是甲骨文发现的开端；自此安阳小屯一带的"龙骨"被有意识收集、释读。再过 1173 年（1928）**国立中央研究院历史语言研究所考古组**由董作宾 + 李济 等启动**殷墟科学发掘**——自此殷墟由"文献中的殷之故都" 转为"考古实证的商代王畿"。再过 1221 年（1976）**中国社会科学院考古研究所安阳工作队郑振香等发掘小屯村西北 M5——即妇好墓**——出土青铜礼器 + 玉器 + 骨器约 1928 件、是商代物质文化最完整的一次发掘。再过 1251 年（2006）殷墟列**联合国教科文组织世界文化遗产**。**唐人在 755 所未识的那片洹水南岸耕地**——千二百多年后是中国考古学的发祥地、是商代文明最重要的物理 anchor。

## 子条目

### 商王朝最后的都城（盘庚迁殷 + 约 250 余年）

**盘庚迁殷**（约 BC 1300）——是商王朝最重要的一次都邑迁徙；自此商至亡约 250 余年皆都于殷（《竹书》系文献作"273 年"；本 doc 取 BC 1300 → BC 1046 约 250 余年这一口径，不同时坐实"BC 1300" 与"273 年"两数）。**《尚书·盘庚》三篇**——记盘庚迁殷前对臣民的诰命；是中国最早的政治文献之一。**殷之名**——商人卜辞多自称"大邑商"或"商邑"、不自称"殷"；"殷"为周人对商之称、含贬意（一说"殷"即"衣"、周人以衣指商）；**今"殷墟"二字**——出自《史记》"周公曰：殷之故墟"之类的早期文献。

**商都于殷约 250 余年间的主要王**：
- **盘庚**（迁殷者）
- **武丁**（中兴）
- **祖庚 + 祖甲**
- **武乙**（"射天" 故事）
- **文丁**
- **帝乙**
- **帝辛（纣）**——亡国王

**这约 250 余年间** 商王朝在洹水南北经营宫殿、宗庙、王陵、族邑、手工业作坊——是中国早期文明物质遗存最完整的一段。**唐人据《史记·殷本纪》知此一段的王世系**——但不知都邑物理位置即在相州安阳县境。

### 武丁中兴与妇好

**武丁**（约 BC 1250-1192 在位）——商代中期最贤明的王；史载"武丁中兴"——商王朝在其手中达鼎盛。**《史记·殷本纪》载武丁夜梦"傅说"**——觉而求之于野、得说于傅险（傅岩）—— 拜为相、商道大兴。**武丁亦有大规模征伐羌方、土方、鬼方等周边方国** 的卜辞记录——但这些卜辞 755 时尚未为人知。

**妇好**——武丁王后、商代最有名的女性。**《史记》《尚书》皆不载妇好**——唐人不知此人；**1976 年妇好墓发掘**后才据墓中出土青铜器铭文（约 109 件器铭"妇好"二字）确认其人。**妇好领兵征伐**（卜辞记载妇好曾领兵 13000 人征羌方——是商代规模最大的一次军事行动之一）+ **妇好主祭祀**——是商代少有的有完整物质实证的女性政治人物。

**注**：本子条目所述武丁中兴、妇好领兵 等史实——大部分依赖 20 世纪甲骨文释读 + 妇好墓考古；**755 时唐人仅能从《史记·殷本纪》知武丁"中兴"两字、不知其具体内容**。本子条目以"今所知" + "唐时所未知" 双重时态写——是 doc 的核心张力。

### 帝辛与牧野之战

**帝辛**（商纣王、商纣同人——三者同一人、不同语境）——商末王；约 **BC 1046 武王伐纣 / 牧野之战** 中亡国。**夏商周断代工程（1996-2000）定武王克殷于 BC 1046**——是今学界主用年代；早年有 BC 1027（《竹书纪年》古本一说）+ BC 1122（古传一说）等说、本 doc 取 BC 1046（标"夏商周断代工程"）。

**唐人据《史记·殷本纪》《尚书·牧誓》深知帝辛**——但唐人所知的帝辛是**"暴君叙事下的帝辛"**：
- **酒池肉林**——《史记》原文；
- **炮烙之刑**——《史记》《列女传》载；
- **剖比干心**——《史记》载；
- **囚箕子**——《史记》载；
- **宠妲己**——《史记》载。

**这些叙事**——是周人对商末王的政治评价、唐人继之；今学界对帝辛的评价更复杂（认为帝辛征伐东夷、扩商朝版图至淮河—— 但因东征过度耗损、为周所乘）。**牧野之战**——武王伐纣的决定性战役；《尚书·牧誓》载之；**牧野位置一般定在朝歌郊外**（今河南淇县南）——而非安阳殷墟。**这进一步加深唐人"殷之故都在朝歌" 的印象**。

### 唐人对殷墟的认知（一片空白 + 朝歌印象）

**这是本 doc 最重要的子条目**——唐人对"殷之故都"的物理认知与今所知错位巨大。

**唐人认知的主要来源**：
1. **《尚书·盘庚》**——盘庚迁殷叙事；但《盘庚》不明指地望。
2. **《史记·殷本纪》**——商王世系 + 主要事件；但《殷本纪》不明指"殷"之具体方位。
3. **《竹书纪年》**——殷王世系 + 都邑迁徙；古本残文颇异；唐时已不完整。
4. **《吕氏春秋·慎大览》**——殷之故墟在"邺"、朝歌一带；"邺"为宽泛地域。
5. **《尚书·牧誓》**——牧野之战；牧野在朝歌郊外（今淇县南）。

**唐人通识结论**：**殷之故都在卫州卫县朝歌**（今河南淇县）——这是盛唐至中晚唐学者的主流看法。**李白、杜甫、王维 等盛唐诗人** 若有"殷墟"或"殷之故都"叙事——多投向朝歌方向；安阳一带的怀古诗几乎不涉及商殷。

**相州 / 邺郡的怀古主调**：
- **曹魏邺都**（曹操 204 取邺为本据；建安文学的舞台）；
- **铜雀台**（曹操 210 筑；建安文学的物理 anchor）；
- **东魏邺都**（534-550）+ **北齐邺都**（550-577）；
- **北齐高氏宫殿、佛寺、北响堂山石窟**等。

**这些怀古主题** 是盛唐相州 / 邺郡文人最熟悉的；**商殷在此地的物理存在** 完全不在唐人视野——是中国早期史实物理 anchor 长期失落的一个典型案例。

### 殷墟与相州 / 邺郡的关系

**相州治安阳县**——盛唐 755 时邺郡太守治安阳县（天宝元年 742 改州为郡之后只设郡太守、不并列"刺史"）；安阳县治在今安阳市区；殷墟在安阳县城西北郊。

**邺县**——相州属县、邺郡郡名所本；位于今河北临漳县邺镇一带；是曹魏邺都 + 东魏 / 北齐邺都所在；与殷墟分属两个地理点（**邺县在殷墟东北约 30-50 里**——具体里数本 doc 不坐实、唐人无此明确记载）。

**唐人言"邺"时**：
- 指**邺郡**（= 相州、天宝元年 742 改名）—— 治安阳；
- 指**邺县**（相州属县）—— 是曹魏 + 北齐邺都；
- 指广义的"古邺地"—— 含安阳 + 临漳一带；
- **不指**商殷故都。

**这种地名错位** 是 755 时殷墟"完全不被识别"的另一层原因——商殷的物理 anchor 在唐人"邺"的语义之外；唐人"邺"主指曹魏 + 北齐邺都。

### 1899 甲骨文发现至今 殷墟的"被发现"史

**1899 王懿荣识药铺"龙骨"为商代刻辞** —— 是殷墟"被发现"的开端；自此小屯村"龙骨"被有意识收集、释读。**王懿荣 1900 庚子之变中殉国**——其所藏龙骨归刘鹗；**刘鹗 1903 《铁云藏龟》**——首本甲骨文著录。

**1928 国立中央研究院历史语言研究所考古组** 由董作宾 + 李济 等启动**殷墟科学发掘**——是中国考古学的开端；自 1928 至 1937（抗战中止）共发掘 15 次；**1928-1937 间发掘** 含**小屯村宫殿宗庙区 + 西北冈王陵区** + 大量甲骨 + 青铜器。**李济 + 董作宾 + 梁思永 等** 是中国考古学第一代主将。

**1950 后中国科学院（后中国社会科学院）考古研究所安阳工作队**——延续殷墟发掘至今 80 余年。**1976 妇好墓发掘**（小屯村西北 M5、郑振香主持）——出土 1928 件商代器物；**1990s 后陆续发掘洹北商城**（殷墟外围一座更早的商代都城、可能为盘庚迁殷的初都）。**2006 殷墟列联合国教科文组织世界文化遗产**——是中国早期文明最重要的物理 anchor。

## 历史长河（755 → 今）

**中晚唐**　**756-763 安史之乱中相州反复易主**——**乾元二年（759）邺城之战**（约二十万唐军围邺、号称六十万、败于史思明）即在相州；**但唐军 + 叛军间反复争夺的是邺城（北齐邺都旧址、邺县）**、不是殷墟。**殷墟仍在洹水南岸默默躺着、不被识别**。中晚唐相州 / 邺郡 的怀古诗一如盛唐——多咏曹魏邺都 + 北齐邺宫；商殷不在视野。

**两宋至元**　**北宋初相州改安阳府**——治安阳县；殷墟一带仍是农田。**宋金间金石学兴起**——欧阳修《集古录》、赵明诚《金石录》 等收录古代铜器铭文、含商代器——但宋人多不知商代铜器具体出土地；**安阳一带间或出土商代铜器** + 卜骨——但宋人或归为"古器物"、不识为"殷之故都遗物"。**金元间**安阳为彰德府治；殷墟一带继续作为农田 + 村落。

**明清**　**明清间安阳为彰德府治**；殷墟一带继续作为农田。**明清地方志**（如《彰德府志》《安阳县志》）有"古殷墟"二字之零散记载——但具体物理范围、文物归属皆不明。**清乾隆 + 嘉庆间**安阳小屯一带间或出土商代青铜器 + 龟甲——多被当作"古铜器"或"龙骨" 入药铺；**直至 1899 王懿荣识"龙骨"为商代刻辞**——殷墟方由"地方志中的模糊地名" 转为"实证的商代王畿"。

**民国至今**　**1928 国立中央研究院历史语言研究所考古组启动殷墟科学发掘**——是中国考古学的开端；**1928-1937 间 15 次发掘**；**1950 后中国社会科学院考古研究所安阳工作队延续发掘至今**；**1976 妇好墓**；**1990s 洹北商城**；**2006 列联合国教科文组织世界文化遗产**。**今游人来安阳殷墟博物苑**——所见的是**宫殿宗庙区遗址 + 妇好墓 + 王陵区西北冈 + 甲骨文展厅**——**盛唐相州人在 755 所走过的那片洹水南岸耕地**——千二百多年后是中国早期文明最重要的物理 anchor；**而 755 时这一切尚在地下、唐人不识**——这是中国早期史实物理 anchor 失而复得的最大的一例。
