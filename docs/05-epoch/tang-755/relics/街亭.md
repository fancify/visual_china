# 街亭

## 一句话

秦州治上邽东北 / 北侧陇城县境的陇坻关道——**蜀汉建兴六年春、马谡舍水上山失街亭于张郃**；诸葛亮一出祁山自此功败、退军汉中、夏戮马谡以谢众——是中国军法严明叙事的物理 anchor，与五丈原同为诸葛亮悲剧的两端。

## 形貌

街亭在陇右道**秦州天水郡治上邽东北 / 北侧陇城县境**——**陇山西麓、略阳川北岸**，是一处当道扼守的关隘亭障。**今属甘肃天水市秦安县陇城镇一带**（《元和郡县图志》卷三十九陇右道秦州条记"街泉故城在陇城县"；唐时陇城县属秦州、街亭即在陇城西郊）。

地势：东接**陇坻**（陇山主脉西麓）、西通**略阳川**（清水河支流）河谷、北依**陇山北支**、南瞰**渭水北岸支流**——四面为山所夹的一处东西向河谷通道。**街亭本意是"街泉亭"**——"街"为地名、"亭"是汉魏军事建制（十里一亭）；故其本名"街泉亭"、《三国志》原文如是；后世简称"街亭"乃约定俗成、二名通行至今。**此处的地理意义**：是关中入陇右后、自陇山西麓南下取秦州（上邽）、北上取略阳的咽喉一段——**陇右地形上的关道交叉点**。

亭障本身——汉魏间为军事亭、唐时已为乡级聚落（陇城县下属），北接陇山牧场、南连渭水河谷。物候：陇坻一带海拔约 1500 米、四时分明、冬寒早雪、夏短秋长；山中多松柏、川中多麦粟——是关陇间典型的高原河谷小镇。

**关于地名辨**：街亭、上邽、祁山三处皆在秦州境、但不同——**祁山**在祁山堡（今甘肃礼县祁山镇）秦州西南、是蜀军 228 北伐自汉中出陇右的"前进基地"；**上邽**是秦州治所（今甘肃天水）秦州城所在；**街亭**在秦州治上邽东北 / 北侧陇城县境（今秦安县陇城镇）秦州北境。三地呈三角分布、各扼陇右一隅——盛唐读《三国志》者皆需辨。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

街亭在唐人的回望里是**诸葛亮一出祁山失败的转折点、马谡违节度致军溃、张郃断汲道大破之、街亭失即陇右失**——**距 755 已 527 年**。

**蜀汉建兴六年（魏太和二年；228）春**——诸葛亮**第一次北伐**自汉中出兵。**扬声由斜谷道取郿、使赵云、邓芝为疑兵据箕谷**；**亮身率诸军攻祁山**——**蜀汉 228 一出祁山即此**（《三国志·诸葛亮传》"亮身率诸军攻祁山、戎陈整齐、赏罚肃而号令明"）。**南安、天水、安定三郡叛魏应亮**——蜀军势头一时甚锐、陇右大震。

**魏明帝亲征至长安**——**遣右将军张郃督诸军拒亮**（张郃官品本 doc 严守"右将军"口径——非"魏国大将军"；张郃此役以陇右行军总管身份西出、本传："郃督诸军、拒亮将马谡于街亭"）。**诸葛亮违众拔马谡督诸军在前、与郃战于街亭**——这是诸葛亮第一次北伐用人最具争议的一着。

**马谡（190-228）**——襄阳宜城人、**马良弟**（马氏五常之末、白眉马良为兄；本 doc 严守"马谡 = 马良弟"口径——不要把马良与马谡混为一人）；**才器过人、好论军计**——刘备临终曾谓诸葛亮："马谡言过其实、不可大用、君其察之"；**诸葛亮素器异之**、北伐前为参军、违众用之督街亭。

**街亭之战的关键**：**马谡舍水上山、不据当道扎营**——《三国志·马谡传》"谡舍水上山、举措烦扰"；**裨将王平连规谏谡**——谡不能用（《三国志·王平传》"连规谏谡、谡不能用、大败于街亭"）。**张郃绝其汲道、击、大破之**——蜀军溃、士卒离散。**街亭失守**——陇右关道即开、张郃魏军可长驱直入秦州。**亮无所据**——南安、天水、安定三郡叛魏者复降；**亮拔西县千余家、还于汉中**（《三国志·张郃传》——西县即今甘肃礼县红河乡一带、亮拔其民南迁汉中）。**一出祁山自此功败**。

**夏五月**——**诸葛亮收马谡下狱、戮之以谢众**（《三国志·诸葛亮传》《马谡传》本传）。**"挥泪斩马谡"是后世文学化用**——正史原文仅"戮谡以谢众"；**"挥泪"二字、出《三国演义》及南北朝以下杂史**、本 doc 严守"正史无挥泪"口径。但**马谡死时三十九岁、与诸葛亮相交甚厚、亮自请贬丞相为右将军三级**——是中国军法史上最沉重的一次执法。**自此后**——蜀汉建兴七年（229）二出（陈仓道）、九年（231）三出（卤城）、十二年（234）五出**至五丈原**——亮卒。**一出祁山失街亭、是诸葛亮北伐五次中最痛的一次失败**。

**自三国魏蜀皆亡后**——街亭作为"马谡失节、孔明斩谡"的物理 anchor、在陇右地理志中留名。**北魏《水经注·卷十七·渭水》**记"街泉亭故城"——是地名延续的早期史源。**唐《元和郡县图志》卷三十九陇右道秦州条**记"街泉故城在陇城县"——是唐人对街亭地望的明确指认。

### 此刻状态

天宝十四载这一年的街亭——**陇坻关道边一处普通乡镇**。

街亭已不复汉魏军事亭障的建制——盛唐时为**秦州陇城县下属一处聚落**（陇城县治在今秦安县陇城镇）。**故亭故城遗迹**——《元和郡县图志》记"街泉故城"在陇城县境、规模史载未详；**盛唐时是否有具名"马谡故垒""孔明斩谡台"等纪念建筑、本 doc 不预设**——这些具体物事多是宋元以下方志附会、不是 755 时所见。**755 时的街亭**：山在、川在、陇坻关道在、街泉故城地望可指——但作为"诸葛亮一出祁山失败地"的纪念性，多在士庶口耳之间，物理纪念物不显。

**秦州（治上邽）在 755 时归唐辖**——**陇右节度使治鄯州（今青海乐都）**、秦州为陇右道属州、是关中通陇右的咽喉。**哥舒翰（约 699-757）以陇右节度使兼河西节度使**屯重兵于河西陇右、声威甚盛——**天宝十二载（753）破吐蕃于洪济、九曲，唐与吐蕃陇右势力此刻处唐攻势的高峰**。秦州一带 755 年仍是**唐治、唐军、唐税、唐邮**的正常运转区——**街亭所在的陇城县**亦不例外。

**盛唐人凡过秦州陇道者**——多有访街亭故地、凭吊马谡失节、孔明斩谡的典故。陇右驿路自长安出潼关入岐州、循渭水西行至秦州、再北行入陇道——**街亭即在秦州北至陇州的驿路中段**。盛唐边塞诗中咏陇头、陇坻者甚众（"陇头流水、其声幽咽"）——但**专写街亭凭吊的盛唐诗作不多**；**杜甫的秦州诗**（《秦州杂诗二十首》乾元二年/759 秋客秦州作）写陇右风物甚详——**但杜甫秦州行在 759 年、不在 755 现场**、本 doc 严守此口径。

### 此刻意味

对天宝十四载的秦州陇城县人来说，街亭是**县西一处"诸葛丞相一出祁山所失"的故亭**——日常田耕、岁岁有过陇道者凭吊。对盛唐文人来说——街亭是**"军法严明 + 用人不当"两端典故的物理 anchor**：诸葛亮违众用马谡、马谡违节度致军溃、孔明戮之以谢众、自贬三级——这一段是中国君臣赏罚叙事的极致。**与五丈原合观**——失街亭是诸葛亮北伐五次中最大的一败、失街亭后六年诸葛亮死于五丈原；**唐人凭吊街亭即是凭吊"蜀汉北伐之业败垂成"**——这是街亭在盛唐 755 时的核心意义。

唐人尚不知道——再过半年安禄山起兵、再过两年（757）吐蕃乘安史之乱东进、**广德元年（763）秦州陷吐蕃**——**自此街亭一带为吐蕃辖地近九十年**（至大中三年/849 唐复秦州）。**再过四年（759 秋）杜甫弃华州官、自秦州入陇道、过街亭一带、再南下入蜀**——**杜甫《秦州杂诗》《赤谷》《铁堂峡》《盐井》《寒峡》《法镜寺》《青阳峡》《龙门镇》《石龛》《积草岭》《泥功山》《凤凰台》一组陇蜀纪行诗、是中国文学史上最重要的陇右纪行**——但作于 759 年、不在 755 现场。再过千年——街亭故地会陆续立"诸葛庙""马谡墓""街亭古战场"等纪念物事——多为明清以下方志附会。**唐人在 755 所见的那片陇坻关道**——山在、川在、街泉故城地望在；但具体纪念物事是后世累积的、不是 755 时所见。

## 子条目

### 街泉亭故城

陇城县境（今甘肃天水市秦安县陇城镇）——**汉魏军事亭障的故址**。"街"为地名、"泉"为水名、"亭"是汉制十里一亭——三字合而为军事建制名。**《水经注·卷十七·渭水》**记"略阳川"一带有"街泉故城"——是地名延续的早期史源；**《元和郡县图志》卷三十九陇右道秦州条**"街泉故城在陇城县"——是唐人指认的明确出处。**盛唐时故城规模史载未详**——本 doc 不预设盛唐街泉故城已成纪念地标；今所见街亭古战场纪念地多为明清后陆续修建。

### 马谡舍水上山处

街亭城南一处山地——**传为马谡 228 春违令舍水上山扎营的地点**。《三国志·马谡传》"谡舍水上山、举措烦扰"——**正史无具体山名**；**今陇城镇南有"马谡崖""断山"等地名**、多为后世附会、史源不出唐宋。**盛唐 755 时是否已立具名纪念点、本 doc 不预设**。**这一山地的意义**：是中国军事史上"违节度、违地形"的经典反面教材——后世兵书多引此为戒。

### 张郃断汲道（魏军绝水处）

街亭山下一处水道——**传为张郃 228 春断马谡军汲道之处**。《三国志·张郃传》"郃绝其汲道、击、大破之"——**正史无具体水源名**；**今陇城镇南略阳川一支流**、地方志或附会为当年汲道。**张郃（？-231）**是魏五子良将之末、**228 时为右将军**（本 doc 严守此口径——非"魏国大将军"；张郃后于建兴九年/231 木门道追蜀军中流矢卒），断水一着定街亭胜负——是其平生最重要的一次战功。**盛唐人来街亭凭吊**——亦有念及张郃此役。

### 王平规谏处（蜀军裨将谏不从）

街亭蜀军营中——**王平（？-248）当时为马谡裨将、连规谏谡、谡不能用**。《三国志·王平传》"连规谏谡、谡不能用、大败于街亭；众尽星散、惟平所领千人、鸣鼓自持、魏将张郃疑其伏兵、不往逼也"——**王平千人独全军、是街亭之败中唯一的亮点**。亮以平为参军、统五部兼当营事——**王平自此累迁至镇北大将军、督汉中**。**盛唐人读《三国志》者**——多以王平为"军中守节"的典范；街亭凭吊时常念其与马谡的对照。

### 陇坻关道（街亭的战略意义）

街亭所在的陇坻——**陇山西麓南北纵贯的关道**。陇坻东接关中、西连河西、南通秦州、北达灵州——**是关中与陇右、河西、朔方四方交通的咽喉**。**街亭失即陇右失**——陇坻关道一开、张郃魏军可长驱秦州、亮无所据——**这是诸葛亮 228 一出祁山自此功败的根本原因**。**盛唐陇坻**仍是关中入陇右的主驿路——**秦州为陇右道东端门户**、街亭在其北、扼陇坻关道一段；**盛唐边塞诗中"陇头""陇坻""陇右"皆此一带**。

### 戮马谡（建兴六年夏五月）

**建兴六年（228）夏五月**——诸葛亮收马谡下狱、戮之以谢众（《三国志·诸葛亮传》《马谡传》本传 + 裴注引《襄阳记》）。**正史原文仅"戮谡以谢众"**——"挥泪斩马谡"是后世文学化用、本 doc 严守此口径。**诸葛亮自请贬丞相为右将军三级**——以负街亭失利之责。**这一段是中国军法史上最沉重的一次执法**：诸葛亮与马谡相交甚厚、马谡死时三十九岁——**孔明军法之严、自此为中国军政叙事的极致**。**盛唐人凡读《三国志》及裴注者皆知此典**。

## 历史长河（755 → 今）

**中晚唐**　**广德元年（763）秦州陷吐蕃**——**街亭一带自此为吐蕃辖地近九十年**（至大中三年/849 唐宣宗收复秦州）。**乾元二年（759）秋杜甫弃华州官、自秦州入陇道**——**《秦州杂诗二十首》《赤谷》《铁堂峡》《青阳峡》《龙门镇》等一组陇蜀纪行诗**——是中国文学史上最重要的陇右纪行；杜甫南行所循即陇坻关道、过街亭一带（具体是否登街亭故城凭吊、史源未明、本 doc 不预设）。**晚唐**——街亭区为唐蕃边争反复地、士庶凭吊大减；**大中二年/848 以后张议潮起于河西沙瓜诸州（归义军）渐次收复河西**；**秦州复归唐属另一脉络**——属陇右收复与唐蕃边地互动的进程（大中三年/849 唐宣宗收复秦州），**不直接归张议潮功业**。

**两宋**　**北宋初秦州为关中西门、街亭一带为汉蕃杂处区**；**《太平寰宇记》《元丰九域志》皆记街泉故城在陇城县**——史志层面地名稳定。**北宋元丰间苏轼《诸葛武侯画像赞》**等多有念及街亭典故——但**专写街亭凭吊的宋人诗作不多**；**南宋陆游《关山月》《观大散关图有感》**等陇右诗咏陇头、未具写街亭。**金元**——街亭区为金蒙交战边地、纪念物事多废。

**明清**　**明嘉靖间秦州志**记街亭故地、立"诸葛丞相庙"于陇城；**清乾隆、嘉庆间秦州知府数次重修街亭古战场纪念地**——今所见"街亭古战场"主体格局多定于此时。**清陇城镇形成街亭村、马谡崖、断山、汲水沟等系列附会地名**——多不出明清方志、不是 228 当年原物、亦不是 755 唐人所见。

**民国至今**　1949 年后街亭古战场列甘肃省文物保护单位、**2013 年陇城街亭古战场遗址列全国重点文物保护单位**。**2010s 多次考古调查**——确认陇城镇北有汉魏夯土遗迹、街泉故城地望可指；但**马谡当年扎营的具体山头、张郃断汲道的具体水源**——考古至今未确认。**今街亭古战场在甘肃省天水市秦安县陇城镇**——包括诸葛丞相庙、马谡墓（附会）、街亭古战场纪念馆、汉代亭障遗址等——主体为明清重修与现代复建。**唐人在 755 所见的那片陇坻关道**——山在、川在、街泉故城地望在；但具体纪念物事是后世累积的、不是 755 时所见。

## 引文

> （建兴）六年春，亮身率诸军攻祁山……南安、天水、安定三郡叛魏应亮，关中响震……魏明帝西镇长安，命张郃拒亮。亮使马谡督诸军在前，与郃战于街亭。**谡违亮节度，举动失宜，大为郃所破。**亮拔西县千余家，还于汉中，戮谡以谢众。
> ——《三国志·卷三十五·蜀书·诸葛亮传》（陈寿）

> 谡舍水上山、举措烦扰……张郃绝其汲道、击、大破之；士卒离散。亮进无所据，退军还汉中。**谡下狱物故**（一作"亮戮谡以谢众"）。
> ——《三国志·卷三十九·蜀书·马良马谡传》（陈寿）

> 平连规谏谡，谡不能用，大败于街亭。众尽星散，惟平所领千人，鸣鼓自持，魏将张郃疑其伏兵，不往逼也。
> ——《三国志·卷四十三·蜀书·王平传》（陈寿）

> 满目悲生事，因人作远游。迟回度陇怯，浩荡及关愁。
> ——杜甫《秦州杂诗二十首》之一（乾元二年/759 秋客秦州作；755 时未作）
