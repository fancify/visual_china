# 灵岩寺

## 一句话

齐州济南郡长清县灵岩山中一座**北魏太延元年（435）法定禅师所立**的山中禅寺——北魏至北齐间渐成山东第一禅寺，隋唐再修、玄宗朝鼎盛；755 时是齐鲁佛教的物理 anchor，**距法定始建已三百二十年**；后世（北宋 1090s 起）渐与天台国清、当阳玉泉、南京栖霞并称为"海内四大名刹"——但**这一并称非盛唐唐人语**。

## 形貌

灵岩寺在齐州济南郡治历城县西南约一百里——长清县东南、**泰山西北麓灵岩山中**，一处山间盆地。寺距济南郡治历城（今山东济南）西南行约一百里、距兖州治瑕丘（鲁郡治、今山东济宁兖州区）东北约一百五十里——是河南道东部齐鲁山地最大的一处禅寺。

寺址在**灵岩山主峰之下**——南依泰山主脉、北望黄河（黄河 755 时北流于齐州北、距灵岩寺约百余里）、西俯长清平地、东接泰山西麓诸峰。**整个山中分内外两层**：山外是**长清县城**（齐州属县治所，距灵岩寺西北约二十里）、山外缓坡是**寺前驰道**；山内是**灵岩寺主体**——四面青石山壁围合的一片小盆地，南北约一里、东西半里。

寺布局沿**南北轴线**——寺门朝南、面对长清平地；门内是**前殿**（盛唐殿名史源未详，本 doc 不预设）；其后是**主殿**——盛唐人称"佛殿"或"释迦殿"，史源略异；殿后是**法堂**与**僧院**；最北高台之上是**塔基台**——塔为方塔，盛唐塔形史源不稳（今所见辟支塔为北宋淳化五年/994 始重建之八角九层砖塔、与盛唐塔非同一物）。**塔东侧另立"墓塔林"**——历代住持高僧塔的累积、北朝至盛唐间已有数十座（现存灵岩寺塔林共 167 座、起自唐、止于清，是国内现存最大墓塔林之一）。

寺周物候——背靠泰山西北、海拔约 400 米、四时分明：春有山桃野杏、夏多云气、秋叶遍山、冬有薄雪。寺所在的灵岩山，山石多**青灰石灰岩**——千岩万壑、状如莲台；**寺南山门外有"镜池"一泓**——是灵岩寺物候核心地标之一（"灵岩"之名一说即源于此池如镜映岩）。

寺前**驰道自长清县城东南行约二十里入山**——过镜池、上石阶、至山门。盛唐凡入灵岩者，多自齐州治历城西行约一百里至长清、再东南行入山——日程约一日半。寺东**通泰山驰道一支**——是齐鲁佛教与封禅地理的交汇线；寺西**通济水（隋唐济水自荥阳分黄入海、755 时独流未夺）**——济水至齐州西北约百里入海口为蒲台县。**这是灵岩寺在 755 时的整体地理位置**：泰山西北、济水南、黄河南、齐州西南、兖州西北——河南道东部最核心的一处禅寺。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

灵岩寺的来路在唐人记忆里是**两条线交织**——北魏法定立寺 + 北齐隋唐三朝重修。

**北魏始建**：**北魏太武帝太延元年（435）法定禅师立寺于灵岩山中**——《灵岩寺碑》《灵岩志》皆载此事。法定禅师生平史源略简，据《续高僧传》及后世灵岩寺志：法定自西天竺来，渡岭南上、经河洛、至山东、择灵岩为道场——这是灵岩寺最早的明确始建记录。**注：东晋僧朗（4 世纪末）在山东讲学**——后世灵岩寺志或将寺源上推至僧朗，但**僧朗为讲学非建寺**，本 doc 取北魏法定 435 始建口径。

**北朝至隋**：法定立寺后，**北魏正光至孝昌间（520s）灵岩寺渐成山东禅寺第一**——这是寺通名"灵岩"的初行时期。**北齐天保间（550s）有重修记录**——北齐文宣帝高洋崇佛、灵岩寺得朝廷供养；**北周武帝建德三年（574）灭佛**——灵岩寺殿宇毁、僧散，与北齐境内其他大寺同遭其难。

**隋开皇初（580s）文帝复佛**——灵岩寺得重修；**隋开皇间杨广（后炀帝）为晋王时**亦曾过灵岩、施供——这是灵岩寺与隋皇室的第一次接触。隋大业间炀帝东巡过齐鲁、未必再幸灵岩；但隋代灵岩寺殿宇规模较北齐为大。

**唐前期**：**唐贞观间（627-649）太宗朝重修灵岩寺**——这是灵岩寺入唐第一次大修。**唐高宗、武则天朝**屡有敕修；**唐高宗麟德二年（665）封禅泰山经长清、过灵岩山下**——封禅大队顺路供养（武则天 696 年封禅在嵩山而非泰山——《资治通鉴》卷二〇五"神功元年腊月封神岳"——故不及齐州灵岩）。

**玄宗朝**——开元至天宝间灵岩寺为齐鲁第一禅寺，僧众数百、殿宇数十；**天宝元年（742）李邕撰《灵岩寺颂》一通**（李邕时为北海郡太守、距灵岩寺不远，文人碑铭风骨——但原碑存佚有异文，本 doc 取 likely 口径）——这是盛唐灵岩寺最直接的文人碑文。**玄宗本人是否亲幸灵岩**——史载未详；玄宗一生封禅、游赏多次东行至齐鲁、但具体过灵岩寺礼佛史源不稳、本 doc 不预设。

### 此刻状态

天宝十四载这一年的灵岩寺——**法定 435 始建后第 320 年**、唐玄宗朝鼎盛期之末。

寺**在齐州济南郡长清县境**——**齐州治历城县**（今山东济南）、**天宝元年（742）改齐州为临淄郡、天宝五载（746）改临淄郡为济南郡**（《旧唐书·地理志》《新唐书·地理志》两阶段沿革本 doc 取此口径——755 时官称济南郡已九年）；**长清是济南郡属县**、治所在今山东济南长清区；灵岩寺在长清县东南约二十里灵岩山中。

距齐州治历城西南、距兖州治瑕丘（鲁郡治）东北约百五十里——是河南道东部齐鲁山地的核心位置。**寺与州、郡治的距离**——东北行至历城（齐州治）约一百里、东南行至瑕丘（鲁郡治）约一百五十里、南行至泰山主峰约六十里、北行至济水（隋唐济水独流入海）岸约百里——盛唐人入灵岩、可自这四向择其一。

寺殿盛唐主体**已立**——前殿、佛殿、法堂、僧院、塔基。**僧众数百**——具体住持是法定第几代法嗣盛唐史源未详（本 doc 不预设）；殿前柏树成林（部分至今犹存、有株经现代植物学测定上溯至北朝）；塔东墓塔林已累数十座。**寺前驰道与镜池**——是盛唐人入寺第一眼所见。

**寺的日常**——僧众晨钟暮鼓、四时讲经、岁岁朝礼；州、郡级官员春秋两遥礼；远来禅僧投宿讲席。盛唐灵岩寺在禅宗谱系上的位置——介于北宗（神秀一系、长安洛阳间）与南宗（慧能一系、岭南江南间）之间——**地处北方但非北宗根据地、与南宗亦无直接师承**；是齐鲁地区一脉相承的**北朝至唐独立禅院**。

**山东佛教 755 时的整体语境**——齐鲁地区的禅寺密度高、灵岩为首、之后是兖州的兴国寺、青州的龙兴寺、登州的崂山一带诸寺。**灵岩寺是当时山东境内最大的禅院**——但**"四大名刹"（灵岩 / 国清 / 玉泉 / 栖霞）并称是后世（约北宋 1090s）形成的说法**——盛唐人尚无此并称、本 doc 不预设。755 时唐人称灵岩，多言"齐鲁第一禅寺"或"灵岩之妙"——而非"四大名刹"。

**寺与封禅地理的关联**——灵岩山是泰山西北余脉、距泰山主峰东行约六十里。**秦汉唐诸帝多次封禅泰山**，封禅大队多自齐州治历城南行至泰山——经长清、过灵岩山下；**唐高宗麟德二年（665）封禅、玄宗开元十三年（725）封禅**——两次封禅大队过灵岩山下，灵岩寺得**蒙朝廷顺路供养**。盛唐人嘴里的"灵岩"——既是禅寺、亦是封禅地理上的一站。

天宝十四载的灵岩寺——**香火依旧、殿宇依旧、僧众依旧**。再过半年——安禄山起兵于范阳；安史叛军南下河北、河南、山东俱受兵燹；**齐州、兖州一带是河南道东部、距叛军主战场（河北、河南西部）相对偏东**——灵岩寺虽不入主战线、但齐鲁道路阻塞、香火渐稀。**这是灵岩寺在中唐前最后一段平静**。

### 此刻意味

对天宝十四载的长清人来说，灵岩寺是**县东南山中那座最大的禅寺**——岁岁朝礼、四时香火。

对齐州济南郡那一片士人来说，灵岩寺是**齐鲁第一禅院 + 北朝至唐三百年一脉相承的物理 anchor**——天宝元年李邕撰颂、是文人入此寺的体面。对长安那一代禅僧来说——灵岩寺是**东行齐鲁的必谒之寺**——但齐州远在长安东三千里、能至者少；盛唐人嘴里的"灵岩"，与曹溪法泉寺类似，**是个地名兼宗门符号**——比实地远、比传说近。

唐人尚不知道——再过二百四十年（北宋淳化五年/994），辟支塔将重建为八角九层砖塔——盛唐塔基之上立宋塔；再过三百一十年（北宋治平三年/1066），灵岩寺千佛殿内将立**四十尊罗汉彩塑**——是宋代雕塑史最著的群像；再过千年，**寺前柏林、塔林一百六十七座墓塔、千佛殿四十罗汉塑像**仍立——**唐人在 755 见过的灵岩，物理 anchor 未断**。

而 755 这一刻——灵岩仍是齐鲁山中那座静寺；殿前柏荫、塔下香烟、镜池映岩——日子如常、香火如常。寺主持已不知是法定第几代法嗣、寺东墓塔林已累数十座、寺中僧众仍守着北朝至唐三百年一脉相承的禅业。这是灵岩寺在中唐前最后的一段"在场"——再过半年，安史乱起；再过两年，齐州陷叛军；再过九十年，会昌灭佛——灵岩寺将在 755 之后的一千二百七十年里经几度起伏、几度重修、几度断裂。**但灵岩山仍在、柏仍在、池仍在**——这是盛唐人最不及知、亦最难以预料的一份长情。

## 子条目

> 子条目按"盛唐物理 + 历史人物 + 物候 anchor"分层：佛殿 / 辟支塔 / 墓塔林 / 镜池 / 法定 / 李邕 / 灵岩山 / 古柏。其中**佛殿、塔、塔林、镜池**在盛唐已立、是 755 时的物理对象；**法定、李邕**是历史人物层；**灵岩山、古柏**是物候 anchor 层。

### 佛殿 / 释迦殿（盛唐主殿）

**寺主体核心殿——立释迦像之处**。北魏始建、北齐重修、隋开皇复修、唐贞观再修——盛唐殿宇为这一传承的累积形态。**755 时殿内主像盛唐史源未详**——本 doc 不预设具体形制；殿前柏树成林、其中数株经现代植物学测定可上溯至北朝（500s）——盛唐人在 755 所见的柏，与今游人所见的有数株是同一株树。**今所见千佛殿（千佛殿内宋塑四十罗汉为北宋治平三年/1066 立、非唐塑）**为明嘉靖至清乾隆间重修，盛唐主殿原貌已不可考。

### 辟支塔 / 灵岩寺塔（盛唐塔基 + 北宋重建）

**寺最北高台之上**——盛唐塔形史源不稳、塔为方塔抑或楼阁式、塔层数史载未详，本 doc 不预设。**北宋淳化五年（994）始重建为八角九层砖塔**——这是今所见辟支塔的形制起点；北宋崇宁至宣和间（1100s）大修；明清续修。**盛唐塔基至今仍在塔下**——是灵岩寺与北朝隋唐一脉相承的另一物理 anchor。"辟支"二字取"辟支佛"——独觉成佛之意。

### 墓塔林（北朝至明清累积）

**寺东侧高台之上**——历代灵岩寺住持高僧塔的累积。**始自唐代**（盛唐前北朝高僧塔史源未详、可能已在但已毁）——盛唐人在 755 所见塔林尚未成大规模、但已有数十座唐塔；中唐至北宋累积加速；元明清续立。**今存灵岩寺塔林 167 座**——起自唐、止于清，**是国内现存最大墓塔林之一**（次于嵩山少林寺塔林）。塔林的存在说明灵岩寺**自唐至清传承未断**——这是与曹溪南华寺、嵩山少林寺并立的三大禅宗物理 anchor 之一。

### 镜池（"灵岩"之名的物候来源）

**寺南山门外一泓水池**——"灵岩"之名一说源于此池水如镜映岩。盛唐人入寺前必经此池——是入山的第一眼物候。**今镜池仍在**——为灵岩寺景区核心地标；池水之上、山门之外、灵岩诸峰倒映水中，与盛唐人在 755 所见在物候层面有真实连续。

### 法定禅师（北魏始建者）

**北魏太武帝太延元年（435）立寺者**——《灵岩寺碑》《灵岩志》皆载。法定自西天竺来、渡岭南上、经河洛、择灵岩为道场——这是灵岩寺最早的明确始建记录。**法定生平史源略简**——《续高僧传》卷十六有传但记事零散；法定与北魏佛教的整体语境关联紧密（北魏太武帝太延间正处佛道之争中——435 立寺距太武帝灭佛（446-452）尚有 11 年，灵岩寺得在灭佛前立基）。**注：东晋僧朗（4 世纪末）在山东讲学**——后世灵岩志或将寺源上推至僧朗，但**僧朗为讲学非建寺**，本 doc 取北魏法定 435 始建口径。

### 李邕《灵岩寺颂》（天宝元年/742）

**盛唐灵岩寺最直接的文人碑文**——天宝元年（742）李邕为时北海郡（青州改名）太守、距灵岩寺约二百里、亲撰《灵岩寺颂》一通立于寺中。**李邕（678-747）**——是盛唐第一流的碑铭文家、与杜甫、李白、王维同时；其碑铭文体在盛唐文人碑中**与颜真卿、张说并立**。**注：原碑存佚有异文**——一说碑在唐末战乱中毁、一说至北宋仍立，本 doc 取 likely 口径。**这通颂文是灵岩寺入唐文人撰碑文体的关键**——比柳宗元为曹溪撰《大鉴禅师碑》（815）尚早 73 年。

### 灵岩山（寺所依之山）

寺所在的山——**泰山西北余脉、海拔约 600 米**。山势两层：外环青石壁、内含小盆地；**盆地中即灵岩寺址**。**"灵岩"二字的来源**——一说自镜池如镜映岩、一说自山石之灵秀；盛唐人在山中行、远见诸峰如莲、近见石壁如屏——"灵岩"之名乃成。**灵岩山在唐时是齐鲁佛教的物理 anchor**——山下即寺、寺下即镜池、镜池外即长清平地。盛唐人若自齐州治历城西行百里，远见灵岩诸峰自泰山西北突出，便知灵岩已近。

### 寺前古柏（北朝至盛唐间植）

**寺前及殿前柏林**——盛唐人在 755 所见时已是茂林；现存数株经现代植物年龄学测定**可上溯至北朝（500s）**——是与法定 435 始建仅相距百年内的物理 anchor。**注：寺前现存古柏的"汉柏"等通名是后世传说**——本 doc 取"北朝至隋唐间植"的科学口径。盛唐人在 755 所见的柏，与今游人所见的灵岩寺殿前数株古柏——**至少有数株是同一株树**；这是灵岩与楼观古柏、嵩山少林古柏并立的中国佛道物理 anchor 之一。

## 历史长河（755 → 今）

灵岩寺自 755 起的物理命运，与曹溪南华寺、嵩山少林寺并列——是中国佛教史上**自北朝立寺一脉相承至今的少数大寺**之一，其间数经兵燹、数经重修、数经断裂——但寺址未移、山未易、僧塔林累至今。

**中晚唐**　安史乱中灵岩寺所在齐州一带受兵燹有限——**齐州 755-757 间一度为叛军（史思明部）所据**、寺中香火减、僧众散；**乾元元年（758）唐军收复齐州**——灵岩寺得复修、但规模已不及天宝鼎盛。**唐宪宗元和间（800s-810s）灵岩寺渐复**——这一时期墓塔林开始累积、是塔林的中段起点。**晚唐文宗、武宗朝**——文宗朝（830s）尚有齐鲁士人入寺；**唐武宗会昌灭佛（842-845）**——灵岩寺为山东最大禅寺、首当其冲，殿宇毁过半、僧众强令还俗；**宣宗大中元年（847）复佛**——灵岩寺得复修，但盛唐殿宇规模已大部不存。**唐末战乱（黄巢之乱 880s）**——黄巢军北上山东、齐州一带受兵燹再次甚重；灵岩寺殿宇再毁。

**两宋**　**北宋初灵岩寺再次中兴**——**淳化五年（994）真宗朝重建辟支塔**为八角九层砖塔；**治平三年（1066）千佛殿立宋塑四十罗汉**——是宋代雕塑史最著的群像之一、今犹存。

**宋代灵岩寺为山东第一禅院**——并与天台国清寺、当阳玉泉寺、南京栖霞寺**约北宋 1090s 起渐被并称为"海内四大名刹"**——这是后世说法、盛唐唐人尚无此并称。**北宋士大夫俱有过灵岩之记或诗**（具体作者与诗作系于灵岩寺的史源有详略、本 doc 不预设单一代表作）——这一时期灵岩寺为山东文人题咏之地。**南宋初山东陷于金朝**（1127 靖康之变后）——灵岩寺名属金、但寺存；南北分治之间灵岩寺地位未坠。

**金元**　**金大定间（1160s-1170s）灵岩寺续修**——金代灵岩寺地位虽不及北宋、但寺主体未毁。**元代灵岩寺仍属山东禅院第一**——元成宗、武宗朝俱有敕修记录；**元代墓塔林续累**——是塔林从唐至清一脉相承的中段。**至元间（1280s）灵岩寺一度为元廷崇佛重镇之一**——但全国佛教中心已转至大都（今北京）一带、灵岩地位渐成地方。

**明清**　**明洪武初敕修灵岩寺**；**明嘉靖间（1530s）僧明喻撰《灵岩志》**——是明代灵岩寺通志、溯及唐宋；**明万历间多次重修千佛殿、辟支塔**——这是今所见千佛殿与辟支塔形制的明清定型期。**清康熙、雍正、乾隆三朝俱有敕修记录**——清代灵岩寺为山东境内仍为首要禅院之一、但全国地位已让于嵩山少林、当阳玉泉等。**清乾隆南巡过山东**——乾隆十三、二十一、二十七、三十、三十六、四十一、四十五、四十九年八次南巡中数次过灵岩、留有御笔诗碑（部分至今存于寺中）——是灵岩寺与清皇室的最后一次重要交涉。

**民国至今**　民国十年代以来灵岩寺续有修缮、但兵燹与失养交替。**1956 年灵岩寺列山东省重点文物保护单位**；**1982 年国务院列灵岩寺为全国重点文物保护单位**——与 **1987 年泰山列世界文化与自然遗产**形成地理上的联动保护；**2010 年代起灵岩寺塔林、千佛殿、辟支塔多次大修**——以保护宋塑四十罗汉为重；**2018 年灵岩寺塔林整体加固工程**——是塔林一千余年以来最大一次保护性修缮。**今灵岩寺仍在济南市长清区万德镇灵岩山中**——千佛殿宋塑、辟支塔、塔林一百六十七座、镜池、寺前古柏——这些物理 anchor 与盛唐人在 755 所见相去千二百七十年，**但其中数项（柏林、塔基、镜池、寺址、灵岩山本体）在物理层面有真实连续**。

这是灵岩寺区别于楼观、九成宫、玉华宫的关键——**它的物理 anchor 未断**；但**唐时殿宇形制已散**（盛唐主殿、盛唐塔形均已不存，今所见千佛殿、辟支塔均为宋至明清重建）——故归 **A4 唐态断裂但今存** 而非 A1。盛唐人若于 755 在灵岩、与今游人于灵岩——**所见之山、所见之池、所见之柏（部分）、所见之寺址**——是同一处；**所见之殿、所见之塔、所见之像**——则已两易其形。
