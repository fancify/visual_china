# 官渡

## 一句话

**汴水北侧的"以弱胜强"古战场**——建安五年（200）曹操以二三万守此、败袁绍十万南下之师；自此曹操由群雄之一转为未来天下半壁；755 时唐人见的是郑州中牟县东北约三十里一片平畴间的古战场遗址 + 鸿沟分支官渡水东流。

## 形貌

官渡在唐河南道**郑州（天宝元年/742 改荥阳郡，治管城县）东属县中牟县**境内——具体位置在**中牟县东北二十二唐里**（《元和郡县图志》郑州中牟县条作"官渡水在县东北二十二里"，按今制约合十二公里）。**地势**：黄河南岸冲积平原中段、华北平原最腹地的低洼地；**官渡水**——是古**鸿沟（即蒗荡渠）**自荥阳分出向东南的一支水道、在中牟县东北分流、东流入汴；**水道两岸地势平、利屯军、利运粮、亦利退守**——这是曹操选官渡为防线的物理基础。

**官渡周边水系**——是华北平原中最密的一段：**北约一百二十里为黄河**（汉魏故道、自荥阳东北流过）；**西约三十里为鸿沟分流处**（古荥阳故城南）；**南约六十里为汴水主流**（鸿沟分出之主线、东南经陈留至彭城入泗）；**官渡北至东北方向约四十里为乌巢泽**（古泽薮、袁绍粮屯所在；具体地望封丘、延津两说，详见下"乌巢"子条目）。**官渡正在这片密水网中**——南北可凭水为限、东西可凭水转运——是中牟县境内最适合大军屯守的一处。

**唐 755 时官渡**——是郑州中牟县境内一片"古战场"故地。**官渡水仍东流**——是这片古战场最稳的物理 anchor；**故垒残基**——古战场遗址（土垒、烽燧基）在田畴间可辨；**鸿沟分流处的荥阳故城**——在官渡西约三十里、亦是唐人凭吊官渡时常一并到的关联地。**唐中牟县治在今河南中牟县老县城**（距官渡遗址约三十里西南）——县治不在官渡遗址内；**官渡作为中牟县东北一片旷野**——岁岁田耕、过客时来。

物候：华北平原中段、暖温带季风——四时分明、春多旱、夏有汴水暴涨、秋高、冬寒——盛唐人秋日东出洛阳至郑州凭吊官渡多记"白杨疏落、官渡水浩"。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

官渡在唐人的回望里——是**中国战争史"以弱胜强"最经典的一笔 + 曹操由群雄之一转为天下半壁的转折点**。**整场战役从建安五年（200）二月至十月、跨九个月**、由数次前哨战 + 主战 + 决战构成，是唐人通过陈寿《三国志》得以系统知晓的一段战争史。

**战前格局（199 末-200 初）**。**汉献帝建安四年（199）**——袁绍灭公孙瓒、并冀州 + 青州 + 幽州 + 并州、号"四州之主"、是河北最大势力；**曹操**——奉汉献帝都许（许都 = 今河南许昌东）、领兖州 + 司隶东部 + 豫州一带、势力远小于袁绍；**袁绍兵号十万**（《三国志·袁绍传》"简精卒十万、骑万匹、将以攻许"——实际史载约十万、号称略有夸张）；**曹操兵约二三万**（《三国志·武帝纪》"时公（曹操）兵不满万、伤者十二三"——本 doc 取主流史源：曹操总兵约二三万、官渡前线不满万）；**强弱悬殊**——是 200 年初河南河北两大势力的实情。

**前哨：白马 + 延津（200 二月-四月）**。**建安五年（200）二月**——袁绍遣大将**颜良**渡黄河攻东郡白马（今河南滑县）；**四月**——曹操救白马、用荀攸"声东击西"计、佯攻延津（白马西约六十里黄河南岸）、回师袭白马；**关羽斩颜良于白马**（《三国志·关羽传》"羽望见良麾盖、策马刺良于万众之中、斩其首还"——是中国军事史最有名的单骑刺将一例）；**白马解围**。**袁绍闻颜良死、即率大军南渡黄河追曹操**；曹操佯弃辎重诱袁绍军、**张辽 + 徐晃部斩文丑于延津**（《三国志·张辽传》《徐晃传》——延津之战文丑死、袁绍军气受挫）。**注**：**颜良 + 文丑皆死于白马 + 延津、非死于官渡本战**——这是唐人读《三国志》最易混的一笔、本 doc 严守史源、二将之死属"官渡前奏"。

**相持：官渡（200 五月-八月）**。**五月**——袁绍军南进至阳武（官渡北约三十里）扎营、欲与曹操决战；**七月**——曹操军退守官渡水北岸、构筑长围、与袁军对峙；**八月**——袁绍军逼近官渡、两军隔水相持、互攻数旬。**相持期最危险的一刻**：**曹操军粮将尽**、曹操致书荀彧（在许都坐镇）欲还许；**荀彧答**："今军食虽少、未若楚汉在荥阳成皋间也……公以至弱当至强、若不能制、必为所乘——是天下之大机也"（《三国志·荀彧传》原文）——**荀彧此一书**是官渡相持期曹操坚守的决心关键；曹操读毕、决意不退。

**决战：乌巢焚粮 + 袁绍军溃（200 十月）**。**关键转折**：**袁绍谋士许攸**——因家属在邺城犯法被审配收押、怨袁绍、夜出袁营降曹操、**告以乌巢屯粮**：袁绍十万大军的粮食皆屯于官渡北约四十里的乌巢泽（今河南封丘县东南、一说延津县东南——本 doc 取封丘说为主、延津说为附）、由淳于琼率万余人守；**曹操即夜率轻骑五千、衣袁军服、举袁军旗、衔枚疾趋乌巢**；**十月凌晨**——曹军至乌巢、纵火焚粮、斩淳于琼；**袁绍闻乌巢失**——遣张郃 + 高览攻官渡曹营、不下；**张郃 + 高览闻乌巢已焚、降曹**；**袁军大溃**——袁绍与子袁谭仅率八百骑渡黄河北逃；**曹操尽收袁绍辎重 + 图书 + 珍宝**。**这一夜**是官渡之战的决战夜——**曹操以弱胜强、自此立北方霸权基础**。

**战后：曹操定河北（200-207）**。**建安七年（202）五月**袁绍忧愤而卒；**袁谭 + 袁尚兄弟相争、曹操乘虚伐之**；**建安九年（204）八月**曹操克邺城（详 [邺城](./邺城.md)）、自此以邺为霸府；**建安十二年（207）**曹操北征乌桓、彻底平定河北——**自官渡至定河北、凡七年**。**官渡之战是这一切的起点**——是中国战争史"以弱胜强"最经典的奠基战。

**官渡之后**——曹操势力骤张；**建安十三年（208）赤壁之战**曹操败于孙刘联军、止步于长江；**建安二十二年至二十五年（217-220）**曹操进爵魏王、子曹丕代汉立魏——**三国格局自此定型**。**注**：**"三国"作为完整时代名**——是后世从陈寿《三国志》（西晋太康间 280s 撰）得来；755 时唐人按《三国志》知"魏吴蜀三国"——这是唐人对官渡之战时代位置的认知框架。

### 此刻

天宝十四载这一年的官渡——**已 555 年远**（200→755）；属唐河南道**郑州（荥阳郡）中牟县**境内、距中牟县治约三十里东北。

**官渡水**：仍自鸿沟分流处东流、注入汴水——755 时官渡水水量稳定、是中牟县境内最稳的水道之一；**故垒残基**：古战场遗址（土垒、烽燧基）在田畴间隐约可辨——夯土残基高数尺、与周边平畴形成微弱起伏；**乌巢故地**：在官渡北约四十里（封丘县东南或延津县东南——唐人对乌巢具体位置已不甚明、《元和郡县图志》亦不能确指）、是袁绍粮屯被焚处——是官渡战役决战的物理 anchor 之一、但 755 时仅一片泽薮旷野、无显著遗存。

**唐郑州 + 中牟县治所**：**郑州（荥阳郡）治管城县**（即今河南郑州、距官渡遗址约五十里西）；**中牟县治在今河南中牟县老县城**（距官渡遗址约三十里西南）——县治皆不在官渡遗址内。**官渡作为中牟县东北一片旷野**——田耕、放牧、岁岁有过客来凭吊。

**唐人对官渡的认知**——**以弱胜强 + 三国奠基 + 曹操转折**三重叠加：
- **战争史的物理 anchor**——盛唐文人读《三国志》《后汉书》及裴注皆知"曹操 vs 袁绍"决战之地在汴水北侧；
- **三国格局起点**——官渡 → 赤壁（208）→ 夷陵（222）是三国三大战役、官渡是奠基；
- **曹操由弱转强的转折点**——官渡之前曹操是群雄之一、之后曹操是未来天下半壁——唐人凡论曹操、必绕不开官渡。

**安史前夕**（即 755 这一刻）——再过半月（农历十一月初九）安禄山起兵于范阳；**自范阳南下取河北诸州 + 渡黄河 + 取河南**——**郑州 + 中牟县皆在叛军南下必经之路上**；**官渡遗址本身是古战场旷野、无守军**——但其所在的郑州 + 中牟县是安史叛军南下河南的咽喉之一；**安史叛军 756 正月渡黄河南下**——所经路线与官渡之战 200 年袁绍南渡黄河的路线大体重合——这是 755 时唐人尚不知、却即将在五个月后重演的一段路线。

### 此刻意味

对天宝十四载的中牟人来说——官渡是**县东北三十里那片"曹公破袁绍"故地**：田耕岁岁、过客时来；官渡水东流 + 故垒残基——构成日常生活背景里的古战场剪影。对盛唐文人来说——官渡是**中国战争史"以弱胜强"最重的一个 anchor**：曹操以二三万破袁绍十万、自此立北方霸权——是别处难见的戏剧密度。**盛唐文人凡论用兵之道**——必引官渡为例；**盛唐文人凡过郑州东行至中牟**——必至官渡凭吊。

唐人尚不知道——**再过半年（至德元载/756 正月）**，安史叛军将自河北渡黄河南下、过郑州 + 中牟、攻陷东都洛阳——**与 200 年袁绍南渡黄河攻官渡的路线大体重合**；**至德二载（757）九月**唐军收复东都、是在官渡战役 557 年之后的同一片华北平原。**唐人更不知道**——再过千余年（20 世纪），中牟县东北古城村一带将经现代考古确认为官渡古战场遗址核心区——**与 755 唐人所见的物理景观在大轮廓上是连续的**：以弱胜强、千八百多年、未易。

## 子条目

### 官渡水（鸿沟分流东支）

**官渡水 = 古鸿沟（即蒗荡渠）自荥阳分出向东南的一支水道**——在中牟县东北分流、东流入汴。**鸿沟**——是战国魏惠王开凿的大运河、自荥阳引黄河水东南流、至陈（今河南淮阳）东入颍水——是先秦至秦汉黄河中游最重要的人工水道；**官渡水**——是鸿沟在中牟县东北分出的东支、长约数十里、东注汴水。**官渡之战**——之所以以"官渡"为名、即因曹操在官渡水北岸扎营、与袁绍隔水相持；**水道两岸地势平、利屯军、利运粮、亦利退守**——这是曹操选官渡为防线的物理基础。**755 时官渡水仍东流**——是这片古战场最稳的物理 anchor；**今官渡水大部河道已淤塞**、仅余局部水脉、在中牟县东北古城村一带可见。

### 乌巢（袁绍粮屯被焚处）

**乌巢 = 袁绍粮屯**——在官渡北约四十里、是袁绍十万大军的粮食总仓；由淳于琼率万余人守。**建安五年（200）十月**——曹操夜率轻骑五千袭乌巢、纵火焚粮、斩淳于琼——是官渡之战决战夜的关键。**乌巢具体位置历来有争**：
- **封丘说**（主流）：今河南封丘县东南；
- **延津说**（次）：今河南延津县东南；
- **两说皆在官渡北约四十里**、皆位黄河南岸古泽薮中——两说差距不大。

**注**：**乌巢不等于官渡**——乌巢是官渡北约四十里的粮屯地、是官渡之战的决战夜物理 anchor、但本身是独立地点；唐人凡论官渡之战必提乌巢、但不应把乌巢等同于官渡本身。**755 时乌巢故地**——是一片泽薮旷野、无显著遗存；唐人对乌巢具体位置已不甚明、《元和郡县图志》亦不能确指。

### 白马 + 延津（官渡前奏）

**白马** = 今河南滑县东、黄河南岸渡口；**延津** = 今河南延津县北、黄河南岸渡口（白马西约六十里）——皆是黄河南岸古渡口、是袁绍军南渡黄河的主要节点。

- **白马之战（200 二月-四月）**：袁绍遣颜良攻白马；曹操救白马、用荀攸声东击西计、回师袭白马；**关羽斩颜良于白马**——是中国军事史最有名的单骑刺将一例。
- **延津之战（200 四月）**：袁绍闻颜良死、率大军南渡黄河追曹操；曹操佯弃辎重诱袁绍军；**张辽 + 徐晃部斩文丑于延津**——袁绍军气受挫。

**注**：**颜良 + 文丑皆死于白马 + 延津、非死于官渡本战**——这是唐人读《三国志》最易混的一笔；二将之死属"官渡前奏"、不入官渡本战。**755 时白马 + 延津**——皆是黄河南岸渡口、唐河南道滑州（白马所在）+ 滑州酸枣县（延津所在、唐属滑州）境内、与官渡为关联三处古战场。

### 许攸夜降 + 荀彧坚守书（两个关键转折）

**官渡战役有两个非物理性的关键转折**——皆是文字与人事、不是兵戈：

- **荀彧坚守书（200 七月-八月相持期）**：曹操军粮将尽、致书荀彧欲还许；荀彧答以"以至弱当至强、若不能制、必为所乘——是天下之大机也"（《三国志·荀彧传》）；曹操读毕、决意不退——**这一书**是官渡相持期曹操坚守的决心关键。
- **许攸夜降（200 十月）**：袁绍谋士许攸因家属在邺城犯法被审配收押、怨袁绍、夜出袁营降曹操、告以乌巢屯粮；曹操即夜袭乌巢——**这一降**是官渡决战夜的关键转折。

**这两个转折**——构成官渡之战戏剧性的内核：**一篇书 + 一次降**、决定了一场十万 vs 二三万的决战走向。**盛唐文人凡论官渡、必论荀彧 + 许攸**——是别处难见的人事深度。

### 三国三大战役（官渡 → 赤壁 → 夷陵）

**三国格局由三大战役奠基**：

- **官渡之战（200）**：曹操 vs 袁绍、定北方；
- **赤壁之战（208）**：曹操 vs 孙刘联军、定南方止于长江；
- **夷陵之战（222）**：刘备 vs 孙权、定蜀汉止于巴蜀。

**官渡是三大战役的奠基战**——之前曹操是群雄之一、之后曹操是未来天下半壁、之后才有赤壁 + 夷陵。**盛唐文人对三国时代的认知**——按陈寿《三国志》（西晋太康间 280s 撰）+ 裴松之注（南朝宋）构建；**"三国"作为完整时代名**——是后世定型、755 时唐人按《三国志》知"魏吴蜀三国"——这是唐人对官渡之战时代位置的认知框架。

## 历史长河（755 → 今）

**中晚唐**　**至德元载（756）正月**——安史叛军自河北渡黄河南下、过郑州 + 中牟、攻陷东都洛阳——**与 200 年袁绍南渡黄河攻官渡的路线大体重合**；**至德二载（757）九月**唐军收复东都；**乾元二年（759）九月**史思明再陷洛阳——郑州 + 中牟县在安史之乱中数度易手。**中晚唐凡至河南凭吊官渡者**——皆叹"曹公以弱胜强、今日何不可"；**晚唐杜牧《题乌江亭》**（会昌间作）"江东子弟多才俊、卷土重来未可知"——虽不直写官渡、但"以弱可胜强"的史观与官渡同一脉络；**晚唐胡曾《咏史诗·官渡》**（晚唐 870s）"东汉中平日、桓灵失驭年……曹瞒夜出营、火烧乌巢粮"——是晚唐官渡凭吊诗的代表。

**五代 + 两宋**　**五代后梁后唐间**——中原战乱、官渡遗址在郑州（汴梁西约六十里）辖境、岁岁有过客凭吊；**北宋初**——汴梁（即东京开封府）距官渡遗址仅约百余里、是北宋都城腹地；**北宋庆历八年（1048）黄河北流改道**——黄河自澶州（今河南濮阳）北决、改东北向流入海——**这次改道令官渡水水系大变**、鸿沟分流处的水量骤减、官渡水大部河道淤塞。**北宋诗人苏轼、苏辙、王安石**等过郑州凭吊官渡留诗——**苏轼《赤壁怀古》**（元丰五年/1082 黄州作）虽是写赤壁、但"以弱胜强"的史观在宋人对三国战役的总观中、官渡常与赤壁并提；**北宋《武经总要》**（庆历四年/1044 曾公亮等撰）多次引官渡之战为以弱胜强的兵法范例。

**金元**　**金代官渡遗址在郑州辖境**——金人对官渡没有大规模营建、故垒残基续废；**金元间黄河多次改道**——黄河主流自荥阳东北南移至开封以南、官渡水水系彻底破坏、大部河道淤塞；**元代官渡已成田野**——故垒基址上有农人耕作、放牧；**元代地方志记官渡"古战场残基、土堆数尺"**——是官渡物理保存状态的中期记录。

**明清**　**明嘉靖间**官渡遗址在开封府郑州中牟县境内——**嘉靖三十六年（1557）立"曹公破袁绍处"碑**于官渡故地；**清乾隆四十年（1775）**陕西巡抚毕沅过中牟考察官渡（毕沅作《中州金石记》收录官渡碑刻）；**清末官渡已是河南中部一处"考古名胜"**——但因没有大型纪念建筑（不同于赤壁、夷陵均有重修纪念物），凭吊者主要是文人士大夫。**清末民国间地方志**——中牟县东北古城村一带被指为官渡古战场核心区、与"官渡水"故道沿线相连。

**民国至今**　1949 年后官渡古战场遗址列河南省重点文物保护单位、**2006 年官渡古战场遗址列全国重点文物保护单位**。**20 世纪 80 年代起**——河南省文物考古研究所等单位对中牟县东北古城村一带的官渡古战场遗址进行多次调查、确认故垒基址 + 古城村古城（一说为曹操官渡大营所在）+ 官渡水故道——是华北平原最重要的一处古战场遗址之一。**2003 年起中牟县始建官渡古战场旅游区**——展示官渡之战遗存与发掘出土物；**官渡古战场遗址跨中牟县东北古城村 + 邵岗乡一带**——是河南中部最重要的一处三国遗址。**今游人来官渡古战场遗址**——所见的是田野中的故垒残基 + 古城村古城轮廓 + 官渡水故道——**与 755 唐人所见的物理景观在大轮廓上是连续的**：千八百多年、以弱胜强、官渡以古战场之姿等待至今。
