# 黄鹤楼

## 一句话

鄂州江夏县治西二里黄鹄矶上的瞭望楼——**孙权魏黄初二年（221）自公安都鄂、改鄂名武昌；越二年吴黄武二年（223）筑夏口城、楼立西南角矶上为军事用**；至南朝齐始作观景楼用、称黄鹤楼；755 时是盛唐江南名楼，**崔颢《黄鹤楼》"昔人已乘黄鹤去"已传诵二十年**。

## 形貌

黄鹤楼立**鄂州江夏县治西二里黄鹄矶**——**黄鹄山**（亦名蛇山）西端伸入长江中的一道矶石。

矶下即是长江——**江北对岸为汉水入江之口**（即"汉口"，但 755 时此处尚是江湖泽国、无大型聚落，"汉口"作镇市要到一千年后明代成化间）。**江汉合流自此向东**——南岸鄂州、北岸沔州（治汉阳，今武汉汉阳）、隔江相望；从矶上向北看，长江浩瀚、汉水自西北注入江、两水相汇激起波澜。**站在黄鹄矶上看西、看北、看东**——是当时整个江汉交汇带最佳的瞭望角度：故吴黄武二年（223）孙权筑夏口城时（其都鄂改武昌已在两年前 / 魏黄初二年 221），特立此楼于城西南角。

楼形——三国吴始建时是**军事瞭望楼**：木构、高数丈、上设守望，下为兵戍。至南朝齐（479-502）渐转为观景楼用、楼上设栏可凭眺，**755 时所立者**应是历代屡修中的某一座（具体唐楼形制盛唐史源未详；今所见 1985 年重建之楼非古制）。唐人凡至鄂州者必登；登楼向东可望长江浩荡东去、向北可望汉水入江、向南可俯瞰江夏县城坊里、向西可见蛇山曲折东延。

矶下江水深而急——**这是长江中游一处天然的扼江险地**：上接荆州（江陵）、下达九江浔阳；楼立矶上、是江汉水路的视觉地标。**自上游来船**——出三峡过夷陵（峡州）、东下荆州江陵、再东即至鄂州黄鹤楼前；**自下游来船**——逆江上九江、武昌（鄂州）、再溯而上荆州、入峡至蜀；**自汉水来船**——自襄州顺汉水南下、入江口即望黄鹄矶头楼角。**江夏因此是长江中游三向水路的交汇点**——而黄鹤楼是这一交汇点的视觉中心。

矶头眼前另有一道沙洲——**鹦鹉洲**横亘江中、与黄鹄矶隔江相对；洲上芳草萋萋、是黄鹤楼眼下最显著的中景。再北望、隔江是沔州汉阳县——汉阳县城外是江汉间的湿地林木、即崔颢诗"晴川历历汉阳树"所写。**江、汉、洲、阳——这四景同时入眼**——是黄鹤楼区别于其他江楼的根本特征。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

黄鹤楼在唐人的回望里有两层——**三国吴的军事瞭望楼 + 南朝以后的观景楼**。

**第一层：三国吴始建（221→223 两事分系）**。**魏黄初二年（221）**——孙权自公安移都于鄂、改鄂名"武昌"（**注：孙吴武昌 = 今湖北鄂州市；与今武汉武昌区是两地——唐 755 时鄂州治"江夏县" = 今武汉武昌区，与孙吴所都"武昌"非同一地，本 doc 后文凡称"武昌"皆为孙吴武昌即今鄂州市**）。**越二年——吴黄武二年（223）**——孙权于黄鹄山西端江岸（今武汉武昌区一带——即唐江夏县治）**筑夏口城**；夏口城西南角即黄鹄矶——**孙权命于矶上立军事瞭望楼**（即黄鹤楼前身），用以监江汉水道、防上游蜀汉（刘备）来犯。**这是黄鹤楼最早的物理实体**——是兵戍设施、不是文人观景台。"黄武"为孙权年号——**不是"魏黄武"**（魏无此年号，魏文帝同期为黄初；本 doc 严守史源）。**都鄂改武昌（221）与筑夏口城兼立瞭望楼（223）是两事、相隔两年、勿混系**。

**第二层：南朝转作观景楼（南齐前后）**。**三国归晋（280）后**——夏口城转为江夏郡治；**南朝齐时（479-502）**——《南齐书·州郡志》记载江夏郡条已言楼之事；**与此前后《述异记》（南朝梁 任昉撰）**载费祎乘黄鹤至此升仙、楼因得名——传说之具体生成年代不可考、但至迟在南朝齐梁之际、楼已从军事瞭望转为观景凭吊之用。"**黄鹤楼**"之名——三说并存：（a）因下有黄鹄矶而得名（鹄为鹤之异写）；（b）因仙人费祎乘黄鹤至此说而得名；（c）因唐人崔颢诗"昔人已乘黄鹤去"之后、仙鹤说固定下来——盛唐人多取仙鹤说。

**第三层：盛唐文学定型（730s）**。**约开元天宝间（730s 末—740s 初）**——崔颢登黄鹤楼作七律《黄鹤楼》："昔人已乘黄鹤去、此地空余黄鹤楼。黄鹤一去不复返、白云千载空悠悠。晴川历历汉阳树、芳草萋萋鹦鹉洲。日暮乡关何处是、烟波江上使人愁。"——**自此黄鹤楼进入中国文学的最高梯队**。严羽《沧浪诗话》（南宋）评此诗"**唐人七律第一**"。**约同时段（开元十八年/730 前后）**——李白送孟浩然自鄂赴广陵、作《黄鹤楼送孟浩然之广陵》"故人西辞黄鹤楼、烟花三月下扬州；孤帆远影碧空尽、惟见长江天际流"——又一首盛唐七绝顶峰。**至 755 时**——崔诗已传二十年、李诗已传二十五年——黄鹤楼是盛唐**江南诗学最浓的物理 anchor 之一**。

### 此刻状态

天宝十四载这一年的黄鹤楼——**立矶上、面长江、楼形依旧、诗已名世二十年**。

**行政归属**：鄂州（**天宝元年/742 改江夏郡**、至德二载/757 复鄂州）——天宝十四载这一年正在"江夏郡"称之下。郡治在**江夏县**（今武汉武昌）。**江南西道**——道治在洪州（今南昌）。鄂州在江南西道东北、长江中游、是江南西道与江南东道、淮南道交界的水运枢纽。

**楼的状态**：楼立黄鹄矶头——**自三国吴始建至 755 已 532 年**；其间屡修屡毁、屡毁屡修。盛唐时楼当尚可凭眺、为江夏县境最显著的物理 anchor 之一（具体盛唐时此楼建筑形制史源未详——本 doc 不预设）。**唐人登黄鹤楼实景**——盛唐文集中黄鹤楼诗已成传统：崔颢、李白以外、孟浩然《鹦鹉洲送王九之江左》、王维《送康太守》、岑参等盛唐诸大家皆有过黄鹤楼或周边之作。**安史前夕**，长江中游水道仍稳——洛阳东都、长安西京尚未被战火惊扰，南来北往的舟楫照常自江夏过、登楼者照常凭眺。

**江汉景观**：黄鹤楼眼前的**鹦鹉洲**——是长江中一道沙洲、传说东汉末祢衡作《鹦鹉赋》于此被黄祖所杀、因得名。755 时鹦鹉洲尚立江中（明清后渐淤没、清乾隆间故洲已不存、今武汉鹦鹉洲为新洲）——崔颢"芳草萋萋鹦鹉洲"看的便是这一道古洲。**江中常有舟楫往来**——商船、官船、漕船、渔舟混杂；矶下水深浪急、是楼上凭眺时最有动感的一处。

**与盛唐三大江南楼的并称**——后世并称"江南三大名楼"为黄鹤楼（鄂州）+ 岳阳楼（岳州）+ 滕王阁（洪州）；但**"三大名楼"作为并称概念实定型于明清**——盛唐时三楼各自为名、并未并称。**755 时盛唐人提江南名楼**——黄鹤楼是首列、岳阳楼次之（杜甫《登岳阳楼》要到大历三年/768 才作）、滕王阁因王勃《滕王阁序》（约上元二年/675 作）已传八十年——三楼皆有诗为锚、但"三大名楼"作为一组并称之事属后世。

### 此刻意味

对天宝十四载的江夏县民来说，黄鹤楼是**县治西二里那座江上瞭望楼**——日常无大事、岁岁有过客来登。对盛唐过鄂州的诗人来说——黄鹤楼是**必登的诗学场所**：崔颢已立标杆于此、李白送别孟浩然亦在此、王维孟浩然岑参皆有诗——**登楼即是与盛唐前辈对话**。

唐人尚不知道——天宝十四载十一月初九（公历 755 年 12 月 16 日）安禄山起兵于范阳、北方陷于战火；江南西道、鄂州因不在主战场而保全；**长江中游水道成北方士庶南奔的主要通道**——杜甫晚年（大历三年/768）自夔州出三峡过荆州下江陵、本欲过鄂州登黄鹤楼、终未成行而北返。**李白晚年**——乾元二年（759）流夜郎遇赦、东归过江夏作《与史郎中钦听黄鹤楼上吹笛》"一为迁客去长沙、西望长安不见家；黄鹤楼中吹玉笛、江城五月落梅花"——这是李白与黄鹤楼最后一次相遇、距 755 仅四年、却已是另一个时代。

再过千二百多年——黄鹤楼屡毁屡建、清光绪十年（1884）后毁、1985 年于蛇山新建处重立——**今游人来武汉看的黄鹤楼是 1985 年钢筋混凝土仿宋楼**、非古楼；但脚下蛇山西端伸入长江的那道矶石、与楼眼前长江、汉水合流的形势——与 755 时唐人所见在物理层面是连续的：**山仍是山、江仍是江、汉水仍自西北注入长江**。

天宝十四载这一年——黄鹤楼最深的意义、不在楼之高、不在江之阔、而在**崔颢二十年前那首七律已经把这座楼写进了不可逾越之处**：自此盛唐凡至江夏者、登楼皆要默念那二十八字、再决定写不写。**李白也好、孟浩然也好、王维也好——盛唐顶级诗人在这座楼前都必须先向崔颢点头**。这是 755 时黄鹤楼独有的、与其他江南名楼不同的存在感：**它已经不是一座楼、而是一首诗的物理 anchor**。

## 子条目

### 黄鹄矶（蛇山西端入江矶）

**黄鹤楼立矶之上**——黄鹄矶是黄鹄山（蛇山）西端伸入长江中的一道矶石。"鹄"为鹤之古异写——矶因山形似鹤栖江上之状得名（一说）。**矶上即楼基**——矶下即长江水道；自江上溯来船看见的是矶头楼角、自岸上来看的是楼立矶顶、楼下即激流。**矶之险**——三国吴筑夏口城择此为西南角、即取矶头扼江之势。**今黄鹄矶**——仍在武汉蛇山西端、即古黄鹤楼旧址所在；**1985 年新黄鹤楼并未立于古矶之上、而是东移约 1 公里于蛇山东段另择新址重建**（1957 年武汉长江大桥武昌桥头堡占用了原矶旧址）——故古矶为旧址、新楼非原址；矶石与江水之间已隔大堤、不复唐人所见"矶下即江"之险。

### 夏口城（三国吴黄武二年/223 筑）

**孙权于黄鹄山西端筑此城**——东西二里、南北二里余的小城；**城西南角即黄鹤楼之楼址**。夏口城是孙权移都武昌后的江汉前沿据点——**军事用途明显**（防上游蜀汉刘备 + 监江汉水道）。**晋平吴（280）后**——夏口城转为江夏郡治；**南朝齐梁间**——城已扩为郢州州城；**至唐 755**——已是鄂州州城（治江夏县）的旧基所在；州城规模远大于三国旧夏口城、但旧城的"西南角黄鹤楼"位置一直被保留沿用。

### 崔颢《黄鹤楼》

**约开元天宝间（730s 末—740s 初）作**——崔颢登黄鹤楼留诗七律一首："昔人已乘黄鹤去、此地空余黄鹤楼。黄鹤一去不复返、白云千载空悠悠。晴川历历汉阳树、芳草萋萋鹦鹉洲。日暮乡关何处是、烟波江上使人愁。"——前四句**破七律对仗常例**、连用三个"黄鹤"、一气直下；后四句**回到工整对仗**——是七律史上最有名的"破体之作"。**严羽《沧浪诗话》评"唐人七律第一"**——南宋以来近八百年公论。**755 时此诗已传二十年**——是黄鹤楼最重要的文学 anchor。

### 李白《黄鹤楼送孟浩然之广陵》

**约开元十八年（730）作**——孟浩然自鄂赴广陵（今扬州）、李白送别于黄鹤楼之时作七绝："故人西辞黄鹤楼、烟花三月下扬州；孤帆远影碧空尽、惟见长江天际流。"——盛唐七绝顶峰之一。**李白"眼前有景道不得、崔颢题诗在上头"**——传说出元代辛文房《唐才子传·崔颢》、晚出史源、不一定是史实；但李白确实多次过黄鹤楼、与此楼关系极深。**755 时此诗已传二十五年**——盛唐人多以此诗为送别诗的标杆。

### 鹦鹉洲（黄鹤楼前长江沙洲）

**长江江夏段中的一道沙洲**——汉末祢衡（173-198）作《鹦鹉赋》于此、被江夏太守黄祖所杀、洲因得名。**崔颢"芳草萋萋鹦鹉洲"**看的便是这一道古洲。**755 时鹦鹉洲尚立江中**——是黄鹤楼眼前最重要的视觉景观之一。**至明清后渐淤没、清乾隆间古洲已不存**——今武汉所谓"鹦鹉洲"为后世新淤之洲、非崔颢所见之古洲。

### 江汉合流 / 三水交汇

**黄鹤楼前是中国中部最重要的水路交汇**——**长江自西来**（出三峡过荆州东下）、**汉水自西北来**（自襄州顺汉水南下）、**两水于汉阳之北合流**继续向东。**这一合流形势是江汉平原与长江中游的核心地理 anchor**——楚地、荆襄、江夏自古因水而立。**唐时鄂州即因这一交汇而为江南西道东北重镇**——上控荆州、下扼九江、北通襄阳；登黄鹤楼可同时看见这三向水道的来去。**755 时三向水道俱通**——荆襄漕粮、淮南盐铁、岭南贡物多经此转运——黄鹤楼眼下的舟楫密集程度远超后世人之想象。

### 汉阳与汉水入江口

**黄鹤楼对岸即沔州治汉阳县**——隔长江相望。**汉水自西北注入长江**——入江口即在汉阳之北。**唐时汉阳为沔州州治**——州城规模远小于鄂州（鄂州为江南西道东北重镇、汉阳为小州）。**"汉口"作镇市**——要到一千年后明代成化间（1465-1487）汉水改道、北岸成镇、始有"汉口"之名；**755 时此处尚是江湖泽国、无大型聚落**——崔颢诗"晴川历历汉阳树"看的是江北汉阳县城外那片江汉间的湿地与林木、不是后世的"汉口镇"。

### 黄鹄山 / 蛇山

**黄鹤楼所立之山**——黄鹄山是唐时通名、亦称蛇山（因山形蜿蜒似蛇）。**山东西延伸约数里**——西端即黄鹄矶、伸入长江；东端连接江夏县城东郭。**唐时山上以黄鹤楼为最显著建筑**——其余多为林木与寺观（具体寺名盛唐史源未详）。**至明清后山上渐多寺院、亭台**——黄鹤楼始与周边亭阁组成一片山头景观。**今武汉蛇山黄鹤楼公园**沿山东西展开、园内除 1985 年新黄鹤楼外、另有岳飞铜像、白云阁、落梅轩、搁笔亭等附属建筑——皆 1980s 后规划之产物、非唐迹。

### 楼名的三说：黄鹄、仙鹤、崔诗

**第一说："黄鹄矶"得名说**——楼立于黄鹄矶上、矶以山得名（山如鹤栖江上）；"鹄"为"鹤"古异写、汉魏间通用。**这是最早、最朴素的解释**——见《元和郡县图志》追述。**第二说："费祎乘黄鹤"仙说**——《述异记》（南朝梁 任昉撰）等载费祎（一作"费文祎"、三国蜀汉名臣）在此乘黄鹤升仙、楼因得名。**第三说："崔颢诗后定型"说**——黄鹤楼之"鹤"自崔颢诗"昔人已乘黄鹤去"后、仙鹤说固化、楼之诗意自此定型。**755 时盛唐人多取仙鹤说**——但学者多兼采前两说；本 doc 严守三说并存、不作断语。

## 历史长河（755 → 今）

**中晚唐**　**安史乱中（755-763）**——江南西道未陷大乱、鄂州与黄鹤楼俱保全；**长江中游水道成北方士庶南奔的主要通道**——黄鹤楼一时为流寓诗人凭吊乡关之处。**李白乾元二年（759）**流夜郎遇赦、东归过江夏作《与史郎中钦听黄鹤楼上吹笛》"一为迁客去长沙、西望长安不见家；黄鹤楼中吹玉笛、江城五月落梅花"——是李白与黄鹤楼最后一次相遇、距 755 仅四年、却已是另一个时代。**杜甫大历三年（768）**出三峡至江陵、本欲东下过鄂州、终未抵江夏；**杜甫《黄鹤楼》诗实未作**——他与此楼终是失之交臂。**中晚唐凡过江夏者必登黄鹤楼**——白居易、刘禹锡、张籍、王建、温庭筠等中晚唐诸家皆有黄鹤楼诗；**白居易《卢侍御与崔评事为予于黄鹤楼置宴宴罢同望》**等皆是中唐黄鹤楼诗代表。**晚唐崔涂、罗隐**等亦有黄鹤楼咏；**晚唐五代间楼累毁累修**——具体何代何月的修建史源不密。

**两宋**　**北宋**鄂州为荆湖北路重镇、黄鹤楼累修；**陆游入蜀途中过黄鹤楼**作《入蜀记》记其形——南宋绍兴间楼已与盛唐异、但仍为江夏地标。**北宋元丰年间（1078-1085）**重建之楼形制宏大、楼制三层、画栋飞檐——**这是黄鹤楼史上第一次有详细文献记载形制的时代**。**北宋诗人苏轼、王安石、黄庭坚**等皆有黄鹤楼诗；**南宋诗人陆游、范成大**等过鄂州亦皆登。**南宋绍兴间岳飞驻军鄂州**（1134-1140）——三次率岳家军北伐皆以鄂州为大本营、相传（一作托名）登黄鹤楼留《满江红·登黄鹤楼有感》"遥望中原、荒烟外、许多城郭"（此词作者归属至今学界仍有争议、明清以来多系于岳飞名下、本 doc 不作断语）——若此词确为岳飞登楼所作、则是黄鹤楼史上最有军事悲壮气的一首词、与盛唐崔诗李诗合为"楼之三声"。

**金元**　北方金朝未及江南、鄂州一直在南宋治下；**元代**江夏归鄂州路；**元末徐寿辉、陈友谅起义军以江汉为基地**、鄂州一带常为战场；楼累毁累修。**元代诗人虞集、揭傒斯**等过江夏皆有黄鹤楼诗——但元代黄鹤楼的具体物理状态史源不详。

**元明清**　**元代**江夏为鄂州路治、楼累修累毁——元末战乱（1356-1368）间楼一度倾废。**明洪武初**重建——**明楼形制为楼台三层、覆以铜瓦、楼前石碑林立**；**嘉靖年间所立之楼**楼上有铜瓦黄琉璃饰、楼前石碑数十座——是明楼形制最盛之期；**万历后楼几次焚毁、崇祯年间重修；清初顺治、康熙间屡修**。**明清间黄鹤楼累毁累建凡七次**——每次形制略异、是历代毁建最频之时代。**清同治七年（1868）**所建之楼是**最后一座古黄鹤楼**——三层、楼高约九丈、八角攒尖顶、覆铜瓦——光绪十年（1884）夏秋间因近邻民居失火延及、**楼焚毁**；**仅余夯土石基**。**自此至 1955 年武汉长江大桥兴建**——黄鹤楼旧址七十余年间为空地。**长江大桥（1955 动工、1957 通车）武昌桥头堡选址占用原黄鹤楼旧址**——故古矶旧址不复可建；**1985 年于蛇山东段新址（距旧址东约 1 公里）重建黄鹤楼**——钢筋混凝土仿宋楼、楼高 51.4 米、五层、与旧址不同处。

**民国至今**　**今游人来武汉所见黄鹤楼**是 1985 年重建之楼——非古楼。**蛇山黄鹤楼公园**与下方武汉长江大桥共同构成武汉最显著的城市天际线。**站在 1985 年的黄鹤楼上**——眼下江汉合流、对岸汉阳龟山与晴川阁、北岸汉口高楼林立——形势与 755 唐人所见在山水大势上仍连续，但城市景观已是另一个时代。**黄鹤楼今为湖北省重点文物保护单位 + 国家 5A 级景区**。**今武汉三镇**（武昌 + 汉阳 + 汉口）——是 1927 年武汉三镇合称之产物——**755 时此三镇尚不并称**：武昌（江夏）是鄂州州治、汉阳是沔州州治、汉口尚未发展为镇市；**唐人若听到"武汉"二字会茫然不解**。本 doc 严守 755 时态、不用"武汉"作 755 历史叙述、只在涉及今址定位时偶用作现代参照。

**今登 1985 年黄鹤楼者**——多数人不知崔颢七律的破体结构、不知李白与孟浩然在此别于开元十八年的具体情境、不知"昔人已乘黄鹤去"的"昔人"究竟是费祎还是某位无名仙人——但**站在五层楼顶向北望去、江汉合流的形势仍在**——这是黄鹤楼跨越一千二百七十年、仍能维持其文学锚点功能的根本：**地理形势比楼本身更耐久**。
