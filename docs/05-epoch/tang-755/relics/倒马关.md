# 倒马关

## 一句话

**太行八陉之飞狐陉东端咽喉、河北平原通代北的物理 anchor**——立定州 / 博陵郡唐县境太行东麓一道狭谷里、北望蔚州飞狐县、西通河东代北；**Tang 755 时尚不称"倒马关"——唐人在文牍里写的是"鸿上关"或"常山关"**；**再过半年安禄山自范阳起兵、叛军一部经飞狐陉西出代北召集塞外援军**——这道在 755 仍以鸿上 / 常山为名的关塞、半年后即成安史之乱的物理通道之一。

## 形貌

倒马关在唐河北道**定州 / 博陵郡唐县境**——具体在**定州治安喜县**（今河北定州）**西北约百五十里**、太行山东麓一道狭谷里（今河北唐县西北倒马关乡）；与**蔚州/安边郡飞狐县**（今河北蔚县）相距百余里 / 数百唐里、与**常山郡**（治真定县、今河北正定）西北边境相接。**地势**：**太行山自西南向东北横亘**——西面是河东道代北高原；东面是河北平原；倒马关立**太行东麓一道狭长峡谷里**——南北约十余里、东西约二三里、两侧山势峻拔；**这道峡谷接飞狐陉东端**——飞狐陉自蔚州境向东南穿太行、东端开口即在倒马关一带。

**倒马关与飞狐陉的关系**：**飞狐陉本身**——是太行八陉自南而北第六陉、主体在蔚州境（今河北蔚县至涞源一带）；陉道自代北飞狐县东南行、穿太行山至河北平原；**陉道东端开口在定州西界 + 唐县西北**——**倒马关即扼此东端开口**。**这是一个常被混淆的细节**：飞狐陉 ≠ 飞狐县——飞狐县是蔚州属县、飞狐陉是整段陉道；倒马关在飞狐陉东端、不在陉道中段。

**与其他太行关塞的格局**：**北面易州 / 上谷郡境有五阮关**（即后世"紫荆关"——在易州西五回县境、太行蒲阴陉咽喉、距倒马关约百里）；**幽州西北有居庸关**（妫州境、军都陉咽喉）；**三关南北一线护幽燕西门**——但**"内三关"（居庸 + 紫荆 + 倒马）并称是明代命名**、Tang 755 时三关各自分立、未有并称之名。**这一点对 doc 的诚实很关键**：755 时唐人不会说"内三关"——他们说的是"居庸"、"五阮"（或"子庄"）、"鸿上"（或"常山"）；"紫荆关"一名要到金元以后才行用。

**关城形制**：盛唐倒马关的具体形制 + 周长 + 门数史载零散——本 doc 不预设细节（**今所见倒马关楼是明景泰 / 嘉靖以后重修、不是唐关原物**）。唐倒马关（鸿上关 / 常山关）大体应有**关墙 + 关门 + 戍兵营 + 关吏馆**；具体规模未见可信细节。**关塞外**：西通飞狐陉中段 + 蔚州 + 河东道云州；**关塞内**：东通定州 + 易州 + 河北平原。

**山势**：关谷两壁——**北壁陡而短、南壁缓而长**；岩层主体为太行常见的**寒武奥陶纪灰岩**——风化后多形竖崖与碎石坡；谷底有马道一条、宽不过丈余、随溪曲折；过关者**牵马而行、不能并辔**——"倒马"之名的民间附会（马倒不能起）即由此而来、但 755 时尚无此名。**关谷东出口**——南面唐县方向地势渐缓、过一道丘陵即入河北平原；**西入口**——西北面五回 / 涞源方向地势渐高、过一道分水岭即入飞狐陉中段、再西北即蔚州境飞狐县（今河北蔚县）。

**物候**：太行东麓——温带季风气候过渡带；冬寒夏凉、关谷内常有山溪。**关谷内的水**——是**唐河（即滱水）支流**；滱水自代郡灵丘县东南流、经飞狐陉至倒马关一带、再东南入定州境（今唐河水系）。**这条水自唐前即是飞狐陉的物理 anchor**——《水经注》载之甚详。**四时**：春多风沙（来自代北高原）；夏多骤雨（关谷内雨势集中、易致山洪）；秋短而凉、关山红叶；冬寒、关谷内有薄冰、但因山溪冬不冻、过关驿路仍可通——这一点与更北的居庸关、更西的飞狐陉中段不同（后两者冬季冰雪封山时间更长）。**因此倒马关在四时之中、是太行八陉中通行时间最长的关塞之一**——这亦是其战略价值的物理基础。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

倒马关在唐人的回望里——是**太行飞狐陉东端的千年关塞**。

**关塞的设立**：**战国时期**——此一带为**中山国**（BC 414 立国、BC 296 为赵所灭）疆域；中山国都灵寿（今河北平山）、领有今河北中部太行东麓诸城；**倒马关一带为中山国西部山口、扼太行陉道**。**汉代**——属中山国（汉景帝置）；**关塞的具体名称**——汉代史料零散、未必已立专名。

**北朝至隋的关名**：**北魏置关、名"鸿上关"**——亦称"常山关"（因关塞西邻常山郡）；《魏书·地形志》载之；**这是倒马关在 Tang 755 时实际使用的名号**。**隋承北朝制**——关名沿用；**唐承隋制**——755 时官府文牍当书"鸿上关"或"常山关"、不书"倒马关"。**"倒马关"之名**——起于**北宋以后**（北宋 + 金 + 元）、至明景泰 / 嘉靖间重修后正式定名；明长城"内三关"格局下、倒马关与居庸 + 紫荆并称。**本 doc 标题用"倒马关"为现代通行名、但行文中提示 755 时的旧名**。

**唐倒马关的行政归属**：**唐承隋制、置定州**——治安喜县（今河北定州）；**天宝元年（742）改定州为博陵郡**——倒马关属博陵郡境。**定州 / 博陵郡**——领安喜 + 北平 + 鼓城 + 恒阳 + 新乐 + 义丰 + 深泽 + 唐县 + 望都 + 陉邑等县；**倒马关一带属唐县境**——唐县（今河北唐县）领飞狐陉东端山地与倒马关一带。**北面五回县（属易州 / 上谷郡）与唐县接壤**——五回县治在今河北涞源东南、辖蒲阴陉东段山地；**倒马关在唐县境、五阮关（即后世紫荆关）在五回县境、两关分属两州两郡**。

**与河东代北的关系**：**飞狐陉是河北道定州 / 易州 ↔ 河东道云州 / 蔚州的核心通道**；**唐时蔚州**——天宝元年改为安边郡、领飞狐县 + 灵丘县 + 兴唐县；**飞狐陉自蔚州境向东南穿太行**——倒马关在陉道东端、是出陉入河北平原的物理门户。**唐时此道驿路**——通蔚州 + 云州 + 朔州 + 代州——是河东代北与河北平原的最直接通道之一。

**安史前夕**（即 755 这一刻）——再过半年安禄山起兵于范阳；**叛军主力沿河北平原南下取洛阳**；**但叛军一部需经飞狐陉西出代北、召集同罗 + 奚 + 契丹塞外援军**——倒马关因此成为叛军物理通道之一。**755 时倒马关（鸿上关 / 常山关）仍是和平年代的关塞**——半年后即成安史之乱的物理通道。

### 此刻

天宝十四载这一年的倒马关——**关塞严整、守军按例屯戍、车马驿西通蔚州代北、东通定州河北平原**。

**关城状态**：盛唐 755 时倒马关（鸿上关 / 常山关）主体仍立、有关门 + 戍兵营 + 关吏；具体形制 + 守军规模史载零散。**关谷内外**：东门通飞狐陉东出口（今河北唐县西北），驿路至**定州 / 博陵郡 + 易州 + 常山郡**；西门通飞狐陉中段（今河北涞源 + 蔚县境），驿路至**蔚州治灵丘县 + 河东道云州**。**关使**为唐廷或节度使派遣——具体在 755 这一刻谁任关使、史无确载。

**755 时倒马关的政治氛围**：唐廷对河北 + 河东方向的安全意识在玄宗朝相对放松——**因北疆已无大战二十余年**（自开元末至天宝十四载）；安禄山节度使专制范阳 + 平卢、兼任河东节度使（751 起）、已专制北边十余年；**唐廷未察觉安禄山有反意**——倒马关一带亦无特别戒备。**这是 755 时唐廷对河北河东方向的盲点**——半年后即知不可挽回。

**755 末 + 756 的倒马关命运**：**安禄山十一月起兵于范阳**——主力**南下**沿河北平原攻取洛阳 + 西攻潼关；**但叛军一部经飞狐陉西出代北**——召集同罗 + 奚 + 契丹援军；倒马关因此成为叛军塞外援军南下的通道之一。**至德二载（757）史思明仍据范阳**——**倒马关一带成唐军 vs 叛军反复争夺地**：唐廷北征叛军、叛军塞外援军南下、皆经此关。**这是倒马关在盛唐唯一一段直接关涉天下大势的时间——但发生在 755 之后**。

**755 时的过关者**：**驿使**——唐廷自长安发往幽州 / 范阳节度使府的官文、有时取此路（飞狐陉 → 倒马关 → 定州 → 易州 → 范阳），较经太原 + 井陉 + 河北略远但避井陉险路；**商旅**——河东代北的盐、铁、毛皮南下入河北平原、常经此关；**戍卒**——河北道征发的戍卒赴蔚州 / 云州、亦经此关西出；**僧道行脚**——五台山在河东道代州、自河北平原朝五台、有一道驿路经倒马关 + 飞狐陉 + 蔚州 + 五台——这是盛唐河北僧侣朝五台的常路。**755 这一刻、过关之人或不知半年后这道驿路将载满叛军与援军的脚步**。

**755 时关塞的物质生活**——盛唐边防关塞、戍兵或自博陵郡境征发、或自河北道他州轮调；给养自定州 + 易州 + 常山郡转运；关吏馆在关城内、用于盘查行人 + 收取关税（盛唐关市税制零散、本 doc 不预设具体税则）；关塞外的烽燧 + 戍堡——分布于飞狐陉沿线、具体位置史载零散；**755 这一刻关塞外的小烽**——或仍按例燃柴、或已渐废、史无确载。

### 此刻意味

对天宝十四载的定州人 / 唐县人来说——倒马关（鸿上关 / 常山关）是**自家境内的关塞**：日常驿路过往、戍兵守关、商旅入关纳税。对河东代北的士人来说——这关是**东出河北平原的物理门户**；对河北平原的士人来说——这关是**西入代北高原的物理门户**。**盛唐诗中倒马关的诗咏**——零散；**飞狐陉**偶见于边塞诗（如高适、岑参涉燕代之作）、**但倒马关作为独立诗题密集化在明清**——盛唐时尚不是诗中常见地标。

唐人尚不知道——再过半年、安禄山将自范阳起兵、河北一路糜烂；倒马关将成为塞外援军南下 + 唐军反攻的通道；**这道太行东麓的狭谷**——半年后将见证盛唐最沉重的转折。**今倒马关古关已不复唐时形制**：**现存关楼主体始建于明景泰元年（1450）**、**嘉靖间大规模重修**；今游人来倒马关、所见的是**明长城段的倒马关楼 + 现代游览道**——**唐人在 755 所见的那座鸿上关 / 常山关**：千二百多年后**形制已不可复原**——盛唐倒马关的物理终点是明代长城工程。

## 子条目

### 太行八陉之飞狐陉

**太行八陉**——自南而北八条穿越太行山脉的天然孔道：轵关陉、太行陉、白陉、滏口陉、井陉、**飞狐陉**、蒲阴陉、军都陉。**飞狐陉**即倒马关所在——位太行八陉自南而北第六陉。**飞狐陉的地形**：自代北蔚州飞狐县（今河北蔚县一带）向东南穿太行山——至河北平原定州西界（今河北唐县西北）出山；**陉道全长约二三百里**、是**代北与河北平原的最直接通道**之一。**倒马关扼飞狐陉东端**——立陉道东出口处。

### 倒马关 ≠ 飞狐县 / 飞狐陉中段

**飞狐县是蔚州属县**（今河北蔚县一带、Tang 755 时属蔚州/安边郡）；**飞狐陉是整段陉道**——主体在蔚州境、东端开口在定州唐县境；**倒马关只扼飞狐陉东端**——不在陉道中段、更不在飞狐县内。**这一点常被混淆**：现代地图上"飞狐口"或指陉道某一段、或指倒马关本身、或指飞狐县；本 doc 严格区分——飞狐陉是陉道、飞狐县是县名、倒马关是东端关塞。

### 倒马关 ≠ 五阮关（南北分列、两州两郡）

**五阮关**（即后世"紫荆关"——Tang 755 时此关本名"五阮关"或《水经注》之"子庄关"）在**易州 / 上谷郡五回县**（今河北易县西北、太行山中）、扼**蒲阴陉**咽喉——蒲阴陉是太行八陉自南而北第七陉、主体在易县至灵丘一线；**倒马关**在**定州 / 博陵郡唐县**（今河北唐县西北）、扼**飞狐陉**东端；**两关南北相距约百里**、分属两州两郡（北易州 / 上谷郡 ↔ 南定州 / 博陵郡）、亦分属两陉。**唐时两关分立、各有守军**；**明代起紫荆 + 倒马 + 居庸并称"内三关"**——但 Tang 755 时尚未有此并称、亦不书"紫荆"之名。

### 唐时关名"鸿上"或"常山"——不叫"倒马"

**Tang 755 时此关官方文牍当书"鸿上关"或"常山关"**——前者沿北魏《地形志》关名、后者因关塞西邻常山郡（治真定县、今河北正定）。**"倒马关"之名起于北宋以后**——北宋 + 金 + 元间渐通行；明景泰 / 嘉靖重修后正式定名。**因此 Tang 755 这一刻、此关名不叫"倒马关"**——本 doc 行文中提示这一点；标题用"倒马关"是现代通行名、便于检索。

### 中山国西部山口（战国遗迹背景）

**战国中山国**（BC 414-296）——白狄族建立、领有今河北中部太行东麓诸城；都灵寿（今河北平山）；**中山国西部疆域以太行山脉为屏**——倒马关一带为中山国西部山口、扼太行陉道。**赵武灵王 BC 296 灭中山**——中山国故地入赵；**汉景帝复置中山国**（同名异质——为汉宗室封国、非战国中山国）；**倒马关一带历经战国中山 → 战国赵 → 汉中山国 → 北朝**——历代皆视为战略要地。

### 安禄山起兵与倒马关（755 末 + 756-757）

**天宝十四载十一月（755 末）**——安禄山起兵于范阳；**叛军主力沿河北平原南下取洛阳**；**但叛军一部经飞狐陉西出代北**——召集同罗 + 奚 + 契丹塞外援军南下；**倒马关因此成为叛军塞外援军南下的物理通道**。**至德二载（757）**——史思明仍据范阳、叛军在河北一带反复；**倒马关一带成唐军 vs 叛军反复争夺地**——具体战事史载零散。**这是倒马关在盛唐唯一一段直接关涉天下大势的时间——但发生在 755 之后**。

### 关谷内的水（滱水支流）

倒马关所在的飞狐陉东端——**沿一条山溪展开**；山溪自西北而东南穿关谷、出关谷入河北平原；**这条山溪属滱水水系**（即今唐河上游）——《水经注·滱水》载滱水出代郡灵丘县（今山西灵丘）、东南流经飞狐陉、东南入定州境（今河北唐河）。**盛唐过关者**——必沿此溪行走；溪水冬不冻、夏不涸——是关谷驿道的物理 anchor。**今关谷内仍有此溪**——但水量较唐时减少；溪两岸有明代关楼 + 现代公路共构一道**两千年关谷交通史的物理叠层**。

### 唐代定州（博陵郡）的关塞地位

**定州 / 博陵郡**——河北道上州、领县十；治安喜县（今河北定州）；**州境太行东麓、跨平原与山地**——东半在河北平原、西半在太行山中；**境内一大关**：**倒马关**（西北、扼飞狐陉东端、属唐县境）；**北邻易州 / 上谷郡**——其境内另有**五阮关**（即后世紫荆关、扼蒲阴陉、属五回县境、距倒马关约百里）；**两州两关合护河北道西缘**——定州治安喜在倒马关东南约百五十里、为关后补给与节度的物理 anchor。**755 时博陵郡太守**——具体姓名史载零散；本 doc 不预设。**定州军政**——属河北道、并非直归范阳节度使专辖；但安禄山兼范阳 + 平卢 + 河东节度、势力实及定州一线、安史之乱起即叛军前锋必经之地。**这是 755 时唐廷对河北西缘关塞最大的安全隐患**——但唐廷未察。

### 唐县与倒马关（行政细节）

**唐县**——唐定州 / 博陵郡属县；治所在今河北唐县；**辖飞狐陉东端山地 + 倒马关**——县境跨太行东麓与河北平原西缘。**"唐县"县名**——传起于尧封丹朱于此 / 古唐国故地之意；汉唐相沿、Tang 755 时仍循旧名。**Tang 755 时唐县属定州 / 博陵郡**；**乾元元年（758）复定州**——唐县归属未变。**唐县是倒马关的直接行政母县**——关使、关吏、戍兵的籍贯与给养、皆由唐县与博陵郡承担。**注**：北面五回县（属易州 / 上谷郡）辖蒲阴陉东段山地与五阮关（后世紫荆关）、与唐县在太行东麓接壤——两县常在文献中相邻并提、但分属两州。

### 关塞的几个名号（鸿上 / 常山 / 倒马 之辨）

**鸿上关**——北魏《魏书·地形志》正式关名；"鸿上"或与境内"鸿上山"（即太行东麓某段山脊）相关；亦或与"鸿"字（大、雁、北方意象）相关、史无确证。**常山关**——因关塞西邻**常山郡**（治真定县、今河北正定）得名；常山郡是西汉以来河北中部大郡、**常山赵子龙**（即三国赵云）即此郡人；唐时常山郡天宝元年改为恒山郡（避玄宗讳）、**乾元元年（758）复为常山郡**。**倒马关**——名见于北宋以后；民间附会"过关马倒不能起"；正史中"倒马关"作为定名见于明代。**Tang 755 时此关的官方名号**——文牍当书"鸿上关"或"常山关"；俗称或已有"倒马"之名萌芽、但未通行。**本 doc 标题用"倒马关"**——为现代检索便、亦因明清以来已成定名；行文则反复提示 755 时的旧名、不让现代名遮盖唐代真实。

### 北方民族经此关南下的千年史

**倒马关 + 飞狐陉 + 五阮关（后世紫荆关）**——共同构成**北方民族南下中原的物理通道集**之一。**战国**——林胡 + 楼烦经此南下骚扰赵燕；**汉**——匈奴经此南下；**北朝**——鲜卑 + 柔然 + 突厥经此南下；**唐**——回纥 + 契丹 + 奚的使节 + 商旅 + 兵马经此往返；**辽金元**——契丹 + 女真 + 蒙古经此南下立都北京；**清**——满洲已自东北入关、倒马关南向作用减；但塞外蒙古朝贡 + 商旅仍经此关。**这道太行东麓的狭谷**——见证了北方民族南下中原**两千年**的物理通道史；**755 这一刻**——是这条通道史上一个仍未爆裂的节点、半年后即裂。

### 唐河（滱水）水系与倒马关（地理纵深）

**滱水**——《水经注》立专卷（卷十一）；源出代郡灵丘县高氏山（今山西灵丘西北）；自西北而东南、穿太行飞狐陉、经倒马关一带入河北平原；**至定州境**——汇入沙河（今唐河 + 沙河汇流）；**再东南入瀛州**——经河北平原诸郡、最终入海。**滱水自唐前即是飞狐陉的物理 anchor**——陉道沿溪行、关塞依溪立；**倒马关一带的山溪**——即滱水上游或支流。**今唐河水系**——上游在山西灵丘 + 河北涞源；中游在河北唐县 + 定州 + 望都；下游在保定 + 雄县入大清河（白洋淀水系）；**1958-1960 间建唐河水库（今西大洋水库）**——库区在唐县境、距倒马关数十里；**滱水自唐至今水系基本未大改**——但水量较唐时减少甚多。

## 历史长河（755 → 今）

**中晚唐**　**安史之乱（755-763）期间**——倒马关（鸿上关 / 常山关）是叛军塞外援军南下 + 唐军反攻的物理通道；**763 年安史之乱结束**——关塞回归唐廷。**中晚唐**——河朔三镇（成德 + 魏博 + 卢龙）与唐廷反复；倒马关一带处**成德 + 卢龙**两镇接壤地、常被卷入河朔藩镇与中央军的拉锯。**唐宪宗元和年间**（806-820）——讨成德 + 平卢之役、唐军屡经此关；**唐末黄巢之乱**（875-884）——黄巢军未涉此关、但藩镇间割据加剧、关塞控制者多次易主。

**五代**　**五代间倒马关多次易主**——后梁、后唐、后晋皆控之；**后晋天福三年（938）石敬瑭割燕云十六州予契丹**——**北面易州 / 蔚州随幽云归契丹（辽）**——倒马关一带（定州境）虽未直接入辽、但飞狐陉北段已在塞外政权手中、倒马关由此成中原王朝与契丹的前沿。**后周显德六年（959）周世宗北伐收复瀛州 + 莫州 + 易州**——河北西缘暂归后周、旋归北宋。**北宋时倒马关成宋辽边境关塞之一**——宋廷以之防辽南下；**"倒马关"之名渐通行**——北宋 + 金 + 元间。

**辽金元**　**金破北宋后**——倒马关归金；**元承金制**；**辽金元三朝、倒马关时为边塞时为内地**——具体形制 + 守军史载零散。**金中都**（今北京）立国后——倒马关在中都西南、为内地关塞；**蒙古破金**——成吉思汗 + 木华黎部多次经此一带；**元大都**（今北京）立国后——倒马关在大都西南、为内地关塞、军事意义渐弱。

**明**　**洪武年间明军北伐、收复倒马关**；**景泰元年（1450）大规模重修**——立今所见倒马关楼之始；**嘉靖间扩建关墙 + 加固关城**——形成今所见**明长城倒马关段**的格局。**明倒马关与紫荆 + 居庸并称"内三关"**——属真保镇辖境、守北京西南门；**明代倒马关守军常额数千**——是九边重镇之一的物理 anchor。

**清**　**清初倒马关沿明制**；**清中后期渐失军事意义**——清廷与蒙古结盟、塞外不再是边境；倒马关 + 长城渐废。**清代地方志**——《直隶志》《保定府志》《唐县志》皆有倒马关条；**清人怀古诗**——多咏明代关塞 + 鸿上 / 常山旧名；**清末民国间倒马关一带**——为太行东麓商旅集散地、关楼仍立但戍兵已撤。

**民国前期**　**1909 京张铁路通车**——线在妫州 / 居庸关一带；倒马关无铁路、仍为公路 + 驮路；**民国 1928 直奉战争 + 1933 长城抗战**——倒马关一带亦有战事；但倒马关本身未成主战场。

**民国至今**　**1937 年抗战间倒马关一带**——是晋察冀边区的物理 anchor；八路军在飞狐陉一带活动；**著名的"狼牙山五壮士"**事件（1941 年 9 月）即发生在易县狼牙山——距倒马关约一二百里、属同一片太行东麓战区；**倒马关明长城段 2001 年列全国重点文物保护单位**（与紫荆关 + 内长城段合并）；**2010s 起倒马关明长城逐步修缮**——今游人来倒马关、所见的是**明长城段的倒马关楼 + 现代游览道**。**唐人在 755 所见的那座鸿上关 / 常山关**——千二百多年后**形制已不可复原**：盛唐倒马关的物理终点是明代长城工程；今所见皆明物。

## 与其他关塞的并立

**与居庸关（北门）**——居庸关在妫州境军都山中、控军都陉；倒马关在定州境太行东麓、控飞狐陉东端；**两关南北相距约二三百里**、皆护幽燕方向。**与五阮关 / 后世紫荆关（中门）**——五阮关在易州 / 上谷郡五回县境、控蒲阴陉、距倒马关约百里；**两关分属易州 / 上谷郡 ↔ 定州 / 博陵郡**、分扼两陉。**与井陉关（南门）**——井陉关在恒州 / 常山郡西、控井陉（太行八陉第五陉）；倒马关在井陉北面约二百里；**两关相距虽远、但同为河北 ↔ 河东的物理通道**。**这四关合看**——居庸 + 倒马 + 五阮 + 井陉——构成**Tang 河北道西缘的关塞链**；755 之后、这条关塞链将依次见证安史叛军的物理通过。
