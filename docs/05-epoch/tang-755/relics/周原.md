# 周原

## 一句话

**周天命所自来的物理 anchor**——岐山以南、漆沮二水之间的一片台塬；**约前 12 世纪中叶（具体年不可坐实）古公亶父自豳迁岐、营建邑居 → 王季 → 文王三代经营约数世（一说约 80 年）→ 文王晚年作丰、武王迁镐**；**自古公迁岐（约前 12 世纪中叶）至 755 约 1890 余年；自武王迁镐（BC 约 1046）周原退为旧邦、至 755 约 1800 年**——岐山南麓的田野上仅余地名、周公庙、与《诗经·绵》所述的山形水势；是岐州扶风郡治雍县以东 80-100 里的一片怀古之地。

## 形貌

周原在唐关内道**岐州扶风郡**——**天宝元年（742）改岐州为扶风郡、治雍县**（今陕西凤翔县），**至德二载（757）升凤翔府**——755 时尚未升府。周原的范围在**雍县以东**——具体在**岐山以南、漆水以西、沮水以东**的一片台塬上（今陕西宝鸡市岐山县京当镇 + 扶风县黄堆乡一带）；**距岐州治所雍县东约 80-100 里**。

**地势**：**北倚岐山**（海拔约 1600-1900 米的关中北山支脉、东西横亘）、**南面渭河谷地**——周原本体是**岐山南麓的一片黄土台塬**、海拔约 600-900 米；**两条小水夹其间**——东为漆水（今漆水河、北流入渭）、西为沮水（今沮水/雍水、亦北流入渭）——"漆沮"二字 在《诗经·绵》"民之初生、自土沮漆" 句中作为周人发祥地的水文标识。

**山势**：**岐山** 是周原的北靠——岐山自西向东走、主峰在岐山县城北约 30 里；**岐山有两峰对峙**、《国语》所谓"鸑鷟鸣于岐山"即指此。**南面渭河**——周原台塬下是渭河北岸的冲积地、是关中最肥沃的一段田野。

**周原本体**：**东西约 70 里、南北约 20 里**的一片台塬——**主体在岐山南麓 京当镇 + 扶风黄堆 一带**；台塬高出渭河谷地约 50-150 米——视野开阔、西北可望岐山主峰、东南可望渭水。

物候：关中西部、温带季风气候——四季分明、春夏多风、冬寒不烈；**周原台塬上**——黄土肥沃、宜麦粟；岐山南麓 + 渭河北岸 是关中最早的农业核心区之一。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

周原在唐人的回望里——是**周天命所自来的物理起点**、**关中最古的一片"圣地"**。

**周人先世**——**远祖弃**（后稷）佐尧舜稷农事、封于邰（今陕西武功）；**自弃以下十余世**——**公刘** 自邰迁豳（今陕西旬邑县一带）——《诗经·大雅·公刘》载之。**自公刘以下又十余世**——**古公亶父**（后追尊周太王）继位时、戎狄侵扰——**约前 12 世纪中叶（具体年不可坐实、旧说约 BC 1140 系高度估算）古公亶父率族人自豳南迁、度漆沮、踰梁山、止于岐下**——**这便是《诗经·绵》所记的"古公亶父, 来朝走马, 率西水浒, 至于岐下, 爰及姜女, 聿来胥宇"**。

**古公亶父在周原营建邑居**——《诗经·绵》详记其事：**"乃慰乃止, 乃左乃右, 乃疆乃理, 乃宣乃亩"**——划疆、理田、宣土、亩制；**"乃召司空, 乃召司徒, 俾立室家"**——置司空、司徒以筑室；**"其绳则直, 缩版以载, 作庙翼翼"**——夯版筑墙、立宗庙。**这是周人自游牧戎狄之地（豳）转入农耕文明的关键一步**——亦是周人"国家形态"的起点。

**古公亶父 → 王季（季历）→ 文王（姬昌）三代经营周原约数世（一说约 80 年）**——**王季继位约 BC 1100 前后**、继续在周原营建；**文王继位、在周原行政（具体年不可坐实）**——**周原达鼎盛**——是岐周（周原又称岐周）作为周人都邑的最后阶段。**文王晚年作丰**（具体年不可坐实、丰京在今西安西南沣水西岸）——**周原退为周人的旧邦**；**武王克商后迁镐京**（沣水东岸、与丰京隔水相望）——**周原虽已非都、但宗庙仍在岐**——周王室"岁时祭于岐周"。

**周原与镐京的关系**——周原是**周人自古公迁岐至文王作丰约数世（一说约 80 年）的发祥地 + 古都**；丰京是文王晚年新都；镐京是武王新都；**三者不可混**——后人或称"丰镐"指文王武王两京、或称"岐丰镐"包含周原；但周原是最早、最远、最古的那一处。

**唐人对周原的认知**——**自武王迁镐（BC 约 1046）周原退为旧邦、至 755 约 1800 年；若自古公迁岐（约前 12 世纪中叶）起算则约 1890 余年**；**确切位置已模糊**——但**岐山南麓 + 漆沮之间** 的大方位 唐人按《诗经·绵》《史记·周本纪》《元和郡县图志》 仍可考；**岐山县 + 扶风县** 一带田野上有"周公庙" + "古周原"地名、有相传为周公旦故宅的祠庙（即今岐山周公庙、唐代已立、玄宗朝有奉祀）。

### 此刻

天宝十四载这一年的周原——**自武王迁镐（BC 约 1046）周原退为旧邦、至此已约 1800 年**（若自古公迁岐 约前 12 世纪中叶 起算则约 1890 余年）；以**岐州扶风郡境内的一片古地**身份存在。

**岐州扶风郡的行政分量**——**天宝元年（742）改岐州为扶风郡**、辖九县（雍 / 岐山 / 扶风 / 麟游 / 普润 / 郿 / 虢 / 陈仓 / 岐阳 等——按《元和郡县图志》《新唐书·地理志》、755 时 雍县为治所、陈仓由武德元年/618 改虢县置、凤翔县/宝鸡县皆为 至德二载/757 以后改名、755 时尚未见）；**治雍县**（今陕西凤翔县）——**雍县是岐州治所、但周原本体在岐山县 + 扶风县境**。**周原距岐州治所雍县东约 80-100 里**——是岐州东境的核心古迹区。

**唐玄宗与岐州的渊源**——**玄宗朝多次西巡至岐州扶风郡（治雍县）**（开元 + 天宝间；755 时尚无凤翔府——凤翔府为至德二载/757 后建制）；**玄宗以"周文王受命"自况**——开元 + 天宝间册命中多以"周文王 + 武王"为典故——**周原作为"文王受命"的物理 anchor**——在玄宗朝的文化想象里分量极重。**注**：玄宗本人是否亲至周原 史源略零散；但**岐山周公庙 + 古周原 在玄宗朝有奉祀**——是确凿的。

**唐人的周原**——主要是三种存在形态：
- **地名**：岐山、扶风、雍、漆、沮 等地名皆可追到《诗经》《史记》——是"活的古地名"。
- **祠庙**：岐山周公庙（祀周公旦）、扶风一带相传文王 / 武王 / 周公旁支祠庙——唐代已立。
- **文学典故**：盛唐诗 + 赋中 "岐周 / 岐下 / 凤鸣岐山 / 周原" 是常见的怀古意象。

**"凤鸣岐山" 典故的源头**——**不是《诗经·绵》原文**（《绵》无"凤鸣"二字）、而是**《国语·周语上》"周之兴也, 鸑鷟鸣于岐山"**——后世化用为"凤鸣岐山"、成为"周受命" 的文学符号。**755 时盛唐人** 嘴里的"凤鸣岐山"——大体已是 周原 = 周受命之地 的标准修辞。

### 此刻意味

对天宝十四载的岐州人来说——周原就是**自家东边的古地**：田野上的黄土台塬、岐山南麓的山形、周公庙的香火、漆沮二水的流向——皆与《诗经·绵》对得上。对长安那一代士人来说——周原是**"周天命所自来" 的物理根**：玄宗以周文王自况、册命典故频出"岐周"、《诗经》是太学必读、岐山 + 周原是关中怀古的最深一层。

唐人尚不知道——再过 200 年（北宋初），周原仍是关中怀古之地、宋人多有过岐山赋诗者；再过千年（清乾隆 + 嘉庆间），关中考古学渐兴、周原一带始有零星青铜器出土——**清道光二十三年（1843）毛公鼎在岐山县董家村出**、**道光二十九年（1849）大盂鼎在岐山县礼村出**、**清光绪十六年（1890）大克鼎在扶风县任家村出**——三大鼎皆出周原。**1976-1977 陕西省考古研究院在岐山京当 + 扶风黄堆发掘**——出土**西周早期凤雏甲组建筑基址**（典型四合院、前殿后寝）+ **周原甲骨**（与殷墟甲骨并列的西周甲骨）+ 大量青铜器窖藏——**周原作为西周早期都邑的物理实证由此确立**。**唐人在 755 所见的那片黄土台塬**——一千二百多年后仍立、岐山未易、漆沮仍流、京当 + 黄堆 的田野下藏着古公亶父 + 王季 + 文王 三代经营的实物遗存。

## 子条目

### 古公亶父迁岐（《诗经·绵》核心）

**约前 12 世纪中叶（具体年不可坐实、旧说约 BC 1140 系高度估算）古公亶父率族人自豳南迁至岐下** 是周原作为周人发祥地的起点。**《诗经·大雅·绵》是这段历史的最早记录**——共九章、详记古公亶父自豳迁岐、营建周原、立宗庙、置百官 的全过程。**"古公亶父, 来朝走马, 率西水浒, 至于岐下, 爰及姜女, 聿来胥宇"**——是《绵》最有名的一句、述古公亶父与其妃太姜（姜女）同行迁岐、相宅而居。**《史记·周本纪》补述**——**古公亶父去豳、度漆沮、踰梁山、止于岐下**——"度漆沮"指渡过漆水沮水（周原东西两侧的小水）、"踰梁山"指越过梁山（豳与岐之间的山）。**孟子也述此事**——**《孟子·梁惠王下》"昔者大王居邠, 狄人侵之, ...去之岐山之下居焉"**——大王即古公亶父、邠即豳。**古公亶父被周武王追尊为周太王**——是周人始祖系的关键一代。

### 王季 + 文王在周原（约前 12 世纪后期—前 11 世纪中叶）

**古公亶父三子**——长太伯 + 次仲雍 + 季季历；**古公亶父欲传位于季历**（因季历子姬昌有圣德）、**太伯与仲雍乃让国南奔**（《史记·吴太伯世家》载之、后为吴国之祖）。**季历继位即王季**——继续在周原经营、扩张周人势力；**王季为商王文丁所杀**（《竹书纪年》载）——**子姬昌继位即周文王**。**周文王在周原行政数十年**（具体年不可坐实）——周原达鼎盛；**文王晚年作丰**（丰京在今西安西南沣水西岸、具体年不可坐实）——周原退为旧邦。**文王在周原的经营** 是周人"翦商"准备期的物理舞台；**《诗经·大雅·皇矣》《大雅·文王》** 详记文王受命于天 + 在周原行政 的事。

### 文王迁丰 + 武王迁镐（周原退为旧邦）

**周文王晚年作丰**（具体年不可坐实）——**丰京在沣水西岸**（今陕西西安市西南）——**周原 → 丰京** 是周人自岐山南麓向渭河下游核心区的迁移、是周人"东向克商"的物理准备。**武王克商（约 BC 1046）后迁镐京**——**镐京在沣水东岸、与丰京隔水相望**——**丰 + 镐 合称"宗周"**。**周原虽已非都**——但**周王室宗庙仍在岐周**——**"岐周宗庙"** 在西周一代受到周王岁时祭祀（《尚书》《诗经》多处述及）；**周原作为周王室宗教意义上的"祖地"** 一直延续到西周末。**西周亡（BC 771）后** 周原渐废、东周王室东迁洛邑、岐周宗庙不再受朝祀。

### 漆沮二水（周原的水文标识）

**漆水 + 沮水** 是周原的两条小水——**漆水** 在周原东侧（今漆水河、北流入渭）、**沮水** 在周原西侧（今沮水/雍水、亦北流入渭）。**《诗经·大雅·绵》"民之初生, 自土沮漆"**——以"漆沮"二字代指周人发祥地。**《史记·周本纪》"古公亶父...度漆沮"** 亦以漆沮二水作为豳与岐之间的地理标识。**唐 755 时漆沮二水仍流**——是岐州东境的两条小川、漆水东 + 沮水西、夹着周原台塬。**注**：唐代"漆沮"具体所指 与今水道略有出入——古"沮水"可能指今之雍水或更小的支流；本 doc 取大方位、不细辨。

### 凤鸣岐山（典故源头）

**"凤鸣岐山" 是后世化用的典故**——**不是《诗经·绵》原文**（《绵》无"凤鸣"二字）。**典故源头是《国语·周语上》**——"**周之兴也, 鸑鷟鸣于岐山**"（鸑鷟即凤凰一类的瑞鸟、《国语》以之喻周受命）。**自《国语》以下** "凤鸣岐山" 渐成"周受命" 的文学符号；**汉魏六朝以来诗赋中"凤鸣 + 岐山"频出**——盛唐 755 时已是常见怀古意象。**唐玄宗朝以周文王自况**——"凤鸣岐山" 是玄宗朝册命 + 文学中最常见的典故之一；**周原作为这一典故的物理 anchor**——在盛唐的文化想象里分量极重。**注**：凤鸣岐山 原文是"鸑鷟鸣于岐山"、不是"凤凰"——但唐人 + 后人 多以"凤"代之、本 doc 沿用"凤鸣岐山" 唐人通用语。

### 岐山周公庙（唐代已立）

**周公庙在岐山县城北约 7 里的凤凰山南麓**——**相传为周公旦故宅** + **后世周公祠庙所在**。**周公旦** 是文王第四子、武王之弟——**辅佐武王克商 + 摄政成王 + 制礼作乐**——是西周开国元勋。**周公庙在唐代已立**——**唐玄宗开元年间有诏修**（《元和郡县图志》《旧唐书·礼仪志》略涉）；**玄宗以周文王自况** + **以周公旦为辅政典范**——周公庙在玄宗朝的奉祀分量重于前代。**755 时周公庙是岐州东境最大的一处先秦祠庙**——是周原作为"周天命所自来" 的物理标识之一。**注**：周公庙具体何时始建史源略零散——但唐代已立 是确凿的。

### 周原作为"周天命"物理 anchor（唐玄宗朝文化语境）

**唐玄宗一脉以"周文王 / 武王"自况**——**开元 + 天宝间册命、改元、礼仪 多用《诗经》《尚书》周代典故**；**周原作为"文王受命" 的物理起点**——在玄宗朝的文化想象里是**最深的根**。**玄宗 + 杨贵妃 的"开元盛世 → 天宝由盛转衰" 叙事**——后人多以"周厉幽 + 唐玄"对举（《史记·周本纪》《国语·周语》记周厉王 + 周幽王亡国事——西周末与玄宗朝晚期有政治叙事上的呼应）；**周原是这条叙事的物理起点**——周人自岐周兴 → 镐京盛 → 镐京衰 → 东迁洛邑 → 周亡；**这条线在玄宗朝结束后约半年的安史之乱中**——以**玄宗 756 年西奔经岐州扶风郡（雍县；翌年 757 升凤翔府）**——形成奇特的物理回响。

## 历史长河（755 → 今）

**中晚唐**　**至德二载（757）升岐州为凤翔府**——是玄宗西奔（756 经岐州扶风郡、雍县）后肃宗 朝廷的政治选择；**自此凤翔府成为关中西部的核心州**——周原所在的岐山县 + 扶风县 皆属凤翔府辖。**中晚唐凤翔府** 是长安西门户、亦是周原所在——杜甫、白居易、李商隐等多人 有过凤翔（岐州）赋诗、涉周原 + 岐山者不少。**中晚唐周原** 作为怀古之地 + 周公庙 + 古地名 形态延续。

**两宋**　**宋仁宗 + 神宗朝**——**苏轼** 嘉祐六年（1061）任凤翔府签判——**苏轼有《凤翔八观》组诗**——其中《李氏园》《维摩像》等涉周原 + 岐山；**苏轼之周原诗** 是宋代关中怀古文学的代表。**北宋金石学渐兴**——**周原 + 关中一带始有零星青铜器出土**（北宋《考古图》《宣和博古图》录关中出土青铜器若干、其中部分今考为周原所出、但宋时尚未明）。**南宋至元** 周原作为怀古之地分量渐降。

**明清**　明清间周原一带渐复——岐山 + 扶风 + 凤翔 三县 皆有周原古地名 + 周公庙奉祀；**清乾隆 + 嘉庆间关中考古学渐兴**——**清道光二十三年（1843）毛公鼎在岐山县董家村出**、**道光二十九年（1849）大盂鼎在岐山县礼村出**、**清光绪十六年（1890）大克鼎在扶风县任家村出**——**三大鼎皆出周原**——是清代金石学的高峰；**自此周原作为西周青铜器窖藏区** 的物理地位确立。

**民国至今**　**1949 年后周原列陕西省重点文物保护单位**；**1976-1977 陕西省考古研究院在岐山京当 + 扶风黄堆发掘**——出土**西周早期凤雏甲组建筑基址**（典型四合院、前殿后寝、夯土基址保存完好）+ **周原甲骨**（与殷墟甲骨并列的西周甲骨、刻有占卜辞、是西周初年的实物文献）+ 大量青铜器窖藏；**周原作为西周早期都邑的物理实证** 由此确立。**1980s-2010s 陕西省考古研究院持续发掘**——周原范围内已发现 西周建筑基址 + 墓葬 + 作坊 + 甲骨 + 青铜器窖藏 数千处；**今周原遗址列国家级重点文物保护单位 + 周原博物馆 + 周原国家考古遗址公园**。**唐人在 755 所见的那片黄土台塬 + 岐山南麓 + 漆沮二水**——一千二百多年后仍在；**唐人未见的周原甲骨 + 凤雏建筑基址 + 西周窖藏青铜**——以考古的方式 自地下重出。**周原作为"周天命所自来" 的物理 anchor**——自古公亶父营邑（约前 12 世纪中叶、具体年不可坐实）至今约三千一百余年——是中国"文化记忆 + 考古实证" 最深的一片地。
