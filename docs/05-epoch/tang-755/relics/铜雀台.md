# 铜雀台 / 邺城三台

## 一句话

邺都西城墙上的曹操三座高台——**建安九年取邺、十五年筑铜雀**，自此邺城成为曹魏霸府、建安文学之核；755 时唐人见的是相州境内一片废城——**北周大象二年焚邺已 175 年**。

## 形貌

邺城在相州（治安阳）北约四十里——**漳水（浊漳水）南岸**，分**邺北城**（曹魏旧城）+ **邺南城**（东魏北齐扩建）两片相邻。

铜雀、金虎、冰井三台都立在**邺北城西墙**上——**自北向南依次为冰井台、铜雀台、金虎台**（《水经注·浊漳水》《邺中记》明文）。**铜雀台居中、最高**：《邺中记》记"铜雀台高十丈"（按汉魏一丈约 2.3 米，约 23 米）、"有屋一百二十间"；**冰井台居北**：高八丈（约 18 米）、屋一百四十间、上有冰室三所——是曹魏宫廷夏日藏冰之处；**金虎台居南**：高八丈、屋一百九间。

三台台基广而高——西望漳水、北望河朔平原；台上殿宇连甍、楼阁飞檐——是建安年间中原最壮丽的宫廷建筑群之一。**邺北城本身**："东西七里、南北五里"（按《邺中记》《水经注》）——是汉末曹操霸府所在；曹操**建安九年（204）八月**攻克邺城后即以此为大本营、十数年间扩为霸府。

唐时邺城是**相州境内一片废城**——城墙残墙、台基残土、漳水仍流。盛唐人自相州（安阳）北行四十里渡漳水、即至邺城遗址；登铜雀台残基西望漳水、北望平畴——这是 755 时凭吊邺都的实际行程。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

邺城与三台在唐人的回望里是**两段历史的叠加**——汉魏曹氏霸府 + 北朝东魏北齐都城——**两次为天下中心、两次成废都**。

**第一段：曹操霸府 + 建安文学（204-220）**。**建安九年（204）八月**——曹操攻克袁绍据守的邺城。**自此以邺为大本营**——朝廷虽仍在许都（许昌），曹操实际坐镇邺城。**建安十五年（210）冬**——曹操命作铜雀台、立邺城西墙之上；其后**十八年（213）作金虎台**、**十九年（214）作冰井台**——三台合称"邺中三台"。**建安十七年（212）冬**——曹操命三子曹丕、曹植、曹熊登铜雀台赋诗——**曹植《铜雀台赋》**自此传天下："立中天之华观兮、连飞阁乎西城；临漳水之长流兮、望园果之滋荣"。

**建安七子**——曹丕《典论·论文》所列：孔融、陈琳、王粲、徐干、阮瑀、应玚、刘桢。**邺下文人集团**实际以**三曹**及**王粲、陈琳、徐干、阮瑀、应玚、刘桢**等曹氏幕府文人为核心——**孔融在朝廷为官、与曹操不合、建安十三年（208）为曹操所杀，并非邺下幕宾**。"建安风骨"自此成为后世文学评价的最高标杆之一。**建安二十五年（220）正月**——曹操卒于洛阳，葬于邺西"高陵"（**今河南安阳西高陵村曹操高陵遗址、2009 年确认**）；**同年十月**——曹丕代汉立魏、迁都洛阳；邺城退为"五都"之一（魏五都：长安、谯、许、邺、洛阳）。

**第二段：东魏北齐都城（534-577）**。**东魏天平元年（534）**高欢迁孝静帝自洛阳至邺、立东魏；自此邺城再为天下中心。**东魏北齐间扩建邺城**——在邺北城南墙外另筑**邺南城**——两城相邻、合称"邺都"。东魏北齐 43 年间皆都邺、是北朝鼎盛文化中心之一。**北齐承光元年（577）**北周武帝灭北齐、邺城归北周。

**第三段：邺城焚毁（580）**。**北周大象二年（580）八月**——尉迟迥据邺城反杨坚、杨坚遣韦孝宽平之；**乱平后杨坚下令焚邺城**——殿宇尽毁、徙居民四十六里南至安阳新城。**自此邺都为废城**——三台残基仍立、城墙仍存、但已无主、无民、无生气。

**自隋至唐**——邺城作为废城 175 年（580→755）。隋唐两朝皆未再建邺城——相州治所在安阳新城。**唐人来邺城仅为凭吊**——见的是漳水南岸一片旷野中的残墙台基、与三国曹氏霸府旧址。

### 此刻状态

天宝十四载这一年的铜雀台 / 邺城三台——**夯土残基已废 175 年**。

**铜雀台基**仍立邺北城西墙上——夯土堆高约数丈（不复《邺中记》记的十丈）、上无屋宇；**金虎台基、冰井台基**亦如是。三台之间的西城墙残段仍可辨——东西七里南北五里的邺北城轮廓在田野间隐约成形。**漳水仍自北缘东流**——是这片废城最稳的物理 anchor。

**盛唐人凡至相州、必登三台凭吊**——但具体盛唐文集中咏邺三台的诗作并不密集；**主要凭吊集中在中晚唐**（参 历史长河段）。**755 时邺城的核心叙事**：**"曹操霸府之废墟、建安文学之故地"**——这一点对盛唐文人是基本共识。

**安史前夕**（即 755 这一刻）——再过一年（至德元载/756）安禄山在洛阳称帝、年底安军北返河朔；**至德二载（757）九月、十月**唐军郭子仪 + 李光弼率朔方军、回纥兵东出潼关、复东都洛阳——**这一年的相州、邺城旧址成唐军前进基地**——但唐军未在邺城驻、邺仍是废城。**乾元二年（759）九月相州大战**——郭子仪、李光弼九节度使七万兵围邺、安庆绪固守——后唐军溃、史思明南下、邺城再陷敌手。**这场战便是发生在 755 时空着的邺城遗址周围**——但本 doc 严守 755 时态、相州大战属"755 后"事件。

### 此刻意味

对天宝十四载的相州人来说，铜雀台与邺城三台是**州北四十里那片"曹操旧京"**——日常田耕、岁岁有过客来凭吊。对盛唐文人来说——邺城是**建安文学的物理 anchor**——读《三国志》《典论》《建安七子集》的人，都知道这一段诗酒酬唱的场所就在漳水南岸。

唐人尚不知道——再过五年（乾元二年/759）这片废城会成相州大战的中心；再过 90 年（晚唐会昌二年至四年/842-844），杜牧会在黄州（即齐安）刺史任上作《赤壁》"东风不与周郎便、铜雀春深锁二乔"——把铜雀台写进中国文学最有名的反讽里（注：**杜牧《赤壁》写的是"假设曹操胜、二乔成铜雀台之妾"的反事实——二乔是孙策、周瑜之妻，本 doc 严守史实**）。再过千年——三台基址仍立田中、夯土堆减半、上长荒草——**今游人来铜雀台遗址，所见的便是 755 唐人所见的同一片夯土**。

## 子条目

### 铜雀台（建安十五年/210 立）

**邺北城西墙上居中之台**——曹操建安十五年（210）冬命作。**《邺中记》记：高十丈（约 23 米）、有屋一百二十间**。**铜雀台是三台中最高、最壮、最有名**——曹操命三子登台赋诗、曹植作《铜雀台赋》于此。**台名"铜雀"**——传说因台屋上立大铜雀像（一说为"铜爵"——爵即雀的古字、汉魏间通）。台至北齐时仍立；北周 580 焚邺时台殿被毁、台基仍存。**今铜雀台基址**：在河北临漳县三台村西、夯土残基高约 4 米、面积约 50×50 米——是 755 时唐人所见同一夯土的现在。

### 金虎台（建安十八年/213 立）

**邺北城西墙上居南之台**——曹操建安十八年（213）命作。**《邺中记》记：高八丈（约 18 米）、有屋一百九间**。**金虎台得名于台屋上立金虎之状物**——**后赵建武帝石虎时（333-349 在位）因避石虎讳改"金凤台"**（《邺中记》后赵石虎改名记录；金凤台之名在后赵北齐间通行）。**唐人称谓既有"金虎"也有"金凤"**——本 doc 取唐人主流口径"金虎台"。**今金虎台 / 金凤台基址**与铜雀台南北相邻、同在三台村；夯土残基与铜雀台基相去约 50 米。

### 冰井台（建安十九年/214 立）

**邺北城西墙上居北之台**——曹操建安十九年（214）命作。**《邺中记》记：高八丈、有屋一百四十间、上有冰室三所**——"冰井"二字得名于台上的冰室：夏日藏冰、冬日取冰、是曹操霸府宫廷夏日饮馔的实用工程。**冰井台是三台中最具"实用功能"的一座**——专供藏冰；铜雀以登赋著名、金虎功能盛唐史源未详（或为日常宫廷台殿），冰井独以实用入史。**今冰井台基址**已大半倾覆——夯土残基比铜雀、金虎更不显。

### 邺北城与邺南城

**邺北城**：曹操霸府旧城——东西七里、南北五里、漳水流城西墙外。建安九年（204）攻克后曹操扩为霸府；自此七十余年（204→273 西晋取邺）皆为北方重镇。**邺南城**：东魏天平元年（534）高欢建——南接邺北城南墙、规模大于邺北、是东魏北齐都城主体。两城合称"邺都"。**北周大象二年（580）焚邺**——两城殿宇尽毁。**今邺城遗址**跨河北临漳县与河南安阳市——**1983 年起中国社科院考古所邺城考古队系统发掘**——确认邺北城西墙 + 三台基址 + 邺南城宫城基址。

### 曹植《铜雀台赋》

**建安十七年（212）冬**——曹操命三子曹丕、曹植、曹熊登铜雀台赋诗。**曹植 21 岁**——立成《铜雀台赋》："立中天之华观兮、连飞阁乎西城；临漳水之长流兮、望园果之滋荣……"——盛唐人读《文选》《曹子建集》皆知此赋。**这是邺中三台传至盛唐最重要的一篇文学文献**——比《邺中记》记录三台规模数字更接近 755 唐人对三台的实际感觉。

### 建安文学（建安七子 + 三曹）

**邺下文人集团**：曹操、曹丕、曹植 "三曹" + **王粲、陈琳、徐干、阮瑀、应玚、刘桢**（七子去孔融六人）等曹氏幕府文人——在邺城形成**中国文学史第一个"集群创作"现象**。**孔融**虽列建安七子（曹丕《典论·论文》），但在朝廷为官、与曹操不合、建安十三年（208）为曹操所杀，**不入邺下幕宾**。**"建安风骨"**自此成为后世文学评价的最高标杆之一。**盛唐文人皆习建安**——李白尤其崇敬建安诗、自比"咏怀九首"于阮籍。**755 时邺城对盛唐人最深的意义**：不是曹操的兵法、不是高欢的政权——是**建安文学的物理 anchor**。

## 历史长河（755 → 今）

**中晚唐**　**乾元二年（759）九月相州大战**——郭子仪、李光弼九节度使七万兵围邺、安庆绪固守；后唐军溃、邺城旁的战乱继续；至广德元年（763）安史之乱方平。**安史中邺城三台残基应再受过物理冲击**——但具体何处坍塌史载未详。**中晚唐凡至河北者皆凭吊邺都**——李白《邺都引》（盛唐至中唐间作）、王昌龄、刘禹锡、温庭筠等多有铜雀台咏；**晚唐杜牧《赤壁》（会昌二年至四年/842-844 黄州任上作）**"东风不与周郎便、铜雀春深锁二乔"——**自此铜雀台进入中国文学最有名的反讽典故**。

**两宋**　**北宋初期**——河北仍是辽宋边境、邺城距辽不远、为军事缓冲带；三台基址仍立田中、岁岁有耕作于上。**北宋诗人范成大、梅尧臣、欧阳修**等过相州凭吊三台留诗。**南宋失中原**后邺城在金代——金人对邺城没有大规模营建、三台续废。

**金元**　元代邺城旧址完全归于田野——三台基址上有农人耕作、有牧人放牧；**元代邺城仅在地方志中作为"古迹"提及**。**马端临《文献通考》**等元代史志记三台"夯土残基、土堆数丈"——可见物理保存的总体状态。

**明清**　明代邺城遗址在彰德府临漳县（明嘉靖后属彰德府）——**明嘉靖间立"曹魏邺都故址"碑**于铜雀台基；清代续立"魏武三台"碑等。**清代邺城已是河北南部一处"考古名胜"**——但因没有大型纪念建筑（不同于隆中、五丈原均有重修纪念物），凭吊者主要是文人。

**民国至今**　1949 年后邺城遗址列河北省重点文物保护单位、1988 年列全国重点。**1983 年起中国社科院考古所邺城考古队系统发掘**——确认邺北城西墙位置 + 铜雀 / 金虎 / 冰井三台基址 + 邺南城宫城与朱明门基址。**今邺城遗址跨河北临漳县（三台村）+ 河南安阳市**——三台基址在三台村西的田中、夯土残基高 4 米、面积约 50×50 米（铜雀、金虎、冰井各 一处）。**2008 年起临漳县始建邺城博物馆 + 邺城遗址公园**——展示三台基址与发掘出土物。**今游人来铜雀台遗址，所见的便是一片夯土堆**——与 755 唐人所见的"夯土残基"在物理层面是连续的——一千四百多年，未易。
