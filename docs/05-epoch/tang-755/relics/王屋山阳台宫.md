# 王屋山阳台宫

## 一句话

河南道怀州（河内郡）王屋县境王屋山的道教祖庭——**开元十二年（724）玄宗命茅山宗第十二代宗师司马承祯于此置阳台观**——王屋山为道教"十大洞天"之首"小有清虚天"，与终南楼观并称玄宗朝两大国家级道观。

## 形貌

阳台宫在怀州（河内郡）西北——治河内县（今河南沁阳）以西北约一百二十里。**王屋县是怀州属县**——县城在太行南麓与王屋山东麓之间的一片台塬上；阳台宫则在县境西部、**王屋山主峰天坛山东南麓**的一处山阶台地。

王屋山——太行山脉西南端的一支独立山系，**主峰天坛山海拔约一千七百米，南俯怀庆平原 / 黄河北岸**；山势孤峻、北背太行、南俯黄河北岸的怀庆盆地。盛唐人由洛阳出，渡黄河北上孟津，经河阳、温县、河内县（怀州治），再西北行入王屋县——是一段约三百里的山阶之路。**洛阳到阳台宫日行可三日**——这是阳台宫作为玄宗朝国家级道观的地理便利：离两京（长安、洛阳）皆不远，又有名山幽奇之实。

宫址在一处山间台地——南望黄河北岸怀庆平原、北倚天坛山主脊、东西两侧界天坛山的支谷。**唐时殿宇、斋堂、道士院凡数十座**（具体殿名盛唐史源未详，今所见殿宇多为元明清重修）；山中另有**天坛**——传为黄帝祭天之坛、亦为道教斋醮场所。两处由石阶相连、相距约十余里。

王屋一带的物候——背靠太行、面向黄河，夏日山间云气常起、冬日山顶有雪；秋后山中红叶层染，是太行南麓秋色最盛处之一。**唐人以"小有清虚天"称之**——意即山中别有一片清虚胜境，与人世之尘嚣相隔。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

王屋山阳台宫的来路两层——**先秦道教传说 + 玄宗朝敕建**。

**先秦传说**：王屋山自先秦已为道教名山——传说**黄帝于王屋山天坛祭天问道**（见《云笈七签》引古道经；先秦至两汉无明确史源、属道教托古叙事）。东汉以来王屋山渐为隐者修道之地——但**唐前王屋山的道教地位远不及终南、华山**。**南朝梁陶弘景（456-536）**创茅山宗（上清派之分支），传王远知、潘师正等——形成上清派"茅山一脉"的师承链；**司马承祯是潘师正的弟子、上清派第十二代宗师**——但他选择王屋山而非茅山为隐居与传道之地，是王屋山真正进入唐皇室视野的关键一步。

**玄宗朝敕建**：**司马承祯（646-735）**字子微、河内温人（今河南温县）——少时学道、事茅山宗第十一代宗师潘师正；学成后游历名山、最终隐于王屋山。**则天、睿宗、玄宗三朝俱召至京**——武周时期已有名；睿宗朝召至长安、命撰道经；**玄宗朝**两次召承祯至京问道（一次在开元九年/721、一次在开元十二年/724）。**开元十二年（724）**——玄宗命于王屋山为承祯置**"阳台观"**（一名"阳台宫"——后世升格通称；本 doc 标题用通行名）。**自此王屋山阳台宫便是李唐皇室的道教北线祖庭**——与终南楼观（关中线）并列。

**承祯之死（开元二十三年/735）**——年九十。葬于王屋山。其弟子**李含光（683-769）**继承上清宗师之位、续主茅山宗事；玄宗朝后期道教高层多出承祯门下。**阳台宫由承祯弟子续主持**——天宝间宫中道士百余、是关东北道教中心。

**玉真公主（持盈）**——玄宗皇妹、睿宗第九女——少时即入道、受度于司马承祯；**史载玉真常往来王屋阳台宫与京畿玉真观之间**——具体常驻王屋的年月程度、史源不稳，本 doc 不预设。盛唐李白、王维等诗人皆有献玉真公主诗——玉真公主是盛唐道教与文学交汇的核心人物之一。

**玄宗朝是李唐道教的鼎盛期**——玄宗自幼崇道、**亲注《道德经》**（《御注道德经》流传至今）、**开元二十九年（741）诏天下州县立玄元皇帝庙**（祀老子）、**天宝元年（742）改州为郡**（怀州改河内郡）即此一阶段道教措施同期；老子的官号自高宗乾封元年（666）"太上玄元皇帝"起、至天宝十三载（754）加号"大圣高上大道金阙玄元皇帝"——这一官号链下，**阳台宫所祀的老子在 755 这一刻是"大圣高上大道金阙玄元皇帝"**。

### 此刻状态

天宝十四载这一年的王屋山阳台宫——**是李唐皇室道教北线的巅峰一刻**。

**阳台宫**在 755 时是河南道（怀州/河内郡）最大的道观——殿宇数十、道士过百。**主殿祀老子**（"大圣高上大道金阙玄元皇帝"，天宝十三载新加官号定后的祀奉）——殿名盛唐史源未详；今所见阳台宫殿宇皆元明清后重建。**司马承祯墓**在宫西山中——开元二十三年（735）葬至 755 已二十年，弟子岁岁致祭。**天坛**则是宫之外山中的核心斋醮地——盛唐凡入王屋者必登天坛、礼老子、望黄河。

**"十大洞天"之说**——道教地理体系中"十大洞天 + 三十六小洞天 + 七十二福地"成形于唐——司马承祯著**《天地宫府图》**（已佚，《云笈七签》转录）即此一体系的奠基文本；**王屋山小有清虚天**列为十大洞天之首。755 这一刻的唐人——盛唐道教高层皆以王屋山为道教第一名山。**这是楼观（关中李唐祖庙）与王屋（道教第一洞天）的分工**：楼观重皇室祭祀（追祖老子）、王屋重道教内部宗师传承（上清派祖庭）。

盛唐文人来王屋访道者甚众——**李白曾有访司马承祯诗**（《大鹏赋》序载承祯赞李白"有仙风道骨"）——但承祯死于 735、李白与之相会在更早；**王维、岑参、刘禹锡等盛唐至中唐诗人**皆有王屋题咏。**专写阳台宫的盛唐前诗作不密集**——王屋的密集咏写多在中唐及以后。

天宝十四载的阳台宫——**朝廷玄元皇帝祀按例举行**；安史前夕（即 755 这一刻）——玄宗已 71 岁、仍在长安兴庆宫；玉真公主仍在世（约 754 改名"持盈"，约 762 卒）；李含光仍主上清宗师事（住茅山）。**阳台宫道众主持祭祀如常**——再过半年，安禄山起兵于范阳，铁骑南下渡黄河——**怀州、河内郡正在叛军南渡渡黄河攻洛阳的主路线上**；至德二载（757）唐军反攻、怀州一带反复争夺。**阳台宫在山中、远离怀州治及黄河渡口主路**——史载未直接受兵燹，但朝廷大祀中断、宫中道众散去甚众。

### 此刻意味

对天宝十四载的王屋县人来说，阳台宫是**县西北山中那座皇家敕建的道宫**——日常香火、岁岁朝礼，山中清虚天的传说由来已久。对洛阳那一代士人来说，阳台宫是**司马承祯曾隐之地、玉真公主所修之所**——离洛阳三百里、可数日还，是访道短游的首选之一。对朝廷来说——阳台宫是**李唐道教北线的祖庭**——其分量等于楼观之于关中，是玄宗朝"两京 + 两道观"格局的北翼。

唐人尚不知道——再过半年安史乱起、阳台宫祀渐减；再过百余年，唐末黄巢之乱中王屋山中道士散尽、宫宇荒废；再过五百年（金元间），全真道兴于终南——王屋作为上清派祖庭的地位渐降。**再过千二百年，阳台宫遗址尚存元明清各代殿宇、山中尚有司马承祯墓**——盛唐人在 755 所登的山、所朝的宫，与今人所见在物理层面仍有可见的连续。这是阳台宫与楼观共有的——**两座唐道教祖庭的物理 anchor 未断**。

## 子条目

### 阳台宫主殿 / 老子祀殿

**阳台宫的核心祀殿——立老子像之处**。开元十二年（724）司马承祯始建、开元、天宝间多次扩修。**755 时殿内祀奉的是"大圣高上大道金阙玄元皇帝"**（天宝十三载新加官号定后的祀奉）——这是与终南楼观同步的官称。**像之具体规模、衣冠形制盛唐史源未详**——今所见阳台宫遗址内的殿宇与像为元明清重修，盛唐原貌已不可考。**殿之具名"老君殿"或"玄元殿"是后世通称**——盛唐殿名本 doc 不预设。

### 天坛 / 天坛山顶

**王屋山主峰之巅**——传为黄帝祭天问道之坛（道教托古传说，先秦无明确史源）。盛唐时为阳台宫所属斋醮地——**道士岁时登坛、朝礼、望黄河**。天坛山顶视野极阔——南望黄河、北望太行、东望大伾、西望中条——**是太行南麓最壮观的一处天文望气之所**。今天坛山顶尚存历代道教刻石与坛址、为王屋山主要朝礼地。

### 司马承祯墓 / 子微洞

**司马承祯之墓**——开元二十三年（735）葬于阳台宫西山中。墓侧山崖有**子微洞**（一作"承祯洞"）——传为承祯生前修道之所；盛唐弟子岁岁致祭。**今墓址尚存**（地表标记为后世重立，墓主之骨是否仍在原位无确证）——为王屋山道教传承的物理 anchor。

### 司马承祯《天地宫府图》与"十大洞天"

**司马承祯著《天地宫府图》**——道教十大洞天 + 三十六小洞天 + 七十二福地体系的奠基文本；**王屋山列十大洞天之首"小有清虚天"**。原书已佚——北宋张君房辑《云笈七签》卷二十七转录其要。此体系自唐至今为道教地理之根本——盛唐人入阳台宫者、皆知王屋为"第一洞天"。**这是阳台宫之外、王屋山在道教史上的核心地位的文本依据**。

## 历史长河（755 → 今）

**中晚唐**　安史乱中怀州（河内郡）反复争夺——叛军、唐军、回纥援军皆经此线；阳台宫在山中受兵燹间接（具体损毁程度史载零散）。**自德宗朝起**朝廷道教祀渐减；但**唐宪宗、武宗朝**因崇道复兴有新致祭记录。**武宗朝（840-846）灭佛**——这一时期阳台宫地位再升、王屋山道众增；但武宗崩、宣宗复佛后阳台宫地位渐降。**唐末黄巢之乱（880s）**——怀州一带战乱、王屋山中道士散尽、宫宇荒废过半。

**五代北宋**　唐亡（907）后阳台宫失去"李唐皇室道观"的政治地位——五代间宫宇荒。**北宋初道教仍兴**（宋真宗朝崇道；徽宗朝奉道甚专）——大中祥符、政和年间皆有诏修王屋记录。**但宋代道教中心已转至武当、龙虎山、茅山**——阳台宫从盛唐"道教第一洞天 + 上清北祖庭"退为河南地方道观。

**金元**　**金大定七年（1167）王重阳（1112-1170）于终南山创全真道**——全真兴于终南、与王屋地理相去甚远、宗派亦不同（全真奉钟离权、吕洞宾等八仙为祖）。**元代全真兴盛**——王屋山地位本应衰，但**元世祖至元间敕修阳台宫**（元代正式定名"阳台宫"，自此"宫" 称固定）——阳台宫得以续传。**今阳台宫遗址内主体殿宇即元代格局基础上后世重建**。

**明清**　明洪武初敕修阳台宫；明嘉靖、万历间多次重修；清康熙、雍正、乾隆三朝俱有修缮记录。**清代阳台宫为河南道教重要场所**——但全国地位远不及武当、龙虎、茅山。

**民国至今**　1949 年后阳台宫先列河南省重点文物保护单位、后列全国重点。**1980 年代起**济源县（1988 撤县置市）始建**王屋山国家森林公园**——以天坛山为核心、阳台宫为山门入口。**2000s 河南省考古研究所发掘阳台宫遗址**——确认唐至元代道宫规模、出土开元至天宝间器物。**今游人入王屋山，可登天坛、观阳台宫殿（元明清主体）、谒司马承祯墓**——盛唐人在 755 所登的山、所朝的宫，与今人所见在物理层面有真实连续。这是王屋山阳台宫的**物理 anchor 未断**——与楼观（终南）并称两座可以亲履的唐道教祖庭。
