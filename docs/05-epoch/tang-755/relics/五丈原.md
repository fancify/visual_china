# 五丈原

## 一句话

岐州郿县斜谷口外的一处台原——**蜀汉建兴十二年八月、诸葛亮以五十四岁卒于军中**；与魏军司马懿对峙百余日的最后一阵未及打、北伐自此终——是中国军政史上最有名的一处"未竟之地"。

## 形貌

五丈原在岐州扶风郡**郿县与武功界**——**渭水南岸、斜谷口外**，是秦岭北麓的一处台原（黄土台塬）。**今属陕西宝鸡市岐山县蔡家坡镇五丈原村一带**（旧志多记"原在武功西十里余"，《水经注·渭水》引诸葛亮书；唐代地望近郿县/武功二县之界）。

地势：北临**渭水**、南依秦岭北麓、西界**斜谷**（褒斜道北口）、东界**石头河**——四面边界清晰。**台原本身**：海拔约 750 米、比周边渭水谷地高出约 100 米；东西约九里、南北约三里——是渭河南岸众多台原中最显眼的一处。**"五丈"得名**——按古制台高约五丈（汉魏一丈约 2.3 米、约 11 米）；但"五丈"二字在《水经注》《三国志》原典中已并立、是地名不必拘字面高度。

台原顶部相对平坦——可立营、可练兵、可观渭水北面魏军动向。**渭水自西向东在台原北缘流过**——盛唐时河面宽广、深可行舟。**斜谷自南而北、在台原西缘出秦岭**——这便是**褒斜道**北口，连接关中与汉中。**五丈原的战略位置**：扼**褒斜道北口、渭水南岸、关中入汉中咽喉**——是蜀汉北伐军可以扎营、可以观察魏军、可以选择进退的一处天然舞台。

唐时五丈原是**一片有名的台原**——上有荒草、间或有牧人放牧、间或有耕者播粟；**台上是否已立诸葛纪念建筑、史载未详**——本 doc 严守"盛唐时未见明确武侯纪念建筑"口径。台原本身、渭水、斜谷三件物事——是 755 时唐人凭吊诸葛亮的实际依凭。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

五丈原在唐人的回望里是**诸葛亮第五次北伐病卒之地**——**距 755 已 521 年**。

**蜀汉建兴十二年（魏青龙二年；234）二月**——诸葛亮自汉中出兵第五次北伐。**军凡十万**——出**斜谷**（褒斜道）、过**秦岭**北口、至关中。**诸葛亮据武功五丈原**——与司马懿魏军**对峙于渭南**（《三国志·诸葛亮传》"与司马宣王对于渭南"）。**司马懿引军济渭、背水为垒**（《晋书·宣帝纪》"百姓积聚皆在渭南……遂引军而济、背水为垒"）——是司马懿主动放弃渭北防线、移驻渭南与蜀军直接对峙。**这是诸葛亮北伐的最远点、亦是与魏军最近距离的对峙位置**。

**自二月至八月、对峙百余日**——诸葛亮多次挑战、司马懿坚守不出。**诸葛亮屯田于五丈原**——令士兵"杂耕渭滨、为久驻之计"。**司马懿则坚壁清野**——固守军营、不应战。**建兴十二年秋八月**——**诸葛亮疾病、卒于军、时年五十四**（《三国志·诸葛亮传》原文；具体日不见正史，本 doc 严守"秋八月"口径）。**遗命**：长史杨仪、护军姜维、丞相参军费祎等密不发丧、徐徐退军入斜谷。**司马懿闻蜀军退、追之至赤岸**——见蜀军反旗鸣鼓、疑亮未死、不敢逼近——史称"死诸葛走生仲达"（《晋书·宣帝纪》）。**蜀军安然回到汉中**。

**自 234 后**——蜀汉无再大规模北伐（姜维后期九伐中原是另一阶段）；**蜀汉景耀六年（263）魏军灭蜀**；**晋武帝太康元年（280）灭吴、三国统一**；**自西晋起五丈原成"诸葛卒地"凭吊符号**——但具体何时立武侯纪念建筑、史载未明。**蜀汉景耀六年（263）于沔阳（今陕西勉县）立武侯庙**——是史载最早的官立武侯庙、但**不在五丈原**。**西晋永兴中（304-306）李雄于成都立武侯庙**——亦不在五丈原。

**自东晋以下至唐——五丈原作为"诸葛卒地"是士庶共识**；**北魏《水经注》载"五丈原"地名、述诸葛旧迹**——可见盛唐前 五丈原 在地理志层面是已知坐标。**但盛唐前是否在原上立武侯祠**、史载未详——本 doc 严守。

### 此刻状态

天宝十四载这一年的五丈原——**一片台原荒坡**。

**台原顶部**——荒草过膝、间或有牧人放牧、间或有耕者；**台上是否有具名纪念建筑**——盛唐文集中明确写"五丈原武侯祠"的诗文极少，本 doc 不预设盛唐五丈原已立武侯庙。**渭水仍自西向东流过台原北缘**——是 755 时五丈原最稳的物理 anchor。**斜谷自南而北在台原西缘出秦岭**——盛唐褒斜道（即斜谷北段）仍是关中入汉中的主要驿路之一。

**盛唐人凡至岐州郿县者**——多有过五丈原驻足凭吊。**李白、王维、岑参**等盛唐诗人多写过诸葛亮——但**专写五丈原的盛唐前诗作不多**；**杜甫的"诸葛三诗"**（《蜀相》《八阵图》《武侯庙》）皆作于 755 后（《蜀相》上元元年/760 客成都作、《八阵图》大历元年/766 夔州作）——本 doc 严守"杜甫诸葛诗不在 755 前"口径。

**755 时五丈原的核心意义**：**"诸葛亮卒于此"** + **"出师未捷身先死"**——这是中国历史上最有名的"未竟之地"叙事。**台原本身不算大景**——盛唐人在郿县停留时上原一望、凭古意而已；真正的诸葛凭吊地仍以**成都武侯祠**（蜀汉武侯祠至唐已四百余年）与**隆中**（诸葛出山地）为主轴。

### 此刻意味

对天宝十四载的郿县人来说，五丈原是**县境西南那片"诸葛丞相卒地"的台原**——日常田耕、岁岁有过客来凭吊。对盛唐文人来说——五丈原是**"出师未捷"的物理 anchor**：诸葛亮辅蜀二十七年、北伐五次、最终在这片台原上五十四岁卒于军中——这是中国君臣相得叙事的悲剧收尾。

唐人尚不知道——再过半年安禄山起兵；再过一年潼关失、玄宗自长安出延秋门西奔、过郿县入散关南下——**玄宗西奔路线在郿县境内、距五丈原近**（具体玄宗是否登原一望、史载未详；本 doc 不预设）。再过 11 年（大历元年/766）杜甫在夔州作《八阵图》"功盖三分国、名成八阵图；江流石不转、遗恨失吞吴"——把诸葛亮悲剧推向中唐文学顶峰。**再过千年——五丈原台上会立起明清重建的诸葛庙**——今五丈原诸葛庙（今岐山县蔡家坡镇五丈原村）为明清重建、清咸丰、光绪间多次修缮、台原顶部"武侯衣冠冢"亦为后世附会。**唐人在 755 所见的那片荒坡**——位置在、台原在、渭水在；但具体物事是后世累积的、不是 755 时所见。

## 子条目

### 五丈原台原顶部

**海拔约 750 米、东西约九里、南北约三里**——是诸葛亮 234 北伐屯军、与司马懿对峙百余日的实际地。**台顶相对平坦**——可立营、可练兵、可观渭水北面魏军动向。**诸葛亮当年屯田于此**——史载"杂耕渭滨、为久驻之计"——盛唐人来五丈原、看到的便是这一片可耕可牧的台原。**今台原顶部多为农田 + 现代道路 + 武侯庙建筑群**。

### 诸葛亮卒地（建兴十二年秋八月）

**蜀汉建兴十二年（234）秋八月**——诸葛亮病卒于军中、年五十四（《三国志·诸葛亮传》"亮疾病、卒于军、时年五十四"——具体日不见正史）。**具体卒于五丈原何处、史载未详**——盛唐人凭吊时所对的是整片台原、不是某一精确点。**今五丈原诸葛庙内有"诸葛亮衣冠冢"**——但这是明清后附会的纪念物、不是诸葛亮真葬地（**诸葛亮葬于汉中定军山——其遗命"葬于定军山、依山为坟、冢足容棺、敛以时服、不须器物"——是中国君臣俭葬的典范**）。

### 死诸葛走生仲达（蜀军撤退典故）

诸葛亮卒后**遗命**：长史杨仪、护军姜维、丞相参军费祎等**密不发丧、徐徐退军入斜谷**。**司马懿闻蜀军退**——率军追之至赤岸（一作赤崖）；蜀军**反旗鸣鼓、若将向懿者**——懿疑亮未死、不敢逼近、退军。**当时百姓为之谚曰："死诸葛走生仲达"**（《晋书·宣帝纪》）。**这一段是诸葛亮临终最后一阵谋略**——亦是司马懿生平最有名的一次失误。盛唐人凡读《晋书》《通鉴》者、皆知此典。

### 斜谷（褒斜道北口）

**五丈原西界**——斜谷自南而北在台原西缘出秦岭。**斜谷是褒斜道的北段**——褒斜道是关中入汉中四条主要古道之一、与陈仓道、傥骆道、子午道并列。**诸葛亮 234 北伐即出斜谷至五丈原**——这是斜谷在三国军事史上最重要的一次走兵。盛唐褒斜道仍是驿道、商旅、军队往来频繁——但**斜谷此刻不是 755 时人的关注焦点**（玄宗 756 西奔走的是陈仓道而非斜谷）。

### 司马懿大营（渭南、背水为垒）

**渭水南岸**——是 234 司马懿魏军大营所在（《晋书·宣帝纪》"百姓积聚皆在渭南……遂引军而济、背水为垒"——司马懿主动放弃渭北防线、移驻渭南、与蜀军直接对峙）。**司马懿坚壁不出**——任凭诸葛亮挑战、屡次以"巾帼妇人之服"激将——皆不动。**这种"以静制动"是司马懿与诸葛亮百余日对峙的核心**——也是诸葛亮无法迫敌出战的根本原因。**盛唐人来五丈原**——可登台北望渭水、想象当年魏军背水为垒之地——但具体魏营位置史载不详、考古至今未确认。

## 历史长河（755 → 今）

**中晚唐**　五丈原未受安史之乱直接兵燹——盛唐至中唐间仍是台原荒坡。**杜甫的"诸葛三诗"**（《蜀相》上元元年/760、《八阵图》大历元年/766、《武侯庙》大历间作）——把诸葛亮悲剧推向中唐文学顶峰。**特别是《八阵图》**："功盖三分国、名成八阵图；江流石不转、遗恨失吞吴"——后两句的"遗恨"二字、是中国文学史上对诸葛亮最沉重的一笔。**中晚唐凡过郿县者多有五丈原凭吊**——韩愈、刘禹锡等有诗。

**两宋**　**北宋天禧二年（1018）**——朝廷诏修五丈原武侯祠（《宋会要辑稿·礼》——这是史载五丈原始立武侯庙的明确记录、但具体规制不详）；**北宋元丰间（1078-1085）**苏轼过五丈原、留诗"忆昨五丈原、骂尽中原贼"（一系于苏轼、史源略异）。**南宋初**——金兵屡至关中、五丈原区一度沦为战场；但因五丈原台原本身无大型纪念建筑、物理破坏不甚。

**金元**　元代五丈原诸葛庙规模史载未详。**马端临《文献通考》**等元代史志记五丈原"诸葛旧迹、有祠存"——可见元代台原上已有武侯祠。**元末明初战乱**——五丈原诸葛庙又一度衰。

**明清**　**明嘉靖、万历间**关中地方官多次重修五丈原武侯祠——今所见**五丈原诸葛庙主体格局为明清重修**。**清咸丰、光绪间**俱有修缮记录。**清代五丈原已是关中诸葛凭吊三大主轴之一**（另两处：成都武侯祠 + 隆中诸葛庐）。

**民国至今**　1949 年后五丈原诸葛庙先列陕西省重点文物保护单位、2013 年列全国重点。**1990s-2010s 多次考古调查**——确认台原顶部范围、渭水南岸地貌、斜谷北口位置；但**诸葛亮当年屯军的具体营垒位置——考古至今未确认**。**今五丈原诸葛庙在陕西宝鸡市岐山县蔡家坡镇五丈原村**——包括献殿、正殿（祀诸葛亮）、衣冠冢、八阵图苑等——为明清重修；台原顶部部分辟为蔡家坡镇一带耕地与现代道路。**今游人来五丈原**——所见的是明清重建的纪念建筑 + 千年未易的台原 + 渭水 + 斜谷——**台原、水、谷三件物事，与 755 唐人所见在物理层面是同一片**；具体纪念物事、是后世累积的。
