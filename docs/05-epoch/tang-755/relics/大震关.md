# 大震关

## 一句话

**关中入陇右门户、长安 ↔ 鄯州 物理 anchor**——陇州汧阳郡境陇山中一道关塞、东接关中、西通陇右；**唐 755 时仍是哥舒翰节度陇右往返长安的必经之关**；**再过七年（762）吐蕃陷陇右、大震关入吐蕃**——盛唐西门户的物理终点。

## 形貌

大震关在唐关内道**陇州汧阳郡境**——具体在**汧源县西陇山中**（今陕西省宝鸡市陇县西）；距陇州治**汧源县**约**六十至八十里**。**地势**：**陇山自北而南横亘**——东面是关中盆地西缘（汧水 + 渭水上游谷地），西面是陇右黄土高原（陇坻台塬 + 渭水北源）；**关塞立陇山东坡的山口**——南北山势峻拔、关沟自东而西穿陇山一线——**是关中盆地通往陇右高原的天然孔道**。

**大震关之险**：**东俯汧水谷地**（汧水东南流入关中）、**西望陇右高原**（黄土塬接渭水上游）；**关城就在陇山东坡的隘口**——东依汧水、西通秦州（陇右节度南境州治）方向；**一关守住关中-陇右两片大地的西方通道**。

**陇山即六盘山南段**——自宁夏南境向南延伸至秦岭北麓，**南北约六百里、东西约百里**；**陇山在关中盆地以西形成一道天然屏障**——古人称"陇坻"、"陇坂"。**大震关即扼陇山东西孔道**——是关中入陇右的最直接关塞。

**关城形制**：盛唐大震关具体形制 + 周长 + 门数史载零散——本 doc 不预设细节（**今所见大震关楼为清代重建、不是唐关原物**）。唐大震关大体应有**关墙 + 关门 + 戍兵营 + 关吏馆 + 驿馆**；具体规模未见可信细节。**关城外**：东通汧源县 + 陇州 + 长安方向；西通秦州 + 渭州 + 兰州 + 鄯州方向。

**关中三向西路中的位置**——关中盆地向西外开有三条主要通道：**大震关偏西**（陇州陇山中）扼入陇右 + 河西南路；**萧关偏北**（原州六盘山东麓）扼入河西北路 + 灵州；**散关偏西南**（凤州大散岭中）扼入山南 + 蜀地。**三关分守关中三向西出**——盛唐 755 时三关皆立、形势完整。

物候：陇山一带——温带半干旱 + 季风过渡气候；冬寒夏凉、春秋多风；山中有泉水 + 山溪——是过往兵旅 + 商旅的天然水源。陇山东坡较湿润、西坡入黄土塬较干——大震关恰立两侧气候转折的物理 anchor。**关城内外常有戍兵 + 驿马 + 商旅**——是盛唐陇右驿道的核心关塞之一。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

大震关在唐人的回望里——是**关中西门户 + 千年陇山关塞**。

**大震关的设立**：**汉初置西陇关**——是汉廷在陇山一线置关之始；《汉书·武帝纪》载**元鼎五年（BC 112）冬十月、武帝自雍始巡、西出陇关**——是汉武西巡至陇山一线的最早可考一次；**汉武帝出关传说不可坐实大震关名所本**——大震关名意，一说源于汉武西巡雷震、一说源于山势震慑——**皆传说**；**北朝改西陇关为大震关**——名遂沿用至唐。**自汉至北朝**，大震关一直是关中王朝西出陇右的关塞——多次易主、多次重修。

**唐大震关的政治军事地位**：**唐承前代之制、置大震关为军事关塞**；**大震关守军由唐廷直接指挥**——**大震关属陇州 + 与陇右节度（治鄯州）军政体系隔陇山相望**。**自唐初至盛唐 138 年（618-755）大震关无大战事**——是太平时代；但**大震关常驻兵约数百**——是日常守关 + 验商旅 + 收关税 + 备驿馆。

**大震关与长安、鄯州的关系**：**长安西行 + 经咸阳 + 凤翔（扶风郡岐州）+ 陇州 + 大震关**——出关后**沿陇山西麓南下经秦州 + 渭州 + 兰州 + 至鄯州**（具体里程史源略异，约一千八百唐里）——**这是盛唐长安通陇右节度治所鄯州的最直接驿路**。**陇右节度使**——自开元二年（714）正式设置、治鄯州；**辖境跨陇山东西**——**大震关是节度使往返京师的物理咽喉**。

**755 时陇右节度使是哥舒翰**——天宝十一载（752）拜陇右节度；**自此至 755 哥舒翰多次往返长安 + 鄯州、必经大震关**。**哥舒翰是盛唐西陲第一名将**——破吐蕃、收九曲、置宁塞军；**大震关是他往返京师与节度治所的物理 anchor**。**755 这一刻哥舒翰在长安养病**——半年后他将自长安出潼关御安禄山叛军，与大震关再无关涉。

**安史前夕**（即 755 这一刻）——**长安 + 陇右兵东调救京、自鄯州东行过大震关者罕**（多直接东行）；**大震关本身仍是平时关塞**。**大震关在 755 前夕的状态**——驻兵约数百、关楼严整、商旅通行不阻、驿馆按日轮值。

### 此刻

天宝十四载这一年的大震关——**关塞严整、驻兵数百、扼陇山东西孔道**。

**关城状态**：盛唐 755 时大震关主体仍立、有戍兵 + 关吏 + 驿馆；具体形制史载零散。**关城外**：陇山东西延伸；**关城内外的驿路**——东接汧源县驿道至陇州治 + 长安；西接秦州驿道至兰州 + 鄯州。**关使**为唐廷或陇右节度使派遣——具体在 755 这一刻谁任关使、史无确载。

**755 时大震关上的政治氛围**：**大震关是平时关塞**——关税收 + 商旅过关 + 戍兵巡逻；**关吏对入陇右的官员、戍卒、商旅** 多有验牒；**西向陇右 + 河西的人流密度** 远不如东向洛阳的潼关、但作为陇右第一关、地位重要。**陇右节度使军政文书 + 戍卒轮换 + 陇右商队 + 吐蕃朝贡使**——皆经大震关往返。

**755 时关下的陇右商旅**：**唐蕃在赤岭（今青海日月山）设互市**——开元二十二年（734）金城公主请于赤岭立碑、置互市（《新唐书·吐蕃传》）；**陇右商旅往返**——经大震关入陇右、过秦州 + 兰州 + 鄯州、抵赤岭互市；**关吏验商货 + 收关税**（《唐六典·卷六·尚书刑部·司门郎中》载关津之制）。盛唐大震关的商旅来去——是中原与吐蕃 + 河西诸胡 + 西域物货往来的物理 anchor。（注：**"茶马互市"制度名属北宋以后正式建制**——盛唐 755 时尚未成名、不宜用此词；唐蕃互市以缯帛 + 金银 + 马匹为主、茶虽已传入吐蕃但非互市主品。）

**755 末 + 756 的大震关命运**：**安禄山十一月起兵于范阳**——主力**南下**沿河北 + 河南方向；**大震关未涉直接战事**。但**756 七月玄宗西奔入蜀** —— 玄宗一行经岐州 + 散关入蜀、**未走大震关**（大震关向陇右、不向蜀地）；**756 七月太子李亨（后肃宗）北奔灵武**——**亦未走大震关**（取邠州、宁州、庆州、灵武一线、走萧关方向）。**大震关在 756 这一年——并非主战场、也非主驿路**；但在 757 后逐渐凶险。

### 此刻意味

对天宝十四载的陇州人 / 汧源人来说——大震关是**自家境内的关塞**：日常入陇右商旅必过、戍兵守关、关吏验牒、驿马轮换。对长安那一代士人来说——大震关是**关中西门户**：通往陇右 + 河西 + 西域的物理 anchor；**盛唐诗中"陇关"、"陇山"二字常出现**（王维、高适、岑参等边塞诗或涉），**"陇头流水、流离山下"**——是盛唐征戍诗的核心意象之一；大震关即"陇关"实指。

唐人尚不知道——再过七年（762）吐蕃东扩、陷陇右诸州、**大震关将随陇右失去唐控制**——自此约**百年**间大震关在吐蕃手中（762-849）；**唐宣宗大中三年（849）张义潮归义** + 唐收复陇右部分州县、大震关一度复入唐境；但**长安 ↔ 鄯州的旧驿路已断**——盛唐那条经大震关西去鄯州的物理通道、在中晚唐基本残废。

## 子条目

### 陇山即六盘山南段（不是秦岭）

**陇山**——自宁夏南境固原一带、向南延伸至陕西陇县西、长约六百里；**陇山即六盘山南段**——并不是秦岭。**这一点常被现代地理混淆**：**陇山在关中西北**、**秦岭在关中正南**；**大震关在陇山中**（关中西门）、**散关在秦岭北麓**（关中西南门）——**两关相距数百里、是关中西出两个独立的门户**。**陇山西麓即陇右高原 + 渭水北源**；秦岭北麓即关中盆地西南端。**唐人"陇头"、"陇坻"、"陇坂"皆指陇山**——不指秦岭。

### 大震关 ≠ 散关 ≠ 萧关（关中三向西路）

**关中盆地向西外开有三向通道、对应三关**：
- **大震关**（陇州汧阳郡境陇山中）——**关中入陇右**：长安 → 凤翔 → 陇州 → 大震关 → 秦州 → 渭州 → 兰州 → 鄯州；
- **萧关**（原州平凉郡境六盘山东麓）——**关中入河西北路 / 灵州**：长安 → 邠州 → 泾州 → 原州 → 萧关 → 灵州 / 凉州；
- **散关**（凤州河池郡境大散岭中）——**关中入山南 + 蜀地**：长安 → 凤翔 → 陈仓 → 散关 → 凤州 → 兴州 → 利州 → 益州。

**三关分守关中三向西出**——盛唐 755 时三关皆立、形势完整。**长安 ↔ 鄯州主线过大震关**；**长安 ↔ 凉州主线常过萧关**（亦可经大震关 + 兰州 + 凉州、稍远）；**长安 ↔ 益州主线过散关**。**这是唐代行政地理的物理细节、不可混淆**。

### 大震关名意（汉武西巡传说 / 山势震慑）

**"大震关"名意**——**汉武帝出关传说不可坐实大震关名所本**：一说源于汉武西巡至此雷震大动、一说源于陇山山势震慑——**皆传说**。《汉书·武帝纪》明载者为元鼎五年（BC 112）冬十月武帝自雍始巡、西出陇关；"元封元年（BC 110）雷震名关"实为附会。**汉旧名"西陇关"**——北朝改"大震关"——名遂沿用至唐。**"大震"二字**——既是地理 anchor、也是声音 anchor、还是历史 anchor；**盛唐人过大震关、未必能不想到汉武西巡这一历史回声**——唐人怀古、常以"大震"扣此意。

### 哥舒翰与大震关（752-755 陇右节度）

**天宝十一载（752）哥舒翰拜陇右节度使**——治鄯州；**自此至 755 哥舒翰多次往返长安 + 鄯州、必经大震关**。**哥舒翰是盛唐西陲第一名将**——**天宝十二载（753）破吐蕃、收九曲**；**天宝十三载（754）置宁塞军**——皆是盛唐西陲军政的高峰一刻。**大震关是哥舒翰这一段军政生涯的物理 anchor**——他每一次自长安出发去鄯州接节度、或自鄯州回京述职、皆要过此关。

**755 这一刻哥舒翰在长安养病**——年初他自陇右回京、病居宅邸；**安禄山起兵后**（755 十一月）—— 唐廷起复哥舒翰**领潼关防线、御安禄山叛军**（756 春哥舒翰自长安出潼关、守潼关）；**756 六月哥舒翰兵败潼关 + 被俘**——**自此哥舒翰再未西归大震关**。**大震关 ↔ 哥舒翰**这一段物理关联——是盛唐西陲军政与天下转折的微妙交集。

### 长安 → 鄯州主线（过大震关）

**盛唐长安通鄯州的主驿路**——具体里程史源略异，本 doc 取通称：
- **长安 → 凤翔（扶风郡岐州）**——约 300 唐里、沿渭水北岸西行；
- **凤翔 → 陇州（汧阳郡）**——约 200 唐里、沿汧水西行；
- **陇州 → 大震关**——约 60-80 唐里、入陇山东坡；
- **大震关 → 秦州（天水郡）**——约 200 唐里、出陇山西麓南下；
- **秦州 → 渭州（陇西郡）**——约 200 唐里、沿渭水西行；
- **渭州 → 兰州（金城郡）**——约 300 唐里、沿渭水入黄河；
- **兰州 → 鄯州（西平郡）**——约 400 唐里、沿湟水西行。

**全程约 1700-1800 唐里**——**大震关在此驿路的关中末端、是长安出发后第一个"出关"节点**。**这条驿路与河西路（经萧关 + 灵州 / 凉州）的分歧点**——在长安出发后即分；**两路在兰州或凉州一带重新相会**——是盛唐西陲驿路网的核心一段。

### 大震关税与商旅 / 唐蕃赤岭互市

**唐大震关收商旅过关税**——这是关塞的常规财政职能（《唐六典·卷六·尚书刑部·司门郎中》载关津之制）；**大震关税**主要针对入陇右 / 出陇右的商货 + 公私往来。**盛唐大震关税收**史载零散；**与潼关、玉门关等比、大震关税收规模较小**——因入陇右商旅密度不如东出洛阳；但**唐蕃赤岭互市**——开元二十二年（734）金城公主请于赤岭立碑、置互市（《新唐书·卷二一六·吐蕃传》）——其中原段商旅过大震关入陇右、关吏验商货 + 收关税。（注：**"茶马互市"为北宋以后正式建制**——盛唐 755 时尚无此制度名；唐蕃互市以缯帛 + 金银 + 马匹为主。）

### 762 吐蕃陷陇右与大震关失唐控制

**广德元年（763）吐蕃东扩大举 + 陇右诸州陆续陷蕃**——**大震关具体陷年待考**（一说宝应元年/762、一说广德元年/763）；**自此约百年**间大震关失唐控制（陷蕃—849）。**唐宣宗大中三年（849）张义潮归义军起兵 + 收复河西部分州县**——大震关一度复入唐境；但**长安 ↔ 鄯州的旧驿路已断**——盛唐那条经大震关西去鄯州的物理通道、在中晚唐基本残废。**这是大震关在 755 之后最沉重的转折**——盛唐西门户的物理终点是吐蕃东扩。

### 陇头流水——盛唐征戍诗中的大震关

**"陇头流水、流离山下"**——是盛唐征戍诗的核心意象之一；**大震关即"陇关"实指**。**王维《陇头吟》**："长安少年游侠客、夜上戍楼看太白"——所写戍楼即陇山一带关塞之一；**高适、岑参边塞诗**——多次涉陇关 + 陇头 + 陇坂。**盛唐人过大震关西行**——多有"出陇"之叹；**关东是关中故乡 + 长安繁华**、**关西是陇右黄土塬 + 戍卒征人**——大震关在物理上是这一文化分界的 anchor。**唐人陇头诗的密集化**——自盛唐始、至中晚唐延续；**大震关在唐诗中虽不如潼关、玉门关密集**、但**作为"陇关"实体**——是盛唐征戍诗最重要的物理 anchor 之一。

## 历史长河（755 → 今）

**中晚唐**　**至德二载（757）唐军收复长安**——大震关仍在唐境；**乾元二年（759）杜甫弃华州司功参军、西赴秦州**（或经陇坂 / 大震关一线、或别道）——**自秦州再南奔同谷不经大震关**（南奔同谷取祁山道、大震关在秦州东北方向）；**这是盛唐 → 中唐过渡的一段陇右诗踪**。**广德元年（763）吐蕃东扩大举 + 陇右诸州陆续陷蕃**——**大震关具体陷年待考**（一说宝应元年/762、一说广德元年/763）；**自此约百年**间大震关失唐控制。**唐宣宗大中三年（849）张义潮归义军**——大震关一度复入唐境；但**关中 ↔ 陇右的旧驿路基本残废**。

**五代**　**五代间陇山一带战乱频仍**——大震关多次易主、关塞规模渐降。**后唐 + 后晋 + 后汉**皆曾控大震关——但**陇右节度治所（鄯州）早已不在唐廷手中**、大震关失去战略价值。

**两宋**　**北宋初大震关重修**——属秦凤路；**北宋仁宗 + 神宗间**——大震关曾是宋与西夏 + 吐蕃诸部对峙的边境关塞之一；**南宋绍兴间**——大震关属金（北宋亡 1127 后）；**南宋亡后大震关失军事意义**。

**金元明清**　**元明清间大震关续作关塞、但规模渐降**；**明代大震关属凤翔府陇州**——是陇州西界关塞；**清初大震关重修关楼**——今所见关楼即清代遗构（不是唐关原物）；**清中后期大震关渐失军事意义**——清廷与西陲诸部结盟、陇山一线不再是边境。

**民国至今**　**1949 年后大震关古关遗址列陕西省重点文物保护单位**；**1990s-2010s 多次考古勘察**——确认大震关古关墙、关沟驿道、烽燧遗迹；**今宝鸡市陇县境**——所见的是**大震关古道遗址 + 关楼为清代重建 + 部分关墙犹存**。**唐人在 755 所见的那座大震关**——千二百多年后**主体已残**；盛唐那条经大震关西去鄯州的驿路——今已成宝中铁路 + 国道 G310 的现代驿道。**唐人在 755 所见的那座大震关 + 哥舒翰往返京师与鄯州的关沟**——盛唐西门户的物理终点是吐蕃东扩与百年陷蕃。
