# 统万城

## 一句话

**大夏赫连勃勃白城**——陕北靖边无定河北岸；**凤翔元年（413）始筑、约真兴年间（419 前后）宫城落成、北魏始光四年（427）攻克统万、大夏 431 亡国后退为废都**；**755 时已废 324 年**——白色夯土城墙仍立、唐人称"白城"、是夏州朔方郡治朔方县（隋岩绿县、贞观二年/628 改名朔方）所在。

## 形貌

统万城在唐关内道**夏州朔方郡治朔方县（隋岩绿县、贞观二年/628 改名朔方）**——具体在**无定河北岸的台塬**上（今陕西省榆林市靖边县红墩界镇白城子）。**地势**：陕北黄土高原南缘、无定河谷北侧的一片高台塬——海拔约 1300 米；东、西、北三面是黄土沟壑、南面是无定河——是十六国时期赫连勃勃选址的天然险要。

**统万城分东西二城**——东城东西约 850 米、南北约 500 米；西城东西约 800 米、南北约 770 米。**两城连成一体**——是十六国时期规模最大、保存最完好的都城遗址之一。**城墙**：**夯土掺白石灰**——筑成后呈白色、唐人遂以"白城"代统万；墙基厚约 16 米、残高 5-15 米。**赫连勃勃所谓"蒸土为城、其坚可砺刀斧"**（《晋书·赫连勃勃载记》原文）——是大夏立国的技术象征。

**城制特点**：
- **东西二城连体**——是早期都城规制的代表。
- **马面 + 角楼**：城墙上有夯土马面（外凸碉楼）数十处、角楼四处——是防御工事的代表。
- **西城西南角的"南宫"高台**：传为赫连勃勃宫殿主体；今残高约 25 米——是统万城最显的物理标识。
- **城内有大型礼制建筑基址**：含郊祀坛、宗庙等。

物候：陕北黄土高原南缘、温带大陆性气候——干燥、风大、冬寒夏热；周边无树木、是黄土沟壑与草原过渡地带。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

统万城在唐人的回望里——是**大夏国都的物理废墟**、**陕北最古的一座白色城**。

**赫连勃勃（约 381-425）**——**铁弗匈奴**人（南匈奴后裔与鲜卑混血、原姓刘）；东晋义熙三年（407）起兵于陕北、立**大夏国**（"夏"取自"大禹之后" 政治叙事）、自称大夏天王。**义熙九年（413）**——勃勃命**叱干阿利**为营宫监、于无定河北岸**始筑统万城**；前后历**约 6 年（413-419 间）筑成**；**"统万"二字意"一统万邦"**——是赫连勃勃政治抱负的体现。

**赫连勃勃的暴政**：营宫监叱干阿利**以严刑筑城**——史载"蒸土为城"（《晋书》原文）：实际工艺为以黏土掺白石灰夯筑（现代考古确认为白灰夯土 / 三合土）——筑成后墙体可砺刀斧。**叱干阿利筑城时若锥土能入一寸、即杀筑者及监督**（《晋书》原文）——筑城死者万计；**"夯土中混有人骨"为后世传说**，本 doc 不坐实考古结论。**这种暴政**是大夏立国 22 年间最阴暗的一面、亦是赫连勃勃名声"残虐"的来源。

**大夏的兴亡**：**义熙九年（413）建都统万**；**永和六年（420）取长安**——大夏达鼎盛；**勃勃于真兴七年（425）卒、子赫连昌继位**；**北魏太武帝始光四年（427）魏军攻克统万**、赫连昌被擒、大夏灭于北魏。**统万自此 22 年后变为废都**；北魏改置统万镇、后再立为夏州——**北魏、北周、隋、唐皆置夏州于统万旧址**。

**自北魏起统万作为夏州州治延续 300 余年**——但**统万的政治分量自大夏亡后骤降**：自十六国都城退为北朝边州、隋唐边境州。**盛唐 755 时统万城是夏州朔方郡治朔方县（隋岩绿县、贞观二年/628 改名朔方）所在**——但城内的人口、政治分量远不及大夏鼎盛期。

### 此刻

天宝十四载这一年的统万城——**已废 324 年（431 大夏亡 → 755）**；以**夏州朔方郡治朔方县（隋岩绿县、贞观二年/628 改名朔方）**身份存在。

**城墙、马面、角楼、南宫高台**——盛唐 755 时主体仍立、白色夯土未褪。**唐夏州的州治**：在统万城西城内——**夏州刺史驻、岁掌朔方郡军政**。**朔方节度使**（朔方军最高军政长官）治灵州（今宁夏灵武）、**夏州属朔方节度辖境**——是盛唐西北军事体系的重要一环。

**唐人对统万的认知**——**白城**：唐诗多以"白城"代统万。盛唐边塞诗 + 怀古诗中"白城"二字出现频繁：
- **李益**（748-829）有《暮过回乐峰》等多首边塞诗涉白城；
- **王建**（约 766-830）有《白城寨》等；
- **白居易**有"白城子下年年战"句（《新乐府》一系）；
- **岑参、高适**等盛唐边塞诗中亦有提及。

**注**：李益、王建、白居易诗 多作于 755 后（中晚唐）；盛唐前期边塞诗对统万的具体提及史源略零散。

**唐人对统万的政治记忆**——**赫连勃勃残虐筑城**的故事在盛唐已是边塞文学典故；**蒸土为城、人骨夯墙**的传说自十六国至唐宋持续传播。**统万作为"匈奴帝都" 的物理纪念**——是关内道北部最特殊的一座城。

### 此刻意味

对天宝十四载的夏州人来说——统万城就是**自家的州城**：白色夯土墙仍立、唐人居城内、官署驻南宫高台旁。对长安那一代士人来说——统万是**陕北边塞的物理 anchor**：边塞诗的实景、赫连勃勃故事的舞台、北朝至唐皇室"出身记忆"（武川镇 + 朔方 + 统万 等北方边镇）的西段。

唐人尚不知道——再过 50 年（约 9 世纪初），统万城仍是夏州州治；再过 200 年（北宋初），统万城被宋廷废为军州；**北宋淳化五年（994）宋太宗诏毁统万城**——以城墙过坚、防党项利用之；**自此统万自废**——白色夯土墙逐渐荒芜。再过千年（1996-2010s），**陕西省考古研究院多次发掘统万城遗址**——确认东西二城、南宫高台、城内礼制建筑基址；**今统万城遗址列国家级重点文物保护单位、2014 年列联合国教科文组织世界文化遗产预备名录**。**唐人在 755 所见的那座白色夯土城**——**主体犹存**、千二百多年未易；今游人来靖边白城子、所见的是十六国时代的真物理遗存。

## 子条目

### 东西二城连体

**统万城分东西二城**——东城 + 西城连成一体；东城为外郭、西城为内郭（宫城）。**东城东西约 850 米、南北约 500 米**；**西城东西约 800 米、南北约 770 米**——**两城以中部一道城墙分隔**。**这种"东西二城连体"** 是十六国都城规制的代表——与同期前秦长安城、后秦长安城（皆改建汉长安城旧）+ 北魏平城（早期）等都城形制相似。

### 南宫高台（赫连勃勃宫殿基址）

西城西南角的一处大型夯土高台——传为赫连勃勃**南宫**（主宫殿）基址。**今残高约 25 米**——是统万城最显的物理标识；**这是十六国时代最大的一处宫殿夯土遗存之一**。**1996-2010s 考古发掘**——确认台上有大型殿宇基址、出土十六国陶片、瓦当等。**南宫高台朝南**——可俯瞰无定河南面的草原与黄土沟壑。

### 蒸土为城 + 砺刀斧（《晋书》原文）

**赫连勃勃命叱干阿利筑统万城** 时所用工艺——史载**"蒸土为城"**（《晋书·赫连勃勃载记》）；**现代考古口径**：以黏土掺白石灰 + 砂粒夯筑（白灰夯土 / 三合土工艺）——筑成后墙体可砺刀斧。**"砺刀斧"** 是史书对统万城坚硬的形容。**唐人在 755 见到的统万城墙**——便是这种夯土筑成的、324 年未坏；**今统万城墙残基仍可见此种夯土工艺**——**夯土中含白石灰晶体**、敲击有金属声。

### 叱干阿利筑城暴政（"锥土入一寸即杀"）

**营宫监叱干阿利**——大夏立国时期最残虐的官员。**筑城制度**：城墙夯土筑成后、用锥试夯土密度——若锥能入一寸即杀筑者及监督（《晋书·赫连勃勃载记》原文）。**这种暴政导致筑城死者万计**——是历代记录的可怕场面。**"夯土中混有人骨"为后世传说**——本 doc 不坐实考古结论；现代考古口径为白灰夯土 / 三合土。**叱干阿利的暴政**——是赫连勃勃时代最阴暗的一面、亦是盛唐边塞文学"白城残虐故事" 的来源。

### 白城（唐人通称）

**统万城在唐人口中通常作"白城"**——以城墙夯土掺白石灰呈白色得名。**唐边塞诗 + 怀古诗中"白城"二字出现频繁**：李益、王建、白居易、岑参等多人有用。**白城作为陕北边塞的文学符号**——与凉州 + 玉门关 + 阳关 + 萧关 等并列。**注**：李益、王建、白居易等诗多作于 755 后；盛唐前期对白城的具体诗咏史源略零散。**755 时盛唐人嘴里的"白城"**——主要是地理认知层面 + 边塞印象层面。

### 大夏国（407-431）的兴亡

**大夏国是十六国后期短暂的一个匈奴系政权**——**407 赫连勃勃立国**、**413 筑统万**、**420 取长安达鼎盛**、**425 勃勃卒、子赫连昌继位**、**427 北魏攻克统万、赫连昌西走**、**431 赫连定被吐谷浑所俘——大夏亡**——**大夏立国 24 年**（407-431；都于统万 18 年/413-431）。**赫连勃勃的"夏"号**——取自"大禹之夏" 政治叙事、自称匈奴是夏后氏遗裔；这种"自我中国化"是十六国诸胡政权的常见模式。**大夏亡后，统万退为北魏统万镇 → 北周夏州 → 隋夏州 → 唐夏州**——皆置州治于旧统万城。

## 历史长河（755 → 今）

**中晚唐**　**广德元年（763）后吐蕃东扩 + 党项内迁**——夏州一带渐成党项核心区；**唐末党项李思恭据夏州**（881 起），实际控制统万城。**自此统万自唐廷边州转为党项独立势力的核心**——是西夏立国（1038）的物理根基（西夏定都兴庆府/银川、但其势力源头在统万）。

**两宋**　**北宋初党项渐独立**——李继迁（963-1004）以统万为基地反宋。**北宋淳化五年（994）宋太宗诏毁统万城**——以城墙过坚、防党项利用之；**自此统万被废**——白色夯土墙逐渐荒芜。**北宋元符元年（1098）党项再据统万旧址筑新城**——但规模远不及大夏旧统万。

**金元明清**　金元间统万一带渐荒废；**元代统万城旧址已成无人区**；明清间仅有少量蒙古族、汉族游牧者经过。**清乾隆四十年（1775）陕西巡抚毕沅至靖边考察**——立"统万城故址"碑——是清代"复古考"的一笔。

**民国至今**　1949 年后统万城遗址列陕西省重点文物保护单位、1996 年列首批全国重点。**1996-2010s 多次考古发掘**——确认东西二城、南宫高台、马面、角楼、城内礼制建筑基址；出土十六国至唐代陶片、瓦当、人骨等。**2014 年统万城遗址列联合国教科文组织世界文化遗产预备名录**。**今游人来靖边白城子**——所见的是**白色夯土城墙、25 米高的南宫高台、四面 30 余处马面**——**是十六国时代保存最完好的都城遗址**。**唐人在 755 所见的那座白色夯土城**——一千二百多年后绝大部分仍立、是中国十六国都城遗存中规模与完整度兼备的孤例。
