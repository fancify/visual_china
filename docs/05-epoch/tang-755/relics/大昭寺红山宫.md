# 大昭寺 + 红山宫

## 一句话

吐蕃逻些（拉萨）城心——**松赞干布七世纪中叶所建**：山下大昭寺供文成 / 尺尊公主入藏所携释迦摩尼像、红山顶宫殿（布达拉宫前身）九层千室；**755 时俱已百年，吐蕃赞普赤德祖赞此年前后卒、赤松德赞即将即位**。

## 形貌

逻些（藏文 lhasa；唐代汉译"逻些"或"逻娑"）在**雅鲁藏布江中游北岸的拉萨河谷**——海拔约 3650 米，是吐蕃帝国 7 世纪中叶迁都后的核心城市。城中有两座建筑构成 7-8 世纪吐蕃帝国的物理 anchor：**山下大昭寺** + **红山顶宫殿（红山宫）**。

**大昭寺**——在拉萨河北岸一片平地上，**主殿坐东面西**（这是藏传佛教寺院少见的方位——与汉地寺院常见的"坐北朝南"不同；藏文文献口径并见拉萨官方解说）。**寺主体为方形院落**——正殿主供释迦摩尼像；周围有数十间附属殿堂；屋顶有金色塔顶（**今所见金顶为后世**，盛唐原貌史源未详）。**寺得名**："大昭"是后世汉译——藏文称**"惹萨幻显寺"**（ra-sa 'phrul-snang，"惹萨"是拉萨古称，"幻显"言释迦像神显之意）或**"觉康"**（jo-khang，"觉沃之家"——觉沃即释迦像）。

**红山宫**——拉萨城中央一座红色岩山（藏文"玛布日"，意"红色山头"）之顶。**松赞干布七世纪中叶在此山顶建九层楼宫**——是吐蕃赞普的政事 + 寝宫所在；千间宫室、连塔环屋、与城下大昭寺隔一段平地相望。**红山宫是吐蕃帝国的政治中心**——文成公主入藏后即居于此（藏文文献记）。

物候：拉萨海拔 3650 米——高原气候、紫外强烈、四时寒凉、夏日多雨午后日照、冬日极干燥。盛唐人来逻些——多自鄯州西出唐蕃界、过赤岭、青海湖、那曲一线南下——是唐与吐蕃之间使节、商旅的常行路线。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

逻些在唐人的回望里，是**吐蕃帝国都城**——唐与吐蕃 130 年外交往来的核心地标。

**吐蕃兴起**：松赞干布（约 605-650）即位后**迁都于逻些**（具体年份藏文文献略异，约 633 前后）；**贞观十五年（641）唐太宗以宗室女文成公主出降松赞干布**——文成公主自长安出发、经鄯州西出唐蕃界、过赤岭、入吐蕃境、抵逻些——是唐与吐蕃和亲之始。**藏文《吐蕃王统记》记**：大昭寺、红山宫两座建筑皆在松赞干布朝营建——大昭寺由尺尊公主（尼婆罗、即尼泊尔）+ 文成公主合力主持立寺、安置两公主入藏所携佛像；红山宫由松赞干布在玛布日山顶筑九层楼。

**唐高宗朝至武则天朝**——唐蕃关系从和亲转为军事对峙；**670 大非川之战**、**678 青海之战**等多次大战、唐军多败。**唐中宗朝**——景龙四年（710）唐以**金城公主**出降吐蕃赞普赤德祖赞——是唐与吐蕃第二次和亲。**藏文文献记金城公主入藏后**，将原供小昭寺的释迦摩尼**十二岁等身像（觉沃释迦牟尼）迁至大昭寺**——自此 755 时大昭寺主供的是十二岁觉沃像（**本 doc 严守此通行口径**，但具体迁移年份藏文异说颇多）。

**玄宗朝**——开元二年（714）唐蕃武功县战、开元二十一年（733）**金城公主请定唐蕃疆界、次年（734）于赤岭立界碑**（见 [赤岭碑](./赤岭碑.md)）；开元末天宝间唐蕃多次冲突——**天宝十二载（753）哥舒翰兼河西、陇右两镇节度使**、对吐蕃压力达盛唐最高峰。**天宝十四载（755）前后**——吐蕃赞普**赤德祖赞（约 704-755）**卒（具体月份、卒地藏文异说，本 doc 不坐实）；**赤松德赞（742-797）**少年即位前夜——这一年逻些正在赞普换代关头。

### 此刻状态

天宝十四载这一年的逻些——**吐蕃帝国都城、赞普换代之年**。

**大昭寺**：建寺已百余年（约 642→755 = 113 年）。寺主供**释迦摩尼十二岁等身像**（觉沃释迦牟尼）——金城公主 710 入藏后从小昭寺迁来（藏文文献口径）。寺规模在松赞干布朝奠定，金城 / 文成两次入藏均有扩建；**755 时大昭寺已是吐蕃佛教中心**——但吐蕃佛教正式立国教 是 779 年赤松德赞的事，755 时大昭寺更多是**赞普家寺**性质。

**红山宫**：建宫已百年余（约 637→755 = 118 年）。**九层楼宫**仍是吐蕃赞普政事 + 寝宫所在——**赤德祖赞 755 年前后卒**（具体月份、卒地藏文文献异说，**本 doc 不坐实卒于红山宫**）；逻些进入赞普换代关头、赤松德赞即将即位。**红山宫与大昭寺之间**：一段平地、有道相连——盛唐时这段路是逻些城内的主轴；今布达拉宫与大昭寺之间的"八廓街"区便是这一段路的现代延续。

**这一年的吐蕃**——**赤德祖赞 755 年前后卒**（具体月份、卒地藏文文献异说），**赤松德赞少年即位（按 742 生约 13 岁）**、由权臣摄政；同年 11 月（汉历）安禄山起兵于范阳——**两个帝国同时进入转折期**。**至德元载（756）吐蕃趁安史之乱东扩**——再过几年（762-763）吐蕃将攻陷长安一日。**755 时这一刻**——逻些尚是平静的都城、唐蕃边境暂稳。

### 此刻意味

对天宝十四载的吐蕃人来说，大昭寺 + 红山宫是**逻些城心的两座最神圣建筑**——日常礼佛 + 朝廷议政。对唐人来说——逻些是**远在四千余里外的吐蕃都**；只有曾任唐蕃使节者（如刘元鼎元和十四年/819 入吐蕃、《新唐书·吐蕃传》记其行程）方能亲见。盛唐人嘴里的"逻些"——主要是地理认知层面的吐蕃都名；具体大昭寺 + 红山宫的形制，**唐代汉文献载之甚少**。

唐人尚不知道——再过 24 年（779 赤松德赞朝），吐蕃正式立佛教为国教；再过 60 年（842 朗达玛灭佛），红山宫在吐蕃帝国崩溃中**毁于雷击大火 + 兵燹**——九层楼宫只余山顶残基；再过 800 年（17 世纪），五世达赖喇嘛在红山宫原址重建**布达拉宫**——但红山宫的盛唐原貌已不可见。**大昭寺则千年屡修屡毁、屡毁屡修——主体犹存**——觉沃释迦牟尼像仍坐于寺心、千二百多年未易。**唐人在 755 所见的那尊像，与今游人所见的，藏文文献口径是同一尊**。

## 子条目

### 大昭寺正殿（觉沃释迦牟尼像）

大昭寺中心——主殿主供**释迦摩尼十二岁等身像**（觉沃释迦牟尼）。**藏文文献记**：此像本是文成公主 641 入藏时带去的；初供小昭寺；金城公主 710 入藏后迁至大昭寺——自此 755 时大昭寺主供即此像。**像质材**：传为铜质鎏金；**像高约 1.5 米**（坐姿）；释迦摩尼 12 岁童子相、跏趺坐。**这是中国境内最神圣的一尊佛教造像之一**——藏传佛教信徒视为"释迦摩尼本尊"。**像至今犹存**（屡经修补、镶金嵌玉，今所见已非盛唐原貌；但主体未易）。

### 释迦摩尼八岁等身像（小昭寺主供）

**尺尊公主入藏所携——释迦摩尼八岁等身像**（觉沃米觉多吉）。**初供大昭寺**；八世纪初（金城公主入藏 710 前后）**与十二岁觉沃像互换**——**自此 755 时小昭寺主供八岁觉沃像、大昭寺主供十二岁觉沃像**。**本 doc 严守此藏文文献通行口径**（具体互换年份藏文异说较多）。**今小昭寺仍以释迦摩尼八岁等身像为主圣**——但**原盛唐像与今像的延续情况须 hedge**：一说原像于文革中毁、今为后世重塑；一说原像主体犹存。盛唐人若至逻些礼佛、八岁像在城西小昭寺、十二岁像在城心大昭寺。

### 红山宫九层楼

**玛布日山（红山）顶上的九层楼宫**——松赞干布七世纪中叶建。**九层楼**：藏文文献记宫凡九层、千间室；山下平地有夯土城墙连山顶宫宇；西南向有正门。**红山宫是吐蕃帝国 200 年的政治核心**——历代赞普在此议政、起居、出生、即位、死亡。**755 时**赤德祖赞前后卒、赤松德赞少年（约 13 岁）即将即位（具体卒地与日月藏文文献异说，本 doc 不坐实卒于红山宫）。**红山宫毁于 9 世纪中叶**（842 朗达玛之乱 + 雷击大火）——之后山顶宫宇尽毁、仅余夯土残基。

### 八廓街（大昭寺与红山宫之间的城内主轴）

大昭寺前的**转经道**——藏文称"八廓"（bar-skor，"中圈"）——是大昭寺周围一圈环形朝拜道。**红山宫与大昭寺之间** 有一段平地、是 7-9 世纪吐蕃逻些城内的主轴——后世发展为**八廓街**。**盛唐时这段路上是何状况**——史载未详；藏文文献记为"市集 + 朝拜 + 赞普车驾出入"地。**今八廓街**是拉萨老城区的核心、围绕大昭寺的环形商业街。

### 文成公主与金城公主（两次和亲）

**文成公主**（约 625-680）——唐宗室女，**贞观十五年（641）出降松赞干布**——是吐蕃 + 唐第一次和亲。藏文文献记文成公主主持大昭寺立寺 + 选址、带去释迦摩尼十二岁等身像。

**金城公主**（约 698-741）——唐中宗朝宗室女，**景龙四年（710）出降吐蕃赞普赤德祖赞**（赤德祖赞是松赞干布之曾孙；金城是赤松德赞之母）——是吐蕃 + 唐第二次和亲。**金城公主入藏后将十二岁觉沃像自小昭寺迁至大昭寺**——是大昭寺主供的关键时间点。

## 历史长河（755 → 今）

**中晚唐**　**赤松德赞朝（755-797）**——吐蕃达鼎盛、立佛教为国教（779）、建桑耶寺（779-787）——**藏传佛教正式确立**。**安史中吐蕃东扩**——762-763 吐蕃陷长安一日；广德元年（763）后唐蕃疆域大变。**赤祖德赞朝（815-838）**——吐蕃由佛教达鼎盛；**长庆会盟（821-822）唐蕃立碑**于大昭寺前（即"唐蕃会盟碑"——今犹存于大昭寺前）。**朗达玛朝（838-842）**——朗达玛灭佛、842 被刺；吐蕃分崩离析；**红山宫此时被毁**——九层楼宫被雷击大火 + 兵燹毁尽。

**两宋至元**　**吐蕃分裂期**（842-1239）——萨迦派、噶举派、宁玛派等各派禅修传承、政治势力林立。**大昭寺在各派施主之间多次维修**——主体犹存。**元朝**——元世祖封萨迦派八思巴为帝师；元代多次诏修大昭寺。

**明清**　**明洪武初**封宗喀巴（1357-1419）格鲁派——格鲁派渐为西藏第一大宗派。**清顺治九年（1652）五世达赖喇嘛入京**——清廷封达赖喇嘛、**清顺治至康熙间布达拉宫在红山宫原址重建**（1645-1693 主体完成）——**红山宫的盛唐九层楼原貌已不可见、布达拉宫是新建**。**清代大昭寺为西藏佛教中心**——历代修缮、规模渐大。

**民国至今**　1949 年后大昭寺、布达拉宫均列国家级文物保护单位。**2000 年大昭寺列联合国教科文组织世界文化遗产**（作为布达拉宫历史建筑群的扩展）；**1994 年布达拉宫列世界文化遗产**。**今大昭寺主供觉沃释迦牟尼像**——藏文文献口径与盛唐 755 时为同一尊；周围殿宇多为后世修复。**今布达拉宫**——是 17 世纪重建的，红山宫盛唐原貌已不可见、仅留遗址性记忆。**唐人在 755 所见的那座红山顶九层楼宫——千二百多年后已成另一座宫**；但**山下那尊觉沃像、藏文文献记为同一尊**——这是大昭寺最稳的物理 anchor。
