# 北魏邙山陵群

## 一句话

**北魏迁都洛阳后的五帝陵**——孝文帝长陵 + 宣武帝景陵 + 孝明帝定陵 + 孝庄帝静陵 等；**499-531 年间陆续葬入**（499 长陵起、至 531 静陵）；**755 时已立 220-260 年、邙山土堆累累**——唐人在邙山耕作 / 出洛阳东行时**每见的废陵**；是北朝中期最重要的陵群之一。

## 形貌

北魏邙山陵群在唐河南道**河南府偃师县境**——具体在**洛阳北邙山中段**（今河南省洛阳市偃师区邙山乡 + 洛阳市孟津区一带）。**邙山**——洛阳北的一片黄土丘陵、海拔约 250-350 米、东西延绵约 100 公里——**自周秦至北朝、邙山是华北最大的帝王陵区**：

**邙山陵区的多代叠加**：
- **东周王陵**（约 770-256 BCE）
- **东汉光武帝原陵 + 历代东汉帝陵**（25-220 CE）
- **曹魏 + 西晋帝陵**（220-316）
- **北魏迁洛后五帝陵**（499-531）—— **本 doc 主写**
- **唐人对邙山的认知**：是"北邙"——历代帝王长眠地、凭吊文学的高频地点。

**北魏五帝陵的具体分布**：
- **孝文帝长陵**：偃师邙山乡官庄村——长陵主峰；**长陵是北魏迁洛后第一座帝陵**；**与文昭皇后高氏合葬**。
- **宣武帝景陵**：洛阳市北、邙山中段——**1991-1992 考古发掘**；是**陵主明确且较早系统发掘 + 开放地宫的北魏帝陵**（2013 后又有疑似节闵帝元恭墓发掘报道——洛阳衡山路北魏大墓）。
- **孝明帝定陵**：邙山——具体位置考古勘察中。
- **孝庄帝静陵**：邙山——具体位置史源略异；亦或在偃师邙山东段。
- **其他次要陵**：北魏后期诸帝（孝庄帝建陵等）亦多葬邙山附近、史载零散。

**陵冢形制**：北魏帝陵**封土为坟**（非依山为陵；与汉魏旧制延续）；底边约 100-150 米、高约 10-25 米——比唐六陵略小但与汉陵相近。**陵区布局**：陵以陵主穴为中心、四面有围墙、神道、献殿、陵令所驻——但**北魏帝陵神道石刻不如唐陵密集**（北魏礼制相对简朴）。

**邙山陵区的总规模**：千余年间各代帝陵 + 王公贵族墓陆续葬此——**今考古勘察邙山陵区共有古墓约 1.7 万座**（汉至北朝至唐宋陆续累积）；**北魏五帝陵是其中政治分量最高的几座**。

物候：黄土高原南缘、温带季风气候——夏热冬寒、雨集中夏季；邙山为黄土丘陵、易耕作、唐时多为农田 + 陵区混合景观。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

北魏邙山陵群在唐人的回望里——是**北魏迁都洛阳后的五帝陵 + 北朝政治史的物理纪念**。

**北魏孝文帝迁都洛阳**：**北魏太和十八年（494）**孝文帝**自平城（今山西大同）迁都洛阳**——是中国历史上最有名的胡族汉化政治改革。**迁都的政治背景**：（1）北魏自拓跋珪开国（386）至孝文帝（471 即位）已百余年；（2）平城（在阴山以南、北方边境）作为首都不利于统治中原；（3）孝文帝（自身鲜卑、汉化深）以"开万世之基" 推动迁都 + 全面汉化。**迁都后的政治措施**：改鲜卑姓氏为汉姓（拓跋→元、独孤→刘 等）、易胡服为汉服、禁鲜卑语、定郊祀礼于洛阳南郊。

**孝文帝太和二十三年（499）四月**——孝文帝崩于**南伐齐途中谷塘原之行宫**（《魏书·孝文帝纪》原文）；**同年八月葬于邙山长陵**——是北魏迁洛后第一座帝陵。**自此北魏皇室明诏：迁洛后诸帝皆葬邙山、不归平城旧陵**（这是孝文帝迁都的最终礼制）。

**北魏五帝陵的次第葬入**：
- **太和二十三年（499）孝文帝长陵**
- **延昌四年（515）宣武帝景陵**
- **武泰元年（528）孝明帝定陵**（"河阴之变" 后葬）
- **普泰元年（531）孝庄帝静陵**

**北魏后期的内乱**：**孝明帝 528 被毒杀（一说被胡太后毒杀）**——**同年河阴之变**：尔朱荣大杀北魏宗室与朝臣 1300 余人于河阴（黄河北岸）；自此北魏分裂。**永熙三年（534）北魏分裂为东西**——**东魏迁都邺、西魏（实由宇文泰把持）都长安**。**自此邙山北魏陵群再无新葬入**——成废陵区。

### 此刻

天宝十四载这一年的北魏邙山陵群——**已立 220-260 年（499 长陵→755 = 256 年；531 静陵→755 = 224 年）**；以**废陵群**身份存在。

**陵冢仍立邙山**——长陵、景陵、定陵、静陵 + 北魏其他帝陵、王公墓——盛唐 755 时主体犹存；具体盛唐时是否有盗扰、史载未详。**北魏礼制相对简朴**——神道石刻不如唐陵密集、陵区围墙多数已坍。

**唐人对北魏邙山陵群的认知**——主要通过：
1. **史书记载**：《魏书》（北齐魏收撰）+ 《北史》（唐李延寿撰）——盛唐已成书 + 广流；
2. **物理凭吊**：盛唐人出洛阳北行、必过邙山——可见北魏陵冢；
3. **唐诗**：盛唐诗中"邙山"作为意象高频——多咏帝王陵墓荒废、人生无常等。

**唐人凡至邙山者多有怀古**——王勃《滕王阁序》虽不写邙山、但同期文学有大量邙山怀古诗。**杜甫《忆昔二首》《饮中八仙歌》等盛唐诗**——多用"陵阙" 二字暗指汉魏北魏诸陵；**盛唐凭吊邙山的密度** 远超后世。

**这一年的邙山陵群** 处于**北魏废陵 + 唐人凭吊 + 农田混合**的稳定状态——再过半年安禄山起兵；**至德二载（757）十月唐军复东都洛阳**——**邙山陵群一带反复易主**；具体安史中是否有大规模盗扰、史载零散。

### 此刻意味

对天宝十四载的偃师 / 河南府人来说——邙山陵群是**自家土地上的祖辈遗物**：北魏帝陵 + 历代帝王陵 + 王公墓——是中原最大的陵区。对长安那一代士人来说——邙山是**汉魏六朝北方文明的物理纪念**：东汉光武原陵 + 曹魏文帝首阳陵 + 西晋帝陵 + 北魏五帝陵 等——千年间数十帝长眠于此。**唐人凡过邙山者皆有怀古**——是盛唐文学中最浓厚的"凭吊地"之一。

唐人尚不知道——再过 130 余年（885 黄巢之乱）邙山陵群将被陆续盗掘；**五代温韬大规模盗发关中唐陵**——但温韬主要在关中、邙山陵群被盗多发生于黄巢乱后 + 北宋初。**北魏景陵在唐末或北宋间被盗发**——这是 1991-1992 现代考古发掘时确认的。再过千二百年（1991-1992），**洛阳市文物考古研究院发掘景陵地宫**——出土北魏陶俑、壁画、墓志等——是北魏帝陵唯一一次科学发掘。**唐人在 755 所见的邙山土堆累累**——千二百多年后陵冢主体犹存、地表景观依然；但**北魏礼制简朴的神道石刻群** 大多已不存。

## 子条目

### 孝文帝长陵（迁洛后第一座帝陵）

**北魏孝文帝拓跋宏（467-499）**——北魏第六代皇帝、迁洛主导者；**太和二十三年（499）四月崩于南伐齐途中**（地点史源略异）；**同年葬长陵**——是**北魏迁洛后第一座帝陵**。**位置**：今河南省偃师区邙山乡官庄村——长陵主峰。**孝文帝与文昭皇后高氏合葬**——**高氏即宣武帝生母、生时已卒（497 卒）**、孝文帝崩后合葬。

**长陵的政治意义**：孝文帝迁都洛阳是中国历史上最有名的胡族汉化改革；其葬于邙山而非平城——是迁都决心的最终物理体现。**唐人来邙山多瞻仰长陵**——是北魏皇室"承汉魏旧制" 的物理纪念。

### 宣武帝景陵（北魏帝陵唯一发掘）

**北魏宣武帝元恪（483-515）**——孝文帝次子、北魏第七代皇帝；**延昌四年（515）正月崩**、葬景陵。**位置**：今洛阳市北邙山中段。

**景陵是陵主明确且较早系统发掘 + 开放地宫的北魏帝陵**——**1991-1992 年洛阳市文物考古研究院发掘地宫**（墓被盗严重）——出土：
- **青瓷器、陶器、石器 等 50 余件**：含侍立俑、骑马俑、武士俑等北魏造像；
- **石椁基址 + 砖室结构**：确认北魏帝陵地宫形制。

**景陵出土物今藏洛阳博物馆**——是北魏物质文化研究的核心实物。**2013 年洛阳衡山路北魏大墓发掘报道**——按"疑似节闵帝元恭墓"对外公布；景陵不再是"唯一" 北魏帝陵已系统发掘地宫者。

### 孝明帝定陵（"河阴之变" 后葬）

**北魏孝明帝元诩（510-528）**——宣武帝长子；**武泰元年（528）二月被毒杀**（一说被其母胡太后毒杀）、年 19；**同年葬定陵**——位于邙山。**定陵的特殊背景**：孝明帝被毒杀同年发生**"河阴之变"**——**尔朱荣大杀北魏宗室与朝臣 1300 余人于河阴**（黄河北岸）；**自此北魏分裂**。**定陵的葬礼**在内乱中进行——具体规模史载未详。

### 孝庄帝静陵 + 北魏其他帝陵

**孝庄帝元子攸（507-531）**——北魏末年皇帝、被尔朱兆所杀；**永安三年/普泰元年（531）卒、葬静陵**（《魏书·孝庄帝纪》）。**静陵在邙山**——具体位置在洛阳邙山。**节闵帝元恭（498-532）**——北魏另一末年皇帝、被高欢所废所杀；**葬地有疑似论**：2013 洛阳衡山路北魏大墓发掘报道按"疑似节闵帝元恭墓"对外公布。**北魏其他葬邙山附近的帝陵**：前废帝（元晔）、后废帝（元朗）等——史源零散。

### 邙山的多代陵区（东周至北魏）

**邙山是华北最大的帝王陵区**——自东周至北魏千余年累积：
- **东周王陵**（770-256 BCE）
- **东汉光武帝原陵 + 历代东汉帝陵**（25-220）
- **曹魏文帝首阳陵**（226 葬）+ 其他曹魏陵
- **西晋武帝峻阳陵**（290 葬）+ 其他西晋陵
- **北魏迁洛后五帝陵**（499-531）—— 本 doc 主写
- **王公贵族墓**：千余年间数千座

**唐人对邙山的整体认知**：**"北邙"是凭吊地、是历代帝王陵的代名词**——盛唐诗中"邙山"或"北邙"二字出现频繁、多用作"人生无常 / 王朝兴亡"的意象。

### 北魏孝文帝迁都洛阳的政治意义

**孝文帝拓跋宏（467-499）**：北魏第六代皇帝、最有名的汉化改革者。**太和十八年（494）迁都洛阳**——自平城（今大同）迁洛阳；**改革内容**：
- 改鲜卑姓氏为汉姓（拓跋→元、独孤→刘 等）；
- 易胡服为汉服、禁鲜卑语；
- 定郊祀礼于洛阳南郊；
- **诏迁洛后诸帝皆葬邙山、不归平城旧陵**——是迁都礼制的最终。

**孝文帝改革的政治后果**：（1）北魏中央政权汉化深入；（2）但鲜卑老贵族（多在六镇）不满——为 524 六镇之乱埋下种子；（3）孝文帝崩后 35 年（534）北魏分裂——东西两魏延续至 577 北周灭北齐。**孝文帝是中国历史上最大胆的汉化政治家之一**——其迁都的物理 anchor 即邙山陵群。

## 历史长河（755 → 今）

**中晚唐**　**至德二载（757）十月唐军复东都洛阳**——邙山陵群一带反复易主；具体安史中是否有大规模盗扰史载零散。中晚唐邙山仍是凭吊地——白居易、刘禹锡、温庭筠 等多有邙山诗。**长安诗坛对邙山的认知** 持续至唐末。

**唐末黄巢 + 五代**　**广明元年（880）黄巢入长安**——洛阳邙山陵群在乱中受波及；**唐末至五代间邙山陵群陆续被盗掘**——具体何陵何年史载未详。**北魏景陵在唐末或北宋间被盗发**——1991-1992 考古发掘时确认。

**两宋**　北宋初邙山陵区仍是凭吊地、但北宋的陵区已转至河南巩义（北宋皇陵）；邙山陵区渐失"现役" 地位。**北宋金石学家欧阳修、赵明诚等**对邙山碑刻有研究——但实物多被盗损。

**金元明清**　金元明清间邙山陵群地表渐成农田、王公墓被盗掘多次。**明清间邙山仅余土堆累累 + 部分碑刻 + 农民耕作时偶有出土**。

**民国至今**　**1947 年 + 1980s 起多次考古勘察邙山陵区**——确认北魏长陵、景陵、定陵、静陵 等位置；**1991-1992 年洛阳市文物考古研究院发掘景陵地宫**——出土北魏陶俑、壁画、墓志等。**2008 年起邙山陵区列国家级重点文物保护单位**（北魏长陵、景陵等多座单列）。**2009 年曹操高陵（亦邙山陵区周边）确认**——使邙山陵区的研究再受关注。**今游人来邙山**——所见的是**长陵 + 景陵 + 定陵 等土堆 + 洛阳博物馆藏景陵出土物**——**唐人在 755 所见的邙山土堆累累、与今相似**；具体陵冢主体犹存、但盛唐人当年所见的神道石刻、围墙、献殿——多已不复。
