# 福州港

## 一句话

**江南东道东南海岸的新兴海港**——福州港在长乐郡闽县治东、闽江入海口一带；是闽中诸州（福州 + 建州 + 汀州 + 漳州 + 泉州）海上出口与江南、岭南、海外通商的物理 anchor。与广州港的胡舶云集、明州港的江南近海枢纽、登莱二州的渤海北通新罗渤海相比——**755 时福州港尚是新港**：闽中开发自永嘉八姓入闽（311 后）以来缓慢累积，至开元天宝之际方初见海港气象；其大港地位要等到中晚唐 + 五代闽国 + 北宋方真正成形。盛唐六港（广州 + 交州 + 明州 + 登州 + 莱州 + 福州）之一席——但 755 这一刻仍以转口闽中山货海产为主，胡舶偶来而非常驻。

## 形貌

福州港在 755 时——**不是一处方城式港口、而是闽江入海口两岸一段散布的津渡与泊舶之地**。

**地理位置**：**江南东道福州长乐郡治闽县东郊** + **闽江入海前最后一段**。福州城在闽江北岸三山（屏山 + 于山 + 乌山）环抱的台地上；自城东南行约三十唐里、闽江已转入入海前的最后一段——江面骤阔、潮汐可上溯至城南郊；自城东南行约六十至百余唐里、便至闽江入海口。**港口的物理形态**——非单一津渡、而是自城东闽县东郊（今福州市仓山区一带）、东南顺闽江下行至马江一带（今福州市马尾区）、再至闽江口（今福州市连江县与长乐县交界、闽江入海处）、共数十唐里岸线上散布的津渡与泊舶之地。

**港口分段**：
- **闽县东津**——闽江流过福州城南、城东南郊的津渡；中小江船的总集散；闽中山货（茶 + 木 + 漆 + 铁 + 银 + 楠木 + 杉木）由此下江、海产（盐 + 鱼 + 蚌 + 珠 + 玳瑁）由此上岸；
- **马江段**——闽江下行至今马尾一带（盛唐尚无"马尾"之名、唐人多称"东港"或随地名而呼）；江面阔、可泊大船；江南海船 + 岭南海船 + 偶尔的胡舶在此停靠；
- **闽江口外港**——闽江入海处一带；海船候风、候潮；与岭南广州港 + 江南明州港的航路在此衔接。

**港口设施**：盛唐 755 时福州港——**官私并立、津吏稀少**。**长乐郡 + 闽县**辖港务、津吏数人；江船 + 海船 + 渔舟数百艘上下；**贡品集散**——《新唐书·地理志》载福州土贡有"荔枝 + 龙眼 + 蕉布 + 葛布 + 海蛤"——这些贡品自闽中各县汇于福州、再经港口北运至洪州（江南西道），转赣江入大庾岭通广州、或转长江入扬州北上洛阳。**官设市舶使**——盛唐市舶使主驻广州（开元二年 / 714 市舶使始见于安南、广州后成主设地）；福州港 755 时**未置市舶使**、偶有外舶来此**由长乐郡、闽县地方官按关津常制处理；是否上报广州市舶使、史阙不可断言**——这是 755 时福州港与广州港规格差的具体体现。

**周边水道**：
- **闽江**——闽中第一大江、自西北三源（建溪 + 富屯溪 + 沙溪）汇流而下、过建州 + 剑州山谷、至福州西郊切出冲积平原、再折东南入海；江面在福州城外汛期可达数里；
- **乌龙江**（闽江南汊）——自福州城南支分而出、绕南台岛南侧入海；江面较窄；与闽江本支夹南台岛形成"南北二港"格局——盛唐时尚未有"南台岛"专称、唐人通称"江南岸"；
- **海岸**——闽江口南北各有海湾 + 沙嘴 + 礁岛；北有罗源湾 + 三都澳一带（盛唐时尚未开发为良港）；南有兴化湾 + 湄洲湾一带（亦未开发）；唯闽江口为闽中海贸的总出口。

**物候**：盛唐时福州港一带——**亚热带海洋性 + 季风明显**。**春**——多雨 + 雾；江口大雾时船不出；**夏秋**——**飓风（即今之台风）多发**——《元和郡县图志》《通典》皆载闽中"夏秋多飓"——此为闽地海港最大威胁；飓来则船舶 + 津渡尽毁；**冬**——东北风强、海船南下广州正合季风；**江口潮汐**——一日两潮、潮差大；海船须候潮入江、候潮出海；唐人航海于福州港、最重季风 + 潮汐 + 飓风三事。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

福州港在唐人的回望里——**是闽中海岸缓慢开发千余年的累积 + 永嘉以后衣冠南渡的物理出口**。

**闽中开发史的回望**：
- **春秋战国**——闽越人居此、海岸渔捞 + 山间刀耕；与中原近无直接通商；
- **秦置闽中郡（前 222 / 前 221）**——名义上入中原版图、实际上自治；
- **汉武帝元封元年（前 110）灭闽越**——闽越王无诸故都东冶（今福州一带）被废、徙其民于江淮间——闽中一度近乎空地；东冶港 / 东冶县名号见《史记·东越列传》——是福州海港最早的史载渊源；
- **东汉 + 三国吴**——复置冶县、东侯官；孙吴永安三年（260）置建安郡——闽中复入华夏版图、海岸开发缓慢复苏；
- **晋永嘉之乱（311）+ 衣冠南渡**——"**八姓入闽**"——**林 + 陈 + 黄 + 郑 + 詹 + 邱 + 何 + 胡**八姓士族避乱入闽——这是闽中汉人构成的主要来源、亦是闽中开发的真正起点（八姓入闽史载见于宋人 + 明人方志、最早溯及唐林谞《闽中记》；东晋至隋间确有大批中原士族南徙入闽是事实、唐人通认此事）；
- **隋开皇九年（589）平陈**——闽中复入中原王朝；**开皇九年（589）改置泉州**——州治在今福州（隋初改丰州为泉州）；**开皇十二年（592）置泉州总管府**——州治不变、加置总管府以统军政；
- **入唐**——名号几度更易（建州 → 泉州（隋至唐武德）→ 闽州（711）→ 福州（725）→ 长乐郡（742））；**开元十三年（725）福州置都督府**（《新唐书·地理志六》）——这是福州在盛唐海岸防务 + 海贸地位上升的标志。

**港口史的回望**：
- **汉东冶港**——闽越故港、规模有限；
- **三国吴**——孙吴海贸初兴、东冶港始通海外（《吴志》载孙权遣康泰 + 朱应通南海、其出海点未明、福州 / 广州皆有可能）；
- **东晋南朝**——衣冠南渡后、闽中海岸开发加速、福州一带始有较规则的海港；
- **隋至唐初**——海贸初规模、但仍以广州 + 交州 + 扬州为主、福州港规模有限；
- **盛唐开元天宝间**——闽中户口积累、海贸开始向福州港集中；**开元二年（714）市舶使始见于安南、广州后成主设地**——市舶使是中国官方海贸最早的官名、福州港尚附属其下；**开元十三年（725）福州置都督府**——福州港的官方地位上升；天宝间海贸初具气象、但仍以转口为主。

**港口的具体使用者**——**闽中山货商**（茶 + 木 + 漆 + 铁 + 银 + 楠木 + 杉木自闽中山间汇于福州港、北运 + 海运）+ **江南海船**（自明州 + 扬州 + 楚州南下、至福州港转口闽货）+ **岭南海船**（自广州 + 潮州北上、至福州港转口）+ **偶尔的胡舶**（自南海航来、但 755 时多直入广州、福州港胡舶非常驻）+ **闽中渔民**（江口渔捞 + 海岸渔捞）+ **僧侣**（闽中佛寺与江南天台 / 牛头宗的来往、自福州港北行至明州或南下至广州西游）——是盛唐闽中海贸的物理出口。

### 此刻

天宝十四载这一年的福州港——**繁忙、按例运行、闽中山货海产集散、海舶偶来、即将进入安史之后中晚唐 + 五代闽国 + 北宋的大港时代**。

**港口状态**：盛唐 755 时福州港——闽县东津 + 马江段 + 闽江口外港三段并行；江船 + 海船 + 渔舟数百艘上下；闽中山货自上游沿闽江而下、海产自外海上岸；**长乐郡 + 闽县**辖港务、津吏数人；外舶**由长乐郡、闽县地方官按关津常制处理**——是否报广州市舶使、史阙不可断言。**港口的实际规模**——较广州港 + 扬州港 + 明州港小一档；较登州 + 莱州对新罗渤海贸易的规模略大、但不及岭南海贸的胡舶云集。

**与盛唐六港的相对地位**：
- **广州港**（岭南道；南海大港；714 市舶使始见、广州后成主设地）——盛唐第一大港；胡舶云集；外贸总枢；
- **交州港**（岭南道；红河三角洲）——南海西线大港；与广州港齐名；
- **扬州港 / 江都港**（淮南道；长江下游 + 大运河南端）——内河第一大港 + 长江口海贸；"扬一益二"之扬；
- **明州港**（江南东道；今宁波）——江南东海岸主港；与新罗 + 日本的航路起点；
- **登州港 + 莱州港**（河南道；山东半岛）——渤海北通新罗 + 渤海国的主港；
- **福州港**（江南东道；闽江口）——**新兴 + 第六**；闽中诸州的海上出口；755 时规模最小、地位最末；中晚唐 + 五代 + 北宋方上升。

**唐人对福州港的认知**：**"福州港"三字 755 时尚非常用地名**——唐人多称"闽县东津""闽江口""东冶港"（用古名）；正式行政文书写"长乐郡闽县某津"。**盛唐文献中福州港的著名度**——远不及广州 + 扬州 + 明州；杜甫 + 李白 + 王维诸盛唐大家无一咏福州港 / 闽江口（盛唐诗中咏闽中者极少、要到中晚唐 + 五代 + 北宋方多）；这是福州港 755 时"新港"地位的文学体现。

**港口的实际功能**：
- **山货下海**——闽中山间的茶 + 木 + 漆 + 铁 + 银 + 楠木 + 杉木 + 荔枝 + 龙眼 + 橄榄、自建州 + 剑州 + 汀州顺闽江而下、汇于福州港、再海运 / 江运北上；
- **海产上岸**——闽中海岸 + 闽江口 + 兴化湾 + 湄洲湾的盐 + 鱼 + 蚌 + 珠 + 玳瑁 + 海蛤、自江口上岸、转闽中山间或北上江南；
- **转口贸易**——岭南海船北上 + 江南海船南下、于福州港中转闽货 + 广货 + 江南货；
- **闽中贡赋**——福州土贡 + 长乐郡贡赋经此港北运（一道走海上至明州 + 扬州、一道走陆路 / 内河至洪州转长江）；
- **僧侣 + 使节往来**——闽中佛寺与江南 / 岭南的来往、自福州港中转；唐前期日本 + 新罗使节多走明州 + 登州、福州港此类往来较少。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九即起）；**天宝十四载十二月丁酉陷东京洛阳**（公历 756 年 1 月）——通济渠北端起点失；**江淮漕路自此中断**——但福州港远在闽中海岸、安史兵祸未直接波及；**长乐郡 + 闽中诸州**奉唐正朔、按例运行；**至德至乾元间（756-758）**江南诸道连年漕粮北运援唐军——福州贡赋多走陆路 / 山道经洪州北上、海路 + 内河漕路皆有；**福州港在安史中未陷、未毁、未中断**——是盛唐唐廷江南半壁的一处稳固后方海港。

### 此刻意味

对天宝十四载的长乐郡闽县人民来说——福州港是**自家东郊的物理生计**：闽中山货海产经此集散、衣食所赖；江南 + 岭南海船带来盐 + 米 + 丝 + 瓷、本地海产 + 山货换出。对盛唐唐廷来说——福州港是**江南东道东南海岸的一处新兴海港**——规模不及广州 + 扬州 + 明州、但已具雏形；闽中诸州贡赋经此或经陆路北上洛阳；海防上是闽江口的物理 anchor。对盛唐六港来说——福州港是**最末、最新、最未定型的一席**——其大港地位要等到中晚唐 + 五代闽国 + 北宋方真正成形。

唐人尚不知道——**晚唐乾宁三年（896）王潮 + 王审知兄弟入闽**奏置威武军节度使、治福州；**五代后梁开平三年（909）王审知封闽王**——闽国（909-945）以福州为都；**王审知治闽期间**（898-925）大力发展福州海贸 + 修筑罗城 + 招徕海舶——福州港自此进入大港时代；**北宋元祐二年（1087）置福建路市舶司于泉州**——闽中海贸的最高官方规格设于泉州、福州港作为次级节点；**南宋至元间泉州港跃升中国第一大港**——福州港作为闽中山货海产的次级港退居二线；**明成化间（1465-1487）福建市舶司由泉州迁福州**——福州港方在官方海贸体系中代泉州为闽中首港、但中国海贸主轴已移至广州 + 月港；**明清间福州港**仍为闽中政治 + 军事 + 文化中心、海港地位次于泉州 + 漳州。**755 一刻**的福州港——尚不知自己将经历这八百年的兴衰起落、当下只是一处新兴海港、按例运行。

## 子条目

### 闽县东津（福州港的核心津渡）

**闽县东郊、闽江北岸**——福州城东南郊的津渡；中小江船的总集散；闽中山货下江 + 海产上岸的主要节点。**津吏 + 渡舟 + 仓房 + 集市**——按例运行；**长乐郡 + 闽县**辖；**755 时格局**——较广州港的胡舶云集、扬州港的漕舟接连、明州港的与新罗日本通商相比、规模小一档；但作为闽中海贸的物理出口、已具雏形。**今闽县东津的地理位置**——大致在今福州市仓山区闽江北岸 + 台江区东南郊一带；盛唐津渡早已不在、但作为福州港史的物理原点仍可追溯。

### 马江段（闽江下行至今马尾一带）

**闽江下行至福州城东南约二十余唐里、江面骤阔**——可泊大船；江南 + 岭南海船在此停靠；**755 时唐人尚无"马尾"之名**——此名出现于清同治年间（1860s）福州船政局始建之际、是近代名号；**唐人通称"东港"或随地名而呼**（"闽江下游"、"闽口"）；马江段是福州港承担海船泊舶功能的核心区域、与闽县东津承担江船 + 集散功能形成分工。**今马尾港的地理位置**——大致在今福州市马尾区闽江北岸；盛唐泊舶处早已不在、但作为福州港史的物理原点仍可追溯。

### 闽江口外港（闽江入海口一带）

**闽江入海处**——海船候风 + 候潮的物理节点；与岭南广州港 + 江南明州港的航路在此衔接；**755 时唐人称"闽江口""闽口"**——盛唐尚无统一名号；**今闽江口的地理位置**——大致在今福州市连江县东南 + 长乐区东北、闽江入海处一带；包括今琅岐岛 + 川石岛 + 壶江岛诸岛屿；盛唐时这些岛屿是海船候风 + 候潮 + 避风的物理 anchor。

### 闽中山货（福州港的核心物流）

**茶 + 木 + 漆 + 铁 + 银 + 楠木 + 杉木 + 荔枝 + 龙眼 + 橄榄**——闽中山间的物产；自建州 + 剑州 + 汀州顺闽江而下、汇于福州港、再海运 / 江运北上。**福州土贡**（《新唐书·地理志六》）：**荔枝 + 龙眼 + 蕉布 + 葛布 + 海蛤**——这些贡品自闽中各县汇于福州、再经港口北运。**山货之于福州港**——是其立足的物理基础；755 时山货的规模决定了福州港的规模、随着中晚唐 + 五代闽中开发的深入、山货流量上升、福州港随之扩张。

### 八姓入闽（闽中汉人构成的主要来源）

**晋永嘉之乱（311）+ 衣冠南渡**——"八姓入闽"——**林 + 陈 + 黄 + 郑 + 詹 + 邱 + 何 + 胡**八姓士族避乱入闽。**史载渊源**——最早见于唐林谞《闽中记》（已佚、唐人著作）、宋人方志（《三山志》《八闽通志》）追述；明人确立"八姓入闽"之说。**史实评价**——东晋至隋间确有大批中原士族南徙入闽是事实、唐人通认此事；具体八姓名单史载存疑、但作为闽中汉人构成的主要来源、是闽人 / 闽中开发史的核心叙事之一。**与福州港的关系**——八姓入闽是闽中开发的起点、福州港的兴起最终建立在闽中汉人士族 + 山间农耕 + 海岸渔捞数百年累积之上。

### 闽中诸州 / 江南东道东南诸州（755 时福州所属）

**755 这一刻福州的上级行政归属**——**江南东道东南诸州**（福 + 建 + 泉 + 漳 + 汀）之首；福州长乐郡治闽县、为闽中诸州中户口最盛 + 海港最具雏形之州。**"福建"作为一组州的合称**——盛唐文献已偶用。**福建经略使**——**755 时福建经略使已置（《新唐书·方镇表》开元二十一年/733 已见）**；但**福建道作为完整一级行政道**——是 758 / 759（乾元元年至上元元年间）析江南东道而置福建经略观察使时才正式分出、地位介于道与州之间；**正式立"福建道"**则要等到中唐以后。**正式独立的"福建节度使"**——要到晚唐乾宁三年（896）王潮奏置威武军节度使时才出现——755 当年这些官名皆不存在。**与福州港的关系**——755 时福州港的行政管辖即长乐郡 + 闽县直辖、上对江南东道；港务、海防由地方官按常制处理、尚无"福建"层级的统筹机构。

### 海上飓风（闽中港口最大威胁）

**夏秋多飓**——《元和郡县图志》《通典》皆载闽中"夏秋多飓"——盛唐时尚无"台风"之名、但飓风之实已为唐人熟知。**飓风的破坏**——飓来则船舶 + 津渡 + 沿岸聚落尽毁；闽中海港每年须经历数次飓风的洗礼；**与福州港的关系**——飓风是福州港最大的物理威胁、亦是闽中海港兴起较江南 + 岭南海港滞后的部分原因。**唐人应对**——海船候风 + 候潮 + 避风岛；闽江口诸岛（琅岐岛 + 川石岛 + 壶江岛）是飓风季节的避风物理 anchor。

## 历史长河（755 → 今）

**中晚唐**　**安史中福州港未陷、未毁**——闽中远在战火之外、福州港按例运行；**至德 / 乾元间**江南漕运反复中断、福州贡赋多走陆路 / 山道经洪州北上、海路亦有；**中唐元和间**（806-820）福州港规模缓慢上升、闽中户口积累；**晚唐**——福州港作为闽中海贸的核心地位逐渐确立；**晚唐乾宁三年（896）王潮奏置威武军节度使治福州**——福州港的政治地位上升。

**五代闽国**　**后梁开平三年（909）王审知封闽王**——闽国（909-945）以福州为都；**王审知治闽期间**（898-925）大力发展福州海贸——招徕海舶 + 修筑罗城 + 立招贤馆 + 通使新罗 + 日本 + 南海诸国；福州港自此进入大港时代；**闽国时期福州港**是中国东南海岸最重要的海贸 + 文化 + 宗教枢纽之一。

**两宋**　**北宋**——广州港 + 明州港 + 杭州港 + 泉州港等并立诸路市舶司、福州港为闽中海贸的次级节点；**北宋元祐二年（1087）置福建路市舶司于泉州**——这是闽中海贸的最高官方规格、设司地为泉州而非福州（北宋熙宁置司之说不确）；**南宋至元间泉州港跃升中国第一大港**——福州港作为闽中山货海产的次级港退居二线；**南宋福州淳熙九年（1182）梁克家修《三山志》**——这是福州城最早的现存方志、保留福州港中晚唐至南宋的丰富史料。**明成化间**（1465-1487）**福建市舶司由泉州迁福州**——福州港方在官方海贸体系中代泉州为闽中首港、但此时中国海贸主轴已移至广州 + 月港 + 后来的厦门。

**元代**　元代福州港规模次于泉州港、但仍为东南海贸的核心节点之一；**元置福建行省**——福州为省治；福州港承担省内贡赋 + 海贸 + 海防多重功能。

**明清**　**明代**——福州港继续作为福建省治的核心海港；**明永乐三年至宣德八年（1405-1433）郑和七下西洋**——船队多于福州长乐太平港候风出海——**长乐太平港**（今长乐区闽江口）是郑和船队的重要锚地之一；**明清间福州港**仍为闽中政治 + 军事 + 文化中心、海港地位次于泉州 + 漳州；**清同治五年（1866）福州船政局成立**——闽江马江段成为中国近代海军 + 造船业的发祥地之一；**清光绪十年（1884）马江海战**——中法战争中福州马尾港遭法军舰队袭击、福建水师覆没——这是福州港在中国近代史上的一个 dramatic moment。

**民国至今**　**1949 年后**——福州港作为福建省治的核心港口；**1980s 改革开放后**——福州港 + 马尾港 + 长乐松下港 + 江阴港逐步现代化；**今福州港**是中国东南海岸的重要海港之一、规模次于上海 + 广州 + 深圳 + 厦门、但作为福建省治港仍具核心地位。**今游人重访福州港一带**——所见的是闽江下游沿岸的现代港区 + 马尾船政文化遗址 + 长乐郑和广场 + 闽江口诸岛——盛唐时的福州港作为新兴海港的物理形态早已不在、但作为闽中海贸史 + 中国海港史的物理原点仍在福州市 + 闽江口的地理记忆中。

## 引文

据《元和郡县图志·卷二十九·江南道五·福州》大意——福州治闽县、在闽江北岸；闽江自西北山中三源汇流而下、过建州 + 剑州至福州西郊切出冲积平原、再折东南入海；闽县东郊为津渡 + 集散之地；闽中山货海产由此出入；夏秋多飓、海船候风 + 候潮（闽县附郭 + 闽江入海口 + 海岸气候）。

> 福州长乐郡，中都督府。本闽州，开元十三年更名……
> ——《新唐书·卷四十一·地理志六》（江南东道·福州长乐郡；都督府建置）

> 福州，秦闽中郡地……隋开皇九年平陈，改为泉州；十二年置泉州总管府……武德六年置泉州，景云二年改为闽州，开元十三年改为福州，天宝元年改为长乐郡，乾元元年复为福州。
> ——《旧唐书·卷四十·地理志三》江南东道·福州（长乐郡）（福州名号沿革）

> 闽中夏秋多飓风。
> ——《通典》卷一八二·州郡十二·古扬州下·福州（闽中海港的气候威胁）

> 闽越王无诸者……汉五年，复立无诸为闽越王，王闽中故地，都东冶。
> ——《史记·卷一一四·东越列传》（汉东冶——福州一带海港的最早史载渊源）

> 朕闻闽王王审知，治闽三十年，岁修职贡，海舶遥来，市舶之利、岁入数百万。
> ——《十国春秋》卷九十·闽世家（王审知治闽期间福州港的繁荣；属 755 之后两百年的回响）
