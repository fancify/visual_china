# 龙城

## 一句话

**三燕（前燕、后燕、北燕）都城**——东晋咸康七年（341）慕容皝徙都于此；自此前燕、后燕、北燕三朝相继都龙城；**北魏太延二年（436）灭北燕、龙城退为废都**；**755 时是营州（柳城郡）治柳城——平卢节度治、安禄山起家地**。

## 形貌

龙城在唐河北道**营州柳城郡治柳城县**——具体在**大凌河中游北岸的台塬**（今辽宁省朝阳市）。**地势**：辽西走廊东端、大凌河谷北岸——海拔约 170 米；东接松辽平原、西出大凌河谷可至蒙古高原、南循辽西走廊可至中原——是辽西最重要的地理枢纽。

**三燕龙城的形制**——东晋至北朝时期的都城；今考古勘察显示**主城规模约南北 1500 米、东西 1200 米**——是十六国晚期至北朝早期辽西的最大城。**城墙**：夯土筑成——比统万城（白色夯土）略简、但规模相当。**城内主要遗存**：
- **太极殿基址**：前燕、后燕、北燕三朝皇宫主殿——今考古确认夯土殿基；
- **和龙宫**：北燕冯跋所建寝宫；
- **东、西、北三大佛寺**：北魏至唐代陆续建——含**朝阳北塔**（始建于北魏太和年间、唐代重修）；
- **三燕宗庙**：祭祀慕容氏 + 冯氏祖先。

**朝阳北塔**：今朝阳市内一座大型方形砖塔——**北魏思燕佛图为前身**（北魏冯太后时期始建，约 480s）；**隋仁寿年间建舍利塔于此**；**唐代修饰 / 改筑**；**辽代再修、形成现存十三檐密檐塔外观**——是**叠加北魏-隋-唐-辽四代遗迹**的复合古塔；今塔高 42.6 米、十三檐、方形——是辽宁省现存最古塔之一。**今塔身外观主体为辽代**，盛唐 755 时所见非今塔模样。

**安禄山时代的龙城**：755 时柳城是**营州**（平卢节度治）所在；**安禄山兼平卢节度自天宝间起**——其起家地即营州柳城。

物候：辽西走廊东端、温带季风气候——夏温冬寒、四时分明；山地丘陵 + 大凌河谷地——是中原通往蒙古高原与朝鲜半岛的天然孔道。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

龙城在唐人的回望里——是**三燕故都 + 自家平卢节度治 + 安禄山起家地** 的三重身份。

**三燕（前燕、后燕、北燕）**：**慕容鲜卑**部落——出辽西、自西晋末渐立。**前燕（337-370）**：慕容皝立国、**东晋咸康七年（341）徙都龙城**；**永和八年（352）前燕慕容俊迁都邺**（今河北临漳）；**前燕亡于前秦苻坚 370**。**后燕（384-407）**：慕容垂建立、**太元九年（384）建都中山**（今河北定州）；**永康二年（397）慕容宝北奔龙城**——龙城自此成后燕北部政权中心；**光始元年（401）后燕迁都龙城**（一说光始七年/407 末叶），是后燕末年回光返照。**北燕（407-436）**：**汉人冯跋**杀慕容熙、立北燕、**都龙城 29 年**——是十六国后期辽西的独立汉人政权。**北魏太延二年（436）灭北燕**——龙城自此退为废都。

**自十六国至北朝 + 隋 + 唐**——龙城旧址一直作为辽西军政中心存在：**北魏置营州于龙城**；**北周、隋、唐相继置营州**——治**柳城**（即龙城旧址）。**唐营州**（柳城郡）：**州郡本身领柳城一县**；营州都督府 + 平卢节度使治此——辖境含羁縻州（奚、契丹、室韦等）；平州、檀州为独立别州——是唐东北边军的重镇格局。

**安禄山与营州**：**安禄山（约 703-757）**——出生在**营州柳城**；**本姓康氏**（粟特康国系）、**母阿史德**（突厥）；母后改嫁突厥将军安延偃、**禄山遂冒姓安**（《旧唐书·安禄山传》明文）。**安禄山幼时即在营州一带成长**——通六种胡语、为蕃将。**开元二十年代（720s 末）起渐升营州军职**；**天宝元年（742）兼任平卢节度使、治营州（柳城）**；**天宝三载（744）兼范阳节度**；**天宝十载（751）兼河东节度**——一身三镇、辖 18 万兵、是盛唐边军最大势力。

**安禄山的"营州起家"**——意味着**唐天宝十四载（755）十一月起兵的安禄山**，其家、其将、其部、其根都在营州柳城（龙城旧址）。**这是龙城在 755 时最深的政治意义**：从十六国三燕故都，变为唐安史之乱起兵的根据地。

### 此刻

天宝十四载这一年的龙城（柳城）——**已废 319 年（436 北燕亡 → 755）**；以**营州柳城郡治柳城县**身份存在；同时是**平卢节度治、安禄山起家地**。

**三燕宫殿基址**：盛唐 755 时太极殿基址、和龙宫等遗迹应仍可辨——但已 319 年未维护、夯土残败。**朝阳北塔**：自北魏太和年间始建至 755 时已约 270 年——盛唐多次重修、塔身完整。

**唐营州的政治布局**：
- **营州刺史驻柳城**；
- **平卢节度使治此**；
- **柳城是营州唯一的"州治县"**——其他属州（平州、檀州等）有自己的州治。

**安禄山起兵前夜（755 末）**：营州柳城是**安禄山部众的大本营**——大批粟特、突厥、契丹、奚等胡兵驻屯于此；**安禄山本人时居范阳（幽州）**——但其根、其家、其后方都在营州柳城。**11 月初九安禄山在范阳起兵**——首批部众多自营州 + 范阳调动；**营州在安史之乱前期是叛军后方核心**。

### 此刻意味

对天宝十四载的营州人来说——龙城旧址是**自家州城**：三燕故都的物理遗存、唐营州州治、平卢节度的中心；岁岁有州县官署运作。对长安那一代士人来说——龙城是**辽东最远的一座军州**：盛唐边塞诗中"营州"二字常出现、但具体熟知龙城三燕故事的不多。对**安禄山本人**——龙城是出生地、是起家地、是 755 起兵后的"根据地"。

唐人尚不知道——再过半年安禄山起兵已成 + 半年后玄宗西奔 + 8 年后安史平；**营州 + 平卢节度在安史中反复易主**——**至德二载（757）唐军收复河北、营州归唐**；**贞元间（780s）契丹兴起、营州渐成唐与契丹的争夺之地**；**唐末营州为契丹所据**——是辽东边境的最终转手。**北宋金元间龙城旧址成游牧汉地交界**；**辽兴中府（1041）建于柳城**——契丹辽朝在此建北方陪都；**金兴中府、元大宁路**继续此地的军政重镇身份。**今游人来朝阳**——所见的是**朝阳北塔（北魏-唐-辽四代叠加）、三燕龙城遗址（1990s-2010s 发掘）+ 朝阳博物馆**——**唐人在 755 所见的龙城旧址主体犹存**。

## 子条目

### 三燕宫殿基址（太极殿 + 和龙宫）

**前燕、后燕、北燕三朝在龙城的皇宫**：
- **太极殿**：前燕慕容皝徙都龙城后建（约 341 年间筑）——是太极殿主殿；后燕、北燕沿用。今**朝阳市内三燕龙城遗址公园**展示有夯土殿基（1990s-2010s 考古发掘确认）。
- **和龙宫**：**北燕冯跋所建**——是北燕政治中心。"和龙"是龙城别名（北燕通称"和龙"）；后世亦称北燕为"和龙国"。
- **三燕宗庙**：祭祀慕容氏（前燕、后燕）+ 冯氏（北燕）祖先。

**今三燕龙城遗址**：1990s-2010s 多次考古发掘——确认主城范围、太极殿基址、城门遗迹；**核心遗存出土物**：北魏至唐代陶片、瓦当、佛教造像残件等——证实自三燕至唐代龙城延续不断。

### 朝阳北塔（北魏-唐-辽四代叠加）

**朝阳市内一座方形砖塔**——**北魏思燕佛图为前身**（北魏冯太后时期始建、约 480s）；**隋仁寿年间建舍利塔**；**唐代修饰 / 改筑**；**辽代再修、形成今十三檐密檐塔外观**。**塔身考古特征**：内核为北魏夯土塔基 → 外砌隋唐砖石 → 辽代砖檐 + 顶刹——是**单座塔上五代遗存叠加**的代表。**今塔高 42.6 米、十三檐、方形**——是辽宁省现存最古塔之一。**1980s-1990s 朝阳北塔修缮发掘**——出土**佛骨舍利 + 数百件北魏至辽代佛教造像、玻璃器、金银器**——震动当时考古界。**盛唐 755 时**——此处有佛塔（北魏-隋-唐叠加状态）、是龙城最显的物理标识；但**今游人所见的辽代外观主体非盛唐原貌**。

### 安禄山起家地（营州柳城）

**安禄山（约 703-757）**——出生在**营州柳城**（即龙城旧址）；幼时即在营州一带成长。**身世**：**本姓康氏**（粟特康国系）、**母阿史德**（突厥）；母改嫁突厥将军**安延偃**——禄山遂冒姓安（《旧唐书·安禄山传》明文）；**通六种胡语**——是营州边境多民族环境的产物。**渐升边军职**：开元二十年代起在营州军中任职、屡立战功；**天宝元年（742）兼平卢节度使、治营州**；**天宝三载（744）兼范阳**；**天宝十载（751）兼河东**——一身三镇。**起兵之前的安禄山**：**居范阳幽州**（其平卢节度治在营州、但范阳节度治在幽州、其常驻地是幽州牙城）；但**营州柳城仍是其家族 + 旧部的根**。

**755 时营州的实际状态**：粟特、突厥、契丹、奚等胡兵驻屯密集；汉族官署运作正常；安禄山起兵前夜各方面物资在大规模调动——**这是 755 时龙城旧址最特殊的政治时刻**。

### 三燕（前燕、后燕、北燕）的政治史

**前燕（337-370）**：慕容皝立国——慕容鲜卑部落势力；东晋咸康七年（341）徙都龙城；永和八年（352）慕容俊迁都邺；370 前燕亡于前秦苻坚。

**后燕（384-407）**：慕容垂建立——慕容皝之子；太元九年（384）建都中山（河北定州）；后期慕容熙末叶（406-407）复都龙城；407 慕容熙被冯跋所杀、后燕亡。

**北燕（407-436）**：**汉人冯跋杀慕容熙、立北燕**——是十六国后期辽西的独立汉人政权；**都龙城 29 年**；436 北魏灭北燕。

**三燕的特殊性**：是十六国唯一一个**鲜卑（慕容氏）→ 汉人（冯氏）** 转手的政权系列；**冯跋的"汉人立国" 形象**——在十六国胡族政权中独特。**冯跋后裔出冯太后（北魏献文帝拓跋弘皇后、孝文帝拓跋宏祖母）**——是北魏孝文帝改革的关键人物；**冯氏与北魏的政治联姻** 是 5 世纪北方政治的核心一环。

### 龙城-柳城-营州-朝阳（名称延续）

- **龙城**（337-436）：三燕都；
- **统万镇 + 营州**（436-唐初）：北魏始置；
- **柳城县**（唐）：营州治所县；
- **营州**（唐）：州郡本身领柳城一县；都督府辖境含奚、契丹等羁縻州；
- **平卢节度治**（742-755）：营州的军镇身份；
- **兴中府**（辽）：契丹辽朝陪都；
- **大宁路**（元）：元代重镇；
- **朝阳**（今）：今辽宁省朝阳市。

**这一系列名称延续 1400 余年——同一座城基**。**唐人在 755 嘴里的"龙城"或"柳城"或"营州"**——指的都是同一片台塬；今游人来朝阳所见的也是同一片地。

## 历史长河（755 → 今）

**中晚唐**　**安史之乱后平卢重心南移**：营州受奚、契丹等周边部族压力增大；**8 世纪 60 年代后唐在营州控制趋失**；**贞元间（780s）契丹兴起**、营州成唐与契丹的争夺之地；**唐末营州为契丹所据**——是辽东边境的最终转手。中晚唐营州的具体物理状态——朝阳北塔仍立、三燕宫殿基址渐荒。

**辽 + 金 + 元**　**辽兴中府（1041 建于柳城）**——契丹辽朝在此建北方陪都之一；**朝阳北塔在辽代大规模重修**（建中圣寺塔、即今朝阳北塔的辽代外形主体）。**金兴中府、元大宁路**——继续辽东重镇身份。**辽金元三朝**——朝阳一带为东北重镇、城内人口稠密、佛教兴盛。

**明清**　**明洪武初置朝阳卫**；**清乾隆三十九年（1774）设朝阳县**——朝阳之名自清乾隆起。**清代朝阳为辽西汉蒙交界重镇**——但三燕龙城遗存已大半荒。

**民国至今**　**1949 年后朝阳为辽宁省辽西重镇**；**1980s 起多次考古发掘三燕龙城遗址**——确认太极殿基址、城门、城墙等。**朝阳北塔 1985-1992 大修**——出土北魏、唐、辽佛教文物 + 佛骨舍利；**今朝阳北塔列国家级重点文物保护单位**。**2000s 起朝阳市建设三燕龙城遗址公园**——以三燕宫殿基址 + 朝阳北塔为核心。**今游人来朝阳**——所见的是**朝阳北塔（北魏-唐-辽四代叠加）、三燕龙城遗址、朝阳博物馆**——**是十六国三燕都城遗存的最完整一处**。**唐人在 755 所见的那座三燕故都旧址 + 营州柳城治所**——一千二百多年后主体犹存。
