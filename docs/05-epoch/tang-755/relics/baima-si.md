# 白马寺

## 一句话

河南府洛阳东郊古刹，汉传佛教祖庭；按永平求法旧说，白马驮经入洛——唐人到此先见的是佛教入华的起点。

## 形貌

白马寺不在洛阳城里，在城东。

自隋唐洛阳外郭城东郭出建春门，循洛水北岸向东。再走十二公里，过几片麦垄、几道渠塘——便是寺。这地方原属汉魏洛阳故城的西郊，那座旧城就在白马寺再向东三公里——铜驼街的台基、永宁寺塔的故基，皆露在野草里。寺立的时候（68），汉魏旧城正是当时的洛阳；六百余年后唐人再来，旧城已是废墟，寺却仍立——废墟边一处还在燃香的院落，是它独特的姿态。

寺占地不大，约二百亩。中轴一线南北贯穿：南为山门，门内东西分置摄摩腾、竺法兰二高僧的瘗骨墓——一座寺以两座墓做开门，是因为这两个人是寺立之由。再北为天王殿、大佛殿、大雄殿、接引殿、毗卢阁——这一套五进格局是后世明清营造的样子，盛唐时形制不全如此，但中轴的骨架自始未变。寺东南独立着一座方塔——唐人见到的塔已是北魏所重修，砖石叠涩、九层方形。塔名"齐云"是后起，这名字到金代才定。

寺的"白马"得名于驮经的白马——传永平十年（67），蔡愔、秦景于大月氏遇摄摩腾、竺法兰二僧，以白马驮《四十二章经》及佛像东归，永平十一年（68）汉明帝敕建此寺以居之。寺立后白马即坐化于此，葬于寺门外——后世立白马石雕一对作纪念，现存山门外两侧的白马即为宋代所凿。这两匹石马上承汉，下接千年，是寺中最朴素的一处实物。

寺的特别之处不在建筑——明清以后历代重修，木构非汉非唐；今日所见地面建筑最古的主要是金代齐云塔（大定十五年 1175 重修），其余殿宇多为元明清重修后的样子。**特别之处在于"立"**——按永平十一年立寺旧说，它被尊为汉传佛教"祖庭""释源"；官方叙述中常称"中国第一古刹"。

## 大唐天宝十四载（755年）

### 永平求法传说

755 年的白马寺已经七百岁。

唐人对这寺的所有想象都从一个传说开始——后汉永平七年（64）正月某夜，明帝梦见一金人在殿庭上飞行，头顶有日光。次日朝问群臣，太史傅毅答："西方有神，其名曰佛——陛下所梦，将无是乎？"明帝乃遣中郎将蔡愔、博士秦景等十八人西使天竺求法。三年后（67），使团于大月氏国遇见两位天竺僧人——摄摩腾（Kāśyapa Mātaṅga）、竺法兰（Dharmaratna）。使团以白马驮经像东归，永平十年（67）冬至洛阳；明帝命于洛阳城西雍门外别立精舍以居二僧——次年（68）寺成，因白马驮经之故，命名"白马寺"。

这个故事最早见于东汉末牟子《理惑论》、再见于《后汉书·西域传·天竺国》，到北齐魏收《魏书·释老志》成定型版。**唐人接受这个版本**——他们读《魏书》、读杨衒之《洛阳伽蓝记》（北魏 547 年成书），白马寺的来历自此以"永平求法"为正史叙事。

后世学者对这个传说颇有疑议——汉明帝感梦的具体年月、十八使臣的姓名、白马驮经的细节，皆有夸饰之嫌。但**佛教自西域东传一事大致不诬**——东汉确有西域僧侣到洛阳译经，**永平、建初年间是汉传佛教的真正起点**。白马寺即使不是 68 年立的，也是这一程东传的最早地面纪念碑。**至于具体年份，唐人不问；他们只知道这是中国第一寺**。

### 摄摩腾 竺法兰 译经处

寺立之初最大的功绩是译经。

摄摩腾与竺法兰在白马寺译出《四十二章经》——**这是第一部汉译佛经**。该经辑录佛说四十二段简短教义，文风近《老子》《论语》语录，是为初传时面向汉人士大夫的入门读本。译笔朴素，几乎是逐字"道""无为""清净"等汉地哲学词汇套用——这种译法后世称"格义佛教"，是佛教在中国扎根之初最自然的姿态。

竺法兰后续又译《十地断结》《佛本生》《佛本行》《法海藏》等数部——大半已亡佚。摄摩腾、竺法兰各自卒葬白马寺，墓在山门内东西两侧——这是唐人所能见到的两座最古老的佛门墓塚。中唐宪宗朝有僧澄观（740-839）过白马寺，为二高僧墓写赞——这种"过寺写赞"是唐人对祖庭最朴素的致敬。

寺内"清凉台"传为摄摩腾、竺法兰译经之处——是一处砖砌高台，约六米。这台子的具体始建年代不可考，但唐人见到的清凉台已是台。台上据传原有"明帝问梦"的塑像与碑——这些都属传说附会，但**台与传说同代**——两者都不晚于北魏。

### 北朝 隋 唐 重修

寺在汉之后，先入魏。

魏明帝青龙三年（235）于寺中立浮屠九级。北魏迁洛（494）后，孝文帝、宣武帝、灵太后皆崇佛——白马寺一度更名"招提寺"，旋复旧名。北魏熙平元年（516）灵太后于洛阳建永宁寺塔（高约一百四十米，三十七年后焚于火），白马寺塔虽不及永宁寺塔之高，但作为更古老的存在，地位反而稳。

北魏 547 年杨衒之作《洛阳伽蓝记》追忆都城旧观——其卷四"城西"开篇即白马寺，写道："白马寺，汉明帝所立也，佛入中国之始。"——这一句**为后世千五百年的白马寺地位定调**。

北周武帝建德三年（574）大兴灭佛——洛阳一时寺毁僧散，白马寺亦在劫中。但七年后周宣帝大象元年（581）即复佛，旋入隋——文帝、炀帝两代崇佛，白马寺得隋大业重修。

唐承隋制，白马寺由长安、洛阳两京僧录司双管。唐代它更像祖庭象征——玄奘以后唐代主要译场在长安大慈恩寺、慈恩寺，但白马寺作为"祖庭"在唐人心中地位独存。武则天朝、玄宗朝皆有续修；至天宝十四载（755），寺立于洛阳东郊——具体殿宇格局不可按今日明清中轴倒推。

### 薛怀义 武则天朝

武周一朝，白马寺出过一个奇人。

薛怀义本名冯小宝，是市井卖药的恶少。据《旧唐书·薛怀义传》《资治通鉴》卷二〇三，他被武则天的女儿千金公主举荐入宫——武则天召幸后令出家，剃发为僧，赐姓薛，名怀义，**令出入禁中**。**垂拱初**（《资治通鉴》系于垂拱元年 685 冬，《旧唐书·薛怀义传》作"垂拱初……修故白马寺，寺成，自为寺主"，两源略异），薛怀义主持修故白马寺、寺成后自为寺主——这是白马寺历史上最具戏剧性的一段。

薛怀义在白马寺任内大兴土木：**修明堂、天堂**——**垂拱四年（688）**拆乾元殿造明堂，天授元年（690）前后续督造天堂，是武周营造紫微城的实际主持者；至于"修白马寺塔"一节，今所见唐宋一手史料未见明确支撑。这种"以白马寺主持兼营禁中"的身份在中国佛教史上是独例。

但故事的尾声不漂亮。武则天后宠日渐转向御医沈南璆——薛怀义嫉妒。证圣元年（695）正月十六日夜，**薛怀义纵火烧天堂——火延明堂，二殿一夜俱烬**——这是中古第一木构、武周第一象征的灾难夜。武则天虽心痛，仍命重建明堂；同年二月，**令薛怀义之死于宫中**——具体死法《资治通鉴》《旧唐书》两源略异，但都指向被太平公主一系所杀。**死后葬白马寺**——这位住持以纵火洛阳两大殿、终被宫中所杀，反过来葬入自己主持的祖庭。

755 年的盛唐僧人路过白马寺，看到的塔、看到的部分院落，皆有薛怀义遗迹。武周已过五十年，这段故事还在被讲。

### 玄奘归洛阳与白马寺

贞观十九年（645）正月二十四日，玄奘自天竺归——抵长安朱雀大街，迎者倾城。

但玄奘归国后，最早抵达的不是长安，而是洛阳。彼时太宗驻跸洛阳行宫（隋唐洛阳城紫微城），玄奘奉诏先至洛阳谒见——根据慧立《大慈恩寺三藏法师传》卷六，玄奘于洛阳谒见太宗于仪鸾殿，太宗劝其还俗辅政，玄奘辞——其后乞还嵩山少林寺译经，太宗不许，命其至长安弘福寺译。

**关于玄奘在洛阳期间是否驻锡白马寺**——传世史料无直接记载，但《大慈恩寺三藏法师传》载玄奘在洛阳期间"宿于宫禁、出入仪鸾"——出入紫微城。从紫微城至白马寺约十二公里，玄奘是否曾过寺礼祖庭，唐宋两代史料皆未明记。**唐人传说言玄奘曾过白马寺礼摄摩腾、竺法兰墓**——这种传说在宋以后白马寺碑铭中渐多，但盛唐时未必有确证。

可以确证的是：玄奘归后所译《大唐西域记》自序中明引摄摩腾、竺法兰故事——他自觉地把自己接在永平求法这条线索之末。**自摄摩腾至玄奘，六百年——白马寺立于这条线索的起点**。755 年的洛阳僧人讲玄奘故事，必从白马寺讲起。

### 此刻

755 年的白马寺由洛阳僧录司管，住持是何人不可考。寺仍是东都佛教中心之一——但论盛唐两京佛门重镇，长安诸寺（大慈恩寺、西明寺、青龙寺）的体量远过白马；洛阳本地的福先寺、敬爱寺也已超白马的现役规模。

**白马寺胜在一件事——古**。

唐人来白马寺，不是为了听最新经义，是为了**站在中国佛教的起点**。摄摩腾、竺法兰墓在山门内，清凉台在寺内深处，塔在东南——这三件东西不可复制。盛唐有钱，可以再建一座更大的塔、修更精的塑像；但**"汉永平十一年立"这五字，无可重复**。

天宝十四载十一月，安禄山自范阳起兵——十二月十二日叛军破上东门入洛阳。叛军入城后烧掠数日，白马寺距外郭东郭十二公里，火势所及但不严重。安史乱中白马寺多次易手——叛军以洛阳作伪燕都，寺为伪燕僧录司所辖；唐军两次收复，寺归唐——历乾元、宝应、永泰诸年，最终仍存。这种"乱而不毁"的命运将贯穿白马寺整个余生——它太早立、太牢、又不在城里——每次大乱都被掠，但都没烧透。

## 历史长河（755 → 今）

**中晚唐**　至德二载（757）十月唐军收复洛阳——白马寺归唐。但盛唐两京双都格局已散，洛阳由帝国中心退为副京，白马寺的地位也随之回落——不再是"东都之首寺"，而是"东都之一寺"。中晚唐文献中白马寺的出场渐少。会昌五年（845）武宗灭佛——洛阳大寺多毁，白马寺幸存——大约因为它"祖庭"的地位让朝廷不忍尽毁，也或因为它在城外、容易被疏漏。宣宗大中元年（847）复佛，寺得修——但已不复盛唐之盛。

**五代北宋**　五代乱世洛阳易主五次，白马寺多有损坏。北宋建立后，太宗淳化三年（992）大修白马寺——这是它在两宋的第一次重生。北宋"齐云塔"始建年份在两说之间——一说后周显德二年（955）始造，一说北宋开宝至太平兴国年间（968-984）始造；至**金大定十五年（1175），齐云塔最终重修为今存形制**——十三层密檐砖塔，高约二十五米，塔基沿用唐代旧位。这座塔今天看到的就是金代的样子——**它本身已经八百五十岁**，是寺中现存最古老的地面建筑。塔名"齐云"出自宋金人——盛唐人见到的塔不叫这个名字。

**元明清**　白马寺历元、明、清三朝皆有重修，但每一次都越来越偏离唐宋旧制。元至正间一修，明嘉靖三十四年（1555）司礼监掌印太监黄锦大修——今寺中央大佛殿、毗卢阁等主要殿宇皆为黄锦此次所立的基础。清康熙、乾隆间皆有续修。**寺的整体地位在元明清渐有下降**——一是中国佛教重心南移、北方寺院普遍式微；二是洛阳自宋以后已不复为都城，**"东都首寺"的政治依托没了**；三是齐云塔之外的木构皆为明清新建，文物价值不及南方诸宋元古寺。乾隆四十一年（1776）毕沅修《关中胜迹图志》——同代洛阳学人补撰《洛阳古迹考》，白马寺仍列祖庭，但篇幅已不长。

**民国至今**　1908 年伯希和过洛阳，访白马寺，记其残破。民国二十年代寺已残败——1931 年蒋介石、宋美龄至洛阳曾过白马寺，捐资小修。抗战期间寺受兵火波及，多殿损坏。

1949 年后，1961 年白马寺列入第一批全国重点文物保护单位。1973 年起开始全面大修——这是白马寺自明嘉靖以来最大规模的一次工程，恢复殿宇、补塑佛像、修整中轴。1980 年代日本佛教界以"白马寺是中日佛教共同祖庭"为契入点频频访问；1992 年印度政府以**释源**之地为由捐建白马寺"印度风格佛殿"——2010 年落成；2014 年泰国佛殿、2014 年缅甸佛殿陆续落成——白马寺由此成为**世界上唯一一座汇集中、印、泰、缅四国佛殿的寺**。这种格局是 755 年的人完全想不到的——汉传佛教的祖庭，反过来成了多国佛教的会客厅。

至于香火——**1958 年至今从未真正断过**。文革期间寺有损坏但主体存——1972 年柬埔寨西哈努克亲王访华提出参观白马寺，周恩来命紧急修整。这种"因外宾访问得救"的命运曾让白马寺成为彼时仅存数座保持完整的中国古寺之一。

**1958 年到 2026 年，整整一个甲子又十年——寺仍在烧香。从永平十一年（68）算起，凡 1958 年**。这是中国地面上香火最久的一处佛门——比任何一座宋元木构都老六百年，比任何一处明清寺院都老一千一百年。盛唐时它已七百岁；今天它即将两千岁。

## 引文

杨衒之《洛阳伽蓝记·卷四》（北魏 547）：

> 白马寺，汉明帝所立也，佛入中国之始。寺在西阳门外三里御道南……寺上经函至今犹存，常烧香供养之。

——这一句"佛入中国之始"为白马寺定调，唐人读此句，自此白马寺的祖庭身份再无可议。

《魏书·释老志》（北齐 魏收）：

> 后孝明帝夜梦金人，项有日光，飞行殿庭……乃遣郎中蔡愔、博士弟子秦景等使于天竺，写浮屠遗范……愔之还也，以白马负经而至，汉因立白马寺于洛城雍关西。摩腾、法兰咸卒于此寺。

——永平求法故事的定型版。

慧立《大慈恩寺三藏法师传·卷六》（玄奘归洛阳）：

> 法师既至洛阳，谒太宗于仪鸾殿……太宗劳问甚厚，问以彼国风土，法师对扬甚悉。

——玄奘归后第一站是洛阳，而非长安。白马寺与玄奘的具体关联未直书。

王昌龄《诸官游招提寺》（唐人写白马寺一别名"招提"的诗篇之一）：

> 招提此山客，绳床见徐果。
> 旧识半生缘，何如顿成果。

——盛唐诗人写白马寺的诗存世不多，王昌龄此首为其一。

### 白马寺的时间跨度对照

| 年份 | 事件 | 寺龄 |
|---|---|---|
| 永平十一年（68） | 白马寺立 | 0 |
| 北魏 547 | 杨衒之作《洛阳伽蓝记》载白马寺 | 479 |
| 武周 688 | 薛怀义任住持 | 620 |
| **天宝十四载（755）** | **盛唐 doc 主场——寺已七百岁** | **687** |
| 唐会昌灭佛 845 | 白马寺幸存 | 777 |
| 金大定十五年 1175 | 齐云塔重修为今存形制 | 1107 |
| 明嘉靖三十四年 1555 | 黄锦大修，主要木构改面今貌 | 1487 |
| 民国二十年代 1930s | 寺残败，蒋介石、宋美龄捐修 | 1860s |
| 1961 | 全国重点文物保护单位 | 1893 |
| **2026 年（今）** | **寺仍在烧香** | **1958** |

**盛唐时它七百岁，今天它两千岁差四十二年**——这是项目所有 POI 中时间跨度最长的几座之一。盛唐距 2026 凡一千二百七十一年，而白马寺自立寺至盛唐之间也已七百年——**两段时间叠起来，等于地球文明史的一片可观切片**。

### 三个译经传统的源头

| 译者 | 时代 | 主场 | 关系 |
|---|---|---|---|
| 摄摩腾 / 竺法兰 | 东汉永平 (67-) | 白马寺清凉台 | 起点——《四十二章经》 |
| 鸠摩罗什 | 后秦 401-413 | 长安草堂寺、逍遥园 | 中段——《法华经》《维摩诘经》《金刚经》 |
| 玄奘 | 唐贞观-麟德 (645-664) | 长安大慈恩寺翻经院 | 顶峰——《瑜伽师地论》《大般若经》 |

**这三个译场是中国佛教史的三个里程碑**——而白马寺是这条线索的起点。**摄摩腾、竺法兰之于鸠摩罗什、玄奘，是汉文之于盛唐文学的关系**：可以说粗朴，但不可能不在。
