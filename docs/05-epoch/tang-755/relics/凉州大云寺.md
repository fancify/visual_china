# 凉州大云寺

## 一句话

**武周遗产 + 河西陇右佛教 + 凉州城内物理 anchor**——武则天天授元年（690）颁《大云经》于天下、诏诸州各置大云寺一所；凉州大云寺即其中一所、立在陇右道凉州武威郡治姑臧县城内；**唐 755 时是哥舒翰节度河西陇右、戍卒商旅过凉州的城内日常**；**再过八到十年（763-765）凉州陷蕃**——大云寺随凉州入吐蕃辖境、唐廷之外八十余年。

## 形貌

凉州大云寺在唐陇右道**凉州武威郡治姑臧县城内**——天宝元年（742）改凉州为武威郡、治姑臧县（今甘肃省武威市凉州区）；具体位置在姑臧县城内一隅、史源记载零散，**一说在城东门内、一说在城北街**——本 doc 不预设具体方位。**寺基由前代旧寺改置而来**——一说是改自姑臧县城内一座旧寺（**弘藏寺**之说见于地方碑刻 + 后世志书、不能完全坐实**；可能性较高但本 doc 不当作定论）；自天授元年（690）改名"大云寺"——此后名遂沿用。

**寺院规制**：盛唐凉州大云寺具体形制 + 殿堂 + 塔院数 史载零散——本 doc 不预设细节（今所见凉州大云寺残迹寥寥、主体已不存）。唐代大云寺大体应有**山门 + 前殿 + 大殿 + 经楼 + 钟楼 + 塔院 + 僧房**；据武则天 690 诏令、各州大云寺须**藏《大云经》+ 设升高座 + 由僧讲解**——故大云寺有讲经常制；具体规模未见可信细节。**寺东西南北邻**——皆姑臧县城内街巷 + 民居 + 市井；大云寺与凉州其他寺院（如罗什寺、北凉旧寺等）共构姑臧城内的"佛寺底子"。

**钟楼与铜钟**：凉州大云寺最具象的物理遗存是**大云寺铜钟**——唐代铸、具体铸年史载不详、今藏甘肃省武威博物馆；钟通高约 2.4 米、口径约 1.45 米、重约 4 吨、钟身饰云雷纹 + 八卦纹 + 飞天纹；是**唐代凉州大云寺唯一存世可见实物**。**唐代铜钟可为唐代大云寺遗物；不能确证 755 已在寺中**——具体铸年不可考、是否 755 已悬挂钟楼亦无确证；本 doc 不预设其在 755 一刻已是大云寺钟楼实物。

**姑臧城内的佛教底层**：盛唐凉州城内寺院众多——除大云寺外、有**罗什寺**（鸠摩罗什 后秦 401 入长安前居此 17 年的旧地）+ **北凉时代旧寺**（沮渠蒙逊 412 都姑臧时所置）+ **诸唐寺**等；凉州是河西陇右佛教重镇——大云寺嵌入其中、不孤立。**罗什寺 ≠ 大云寺**——是两寺、唐人 755 时皆在；二者皆是姑臧城内佛教遗产的不同层位。

物候：河西走廊东端——温带半干旱大陆性气候；夏短秋早、冬寒久长；春迟夏短、四月城外杏花始开；八月霜降、胡杨变金。**寺院内外的钟磬声 + 商旅声 + 戍卒声 + 市井声**——按唐代州城寺院常制可推；具体大云寺钟磬声形 史载零散——**今藏武威博物馆的那口"凉州大云寺铜钟"为后世可见遗物、不能坐实 755 现场已悬用**（见下"钟楼与铜钟"）；本 doc 不预设 755 凉州人晨昏所闻的钟声即出此钟。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

凉州大云寺在唐人的回望里——是**武则天武周遗产 + 河西陇右佛教中心 + 凉州城内物理 anchor**。

**大云寺的设立**：**武则天天授元年（690）秋七月**——**沙门十人伪撰《大云经》表上**（《资治通鉴》卷二〇四）；**实际上《大云经》早在北凉 + 唐初已由昙无谶 + 后世译师 译出**——武周改撰 + 附会其中"女王统治"经文以助武则天称帝之论；**同年九月武则天受尊号"圣神皇帝"、改唐为周**——大云寺即此政治-宗教转折的物理产物。**九月制颁《大云经》于天下 + 诏两京诸州各置大云寺一所**——是武周政治神学的全国布点。**自此各州（包括凉州）皆有大云寺一所**——凉州大云寺即此一所；具体由凉州都督府 + 凉州地方官（690 时凉州刺史；755 时**武威郡太守 / 凉州都督府长官**——天宝元年/742 改州为郡、刺史改称太守）奉诏改置一座旧寺为大云寺。

**大云寺与凉州佛教旧底子**：**凉州自十六国五凉以来即是河西佛教重镇**——**前凉张轨 + 后凉吕光 + 北凉沮渠蒙逊**皆都姑臧、皆崇佛；**鸠摩罗什 384-401 在凉州 17 年**——后秦 401 入长安前居此（《高僧传》卷二）；**昙无谶 412-433 在北凉译《大般涅槃经》**——是凉州在北凉时代的佛教高峰。**唐初 + 武周 + 盛唐**——凉州佛教底子未断；**武则天天授元年置大云寺**——是在凉州本有的佛教底层上再叠一层武周色彩。**唐人 755 时看凉州大云寺**——既看见武则天 690 诏令、也看见前代凉州佛教的连续。

**大云寺的政治意义**：**武则天颁《大云经》 + 置大云寺**——是**女主称帝的政治神学**：经中"女王统治转轮王法"等语被引为武则天受命之凭；**两京（长安 + 洛阳）+ 诸州大云寺**形成全国布点——是武周政治在地方的物理 anchor。**705 武周亡 + 唐中宗复辟**——**李唐复唐后部分大云寺改名（如改为龙兴寺）+ 部分仍存原名**；**凉州大云寺在 705 之后是否改名 史源不详**——本 doc 取通说"凉州大云寺名沿用至盛唐"（与"凉州大云寺铜钟"等物证一致）；**至少 755 时凉州城内仍有"大云寺"在**。

**755 时凉州大云寺的状态**：**凉州是河西节度治 + 西陲第一城**——"七里十万家"（**"七里十万家"是盛唐凉州城规模的概括语 + 文学修辞、与户籍口数不必逐字对照**；《新唐书·地理志》载凉州武威郡天宝间户二万二千四百六十二 + 口十二万二百八十一）；**大云寺在姑臧县城内一隅**——是凉州城内常驻的中等规模寺院；**寺内有住持 + 僧众数十至百余**（具体人数史载零散）+ **藏《大云经》+ 升高座讲经**（按 690 诏制）；**寺当有钟楼 + 寺钟为唐代寺院常制**，但**今藏武威博物馆的那口"凉州大云寺铜钟"为后世可见遗物、不能坐实 755 现场已悬用**（见上"钟楼与铜钟"）。**寺前街市**：商旅过往、戍卒进出、香客信众——是姑臧城内日常的一片市井角。

**哥舒翰与凉州大云寺**：**天宝十二载（753）哥舒翰兼河西陇右两镇节度使**（《资治通鉴》卷二一六）——**自此至 755 哥舒翰多次往返长安 + 鄯州 + 凉州**（河西节度治凉州、陇右节度治鄯州；两镇节度治所相距数百里、皆需哥舒翰巡视）。**哥舒翰过凉州时多驻河西节度使府**——位姑臧县城内；**大云寺与节度使府皆在城内**——哥舒翰过凉州之日、未必每过必入大云寺、但**大云寺寺檐 + 寺前街市 + 武周遗产**——是哥舒翰在凉州城内日常视野的一部分（**大云寺寺钟之声 754-755 是否确入哥舒翰耳、无证可凭**；唐代铜钟为后世可见遗物、不能坐实 755 现场已悬用 / 听过）。**755 这一刻哥舒翰已年初自陇右回长安养病**——再未西归凉州；**756 春自长安出潼关御安史叛军 + 756 六月兵败潼关被俘**——自此哥舒翰与凉州大云寺、再无关涉。

### 此刻

天宝十四载这一年的凉州大云寺——**寺院严整、武周遗产 + 河西陇右佛教 + 凉州城内日常**（寺院晨昏当有钟磬声、按唐代州城寺院常制可推；**今藏武威博物馆的那口"凉州大云寺铜钟"为后世可见遗物、不能坐实 755 现场已悬用 / 闻过**）。

**寺院状态**：盛唐 755 时凉州大云寺主体仍立；具体形制史载零散。**僧众**——按 690 诏制大云寺有住持 + 僧众讲《大云经》；具体在 755 这一刻凉州大云寺住持是谁、史无确载。**寺内日常**——晨钟暮鼓 + 讲经 + 香客 + 寺前市井；**寺外街市**——姑臧县城内民居 + 商铺 + 戍卒营。

**755 时大云寺的政治氛围**：**武则天 690 诏置大云寺**距 755 已**六十五年**——武周亡（705）已**五十年**；**唐人 755 时**——已不大将大云寺当作武周政治神学的活体载体；**大云寺已渐成"前朝遗产 + 城内常住的中等寺院"**。**但寺名"大云"仍是武周记忆的物理标记**——盛唐人过凉州 + 见大云寺 + 想起武则天 690 诏令——是一种**怀旧而不再政治化**的物质回声。

**755 时大云寺前的实际过往者**：**河西节度戍卒 + 粟特商人 + 西域使节 + 香客 + 姑臧市井百姓** —— **凉州是河西节度治 + 丝路东段的最大中转**；**大云寺正在姑臧城内**——这些人流的一部分必经大云寺门前（具体频次史无确载、但寺当在城内主要街巷之一）。**寺内讲经**——按制有僧讲《大云经》；具体讲座频次 + 听众规模 不可考。

**755 末 + 756 后的凉州命运**：**安禄山十一月起兵于范阳**——**河西陇右兵东调救京**（**756 初哥舒翰自长安守潼关、其麾下河西陇右兵成潼关防线主力**）；**自此河西陇右西陲守备日空**——为 763 吐蕃东扩埋下种子。**广德元年（763）吐蕃东扩大举 + 陇右诸州陆续陷蕃**；**永泰元年/765 凉州陷蕃**（**主流取 763-765 间逐渐陷蕃**——具体凉州陷年史源略异、一说 763、一说 765；本 doc 取 765）；**自此凉州大云寺入吐蕃辖境约八十余年**（765-848）；**唐宣宗大中二年（848）张议潮起义 + 大中五年（851）复凉州**——凉州大云寺一度复归唐廷；但**唐廷已无力恢复盛唐河西节度** + **大云寺旧规模已残**。

### 此刻意味

对天宝十四载的凉州人 / 姑臧人来说——大云寺是**自家城内的常住寺院 + 武周遗产**：日常香客 + 讲经 + 市井——一座"在那里"的寺（寺院晨昏当有钟磬之声、按唐代州城寺院常制可推；**今藏武威博物馆的那口铜钟为后世可见遗物、不能坐实 755 现场已悬用**）。对长安那一代士人来说——大云寺是**全国大云寺布点的一所** + **武则天天授元年（690）诏令的物理 anchor**：盛唐人过凉州 + 见大云寺 + 想起武周 + 想起七十年前的诏令；**怀旧而不再政治化**。

唐人尚不知道——再过八到十年（763-765）凉州陷蕃、**大云寺将随凉州入吐蕃辖境**——自此约**八十年**间大云寺在吐蕃手中（765-848）；**唐宣宗大中五年（851）张议潮复凉州** + 大云寺一度复入唐境；但**唐廷已无力恢复盛唐河西节度** + 大云寺旧规模已残；**唐末五代凉州数易其主**——大云寺亦渐衰。**今与大云寺相关联可见的唐代物质遗存——唯余一口铜钟**（现藏甘肃省武威博物馆）；**但唐代铜钟为后世可见遗物、不能坐实 755 现场已悬用 / 听过**——本 doc 不预设此钟即"唐人在 755 所听的那钟"。盛唐那座大云寺的钟楼 + 大殿 + 塔院——已不复存。

## 子条目

### 大云经与武周天授元年（690）颁经 + 置寺

**《大云经》之颁**——是武则天称帝前夕的政治神学起手：
- **天授元年（690）秋七月**——**沙门法明等十人伪撰《大云经》**（实际《大云经》早有昙无谶译本 + 唐初译本——所谓"伪撰"指武周改撰 + 附会经中"女王统治转轮王法"等语）；
- **七月表上武则天 + 盛言神皇受命**——为武则天称帝铺路；
- **九月武则天受尊号"圣神皇帝"、改唐为周**；
- **同月制颁《大云经》于天下 + 诏两京诸州各置大云寺一所 + 藏《大云经》+ 升高座讲解**（《资治通鉴》卷二〇四）。

**两京诸州大云寺**——是武周政治神学在地方的全国布点；**凉州大云寺即此布点之一**。**705 武周亡 + 唐中宗复辟**——**部分大云寺改名（如改为龙兴寺）+ 部分仍存原名**；**凉州大云寺在 705 之后名号去留 史源不详**——本 doc 取通说"沿用至盛唐"。**至少 755 时凉州城内仍有"大云寺"在**（与"凉州大云寺铜钟"等物证一致）。

### 凉州大云寺铜钟（唐代铸、现藏武威博物馆）

**凉州大云寺铜钟**——是凉州大云寺唯一存世可见实物：
- **铸年**：唐代铸、具体铸年史载不详；通称"唐代铜钟"；
- **形制**：通高约 2.4 米、口径约 1.45 米、重约 4 吨；钟身饰**云雷纹 + 八卦纹 + 飞天纹**；钟体保存完整、唐代铸造工艺成熟；
- **现存**：**甘肃省武威博物馆**藏、国家一级文物；
- **历史**：钟身保存完整、至清代可证仍悬挂大云寺钟楼（清代地方志可证）；**自唐代铸成至何时入大云寺钟楼、史载不详**——唐代铜钟为后世可见遗物、不能坐实 755 现场已悬用 / 听过；近代移入武威博物馆保护。

**这口铜钟今为大云寺最具象的物理遗存 + 唐代铸造工艺标本**——**但唐代铜钟为后世可见遗物、不能坐实 755 现场已悬用 / 听过**：具体铸年不可考、是否盛唐 755 一刻已悬大云寺钟楼、是否 755 凉州人确曾闻此钟声——皆无确证。**本 doc 不预设盛唐人、中晚唐人、西夏人、元明清人"皆听过同一口钟"**——只能说**唐代某一刻起至清末此钟悬挂大云寺钟楼**（清代地方志可证、盛唐段无证）；近代移入武威博物馆保护。**其他大云寺殿堂 + 塔院 + 山门皆不存**——铜钟之于 755 大云寺、最多只能作"唐代凉州大云寺物质遗产的一件可见证物"看待。

### 罗什寺 ≠ 大云寺（姑臧城内两寺）

**罗什寺与大云寺是凉州城内两座不同寺院**：
- **罗什寺**——**鸠摩罗什 384-401 在凉州 17 年的旧地**（后秦 401 入长安前居此）；后秦人为罗什立寺、寺名"罗什"；自后秦至唐——罗什寺一直在姑臧城内（现存凉州区罗什寺塔为后世重建、唐遗构已不存）；
- **大云寺**——**武则天天授元年（690）置**；据"弘藏寺"等旧寺改置（具体史载零散）；自唐至清——大云寺亦在姑臧城内。

**两寺皆在 755 时姑臧县城内 + 皆是凉州佛教遗产的不同层位**：罗什寺承五凉 + 后秦佛教 + 鸠摩罗什译经；大云寺承武周政治神学。**两寺距离 + 具体位置 史载零散**——本 doc 不预设。**现代地方文献偶有混淆罗什寺与大云寺**——两寺确为两座、不可混；皆是凉州佛教史的不同维度。

### 武则天 690 颁经 + 各州大云寺布点

**天授元年（690）置大云寺于天下诸州**——是武周政治神学的全国布点：
- **两京**（长安 + 洛阳）各置大云寺；
- **诸州**各置大云寺一所（约三百余州）；
- **凉州大云寺**即此诸州之一。

**今存可考的大云寺遗存**：
- **凉州大云寺**（武威）——存铜钟；
- **泾州大云寺**（甘肃泾川）——1964 + 2012 出土武周大云寺地宫舍利石函 + 金棺银椁；
- **西安大慈恩寺**之外的长安大云寺（旧址不确）；
- **各州大云寺**——多已不存或并入他寺。

**武则天颁经 + 置寺的政治意义**——是**女主称帝的政治神学**；**705 武周亡后大云寺渐失政治色彩**——但作为"前朝遗产 + 地方寺院"持续存在；**凉州大云寺至盛唐 755 仍存**——即此 doc 所写之物。

### 哥舒翰 + 河西陇右节度 + 凉州大云寺（753-755）

**天宝十二载（753）哥舒翰兼河西陇右两镇节度使**（《资治通鉴》卷二一六）——一身两镇、**河西、陇右重兵**（具体兵数史载零散、《通典·兵典》载河西节度使常备七万三千 + 陇右节度使常备七万五千、合计约十四万余——精确数需出处、本 doc 不预设具体数字）；**自此至 755 哥舒翰多次往返长安 + 鄯州 + 凉州**：
- **鄯州**——陇右节度治；哥舒翰常驻地；
- **凉州**——河西节度治；哥舒翰巡视地；
- **长安**——朝觐 + 述职。

**哥舒翰过凉州时**——驻河西节度使府（位姑臧县城内）；**大云寺与节度使府皆在城内**——哥舒翰过凉州时未必每过必入大云寺、但**大云寺寺檐 + 寺前街市 + 武周遗产**——是哥舒翰在凉州城内日常视野的一部分（**大云寺寺钟之声 754-755 是否确入哥舒翰耳、无证可凭**；唐代铜钟为后世可见遗物、不能坐实 755 现场已悬用 / 听过）。**755 这一刻**——哥舒翰已年初自陇右回长安养病；**756 春自长安出潼关御安史叛军 + 756 六月兵败潼关被俘**——自此哥舒翰再未西归凉州。**大云寺 ↔ 哥舒翰**这一段物理关联——是盛唐西陲军政与天下转折的微妙交集。

### 凉州陷蕃（763-765）+ 大云寺入吐蕃八十年

**广德元年（763）吐蕃东扩大举 + 陇右诸州陆续陷蕃**；**永泰元年（765）凉州陷蕃**（**主流取 763-765 间逐渐陷蕃**——具体凉州陷年史源略异：《资治通鉴》卷二二三广德元年/763 载"吐蕃陷秦、成、渭等州"——尚未及凉州；《新唐书·吐蕃传》载凉州陷于代宗永泰年间/765；本 doc 取 765）。**自此凉州大云寺入吐蕃辖境**——**765-848 约八十余年**间为吐蕃所辖。

**吐蕃辖境下的大云寺**：**吐蕃信佛**（赤松德赞 754 即位后大兴佛教 + 791 桑耶寺立 + 译经事业）——**凉州大云寺在吐蕃辖下并未被毁**——仍作寺院使用；**寺内僧众或汉或藏**。**这一段史载零散**——敦煌文书中偶见吐蕃辖境下凉州寺院的记录。

**唐宣宗大中二年（848）张议潮起义**（敦煌文书 P.2962《张议潮变文》）+ **大中五年（851）复凉州**——凉州大云寺一度复归唐廷；但**唐廷已无力恢复盛唐河西节度** + 大云寺旧规模已残。**张议潮归义军实际控制凉州时间较短**——后凉州数易其主、归义军衰；**大云寺渐衰**——但铜钟存。

## 历史长河（755 → 今）

**中晚唐**　**755 末安史起 + 756 哥舒翰潼关之败 + 河西陇右兵东调救京 + 西陲守备日空**——**广德元年（763）吐蕃陷秦、成、渭等州**；**永泰元年（765）凉州陷蕃**——**凉州大云寺入吐蕃辖境约八十余年**（765-848）。**吐蕃辖下大云寺未被毁** + 寺内僧众或汉或藏。**唐宣宗大中二年（848）张议潮归义军起义**（敦煌文书 P.2962）+ **大中五年（851）复凉州**——大云寺一度复唐廷；但**唐廷已无力恢复盛唐河西节度**。

**五代**　**唐末归义军衰 + 凉州数易其主**——**唐末 + 吐蕃别部 + 党项 + 回鹘** 等先后据凉州；**大云寺渐衰** + 但铜钟存 + 寺基未毁。

**两宋至元**　**北宋初凉州为西凉府** + **景祐五年（1038）党项李元昊建西夏 + 凉州为西夏西凉府**——**大云寺在西夏辖境内续作寺院**；**西夏崇佛 + 凉州佛教再振**（**西夏 天授礼法延祚元年/1038 凉州感通塔碑**——是西夏凉州佛教高峰一刻）。**元代蒙古统一**——凉州为甘肃行省；**大云寺规模渐降**。

**明清**　**明初置凉州卫**——属陕西行都司；**大云寺续作寺院、规模渐降**；**清乾隆朝**——凉州大云寺铜钟仍悬挂寺中钟楼；**清中后期凉州大云寺寺基渐失**——但铜钟存。**民国前后大云寺主体已不存** + 仅余铜钟与残基。

**民国至今**　**1949 年后凉州大云寺铜钟移入甘肃省武威博物馆保护**——是大云寺唯一存世可见实物；**今武威市凉州区**——所见的是**大云寺寺基遗址 + 残碑 + 部分塔基**（具体保存状况已残）；现"罗什寺"在凉州市内（**罗什寺塔后世重建 + 唐遗构已不存**）——是另一寺、不要与大云寺混。**今武威博物馆藏唐凉州大云寺铜钟**——是与盛唐大云寺相关联的最具象物理遗存；**但唐代铜钟为后世可见遗物、不能坐实 755 现场已悬用 / 唐人在 755 一刻已闻此钟**——具体铸年不可考、是否 755 已悬挂大云寺钟楼亦无确证；本 doc 不预设"唐人在 755 所听的那钟声"即此钟。**盛唐那座大云寺的钟楼 + 大殿 + 塔院**——皆已不存。游人今日所见、所抚的、是这口唐代铸钟的器物本身（今已不再敲）。
