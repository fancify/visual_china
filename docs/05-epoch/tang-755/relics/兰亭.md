# 兰亭

## 一句话

越州山阴城西南约十二里**兰渚之畔**的水边小亭——东晋永和九年三月三日王羲之与谢安、孙绰等四十一人于此修禊、流觞曲水、即席作《兰亭集序》——书法第一行书、文学千古名篇之诞生地；**755 时去永和禊集已四百零二年、真迹陪葬昭陵已一百零六年**，唐人凭吊的是摹本与一片荒废故地。

## 形貌

兰亭在越州山阴县城西南约十二里（按《元和郡县图志》卷二十六·江南道二·越州山阴县口径）——**会稽山北麓、兰渚山下**，是一片山中小盆地里的水边亭址。

地势：南依**兰渚山**（海拔约 130 米，孤峰不高，因山中产兰而得名）；山北麓出一带水——曰**兰溪**——西流入鉴湖。亭址在兰溪南岸、兰渚山北麓的山脚——前临溪、后倚山、左右**茂林修竹**——是一片可坐数十人的山间清地。**这便是王羲之《兰亭集序》"此地有崇山峻岭、茂林修竹、又有清流激湍、映带左右"所写的地理底色**——崇山是会稽山主脉、峻岭是兰渚山一线、清流激湍是兰溪自山中下来的一段。

兰溪自山中下来——曲折回环、水势不急——是适合**流觞**（盛酒之觞顺水漂流）的最好地形。永和九年王羲之等人在溪的拐弯处设亭、引溪水绕亭一回——觞自溪上漂下、停于谁前则谁饮酒赋诗——**兰亭使流觞曲水成为后世最著名范式**（流觞习俗有更早源头，但兰亭一会以《序》传世，成为后人凡言"流觞曲水"必溯之范本）。今所见兰亭仍存"流觞亭"，溪水如旧——但**亭体早非晋时原物**。

物候——越州山阴一带气候温润、四时分明；春日三月，山中竹笋初生、兰花初开、桃李在岸——是江南最适合野宴的一段时令。"天朗气清、惠风和畅"——王羲之笔下永和九年那个上巳日的天气，是江南春日的标准模样。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

兰亭在唐人的回望里是**东晋永和九年三月三日王羲之与四十一名士修禊作《兰亭集序》之地**——是中国书法史与文学史共同的圣地。

**王羲之（303-361）**——琅琊临沂（今山东临沂）人，出身**琅琊王氏**——东晋第一流门阀。王羲之少年随家族南渡，居建康（今南京）；中年出任**会稽内史**（约永和七年/351 至永和十一年/355 间）——会稽郡治山阴、即今绍兴。**永和九年（353）三月三日上巳节**——王羲之以**会稽内史**身份、约**谢安、孙绰、谢万、王凝之、王徽之、王献之、孙统、谢藩、徐丰之、袁峤之**等**四十一人**于山阴城南兰亭修禊（按：《兰亭集》载与会者四十一人；后世异文或作四十二人，本 doc 严守四十一人口径）。

**上巳节**——三月上旬巳日（魏晋后渐固定于三月三日）——是先秦以来"祓禊"（临水沐浴、祓除不祥）的古俗。**永和九年这一日"天朗气清、惠风和畅"**——王羲之等人沿兰溪设亭、流觞曲水、即席赋诗——共得诗三十七首、十一人未成诗者各罚酒三杯（《兰亭集》目录可考）。诗成之后——**王羲之即席作序**——这便是**《兰亭集序》**（亦称《兰亭序》《禊帖》），全文三百二十四字。

《兰亭集序》一篇——**前半写景与雅集之乐**："此地有崇山峻岭、茂林修竹、又有清流激湍、映带左右"；**中段转入人生感慨**："夫人之相与、俯仰一世……当其欣于所遇、暂得于己、怏然自足、不知老之将至"；**末段归于生死之叹**："向之所欣、俯仰之间、已为陈迹……死生亦大矣、岂不痛哉"。三百二十四字、由乐转悲、由景入哲——是中国散文中最早把"个人之乐"与"宇宙之短"并置的一篇。

**书法即在这一篇上**——王羲之即席挥毫、二十八行、三百二十四字——传后人称为**"天下第一行书"**。**全篇二十一个"之"字、各不相同**——这是后世品评《兰亭》最常提的细节。王羲之自珍其稿——传家于子孙——至七世孙**智永禅师**（陈隋间永欣寺僧），智永传**辨才禅师**（永欣寺、隋唐间）——**真迹自王羲之家私藏三百余年**。

**贞观间**——唐太宗酷爱王羲之书法、四方求《兰亭》原本不得；闻辨才藏于永欣寺、命**萧翼**（监察御史）**计赚兰亭**——萧翼乔装赴永欣寺、与辨才论书、终以诡道得真迹献太宗——史称**"萧翼赚兰亭"**（唐何延之《兰亭记》详记此事；后世画家（传）阎立本作《萧翼赚兰亭图》（实为宋摹、原作存疑）——是这段故事最有名的图像载体）。**萧翼赚兰亭事**——是唐人传说、史源唯何延之一家，真伪存疑；但**真迹入太宗御府是基本史实**。

**贞观二十三年（649）五月**——太宗崩、遗诏以《兰亭》原本陪葬昭陵——**真迹自此入土、从此不复见于世间**。**自 649 至 755**——已一百零六年。

### 此刻状态

天宝十四载这一年的兰亭——**距永和九年禊集已四百零二年、距真迹陪葬昭陵已一百零六年**。

兰亭故地仍在——兰渚山在、兰溪在、流觞之水绕亭而过——但**晋时所立之亭早已不存**。**自永和九年禊集后**——王羲之仍居山阴数年；永和十一年（355）以与扬州刺史王述不睦称病去会稽内史职、退居山阴东南山中；**升平五年（361）卒于会稽**，年五十九（按 303 生 / 361 卒）。**自王羲之死至 755**——三百九十四年（《兰亭集序》写于 353，到 755 是 402 年；但王羲之本人 361 卒，到 755 是 394 年——两个时距常被混用，本 doc 严守此分）。

**兰亭故地在唐前期渐荒**——南朝宋齐梁陈历经四代、兰亭无主修葺；隋平陈（589）后越州属江南——唐贞观元年（627）分天下为十道、越州属江南道；**开元二十一年（733）析江南为东、西两道**，越州始属**江南东道**——755 这一刻越州为江南东道大州、山阴县境内兰亭故地存而不显。**唐时是否有立纪念建筑、史源未详**——本 doc 不预设盛唐 755 时兰亭故地是否已立"流觞亭""鹅池碑""墨华亭"等今日所见诸物——这些多是明清以来累积形成的纪念地标。**755 时是否有亭未可确证**——参照越州 doc"隋唐时已是地方名胜、州县岁修"之口径，**州县岁修的可能存在**；但具体何制何形史源不足；**唐人凭吊的是地望与摹本**——而非永和原构。

**755 时唐人能见的《兰亭》**——是**摹本**，非真迹。**贞观间太宗命供奉拓书人**——**冯承素、欧阳询、虞世南、褚遂良、赵模、韩道政、诸葛贞**等——**各摹《兰亭》数本**赏赐近臣、藏于内府。**传冯承素摹本（卷首钤神龙小玺、世称"神龙本"；摹成时间不坐实于 705-707 神龙年间）**——以双钩廓填法摹《兰亭》一本——今藏北京故宫博物院——**是公认最接近原作的摹本**。**755 时唐人若欲见《兰亭》**——只能在内府、或得自御赐近臣家中、或见外间流传之摹本（多为虞世南本、褚遂良本系统）；**真迹永不再见**。

**安史前夕**——这一刻的兰亭故地——荒中带静、静中带远——是江南山阴一片可访可不访的去处。再过半年安禄山起兵——但**越州在江南东道、远在战火之外**；**兰亭故地在安史乱中物理保存基本完好**。

### 此刻意味

对天宝十四载的山阴人来说，兰亭是**州城西南十二里那一片"王右军禊集故地"**——日常的山中清地、上巳前后偶有士人来访。对盛唐文人来说——兰亭是**"风流"的具象**：四十一名士、流觞曲水、即席作书——这一段叙事在六朝至唐成为中国"名士雅集"的最高范式。**王羲之 = 书圣**这一称号至唐太宗朝已定（太宗亲撰《王羲之传赞》"尽善尽美、其惟王逸少乎"）——盛唐凡习书者，必摹《兰亭》。

但 755 时唐人凭吊兰亭的感受是**特殊的**——**真迹不在地上、在昭陵之下**；唐人手中的《兰亭》是**摹本**——再好的摹本，毕竟不是王右军本人之笔。**这种"原本永失"的怅惘**——在中晚唐文人诗文中反复出现——是兰亭作为"书法圣地"独特的精神质地。

唐人尚不知道——再过三百余年（北宋），《兰亭》摹本会经苏轼、米芾等人题跋、衍生出"定武本"系统、与"神龙本"并称两大系统；再过八百余年（明嘉靖二十七年/1548），兰亭故地会被山阴知府沈启重建——立流觞亭、御碑亭、鹅池碑、墨华亭——**今所见兰亭主体格局多定于明嘉靖**；再过千年，**永和九年那一片真迹**——仍在昭陵之下，从未再见天日；**唐人在 755 凭吊的那一片荒废故地**——地望大体相承（具体亭址虽屡迁屡建、明嘉靖重建址至今未变），是兰亭传说最稳的物理 anchor。

## 子条目

### 流觞曲水（兰溪绕亭一段）

亭前兰溪自山中下来——曲折回环、水势不急——王羲之等人于溪的拐弯处设亭、引溪水绕亭一回。觞（盛酒之木杯）自溪上漂下、停于谁前则谁饮酒赋诗——共得诗三十七首、十一人未成诗者各罚酒三杯。**"流觞曲水"自此成为中国名士雅集最具象的一个意象**——后世园林（如苏州拙政园、北京潭柘寺、绍兴沈园）多仿之、立"流杯渠"或"流杯亭"——皆出兰亭。**盛唐 755 时溪水仍流**——是兰亭故地最稳的物理元素；**今所见流觞亭为明嘉靖重建**。

### 兰亭集序（《兰亭序》《禊帖》）

王羲之即席所作——**二十八行、三百二十四字**——前半写景、中段转感慨、末段归生死之叹。"天朗气清、惠风和畅"；"崇山峻岭、茂林修竹"；"暂得于己、怏然自足"；"向之所欣、俯仰之间、已为陈迹"；"死生亦大矣、岂不痛哉"——皆出此篇。**全篇二十一个"之"字、各不相同**——是后世品评《兰亭》最常提的细节。**真迹陪葬昭陵后唯余摹本**——755 时唐人见者皆摹本；**今所见《兰亭》皆唐摹**——其中以冯承素**神龙本**（705-707 摹）公认最接近原作。

### 王羲之（书圣、会稽内史）

**王羲之（303-361）**——琅琊临沂人、出身琅琊王氏。**官至右军将军、会稽内史**——故世称"王右军"（**非"王右丞"——王右丞是王维**，两者勿混）。王羲之书法**真行草俱善、自成一家**——破东汉以来钟繇之"古质"、立"今妍"一脉——至唐太宗朝定为"书圣"。**子王凝之、王徽之、王献之**——皆有书名，**王献之**继父风、与父并称"二王"。王羲之**升平五年（361）卒于会稽**、年五十九；葬会稽（具体葬地史源略异）。

### 萧翼赚兰亭（唐贞观间事，史源单一）

**贞观间**——唐太宗酷爱王羲之书法、闻《兰亭》真迹藏于永欣寺辨才禅师手中、命**监察御史萧翼**乔装赴永欣寺与辨才论书——萧翼出示伪本《兰亭》、辨才轻信、出真本相校——萧翼伺机窃归献太宗。**事见唐何延之《兰亭记》**（开元/天宝间作）；**史源单一、真伪存疑**——后世画家**阎立本作《萧翼赚兰亭图》**（一作宋摹本）——是这段故事最有名的图像载体。**但太宗陪葬昭陵 + 真迹自此失传是基本史实**——史源充足；萧翼具体如何"赚"则是唐人小说化的部分。本 doc 严守"萧翼赚兰亭为唐人传说"口径。

### 真迹陪葬昭陵（贞观二十三年/649）

**贞观二十三年（649）五月**——唐太宗崩、**遗诏以《兰亭集序》原本陪葬昭陵**（昭陵在今陕西礼泉九嵕山）——**真迹自此入土、从此不复见于世间**。**自 649 至 755**——已一百零六年。**后世传五代温韬盗昭陵、《兰亭》真迹复出**——是宋以后笔记小说之记、**史源不足、本 doc 不取**；**最稳的口径仍是：真迹陪葬昭陵、自此不见**。

### 神龙本（冯承素摹本，今存最佳）

**传冯承素摹本（卷首钤神龙小玺、世称"神龙本"；摹成时间不坐实于 705-707 神龙年间）**——内府供奉拓书人冯承素以**双钩廓填法**摹《兰亭》一本——因卷首钤"神龙"年号小玺、故世称"神龙本"——今藏**北京故宫博物院**。**神龙本是公认最接近王羲之原作的摹本**——保留涂改痕迹、用笔起讫、墨色浓淡——是研究王羲之书法最重要的物质载体。**盛唐 755 时神龙本已存数十年**（具体摹成年份史源不确，唐摹本系统之内传冯承素本系唐内府摹本之一），唐人能见的最近原作之物即此一脉。其他重要摹本——**虞世南本**（贞观间摹）、**褚遂良本**（贞观间摹）、**赵模本**等——皆唐摹，755 时俱在世。

## 历史长河（755 → 今）

**中晚唐**　安史乱中越州未受兵燹大破坏；兰亭故地在乱中保存如旧。**755 之前盛唐文人过越州者**——**贺知章**（自号"四明狂客"、天宝三载/744 告老归山阴并卒于其岁）有越州之归；**李白**于天宝六载（747）后两度南游剡中、过越州——《送贺宾客归越》"镜湖流水漾清波、狂客归舟逸兴多"即此期作——是盛唐越州山阴的文学余响。**中晚唐文人过越州者**——**元稹**（长庆/宝历间任越州刺史）、**白居易**（同期）——皆有兰亭咏。**晚唐**——《兰亭》摹本流传更广、文人争相题跋——兰亭"凭吊真迹之失"的怅惘渐成中晚唐书学一脉的母题。

**两宋**　**北宋初**——《兰亭》定武本（传为欧阳询所摹、刻于定州武军——故名"定武本"）大行于世——与神龙本并称两大系统。**苏轼、米芾、黄庭坚**等北宋第一流书家皆题跋《兰亭》——**苏轼《题晋人帖》《跋兰亭后》**、**米芾《书史》**——皆深论兰亭之得失。**南宋**——金兵屡至江南、越州一度受兵燹；兰亭故地略有损坏；**南宋后期**——越州为南宋后方、官府重视、兰亭故地始有简略修葺。

**元明清**　元代兰亭无大事；**明嘉靖二十七年（1548）山阴知府沈启大修兰亭**——立**流觞亭、御碑亭、鹅池碑、墨华亭**——**今所见兰亭主体格局多定于明嘉靖**。**清康熙三十七年（1698）**康熙帝南巡过山阴、御书《兰亭集序》立碑——即今**御碑亭**所立"康熙临兰亭序碑"；**清乾隆十六年（1751）**乾隆帝南巡至兰亭、亦御书《兰亭集序》并题诗——立于御碑亭碑阴——故此碑**正面康熙临、背面乾隆题**——清两朝皇帝同立一碑——是兰亭最有名的一件清代遗物。**清代兰亭已是绍兴府/山阴第一名胜**（越州之名南宋以后已废、元代改置绍兴府、清因之）——士大夫南来、贾客过山阴，皆以至兰亭礼王右军为人生一事。

**民国至今**　**1963 年（浙江省人民委员会公布）兰亭为浙江省级文物保护单位**（具体批次以浙江省文物局公布为准）；**2013 年列全国重点文物保护单位**。**1990s-2010s 多次考古勘察**——确认明嘉靖重建址周边有南朝至唐五代陶片、夯土遗痕，但**未发现可直接证明"永和九年禊集亭址"的物理证据**——这是兰亭考古的实际状态。**今兰亭为绍兴市柯桥区兰亭镇兰亭风景区**——流觞亭、御碑亭、鹅池碑、墨华亭、王右军祠等俱为明嘉靖以来历代修葺、清代续修、民国与当代修缮所累积。**唐人在 755 所见的那一片荒废故地**——地望大体相承（明嘉靖重建址至今未易，永和原亭址则未获考古实证）；但**永和九年那一篇真迹**——仍在昭陵之下、唐人见不到，今人也见不到。
