# 垓下

## 一句话

**淮北平原沱河南岸的楚汉决战故地**——汉五年十二月（BC 202）韩信合诸侯兵围项羽于此、四面楚歌、项羽悲歌"虞兮"（虞姬死事从后出文献《楚汉春秋》）、突围至乌江自刎；**自此距 755 已 956 年**（公元无 0 年），唐人对垓下精确位置已不能坐实——一说徐州彭城西南、一说亳州谯县东、田野间一片旷土。

## 形貌

垓下故战场在唐河南道**徐州（彭城郡）与亳州（谯郡）交界**的淮北平原上——**具体位置历来争论**：主流说在今安徽蚌埠固镇县濠城镇韦庄一带（沱河南岸），1965-1980s 当地考古确认有秦汉之际兵器与夯土壁垒遗存；亦有河南鹿邑说、安徽灵璧说。**755 时唐人对垓下精确位置已不可坐实**——《史记·项羽本纪》只记"项王军壁垓下、兵少食尽"，未给地理坐标；唐人凡过淮北凭吊垓下，多在徐州彭城西南至亳州谯县东这一带的旷野间凭气象推度。

**地势**：黄淮平原南缘、淮河北支沱水南岸——华北平原与江淮平原过渡地带、土厚水浅、四野无险。**沱水（今沱河）自西北来、东南去**，与淝水、濠水共同构成淮北平原的三条南北向小水系——垓下故地正在沱水南岸的一片冲积台地上。**地貌**：无山、无大林、无险隘——是华北平原典型的"四战之地"；楚汉两军在此决战、依赖的不是地形、是兵力 + 粮道。

**唐 755 时垓下故地**：田耕岁岁、旷野无垠；沱水仍东南流；**故战场地表无可辨遗物**——汉初战场已 956 年远（BC 202→AD 755；公元无 0 年），夯土早被农田覆盖；唐人来凭吊见的是**一片秋草 + 沱水 + 不知何处是当年项王军壁**的旷土。**唐徐州治彭城（今江苏徐州）**——距垓下故地约一百五十里东北；**唐亳州治谯县（今安徽亳州）**——距垓下故地约一百二十里西北；**两州治皆不在垓下故地**，故地是徐亳两州边缘的旷野。

物候：黄淮平原、暖温带季风——四时分明、春多沙、夏有沱水汛、秋高、冬寒。**汉五年十二月垓下决战正在冬月**——史载夜半起寒风、四野楚歌、汉军围楚军于沱水南岸、是中国军事史上最有名的一个寒冷冬夜。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

垓下在唐人的回望里——是**中国文学史最悲烈的一夜 + 楚汉相争的决定性终章**。**自垓下至 755 共 956 年**（BC 202→AD 755；公元无 0 年）——比邺城至 755（580→755 = 175 年）远五倍有余；唐人凭吊垓下的距离感、近于今人凭吊唐代——遥远、空旷、只凭典籍。

**楚汉相争的脉络**（盛唐文人皆习《史记》《汉书》者必知）：**秦末群雄并起**（BC 209 陈胜吴广起义 + BC 207 项梁项羽起兵 + 刘邦起兵于沛）；**BC 206 项羽分封十八诸侯**——自立西楚霸王、都彭城（今江苏徐州）；分刘邦为汉王、都南郑（今陕西汉中）；**BC 205 楚汉相争**——刘邦自汉中出陈仓还定三秦、东出函谷与项羽战于荥阳—成皋间约三年；**BC 203 鸿沟之约**——楚汉以鸿沟（今河南荥阳—商丘间古运河）为界、中分天下、项羽东归；**BC 203 末**——刘邦背约追击项羽至阳夏（今河南太康）；**BC 202 十二月**——项羽退至垓下、兵少食尽、汉军合围——**垓下决战自此始**。

**垓下决战**（BC 202 汉五年十二月）：**韩信将三十万**（《史记·高祖本纪》载此为号称之数，"诸侯兵皆会"未另给确数）+ 孔将军 + 费将军等合围项羽于垓下；**项羽兵约十万**（亦为《史记》所载号称、非清点之实）——壁垒于沱水南岸。**夜半汉军四面起楚歌**——**这是中国军事心理战最早、最有名的一笔**：《史记·项羽本纪》原文"夜闻汉军四面皆楚歌、项王乃大惊曰：汉皆已得楚乎？是何楚人之多也"——**项羽闻之知大势已去**。**项羽悲歌**——《项羽本纪》原文"于是项王乃悲歌慷慨、自为诗曰：力拔山兮气盖世、时不利兮骓不逝、骓不逝兮可奈何、虞兮虞兮奈若何"——**这首四句"垓下歌"是中国诗歌史上第一首署名个人抒情诗之一**、亦是中国"英雄末路"主题的原型。**虞姬和歌**——《史记》原文未明记和歌词、《楚汉春秋》（陆贾）残卷补记虞姬亦自歌答："汉兵已略地、四方楚歌声、大王意气尽、贱妾何聊生"；**虞姬之死**——《史记》记"美人和之"未明记其死、后世（《楚汉春秋》以下）补叙虞姬自刎于项羽前；**"霸王别姬"**这个文学标题是后世（元杂剧《霸王别姬》以下）才有的——**《史记》原文未写"别姬"二字**、本 doc 严守《史记》口径。

**项羽突围与乌江自刎**：项羽夜半率**八百骑**南突围、过淮水至阴陵（今安徽定远西北）迷道、至东城（今安徽定远）剩二十八骑、最后至**乌江**（今安徽马鞍山和县乌江镇）——乌江亭长舣船待项羽渡江、项羽辞曰"天之亡我、我何渡为"——**自刎于乌江**。**注：乌江与垓下相距约二百里、是两地不可混**——垓下是决战处、乌江是自刎处；后世文学常将"垓下"+ "乌江"作为一段连写、但物理上是两地。

**刘邦称帝建汉**：**汉五年（BC 202）二月甲午**——刘邦即皇帝位于汜水之阳（今山东定陶）；**汉初定都洛阳、五月徙都长安**——自此两汉相续约四百年（西汉 BC 202-AD 8，中断于新莽 9-23，东汉 AD 25-220），皆以"高祖斩白蛇 + 垓下灭项"为开国双典。**垓下决战是汉室开国的决定性一笔**——盛唐文人凡读《史记》《汉书》皆习此节、皆能背"力拔山兮气盖世"。

**唐诗中的垓下与虞姬**——盛唐至中唐，凡过淮北凭吊楚汉故地者皆有诗。**项羽 + 虞姬意象**：在唐诗中频频出现、是"英雄末路 + 红颜薄命"的双重原型；**"霸王"作为称谓**——唐人称项羽多用"项王"或"楚霸王"、近于《史记》原文。**唐人对垓下精确位置的不解**——是 755 时凭吊垓下的一个特殊难点：典籍只记"垓下"二字、不给坐标；唐人多在徐州 + 亳州 + 泗州一带的旷野间凭气象推度、不能坐实。

### 此刻

天宝十四载这一年的垓下故战场——**已远 956 年**（BC 202→AD 755；公元无 0 年）；属唐河南道**徐州（彭城郡）与亳州（谯郡）交界**的淮北平原；**精确位置唐人已不能坐实**。

**主流说**（今地理学界以固镇县濠城镇为定）——在 755 时属**徐州彭城郡 / 亳州谯郡 / 泗州临淮郡一带**（具体地点不可坐实；下邳为泗州属县、非郡名；《元和郡县图志》卷九河南道未明记垓下治、亦未给精确位置；后世地理志各持一说）。**注**：**宿州系唐宪宗元和四年（809）始置**、755 时宿州尚不存在——这是写垓下不可踩的一个时态雷区；755 时此地不属宿州。

**沱水**：仍自西北来、东南去；755 时沱水水量充沛、是淮北平原最稳的小水系之一；**唐人凡过淮北凭吊垓下、多记"渡沱水入垓下"**——但渡哪一处沱水即是当年项王军壁、唐人已说不准。**故战场地表**：**汉初土垒早被农田覆盖**、地表无显遗物；**民间俗称**：垓下附近的旷土民间有"项王城"+ "霸王城"+ "九里山"（民间附会古战场名）等俗称——皆是民间附会、非《史记》原文记载。

**徐州 + 亳州两州的治所**：**徐州治彭城**（今江苏徐州）——距垓下故地约一百五十里东北；**亳州治谯县**（今安徽亳州）——距垓下故地约一百二十里西北；**两州治皆距垓下故地约百余里**——故地是两州边缘的一片旷野。**唐徐州 + 亳州的关注点**：徐州是河南道东部的军事 + 漕运重镇（淮河 + 泗水交汇处）、亳州是河南道的农耕 + 文化要地（曹魏故乡谯县所在）——**唐人对垓下的凭吊**多在自这两州出行的旅途中、不是专程考古。

**唐人对垓下的认知**——**楚汉决战 + 项王末路 + 虞姬悲歌**三重叠加：
- **楚汉决战的物理 anchor**——盛唐文人读《史记·项羽本纪》皆知决战在淮北旷野；
- **项王末路的精神 anchor**——"英雄末路"主题以项羽为原型、"力拔山兮气盖世"是唐诗反复化用的典；
- **虞姬悲歌的文学 anchor**——虞姬意象在盛唐至中唐诗中频频出现、是"红颜薄命"原型。

**安史前夕**（即 755 这一刻）——再过半月（农历十一月初九）安禄山起兵于范阳；自范阳南下取河北诸州、再南渡黄河取河南；**徐州 + 亳州皆在安史叛军南下江淮的必经之路上**；**淮北平原（含垓下故地）**在安史之乱中将受兵燹冲击——但 755 这一刻尚未发生。

### 此刻意味

对天宝十四载的徐州人 + 亳州人来说——垓下是**两州交界那片"楚汉旧战场"**：田耕岁岁、过客时来；沱水仍东南流；**项王军壁早不可见**——只留典籍中那一夜的悲歌。对盛唐文人来说——垓下是**中国"英雄末路 + 红颜悲歌"双重原型的物理 anchor**：《史记》中那四句"力拔山兮气盖世"是中国诗歌史上第一首署名个人抒情诗之一——盛唐文人凡习《史记》皆能背、皆能于此地凭吊。

**这片淮北旷野的特殊感**——不同于邺城（六代古都、废墟可辨）+ 不同于长安洛阳（千年帝都、宫阙犹存）：垓下是**一片"几乎无物可凭"的纯净战场**——典籍中那一夜的密度极高、地理上那一片土的稀薄极广——盛唐文人凡过淮北凭吊垓下、皆有一种"凭气象推度而不可坐实"的怅然。

唐人尚不知道——**再过千余年**（1965-1980s），固镇县濠城镇韦庄一带的考古发掘将确认垓下古战场的物理遗存——出土秦汉之际兵器 + 夯土壁垒 + 战马骨骸；**今游人来垓下遗址**：所见的是淮北平原中的一片普通田野 + 沱河南岸的几处夯土残基 + 当地立的"垓下古战场"碑——**与 755 唐人所见的物理景观在大轮廓上是连续的**：旷野如旧、沱水如旧、英雄末路的悲歌如旧——只是位置今人能定、唐人不能。

## 子条目

### 沱水南岸的决战处（地理 + 位置之争）

**垓下决战的物理舞台是沱水南岸的一片冲积台地**——《史记·项羽本纪》原文"项王军壁垓下"、未给精确地理坐标；**自汉至唐对垓下的具体位置一直有歧**。**历代主要说**：
- **沱河南岸说**（主流）：今安徽蚌埠固镇县濠城镇韦庄一带——1965-1980s 当地考古确认秦汉之际兵器 + 夯土壁垒 + 战马骨骸；今地理学界以此为定；
- **河南鹿邑说**：今河南周口鹿邑县东——以"垓"近"陔"、地名学推度；
- **安徽灵璧说**：今安徽宿州灵璧县东南——以《元和郡县图志》某些版本记"垓下在虹县（今安徽泗县）"推度；
- **其他**：尚有亳州说、徐州说等多家。

**755 时唐人对此已不能坐实**——《元和郡县图志》卷九（成书于唐宪宗元和年间 / 9 世纪初）记"垓下在徐州彭城西南"或"亳州东"等多说、未定一处；盛唐文人凡过淮北凭吊垓下、多在徐州 + 亳州 + 泗州一带的旷野间凭气象推度。**本 doc 取主流（固镇濠城）口径 + 明确标注唐人不知**——这是写垓下不可绕开的史源诚实。

### 四面楚歌（中国军事心理战之祖）

**汉五年十二月（BC 202）夜半**——汉军围项羽于垓下、项王兵少食尽。**汉军令降楚之兵唱楚歌**——《史记·项羽本纪》原文"夜闻汉军四面皆楚歌、项王乃大惊曰：汉皆已得楚乎？是何楚人之多也"。**这是中国军事心理战最早、最有名的一笔**——以"心攻"代替"力攻"、以"乡音"瓦解敌军意志；**自此"四面楚歌"成为中国成语**——形容陷入孤立无援的绝境。

**注**：**"楚歌"具体唱的什么、《史记》未记**；后世（戏曲、小说）多附会具体歌词、皆非《史记》原文。**唐人对"四面楚歌"的理解**——已是成语化的典故、不再纠结具体歌词。

### 项羽垓下歌（中国诗歌史的原型）

**《史记·项羽本纪》原文**：
> 于是项王乃悲歌慷慨、自为诗曰：力拔山兮气盖世、时不利兮骓不逝、骓不逝兮可奈何、虞兮虞兮奈若何。

**这首四句"垓下歌"**是中国诗歌史上第一首署名个人抒情诗之一——以**楚辞体**（兮字句）写**英雄末路**主题；**"骓"**是项羽所乘乌骓马、《史记》记其"常骑之"。**"垓下歌"在文学史上的地位**：与刘邦"大风歌"（汉七年/BC 200 过沛县所作）并列汉初最有名的两首楚辞体诗——**"垓下歌"是失败者的悲歌、"大风歌"是胜利者的豪歌**——构成中国文学"成王败寇"的双重原型。**盛唐文人凡习《史记》皆能背"垓下歌"**——是 755 时垓下在文学史上的重量。

### 虞姬与"霸王别姬"（文学化的层叠）

**《史记·项羽本纪》原文记虞姬**："有美人名虞、常幸从；骏马名骓、常骑之"——**未明记虞姬死**；**项羽悲歌后**"美人和之"——**未明记和歌词**。**《楚汉春秋》（汉 陆贾）残卷补记虞姬和歌**："汉兵已略地、四方楚歌声、大王意气尽、贱妾何聊生"——**这首和歌见于类书所引**、是虞姬悲歌的最早文学化版本。

**"霸王别姬"作为标题**——是**后世（元杂剧《霸王别姬》以下）**才有的文学化标题；**《史记》原文未写"别姬"二字**。**虞姬之死**——《楚汉春秋》以下文献补叙虞姬自刎于项羽前；**民间传说**虞姬墓在垓下故地附近（今安徽灵璧县境内有"虞姬墓"+ 庙）——皆是民间附会、非《史记》原文记载。

**唐人对"霸王别姬"的理解**——尚无元杂剧的"别姬"标题、但虞姬意象已在盛唐至中唐诗中频频出现；**李白、杜甫、白居易**等诗中皆有项羽 / 虞姬意象的化用——是 755 时垓下在文学想象中的密度。

### 项羽乌江自刎（与垓下不同地）

**项羽夜半率八百骑南突围**——过淮水至**阴陵**（今安徽定远西北）迷道、问田父、田父绐之曰"左"、项羽左行陷大泽中、汉骑追及；**至东城**（今安徽定远）项羽剩二十八骑、自度不能脱、谓其骑曰"吾起兵至今八岁、身七十余战、未尝败北";最后**至乌江**（今安徽马鞍山和县乌江镇、长江北岸渡口）——乌江亭长舣船待项羽渡江、项羽辞曰"天之亡我、我何渡为；且籍与江东子弟八千人渡江而西、今无一人还、纵江东父兄怜而王我、我何面目见之"——**自刎于乌江**。

**乌江与垓下**：相距约二百里（垓下在今安徽固镇县濠城镇、乌江在今安徽和县乌江镇、二者中间隔淮河 + 长江两条大水系）——**是两地不可混**。**后世文学常将"垓下"+ "乌江"作为一段连写**——杜牧《题乌江亭》"江东子弟多才俊、卷土重来未可知"即写乌江、非写垓下；**本 doc 严守地理口径**：垓下是决战处、乌江是自刎处。

### 韩信围羽（楚汉决战的军事架构）

**《史记·淮阴侯列传》记**：**汉五年十二月**——韩信合诸侯兵围羽于垓下；**韩信将三十万**（《史记·高祖本纪》载此为号称之数）+ 孔将军 + 费将军等节制诸路。**汉军兵力构成**：刘邦本部 + 韩信齐兵 + 彭越梁兵 + 英布九江兵 + 刘贾别军——**号六十万**（《史记》记"诸侯兵皆会"，"六十万"为号称）；**项羽兵力**：**号十万**（《史记》载此为号称）——壁垒于垓下、兵少食尽。

**注**：**"六十万"+ "十万"+ "三十万"皆为《史记》所载号称**——汉初战争兵力史源多虚夸，《史记》对韩信三十万、项羽十万均明用"号"字；本 doc 据《史记》原文取号称口径，明标"号"/"约"字而不取定数。**韩信此役**——是其军事生涯最辉煌的一战、亦是其后被刘邦猜忌的开端（汉六年/BC 201 刘邦伪游云梦擒韩信）；**垓下决战是韩信 + 项羽 + 刘邦三人最后一次同框**——盛唐文人读《史记》凡至此节皆有"三杰末路"之叹。

### 刘邦称帝建汉（垓下决战的政治后果）

**汉五年（BC 202）二月甲午**——刘邦即皇帝位于汜水之阳（今山东定陶）；**汉初定都洛阳**——汉五年五月置酒洛南宫论功；**汉五年（BC 202）五月徙都长安**——纳娄敬 + 张良议、自此西汉以长安（汉长安城、在今西安西北汉城遗址）为都。**两汉相续约四百年**（西汉 BC 202-AD 8，中断于新莽 9-23，东汉 AD 25-220）——皆以"高祖斩白蛇 + 垓下灭项"为开国双典。

**垓下决战是中国第一个大一统帝国（秦）灭后、第二个大一统帝国（汉）立的决定性一笔**——盛唐文人凡读《史记》《汉书》皆习此节、凡至徐州 + 亳州一带凭吊楚汉故地皆有"两汉自此始"之感。**注**：**秦汉之际"大一统"概念是后世史家追加的**——汉初尚有异姓诸侯王（韩信楚王、彭越梁王、英布淮南王、卢绾燕王、张敖赵王、吴芮长沙王、臧荼燕王等七国）、至汉高祖晚年至文景间次第削平；本 doc 不踩这个雷区。

## 历史长河（755 → 今）

**中晚唐**　**安史之乱中（755-763）**——徐州 + 亳州为河南道东部主战场之一；**至德二载（757）张巡睢阳之战**——睢阳即唐宋州（今河南商丘）、距垓下故地约二百里西北；张巡 + 许远以三千守睢阳十月、城破殉国——**这是安史之乱中淮北最有名的一战**、与垓下决战隔 958 年（BC 202→AD 757；公元无 0 年）遥相辉映；**注**：张巡睢阳之战是安史之乱中事、非 755 当时事、本 doc 严守 755 时态。**中晚唐凡至淮北凭吊楚汉故地者皆叹其旷**——**李白《古风》其十**"秦皇扫六合 + 汉刘起阡陌"等写楚汉之际、**杜牧《题乌江亭》**（杜牧中晚唐过乌江亭作，具体系年 / 任所另核；写乌江非垓下）"江东子弟多才俊、卷土重来未可知"——是中晚唐凭吊项羽诗的代表（注：杜牧《题乌江亭》写乌江、非垓下、但二者文学上连为一段）。

**五代 + 两宋**　**五代后梁后唐间**徐州 + 亳州仍为河南要地、垓下故地仍是旷野；**北宋初**——徐州 + 亳州属京东西路 + 淮南东路、垓下故地仍无人坐实位置；**北宋庆历八年（1048）黄河北流改道**——黄河自澶州（今河南濮阳）北决、流入海于今河北沧州一带；**这次改道令淮北水系大变**——但沱水段位置未改、垓下故地段的沱水仍东南流。**北宋诗人凭吊项羽**——**李清照《夏日绝句》**（南渡后/1127 以后作）"生当作人杰、死亦为鬼雄；至今思项羽、不肯过江东"——**是宋代凭吊项羽诗的高峰**、亦是"项羽精神"在宋代南渡后的复活；**苏轼凡过淮北亦有项羽诗**——是北宋凭吊垓下/乌江的典型。

**金元**　**金代垓下故地在金朝南界**（金宋以淮河为界）——属金朝南京路 + 山东西路边境、是金宋边境的一片旷野；**金元间淮河 + 沱河多次改道**——元至元年间黄河南徙夺淮、淮北水系大变、沱河下游改道；**元代凡至淮北凭吊楚汉故地者**——**元杂剧《霸王别姬》**（元末明初作、作者不详）是"霸王别姬"作为文学标题的起源——**自此"霸王别姬"成为中国戏曲史最有名的剧目之一**；**虞姬作为悲剧女主角的形象**——在元杂剧中最终定型。

**明清**　**明清垓下故地属凤阳府（明）+ 凤阳府/泗州（清）**——是淮北平原的一片普通农田；**明清地方志**：**《凤阳府志》《灵璧县志》《虹县志》**等明清地方志皆记垓下位置、各持一说（灵璧说 / 虹县说 / 固镇说等）——**这是明清地理学界对垓下位置最活跃的一段讨论**；**清乾隆 + 嘉庆间**史地学家 + 金石学家凡考察淮北古战场者皆论垓下——**顾祖禹《读史方舆纪要》卷二十一·凤阳府**对垓下位置有详考；**清末民初**——**王国维 + 罗振玉**等以甲骨金石学方法重审垓下、未有定论；**直至 1965-1980s 固镇县濠城镇考古发掘**——以出土秦汉之际兵器 + 夯土壁垒 + 战马骨骸定垓下位置于固镇濠城——是垓下位置之争千年悬案的考古解。

**民国至今**　**1949 年后垓下遗址属安徽省蚌埠市固镇县**——**1965 年起当地始系统调查**；**1980s 起安徽省考古所与中国社科院考古所合作发掘**——确认垓下古战场遗存；**2007 年起固镇县始建垓下古战场纪念园 + 项王故里 + 虞姬墓**（民间附会，非《史记》原文记载、但今已成旅游 anchor）；**今游人来垓下遗址**：所见的是淮北平原中的一片普通田野 + 沱河南岸的几处夯土残基 + 当地立的"垓下古战场"碑 + 项羽 + 虞姬的雕像群——**与 755 唐人所见的物理景观在大轮廓上是连续的**：旷野如旧、沱水如旧、英雄末路的悲歌如旧——只是位置今人能定、唐人不能。**京沪铁路 + 京沪高铁**穿固镇县境而过——是今人至垓下遗址最便的交通方式；**自固镇高铁站至垓下遗址约二十里**——半小时车程可至。
