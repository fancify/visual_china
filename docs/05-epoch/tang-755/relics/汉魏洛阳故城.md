# 汉魏洛阳故城

## 一句话

**东汉 + 曹魏 + 西晋 + 北魏迁洛后四朝旧都的物理废墟**——河南府洛阳县东、偃师县西郊原；**自东汉建武元年（25）光武迁洛起、至北魏永熙三年（534）东魏迁邺止、为都凡 510 年（中有空缺）**；**隋大业元年（605）炀帝命于其西十八里另筑新都洛阳、徙旧城居民百工**——汉魏故城自此正式废；**755 时已废 150 年**——城垣可辨、永宁寺九级浮图塔基孤立邙山南麓——是唐人出洛阳东行必过的旧都遗址。

## 形貌

汉魏洛阳故城在唐河南道**河南府洛阳县东 + 偃师县西**境郊原——具体在**唐洛阳城东约十五唐里**（今河南省洛阳市偃师区翟镇汉魏故城遗址、距唐洛阳城遗址即今洛阳老城直线约 15 公里）。**地势**：**北邙山南麓 + 洛水北岸 + 谷水（今涧水）经城西南入洛 + 伊水汇于东南**——是邙山与洛水之间的一片狭长台地、海拔约 130-150 米。

**故城规模**：**北魏鼎盛期（494-534）周长约 13 公里**——分**内城（汉魏旧城 + 北魏宫城所在）+ 外郭城（北魏孝文帝 + 宣武帝增筑、东西约 10 公里、南北约 7.5 公里）**两重。**755 时所见**——内城城垣（夯土）大体犹存、残高约 3-7 米；外郭城已多坍；城中阡陌 + 农舍混杂。

**多代叠加的物理痕迹**（755 时尚可辨识者）：
- **东汉南北宫基址**——光武至灵帝百余年所修；南宫为礼制 + 朝会、北宫为宫廷生活；**永嘉五年（311）汉刘曜入洛、焚南北宫**；755 时仅余夯土台基；
- **曹魏太极殿基址**——曹叡青龙年间（233-237）大造洛阳宫、太极殿是其核心；**北魏迁洛后沿用太极殿名号 + 重建**；
- **西晋诸殿基址**——多与曹魏旧址叠加；
- **北魏宫城太极殿 + 阊阖门 + 端门 等基址**——北魏孝文帝迁洛（494）后大规模重建；
- **永宁寺九级木浮图塔基**——夯土方台、底边约 100 米见方、残高约 8 米；**755 时是汉魏故城最显的物理标识**；
- **铜驼街遗迹**——汉魏间皇城南向大街、街旁列铜驼二、是"铜驼荆棘"成语的物理原型；755 时铜驼已不存、街址仍辨。

**城垣的多重痕迹**——**东汉版筑 + 曹魏修缮 + 北魏扩筑**三层夯土叠加；今考古剖面可辨。**1962-2010s 中国社会科学院考古研究所洛阳工作队多次发掘**——确认内城十二门、外郭城十三门 + 数十处宫殿基址 + 永宁寺塔基地宫 等。

物候：洛阳盆地、温带季风、夏热冬寒、雨集中夏季；故城内阡陌纵横、邙山南麓土厚易耕——755 时多为农田 + 废墟混合景观。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

汉魏洛阳故城在唐人的回望里——是**前朝四代旧都的物理遗存**：**东汉 + 曹魏 + 西晋 + 北魏迁洛后**——四个王朝、五个世纪、共一座城。

**东汉定都洛阳（25）**——**建武元年（25）十月**光武帝刘秀**定都洛阳**（东汉建国直接定都洛阳、非由长安迁来；《后汉书·光武帝纪》）；**自此东汉以洛阳为都 195 年（25-220）**——历十二帝；**汉灵帝中平六年（189）董卓入洛、焚南北宫 + 挟献帝西迁长安**——东汉洛阳第一次浩劫。

**曹魏定都洛阳（220）**——**黄初元年（220）十一月癸酉**魏文帝曹丕**受汉禅、定都洛阳**（《三国志·魏书·文帝纪》）；**曹叡青龙年间（233-237）大造洛阳宫 + 太极殿、阊阖门**——是汉魏洛阳的盛期之一；**曹魏以洛阳为都 46 年（220-265）**。

**西晋继都洛阳（265）**——**泰始元年（265）十二月**晋武帝司马炎**受魏禅、仍都洛阳**（《晋书·武帝纪》）；**西晋以洛阳为都约 46 年（265-311）**——历四帝；**永嘉之乱**：**永嘉五年（311）六月**汉（前赵）刘曜陷洛阳、**焚宫庙、虏怀帝**（《晋书·孝怀帝纪》）——洛阳自此陷落、史称"永嘉之乱"；**建兴四年（316）愍帝降于长安、西晋亡**（愍帝在长安受降、不在洛阳）；**自此洛阳沦为前赵、后赵、前燕、前秦、北魏前期诸胡政权的边境废都凡 183 年（311-494）**——是汉魏洛阳最阴暗的一段。

**北魏孝文帝迁洛（494）**——**太和十七年（493）冬十月**孝文帝拓跋宏**诏迁都**（《魏书·孝文帝纪》原文）；**太和十八年（494）九月车驾发平城、迁洛阳**——抵汉魏旧城；**太和十九年（495）九月新作金墉宫成**——北魏在汉魏故城内扩建宫城。**孝文帝迁洛是中国历史上最有名的胡族汉化改革**——改鲜卑姓氏为汉姓、易胡服为汉服、禁鲜卑语、定郊祀礼于洛阳南郊（见 [北魏邙山陵群](./北魏邙山陵群.md)）。

**北魏洛阳的盛期（494-534）**——**北魏在汉魏旧城基础上大规模扩筑外郭城**：东西约 10 公里、南北约 7.5 公里——**周长约 13 公里、户口最盛时近 60 万**——是中国 5-6 世纪最大的都城。**永宁寺**：**熙平元年（516）灵太后胡氏立**——城内中央偏北、九级木浮图、高约 130 米（《洛阳伽蓝记》载"去京师百里、已遥见之"）；**永熙三年（534）二月**——**永宁寺九级木塔被雷击焚毁**（《洛阳伽蓝记》原文）——是北魏洛阳最有名的物理事件。

**北魏分裂（534-535）**——**永熙三年（534）十月**孝武帝元修**西奔长安**（投宇文泰）、**东魏孝静帝元善见迁都于邺**（《魏书·废出三帝纪》）；**自此汉魏洛阳作为都城的历史正式终结**——汉魏定洛 - 北魏迁洛 510 年（25-534；中有空缺约 183 年）告终。

**北朝至隋的废都期（534-605）**——东魏迁邺后汉魏洛阳渐废 71 年；西魏 / 北周间洛阳作为边境州郡；隋开皇间洛阳为豫州州治；至**隋大业元年（605）三月丁未**——**隋炀帝诏营东京**（《隋书·炀帝纪》原文）：**于汉魏故城西十八里另筑新都**（今洛阳老城所在）、**徙天下豪富 + 旧城居民百工于新都**——汉魏故城内的宫殿物料、居民、寺庙陆续迁出。**汉魏洛阳故城自此正式废**——自 605 年起至 755 年凡 150 年。

### 此刻

天宝十四载这一年的汉魏洛阳故城——**已废 150 年（605 隋炀帝徙居 → 755）**；以**废都遗址 + 邙山南麓农田 + 永宁寺塔基孤峰**身份存在。

**所见地表**——**内城城垣**（东汉 + 曹魏 + 北魏三层夯土）**大体犹存**、残高 3-7 米；外郭城已多坍；城中阡陌 + 农舍 + 唐人耕作。**永宁寺塔基**——**夯土方台、底边约 100 米见方、残高约 8 米**——孤立邙山南麓、是汉魏故城最显的物理标识；**唐人称"永宁寺塔基"** 为知名怀古地；塔基周边遗有北魏石刻、瓦当、残砖。**铜驼街遗迹**——汉魏间皇城南向大街、街址仍辨；铜驼自西晋永嘉间被毁、唐时已不存——但"铜驼荆棘"成语在唐人凭吊文学中频出。

**唐河南府的行政归属**——**汉魏故城跨洛阳县东境 + 偃师县西境**：汉魏内城在偃师县西、外郭城东段在偃师县境、西段在洛阳县境；河南府尹治唐洛阳城内、汉魏故城为府境郊野。**755 时无州县治所设于汉魏故城内**——是纯郊野遗址。

**唐人对汉魏洛阳故城的认知**——主要通过：
1. **史书记载**：**《后汉书》**（南朝宋范晔撰）+ **《三国志》**（西晋陈寿撰）+ **《晋书》**（唐房玄龄等贞观二十二年/648 撰）+ **《魏书》**（北齐魏收撰）+ **《洛阳伽蓝记》**（北魏杨衒之 547 撰）——盛唐已成书广流；其中《洛阳伽蓝记》尤为唐人爱读、是汉魏洛阳怀古的核心 textual 来源；
2. **物理凭吊**：盛唐人出唐洛阳城东行十五里、必过汉魏故城——城垣、永宁寺塔基、铜驼街遗址、邙山南麓的北魏陵群（见 [北魏邙山陵群](./北魏邙山陵群.md)）皆历历在目；
3. **玄宗朝东巡**：玄宗朝前期（713-736）多次自长安东巡洛阳、必经长安-洛阳驰道、出陕郡 + 新安函谷新关 + 抵唐洛阳城——**驰道东段最后一程即过汉魏故城北郊**；**开元二十四年（736）十月玄宗最后一次自洛阳还长安**——**自此玄宗朝再未东巡 19 年（至 755）**；汉魏故城遗址在 755 时已 19 年未受帝王车驾经过。

**唐人对汉魏故城的凭吊文学**——盛唐前期已多。**陈子昂《登幽州台歌》《蓟丘览古》**虽不直写洛阳、但同期文学已大量咏汉魏故城。**李白、杜甫、王维 等盛唐诗人** 多有过汉魏故城的旅程——其凭吊汉魏六朝的诗篇 多以汉魏洛阳故城为物理 anchor。

**这一年的汉魏故城** 处于**废都 + 农田 + 凭吊地**的稳定状态——再过半年安禄山起兵；**天宝十四载十二月丁酉（公历 756 年 1 月 18 日）安禄山陷东都洛阳**——叛军入唐洛阳城、汉魏故城地区作为唐洛阳城东郊一并失守。

### 此刻意味

对天宝十四载的洛阳县 + 偃师县人来说——汉魏故城是**自家土地上的旧都遗物**：永宁寺塔基孤立、北魏夯土城垣残存、铜驼街址犹辨——是中原农田中最大的一片废墟。对长安那一代士人来说——汉魏故城是**汉魏六朝的物理纪念**：东汉 + 曹魏 + 西晋 + 北魏四朝旧都凝缩于一城——是**盛唐怀古路线上最浓厚的一站**：与 [灞桥](../transport/长安-洛阳驰道.md) + [邺城](./邺城.md) + 建康（今南京）同列盛唐凭吊的"前朝旧都"系列。

**盛唐人理解的"洛阳"是双层的**——**眼前的唐洛阳城**（隋大业元年/605 炀帝所筑、唐沿用 150 年）是当下的政治 + 经济 + 文化中心；**汉魏故城**（25-534 四朝旧都）是历史的纪念碑、距唐洛阳城仅十五里——**两座城是中国都城史最戏剧的一组"旧城-新城"拼贴**：同一地理空间、同一名号"洛阳"、却是两座独立的城；**605 隋炀帝徙居** 是中国古代都城史最大的一次"主动废都 + 另筑新都"的事件。

唐人尚不知道——再过半年（756 初）汉魏故城地区将随东都洛阳沦陷于叛军；**至德二载（757）十月唐军郭子仪、李光弼、回纥兵复东都洛阳**——汉魏故城再归唐廷；**乾元二年（759）九月史思明再陷洛阳**——汉魏故城再失；**宝应元年（762）十月唐军 + 回纥兵复洛阳**——但**回纥兵纵掠洛阳城内三日**——唐洛阳城遭重创、汉魏故城地区一并波及。**自此中唐汉魏故城**——更显荒残。

再过百余年（唐末），**汉魏故城内仅余永宁寺塔基 + 北魏夯土残垣**；**北宋间洛阳为西京、汉魏故城旁修白马寺**（白马寺自东汉永平十年/67 始建、虽不在汉魏故城内、但相距仅约 10 唐里、是汉魏故城配套的活景）；**金元间汉魏故城地表渐成农田**；**1962 年起中国社会科学院考古研究所洛阳工作队多次发掘**——确认内城、外郭城、永宁寺塔基（1979-1981 发掘）、北魏宫城太极殿基址 等；**1988 年汉魏洛阳故城列国家级重点文物保护单位**。**今游人来偃师翟镇**——所见的是**北魏夯土城垣残段 + 永宁寺塔基（仍立）+ 太极殿基址 + 部分石刻 + 出土北魏陶俑、瓦当等**——**唐人在 755 所见的那座废都**：永宁寺塔基犹存、夯土城垣犹存——千二百多年后地表景观仍可辨；但**汉魏铜驼 + 曹魏太极殿 + 北魏永宁寺九级浮图** 早已不存。

## 子条目

### 东汉迁洛（25-220）

**东汉光武帝建武元年（25）十月**——**刘秀定都洛阳**（东汉建国直接定都洛阳、非由长安迁来；《后汉书·光武帝纪》）；**自此东汉以洛阳为都 195 年**（25-220）——历光武、明、章、和、殇、安、顺、冲、质、桓、灵、献十二帝。**东汉洛阳的盛期**：**明帝永平间（58-75）**修白马寺（在城西十里、汉传佛教第一寺）、扩南北宫；**章帝、和帝间**经学、文学、史学俱盛；**桓帝、灵帝间**外戚 + 宦官交替专权、洛阳成党锢之祸的舞台。**汉灵帝中平六年（189）董卓入洛**——**焚南北宫 + 挟献帝西迁长安**——**东汉洛阳第一次浩劫**。**自此东汉洛阳渐废、献帝于建安元年（196）东归洛阳后再迁许昌**——东汉洛阳实际上自 189 年起即衰、220 年曹丕受禅时洛阳已重修。

**唐人对东汉洛阳的认知**——主要通过《后汉书》（南朝宋范晔撰、盛唐已成书）+ 班固《两都赋》+ 张衡《二京赋》——后两赋是唐前期文学常读、是汉魏洛阳形貌的核心 textual 来源。

### 曹魏定都洛阳（220-265）

**曹魏文帝曹丕黄初元年（220）十一月癸酉**——**受汉禅、定都洛阳**（《三国志·魏书·文帝纪》）。**曹魏修复洛阳**——自董卓焚宫后 31 年、曹丕重修南北宫 + 太极殿。**曹叡（明帝）青龙年间（233-237）大造洛阳宫**——**太极殿 + 阊阖门 + 端门** 等是其核心；**青龙三年（235）治许昌宫 + 洛阳南北宫诸殿**（《三国志·魏书·明帝纪》）。**曹魏以洛阳为都 46 年**（220-265）——历文、明、齐王、高贵乡公、元帝五帝；**司马氏自高平陵之变（249）后渐执魏权、至 265 司马炎受禅**。

**曹魏洛阳的物理遗产**——**太极殿** 是其最显的建筑物：**北魏迁洛后沿用太极殿名号 + 重建**——曹魏 + 北魏两代太极殿叠加于同一基址；**唐人称"汉魏太极殿基址"** 为知名怀古地。

### 西晋继都洛阳（265-311）+ 西晋亡于长安（316）

**西晋武帝司马炎泰始元年（265）十二月**——**受魏禅、仍都洛阳**（《晋书·武帝纪》）。**西晋洛阳的盛期**：**太康年间（280-289）**晋平吴、天下一统、洛阳号"太康之盛"；**陆机、陆云、左思 等"三杨二陆"** 集洛阳、文学盛极一时；**左思《三都赋》成、洛阳纸贵**（《晋书·左思传》）。**西晋洛阳的衰期**：**惠帝间（291-306）八王之乱**——洛阳成内乱中心；**永兴元年（304）匈奴刘渊起兵于离石、立汉国**——为永嘉之乱埋下祸根。

**永嘉之乱**——**永嘉五年（311）六月**——**汉（前赵）刘曜陷洛阳、焚宫庙、虏怀帝**（《晋书·孝怀帝纪》原文）；史称"永嘉之乱"——是**中国历史上最严重的"中原沦陷"事件之一**：西晋宗室南渡江左 + 中原士庶大举南迁、形成东晋"衣冠南渡"。**建兴四年（316）愍帝降于长安、西晋亡**（愍帝在长安受降、非在洛阳；洛阳实际上 311 已陷）。

**自此汉魏洛阳沦为前赵、后赵、前燕、前秦、北魏前期诸胡政权的边境废都凡 183 年**（311-494）——是汉魏洛阳最阴暗的一段。**唐人对永嘉之乱的认知**——是盛唐怀古文学最浓厚的一段：**李白《古风》等** 频用"永嘉" 二字暗指中原沦陷。

### 北魏孝文帝迁洛（494）+ 北魏洛阳盛期

**北魏孝文帝拓跋宏太和十七年（493）冬十月**——**诏迁都**（《魏书·孝文帝纪》原文）；**太和十八年（494）九月车驾发平城、迁洛阳**——抵汉魏旧城；**太和十九年（495）九月新作金墉宫成**——北魏在汉魏故城内扩建宫城。**孝文帝迁洛的政治背景**：（1）北魏自拓跋珪开国（386）至孝文帝（471 即位）已百余年；（2）平城（今山西大同、在阴山以南、北方边境）作为首都不利于统治中原；（3）孝文帝（自身鲜卑、汉化深）以"开万世之基" 推动迁都 + 全面汉化（见 [北魏邙山陵群](./北魏邙山陵群.md) 详）。

**北魏洛阳的扩筑**——**孝文帝 + 宣武帝（483-515 在位）+ 灵太后胡氏（515-528 临朝）三朝大规模营建外郭城**：东西约 10 公里、南北约 7.5 公里、周长约 13 公里——**户口最盛时近 60 万**——是中国 5-6 世纪最大的都城。**北魏洛阳的繁华**：**佛寺 1367 所**（《洛阳伽蓝记》载）——是中国佛教史最盛的城；**永宁寺九级浮图、白马寺、嵩岳寺塔（嵩山）等** 皆这一时期所建或重修。

**北魏分裂**——**永熙三年（534）十月**孝武帝元修**西奔长安**（投宇文泰）、**东魏孝静帝元善见迁都于邺**（《魏书·废出三帝纪》）；**自此汉魏洛阳作为都城的历史正式终结**——汉魏迁洛 - 北魏迁洛 510 年（25-534）告终。

### 永宁寺九级浮图（516 建 / 534 焚）

**永宁寺**——**北魏熙平元年（516）灵太后胡氏立**（《洛阳伽蓝记·城内·永宁寺》原文）；位置在**汉魏洛阳故城内中央偏北**、北魏宫城南、铜驼街旁。**永宁寺九级木浮图**——**高约 130 米**（《洛阳伽蓝记》载"去京师百里、已遥见之"）——是中国古代最高的木塔之一、亦是 6 世纪初最壮丽的佛教建筑。**塔身九级、木构、外覆金箔、夜间塔上铃铎闻数十里**（《洛阳伽蓝记》载）。

**永宁寺浮图的焚毁**——**北魏永熙三年（534）二月**——**九级木浮图被雷击焚毁**（《洛阳伽蓝记》原文："永熙三年二月、浮图为天火所烧"）——是北魏洛阳最有名的物理事件、亦预示北魏分裂（同年十月孝武帝西奔、东魏迁邺）。**自此永宁寺仅余塔基 + 部分石刻**。

**755 时唐人所见**——**永宁寺塔基**：**夯土方台、底边约 100 米见方、残高约 8 米**——孤立邙山南麓、是汉魏故城最显的物理标识；**唐人称"永宁寺塔基"** 为知名怀古地。**1979-1981 中国社会科学院考古研究所洛阳工作队发掘永宁寺塔基**——确认塔基地宫 + 出土北魏石刻、彩塑残片、瓦当等——是北魏佛教物质文化的核心实物；今藏洛阳博物馆 + 中国社会科学院考古研究所。

### 铜驼街 + "铜驼荆棘"

**铜驼街**——**汉魏间皇城南向大街**、自宫城阊阖门南至宣阳门、街旁列铜驼二（西晋陆机《洛阳记》载"汉铸铜驼二、列在宫南四会道头"）。**"铜驼荆棘"成语**——出自《晋书·索靖传》：**索靖（西晋人、卒于永嘉之乱前）有先知之明、知天下将乱、指洛阳宫门铜驼叹曰"会见汝在荆棘中耳"**——**预言西晋将亡 + 洛阳沦为废墟**。

**永嘉之乱（311）后铜驼遭毁**——具体何时不存史源略异；**唐时铜驼已不存** + 铜驼街址仍辨。**"铜驼荆棘"成语在唐人凭吊文学中频出**——是盛唐怀古汉魏六朝最浓厚的意象之一。**李白、杜甫、王维、孟浩然 等盛唐诗人**——皆用过"铜驼"二字。

### 隋大业元年（605）徙居 + 唐洛阳新都

**隋炀帝大业元年（605）三月丁未**——**诏营东京**（《隋书·炀帝纪》原文）；**于汉魏故城西十八里另筑新都**（今洛阳老城所在）。**炀帝徙居**：**徙天下豪富 + 旧城居民百工于新都**（《隋书·炀帝纪》）——汉魏故城内的宫殿物料、居民、寺庙陆续迁出。**自 605 年起至 755 年凡 150 年——汉魏故城正式废**。

**唐洛阳新都的物理位置**——在汉魏故城**西约十八唐里**（今洛阳老城与汉魏故城遗址直线距离约 15 公里）——**两座城同在洛阳盆地、同名"洛阳"、却是两座独立的城**。**唐洛阳城承隋大业新都之制**——周长约 27 公里、外郭城分三重（外郭 + 皇城 + 宫城）、是中国 7-8 世纪最大的城（见 [洛阳](../cities/洛阳.md)）。

**这是中国古代都城史最戏剧的一组"旧城-新城"拼贴**——同一地理空间、同一名号、却是两座独立的城；605 隋炀帝徙居 是中国古代都城史最大的一次"主动废都 + 另筑新都"的事件。

### 白马寺（汉魏故城西约十里）

**白马寺**——**东汉明帝永平十年（67）始建**、是**汉传佛教第一寺**；位置在**汉魏洛阳故城西约十里**（今洛阳市白马寺镇、即唐洛阳城东郊与汉魏故城西郊之间）——**不在汉魏故城内**、但相距仅约 10 唐里、是汉魏故城配套的活景。**白马寺自东汉至唐 700 年间多次毁修**——755 时仍盛、是唐人出洛阳东行往汉魏故城路上的必经一站；**白马寺与汉魏故城是盛唐人凭吊路线上的"活景 + 废墟"二联**。

**注**：白马寺**不在**汉魏故城内、亦不在唐洛阳城内、而在**两城之间的郊原**——是唐人出洛阳东行 + 入汉魏故城的物理过渡点。

## 历史长河（755 → 今）

**安史中（755-763）**　**天宝十四载十二月丁酉（756 年 1 月 18 日）安禄山陷东都洛阳**——汉魏故城地区作为唐洛阳城东郊一并失守；**至德二载（757）十月唐军郭子仪、李光弼、回纥兵复东都洛阳**——汉魏故城再归唐廷；**乾元二年（759）九月史思明再陷洛阳**——汉魏故城再失；**宝应元年（762）十月唐军 + 回纥兵复洛阳**——但**回纥兵纵掠洛阳城内三日**——唐洛阳城遭重创、汉魏故城地区一并波及。**自此中唐汉魏故城**——更显荒残。

**中晚唐**　**贞元 / 元和间唐廷无力修缮汉魏故城**——故城内仅余永宁寺塔基 + 北魏夯土残垣；中晚唐诗人**白居易、刘禹锡、温庭筠 等多有汉魏故城凭吊诗**——白居易尝任河南尹（晚年长居洛阳）、亦多次至汉魏故城；**白居易《永宁寺》《北邙行》等**（中晚唐作；本 doc 历史长河段引用）——是汉魏故城凭吊文学的核心。

**唐末 + 五代**　**广明元年（880）黄巢入长安**——洛阳一带在乱中受波及；**唐末五代后梁、后唐皆都洛阳**——但所都为隋唐洛阳新城、汉魏故城地区仍荒。**五代后晋、后汉、后周间汉魏故城内永宁寺塔基仍立**——但已残破。

**两宋**　**北宋初定都开封**——洛阳退为西京、汉魏故城地区作为西京郊野；**北宋金石学家欧阳修、赵明诚等** 对汉魏故城碑刻有研究——**北宋间汉魏故城渐被农田覆盖**；永宁寺塔基仍立、但木塔早不存。**金元间汉魏故城地表渐成农田**——王公贵族墓 + 北魏寺院基址被盗掘多次。

**金元明清**　金元间汉魏故城地表完全成农田、仅永宁寺塔基 + 部分夯土城垣可辨；**明清间汉魏故城渐成偃师县境内的一片农村**——仅地方志载"汉魏故城遗址"；**清乾隆间地方志载汉魏故城"夯土城垣残存数里、永宁寺塔基孤立"**——可见唐 755 至清乾隆千年间地表景观大体延续。

**民国至今**　**1955 年起中国社会科学院考古研究所洛阳工作队多次勘察汉魏故城**；**1962-1980s 多次发掘**——确认内城、外郭城、北魏宫城太极殿基址等；**1979-1981 发掘永宁寺塔基**——出土北魏石刻、彩塑残片、瓦当 等；**1988 年汉魏洛阳故城列国家级重点文物保护单位**；**2007 年起申报"丝绸之路：长安-天山廊道的路网"世界文化遗产**——**2014 年汉魏洛阳故城（北魏宫城太极殿基址 + 永宁寺塔基）作为丝绸之路东端节点列联合国教科文组织世界文化遗产**。

**今游人来偃师翟镇**——所见的是**北魏夯土城垣残段 + 永宁寺塔基（仍立）+ 太极殿基址 + 部分石刻 + 出土北魏陶俑、瓦当等**——**唐人在 755 所见的那座废都**：永宁寺塔基犹存、夯土城垣犹存——千二百多年后地表景观仍可辨；但**汉魏铜驼 + 曹魏太极殿 + 北魏永宁寺九级浮图** 早已不存。**汉魏洛阳故城是中国都城遗址中保存规模最大、跨代最多（东汉 + 曹魏 + 西晋 + 北魏四朝）的一处**——是 5 世纪以前中国都城史的物理总结。
