# 淝水

## 一句话

**淮南寿州东郊的一条小水**——东晋孝武帝太元八年（383）十一月**谢玄以八万北府兵败前秦苻坚于此**；前秦先军误退、自溃于淝水西岸——"**风声鹤唳、草木皆兵**"出于此战；自此前秦分裂、北方再入后燕、后秦、北凉、夏等割据, 至 439 北魏统一北方、东晋偏安百年——**755 时距此战 372 年、淝水仍东流入淮、八公山仍立寿春北郊**。

## 形貌

淝水在唐淮南道**寿州（天宝元年/742 改寿春郡）**治寿春县境内——具体在**寿春县城东郊**、距县治约五至二十里的浅丘原野间。**地势**：淮河南岸、江淮丘陵北缘——寿春据淮河南岸要冲、东淝河自寿春东南山地北流、绕城东北、汇入淮河；**八公山**横亘寿春北郊、是淮河南岸最显的一处低山屏障；西望大别山余脉、东接江淮平原。

**淝水**——按《水经注》是**肥水三源**：东、中、西三源分流；**东淝河**自寿春东南合肥（今安徽合肥）一带北流、绕寿春城东北、入淮水；**西淝河**自寿春西南另出、独立入淮——**383 年战役在东淝河**、即寿春东郊那一段。**唐 755 时**东淝河仍是寿春东郊的主要水道——河宽数十丈、深浅不一、夏汛漫滩、冬涸见底——是寿春城东最稳的物理 anchor。

**八公山**——**寿春正北约三里**的低山群——**主峰约海拔 200 米左右**；以"八公"得名（西汉淮南王刘安招方士八公修道之地传说所在）；山上多林木、山势平缓但起伏多变——**是 383 年苻坚登寿春城北望淝水西岸、误以"草木皆兵"的物理 anchor**；八公山林木在风中起伏、与淝水西岸前秦军营遥相对望。

**寿春县城**——**淮河南岸的古城**；春秋楚国晚期曾迁都于此（楚考烈王二十二年/前 241 迁寿春、号"郢"）；秦汉九江郡 + 淮南国治；**三国魏吴交兵的前线**（合肥与寿春相距 200 余里、是淮南军事重镇）；**383 时是东晋"淮南前线"的最北一镇**、**755 时是唐寿州治、淮南道腹地的一座中等州城**。

**755 时所见的淝水古战场**——是寿春县城东郊的**一片平阔河滩 + 低岗 + 林莽 + 农田**：东淝河仍东北流入淮、八公山仍立城北、淝水西岸是 383 年前秦军营所在（已成农田）、淝水东岸是当年东晋北府兵渡水追击的地段；**唐人来寿州凭吊淝水**——多自寿春县东门出城、行数里至淝水岸边、登八公山远望、想象 372 年前那场"风声鹤唳"的夜溃。

物候：江淮分界线的暖温带——四时分明、夏有淮汛、春多东南风、冬冷不极；**383 年十一月正是寒风萧瑟、八公山林木叶落、淝水水浅可涉的时令**——这是唐人凭吊时常想到的物候细节。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

淝水在唐人的回望里——是**中国"以少胜多"最经典的一战 + 南北朝分裂格局的物理定格 + "风声鹤唳""草木皆兵""投鞭断流"三个成语的诞生地**。**383 年这一战**——构成唐人对淝水三层认知：

**第一层：前秦统一北方的尽头**。**前秦苻坚自承平五年（357）继位**——以王猛为相、十数年间次第灭仇池（370 前后）、灭前燕（370）、灭前凉（376）、灭代国（376）——**至建元十五年（379）攻陷襄阳、克梁益**——**北方再度归一**；苻坚自视为**曹操之后少数接近 / 完成北方整合的雄主之一**。**建元十八年（382）十月**——苻坚于长安召群臣议伐晋；**朝臣多谏不可**（王猛遗言"晋虽僻陋吴越、乃正朔相承——不可图也"；苻融、朱彤、石越等皆力谏）；**苻坚不听、独决南征**——**这是淝水之战的政治起点**：一位刚统一北方的胡族雄主、视江左偏安为不臣、欲一战平天下。

**第二层：东晋"北府兵 + 谢氏"的决战**。**东晋孝武帝太元元年（376）**——**谢安以宰相居中、谢玄以前将军居外**；**太元二年（377）谢玄选练北府兵于京口**（今江苏镇江）——以**刘牢之 + 何谦 + 诸葛侃 + 高衡 + 刘轨**等北徐州流民为兵源、号"**北府兵**"——这是东晋立国 60 年后第一次建成真正能战的军队。**太元四年至七年间（379-382）**北府兵在淮南屡破前秦小股南下——为淝水决战做了充分训练。**淝水时东晋兵力**：**谢玄、谢琰、桓伊、刘牢之等率北府兵共约八万**（《晋书》原文八万；分两路：谢玄主力七万从京口溯淮西上、桓伊一万从豫州东出）——**对前秦先锋"苻融三十万"+ 苻坚后军八十余万号称的"百万"**——**以一比十的悬殊**对峙于淝水。

**注**：**前秦"百万"是号称**——实际**前线投入淝水的兵力是苻融率领的三十万先锋**（《晋书》《资治通鉴》原文）；**苻坚自率"百万"后军在长安至项城（今河南沈丘）的途中**——大多数后军尚未到淝水。**东晋"八万北府兵"是实数**——这是淝水以少胜多的真实兵力对比、不是"八万对百万"而是"八万对三十万"（仍是一比四的悬殊）。

**第三层：决战夜的戏剧**（太元八年/383 十一月）：

- **十月**——苻坚自项城进至寿春北、登寿春城北望——**见八公山林木、风中起伏、误以为东晋伏兵**——"**草木皆兵**"出此（《晋书·苻坚载记下》原文）；
- **苻坚与苻融登城观察、问朱序（晋降将、暗通晋廷）**敌情——**朱序劝苻坚"今诸军未集、宜先击晋"**——苻坚不从、令前军于淝水西岸列阵；
- **谢玄遣使请前秦稍退、以决战**——苻坚许之、**令前军后撤**；**苻融驰马整军**——**前秦后军误以为前军已败、争相奔溃**；
- **谢玄渡淝水追击**——**朱序在阵中大呼"秦兵败矣"**——前秦军遂大溃；**苻融策马阻军、马倒被杀**；**苻坚中流矢**、单骑逃归——**昼夜不息、闻风声鹤唳皆以为追兵至**（"**风声鹤唳**"出此）；
- **十一月癸卯朔**——东晋诸军班师、淝水之战毕；**前秦军死伤十之七八**、苻坚仅以身免、逃归长安。

**自淝水至盛唐的余波**——三个时段：

- **前秦分裂（383-394）**——苻坚回长安后、**慕容垂叛归河北立后燕（384）**、**姚苌叛立后秦（384）**、**苻坚 385 年为姚苌所杀**——前秦立国 35 年（351-394）至此终结；**北方再入后燕、后秦、北凉、夏等割据, 至 439 北魏统一北方**——是淝水之战最直接的政治后果；
- **东晋延续（383-420）**——淝水战后东晋偏安江左 37 年、至**晋恭帝元熙二年（420）**为刘裕篡晋立宋止；**衣冠南渡的格局自淝水巩固**——是南北朝分裂格局的物理定格；
- **隋唐统一（589 + 618）**——**淝水以后 206 年（589 隋灭陈）**北南方才重新统一；**唐人对淝水的认知**——是"分裂格局被延长 / 定格"——**苻坚若胜、则南北早在 383 年即可一统、不必等隋；苻坚一败、南北分裂百余年**——是中国历史最大的一次"if"。

**注**：**"投鞭断流"** 出《晋书·苻坚载记下》——苻坚南征前、群臣谏"晋有长江之固"、苻坚答："**以吾之众、投鞭于江、足断其流**"——这是淝水前苻坚自负的写照；**"东山再起"** 出《晋书·谢安传》——谢安 40 岁前隐居会稽东山、升平四年/360 出仕（应桓温辟）、太元二年/377 以中书令秉政期辅佐淝水之战——**与淝水有直接关联**（谢安秉政期即遇淝水）。**这三个成语 + "风声鹤唳""草木皆兵"共五个成语**——皆与淝水之战相关、构成中国成语史上最密集的一战。

### 此刻

天宝十四载这一年的淝水——**距 383 年战役 372 年**；属唐淮南道**寿州（寿春郡）寿春县**境内、东淝河自寿春东郊北流入淮。

**淝水古战场**：**淝水西岸**（今八公山西南、寿春东北约十里）——383 年前秦军营所在、唐时已成农田 + 林莽；**淝水东岸**——东晋北府兵渡水追击的地段、唐时是寿春县乡野；**淝水水道**仍东北流入淮——**水浅可涉**（与 383 年同；东淝河自来水量不大、十一月枯水时确可徒涉）；**淝水两岸**多苇草 + 沼泽 + 农田——是江淮平原典型的湿地景观。**八公山**：寿春正北约三里、林木仍葱郁；或有寺观传说, 755 不坐实；**唐人登八公山**——遥望淝水西岸 + 寿春古城——是淝水凭吊的核心动作。

**寿春县城**：**唐寿州治**、**淮南道腹地的中等州城**；**州城周约九里**（依《元和郡县图志》）、有四门、内有寿春郡太守廨 / 郡府 + 寿州军府；**寿春县令掌州城治理**；**唐玄宗朝寿州为淮南道北部要冲**——扼淮南北上河南的咽喉。**寿春人对淝水的日常认知**：东郊那条小水、八公山下、田间地头——是"晋时谢公破苻坚处"；**地方民间口传**有"谢公祠""谢公台"等遗迹（具体位置 755 时史源不详、本 doc 不强行 anchor）；**唐廷未在淝水设特殊岁修保护**——任战场遗迹在田野中风化。

**唐人凭吊淝水的密度**——**盛唐前期较低**（多在边塞 + 长安洛阳两京 + 蜀道 + 江南等地写诗）；**中晚唐及以后渐高**——**李白《登广武古战场怀古》《永王东巡歌》《登金陵凤凰台》等**虽未直咏淝水、但"晋代衣冠成古丘"的怀古笔法是这一脉的源头；**杜甫、刘禹锡、杜牧、李商隐**等中晚唐人皆有涉及淝水的诗咏。

**安史前夕**（即 755 这一刻）——再过半月（农历十一月初九）安禄山起兵于范阳；**自范阳南下取河南、淮南是江淮防线的北部**——**寿春 / 淝水 / 八公山一带在安史中是江淮防线最北一段**；**江淮之战（756-757）即将在淝水西南数百里的睢阳 + 颍川一带爆发**——是淝水之后江淮再度成为战场的开端。

### 此刻意味

对天宝十四载的寿春人来说——淝水就是**自家县城东郊那条小水**：渡水、捕鱼、汲水、踏雪——是地理日常；八公山是城北那座低山——可登可望。**372 年前那场夜战**——只在地方口传与读书人记忆里、不再有政治意义。对长安那一代士人来说——淝水是**南北朝分裂格局的物理定格 + 中国"以少胜多"的最高范式**：八万对百万（号称）、谢公屐 + 北府兵 + 草木皆兵 + 风声鹤唳——这一长段戏剧皆有此地为物理 anchor。

**盛唐 755 时对淝水的怀古**——**密度不及建康 / 邺城 / 赤壁**：盛唐前期诗人对淝水的具体咏怀史源不密；**怀古文学的密集化在中晚唐及以后**——**李商隐《筹笔驿》《北齐二首》等**、**李白晚期怀古**、**杜甫《咏怀古迹五首》等**——皆涉及南北朝分裂主题、淝水是其中一条暗线。

唐人尚不知道——再过半月安禄山起兵、江淮防线即将再度成为战场；**江淮十年战乱（755-763）以张巡守睢阳为代表**——是淝水之后江淮再度承担"分裂前线"角色的开端。再过 200 余年（北宋 + 南宋之交）——**靖康之难（1127）后**——南宋以淮河为界、淝水再度成为南北对峙的前线；**南宋绍兴十一年（1141）岳飞收复颍昌**——是淝水西北方向的另一段历史；**辛弃疾《永遇乐·京口北固亭怀古》"想当年金戈铁马、气吞万里如虎"**——南宋怀古词以京口（与寿春同在淮南）为 anchor、与淝水的怀古情绪一脉相承。

再过千余年（明清 + 现代）——**淝水仍是寿县（民国废府厅州县后、寿州改寿县）东郊那条东淝河**；八公山仍立寿春北郊；**1949 年后**寿县多次考古勘察淝水古战场 + 八公山 + 寿春古城——确认八公山主峰位置 + 淝水古战场范围 + 寿春战国楚都"郢"遗址；**今游人来寿县**——所见的是**淝水古战场（八公山南麓 + 东淝河沿岸）+ 八公山国家森林公园 + 寿县古城（明代砖城）+ 寿春楚墓**——**多为后世重修或现代复建**；**唐人在 755 所见的那片淝水东郊原野与八公山林莽**——大半已为后代叠加的城乡景观覆盖；**但八公山仍立寿春北郊、淝水仍东北流入淮**——**与 755 唐人所见的物理景观在大轮廓上是连续的**：千四百多年、南北兴亡、未易。

## 子条目

### 淝水西岸（前秦军营 + 后军误溃处）

**淝水西岸**——383 年前秦先锋苻融三十万军列阵处；**自淝水西岸西望**为八公山南麓的低岗 + 林莽；**自淝水西岸东望**为淝水东岸的东晋北府兵阵地——隔水相望、约百余步。**十一月某日**——苻坚许谢玄请前军后撤、苻融驰马整军——**前秦后军误以为前军败、争相奔溃**——**这是淝水之战的关键转折点**：不是被打败、而是自溃。

**自溃的原因**——史源有多说：
- **朱序在阵中大呼"秦兵败矣"**（《晋书·苻坚载记下》《资治通鉴》）——是直接导火索；朱序为东晋襄阳守将、襄阳陷后降前秦、淝水时随苻融在阵中、暗通晋廷；
- **前秦军成分复杂**（氐族 + 鲜卑 + 羌族 + 汉族 + 匈奴等多胡杂集）——临时整合、缺乏统一指挥；
- **前线兵力悬殊不如号称**——苻融三十万先锋已在淝水、苻坚自率的"百万"后军大多在项城至寿春途中、未到淝水。

**淝水西岸的自溃**——是 383 年战役最戏剧的一笔；**755 时唐人来淝水西岸凭吊**——所见是一片农田与低岗、想象 372 年前那场"自踏死者无数"的夜溃。

### 淝水东岸（东晋北府兵渡水追击处）

**淝水东岸**——383 年东晋谢玄率北府兵列阵处；**自淝水东岸西望**——淝水西岸前秦军营隔水可见；**谢玄渡水追击**——**北府兵渡淝水西击前秦**、追至青冈（淝水西约二十里）——前秦死伤十之七八、苻坚单骑遁走。

**北府兵**——**谢玄于太元二年（377）选练于京口的精锐**——以北徐州（今淮南北部至山东南部）流民为兵源、刘牢之 + 何谦 + 诸葛侃 + 高衡 + 刘轨等为将；**淝水之战是北府兵立国 6 年后的第一次大战**——以八万破前秦三十万先锋、奠定了北府兵在东晋至南朝宋之间的核心地位。**北府兵在淝水之后**——续为东晋至南朝宋的主力军、**刘裕（北府兵将领）以北府兵为基础篡晋立宋（420）**——是淝水之后北府兵的长期政治后果。

**755 时唐人来淝水东岸凭吊**——所见是寿春县乡野、淝水东北流入淮——想象 372 年前北府兵渡水追击的那一夜。

### 八公山（"草木皆兵"的物理 anchor）

**八公山**——**寿春正北约三里**的低山群、海拔约 200 米；**得名于西汉淮南王刘安招方士八公修道传说**——《淮南子》成书地之一。**383 年十月**——苻坚自项城进至寿春北、登寿春城北望——**见八公山林木、风中起伏、误以为东晋伏兵**——"**草木皆兵**"出此（《晋书·苻坚载记下》原文："（坚）见八公山上草木、皆以为晋兵"）。**这是苻坚自寿春北望淝水西岸的物理视角**——以低山林木幻作伏兵、是疲将面对未知敌情时的典型心理。

**八公山在 755 时**——仍立寿春北郊、林木仍葱郁；**或有寺观传说, 755 不坐实**（奉淮南王刘安八公修道传说；史源不详, 本 doc 不在 755 时态坐实寺观存在）；**唐人来八公山**——多登山遥望淝水西岸 + 寿春古城——是淝水凭吊的核心动作。

**注**：八公山与"风声鹤唳"是两个不同 anchor——"草木皆兵"在八公山（苻坚登寿春城北望所见）；"风声鹤唳"在苻坚淝水溃败后逃归途中、不在八公山具体某地——是溃军普遍心理状态。

### 寿春古城（楚 + 秦汉 + 三国 + 东晋 + 唐多代相承的州治）

**寿春**——**江淮间最重要的古城之一**：
- **楚考烈王二十二年（前 241）**——楚自陈郢迁都寿春、号"郢"——是楚国最末一段都城（前 241-前 223 楚亡）；
- **秦汉**——置寿春县、属九江郡；**淮南国治寿春**（西汉刘安都此、《淮南子》成书地）；
- **三国魏吴**——寿春是魏吴交兵的前线、与合肥并列为淮南二大军镇；**司马懿、毋丘俭、诸葛诞**等魏将皆驻寿春；**毋丘俭（255）+ 诸葛诞（257-258）二次叛魏**皆以寿春为据点；
- **西晋永嘉乱后**——**东晋以寿春为淮南前线**；**383 年淝水之战**前后寿春是东晋北府兵的前进基地；
- **南朝宋齐梁陈**——寿春仍为淮南前线；
- **隋开皇九年（589）平陈后**——置寿州（治寿春）；
- **唐贞观间**——寿州治寿春；**天宝元年（742）改寿州为寿春郡**（治寿春县）；**755 时正名寿春郡（即寿州），下辖寿春县**；
- **唐肃宗乾元元年（758）**——州郡再改、寿春郡复为寿州。

**755 此刻正名**——**寿州治寿春县（即寿春郡）**；**唐人在 755 来此**——多说"过寿州""至寿春"——**不说"寿县"**（寿县是民国改名）。

**注**：**寿县是 1912 民国废府厅州县后才有的名**——**755 时正名是寿春县**；本 doc 严守 755 时态、不在 755 上下文用"寿县"。

### "风声鹤唳""草木皆兵""投鞭断流"三典（淝水成语史）

**淝水之战**——是中国成语史上最密集的一战，三个核心成语 + 两个延伸成语皆出此战：

- **"草木皆兵"**——出《晋书·苻坚载记下》："（坚）见八公山上草木、皆以为晋兵"；**指苻坚登寿春城北望八公山林木误以为伏兵**；
- **"风声鹤唳"**——出《晋书·苻坚载记下》：苻坚淝水败后逃归途中、"闻风声鹤唳、皆以为追兵"——**指溃军心理状态**；
- **"投鞭断流"**——出《晋书·苻坚载记下》：南征前苻坚答群臣"以吾之众、投鞭于江、足断其流"——**指南征前苻坚自负**；
- **"东山再起"**——出《晋书·谢安传》：谢安年 40 前隐居会稽东山、升平四年/360 出仕（应桓温辟）、太元二年/377 以中书令秉政期辅佐淝水之战——**与淝水直接相关**（谢安秉政期即遇淝水）；
- **"屐齿之折"**——出《晋书·谢安传》：淝水胜报至建康、谢安方与人下棋——读毕信、客问、答曰"小儿辈遂已破贼"——**罢局**、还内、过门限、**屐齿之折而不觉**——**是谢安"喜怒不形于色"的最高戏剧化**。

**这五个成语**——构成淝水之战在中国语言文化中的物理 + 心理 + 政治三层 anchor；**唐人读《晋书》（贞观间房玄龄等修）皆习此五典**——是 755 时盛唐文人的常识。

### 谢安谢玄（陈郡谢氏的家族决战）

**陈郡谢氏**——**东晋至南朝最显的门阀之一**、与琅琊王氏并称"**王谢**"；淝水时谢氏家族在前后线的安排：

- **谢安（320-385）**——东晋宰相、字安石；年 40 前隐居会稽东山、升平四年/360（年 40 余）出仕（应桓温辟）、太元二年/377 以中书令秉政；**淝水时居建康（东晋都城）总揽全局**——史载淝水胜报至建康、谢安方与人下棋（《晋书·谢安传》"屐齿之折"典）；
- **谢玄（343-388）**——谢安之兄谢奕之子（即**谢安侄子**、不是谢安之子）、字幼度；**淝水时为前将军、北府兵主帅**——**主持淝水前线**；
- **谢琰（？-400）**——谢安次子、字瑗度；**淝水时随谢玄出战、有功**；
- **谢石（327-388）**——谢安之弟、字石奴；**淝水时为征虏将军、与谢玄、谢琰并力**。

**陈郡谢氏在淝水之战的安排**——**谢安居中、谢玄居外、谢琰 + 谢石居前线**——**一门四公一战定天下**——是中国门阀家族最辉煌的一战。**谢氏家族在淝水之后**——续为东晋至南朝最显的门阀之一；**谢灵运（385-433）**为谢玄之孙、是南朝山水诗鼻祖；**谢朓（464-499）**为南齐"竟陵八友"之一、是六朝最重要的诗人之一——**谢氏家族在文学史的长尾**至中晚唐仍是诗人凭吊的对象。

**注**：**谢安与谢玄是叔侄、不是父子**——这是常被误说的史源；**谢玄是谢安兄谢奕之子**（《晋书·谢玄传》原文）。

### 苻坚的悲剧（统一北方者败于淝水后被弑）

**苻坚（338-385）**——前秦天王（前秦王统称"天王"不称"帝"）；**继位时年 19**（357）；**以王猛为相、十数年间统一北方**（357-379）；**至建元十五年（379）攻陷襄阳、克梁益**——北方再度归一。**淝水之战时苻坚 45 岁**——**这是他一生最大的政治赌博**：以统一北方者的姿态欲一战平天下。

**淝水败后**：
- **太元八年/建元十九年（383）十一月**——苻坚单骑逃归长安；
- **太元九年/建元二十年（384）**——**慕容垂叛归河北立后燕** + **姚苌叛立后秦**（**后秦姚兴 394 灭前秦末帝苻崇**）+ **慕容冲围长安**——前秦自此分裂；
- **太元十年/建元二十一年（385）**——**苻坚为姚苌所杀、年 48**；**前秦立国 35 年（351-394）至此名存实亡、394 苻登被姚兴所擒、前秦终亡**。

**苻坚的悲剧**——是**十六国时期一次最接近南北统一的尝试**：若苻坚胜淝水、则南北早在 383 年即可一统、不必等隋；苻坚一败——**北方再入后燕、后秦、北凉、夏等割据, 至 439 北魏统一北方**、南北分裂格局延续至 589 隋灭陈才终结。**唐人读《晋书·苻坚载记》**——多对苻坚有同情之笔：**苻坚虽夷狄出身、但崇汉文化、用王猛 + 修学校 + 治内政**——是十六国诸主中最近"汉式雄主"的一位；**淝水之败**——既是政治赌博的失败、也是民族整合尚未成熟的代价。

## 历史长河（755 → 今）

**中晚唐**　**安史十年（755-763）**——江淮成为唐廷的财赋后方；**张巡守睢阳（757）**等是淝水西北方向的另一场以少胜多——是淝水之后江淮承担"分裂前线"角色的延续。**中晚唐淝水凭吊诗**——**李商隐《北齐二首》《筹笔驿》**等涉及南北朝分裂主题；**杜牧《题乌江亭》《赤壁》等**——续中晚唐怀古传统、虽未直咏淝水、但"东风不与周郎便"的反讽笔法与淝水"以少胜多"的母题相通；**晚唐胡曾《咏史诗·淝水》**"苻坚百万岂能成、一阵风声鹤唳惊——堪笑当时谢安石、不开关塞御边兵"——**是晚唐对淝水最直接的咏怀**。

**两宋**　**北宋**——寿春仍是淮南西路的州城；**北宋元丰年间（1078-1085）**——王安石、苏轼等过寿春多有凭吊诗；**靖康之难（1127）**后——**南宋以淮河为界、寿春再度成为南北对峙的前线**；**南宋绍兴十一年（1141）岳飞收复颍昌**——是淝水西北方向的另一段抗金；**辛弃疾《永遇乐·京口北固亭怀古》"想当年金戈铁马、气吞万里如虎"**——南宋怀古词以京口为 anchor、与淝水的怀古情绪一脉相承；**陆游《书愤》"楼船夜雪瓜洲渡、铁马秋风大散关"**——续南宋抗金前线的怀古传统。**南宋开禧二年（1206）**——韩侂胄北伐、寿春再度成为前线；**金哀宗天兴元年/南宋绍定五年（1232）**——蒙古攻金、寿春一带亦受波及。

**元明清**　**元代寿春**——降为安丰路寿春县；**明洪武四年（1371）**——置寿州（属凤阳府）；**明清间寿州古城重修**——今寿县所见的明代砖城即此时所筑；**清乾隆年间**——寿州 + 八公山 + 淝水古战场是清代"江淮怀古"的核心；**清袁枚 + 赵翼 + 王士禛**等过寿州多有诗咏。**注**：**1912 年中华民国废府厅州县**——寿州改寿县——**寿县是民国名、不能在 755 上下文用**。

**民国至今**　**民国间寿县**——是江淮间的中等县城；**1949 年后多次考古**——**安徽省文物考古研究所 + 寿县文物管理所 1980s-2010s** 系统勘察**淝水古战场 + 八公山 + 寿春古城 + 寿春楚墓**——**确认八公山主峰位置 + 淝水古战场范围 + 寿春战国楚都"郢"遗址**；**1986 年寿县古城列国家级重点文物保护单位**；**2001 年八公山国家森林公园开放**；**2014 年寿县历史文化名城**列国家级。**今游人来寿县**——所见的是**淝水古战场（八公山南麓 + 东淝河沿岸）+ 八公山国家森林公园 + 寿县古城（明代砖城）+ 寿春楚墓 + 安丰塘（春秋楚国水利）**——**多为后世重修或现代复建**；**唐人在 755 所见的那片淝水东郊原野与八公山林莽**——大半已为后代叠加的城乡景观覆盖；**但八公山仍立寿春北郊、东淝河仍东北流入淮**——**与 755 唐人所见的物理景观在大轮廓上是连续的**：千四百多年、南北兴亡、淝水以一战定中国南北分裂百年——这片小水的分量、至今未减。
