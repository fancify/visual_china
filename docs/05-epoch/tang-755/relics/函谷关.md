# 函谷关

## 一句话

中国关塞史最复杂的"三关一名"——**战国秦古函谷关**（虢州灵宝县境）+**汉武帝元封间东徙的函谷新关**（河南府新安县）+**曹魏黄初间复立的魏函谷关**（古关南）；**唐 755 时三关皆为遗址、不复要塞**，要塞职能由 70 里西的潼关接替；老子西出 + 尹喜留经 + 玄宗开元二十九年/741 立玄元皇帝庙——是盛唐最重的怀古 anchor。

## 形貌

函谷关是**一个名、三处址**——这是它在中国关塞史中独有的复杂。

**古函谷关（秦关 / 旧函谷关）**——战国秦置，原址在**今河南省灵宝市东北、稠桑原（函谷）一带**（今灵宝市函谷关镇王垛村附近）。**地势**：**北临黄河、南依秦岭余脉（崤山）**——黄河自陕北南流至此转向东、与崤山北麓夹成一道狭谷；**关道**就走在这条狭谷中——**两壁高十余丈、谷底窄不过容车，深险如函**——"函谷"之名即由此来。**这是中国地理上最早的"地形—关塞"耦合典型**——关因谷而立、谷因关而名。**唐 755 时古函谷关属河南道虢州弘农郡灵宝县**（天宝元年/742 改虢州为弘农郡；治弘农县——灵宝是属县）。

**函谷新关（汉关 / 新安关）**——**西汉武帝元封间**（约前 110-前 109）东徙至新安。原因记于《史记·杨仆传》：**楼船将军杨仆**（汉武帝时名将、平南越功臣）因关中—中原界以函谷关为分，**其家在新安、属关外**——杨仆耻为"关外人"、上书请徙关；**武帝从之，移关至新安东**。新关址**在今河南省洛阳市新安县东、青龙山一带**——距古函谷关东北约 300 里。**唐 755 时函谷新关故址属河南府新安县**（东京洛阳所辖；新安县直属河南府、不属虢州）——这是两关在唐代政区上的根本分别。

**魏函谷关**——**三国魏黄初年间**（220-226）曹魏又在古函谷关稍南立关，是为"魏函谷关"。**意图**：复古函谷关的天险传统、防御西方蜀汉；位置约在古关南数里、扼新的崤函山道。唐 755 时魏函谷关亦废、与古关并为遗址。

**三关的位置关系**——古关在西、魏关在古关南数里、新关在东北 300 里；**三关皆守崤函古道**（关中—中原大动脉），但具体扼点不同：**古关扼稠桑原狭谷、魏关扼其南山道、新关扼新安东青龙山口**。**唐 755 时唐人若从洛阳西行至长安**——走的是**两京驰道**（北线）+ 经新安 + 渑池 + 陕州 + 灵宝 + 华阴 + 潼关 + 至长安——驰道**绕过函谷古道**（古道险窄，唐时已不为主驿）；函谷三关皆在驰道附近、但驰道主干已不经三关。

**函谷古道**——古函谷关之所险，全在那道"深险如函"的稠桑原狭谷；自战国至秦汉、是关中东出的唯一通道；隋唐两京驰道修后、古道渐为辅道。**唐 755 时古函谷关道仍存**——是地方驿支线、商旅偶过、不复是国家级要塞驿。**今稠桑原函谷古道遗址**（灵宝函谷关镇境）仍可见——两壁削立、谷底狭长——是中国地理上最有名的"天然关塞" 之一。

**物候**——崤函山地为暖温带季风气候——夏多雨、冬寒、春秋多风沙；黄河南岸地势低、夏可耕；崤山林木较丰、有古道两侧的杨柳秋色——是唐人"过函谷"诗中的常景。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

函谷关在唐人的回望里——是**前代关塞 + 怀古 anchor**：1100 余年的关塞史、三关之徙、老子尹喜传说、玄元皇帝庙——堆叠在崤函山间这片小小狭谷里。

**战国秦古函谷关**：**秦孝公商鞅变法后置函谷关**——是关中东大门，扼六国西进咽喉；**"秦不开关、六国莫敢西"** 见《史记·六国年表》——这是中国军事地理史上最有名的一句"地形决定论"。战国合纵抗秦六次、**联军兵至函谷关下而退者数次**——古函谷关是六国军事意志的物理终点。**秦统一后、函谷关仍为关中东大门**；**陈胜吴广起义后**——**刘邦 BC 206 由武关入关、至灞上、秦王子婴投降**（不是经函谷关）；**项羽 BC 206 率诸侯兵后至古函谷关、闻刘邦已定关中、怒而强行突入**——古函谷关之名因此役复显。

**汉武帝元封东徙函谷关**——**约前 110（元封元年）楼船将军杨仆因家在新安、属关外、耻之、上书请徙关于新安**；**武帝从之、约前 109（元封二年）后实际徙关于新安**——成"函谷新关"（汉关）。**这是中国关塞史最戏剧的一次"为一人徙一关"**——因杨仆的个人荣耻、改变了关中—中原 600 年的地理界。**徙关之后**——古函谷关废、新安关继其名号；但**关塞地理优势远不如古关**（新安青龙山不如稠桑原险），自此函谷关在军事意义上渐衰。

**东汉建安潼关代起**——**建安元年（196）曹操弃函谷新关、于黄潼会合处筑潼关**；**自此潼关取代函谷关成关中东大门**——函谷新关退冲要、函谷古关已废 300 年、函谷之名渐为怀古之名。

**三国魏函谷关**——**黄初年间（220-226）魏文帝曹丕在古函谷关南复立关**（魏函谷关）——意图复古关之险；但**实际地位远不如潼关**——很快又废。**自魏晋以下、函谷三关皆为遗址**；隋唐两京驰道修后、函谷古道渐为辅道；**唐 755 时函谷关塞职能完全由 70 里西的潼关承担**。

**老子 + 尹喜 + 古函谷关传说**——《史记·老子韩非列传》："老子修道德、其学以自隐无名为务……见周之衰、乃遂去、至关、关令尹喜曰：'子将隐矣、强为我著书。'于是老子乃著书上下篇、言道德之意五千余言而去、莫知其所终。"——**《史记》未明指此"关"即函谷关**；但**自汉以下道教传统、皆以此"关"为古函谷关**（亦有说楼观台说经者——见《楼观本起内传》）。**唐皇室追祖老子、尤重老子说经地点**——**玄宗开元二十九年/741 制两京及诸州各置玄元皇帝庙**；**虢州函谷关故址即立玄元皇帝庙一座**（见《唐会要》卷五十）——这是唐 755 时古函谷关在唐廷视野中的核心地位：**不是要塞、而是道祖庙 + 怀古地**。

### 此刻

天宝十四载这一年的函谷关——**三关皆已为遗址、皆已不立要塞**。

**古函谷关故址（虢州灵宝县境）状态**：稠桑原狭谷仍在；古关楼遗址犹存（具体存留程度史载零散——本 doc 不预设）；**关故址侧立有玄元皇帝庙**（开元二十九年/741 玄宗敕立）；**唐廷遣官修庙、岁时祭老子**；**关道仍可通行**、但已为地方驿支线。

**函谷新关故址（河南府新安县境）状态**：青龙山口关楼遗迹犹存；**两京驰道（北线）经新安县城**——但驰道主干**不经新关故址**（新关在新安东、驰道在新安县城北）；**新关故址在唐 755 时已不为要塞**——是地方旧迹。

**魏函谷关故址**：在古关南数里；**唐 755 时已废 500 余年**——遗址零散。

**755 时唐人过函谷的实际路线**：**自洛阳西行**——经新安县 + 渑池县 + 陕州 + 灵宝县 + 华阴县 + 潼关 + 至长安——**这是两京驰道（北线）的主干**；**过新安时**经过函谷新关故址附近、但驰道不入旧关；**过灵宝时**——可见古函谷关故址 + 玄元皇帝庙；**过陕州**——是黄河漕运咽喉；**过潼关**——是真正的关中东大门。**整条驰道 7-8 日可达**（具体里程史源略异）。

**755 时函谷关上的政治氛围**——**唐廷对函谷关的核心关注是道祖庙 + 玄元皇帝祭祀**——不是要塞防御；**安史前夕函谷三关无戍兵 + 无关吏 + 无关税收**——这是真正的"前代关塞" 状态。**与潼关那种"严整守备"的现役关塞完全不同**——自东汉建安元年（196）潼关取代函谷关为关中东大门后、函谷三关渐退为旧关 / 怀古地、至唐 755 已逾 550 年。

**755 末 + 756 初函谷关的命运**：**安禄山十一月起兵 → 十二月陷洛阳 → 756 西向潼关攻击**——**叛军西进途中必经新安 + 渑池 + 陕州 + 灵宝（古函谷关故址）**——**但叛军的真正目标是潼关**、不是函谷；**叛军过函谷如过空关**——这是唐 755 函谷关在安史之乱中的真实角色：**只是叛军行军路线上的一片旧迹**。

### 此刻意味

对天宝十四载的虢州灵宝县人来说——古函谷关是**自家境内的怀古地**：玄元皇帝庙岁时香火、唐廷遣官祭老子、旧关道偶过商旅。对河南府新安县人来说——函谷新关故址是**家门口的旧关楼**：驰道过新安、新关在县东、平日少人问津。对长安那一代士人来说——函谷关三关是**史书 + 道经里的关塞**：战国合纵 + 秦汉东冲 + 杨仆徙关 + 老子西出 + 玄元庙——**这些故事比关本身更重要**。

唐人尚不知道——**函谷之名将在中国地理史中长存**：自先秦至今、"函谷"二字一直是**中国关塞文化的最深 anchor**——比潼关、剑门关、玉门关都更早、都更老；**老子《道德经》传说**——使函谷关脱离单纯军事关塞、成**道家文化的物理原点**；**杨仆徙关**——成**中国"为一人徙一关"的成语典故**（"耻为关外人"）；**"秦不开关、六国莫敢西"** 成**中国军事地理的格言**。**今古函谷关遗址（灵宝市函谷关镇）**——经 1990s-2010s 多次勘察发掘、稠桑原函谷古道遗址保护性展示；**新建有函谷关历史文化旅游区 + 老子文化园**——是中国关塞旅游 + 道家文化的核心地之一。**今函谷新关遗址（新安县）**——青龙山口存少量关楼基址；**魏函谷关遗址**已不显。**唐人在 755 所见的那座玄元皇帝庙**——千二百多年后已不存（庙宇主体多次毁于战乱 + 重修），今函谷关景区内有近年新建的太初宫——是怀古性的复建。

## 子条目

### 古函谷关（秦关）+ 稠桑原狭谷

**古函谷关之险**——**全在稠桑原狭谷**：黄河南岸 + 崤山北麓夹成一道狭谷、两壁削立十余丈、谷底窄不过容车、深险如函；**函谷古道**就走在谷中——**是战国秦关中东大门、是秦汉关中东冲的唯一通道**。**"秦不开关、六国莫敢西"** 是中国军事地理史最有名的一句——**指的就是这道狭谷**。**战国六国合纵抗秦六次**——联军兵至函谷关下而退者数次（《史记·六国年表》）；**秦兵出关 + 六国兵退**——这道狭谷见证了战国 100 余年的西东对峙。**唐 755 时古关故址属虢州灵宝县境**——玄宗开元二十九年/741 立玄元皇帝庙一座、岁时祭老子；**关道仍可通行**、是地方驿支线。

### 杨仆徙关 + 函谷新关（汉关）

**汉武帝元封间徙函谷关于新安**——**中国关塞史最戏剧的一次"为一人徙一关"**：**楼船将军杨仆**（汉武帝时名将、平南越功臣）因家在新安、属关外、**耻为"关外人"、上书请徙关于新安**；**武帝从之**——**约前 110 杨仆上书、约前 109 后徙关至新安东**（《史记·杨仆传》+《汉书·武帝纪》元封纪）。**新关址在今河南省洛阳市新安县东、青龙山一带**——距古关东北约 300 里。**徙关后**：关中—中原界向东推 300 里、古函谷关废、函谷新关继其名号。**杨仆耻为关外人** 成中国成语典故——**是个人荣耻凌驾地理大势的极端例证**。**唐 755 时函谷新关故址属河南府新安县**——已废 800 余年、唯余青龙山口的旧关楼基址。

### 魏函谷关（黄初复立的"复古之关"）

**三国魏黄初年间（220-226）曹魏在古函谷关南数里复立关**——是为"魏函谷关"。**意图**：**复古关之险 + 防御西方蜀汉**——曹魏在与蜀汉对峙之际、欲重建函谷的天险传统。**但实际地位远不如潼关**——**潼关已在建安元年/196 立**（曹操弃函谷新关、于黄潼会合处筑潼关）；**潼关地势比函谷更险、扼关中东大门更直接**——**魏函谷关在潼关阴影下、很快废**。**自魏晋以下、函谷三关皆为遗址**——魏函谷关存在时间最短、地位最弱、在中国关塞史中最隐。唐 755 时已废 500 余年——遗址零散、唐人罕至。

### 老子西出函谷关 + 尹喜留经

**《史记·老子韩非列传》**：**"老子修道德、其学以自隐无名为务……见周之衰、乃遂去、至关、关令尹喜曰：'子将隐矣、强为我著书。' 于是老子乃著书上下篇、言道德之意五千余言而去、莫知其所终。"**——**《史记》未明指此"关"即函谷关**；但**自汉以下道教传统、皆以此"关"为古函谷关**（亦有说楼观台说经者——见《楼观本起内传》）。**唐 755 时唐皇室追祖老子**——**玄宗开元二十九年/741 制两京及诸州各置玄元皇帝庙**；**虢州函谷关故址即立玄元皇帝庙**（《唐会要》卷五十·尊崇道教）。**自此函谷关在唐人视野中转型**——从"前代关塞" 转为"道祖庙 + 怀古地"；**李白、岑参等盛唐诗人路过函谷皆作诗**——但 755 时具体诗证史源不稳、本 doc 不预设。

### 函谷关与潼关（古关 vs 新关）

**函谷关与潼关的关系是中国关塞史最重要的一对"古关—新关"承继**：
- **战国秦至西汉初**：**古函谷关**是关中东大门——稠桑原狭谷扼咽喉。
- **西汉武帝元封间至东汉建安**：**函谷新关**（新安）继其名——但地势已远不如古关。
- **东汉建安元年/196 至今**：**潼关**（黄潼会合处）取代函谷关——**潼关在函谷西约 70 里**（直线距离）、扼黄河南岸 + 秦岭北麓夹峙处——地势比函谷更险、更直接守关中东大门。**自此函谷之名退冲要、潼关接续之**——这是中国关塞地理的一次"东西大转移"。**唐 755 时**：**潼关是现役关中东大门**（半年后哥舒翰守关、败战、玄宗西奔）；**函谷三关是前代遗址 + 怀古地**。这种**"古关怀古、新关守险"** 的格局——是 755 时函谷 / 潼关并立的真实状态。

### 玄元皇帝庙 + 唐皇室道教与函谷关

**唐皇室自高祖始追祖老子**——**李渊与老子（李耳）同姓**、即追为远祖；**武德间在楼观台立宗圣观奉老子为道祖**；**玄宗朝道教尊崇达盛唐顶峰**——**开元二十九年/741 正月制"两京及诸州各置玄元皇帝庙"**（《旧唐书·玄宗本纪下》）——**两京玄元皇帝庙立于长安大宁坊、洛阳积善坊**；**诸州玄元皇帝庙则按地择址**——**虢州即立于函谷关故址**（依老子西出说经传说）。**唐 755 时虢州函谷关玄元皇帝庙岁时祭老子**——是唐皇家道教体系在地方的物理 anchor；**与楼观台（京兆府盩厔县终南山北麓）+ 玄元皇帝庙（长安 + 洛阳）+ 函谷关玄元庙**——构成唐皇家道教三大空间节点。**这是唐 755 时函谷关的真实身份**：**不是关塞、是道祖庙**。

## 历史长河（755 → 今）

**中晚唐**　**756 安史叛军过函谷西攻潼关**——函谷三关皆为叛军行军路线上的旧迹、无战事；**757 唐军郭子仪 + 李光弼 + 回纥东出潼关复长安洛阳**——亦经函谷而过、不停。**中晚唐函谷玄元皇帝庙岁时祭老子不辍**；**杜甫、岑参、李白等诗人路过函谷皆作诗**（具体诗证略异）——函谷在中唐诗中是"前代怀古"地、远不如潼关分量重。**唐末黄巢之乱**——黄巢军过函谷西攻潼关（广明元年/880）；函谷三关皆为旧迹、无所阻。

**两宋**　**北宋函谷玄元皇帝庙废**（具体年份史载略异）；**北宋灵宝改属陕州**（虢州废）；**北宋间函谷古道仍存**、但已非主驿。**南宋金人占函谷一带**——金灭北宋后函谷三关皆在金境、无重立。

**金元明清**　**元明清间函谷古道仍存**——是地方驿道；**明清间在古函谷关故址附近多次重修关楼 + 老子庙（太初宫等）**——但形制远不如盛唐时严整。**清代灵宝、新安二县皆为河南府所辖**——函谷三关皆为地方旧迹。

**民国至今**　**1949 年后函谷三关初列省级文物保护**。**1990s-2010s 河南省文物考古研究院 + 灵宝市文管所多次勘察古函谷关遗址**——稠桑原函谷古道遗址保护性展示；**新建函谷关历史文化旅游区 + 老子文化园**（含近年新建的太初宫——是怀古性复建、不是唐玄元庙原物）。**今函谷新关遗址（新安县）**——青龙山口存少量关楼基址；**魏函谷关遗址**已不显。**唐人在 755 所见的那座古函谷关 + 玄元皇帝庙**——千二百多年后**关楼主体已不存、玄元庙原物已不存**——今所见多为明清及近代复建/重修。**函谷之名仍在**——"函谷"二字穿越 2300 年、仍是中国关塞文化的最深 anchor。**这是函谷三关的物理终点**：**关楼或不存、关道犹在、关名不灭**。
