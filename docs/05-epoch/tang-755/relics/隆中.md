# 隆中

## 一句话

襄州襄阳城西二十里的山中——诸葛亮自荆州避乱躬耕于此十年、刘备三顾草庐、孔明出"隆中对"定三分天下——**盛唐人凭吊三国地最盛者**，李白杜甫多有题诗。

## 形貌

隆中在襄州襄阳县城西二十里——**汉水南岸、隆中山下**，是一片山间小盆地。

地势：北临汉水（沔水）支流檀溪、南依隆中山主峰（海拔约 306 米）、西接虎头山、东接乐山——四山围合一片东西约二里、南北一里余的小盆地。**诸葛亮当年的草庐传说便在这片盆地中央**——东晋习凿齿《襄阳耆旧记》"亮家于南阳之邓县、号曰隆中"（汉时此地属南阳郡邓县、东汉末曹魏分荆州后归襄阳郡；盛唐时属山南东道襄州襄阳郡——历代行政区划之变是地名争议的种子）。

**隆中山是这一带地理标识**——主峰不高，但孤峰挺出于襄阳城西平畴之上、远望可辨。山下檀溪水绕隆中盆地北缘东流入汉水。盛唐人自襄阳城西出，循檀溪、过万山一线西行十五里至檀溪渡、再南行五里入隆中山口——这是 755 时凭吊诸葛庐的实际行程。

物候——汉水中游南岸气候湿润、四时分明；隆中山中竹林、柏林、桂林为主，是襄阳城外一处可日游可半日还的山中清地。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

隆中在唐人的回望里是**三国诸葛亮的躬耕地、三顾茅庐发生地、隆中对策提出地**——是中国历史上"君臣相得"叙事的物理 anchor。

**诸葛亮（181-234）**——琅琊阳都（今山东沂南）人，幼孤。**叔父诸葛玄**带诸葛亮南投荆州刘表——玄殁后诸葛亮**约建安二年至四年（197-199）间**移居隆中、自此**躬耕陇亩、博览群书、自比管仲乐毅**。**居隆中约十年**——其间结交荆州本地名士司马徽、庞德公、徐庶等。

**建安十二年（207）前后**——刘备时驻新野（今河南南阳新野，时属荆州）；徐庶向刘备推荐诸葛亮："此人可就见、不可屈致"。刘备**三顾隆中草庐**——《三国志》原文仅"凡三往、乃见"（具体季节、何月，正史无明记）。**诸葛亮为刘备分析天下形势、提出"跨有荆益、待时北伐"的战略**——史称**"隆中对"**（《三国志·诸葛亮传》记载完整）。**孔明出山辅刘备**自此始——时年二十六、刘备时年四十六。

**自 207 出山**——诸葛亮先后辅刘备取荆州、定益州、立蜀汉；**建兴元年（223）刘备崩、刘禅即位**，诸葛亮领丞相、辅政；**建兴五年至十二年（227-234）**主持北伐五次；**建兴十二年（234）八月**——诸葛亮卒于**五丈原**（另立 doc 详）。

**自三国吴蜀皆亡后**——隆中作为"诸葛躬耕地"开始被纪念。**东晋习凿齿《襄阳耆旧记》（约 360s）首次明确记隆中为"亮家"**——这是隆中具体地点定于襄阳城西二十里的最早具名史源。**北魏郦道元《水经注·沔水》**沿承之、记"沔水南、隆中山下、诸葛庐故宅"。**自东晋以下至唐——襄阳隆中即诸葛躬耕地、是士庶共识**。

### 此刻状态

天宝十四载这一年的隆中——**距诸葛亮出山辅刘备已 548 年、距诸葛亮卒于五丈原已 521 年**。

隆中盆地中央**诸葛庐故宅地望可指**——东晋《襄阳耆旧记》、北魏《水经注》皆指此为"亮家"；东晋以来历代当有过简略修葺（具体盛唐时是何状态、史源未详）。**盛唐时是否已立"三顾堂""梁父吟碑""诸葛井"等具名纪念建筑——本 doc 不预设**；这些具体物事多是明清至今陆续附会形成的纪念地标，不是盛唐 755 时的实际存在。**755 时隆中的实际状态**：草庐故宅地望可指、隆中山在、檀溪在——但具体纪念建筑可能仅有简陋一二、远非今古隆中规模。

**盛唐人来隆中凭吊者甚众**。**李白《读诸葛武侯传书怀》《赤壁歌送别》**等多首诗咏诸葛亮——李白对诸葛亮的崇敬贯穿盛唐文学；**杜甫此时尚在长安**——他的《蜀相》（上元元年/760 客成都作）、《八阵图》（大历元年/766 夔州作）、《武侯庙》等"诸葛三诗"是中唐凭吊诸葛亮的最高峰，**但都不写于 755 之前**——本 doc 严守"杜甫诸葛诗系于 755 之后"口径。

**755 时的隆中** 是**山中一处供凭吊的纪念地**——日常香火、岁岁有襄阳士庶来访。再过半年安禄山起兵——但荆州、襄阳此时未直接受兵燹影响；**隆中在安史乱中物理保存基本完好**。

### 此刻意味

对天宝十四载的襄阳人来说，隆中是**州城西二十里那片"诸葛躬耕地"**——日常文教与凭吊的去处。对盛唐文人来说——隆中是**"君臣相得"的具象**：刘备三顾、诸葛对策、出山辅佐——这种"诚意被回应"的叙事在盛唐文人中持续共鸣，**李白尤其崇敬诸葛亮**。

唐人尚不知道——再过 500 年（南宋），南阳卧龙岗会兴起为另一处"诸葛躬耕地"、与襄阳隆中并争；再过千年，隆中山下的草庐、三顾堂、梁父吟碑会经过明清数次大修；**今所见的"古隆中"是明清以来历代纪念地的累积**——诸葛亮 207 出山时的草庐，物理上的真实遗物——一片都没有；但**这片盆地的位置一千八百多年不易**——是诸葛庐传说最稳的物理 anchor。

## 子条目

### 诸葛草庐遗址

盆地中央——传为诸葛亮 197-207 间躬耕居处。**东晋习凿齿《襄阳耆旧记》、北魏《水经注》皆指此地为"亮家"**——是隆中诸葛庐说的两大早期史源。**盛唐草庐主体不是诸葛亮当年所居原物**——而是东晋以来历代修葺的纪念建筑。今所见草庐为明清重修、复原唐宋人想象中的"草庐"。

### 三顾堂

草庐之东——传为刘备三次顾访诸葛亮的具体地点。**建安十二年（207）前后**刘备自新野来此、"凡三往、乃见"——闻"隆中对"。三顾堂建年史载未详——**盛唐 755 时是否已立"三顾堂"具名建筑、本 doc 不预设**；今所见三顾堂为明清重修。

### 隆中山主峰

隆中盆地南屏——海拔约 306 米。**山非五岳、亦非道教祖庭**，但因诸葛亮"躬耕陇亩、博览群书"于此山下，山名"隆中"自三国后随诸葛亮典故传天下。山中竹柏成林、四时常绿——是隆中盆地的天然背景。

### 隆中对（孔明对策原文出处）

**建安十二年（207）前后**——刘备三顾隆中得见诸葛亮、问"汉室倾颓、奸臣窃命……欲信大义于天下"之策（正史无具体季节、《三国志》仅"凡三往、乃见"）。诸葛亮分析：曹操不可与争锋于北、孙权可以为援不可图于东、荆州可取、益州可入；待时北伐——**史称"隆中对"**，《三国志·诸葛亮传》载全文。**这一段对策是中国军事政治史最有名的"未来推演"**——是隆中作为纪念地的核心叙事 anchor。

### 梁父吟（诸葛亮自比管仲乐毅）

《三国志·诸葛亮传》："（亮）身长八尺、容貌甚伟、时人异焉。躬耕陇亩、好为《梁父吟》——每自比于管仲、乐毅、时人莫之许也。"**《梁父吟》是齐鲁古乐府**——内容多悲歌、咏古。诸葛亮"好为《梁父吟》" 是文学典故、自此"管乐自比"成中国士人理想抱负的代名词。盛唐李白《梁甫吟》即取此典——但**李白《梁甫吟》主旨是抒怀、非凭吊隆中**。

## 历史长河（755 → 今）

**中晚唐**　安史乱中襄阳为后勤要地、未直接受兵燹大破坏；隆中在乱中保存完好。**上元元年（760）杜甫客成都过武侯祠作《蜀相》**——"丞相祠堂何处寻、锦官城外柏森森"——成都武侯祠与襄阳隆中是中唐凭吊诸葛亮的两大地标。**中晚唐韩愈、刘禹锡、白居易**等过襄阳皆有隆中咏。

**两宋**　**北宋大中祥符间（1008-1016）**朝廷诏修隆中诸葛庐——这是隆中第一次入官修维护体制。**南宋初**——金兵屡至襄阳、隆中略有兵燹；但因襄阳南宋长期为军事重镇、官府重视、隆中得续修。**南宋后期至元间**——南阳卧龙岗作为"另一处诸葛躬耕地"开始兴起；**自此襄阳隆中 vs 南阳卧龙岗的"躬耕地之争"延续至今**。

**明清**　明初敕修隆中诸葛庐；**明嘉靖间（1522-1566）襄阳府大修隆中**——草庐、三顾堂、梁父岩、诸葛井等系列纪念建筑陆续完善——今所见"古隆中"主体格局多定于此时。清康熙、雍正、乾隆三朝俱有修缮记录。**清代隆中已是襄阳第一名胜**——士大夫贬荆湖、贾客过襄阳，皆以至隆中礼诸葛为人生一事。

**民国至今**　1949 年后隆中列湖北省重点文物保护单位、1996 年列全国重点。**1990s-2010s 多次考古勘察**——确认隆中山下盆地有汉魏六朝陶片、夯土遗痕，但**未发现可直接证明"诸葛亮原居住所"的物理证据**——这是隆中考古的实际状态。**今古隆中为襄阳市襄城区古隆中风景名胜区**——草庐、三顾堂、梁父岩、诸葛井、武侯祠俱为明清重修。**唐人在 755 所见的那座山中诸葛庐**——位置在、传说在、具体物事一片真的也没有；但**这片盆地的位置一千八百多年未易**——是隆中传说最稳的物理 anchor。
