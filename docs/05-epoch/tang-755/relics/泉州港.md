# 泉州港

## 一句话

**福建新兴的海港**——泉州港在江南东道泉州清源郡晋江县治南郊、晋江入海口与后渚一带；**唐景云二年（711）方自武荣州改置泉州、至 755 时设州才四十四年**——是盛唐福建沿海一处仍在起步的新兴港务，远不如同道北的福州港繁盛、更不及岭南广州的市舶巨埠。**755 时无市舶使**（市舶使制此期专设于广州，泉州设市舶司要待北宋元祐二年/1087）；海舶以大食、波斯、师子国偶到，**福建三港（福州 + 泉州 + 漳州）中泉州居中**；**中晚唐渐兴 + 五代闽国奠基 + 两宋成为中国第一大港 + 元代马可·波罗见其盛景**——但盛唐 755 这一刻、泉州港尚是江南东道海岸线上一处年轻的津口。

## 形貌

泉州港在 755 时——**不是一座城南门外的渡口、而是一片散落在晋江入海口南北两岸的港埠群**。

**地形位置**：**晋江县治**（今泉州市鲤城区一带）城南三五唐里至晋江北岸；**晋江**自西北戴云山诸支汇流而东南入海、入海口北岸是**后渚港**一带（今泉州市丰泽区东海街道）、南岸是**法石**一带（今泉州市丰泽区南）；**港务核心**在晋江入海口南北两岸三五唐里范围内——船泊处分散、不集中于一处。**北面**——后渚港深水、可泊海舶；**南面**——法石浅水、多泊江舟与近海小船；**港西**——晋江上溯可至晋江县治、再上至南安县（今南安市）；**港东**——出海口外为**台湾海峡**（唐人称"东海"或"涨海"南段）、东南可达林邑（占婆）+ 真腊 + 师子国 + 大食。

**港埠形制**：盛唐 755 时泉州港——**官设港务初立 + 民间码头并存**。**无独立市舶使**（市舶使专设于广州、自开元间制度化）——港务由**清源郡太守 / 晋江县令**监督（755 时正名清源郡、长官为太守而非刺史；天宝元年/742 改州为郡、刺史改太守；乾元元年/758 复州时方复刺史）；蕃舶到岸先报清源郡郡衙、再转报江南东道采访处置使（治苏州）。**码头形制**：石阶 + 木栈道 + 江岸夯土堤；海舶系缆于江岸或锚泊江中、小舟驳运货物上岸。**港务设施**：泊船区 + 仓房（多为夯土 + 木构）+ 蕃客馆驿（粗具雏形、未及广州蕃坊之规模）。

**周边水道与陆路**：
- **晋江**——主流自戴云山东南流、过南安县、晋江县治、入海于后渚港 + 法石之间；江面宽约一二唐里、潮汐影响达晋江县治；
- **海岸线**——南北蜿蜒、多港湾（**北至福州港约三百唐里、南至漳州港约三百唐里**）；**福建三港**（福州 + 泉州 + 漳州）沿海岸线南北排列；
- **陆路**——晋江县治 → 西北过仙游 → 福州（约五百唐里）；晋江县治 → 西南过漳浦 → 漳州（约四百唐里）；陆路艰险、**闽中（江南东道福建诸州）**入唐前后皆以海路 + 内河为主（按：**福建经略使 733 已置（《新唐书·方镇表》）；但福建道作为完整一级行政道 是 758/759 才正式分**——755 一刻闽中诸州（福州、建州、泉州/清源郡、漳州、汀州——**汀州开元二十四年/736 已置**）皆归江南东道采访处置使监临、治苏州）。

**物候**：盛唐时泉州港一带——**亚热带海洋季风、四时如春**；冬无霜雪 + 夏多台风（六至九月）+ 春秋多雨；海舶到港**多在春夏信风期**（南海西南风、五至八月）、归航**多在秋冬信风期**（东北风、十至次年二月）。**台风**——是泉州港最大的物候威胁；唐人尚不识"台风"二字（"台风"作专名要至明清；唐人称"飓风"）；港务每年六至九月须备避风。**潮汐**——晋江入海口潮汐明显、海舶到港须候潮入港 + 候潮出港；是泉州港务的日常节奏。

**周边聚落**：**晋江县治**（今泉州市鲤城区一带）城周约六唐里、夯土城墙、唐景云二年/711 改武荣州为泉州时州治寄于此处、开元六年/718 析南安置晋江县后正式作为泉州 / 清源郡 + 晋江县治、城内有清源郡郡衙 + 晋江县衙 + 武荣州旧址（武荣州先于此、唐圣历二年/699 自南安县析置、唐景云二年/711 改泉州）；**南安县**（今南安市）在晋江上游、是泉州境内古县；**惠安县**在港北、是后渚港的北翼陆地（惠安县唐时尚未单置、属晋江县管辖；惠安县置于北宋太平兴国六年/981）。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

泉州港在唐人的回望里——**是一处新置不到半个甲子的福建海港 + 一处从内陆县治派生出的港务**。

**州名沿革**——这一点须细辨、因唐前期"泉州"曾有二地：
- **武德六年（623）至景云二年（711）间**——"泉州" 治**闽县**（今福州治）；此时之"泉州"即后世之福州、与晋江一带无涉；
- **景云二年（711）**——**自闽县处之泉州改名"闽州"（非"福州"）、同年于晋江一带新置泉州、治晋江县**——自此"泉州"专指晋江一带（《新唐书·地理志六》、《旧唐书·地理志三》皆载）；**福州之名要待开元十三年（725）方自闽州改置**——711-725 间闽县治处之州正名为"闽州"、非"福州"；常见误植"711 改福州"——实为 711 改闽州、725 改福州；
- **天宝元年（742）**——玄宗诏天下州一律改郡、泉州改称**清源郡**——这是 755 年当年的官称；
- **乾元元年（758）**——肃宗复改郡为州、清源郡复称泉州。

**755 时正名"清源郡"**——治晋江县；行政文书写"清源郡晋江县泉州港"；民间俗称"泉州""晋江口""后渚港"皆可。

**晋江县名沿革**：**唐开元六年（718）析南安县置晋江县**（《新唐书·卷四十一·地理志六》系于"开元六年"；《元和郡县图志·卷二十九》系于"开元八年/720"——异文并存、姑取开元六年为主、开元八年备录）——因晋江得名；自此晋江县治成为泉州 / 清源郡的州治核心；**755 时**晋江县置 37 年（取开元六年/718 算）至 35 年（取开元八年/720 算）、属新县。**前此 711-718 间**——泉州虽已置、晋江县尚未析出、州治寄于南安县之东境。

**注意**：**不要写"鲤城"作 755 名**——"鲤城"是后世（北宋以后）泉州城形如鲤鱼的别称、唐人不用此名；**不要写"刺桐城"作 755 名**——"刺桐"作泉州别称大抵起于**五代**：**闽国王延彬**（约 904-930 间长期为泉州刺史、号"招宝侍郎"）经营海贸开其端；**清源军节度使留从效**（949-962 节度）传称遍植刺桐于城周——"刺桐城"作泉州别称由此渐起；盛唐 755 一刻刺桐尚未植、亦无"刺桐城"之名。

**港务沿革**——
- **秦汉**——晋江一带属**会稽郡冶县**（汉武帝平闽越后）、再属**会稽南部都尉**；港务无明载；
- **三国吴**——孙吴永安三年（260）置**建安郡**于福州一带、晋江口属之、为孙吴海防之远郊；
- **西晋**——**晋太康三年（282）置晋安郡**于福州、晋江一带属之；**"晋江"得名亦在此期**——晋人南迁、江得晋名（民间附会、史源可疑、姑存）；
- **南朝陈**——光大二年（568）置**南安郡**于南安一带、晋江口属之；
- **隋**——开皇九年（589）平陈、改南安郡为**南安县**、属泉州（治闽县/今福州）；
- **唐圣历二年（699）**——自南安县析置**武荣州**于晋江一带；**景云二年（711）**——改武荣州为**泉州**、州治寄于南安县之东境、闽县治处之旧"泉州"同时改名**闽州**（**福州之名要待开元十三年/725 始定**）——**自此晋江一带始称泉州**；**开元六年（718）**——析南安县置**晋江县**（《新唐书·地理志六》；《元和郡县图志》作开元八年/720——异文并存）、自此泉州治晋江；
- **开元 + 天宝间**——泉州港务渐开、海舶偶到、为福建三港之一；755 时港务初具雏形、远不及广州市舶之盛。

**唐人对泉州港的认知**——盛唐 755 一刻泉州港不在李白杜甫的诗中、不在唐玄宗的视野里；是江南东道海岸线上一处年轻的津口。**主要使用者**：**福建本道**商旅（盐 + 茶 + 瓷器 + 木材）+ **江南诸道**南来海商（杭州 + 明州 + 福州一带的中转海舶）+ **岭南广州**的二级蕃舶（自广州转泊福州 / 泉州的小型海舶）+ **海外蕃舶**偶到（大食 + 波斯 + 师子国 + 林邑 + 真腊；但远不如广州 + 数量极少）。**主要货物**：进口——香料（沉香、乳香、苏合香）+ 象牙 + 犀角 + 珊瑚 + 玳瑁；出口——丝、绢、瓷（越州青瓷 + 长沙窑器经此南转）、茶、铜钱。

### 此刻

天宝十四载这一年的泉州港——**港务按例运行、海舶偶到、福建本道商旅密集、远不识安史浩劫的暗影**。

**港务的实际规模**：盛唐 755 时泉州港——**远不及广州**（广州一年蕃舶可至数十艘、市舶岁入巨）；**约略接近福州港**（福州港稍北、是江南东道老港、自孙吴 + 东晋以来即为海贸节点）；**胜于漳州港**（漳州港更南、置州才更晚、唐垂拱二年/686 置漳州、港务薄）；**福建三港**中泉州居中、是新兴的中位港。**港务岁入**——无明载、当不至州县财赋大宗；**蕃舶岁数**——当不过数艘、远不及广州之盛。

**蕃客的实际数量**：盛唐 755 时泉州港——**蕃客馆驿粗具雏形、未及广州蕃坊之规模**；大食、波斯、师子国、林邑商人偶聚港西、人数不多；**唐人对蕃客的处置**——按唐廷羁縻常制，蕃客以**临时馆驿 + 港西散居**为主、未成蕃坊制度；尚无独立蕃坊建制（蕃坊作专名 + 蕃长作专职、是中晚唐至五代以后泉州港随蕃舶日繁而陆续形成的）。

**港务与朝廷的关系**——**755 时无市舶使**：市舶使制专设于广州（开元间制度化）、泉州港务由**清源郡太守 / 晋江县令**直辖（755 时正名清源郡、长官为太守）、再上报江南东道采访处置使（治苏州）。**舶脚 + 进奉**——港务收的关税与朝廷贡献、当循广州市舶之例、但规模小得多；**蕃货 + 蕃客**——多自广州或经海路直至福建、再分流至福州、泉州、漳州。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九即起）；**天宝十四载十二月丁酉陷东京洛阳**（公历 756 年 1 月）——中原大震；**但江南东道及闽中一带远在江南**——安史叛军未能南下过淮、**闽中（江南东道福建诸州）未陷**（按：**福建经略使 733 已置（《新唐书·方镇表》）；但福建道作为完整一级行政道 是 758/759 才正式分**、755 时闽中诸州归江南东道采访处置使监临）；**泉州港 + 福州港 + 漳州港皆完整保全**——成为唐廷江南半壁的物理 anchor。**至德 / 乾元间**——唐廷半壁赖江淮租赋苦输支撑、海贸路径未受直接战火；**福建三港的海舶反而稍增**——因北方陆路 + 江淮漕路反复中断、南方海贸的相对重要性上升；**这是泉州港中晚唐兴起的远因**——但 755 这一刻、唐人尚不能预见。

### 此刻意味

对天宝十四载的清源郡晋江县人民来说——泉州港是**自家城南郊的港务**：日常商旅出入 + 海舶偶到 + 蕃客聚港；晋江县治南门外舟车辐辏、皆为入泉州港务之用。对江南东道的人民来说——泉州港是**福建三港之一**：自福州南行海路、过泉州、至漳州、再南可达广州；是江南东道海岸线上的中位节点。对盛唐唐廷来说——泉州港**尚不构成财政命脉**——盛唐财政命脉在江淮漕运（瓜洲 + 邗沟 + 通济渠）、在关中粟米、在岭南广州市舶；泉州港此刻只是江南东道海岸线上一处年轻的辅港。

唐人尚不知道——
- **中晚唐**（约 9 世纪）——海舶渐多至泉州、福建三港中泉州渐与福州齐名；
- **五代闽国**（909-945）——王审知 + 王延翰 + 王延钧 + 王延曦 等闽国诸王以泉州为东南海贸重镇、植刺桐 + 立蕃客馆驿 + 海舶日盛；**"刺桐城"作泉州别称自此始**；
- **北宋元祐二年（1087）**——朝廷于泉州设市舶司、自此泉州港成为中国主要海贸港之一、与广州并称；
- **南宋**——泉州港继续繁荣、与广州并为南方两大海港；**南宋宝庆元年（1225）赵汝适为泉州市舶提举、撰《诸蕃志》**——记 50 余国海贸；
- **元代**——泉州港跃升中国第一大港、亚洲乃至世界海贸枢纽之一；**元马可·波罗见其盛景**（约 1292 经此回西方）——记"刺桐港"商船云集、远超广州；**元至正九年（1349）汪大渊撰《岛夷志略》**——亦于泉州写就；
- **明代以降**——明初海禁 + 月港（漳州）兴起、泉州港渐衰；**清代**——闽南海港重心转至厦门；
- **今**——泉州港遗存以**后渚港 + 法石 + 真武庙 + 老君岩 + 清净寺 + 开元寺 + 洛阳桥**等串联、为联合国教科文组织"宋元中国的世界海洋商贸中心"世界文化遗产（2021）——但**这些遗存绝大部分是宋元时期的、不是盛唐 755 的物理形态**。

**755 一刻的泉州港**——只是**一处置州才 44 年的福建新港、一片晋江入海口南北两岸散落的码头、几艘自广州或江南来的中转海舶、一二处粗具雏形的蕃客馆驿**——是中国海贸史上一个尚未启动的早期节点。

## 子条目

### 武荣州 + 泉州 + 清源郡（州名沿革三段）

**唐圣历二年（699）置武荣州**——自南安县（隋平陈后置、属泉州/治闽县）析置、治今晋江县治一带；是晋江一带始有独立州级建制；**唐景云二年（711）改武荣州为泉州**——同年闽县治处之旧"泉州"改名**闽州**（**非"福州"**——福州之名要待开元十三年/725 始定）、晋江一带始称泉州；**天宝元年（742）改泉州为清源郡**——755 时正名；**乾元元年（758）复泉州**——延至北宋初。**州名沿革的意义**：晋江一带从隋至唐前期是一处不入州级建制的远郊、自武荣州 + 泉州 + 清源郡三段才陆续登上中国行政地图的州级层面；755 时设州才 44 年、是新州。

### 晋江县（开元六年/718 析南安置、因晋江得名）

**唐开元六年（718）析南安县置晋江县**（《新唐书·卷四十一·地理志六》系于开元六年；《元和郡县图志·卷二十九》作开元八年/720——异文并存、姑取开元六年为主、开元八年备录）——前此景云二年（711）改武荣州为泉州时、州治尚寄于南安县之东境、未有"晋江县"之名；718 析置后泉州治始定于晋江；**晋江县名的由来**——因晋江得名；**晋江得名的由来**——民间传说晋人南迁、江得晋名（**晋太康三年/282 置晋安郡** 或两晋永嘉间南迁北人到此一带、江因之得名）；史源不甚精、姑存民间附会。**755 时晋江县治**——置县 37 年（开元六年/718 算）；城周约六唐里、夯土城墙、城内有清源郡郡衙 + 晋江县衙 + 武荣州旧址；县治南门外三五唐里至晋江北岸、再南至后渚港 + 法石。

### 后渚港（晋江入海口北岸的主要泊船区）

**位置**：晋江入海口北岸、今泉州市丰泽区东海街道一带；**港务**：深水可泊海舶、是盛唐 755 时泉州港的主要泊船区；**沿革**：唐前中期开发、中晚唐 + 五代闽国 + 两宋繁盛、元代成为泉州港务的核心；**后世**：1974 年于后渚港发掘的"南宋海船"（泉州海外交通史博物馆藏）是中国海贸考古的标志性发现——是南宋遗物、不是盛唐 755 现场。**唐人对后渚港的认知**——是泉州港的主要泊船区、海舶到港多于此抛锚；但 755 时港务仍粗。

### 法石港（晋江入海口南岸的辅助泊船区）

**位置**：晋江入海口南岸、今泉州市丰泽区南一带；**港务**：浅水、多泊江舟与近海小船、是后渚港的辅港；**沿革**：唐前中期开发、与后渚港并立、中晚唐以后渐弱、宋以后法石港的真武庙（南宋）成为海舶到港的祭祀场所。**755 时**——法石港仅是港务辅助节点、规模远不及后渚。

### 福建三港（福州 + 泉州 + 漳州 的排位）

**福州港**——江南东道老港、自孙吴 + 东晋以来即为海贸节点；**755 时福建三港之首**；治闽县（今福州）；置州最久；
**泉州港**——景云二年/711 新置之州、设州才 44 年；**755 时福建三港之中位**；治晋江县；港务初兴；
**漳州港**——垂拱二年/686 置漳州（治漳浦）、置州也不久；**755 时福建三港之末**；港务薄。

**福建三港的格局**——盛唐 755 一刻自福州→泉州→漳州 沿海岸线南北排列、海路相通、是江南东道海岸线南段的三处主要港务节点；其上为江南东道之明州 + 杭州、其下为岭南广州。**中晚唐 + 五代以后**——泉州港后来居上、北宋以后跃居福建三港之首 + 中国第一大港；但 755 这一刻、福州港仍是福建三港之首。

### 海舶与蕃客（大食 + 波斯 + 师子国 + 林邑 偶到）

**海舶种类**——盛唐 755 时到泉州港的海舶——以**大食 / 阿拉伯商舶** + **波斯**（萨珊余裔、入大食属境）+ **师子国**（斯里兰卡）+ **林邑**（占婆）+ **真腊**（柬埔寨）为主；数量极少、多为自广州中转的二级海舶。**蕃货**——香料（沉香、乳香、苏合香、丁香）+ 象牙 + 犀角 + 珊瑚 + 玳瑁；**唐货**——丝、绢、瓷（越州青瓷 + 长沙窑器）、茶、铜钱。**蕃客馆驿**——盛唐 755 时泉州港蕃客馆驿粗具雏形、未及广州蕃坊之规模；蕃客在港西聚居、人数不多、唐廷按羁縻常制处置。

### 港务与朝廷（755 时无市舶使）

**市舶使制**——专设于广州、自开元间制度化（《唐会要·卷六十六·市舶使》载）；**泉州港 755 时无市舶使**——港务由清源郡太守 / 晋江县令直辖（755 时正名清源郡、长官为太守而非刺史；天宝元年/742 改州为郡时刺史改太守、乾元元年/758 复州时方复刺史）、再上报江南东道采访处置使（治苏州）。**舶脚 + 进奉**——港务收的关税 + 朝廷贡献、循广州市舶之例但规模小得多。**泉州设市舶司要待北宋元祐二年（1087）**——南宋以后泉州市舶司岁入达数十万缗、与广州并称；但盛唐 755 这一刻泉州港尚不构成朝廷海贸的财政重镇。

## 历史长河（755 → 今）

**中晚唐**　**755-907**——海贸路径自北方陆路转向南方海路、福建三港的相对重要性上升；**泉州港渐与福州港齐名**；**晚唐**（约 9 世纪）——大食、波斯海舶到泉州渐多；**蕃客馆驿渐成形**——但仍未达广州蕃坊之规模。

**五代**　**909-945 闽国王审知 + 诸子时期**——**王审知**为泉州一带海贸的重要奠基者、广纳蕃舶 + 优待蕃客 + 整顿港务；**王延彬**（王审知侄、闽国时长期任泉州刺史、约 904-930 间 / 历闽王审知至王延钧朝、号"招宝侍郎"、于泉州 + 海港大兴海贸）——是闽国时期泉州港的关键经营者；**清源军时期**（945-978）——**闽国灭于南唐（945）后**、**留从效**（约 906-962）于 945 之后割据泉、漳、即清源军节度使（清源军 949 始置、留从效 949-962 节度）、传称其曾于泉州城周遍植刺桐——**"刺桐城"作泉州别称大抵自留从效植刺桐 + 王延彬经营之后渐起**（此前盛唐 755 一刻无此名）；留从效卒后**陈洪进**（962-978 清源军节度使、978 纳土归宋）继续经营泉州港、保持东南海贸重镇地位；**五代是泉州港跃升中国主要海港的关键期**。

**两宋**　**北宋元祐二年（1087）于泉州设市舶司**——自此泉州港正式进入中国主要海贸港行列、与广州并称；**1087 之后**——泉州市舶司岁入与海贸规模渐与广州并称（具体数额《宋史·食货志》系于多年累计、非熙宁五年/1072 泉州数据——1072 时泉州尚未设司、彼时所记岁入当为广州市舶司或诸路市舶总数）；**南宋宝庆元年（1225）赵汝适为泉州市舶提举、撰《诸蕃志》**——记 50 余国海贸；**南宋**——泉州港继续繁荣、舶来香料 + 宝石 + 象牙、舶去丝绢 + 瓷器 + 茶叶；**蕃坊 + 清净寺 + 开元寺 + 老君岩 等遗存大半成形于此期**。

**元代**　**元 1271-1368 泉州港跃升中国第一大港**——亚洲乃至世界海贸枢纽之一；**元马可·波罗约 1292 经此回西方**——记"Zaitun"（刺桐）港"商船云集、远超 Alexandria"——是中外海贸史上 dramatic 的一笔；**元至正九年（1349）汪大渊撰《岛夷志略》**——亦于泉州写就、记 200 余国海贸；**元末**——亦思巴奚战乱（1357-1366）—— 港务大伤、元末明初泉州港衰落的关键节点。

**明清**　**明初**——洪武十四年（1381）禁海、市舶司废、泉州港大衰；**明中后期**——月港（漳州海澄、福建漳州一带）兴起、泉州港的海贸地位渐让；**清代**——海港重心转至厦门（康熙二十三年/1684 开海、厦门为闽南主港）；**清乾隆以后**——泉州港彻底退出中国海贸主港行列。**今泉州港遗存**——以**后渚港 + 法石 + 真武庙 + 老君岩 + 清净寺 + 开元寺 + 洛阳桥 + 安平桥** 等串联、为联合国教科文组织"宋元中国的世界海洋商贸中心"世界文化遗产（2021）——**但绝大部分遗存是宋元时期的、不是盛唐 755 的物理形态**。

**民国至今**　**今泉州港**位于泉州市丰泽区 + 鲤城区一带、晋江入海口南北两岸；**后渚港**今仍为渔港 + 小型货港；**法石**今为遗址；**晋江县治**今为泉州市鲤城区；**唐代泉州港的物理形态**——大半为后世港务变迁所湮、仅"后渚港""法石"等地名仍存。**今游人重访泉州一带**——所见的是**宋元泉州的海洋商贸遗存 + 后世的闽南文化层叠**——盛唐 755 一刻那处置州才 44 年的福建新港、那些晋江入海口南北两岸的散落码头、那几艘自广州或江南来的中转海舶、那一二处粗具雏形的蕃客馆驿——皆已不可见。**唐人那一刻看到的泉州港**——是一处尚未启动的早期海贸节点、是一片福建沿海的年轻港埠；那种"未启动"的清淡气象、是 755 一刻泉州港独有的、亦是后世（宋元繁华以后）无法复现的。
