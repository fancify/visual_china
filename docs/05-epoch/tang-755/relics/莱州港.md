# 莱州港

## 一句话

**胶东半岛西部的渤海湾口**——莱州港在河南道莱州/东莱郡治掖县的东海岸（今山东烟台莱州市一带、临今莱州湾），与登州港（蓬莱）同在胶东半岛、合称胶东两海口；**登州偏东北、临黄海、北航日本朝鲜 + 渡渤海至辽东**；**莱州偏西、临渤海湾、北航辽东半岛 + 渤海湾沿岸 + 渤海国**——是 755 时唐廷与**渤海国**（698-926）+ **辽东 / 平卢节度**沿海联络的主要海口之一；755 时正名"东莱郡"（天宝元年/742 改）、治掖县（汉以来旧县）；**行政属河南道；与平卢节度仅有海路联动（平卢治营州、765 后才南迁青州）**——755 十一月安禄山起兵后平卢节度由侯希逸守、又自营州南渡海至青州、莱州港一带正是其海路撤退的物理走廊。

## 形貌

莱州港在 755 时——**胶东半岛西岸的官设海口 + 民间渔泊**。

**地理位置**：**河南道莱州/东莱郡治掖县**——治在今山东烟台莱州市一带；州治东行至海约二十至三十唐里、即莱州港所在；**莱州湾**（今山东沿海西部 + 渤海湾南缘）三面陆地、唯北面开向渤海湾；**胶东半岛西部**——与登州（蓬莱、在半岛北端）相距约一百五十唐里、与青州（北海郡治、半岛西邻）相距约一百八十唐里；**临渤海湾**——隔海北望辽东半岛（约三百唐里）+ 东北望渤海国南境（约八九百唐里、自登州出海者亦取此道）+ 西北望河北道沧 / 棣州海岸。

**港口设施**：盛唐 755 时莱州港——**官设海口 + 民间渔泊**并立。官设海口属**河南道莱州**辖、**东莱郡太守，或长史 / 司马**督；**津吏 + 海舶 + 渔舟 + 行旅**为常态；唐廷与渤海国 / 辽东的官使、民间海商的鱼盐贸易 + 客商往返、皆于此换舶。**津渡通制**：《唐六典》卷七·水部郎中员外郎条载关津水驿之制——三十里一水驿、海口与漕路同样设管。但 **海舶非内河舟**——船型大、吃水深、出海须候风候潮、不同于江河漕渡的密集时刻；莱州港的实际节奏是**春夏出海 + 秋冬避风**——盛唐时北方海船受信风约束、年有定时。

**周边水道与海道**：
- **渤海湾**——北望辽东半岛（旅顺、都里镇一线）；唐时**都里镇**（今辽宁大连旅顺一带）为渤海湾北岸第一登陆点；**自莱州港出海北航辽东** + 沿辽东半岛东行 + 越鸭绿江口 + 沿朝鲜半岛西海岸北上 + 至渤海国南境（今朝鲜咸境 / 中国吉林一带海岸）——是唐 ↔ 渤海国主要海路；
- **登州海口（蓬莱）**——东北一百五十唐里、胶东半岛北端、临黄海；登州偏北航 + 莱州偏西航、两港分工 + 互补；
- **掖水（今胶莱河北段 + 掖水诸支）**——自掖县境内出、东北流入海；莱州港即在掖水入海口附近的海岸；为内陆漕物经掖县至港的水道；
- **莱州湾沿岸沙嘴 + 浅滩**——海舶须循深泓出湾、岸边多设标识（古谓"望斗"）。

**物候**：莱州港一带——**暖温带季风、四时分明**；春多东风 + 海雾；夏多霖 + 海风急 + 暑湿；秋高气爽 + 海舶北上之时（北方信风顺）；冬有薄冰 + 港口结冰（盛唐时莱州湾冬季可结薄冰、海舶停航近三月）。**渤海湾的航季**——**春末至秋初为主要出航期**、冬季 + 春初停航；唐与渤海国的官使、民间海商皆循此节奏。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

莱州港在唐人的回望里——**是汉以来胶东西部的旧海口 + 与辽东渤海湾沿岸通海的物理走廊**。

**州县史的回望**：
- **汉**——掖县为汉东莱郡治（西汉武帝元封间立东莱郡、治掖县）、**胶东半岛西部的核心县**；
- **三国魏 + 西晋**——东莱郡仍治掖县、属青州；
- **东晋十六国**——前燕 + 前秦 + 后燕诸朝迭领；
- **南北朝**——北魏立光州（治掖县）、东莱郡为属郡；
- **隋开皇五年（585）**——**改光州为莱州**、治掖县——**莱州**之名自此始；隋大业三年（607）改州为东莱郡；
- **唐武德四年（621）复莱州**——治掖县；**天宝元年（742）改莱州为东莱郡**（汉旧名）；**乾元元年（758）复莱州**——**755 时正名"东莱郡"、治掖县**；行政文书写"东莱郡掖县"，民间俗称"莱州""掖城"皆可；州境领掖、昌阳、即墨、胶水四县（《新唐书·卷三十八·地理志二》东莱郡条）。

**海路史的回望**：
- **汉武帝东巡 + 楼船军征朝鲜**——元封二年（前 109）汉武帝遣楼船将军杨仆自齐地浮渤海至朝鲜半岛、立四郡（乐浪 + 玄菟 + 真番 + 临屯）；楼船军出海口当在齐地东莱一线、莱州港一带即为先声；
- **三国魏**——青州刺史辖东莱、北航辽东（公孙渊据辽东时双方海路往来）；
- **南北朝**——北魏 + 北齐皆以莱州为北航之口；
- **隋**——**隋大业八年至十年（612-614）三征高句丽**——隋炀帝命水军自东莱出海、来护儿 + 周法尚率舟师自东莱浮海至浿水（今朝鲜大同江）——**东莱即莱州、莱州港在此役中是隋水军总出海口**；
- **唐**——**贞观十九年（645）唐太宗征高句丽**——张亮率水军自东莱浮海至卑沙城（今辽宁大连金州一带）；**总章元年（668）唐高宗灭高句丽**——刘仁轨等水军亦自登 / 莱出海；**自此以后唐与新罗 + 渤海国的官使海路**——多从登 / 莱二港出入，**登州港**承新罗 + 日本一线、**莱州港**承辽东 + 渤海国一线（**实际两港在不同时期皆兼办**、不绝对分工）。

**渤海国（698-926）的成立与入唐道**：
- **武则天圣历元年（698）**——**大祚荣**（高句丽遗民 / 粟末靺鞨）于东牟山（今吉林敦化一带）建震国、自立为震国王；
- **唐玄宗先天二年（713）**——遣崔忻册大祚荣为**渤海郡王**、**忽汗州都督**——此时正式为唐羁縻；自此**渤海国**之名定（《旧唐书·卷一九九下》北狄·渤海靺鞨条）；
- **武王大武艺（719-737）+ 文王大钦茂（737-793）**——渤海国向唐通使频繁；都城由旧国（敦化）迁至**上京 龙泉府**（今黑龙江宁安渤海镇）；**755 时正值文王大钦茂在位（737-793）**——是渤海国的全盛期、年与唐遣使数次；
- **入唐道**——渤海国遣使入唐主要走两道：**陆道**经辽东 + 营州（平卢节度治）+ 范阳 + 入两京；**海道**自渤海国南境海岸出海、沿辽东半岛东岸南下 + 越鸭绿江口 + 至辽东半岛南端都里镇 + 横渡渤海湾至**莱州 / 登州登陆** + 陆行入两京——**莱州港在 755 时正是渤海国海路入唐的主要登陆港之一**。

**平卢节度的形势**：
- **开元七年（719）置营州都督府**——治营州（今辽宁朝阳）、辖东北诸羁縻州；
- **开元二十八年（740）置平卢军节度使**——治营州、辖营 + 平 + 卢龙诸州 + 兼押两蕃（奚 + 契丹）+ 渤海 + 黑水靺鞨四府经略使；
- **天宝间**——平卢节度由**安禄山**遥领（兼范阳 + 河东三镇）；
- **平卢节度与莱州港**——莱州虽属河南道、行政上不隶平卢；但海路上**莱州港 + 登州港**是平卢节度（辖辽东）+ 渤海国 + 唐两京间的主要海路接点；**755 时正是平卢节度的全盛期、莱州港的渤海湾航运亦因之繁忙**。

### 此刻

天宝十四载这一年的莱州港——**按例运行、海舶北航、唐与渤海国官使往来、即将经历安史浩劫**。

**港口状态**：盛唐 755 时莱州港——官设海口 + 民间渔泊并立；春夏海舶接连出航 + 秋冬避风停泊；津吏验牒、海商纳钱、渤海国使臣或唐使节渡海登陆者偶至。**官道接陆**——自莱州港西行二十余唐里至掖县城（莱州治）+ 再西行经青州（北海郡治）+ 至兖州 + 至洛阳——是渤海国 / 辽东客商入两京的陆道延伸。

**与登州港的分工**：**登州港**（蓬莱、胶东半岛北端、临黄海）——主承**新罗 + 日本**一线、亦兼办渤海国海路；**莱州港**（掖县、胶东半岛西部、临渤海湾）——主承**辽东 + 渤海国** + 渤海湾沿岸（沧 / 棣等州）一线；**两港在不同时期与不同政治形势下职能略有调换**、文献未严格区分（《新唐书·渤海传》仅言"自登州东北海行"、未细分登 / 莱）；但**地理上**——莱州偏西更近渤海湾内陆 + 登州偏北更近黄海开洋——**莱州航辽东更便、登州航日本朝鲜更便**——大致格局。

**渤海国海路的实际**：**755 时正值渤海文王大钦茂在位（737-793）**——是渤海国与唐通使的高频期；**渤海国遣使入唐**——海路自渤海国南境海岸出海 + 沿辽东半岛东岸南下 + 越鸭绿江口 + 至都里镇 + 横渡渤海湾 + 至莱州 / 登州登陆 + 陆行入两京；**唐遣使赴渤海**——亦循反道；**民间海商**——以盐 + 丝 + 布 + 米 + 海产（昆布 + 海藻 + 海贝）+ 北珠 + 渤海特产（貂皮 + 鹰）等为商品；**莱州港是这一海路的南端登陆口之一**——其繁忙程度依年份与政治形势而异、文献未载具体数字。

**唐人对莱州港的认知**：莱州港是**北航辽东 + 渤海湾沿岸要冲**；江北赴辽东 + 渤海国必经莱州 / 登州出海；辽东 + 渤海国客使入唐亦经此登陆。**唐人尚不以"渤海湾航线"为术语**——而以"北航辽东""自东莱浮海""自登州东北海行"等具体路名指称（《新唐书·渤海传》："其北界曰栅城府、为东京、龙原府……自登州东北海行……"——可见登州是文献中渤海国海路的主要标识、莱州为伴行港）；755 时莱州港尚未入唐诗、是纯然的官道海口 + 商旅渔泊。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九/丙寅即起）；**安禄山兼领平卢 + 范阳 + 河东三镇**——平卢军本部于营州、但安禄山叛起后**平卢节度由侯希逸代守**（《旧唐书·卷一二四·侯希逸传》）；**永泰元年/765 前后南迁青州**——侯希逸率平卢军残部**自营州南渡海至青州**、走的正是辽东 → 莱州湾沿岸 → 青州一线；**莱州港在此一役中是平卢军海路撤退的物理走廊** + 唐廷河南道东端最重要的物资集散点之一；**永泰元年/765 前后南迁青州**之后——平卢军已南徙、营州沦于奚 / 契丹、**辽东海路自此衰**——莱州港的繁忙骤减。

### 此刻意味

对天宝十四载的东莱郡掖县人民来说——莱州港是**自家东郊的海口**：日常海舶出入 + 渔泊营生 + 渤海国客商偶至。对河南道东端的人民来说——莱州港是**北航辽东的第一站**：商旅货物经此出海、北上渤海湾。对盛唐唐廷来说——莱州港是**与渤海国 + 辽东海路联络的物理 anchor 之一**——与登州港并立胶东两海口、共同支撑盛唐东北海防 + 海贸 + 外交。

唐人尚不知道——天宝十四载十一月丙寅安禄山将于范阳起兵、十二月丁酉将陷东京洛阳；平卢军本部营州亦将动荡；**永泰元年/765 前后南迁青州**——侯希逸将率平卢军残部自营州南渡海至青州、莱州港一带正是其海路撤退的物理走廊；其后平卢军南徙青州、营州沦于奚 / 契丹、**辽东海路自此衰**；**中晚唐莱州港仍存**——但与渤海国的海路日渐稀少、转为以登州港为主；**渤海国大彝震（830-857）以后**——与唐海路通使更倚登州、莱州港的渤海湾航运地位渐为登州取代；**唐末五代以后**——海岸线变迁（莱州湾西部岸线北退 + 沙嘴变迁）+ 内陆交通改道 + 海上贸易重心南移至明 / 泉 / 广，**莱州港作为大港的角色自此衰退**——但作为渤海湾沿岸渔港 + 鱼盐市仍存至明清。

**今游人重访莱州港一带**——所见的是**山东烟台莱州市**（1988 年撤掖县置莱州市；**掖县**之名自汉至 1988 年共行二千余年）+ **莱州湾**（今为渤海三大海湾之一、海岸线略有北退）+ **盛唐莱州港的物理位置**已因岸线变迁略移、今莱州市东海岸的太平湾 / 三山岛一带或为其大致位置；**渤海湾航线**早已转为现代海运 + 莱州港仍为今山东重要港口（今"莱州港"为现代规划港口、与盛唐莱州港地理位置接近但非同一物理设施）。

## 子条目

### 莱州（东莱郡）的州县沿革

**州县沿革**：
- **汉武帝元封间立东莱郡**——治掖县；
- **三国魏 + 西晋**——东莱郡仍治掖县；
- **北魏立光州**——治掖县；
- **隋开皇五年（585）改光州为莱州**——治掖县；隋大业三年（607）改州为东莱郡；
- **唐武德四年（621）复莱州**——治掖县；**天宝元年（742）改莱州为东莱郡**；**乾元元年（758）复莱州**；
- **元 + 明 + 清**——莱州 / 莱州府皆治掖县；
- **民国 + 中华人民共和国**——掖县属山东；**1988 年撤掖县置莱州市**（县级、由烟台代管）——**掖县**之名自汉至 1988 年共行二千余年；2026 今为山东烟台莱州市。

**领县**：755 时莱州/东莱郡领**掖、昌阳、即墨、胶水**四县（《新唐书·卷三十八·地理志二》东莱郡条；户二万六千九百九十九、口十六万八千五十九）；**掖县**为州治（汉以来旧县）；**即墨**在半岛南部、临今胶州湾（即墨港后来另成一海口、但唐时尚不显）；**胶水**因胶水（今胶莱河）得名；**昌阳**在掖县南。

### 莱州港（掖县东海岸的官设海口）

**港口的形成**：**汉武帝东巡 + 楼船军征朝鲜（前 109）**——东莱即为楼船军出海口；**隋大业八年至十年（612-614）三征高句丽**——隋炀帝命水军自东莱出海、来护儿 + 周法尚率舟师浮海至浿水——**东莱即莱州、莱州港在此役中是隋水军总出海口**；**唐贞观十九年（645）唐太宗征高句丽**——张亮率水军自东莱浮海至卑沙城；**自此盛唐莱州港作为北航辽东 + 渤海国的官设海口完备**。

**港口的设施**：盛唐 755 时——官设海口 + 民间渔泊并立；津吏 + 海舶 + 行旅 + 渤海国客使为常态；春夏出航 + 秋冬避风。**港口费用**：**《唐六典》卷七·水部郎中员外郎条**载关津水驿之制——海口与漕路同样设管；商旅按舶纳钱、官差持牒免费。

**港口的地位**：莱州港是**胶东半岛西部的渤海湾口**——**与登州港分工 + 互补**：登州偏北航日本朝鲜、莱州偏西航辽东渤海湾；**755 一刻**仍按例运行、安史起后渐衰；**唐末以后辽东海路与渤海国一并衰**、莱州港的核心地位为登州 / 明州 / 泉州 / 广州所取代。

### 渤海国（698-926；唐羁縻 + 海路通使）

**渤海国沿革**：
- **武则天圣历元年（698）**——**大祚荣**（高句丽遗民 / 粟末靺鞨）于东牟山建震国、自立为震国王；
- **唐玄宗先天二年（713）**——遣崔忻册大祚荣为**渤海郡王**、**忽汗州都督**——自此正式为唐羁縻；**渤海国**之名定；
- **武王大武艺（719-737）**——开拓疆土、与唐时有摩擦（开元二十年/732 渤海袭登州事）+ 时复通使；
- **文王大钦茂（737-793）**——**755 时正值在位**——是渤海国与唐通使的高频期；都城由旧国（敦化）迁**上京 龙泉府**（今黑龙江宁安渤海镇）；又一度迁**中京 显德府**（今吉林和龙西古城）；
- **宣王大仁秀（818-830）+ 大彝震（830-857）**——渤海国全盛、号"海东盛国"（崔致远《入唐表》）；
- **辽天显元年（926）**——辽太祖耶律阿保机灭渤海、改称东丹国；渤海国终。

**渤海国与莱州港**——海路入唐主道之一；自渤海国南境海岸出海 + 沿辽东半岛东岸南下 + 越鸭绿江口 + 至都里镇 + 横渡渤海湾 + 至莱州 / 登州登陆 + 陆行入两京；**755 时正值渤海国 - 唐通使高频期、莱州港是这一海路的南端登陆口之一**。

### 平卢节度 + 营州 + 都里镇（渤海湾北岸 + 莱州海路对岸）

**平卢节度**：**开元二十八年（740）置平卢军节度使**——治**营州**（今辽宁朝阳）；辖营 + 平 + 卢龙诸州 + 兼押两蕃（奚 + 契丹）+ 渤海 + 黑水靺鞨四府经略使；**天宝间由安禄山遥领**——是安禄山三镇之一；**安史起后平卢军本部于营州**——由**侯希逸**代守；**永泰元年/765 前后南迁青州——侯希逸率平卢军残部自营州南渡海至青州**——是辽东海路撤退的实例；**永泰元年/765 前后南迁青州**、**营州沦于奚 / 契丹**——辽东海路自此衰。

**都里镇**：**今辽宁大连旅顺一带**——渤海湾北岸第一登陆点；**唐时为辽东半岛南端的海口**；自莱州港浮海北航三百唐里至都里镇、再陆行或循辽东半岛东岸北上至渤海国 / 营州——是莱州港的对岸接点。

### 与登州港的分工（胶东两海口）

**地理与职能分工**：
- **登州港（蓬莱）**——胶东半岛北端、临黄海；**北航日本朝鲜**为主（自登州东南海行至新罗 + 自登州东海行至日本）+ 兼办渤海国海路；
- **莱州港（掖县）**——胶东半岛西部、临渤海湾；**北航辽东 + 渤海湾沿岸**为主（自莱州北航三百唐里至都里镇 + 沿辽东半岛东岸至渤海国）+ 兼办新罗 + 日本一线（但实际较少）；
- **文献的提法**——《新唐书·渤海传》"自登州东北海行"——可见登州为文献中渤海国海路的主要标识；但**实际航线**——海舶自莱州港出洋 + 沿渤海湾北上者亦多；**两港在不同时期 + 不同政治形势下职能略有调换**、文献未严格区分。

**两港同属河南道 + 海路上与平卢军镇相接 + 同为唐廷东北海防 / 海贸 / 外交的物理枢纽**——可视作胶东两海口的整体。

## 历史长河（755 → 今）

**中晚唐**　**755-763 安史乱中**——平卢军本部于营州动荡、海路撤退之议渐起；**永泰元年/765 前后南迁青州**——侯希逸率平卢军残部自营州南渡海至青州、走莱州港 + 渤海湾沿岸海路撤退；其后平卢军南徙青州、营州沦于奚 / 契丹、**辽东海路自此衰**——莱州港的繁忙骤减；**中晚唐莱州港仍存**——但与渤海国的海路日渐稀少、转为以登州港为主；**晚唐 9 世纪**——日僧圆仁《入唐求法巡礼行记》数次记登州 + 文登 + 赤山诸海口、但莱州港未单记——可见莱州港此时已退居次要。

**五代 + 北宋**　**辽天显元年（926）辽太祖灭渤海**——东北海路全断；**五代乱中胶东沿海多盗**——莱州港更衰；**北宋**——莱州属京东东路、莱州港仍为渤海湾沿岸鱼盐市 + 渔泊；与契丹（辽）+ 女真（金）的海上来往多自登州 + 密州、莱州港非主道。**金 + 元**——莱州为山东东路 / 般阳路、海口转衰。

**明清**　**明洪武年间筑莱州城**——治掖县；**明清间莱州为山东沿海重镇** + 备倭城之一；但**海运重心南移至明州 / 泉州 / 广州**——莱州港仅为地方鱼盐市 + 渔泊；**清雍正间升莱州府**——治掖县；**清末 + 民国**——莱州港作为大港的历史已终。

**民国至今**　**1988 年撤掖县置莱州市**（县级、由烟台代管）——**掖县**之名自汉至 1988 年共行二千余年；**今莱州市**为山东烟台代管县级市、临莱州湾；**今"莱州港"**——为现代规划港口（位于莱州市西北的太平湾 + 三山岛一带）、与盛唐莱州港地理位置接近但非同一物理设施；**渤海湾航线**早已转为现代海运（青岛 / 烟台 / 天津诸大港）+ 莱州港仅为山东沿海中型港口。**今游人重访莱州港一带**——所见的是莱州湾 + 现代港口 + 莱州市城区（盛唐掖县大致位置）；**盛唐莱州港的物理形态**已因岸线变迁 + 海运重心转移而难寻——但作为渤海湾沿岸古海口的地理走廊仍在。
