# 平城

## 一句话

**北魏前期都城**——拓跋鲜卑入主华北的物理 anchor；**天兴元年（398）拓跋珪自盛乐迁都平城、至太和十八年（494）孝文帝迁洛阳——为北魏都 96 年**；**755 时旧都已废 261 年**——以**云州云中郡治云中县**身份存在（属河东节度），平城旧城大半坍废、宫城基址犹立、方山永固陵 + 武州山云冈石窟仍在城西北、是盛唐北方塞下"代北旧都"最完整的物理遗存。

## 形貌

平城在唐河东道**云州云中郡治云中县**——具体在**桑干河北岸、武州山东南**（今山西省大同市操场城一带）。**地势**：山西最北的一片高平地——海拔约 1050 米；**南面隔桑干河望恒山余脉**、**北面是阴山南麓的丘陵草原**、**西面是武州山（云冈石窟所在）**、**东面是御河谷**——是 4 世纪末拓跋鲜卑南下时选中的天然要塞 + 农牧过渡地。

**北魏平城的城制**——史载分**外城 + 京城 + 宫城**三重：
- **外城**：周回约 32 里（《水经注·㶟水》）——东西约 6500 米、南北约 6500 米；
- **京城（内城）**：方约 9 里——是百官宅第 + 礼制建筑所在；
- **宫城**：在京城北部——含太极殿 + 西宫 + 北宫 + 永安殿等北魏宫廷主体建筑。

**宫城遗址**：今大同**操场城**——夯土城墙残存、东西约 1800 米、南北约 1500 米；**是北魏宫城基址**。**1980s-2010s 山西省考古研究院多次发掘操场城**——确认北魏宫城范围、出土北魏瓦当（含"大代万岁" + "传祚无穷"瓦当）、陶器、墓志等。

**北魏平城周边的礼制 + 陵区**：
- **武州山（今云冈）**：平城西约 16 公里——**云冈石窟所在**；
- **方山**：平城北约 25 公里——**冯太后永固陵 + 孝文帝预营永泰陵基址**；
- **明堂 + 灵台 + 辟雍**：平城南郊——北魏礼制建筑群（孝文帝太和十年/486 建）；
- **白登山**：平城东北——汉高祖被匈奴困之处，北魏时为皇室祭祀地。

物候：山西最北、温带大陆性气候——干燥、风大、冬寒夏凉；春秋多沙尘、冬季严寒——是北方塞下的代表气候。农牧过渡地带、宜麦黍 + 牲畜放牧。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

平城在唐人的回望里——是**北魏前期的都城 + 拓跋鲜卑入主华夏的物理 anchor**。

**拓跋鲜卑入主**：**北魏太祖道武帝拓跋珪（371-409）**——鲜卑拓跋部首领；**登国元年（386）于盛乐（今内蒙古和林格尔）立代国（改国号北魏）**；**天兴元年（398）自盛乐迁都平城**——是拓跋鲜卑从草原部族向中原王朝转型的物理标志。**自此北魏以平城为都 96 年**。

**北魏前期六帝在平城**（398-494，96 年）——道武、明元、太武、文成、献文、孝文六帝皆在平城即位 + 主政（孝文帝 494 迁洛、499 崩，最后约 5 年在洛阳）：
- **道武帝拓跋珪**（在位 386-409）：开国、迁都平城、定礼制；
- **明元帝拓跋嗣**（409-423）：南拓中原、克黄河；
- **太武帝拓跋焘**（423-452）：**太延五年（439）灭北凉 + 统一北方**——是北魏最辉煌的时刻；亦行"太武灭佛"（446-452）；
- **文成帝拓跋濬**（452-465）：复佛、**和平元年（460）始凿武州山石窟（云冈）**；
- **献文帝拓跋弘**（465-471）：传位孝文帝、自为太上皇；
- **孝文帝拓跋宏**（471-499）：**太和十八年（494）迁都洛阳**——平城作为都的终结。

**北魏的鲜卑化 vs 汉化张力**——是北魏 96 年平城都期间的核心政治主题：
- **太武帝时期**：以鲜卑武力定天下、灭北凉、统一北方；
- **冯太后摄政期（466-490）**：开始全面汉化——均田制、三长制、班禄制等；
- **孝文帝亲政后**：**太和十七年（493）下诏迁都**——**太和十八年（494）正式迁洛**；**改鲜卑姓为汉姓**（拓跋→元、独孤→刘 等）、**易胡服为汉服**、**禁鲜卑语**——是中国历史上最有名的胡族汉化政治改革。

**孝文帝迁洛之后**——**平城自此自都退为代北边镇**：
- **太和十八年（494）后**——平城所在司州改置为恒州；洛阳一带改为司州（京畿）；平城为北魏"北京"陪都；
- **北魏后期六镇之乱（524-525）**——平城为乱军反复争夺地、几被破；
- **东魏 / 北齐时**（534-577）——平城为北齐恒州治；
- **北周灭北齐（577）后**——平城所在为北周恒州；
- **隋初（开皇中）**——改置云州；
- **隋大业 + 唐初**——平城为云州治所、但城内已大半坍废。

**唐人对北魏的总体印象**：盛唐人凡读《魏书》（北齐魏收撰）+ 《北史》（唐李延寿撰）者，皆知**平城作为北魏前期都城的 96 年**；具体到北魏六帝、孝文帝迁洛、太武灭北凉、文成凿云冈 等大事——士人通晓。**唐人未必对北魏六帝皆有具体感**——但**孝文帝迁洛 + 太武灭北凉 + 云冈石窟**是盛唐士人共通的北朝记忆。

### 此刻

天宝十四载这一年的平城——**已废为代北旧都 261 年（494 孝文帝迁洛→755）**；以**云州云中郡治云中县**身份存在（属河东节度）。

**云州的唐廷建制**：**贞观四年（630）唐廷置云州**于平城旧址；**永淳元年（682）突厥扰边、云州一度废**；**开元十八年（730）复置云州**——天宝元年（742）改云中郡、**乾元元年（758）复称云州**。**755 时云州属河东节度**——河东节度治太原、节度使范围含云州、朔州、代州、忻州等北方边州。**云州的边军职能**：北御契丹 + 突厥后部 + 回纥；屯田、戍守、互市——是盛唐北方边防的重要一环。

**平城旧城状态**：盛唐 755 时——
- **外城（周回 32 里）**：大半坍废、夯土墙残存仅段落；
- **京城（方 9 里）**：已渐为云州城所占——云州治所即在平城内城基址；
- **宫城（操场城）**：北魏太极殿、西宫 + 北宫 等主体建筑早废——但**部分夯土基址仍立**；
- **明堂、灵台、辟雍**（北魏礼制建筑）：早废、唐人 755 时只见土堆；
- **方山永固陵**（冯太后陵）：盛唐 755 时陵冢仍立、地表景观未大变；
- **武州山云冈石窟**：盛唐 755 时石窟仍存——但**自孝文帝迁洛后（494）已无新凿、寺院渐衰**；唐时仍有僧人居云冈、规模远不及北魏鼎盛期。

**唐人对平城的认知**——**"代北旧都"**：唐诗、唐人散文中"代北 / 平城 / 云中"是边塞文学的高频意象；**盛唐边塞诗**中"代北"二字出现密集：
- **王维**（约 701-761）有边塞诗涉云中；
- **高适**（约 704-765）《燕歌行》等边塞诗涉北边；
- **岑参**（约 715-770）多有北边诗——但岑参主写西域、对代北的具体提及较少；
- **后世李益、卢纶 等中晚唐边塞诗**——多咏代北、云中。

**唐人对北魏的"长安记忆"**：盛唐人凡至云州者多有怀古——但**唐人对北魏的具体感** 远不及对汉、隋的具体感（北魏在唐人心中是"前朝之前的胡朝"）。**孝文帝迁洛 + 全面汉化**——是盛唐士人最常引用的北魏典故；**李唐皇室祖出武川**（北魏六镇之一）——盛唐士人通读《周书》《北史》者皆知。

**这一年的平城** 处于**云州州治 + 北魏旧都遗址 + 边军屯戍**的稳定状态——再过几个月安禄山起兵于范阳；**至德二载（757）唐军反攻**——云州一带反复易主；**广德元年（763）后吐蕃 + 回纥渐扰**——云州为唐廷北边争夺地之一。

### 此刻意味

对天宝十四载的云州人来说——平城就是**自家州城的旧底**：北魏前期 96 年的都城基址、夯土残墙立于桑干河北岸、宫城基址 + 方山陵 + 云冈石窟皆在视线之内。对长安那一代士人来说——平城是**北魏前期 96 年的物理纪念**：拓跋鲜卑从盛乐南下、入主中原、定都平城、统一北方、最终汉化迁洛——这一段 96 年的政治史**全部以平城为舞台**。**鲜卑入主华夏的物理 anchor**——是平城最特殊的政治分量。

但**唐云州不是"鲜卑都城"**——唐云州是**盛唐边军镇守 + 农牧过渡地的州治**；旧平城的政治记忆已被埋入云州的边防日常。**两重身份的张力**——是平城在 755 时最深的意味：既是 4-5 世纪鲜卑入主华夏的物理 anchor，又是 8 世纪盛唐边军戍守的塞下州治。**这种张力以物理形式存在**：操场城夯土基址 + 云冈石窟 + 方山永固陵 + 唐云州城——四者皆在同一片平地上。

唐人尚不知道——再过千二百年（20 世纪），**云冈石窟列联合国教科文组织世界文化遗产**（2001）；**北魏平城遗址**成为研究 4-6 世纪胡汉合流的核心实物 anchor。**唐人在 755 所见的代北旧都**——千二百多年后**主体已为大同市覆盖**、但**操场城（宫城基址）+ 方山永固陵 + 云冈石窟**——三处北魏遗存犹存、且皆列国家级重点文物保护单位。**唐人在云州城内所见的北魏夯土残墙** 多已不复——但**云冈石窟的北魏造像** 千二百年未易、是 4-5 世纪鲜卑入主华夏最完整的物理纪念。

## 子条目

### 道武帝迁都平城（天兴元年/398）

**道武帝拓跋珪（371-409）**——鲜卑拓跋部首领、北魏开国君主；**登国元年（386）于盛乐（今内蒙古和林格尔）立代国、改国号北魏**；**皇始元年（396）入河北、克邺城**；**天兴元年（398）七月自盛乐迁都平城**——是拓跋鲜卑从草原部族向中原王朝转型的物理标志。**迁都的政治背景**：（1）盛乐在阴山以北、不利于统治中原；（2）平城在阴山以南、桑干河北岸——既近草原又近中原、是农牧过渡地；（3）道武帝以"开万世之基" 推动迁都。**自此北魏以平城为都 96 年**（398-494）。

### 太武帝灭北凉 + 统一北方（太延五年/439）

**太武帝拓跋焘（408-452）**——北魏第三代皇帝、最有军事才能的鲜卑君主；**始光四年（427）攻克统万城 + 灭大夏**；**太延二年（436）灭北燕**；**太延五年（439）灭北凉沮渠氏 + 统一北方**——是北魏 96 年平城都期间最辉煌的时刻。**自十六国之乱（304-439）135 年后**——华北重归一统、由北魏主之。**太武帝的鲜卑武力定天下**——是平城作为都城最有政治分量的一段。

**太武帝亦行"太武灭佛"**（太平真君七年/446 至正平二年/452）——是中国历史上最早的大规模灭佛运动；其子文成帝即位后复佛、并**和平元年（460）始凿云冈石窟** 以补灭佛的礼制损失。

### 文成帝 + 昙曜五窟 / 云冈石窟（和平元年/460 始凿）

**北魏文成帝拓跋濬（440-465）**——太武帝之孙；即位后复佛、扶植佛教。**和平元年（460）沙门统昙曜奏请凿武州山石窟**——即**云冈五窟（昙曜五窟，今 16-20 窟）**：一说象征道武以来五位帝/追尊帝的皇家谱系——是中国早期最大规模的皇家石窟造像。

**云冈石窟的开凿延续至孝文帝迁洛（493）前**——主体工程于**孝文帝迁洛前完工**；**迁洛后云冈石窟规模渐冷落**——但部分续凿至北魏末。**云冈现存洞窟 252 个、造像约 5.1 万尊**——是中国早期石窟艺术的代表 + 北魏前期佛教的物理 anchor。

**注**：云冈石窟的"昙曜五窟" 是文成帝时（460）始凿、非孝文帝时；本 doc 严守"460 始凿、孝文帝迁洛前主体完工"口径。

### 冯太后 + 方山永固陵（太和十四年/490 葬）

**冯太后（文明皇后冯氏，441-490）**——文成帝皇后、北魏摄政 20 余年（466-490 主政）；是北魏前期最强势的女政治家。**冯太后的政治遗产**：
- **均田制**：太和九年（485）颁——影响中国土地制度千余年；
- **三长制**：太和十年（486）颁——以邻、里、党三长为基层组织；
- **班禄制**：太和八年（484）颁——官吏始有俸禄；
- **培养孝文帝**：自孝文帝即位（471）起冯太后亲教 + 主政；孝文帝亲政后的全面汉化改革——多承冯太后基业。

**冯太后崩 + 永固陵**：**太和十四年（490）九月冯太后崩**、年 49；**同年葬方山永固陵**——位于平城北约 25 公里的方山（今大同市方山）。**永固陵依山为陵**——是北魏前期皇家陵制；**与"方山思远佛寺"** 配合——是冯太后陵区礼制建筑群的代表。

**孝文帝预营永泰陵**：太和十四年（490）冯太后崩后、孝文帝预营自身陵于永固陵之西——号"永泰陵"；**但孝文帝太和二十三年（499）崩时已迁洛 5 年、葬洛阳长陵**；**永泰陵基址为虚陵 + 未完成**——史载零散。**注**：本 doc 严守"孝文帝实际葬洛阳长陵、永泰陵为预营未葬之虚陵"口径。

**方山永固陵 1976 年发掘**：山西省考古研究院发掘永固陵地宫——出土北魏石椁 + 部分陶器 + 墓志——是北魏前期皇陵唯一一次科学发掘。

### 孝文帝迁洛阳（太和十八年/494）

**孝文帝拓跋宏（467-499）**——北魏第六代皇帝、最有名的汉化改革者。**太和十七年（493）八月**孝文帝以"南伐萧齐"为名南下、至洛阳后下诏迁都；**太和十八年（494）正式迁都洛阳**。**迁都的政治背景**：（1）平城在阴山以南、北方边境——不利于统治中原；（2）孝文帝（自身鲜卑、汉化深）以"开万世之基" 推动迁都 + 全面汉化；（3）冯太后摄政期已奠定汉化基础——孝文帝亲政后推向全面。

**迁洛后的全面汉化**：
- 改鲜卑姓氏为汉姓（拓跋→元、独孤→刘 等）；
- 易胡服为汉服、禁鲜卑语；
- 定郊祀礼于洛阳南郊；
- 诏迁洛后诸帝皆葬邙山、不归平城旧陵——是迁都礼制的最终。

**迁洛后的平城**——自此从都城退为代北边镇；**北魏后期六镇之乱（524-525）** 中平城几被破。**孝文帝迁洛**——是平城作为都城的物理终结、亦是北魏中央政权全面汉化的物理起点。

### 北魏六帝在平城（398-494, 96 年）

**北魏前期六帝皆在平城即位 + 主政**：
- **道武帝拓跋珪（386-409）**：开国、迁都平城、定礼制；
- **明元帝拓跋嗣（409-423）**：南拓中原、克黄河；
- **太武帝拓跋焘（423-452）**：灭北凉 + 统一北方；
- **文成帝拓跋濬（452-465）**：复佛、始凿云冈；
- **献文帝拓跋弘（465-471）**：传位孝文帝、自为太上皇；
- **孝文帝拓跋宏（471-499，前 23 年在平城、后 5 年在洛阳）**：迁都洛阳、全面汉化。

**这六位皇帝是北魏前期 96 年的政治核心**——其中道武、明元、太武、文成、献文五帝**皆葬云中金陵或方山一带**（北魏前期皇陵在平城北）；**孝文帝是唯一葬洛阳长陵的北魏皇帝**——是迁都决心的最终物理体现。**唐人对北魏六帝皆有具体感的可能性较低**——盛唐士人通晓的主要是**太武帝灭北凉 + 孝文帝迁洛 + 文成凿云冈** 等几件大事。

## 历史长河（755 → 今）

**中晚唐**　**至德二载（757）安史中云州一带反复易主**；具体平城旧址在安史中是否有新破坏史载零散。**广德元年（763）后回纥 + 吐蕃渐扰北边**——云州为唐廷北边争夺地。**唐末沙陀崛起**——**李克用为代北沙陀首领、根据地即在云州 / 代北一带**；**唐末沙陀据云州**——是唐末"代北势力"的起点。

**五代 + 辽**　**后晋天福元年（936）石敬瑭割燕云十六州予契丹**——**云州（即平城旧址）入辽**；**辽改云州为西京大同府**——**自此"大同"之名始**。**辽代大同府是辽五京之一**——平城旧址重获政治分量；辽代续凿云冈部分洞窟、修方山一带寺院。

**金 + 元 + 明清**　**金代大同府仍为重镇**；金人多次修云冈寺。**元代大同路 + 明代大同府**——是华北北部最大的军事重镇；明代设大同总兵镇守、为九边重镇之一。**清代大同府** 渐失军事分量——为山西北部农牧贸易中心。

**民国至今**　**1949 年后大同市继承平城旧址**——**操场城（北魏宫城基址）+ 方山永固陵 + 云冈石窟**——三处北魏遗存皆列国家级重点文物保护单位。**1980s-2010s 山西省考古研究院多次发掘平城遗址**——确认外城、京城、宫城范围、出土北魏瓦当、陶器、墓志等。**2001 年云冈石窟列联合国教科文组织世界文化遗产**——是中国早期石窟艺术的代表。**今游人来大同**——所见的是**操场城夯土基址 + 方山永固陵 + 云冈石窟北魏造像**——**唐人在 755 所见的代北旧都主体已为现代大同市覆盖**；但**三处北魏遗存的物理 anchor 犹存**——是 4-5 世纪鲜卑入主华夏最完整的物理纪念。
