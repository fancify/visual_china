# 明州港

## 一句话

**唐 东海航线的江南启动点**——明州港在 江南东道明州（755 时正名余姚郡）治鄮县东海岸、甬江入海口一带；明州州本身是开元二十六年（738）才自越州东境析出的新州，至 755 年仅立 17 年——州与港俱为盛唐玄宗朝的新生事物；与南海路的广州港 + 北海路的登州港并立，**广州主南海、登州主朝鲜半岛与黄海北岸、明州主东海与日本**——盛唐三向海贸的东向起点；**但盛唐 755 时明州港的真正繁盛尚未到来**——日本遣唐使以明州为主登陆港、是中晚唐（9 世纪起、承和五年/838 小野篁那批后期才确立）的事；755 这一刻的明州港是一座**新港**、海舶虽已往来、规模远未到中晚唐"四明海舶岁岁来"的盛境。

## 形貌

明州港在 755 时——**是江南东南角一座立州才 17 年的新港，甬江出海口一段海岸**。

**地理位置**：**江南东道余姚郡治鄮县**（今浙江省宁波市鄞州区一带）东海岸——自鄮县城东北行约数十唐里即至海；**州治鄮县城**压在四明山东北麓与杭州湾南岸之间的一片冲积台地上；**州城西南**为四明山（主峰金钟峰一千余米；山势虽不甚高、雨水充沛、是浙东青瓷与名茶产山）；**州城北 + 东**是水网平原——**奉化江、姚江两流自西南、西北合于州城东北**、其合流处后世名"三江口"（合流形态唐时已成、但"三江口"作宁波地标的写法是宋以后事）；**自合流处东注入海、即甬江**——一州之水皆归此一江入东海。**明州港**——即在甬江入海口一带的海岸；**距越州治山阴县**（今绍兴）约二百唐里、**距杭州治钱塘县**（今杭州市区）约三百唐里。

**港的物理形态**：**唐 755 时明州港尚无后世意义上的城港分立**——州治鄮县是行政中心、港埠是州东北海岸一线的若干登陆点；**主登陆点**约在甬江入海口南北两岸（今宁波镇海、北仑一带）；**港埠**——海舶下锚处水深 + 港湾内可避东南风 + 沙岸便于装卸；**唐时港埠规模有限**——海舶寥落、商舶多自江南河北上至扬州或南海路至广州；明州港是这两大主航线之外的第三个海路口岸。**港埠的设施**——盛唐 755 时无独立的市舶机构（**唐于明州正式立市舶机构是中晚唐至五代的事**、本 doc 不预支）；州县衙督收市税 + 关吏验牒 + 海舶下锚 + 商旅登岸——是州东境的一处通海口岸、非独立官署。

**港的腹地水道**——**甬江 + 奉化江 + 姚江三水合于鄮县城东北三江口、东流入海**；**港埠** 自三江口东行数十里至海；**州境内陆水道与港埠通联**——海舶卸货后可沿甬江上溯至鄮县城下、再分入奉化江 + 姚江通往内陆；**州西南陆道通越州**——自鄮县西行经余姚县至越州治山阴县约二百唐里；**州西北水道通杭州**——循杭州湾南岸西行至杭州治钱塘县。

**港的对外海路**：盛唐 755 时明州港的海路网络——
- **北路**——沿杭州湾北岸 + 黄海西岸北上至登州（蓬莱），转登州港至山东半岛诸港；**这是盛唐 江南海舶北上的主路**；
- **东路**——自甬江入海口东渡东海至日本（**北九州博多湾**，或南九州坊津/萨摩一带）；**这是盛唐至中晚唐 东海航线南路的起点**；
- **北东路**——自明州港北上转杭州湾、出海后东北行至朝鲜半岛西南沿岸（新罗）；新罗商舶在盛唐至中晚唐间频繁往来明州 + 登州；
- **南路**——沿浙东 + 闽东海岸南下至福州 + 泉州 + 广州；**这是江南海舶南下与南海路衔接的路线**；
- 与广州港的**南海路**（自南海经马六甲海峡通天竺、波斯、大食）相对——**广州主南海、登州主朝鲜半岛与黄海北岸、明州主东海与日本**——三港分立的盛唐三向海贸格局、于明州港新立的开元天宝间渐成。

**物候**：**江南偏东南**——亚热带季风、四时分明、春雨绵绵、梅雨长、夏热伴东南风、八月台风偶至、冬不甚寒；**沿海一带咸雾偏多**——海舶出航多候清明（春）或重阳（秋）季风。**东海航行的季候规律**——**春夏东南风可东渡日本**（顺风）、**秋冬西北风可西归大唐**（顺风）——但海路凶险、东海航程动辄半月、海难屡有；遣唐使一行常船只散失、人多有不归。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

明州港在唐人的回望里——**是一座新港**。**港埠所在的鄮县是汉魏旧县**（秦始皇灭楚后/前 223 置会稽郡时已立、"鄮"字取"贸"音、因鄞东海岸有海贸之集而得名）——但**作为一州治所的明州、是 738 才有的事**；港埠虽久、州治才新。

**港埠的历史长河**：
- **秦汉**——鄮县已为会稽郡东境一海滨小县、海贸初萌；"鄮"字"贸"音、是这片海岸早期海贸活动的语义化石；
- **东汉至三国吴**——孙吴据江东、鄮县属会稽郡、东海航线初具雏形；孙吴曾有"卫温浮海至夷洲"（230，即台湾或琉球）之类的远洋尝试——但盛唐人对此未必深究；
- **东晋南朝**——鄮县为南朝海防要地之一、海贸 + 海路有断有续；**梁陈间**鄮县东海岸是江南与日本（倭国）、新罗、百济间海路的若干登陆点之一——但规模远不及后世；
- **隋开皇九年（589）**——平陈、废鄮县入句章；隋一代鄮县并入句章约 30 年；
- **唐武德四年（621）**——析句章复立鄮县、属越州；
- **开元二十六年（738）闰八月**——**采访使齐澣奏析越州东境置明州**（《新唐书·地理志五》《旧唐书·玄宗本纪》）——州名取**四明山**之"明"（山顶四窗岩有四方天窗似的洞、月光穿照如四明、故山名"四明"）；新立的明州领**鄮、奉化、慈溪、翁山**四县——鄮县为治所；
- **天宝元年（742）**——玄宗诏天下州一律改郡、明州改称**余姚郡**——是 755 时的官称（**乾元元年/758 复明州**——755 后三年事）；
- **明州港**——随明州州的设立而正式作为一州海港显形；港埠虽是汉魏以来旧地、但**作为"明州港"被唐廷正式纳入州级海贸管理体系是 738 后的新事**。

**明州港 + 广州港 + 登州港的盛唐三向海贸**——
- **南海路 / 广州港**：**广州**为岭南道治、隋唐间南海第一大港；**唐开元间始置市舶使**（一说开元二年/714、文献不一）—— 是唐廷海贸最重要的口岸；**南海路**通占婆 + 真腊 + 室利佛逝 + 师子国（今斯里兰卡）+ 天竺东岸 + 波斯湾大食；**广州主南海**——是盛唐对外海贸的第一极；
- **黄海北岸 / 登州港**：**登州**为河南道治、唐贞观元年（627）置；登州港（蓬莱）是**北路 + 朝鲜半岛**航线的主港——日本遣唐使**北航路**自北九州 → 朝鲜海峡 → 山东登州一线入唐；**登州主朝鲜半岛与黄海北岸**——是盛唐对外海贸的第二极；
- **东海路 / 明州港**：**明州**为江南东道新州、东海路通日本南九州的**南航路**——**自北九州 → 东海 → 明州或苏州（江南东道）** 一线入唐；**明州主东海与日本**——是盛唐对外海贸正在兴起的第三极、远未到广州 + 登州的规模。

**与遣唐使的关系**——**这是 755 时态的核心区分**：
- **日本遣唐使自舒明天皇二年（630）始派、至宇多天皇宽平六年（894）废、共派 19 次**（其中实际成行约 14 次）——**755 时已派至第十二次**（**孝谦天皇天平胜宝四年/752 派出、天平胜宝六年/754 归国**）；
- **第十二次遣唐使（752-754）**——**正使藤原清河 + 副使大伴古麻吕 + 副使吉备真备**——是**鉴真东渡同时归航**的一批；**鉴真随副使大伴古麻吕船自苏州黄泗浦（今张家港）启航、753 年十二月二十六日抵日本萨摩国坊津**——**鉴真东渡第六次成功的起点是扬州 → 苏州黄泗浦、非明州港**；本 doc 严守史源、不作"鉴真自明州东渡"之误；
- **遣唐使的两条航路**——**北航路**（630-702 年间多用、自北九州 → 朝鲜海峡 → 山东登州）+ **南航路**（702 后渐用、自北九州 → 东海 → 明州或苏州一线）；**南航路在 8 世纪上半叶（玄宗朝）渐成主路**——但**以明州港为南航路主登陆港**是中晚唐（9 世纪起、承和五年/838 小野篁那批后期）才完全确立的事；**755 时明州港虽已通日本、规模不大**——遣唐使的南航路常自苏州 + 扬州 + 明州 + 楚州几处轮换登陆。

### 此刻

天宝十四载这一年的明州港——**一座新港、按例运行、海舶寥落却已显形、繁盛尚在前方**。

**港的状态**：**州治鄮县城周回不大**——压在四明山东北麓与杭州湾南岸之间的冲积台地上；**州城东北三十里至甬江入海口**——是明州港的核心登陆点；**港埠规模有限**——海舶下锚处水深 + 港湾避东南风 + 沙岸便于装卸；**州 + 县官督**——盛唐 755 时无独立市舶机构、州县衙督收市税 + 关吏验牒；**海舶**——日本遣唐使的南航路偶有自此入唐（远不如苏州 + 扬州 + 登州多）、新罗商舶往来贸易 + 江南海舶北上登州或南下福建。

**港的对外海路**（755 时实际运行）——
- **东渡日本**——明州港至日本**北九州博多湾**（或南九州坊津/萨摩一带）的东海航程、顺风约半月（春夏东南风）、逆风更久；遣唐使偶有自此入唐——但 755 时已发出第十二次（752-754）、第十三次要到代宗永泰元年（765）才出（实际未成行）；755 时无遣唐使在港；
- **北上登州**——明州港沿杭州湾北岸 + 黄海西岸北上至登州、转往朝鲜半岛或山东半岛诸港——**这是盛唐 江南海舶北上的主路**；
- **新罗商舶**——新罗（朝鲜半岛南部）与唐贸易密切、新罗商舶往来明州 + 登州；**这是唐对外海贸中**最稳定的一支**——比日本航线更密 + 比南海大食航线更近；
- **南下福建广州**——沿浙东 + 闽东海岸南下至福州 + 泉州 + 广州、与南海路衔接；这是江南海舶南下的支路、规模不大。

**港的商品流通**——**明州港的核心物产是青瓷 + 茶 + 海产**：
- **越窑青瓷**——**上虞、慈溪、余姚一线的越窑窑场盛唐时窑火正盛**；**上林湖窑址在今慈溪境内、唐时属余姚县**——是盛唐至晚唐间越窑青瓷的核心烧造地；**青瓷自明州港 + 越州出海**通日本 + 新罗 + 南海——是盛唐对外海贸最具代表的物质载体之一；
- **四明山茶**——四明山下出茶、海舶出口至日本 + 新罗；
- **海产**——明州近海产盐 + 鱼蚌 + 海产珍贝；
- **进口物**——明州港自日本 + 新罗 + 南海进口的物产较少、规模不及广州 + 登州；以日本带回的工艺品 + 新罗带来的人参 + 海产为多。

**唐人对明州港的认知**——**江南面海的新港 + 日韩航线的南端起点 + 越窑青瓷出海的口岸**。**唐人对明州的整体观感**——**新州、海港之州、四明山下的青瓷与佛寺之乡**；**港的繁盛**——尚不如广州 + 登州、亦不及扬州（虽扬州非海港、但是江南漕运 + 商业枢纽、海舶北上常以扬州为中转）；**港的潜力**——已显形、待中晚唐渐起。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九即起）；**天宝十四载十二月丁酉（公历 756 年 1 月）陷东京洛阳**——通济渠北端起点失、漕路中断；**明州港距前线四千唐里**、未直接受影响——但江南作为唐廷漕赋命脉、地位骤升；**至德 / 乾元间江南诸道租赋苦输支撑唐廷半壁**——明州港作为江南东南海港 + 海贸口岸的份量、自此渐显；**中晚唐至五代**——明州港逐渐成为东海贸易的第一港、最终在宋元间立为四大市舶港之一。

### 此刻意味

对天宝十四载的余姚郡鄮县人民来说——明州港是**自家东郊的海口**：海舶寥落却已往来、商旅登岸 + 越窑青瓷出海 + 日韩商舶入港；州城东北三十里的海岸线、是州境与外海联通的物理 anchor。对盛唐唐廷来说——明州港是**东海航线的新起点**——尚未到广州 + 登州的规模、但已显形；广州主南海 + 登州主朝鲜半岛 + 明州主东海与日本——盛唐三向海贸的东向起点、于玄宗朝渐成。对江南海舶 + 日韩商舶来说——明州港是**江南东南海岸最便利的登陆点之一**——避东南风 + 通甬江上溯 + 入江南内陆腹地；是东海航线的物理实现。

唐人尚不知道——天宝十四载十二月丁酉（公历 756 年 1 月）通济渠北断、漕路停输；江南诸道租赋的份量将骤升；**中晚唐至五代**——明州港随海贸之盛而起、**承和五年（838）小野篁那批日本遣唐使后期**以明州为主登陆港、**晚唐五代上林湖越窑**烧出秘色瓷自明州港出海——明州港的盛况正在前方。**南宋绍熙五年（1194）**——明州升庆元府、**与广州 + 泉州 + 华亭并列为南宋四大市舶港**；**元** 改庆元路；**明洪武十四年（1381）**——改庆元路为**宁波府**——"宁波"二字自此立（取"海定则波宁"意）。盛唐 755 时的明州港、尚未到这些盛况——但作为东海航线的物理起点、已显形。

## 子条目

### 鄮县（明州治；秦汉旧县；港埠所在）

**鄮县**——秦始皇灭楚后（前 223）置会稽郡时所立、治所即今宁波一带；**"鄮"字取"贸"音**——因鄞东海岸有海贸之集而得名——是这片海岸早期海贸活动的语义化石。**汉承秦制**——鄮县属会稽郡；**东汉、三国吴、东晋南朝相沿**——这一带千年间一直是会稽郡东境的一个海滨小县、治所微移而县名不废。**隋开皇九年（589）**——平陈、废鄮入句章；**唐武德四年（621）**——析句章复立鄮县、属越州；**开元二十六年（738）**——明州立、鄮县升州治；**至晚唐永贞元年（805）**——析鄮县置鄞县、州治移驻鄞县（今宁波市区一带）——但 755 时鄞县尚未析出、州治仍在鄮县。**明州港**——以鄮县东海岸为核心；州治在鄮县城（即今宁波鄞州区中心一带）、港埠在城东北甬江入海口（今宁波镇海、北仑一带）。

### 甬江入海口（明州港的物理 anchor）

**甬江**——明州州境内最重要的水道、自三江口（鄮县城东北、奉化江 + 姚江合流处）东注入海。**入海口**——杭州湾南岸最深的一段港湾；**港湾**——可避东南风、水深可下锚海舶、沙岸便于装卸；**唐 755 时**——明州港的主登陆点即在甬江入海口南北两岸。**自三江口至入海口**——约数十唐里、海舶卸货后可沿甬江上溯至州治鄮县城下、再分入奉化江 + 姚江通往内陆。**今甬江仍是宁波港的核心水道**——盛唐时的物理形态与今大致同地、但海岸线因后世海塘 + 围垦渐东推。

### 越窑青瓷（自明州港出海的盛唐核心物产）

**越窑**——浙东上虞、慈溪、余姚一线的瓷窑窑场、自东汉以来烧造青瓷、至盛唐窑火正盛；**上林湖窑址**（在今慈溪境内、唐时属余姚县）——是盛唐至晚唐间越窑青瓷的核心烧造地。**青瓷**——自明州港 + 越州出海通日本 + 新罗 + 南海——是盛唐对外海贸最具代表的物质载体之一；**1957 年起多次发掘上林湖窑址、2015 年于上林湖后司岙出土晚唐五代秘色瓷实物**——与法门寺地宫 1987 年出土的咸通十五年（874）秘色瓷恰能对应；是明州港境内最重要的盛唐至晚唐物质遗存。**注**：**秘色瓷是晚唐五代物**——盛唐 755 时尚未烧出；本 doc 严守不预支。

### 日本遣唐使的两条航路 + 鉴真东渡

**遣唐使**——**日本遣唐使自舒明天皇二年（630）始派、至宇多天皇宽平六年（894）废、共派 19 次**（其中实际成行约 14 次）；**755 时已派至第十二次**（752-754）。**两条航路**——
- **北航路**（630-702 年间多用）——自北九州 → 朝鲜海峡 → 山东登州；
- **南航路**（702 后渐用）——自北九州 → 东海 → 明州或苏州一线；

**南航路以明州港为主登陆港**——是**中晚唐（9 世纪起、承和五年/838 小野篁那批后期）才完全确立**；755 时南航路常自苏州 + 扬州 + 明州 + 楚州几处轮换登陆。

**鉴真东渡**（743 始 + 754 抵日本）——**第六次东渡 753-754 成功的起点是扬州 → 苏州黄泗浦（今张家港）、非自明州港**；**鉴真随日本遣唐使副使大伴古麻吕船自苏州黄泗浦启航**——753 年十二月二十六日抵日本萨摩国坊津；本 doc 严守史源、**不作"鉴真自明州东渡"之误**。明州港在鉴真东渡中**非起点**、但与第十二次遣唐使的整体南航路相关联。

### 新罗商舶（盛唐至中晚唐东海航线最稳定的一支）

**新罗**（朝鲜半岛南部）与唐贸易密切——**新罗商舶往来明州 + 登州**——是唐对外海贸中**最稳定的一支**；比日本航线更密、比南海大食航线更近。**755 时**新罗商舶已频繁出入明州港；**中晚唐**——新罗商人在明州 + 登州 + 楚州沿海岸形成"新罗坊"聚居——是中晚唐东海贸易的核心载体之一。

### 明州港 + 广州港 + 登州港（盛唐三向海贸格局）

**南海路 / 广州**——岭南道治、隋唐间南海第一大港；通占婆 + 真腊 + 室利佛逝 + 师子国（今斯里兰卡）+ 天竺东岸 + 波斯湾大食；**主南海**。

**黄海北岸 / 登州**——河南道治、唐贞观元年（627）置；登州港（蓬莱）通日本北航路 + 朝鲜半岛 + 山东半岛诸港；**主朝鲜半岛与黄海北岸**。

**东海路 / 明州**——江南东道新州（738 立）；通日本南航路 + 新罗 + 江南内陆腹地；**主东海与日本**。

**三港分立**——**广州 + 登州 + 明州**——是盛唐对外海贸的三向格局；**广州 + 登州**两港早在唐前期（开元前后）已立、明州港随明州州 738 立而显形、是三港中最年轻的一支。**安史之后**——南海路 + 东海路渐显形、北路因新罗 + 渤海局势波动而起伏；**中晚唐至五代**——三港格局调整、明州渐升、广州续盛、登州渐衰。

## 历史长河（755 → 今）

**中晚唐**　**乾元元年（758）**复改余姚郡为明州——州名"明"自此定。**大历六年（771）**——翁山县（海中）因海寇袭扰而废、明州自此领三县（鄮、奉化、慈溪）。**永贞元年（805）**——析鄮县置**鄞县**、州治移驻鄞县（今宁波市区一带）。**长庆元年（821）**——州城自小溪（今鄞江镇）迁至**三江口**（即今宁波老城）——是明州城作为"三江口之城"的肇始；自此一千二百余年州治再未大迁。**晚唐至五代**——明州港随海贸之盛而起；**承和五年（838）小野篁那批日本遣唐使**——明州为主登陆港确立；**圆仁《入唐求法巡礼行记》**（开成三年至大中元年/838-847）——记述这一时期东海航线 + 唐沿海港埠的运行；**新罗、日本商舶岁岁来**——明州自此成中晚唐至五代东海贸易的第一州。**上林湖越窑**在晚唐五代烧出**秘色瓷**——与法门寺地宫 1987 年出土的咸通十五年（874）秘色瓷恰能对应；这是明州港境内最重要的盛唐至晚唐物质遗存。

**五代至宋**　**五代**——明州属吴越国（907-978）、钱氏定都杭州、明州为东境海港重镇；**北宋**——仍号明州、海贸更盛；**北宋淳化/咸平间（992 或 999）**——立**明州市舶司**（与广州 + 杭州市舶司并立）——是唐 755 时尚未有的官署、北宋始立、本 doc 严守不预支。**南宋绍熙五年（1194）**——明州升庆元府；南宋间庆元府是**市舶司所在**之一、**与广州 + 泉州 + 华亭并列为南宋四大对外贸易港**——这是明州港在中国海贸史上的最高峰之一。

**元明清**　**元**——改庆元路；**明洪武十四年（1381）**——改庆元路为**宁波府**——"宁波"二字自此立（取"海定则波宁"意）、"明州"之名自洪武十四年正式退场——明州之名共立 643 年（738-1381）。**明嘉靖间倭乱沿海**——宁波府是首当其冲、府城海塘屡修；**明万历间**宁波府人口逾百万。**清沿之**——**鸦片战争后宁波为 1842 年《南京条约》五口通商口岸之一**——开埠岁月由此起；宁波港自此进入近代海港时代、但物理位置仍在甬江入海口一带、与盛唐明州港同地。

**民国至今**　**1912 年废府置县**。**1949 年后**——改设宁波专区；**1983 年**——地市合并设地级宁波市；**今宁波市**——浙江省辖**副省级、计划单列市**、面积九千八百余平方公里、人口约九百四十万；市辖海曙、江北、北仑、镇海、鄞州、奉化、余姚、慈溪、宁海、象山——大体覆盖盛唐余姚郡境、外加翁山旧县（今舟山、1987 年自宁波析出为单设地级舟山市）。**北仑港**（位于今宁波市北仑区、甬江入海口南岸）——是今中国大陆第一大港、世界第一大港之一（货物吞吐量计）——盛唐 755 年那座面海新州的港埠雏形、于一千二百余年后成为东亚海贸的物理中枢。**上林湖越窑遗址**（1957 年起多次发掘、2015 年后司岙秘色瓷出土）——是明州港腹地最重要的盛唐至晚唐考古成果。**今游人重访宁波港**——所见的是甬江入海口 + 北仑港 + 镇海港 + 三江口老城——盛唐时的明州港作为物理形态早已扩展至今港 + 古港不再分立、但作为东海航线起点的地名记忆仍在。
