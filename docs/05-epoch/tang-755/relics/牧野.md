# 牧野

## 一句话

**周武王伐纣决战地**——BC 1046（夏商周断代工程主张；早年有 BC 1027 / BC 1122 等异说）周武王率诸侯之师渡孟津、北上至商郊牧野与商纣王（帝辛）会战；商军倒戈、商亡、周立；**755 时已废约 1800 年**——城垣无存、唐人据《尚书·牧誓》《史记·周本纪》知此地大略在卫州汲县北郊原；牧野是中国"天命"之物理 anchor——周代 800 年礼乐 + 历代"汤武革命"叙事皆从此一战出。

## 形貌

牧野在唐河北道**卫州汲郡治汲县**境——具体在**汲县北约 30 唐里、太行山东麓与黄河北岸之间的一片平野**（今河南新乡市牧野区一带、与卫辉市相邻）。**地势**：太行山东南余脉缓缓没入华北平原、海拔约 80-100 米；地势开阔、东南为黄河、东北为永济渠（隋唐大运河北段）、北为太行——是中原通往河北的咽喉之一。**今所见**：盛唐 755 时牧野旧地一片郊原——城垣无、营垒无、战壕无；地表为农田 + 桑林 + 间或起伏的土丘——与殷墟、镐京同样属"地表无形 + 仅存方位记忆" 一类的物理 anchor。

**与卫州治汲县的方位**：**卫州治汲县**（今河南卫辉市）——盛唐 755 时汲郡太守治汲县（天宝元年 742 改州为郡之后只设郡太守、不并列"刺史"）；牧野在汲县城北郊约 30 唐里许（具体里数本 doc 不坐实、唐人无此精确记载）。**与朝歌 + 殷墟的方位**：**朝歌**（今河南淇县）—— 唐人通识中的"殷末都"、距牧野约百余唐里量级；**殷墟**（今河南安阳小屯）—— 盘庚迁殷之地、距牧野更远约百余至二百唐里量级 + 与朝歌分属两个地理点（具体里数本 doc 不坐实、唐人无精确记载）。**注**：朝歌 与 殷墟 在 唐人 认知 中 多被 等同 / 互换 —— 唐人 多以 朝歌 为 商末 都、不识 安阳 一带 为 殷墟；**牧野 与 朝歌 的方位关系** 是 唐人 最 熟悉 的 一对 —— "武王 渡 孟津、北上 至 牧野、击 朝歌"。

**与黄河 + 孟津的方位**：**孟津**（今河南孟津县、洛阳东北）—— 武王伐纣 渡黄河 之处；**自孟津渡河北上至牧野约 300 唐里**——武王 大军 自周原 + 镐京 一路 东行、经 洛水 一带（按：成周 / 洛邑 为 周公 克 商 后 营建、武王 伐纣 时 尚 未 立）、北 渡 孟津、再 沿 太行 东 麓 北上 至 商 郊 牧野。

物候：太行 东南 余脉 + 华北 平原 过渡 带、温带 季风 气候——夏 热 多雨、冬 冷 干燥；属 河北道 腹 地 农耕 区；间 或 有 桑林 + 麻 田 + 禾 黍——是 唐 卫州 一带 最 熟悉 的 乡野 景观。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

牧野 在 唐人 的 回望 里——是**"汤武革命" 之 "武革命" 的 物理 anchor + 周天命 的 起 点**。

**武王伐纣的来路**（盛唐人据《尚书》《史记》《诗经》可知）：
- **周文王末年**——三分天下有其二、未灭商而崩；**武王嗣位**、以太公望（吕尚、姜尚、师尚父）为师、周公旦为辅；
- **武王九年（约 BC 1048）孟津之会**——武王 率 周师 + 八百 诸侯 会盟 于 孟津、观兵 而 还（"诸侯 不期 而 会 者 八百"——《史记·周本纪》）；**孟津 之 会 是 牧野 之 战 的 前奏**——周 武王 试探 诸侯 + 商纣 反应、未即 灭商；
- **武王十一年（BC 1046、夏商周断代工程主张）**——武王 自 镐京 出师、率 戎车 三百 乘 + 虎贲 三千人 + 甲士 四万 五千人（《史记》数）；经 洛水 一带（后世 成周 / 洛邑 附近——成周 为 周公 克 商 后 营建、武王 此时 尚 未 立）+ 北 渡 孟津、再 沿 太行 东 麓 北上 至 商 郊 牧野；
- **二月 甲子 朝**（《尚书·武成》载、夏商周断代工程定为 **BC 1046 年 1 月 20 日**）——武王 在 牧野 誓师、是 为 《牧誓》；
- **甲子 日 决战**——商 纣 调 商军 / 徒众 七十 万（《史记》"发 兵 七十 万 人"、《尚书·武成》"前 徒 倒 戈"；"奴隶、俘虏" 为 后世 解释、后世 多 疑 此 数 之 巨）拒 周；商 军 倒戈 相向、**血流漂杵**（《尚书·武成》原文、孟子 后 以 "尽信 书 不如 无 书" 质之）；
- **当夜**——商 纣 自焚 于 鹿台、商 亡；
- **武王 入 商邑**（朝歌）—— 释 箕子 之 囚 + 表 商容 之 闾 + 封 比干 之 墓；
- **武王 克商 后**：**封 纣王子 武庚 于 殷商 故地**（朝歌 / 殷墟 一带 地望 需 分辨——皆 商 王 畿、安抚 殷 遗 民）、**分 管叔 鲜 + 蔡叔 度 监 之**（"三 监"）；**微子 启** 此 时 尚 未 受 封——**周 公 平 三 监、诛 武庚 后（约 BC 1043）始 封 微子 于 宋**；
- **武王 返 镐**——立 周 王朝；**两 年 后 武王 崩**——成 王 即 位、周 公 摄政；**翌 年 武庚 + 三监 之 乱**——周 公 东 征 三 年、平 之、**再 营 洛邑 + 分封 诸侯**——周 礼 800 年 自此 出。

**唐人对牧野的认知**：
- **位置认知**——唐人 据《尚书·牧誓》"武王 戎车 三百乘、虎贲 三百人、与 受 战 于 牧野" + 《元和郡县图志》卷十六 卫州 沿革 知 牧野 在 卫州 汲县 北 郊原；**李吉甫《元和郡县图志》813 撰**、晚于 755、本 doc 引用 以 载 唐人 地理 记忆；
- **文化认知**——**"汤武革命" 的 "武革命"**：商汤 灭夏 + 武王 灭商 + 周公 制礼 作乐——是 中国 早期 政治 哲学 中 "天命 转移" 的 两 例 范式；**唐 玄宗 一脉 以 "承周 天命" 自居**、牧野 是 这 文化 想象 的 根源 之 一；
- **物理认知**——城垣 无、地表 无 战 迹；唐人 所 辨识 者、主要 是 **方位** + **历代 文献 中 的 牧誓 + 大明 之 章**——以及 卫州 汲郡 一带 间 或 起伏 的 土丘。

### 此刻

天宝十四载这一年的牧野——**已废约 1800 年**（BC 1046 → AD 755、公元 无 0、合计 约 1800 年）；**地表 无 战 迹、属 唐 卫州 汲郡 治 汲县 北 郊原**。

**地表景观**：太行 东南 余脉 + 华北 平原 过渡 带 的 一片 郊原——农田 + 桑林 + 间 有 土丘 起伏；周边 为 卫州 汲县 + 共县 + 新乡县 + 卫县 等 的 乡野；**唐人 不能 见 到 战 阵 + 不能 见 到 商军 倒戈 之 地 + 不能 见 到 武王 誓师 之 坛**——只 能 见 到 **方位 + 一片 郊原 + 远处 太行 的 天际 线**。

**唐人 到 牧野 一带 的 实际 体验**：**盛唐 卫州 汲郡 是 永济渠（隋唐 大 运河 北 段）经 过 的 重要 州**——汲县 + 共县 + 新乡县 皆 在 永济渠 沿线、是 河北 道 商旅 + 漕运 的 要 道；**唐人 自 洛阳 经 永济 渠 北上 至 幽州 / 范阳 郡**——途中 经 卫州、当 不 难 见 到 "牧野" 这 一 地名 + 方位 印象。**李白（701-762）、杜甫（712-770）、王维（约 701-761）等 盛唐 文人**——若 经 永济 渠 北上 河北、或 在 卫州 怀 古、皆 当 知 牧野 即 在 此 北 郊。

**755 时 的 "牧野 意识"**：是**盛唐 文人 怀古 的 重要 题材 之 一**——但 不 是 一种 考古 意识、是 一种 政治 哲学 上 的 自我 认同。**唐 玄宗 一脉 以 "承周 天命" 自居**——封禅 + 郊祀 + 明堂 + 籍田 + 大武 之 乐 等 礼制 复古、皆 取意 于 周；**牧野 是 周 礼 800 年 的 起 点、亦是 唐 礼 复古 的 远 源**。

**这一年 的 卫州 汲郡** 处于**唐 玄宗 朝 末年 河北 道 腹 地 州 城**的 稳定 状态——再过 半年 安禄山 起 兵 于 范阳、自 范阳 南下；**755 末 至 763 安史 八 年 间 河北 道 反复 易主**——卫州 + 汲县 + 牧野 旧地 多次 处于 战乱 中。但 唐军 + 叛 军 间 反复 争夺 的 是 卫州 城 + 永济 渠 沿线 的 漕 运——**牧野 旧地 仍 在 汲县 北 默默 躺 着、不 被 重新 识 为 牧野 战 场**。

### 此刻意味

对天宝十四载的卫州 / 汲县人来说——汲县 北 30 里 的 那 一片 郊原 就 是**自家 城 北 的 乡野**：田 垄、桑 林、远处 太行 的 天际 线——是 熟悉 的 乡野 景观。对 长安 那 一 代 士人 来说——牧野 是**周 武王 克商 之 地 + "汤武 革命" 的 物理 anchor + 周 礼 800 年 的 起 点**：从 唐 玄宗 到 杜 甫 王 维 李 白、皆 以 "承 周" 自居——牧野 是**这种 自我 认同 的 远 源**。

唐人 心中 的 牧野——是**《尚书·牧誓》的 战前 誓师 辞 + 《诗经·大雅·大明》的 战阵 乐 章 + 《史记·周本纪》的 战 后 分封**：**这 三 重 文本 叙述** 构成 唐人 对 牧野 的 全部 物理 想象。**唐人 在 卫州 汲县 北 郊原 走 过**——脚下 是 1800 年 前 周 武王 + 太公 望 + 周公 旦 + 八百 诸侯 + 商 纣 王 帝辛 + 七十 万 商军 / 奴隶 的 战 场；**但 唐人 见 不 到 任何 物理 痕迹**——只 能 凭 文本 与 方位 神游。**这是 中国 早期 史 实 物理 anchor 与 文本 anchor 的 分裂**：牧野 的 文本 anchor 在 《牧誓》《大明》《周本纪》清晰 至 极——而 物理 anchor 只 余 方位。

唐人 尚 不 知道——再 过 半 年（755 末 安史 起 兵）、再 过 千 二百 多 年（1990 年 起 河南 新乡 牧野 区 一带 陆续 有 商 周 之 际 遗 物 发掘）——**牧野 古 战场 的 物理 痕迹 由 文献 转 为 考古**。**唐人 在 755 所 见 的 那 一片 郊原**——千 二百 多 年 后 是 新乡 市 区 + 牧野 区 的 一部 分；**今 游人 到 新乡**——所 见 的 是 现代 城市 + 间 或 的 考古 公园 + 街道 名 "牧野 大道"——**而 755 时 这 一切 尚 在 地下、唐人 不识**——这是 中国 早期 史 实 物理 anchor 失 而 复 得 的 又 一 例。

## 子条目

### 牧誓（中国最早的战前檄文）

**《尚书·周书·牧誓》**——武王 战前 誓师 辞、是**中国 现存 最 早 的 战前 檄文 之 一**。**原文 开篇**："时 甲子 昧 爽、王 朝 至 于 商 郊 牧野、乃 誓"——**武王 在 牧野 集 诸侯 之 师、左 杖 黄钺、右 秉 白旄 以 麾**；**檄 商 罪**："今 商王 受、惟 妇言 是 用、昏 弃 厥 肆 祀 弗 答、昏 弃 厥 遗 王父母 弟 不 迪、乃 惟 四方 之 多 罪 逋 逃、是 崇 是 长、是 信 是 使、是 以 为 大夫 卿 士、俾 暴 虐 于 百 姓、以 奸 宄 于 商邑"——**武王 列 商 纣 之 罪 + 召 诸侯 之 师 + 誓 同 心 灭 商**。**檄 阵 法**："不 愆 于 六 步 七 步、乃 止 齐 焉"——**周师 阵 法 整 齐 + 不 贪 杀 + 不 滥 进**。**这 是 中国 早期 政治 文 学 最 重要 的 篇 章 之 一**——盛唐 士人 自 童 蒙 起 即 诵 之；**唐 玄宗 朝 礼制 复古** 时 多 引 《牧誓》 之 语。

### 牧野之战（甲子日 + 商军倒戈 + 血流漂杵）

**甲子 日 决战**（夏商周 断代 工程 定 为 BC 1046 年 1 月 20 日）—— 武王 率 周师 + 八百 诸侯 与 商 军 决战 于 牧野。**周师 数**：戎车 三百 乘 + 虎贲 三千人 + 甲士 四万 五千人（《史记·周本纪》）；**商 军 数**：纣 调 商军 / 徒众 七十 万 拒 周（《史记·周本纪》"发 兵 七十 万 人"、《尚书·武成》"前 徒 倒 戈"；"奴隶、俘虏" 为 后世 解释；后世 学者 多 疑 此 数 之 巨——或 实际 仅 数 万、《史记》数 为 夸大）。**战 况**：**商 军 倒戈 相 向**——商军 / 徒众（后世 解 为 奴隶、俘虏）不 愿 为 商 拒 周、在 阵 前 反 戈；**周师 长驱 直 入**；**血流 漂 杵**（《尚书·武成》原文、形容 战 之 烈）——**孟子 后 以 "尽信 书 不如 无 书"（《孟子·尽心 下》）质 此 句**：以 武王 仁 义 之 师 + 商军 倒戈 + 战 局 速 决、不应 有 "血流 漂 杵" 之 烈；**这 是 孟子 对 《武成》 的 著 名 文 本 批 评**——盛唐 士人 多 知。**当 夜**——**商 纣 王 帝辛 自焚 于 鹿台**（鹿台 在 朝歌 / 今 河南 淇县）；商 亡。

### 牧野与朝歌、殷墟的方位关系

**牧野** + **朝歌** + **殷墟**——是 商 末 + 周 初 三 个 紧密 相关 的 地 名。**牧野**（今 河南 新乡 牧野 区）——决战 地、在 商 郊；**朝歌**（今 河南 淇县）——唐人 通识 中 的 "商 末 都"、纣 王 自焚 之 鹿台 所 在；**殷墟**（今 河南 安阳 小屯）——**盘庚 迁 殷** 后 的 商 王 畿、商代 中 晚 期 主要 都 邑。**三 地 方位**：**朝歌 与 殷墟 分 属 两 个 地理 点**——朝歌 在 殷墟 南、两 地 相 距 约 百余 唐里 量级；**牧野 在 朝歌 南、距 殷墟 约 百余 至 二百 唐里 量级**（具体 里 数 唐人 无 精确 记载、本 doc 不 坐 实）。

**唐人 对 三 地 的 认知**：**朝歌 = 商末 都** + **殷墟 在 唐人 视野 之 外**——唐人 多 据 《尚书·牧誓》《史记·殷本纪》 + 《吕氏春秋·慎大览》 认 朝歌 为 商末 都；**安阳 一带 在 唐人 认知 中 是 曹魏 邺 都 + 北齐 邺 都**、不 是 殷 墟。**故 唐人 心 中**：**牧野 之 战** = 武王 自 孟津 渡 河 北上、至 朝歌 郊 外 之 牧野、击 朝歌 + 杀 纣——**与 殷墟 无 关**。**这 是 唐人 早期 史 实 物理 anchor 失 准 的 又 一 例**。

### 武王克商后的分封（微子启 + 武庚 + 三监）

**武王 克商 之 后**——立 即 处理 殷 遗 民 + 商 王 畿 + 商 之 后：

1. **封 微子启 于 宋**（**按 时序**：微子 封 宋 不 在 武王 克 商 当 时——而 在 **周 公 平 三 监、诛 武庚 之 后（约 BC 1043）**；武王 克 商 直 后 只 立 武庚 守 殷、未 即 封 微子）——微子 启 是 商 纣 王 同 父 异 母 兄、商之 后；周 公 平 乱 后 封 之 于 **宋**（今 河南 商丘）——以 续 商汤 之 祀。**宋 国** 自 此 始 + 春秋 时 宋襄公 为 五 霸 之 一 + 战国 末 灭 于 齐；**孔子 + 庄子 + 墨子** 皆 宋 人 / 其 先 为 宋。
2. **封 纣王子 武庚 于 殷商 故地**（朝歌 / 殷墟 地望 需 分辨——朝歌、殷墟 皆 商 王 畿、不 宜 直接 等同 于 今 安阳 一 点）——武王 不 绝 商 祀、以 武庚 守 商 王 畿、安抚 殷 遗 民。
3. **分 管叔 鲜 + 蔡叔 度 监 武庚**（"三 监"；一说 含 霍叔 处 / 一说 含 武庚 自身）——分 商 王 畿 为 三、以 武王 弟 鲜 + 度 + 处 三 人 监 之；是 周 初 防 殷 遗 民 复 国 的 制度 安排。

**两 年 后 武王 崩**——成 王 即 位、周 公 摄政；**翌 年 武庚 + 三 监 之 乱**——武庚 联 管叔 + 蔡叔 + 殷 遗 民 反 周；**周 公 东 征 三 年、平 之**——杀 武庚 + 杀 管叔 + 囚 蔡叔；**再 营 洛邑 + 分封 诸侯 + 制 礼 作 乐**——**周 礼 800 年 自此 出**。**注**：周 公 平 管 蔡 之 乱 在 BC 1043 武王 死 后、**不 是 牧野 之 战 当 时**——**微子 启 受 封 于 宋 亦 在 周 公 平 三 监、诛 武庚 之 后**（非 武王 克 商 当 时）；本 doc 严守 此 时 序、不 混 两 事。

### 唐人对牧野的"承周天命"想象

**唐 玄宗 一脉 以 "承周 天命" 自居**——是 唐 立国 叙事 的 核心 之 一（详见 [镐京](./镐京.md) 子 条目）。**牧野 在 这 想象 中 的 位置**：

- **"汤武 革命" 的 "武 革命"**——《周易·革卦·彖辞》"汤 武 革命、顺 乎 天 而 应 乎 人" 是 中国 早期 政治 哲学 中 最 重要 的 一 句；**牧野 之 战 是 "武 革命" 的 物理 anchor**——商 失 天命 + 周 受 天命 + 武王 + 太公 望 + 周 公 旦 是 天 命 转 移 的 三 个 物理 执 行 者。
- **周 礼 800 年 的 起 点**——周 公 制 礼 作 乐 是 周 武王 克 商 + 武 王 死 后 + 周 公 平 管 蔡 之 乱 之 后 的 制度 大 整理——**而 这 整 一 链 的 起 点 是 牧野 之 战**：无 牧野 之 战 即 无 周 王 朝、无 周 王 朝 即 无 周 公 制 礼 作 乐、无 周 公 制 礼 作 乐 即 无 中国 礼 乐 文 明 的 主 流。
- **唐 玄宗 朝 礼制 复古**——开元 + 天宝 间 礼 部 + 太 常 寺 屡 次 议 礼、玄宗 屡 行 封禅 + 籍田 + 大武 之 乐——**牧野 + 周 公 + 武王** 是 这 些 礼制 的 远 源。**《大武》之 乐**——《吕氏春秋·古乐》载 武王 伐 纣、渡 孟津、奏 《大武》之 乐；**唐 玄宗 朝 雅 乐 体 系 中 的 "周 雅" 部 分** 取 意 于 《大武》。

**故 唐人 在 卫州 汲县 北 郊原 走 过**——所 怀 之 古 不 仅 是 一 场 战 事、更 是 一 套 政治 哲学 + 一 个 文化 自我 认同 的 物理 根。

### 牧野的年代争议（BC 1046 vs 1027 vs 1122）

**武王 克商 / 牧野 之 战 的 公元 年 代**——是 中国 早期 史 学 中 争议 最 大 的 公案 之 一。**主要 异 说**：

- **BC 1046**——**夏商周 断代 工程**（1996-2000）主张；今 学界 主用 年代；本 doc 取 此 数；
- **BC 1027**——**《竹书纪年》古本 残 文** 之 一说；20 世纪 王国维 等 学者 取 之；
- **BC 1122**——**汉 儒 古 传 一说**；唐人 据 《史记》 + 《竹书纪年》 各 有 所 本、未 定 一 年；
- **其他 异 说**——BC 1075、BC 1059、BC 1050、BC 1045 等 数 十 种。

**唐人 对 此 年代 的 认知**：**唐人 未 有 定 年**——唐人 据 《史记·周本纪》 + 《竹书纪年》 + 《尚书》 等 各 有 所 本；**李 吉甫 《元和 郡县 图志》卷十六 卫州 条** 不 载 牧野 之 战 具体 年 代；**盛唐 士人 心 中** 的 牧野 是 "约 千 八百 年 前" 的 一 场 战 事——具体 公元 年 不 可 确 知。

**本 doc 取 BC 1046**（夏商周 断代 工程 主张）——但 不 排 斥 其他 异 说；**与 唐人 认知 的 差距**：唐人 不 知 公元 年 + 不 知 "BC 1046" 这 一 数——但 唐人 知 武王 伐 纣 在 商 末 + 距 唐 约 千 八百 年。

## 历史长河（755 → 今）

**中晚唐**　**756-763 安史 之 乱 中 河北 道 反复 易主**——卫州 + 汲县 + 牧野 旧地 多次 处于 战乱 中；**乾元 二 年（759）史思明 自 范阳 南下 + 邺城 之 战**（约 二十 万 唐军 围 邺 + 败 于 史思明）即 在 相州 邺城——**距 牧野 约 100 唐里**；**牧野 旧地 仍 在 汲县 北 默默 躺 着**。**中晚 唐 卫州 一带**——永济 渠 漕运 时 通 时 断、卫州 + 汲县 反复 在 唐军 + 藩镇 之 间 易 手；**牧野 作为 物理 anchor 始终 不 在 唐人 战略 视野 中**——唐军 + 叛 军 间 反复 争夺 的 是 永济 渠 + 卫州 城、不 是 牧野 旧地。

**两宋**　**北宋 仍 设 卫州、治 汲县**——**牧野 旧地 仍 是 汲县 北 郊原**。**宋 金 间 金 石 学 兴 起**——欧阳修《集古录》、赵明诚《金石录》等 收录 古 代 铜 器 铭文、含 周 代 器——但 宋人 多 不 知 周 代 铜 器 具体 出 土 地；**卫州 + 汲县 一带 间 或 出 土 商 周 之 际 铜 器**——但 宋人 或 归 为 "古 器物"、不 识 为 "牧野 战 场 遗 物"。**金 元 间**汲县 为 卫辉 路 / 卫辉 府 治；牧野 旧地 继续 作为 农 田 + 村落。

**明清**　**明 清 间 汲县 为 卫辉 府 治**；牧野 旧地 继续 作为 农 田。**明 清 地 方 志**（如 《卫辉 府 志》《汲县 志》《新乡 县 志》）有 "牧野" 二字 之 零散 记 载 + 多以 汲县 北 / 新乡 县 东 一带 为 "古 牧野 战 场"——但 具体 物理 范围 + 文物 归 属 皆 不 明。**明 清 间 文人 到 汲县 + 新乡 一带 怀 古**——多 凭 《尚书·牧誓》《史记·周本纪》文 本 神 游、地表 仍 无 物理 痕迹。

**民国至今**　**1949 后 河南 新乡 设 市**——**汲县 北 郊 一带 划 入 新乡 市 区**；**牧野 旧地 渐 为 新乡 市 区 + 牧野 区 的 一部 分**。**1990 年 起 河南 新乡 牧野 区 + 卫辉 市 一带 陆续 有 商 周 之 际 遗 物 发 掘**——含 兵 器 + 车 马 器 + 铜 器 + 卜 骨 等；**但 牧野 古 战场 的 物理 范围 + 具体 战 阵 位置 仍 不 完全 明 确**——考古 学界 尚 在 讨论 中。**今 游人 到 新乡**——所 见 的 是 **现代 城市 + 牧野 大道 + 牧野 广场 + 间 或 的 考古 公园**——**而 755 时 这 一切 尚 在 地下、唐人 不 识**——**牧野 由 文本 anchor 渐 转 为 文本 + 考古 双 重 anchor**。
