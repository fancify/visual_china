# 阳平关

## 一句话

兴州顺政郡长举县境嘉陵江西岸的关塞——**三国张鲁据汉中二十余年筑关于此 + 215 曹操西征张鲁攻拔之 + 219 刘备夺汉中黄忠斩夏侯渊于定军山**；唐 755 时是金牛道与沮道（汉中-武都段）分歧的物理 anchor，是汉中盆地的西门户、入蜀第二关。

## 形貌

阳平关在唐山南西道**兴州顺政郡长举县境**——具体在嘉陵江西岸、沔水（汉水）上游北岸、白马河汇入嘉陵江口的一片台地上（今陕西汉中市宁强县阳平关镇一带）。**地势**：**北倚秦岭南坡, 南接米仓山 / 大巴山西端、南瞰沔水（汉水）上游河谷、东扼汉中盆地西口、西接武都氐羌故地**——是嘉陵江与汉水分水岭附近最低、最易设关的一处咽喉。

**阳平关之险**：**关城西负群山、东临沔水、北俯嘉陵江白马河汇流口**——一关守住汉中盆地的西门。**自陈仓道南来过散关、入凤州、过河池、至兴州（顺政）后**——可分两路：**东南循金牛道**（经金牛驿、五丁关）入梁州（汉中郡治南郑县）；**或西南循沮道**（汉中-武都段）入武都郡（今甘肃武都一带）——**阳平关恰在此分歧前一段、是金牛道与沮道共扼的物理 anchor**。

**关城形制**：盛唐阳平关具体周长 + 门数 + 关楼规模史载零散——本 doc 不预设细节（今所见关楼 + 关墙多为近代重建 + 部分汉魏夯土残基；陕西省考古研究所 1980s-2010s 多次勘察确认有夯土城基 + 关楼遗址、具体唐时形制仍未坐实）。**关楼立沔水北岸台地上**——可俯瞰沔水东流 + 白马河汇嘉陵江一片；**金牛道**自关东南折入汉中盆地、**沮道**自关西南折入武都谷地。

**白马河、沮水、嘉陵江三水**：阳平关一带是嘉陵江上游南岸最重要的水系交汇点。**白马河**自西北山中流出、在阳平关北侧汇入嘉陵江；**沮水**（古名"沮水"，今亦称"沮河"，与关中沮水非一脉）自西南山中流出、在阳平关南侧汇入沔水（汉水）上游；**沔水（汉水）**自西向东流过关南——三水交汇之地、自古为兵家必争。**《华阳国志·汉中志》**"阳平关在汉中郡之西、白马河、沮水会嘉陵江处"——是阳平关地望的早期史源。

**阳平关与汉中、武都的关系**：**阳平关东行约 200 唐里至梁州（汉中郡治南郑县）**；**阳平关西行约 250 唐里至武都郡（今甘肃武都）**；**阳平关北行约 80 唐里至兴州（顺政郡治顺政县）**——是兴州东南境、汉中盆地西门、武都东门的三角交汇。**关于"阳平关 vs 阳安关"**：一说北周以后改名"阳安关"、二关同地异名；一说阳平关在汉魏 + 阳安关在北周以下为分立关塞——本 doc 严守"唐 755 时主流仍称阳平关"口径、不预设二关分立。

物候：秦巴山地、汉江上游北岸——夏温多雨、冬寒少雪；山中林木茂盛、白马河、沮水间多稻田 + 桑麻。**关城内外多有戍兵 + 关吏 + 驿马 + 商旅**——是盛唐入蜀第二关、地位较散关稍轻、较剑门关又重。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

阳平关在唐人的回望里——是**三国汉中拉锯战的物理 anchor + 张鲁据汉、曹操攻拔、刘备夺汉中三大事件的共同舞台**。

**汉魏间张鲁据汉中**（约 191-215）——**张鲁（？-216）字公祺、沛国丰人、五斗米道第三代天师**（祖张陵创教于蜀汉鹄鸣山、父张衡传教、鲁继任）；**初平二年（191）前后张鲁据汉中**——以"五斗米道"治民、行义舍米肉、不设官吏、号"师君""治头大祭酒""祭酒""鬼卒"——是汉末割据中最有宗教色彩的一支。**张鲁在汉中二十余年**——筑关于阳平、与关中曹操、巴蜀刘璋皆有交涉；**阳平关之筑、张鲁居其首功**（《三国志·张鲁传》"鲁在汉中二十余年"）。

**建安二十年（215）秋七月**——**曹操西征张鲁**：**自陈仓道南下、过散关、至河池**——**攻阳平关**。**曹操初攻不利**——《三国志·武帝纪》记"始至关，鲁欲降；其弟卫不肯，将兵数万拒关、坚守。攻之不能拔"——**曹操叹"难于上青天"**（注：此叹见后世注引、正史《武帝纪》《张鲁传》原文皆无此六字、本 doc 严守"叹语后世附会"口径）。**后曹操军趁夜分路袭关、张卫军溃**——**张鲁奔南山入巴中、献宝藏归曹操**；**鲁拜镇南将军、封阆中侯**——**阳平关自此入曹魏**（《三国志·武帝纪》《张鲁传》本传）。

**建安二十四年（219）春**——**刘备北攻夏侯渊于阳平关、夏侯渊定军山**：**先主自阳平南渡沔水、缘山稍前、于定军山势作营**（《三国志·先主传》本传原文）；**夏侯渊（？-219）魏征西将军、镇汉中、与张郃并将魏军**；**法正智、黄忠勇——一战斩夏侯渊于定军山**（《三国志·黄忠传》"忠推锋必进、劝率士卒、金鼓振天、欢声动谷、一战斩渊"；《法正传》载**曹操**闻刘备斩夏侯渊后所叹"吾故知玄德不办此、必为人所教也"——是曹操对刘备此役胜由法正智的判断、非先主自言）。**定军山在阳平关东北数十里、汉中盆地西部**——是阳平关战区的核心战场；**斩夏侯渊后**——曹操亲征汉中、与刘备相持数月、自度不能克、引军退回长安。**阳平关入蜀汉**——**自此汉中入蜀汉、与曹魏 在阳平关 + 武兴 + 沔阳一带兵锋反复**。

**三国后阳平关续作关塞**——曹魏景元四年（263）钟会、邓艾灭蜀汉、亦经阳平关入汉中；**晋以下递有**。**北魏、北周、隋递筑关**——**北周或改"阳平"为"阳安"**（史源不一、本 doc 不预设）。**隋大业三年（607）罢州为郡、阳平关属顺政郡长举县**。**唐武德元年（618）复州、阳平关属兴州长举县**——**唐 755 时阳平关之名仍为主流**（本 doc 严守此口径）。

**唐人对阳平关的认知**：**755 时唐人凡读《三国志》者皆知阳平关之名**——是三国汉中拉锯战的核心物理 anchor。**自唐初至盛唐 138 年（618-755）阳平关无大战事**——是太平时代；**关城常驻兵约数百**——是日常守关 + 验商旅 + 收关税。

### 此刻

天宝十四载这一年的阳平关——**关塞严整、驻兵数百、扼金牛道与沮道分歧**。

**关城状态**：盛唐 755 时阳平关主体仍立、有戍兵 + 关吏 + 驿馆；具体形制史载零散。**关城外**：金牛道东南延伸、沮道西南延伸；**金牛道北口的散关 + 中段的阳平关 + 南口的剑门关**——是关中入蜀的三大关塞咽喉。**阳平关与剑门关合观**——剑门关守"绵州入益州"一关、阳平关守"兴州入梁州"一关、二关皆是入蜀必经。

**755 时阳平关上的政治氛围**：**阳平关是平时关塞**——关税收 + 商旅过关 + 戍兵巡逻；**关吏对入蜀的官员、僧侣、商旅** 多有验牒；**入蜀商旅自陈仓道南来、至兴州后**——多数转金牛道东折入汉中、少数循沮道西折入武都；**阳平关一关而扼两道**——日常驿务繁忙。

**755 末 + 756 初的阳平关命运**：**756 七月玄宗西奔入蜀**——**自长安出延秋门 + 西行经咸阳 + 马嵬 + 扶风 + 陈仓 + 入散关 + 至凤州河池 + 至兴州顺政**——**自兴州转金牛道东折经阳平关 + 经金牛驿 + 五丁关 + 至利州（广元）+ 剑州（剑阁，过剑门关）+ 绵州 + 汉州 + 至益州（成都）**。**玄宗一行过阳平关具体经历**——《资治通鉴》卷二一八记"丙戌至兴州"后未再细列阳平关一节、但**自兴州东南入金牛道必经阳平关**——本 doc 严守此口径、不预设玄宗在阳平关具体驻跸。

### 此刻意味

对天宝十四载的兴州 / 长举县人来说——阳平关是**自家境内的关塞**：日常入蜀商旅必过、戍兵守关、关吏验牒。对长安那一代士人来说——阳平关是**三国汉中拉锯战的物理 anchor**：**张鲁据汉、曹操西征、刘备夺汉中、黄忠斩夏侯渊于定军山**——这一系列三国关键事件都以阳平关 / 定军山为核心舞台。**盛唐诗中"阳平关"三字出现不多**——但与定军山合观、是诸葛亮、黄忠、夏侯渊、张鲁、曹操诸多三国人物的共同 anchor。

唐人尚不知道——再过半年、阳平关将经历**唐玄宗西奔入蜀**：**关塞从平时驿路转为皇帝逃难必经**；再过四年（759 冬）杜甫自秦州赴同谷再入蜀、亦经兴州境南下；再过 130 年（885）僖宗避黄巢入蜀、亦循陈仓—金牛线、过阳平关入蜀。**今阳平关镇在陕西汉中市宁强县**——**部分关墙汉魏夯土犹存 + 关楼为近代重建 + 三国汉中战场遗迹散布周边**。**唐人在 755 所见的那座阳平关**——千二百多年后**主体已残**：盛唐阳平关的物理终点是中国关塞史的常态。

## 子条目

### 张鲁据汉中（约 191-215）

**张鲁（？-216）字公祺**——五斗米道第三代天师；**初平二年（191）前后据汉中**——以宗教治民、行义舍米肉、不设官吏；**号"师君""治头大祭酒""祭酒""鬼卒"**——是汉末割据中最有宗教色彩的一支。**张鲁在汉中二十余年**——筑关于阳平、与曹操、刘璋皆有交涉；**阳平关之筑、张鲁居其首功**。**建安二十年（215）秋曹操西征**——张鲁初欲降、弟卫不从、拒关坚守；**关破后张鲁奔南山入巴中、献宝藏归曹操**；**曹操拜张鲁为镇南将军、封阆中侯**——**张鲁次年（216）卒**。**盛唐人对张鲁的认知**——多以其"宗教治国 + 阳平筑关 + 降曹保民"三事并提。

### 215 曹操西征张鲁（阳平关之战）

**建安二十年（215）秋七月**——**曹操自陈仓道南下、过散关、至河池、攻阳平关**：
- **初攻不利**：《三国志·武帝纪》"始至关、鲁欲降；其弟卫不肯、将兵数万拒关、坚守。攻之不能拔"；
- **传"难于上青天"叹**——后世注引、正史《武帝纪》《张鲁传》原文皆无此六字（本 doc 严守"叹语后世附会"口径）；
- **后操军趁夜分路袭关**——张卫军溃；
- **张鲁奔南山入巴中、献宝藏归曹操**——阳平关入曹魏。

**这一战是阳平关在三国史上的第一次大战**——曹操西征张鲁的核心一关；**自此后阳平关在中国军事史上反复出现**。

### 219 刘备夺汉中（定军山之战）

**建安二十四年（219）春**——**刘备自葭萌（今四川广元昭化）北攻夏侯渊于阳平关 + 定军山**：
- **先主自阳平南渡沔水、缘山稍前、于定军山势作营**（《三国志·先主传》本传原文）；
- **法正智**——劝刘备据定军山高地、夏侯渊不能拔；
- **黄忠勇**——一战斩夏侯渊于定军山（《三国志·黄忠传》"忠推锋必进、劝率士卒、金鼓振天、欢声动谷、一战斩渊"）；
- **斩夏侯渊后**——曹操亲征汉中、与刘备相持数月、自度不能克、引军退回长安；
- **刘备遂有汉中**——**七月自立汉中王**（《三国志·先主传》本传）。

**定军山在阳平关东北数十里、汉中盆地西部**——是阳平关战区的核心战场；**与阳平关合观、即三国汉中拉锯战的完整战场**。**盛唐人凡读《三国志》者皆知此役**——是黄忠"老当益壮"的物理 anchor。

### 金牛道与沮道的分歧

**金牛道**：**关中入蜀的"四道"之一**（金牛道、米仓道、荔枝道、子午道——以入梁州为枢纽计）；**自陈仓道南下、过散关、过河池、至兴州**——**自兴州东南循金牛道经阳平关 + 金牛驿 + 五丁关至利州**（具体里程史源略异）；**金牛道是关中入蜀的最稳定驿路**——也是 756 玄宗西奔入蜀的实际路线。

**沮道**：**汉中-武都段的山道**——**自阳平关西南循沮水 + 白马河谷地 + 入武都郡**（今甘肃武都一带）；**沮道在唐时为支线驿道**——较金牛道流量小、但是入武都氐羌故地的最直接通道；**沮道东端的物理 anchor 即阳平关**。

**阳平关恰在此两道分歧前一段**——一关而扼两道、是唐 755 时阳平关地位的根本。

### 阳平关 vs 阳安关（名号辨）

**关于"阳平关 vs 阳安关"**——**一说北周以后改"阳平"为"阳安"**（《通典》《元和郡县图志》一带或有此说）；**一说阳平关在汉魏 + 阳安关在北周以下为分立关塞**（地方志或有此说）；**一说二关同地异名、北周后通称阳安、唐 755 时民间口语仍呼阳平**。**本 doc 严守"唐 755 时主流仍称阳平关"口径**——史源不一、不预设二关分立、亦不预设北周以后已改名。**盛唐 755 时阳平关之名仍见于《元和郡县图志》(813)之前的隋唐地理书**——本 doc 取通称作历史 anchor。

### 阳平关税与商旅

**唐阳平关收商旅过关税**——这是关塞的常规财政职能；**阳平关税**主要针对入蜀 / 出蜀 + 入武都 / 出武都的商货 + 公私往来。**盛唐阳平关税收**史载零散；**与潼关、散关、玉门关等比、阳平关税收规模较小**——因兴州境本身户口稀、入蜀商旅密度较散关已减；**但阳平关一关扼两道**——日常驿务仍重。

### 三国阳平关地望辨

**阳平关地望历来有争论**——**一说在陕西宁强县阳平关镇**（主流说，本 doc 取此）；**一说在甘肃成县境**（少数说，史源不出唐宋）；**陕西省考古研究所 1980s-2010s 多次勘察**——确认陕西宁强县阳平关镇有汉魏夯土城基 + 关楼遗址 + 三国战场散布；**本 doc 严守"陕西宁强阳平关镇"主流说**——不预设其他地望。

## 历史长河（755 → 今）

**中晚唐**　**756 七月玄宗西奔过阳平关**——是阳平关在盛唐唯一一次最有名的过往；**至德二载（757）末玄宗返长安**——亦由原路过阳平关回京。**广明元年（880）黄巢入长安**——**唐僖宗西奔成都**、亦走陈仓—金牛线、过阳平关一路；**这是盛唐"玄宗 → 僖宗" 两次天子过阳平关入蜀的连续**。**中晚唐阳平关仍为兴州长举县关塞**——但因兴州户口稀、阳平关地位较散关、剑门关已减。

**两宋**　**北宋阳平关属兴州长举县**；**南宋绍兴 - 乾道间**阳平关为南宋川陕防线后方关塞——**吴玠、吴璘兄弟守关陕四十年**、与和尚原、仙人关、阳平关一线呼应；**陆游守散关时**——亦或念及阳平关三国旧迹（《剑南诗稿》或有涉、本 doc 不具引）。**元至元二十年（1283）废兴州为略阳县、阳平关属略阳县**——"阳平关"地名延续。

**金元明清**　元明清间阳平关续作关塞、但规模渐降；**明代阳平关属汉中府略阳县**；**清乾隆、嘉庆间数次重修阳平关关楼**——今所见关楼主体格局多定于此时。**清陕西宁强县形成阳平关镇地名**——是清代行政区划的延续。**明清间三国阳平关 + 定军山 + 武侯墓 + 黄忠庙等纪念物事**——多在汉中盆地西部陆续修建；**阳平关镇内或外、附会"张鲁城""曹操营""黄忠斩夏侯渊处"等地名**——多为明清方志附会、不是 755 唐人所见。

**民国至今**　**1949 年后阳平关古城遗址列陕西省文物保护单位**。**1980s-2010s 陕西省考古研究所 + 宁强县文物管理所 多次勘察 + 部分修缮**——确认阳平关汉魏夯土城基 + 关楼遗址 + 三国战场散布；**2013 年阳平关古城遗址列全国重点文物保护单位**（与定军山战场遗址合并申报）。**今陕西汉中市宁强县阳平关镇**——所见的是**阳平关古道遗址 + 关楼为近代重建 + 部分关墙汉魏夯土犹存 + 三国战场散布周边**。**今宝成铁路 + 国道 G108 + 川陕公路过阳平关镇**——是现代驿道的延续；唐人在 755 所见的那座阳平关，千二百多年后主体已残，盛唐阳平关的物理终点是中国关塞史的常态。
