# 三门峡砥柱

## 一句话

**黄河中游最险一段** + **唐漕运最大险关**——**陕郡治陕县 + 平陆县境黄河中流**、河中三巨石（人门 / 神门 / 鬼门）分流成"三门"、三门西约一二唐里中流有孤峰"砥柱"——**《尚书·禹贡》"至于砥柱"是中国地理的最早 anchor 之一** + **盛唐裴耀卿三门转运法的物理 anchor**；**1957 三门峡大坝始建 + 1960 蓄水后没入水下**——4 千年地理一坝改变。

## 形貌

三门峡砥柱在黄河中游——**陕郡治陕县 + 平陆县境黄河中流**、距陕县治约三十唐里（今河南三门峡市黄河三门峡段 + 砥柱）。**黄河自西向东**——出潼关、过陕郡、河道急转、自此入崤山一线丘陵；**河道中流有 3 巨石**——南北分立、将黄河分为三股急流：

- **人门**（南股、最缓）——唐时漕船可勉强通行；
- **神门**（中股、最险）——唐时漕船多覆没于此；
- **鬼门**（北股、亦险）——唐时基本不行船。

**3 巨石合称"三门"**——是黄河中游最狭最险一段；**三门西约一二唐里**——中流有孤峰一座、谓之"**砥柱**"。砥柱与三门**同段不同石**：**三门**是河中 3 巨石分立、分流；**砥柱**是三门西的中流孤峰、特立中流不动。

**砥柱形貌**：唐时中流孤峰一座、高出水面数十丈、左右黄水奔流、孤峰岿然——是"中流砥柱"成语的物理来源。**砥柱与三门同段、皆陕郡治陕县 + 平陆县境**——陕县在黄河南岸、平陆在黄河北岸（**755 时平陆县属河东道河东郡（蒲州）**——天宝元年/742 改蒲州为河东郡、至德元载/756 才升河中府；陕县属河南道陕郡）；砥柱 + 三门跨南北两岸、是两州（郡）两道的天然界限。

**唐人入陕郡所见**：自长安东出 + 过潼关 + 入弘农郡 + 至陕郡治陕县——驰道在黄河南岸黄土塬上、距河岸约一二唐里高；自陕县北望黄河、河中三门 + 砥柱皆在目；**北岸是平陆县 + 中条山一线**——南北两岸隔河相望。**唐人最深的印象**：黄河水黄、三门湍急、砥柱孤立——是"中流砥柱"四字的物理来源。

**地势**：黄河中游、自西向东、过陕郡入崤山——**南岸是崤山一线**、**北岸是中条山一线**；**砥柱 + 三门**所在的河段——崤山 + 中条夹峙、河道最狭、水流最急；**自此东出**——至洛阳一线渐入伊洛盆地、河道渐宽。

**物候**：黄河水量**春枯夏涨**——**春**水浅、漕船可通三门；**夏秋**水涨、三门险倍增、漕船多覆；**冬**水浅、三门可见。**唐漕运多择春行 + 秋末行**——避三门夏涨。

**气候**：温带季风、四时分明——**春**多沙、河水带泥；**夏**多雨、黄河水涨、三门险段尤甚；**秋**高、水退、漕运渐复；**冬**有薄雪、黄河可冰、漕船停泊。**三门峡一段的水文**——是黄河中游 + 黄土高原下泄的总枢、含沙量极高、是"黄河之黄"的物理 anchor。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

砥柱在唐人的回望里——是**中国地理的最古 anchor 之一** + **黄河漕运最大险段** 两重。

**砥柱的最古文献**：
- **《尚书·禹贡》**——载大禹"导河积石、至于龙门、南至于华阴、**东至于砥柱**"——砥柱是大禹治水的下游 anchor 之一；中国"九州"地理的最早纪念地；
- **《史记·夏本纪》《汉书·地理志》**等皆承禹贡之载——砥柱作为中国地理的不变 anchor、自先秦至汉魏一以贯之；
- **《水经注·河水》**——北魏郦道元详载"河水又东、至砥柱山"——砥柱 + 三门作为黄河中游最险段的描述、自此入中古地志之常制。

**砥柱的漕运史**：
- **战国 + 秦汉**——黄河漕运已用、但砥柱险段为漕运瓶颈、官民皆苦之；
- **东汉建武十一年（35）**——光武帝令在三门凿石、修河道——是历史上第一次大规模凿砥柱、险减但未绝；
- **北魏 + 北齐**——多次凿三门、修漕道——皆无大成；
- **隋大业元年（605）**——隋炀帝大规模开通济渠 + 永济渠——**自此黄河漕运的核心改至通济渠**（洛阳至淮水），砥柱险段为陆运 + 水运并行的过渡段；
- **唐承隋制**——长安 + 洛阳间漕运沿黄河中游、过砥柱 + 三门是必经险段；**唐前期漕运每年损耗约 8 万石**（《新唐书·食货志》）——皆因砥柱 + 三门覆没。

**开元二十二年（734）裴耀卿三门转运法**——是唐漕运史最大一次改革：
- **裴耀卿任江淮、河南转运使**——奏改漕运法；
- **置四仓**：**河阴仓**（黄河南岸；汜水东、广武山下；距洛阳约二百里）——东端起点、纳江淮 + 河南漕米；**柏崖仓**（黄河北岸；河清县境）——河阴对岸之北岸仓；**集津仓**（三门东岸；今河南三门峡市集津渡一带）——三门东仓、漕船至此卸粮；**盐仓**（三门西岸；今山西平陆县境）——三门西仓、陆运过三门后再装船西上；
- **陆运十八里绕三门险段**——**自集津仓陆运至盐仓**、绕过三门险段；**节级转输**——河阴仓 → （水运）→ 集津仓 → （陆运十八里）→ 盐仓 → （水运）→ 太仓（长安）；
- **效果**：自此**凡三岁、运米七百万石**（《新唐书·食货志》：「凡三岁、运米七百万石」）、漕运损耗大减——**关中粮食有保障**；**自此玄宗朝再未"就食洛阳"**（开元二十四年/736 十月玄宗最后一次自洛阳还长安 + 自此再未东巡 19 年至 755）。

**裴耀卿三门转运法的历史意义**——是唐廷在地理 + 制度间的一次大调适：黄河中游有砥柱 + 三门险段、不可改变；唐廷以**陆运十八里**绕之、节级转输——以制度补地理之不可改。**自此中国漕运的核心范式**——"水陆联运 + 节级转输 + 仓储中转"——为后世宋元明清漕运体系奠定基础。

**砥柱 + 三门的文化意义**：
- **"中流砥柱"成语**——自先秦至唐已成定语、谓中流孤峰特立、风浪不动；引申为"困境中坚守不动者"——是中国文人最常用的怀古 + 寄志意象之一；
- **唐人咏砥柱**——李白 + 杜甫 + 韩愈等皆有咏砥柱诗（部分作于 755 之后、但 755 时砥柱已是怀古 anchor）。

### 此刻

天宝十四载这一年的砥柱——**盛唐漕运按例运行 + 中流砥柱依然 + 即将随安史之乱而漕运中断**。

**砥柱与三门**：盛唐 755 时砥柱 + 三门依然在陕郡治陕县 + 平陆县境黄河中流；**裴耀卿三门转运法已运行 21 年**（开元二十二年/734 至天宝十四载/755）；**四仓体系按例运行**——河阴仓 + 柏崖仓 + 集津仓 + 盐仓 四仓昼夜不息、漕米经陆运十八里（集津 → 盐仓）绕三门、节级西上至长安太仓。

**漕运状态**：盛唐 755 时**三年累岁七百万石**（《新唐书·食货志》「凡三岁、运米七百万石」、即三年合计七百万、岁均约二百三十余万）——是中国漕运史的一次高峰；**漕运者**——江淮漕米 + 河南漕米 + 河北漕米皆经黄河中游过砥柱段；**漕夫 + 役夫**——四仓各驻数千人、集津 → 盐仓陆运一段最为艰苦。

**砥柱的可见状态**：盛唐 755 时砥柱孤峰依然立于黄河中流——**唐人过陕郡 + 平陆**、皆可见砥柱；**长安-洛阳驰道**（详见 [长安-洛阳驰道](../transport/长安-洛阳驰道.md)）过陕郡治陕县——驰道距砥柱约三十唐里、不直接经砥柱、但是唐人入陕郡必登高一望砥柱、是盛唐两京间的怀古 anchor。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳已成必然**（11 月初九即起）；**叛军自范阳南下**——**十二月丁酉陷东京洛阳**（公历 756 年 1 月）；**自此漕运东段失**——黄河漕运中断；**至德元载（756）六月哥舒翰潼关之败**——关中失、漕运全断；**至德二载（757）九月、十月**唐军收两京——漕运复、但已不复盛唐繁华。

### 此刻意味

对天宝十四载的陕郡 + 平陆人民来说——砥柱是**最熟悉的乡土 anchor**：朝起暮归、抬眼即见；漕粮过境、是赖以为生的物理空间。对长安那一代士人来说——砥柱是**大禹治水的最古 anchor + "中流砥柱"成语的物理来源**：**自禹贡 + 至盛唐 + 已逾 3 千年**、砥柱依然孤立中流、是中国地理最不变的纪念地。**盛唐人在 755 所见的那砥柱**——是 4 千年来未改的孤峰；**漕运四仓**（河阴 + 柏崖 + 集津 + 盐仓）——是裴耀卿 21 年前的改革遗产、关中粮食的物理保障。

唐人尚不知道——再过半年漕运将断；至德 / 乾元间三门转运法虽复、但漕运量已减至盛唐三分之一；**自此中唐漕运的核心改至江淮经汴州 + 徐州 + 运河至洛阳**——砥柱段地位下降。

**今游人寻砥柱**——所见的是**三门峡水库 + 1957 三门峡大坝**：**1957 年三门峡水利枢纽始建 + 1960 蓄水**——砥柱与三门皆没入水下；**今三门峡市黄河三门峡景区**——可乘船至库区、低水位时可见砥柱顶端、高水位时全没——**4 千年地理一坝改变**、是 20 世纪中国水利工程对古地理最大的一次改写。**自禹贡 + 至盛唐 + 至宋元明清** + 砥柱孤立中流不改；**1957-1960 一坝变之**——是中国地理史最大的一次"古今分野"。

## 子条目

### 三门（人门 + 神门 + 鬼门）

**黄河中游 + 陕郡治陕县 + 平陆县境河中 3 巨石**——南北分立、将黄河分为三股急流：**人门**（南股、最缓）——唐时漕船可勉强通行；**神门**（中股、最险）——唐时漕船多覆没于此；**鬼门**（北股、亦险）——唐时基本不行船。**3 巨石合称"三门"**——是黄河中游最狭最险一段。**三门** + **砥柱**同段、不同石——三门是河中 3 巨石分立、分流；砥柱是三门西的中流孤峰、特立中流不动。**三门的物理状态**：唐时河中 3 巨石高出水面数丈至数十丈、左右奔流；**1957 三门峡大坝始建 + 1960 蓄水**——三门皆没入水下。

**三门的命名**：人门 + 神门 + 鬼门——是唐前 + 唐时已成定名（《水经注》载"三门"、未细分"人 / 神 / 鬼"；细分应在中古之间渐成）；**唐人过三门** + 多自人门勉强通行、神门 + 鬼门则视为绝地；漕夫 + 役夫每过三门、必先祭河神 + 焚香——是黄河漕运的物理仪式 anchor。

### 砥柱（中流孤峰）

**三门西约一二唐里 + 黄河中流孤峰**——**砥柱**。砥柱与三门**同段不同石**：三门是 3 巨石分流；砥柱是中流孤峰特立。**砥柱的最古载**：《尚书·禹贡》"导河积石、至于龙门、南至于华阴、**东至于砥柱**"——是中国地理的最古 anchor 之一、大禹治水的下游纪念地。**"中流砥柱"成语**——自先秦至唐已成定语；引申为"困境中坚守不动者"。**砥柱的物理状态**：唐时中流孤峰一座、高出水面数十丈、左右奔流、孤峰岿然；**1957 三门峡大坝始建 + 1960 蓄水**——砥柱没入水下、低水位时可见顶端。

### 裴耀卿三门转运法（开元二十二年/734）

**唐漕运史最大一次改革**——开元二十二年（734）**裴耀卿**任**江淮、河南转运使**——奏改漕运法、置四仓节级转输 + 陆运十八里绕三门：
- **河阴仓**（黄河南岸；汜水东、广武山下；距洛阳约二百里）——东端起点仓、接江淮 + 河南漕米、再沿黄河西上；
- **柏崖仓**（黄河北岸；河清县境）——河阴对岸之北岸仓、与河阴仓互为犄角；
- **集津仓 = 三门东仓**（三门东岸；今河南三门峡市集津渡一带）——漕船至此卸粮、起陆运过三门；
- **盐仓 = 三门西仓**（三门西岸；今山西平陆县境）——陆运过三门后、漕米至此再装船西上；
- **陆运十八里**——**自集津仓陆运至盐仓**、绕过三门险段；节级转输：河阴 →（水）→ 集津 →（陆十八里）→ 盐仓 →（水）→ 太仓（长安）；
- **效果**：自此**凡三岁、运米七百万石**（《新唐书·食货志》「凡三岁、运米七百万石」）、漕运损耗大减——**关中粮食有保障**；**自此玄宗朝再未"就食洛阳"**。

**注**：**裴耀卿三门转运四仓** = **河阴仓 + 柏崖仓 + 集津仓 + 盐仓**——是开元二十二年三门转运法的核心仓储体系；**陆运十八里 = 集津 ↔ 盐仓**（绕三门险段）。**河阴仓** ≠ **含嘉仓**：河阴仓在汜水东黄河南岸（距洛阳约二百里）；含嘉仓在洛阳东都郭城内（详见 [洛阳](../cities/洛阳.md)）——二者不同仓。

### 漕运险段（盛唐年损 8 万石 → 改后三年七百万石）

**唐前期漕运的物理瓶颈**——黄河中游过砥柱 + 三门段、漕船覆没、每年损耗约 8 万石（《新唐书·食货志》）；**开元二十二年裴耀卿改漕运法**——以陆运十八里绕三门险段、节级转输；**改后三年累岁七百万石**（《新唐书·食货志》「凡三岁、运米七百万石」）——是中国漕运史的一次高峰。**漕运的物理 anchor**：自江淮经通济渠至洛阳含嘉仓 + 自洛阳沿黄河西上至河阴仓 + 再西上至集津仓 + 陆运十八里至盐仓 + 再装船西上至长安太仓——**节级水陆联运**。

### 砥柱与"中流砥柱"

**"中流砥柱"成语**——自先秦《晏子春秋》《史记》等已成定语、谓中流孤峰特立、风浪不动；引申为"困境中坚守不动者"。**砥柱作为物理来源**——黄河中游孤峰一座、左右奔流、孤峰岿然——是成语的物理 anchor。**唐人咏砥柱**：李白《公无渡河》《行路难》中有黄河 + 砥柱意象；杜甫《北征》《潼关吏》等亦写黄河中游；韩愈《送水陆运使韩侍御归所治序》中亦载砥柱漕运（韩愈为中唐人、写漕运 anchor）。**砥柱是中国文人最常用的怀古 + 寄志意象之一**——自先秦至盛唐至中晚唐至今、3 千年未改。

### 砥柱与三门的天然界限（陕州 + 蒲州两州两道）

**砥柱与三门所在的黄河中游一段**——**南岸是河南道陕郡治陕县**、**北岸是河东道河东郡（蒲州）+ 平陆县**；**砥柱 + 三门跨南北两岸**——是两州（郡）两道的天然界限。**唐时陕郡太守驻陕县**（详见 [陕州](../cities/陕州.md)）——管南岸；**河东郡太守驻河东县**（今山西永济；天宝元年/742 改蒲州为河东郡、至德元载/756 升河中府——755 此刻正名为河东郡）——管北岸 + 平陆县。**砥柱 + 三门**——是关内道 + 河南道 + 河东道三道交界一段（潼关 + 陕郡 + 河东郡 + 平陆一线）的物理 anchor。

### 安史中漕运中断（756-757）

**天宝十四载十一月**（农历；公历 755 年 12 月）安禄山起兵于范阳——**十二月丁酉陷东京洛阳**（公历 756 年 1 月）——**自此黄河漕运东段失** + 含嘉仓 + 集津仓皆入叛军；**至德元载（756）六月哥舒翰潼关之败**——关中失、漕运全断；**至德二载（757）九月、十月**唐军郭子仪 + 李光弼 + 回纥兵收两京——漕运复、四仓体系亦复。**自此中唐漕运**——量减至盛唐三分之一；**贞元 / 元和间**漕运改至江淮经汴州 + 徐州 + 运河至洛阳——**砥柱段漕运地位下降**；但四仓体系仍间断运行至唐末。**砥柱 + 三门**——作为漕运险段的角色渐淡、但作为怀古 anchor 不改。

## 历史长河（755 → 今）

**中晚唐**　**至德二载（757）秋唐军收两京**——漕运复；但漕运量已减至盛唐三分之一；**贞元 / 元和间**漕运改至江淮经汴州 + 徐州 + 运河至洛阳——**砥柱段漕运地位下降**；但三门转运法的四仓体系仍间断运行至唐末。**唐末黄巢起义**——漕运再断；**砥柱 + 三门**作为漕运险段的角色渐淡、但作为怀古 anchor 不改。

**五代 + 两宋**　**北宋初定都开封**——漕运的核心改至汴河 + 黄河下游一线；**砥柱段的漕运**渐废；但作为"中流砥柱"的文化 anchor、宋人咏砥柱诗仍多——苏轼 + 黄庭坚 + 陆游等皆有咏砥柱句。**北宋元丰间**——曾再凿三门、修漕道、亦无大成。

**金元明清**　**元代**漕运改至大运河（济宁至大都）一线——砥柱段已非主要漕运通道；**明清间**砥柱 + 三门作为"陕州八景"之一——成为地方文化 anchor；**清乾隆间**——砥柱 + 三门作为山陕商旅 + 漕运残余的物理空间继续存在。

**民国至今**　**1934 年陇海铁路通车至潼关**——铁路过陕州 + 三门峡一线、与古黄河漕运大体平行（但走南岸黄土塬）；**1955 年**——中苏合作的"三门峡水利枢纽"立项；**1957 年 4 月**——三门峡大坝始建；**1960 年 9 月**——蓄水——**砥柱 + 三门皆没入水下**；**1960s 后**——库区淤积严重、坝功能改造、低水位时砥柱顶端可见、高水位时全没。**今三门峡市黄河三门峡景区**——可乘船至库区、寻砥柱顶端；**自禹贡 + 至盛唐 + 至宋元明清 + 4 千年砥柱孤立中流**——**1957-1960 一坝改变**：是 20 世纪中国水利工程对古地理最大的一次改写。**今游人至三门峡**——所见的是**水库 + 大坝**：盛唐人在 755 所见的那砥柱 + 三门、已成水下遗迹。
