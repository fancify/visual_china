# 赤岭碑

## 一句话

鄯州西陇右道与吐蕃边界**赤岭**（今青海日月山）——**唐开元二十一年（733）金城公主请定唐蕃疆界、次年唐蕃于赤岭分界立碑**；碑文与形制未传、今实物已不存——**755 时仍立、是唐蕃边境的物理 anchor**。

## 形貌

赤岭在**鄯州（陇右节度治）西约 200 里**——海拔约 3500 米的一道高原山岭，是**青海湖东南屏 + 黄河上游分水岭**。**今地理坐标**：青海省海南藏族自治州共和县与湟源县交界一带的**日月山**（明清后通称"日月山"，唐人称"赤岭"——以山土赭红得名）。

地势：**赤岭由东向西**——东为黄土高原边缘（陇右道核心区，麦田、村落、驿道）；**西为青藏高原边缘**——草原、湿地、青海湖东南畔。**这是中国地理的一道重要分水岭**——岭东属黄河上游农业带、岭西属青藏高原牧业带；亦是**农耕文明与游牧文明的天然界**。

**唐蕃以赤岭为界**——意指岭东归唐、岭西归吐蕃。**唐蕃使节往来必经此岭**——长安→鄯州→赤岭→青海湖→那曲→逻些是唐蕃常行驿路；**赤岭碑就立在赤岭山口**——是唐蕃使节经过时必见的边境标识。

**赤岭碑的形制 + 碑文**——史载未传、实物不存；后世推测有"双方各立一碑（一汉文、一吐蕃文）"或"一碑双语刻"两说，本 doc 不坐实。**碑材**：当地花岗岩或石灰岩（推测）。**碑高约盛唐石碑常规 1.5-2.5 米**——具体史载未详。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

赤岭碑在唐人的回望里——是**唐蕃 130 年外交关系中具体的物理边界**。

**唐蕃边境的早期**：自**贞观十五年（641）文成公主入藏**起、唐蕃和亲——边境模糊；**贞观至高宗朝（641-683）**唐蕃多有边境摩擦但未明界；**高宗咸亨元年（670）大非川之战 + 仪凤三年（678）青海之战**等大战、唐蕃多次冲突——**边境无明界、双方各按实际控制线行事**。

**金城公主入藏与立界**：**景龙四年（710）金城公主出降吐蕃赞普赤德祖赞**——是唐蕃第二次和亲。**开元二十一年（733）正月金城公主请定唐蕃疆界**（《旧唐书·吐蕃传》《资治通鉴》明文）——金城公主自吐蕃廷请立界以减边境冲突。**唐玄宗准许**——派**工部尚书李暠为使**赴吐蕃定盟；**次年（开元二十二年/734）唐蕃于赤岭分界立碑**——**双方约定以赤岭为界、岭东归唐、岭西归吐蕃**（"请立"与"实际立碑"年份史源略异、本 doc 取"733 请、734 立"通行口径）。

**这是唐蕃外交史上第一座有具体地理坐标的界碑**——是金城公主作为"和平公主"的实际外交成就。**自 734 立碑至 755**已约 21 年——**期间唐蕃数次冲突**：
- **开元二十五年（737）河西节度使崔希逸破吐蕃**——**赤岭碑未毁但盟实际已破**；
- **天宝间哥舒翰兼河西、陇右节度（753）**对吐蕃压力达盛唐最高峰；
- **天宝十四载（755）九月赤德祖赞卒**——唐蕃关系再变。

### 此刻状态

天宝十四载这一年的赤岭碑——**仍立赤岭山口、约 21 年之实物**。

**碑的物理状态**：开元二十二年（734）立、至 755 已约 21 年——经青海高原二十余个寒暑、风沙雪霜——具体表面磨损史载未详；**碑作为"唐蕃界碑"的地标性意义仍在**——盛唐人 / 吐蕃人 / 商旅 / 使节经过赤岭山口时仍以此碑为唐蕃边境标识。

**实际边境状态**：**734 立碑后双方多次冲突**——但赤岭这条物理界线大致维持；**755 时双方以赤岭为界**——具体边防部署、驻营位置史载未详。**这一年的赤岭山口**——商旅小心通行；唐蕃使节定期往返——金城公主已卒于天宝元年（741；藏文文献略异），但其立碑事 + 唐蕃和亲传统仍在。

**这一年的赤岭——是唐蕃边境最具象的物理 anchor**。再过几个月——安史之乱起、河西陇右兵东调救京、**赤岭边境守备日空**；**广德元年（763）吐蕃东扩**——攻陷河西陇右诸州、**唐蕃实际接触线东移**——赤岭碑随之失去"边界"的实际功能；之后吐蕃东境推至秦州、凤翔一线、**赤岭碑废为"前唐蕃界"的遗存**。

### 此刻意味

对天宝十四载的鄯州人来说——赤岭碑是**州西二百里的国际界碑**：自此岭东、即唐；岭西、即吐蕃；商旅、使节、戍兵 21 年间都过此碑而行。对长安那一代士人来说——赤岭碑是**唐蕃和亲与外交的具象**：金城公主 710 入藏的远行、733 请 734 立的盟约、汉蕃文字（双语 / 各立 二说）的国际界碑——皆系于此一碑。**盛唐人嘴里的"赤岭"**——主要是**唐蕃边界**的代名词、有时直接以"赤岭"代"唐蕃边境"。

唐人尚不知道——再过 8 年（763）赤岭碑会失去边界功能；再过 50 年（约 9 世纪初）唐蕃边境彻底重定（长庆会盟 821-822 + 大昭寺前立"唐蕃会盟碑"）——但**赤岭碑的实物在中晚唐至五代间渐毁**（具体何年史载未详；可能毁于唐末战乱或宋元）；再过千余年（明清），**赤岭因山土赭红 + 日月双形**通称"日月山"——民间附会"文成公主在此泣别故乡" 的传说渐成；再过千二百年（21 世纪）——**唐蕃界碑实物已不存**；今日月山有现代立"日月山碑"作纪念——非盛唐原物。**唐人在 755 所见的那块汉蕃文字界碑**——千余年间彻底湮没。

## 子条目

### 汉蕃文字（双语 / 各立 二说）

**赤岭碑的形制特点**——盛唐立碑事的具体形制史载未详：后世推测有"双方各立一碑（一汉文、一吐蕃文）"或"一碑双语刻"两说、史源不一、本 doc 不坐实。**汉蕃文字碑在唐代外交史上的位置**：与中晚唐**"唐蕃会盟碑"**（长庆元年/821 + 二年/822 立、今犹存于大昭寺前——三面分刻汉文、藏文、唐蕃使节署名）形成"唐蕃外交两碑"。**赤岭碑是首碑（733 请 / 734 立）、会盟碑是次碑（821-822）**——两者勿混。**赤岭碑的具体碑文内容**——史载未传、本 doc 不预设。

### 金城公主（赤岭碑的发起人）

**金城公主（约 698-741）**——唐中宗朝宗室女、**景龙四年（710）出降吐蕃赞普赤德祖赞**；婚后留吐蕃约 30 年；卒于天宝元年（741）。**金城公主入藏后扮演唐蕃和平的关键人物**：
- 入藏后**将释迦摩尼十二岁等身像自小昭寺迁至大昭寺**（见 [大昭寺红山宫](./大昭寺红山宫.md)）；
- **开元二十一年（733）请立赤岭碑**——是其外交最具体的成就；
- **天宝元年（741）卒于吐蕃**——其卒后唐蕃关系再变。

**金城公主的历史地位**——介于文成公主（641 入藏的政治和亲）与中晚唐和亲公主之间；其立碑赤岭一事是唐蕃外交史的关键节点。

### 赤岭（今日月山）地理

**赤岭得名**：以山土赭红得名（"赤"即红）。**明清后通称"日月山"**——传说山口有圆形大石如日 + 弯形如月、并以文成公主在此过岭时泣别故乡、掷手中日月双镜化山的传说。**实际地理**：海拔约 3500 米、海拔陡然抬升的高原边缘——**是黄土高原与青藏高原的天然界**。**今日月山**是青海重要的旅游与历史地标——青海湖东 + 鄯州故地 之间。

### 与"唐蕃会盟碑"区分（勿混）

**赤岭碑（733）**：唐玄宗朝、立于鄯州西赤岭山口、唐蕃以赤岭为界——是唐蕃边境的物理 anchor。**今实物已不存**。

**唐蕃会盟碑（821-822）**：长庆元年（821）唐穆宗与吐蕃赞普赤祖德赞和议、长庆二年（822）双方使节在长安 + 逻些两地分别盟誓——**逻些碑立于大昭寺前**（三面分刻汉文、藏文、双方使节署名）、**长安碑立于唐廷**。**今逻些碑犹存于大昭寺前**（西藏自治区第一批重点文物保护单位），长安碑已不存。

**两碑分属唐蕃外交的两个时期**——赤岭碑是开元盛期、会盟碑是中晚唐安史后；两者勿混。

### 鄯州 + 陇右节度（唐蕃前线）

**鄯州（陇右节度治、治湟水县/今青海乐都）** 是 755 时唐蕃边境最重要的州——**辖鄯、廓、河、洮、岷、秦、渭、临、武、成 等 10 余州**。**天宝十二载（753）哥舒翰兼河西、陇右两镇节度使**——13 万兵驻河西陇右、是盛唐西部所有兵力的最前线总根脚。**赤岭碑在鄯州西**——是陇右节度的西界标识；**755 时鄯州 + 赤岭一线是唐蕃前线最稳的物理 anchor**。再过几个月——河西陇右兵东调救京、**赤岭边境守备日空**。

## 历史长河（755 → 今）

**中晚唐**　**广德元年（763）后吐蕃陷河西**——赤岭碑随**唐蕃实际接触线东移**失去"界碑"实际功能；**唐蕃东境推至秦州、凤翔一线**。**长庆元年/821 + 二年/822 唐蕃会盟**——立"唐蕃会盟碑"于长安 + 大昭寺前；自此唐蕃边境重定。**赤岭碑在此期间** 仍立赤岭、但已是"前界碑"性质；具体何时毁、史载未详。

**两宋至元**　**吐蕃分裂期（842-1239）**——青藏高原各派禅修传承立；**赤岭一带归吐蕃 / 各派势力**；**赤岭碑可能毁于唐末至五代或宋元间**——具体史载未详。**唐宋人对赤岭碑的具体认知** 渐淡——成为"唐蕃外交的一段史志记忆"。

**元明清**　元代青藏高原归元；**明清间"赤岭"渐通称"日月山"**——明万历《青海行都司志》等地志已用"日月山"。**清代日月山是茶马互市、汉藏贸易的重要关道**——农业带与牧业带的交界仍在此线。**清乾隆间**有官员到日月山考察、寻赤岭碑遗——**未得**。

**民国至今**　1949 年后青海建省。**1984 年青海省在日月山立现代纪念碑**——以纪念赤岭碑（733 请 / 734 立）+ 文成公主 + 金城公主和亲事——非盛唐原物。**今日月山是青海热门景点**——有"日月亭"、"回望长安"碑、"文成公主像"等现代纪念建筑。**今游人来日月山**——所见的是高原山口 + 现代纪念碑 + 远眺青海湖；**盛唐人当年所见的那块汉蕃文字界碑实物——千余年间彻底湮没**。**赤岭这一物理界标本身仍在**——黄土高原与青藏高原的分水线、一千二百多年未易；但**那块碑——唐蕃外交的具体物证——今游人只能想象其曾立**。
