# 镐京

## 一句话

**西周宗周王都遗址**——周武王灭商后立都于沣水东岸、西周历十二王（十一世、孝王是夷王叔父、支系）居此 275 年；**BC 771 犬戎入镐、周幽王死于骊山下、平王东迁洛邑、镐京自此废**；**755 时已废约 1524-1525 年**（BC 771 犬戎入镐约 1525 年、BC 770 平王东迁约 1524 年；无公元 0）——城垣无存、仅余沣水东岸郊原一片；唐人知其大概方位、视之为"周天子故都"——盛唐天朝叙事的根。

## 形貌

镐京在唐京兆府**长安县境郊原**——具体在**长安城西南约 30 唐里、沣水东岸**（今西安市西南斗门镇一带）。**地势**：关中平原中部、渭水南岸的一片冲积平原；沣水自南山（终南山）北流、过镐京旧地、再北汇入渭水——西周时镐京正在沣水东岸的台地上。**今所见**：盛唐 755 时镐京旧地一片郊原——**城垣已废千余年**、地表无明显城迹；唐人所辨识者、主要是**沣水**这条河 + **镐池**这一汪水——皆周天子时代的物理遗存。

**与丰京的方位**：**丰京在沣水西岸**——周文王末年所建（"文王作邑于丰"）；**镐京在沣水东岸**——周武王灭商后所建。**两京隔沣水相望**——合称"丰镐"。**功能上**：传统说法以丰京为宗庙所在、镐京为天子治所——但近世考古口径以两京功能多有重叠、共同构成西周早期的双京制。**两京间距约数里**——周天子日常往来如同一城。

**镐池 + 镐京方位**：唐《元和郡县图志》卷一载"周武王宫即镐京、在县西北十八里"——**以 唐长安县治 为参照点**记"县西北十八里"；**以 唐长安城外郭 为参照点**、镐京遗址实际方位在西南方向（沣水东岸）——**参照点不同导致表述差异**、并非矛盾。**镐池**则是西周镐京旧地附近的一汪水池、**唐人犹见**；今已湮。**镐池作为镐京的物理 anchor**——盛唐人到沣水郊游、所见的"周天子故迹" 主要就是这一汪水 + 沣水。

物候：关中平原中部温带季风气候、四时分明——春多沙、夏多雨、秋高、冬有薄雪；沣水两岸郊原、桑麻 + 禾黍 + 桃李——是关中农耕区的腹地。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

镐京在唐人的回望里——是**周天子故都 + 大唐"天朝故里"的物理根**。

**周人迁岐至灭商**：
- **古公亶父率周人迁岐**（约 BC 12 世纪、周原所在的岐山下、今陕西宝鸡岐山扶风一带）——**周原是周人发祥地**；
- **王季 + 周文王居岐**——周人在周原立国之基；
- **周文王末年作邑于丰**（沣水西岸）——**丰京始**；
- **周武王灭商**（**BC 1046**、夏商周断代工程主张；牧野之战克商纣王）；
- **武王灭商后定都镐京**（沣水东岸）——**镐京始**；
- **自此周人都丰镐**——西周王都 275 年（BC 1046 - BC 771）。

**镐京作为西周王都的 275 年**：
- **周武王**（BC 1046-1043）——立周、定鼎镐京；
- **周成王 + 周公旦摄政**（BC 1042-1021）——制礼作乐、镐京制度大备；同时营建洛邑（成周）为东都、安置殷遗民——**自此西周"两都制"**：宗周（镐京）+ 成周（洛邑）；
- **周康王**（BC 1020-996）——"成康之治"、西周鼎盛；
- **周昭王、穆王、共王、懿王、孝王、夷王**——西周中期；
- **周厉王**（BC 877-841）——国人暴动、厉王奔彘、"共和行政"（BC 841 是中国信史可考最早年份）；
- **周宣王**（BC 827-782）——宣王中兴；
- **周幽王**（BC 781-771）——西周末王。

**幽王之难（BC 771）**：**周幽王宠褒姒、废申后 + 太子宜臼**——申侯怒、联犬戎入镐京；**幽王 + 太子伯服死于骊山下**（骊山在长安东、今陕西临潼）——**史称"骊山烽火"传说**（实际"烽火戏诸侯"为后世附会、《史记》载褒姒"不好笑"是史源、烽火事《吕氏春秋》《史记》皆有、但实事可疑）；**犬戎入镐京、镐京焚毁**——西周亡。

**平王东迁（BC 770）**：**申侯 + 鲁侯 + 许文公等立宜臼为王、是为周平王**；**平王自镐京东迁洛邑**——东周始。**自此镐京废**——周天子再不居丰镐、丰镐旧地渐为农田。**自 BC 770 平王东迁至 755 约 1524 年**（无公元 0；BC 771 犬戎入镐计约 1525 年）——镐京废墟已埋于关中平原下、地表仅余沣水 + 镐池 + 郊原。

**秦汉至北朝镐京一带的沿革**：
- **秦昭王**（BC 306-251）于沣水东岸立**上林苑**——镐京旧地入秦苑囿；
- **汉武帝**（BC 141-87）扩建**上林苑**——含昆明池、镐池 等——丰镐旧地皆在上林苑内；
- **东汉、魏晋、十六国、北朝**——长安城兴废多次、丰镐旧地一直是长安城西南的郊野；
- **隋开皇二年（582）隋文帝营建大兴城**（即唐长安城前身）——长安城在丰镐东北约 30 里、丰镐旧地遂为长安西南郊外的农田。

**唐人对镐京的认知**：
- **位置认知**——唐人知镐京大略在沣水东岸、长安西南；**《元和郡县图志》卷一载"周武王宫即镐京、在县西北十八里"**——**以 唐长安县治 为参照点** 记"县西北十八里"；**以 唐长安城外郭 为参照点**、镐京遗址实际方位在西南方向（沣水东岸）——**参照点不同导致表述差异**（注：李吉甫书在 9 世纪初成、本 doc 引用以载唐人地理记忆）；
- **文化认知**——**周天子故都 + 大唐"丰镐故乡"**：唐皇室自李渊起即以"关中本位 + 周丰故里"立国叙事；唐玄宗在位 44 年、多次到沣水郊游、提及"周丰";
- **物理认知**——城垣 已废 千余 年、地表无 城迹；唐 人所见 主要 是 沣水 + 镐池 + 郊原 这三 样自然 物。

### 此刻

天宝十四载这一年的镐京——**已废约 1524-1525 年**（BC 771 犬戎入镐约 1525 年、BC 770 平王东迁约 1524 年；无公元 0）；**地表无城迹、属唐京兆府长安县郊原**。

**唐皇室与丰镐的关系**：**李唐自言"陇西李氏 + 关陇集团" 出身**——以**关中本位 + 周天子故里**为立国叙事；**唐玄宗一脉以"承周天命" 自居**：开元天宝间多次到沣水郊游、提及"周丰"；**玄宗朝礼制复古、改革郊祀制度**——是仿周公"制礼作乐" 的政治符号（**注**：唐代明堂在东都洛阳、武则天垂拱四年/688 建；玄宗朝几度改建——开元 5 年/717 改"明堂"为"乾元殿"、开元 10 年/722 复"明堂"、开元 27 年/739 拆上层去柱心木仍为"乾元殿"——**长安城内不建明堂**）；**长安城本身的规制**（外郭城 + 皇城 + 宫城 的三重制）——亦取意周制（一说承自《周礼·考工记》"匠人营国" 之制）。

**盛唐人到沣水郊游**——所见者是**沣水东岸的一片郊原 + 镐池一汪水 + 远处终南山的天际线**。**沣水自南山北流、过昔日丰镐之间、再北汇入渭水**——这条河仍在；**镐池**犹存（《元和郡县图志》 载长安西北十八里、亦或盛唐前期人记忆中犹在）。**唐人在沣水郊原、不能见到城垣、不能见到宫殿、不能见到丰镐之间的桥梁**——只能见到**沣水 + 镐池 + 终南山 + 田畴**。

**755 时的"丰镐意识"**：是**盛唐文人怀古的题材**——但不是 一种 考古意识、是一种 文化 自我 认同。**杜甫（712-770）**——成长于盛唐、其诗多涉关中地理、含丰镐遗意；**王维（约 701-761）**——居终南山、其诗多涉沣水 + 周丰；**李白（701-762）**——其《古风》多咏周丰故事。**盛唐文人到沣水郊游、见镐池**——常感"千载之上、宗周遗迹、不可复见"之意。

### 此刻意味

对天宝十四载的长安人来说——丰镐旧地就是**长安西南郊外的一片农田 + 一条河 + 一汪水**：沣水东岸的郊原、镐池的清波、终南山下的天际线。对长安那一代士人来说——丰镐是**大唐"天朝故里"的物理根 + 西周十二王（十一世）故都的怀古地**：从唐玄宗到杜甫王维李白、皆以"承周" 自居——丰镐是**这种自我认同的物理 anchor**。

唐人尚不知道——再过半年（755 末安史起兵）、再过六十年（781 唐德宗朝）、再过 150 年（907 唐亡）——丰镐旧地仍是郊原；**自唐至宋元明清** 千年 间、丰镐 旧地 始终 是 长安 / 西安 城 西南的农田、再无政治分量。再过千二百多年（1933 起 + 1950s 后），**陕西省考古研究院 + 中国社科院考古所** 在西安沣东沣西多次发掘——**确认沣水西岸为丰京、沣水东岸为镐京**；出土西周宫殿基址、青铜器、车马坑、卜骨等——**镐京遗址 1961 年列首批全国重点文物保护单位、与丰京合称"丰镐遗址"**。**唐人在 755 所见的那一片郊原 + 沣水**——千二百多年后**沣水仍在、镐池已湮**；今游人到斗门镇——所见的是**沣水东岸的考古遗址公园 + 现代西安西南郊的城市边缘**。

## 子条目

### 丰镐双京（沣水西丰、沣水东镐）

**西周早期实行"丰镐双京制"**——**丰京在沣水西岸**、周文王末年作邑于丰；**镐京在沣水东岸**、周武王灭商后立都。**两京隔沣水相望、间距约数里**——周天子日常往来如同一城。**功能分工**：传统说法以**丰京为宗庙所在**（"文王作邑、武王迁都" + 文王宗庙在丰）、**镐京为天子治所**（武王立国后的政治中心）；**近世考古口径**：两京功能多有重叠、共同构成西周早期双京。**自周武王至周幽王 西周 历 十二 王（十一 世；孝王 是 夷王 叔父、属 支系）居丰镐**——西周 王都 275 年（BC 1046-771）。**注**：周共王（恭王）以后或有徙居槐里、犬丘 等说、本 doc 不展开。

### 周原 + 丰镐（周人发祥与都城的两段）

**周原 + 丰镐 是 周人 立国 史 的 两 段**——**周原**（今陕西宝鸡岐山扶风一带）是**周人发祥地**：古公亶父率周人**迁岐**、王季 + 文王 居岐——**周人在周原 形成 部族 + 立国 之 基**；**丰镐**（今西安沣东沣西）是**周天子都城**：文王末年作邑于丰、武王灭商立都于镐——**周人自周原 东进 + 立都丰镐 + 统治 天下**。**两段 不可 混 为 一段**——**周原 是 部族 阶段 / 王季文王 时代**、**丰镐 是 天子 阶段 / 武王至幽王 时代**。**唐人对 周原 + 丰镐 的 认知**——多以 "周丰" 代 整个 西周 故 都 + 故 里、不细分 周原 与丰镐。

### 周幽王之难 + 平王东迁（BC 771-770）

**周幽王十一年（BC 771）犬戎入镐京**——**幽王 + 太子伯服 死于 骊山下**（骊山在长安东、今陕西临潼）；**镐京焚毁**——西周亡。**翌年（BC 770）申侯 + 鲁侯 + 许文公等立宜臼为王、是为周平王**；**平王自镐京东迁洛邑**——**东周始**。**自此镐京废**——周天子 再 不居 丰镐。**"骊山烽火戏诸侯" 传说**——《吕氏春秋》《史记》皆有、但 实事 可疑（西周 时 烽火 制度 尚 不 见 史载）；本 doc 不坐实。**真实 史源**：**幽王 宠 褒姒、废 申后 + 太子 宜臼、申侯 怒 联 犬戎 入镐** 是 主流 史 叙述。

### 镐池（唐人犹见的西周遗物）

**镐池**——西周 镐京 附近 的一 汪 水池、**唐人 犹见**。**《元和郡县图志》卷一 载**："周 武王 宫 即 镐京、在 县 西北 十八 里"——**是 西周 镐京 旧地 在 唐人 地理 系 中 的 物理 anchor**（**以 唐 长安县 治 为 参照点** 记 "县 西北 十八 里"；**以 唐 长安城 外郭 为 参照点**、镐京 遗址 实际 方位 在 西南 方向 / 沣水 东岸——**参照点 不同 导致 表述 差异** / 今 西安 西南 郊 沣水 一带）；**镐池 即 在 此 范围 内**。**汉武帝 扩建 上林苑** 时 镐池 入 苑囿；唐 时 上林苑 已废、镐池 作为 一汪 自然 水 池 仍在。**镐池 是 唐人 到 沣水 郊游 时 能 见 到 的 西周 物理 痕迹 之一**——亦是 盛唐 文人 怀古 的 题材。**今镐池 已 湮**——20 世纪 沣水 改道 + 周边 农 田 开垦 + 现代 城市 扩张 后、镐池 不复 可 辨。

### 唐皇室的"承周"叙事

**李唐自言"关中本位 + 周丰故里"**——是唐 立国 叙事 的 核心。**唐高祖李渊 起兵 太原 + 入长安 立 唐**（618）——以**关陇集团 + 关中本位** 立国；**唐太宗李世民 + 高宗李治 + 武则天 + 玄宗李隆基** 一脉——皆以**承周天命** 自居：**长安城 的 规制** 取意 周制（《周礼·考工记》"匠人营国" 之 制 的 一种 实现）、**封禅 + 郊祀** 制度 多 沿用 周礼 + 汉魏 旧 制；**明堂 制度** 仿 周公"制礼作乐"——**注**：唐代 明堂 在 **东都 洛阳**、武则天 垂拱 四年/688 建；玄宗朝 几度 改建——**开元 5 年/717 改"明堂"为"乾元殿"、开元 10 年/722 复"明堂"、开元 27 年/739 再 改 乾元殿**——**长安城 内 不 建 明堂**。**玄宗朝（713-755）"承周" 叙事 达 鼎盛**——**礼制 复古、改革 郊祀 制度、复 籍田 礼**；玄宗 多次 到 沣水 郊游、提及 "周丰"——是**盛唐 天朝 叙事 的 物理 根**。

### 西周王都 275 年 + 十二王（十一世）

**自周武王（BC 1046 立国）至 周幽王（BC 771 死于骊山下）共 275 年**——西周 王都 镐京 历 **十二 王**：**武王、成王、康王、昭王、穆王、共王、懿王、孝王、夷王、厉王、宣王、幽王**；**按 世 系 计 为 十一 世**——**孝王 是 夷王 叔父、属 共王 之 弟、为 支 系 旁 出**（《史记·周本纪》载 懿王崩、共王 弟 辟方 立 是 为 孝王；孝王崩、诸侯 复 立 懿王 太子 燮 是 为 夷王）——故 "十二 王 / 十一 世" 是 数量 上 自洽 的 主流 口径。**自周成王起 + 周公旦 制礼作乐**——**镐京制度 大备**：宗庙 + 社稷 + 明堂 + 辟雍 + 灵台 + 籍田 + 蒐狩 等 周礼 系统、皆 以 镐京 为 中心 + 配 以 洛邑 成周。**自周成王 营建洛邑（成周）后**——西周 "两都制"：**宗周（镐京）** 在 西、**成周（洛邑）** 在 东——是 中国 历史 上 最早 的 两 都 制 模式。**这种 两 都 制**——后世 汉 长安 + 洛阳、隋 唐 长安 + 洛阳、皆 取意 于 此。

## 历史长河（755 → 今）

**中晚唐**　**广德元年（763）吐蕃陷长安 15 日**——丰镐 旧地 一并 入 吐蕃 短暂 控制；**贞元 / 元和 间 长安 复 安**——丰镐 旧地 仍 是 长安 西南 郊外 的 农田。**唐末 黄巢 入 长安（880）+ 朱温 迁 帝 + 长安 焚毁（904）**——丰镐 旧地 始终 是 郊野、**唐 亡（907）** 后 长安 城 自身 已 废、丰镐 旧地 更 无 政治 分量。

**两宋**　**北宋 定都 开封**——长安 退 为 京 兆 府 + 西北 重镇；**金 + 元 + 明 + 清** 长安 改 名 西安、丰镐 旧地 始终 是 西安 西南 郊外 的 农田。**镐池 在 五代 至 元 间 渐 湮**——20 世纪 已 不 复 可 辨。

**金元明清**　**清乾隆 间 + 嘉庆 间 关中 学派** 兴 复古 之 风——**毕沅 + 钱大昕 + 阮元 等 考 丰镐 故 地**；**清 道光 二年（1822）陕西 巡抚 卢坤 至 沣水 立 "周丰镐故都" 碑**——是 清代 复古 考 的 一笔。

**民国至今**　**1933 年 起 中央 研究院 历史 语言 研究所 + 北平 研究院** 在 西安 沣东 沣西 多次 调查；**1950s 后 陕西省 考古 研究院 + 中国 社科院 考古所** 系统 发掘——**确认 沣水 西岸 为 丰京、沣水 东岸 为 镐京**；出土**西周 宫殿 基址 + 青铜器 + 车马坑 + 卜骨**；**镐京 遗址 1961 年 列 首批 全国 重点 文物 保护 单位**、与 丰京 合称 "**丰镐遗址**"。**今 游人 到 西安 西南 斗门镇**——所见 的 是**沣水 东岸 的 考古 遗址 公园 + 1933 年 以来 的 考古 探方 + 现代 西安 西南 郊 的 城市 边缘**。**唐人 在 755 所见 的 那 一片 郊原 + 沣水**——**沣水 仍在**、**镐池 已 湮**、**地表 城迹 仍 无**——千 二百 多 年 来 镐京 始终 是 一座 "地表 无形 + 地下 有 物" 的 遗址。
