# 巨鹿

## 一句话

**漳水南岸的灭秦决战地**——秦二世三年（前 207）项羽率楚军救赵、**渡漳水、破釜沉舟、以一当十、九战章邯王离军**、王离军败、章邯军后降；秦军主力覆灭于此、灭秦序幕由此开；755 时唐人见的是邢州钜鹿郡境内龙冈县东南一片旷野中的旧城残迹、距决战 961 年（公元纪年无 0 年、前 207 至 755 实历 961 年）。

## 形貌

巨鹿故地在唐河北道**邢州（天宝元年/742 改为钜鹿郡；州治龙冈县，今河北邢台）**东南——具体在**龙冈县治东南约三十至五十里**（今河北邢台市平乡县 + 巨鹿县跨界一带）。**地势**：华北平原中部、太行山东麓的冲积平原；**漳水（浊漳水）**自西北来、流经故城以北或西北一带、东去入海；**滏水（今滏阳河）**自西南来、与漳水汇于平原。**这是太行山东麓最典型的冲积平原 — 战国时赵国东境、秦置巨鹿郡治所所在**。

**巨鹿故城**——战国赵巨鹿、秦巨鹿郡治：**秦巨鹿郡（BC 222 秦灭赵后置）治巨鹿县**——是关东与河北平原咽喉之地。**BC 207 巨鹿之战时**赵王歇 + 张耳据此城；章邯 + 王离的秦军围城。**汉初**巨鹿仍为巨鹿郡治；**东汉以后**巨鹿郡治多次东南迁——至隋唐间巨鹿县已是邢州（钜鹿郡）下属一县、非郡治；**郡治在龙冈县**。**唐巨鹿县治**：在故城东南另立（即今河北巨鹿县老县治一带）；**巨鹿故城遗址**当时已是城外旷野中的一处土墙残迹——唐人凡至邢州凭吊巨鹿者所见即此。

**今巨鹿故城遗址**：在河北邢台市平乡县境内（部分跨巨鹿县），1995-2010s 考古调查确认夯土残墙轮廓——但**唐人不知考古**；唐人见的是田畴间一片可辨的方形城基。

物候：华北平原中部、暖温带季风——四时分明、春多沙、夏秋漳水滏水易涨、冬寒；盛唐人秋日凭吊巨鹿、所见多是"白茅 + 黄叶 + 漳水浩荡"——与凭吊邺城同类气候。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

巨鹿在唐人的回望里——是**中国战争史最著名的"以少胜多"现场**。**秦二世三年冬（颛顼历以十月为岁首、即前 207 年末至前 206 年初；按公历亦在前 207 年末稍延入次年）**——这场战役构成唐人对巨鹿三层认知：

**第一段：秦末天下大乱（BC 209-207）**。**秦二世元年（BC 209）七月**——陈胜、吴广起于大泽乡；**张楚政权**立、关东诸侯并起。**BC 208 年**——项梁立楚怀王熊心、自立为武信君；同年九月**项梁战死定陶**（章邯击破楚军、项梁被杀）；楚军元气大伤、退守彭城。**章邯击破楚军后北上击赵**——赵王赵歇（赵王武臣已死）+ 丞相张耳退守巨鹿；**章邯命王离（王翦之孙、上郡守）率秦边军南下围巨鹿**——王离军约二十万、章邯军屯巨鹿南棘原约二十万、合围巨鹿。**陈余**（张耳故友、时为赵将军、拥兵数万于巨鹿北）**畏秦不敢进救**——是巨鹿之围的紧迫情势。

**第二段：项羽夺权（BC 207 冬）**。**楚怀王熊心命宋义为上将军、项羽为次将、范增为末将——率楚军北上救赵**。**宋义至安阳（楚地、非河北安阳；约今山东曹县东）逗留四十六日不进**——欲坐观秦赵相敝、再图取利；**项羽于帐中斩宋义**（《史记·项羽本纪》"于是项羽晨朝上将军宋义、即其帐中斩宋义头、出令军中曰：宋义与齐谋反楚、楚王阴令籍诛之"）——**自立为上将军**。**这是项羽夺取楚军兵权的关键一笔**——以一军中之诛主帅、立信于诸将；楚怀王闻之、不得已正式拜项羽为上将军。

**第三段：破釜沉舟（BC 207 冬末至次年初）**。**项羽率楚军北上、至漳水南岸**——决战在即。**项羽先遣英布、蒲将军率二万人渡漳水试战**——破秦军一阵、王离 + 章邯军震恐；**项羽自率全军渡漳水**——下令"皆沉船、破釜甑、烧庐舍、持三日粮、以示士卒必死、无一还心"（《史记·项羽本纪》原文）——**这一笔是"破釜沉舟"典故的本源**。**注**：项羽渡漳是**漳水（浊漳水）**、不是黄河——汉以后讹传渐多、唐人读《史记》原文者皆知是漳水；本 doc 据《史记·项羽本纪》原文取漳水口径。

**九战之后**——"楚战士无不一以当十、楚兵呼声动天、诸侯军无不人人惴恐"（《史记·项羽本纪》原文）——这是"**以一当十**"典故的本源。**项羽军绝王离军粮道、九战、大破之**——**王离被擒**（《史记》"乃相与共立羽为诸侯上将军、诸侯皆属焉、于是已破秦军、项羽召见诸侯将、入辕门、无不膝行而前、莫敢仰视"）、苏角战死、涉间自焚。**章邯军屯棘原**——与项羽相持月余；**BC 207 七月章邯降项羽**（殷墟之盟）——秦军主力自此覆灭。**这是秦灭的真正一战**——王离军（约二十万）败于巨鹿、章邯军（约二十万）后降于殷墟（秦军 40 万主力先败后降、章邯降卒后于次年十二月在新安被坑）；自此关东至关中再无秦军大阵、刘邦得以西入武关。

**巨鹿之战之后**——**BC 206 十二月项羽至新安、坑杀秦降卒二十万**（章邯降卒）——是项羽残暴最显的一笔。**BC 206 十二月项羽入关、咸阳屠城、杀秦王子婴、焚秦宫**；**BC 206 二月项羽自立西楚霸王、分封十八诸侯**——**巨鹿一战是项羽霸业的奠基战**。

**唐人对巨鹿的认知**——主要源自**《史记·项羽本纪》《史记·秦始皇本纪》《史记·张耳陈余列传》《汉书·陈胜项籍传》**四篇。**司马迁《项羽本纪》是巨鹿之战最完整的记录**——盛唐文人凡读《史记》者皆熟此战；**"破釜沉舟"+ "以一当十" + "膝行而前、莫敢仰视"** 三处文字是中国战争文学最著名的几笔之一。

### 此刻

天宝十四载这一年的巨鹿——**距决战已 961 年**（前 207 至 755；公元纪年无 0 年、不作 962 计）；属唐河北道**邢州（钜鹿郡）巨鹿县**境内、距钜鹿郡治龙冈县（今河北邢台）东南约三十至五十里。

**巨鹿故城遗址**：方形城基轮廓在田畴间可辨；夯土残墙数尺至丈余、断续；城内已成耕地、田中偶见秦汉时期残砖瓦。**漳水**：自西北来、东去入海；**755 时漳水正流位置在故城以北或西北一带**——是巨鹿之战项羽渡漳的物理 anchor、唐人凭吊巨鹿者多至漳水边凭眺。**棘原**：章邯军屯驻地、在巨鹿南、唐时已无显迹；**项羽渡漳处**：唐人多指漳水南岸今平乡县境内一处——但具体地点已不可详（《元和郡县图志》《通典》皆未确指）。

**唐邢州 + 巨鹿县治**：**邢州治龙冈县**（今河北邢台市；天宝元年/742 改州为郡称"钜鹿郡"——是 755 时正式名称）；**巨鹿县治**在故城东南另立（今河北巨鹿县老县治一带）；**巨鹿故城遗址**当时是巨鹿县境内的一处旷野古迹——非县治、非郡治。**唐巨鹿县属钜鹿郡（邢州）下属**——是钜鹿郡名所自来、但县治已不在故城。

**唐人对巨鹿的凭吊**——三重叠加：
- **灭秦决战的物理 anchor**——盛唐文人读《史记》皆知"破釜沉舟"在此漳水南岸；
- **项羽霸业奠基战**——与四年后（BC 202）项羽败于垓下形成戏剧 arc、是唐人读《史记·项羽本纪》最强的两个 anchor；
- **战国赵巨鹿 + 秦巨鹿郡的旧城**——盛唐人对赵国东境与秦关东郡治的物理参照亦在此。

**安史前夕**（即 755 这一刻）——再过半月（农历十一月初九）安禄山起兵于范阳；自范阳南下取河北诸州、邢州（钜鹿郡）皆在叛军南下必经之路上。**巨鹿故城自身是废墟、无守军**——但邢州治龙冈县是河北道南部至河南道的咽喉之一。

### 此刻意味

对天宝十四载的邢州人来说——巨鹿故城是**州治东南那片"项羽破秦处"**：田耕岁岁、过客时来；漳水东流 + 城基方形夯土 + 棘原已无显迹——构成日常生活背景里的灭秦决战剪影。对盛唐文人来说——巨鹿是**中国战争史最戏剧的"以少胜多"现场**：项羽以五万楚军破章邯 + 王离四十万秦军 + 王离被擒 + 章邯降 + 秦军主力歼于此——一城承载灭秦的决定性一战。

**与同时代凭吊点的对照**——
- **鸿门**（今陕西临潼东）：BC 206 十二月项羽设宴鸿门、刘邦脱险——是项羽放刘邦走、楚汉相争分水岭、距巨鹿一年；唐时鸿门在京兆府新丰县境、与巨鹿同为楚汉故事的两端；
- **垓下**（约今安徽蚌埠固镇濠城镇一带；主流说，另有灵璧、鹿邑诸说）：BC 202 十二月项羽四面楚歌、项羽悲歌、决战于垓下、突围至乌江自刎——垓下为四面楚歌决战处、乌江为自刎处；是项羽身死的最后一程、距巨鹿五年；唐时垓下故地在徐州、亳州、泗州一带，唐人不能坐实；
- **三处构成楚汉相争的三个 anchor**——巨鹿（前 207）+ 鸿门（前 206）+ 垓下（前 202）——盛唐文人凡读《史记·项羽本纪》皆知三地相距不过五年、却把项羽从二十六岁霸业奠基带到三十一岁乌江自刎。**项羽在巨鹿城下完成了从将军到霸王的蜕变**——这是 755 时凭吊巨鹿者最深的一笔。

唐人尚不知道——**自宋元间漳水多次改道**、漳水正流位置渐离巨鹿故城；**明清间巨鹿故城渐为田畴所没**——城基轮廓在 19 世纪后已难辨；**1995-2010s 考古调查**始系统确认巨鹿故城遗址位置——是中国战争史最著名一战的物理 anchor 的现代认定。

## 子条目

### 巨鹿故城

**战国赵巨鹿 + 秦巨鹿郡治 + 汉初巨鹿郡治**——是 BC 207 巨鹿之战时赵王歇 + 张耳所守之城。**故城位置**：今河北邢台市平乡县境内（部分跨巨鹿县）、距唐邢州治龙冈县东南约三十至五十里。**形制**：方形夯土城、规模不详（考古调查中）。**BC 207 巨鹿之围**——王离军围城、城内赵王歇 + 张耳守备、援军（陈余等）观望于巨鹿北；项羽渡漳之后于城外破王离军、解巨鹿之围。**汉以后**——巨鹿仍为巨鹿郡治至东汉、晋后郡治多次迁徙；**隋开皇间**巨鹿县并入南宫县、再后复立；**唐巨鹿县治**在故城东南另立（今河北巨鹿县老县治一带）——**故城自唐起已是城外旷野中的旧城遗迹**。**今巨鹿故城遗址**：1995-2010s 考古调查确认夯土残墙轮廓——**与 755 唐人所见的物理景观在大轮廓上是连续的**。

### 漳水南岸渡口（破釜沉舟处）

**项羽渡漳水的具体地点**——《史记·项羽本纪》记"项羽乃悉引兵渡河、皆沉船、破釜甑、烧庐舍、持三日粮、以示士卒必死、无一还心"——**"河"在此处指漳水**（浊漳水）、非黄河；汉以后讹传渐多、唐人读《史记》原文 + 《水经注·卷十·浊漳水》对照者皆知是漳水。**渡口具体位置**——《元和郡县图志》《通典》皆未确指；唐人凭吊者多至漳水南岸今平乡县境内一处。**漳水南岸**：是项羽下令破釜沉舟之地、是 BC 207 冬末楚军决死之心的物理 anchor；**唐 755 时漳水仍东流**——是这片战场最稳的物理事实。**注**：宋元以后漳水多次改道——南宋绍兴间 + 元至元间 + 明嘉靖间均有大改道；唐 755 时漳水正流位置与汉代相近、仍可指证"项羽渡漳处"。

### 棘原（章邯军屯驻地）

**章邯军屯驻巨鹿南棘原**——是 BC 207 巨鹿之战时章邯主力（约二十万）的屯驻点；与项羽军隔漳水或棘原一线相持月余。**BC 207 七月章邯于殷墟（今河南安阳一带）降项羽**——自此王离军败于巨鹿、章邯军后降于殷墟；秦军主力星散：降卒后在 BC 206 十二月被项羽坑于新安（今河南渑池东）二十万、章邯等三将转封雍王 + 塞王 + 翟王、秦兵主力自此覆灭。**棘原唐时已无显迹**——唐人凡至巨鹿凭吊棘原者所见即田畴、《元和郡县图志》《通典》皆未列棘原条；**棘原大致在今河北平乡县与广宗县一带**（汉代史地考证）——是巨鹿之战秦军屯驻的物理 anchor。

### 破釜沉舟（中国战争最著名典故）

**"破釜沉舟"出《史记·项羽本纪》**——"项羽乃悉引兵渡河、皆沉船、破釜甑、烧庐舍、持三日粮、以示士卒必死、无一还心"。**这是"毕其功于一役"的物理 anchor**——以毁己之退路、立必死之心；是中国战争史最著名的几个典故之一。**与"以一当十"并列**——"楚战士无不一以当十"亦出《项羽本纪》同段；两典构成项羽巨鹿战的双重 emblem。**唐人对破釜沉舟的引用**——盛唐边塞诗人凡写决死战意者多隐用此典（具体名篇逐字引证尚需逐首核校、本 doc 不强指）；李白、杜甫、王昌龄、高适等集中咏巨鹿或明引"破釜沉舟"四字之诗存疑、不在此列具体诗题、以免伪引。

### 项羽斩宋义（夺权于安阳）

**BC 207 冬项羽在楚地安阳斩上将军宋义**——是巨鹿之战前奏。**安阳**：在此指楚地安阳（约今山东曹县东、楚地）、**非河北安阳**（北周以后的相州治）——这是唐人读《史记》最易混淆的一处地名、本 doc 据《史记·项羽本纪》原文取楚地安阳口径。**宋义逗留安阳四十六日不进**——欲坐观秦赵相敝；项羽以"先破秦入关、定关中"为志、晨入宋义帐中斩之；**自立为上将军、楚怀王闻之不得已正式拜项羽**——这是项羽夺取楚军兵权的关键一笔、为巨鹿之战奠定指挥统一。

### 巨鹿之战的史地诸事

**几处易混淆之点**——本 doc 据《史记·项羽本纪 + 秦始皇本纪 + 张耳陈余列传》取严密口径：
- **章邯**：秦少府（非将军本职、以少府权将军）；BC 208 击破陈胜军、击破项梁军于定陶；BC 207 围巨鹿；BC 207 七月降项羽；BC 206 项羽分封章邯为雍王。
- **王离**：秦上郡守、王翦之孙；率秦边军（长城军、约二十万）南下围巨鹿；BC 207 巨鹿之战被项羽擒。
- **苏角 + 涉间**：王离军部将；苏角战死、涉间自焚。
- **陈余**：赵将军、拥兵数万于巨鹿北、畏秦不敢进救；与张耳后因此结怨、构成《史记·张耳陈余列传》主线。
- **巨鹿之围历时**：约半年（BC 208 末王离围城至 BC 207 末项羽破之）；项羽渡漳至破王离不过数十日。
- **"破釜沉舟"渡的是漳水**（《史记》原文"河"指漳水）——**不是黄河**；唐人读《史记》原文者皆知。
- **"以一当十"原文**："楚战士无不一以当十、楚兵呼声动天、诸侯军无不人人惴恐"——出《项羽本纪》同段、是巨鹿之战正面战记述。

## 历史长河（755 → 今）

**中晚唐**　唐邢州（钜鹿郡）在安史之乱中两度易手——**至德二年（757）**叛军安庆绪据邢州；**宝应元年（762）**唐军收复邢州。**中晚唐凡至邢州凭吊巨鹿者**——主要是过路使臣 + 文人：**李白《登广武古战场怀古》**虽写广武（鸿沟）但同类怀古情境亦含巨鹿；**杜甫凡过河北诸州未至巨鹿**（杜甫天宝末至河南、未至河北）。**晚唐胡曾《咏史诗·巨鹿》**"巨鹿之战"题诗——是晚唐凭吊巨鹿的代表（胡曾约咸通至广明间作《咏史诗》一百五十首）。

**五代 + 两宋**　**五代后梁后唐间**邢州在河北道南部——**后唐天成四年（929）**改邢州为安国军节度；**北宋初**邢州仍为军节度治所。**北宋庆历八年（1048）黄河北流改道**——黄河自澶州（今河南濮阳）北决、改东北向流入海——**这次改道与漳水合流于今河北沧州一带、但邢州 + 巨鹿故城段的漳水主流位置未改**；**北宋大观二年（1108）漳水大改道**——自原巨鹿故城以北南移、巨鹿故城逐渐脱离漳水主流。**北宋诗人**凡过邢州凭吊巨鹿者留诗——**苏轼《项籍庙》《巨鹿之战》**等是宋代凭吊巨鹿的代表。**南宋陆游《入蜀记》**凡过中原凭吊楚汉故事者亦多含巨鹿一笔。

**金元**　**金代邢州在大名府路 + 真定府路间转隶**；**金元间漳水多次改道**——元至元间漳水自原邢州东境南移、巨鹿故城脱离漳水正流；**元代巨鹿县属顺德路（治邢台）**——巨鹿县治在元末再次东迁、距巨鹿故城更远。**元代地方志记巨鹿故城**——"夯土残基、城周数里、田中残瓦犹存"——是巨鹿故城物理保存状态的中期记录。

**明清**　**明代巨鹿县属顺德府**；**清初顺治十二年（1655）**巨鹿县因水灾（漳水溢）治所东迁至今河北巨鹿县城——**与巨鹿故城遗址距离更远**。**清代凡过邢州凭吊巨鹿故城者**：**清乾隆四十年（1775）**陕西巡抚毕沅过河北时考察邢州、留《巨鹿怀古》诗；**清嘉庆二十年（1815）**直隶布政使方观承编《畿辅通志》收录巨鹿故城条目。**清末巨鹿故城已是田野中一处"考古名胜"**——但因没有大型纪念建筑（不同于隆中、五丈原均有重修纪念物），凭吊者主要是文人士大夫与地方官。

**民国至今**　**宋代巨鹿县城在北宋大观二年（1108）漳河大水中被冲毁淤埋**——是为"水淹巨鹿"之事；**民国 9 年（1920）当地农民掘井偶得宋瓷**——遂使宋代巨鹿县城遗址重见于世；**1921 京津外人 + 1923 北平历史博物馆**先后入村抢救发掘、所出宋代瓷器器具大量流入中外公私藏家、是为"巨鹿宋瓷"。**1929 年并无"震出宋瓷"一事**——本 doc 据民国发掘记录订正。**这批宋代古城是宋代巨鹿县城、非战国秦汉巨鹿故城**——本 doc 严守二者之别。**1949 年后**巨鹿故城遗址列河北省重点文物保护单位；**1995-2010s 考古调查**确认巨鹿故城遗址跨平乡县 + 巨鹿县境的具体位置——夯土城墙残基 + 城周战国秦汉残瓦——是巨鹿之战物理 anchor 的现代认定。**今游人来巨鹿故城遗址**：所见的是华北平原中部一片田野中的方形夯土残墙 + 漳水（虽已远离原河道）东流 + 城内残瓦——**与 755 唐人所见的物理景观在大轮廓上是连续的**：两千二百多年、灭秦决战之地以废墟之姿等待至今。**项羽渡漳处**——今河北平乡县境内立"破釜沉舟纪念碑"（1990s 立）；**巨鹿之战陈列馆**——2000s 始建于今河北巨鹿县境内——是巨鹿之战历史叙述的现代物理 anchor。
