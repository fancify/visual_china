# 涿鹿

## 一句话

**华夏始战之野**——燕山以北、桑干河北岸的一片山前盆地；**上古黄帝战蚩尤、又战炎帝之地**（《史记·五帝本纪》所记、传说年代约前 26 世纪、具体年不可坐实）；**755 时在河北道妫州妫川郡怀戎县境内**——**唐 武德七年（624）置涿鹿县、贞观八年（634）省入怀戎县**——故 755 时**涿鹿县已废 121 年**、仅作为怀戎县境内的旧地名 + 上古黄帝战蚩尤典故的物理 anchor 存在；距长安东北约一千余里、是中国军事史最古的一处物理起点。

## 形貌

涿鹿在唐河北道**妫州妫川郡**——**天宝元年（742）改妫州为妫川郡**（《旧唐书·地理志》《新唐书·地理志》皆载）——**州（郡）唯一属县即怀戎县**（治今河北怀来县大古城一带）；**涿鹿故地在怀戎县东南境**——具体在今河北涿鹿县矾山镇 + 县城一带；距怀戎县治东南约 60-80 里。

**地势**：**北倚燕山**支脉——**燕山 / 军都山支脉** 自西向东走、阻隔妫州与漠南草原；**南界桑干河**——桑干河自西南来、北流入湿余水（今永定河上游）；**涿鹿故地夹在燕山与桑干河之间**——是一片山前缓坡 + 河谷盆地——海拔约 500-800 米；**西望涿鹿山**（俗称矾山、今河北涿鹿县矾山镇主峰）—— 海拔约 900 米；**东南为桑干河谷地**——可东南通幽州（范阳郡）方向。

**水文标识**：
- **涿水**——《史记·五帝本纪》集解 + 《水经注》载涿鹿一带有涿水；今矾山镇南有黄帝泉（俗称阪泉），泉水北流；
- **桑干河**——涿鹿南界、自西向东北流；
- **阪泉**（传说水文）——《史记》"黄帝战炎帝于阪泉之野"——一说阪泉即在涿鹿东南、今矾山黄帝泉一带；另说阪泉在山西阳曲（与涿鹿之野是两处或一处、史源说法不一、本 doc 取"阪泉与涿鹿同区"的主流说而不坐实具体水点）。

**山势**：
- **涿鹿山**（矾山）——传说黄帝合符釜山即在附近；今河北涿鹿县矾山镇南有"釜山"地名；
- **燕山支脉** 自西向东横亘——是涿鹿之野的北屏；
- **军都关**（居庸关前身、唐时居庸关已立）——在涿鹿东南、是涿鹿通幽州方向的关口。

**周边地理**：
- **东南距幽州（范阳郡）治蓟县约 200 唐里**——出居庸关沿桑干河谷可至幽州；
- **西南距代州雁门郡约 300 唐里**——通山西方向；
- **北越燕山为漠南草原** —— 是奚 + 契丹（突厥溃后两族在燕山以北经营）的游牧之地；
- **西距妫州治怀戎县约 60-80 里**——是怀戎县东南境的核心古迹区。

物候：燕山以南、塞外气候——冬寒长、夏短热；桑干河北岸宜麦粟 + 牧、是河北道西北边境州县的混合农牧带；与河北道腹地（相州 / 魏州 / 邢州）的纯农耕景观不同。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

涿鹿在唐人的回望里——是**中国军事史最古的一处物理 anchor**、**黄帝战蚩尤的传说之地**、**华夏始战之野**。

**黄帝战蚩尤叙事**（盛唐人据《史记·五帝本纪》《国语·楚语下》《山海经·大荒北经》《逸周书·尝麦解》等可知）：

**上古之时**——传说"三族鼎立"于中原 + 北方：
- **黄帝部落**——以姬姓为主、居中原；
- **炎帝部落**——以姜姓为主、居黄河中游；
- **蚩尤部落**——所谓"九黎"（《国语·楚语下》观射父语"九黎乱德"）——居中原与东方之间。

**《史记·五帝本纪》载**——**黄帝**（轩辕氏）"与炎帝战于阪泉之野、三战、然后得其志"——是黄帝与炎帝的"阪泉之战"；**继而**"蚩尤作乱、不用帝命、于是黄帝乃征师诸侯、与蚩尤战于涿鹿之野、遂禽杀蚩尤"——是黄帝与蚩尤的"涿鹿之战"。**两战合称**——是华夏文明早期"族群整合"的物理起点；**唐人据此**——以涿鹿为华夏始战之野、阪泉为华夏第二战之野（一说两地实一区）。

**《山海经·大荒北经》补述**——"蚩尤作兵伐黄帝、黄帝乃令应龙攻之冀州之野"——蚩尤先动；"黄帝乃下天女曰魃、雨止、遂杀蚩尤"——以神话色彩浓厚的笔法记此战。**《逸周书·尝麦解》补述**——"蚩尤乃逐帝、争于涿鹿之阿、九隅无遗"——"九隅"指九州、强调此战的范围之广。**《国语·楚语下》观射父答昭王问**——以"九黎乱德、民神杂糅、不可方物" 述蚩尤族群的"乱德" 性质——是唐人理解蚩尤一族的主要文献依据。

**唐人对涿鹿地望的认识**：
- **《史记·五帝本纪》集解**（南朝宋裴骃）引服虔曰"涿鹿山在涿郡"——但**服虔此说有误**：涿郡治范阳县（今河北涿州市）、与涿鹿（今河北涿鹿县矾山一带）不为同一地；**唐人据此或一时混淆**——但**唐 武德七年/624 置涿鹿县于妫州**之事 + 后出《元和志》可证唐代地理传统——**已渐厘清"涿鹿在涿郡" 之误**；**盛唐通识**——涿鹿在妫州怀戎县境内、不在涿州范阳县境内。
- **涿鹿县沿革**——**隋开皇十六年（596）置涿鹿县**、属妫州；**大业初省**；**唐 武德七年（624）于妫州复置涿鹿县**、**贞观八年（634）省入怀戎县**——自此涿鹿县不复立。**755 时涿鹿县已废 121 年**——仅作为怀戎县境内的旧地名 + 黄帝战蚩尤典故 anchor 存在。
- **唐人言"涿鹿"时**——主指**怀戎县境内的涿鹿故地**（涿鹿山 + 涿水 + 阪泉所在）；**不指涿州/范阳郡**（在今河北涿州市）—— 两地相距约 200 唐里、不可混。

### 此刻

天宝十四载这一年的涿鹿——**作为县已废 121 年**、作为传说之地已传约 2700 年（若取传说黄帝活动年代约前 26 世纪估算）；以**妫州妫川郡怀戎县境内的上古传说 anchor**身份存在。

**妫州妫川郡的行政分量**——**天宝元年（742）改妫州为妫川郡**；**州（郡）领县一**——即**怀戎县**（治今河北怀来县大古城一带）；**妫州是河北道西北边境州**——北越燕山为奚 + 契丹之地；**州治怀戎县在桑干河北岸**——是河北道西北边防的中等州。**涿鹿故地** 在怀戎县东南境、距州治约 60-80 里——是怀戎县境内最重要的怀古地。

**唐人对涿鹿的视野** 主要是三种存在形态：
- **典故** —— 黄帝战蚩尤、阪泉战炎帝——是盛唐文人 + 朝廷册命 + 兵家叙事中最常用的上古典故之一；
- **旧县名** —— 隋 596 置 + 唐 624 复置 + 唐 634 再省的涿鹿县——是 755 时唐人知道的"曾有此县"的旧地名；
- **物理地形** —— 涿鹿山 + 涿水 + 阪泉 + 釜山 等 与《史记》《山海经》对得上的地名物候——唐人按文献仍可考；但**地下夯土遗存**（即今矾山黄帝城遗址、1990s 后调查发现）**唐人不知**——本 doc 仅在末段作物理回响引用。

**这一年的妫州** 处于**唐玄宗朝末年河北道西北边境州**的状态——**再过半年安禄山起兵于范阳**（范阳郡 = 幽州、距妫州东南约 200 里）；**妫州作为范阳节度使辖区**（开元后期 + 天宝间妫州属范阳节度使、安禄山兼任）—— 是安禄山起兵的腹地之一；**755 末至 763 安史八年间**——妫州 + 怀戎县 + 涿鹿故地 皆在叛军控制下；唐军未尝克之。**涿鹿之野** 在这一阶段重新成为军事舞台——但与上古黄帝战蚩尤无涉、是天宝末安史叛军 + 奚契联兵的物理战场。

### 此刻意味

对天宝十四载的妫州 / 怀戎县人来说——涿鹿就是**自家东南边的古地**：涿鹿山起伏、桑干河北流、釜山耸立、阪泉涌涌——是熟悉的塞外乡野景观；田间或闻"古战场"传说——但具体何战、何年、何方、皆无确证。

对长安那一代士人来说——涿鹿是**华夏始战的物理 anchor**：黄帝战蚩尤、阪泉战炎帝——《史记》《国语》《山海经》《逸周书》所记 的上古战事——其物理终点即在妫州怀戎县境的涿鹿之野；**这是中国军事史最古的一处地名**——比殷周牧野早约一千八百年、比楚汉鸿门早约二千四百年、比唐代任何战场都古。**唐人写兵论、咏边塞、述武功**——若推到上古之极、必落于"涿鹿"二字。

盛唐边塞诗中"涿鹿"是常见意象——**李白《古风》"秦皇按宝剑、赫怒震威神"** 等推到上古的诗作多有涿鹿影；**杜甫、高适、岑参** 等边塞诗人 若言"扫胡尘、定边塞"、**多以"涿鹿 + 阪泉"为最古的物理 anchor**。**涿鹿** 对盛唐人的意义——不在地理本身（妫州不过是河北道西北一个小州）、而在**它是华夏自我认同的"始战"** ——黄帝战蚩尤是华夏文明早期"族群整合"的起点、是"我们是华夏" 这一身份的最古物理证据。

唐人尚不知道——再过 200 年（北宋初），涿鹿仍作为黄帝战蚩尤之地存于文献怀古、但物理痕迹愈淡；再过千年（清乾隆 + 嘉庆间），考据学渐兴、顾祖禹《读史方舆纪要》、阎若璩、王鸣盛 等 多有辨涿鹿 / 涿郡 / 涿州之别、最终确认涿鹿在今河北涿鹿县矾山一带；再过一千二百年（1990s），河北省文物部门在涿鹿县矾山镇调查 + 试掘——发现**黄帝城遗址**夯土城垣（推测年代上限不一、学界争议大）——但此遗址与上古传说的关联性 尚无定论；**唐人在 755 所未见的涿鹿地下** ——一千二百多年后是中国上古史 + 考古学最具争议的一处地。

## 子条目

### 黄帝战蚩尤（涿鹿之战核心）

**黄帝战蚩尤** 是中国军事史最古的一战——传说约前 26 世纪（具体年不可坐实、本 doc 取"约前 26 世纪"作为传说年代之上限、不同时坐实具体年）。**《史记·五帝本纪》载**——"蚩尤作乱、不用帝命、于是黄帝乃征师诸侯、与蚩尤战于涿鹿之野、遂禽杀蚩尤、而诸侯咸尊轩辕为天子、代神农氏、是为黄帝"——**此战是黄帝即"天子" 位的物理标志**。

**蚩尤的身份**——《国语·楚语下》观射父语"九黎乱德、民神杂糅、不可方物、夫人作享、家为巫史"——"九黎"指蚩尤所属族类（不是"九黎国"作为政治建制——本 doc 严守此一区分）；**九黎** 一般认为是黄河中下游与东方之间的一支古老族群、与黄帝 / 炎帝两大部落鼎足三立。**《山海经·大荒北经》载**——"蚩尤作兵伐黄帝、黄帝乃令应龙攻之冀州之野"——"冀州" 即古九州之北—— 与涿鹿地望相符。

**战法**——文献多有神话色彩：**应龙** + **天女曰魃** + **雷雨** + **大雾** + **指南车**（《古今注》后出说法、本 doc 不采）等——是后世化用的层累叙事。**唐人据《史记》《山海经》《逸周书》** 知此战大略——但**唐人并不将其作为信史的"实战"对待**——而是作为华夏文明早期"始战" 的物理 anchor；**这是 doc 的核心张力**——涿鹿之战既是传说、又是华夏自我认同的最深一层。

### 阪泉战炎帝（与涿鹿同区？）

**《史记·五帝本纪》载**——"轩辕乃修德振兵、治五气、艺五种、抚万民、度四方、教熊罴貔貅貙虎、以与炎帝战于阪泉之野。三战、然后得其志"——**这是黄帝即位之前的另一战**、与蚩尤涿鹿之战合称黄帝"二战定天下"。

**阪泉地望**——史源有歧：
- 一说**阪泉在涿鹿东南**（《史记》集解引服虔 + 《水经注》系——今矾山黄帝泉一带、距涿鹿故地不远；本 doc 取此一说为主、按"阪泉与涿鹿同区"立论）；
- 一说**阪泉在山西阳曲**（《括地志》《元和郡县图志》部分章节系——与涿鹿是两地）；
- 一说**阪泉在卫州**（北宋以后某些注本系——本 doc 不采）。

**本 doc 立场**——盛唐人对阪泉地望或有分歧、但"阪泉与涿鹿同区"是主流之一；不坐实具体水点、只取"阪泉之野 与 涿鹿之野 相距不远"的大方位。**炎帝身份**——一般认为是姜姓部落的首领、与黄帝同源（一说黄帝、炎帝皆少典之子、同父异母——《国语·晋语四》载）——**阪泉之战 是黄帝 / 炎帝两大同源部落的整合战**；自阪泉以下、**炎黄合流** 成为华夏共同祖先的核心叙事——**后世炎黄共同祖先叙事由此派生**（"炎黄"作为合称在战国后渐成熟、本 doc 取此一通识）。

### 唐人对涿鹿地望的认识（与涿州/涿郡之别）

**这是本 doc 最重要的子条目之一**——涿鹿 / 涿州 / 涿郡 三名易混、唐人据当代地理志书 + 后出《元和志》可证唐代地理传统 已渐厘清前代之误。

**三名之别**：
1. **涿鹿**——上古传说中的黄帝战蚩尤之地；**唐 755 时**在河北道**妫州妫川郡怀戎县**境内（今河北涿鹿县矾山一带）；**隋开皇十六年/596 置涿鹿县、大业初省；唐 武德七年/624 复置、贞观八年/634 再省入怀戎县**——755 时涿鹿县已废 121 年。
2. **涿州**——**唐时未立涿州**（涿州为后周 + 辽 始置；唐时此地属幽州 + 范阳郡）；**唐 755 时** 涿州故地是**幽州范阳郡涿县**（治今河北涿州市）。
3. **涿郡**——**隋大业三年（607）改幽州为涿郡**、**唐武德元年（618）复改幽州**；**唐 755 时** "涿郡" 一名已废 137 年——但作为前代地名仍见于唐人怀古文。涿郡治蓟县（今北京西南）、涵盖范阳县（今河北涿州市）。

**南朝服虔注《史记》误以涿鹿在涿郡**——是历史上一桩较大的地名混淆；**唐人据隋唐涿鹿县建置史 + 后出《元和志》可证唐代地理传统 已渐厘清**——盛唐通识中涿鹿在妫州怀戎县境内、不在涿郡 / 涿州境内。**两地相距约 200 唐里、不可混**——本 doc 严守此一区分。

### 涿鹿县沿革（隋 596 + 唐 624 + 唐 634）

**涿鹿县的两度立废** 是 755 时涿鹿"作为旧地名" 的关键背景——本 doc 严守史源、不模糊处理。

**第一度（隋）**：
- **隋开皇十六年（596）置涿鹿县**、属妫州（妫州亦同年置）；
- **大业初（约 605-608 间）省涿鹿县**——大业三年（607）改州为郡之改制中、妫州亦废、涿鹿一并省。

**第二度（唐）**：
- **唐武德七年（624）复置涿鹿县于妫州**——按《新唐书·地理志》河北道妫州条；
- **贞观八年（634）省涿鹿县入怀戎县**——按同上、亦见《旧唐书·地理志》。

**755 时涿鹿县已废 121 年**——仅作为怀戎县境内的旧地名 + 黄帝战蚩尤典故 anchor 存在。**唐人写"涿鹿"时**——主指**怀戎县境内的涿鹿故地**（涿鹿山 + 涿水 + 阪泉 + 釜山所在）、不指作为现行政区的"涿鹿县"——因为 755 时并无涿鹿县。**本 doc 严守此一时点—— 不写"妫州涿鹿县"作为 755 时的政区**。

### 涿鹿之野作为华夏始战的文化意义

**涿鹿之战** 在中国军事史 + 文化史中的地位——是**华夏自我认同的物理起点**。**理由**：
- **时间最早**——传说约前 26 世纪、比殷周牧野（BC 约 1046）早约一千八百年、比楚汉鸿门（BC 206）早约二千四百年、比唐代任何战场都古；
- **族群整合意义最重**——黄帝战蚩尤 + 阪泉战炎帝 合称"二战定天下"——是华夏部落联盟的物理起点；自此**"华夏" 作为族群意识的最早源头** 即在涿鹿；
- **天子位的物理 anchor**——黄帝因此战"代神农氏" 即天子位——是中国"天子" 概念的最早物理 anchor；
- **后世兵法的典故源头**——后世兵家偶以黄帝兵事作远古象征 + 战国诸子 多以涿鹿战 作为"始战"的文化符号（虽然战法多神话色彩、但作为"始战"的文化符号一直延续）。

**盛唐文化语境中的涿鹿** —— 是边塞诗 + 兵法论 + 朝廷册命 中最古的物理 anchor；**李白、王昌龄、岑参、高适** 等边塞诗人 若推到上古之极、多以"涿鹿 + 阪泉"为最深一层；**唐玄宗朝以"承黄帝 + 周文王 + 汉武帝"自况** —— 涿鹿作为"黄帝即天子位"的物理 anchor、在玄宗朝的文化想象里分量极重。**注**：玄宗本人是否亲过涿鹿 史源略零散；但**妫州怀戎县在范阳节度使辖区**、安禄山兼任范阳节度使——涿鹿在天宝末是安禄山的腹地。

### 妫州怀戎县与涿鹿故地的关系

**妫州妫川郡 = 唐玄宗朝末年河北道西北边境州**——天宝元年（742）改妫州为妫川郡——州（郡）唯一属县即**怀戎县**（治今河北怀来县大古城一带、桑干河北岸）。**涿鹿故地** 在怀戎县东南境、距州治约 60-80 里——具体含涿鹿山 + 涿水 + 阪泉 + 釜山 + 黄帝泉等地名。

**怀戎县名的来历**——"怀戎" 意指"招怀戎狄"——是唐人对此地"塞外/边境"地理身份的命名；**与上古黄帝战蚩尤"始战"** 形成有趣的呼应——同是中国"族群整合 + 边境管理"的物理舞台、相距约 2700 年。

**妫州在范阳节度使辖区**——开元后期至天宝间妫州属范阳节度使（安禄山兼任）——**涿鹿故地** 在安史叛军起兵之前已在叛军实际控制的腹地之内；**755 末安禄山起兵后**——妫州 + 涿鹿一带成为叛军兵源补给地之一；**唐军未克妫州** 直至 763 史朝义败死。**涿鹿之野** 在天宝末重新成为军事舞台——但与上古黄帝战蚩尤无涉、是不同语境下的"涿鹿"。

### 1990s 矾山黄帝城遗址（唐人不知、本 doc 末段引用）

**唐人未见涿鹿地下任何夯土遗存**——但本 doc 在末段引用一处考古发现以构成"物理回响"。

**矾山黄帝城遗址**——位于今**河北省涿鹿县矾山镇** 北约 1.5 公里、桑干河支流南——**1990s 河北省文物部门调查 + 试掘**——发现**夯土城垣残址**（推测城周约 1200 米、夯土厚约 15-20 米）；**学界对此遗址的年代上限争议大**——一说为新石器晚期至夏代之间（约前 3000-前 2000 年）、与上古黄帝战蚩尤传说时段可能有交集；另说为战国 + 秦汉间夯土、与上古传说无关。**本 doc 立场**——**不坐实矾山黄帝城遗址与黄帝传说的关联**、仅作为"涿鹿在 755 时未被识别的物理遗存" 作末段物理回响引用。**唐人不知此地下有夯土城垣**——是中国上古传说 anchor 与考古遗存关系的又一例。

## 历史长河（755 → 今）

**中晚唐**　**756-763 安史八年间妫州 + 怀戎县 + 涿鹿故地** 皆在叛军控制下——是范阳节度使腹地；唐军未克。**763 史朝义败死后**——妫州归唐、但**河北道实际处于河朔三镇半割据状态** 至唐末；**妫州属卢龙节度使（范阳节度使后身）辖区**——涿鹿故地仍是怀戎县东南境的怀古地。**中晚唐边塞诗中涿鹿一词频出**——多用于推到上古之极的诗作；李益、卢纶、戎昱 等中唐诗人多有涉。**晚唐五代间妫州沿革** 略有变动——**后唐 + 后晋间妫州一度入契丹**（936 石敬瑭割幽云十六州予契丹、妫州为十六州之一）——自此妫州 + 涿鹿故地入辽国疆域、属辽南京道。

**辽金元**　**辽南京道妫州** 治怀戎县——涿鹿故地继续作为塞外古地存在；**辽统和年间（10 世纪末）置奉圣州**（治怀来县大古城一带）——涿鹿故地属奉圣州；**金代沿辽制**——奉圣州属西京路；**金大定 + 大安 年间** 涿鹿一带为金蒙 / 金元边争的战场（虽非黄帝战蚩尤之野、是另一层"涿鹿之野"的物理重叠）。**元代** 涿鹿故地属上都路保安州——**保安州** 治今涿鹿县；**自元至清初保安州沿置**——涿鹿县之名不复立。

**明清**　**明初保安州属万全都司**——后改属宣府镇——是明长城外的边防州；**涿鹿故地** 继续作为塞外古地、间有方志载"黄帝战蚩尤之野"。**清代沿明制**——**清乾隆 + 嘉庆间考据学渐兴**——**顾祖禹《读史方舆纪要》、阎若璩《尚书古文疏证》、王鸣盛《十七史商榷》** 等 多有辨涿鹿 / 涿郡 / 涿州之别、**最终确认涿鹿在今河北怀来 + 涿鹿县一带**——是清代历史地理学的一项成果；**清代涿鹿县**——**民国二年（1913）改保安州为保安县、民国三年（1914）改保安县为涿鹿县**——自此"涿鹿"作为县名 重新立于地图上、是清亡后中华民国地理建置中"恢复古地名"的一例。

**民国至今**　**民国三年（1914）置涿鹿县**——属察哈尔特别区；1949 年后涿鹿县属察哈尔省、1952 年并入河北省、属张家口专区——**涿鹿县治在矾山镇北约 30 公里的涿鹿镇**（不在矾山）。**1990s 河北省文物部门在涿鹿县矾山镇调查 + 试掘**——发现**黄帝城遗址**夯土城垣；**1990s 矾山镇修黄帝庙 + 黄帝陵 + 三祖堂**（祀黄帝 + 炎帝 + 蚩尤三祖）——是当代官方与民间共同构建的"华夏始祖" 物理 anchor。**今游人来矾山** —— 所见的是黄帝庙 + 黄帝城遗址 + 釜山 + 阪泉（黄帝泉）+ 三祖堂——**盛唐妫州人在 755 所走过的涿鹿之野**——一千二百多年后是中华民族"始祖纪念" 的物理舞台；**而 755 时这一切尚在地下 / 在传说 / 在文献——唐人见涿鹿山、知此地与黄帝战蚩尤有关、但不知地下有夯土城垣**。涿鹿——是中国上古传说与考古遗存关系最深 + 最有争议的一片地。
