# 邺城

## 一句话

**漳水南岸的六朝霸都**——曹魏 + 后赵 + 冉魏 + 前燕 + 东魏 + 北齐 六代以此为都；**北周大象二年（580）杨坚焚邺**、自此约 175 年废墟；755 时唐人见的是相州邺县境内一片夯土残墙 + 三台基址 + 漳水仍流。

## 形貌

邺城在唐河北道**相州（州治安阳县）**北约四十里——具体在**漳水（浊漳水）在邺北城北侧 / 北墙外东流；邺城位漳水南岸**的台地上（今河北省邯郸市临漳县西南 + 河南省安阳市北郊跨界一带）。**地势**：华北平原西缘、太行山东麓的冲积平原；漳水自西北来、东南去；邺城选址正在漳水南岸沃野、北望河朔平畴、西望太行余脉——是东汉末曹操选址的水陆要冲。

**邺城分邺北城 + 邺南城两片相邻**——这是邺城最核心的物理事实，盛唐人凡至邺城遗址必先认这两片：

- **邺北城**：曹魏旧城——曹操建安九年（204）攻克袁绍后扩建；**东西七里、南北五里**（按《邺中记》《水经注》原文）。**漳水沿邺北城北墙外东流**；曹操时期始筑、曹魏五都之一、后赵 + 冉魏 + 前燕续都于此。**铜雀 / 金虎（金凤）/ 冰井三台立邺北城西墙之上**（详 [铜雀台](./铜雀台.md)）。
- **邺南城**：东魏 + 北齐都城——**东魏元象元年（538）始议、兴和元年至武定元年间（539-543）建成**；**位于邺北城南墙外、与邺北城共享城墙**（即邺北城南墙是邺南城北墙）；规模略大于邺北、是东魏北齐 43 年都城主体。**邺南城形制取法北魏洛阳城**——单一宫城居北中、御街南北贯通至朱明门、是中古都城规制承先启后的关键一节。

**两城合称"邺都"**——南北延伸约 6 公里、东西约 4 公里；漳水从城北穿过、为两城共用的天然水道。**邺都鼎盛时（北齐末）**人口逾二十万、是当时北方最大的都城之一。

唐时邺城是**相州境内一片废城**——经北周大象二年（580）杨坚焚毁后已 175 年（580→755）。**城墙残墙**：邺北城轮廓仍可辨、邺南城朱明门基址仍立；**宫殿尽毁**；**三台基址**：自北向南依次为冰井 + 铜雀 + 金虎（金凤）三处夯土台基、铜雀居中最高、约存数丈；**漳水仍北侧东流**——是这片废城最稳的物理 anchor。**唐相州治在安阳县**——距邺城遗址约四十里；唐相州下属**邺县**——县治在邺城东南另立（即今河北临漳老县城旧址、距邺城遗址约二十里）；**唐邺县并不在邺城遗址里**。

物候：华北平原西缘、暖温带季风——四时分明、春多沙、夏有漳水汛、秋高、冬寒——盛唐人秋日北上凭吊邺城多记"白茅黄叶、漳水浩荡"。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

邺城在唐人的回望里——是**六朝相继定都的废墟 + 中国中古最有戏剧性的"焚都"现场**。**两段六朝霸都 + 一次焚毁**构成唐人对邺城三层认知：

**第一段：曹魏霸府（204-220 + 五都之一 213-）**。**建安九年（204）八月**——曹操攻克袁绍据守的邺城、**自此以邺为大本营**；朝廷虽仍在许都，曹操实际坐镇邺、十数年间扩为霸府。**建安十五年（210）冬作铜雀台**、**十八年（213）作金虎台**、**十九年（214）作冰井台**——三台合称"邺中三台"（详 [铜雀台](./铜雀台.md)）。**建安二十一年（216）曹操进爵魏王**——以邺为魏王国都。**邺下文人集团**："三曹"（曹操、曹丕、曹植）+ **王粲、陈琳、徐干、阮瑀、应玚、刘桢**等曹氏幕府文人——形成中国文学史第一个集群创作现象；**孔融**虽列建安七子（曹丕《典论·论文》），但在朝廷为官、与曹操不合、建安十三年（208）为曹操所杀，**不入邺下幕宾**——这是唐人读《典论》《文选》时已知的史实。**建安二十五年（220）正月**——曹操卒于洛阳、葬邺西高陵；**同年十月**——曹丕代汉立魏、迁都洛阳；**邺退为曹魏五都之一**（长安 + 谯 + 许 + 邺 + 洛阳）。

**第二段：十六国诸都（335-370 间）**。曹魏 + 西晋之后邺城在十六国间三度为都：
- **后赵建武元年（335）**石勒之子石虎自襄国（今河北邢台）迁都邺；后赵都邺十六年（335-351 间）；
- **冉魏永兴元年（350）**冉闵杀石氏立冉魏、都邺；冉魏立国二年余（350-352）；
- **前燕元玺六年（357）**慕容俊自蓟（今北京）迁都邺；前燕都邺十三年（357-370）、至前秦建元六年（370）灭前燕、邺归前秦。

**这一段是邺城作为北方多胡都城的高峰**——后赵 + 冉魏 + 前燕皆鲜卑 / 羯 / 汉系政权、相继以邺为霸都。**后赵石虎时金虎台讳名改"金凤台"**——因石虎名"虎"、避讳改三台居南者为"金凤"；**这一讳与一百多年后北周武帝宇文邕（560-578 在位）的"虎"讳无关**——是中古最容易混淆的两个讳点之一，本 doc 取《邺中记》后赵讳口径。

**第三段：东魏北齐都（534-577）**。**东魏天平元年（534）十月**——高欢以洛阳近西魏宇文泰、不安、迫孝静帝自洛迁邺；自此**邺再为天下中心**。**东魏元象元年至武定元年间（538-543）**——为容纳自洛阳东迁的四十万官民、于邺北城南墙外新筑**邺南城**；自此邺都南北二城连体、规模远超曹魏旧制。**北齐天保元年（550）**——高洋受东魏禅、立北齐、续都邺；**北齐建都邺 27 年（550-577）**——是北朝最盛的文化中心之一：**佛教兴盛**（响堂山石窟在邺都西、北齐皇室开凿）、**胡汉文化交融**（北齐宫廷胡风浓厚、宫体诗发轫）、**律令制度**（《北齐律》为唐律之先声）。**北齐承光元年（577）**——北周武帝灭北齐、邺归北周。

**焚毁：北周大象二年（580）**。**这是邺城最戏剧的一笔**——北周宣帝早卒、外戚杨坚专政、相州总管**尉迟迥**据邺反；杨坚遣**韦孝宽**讨之、八月败迥于邺城下、迥自杀。**乱平后杨坚下令焚邺城**——**十月辛未诏毁邺城宫殿、徙居民四十六里南至安阳新城**（《资治通鉴》卷一七四原文）；自此邺都殿宇尽毁、城墙残破、相州治徙至安阳。**这一焚是中国中古最有名的"焚都"事件之一**——以政治动机毁掉一座六朝旧都、是杨坚开皇前夜对北齐残余势力的最后清算。**注**：焚邺城是 580 年杨坚作北周丞相时的政令（次年/581 杨坚代周立隋）——**不是隋炀帝焚邺**；这是唐人凭吊邺城时常混淆的史源、本 doc 据《通鉴》原文取杨坚 580 焚邺。

**自隋至唐**——邺城作为废城 175 年（580→755）。**隋开皇十年（590）**改相州治安阳、自此**相州 + 邺县治皆离邺城遗址**；隋唐两朝皆未再建邺城。**唐人来邺城仅为凭吊**——见的是漳水南岸一片旷野中的残墙台基。

### 此刻

天宝十四载这一年的邺城——**已废 175 年**；属唐河北道**相州邺县**境内、距相州治安阳约四十里。

**邺北城**：东西七里南北五里的轮廓在田野间隐约可辨；西墙上的三台基址（冰井 + 铜雀 + 金虎/金凤）仍立、夯土残基高数丈；东 + 南 + 北三墙残断在田畴间隔出方形格局；曹魏宫城基址在邺北城北中、已成耕地。**邺南城**：东西略宽于邺北、南墙朱明门基址犹存——是东魏北齐宫城与皇室宗庙的物理 anchor；**邺南城宫城朱明门遗址**在 1983-2010s 邺城考古队发掘中是出土物最丰富的一处。**漳水**：仍自邺北城北墙外东流——755 时漳水水量充沛、是河北道北部最稳的水道之一；唐人凭吊邺城多记"渡漳水入邺"。

**唐相州 + 邺县的治所**：**相州治安阳县**（距邺城遗址约四十里南）；**邺县治在今河北临漳老县治**（距邺城遗址约二十里东南）——两县治皆不在邺城遗址内。**邺城遗址当时是邺县属地的一片废墟**——田耕、放牧、岁岁有过客来凭吊。

**唐人对邺城的认知**——**六朝古都 + 建安文学故地 + 焚都现场**三重叠加：
- **建安文学的物理 anchor**——盛唐文人读《三国志》《典论》《建安七子集》皆知诗酒酬唱之地在漳水南岸；
- **东魏北齐都城的废墟**——盛唐人对北齐宫体诗 + 胡风的认知亦以邺都为物理参照；
- **杨坚焚邺的现场**——盛唐人读《隋书》《通鉴》皆知 580 年这一笔；邺城与**汉魏洛阳故城**、**建康**并列为中古重要废都（盛唐人通常以邺 + 建康并称——汉魏洛阳故城唐时仍部分有人居）。

**安史前夕**（即 755 这一刻）——再过半月（农历十一月初九）安禄山起兵于范阳；自范阳南下取河北诸州、邺城旁的相州、邺县皆在叛军南下必经之路上。**邺城自身是废墟、无守军**——但其周边的相州 + 邺县是安史叛军南下河南的咽喉之一。

### 此刻意味

对天宝十四载的相州人来说——邺城是**州北四十里那片"六朝旧京"**：田耕岁岁、过客时来；曹魏铜雀基 + 北齐朱明门 + 漳水东流——构成日常生活背景里的废都剪影。对盛唐文人来说——邺城是**中国中古文学与政治的双重 anchor**：建安诗酒在此 + 北齐宫体在此 + 杨坚焚都亦在此——一座城承载三段截然不同的历史质感、是别处难见的密度。

唐人尚不知道——**再过四年（乾元二年/759）三月**，这片废城周围将爆发安史之乱中唐军最大的一场败仗。**乾元元年（758）十月**——郭子仪、李光弼、王思礼、鲁炅等九节度使奉肃宗诏围安庆绪于邺；**约二十万唐军围邺（号称六十万）四个月**、城中粮尽人相食；**乾元二年（759）三月**史思明自范阳率十三万救邺——会战于邺城北、唐军大溃、节度使各还本镇、邺城战败——史称**邺城之败 / 相州大败**。**这是 755 时唐人想不到、却即将发生在邺城废墟周围的最大一战**——本 doc 严守 755 时态、相州大败属"755 后"事件、归历史长河段。

唐人更不知道——再过千余年（1983-2010s），中国社科院考古所邺城考古队在临漳 + 安阳跨界一带系统发掘、确认邺北城西墙 + 三台基址 + 邺南城朱明门与宫城；**今游人来邺城遗址**：所见的是一片广阔田野中的夯土残基 + 漳水东流 + 三台村西的三处台基——**与 755 唐人所见的物理景观在大轮廓上是连续的**：六朝兴亡、千四百多年、未易。

## 子条目

### 邺北城（曹魏 + 后赵 + 冉魏 + 前燕旧城）

**东西七里、南北五里**（《邺中记》《水经注》）——曹操建安九年（204）攻克扩建、十数年间为霸府；**建安二十一年（216）以后为魏王国都**；曹丕代汉迁洛阳后退为曹魏五都之一。**后赵 + 冉魏 + 前燕** 间相继都邺、皆居邺北城。**邺北城形制**：**东西轴向 + 南北轴线 + 宫城居北中** + **铜雀 / 金虎 / 冰井三台立西墙之上**——是早期都城规制的代表（与汉魏洛阳城形制相近、是从汉式都城向中古都城转型的关键节点）。**今邺北城遗址**：跨河北临漳县三台村 + 河南安阳市北郊——城墙残基在田野中可辨；西墙上的三台基址是邺北城最显的物理标识。

### 邺南城（东魏 + 北齐都城）

**东魏元象元年至武定元年间（538-543）建成**——为容纳自洛阳东迁的四十万官民、于邺北城南墙外新筑。**邺南城规模略大于邺北**——东西约 2800 米、南北约 3460 米；**南墙开三门、中为朱明门**（正南门）；**邺南城形制取法北魏洛阳城**——单一宫城居北中、御街南北贯通至朱明门、是中古都城规制承先启后的关键一节（前承北魏洛阳、后启隋唐长安洛阳）。**北齐 27 年都城**：高氏皇室宫殿 + 太庙 + 灵台皆在邺南城内；**胡汉文化交融的宫廷**——胡乐 + 胡舞 + 胡服 + 佛教兴盛皆以邺南城为物理舞台。**今邺南城遗址**：朱明门基址是出土物最丰富的一处——1983-2010s 邺城考古队系统发掘、确认朱明门 + 宫城南门 + 御街等多处遗存；**邺南城是中古都城形制研究的最重要 anchor 之一**。

### 漳水穿邺

**漳水（浊漳水）自邺北城北墙外东流**——是邺城最稳的物理 anchor、亦是邺都赖以为都的水源。**曹魏时漳水于邺北城西墙外引水入城**——曹操建"白沟"等水系连漳水至黄河；**北齐时漳水亦为邺南城供水**。**漳水南岸即邺南城**——南北二城共享漳水。**漳水汛期**：夏秋多涨、是河北道北部最不安定的水道之一；**北齐天保间（550-559）漳水多次涨溢**——北齐皇室在邺南城北筑堤防漳。**唐 755 时漳水仍东流邺北**——是这片废城唯一活的物理事实；**今漳水仍在原河道附近**——是邺城遗址最稳的地理标志。

### 邺都焚毁（北周大象二年/580 杨坚）

**这是邺城最戏剧的一笔**：

- **大象二年（580）五月**——北周宣帝宇文赟卒、太子静帝（八岁）立、外戚杨坚（隋文帝、时为静帝外祖父）以左大丞相专政；
- **大象二年（580）六月**——相州总管**尉迟迥**（北周宿将）据邺反、河北 + 山东 + 河南诸州响应；
- **大象二年（580）八月**——杨坚遣**韦孝宽**讨之、败迥于邺城下、迥自杀；
- **大象二年（580）十月辛未**——杨坚下令焚邺城宫殿、徙居民四十六里南至安阳新城（《资治通鉴》卷一七四原文）；
- **大象二年（580）以后**——邺城自此为废墟；**次年（开皇元年/581）二月**杨坚代周立隋；**开皇十年（590）**正式改相州治至安阳。

**焚邺的政治意义**：杨坚以焚都的方式断绝北齐 + 北周残余势力的物理凝聚点——这是中国中古最有名的"焚都"事件之一。**注**：焚邺是杨坚 580 年作北周丞相时的政令、**不是隋炀帝焚邺**——这是唐人凭吊邺城时常混淆的史源、本 doc 据《通鉴》《隋书·尉迟迥传》取杨坚 580 焚邺口径。

### 六朝相继定都（一城六代之奇）

**邺城是中国中古最特殊的一座"六代古都"**——曹魏（霸府 + 五都之一 204-220 + 220 后）+ 后赵（335-351）+ 冉魏（350-352）+ 前燕（357-370）+ 东魏（534-550）+ 北齐（550-577）——**六代为都、跨 373 年**（204→577）。**这是中国都城史上唯一一例**——六代相继定都同一城；与其他古都（长安十三朝 + 洛阳九朝 + 建康六朝）的"多朝一都"相比，邺城的"六代一都"更连续、更密集、更高 drama——**自曹魏霸府至北齐 文化中心、一脉相承的都城传统在此延续 373 年**——焚邺意味着一次彻底的物理终结。**755 时盛唐文人凭吊邺城**——意识到这是一座"一城承六朝"的废都。

### 邺下文学（建安 + 北齐两段）

**邺下文人集团**有两段高峰：
- **建安邺下**（204-220）：曹操 + 曹丕 + 曹植 + 王粲 + 陈琳 + 徐干 + 阮瑀 + 应玚 + 刘桢——形成"建安风骨"；**孔融**虽列建安七子（《典论·论文》），但不入邺下幕宾（在朝廷为官、208 为曹操所杀）；
- **北齐邺下**（550-577）：**邢邵 + 魏收 + 温子升**（"北地三才"）+ **颜之推**等——是北朝后期文学最盛之处；**《文选》编者萧统在建康、但《文选》传入北方后即以邺下为传习中心**；**颜之推《颜氏家训》**作于邺都（北齐末至北周初）。

**两段邺下文学的传统延续**——构成邺城作为"文学之都"的双重身份；**盛唐文人皆习建安 + 部分习北齐**——是 755 时邺城在文学史上的重量。

### 三台（详 [铜雀台](./铜雀台.md)）

**邺北城西墙上的三座高台**——自北向南依次为**冰井 + 铜雀 + 金虎（后赵石虎讳改"金凤"）**。**铜雀居中、最高**：《邺中记》记"高十丈"（约 23 米）、"有屋一百二十间"；曹植《铜雀台赋》作于建安十七年（212）。**金虎（金凤）居南**：建安十八年（213）作、高八丈；**后赵石虎在位时（333-349）因避石虎讳改"金凤台"**（《邺中记》后赵讳口径）。**冰井居北**：建安十九年（214）作、高八丈、上有冰室三所——曹魏宫廷夏日藏冰处。**三台在 755 时仍存夯土基址**——是邺城最显的物理标识。**详 [铜雀台](./铜雀台.md)**——本 doc 不重复展开。

## 历史长河（755 → 今）

**中晚唐**　**乾元二年（759）三月相州大败**——郭子仪、李光弼、王思礼、鲁炅等九节度使奉肃宗诏围安庆绪于邺（约二十万、号称六十万）、史思明自范阳南下救邺、唐军大溃。**这是安史之乱中唐军最大的一场败仗**——节度使各还本镇、肃宗丧失统一指挥诸将的能力、自此唐廷在河北的权威动摇；邺城废墟在战后受物理冲击进一步加重（邺南城朱明门一带在安史末考古层面有焚烧痕迹）。**中唐凡至河北凭吊邺城者皆叹其废**——**李白《邺都引》**（约盛唐至中唐间作、具体年代史源有歧）"君不见魏武营邺水……雄豪正自慕、何苦怨青楼"——是邺都凭吊诗的代表；**晚唐杜牧《赤壁》**（会昌二年至四年/842-844 黄州即齐安刺史任上作）"东风不与周郎便、铜雀春深锁二乔"——把铜雀台 / 邺城写进中国文学最有名的反讽典故。

**五代 + 两宋**　**五代后梁后唐间**邺城旧址在魏州（今河北大名）以南——**后唐同光元年（923）魏州升兴唐府、后晋天福二年（937）改邺都**——但**这个"邺都"是魏州升府的虚名、不是邺城遗址**；魏州城距邺城遗址约一百五十里、二者不同地。**北宋初**——河北仍是辽宋边境、邺城旧址在相州治安阳辖境；**北宋庆历八年（1048）黄河北流改道**——黄河自澶州（今河南濮阳）北决、改东北向流入海、与漳水合流于今河北沧州一带——**这次改道令漳水下游水系大变**、但邺城遗址段的漳水主流位置未改。**北宋诗人范成大、欧阳修、苏轼**等过相州凭吊邺都留诗——**苏轼《七绝·铜雀台》**"人间俯仰已千年"等是宋代邺都凭吊的代表。

**金元**　**金代邺城旧址在彰德府（今安阳）辖境**——金人对邺城没有大规模营建、三台基址续废；**金元间漳水多次改道**——元至元年间漳水大改道、自原邺北城北侧南移、邺城遗址逐渐脱离漳水主流。**元代邺城已成田野**——三台基址上有农人耕作、放牧；**元代地方志记三台"夯土残基、土堆数丈"**——是邺城物理保存状态的中期记录。

**明清**　**明嘉靖间**邺城遗址在彰德府临漳县与安阳县界——**嘉靖三十六年（1557）立"曹魏邺都故址"碑**于铜雀台基；**清乾隆四十年（1775）**陕西巡抚毕沅过临漳考察邺城（毕沅作《中州金石记》收录邺城碑刻）；**清末邺城已是河北南部 + 河南北部一处"考古名胜"**——但因没有大型纪念建筑（不同于隆中、五丈原均有重修纪念物），凭吊者主要是文人士大夫。

**民国至今**　1949 年后邺城遗址列河北省 + 河南省重点文物保护单位、**1988 年邺城遗址列全国重点文物保护单位**。**1983 年起中国社会科学院考古研究所邺城考古队系统发掘**——确认邺北城西墙 + 三台基址 + 邺南城朱明门与宫城 + 中轴御街——是中国都城考古最重要的工程之一。**邺城考古队主持的"邺南城朱明门遗址"发掘**（1983-1990s）——出土北齐瓦当、龙凤纹砖、宫殿建筑构件——是中古都城形制研究的重要资料。**2008 年起临漳县始建邺城博物馆 + 邺城遗址公园**——展示六朝遗存与发掘出土物；**邺城遗址跨河北临漳县（三台村 + 邺镇）+ 河南安阳市（北郊豫北农村）**——是华北平原最大的一处都城遗址群之一。**今游人来邺城遗址**——所见的是田野中的夯土残基 + 三台村西的三处台基 + 漳水东流（虽已远离原河道）——**与 755 唐人所见的物理景观在大轮廓上是连续的**：千四百多年、六朝兴亡、邺城以废墟之姿等待至今。
