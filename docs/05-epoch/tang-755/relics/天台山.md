# 天台山

## 一句话

台州临海郡天台县境内的浙东佛道圣山——**隋开皇十一年（591）智顗大师与晋王杨广共建国清寺、立中国佛教第一个本土宗派"天台宗"**——盛唐 755 时是江南东道佛教祖庭、亦是司马承祯早年隐修的道教圣地、佛道共构。

## 形貌

天台山在江南东道台州天台县东北约二十里——**自台州治临海县（今浙江台州临海市）西北行约五十里、入天台县；自县城北行二十里、即入天台山**。

山系：天台山主脊东西延展、北接四明山系、南连括苍山系——是浙东沿海一线山地之一段、海拔自数百米至最高华顶峰（约一一○○米）。山中谷壑深邃、林木茂盛、瀑泉密布——**石梁飞瀑、华顶望日、赤城栖霞**为天台山三大胜景；其中石梁飞瀑是一条天然石桥横跨涧上、瀑水自桥下奔涌而出，是盛唐人入山必访之处。

**国清寺在山南麓**——背依五峰（祥云、灵芝、灵禽、映霞、八桂）、面对双涧（北涧、西涧汇合处）。**寺址在天台山诸寺中最大、亦是天台宗主寺**——隋开皇十一年（591）智顗与晋王杨广共建、初名"天台寺"；隋大业元年（605）杨广即位为炀帝、敕改"国清寺"，取"寺若成、国即清"之义（《国清百录》记智顗临终遗书与炀帝敕改寺名经过）。**盛唐 755 时国清寺殿宇规模已具**——殿、堂、塔、廊数十座（具体盛唐殿名史源略缺、今所见为南宋至清历代重修）；寺中藏天台宗历代祖师章疏、《摩诃止观》《法华玄义》等智顗讲述、灌顶笔录的天台宗根本经典皆藏于此。

**桐柏宫在山西南**——道教福地、玄宗朝司马承祯长期隐居之处；与国清寺相距约十里、佛道两山共构。

物候——天台山近海、四时温润、夏不甚热、冬不甚寒；山中云海多、雾深、瀑响——是浙东最显著的山水佛道道场。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

天台山在唐人的回望里是**两重身份并立**——**中国佛教第一个本土宗派"天台宗"的祖庭** + **道教浙东福地（桐柏宫所在）**。

**佛教这一脉**——**智顗（538-597）**字德安、荆州华容（今湖北潜江）人、南陈名僧。先师承南岳慧思（515-577）、习禅与《法华》；陈太建七年（575）入天台山——自此**修禅二十年、立"天台宗"教观**。**天台宗以《法华经》为宗本、以"一心三观、三谛圆融"为教义、以"止观双修"为实修**——是中国佛教史上**第一个由中国僧人自创的本土宗派**（非自印度直接传入、亦非以单一经论为基的纯翻译宗派）。**隋开皇十一年（591）智顗与晋王杨广共建国清寺**——杨广时为晋王、镇扬州，受智顗菩萨戒、为之外护；寺始建时智顗尚在、卒于开皇十七年（597）；寺成于智顗身后、由弟子灌顶（561-632）主持完成（《国清百录·智顗临终遗书》记建寺始末）。**自智顗以下、灌顶 → 智威 → 慧威 → 玄朗 → 湛然**——天台宗历代祖师皆居天台山国清寺；**至 755 时湛然为天台八祖**（按天台宗八祖传统祖次：北齐慧文为初祖、南岳慧思二祖、智顗三祖、灌顶四祖、智威五祖、慧威六祖、玄朗七祖、湛然八祖；**自智顗以下为六传**——即天台第六代弘法人）、是**类似宗主 / 法主地位**的法系传承。

**道教这一脉**——天台山自南朝以来即为道教福地、有"桐柏山"之称（道教典籍中天台与桐柏常并称、亦或指山中桐柏宫所在那片山）。**司马承祯（646-735）**字子微、河内温（今河南温县）人——盛唐第一道士、是上清派十二代宗师。承祯先后隐居**嵩山、天台山**——在天台山隐居最久、达数十年；**唐睿宗景云二年（711）召入长安、玄宗开元十二年（724）命建王屋山阳台宫**——承祯自天台移居王屋。**至 755 时承祯卒已二十年（735 卒于王屋阳台宫）**——但天台山道教传统未断、桐柏宫仍是浙东道教中心。

**佛道并存**——盛唐人不视为冲突，国清寺与桐柏宫山中相邻；唐皇室尊道、又敬佛——玄宗朝既崇司马承祯、亦敬天台宗诸祖（玄宗于开元二十二年/734 召天台宗七祖玄朗入京问法、《佛祖统纪》记）。

### 此刻状态

天宝十四载这一年的天台山——**是江南东道佛道二教并隆的浙东圣山**。

**国清寺**在 755 时已立寺一百六十四年——智顗逝后一百五十八年；寺中**天台八祖湛然（711-782，自智顗以下为六传）正值壮年**、在山弘法 / 主导天台中兴（湛然《止观辅行》《法华文句记》等论著多撰于 750s-770s——其中部分论著正于 755 前后撰成、与本 doc 此刻状态相合）。**国清寺所藏的天台宗根本经典**——《摩诃止观》《法华玄义》《法华文句》等智顗讲述、灌顶笔录的"天台三大部"——皆于此时供养、传习。

**桐柏宫**在 755 时是司马承祯门下后学（如李含光、王屋派一系）维持的道教中心——但司马承祯本人已殁二十年；天台道教进入"祖师不在、法脉续存"的阶段。

**石梁飞瀑** 在 755 时已是浙东名景——李白《梦游天姥吟留别》（约开元末天宝初作，系年早于 755）中"天台四万八千丈、对此欲倒东南倾"——这是盛唐文学中天台山最有名的一笔；但**李白本人是否亲至天台山、史源不稳**（多数学者以为李白作此诗时未必亲至、是诗人想象 + 浙东传说之合）。此后中唐**白居易、元稹、皮日休、陆龟蒙**等诗人确有亲临天台山的记录——但**755 时尚在李白盛唐叙事时段**、天台山的文学化尚在累积期。

**佛教国际传播**——天台山在 755 时尚未传入日本。**日本最澄（767-822）入唐**是后事——延历二十三年（804）最澄随遣唐使船至明州、九月入天台山学法、次年（805）返日立**日本天台宗**——是天台山佛教影响日本之始。**至 755 时这一切尚未发生**——本 doc 严守"755 这一刻天台山尚未东传"口径；最澄事归历史长河段。

天宝十四载的天台山——**是浙东佛教祖庭 + 道教福地 + 山水名胜三合一**的圣山。再过半年安禄山起兵——但**江南东道远离两京、安史乱中相对稳定**；天台山未受直接兵燹、国清寺与桐柏宫续传不断。

### 此刻意味

对天宝十四载的天台县人来说，天台山是**县北那座深山佛道圣地**——日常香火、岁岁朝礼。对台州、临海郡的士庶来说——天台山是**州境内最重的山水 + 佛教祖庭**；州官岁有公祭。对江南东道的文人来说——天台山是"梦游天姥"的诗意化身、是"石梁飞瀑、华顶望日、赤城栖霞"的山水画卷。对天台宗法系内的僧人来说——天台山是**祖庭、是法脉之所系**；至湛然时代是天台宗中兴的现场。

唐人尚不知道——再过 50 年（804），日本僧人最澄会自明州登岸、入天台山学法、805 返日立日本天台宗——**自此天台山是中日佛教交流的物理 anchor**；再过 200 年（北宋初），天台宗经螺溪义寂、宝云义通、四明知礼等中兴、北宋中后期再度成为东亚佛教重镇；再过 1200 年——**国清寺仍在天台山南麓、唐代石经幢与塔基尚存、日本天台宗信徒至今每年至国清寺朝礼祖庭**。**这是天台山区别于楼观台（道教祖庭）/ 隆中（凭吊地）的关键**——它的**法脉 anchor 与物理 anchor 双重未断**：天台宗教法 1400 年承传不绝、国清寺寺址 1400 年位置未易。

## 子条目

### 国清寺

**天台宗主寺、智顗大师手立**。隋开皇十一年（591）始建、智顗逝后由灌顶完成、隋大业元年（605）炀帝敕改"国清寺"。**755 时天台八祖湛然（711-782，自智顗以下为六传）在山弘法 / 主导天台中兴**——湛然字辅元、号"荆溪尊者"，是中唐天台宗中兴之祖；其《止观辅行》《法华文句记》《法华玄义释签》等"天台五小部"论著正于 750s-770s 间撰成（具体 755 时是否任国清寺住持、史源未明确记载、本 doc 不预设）。**盛唐国清寺殿宇格局**——具体盛唐殿名史源略缺、本 doc 不预设；今所见国清寺主体为南宋以来历代重修、清雍正间大修定今格局；**唐代石经幢、塔基仍存**（1980s-2010s 浙江省考古研究所发掘确认）——是国清寺最重的物理 anchor。

### 智者塔院 / 智顗塔

**智顗大师葬塔之处——在天台山华顶峰下、距国清寺约二十里**。**智顗于隋开皇十七年（597）十一月卒于天台西门石城寺**（今天台山佛陇）——遗体归葬天台山中、立塔供奉。**唐 755 时智者塔已立 158 年**——是天台宗祖庭的核心朝礼地；凡入国清寺者多有礼智者塔。今所见智者塔为唐代塔基 + 后世重修结合的建筑——塔址未易。

### 石梁飞瀑

**天台山三大胜景之一**——天然石桥横跨涧上、瀑水自桥下奔涌而出。**盛唐文学化已起**——李白《梦游天姥吟留别》（约开元末天宝初）有"天台四万八千丈"语；中唐至宋则成"凡入天台必访石梁"的定式。**755 时石梁已是入天台山的核心物景**——但具体盛唐人题刻、本 doc 不预设；今石梁飞瀑仍是天台山国家级风景区核心。

### 桐柏宫

**天台山西南、道教福地**——南朝以来即有道观、唐玄宗朝因司马承祯而显。**司马承祯（646-735）**在赴长安（711）前长期隐居桐柏宫一带——其上清派道法多于此修成；后玄宗开元十二年（724）命承祯建王屋山阳台宫——承祯自天台移居王屋。**755 时桐柏宫仍是浙东道教中心**——但承祯本人已殁二十年；门下后学维持法脉。今桐柏宫为清雍正间重建、原唐代道观规模史源未详。

### 华顶峰

**天台山主峰**——海拔约一一○○米。**"华顶望日"为天台三大胜景之一**——盛唐人登华顶观日出、云海。智顗大师晚年常在华顶修禅——华顶峰下即智者塔院。山中云雾深、四时不一——盛唐文人凭吊天台、常以华顶为山之主象。

### 天台宗法系（智顗 → 灌顶 → 智威 → 慧威 → 玄朗 → 湛然）

**至 755 时自智顗以下已六传、湛然为天台八祖**（按天台八祖传统祖次：北齐慧文初祖、南岳慧思二祖再下接智顗以下汉传六代；下表"汉传 N 代"以自智顗起记）——
- **智顗**（538-597，汉传一代 / 八祖中之三祖）—— 立宗
- **灌顶**（561-632，汉传二代 / 八祖中之四祖）—— 智顗弟子、笔录《天台三大部》、完成国清寺
- **智威**（？-680s 间，汉传三代 / 八祖中之五祖）
- **慧威**（？-720s 间，汉传四代 / 八祖中之六祖）
- **玄朗**（673-754，汉传五代 / 八祖中之七祖）—— 玄宗开元二十二年（734）召入京、卒于天宝十三载（754，即 755 前一年）
- **湛然**（711-782，汉传六代 / **天台八祖**）—— 755 时正住天台、中兴天台宗

**这是中国佛教史上史载最完备 / 文献最齐备的本土宗派法系之一**——后世日本天台宗、宋四明天台宗皆从此脉派生。

## 历史长河（755 → 今）

**中晚唐**　安史乱中天台山未受直接兵燹——江南东道相对稳定。**贞元二十年（804）日本僧最澄入唐**——九月自明州（今宁波）登岸、入天台山随湛然弟子道邃、行满学天台法；**贞元二十一年（805）最澄返日、立日本天台宗、以比叡山延历寺为主寺**——自此**天台山成中日佛教交流的物理 anchor**；**最澄传日本天台、空海另传真言宗**（空海与最澄同期入唐、但在长安青龙寺从惠果学密法、归国后立真言宗于高野山、与天台宗并立而非天台支系）。**唐文宗、宣宗朝**天台宗续盛；**唐武宗会昌灭佛（840s）**——天台山诸寺受波及、但因偏远、损毁较北方诸寺为轻；**唐末战乱**（黄巢之乱 880s）波及浙东、国清寺一度衰落。

**五代北宋**　**吴越国（907-978）**统辖浙东、王朝崇佛——**钱镠、钱俶（吴越末主）大兴天台山**、国清寺续修、天台宗一度中兴。**北宋初（10 世纪末）**——天台宗经螺溪义寂（919-987）、宝云义通（927-988）、**四明知礼（960-1028）**中兴——北宋天台宗有"山家、山外"之争（知礼主"山家派"为正统）；天台山国清寺仍为北宋天台宗主寺。**北宋中期**——天台宗与禅宗、净土并立、是宋代汉传佛教三大宗派之一。

**南宋元**　**南宋绍兴二十一年（1151）**国清寺大修；南宋天台山一度因临安（杭州）近便、为南宋皇室经常驻跸朝礼之地。**元代**国清寺仍存——但天台宗整体随元代禅净合流而渐式微。

**明清**　**明洪武初敕修国清寺**；明嘉靖、万历间多次重修。**清雍正十二年（1734）国清寺大修**——奠定今主体格局。**清代**天台山为浙江名山、士大夫游宦江南者常至国清寺、智者塔礼祖。

**民国至今**　1949 年后国清寺先列浙江省重点文物保护单位、1983 年列汉族地区佛教全国重点寺院。**1980 年代起**国清寺逐步复修——殿宇、僧寮、藏经楼陆续重建；**日本天台宗信徒自 1980 年代起每年至国清寺朝礼祖庭**——日本比叡山延历寺与国清寺立"中日天台宗祖庭"关系。**1990s-2010s 浙江省考古研究所发掘**——国清寺东侧出土唐代石经幢、塔基；智者塔院为唐代塔基 + 历代重修结合。**今国清寺为天台山国家级风景名胜区核心**——殿宇、智者塔、石梁飞瀑、华顶峰俱在。**唐人在 755 所登的山、所礼的寺、所观的瀑**——位置在、传说在、唐代石经幢与塔基在；**这是天台山区别于多数 755 圣地的关键**——**法脉 1400 年承传不绝、物理 anchor 1400 年位置未易**。
