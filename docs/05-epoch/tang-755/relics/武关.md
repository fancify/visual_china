# 武关

## 一句话

**关中南大门、长安南出荆襄的咽喉**——上洛郡商洛县境武关河上游的山间关塞；**汉元年（BC 206）冬刘邦由武关入关、灞上受秦王子婴降**——汉室代秦的物理起点；**战国楚怀王 BC 299 入此被秦昭襄王扣留**——楚秦千年交界最沉痛的一笔；**755 时是长安南向江汉的物理 anchor**——关中四关（东潼、南武、西散、北萧）之南关。

## 形貌

武关在唐山南东道**商州上洛郡境**——具体在**商洛县（一说商南县）境武关河上游**（今陕西省商洛市丹凤县武关镇一带）。**地势**：秦岭东南余脉与丹江支流武关河切出的山间谷地——**北面是秦岭主脊**（峻峭难越）、**南面是商於山地**（连接南阳盆地）；**关城就立武关河谷北岸的山口处**——一关守住关中—商於—南阳—荆襄一线的咽喉。

**武关之险**：**北倚秦岭、南扼商山**——关城所在山口宽不过数百步；**武关河自关下东南流**——汇入丹江，再东南入汉水。**这是关中盆地与南阳盆地之间唯一可大军通行的天然孔道**——南北两侧群山合围，仅武关河谷一线可通。

**武关道**：**长安南出荆襄、江汉的主驿道**——自长安春明门东南行、过蓝田关入商州境、抵上洛县城、再东南经商洛县抵武关、出武关入南阳郡（邓州）、再南经襄州抵江陵 / 江夏。**武关在武关道中段偏南**——距上洛城约百唐里、距邓州约三百唐里。**武关道与潼关道并列为长安东出二大孔道**——潼关道东出洛阳 + 中原；武关道南出邓州 + 荆襄。

**关城形制**：盛唐武关具体形制 + 周长 + 门数史载零散——本 doc 不预设细节。**今所见武关关楼 + 关墙**：主体为明清重修 + 1980s 部分修复——非唐时形制；唐时武关的关楼夯土包石、关墙沿山口两侧山势延伸、具体尺寸需据考古勘察资料细化。

**与商於之地的关系**：**"商於"** = 武关一带（今陕西商洛市丹凤、商南、商州一带连接河南西峡、淅川的山地）——**战国楚秦的交界**；**张仪诓楚怀王"献商於六百里地"** 即此一带；**商於之地** 之名见《史记·张仪列传》《楚世家》。**武关是商於之地的核心关塞**——扼住武关 = 扼住商於 = 扼住关中南门。

**物候**：秦岭东南余脉——温带季风山地气候；夏雨较多、冬寒；武关河谷山中林木茂盛、**关城内外多有戍兵 + 驿马 + 商旅 + 关吏** —— 是盛唐南向荆襄方向的主要关塞。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

武关在唐人的回望里——是**关中南大门 + 楚秦千年交界 + 汉室代秦物理起点**。

**武关的设立**：**春秋战国间已立**——战国数百年间, 楚秦争夺商於之地（自春秋楚自南阳北扩、秦自关中东扩；楚秦在商於之地反复争夺）。**战国中后期武关定属秦**——成秦东南向楚地的前哨关塞；**秦昭襄王（前 306-前 251 在位）** 时武关已是秦楚关系的物理 anchor。

**武关在战国秦汉间的关键时刻**：
- **战国楚怀王 BC 299 入武关被扣**：**楚怀王熊槐（BC 328-BC 299 在位）** 听信秦昭襄王之约——入武关相会、欲结秦楚之好；**至武关、秦伏兵截楚怀王、押至咸阳**——楚怀王 BC 296 死于秦国客馆。这是中国先秦外交史最有名的"约会即扣留"事件之一。**自此武关 = 楚国国君的死地**——楚人深恨之。
- **汉元年（BC 206）冬刘邦由武关入关**：**秦末楚汉之争中**——**刘邦取武关道 + 不取函谷关道**：自南阳北上、过武关、入商於、抵蓝田、至灞上；**秦王子婴白马素车出降于轵道旁**——秦亡。**自此刘邦由武关入关 = 汉室代秦的物理起点**；**项羽 BC 206 率诸侯兵到函谷关 (古函谷关, 弘农县) + 强行突入关中**——与刘邦走的不是同一条路（楚汉时无潼关——潼关东汉建安元年/196 后才置）。**武关因此在中国历史上**成"秦汉转换的关塞 anchor"。

**两汉至北朝**：**东汉末赤眉入关走武关道**（更始三年/25 赤眉樊崇等自南阳北上、过武关、入关中）——是西汉末关中变局的物理路线。**北朝间武关续作关中南门**——延续秦汉以来的关塞地位。

**唐武关的政治军事地位**：**唐承前代之制、置武关为关塞**；**武关守军由唐廷直接指挥**——属商州上洛郡（天宝元年/742 改商州为上洛郡；乾元元年/758 复商州）；上洛郡又属山南东道。**自唐初至盛唐 138 年（618-755）武关无大战**——是太平时代；**武关有戍兵, 额数史载未详**——日常守关 + 验商旅 + 收关税。

**武关与长安、荆襄的关系**：**长安东南行 + 过蓝田关 + 入商州境 + 至上洛 + 至武关**——约六百唐里；**武关东南出 + 经邓州 + 襄州 + 至江陵 / 江夏**——约一千唐里。**这一条武关道**——是盛唐长安南向荆襄的最直接驿路；亦是江淮、荆襄租赋经汉水北运长安的主要通道之一（漕路经汉水入丹江、走武关道陆运至长安——与黄河漕路潼关线并立）。

**安史前夕**（即 755 这一刻）——武关本身仍是平时关塞；商州 / 上洛郡境内一切如常。**755 末安禄山起兵于范阳**——长安东出潼关道吃紧；**武关道的重要性陡然上升**——朝廷自长安调江淮、荆襄租赋入京、多改循武关道（详见商州 doc 与本 doc"此刻"段）。

### 此刻

天宝十四载这一年的武关——**关塞严整、戍兵数百、扼武关道中段**。

**关城状态**：盛唐 755 时武关主体仍立、有驿馆 + 戍兵营 + 关吏；具体形制 + 守军规模史载零散——本 doc 不预设无源数字。**关城内外**：武关河谷两侧山势夹峙；**驿路自关楼东南门出**——下武关河谷、循丹江东南行至邓州。

**755 时武关上的政治氛围**：武关是平时关塞——关税收 + 商旅过关 + 戍兵巡逻；**关吏对南北过关的官员、商旅** 多有验牒；**南向荆襄 + 江汉的人流密度** 不如东向洛阳的潼关、亦不如西南向蜀地的散关——但作为长安南门、地位仍重。

**755 末 + 756 初的武关命运**：**756 六月哥舒翰潼关之败 + 七月玄宗西奔入蜀** 后——长安失守、关中无险可恃；**武关道在安史乱中的真正分量**——是作为 **江淮 + 荆襄租赋经长江 → 汉江 → 丹江 → 武关 → 长安**（再过武关后由唐廷收京后）的备用转输通道。**注**：玄宗西奔走的是西向陈仓道 + 散关（详见散关 doc）——不取武关道（武关道南向荆襄、非入蜀路）；太子李亨北行灵武走的是关中西北的泾邠路（详见商州 doc）——亦不取武关道。**武关道的安史角色**——是在**至德 / 乾元间唐军复京后**作为江淮 + 荆襄输入长安的备用漕路；具体过关运量、班次史源记载有限，本 doc 不预设细节。

**至德二载（757）十月唐军复长安**——之后武关道作为江汉北运的备用通道激活；**乾元 / 上元间**武关道与潼关道并行作长安补给的双线之一。

### 此刻意味

对天宝十四载的商州 / 上洛郡人来说——武关是**自家郡境的南门**：日常戍兵 + 商旅过关 + 关吏验牒；上洛城里人尚未察觉的是、**再过半年（755 末-756 初）武关道的过境官使将骤增**——朝廷自江淮、荆襄调粮赋北运、武关道驿务进入百年罕见的繁忙期。

对长安那一代士人来说——武关是**关中南大门 + 史书里的"楚秦交界"**：**盛唐诗中"武关"二字偶有出现**（高适、岑参等边塞诗或涉、王维《送杨长史赴果州》"褒斜不容幰、之子去何之" 等南向诗或近）——但**武关的诗咏密度** 远不如潼关、阳关、玉门关——因武关道在盛唐属于"平时驿路、非边塞"——文学密度自然较低。**唐人嘴里的"武关"**——既是地理实体（南向荆襄第一关）、也是史书符号（楚怀王扣留地 + 刘邦入关地）。

唐人尚不知道——再过 100 余年（中晚唐）武关道因江淮租赋北运而长期繁忙；再过 1200 余年（民国 + 1980s）武关古关址被多次勘察 + 部分修缮；今游人来武关——**所见的是明清主体的武关关楼 + 关墙 + 武关河谷古道 + "刘邦入关"主题景观**——**唐人在 755 所见的那座武关**：千二百多年后**唐时形制已不复**——但**关塞位置 + 武关河谷 + 武关道线** 一千二百多年未易。

## 子条目

### 武关河（武关南面的水源）

**武关河**——丹江北支、自秦岭东南余脉发源、东南流入丹江、再东南入汉水。**武关就立武关河上游北岸的山口**——河水绕关城南侧东南去；**水深不大、不通大舟**——但是武关道驿路沿河的天然走廊。**武关河谷的地形**——两侧山势夹峙、谷宽数百步、谷长约百里——是关中盆地与南阳盆地之间最易行的山间通道。

### 商於之地（武关一带的战国地理称）

**商於之地** = **商**（今陕西商洛市商州、丹凤一带）+**於**（今河南西峡、淅川一带）——是战国楚秦交界的山间地带；**约六百里**（《史记·张仪列传》"商於之地六百里"）。**战国楚怀王（BC 328-299）** 时——**张仪诓楚怀王"献商於之地六百里"** 以诱楚绝齐；**楚怀王中计绝齐、张仪反悔（只允六里）**——楚怀王怒攻秦、于丹阳 + 蓝田连败——**自此楚国衰落、武关一带定属秦**。**"商於" 之名**自此与"张仪诈楚" 绑定——成中国先秦政治史最有名的"地理诱饵" 之一。**唐人 755 时凡读《史记》《战国策》者**——必知商於 + 武关的故事。

### 楚怀王 BC 299 入武关被扣（楚秦千年交界最沉痛一笔）

**战国楚怀王熊槐（BC 328-BC 299 在位）**——是楚国在战国后期最重要的国君之一。**BC 299**——**秦昭襄王约楚怀王会于武关**、欲修秦楚之好；**楚怀王自郢都北上、过武关入秦境**——**秦伏兵截之、押至咸阳**；**秦昭襄王要楚怀王割黔中地为质 + 楚怀王不肯**——**被拘秦三年**；**BC 296 楚怀王死于秦国客馆**——**楚人迎丧归葬**——楚国朝野悲愤。**自此楚国对秦的仇恨深化**——**直到 BC 223 秦灭楚**——这中间是 76 年的国仇。**"楚怀王入武关" = 中国先秦外交史"约会即扣留" 的代名词**——亦是武关在历史长河中最沉痛的一刻。

**注**：**楚怀王熊槐** 是战国楚怀王（BC 328-299 在位）——**非春秋楚怀王熊蚡冒**（BC 757-741 在位）——本 doc 严守战国楚怀王身份。

### 刘邦由武关入关（汉元年/BC 206 冬·汉室代秦的物理起点）

**秦末楚汉之争**中——**刘邦与项羽分军入关**：**项羽率诸侯兵**自函谷关方向（古函谷关、弘农县境）入关——**鸿门会后强行突入关中**；**刘邦率偏军**自南阳北上——**取武关道**：**汉元年（BC 206）十月**——**刘邦攻破武关 + 入商於 + 抵蓝田 + 至灞上**；**秦王子婴白马素车、衔璧出降于轵道旁**——**秦亡**。**刘邦由武关入关**——是中国历史上"汉室代秦" 的物理起点；亦是武关在历史长河中最有名的一次过往。

**为何刘邦取武关道不取函谷关**：**函谷关有秦军主力据守**；**武关道一线秦军已弱**——刘邦取易难之路、绕开主力。**自此"由武关入关" 成中国军事史"避正取奇" 的经典案例之一**——唐人 755 时凡读《史记·高祖本纪》者必知。

**注**：**楚汉时无潼关**——潼关 **东汉建安元年（196）后** 才置；项羽 BC 206 后率诸侯兵到 **函谷关（古函谷关、弘农县境）**、被刘邦守军所阻、强行突入关中——不经潼关。本 doc 严守这一时序。

**注**：**项羽 BC 206 率诸侯兵到函谷关（古函谷关、弘农县境）+ 强行突入关中**——**不是武关**；本 doc 严守"刘邦由武关、项羽由函谷"的区分（楚汉时无潼关——潼关东汉建安元年/196 后才置）。

### 武关道（长安南出荆襄的主驿道）

**武关道**——**长安 → 蓝田关 → 商州（上洛）→ 武关 → 邓州（南阳）→ 襄州（襄阳）→ 江陵 / 江夏**——全程约一千五六百唐里。**武关道与潼关道并列**为长安东出二大孔道：**潼关道**东出洛阳 + 中原（东西轴）；**武关道**南出荆襄 + 江汉（南北轴）。**武关在武关道中段偏南**——是这条道的核心关塞。

**盛唐武关道的角色**：**平时是长安南向荆襄的驿路 + 江汉租赋北运的备用漕路（与汉水接驳）**；**安史乱中（756-763）武关道的角色骤升**——成江淮、荆襄租赋经长江 → 汉江 → 丹江 → 武关 → 长安的备用通道；详见 [商州](../cities/商州.md) 与本 doc"此刻"段。

### 关中四关（潼关 + 武关 + 散关 + 萧关）

**关中盆地的天然防御体系**——四关合围：
- **东关：潼关**——黄河南岸 + 秦岭北麓夹峙；扼东向洛阳之冲；
- **南关：武关**——秦岭东南余脉 + 武关河谷夹峙（上洛郡境）；扼南向荆襄之冲；
- **西关：散关**——秦岭主脊北麓凤州境；扼西南向汉中蜀地之冲；
- **北关：萧关**——六盘山一线（原州境）；扼北向河西陇右之冲。

**四关在中国地理上是最完整的"关中盆地防御体系"**——盛唐 755 时四关皆立、守军按例驻屯。**武关在四关中的特殊性**：**其他三关多以"关中防御外敌" 为主**；**武关多以"关中通南方" 为主**——南向荆襄、江汉、岭南；**故武关更接近"驿道关塞" + 较少作"边境关塞"**——这是武关与潼关、萧关、散关在功能上的差别。

**注**：四关之称是后世地理归纳——盛唐 755 时唐人是否以此固定四关称呼史源略异；本 doc 取通称作地理 anchor、不预设唐人固定认知。

### 武关与商州的行政关系

**武关 755 时属商州 / 上洛郡（天宝元年/742 改商州为上洛郡；乾元元年/758 复商州）**；**州治上洛县**（今陕西商洛市商州区）；**武关在商洛县 / 商南县一带**——具体县属史源略异（《元和郡县图志》载在商洛县境；后世志书或入商南）。**商州 + 上洛郡又属山南东道**——开元二十一年（733）分山南为东西二道后、商州属东道。**注**：本 doc 将武关归 山南东道商州（上洛郡），不写"关内道"或"陕州" / "虢州"——商州在 755 时属山南东道而非关内道，是因武关道南下属山南方向。

## 历史长河（755 → 今）

**中晚唐**　**安史乱中（756-763）武关道作江淮、荆襄租赋北运的备用通道**——上洛郡 + 武关一线驿务繁忙；具体班次、运量史源记载有限。**至德二载（757）末玄宗自蜀返长安**——亦循陈仓道 / 散关一线、未取武关道。**至德二载十月长安复**、之后武关道与潼关道并行作江南北运的双线之一。**中晚唐武关道续作驿路**——但安史后唐廷整体南向依赖加深、武关道地位渐升。

**唐末黄巢 + 五代**　**广明元年（880）黄巢入长安**——**唐僖宗西奔成都走散关**（与玄宗同路），**未走武关道**。**五代间武关多次易主**——为后梁、后唐、前蜀等政权所争。

**两宋**　**北宋武关属商州**——续作关塞；**南宋武关一带反复在宋金间易手**——但因武关南向襄阳、不在宋金主战线（宋金主战线在淮河 / 川陕）——武关在南宋的战略地位不如散关。

**金元明清**　**明洪武七年（1374）大规模重修武关 + 关楼**——形成今所见关楼 + 关墙的明代主体；**明嘉靖间复修**。**清代武关续作关塞、规模渐降**——成"关中四关" 中地位较低的一关。**清代商州升直隶州**（雍正三年/1725）——武关在直隶商州境内。

**民国至今**　**1949 年后武关古关址列陕西省重点文物保护单位**。**1980s 部分修复**——含关楼局部重建；**1990s-2010s 多次考古勘察**——确认武关关墙残基、武关河古驿道线、武关道驿堡遗迹。**今陕西商洛市丹凤县武关镇**：所见的是**明清主体的武关关楼 + 关墙 + 武关河谷古道 + "刘邦入关 / 楚怀王扣留" 主题景观**——**唐人在 755 所见的那座武关**：千二百多年后**唐时形制已不复**；但**关塞位置 + 武关河谷 + 武关道线** 一千二百多年未易——**这是中国关塞史的常态**。
