# 草堂寺

## 一句话

京兆府鄠县圭峰山北麓的古寺——后秦弘始三年（401）姚兴迎鸠摩罗什入长安、于此地的逍遥园译经十二载、出大乘经论三十五部凡二百九十四卷；**罗什所译《中论》《百论》《十二门论》为中国三论宗祖典——草堂寺由是为三论宗祖庭、兼为罗什译经所涉法华、般若、净土等共同记忆**。

## 形貌

草堂寺在京兆府鄠县——县城东南约三十唐里、距长安城西南约五十唐里——**圭峰山北麓**，一处山口外的台塬。

寺址在**圭峰**山下——圭峰是终南山西段北伸的一支，山形如圭（玉版）状端立；自终南主脊向北延出、与楼观台所依的就峪、田峪同属终南北麓水系，但圭峰更西、更近鄠县城。**寺所据是一片山口外平地**——南依圭峰、北望渭水、东西两侧界小溪——是关中平原边缘进入终南山的一处天然停顿。寺门朝南、面对圭峰；殿宇沿南北轴线一进、二进——天王殿、大殿、罗什殿；殿后高台之上即**罗什塔**——后秦弘始十五年（413）罗什示寂后立、为罗什墓塔（一名"舌舍利塔"——传罗什生前发愿"若所译经无误，焚身后舌不烂"，死后焚身、舌存、立塔以奉）。

寺名"草堂"——源于后秦罗什在此译经时**结草堂为屋**之故；自后秦至唐沿用此名。隋开皇间一次重修、唐贞观间一次重修、唐玄宗朝复多次修缮——755 时所见的殿宇为唐式样、但具体规模盛唐史源未详（《元和郡县图志》《长安志》仅记寺方位与沿革大概，殿名 / 形制不入册）。今所见草堂寺殿宇为明清之后历次重修，**罗什塔主体亦为后世重建——但塔址未易**。

圭峰山一带的物候——背靠终南主脊、面向关中平原，是终南北麓诸寺观共同的气候带：夏多云气、冬有雪线；山间松柏成林、溪水自圭峰下来绕寺而过。寺前有古柏数株——传为后秦罗什手植或唐人补植，史源不明。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

草堂寺在唐人的回望里是**鸠摩罗什的译经地**——一段从西域龟兹到长安、再到鄠县圭峰山下的传译之路。

**鸠摩罗什（344-413）**——西域**龟兹**（今新疆库车）国师之子，父鸠摩罗炎为天竺人、母为龟兹王妹耆婆。罗什七岁出家、博通大小乘——是西域佛教史上最重要的一位译经师。**前秦建元十八年（382）**——前秦苻坚遣大将吕光西征龟兹、**建元二十年（384）**吕光破龟兹、掳罗什东归；途中苻坚被杀（385 淝水之后），吕光遂据凉州自立后凉——**罗什被困凉州十七年**（385-401），习汉语、待时。

**后秦弘始三年（401）十二月**——后秦姚兴攻后凉、吕隆降（后凉正式亡于弘始五年/403），迎罗什入长安。**姚兴待罗什以国师之礼**——居于长安城西**西明阁**（一作西明寺、非唐代长安西明寺）；译经场则在**鄠县逍遥园**——即此地。逍遥园是后秦皇家苑囿之一、设于鄠县圭峰山下；姚兴以此园为罗什译经场所、令僧肇、僧叡、道生、道融等八百余僧助之——是中国佛教史上第一个大规模的官方译经道场。**自弘始三年至弘始十五年（401-413）**——罗什在长安西明阁与鄠县逍遥园之间往来十二载、**译出大乘经论三十五部凡二百九十四卷**（数依《出三藏记集》）——内含《大智度论》《妙法莲华经》《维摩诘经》《金刚经》《阿弥陀经》《中论》《百论》《十二门论》《成实论》等。**罗什译经文字之畅达、义理之精审——为中国佛教翻译史的第一个高峰**；今汉传佛教所诵《妙法莲华经》《金刚经》《阿弥陀经》皆罗什旧译——千四百年未易。

**罗什译《中论》《百论》《十二门论》（合称"三论"）**——立中观空义、破有破无——是印度中观学派在中国的最早系统传译；后世以此三论为宗者称**"三论宗"**。罗什另译《大智度论》——为《大品般若》释经、亦为三论宗 + 天台宗 + 禅宗共依之大经。**罗什译《阿弥陀经》**——为净土宗源头之一（《无量寿经》系三国魏康僧铠译，非罗什所出）；**译《维摩诘经》《妙法莲华经》**——为禅宗、天台宗共依。故论中国汉传佛教各宗之源——**罗什实为多宗共祖**；草堂寺以罗什译经地、为多宗共同朝礼之处。

**罗什示寂于弘始十五年（413）**——一说卒于长安、一说卒于鄠县逍遥园（《高僧传》记"卒于长安大寺"；《晋书》记"卒"未明地点；本 doc 取"卒于长安、舌舍利归葬鄠县"之通说口径）。**罗什生前自誓**："若所传译无谬，焚身后舌不烂。"——示寂后门人依西域法焚身、**舌果不烂**——遂以舌舍利建塔于鄠县逍遥园（即后草堂寺）；此塔即**罗什舌舍利塔**——一名罗什塔。**413 立、姚秦弘始十五年——是草堂寺最古的纪年遗物**（虽塔体已经后世屡修、但塔址与纪念性未易）。

**三论宗在中国的传承**——罗什门下出僧肇、僧叡、道生、道融（合称"什门四圣"）；僧肇（384-414）著《肇论》——四论（物不迁 / 不真空 / 般若无知 / 涅槃无名）——是中国佛教史上第一部成熟的般若学论著、罗什义学之直传。三论宗经南北朝、隋间传至**隋吉藏（549-623）**——吉藏住会稽嘉祥寺、撰《三论玄义》《中观论疏》《百论疏》《十二门论疏》——是三论宗集大成者、亦称嘉祥宗。**至唐初三论宗已渐衰**——盛唐间被法相宗（玄奘 + 窥基，玄奘 645 归长安、664 卒；窥基 632-682）、华严宗、禅宗（南宗 745 神会入洛阳后大兴）渐取代。**755 时三论宗在两京已非显学**——但**草堂寺作为罗什译经地、罗什塔所在、三论宗祖庭——仍是天下佛门朝礼之处**。

**隋唐间草堂寺**——隋开皇间一次重修；**唐贞观间一次重修**（具体年份史源略异）；**唐玄宗朝复多次修缮**——玄宗朝崇道亦不废佛、对长安西郊诸名寺多有恩赏。盛唐间凡入终南访道者、入鄠县访古者、入圭峰朝礼者——皆以草堂寺为终南北麓与楼观台（道教祖庭）并列的两座祖庭之一。**长安士人往来鄠县草堂寺、楼观台、终南山一线——是盛唐最稳的一条修道访古主线**。

### 此刻状态

天宝十四载这一年的草堂寺——**距罗什示寂三百四十二年；罗什塔仍立；三论宗已渐衰、但寺仍是天下罗什门朝礼之处**。

寺主持是何人、僧团多少——盛唐史源未详、本 doc 不预设。**罗什塔仍在寺后高台之上**——塔体已经隋唐间几次重修、形制非姚秦原貌、但塔址未易、纪念性未易。**罗什殿**（祀鸠摩罗什像）在寺中轴线最北、为唐人朝礼罗什的核心殿；殿前传有罗什当年手植柏（亦或唐人补植——史源不明）。**寺中诸殿日常香火**——非两京大寺规模、但作为鄠县与长安士人间的近郊名寺，岁岁有施主、有朝礼者。

**杜甫（712-770）与草堂寺**——杜甫天宝五载（746）至长安、困居长安十年（746-755）——其间数次至鄠县（杜甫有友人居鄠县、亦有姻亲在此）。**一说杜甫作《题草堂寺》诗——但此诗系年存疑、亦或非杜诗**（杜诗集中关于鄠县诸寺有零星提及、但《题草堂寺》一首是否为杜手笔、是否作于鄠县草堂寺、史源略异——本 doc 不预设其确凿）。**盛唐凡入鄠县者，皆至草堂寺一谒罗什塔——这是 755 时的稳态**。

天宝十四载的草堂寺——香火依旧、罗什塔依旧；但**寺外的天下**——再过几个月，安禄山起兵于范阳；再过半年潼关失守、玄宗西奔入蜀（路线为长安出延秋门 → 咸阳 → 马嵬 → 扶风 → 陈仓 → 散关 → 入蜀，不经鄠县）。再过一两年——长安、洛阳俱沦于乱兵；鄠县因在终南北麓、相对未受重创——草堂寺得以保全。

### 此刻意味

对天宝十四载的鄠县人来说，草堂寺是**县东南圭峰山下那座最古的寺**——日常香火、岁岁朝礼。对长安那一代僧俗来说，草堂寺是**鸠摩罗什译经地、罗什塔所在、汉传佛教多宗之祖庭**——离长安西南五十里、可日还、是访古朝礼短游的首选之一。对天下三论宗、禅宗、净土宗、天台宗门下来说——草堂寺是**罗什的家、汉译佛典的祖庭**。

唐人尚不知道——再过千年，三论宗在汉地早已淡出宗派之列；但**罗什的旧译——《妙法莲华经》《金刚经》《阿弥陀经》《维摩诘经》《大智度论》——成为整个汉传佛教千年之内最常诵之经**。论传承之广、之久——草堂寺所立的纪念，**超越了三论宗一宗、化为整个汉传佛教共同的祖庭**。

## 子条目

### 罗什塔（罗什舌舍利塔）

**寺后高台之上——后秦弘始十五年（413）始立的罗什墓塔**。传罗什生前自誓"若所传译无谬，焚身后舌不烂"——示寂后门人依西域法焚身、舌果不烂——遂以舌舍利建塔于鄠县逍遥园（即后草堂寺）。**413 立、姚秦弘始十五年——是草堂寺最古的纪年遗物**。塔体已经隋唐间数次重修——盛唐 755 时所见塔形非姚秦原貌、形制盛唐史源未详；今所见罗什塔为后世（一说宋金、一说明）重建之八角十二层石塔，但**塔址未易、纪念性未易**。**唐人在 755 所朝礼的这座塔，与今游人所见在地点上同一**——这是草堂寺最重的物理 anchor。

### 罗什殿（祀鸠摩罗什像）

寺中轴线最北——祀鸠摩罗什像之主殿。盛唐殿宇规模、像之衣冠形制——史源未详。**755 时罗什在此殿中接受日常香火**——是三论宗、禅宗、净土宗、天台宗诸宗门下共同朝礼的去处。今所见罗什殿为后世重修。

### 逍遥园（罗什译经场旧址）

**草堂寺前身——后秦姚兴朝皇家苑囿**。姚兴弘始三年（401）迎罗什入长安、即以此园为罗什译经场所；罗什在此与僧肇、僧叡、道生、道融等八百余僧译大乘经论三十五部凡二百九十四卷——历十二载。**逍遥园即后草堂寺地——但今寺名"草堂"自后秦罗什结草堂为屋而得**；逍遥园之名至唐已废、仅存于《晋书》《高僧传》等典籍。**盛唐人立于草堂寺中——脚下即逍遥园故址、罗什所译《妙法莲华经》《金刚经》皆出于此**。

### 圭峰（草堂寺所依之山）

寺所依的山——终南山西段北伸的一支、山形如圭（玉版）状端立而得名。**圭峰在唐时是鄠县与终南山间的天然标识**——自鄠县城东南行三十里，远见圭峰，便知草堂寺已近。圭峰与楼观台所依的终南就峪、田峪同属终南北麓水系——三处距离约六十至一百唐里、构成盛唐"鄠县—周至—终南"一线最稳的访道访古主线。**唐人在 755 所见的圭峰——与今登终南西段所见在地理上同一**。

### 鸠摩罗什的西域 - 凉州 - 长安 - 鄠县路

**罗什东行之路**——龟兹（西域佛教中心）→ 凉州（被掳，居十七年，习汉语）→ 长安（后秦姚兴迎入，居西明阁）→ 鄠县逍遥园（译经场）→ 长安（示寂；舌舍利归葬鄠县）。**自西域至关中，自王子至译师，自十七年的凉州滞留到十二年的长安译经——罗什之一生即一段佛教自西域入中原的活地图**。草堂寺所立的纪念，不只是一寺一塔——而是**汉传佛教第一部成熟译本的诞生处**。

### 三论宗祖庭与多宗共源

**罗什译《中论》《百论》《十二门论》——为三论宗祖典**；其门下僧肇撰《肇论》——立中观般若学；至隋吉藏集三论宗大成。**但罗什译经远不止三论**——《大智度论》《妙法莲华经》《维摩诘经》《金刚经》《阿弥陀经》——分属般若、法华、净土、维摩诸系；今汉传佛教各宗之祖典几乎皆出罗什一人之手。**故草堂寺虽为三论宗祖庭，兼为罗什译经所涉法华、般若、净土等共同记忆**——凡读罗什旧译者，至草堂寺皆有所感。**755 时三论宗在两京已渐衰、但草堂寺仍香火不绝**——这正是因为它的纪念性早已超越一宗。

## 历史长河（755 → 今）

**中晚唐**　安史乱中草堂寺受波及不重——玄宗西奔路线（长安出延秋门 → 咸阳 → 马嵬 → 扶风 → 陈仓 → 散关 → 入蜀）不经鄠县，鄠县因在终南北麓、兵燹未及寺。**自德宗朝起**——三论宗在汉地渐式微、法相宗亦衰、禅宗（南宗）独大；但草堂寺仍是长安西郊朝礼名寺。**会昌五年（845）武宗灭佛**——天下寺院大半被毁、僧尼还俗——草堂寺亦在此次劫中受损（具体程度史载未详）；宣宗复佛后渐复。**唐末战乱（黄巢之乱 880s）**——草堂寺殿宇毁过半、罗什塔受损。

**五代北宋**　唐亡（907）后草堂寺一度荒。**北宋初仍有修缮**——但三论宗在宋已绝迹（仅存罗什所译三论作为佛教共同典籍），草堂寺地位由"宗派祖庭"降为"罗什译经地纪念寺"。**金元间**——金大定 + 元至元两朝有修缮记录；元代圭峰山下的草堂寺与楼观台同为关中朝礼名所。

**明清**　明洪武初重修；明万历间复修；清康熙、雍正、乾隆三朝俱有修缮记录。**清代草堂寺为陕西西郊一处中等规模佛寺**——但**罗什塔的纪念性始终未减**——士人入终南访古、必至草堂寺一谒罗什塔。

**民国至今**　1949 年后草堂寺先列陕西省重点文物保护单位、后**1956 年罗什塔列陕西省重点文物保护单位**；**2001 年草堂寺与罗什塔合并列入全国重点文物保护单位**。**1986 年中日佛教界共同发起修复草堂寺**——因罗什译经对日本天台宗、净土宗、日莲宗（皆依罗什所译《法华经》《阿弥陀经》）同为祖庭，日本佛教界亦多次至草堂寺朝礼。**1964 年户县简化字改自鄠县；2016 年撤户县置鄠邑区——恢复"鄠"字古名**——这是唐代地名经千余年后的一次正式回归。**今草堂寺仍在西安市鄠邑区圭峰山下**——罗什塔仍立、罗什殿仍祀。**唐人在 755 所朝礼的那座塔址，与今游人所至在地理上同一**——一千两百七十年，未易。
