# 商丘

## 一句话

**商汤始都 + 周封宋国都的双重古迹**——豫东平原睢水北岸；**商汤始都亳约 BC 1600（南亳说在此）、周武王伐纣后周公封微子启于宋（约 BC 1043）都商丘、周代宋国约 750 余年、至 BC 286 齐魏楚三国分宋而亡**；**755 时商丘故城在唐河南道宋州睢阳郡治宋城县西南郊原**——商代旧迹已没于平畴、宋国都垣残址间见、唐人凭吊即凭吊商汤始迹 + 宋国微子。

## 形貌

商丘故城在唐河南道**宋州睢阳郡治宋城县**——具体在**睢水北岸的豫东平原**上（今河南省商丘市睢阳区）。**地势**：黄淮海平原南缘、睢水 + 涣水 + 涡水之间的一片低平原——海拔约 50 米；地势平阔、无山、唯睢水自西北而东南、宋城县郭枕睢水北岸。

**755 时商丘有两层叠合**：

- **宋州睢阳郡治宋城县**——盛唐河南道东部大郡之一；下辖宋城、襄邑、宁陵、下邑、谷熟、虞城、楚丘、柘城 八县；州治宋城县是当时实存的州城——城郭完整、市坊井然。
- **商丘故城**——在宋城县治西南约数里的郊原。商代亳都旧址（**南亳说**——亳都位置历来诸说：南亳在商丘、北亳在山东曹县、西亳在偃师；唐人按古书通行取南亳说在此）+ 春秋宋国都垣——叠压在同一片台地之上。**唐时商丘故城已是郊原**——城垣残基隐没于宿麦与桑林之间、土埂可循而无人居。

**物候**：豫东平原、暖温带半湿润季风——夏湿冬寒；周原多桑、宿麦、粟黍；睢水夏涨秋落；唐时此地已是中原腹地、无山可凭，远望唯睢水一带与远处汴梁故汴渠。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

商丘在唐人的回望里——是**前朝多败中最早的一层**——比统万、比邺都、比咸阳，都更古。

**商汤的"亳"**——商人始祖契封于商（《诗经·商颂·玄鸟》"天命玄鸟、降而生商"）；**商汤（成汤）约 BC 1600 灭夏、始都亳**（《史记·殷本纪》）。**亳都位置历来有争论**——南亳（商丘）+ 北亳（山东曹县）+ 西亳（偃师）+ 郑亳（郑州商城）诸说并立；**唐人按古书通行认为亳在商丘**（《史记·宋微子世家》司马贞索隐 + 《元和郡县图志》卷七 皆取此说）；本 doc 取唐人这一认知——同时标注**夏商周断代工程**支持郑州商城为亳都的现代考古口径。**商汤以亳为都**——开启了商代五百余年；**商代五迁都**（汤都亳 → 仲丁迁嚣 → 河亶甲迁相 → 祖乙迁邢/耿 → 南庚迁奄 → 盘庚迁殷）——至盘庚约 BC 1300 迁殷（安阳），商代后半叶都于殷墟。**商丘只是商代早期都**——与晚商殷墟不是一地。

**周封微子启于宋**——**约 BC 1046 周武王伐纣**、灭商；**约 BC 1043 周公平管蔡之乱**后、周公**封微子启于宋**（商纣王庶兄、商之贤者）、都商丘——以**继殷祀**、是周人"兴灭国、继绝世" 政治叙事的核心一笔。**宋为公爵**——周代诸侯中爵位最尊的几国之一（与虢、虞、陈、杞同列）。**周代宋国约 750 余年**（约 BC 1043-BC 286）——历**宋襄公（图霸 BC 638 泓之战败）+ 宋景公（春秋后期）+ 宋康王（战国后期）**等。**BC 286 齐、魏、楚三国分宋**——宋亡。

**秦汉**——**始皇 BC 221 灭天下**、改宋为**砀郡**（治睢阳）；**西汉初**为**梁国**（高帝五年 / BC 202 立彭越为梁王、都睢阳；后彭越诛、文帝二年 / BC 178 封次子梁王、即梁孝王刘武 BC 168 即位）；**汉梁孝王**于宋国都旧址东北筑**梁园**（兔园）+ 招邹阳、枚乘、司马相如等——是汉初最有名的诸侯文学苑囿之一。**东汉为梁国睢阳**；**魏晋南北朝**间为豫州梁郡睢阳。**北周**改置**宋州**——以追宋国旧名（唐沿之）。

**唐人对商丘的认知**——**商丘 = 商汤始都 + 宋国都的双重古迹**——这是盛唐士人通晓的常识；**怀古情绪深**——**李白天宝三载（744）作《梁园吟》于宋州**——"我浮黄河去京阙、挂席欲进波连山……梁王宫阙今安在、枚马先归不相待"——是天宝前夕诗人凭吊宋州梁园（汉梁孝王苑囿）+ 同时回望商宋古都的代表作。**注**：李白《梁园吟》凭吊的是**汉梁园**（梁孝王 BC 168 苑囿、在宋国都旧址东北、不是宋国都本体）——但盛唐士人在宋州的怀古情绪是叠层的：商汤始迹 + 宋国微子 + 汉梁孝王梁园 + 唐宋州——四层叠在一处。

### 此刻

天宝十四载这一年的商丘故城——

**已没约 1040 年（BC 286 宋亡 → 755、无公元 0 年）+ 商汤始迹已约 2354 年（BC 1600 → 755）**——是 doc 库中**绝对年代最深**的一处古迹之一（与殷墟、临淄等同列）。

**物理形态**：商丘故城在宋城县西南郊原——商代亳都旧址 + 宋国都垣 叠压于同一片低岗台地。**亳都遗存**——唐时已无可辨之物（夯土城垣早已削平为田、商代陶片或可零星拾自麦垄间，但唐时无现代考古学）；**宋国都垣**——残基隐于宿麦与桑林、土埂可循、间有断瓦残砖——唐人立马郊原即可远见。

**唐宋州的现实**：**宋州睢阳郡治宋城县**——天宝元年（742）改宋州为睢阳郡（**注**：天宝元年改州为郡是唐廷统一举措、宋州随之改睢阳郡、治所不动）；宋城县城郭完整、是唐代豫东大邑之一；**汴渠（通济渠）**自西北过宋州境（不直经宋城县城、但贯宋州下辖宁陵 + 襄邑 等县）——是隋唐大运河东南段的核心水道；**宋州因汴渠之利**——是关东漕粮 + 江淮舟车至长安洛阳的转输要冲。

**唐人凭吊商丘**——多于宋州州治宋城县驻足、出城西南至商丘故城郊原。**所凭吊者**：商汤始迹 + 微子封宋 + 宋襄公图霸泓水 + 汉梁孝王梁园——四层叠合的怀古。

### 此刻意味

对天宝十四载的宋州人来说——商丘故城就是**自家州境的郊原古迹**：宋城县出西南门数里即至、土埂麦垄间隐约可辨垣基。对长安、洛阳那一代士人来说——商丘是**怀古情绪最深一层的物理 anchor**：在前朝多败的诸古都中、商丘最古——比咸阳早 1400 年、比邺都早 2200 年、比统万早 2000 年。

唐人尚不知道——再过两年（至德二载 / 757），**张巡、许远守睢阳**抗安庆绪 / 尹子奇——以六千兵守孤城十月、城破殉国；这是宋州在唐史上最壮烈的一笔，亦是天宝十四载的宋州人无从前知的命运。再过约 250 年（北宋初），宋州升**应天府**（宋太祖赵匡胤陈桥兵变前曾任宋州节度使、即位后以宋为国号、宋州遂为"龙兴之地"、景德三年 / 1006 升应天府、大中祥符七年 / 1014 再升南京——是北宋四京之一）。**这是商丘历史上的最高政治分量**——较之商汤始都、宋国都、犹有过之。

**今商丘古城**——明代弘治十六年（1503）于唐宋城县旧址重筑（即今商丘古城、内城呈圆形、外圆内方）——已非唐宋城县原物。**商丘故城（商汤亳都 + 宋国都）遗址**——在今商丘市睢阳区西南郊；1990s-2000s 河南省考古所多次发掘、确认**周代宋国城垣 + 汉代睢阳城**叠压；商代亳都层位深于地下数米、考古尚未完全定址；**亳都位置至今学界未有定论**。

## 子条目

### 商汤始都亳（南亳说）

**商汤（成汤）约 BC 1600 灭夏、始都亳**（《史记·殷本纪》）——是商代立国第一都。**亳都位置历来诸说并立**：

- **南亳**：商丘（今河南商丘市睢阳区）——唐人通行说、《史记·宋微子世家》司马贞索隐 + 《元和郡县图志》卷七 取此说；
- **北亳**：山东曹县（今山东菏泽市曹县）——汉以来另一说；
- **西亳**：偃师（今河南洛阳市偃师区）——《汉书·地理志》河南郡偃师条作"尸乡、殷汤所都"；
- **郑亳**：郑州商城（今河南郑州）——夏商周断代工程（1996-2000）现代考古主张。

**本 doc 取唐人通行认知（南亳在商丘）**——但标注亳都位置至今学界未有定论。**商汤以亳为都**——开启商代约 250 年早期都城史（至盘庚 BC 1300 迁殷）；**商代五迁都**（汤都亳 → 仲丁迁嚣 → 河亶甲迁相 → 祖乙迁邢/耿 → 南庚迁奄 → 盘庚迁殷）——商丘亳都仅是首座。

### 周公封微子启于宋（约 BC 1043）

**约 BC 1046 周武王伐纣**、灭商；**约 BC 1043 周公平管蔡之乱**——周公旦平定管叔、蔡叔、武庚（商纣王子）叛乱；**乱平后周公封微子启于宋**（商纣王庶兄、商之贤者、武王灭纣后已归周）——都商丘、以继殷祀（《史记·宋微子世家》）。

**注**：**封微子非武王伐纣后立即所封**——而是周公平管蔡之乱后才落定（武王伐纣后初封武庚于殷以续殷祀；武庚叛、被周公诛；周公另立微子启于宋）；这是关键史实、不可混。

**宋为公爵**——周代诸侯中爵位最尊的几国之一；**周代宋国约 750 余年**（约 BC 1043-BC 286）——历宋襄公图霸 + 宋景公 + 宋康王等；**BC 286 齐、魏、楚三国分宋**——宋亡。

### 宋襄公图霸（BC 638 泓水之战）

**春秋宋国第十六代君**——**宋襄公**（BC 650-637 在位）以"仁义"图霸；**BC 638 泓水之战**——宋楚战于泓水（今河南柘城西北、属唐宋州柘城县）；宋襄公**不鼓不成列**（不击半渡之楚军、待楚军过河列阵）——遂大败、襄公伤股、次年卒。**"宋襄之仁"**——成春秋"假仁义"典故、亦是唐人怀古宋国时常引者。**杜甫、李白、王维**等盛唐诗人有过宋州、过泓水，多有诗（杜甫《遣怀》"昔我游宋中、惟梁孝王都"+《遣怀》"宋公旧池馆、零落首阳阿"等）。

### 汉梁园（汉梁孝王苑囿、BC 168 起）

**汉文帝二年（BC 178）封次子刘武为梁王**——都睢阳（即宋国都旧址）；**BC 168 梁孝王刘武即位**——大兴宫室、于睢阳东北筑**梁园（兔园）**——招邹阳、枚乘、司马相如、公孙诡、羊胜等四方士人；是汉初最盛的诸侯文学苑囿。**注**：梁园不是宋国都本体——而是汉梁孝王在宋国都旧址东北另筑的苑囿；本 doc 与商丘故城（商亳 + 宋都）分开。**李白天宝三载（744）作《梁园吟》于宋州**——凭吊汉梁园；亦是 755 前盛唐士人怀宋州的代表作。

### 商丘故城叠压（亳都 + 宋国都 + 汉睢阳 + 唐宋城）

**商丘故城是一处叠压的古城遗址**——

- **底层**：商代亳都（南亳说、若成立、当 BC 1600 至 BC 约 1400 间使用）——考古尚未完全定址；
- **中层**：周代宋国都（约 BC 1043-BC 286、约 750 余年）——宋国都垣是商丘故城的主体；
- **上层**：汉睢阳（梁国都、BC 202-东汉、约 400 年）——西汉梁孝王扩筑；
- **再上层**：唐宋城县（唐宋州治、约 700-1006）——在汉睢阳基础上沿用；
- **明清**：明弘治十六年（1503）重筑商丘古城——内城呈圆形、外圆内方、是今商丘古城原物。

**唐人在 755 所立**：是商丘故城的**底三层**（商亳 + 宋都 + 汉睢阳）之上、当时的唐宋城县城郭则在故城北稍偏。**这种五层叠压**——是中国少数几处都城连续使用 2500 余年的物理遗存。

### 商丘 ≠ 殷墟

**商代 早期都于亳（商丘说之一）+ 晚期都于殷**——**盘庚约 BC 1300 迁殷**（《史记·殷本纪》）——殷在今河南安阳殷墟；**商代后半叶 273 年**都于殷墟、至 BC 1046 周武王灭商；**商代晚期甲骨文 + 青铜礼器**主要出自殷墟。**商丘只是商代早期都**（亳）——与晚商殷墟不是一地；**两地相距 300 余里**（商丘在豫东、殷墟在豫北）。**本 doc 不混商丘与殷墟**——是必须的史实区分。

### 唐宋州的现实（睢阳郡 + 汴渠转输）

**唐宋州在天宝十四载**——河南道东部大郡、治宋城县；**天宝元年（742）改宋州为睢阳郡**（唐廷统一改州为郡的举措）；下辖**宋城、襄邑、宁陵、下邑、谷熟、虞城、楚丘、柘城 八县**。**汴渠（通济渠）**自西北过宋州境、贯宁陵 + 襄邑 等县——是隋唐大运河东南段的核心水道；**宋州因汴渠之利**——是关东漕粮 + 江淮舟车至长安洛阳的转输要冲。**宋州的现实分量**——是盛唐豫东腹地的转输枢纽、不全是怀古之地。

## 历史长河（755 → 今）

**中晚唐**　**至德二载（757）张巡 + 许远守睢阳**——抗安史叛军安庆绪部尹子奇——以六千兵守孤城十月、城破殉国；是商丘在唐史上最壮烈的一笔。**张巡 + 许远 + 南霁云 等三十六将殉国**——《旧唐书》《新唐书》皆有传；**睢阳之战**保住了唐廷江淮漕道、是安史之乱中决定性一役。**注**：张巡死前与许远议——"睢阳一城、关江淮百城" —— 后世评价张巡守睢阳为"以一城守天下"。**自此商丘 / 睢阳是唐忠烈纪念之地**——韩愈《张中丞传后叙》（贞元十九年 / 803 作）—— 立张巡 + 许远 + 南霁云 三人之名为唐之忠烈正音。

**两宋**　**北宋初宋州升应天府**——宋太祖赵匡胤陈桥兵变前曾任宋州节度使、即位后以宋为国号、宋州遂为"龙兴之地"；**景德三年（1006）升应天府**；**大中祥符七年（1014）再升南京**——是北宋四京之一（东京汴梁、西京洛阳、南京应天、北京大名）。**靖康二年（1127）金兵破汴梁**——宋高宗赵构于**应天府（商丘）即位**、改元建炎、是为南宋开国——这是商丘在中国史上的最高政治分量。**靖康/建炎以后（1127-1130s）入金宋争夺、绍兴和议（1141）后属金**——商丘陷金、自此沦为金、元北方边州。

**金元明清**　金元间应天府降为归德府；**明洪武元年（1368）改归德州**、嘉靖二十四年（1545）复升归德府；**明弘治十六年（1503）重筑商丘古城**——内城呈圆形、外圆内方、城周 9 里 + 13 步——是今商丘古城原物。**清沿明制为归德府**——治商丘县。

**民国至今**　1913 年废府、改商丘县；1949 年后置商丘地区；2000 年升商丘市。**商丘古城（明 1503 城）**——是中国保存最完好的明代府城之一、1986 年列国家级历史文化名城、1996 年列全国重点文物保护单位。**商丘故城（商亳 + 宋国都遗址）**——在今商丘市睢阳区西南郊；1990s-2000s 河南省考古所多次发掘、确认周代宋国城垣 + 汉代睢阳城叠压；**商代亳都层位深于地下数米**、考古尚未完全定址；亳都位置至今学界未有定论。**唐人在 755 所凭吊的商丘故城郊原**——一千二百多年后仍在那片睢水北岸的低岗、垣基与陶片仍可于麦垄间间见。
