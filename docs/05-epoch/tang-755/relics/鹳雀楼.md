# 鹳雀楼

## 一句话

蒲州城西南角黄河东岸的三层瞭望楼——**北周时立、初为黄河前哨**；至盛唐而成北方第一名楼，**王之涣开元间题《登鹳雀楼》"欲穷千里目、更上一层楼"已传 20 余年**——755 时是黄河系名楼的顶峰。

## 形貌

鹳雀楼在蒲州城西南角——**黄河东岸**、楼下即蒲津渡口外缘的河滩。

蒲州城坐黄河大转弯东岸——黄河自龙门峡谷出、南下经河津、龙门、再东折至潼关——蒲州正在黄河"几"字形末段东转之前的最后一段。**鹳雀楼立蒲州城西南角的城墙之上**（一说立城外河岸高地、与城墙相接；唐时具体位置史源略有出入，但都在城西南、临河）——三层高楼、自城上凸出于河岸。

**楼凡三层**（《梦溪笔谈》明记"三层"）——下两层为木构台阁、最上一层挑檐高出。台基广而厚，立黄河东岸高崖。**楼名"鹳雀"**——因鹳雀（一种大型涉禽候鸟）岁岁聚栖于此楼檐下、楼前河滩——蒲州黄河滩涂正是这类水鸟的栖息地，鹳群盘旋成蒲州一景，楼以此得名。**古字"鹳雀"与"鹳鹊"通**——史志间或作"鹳鹊楼"，但唐宋史源多作"鹳雀"。

楼立处的形胜——**西望黄河**：自楼上俯视、即蒲津渡舟楫桥船；河水西去转南、过潼关、入华阴；远眺可至潼关方向、长安在西南千里之外。**东望中条**：蒲州城东即中条山——一道南北横亘的山脊、隔蒲州与河东腹地；自楼上望、中条横列于东。**北望涑水、南望条山入河**——是关中、河东、潼关三向交会的一处至高点。沈括《梦溪笔谈》评此楼"前瞻中条、下瞰大河"——是楼之形胜最精的一句。

蒲州本身在 755 时是**河东道东南门户、京畿外围最要冲的州之一**——蒲津渡是长安经河东入北方的最重要黄河渡口（开元十二年/724 张说《蒲津桥赞》载东西两岸合计 8 铁牛 + 配套铁人 + 铁山等，立浮桥），**鹳雀楼即在蒲津渡口边、是黄河防线的物理 anchor 之一**。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

鹳雀楼的来路可追到**北周**——比江南的滕王阁（唐永徽四年/653 立）、黄鹤楼（三国吴黄武二年/223 立）都不同：北方黄河系名楼的源头是**军事瞭望**而非游赏。

**北周（557-581）始建**——一说为北周大将**宇文护（?-572）**所建。**宇文护是北周宗室宰相**、自西魏末以来掌北周朝政十余年、终为周武帝所杀。北周与北齐间黄河中段是两国边界——蒲州在北周境、对岸（黄河西岸）则是关中通路；**鹳雀楼初立时的功能是黄河前哨瞭望**——立三层高楼于河岸、可观黄河上下游百里之内的舟船与河西动静。这是楼的最早形态——一座军用瞭望塔，而非游观名楼。

**隋至唐前期**——北齐 577 灭于北周、北周 581 禅隋、隋 589 灭陈一统天下——**鹳雀楼作为黄河前哨的军事功能渐褪**。隋代蒲州为**蒲州（开皇间立）/ 河东郡（大业三年/607 改）**；唐武德元年（618）复蒲州、其后历经数次州郡转换——本 doc 取 755 这一刻名号 "蒲州 / 河东郡"（**天宝元年/742 玄宗诏改州为郡、蒲州称河东郡；至德元载/756 蒲州升河中府**——一说乾元元年/758 升府，但通说作 756）。

**自唐初起、鹳雀楼渐成游观之楼**——蒲州为关中入河东第一州、过往士人、官吏、商贾凡至蒲州、登鹳雀楼凭眺成例。**盛唐前期已有诗人题楼**——但 755 时刻所传的鹳雀楼诗、主要是王之涣《登鹳雀楼》（开元间作；详子条目）。**畅当 / 畅诸《登鹳雀楼》"迥临飞鸟上"四句**——史载作者有"畅当"与"畅诸"两说、畅当多系**中唐**（大历-贞元间、760s-790s）人；本诗系**755 之后**作、本 doc 仅在历史长河"中晚唐"段引用、不入 755 此刻状态。

**王之涣（688-742）《登鹳雀楼》**——这是鹳雀楼在唐诗史上、亦是中国诗史上最重的一次定位。

王之涣是并州文水县人（今山西文水）——文水在汾水中游、距蒲州约二百里。**王之涣早岁任冀州衡水县主簿**（县官、九品下），后**遭诬罢官、流寓十数年**——在罢官期间漫游北方、写下大量边塞与游观诗。**《登鹳雀楼》系王之涣**作——具体作年史源未详，但**学界主流系于开元年间（713-741）王之涣流寓期间**——即至迟在开元末（741），其后王之涣天宝元年（742）卒。**自《登鹳雀楼》出、鹳雀楼便不再只是蒲州的一座楼**——四句二十字、写尽北方山河气象与登高所悟："**白日依山尽，黄河入海流；欲穷千里目，更上一层楼**"——前两句是登楼所见的实景（白日西沉、黄河东转南去），后两句是登楼所悟的人理（欲望远、必登高）——是唐五绝的顶峰、亦是中国诗"上进哲学"最深的一句。诗中"依山尽"之"山"史源未指实——本 doc 不坐实为中条（中条在蒲州东、落日方位实在西向）；唐人写诗、山河气象自有整体方向感、不必逐句对地理坐标。

王之涣传世仅六首——除《登鹳雀楼》外、另有**《凉州词》**"黄河远上白云间、一片孤城万仞山、羌笛何须怨杨柳、春风不度玉门关"——同为唐绝句巅峰之一。**王之涣一生未显达、卒时仅任文安县尉**（县官、九品下）——但《登鹳雀楼》《凉州词》两首足以使其名立于盛唐前列。

**注**：**王之涣 ≠ 王昌龄（698-756 前后）≠ 王翰（生卒不详、开元间人）**——三人皆盛唐前期边塞诗人、名字相近，但生平作品各异、史源不可混。

### 此刻状态

天宝十四载这一年的鹳雀楼——**已是黄河系第一名楼**。

**楼仍在蒲州城西南角立**——三层木构、台基坚固、北周以来近 200 年未圮。**蒲津渡每日舟楫不绝**——西去长安、东入河东、北上太原方向的车马商贾凡过蒲州、必经楼下；**登楼凭眺已成蒲州一例**——白昼登楼可见黄河蜿蜒、东望中条横列、西望日沉河西；夜登可闻河水涛声、远见两岸渔火。

**王之涣《登鹳雀楼》在 755 时已传 20 余年**——开元年间题、至 755 已逾 20 年。盛唐文人凡至蒲州、未有不诵此诗、未有不登此楼者。**沈括《梦溪笔谈》后追记"唐人留诗者甚多、惟李益（中唐）、王之涣、畅当（或作畅诸；中唐）三篇能状其景"**——这是北宋人回望整个唐代鹳雀楼诗咏的总评（**李益、畅当二诗皆 755 后作；755 此刻所传仅王之涣一首**）。

天宝十四载 这一刻的鹳雀楼——**地位等于江南的滕王阁、黄鹤楼**，但在唐人心目中、它的"北方黄河"地理身位是江南楼台不可替代的。**江南三楼（滕王阁、黄鹤楼、岳阳楼）**写江湖湖泽、楚天烟波；**鹳雀楼独写北方黄河、条山苍茫**——两套地理意象、两种诗境。

**安史前夕**（即 755 这一刻）——再过一年（至德元载/756）安史叛军南下、潼关失守、玄宗西奔；蒲州作为黄河防线一时为河东方面唐军重镇——**至德元载十二月、肃宗下诏改蒲州为河中府**（此为后事、本 doc 严守 755 时态）。**鹳雀楼此时仍立**——叛军未直接攻蒲州、楼无兵燹之毁。

### 此刻意味

对天宝十四载的蒲州人来说，鹳雀楼是**城西南角那座三层老楼**——日常凭眺、岁岁鹳归、孩童在楼下河滩上戏水；楼下蒲津桥铁牛系铁链浮桥、舟楫日过千百；楼上有蒲州州署或地方文人定期上去观风、记事、酬唱。对盛唐文人来说——鹳雀楼是**北方黄河系第一名楼**——凡过蒲州无不登；登则诵王之涣、诵畅当；不登则有缺。**江南三楼写江湖、鹳雀楼独写黄河中条**——这是盛唐文化地理上两套不可替代的诗境之一。对朝廷来说——蒲州是关中东出第一州、鹳雀楼是这一州的物理 anchor，是黄河防线的瞭望塔之延续——北周立时的军事功能，至盛唐已转为文化与文学的 anchor，但军事意义并未完全消失。

唐人尚不知道——再过 230 多年（北宋淳化三年/992）鹳雀楼会**毁于火**；再过千年（1997-2002）今山西永济会**新建一座现代鹳雀楼**——但与唐楼非物理连续，是 20 世纪末旅游业的复建。**唐人所登的那座鹳雀楼自宋初毁后再未重立**——王之涣"更上一层楼"的那一层楼，自宋亡矣。**但二十个字仍在**——这是楼的另一种 anchor、比木构台阁更耐久的 anchor；千载之后无论楼立或毁、人皆能默诵那二十字、知此地曾有楼、楼前曾有河、河西曾有日落。

## 子条目

### 王之涣《登鹳雀楼》

**白日依山尽，黄河入海流。欲穷千里目，更上一层楼**。

**作者**：王之涣（688-742）、并州文水人。**作年**：**开元年间（713-741）**——具体年份史源未详；学界主流系于开元中后期王之涣罢冀州衡水主簿后流寓期间。**755 时已传 20 余年**——盛唐文人凡至蒲州必诵。

**诗的实景对应**：第一句"白日依山尽"——日西沉、群山尽染余晖；史源未指实何山——本 doc 不坐实（中条在蒲州东、落日方位实在西向；自鹳雀楼上西望、夕阳沉于黄河西岸群山方向、不沉中条）。王之涣写的是登楼所见日落山尽的整体气象、不是某一具体山头。第二句"黄河入海流"——蒲州西望黄河、河水东转南下；但"入海"是**远眺的诗意夸张**——蒲州距黄河出海口直线尚有千余里、河流并不直接"入海"于眼前——王之涣写的是黄河整体的方向感、不是脚下河段的物理 endpoint。第三句"欲穷千里目"——欲见远方更远处。第四句"更上一层楼"——鹳雀楼凡三层、自第二层登至第三层、即"更上一层"。这四句是**实景 + 推理 + 哲理 一气贯通**的二十字。

**注**：**"白日"在中古汉语中即"白昼之日"、不特指正午之日**——王之涣写的是夕阳沉山之景，"白日依山尽"中的"白日"对应的是傍晚渐西的太阳。**这一点近世偶有误读**——本 doc 取唐宋通解。

### 畅当（或作畅诸）《登鹳雀楼》——中唐作、755 之后

**迥临飞鸟上，高出世尘间。天势围平野，河流入断山**。

**作者**：畅当——史载作者有"畅当"与"畅诸"两说（《全唐诗》两收）。畅当多系**中唐**（大历-贞元间、760s-790s）人、与王之涣不同时；史源多记其为河东人、与鹳雀楼地理近。**作年**：**中唐**——具体年份史源未详；**755 此刻此诗尚未作**——本 doc 仅在历史长河"中晚唐"段引用、不入 755 此刻状态。

畅当此诗在中唐之后渐与王之涣《登鹳雀楼》并传——沈括《梦溪笔谈》列"三篇能状其景"中即王之涣、畅当、李益（李益亦中唐人、《登鹳雀楼》同在 755 之后）。**畅当诗写楼之"高"**——"迥临飞鸟上、高出世尘间"是登楼第一感；"天势围平野、河流入断山"则是登楼所见的山河格局——条山如断壁、黄河自断山入。**风骨与王之涣异**——王之涣写"理"、畅当写"势"——是唐代鹳雀楼诗的两个 voice，但**两诗分属盛唐 / 中唐两代、非同代并题**。

### 北周宇文护始建说

**鹳雀楼始建于北周（557-581）**——一说为北周大将**宇文护**（?-572）所建。宇文护是北周宗室宰相、自西魏末以来掌北周朝政十余年、终为周武帝所杀。**宇文护建楼的具体年代史源不详**——南北朝史志记此楼"北周时立"、未系于具体某年；"宇文护建"说在隋唐间渐成传统说法、但严格 verified 程度有限。**本 doc 取"北周（557-581）时立"为 verified、宇文护具名为 likely**。

楼立的最初功能是**黄河前哨瞭望**——北周与北齐间黄河中段是两国边界、蒲州在北周境、楼立东岸可观黄河上下游百里之内的舟船与河西动静。这是楼最早形态——军用瞭望塔。

### 蒲州 / 河东郡

**鹳雀楼所在州**——755 时正官名为**蒲州 / 河东郡**（天宝元年/742 玄宗诏改州为郡、蒲州称河东郡；**至德元载/756 蒲州升河中府**——一说乾元元年/758 升府，但通说作 756）。**755 时是蒲州 / 河东郡、不是河中府**——这是本 doc 严格守的 755 时态。

蒲州治河东县（今山西永济市西约 10 公里黄河东岸）——蒲津渡是长安经河东入北方的最重要黄河渡口。**开元十二年（724）张说《蒲津桥赞》**载东西两岸合计 **8 铁牛 + 配套铁人 + 铁山**等，铸于此、牛系铁链以维浮桥。**1989 年蒲州东岸出土 4 铁牛 + 4 铁人 + 2 铁山**（东岸部分；西岸尚未发掘）——是开元铁牛的物理存证。**鹳雀楼即在蒲津渡口边的城西南角**——铁牛在楼下、舟船在楼下、是蒲州之河的物理坐标。

### 蒲津渡（楼下河流）

**鹳雀楼直接俯瞰的黄河渡口**——是关中东出第一渡。**开元十二年（724）张说《蒲津桥赞》**载东西两岸合计 **8 铁牛 + 配套铁人 + 铁山**等、铁链浮桥维河两岸；自此蒲津渡是**唐代黄河最重要的固定渡口**——非临时浮桥、而是常设。**1989 年蒲州东岸出土 4 铁牛 + 4 铁人 + 2 铁山**（东岸部分；西岸尚未发掘）——是开元铁牛千载之后的物理存证。**鹳雀楼是这一渡口的瞭望塔**——楼上观渡、渡口观楼、二者构成蒲州城西南最完整的一处地理景观。

### 中条山（楼东之山）

**蒲州东即中条山**——南北横亘黄河北岸的一道山脊、隔蒲州与河东腹地（运城盆地）。沈括《梦溪笔谈》记鹳雀楼"前瞻中条、下瞰大河"——是楼形胜最精的一句：自楼上东望、中条横列；俯视、黄河在脚下。

**注**：王之涣《登鹳雀楼》"白日依山尽"之"山"史源未指实——本 doc 不坐实为中条山。中条在蒲州东、夕阳方位实在西向（黄河西岸群山方向）；自鹳雀楼上望、日落与中条不在同一方向。唐人写诗、山河气象自有整体方向感——"依山尽"是日西沉、群山染余晖的整体气象、不是某一具体山头。中条作为"楼东之山"是 verified 的地理；作为"白日依山尽之山"则不是——本 doc 不混二者。

## 历史长河（755 → 今）

**中晚唐**　**至德元载（756）十二月**——肃宗下诏改蒲州为**河中府**——自此名号变；但鹳雀楼物理仍立。**乾元三年（760）河中府名定**。**安史乱中（755-763）蒲州为唐军黄河防线重镇**——鹳雀楼或受过兵燹的轻度冲击、但具体史源未详；楼似未大毁。**中唐至晚唐**——鹳雀楼诗咏渐密：**畅当（或作畅诸；大历-贞元间、760s-790s）《登鹳雀楼》**"迥临飞鸟上、高出世尘间……"——其作年系**中唐**、与王之涣盛唐《登鹳雀楼》前后相隔半世纪。**李益（748-829）《同崔邠登鹳雀楼》**"鹳雀楼西百尺樯、汀州云树共茫茫……"——李益亦中唐边塞诗人、其鹳雀楼诗与王之涣、畅当并称唐三绝（沈括《梦溪笔谈》语）。**司空图、薛能、罗隐**等晚唐文人亦多有鹳雀楼咏；蒲州作为黄河边重镇、过往文人凡至无不登。**晚唐**——鹳雀楼仍是凡至蒲州必登的名楼、但唐末战乱（黄巢之乱 880s）后蒲州残破、鹳雀楼物理状态渐衰；五代间无重大重修记录。

**两宋**　**北宋初**——鹳雀楼仍立、是河中府（北宋仍称河中府）一名胜。**北宋淳化三年（992）—— 鹳雀楼毁于火**（沈括《梦溪笔谈》追述、具体起火条目史源稍后；但"宋初毁、其后久湮"为北宋人主流口径）。**自此鹳雀楼物理终结**——北宋间未重建；南宋 (1127 后) 河中府已属金、更不复重建。**北宋沈括《梦溪笔谈》（约 1086-1093）**记鹳雀楼形制"三层、前瞻中条、下瞰大河"——沈括（1031-1095）出生时楼已毁近四十年、不能亲见原楼，其记必据旧闻或楼基残迹，是**鹳雀楼物理形貌的最后一份近源记录**。

**金元**　金代河中府在金境——金人未重建鹳雀楼；元代亦未建。**整个金元四百年间鹳雀楼仅存于诗中**——王之涣、畅当、李益三诗代为楼之 anchor。元代地方志 / 文献仅记"鹳雀楼故址在蒲州城西南"——具体物理已无可考。**元代《唐才子传》（辛文房）收王之涣传**——其中《登鹳雀楼》仍是王之涣条目下最重的代表作。

**注**：**北宋淳化三年（992）毁于火**之说——主要来源是后世笔记（含沈括《梦溪笔谈》系统的相关条目）追述、严格 verified 程度较《资治通鉴》系编年史略弱；本 doc 取主流口径"宋初毁于火、其后久湮"——具体年份 992 为参考、不绝对。但**鹳雀楼物理在北宋间终结**这一点是确定的——北宋后期至南宋间所有文献记此楼皆作"故址"、再无目击记录。

**明清**　明清地方志（《蒲州志》《永济县志》等）仍记鹳雀楼故址、但物理不存。**明清间偶有人议重建、皆未果**——一是河水冲刷、原址在黄河滩涂；二是明清间蒲州地位远不及唐宋、官方与民间皆无重建的财力意愿。**清代蒲州属山西省蒲州府**——府治在永济县；鹳雀楼仅作"古迹"记于府志、凭吊者主要诵王之涣诗。

**民国至今**　**1949 年后蒲州 / 永济县归山西省**；鹳雀楼故址因黄河改道、明清以来河水冲刷、严格物理位置已不可定。**1997-2002 年**——山西省永济市启动**鹳雀楼复建工程**——在永济市西郊（距古蒲州城遗址不远）按盛唐风格新建一座九层（一说四层带平座、外观九重）大型楼阁——**这座新鹳雀楼是 20 世纪末旅游业的复建、与唐代三层鹳雀楼非物理连续**。**今游人来"鹳雀楼"，所见的是 2002 年完工的现代楼**——楼内陈列王之涣《登鹳雀楼》、有楼层介绍、是山西重点旅游景点；**但与王之涣登的那座楼无物理连续**——是文化 anchor 在物理 anchor 终结千年后的一次现代续接。**王之涣《登鹳雀楼》的二十字仍在**——这是楼真正的 anchor，比物理楼活得更久。
