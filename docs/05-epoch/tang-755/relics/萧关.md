# 萧关

## 一句话

**关中西北门 + 长安通灵凉河西的北道关塞**——原州平凉郡平高县境六盘山东麓的山口关；**王维"萧关逢候骑、都护在燕然" 的那一关**（开元二十五年/737 奉使凉州过萧关作）；**755 时仍立、是关中四关之北关**——再过 7 年（763）吐蕃东扩、原州陷蕃、萧关随之失唐控。

## 形貌

萧关在唐**关内道原州平凉郡境**——具体在**平高县境六盘山东麓**（今宁夏回族自治区固原市西南三关口一带，距平高县治约 30-50 唐里）。**地势**：**六盘山东麓的山间隘口**——是黄土高原向陇山过渡的天然孔道；**北面**接陇山东余脉（向北延伸入清水河谷）；**南面**接六盘山主脊（向南连陇山）；**关口在山口狭窄处**——海拔约 1700-1900 米。

**萧关之险**：**六盘山是陇山主峰段**——盛唐人通称"陇山"，其主脊海拔逾 2900 米；**萧关不在最高处**——而在山东麓的山口隘道上、是天然商旅 / 兵马孔道。**关口两侧山势峻急**——一关守住关中盆地北向通灵州 / 凉州 / 河西的北道。**这是"关中四关"中最北的一关**（东潼关、南武关、西散关、北萧关）。

**萧关南北通道**：**自长安出发** + 西北行 + 过邠州（新平郡）+ 泾州（保定郡）+ 入原州境 + 过平凉县 + 至平高县 + **过萧关** + 北行接灵州 / 凉州——这条路盛唐时是长安通河西陇右的两条主路之一（另一条为西路过大震关入秦州）。**大震关偏南偏西、萧关偏北**——盛唐两关并立、分担长安西北向客流。

**关城形制**：盛唐萧关具体形制 + 周长 + 门数史载零散——本 doc 不预设细节（今所见关楼 + 关墙多为后世重建）。**关城外**：南北通道穿关而过；**关楼立六盘山东麓山口** —— 可俯瞰东向清水河谷一片；**关塞内外多有戍兵 + 驿马 + 商旅** —— 是盛唐西北方向的主要关塞之一。

**萧关具体关址**：**历来争论**——一说六盘山东麓三关口（今固原西南），一说瓦亭关一带（今固原南）；亦或唐时萧关数次小迁。**本 doc 严守"盛唐萧关通认在平高县境六盘山东麓" 宽口径**——不预设单点关楼。

**物候**：六盘山一线高寒——夏温冬寒、雪较多；山中林木相对较盛、关外（北向清水河谷）渐入半干旱黄土草原。**关城内外** 多有戍兵 + 关吏 + 驿马 + 商旅。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

萧关在唐人的回望里——是**关中西北门 + 千年北塞 + 关中四关之北关**。

**萧关的历史**：

- **战国秦昭襄王（前 306-前 251）** —— 北上击义渠戎、**自萧关入塞** —— 是萧关作为关塞最早的文献记录之一；自此萧关进入中国北塞史。
- **西汉文帝十四年（前 166）** —— **匈奴老上单于率十四万骑入朝那、萧关 + 杀北地都尉孙卬 + 烧回中宫**（《史记·匈奴列传》《汉书·文帝纪》原文）—— 是萧关在中国军事史上最早最有名的一战；**文帝亲征至甘泉宫**——萧关由此成为汉廷北防的核心关塞。**回中宫**：秦汉时北地郡境的离宫、与萧关相邻；今址在今陕西陇县一带。
- **东汉至北朝**：萧关续作关塞、但因东汉以来匈奴渐衰 + 北朝政治多变 + 关塞位置略有迁徙。
- **隋唐**：萧关延续作为关中西北门 + 北塞关。**唐承前代之制**——置原州 + 平凉郡、辖萧关。

**唐萧关的政治军事地位**：**唐承隋制 + 置原州**；**萧关属原州 + 关内道辖境**（具体节度归属：盛唐萧关在原州境、原州属关内道；非朔方节度直辖、但与朔方节度（治灵州）军事呼应）。**自唐初至盛唐 138 年（618-755）萧关无大战事**——是太平时代；但**萧关常驻兵约数百**——是日常守关 + 验商旅 + 收关税。

**萧关与长安、河西、陇右的关系**：

- **长安经萧关北行 + 至灵州（朔方节度治）**：约 1000 唐里；
- **长安经萧关西北行 + 至凉州（河西节度治）**：约 1800 唐里；
- **萧关与大震关分担长安西北向客流**：大震关偏南偏西（属陇州 + 秦州一线）、萧关偏北（属原州 + 灵州一线）。

**这是盛唐长安通灵凉河西的北道关塞**——河西陇右兵东调救京、使节南下北上、商旅过关——多过萧关或大震关。

**萧关在文学上**：**王维《使至塞上》"萧关逢候骑、都护在燕然"** —— **开元二十五年（737）王维奉使凉州过萧关时作**（《全唐诗》卷一二六）—— 是中国文学史上萧关诗的最高峰：

> 单车欲问边，属国过居延。
> 征蓬出汉塞，归雁入胡天。
> 大漠孤烟直，长河落日圆。
> **萧关逢候骑，都护在燕然**。

**王昌龄《塞下曲》**"饮马渡秋水、水寒风似刀；平沙日未没、黯黯见临洮" —— 涉陇山 + 河西一带边塞。**盛唐边塞诗中"萧关"二字时有出现**——但与"凉州" "玉门关" "阳关" "雁门" 等并立、是盛唐边塞文学的高频符号之一。

### 此刻

天宝十四载这一年的萧关——**关塞严整、驻兵数百、扼六盘山东麓山口**。

**关城状态**：盛唐 755 时萧关主体仍立、有戍兵 + 关吏 + 驿馆；具体盛唐形制史载零散。**关城外**：南北通道穿关而过；**萧关北接灵州 / 凉州的北道** —— 是河西陇右兵东调救京的核心通道之一。

**驻军**：萧关属原州 + 关内道；按盛唐边军规制驻兵约数百。**与朔方节度（治灵州）军政呼应**——朔方节度负责漠南防御 + 与回纥关系；萧关作为关中通灵州的北门、是朔方军南下救京 / 长安使节北上的物理 anchor。

**755 时萧关上的政治氛围**：**萧关是平时关塞**——关税收 + 商旅过关 + 戍兵巡逻 + 使节验牒；**与回纥的关系**：744 后回纥代东突厥据漠南；**755 时唐回纥关系仍属和好**（回纥与唐玄宗朝多有外交）；**萧关是双方通行的物理 anchor 之一**——回纥使节南下长安 + 唐使节北上回纥的部分客流过此。

**安史前夕**（即 755 这一刻）——**安禄山起兵于范阳（在今北京）**——**自范阳南下取洛阳**——萧关方向未受直接兵燹；但**756 后河西陇右兵东调救京**——朔方节度郭子仪 + 河西节度的部分兵力过萧关南下入关中——萧关由此进入安史战争的物理网络。

### 此刻意味

对天宝十四载的原州 / 平高县人来说——萧关是**自家州境的北门**：日常戍兵 + 商旅过关 + 回纥使节南下偶有。对长安那一代士人来说——萧关是**关中西北门 + 王维"萧关逢候骑" 的那一关**：**盛唐文学最深一句萧关诗**——是开元盛世士人想象边塞的核心物理 anchor。**盛唐人嘴里的"萧关"**——既是地理实体、也是文学符号；与"凉州" "玉门关" "阳关" 并立。

唐人尚不知道——再过 7 年（763 广德元年），**吐蕃东扩 + 大举攻唐 + 陷原州 + 平凉 + 萧关一带**；**763 十月吐蕃陷长安**——之前已经先取关中西北诸州、萧关于 762-763 间随原州陷蕃而失唐控；自此**萧关一带成唐蕃边境**、唐廷再无力收复。盛唐人 755 年所见的那座萧关——**七年后即将成"前线",再后**即将成"敌境"。

## 子条目

### 萧关地名与"萧"之考

**"萧关"之名**——史源略异：

- 一说**"萧" = 萧条之意**——关外北地荒漠、关塞萧索、故名"萧关"；
- 一说**"萧" 是地名**——古萧水（清水河上游一支流）流经此地、关因水得名；
- 一说**"萧" = 萧条草木的草名**——关山萧草丛生。

**本 doc 不预设具体词源**——但"萧关" 作为关塞名、自战国秦汉以来已经定型；盛唐人嘴里的"萧关"即"关中西北门"。

### 汉文帝十四年匈奴入萧关（前 166）

**汉文帝十四年（前 166）冬** —— **匈奴老上单于率十四万骑入朝那、萧关 + 杀北地都尉孙卬 + 烧回中宫**（《史记·匈奴列传》原文："孝文皇帝十四年，匈奴单于十四万骑入朝那、萧关，杀北地都尉卬，虏人民畜产甚多……"《汉书·文帝纪》系于十四年；按二者年份一致、取前 166）。

**汉文帝亲征**——发车骑材官十万 + 至甘泉宫（关中咸阳北 + 萧关之南）—— 与匈奴对峙月余；**匈奴退回漠北**。**这是中国军事史上萧关最早最有名的一战**——也是萧关作为"关中北防咽喉" 在文献中第一次大规模出现。**注**：**汉文帝后元六年（前 158）匈奴亦有一次南下** —— 但规模较小、非本条所记十四年之役；二者不可混。

**回中宫**：秦汉时北地郡境的离宫；今址约在今陕西陇县至甘肃华亭一带（萧关之南）；**匈奴烧回中宫** —— 是汉初匈奴南侵深入关中边境的标志性事件。**唐人 755 时凡读《史记》《汉书》文帝纪 + 匈奴传者** —— 必知萧关 + 老上单于 + 烧回中宫的故事；**这是萧关在盛唐人地理认知中的 historical baseline**。

### 王维《使至塞上》"萧关逢候骑、都护在燕然"（737）

**王维（约 701-761）**——盛唐山水边塞诗大家。**开元二十五年（737）春**——王维奉使凉州（河西节度治）+ 慰劳河西边将；**自长安西北行经原州萧关 + 凉州**——途中作《使至塞上》：

> 单车欲问边，属国过居延。
> 征蓬出汉塞，归雁入胡天。
> 大漠孤烟直，长河落日圆。
> **萧关逢候骑，都护在燕然**。

**"萧关逢候骑、都护在燕然"** —— "候骑" 即斥候骑兵；**王维在萧关一带遇到河西军的斥候 + 告知都护现在燕然（汉代燕然山、东汉窦宪破匈奴勒铭处、王维这里借用为河西军前线的代称）**。**这是中国文学史上萧关诗的最高峰**——盛唐边塞诗中"萧关"二字最有名的一次出现。

**注**：**王维《使至塞上》作于 737 + 是 755 前作**——本 doc 可入"唐人眼里的来路" 段引；是 755 时长安士人读《王右丞集》必知的萧关诗。**"大漠孤烟直、长河落日圆"** —— 是盛唐边塞诗写戈壁 + 黄河的双绝之句；本句虽未直接提萧关、但全诗以"萧关逢候骑" 收束、萧关是这首诗的物理终点。

### 萧关道（萧关南北通道）

**萧关南北通道**——盛唐唐人通称"萧关道"：

- **南端**：自长安出发 + 西北行 + 过邠州（新平郡）+ 泾州（保定郡）+ 入原州境 + 过平凉县 + 至平高县 + 过萧关；
- **北端**：自萧关北出 + 入清水河谷 + 北行接灵州（朔方节度治）/ 凉州（河西节度治）。

**萧关道全程**：自长安至灵州约 1000 唐里、自长安至凉州约 1800 唐里。**与大震关道（长安西行经陇州 + 大震关 + 入秦州 + 转兰州 / 凉州）并立** —— 萧关偏北、大震偏南、两关分担长安西北向客流。

**萧关道盛唐时的客流**：

- **使节**：长安通灵州（朔方节度）、长安通凉州（河西节度）+ 安西大都护府的部分使节过萧关道；
- **商旅**：粟特商、回纥商、汉商等过萧关道；
- **军马**：河西陇右兵东调救京、朔方军南下入关 + 多过萧关道（756 起此线骤忙）。

### 大震关与萧关的分担

**大震关**——盛唐唐人称为"陇关"——在陇州（汧阳郡）境陇山东麓（今陕西陇县西、宝鸡之西）；**大震关与萧关并立**——盛唐长安西北向有南北两线：

- **南线（大震关道）**：长安西行经岐州 + 陇州 + **大震关** + 入秦州（天水郡）+ 转兰州（金城郡）/ 凉州；偏南偏西；
- **北线（萧关道）**：长安西北行经邠州 + 泾州 + 原州 + **萧关** + 入灵州 / 凉州；偏北。

**两关之分担**：**自长安通秦州 / 兰州方向** —— 多走大震关；**自长安通灵州 / 河西 / 漠南方向** —— 多走萧关；**自长安通凉州** —— 两关皆可、视具体行程定。**756 后河西陇右兵东调救京** —— 朔方军（治灵州）+ 河西军 + 部分安西军过萧关或大震关入关中救京。

### 关中四关（潼关 + 武关 + 散关 + 萧关）

**关中盆地的天然防御体系** —— 四关合围：

- **东关：潼关** —— 黄河南岸 + 秦岭北麓夹峙；扼东向洛阳之冲；
- **南关：武关** —— 秦岭南麓商山中（商州境）；扼南向荆襄之冲；
- **西关：散关** —— 秦岭主脊北麓凤州境；扼西南向汉中蜀地之冲；
- **北关：萧关** —— 六盘山一线（原州境）；扼北向灵凉河西之冲。

**四关在中国地理上是最完整的"关中盆地防御体系"** —— 盛唐 755 时四关皆立、守军按例驻屯。**注**：**关中四关之称是后世地理归纳** —— 盛唐 755 时唐人是否以此固定四关称呼史源略异；本 doc 取通称作地理 anchor、不预设唐人固定认知。**萧关是四关中地位最特殊的一关** —— 因 763 后吐蕃陷原州、萧关随之失唐控、是四关中最早 "离开唐廷" 的一关。

### 762-763 间萧关之失（吐蕃东扩）

**安史 + 后续年间** —— 安史战争 + 河西陇右兵东调救京 + 西陲守备日空 + **吐蕃东扩**：

- **756 起**：陇右、河西守备日空、吐蕃逐步东扩；
- **762-763 间**：吐蕃陷原州 + 平凉 + 萧关一带；
- **广德元年（763）十月**：**吐蕃 20 万骑入长安**（《资治通鉴》卷二二三）—— 在此之前已经先取关中西北诸州（原州 + 渭州 + 兰州 等）+ 萧关一带；唐廷再无力收复。

**自此萧关成唐蕃边境**——再后**唐廷边境进一步收缩至邠州 + 泾州一线**；**萧关一带百余年间为吐蕃辖境**；**直至宣宗大中年间（847-859）唐廷收复部分陇右诸州**——但萧关一带的关塞 / 道路体系已与盛唐迥异。**盛唐 755 时的那座萧关** —— 8 年后即失唐控、自此 "脱离盛唐物理网络"；唐人 755 时尚不知。

## 历史长河（755 → 今）

**中晚唐**　**762-763 间萧关一带渐入吐蕃辖境**（具体年月史源略异）；**763 后萧关 100 余年为吐蕃辖境**；**大中年间（847-859）唐廷部分收复陇右诸州**——但萧关一带的关塞体系已经断绝盛唐脉络；**晚唐萧关已不复盛唐之 importance**。**杜甫 + 中唐边塞诗** —— 对萧关的诗咏 较盛唐为衰；**萧关在中唐后渐成"已失之关" 的怀古地**。

**唐末 + 五代**　**五代间萧关一带多次易主** —— 党项、回纥后裔、汉族割据政权等反复争夺；**萧关作为关塞的物理形制** 在五代间已残破。

**北宋 + 西夏**　**北宋初萧关一带为宋夏边境** —— 党项李继迁 + 元昊建立西夏（1038）后，萧关一带长期为宋夏拉锯前线；**北宋仁宗朝在原州 + 镇戎军（今固原）+ 萧关道一线大筑堡寨**——萧关附近的瓦亭关、开远堡、绥宁堡 等皆此时所立。**萧关在北宋成宋夏关塞核心**——但已非唐时之萧关形制。

**金元明清**　**金代萧关属金境**；**元代设开成路 + 镇戎州**——萧关地位续降。**明景泰三年（1452）立固原州**——是今"固原"地名之始；**明嘉靖年间在固原一带筑长城（明长城西段）**——萧关 / 三关口一带成明长城关塞之一；**清代萧关续作关塞、但已退离主要边防**。

**民国至今**　**1949 年后宁夏回族自治区 + 固原市设立**；**萧关 / 三关口古道遗址列宁夏回族自治区重点文物保护单位**。**1980s-2010s 多次考古勘察 + 部分修缮**——确认萧关一带的古关墙、古道路线、汉唐烽燧遗迹；**今所见萧关遗址**：**主体为后世重建** + 部分汉唐遗迹散布六盘山东麓——非盛唐 755 时关楼形制。**唐人在 755 所见的那座萧关** —— 千二百多年后**唐时形制已不复**；但**关塞位置 + 六盘山 + 萧关道**——一千二百多年未易。
