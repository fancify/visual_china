# 鹿门别业

## 一句话

襄州襄阳城东南二十唐里鹿门山中的诗人故居——孟浩然 (689-740) 晚年隐居十数年、于此作《夜归鹿门歌》——**天宝十四载这一刻、距浩然食鲜疾发卒已十五年**；鹿门别业荒废，但盛唐人凭吊浩然故宅、犹是从襄阳出南门、循汉江东岸往鹿门山的一日游。

## 形貌

鹿门别业在襄州襄阳县东南二十唐里——**汉江东岸、鹿门山西麓**，是一片山溪边的台地。

地势：鹿门山主峰海拔不高（约 200 米余）、孤峙汉江东岸、与对岸襄阳城遥望。鹿门山之得名——**据《水经注·沔水》所载，本于东汉光武年间襄阳侯习郁立神祠于苏岭山，祠前刻二石鹿夹道，谓之鹿门庙，山因庙得名**——非由山形似双鹿对峙；自汉末庞德公在此采药不返，遂为襄阳一带的隐士谱系 anchor。**鹿门别业故址在山的西麓**——一处面汉江、背鹿门主峰的台地；台前一道山溪自鹿门山口出、西注汉江。盛唐时台上有草庐三五间、园圃数亩、井一口——这是孟浩然晚年隐居十数年的居处。

**孟浩然别业 与 鹿门寺 是鹿门山中两处主物事**——鹿门寺在山主峰南侧、北周建德间（572-578）由鹿门庙转为佛寺（祀庞德公旧庙改作佛寺；具体年份史源不一，《襄阳府志》明清纂记建德元年/572，是较通行的一说）；孟浩然别业则在山西麓近江一处。两处直线相距约一里、由山间小径相连。**唐人入鹿门凡两条路**：自襄阳南门出、东南循汉江至渔梁渡争渡过江、再归鹿门山（孟浩然《夜归鹿门歌》"渔梁渡头争渡喧、人随沙岸向江村"即此景——渔梁渡在汉江中鱼梁洲西端，为襄阳东南郊主渡）；或循汉江东岸陆路南行二十里——前者是诗中行旅、后者是日常往返。

物候——汉江中游东岸、气候湿润；鹿门山中松柏成林、山溪四时有声。襄阳人凡入夏即往鹿门避暑、秋末凭吊浩然——是襄阳城外一处可日游可住宿的山中清地。

## 大唐天宝十四载（755年）

### 唐人眼里的来路

鹿门别业在唐人的回望里是**盛唐山水田园派代表诗人孟浩然的隐居故宅**——是孟浩然《夜归鹿门歌》的实物来源、亦是盛唐"诗人 + 隐居"谱系最近的一处 anchor。

**鹿门山的隐居谱系自东汉末始**——**庞德公**（东汉末襄阳人、刘表客；与司马徽、诸葛亮、庞统等结交）"携妻子登鹿门山，因采药不返"（习凿齿《襄阳耆旧记》）——这是中国隐士史上"举家入山不返"叙事的早期原型。庞德公在鹿门山中传授门生、品鉴人物（曾誉诸葛亮为"卧龙"、庞统为"凤雏"——庞统是庞德公的从子）——**鹿门山自此即"庞公栖隐处"**，与襄阳城西的隆中（诸葛亮躬耕地）一东南一西、是**襄阳二大隐居 anchor**。

**孟浩然 (689-740)**——襄州襄阳人、盛唐山水田园派代表（与王维并称"王孟"）。早年游学长安、洛阳——**开元十六年（728）**应进士试不第、归襄阳；自此**退隐故乡、于鹿门山中筑别业、潜心诗学**。其间数次出游吴越、岭南；与王维、李白、王昌龄、张九龄等盛唐诗人皆有交往——王维在长安为孟浩然画像、李白《赠孟浩然》"吾爱孟夫子、风流天下闻"是盛唐"诗人致敬诗人"的名章（杜甫 712 生、浩然 740 卒——二人是否曾面晤史源不能定，杜甫对浩然的追忆诗集中作于上元至大历间 760s 入蜀以后）。

**孟浩然《夜归鹿门歌》**——是孟浩然鹿门隐居的代表作；末四句"鹿门月照开烟树、忽到庞公栖隐处。岩扉松径长寂寥、惟有幽人自来去"——这是 755 时已传诵二十余年的盛唐山水诗名章。诗中"庞公栖隐处" 一句——把汉末庞德公与盛唐孟浩然的鹿门隐居谱系直接连起来；这是孟浩然自我定位的诗笔。

**开元二十八年（740）**——王昌龄自岭南北归、过襄阳访孟浩然；浩然时疾疹（背疮）将愈、相见欢甚、浪情宴谑、**食鲜（一说鳊鱼、一说河豚——史源异文，《旧唐书》仅言"食鲜疾动"）疾发**——卒于襄阳冶城南园；享年五十二。**王维 738-740 任殿中侍御史 知南选**——开元二十八年（740）奉使南选过襄阳、闻浩然新故、作《哭孟浩然》："故人不可见、汉水日东流。借问襄阳老、江山空蔡洲。"——这是孟浩然殁后最近的一首悼诗（"蔡洲"是襄阳汉水中一片沙洲、近孟浩然旧庐；非汝南郡蔡州。唐宋传本异文有作"蔡州"者，本 doc 从襄阳沙洲一说与襄州 doc 保持口径一致）。

**自浩然殁后至 755**——十五年间、鹿门别业荒废、但孟浩然集已在士人间传抄。**天宝四载（745）王士源辑《孟浩然集》三卷、并作序**——是孟浩然集**早期的编次本与传本序**、亦是 755 前直接现场的史源（非朝廷或官方所修，王士源为孟浩然殁后宜城里人辑录其遗诗成集）。盛唐文人凡过襄阳必访鹿门 ——凭吊浩然故宅、并循其诗回望庞德公"栖隐处"。

### 此刻状态

天宝十四载这一年的鹿门别业——**距孟浩然食鲜疾发卒已十五年、距浩然初隐鹿门已二十七年**。

**别业故址尚可指**——孟浩然后嗣情况史源不详（《新唐书》《旧唐书》本传未及家室具体；本 doc 不取"无子"硬断）、别业身后由族人或同乡门人略事看守；草庐三五间或已倾颓、园圃荒芜；井是否尚存唐人文献未具言、属传说可指而非考古实证。**鹿门别业前传有浩然手植松柏数株犹立**——属襄阳一带的口碑传说、非史源实证；作为 755 时的物理 anchor 是传闻意义上的。**鹿门寺在山主峰南侧依旧香火**——北周建德间立、隋唐递有修葺、755 时为襄阳一带次要佛寺；浩然在世时常往鹿门寺与僧人往还（《题大禹寺义公禅房》一类诗即此境）。

**盛唐人来鹿门凭吊浩然者甚众**。**755 时凡过襄阳的盛唐诗人**——尚在长安的杜甫此时未南下（杜甫南下入襄阳要到上元至大历间）、王维已 56 岁任给事中（仍在长安）、李白 55 岁此时在宣州一带；**这一刻没有第一流诗人正在鹿门**——但鹿门已是盛唐诗坛的一处"圣地"。**王昌龄此时已遭贬龙标（约 748 贬）**——离襄阳千里；浩然故友散于四方。**襄州本地士庶凡识字者皆诵《夜归鹿门歌》**——孟浩然在天宝间已是"故乡新故诗人"的清晰偶像。

**755 时的鹿门** 是**汉江东岸一处供凭吊的诗人故宅 + 山中隐士谱系 anchor**——日常香火、岁岁有襄阳士庶来访；浩然别业故址草庐倾颓、但鹿门月照开烟树的山形未变。再过半年安禄山起兵——但荆襄此时未直接受兵燹影响；**鹿门别业在安史乱中未见毁坏记载、推测未受大规模兵燹**。

### 此刻意味

对天宝十四载的襄阳人来说，鹿门别业是**州城东南二十里那位"十五年前新故"的本地诗人故宅**——日常凭吊、半日可还的去处。对盛唐文人来说——鹿门是**"诗人 + 隐居"的具象**：庞德公举家入山不返、孟浩然退隐故乡作田园诗——这一脉鹿门隐士谱系自汉末贯穿至盛唐，在 755 这一刻是最完整的一刻。

唐人尚不知道——再过五年（760-770s）杜甫流寓蜀地后多次追忆孟浩然、作《忆襄阳孟浩然》《遣兴五首》等；再过百余年（中晚唐）皮日休、陆龟蒙等过襄阳皆有鹿门咏；再过千年——鹿门别业故址、草庐、园圃、孟浩然手植松柏——**物理上的真实遗物一片都没有**；但**鹿门山西麓那片台地的位置一千三百多年不易**——是浩然别业最稳的物理 anchor，亦是盛唐山水田园派的诞生地之一。

## 子条目

### 浩然草庐故址

鹿门山西麓台地——传为孟浩然 728-740 间隐居居处（具体地望今所指为襄阳一带口碑与明清纂志的相承之说、非考古实证）。**盛唐 755 时草庐主体已倾颓**——浩然身后家事无传、十五年风雨之下、原物难存。**今所见鹿门"浩然故居"为明清重修**——清乾隆间襄阳府重修、立"孟亭"以纪。盛唐人在 755 所见的草庐——地望可指、园圃痕传说尚在；但**草庐木构本身或已是浩然殁后第二代或第三代的修补物，井是否尚存唐人文献未具言**。

### 浩然手植松（"诗人手植"）

孟浩然别业前数株松柏——传为浩然手植。**自 728 浩然初隐鹿门至 755**——若属实则树龄约二十七年；皆青年松柏、不及合抱。**今鹿门山西麓尚有古松数株**——树龄不一、最古者据现代植物学测定约八百至千年；**是否上溯至盛唐浩然手植、史源不能定**——本 doc 仅取"传说 anchor"一节、不作物理实证："浩然手植松"在襄阳一带传诵不绝、是诗心而非考古之物。

### 鹿门寺

鹿门山主峰南侧——**北周建德元年（572）由鹿门庙改作鹿门寺**（鹿门庙原祀庞德公；据《襄阳府志》明清纂载，建德间鹿门庙转作佛寺。按：北周武帝宇文邕建德间实行的是排佛废教之政——建德三年/574 诏废佛道二教——故鹿门庙转佛寺一事的政治背景与《襄阳府志》所记年份尚需进一步考订；本 doc 取《襄阳府志》通行一说为口径，附按语备查）。盛唐 755 时为襄州一带次要佛寺、僧十数人。孟浩然在世时常往鹿门寺访僧——《夜归鹿门歌》"山寺钟鸣昼已昏"即鹿门寺钟声；《登鹿门山怀古》《与诸子登岘山》一类诗皆鹿门-襄阳本地行旅之作。**鹿门寺与浩然别业是鹿门山中两处主物事**——相距约一里、由山间小径相连。

### 庞德公 "栖隐处"（鹿门山顶传说址）

鹿门山主峰附近——传为**东汉末庞德公采药不返之处**。《襄阳耆旧记》"庞德公者、襄阳人也……携妻子登鹿门山，因采药不返"——是鹿门山隐居谱系的源头。**盛唐 755 时具名建筑无**——但作为传说 anchor，唐人入鹿门必登主峰、远眺襄阳、追想庞公。孟浩然《夜归鹿门歌》"忽到庞公栖隐处"——是把自己的鹿门隐居与庞德公直接连起来的诗笔。

### 渔梁渡 / 鹿门渡（汉江渡口）

孟浩然《夜归鹿门歌》"山寺钟鸣昼已昏、渔梁渡头争渡喧、人随沙岸向江村、余亦乘舟归鹿门"——一晚之行经**渔梁渡**（汉江中鱼梁洲西端、襄阳东南郊主渡，与襄州 doc 鱼梁洲一节口径一致），泛舟过江而**再归鹿门**。**渔梁渡与鹿门渡是否同一渡、史源异文**——一说渔梁渡即鹿门山西麓近江渡口的别名、一说渔梁渡在鱼梁洲一侧而鹿门渡在东岸鹿门山下、二者为先后两渡。**盛唐 755 时渔梁/鹿门一线渡船数艘、是襄阳东南郊每日往返人流的去处**——浩然在世时往返鹿门-襄阳城十数年皆循此一线。

## 历史长河（755 → 今）

**中晚唐**　安史乱中襄阳为后勤要地、未直接受兵燹大破坏；鹿门别业在乱中保存。**杜甫流寓蜀地后多次追忆孟浩然**——《忆襄阳孟浩然》（760+ 蜀作）"清诗句句尽堪传……重经访旧山、深愧鬼神俱"——是杜甫追忆故友的代表作；**杜甫诸葛诗（《蜀相》《八阵图》《武侯庙》）与杜甫忆浩然诗皆在 755 之后作**——本 doc 严守"杜甫鹿门诗系于 755 之后"口径。**中晚唐皮日休、陆龟蒙、孟郊**等过襄阳皆有鹿门咏——皮日休 (834-883) 自号"鹿门子"、晚年隐鹿门山著《鹿门隐书》，是鹿门作为隐居 anchor 的第二次诗坛高峰。

**两宋**　**北宋大中祥符间**朝廷敕修鹿门寺——这是鹿门寺第一次入官修维护体制。**鹿门别业故址自唐末起渐与鹿门寺合为一片纪念地**——宋人入鹿门凡访浩然故宅、皆借鹿门寺为入口。**南宋初**金兵屡犯襄阳、鹿门略有兵燹；但因襄阳南宋长期为军事重镇、官府重视、鹿门寺与浩然故址得续修。**南宋陆游《入蜀记》记过襄阳访鹿门**——是宋人凭吊鹿门最详细的一笔。

**明清**　明初敕修鹿门寺；**明嘉靖间襄阳府大修鹿门**——浩然故址陆续完善"孟亭""浩然故居""三高祠"（祀庞德公、孟浩然、皮日休三位鹿门隐士）等系列纪念建筑；**今所见鹿门山主体格局多定于明清**。清康熙、雍正、乾隆三朝俱有修缮记录。**清代鹿门已是襄阳第二名胜（仅次于隆中）**——士大夫贬荆湖、贾客过襄阳皆以至鹿门礼浩然为人生一事。

**民国至今**　1949 年后鹿门列湖北省重点文物保护单位、2006 年列全国重点（鹿门寺及孟浩然纪念地一体）。**1990s-2010s 多次考古勘察**——确认鹿门山西麓台地有唐代陶片、夯土遗痕，但**未发现可直接证明"孟浩然原居住所"的物理证据**——这是鹿门考古的实际状态。**今为襄阳市襄州区鹿门山国家森林公园**——孟亭、浩然故居、三高祠、鹿门寺、庞公祠俱为明清重修。**唐人在 755 所见的那座山中诗人故宅**——位置在、传说在、具体物事一片真的也没有；但**鹿门山西麓那片台地一千三百多年未易**——是鹿门别业最稳的物理 anchor，亦是盛唐山水田园派诗心的一处具体来源。
