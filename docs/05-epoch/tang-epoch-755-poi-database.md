## 14. S4 迁移建议

S4 epoch schema 落地时建议：

1. **保留 modern epoch**：`epochs/modern/` 用现代名 (xian/beijing/...)，所有现有 id 不变
2. **建 tang-tianbao-14 epoch**：`epochs/tang-tianbao-14/cities.json` 用 唐名 + 必要时新 id (changan/youzhou/...)
3. **运行时 epoch 切换**：玩家选择 epoch → ContentRuntime 加载对应数据集
4. **不存在的城市**：tang-tianbao-14 不含 (上海/深圳 等不出现)
5. **非 唐辖**：标记为 `politicalAffiliation: "tubo" | "nanzhao" | "bohai"`，UI 上用不同 marker 颜色或开关
6. **历史描述**：tang-tianbao-14 的 description 字段需重写以 唐 755 视角 (避免提及"宋"/"明"/"近代")
